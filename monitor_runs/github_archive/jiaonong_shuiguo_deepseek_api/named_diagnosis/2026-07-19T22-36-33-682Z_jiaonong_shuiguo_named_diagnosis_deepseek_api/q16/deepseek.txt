佳农水果的售后服务整体口碑较好，通常提供“坏果包赔”政策。如果到货发现坏果，建议按以下步骤处理：

1. **及时联系客服**：收到水果后，尽快（通常建议24小时内）检查品质。通过购买平台（如电商APP、官方商城等）联系客服，或拨打佳农官方售后电话。
2. **提供证据**：清晰拍摄坏果照片/视频，并附上快递面单（显示物流信息）。照片需显示坏果数量及损坏情况（如霉变、腐烂、压伤等）。
3. **说明诉求**：根据实际情况说明初步诉求。常见处理方式包括：
   - **按比例赔付**：根据坏果数量占总数比例，退还部分款项。
   - **全额退款**：若坏果严重（如超过一半以上或整体变质）。
   - **补发**：部分情况可协商补发新果（需库存允许）。
4. **等待审核**：客服通常在1-3个工作日内反馈处理结果。保留沟通记录（聊天记录、邮件等）以备后续需要。
5. **注意时效**：避免因超时或未妥善保存（如未冷藏导致二次损伤）影响理赔。

**提醒**：不同渠道（天猫、京东、拼多多等）的具体售后政策可能略有差异，建议优先查看购买页面的“售后规则”或直接咨询店铺客服。生鲜水果运输中少量损坏（如1-2个）属于正常范围，通常可协商部分赔付。
