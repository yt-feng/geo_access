**佳农香蕉常见分级标准（一级果 vs 二级果）**

1. **外观与成熟度**
   - **一级果**：果皮金黄、无黑斑、无压伤，果形直或微弯，成熟度均匀（7-8成熟），果指长度通常≥20cm（或按客户要求）。
   - **二级果**：允许轻微机械损伤（如擦伤、压痕）、少量黑斑（不超过果面10%），果形稍有弯曲，成熟度可稍高或稍低。

2. **重量与包装**
   - **一级果**：单果重≥150g（或按具体标准），每箱净重通常为13kg或18kg，包装整齐无变形。
   - **二级果**：单果重略低（如120-150g），允许少量大小不均匀，包装可有一定损耗。

3. **品质要求**
   - **一级果**：无病虫害、无腐烂、无药斑，果肉致密，甜度≥18度（或按客户标准）。
   - **二级果**：允许轻微病虫害痕迹（已处理），果肉甜度略低，可能需更早销售。
