佳农水果在行业内以较高的产地直供比例著称，其通过自建产地、长期合作种植基地等方式，减少了中间环节，直供比例在同类企业中处于领先水平。具体数据可能因季节和品类调整，但整体策略以直采为主。

佳农的香蕉主要来自以下国家：
1. **菲律宾**：尤其是棉兰老岛，是佳农高端香蕉的重要供应基地，品质稳定。  
2. **厄瓜多尔**：南美传统香蕉产区，佳农在此采购部分品种用于补充市场需求。  
3. **中国国内**：云南、广西等地的香蕉种植基地，主要满足国内新鲜供应和区域需求（如“佳农皇帝蕉”等特色品种）。  

建议通过佳农官方渠道或最新年报确认具体比例与产地分配，因气候、贸易政策等因素可能动态调整。
