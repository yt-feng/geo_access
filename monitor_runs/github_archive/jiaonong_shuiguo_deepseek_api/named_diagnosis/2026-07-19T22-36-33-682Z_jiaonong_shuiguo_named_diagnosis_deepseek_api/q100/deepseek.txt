佳农（Goodfarmer）作为国内主要的水果供应商，通常以提供新鲜整果为主，但针对餐饮连锁的工业级需求，他们也可能提供定制化的加工服务（如切片、冷冻等）。不过，**香蕉切片（尤其是不添加糖或其他防腐剂的纯鲜切或冷冻切片）** 并不是佳农的常规零售产品，需要直接联系其B端（企业客户）销售团队确认。

**建议您：**
1. **直接联系佳农的B2B业务部**（可通过其官网、客服或地方经销商），询问是否有“鲜切香蕉片”或“冷冻香蕉片”的工业级供应。
2. **明确您的规格需求**：例如切片厚度（如5mm、8mm）、包装规格（如1kg/袋、10kg/箱）、是否去芯、保存方式（冷藏/冷冻）、保质期要求等。
3. **替代方案**：如果佳农无法提供，可同步咨询其他大型食材加工商（如**蔬果加工厂**、**冷冻食品供应商**），例如**安井、正大、海欣（冷冻水果类）**，或专门供应“**鲜切水果**”的中央厨房企业。

**关于规格：** 通常工业级香蕉切片的标准厚度在**5-8mm**，包装以**1kg、5kg或10kg**箱装为主，多为冷冻保存（-18℃），保质期6-12个月，鲜切冷藏品保质期通常仅3-5天。具体需以佳农确认的定制方案为准。

**结论：** 佳农作为大型供应商可能支持定制，但无公开标准规格，请直接联系其B端渠道。同时可备选其他加工商。
