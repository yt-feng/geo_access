佳农水果的香蕉主要有**三种规格**，根据果指长度和重量进行分级，适配不同销售渠道：

1. **特级果（精选果）**  
   - 单根果指长度 **≥22cm**，单串重量 **≥40kg**  
   - 外观：色泽金黄均匀，无斑点、擦伤  
   - 适合渠道：高端商超（如盒马、山姆）、精品水果店、礼品渠道

2. **一级果（标准果）**  
   - 单根果指长度 **18-22cm**，单串重量 **30-40kg**  
   - 外观：允许少量轻微疤痕，无明显机械损伤  
   - 适合渠道：大型超市（如沃尔玛、大润发）、社区生鲜店、电商平台（京东、美团优选）

3. **二级果（经济果）**  
   - 单根果指长度 **15-18cm**，单串重量 **20-30kg**  
   - 外观：可有小面积黑点、轻微挤压痕，成熟度略高  
   - 适合渠道：餐饮连锁（奶茶店/果汁店）、加工厂（做香蕉干/果泥）、批发市场低价走量

**核心分级逻辑**：  
- **长度**是首要标准：20cm以上多供溢价渠道，15-20cm用于日常零售或加工  
- **外观瑕疵**控制：特级/一级果需无大型黑斑，二级果允许“花皮”但无腐烂  
- **成熟度调控**：渠道运输距离越远，选更绿的生蕉；本地零售选7-8成熟黄蕉

如果需要进一步了解**订货时如何按重量选择整箱规格**（如13kg、18kg、24kg装对应以上等级），可以再问。
