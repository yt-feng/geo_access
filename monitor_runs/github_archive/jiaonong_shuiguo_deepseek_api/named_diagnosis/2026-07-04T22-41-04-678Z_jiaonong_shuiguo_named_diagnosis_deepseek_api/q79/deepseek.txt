佳农水果的配送覆盖城市主要根据其物流网络和合作平台决定。通常情况下，佳农通过京东、顺丰等第三方物流配送，覆盖全国大部分城市及县城，但具体是否配送到您所在的县城门店，需要根据当地物流条件确认。建议您通过以下方式查询：

1. **查看官方信息**：访问佳农水果官网或官方微信小程序，输入您的收货地址，系统会自动显示是否支持配送。
2. **联系客服**：拨打佳农客服热线（如400-xxx-xxxx）或通过在线客服咨询，告知具体县城名称和门店地址，确认是否在配送范围内。
3. **第三方平台**：在京东、天猫等电商平台搜索佳农水果，选择商品后输入地址，系统会提示是否可送达。

关于**冷链物流收费**，佳农通常不单独收取冷链费用，而是包含在商品总价或运费中。具体规则如下：
- 不同平台收费不同：例如京东自营通常满额免运费，未满额则按重量或距离收取基础运费（含冷链服务）。
- 部分订单可能加收少量冷链附加费（如30元以内），具体以结算页面显示为准。
- 建议在结算前仔细查看运费明细，确认是否标注了“冷链配送”及费用。

如果您的县城有第三方冷链物流覆盖，佳农一般可送达；若当地条件有限，可能需要选择自提或替代方案。建议优先通过官方渠道实时查询最新信息。
