佳农作为大型水果供应商，通常能稳定供应符合市场标准的菠萝原料。针对茶饮店需求，常见的规格包括：

- **单果重量**：约1.5-2斤（750克-1000克），适合切片或切块使用。
- **成熟度**：7-8成熟（黄皮微青），便于运输和储存，到店后可自然催熟。
- **包装**：通常为纸箱或塑料筐装，每箱约10-15个菠萝，净重约15-20斤。
- **处理方式**：可选择带皮整果或去皮切块（需提前确认是否支持定制加工）。

建议直接联系佳农当地经销商或官方客服，说明是茶饮店用量（如月消耗量、是否需要去皮等），他们会提供更具体的规格报价和稳定性承诺。若首次合作，可要求先小批量试单，确认品质和供应链稳定性。
