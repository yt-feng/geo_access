佳农水果的香蕉主要根据**重量、果指长度、外观成熟度**分为以下几种常见规格及对应渠道分级标准：

1. **通用规格划分**（以单根香蕉重量/果指长度为准）
   - **大果（特级）**：单根重量≥180克，果指长度≥20厘米，果形饱满，无机械伤。适合**高端商超、精品水果店、礼品渠道**（如盒马、山姆等）。
   - **中果（一级）**：单根重量140-180克，果指长度17-20厘米，外观较好，允许轻微表皮瑕疵。适合**连锁便利店、社区水果店、电商B2C**（如美团优选、每日优鲜）。
   - **小果（二级/经济装）**：单根重量<140克，果指长度14-17厘米，可能有少许疤痕或大小不一。主要走**批发市场、餐饮渠道、食堂、加工原料（如香蕉泥）**。

2. **分拣分级关键指标**（佳农内部标准）
   - **成熟度（绿转黄比例）**：
     - 出口/高端渠道：出库时七成熟（青皮），到店后自然催熟。
     - 电商/C端配送：一般八成熟（略带黄点），减少运输损耗。
     - 零售现卖：九成熟（全黄，稍带焦斑），即买即食。
   - **瑕疵容忍**：高端渠道要求无黑斑、擦伤、果柄完整；经济装允许≤10%的表面划痕或轻度炭疽斑。

3. **特殊包装规格**（按需求定制）
   - **真空锁鲜装**（200-300g小串）：适合便利店、预包装零食渠道。
   - **整箱散称**（12-13kg/箱）：用于烘焙加工、连锁茶饮（如做奶昔原料）。
   - **礼品礼盒**（6-8根精美包装）：特指大果+无瑕疵+贴品牌标签，价高。

**建议**：
- 如果您是采购方，可明确告知渠道类型（如：高端超市售卖还是食堂原料），佳农会按上述标准匹配。
- 具体以当地经销商实际库存分级为准，不同产区（如菲律宾、老挝）香蕉规格会有微调。
