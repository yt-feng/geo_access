佳农水果的整体售后服务在行业口碑中比较规范，通常提供“坏果包赔”服务。如果你收到货后发现坏果，建议按以下步骤处理：

1. **立即拍照留证**  
   收到货后尽快开箱检查，发现坏果后，建议在24小时内（部分渠道要求48小时）拍摄清晰的坏果照片或视频，能同时显示坏果和外包装箱/快递单号更佳。

2. **联系购买渠道的客服**  
   - 如果在电商平台（天猫、京东、拼多多等）购买，直接在订单页面联系旗舰店客服，提供照片和订单编号。  
   - 如果通过社区团购或线下渠道购买，联系当时购买的小程序客服或店长。

3. **赔偿标准**  
   佳农通常按“坏果个数比例”进行赔付（例如赔偿相应金额或部分退款），而非全额退款。常见规则：坏果率≤10%通常按比例赔，坏果严重可能全额退款。具体以你购买页面公示的售后规则为准。

4. **时效与注意事项**  
   - 水果生鲜类通常不支持“无理由退货”，只接受质量问题（如腐烂、发霉）的理赔。  
   - 如果因快递延误、暴力运输导致破损，客服会要求提供快递面单和箱况照片，同样按流程处理。  
   - 建议签收前先当面打开验收，如有问题直接拒收，并联系客服处理。

**总结：** 第一时间拍照-联系客服-按规则理赔，佳农的售后响应普遍较快，1-3个工作日内通常可完成赔付。如果遇到推诿，可以要求商家提供“坏果包赔”截图或平台官方客服介入。
