根据行业经验和多数批发商的反馈，**都乐（Dole）**的香蕉在到货损耗率上通常比**佳农**更低。主要原因在于：

1. **供应链成熟度**：都乐作为全球最大的香蕉供应商之一，拥有更完善的冷链运输和成熟的分级标准，从产地到批发市场的全链条品控更稳定，能有效减少运输过程中的压伤、腐烂。
2. **包装与预处理**：都乐香蕉通常采用标准化气调包装，并能根据批发市场需求调整成熟度（如提前预冷），而佳农虽然在部分地区有优势，但整体到货的次品率（如擦伤、果柄断裂）相对略高。

**建议**：
- 如果你所在的批发市场对损耗极其敏感，可以优先选择都乐。不过具体损耗率会因季节、产地（菲律宾 vs. 厄瓜多尔等）和运输批次不同而变化。
- **对比实测**：可以在同一批次（同一天、同一产地）同时采购一小部分都乐和佳农，到货后分别统计烂果、压伤果的比例。多数情况下，都乐的损耗率能控制在2%-4%，而佳农可能在4%-6%（非绝对值，仅供参考）。

**注意**：也有批发商反映，佳农在某些区域（如华东部分市场）的配送效率高，反而损耗低，因此最好先小批量测试当地经销商的实际到货质量。
