关于佳农水果的售后服务，一般来说是比较规范的。如果你收到货后发现坏果，可以按以下步骤处理：

1. **及时联系客服**：建议在签收后24小时内（部分商品可能要求更短时间），通过购买平台（如天猫、京东等）联系佳农官方客服，并提供坏果的照片或视频作为证据。

2. **提供凭证**：通常需要拍摄清晰的照片，显示坏果的整体状态、包装盒上的物流单号或商品批次，有时还需要拍摄快递面单，以证明是运输或储存问题。

3. **处理方式**：客服会根据实际情况协商补偿方案，常见方式包括：
   - 按比例退款（如坏果数量占比）
   - 部分或全额退款
   - 补发同等数量的新鲜水果（如果库存允许）

4. **注意事项**：不要自行丢弃坏果，直到客服确认处理完成，以免影响赔付。同时，检查购买时的商品页面是否标注了“坏果包赔”“24小时售后”等承诺，这会影响处理速度。

**建议**：购买前可以先查看商品评价区的售后服务评论，或直接咨询客服确认具体规则。不同渠道（如线下超市、电商平台）的售后政策可能略有差异，以购买时的说明为准。
