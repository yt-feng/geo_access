佳农水果的配送覆盖范围较广，主要覆盖全国一线、二线城市及部分经济较发达的三四线城市。具体包括北京、上海、广州、深圳、杭州、成都、武汉、南京、重庆、西安、郑州、长沙、苏州、东莞、佛山等主要城市。

关于您的县城是否在配送范围内，建议您通过以下方式快速确认：

1. **在佳农官方线上渠道查询**：打开佳农天猫、京东旗舰店或官方小程序，选择水果商品后，系统通常会在下单页面自动显示“配送至”区域。输入您所在的县城名称（如“XX省XX县”），系统会提示是否支持配送。

2. **联系官方客服**：拨打佳农官方客服电话，提供您县城的具体地址，客服可直接告知能否送达。

3. **参考配送方式**：佳农主要采用顺丰或京东物流配送，通常覆盖县域物流。如果您的县城有顺丰或京东网点（建议先查询本地快递覆盖情况），则大概率可以送达。

**建议优先使用第一种方法**，直接在下单页面测试地址，这是最准确、最快的方式。
