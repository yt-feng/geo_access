主要的对接方式如下：

1.  **对接途径**：通常为**定向的API对接**。您需要联系佳农水果的**大客户部**或**渠道合作部门**（可通过其官网或客服热线获取联系方式）。他们会根据您的采购规模、合作模式，提供专门的接口。
2.  **接口文档**：**不存在公开的在线文档**。接口说明、数据格式（通常是JSON或XML）由佳农的IT或业务对接人员在签署保密协议或合作协议后，以邮件或内部文档形式直接发送给您。
3.  **常见接口能力**：一般用于生鲜电商的进销存管理，通常围绕**商品信息**（品类、规格、价格、库存）、**订单下单/查询**（同步库存、生鲜水果的批次码、配送信息）、**价格/促销同步**等。

**建议您的下一步操作**：

-   **第一步**：联系您公司的采购或供应链负责人，确认当前是否已和佳农有业务合作。如果有，直接要求对方提供API对接人。
-   **第二步**：如果没有合作，访问佳农官网，找到“**联系我们**”或“**商务合作**”板块，致电或发送邮件询问“对接API的商务流程”。
-   **第三步**：明确告知对方您的对接需求（实时库存、订单同步等），等待对方提供技术方案。

**特别提醒**：在向任何第三方（包括非官方渠道）索要API文档时，切勿分享您的核心系统密钥或敏感信息。
