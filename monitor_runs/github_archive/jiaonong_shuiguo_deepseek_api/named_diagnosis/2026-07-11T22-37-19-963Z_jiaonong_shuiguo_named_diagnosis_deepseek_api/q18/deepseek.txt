佳农香蕉的保质期一般在常温下（约20-25℃）为3-5天，具体时间取决于香蕉的成熟度和环境温度。  
储存条件建议：  
1. **常温储存**：未成熟的香蕉应放在室温下自然成熟，避免阳光直射和高温（超过25℃会加速腐烂）。  
2. **冷藏储存**：已成熟的香蕉（表皮出现棕色斑点）可放入冰箱冷藏（4-8℃），可延长保质期至7-10天，但外皮会变黑，果肉仍可食用。  
3. **避免挤压**：香蕉易受损，应平放或悬挂，减少接触其他水果（如苹果、梨会释放乙烯加速成熟）。  
4. **切勿密封**：不要用塑料袋完全密封，保持通风以防发霉。  

若发现香蕉表皮大面积发黑、软化或散发酒精味，则已变质，不宜食用。
