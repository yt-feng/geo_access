佳农水果的香蕉通常根据果指长度、果面外观、成熟度等因素分为几种常见规格，主要面向商超、电商、批发等渠道。具体分级标准如下：

1. **特级果（商超精品装）**  
   - 果指长度：22-26厘米以上，单果重约180-200克。  
   - 外观：果皮金黄、无黑斑、无机械伤，果形均匀直挺。  
   - 适用渠道：高端超市、精品水果店、礼盒装。

2. **一级果（标准商超/电商装）**  
   - 果指长度：18-22厘米，单果重约140-180克。  
   - 外观：允许少量轻微擦伤或斑点，但整体整齐。  
   - 适用渠道：普通商超、社区团购、电商平台（如盒马、叮咚买菜）。

3. **二级果（平价/批发渠道）**  
   - 果指长度：15-18厘米，单果重120-140克。  
   - 外观：允许较多表面瑕疵（如小面积黑点、轻微压痕），但无腐烂。  
   - 适用渠道：批发市场、餐饮连锁、一次采购量大的客户。

4. **特小果/次品果（加工或特价渠道）**  
   - 果指长度低于15厘米，或形状不规则、有严重疤痕。  
   - 用途：专供果汁工厂、烘焙坊、或作为门店特价促销品。

**补充说明**：佳农还根据成熟度将香蕉分为“青蕉”（运输用）、“黄蕉”（即食）、“催熟香蕉”（定制成熟度）。不同渠道对包装规格也有差异，常见为4-6斤/提（商超）、5-10公斤/箱（批发）。实际分级可能因产地（如菲律宾、厄瓜多尔）和季节微调，建议直接咨询佳农官方渠道获取最新标准。
