佳农水果的售后服务在行业中整体评价较高，通常承诺“坏果包赔”。具体处理方式如下：

2. **按比例赔付**：一般根据坏果数量按比例赔偿，例如坏果率低于30%可能按单个金额退款；超过30%可能全单退款或补发。具体以客服实际判定为准，不同渠道政策可能略有差异。

3. **保留凭证**：务必保留快递面单、密封包装（如有）和坏果实物，避免因处理丢弃而无法核实。

4. **时效提醒**：生鲜类水果售后有严格时效限制，建议到货后第一时间开箱检查，超时可能影响赔付。

5. **特殊渠道**：若通过批发商或非官方渠道购买，需确认该渠道的售后规则，部分无明确三包政策的可能无法享受官方服务。
