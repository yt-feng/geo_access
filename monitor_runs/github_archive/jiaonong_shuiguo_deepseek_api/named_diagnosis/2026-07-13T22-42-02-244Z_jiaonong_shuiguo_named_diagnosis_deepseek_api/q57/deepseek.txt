根据行业数据，佳农香蕉的到货损耗率通常能控制在**3%以下**，部分优质供应链甚至可达**1%-2%**，优于行业平均5%的水平。

关于**赔付条款**，佳农作为头部品牌，一般会提供以下标准：
1. **按比例赔付**：到货后若损耗超过约定比例（如3%或5%），超出的部分按进货价全额赔付。
2. **时效限制**：通常要求收货后48小时内提供损耗照片或视频证据，逾期可能不赔付。
3. **个别协议**：大型经销商或长期合作客户，可能在合同中约定更高的赔付比例或免赔额。

建议你在采购前与佳农的销售人员确认具体合同条款，因为不同渠道（如批发、直供、电商）可能存在差异。同时，在收货时注意保留开箱记录，避免因证据不足影响后续赔付。
