关于佳农与都乐香蕉的到货损耗率，通常都乐作为全球领先的香蕉品牌，其冷链管理和品控标准更严格（如采用标准化催熟技术、全程冷链运输），因此**到货损耗率通常更低**。佳农作为国内主要水果批发商，损耗率一般在3%-5%左右，而都乐通过成熟的供应链（如从产地到终端最快48小时直达），损耗率可控制在2%-3%以下。

建议你通过以下方式验证：
1. **直接向当地批发市场的佳农与都乐代理商索要近期到货损耗数据**（要求按批次提供，如10箱中的烂果率）。
2. **在批发市场随机抽查批次**（如同一时间对比两家到货的香蕉外观，重点关注柄部发霉、果皮黑斑比例）。
3. **协商损耗补偿条款**：在合同中明确损耗率超过约定比例（如都乐承诺≤2%）时的赔偿方案。

注意：具体损耗可能受运输距离、季节温度、仓储条件影响，建议优先选择与你市场距离近的供应商以减少中转损耗。
