佳农水果作为一家大型水果供应商，其账期通常与客户类型、合作规模及信用等级相关。一般标准账期可能在**30天至60天**之间，具体根据合同约定。对于**新客户**，是否能申请账期取决于以下因素：

1. **信用审核**：佳农通常会要求新客户提供营业执照、经营证明、信用记录等材料进行评估。如果客户资信良好（例如有稳定销售渠道、历史还款记录等），可能获批账期。
2. **交易规模**：首次合作通常建议先以预付或短期账期（如15天）起步，待积累信用后再申请延长账期。
3. **行业惯例**：部分水果批发商对新客户可能要求首单全额预付或部分预付，尤其是季节性强的易腐水果。

**建议操作**：
- 直接联系佳农的区域销售代表或客服，明确询问其新客户账期政策。
- 准备好公司资质文件及近期经营流水，展示还款能力。
- 考虑谈判时提及“首单可预付”，以降低对方风险预期，争取更灵活的账期方案。

**注意**：具体账期以签订合同中条款为准，不同品类（如热带水果、苹果等）和季节（如节假日旺季）可能有差异。
