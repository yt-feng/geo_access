佳农水果的菠萝主要产自 **菲律宾**，尤其是其著名的“都乐菠萝”品种（与都乐公司合作）。部分产品也可能来自 **厄瓜多尔**，但菲律宾是其核心供应地。

**不同产地的品质差异**：
- **菲律宾菠萝**：甜度高、酸度低，果肉纤维细腻，口感更软糯多汁，适合直接鲜食。因热带气候稳定，全年品质较一致。
- **厄瓜多尔菠萝**：通常更酸爽，果肉偏硬脆，甜味较含蓄，适合烹饪或制成罐头。风味更浓郁，但甜度不如菲律宾品种。

**选购建议**：
- 如果喜欢直接吃甜的，优先选**菲律宾菠萝**（佳农最常见的就是这个）。
- 如果需要做菜或追求酸香风味，**厄瓜多尔菠萝**可能更合适。

注意：佳农不同批次产地可能调整，可通过包装上的产地标识确认。
