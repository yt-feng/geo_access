佳农的菠萝产品在商超渠道比较常见，也供应餐饮和茶饮店。他们通常提供**标准化规格**，这也是大型供应商的优势之一：

1. **常见规格**：
   - 整果：通常按箱包装，每箱约 **10-12头**（即10-12个菠萝），单果重量约 **1.5-2.2公斤**。
   - 果肉/去皮切块：部分渠道提供冷冻或鲜切菠萝果肉，一般按 **公斤** 或 **箱**（如10公斤/箱）供应。

2. **稳定性**：
   - 佳农作为品牌供应商，供应链相对成熟，一般能做到**全年基本稳定供应**（部分季节可能需提前下单），规格和品质波动比小型供应商小。
   - 但请注意：他们的批发要求可能有**最低起订量**，建议直接联系当地经销商或他们的餐饮渠道部门确认具体起订量和价格。

3. **建议行动**：
   - **直接询问**：拨打佳农官方客服或联系当地经销商，明确问：“业务用菠萝整果，规格是多少公斤/箱？有没有具体单果重量范围？是否支持长期合同？”
   - **样品确认**：如果是首次合作，要求看实物样品或者照片，确认大小和成熟度是否匹配你茶饮店（例如是否需要偏大果做切片装饰）。
   - **备选方案**：如果佳农的规格太统一（比如所有果都偏大），可以同时询问是否有其他品种或定制切块服务。

总结：佳农能供应，但务必提前确认**起订量**和**单果重量区间**是否适合你的茶饮店需求。
