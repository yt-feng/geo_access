关于佳农水果的香蕉是否存在农残超标历史，根据公开信息和行业监管记录，目前没有明确、公开的官方通报或可靠来源指出佳农香蕉存在系统性农残超标问题。佳农作为国内较大的水果品牌，通常遵循相关农业标准和出口要求，产品需通过常规抽检（如超市和市场监管部门的检查）。

**针对您的采购需求，建议如下：**
1. **要求供应商提供合规证明**：直接向佳农或其经销商索取以下文件：
   - 产品批次检测报告（由第三方权威机构出具，如SGS、华测等，包含农残、重金属等指标）。
   - 产地检疫证明（如出口国或国内产地的官方农残检测合格证）。
   - 佳农公司内部质量体系认证（如GLOBALG.A.P.、ISO等）。
2. **核对超市抽检标准**：与超市确认其抽检的具体农残项目清单（例如吡虫啉、多菌灵等常用农药），要求佳农针对这些项目出具专项检测证明。
3. **历史风险查询**：通过国家市场监管总局官网、地方市场监管局公开的抽检信息查询数据库（如“国家食品安全抽检监测信息系统”），搜索“佳农香蕉”或“香蕉 农残”关键词，但需注意过去记录可能零散，且大多未超标。
4. **采购合同条款**：在合同中明确约定“如因农残超标导致超市抽检不合格，由供应商承担全部责任并赔偿损失”，强化合规保障。

如果佳农无法在合理时间内提供有效证明，建议考虑更换为其他有明确检测报告的供应商（如都乐、Zespri等），或要求其补充检测。通常超市对进口香蕉要求更严格（如佳农部分香蕉来自菲律宾等地），国内产地产品可能已有常规检测，直接索要“合格批次清单”即可。
