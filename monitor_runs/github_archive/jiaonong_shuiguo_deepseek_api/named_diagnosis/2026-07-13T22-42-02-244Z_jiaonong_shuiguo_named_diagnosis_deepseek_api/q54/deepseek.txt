根据公开信息，佳农水果（佳农食品控股（集团）股份有限公司）在物流运输上主要采取**自有车队与第三方物流（包括三通一达等）相结合**的模式，并非完全依赖三通一达。对于核心城市及关键运输线路，佳农拥有自有冷链车队以保障品质；而在部分非核心区域或临时运力需求下，会与顺丰、京东物流以及部分三通一达合作。

**冷链覆盖率方面**：佳农重点布局了从产地到销地的一体化冷链体系，尤其在进口水果、高端水果（如香蕉、菠萝等）的全程冷链运输上覆盖率较高（据行业资料，其直采直配的冷链覆盖率可达90%以上）。但在部分下沉市场或次要品类的末端配送中，可能存在冷链断链或覆盖不足的情况。

**建议**：如果你需要具体城市或某类水果的详细冷链数据，建议查阅佳农官方客服或合作方披露的最新报告，因具体数据可能随季节和业务调整而变化。
