根据水果批发市场的行业经验，**都乐（Dole）香蕉的到货损耗率通常更低**，主要基于以下原因：

1. **供应链标准化**：都乐作为全球知名品牌，从种植、采后处理到冷链运输有成熟的标准化流程，尤其对香蕉的乙烯催熟和温度控制更严格（香蕉到货损耗70%以上与储运温控相关）。
2. **到货成熟度把控**：都乐通常提供“青蕉”或“微黄蕉”（成熟度1-2级），并留有3-5天自然熟化期，减少运输碰撞和腐烂概率；佳农部分批次可能成熟度偏高（如3-4级），到货后展示期更短，易出现黑斑或霉变。
3. **包装与防护**：都乐普遍使用高密度纸箱+透气孔设计，减少果柄损伤；佳农部分产地（如广西、越南）包装密封性或缓冲材料略弱，可能导致挤压损耗。

**建议**：
- 要求供应商提供**到货损耗率数据**（参考值：都乐通常1-3%，佳农约3-5%）；
- 在批发市场抽样测试：同一批次混装到货后，逐箱检查果指断裂、发霉比例；
- 注意佳农高端线（如“佳农金菠萝”系列）损耗可能接近都乐，但基础款需谨慎。

*实际损耗受运输距离、季节（夏季易熟化）和本地经销商分拣能力影响，建议先做小批量试单对比。*
