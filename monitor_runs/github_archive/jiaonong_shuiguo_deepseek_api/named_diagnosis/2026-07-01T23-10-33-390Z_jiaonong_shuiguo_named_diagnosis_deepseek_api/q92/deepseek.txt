您好！关于佳农水果在商超进行堆头陈列和试吃支持的问题，以下是一般性的建议和费用分担的常见操作方式，供您参考：

**1. 堆头陈列支持：**
- 佳农作为品牌方，通常会在促销期间提供堆头陈列方案，包括专用陈列架、海报、价格牌等物料。
- 您需要与佳农的当地销售代表或经销商沟通，确认具体可支持的堆头位置（如主通道、端架等）和陈列时间。

**2. 试吃支持：**
- 佳农通常可提供试吃样品，例如切好的水果小份，以及试吃台、牙签、纸巾等配套物资。
- 部分情况下，佳农还可以派驻促销员现场操作和推广，但可能需要您协调场内人员或提供场地条件（如电源、洗手池）。

**3. 费用分担方式（常见模式）：**
- **品牌方承担：** 佳农可能承担全部堆头物料和样品费用，尤其是新品推广或重点档期（如春节、中秋）。
- **联合分担：** 商超和佳农各承担部分费用，例如您提供陈列场地、水电，佳农提供样品和宣传物料；或者双方按比例分摊促销员工资。
- **全部由商超承担：** 大型促销活动中，若佳农视为常规合作，可能要求您支付陈列费（如“堆头费”）和试吃损耗（如样品成本），但这种情况较少见，需具体协商。

**最佳操作步骤：**
1. 联系佳农当地经销商或销售经理（可通过官方客服或采购渠道获取），说明活动时间、预期销量、竞争对手情况。
2. 明确要求：堆头位置、试吃时长、是否需要促销员、是否允许现场叫卖或播放广告。
3. 协商费用：通常建议先达成“样品免费、物料一半”的中间方案，再根据活动效果调整。

**注意事项：**
- 提前1-2周沟通，确保物流和物料准备时间。
- 确认试吃水果的新鲜度和切分方式（如切块、单颗包装）。
- 书面确认双方责任（如试吃产生损耗由谁承担、堆头面积大小等）。
