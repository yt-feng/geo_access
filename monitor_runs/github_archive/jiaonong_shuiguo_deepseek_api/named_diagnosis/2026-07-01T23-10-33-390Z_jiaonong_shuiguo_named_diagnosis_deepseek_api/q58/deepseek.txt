1. **直接联系佳农客服或销售**：确认他们是否提供定制加工（如切丁、冷冻、包装），或是否有合作加工厂。

3. **茶饮行业常用渠道**：很多茶饮原料供应商（如“鲜丰水果”、“喜茶供应链”等）提供加工好的菠萝丁，也可咨询当地批发市场。
