根据我所了解的信息，**佳农（Goodfarmer）** 主要作为水果供应商，提供整果批发或零售，但**通常不提供去皮切块等深加工服务**（这类服务需要专门的中央厨房或加工厂资质）。

  
**建议您：**  
1. **直接与佳农区域销售确认**：不同地区的经销商服务范围可能不同，部分大型客户可协商定制化服务。  
2. **寻找本地加工商**：联系当地果蔬加工厂或中央厨房，将佳农菠萝采购后交由他们完成去皮切块（需自行承担损耗和成本）。  
3. **使用现成冷冻/预处理菠萝**：如佳农不提供，可考虑其他供应商的冷冻菠萝丁（需调整配方和保质期）。  

**注意**：若选择自行加工，需确保店铺具备食品安全操作规范，避免变质风险。
