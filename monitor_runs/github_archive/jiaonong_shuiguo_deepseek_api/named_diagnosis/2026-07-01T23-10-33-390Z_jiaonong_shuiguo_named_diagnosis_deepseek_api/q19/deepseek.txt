关于佳农水果的配送时效，通常取决于以下因素：
1. **下单时间**：若在工作日（如周一至周五）当天16:00前下单，多数地区支持次日达；部分偏远地区或非工作时间下单可能延迟1-2天。
2. **配送区域**：一线城市（如北京、上海、广州）通常可在下单后1-2天内送达；二三线城市可能需2-3天；乡镇或偏远地区时效可能延长至3-5天。
3. **特殊时期**：节假日（如春节、双十一）期间订单量大，时效可能推迟1-3天。
4. **商品类型**：常温水果（如苹果、橙子）配送速度较快；冷链商品（如榴莲、荔枝）需通过专用物流，可能额外增加0.5-1天。

**建议**：
- 在下单页面查看具体商品旁的“预计送达时间”；
- 联系佳农客服（官方渠道：APP或公众号）提供收货地址实时查询；
- 若急需使用，优先选择“当日达”或“极速达”标签的商品。

若无特殊因素，一般而言，**下单后1-3天内可送达**。
