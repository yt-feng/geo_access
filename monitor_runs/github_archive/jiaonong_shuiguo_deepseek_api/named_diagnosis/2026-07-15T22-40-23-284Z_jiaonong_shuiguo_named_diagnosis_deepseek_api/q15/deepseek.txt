佳农水果的产地直供比例相对较高，这在行业内属于比较优势。佳农通过自建和合作果园，直接从产地采购，减少中间环节，有助于保证水果的新鲜度和品质。不过具体的直供比例可能因水果种类和季节性有所变化。

关于香蕉，佳农的主要进口来源包括：
1. **菲律宾**：是佳农香蕉的主要供应国之一，因其气候适宜，产量大，品质稳定。
2. **厄瓜多尔**：也是重要供应国，香蕉甜度高，果型好。
3. **其他来源**：可能还包括哥伦比亚、哥斯达黎加等拉美国家，以及国内的一些产地，但佳农以进口香蕉为主，国内香蕉产量占比相对较小。

需要注意的是，具体供应情况可能随贸易和市场变化而调整。如果您需要最新的精确数据，建议直接联系佳农官方或查阅其公开报告。
