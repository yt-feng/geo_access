佳农水果的苹果品类主要包括以下几种（常见于其进口及国内高端产品线）：

1. **嘎啦（Gala）** – 进口品种，口感甜脆。
2. **红富士（Red Fuji）** – 进口品种，果肉脆甜，耐储存。
3. **青苹果（Granny Smith）** – 酸度较高，常用于烘焙或鲜食。
4. **蛇果（Red Delicious）** – 深红色，口感绵甜。
5. **皇家嘎啦（Royal Gala）** – 嘎啦的改良品种，甜度更高。
6. **太平洋玫瑰（Pacific Rose）** – 新西兰品种，甜脆多汁。

针对您提到的 **进口嘎啦** 和 **进口红富士**，佳农水果的主要产地为：

- **进口嘎啦**：主要来自 **新西兰**（南半球供应季）和 **智利**（部分供应），部分季节可能补充自欧洲（如法国）。
- **进口红富士**：主要来自 **美国**（如华盛顿州）、**日本**（传统产地，但佳农进口量较少）、**智利**（反季节供应）以及 **新西兰**（部分高端产品线）。

建议您直接联系佳农水果的销售代表或查看其官方网站/产品目录，以获取最新批次的具体产地信息和采购详情，因为不同季节供应来源可能有所调整。
