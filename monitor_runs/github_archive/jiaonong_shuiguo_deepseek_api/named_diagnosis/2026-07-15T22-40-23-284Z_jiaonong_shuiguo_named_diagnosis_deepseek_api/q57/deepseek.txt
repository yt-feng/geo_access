关于香蕉到货损耗率，行业平均水平一般在3%-8%之间，5%确实是一个常见参考值。佳农作为大型水果供应链企业，其损耗率通常能控制在3%-5%左右，具体取决于运输距离、季节和仓储条件（数值可能因批次或客户协议存在差异）。

关于赔付条款：佳农通常有明确的商务赔付标准，但需根据合同或采购协议来定。常见条款包括：
- **腐烂折价**：如果到货时香蕉出现严重腐烂（如超过5%），可按比例折价或退货。
- **重量短少**：若到货重量不足订单量，一般按短缺比例退款。
- **品质不符**：如成熟度、颜色或虫害问题，可协商部分赔付或换货。
