佳农香蕉的保质期通常取决于成熟度和储存温度：

- **常温储存（20-25℃）**：未成熟的青香蕉可存放约 **3-5天** 成熟，成熟后的黄香蕉建议在 **2-3天内** 食用完。
- **冷藏储存（4-8℃）**：成熟后的香蕉可延长保质期至 **5-7天**，但果皮可能变黑，果肉依然可食用。注意：**不可冷藏未成熟的青香蕉**，否则无法正常成熟且可能冻伤。

**储存建议**：
- 将香蕉悬挂或平放，避免挤压。
- 不要和苹果、西红柿等释放乙烯的水果放在一起（除非你想加速催熟）。
- 如果一次买多，可分几根单独存放，或者把香蕉柄用保鲜膜包裹以减少乙烯扩散。

若发现果肉发黑、变软或散发酒精味，说明已变质，不建议食用。
