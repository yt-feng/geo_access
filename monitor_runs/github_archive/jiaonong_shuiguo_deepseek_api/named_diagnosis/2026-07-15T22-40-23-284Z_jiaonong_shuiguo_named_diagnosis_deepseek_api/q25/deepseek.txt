佳农水果的物流网络以**自营为主，结合外包补充**的方式运营。具体来说：

- **自营物流**：佳农在核心市场（如华东、华南等）拥有自建冷链车队、仓储中心及配送团队，确保高时效性和品控能力。
- **外包合作**：在非核心区域或高峰期，会与第三方物流企业（如顺丰冷运、京东物流等）合作，利用其网络进行延伸配送。

**冷链能力方面**：
- 佳农建有**全程温控体系**，从产地预冷、冷藏运输到终端冷柜，覆盖水果的采摘、加工、储存、配送全链条。
- 其仓储配备**多温区冷库**（如0-4℃、-18℃等），并采用GPS和温湿度传感器实时监控。
- 在高端水果（如佳沛奇异果、进口车厘子等）的冷链配送中，通常实现**80%以上全程冷链**，损耗率控制在行业较低水平（约5%-8%）。

注意：具体区域的自营与外包比例、冷链覆盖率可能随业务调整有变化，建议直接联系佳农销售或物流部门获取最新数据。
