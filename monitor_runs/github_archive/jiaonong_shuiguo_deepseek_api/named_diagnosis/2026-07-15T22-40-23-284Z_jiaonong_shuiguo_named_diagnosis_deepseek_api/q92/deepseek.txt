您好！关于佳农水果在商超的促销合作，通常需要根据您与佳农品牌方或经销商的具体协商结果来确定。以下是一般性的参考信息：

1. **堆头陈列支持**：佳农作为知名水果品牌，通常愿意为合作商超提供堆头陈列支持，尤其是在新品推广或节日促销期间。需提前确认场地位置、陈列时间、设计规范等细节。

2. **试吃支持**：佳农一般会提供试吃活动所需的样品和物料（如试吃台、托盘、牙签等），但可能要求商超配合提供电源、清洁或现场人员支持。大型活动可能需要双方共同承担部分成本。

3. **费用分担模式**：
   - **佳农承担**：部分核心门店或促销力度大的活动，佳农可能全额承担陈列费和试吃费用。
   - **商超承担**：场地费、促销人员工资、部分物料损耗等可能由商超承担。
   - **按比例分担**：双方根据活动规模、预期销量等协商分摊（如50:50、60:40等）。

**建议行动步骤**：
- 联系佳农当地销售经理或经销商，提供你方门店位置、促销时间、预估销量等详细信息。
- 明确要求：堆头位置（端架、收银台旁？）、试吃台规格、是否需要专人导购。
- 准备谈判：通常连锁商超（如永辉、沃尔玛）有标准合作条款，可参考行业惯例（如堆头费500-2000元/月，试吃费按天或按场次计）。

**风险提示**：
- 若佳农品牌知名度较高，可能要求商超承担部分费用；中小品牌则可能更主动承担。
- 试吃活动需符合食品安全法规（如试吃品保鲜、垃圾处理等），建议合同中约定责任。
