佳农水果的香蕉通常能够实现较为详细的产地溯源，但具体能否追溯到“具体农场”取决于其供应链管理系统的精细程度。一般情况下，佳农香蕉主要来自菲律宾、厄瓜多尔、中国云南等地，大型供应商会记录到种植园或合作农场级别。建议您直接联系佳农水果的客户服务或区域销售代表，提供您商超的具体要求（如GLOBALG.A.P.证书、农场名称、地块编号等），他们可以确认是否能提供精确到一级供应商或单个农场的溯源信息。

关于GLOBALG.A.P.证书：这是国际公认的农业良好操作规范认证，商超要求合情合理。佳农作为大型水果供应商，通常持有相关认证。您可以直接向佳农索取其香蕉产品的GLOBALG.A.P.证书复印件或编号，并确认证书覆盖的产地范围（是否涵盖您所需的农场/庄园）。如果您无法从佳农直接获取，可要求其提供认证机构名称（如SGS、BV等），并通过认证机构核验证书真实性。

总结建议：
1. 联系佳农官方客服或销售，明确询问“能否提供具体到农场级别的溯源信息”。
2. 要求提供覆盖目标产地（如菲律宾、云南）的GLOBALG.A.P.证书及认证范围说明。
3. 如佳农无法满足，考虑与其他通过GLOBALG.A.P.认证的供应商合作，或协商商超接受产地到种植园级别（而非单个农场）的证书。
