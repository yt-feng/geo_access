你提到的消费者反馈“佳农更甜”确实是许多人的直观感受。以下是两者在口感上的主要区别：

1. **甜度与熟成度**：佳农香蕉通常被认为甜度更高，尤其是在完全成熟（表皮出现芝麻点）后，甜味会更突出。这与其品种选择和后熟处理（如催熟工艺）有关。都乐的香蕉甜度相对温和，更偏向清甜，口感上不会过于浓郁。

2. **质地与细腻度**：佳农香蕉果肉更软糯、绵密，甚至接近“糯性”，咀嚼时几乎没有纤维感。都乐的香蕉则相对紧实，果肉更脆爽一些，尤其是未完全成熟时，口感更偏硬脆。

3. **香气与酸度**：佳农的香味更浓郁，带有一点热带水果的复合香气，几乎无酸味。都乐的香气较清新，且部分消费者反映在成熟度较高时，可能略带一丝微酸，这反而中和了甜腻感。

4. **熟成窗口期**：佳农的甜度高峰期较短，一旦过熟（表皮完全变黑）会迅速变软、甜度下降并产生发酵味。都乐的甜度保持时间较长，即使在表皮出现较多黑点后，果肉依然能保持较好的质地和风味。

建议：如果你偏好高甜、软糯的香蕉，佳农可能更符合你。如果喜欢清爽、略带弹性、不易过熟的口感，都乐会更稳定。下次购买时，可以注意观察香蕉的成熟度——佳农建议买略带青色的回家自然催熟，都乐则适合直接吃或放几天再吃。
