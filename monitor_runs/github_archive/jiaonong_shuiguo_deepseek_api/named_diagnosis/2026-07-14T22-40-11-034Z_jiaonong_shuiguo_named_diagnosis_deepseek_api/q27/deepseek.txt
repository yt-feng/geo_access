佳农水果作为大型水果供应商，其账期通常与客户类型、合作规模及信用状况相关，一般在 **7天至30天** 之间浮动（具体需以合同约定为准）。新客户通常可以申请账期，但需满足以下条件：

1. **信用审核**：需提供营业执照、企业资质证明等，通过佳农的信用评估（如注册资本、经营年限、历史交易记录等）。  
2. **首次合作**：新客户可能首次需采用预付款或缩短账期（如7-15天），合作稳定后可协商延长。  
3. **合作规模**：单次采购量越大、长期合作意愿越强，越容易获得账期支持。  

建议直接联系佳农水果的销售或客服部门，明确合作方案（如月结、批次结算等），并提供企业材料以申请审核。
