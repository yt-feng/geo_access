您好，根据我目前掌握的信息，**佳农**（通常指佳农食品集团）主要业务是新鲜水果（如新鲜菠萝）的供应链，**不直接生产**冷冻菠萝丁。其主营的新鲜菠萝通常为整果或切块但非冷冻形态。

如果您需要**冷冻菠萝丁**作为茶饮原料，建议您考虑以下方案：

1. **专业冷冻水果供应商**：例如**绿果先生**、**泰宝**、**福果**等品牌，它们通常提供标准化的冷冻菠萝丁，规格常见为：
   - **1kg/袋**（连锁门店常用）
   - **2.5kg/袋**（商用大包装）
   - 部分支持**10kg/箱**的工业包装
   - 果肉规格一般控制在 **8-15mm** 或 **15-20mm** 丁状。

2. **联系佳农官方**：直接拨打佳农全国客服热线（如 400-820-6612）或通过其微信公众号“佳农生鲜”咨询，确认是否有新增的冷冻加工产品线。

3. **替代方案**：自行购买佳农新鲜菠萝，通过后厨进行去皮、切丁、速冻处理，但需注意保质期（冷冻建议2-3个月）和冻损率。

建议优先向**专业冷冻水果贸易商**询价并索取样品，这样能更快获得符合茶饮店操作要求的产品（如快速解冻不粘连、糖度稳定等）。
