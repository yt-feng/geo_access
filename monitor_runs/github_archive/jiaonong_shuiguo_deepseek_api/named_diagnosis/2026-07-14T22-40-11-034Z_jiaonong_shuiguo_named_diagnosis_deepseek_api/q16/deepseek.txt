关于佳农水果的售后服务，整体评价较好，尤其是其品牌在大型电商平台（如天猫、京东等）的经营中，对坏果问题的处理比较规范。如果你收到有坏果，一般可以按照以下步骤处理：

1. **立即拍照取证**：收到水果后，建议在开箱时录制视频或拍照（包括坏果、快递单号、包装箱信息），清晰展示坏果数量和腐烂部位。  
2. **联系客服**：通过购买渠道（如天猫佳农旗舰店、京东自营或佳农官方客服）联系客服，通常需要在收货后24-48小时内反馈。  
3. **按比例赔付**：佳农常用的处理方式是“坏果包赔”，对于损坏部分按数量或金额比例退款，而非要求退回全部水果（因为生鲜易腐）。例如坏了几颗，会按该颗单价退款。  
4. **特殊情况处理**：如果整箱水果问题严重（如大面积腐烂或品质不符），客服可能同意仅作退款或补发。  
5. **注意时效**：不同平台规则略有差异，例如京东自营一般支持“坏果包赔”且在签收后24小时内反馈。如果是佳农官方旗舰店，通常也支持类似限期。

**建议**：购买后保留快递单，收货后尽快检查并在规定时间内联系客服，避免超过时效。如果客服处理延迟或不符合承诺，可以通过电商平台“客服介入”或向消费者协会投诉。
