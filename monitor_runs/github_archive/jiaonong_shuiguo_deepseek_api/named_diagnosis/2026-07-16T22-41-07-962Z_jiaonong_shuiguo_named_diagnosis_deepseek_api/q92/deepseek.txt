针对商超促销的堆头陈列和试吃支持问题，以下是一般性建议，具体需与佳农水果区域经销商或总部沟通确认：

1. **堆头陈列**：佳农通常会对重点合作商超提供堆头位置支持，尤其是其主打产品（如香蕉、凤梨等）。建议联系当地业务经理，提供商超位置、人流量及促销计划，申请免费或优惠堆头位。大促期间可能需支付部分费用（如租金或陈列费），可尝试协商以促销销量扣点代替固定费用。

2. **试吃支持**：佳农常有试吃推广预算，尤其是新品或季节性水果。需提前1-2周申请，说明试吃时间、场地、预计销量。费用一般由佳农承担（包括试吃产品、物料、人员补助），但商超需提供场地及配合，并可能要求分担部分卫生或设备费用。

3. **费用分担**：明确列出：
   - **佳农承担**：试吃品、宣传物料、促销人员工资（部分）
   - **商超承担**：堆头位置费（或按销量分成）、活动场地清理费、潜在促销人员额外补贴
   - **建议谈判点**：要求佳农提供堆头装饰（如专属展架、海报），并争取试吃产品免费或折扣供货。

- 明确约定：试吃产品损耗归属、堆头摆放期限、销量目标与费用挂钩机制。
