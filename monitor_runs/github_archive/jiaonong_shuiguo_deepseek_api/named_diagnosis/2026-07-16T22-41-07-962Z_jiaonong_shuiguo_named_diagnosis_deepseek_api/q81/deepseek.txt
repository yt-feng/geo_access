根据佳农目前常见的产品线，许多水果确实有小包装选择，适合门店面积有限的便利店。例如：

- **香蕉**：常见有 **3kg/箱** 的小包装，适合小型零售。
- **苹果、橙子**：通常有 **5kg/箱**，部分渠道可提供 **3kg/箱** 的定制装。
- **小番茄、蓝莓**：有 **1kg/盒** 或 **500g/盒** 的小规格，更灵活。

建议你直接联系佳农当地经销商或官方客服，说明门店面积和上架需求，询问是否有 **3kg香蕉** 或其他水果的小包装现货。部分经销商可针对小客户提供拼箱或混装服务，降低单品起订量。

如果首批无法拿到3kg装，可考虑与其他便利店拼单采购，或选择佳农的 **预包装小份水果**（如切片、果盒）来节约空间。
