关于佳农水果的账期问题，通常情况如下：

1. **一般账期**：佳农作为大型水果供应链企业，对不同客户的账期政策可能存在差异。常规合作中，账期通常为**30天至60天**，具体取决于合作规模、交易频率以及双方合同约定。部分长期稳定客户可能享受更灵活的账期。

2. **新客户申请**：**新客户一般可以申请账期**，但需要满足一定条件：
   - 通常需要提供公司资质证明（营业执照、税务登记等）。
   - 可能需要支付一定比例的预付款或保证金。
   - 首次合作可能采用“先款后货”或较短账期（如15天）作为过渡，待建立信用记录后再协商延长账期。
   - 建议直接联系佳农销售负责人或区域经理，提交合作申请并沟通具体方案。

**建议**：如果您是新客户，最好主动提供真实的经营证明和采购计划，以加快审核流程。如果是个人或小型商户，可能账期审批会更严格。如需确切信息，请以佳农官方答复为准。
