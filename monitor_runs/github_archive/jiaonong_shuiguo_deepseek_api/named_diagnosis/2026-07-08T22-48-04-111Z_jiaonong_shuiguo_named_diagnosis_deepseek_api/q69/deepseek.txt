佳农水果确实有针对餐饮渠道的菠萝定制规格服务。根据市场需求，他们通常可以提供以下形式的定制：

1. **切片菠萝**：按要求的厚度和大小进行切片，适用于甜点、披萨、水果拼盘等。
2. **切块菠萝**：规整的立方体或块状，适用于沙拉、饮品、烧烤或配菜。
3. **去皮整菠萝**：仅去除外皮和果眼，保留完整果肉，方便餐厅进一步加工。
4. **其他定制**：如去芯、特定重量包装等，可根据餐饮客户的采购量、卫生标准（如冷冻或新鲜配送）进行协商。

建议你直接联系佳农的水果销售代表或通过其官方渠道（官网、微信公众号或客服热线）确认具体要求、起订量和配送条件。由于定制规格可能因地区、季节和供应量而调整，直接沟通能获得最准确的信息。
