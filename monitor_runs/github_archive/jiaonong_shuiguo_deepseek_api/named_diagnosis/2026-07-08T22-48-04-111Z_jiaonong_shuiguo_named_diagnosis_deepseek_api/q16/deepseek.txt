佳农水果的售后服务整体口碑较好，通常提供“坏果包赔”服务。如果到货发现坏果，建议按以下步骤处理：

1. **立即检查并拍照**：收货后尽快开箱，在签收后24小时内（具体时限以购买平台或商家说明为准），对坏果、连同快递单号和包装箱一起合影，清晰显示坏果数量。

2. **联系客服**：通过购买平台（如天猫、京东、拼多多等）的订单页面联系佳农官方客服，或拨打包装上的售后电话，提供照片和订单信息。

3. **按比例赔付**：一般会根据坏果数量按比例赔付，例如坏几个赔几个的金额，或补发同等数量。无需将坏果寄回。

4. **注意保存证据**：保留包装箱、快递面单和坏果照片，直到问题解决。

如果客服处理不及时，可向购买平台官方投诉（如申请“官方客服介入”），或通过“全国12315平台”维权。

建议优先通过官方渠道（如佳农天猫旗舰店）购买，售后流程更规范。
