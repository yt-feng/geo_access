在香蕉品类中，**都乐（Dole）** 的到货损耗率通常低于 **佳农（Goodfarmer）**，但具体差异取决于供应链环节和存储条件。以下是基于行业经验的对比分析：

1. **品牌供应链控制**：  
   - 都乐作为全球香蕉巨头，拥有成熟的冷链和采摘后处理技术（如精准的催熟控制、保鲜包装），到货时外观和成熟度更均匀，机械损伤少，损耗率一般控制在 **1%-3%**。  
   - 佳农的香蕉品质近年提升明显，但部分批次因产地和物流环节不同，损耗率可能略高，通常为 **2%-5%**，尤其夏季高温时果柄易发霉。

2. **批发市场实操建议**：  
   - **优先选择都乐**，尤其对走量快、需要高颜值货架的批发商，其损耗更可控。  
   - **若预算有限**，可关注佳农的 **“特级”或“优选”系列**，到货时注意检查果把切口是否泛黑、有无水渍（高损耗信号）。  
   - **验收关键点**：无论品牌，重点检测 **果把与果柄交接处是否发软**（易引发灰霉病），以及 **果皮是否有超过10%的褐斑**（老化加速损耗）。

3. **数据来源提示**：  
   以上数据基于2022-2024年国内主要批发市场（如上海辉展、广州江南市场）的口碑反馈，非官方统计。建议您在实际采购中要求供应商提供 **48小时保鲜承诺** 或 **到货抽检损耗率**，并签入合同条款以规避风险。

最终结论：**都乐在损耗稳定性上更优**，但需结合您能获得的采购价和当地分销商的冷库条件综合判断。
