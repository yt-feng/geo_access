您好！根据佳农水果常用的产品规格，目前市面上确实有小包装选项，但**3kg装香蕉**通常属于中大型批发或家庭装（常见5kg或10kg），小型便利店更推荐以下规格：

- **1-1.5kg装香蕉（散装或透明袋）**：适合小面积门店，方便顾客即买即走，周转快。
- **前置仓定制小包装**：部分佳农经销商提供**2-2.5kg礼盒或密封袋装**（个别区域有3kg装，建议直接向当地经销商确认）。
- **其他水果小包装**：如佳农蓝莓（125g/盒）、牛油果（保鲜袋装）、玉米（独立包装）等，占地面积小。

**建议动作**：
1. 联系**佳农当地经销商**或**批发市场直供渠道**，询问最小起订量及可定制的小包装（如3kg是否可拆分）。
2. 若货源紧张，可考虑**自备小包装袋**，将大包装香蕉拆分为1-1.5kg份，贴上佳农品牌标识（需取得授权）。

小面积便利店优先选择**高周转、耐储存**的小包装品类，避免积压。
