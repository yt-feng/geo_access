**关于下沉市场县城次日达问题：**

佳农的配送体系主要以**中心仓+前置仓/合作商**模式运作。对于下沉市场（县城）能否实现次日达，主要取决于以下因素：

1. **是否有合作经销商或前置仓**：若该县城有佳农的授权经销商或合作水果店、超市，且该县城距离上述配送中心较近（通常200公里以内），则**可能实现次日达**（通过经销商补货）。
2. **距离与时效**：若县城距离配送中心超过300公里，且无区域中转仓，通常为**2-3天达**（通过干线物流+末端配送）。
3. **电商平台合作**：如果通过京东、天猫、美团优选等第三方平台下单，部分平台在县城可能有次日达服务（需具体查看该县城是否在平台“次日达”覆盖范围）。

**建议操作：**
- 您可以告知具体县城名称，或联系佳农官方客服（400-882-8828）查询当地是否覆盖次日达服务。
- 或通过京东、天猫等平台的佳农旗舰店，输入收货地址测试“预计送达时间”，以获取实时信息。

**注意**：下沉市场县城的冷链配送稳定性可能低于一二线城市，极端天气或物流旺季（如春节）可能延迟。如果您是做水果批发或团购，建议提前确认当地经销商库存。
