佳农水果的香蕉主要分为以下常见规格与分级标准，适用于不同销售渠道：

**1. 常见规格**
- **小果（5-6根/斤）**：单根重量约80-100克，适合便利店、散装零售或小份包装。
- **中果（4-5根/斤）**：单根重量约100-140克，商超、生鲜电商主流规格，兼顾卖相与性价比。
- **大果（3-4根/斤）**：单根重量约140-180克，多用于精品超市、高端水果店或礼盒装。
- **超大果（2-3根/斤）**：单根重量180克以上，常见于节日礼盒、高端渠道。

**2. 分级标准（按渠道需求）**
- **商超/生鲜电商（如盒马、叮咚买菜）**：要求果实饱满、色泽鲜黄均匀、无明显伤痕或黑点，成熟度控制在7-8成（青黄皮），便于运输和货架期管理。通常选用中果或大果。
- **批发市场/餐饮业**：侧重性价比，允许果实略有大小不一或轻微瑕疵（如极少的黑斑），成熟度可略低（6-7成青皮），适合集中采购并自行催熟。常用小果或中果。
- **精品超市/高端渠道（如山姆、Ole'）**：严格筛选外观无瑕疵、果形端正、单果重量均匀的大果或超大果，成熟度控制在8-9成（黄皮带点青），并常附带独立包装或托盘。
- **电商社区团购（如美团优选、多多买菜）**：需兼顾运输抗挤压性和到家后的即食性。多采用中果，单根独立泡沫网套包裹，成熟度控制在7-8成（青黄皮），并减少失水率。

**3. 产区与季节影响**
佳农主要供应厄瓜多尔、菲律宾等进口香蕉，以及国内海南、广西产区。进口果果形更细长，适合中果及以上规格；国产果偏短粗，常作小果或中果。夏季催熟加快，可适度降低成熟度标准；冬季保温和运输成本高，规格更倾向于中大果。

如需特定渠道的详细分级文件（如大小分级表、成熟度色卡标准），建议联系佳农官方区域销售代表获取最新版《香蕉品质执行手册》。
