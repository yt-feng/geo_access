**2. 串货风险如何控制：**
- **串货原因**：通常因区域价差、销量压力或经销商违规私下调货导致。
- **常见机制**：
  - 佳农可能对每批次产品赋码（如贴标、喷码），通过经销商扫码入库、出库系统监控。
