根据公开的监管信息和国家市场监管总局的抽检通报，**截至目前，佳农（Goodfarmer）品牌的香蕉在主流渠道（如商超、电商平台）的官方抽检中未发现农残超标的历史记录**。佳农作为国内头部水果企业，其香蕉产品通常通过全球良好农业规范（GlobalG.A.P.）、中国良好农业规范（ChinaGAP）认证，并且定期接受第三方检测。

为确保采购合规，建议您采取以下步骤：

1. **索要合规证明文件**：直接要求佳农供应商提供近期的**第三方检测报告**（如SGS、华测检测等出具），重点查看**农药残留**（如吡虫啉、苯醚甲环唑、丙环唑等香蕉常用药）是否符合GB 2763《食品安全国家标准 食品中农药最大残留限量》要求。

2. **核查供应商资质**：要求佳农或其经销商提供**营业执照、食品经营许可证**，以及每批次的**进货台账和出厂检验合格证明**。

3. **关注官方抽检动态**：可登录**国家市场监督管理总局**官网，搜索“香蕉 抽检 佳农”关键词，或查询各省市市场监管局的月度/季度食品安全抽检通报，确认有无异常。
