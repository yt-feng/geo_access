根据佳农通常的业务模式，佳农主要是一家果蔬供应商，提供新鲜水果（如菠萝）的批发和零售，但一般不提供去皮的加工服务。佳农的产品通常是整果销售。

**建议方向：**
1. **联系佳农官方客服**：直接咨询是否有加工定制业务（部分大型客户或特殊订单可能支持）。
2. **自行处理方案**：
   - 店内配备去皮切块设备，由员工操作。
   - 寻找本地水果加工厂、中央厨房或第三方果蔬加工企业，洽谈去皮切块的代加工服务，但需注意食品安全和保质期。
3. **替代原料**：如无法处理，可考虑采购市售的预切菠萝块（冷冻或冷藏），但需确认供应商是否符合茶饮店的原料标准。

**建议优先执行第1步**：直接联系佳农的客服或销售负责人，明确需求（如批量、规格、区域），获取最准确的答复。
