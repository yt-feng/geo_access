佳农水果的售后服务整体评价较好，通常采用“按坏果比例赔付”或“坏果包赔”政策。如果到货有坏果，建议按以下步骤处理：

1. **立即拍照留证**：在收货后24小时内（部分政策为48小时），拍摄坏果的清晰照片或视频，同时拍下快递单号和包装箱上的信息。
2. **联系客服**：通过购买平台（如天猫、京东、拼多多等）的订单页面联系佳农官方客服，或拨打包装上的售后电话。说明情况并上传证据。
3. **确认赔付方案**：客服通常根据坏果数量或重量（如每箱坏果比例）进行赔付，常见方式包括：
   - 按比例退款（如坏果占整箱的30%，退还30%货款）。
   - 部分情况下直接补偿红包或优惠券。
   - 少量坏果（1-2个）可能直接赠送或减免。
4. **避免争议**：确保在时效内联系，保留好证据。不要随意丢弃坏果和包装，避免客服要求核实。

**注意事项**：
- 佳农对生鲜水果一般不支持无理由退货（除非整箱腐坏严重），仅接受按坏果比例处理。
- 如果客服拖延或拒绝处理，可通过平台官方投诉渠道或12315维权。

建议收货时立即开箱检查，尤其是夏季或长途运输的水果。
