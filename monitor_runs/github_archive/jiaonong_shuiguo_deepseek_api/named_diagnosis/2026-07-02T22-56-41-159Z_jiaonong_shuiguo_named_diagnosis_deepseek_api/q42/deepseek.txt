根据行业经验和市场普遍反馈，**都乐（Dole）香蕉的到货损耗率通常低于佳农（Goodfarmer）**。原因如下：

1. **供应链成熟度**：都乐作为全球香蕉巨头，在中国的冷链物流、采后处理及仓储标准更严格，从产地到批发市场的品控一致性更高，机械损伤和腐烂率更低。
2. **包装与运输**：都乐香蕉普遍采用更厚实的纸箱和更精细的缓冲材料，减少运输磕碰；佳农部分批次在包装强度上略有不足。
3. **成熟度控制**：都乐的乙烯催熟技术更标准化，到货时绿熟均匀，批发市场卸货后黄熟期更可控，损耗窗口期更长。

**但需注意**：具体损耗率还取决于：
- **产地与季节**：菲律宾、厄瓜多尔等产地的都乐香蕉稳定性高；国产或越南产的佳农香蕉在雨季到货损耗可能上升。
- **批发商自身操作**：卸货速度、存放温度、销售周期（如是否快速分销）都会影响实际损耗。

**建议**：若追求稳定低损耗，优先选都乐；若价格差异大且能快速出货，可小批量试单佳农并对比实际开箱损耗数据（建议每月统计3-5次，计算平均损耗率）。
