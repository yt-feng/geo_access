佳农主要是一家以新鲜果蔬（尤其是香蕉、菠萝等热带水果）的种植、采购和销售为主的企业，通常提供的是整果产品（如整颗菠萝），而非直接针对茶饮店的预制加工原料（如切好的菠萝丁）。

如果需要菠萝丁原料，建议你：
1. **联系佳农官方或当地经销商**：确认其是否单独提供去芯、切丁等定制加工服务，不同区域的业务范围可能有差异。
2. **考虑专业餐饮供应链企业**：这类企业通常提供标准化切丁、速冻或罐头菠萝丁，如“田野股份”、“安德利”或本地果切加工厂，更符合茶饮店批量使用的需求。
3. **自行加工**：如果用量不大且要求新鲜，可购买佳农整果后自行切丁，但需注意人工成本和出成率。

建议优先直接致电佳农客服或查询其官网/合作平台（如美菜网、快驴进货）上是否有“加工菠萝丁”商品，以获得准确答复。
