根据行业公开信息，香蕉的到货损耗率行业平均水平约为3%-5%，佳农作为头部供应商，通常能将到货损耗率控制在**2%以内**（具体指标需以合同为准）。部分合作案例显示，其损耗率可低至1.5%左右，但需结合运输距离、仓储条件等因素调整。

关于赔付条款：  
1. **行业常见做法**：多数供应商（包括佳农）会在合同中约定损耗率阈值（如3%），超出部分按比例赔付（通常为零售价或采购价的1-1.5倍）。  
2. **佳农常规政策**：  
   - 若到货损耗率超过约定标准（如2%），超出的部分会根据合同条款进行赔付（例如，按超出数量×单价补偿）。  
   - 高端渠道（如大型商超、电商平台）可能提供更严格的零容忍赔付方案（如发现单根腐烂即换货）。  
3. **注意事项**：具体赔付条款需与佳农销售或合同确认，例如：  
   - 是否包含“运输途中不可抗力”的免责项；  
   - 是否需要提交到货时的第三方质检报告或现场照片。  

建议：签订合同时明确写明损耗率上限及赔付计算方式，并保留到货时的完整证据链（如装箱单、质检记录、实物照片）。
