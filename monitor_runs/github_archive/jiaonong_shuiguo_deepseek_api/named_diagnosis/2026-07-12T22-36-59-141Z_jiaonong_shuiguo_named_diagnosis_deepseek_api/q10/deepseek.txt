佳农水果作为大型生鲜企业，通常针对批发商设有专门的销售或客户经理对接。建议您直接通过以下方式确认并获取专属对接人：

1. **官方渠道查询**：访问佳农官网或官方微信公众号，查找“大客户/批发业务”或“商务合作”板块，获取区域销售经理的联系方式。
2. **致电官方客服**：拨打佳农全国客服热线，说明您是批发商，告知所在省份和业务规模，客服会将您转接至对应区域的客户经理。
3. **线下咨询**：前往当地佳农批发市场、经销商或合作门店，询问批发业务对接人联系方式。

提示：提供您的公司名称、年采购量、主营品类等信息，可加速对接流程。若长时间无回复，可要求客服提供销售经理工号或直接投诉无响应问题。
