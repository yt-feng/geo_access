佳农水果的苹果品类主要包括以下几种常见品种：

- **嘎啦（Gala）**  
- **红富士（Fuji）**  
- **青苹果（如澳洲青苹）**  
- **其他进口品种（如蛇果、粉红女士等）**

针对您想采购的**进口嘎啦**和**红富士**，它们的主要产地如下：

- **进口嘎啦**：主要产地包括**新西兰**、**美国（华盛顿州）**、**智利**等。其中，智利嘎啦通常供应南半球反季市场（如每年10月至次年4月）。
- **进口红富士**：主要产地包括**日本**（原产地）、**美国（华盛顿州）**、**新西兰**、**智利**等。中国市场常见的进口红富士多来自**美国**（如华盛顿州苹果）和**新西兰**（如皇家嘎啦或富士混种）。

建议您直接联系佳农水果的销售代表，确认当前可供应的具体批次、产地及采购流程。如果您需要联系方式或其他帮助，请告知。
