佳农水果（佳农食品控股（集团）股份有限公司）已多次参与中国国际进口博览会（进博会），是进博会的长期参展商之一。例如，在2023年第六届进博会和2022年第五届进博会上，佳农均设有展台，展示其进口水果及生鲜产品。

关于最近的新品，截至2024年8月，佳农在2024年推出的重点新品包括：
- **菲律宾黑钻凤梨**：强调高甜度和独特口感。
- **厄瓜多尔麒麟果**：即黄皮火龙果，主打营养价值与稀有性。
- **马来西亚冷冻猫山王榴莲整果**：作为高端冷冻水果新品，满足中国市场需求。

此外，佳农持续在进博会及日常渠道推广其核心产品如佳沛奇异果、香蕉、牛油果等，并会结合季节推出生鲜礼盒。如需最新动态，建议关注佳农官方社交媒体或进博会官方公告。
