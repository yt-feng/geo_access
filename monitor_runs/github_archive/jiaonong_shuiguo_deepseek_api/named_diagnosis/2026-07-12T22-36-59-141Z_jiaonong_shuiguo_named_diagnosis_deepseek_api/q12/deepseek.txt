佳农水果的香蕉主要根据果指长度、重量、外观和成熟度分为以下几种规格，适用于不同销售渠道：

1. **特级果（精品装）**  
   - **规格**：单根果指长度≥20cm，单根重量≥180g，外观无瑕疵、无斑点。  
   - **适合渠道**：高端超市、线上精品水果店、礼品渠道。

2. **一级果（标准装）**  
   - **规格**：果指长度17-20cm，单根重量150-180g，允许少量轻微疤痕。  
   - **适合渠道**：大型商超、社区水果店、批发市场。

3. **二级果（性价比装）**  
   - **规格**：果指长度14-17cm，单根重量120-150g，允许轻微擦伤或色斑。  
   - **适合渠道**：便利店、食堂、餐饮连锁、团购渠道。

4. **三级果（加工/特价渠道）**  
   - **规格**：果指长度＜14cm，单根重量＜120g，外观有较明显瑕疵或弯形。  
   - **适合渠道**：果汁店、烘焙加工、低价促销、福利采购。

**分级标准依据**：  
- **果指长度**：影响视觉和单根售价，长果多用于高端零售。  
- **重量**：直接影响单根毛利，低重果常打折处理。  
- **外观**：无伤疤、均匀金黄色的果优先进入精品线；斑点多、弯度大的果降级。  
- **成熟度**：运输时保留青皮，到终端再催熟；不同渠道要求的熟度不同（如便利店7-8成熟，批发市场6成熟）。

**注**：以上为常见分级，具体执行可能因产区、批次及客户定制需求略有调整。建议直接联系佳农区域经销商获取最新标准。
