佳农水果的配送时效通常取决于您所在的地区和下单的具体平台。一般来说：
- **标准配送**：大部分城市在**下单后1-3天**内送达，部分偏远地区可能需要**3-5天**。
- **同城/次日达**：如果您位于一线或新一线城市（如上海、北京、广州等），通过其官网或京东、天猫等合作平台购买，部分商品支持**当日或次日达**（需在指定时间前下单）。
- **特殊情况**：节假日或促销期间，配送时长可能延长1-2天。

  
**建议操作**：  
1. 下单前查看商品详情页的“预计送达时间”或“配送区域说明”。  
2. 选择“立即购买”后，结算页面会显示具体送达日期。  
3. 如需精确到小时，可直接咨询佳农官方客服（如天猫旗舰店在线客服）或拨打配送服务热线。  

**注意**：生鲜水果受天气、库存影响，实际时效可能略有浮动。
