佳农作为大型食品供应商，通常能稳定供应适合茶饮店使用的菠萝原料，以下是一些关键点供参考：

1. **常见规格**：佳农的菠萝一般以“菠萝切块（冷冻/罐装）”或“新鲜菠萝（已去皮切块）”形式供应，常见规格为：
   - **冷冻菠萝丁**：如1kg/袋或20kg/箱，适合长期储存、饮料机直接使用。
   - **新鲜菠萝果肉**：按“箱”为单位，每箱约10-15个整菠萝（可要求加工成切块）。

2. **适合茶饮的规格**：
   - 如果做水果茶或果酱，建议选用“冷冻菠萝丁（无糖或稍加糖浆）”，直接解冻即可使用，损耗低。
   - 如需新鲜菠萝，可要求供应商提供“去皮切块（1cm左右立方体）”，避免自己切耗时且浪费。

3. **稳定供应能力**：佳农有自有种植基地和冷链物流，一般能保证：
   - 全年供应（热带产地可调度）。
   - 符合食品安全标准（如无农药残留、符合出口级品质）。

4. **注意事项**：
   - 具体规格需直接联系佳农官方客服或区域经销商，明确说明“茶饮店用量（如月需求量、是否需小包装）”。
   - 若需求量大（比如月>100kg），可协商定制规格（如甜度、切块大小、包装方式）。

总结：佳农能提供茶饮店常用的菠萝原料，但小店铺建议优先考虑“冷冻菠萝丁（1kg装）”或“罐装菠萝块”，方便且稳定。建议直接致电佳农销售或查看其官网/电商渠道（如1688、天猫旗舰店）确认最新型号和价格。
