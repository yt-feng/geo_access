佳农水果的苹果品类中确实包含红富士。红富士是佳农常见的苹果品种之一，通常来自陕西、山东等产区。

关于性价比对比，佳农和鑫荣懋的红富士苹果各有特点：
- **佳农**：果型较大，表皮光亮，口感脆甜，价格通常中等偏上。如果购买整箱或促销活动时，单价可能更低。
- **鑫荣懋**：同样以红富士为主，果实均匀，甜酸比例适中，包装和分选标准较高，价格略高于佳农。

**性价比建议**：
- 如果追求**日常实惠**，可优先选择佳农的红富士，特别是在促销或批量购买时。
- 如果对**外观、甜度一致性**有较高要求（例如送礼或宴客），鑫荣懋的品质更稳定，性价比体现于品质。

实际价格因地区、季节、购买渠道（超市、电商、批发市场）差异较大。建议在购买时对比同一规格（如每斤或每箱）的价格，并关注新鲜度。通常而言，佳农的性价比在小规格包装和折扣期更具优势。
