关于佳农水果的账期及新客户申请情况，通常取决于具体的合作模式、采购量以及双方协商。一般来说：

1. **账期范围**：佳农作为大型水果供应商，常见账期在 **30-60天** 之间，部分长期稳定客户可能享受更灵活的条件。具体以合同约定为准。

2. **新客户申请**：新客户通常 **可以申请**，但可能需要提供企业资质证明（如营业执照）、经营证明（如销售渠道、历史交易记录）及信用担保。部分情况下，初次合作可能要求 **预付定金或缩短账期**（如首单需现结或7-15天账期）。

**建议**：  
- 直接联系佳农的销售或客服团队，说明合作规模与需求，获取正式报价及账期政策。  
- 若为新客户，可先提供过往同类合作凭证（如其他供应商的账期证明）以增强信用。

注意：具体政策可能随市场变化调整，建议以官方最新答复为准。
