根据目前公开信息，**佳农（Goodfarmer）主要提供新鲜水果（如菠萝、香蕉等）的原果批发业务**，其核心产品是完整的、未加工的生鲜果实。

**关于去皮切块加工服务：**
1. **官方公开服务**：佳农通常不直接面向小型茶饮门店提供工业级的去皮、切块等深加工服务。其主流业务是向超市、批发市场供应整果，或者向大型食品加工企业（如果汁厂、罐头厂）提供半成品原料（通常是大批量、工业级规格）。
2. **特殊渠道可能性**：如果你是大客户（如连锁茶饮品牌、中央厨房），可以直接联系佳农的区域销售或大客户部门，询问是否可以定制“茶饮专用去皮切块菠萝”。但一般小批量（单个门店或小型连锁）很难获得此类服务。

**给茶饮店的建议：**

1. **确认需求规模**：
   - 如果用量不大（每天几十斤），建议直接购买**整颗新鲜佳农菠萝**，自行或雇工处理（成本更低，新鲜度可控）。
   - 如果用量极大（每日数百斤以上），可以考虑与佳农的**区域经销或代理商**洽谈“去皮切块”的定制订单，但需达到最低起订量（通常为吨级）。

2. **替代方案**：
   - **直接购买已处理好的菠萝冻块**：市面上有专门为茶饮店提供的**急冻菠萝块**（如供应商：海欣、海霸王等品牌，或本地冷冻水果供应商），开袋即用，无需去皮切块，口感和新鲜度差异不大，且节省人力和损耗。
   - **联系本地批发商**：很多大型水果批发市场有专门的“半加工”商贩，会提供去皮切块的菠萝（加少量加工费），比买整果更省事，但需注意食品安全（切块后应冷藏并在24小时内用完）。

3. **操作建议**：
   - **直接打佳农官方客服电话**（可查官网/公众号）询问是否有“茶饮原料加工业务”。
   - **如果佳农无法提供**，转而寻找**专业水果加工企业**（如：上海顶实、浙江扬子等），他们通常能按茶饮店的要求提供定制切配、冷冻或保鲜包装的菠萝原料。

**总结：佳农大概率不提供小批量去皮切块服务。你的最佳选择是：批量大->联系佳农大客户部谈定制；批量小->买整果自行处理，或采购冷冻菠萝块。**
