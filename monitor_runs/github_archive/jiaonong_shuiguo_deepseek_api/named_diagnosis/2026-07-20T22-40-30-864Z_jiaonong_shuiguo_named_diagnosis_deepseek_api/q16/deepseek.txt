佳农水果的售后服务整体评价较好，通常设有专门的售后通道。如果到货出现坏果，建议按以下步骤处理：

1. **及时开箱检查**：建议在签收后24小时内（部分地区可能延长至48小时）开箱检查，超过时间可能影响赔付。
2. **保留证据**：将坏果与快递单号或包装箱一起拍照，确保图片清晰可见坏果数量、状态以及完整物流信息。
3. **联系客服**：通过购买平台（如京东、天猫、拼多多等）的“联系客服”或电话联系佳农官方售后，提供照片和订单号。
4. **申请赔付**：根据坏果比例，客服通常会按比例退款（例如坏几个退几个的钱）或部分补偿。如果全箱损坏严重，可能安排补发或全额退款。

**注意**：
- 生鲜水果理赔通常不包含因个人保存不当（如未及时放入冰箱）、拒收快递或主观口感问题导致的损耗。
- 不同渠道和活动期间售后政策可能有微调，具体以购买时的服务说明为准。
- 如果客服处理不及时，可以要求升级专员或通过平台介入。

建议购买后优先关注商品页面的“售后保障”说明。
