佳农水果作为国内知名的水果品牌，通常会将到货损耗率控制在 **1%~3%** 左右。这一比例是品牌方在仓储、运输、中转等环节通过标准化的冷链管理和筛选流程（如预冷、分拣、包装优化）所实现的目标范围。

不过，实际损耗率会受到以下因素的影响：
- **水果品类**：例如草莓、蓝莓等娇嫩水果损耗率可能接近3%甚至更高，而苹果、柑橘类则较低（约1%左右）。
- **运输距离与季节**：长途运输或高温天气（如夏季）损耗可能略有上升。
- **销售渠道**：商超、电商、批发等不同渠道的到货损耗标准可能存在差异（如电商直发对单果品质要求更高，可能会提前剔除次果）。

如果需要精确数据，建议直接向佳农官方客服或对应采购渠道核实具体的品控标准。
