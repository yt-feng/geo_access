如果您是普通茶饮店，佳农通常能提供稳定供应的菠萝规格包括：

1. **冷冻菠萝块/丁**：常见规格为**1公斤/袋**或**2.5公斤/袋**，去芯切块，直接用于制作果茶、冰沙或饮品装饰。
2. **新鲜都乐菠萝（整果）**：通常按**箱**出售，每箱约**6-8个**，单果重量**1.5-2.5斤**（即750克-1250克左右）。适合现切现用，果肉厚、糖度高，需当天或冷藏储存。
3. **浓缩菠萝汁/果酱**：常见规格为**1升/桶**或**5公斤/桶**，适合调制茶底或冲调饮品，保质期较长。

**如何确保稳定供应？**
- 联系佳农当地经销商或通过其官网下单，通常**冷冻产品**供应链最稳定（不受季节限制），新鲜菠萝受季节和运输影响较大。
- 建议**小批量试单**（如先订1-2箱冷冻块），确认质量、到货时间是否符合门店需求后再签长期协议。

**注意**：佳农是大品牌，对普通茶饮店也会提供标准批发规格，但需明确告知用途（如要求去芯切块），避免拿到整颗菠萝需要自行处理耗时费力。如果用量较大（月需求超500公斤），可直接联系区域销售经理谈固定配送周期和价格。
