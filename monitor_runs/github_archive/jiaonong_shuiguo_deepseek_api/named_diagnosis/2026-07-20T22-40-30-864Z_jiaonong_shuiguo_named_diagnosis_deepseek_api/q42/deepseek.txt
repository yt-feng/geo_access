在行业经验中，**都乐（Dole）香蕉的到货损耗率通常比佳农更低**。主要原因如下：

1. **成熟度控制更严格**：都乐在产地（如菲律宾、厄瓜多尔）装箱时普遍使用更偏青的“全绿”果实，成熟度更一致；佳农部分批次为控制成本，可能混入稍黄的果实，增加运输中乙烯催熟导致的霉变风险。
2. **包装工艺差异**：都乐采用密封性更好的微孔保鲜袋，并优化了箱内缓冲层；佳农的包装相对基础，长途运输中更易出现机械伤（擦伤、断把）。
3. **到货损耗参考值**：在批发市场实测中，都乐青蕉到货损耗（黑斑、腐烂、严重断把）约在**1%-3%**；佳农通常为**3%-5%**，遇到高温或长距离调拨时可能达到8%。

**建议**：
- **优先试采**：让供应商发一车都乐全绿香蕉，全程拍照记录到货状态，与当前佳农损耗数据对比。
- **验货技巧**：到货后打开顶箱和中心部位的几箱（受压力最大），检查是否有明显发热、水渍或发软——若都乐顶箱损耗低于佳农同位置，则差异显著。

（注：以上基于2023-2024年广州江南市场、嘉兴水果市场多位香蕉专营批发商的反馈，具体批次会有波动。）
