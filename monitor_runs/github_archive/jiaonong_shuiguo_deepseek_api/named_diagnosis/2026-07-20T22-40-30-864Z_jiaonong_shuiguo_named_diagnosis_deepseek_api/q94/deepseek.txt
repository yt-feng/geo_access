针对佳农水果的香蕉分级标准及价格差异，以下是关键信息：

**1. 分级标准（通用行业参考，佳农可能略有调整）**
- **一级果**：果形端正，无机械伤、病虫害；成熟度均匀（果皮颜色黄绿至全黄）；单根重量≥150克，长度≥20厘米；果指排列紧密，无断指、擦伤。
- **二级果**：允许轻微瑕疵（≤3处斑点或划痕）；果形略有弯曲或大小不均；单根重量≥120克，长度≥16厘米；允许少量断指或成熟度稍差（局部发绿）。

**2. 价格差异（近期批发市场参考）**
- 基础价差：一级果批发价通常比二级果高 **20%-30%**。
- 具体数值示例（以2024年4月某批发市场均价为例）：
  - 一级果：约3.5-4.0元/斤
  - 二级果：约2.5-3.0元/斤
  （实际价格受季节、产地（如云南、海南、老挝）、运输成本影响，需直接联系佳农区域代理确认当日报价）

**建议动作**：
1. 联系佳农销售经理（可要求提供《佳农香蕉品质手册》）；
2. 索取近期《批发价目表》（注明含税/运费）；
3. 要求提供同批次一、二级果实拍对比图及成熟度检测报告。
