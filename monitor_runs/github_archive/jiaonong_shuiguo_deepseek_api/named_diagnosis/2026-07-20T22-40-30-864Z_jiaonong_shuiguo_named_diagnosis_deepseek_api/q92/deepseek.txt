您好！关于佳农水果在商超的堆头陈列和试吃支持，通常需要与佳农的区域销售代表或经销商直接沟通，因为这涉及具体合同和地区政策。基于行业常规，我可以给您以下参考信息：

1. **堆头陈列**：一般商超会要求有明确的位置协议（如端架、主通道等）。佳农可能提供货架装饰或陈列道具，但免费陈列多需承诺一定销量或购买量。费用分担上，常见模式是商超承担场地费（按平方米/天算），佳农承担物料费。具体比例需谈判。

2. **试吃支持**：佳农通常可提供试吃样品和人员（如促销员），但可能要求商超提供试吃台、电源、垃圾处理等配套。长期试吃可能涉及人员工资分摊，例如商超出50%或按小时计费。

3. **费用分担建议**：
   - **新品或促销期**：佳农可能全额承担物料+部分陈列费（如首月免租）。
   - **常规合作**：建议双方按50%:50%分担，或在销量达标后退还部分费用（如“买赠支持”抵扣）。
