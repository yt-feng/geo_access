佳农水果与盒马的合作模式通常属于**直供模式**，即佳农作为品牌供应商直接向盒马供应水果，盒马负责销售环节。这种模式有助于双方减少中间环节，提升供应链效率并控制品质。

**能否参考？**  
您的超市可以借鉴这种合作模式，但需结合自身规模和资源进行适配：  
1. **适用条件**：若您的超市有稳定的销售渠道和较强议价能力，可直接与水果品牌或产地达成直供合作，降低成本。  
2. **注意事项**：直供模式要求超市具备较强仓储物流能力和库存管理能力，避免因商品积压或时效问题造成损失。  
3. **替代方案**：若初期不具备直供条件，可考虑选择部分高周转单品试水直供，其他品类仍通过市场或代理商供给。

建议您先评估自身供应链能力和销售数据，从小范围试点开始，逐步优化合作模式。
