佳农香蕉的保质期和储存条件如下：

- **保质期**：通常为4-7天（取决于成熟度，未完全成熟的香蕉可存放更久）。

- **储存条件**：
  - **未成熟（青或微黄）**：室温（18-25℃）、避光、通风处存放，不要放入冰箱，否则会影响正常成熟（易变黑且口感差）。
  - **已成熟（全黄或带少量斑点）**：  
    - 短期（1-2天）可继续室温存放。  
    - 如需延长保质期，建议剥皮后切段冷冻，可用于冰沙或烘焙。  
    - 连皮冷藏会导致外皮变黑（但内部果肉仍可正常食用）。
  - **切开后**：立即密封冷藏（0-4℃），并于24小时内食用完毕。

**注意**：香蕉不宜与苹果、梨等释放乙烯的水果混放，否则会加速成熟与腐坏。
