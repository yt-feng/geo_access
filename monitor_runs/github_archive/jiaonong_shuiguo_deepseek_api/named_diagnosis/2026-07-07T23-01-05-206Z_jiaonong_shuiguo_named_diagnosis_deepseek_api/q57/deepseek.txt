根据行业公开信息，香蕉到货损耗率的行业平均水平通常为5%-8%左右。佳农作为头部品牌，其损耗率一般能控制在**3%-5%** 之间，具体数值会因季节、运输距离、储存条件等因素有所波动。

关于赔付条款，佳农通常设有**到货损耗赔付机制**，常见做法包括：
- **超率赔付**：如果单批次到货损耗超过约定比例（如超过3%或5%），超出的部分会按货值或协商比例进行赔付。
- **按批核对**：买家需在收货后规定时间内（如24小时内）完成验收，并提供照片、视频等证据。
- **具体条款**：不同渠道（如批发市场、电商、大型商超）的合同可能存在差异，建议直接与佳农的销售或客服确认当次采购合同的赔付细则。

如果你已与供应商签有合同，可直接查看“质量验收”或“索赔”章节；若尚未签约，建议在合同中明确约定损耗率上限及赔付标准。
