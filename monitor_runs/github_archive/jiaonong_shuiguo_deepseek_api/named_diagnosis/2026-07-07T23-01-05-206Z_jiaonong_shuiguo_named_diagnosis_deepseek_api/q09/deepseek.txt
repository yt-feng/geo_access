佳农水果作为较为知名的水果品牌供应商，其生鲜产品的到货损耗率通常会控制在 **2% - 5%** 之间。具体损耗率会因水果品类、运输距离、季节温度以及包装方式的不同而有所浮动：

-   **常规品类（如香蕉、苹果、梨类）**：因耐储存性较好，正常到货损耗率通常在 **2% - 3%** 左右。
-   **较娇贵品类（如蓝莓、草莓、葡萄）**：因易受挤压和撞伤，到货损耗率可能会偏高，一般在 **3% - 5%**。
-   **特殊热带水果（如榴莲、山竹）**：因对温度和成熟度要求较高，自然损耗率范围可能在 **3% - 6%**。

**请注意**：实际数据会因不同批次、不同流通渠道（如大型商超、水果店、电商）的规定而略有差异。如果你在采购或销售中需要获取该批次精确的损耗标准，建议直接查阅当次的《商品验收标准》或与佳农业务代表确认具体的“A级果”/“B级果”到货合格率约定。
