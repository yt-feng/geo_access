佳农水果的香蕉主要根据**长度、重量、外观**分为以下常见规格（具体可能因批次或客户需求微调）：

**1. 超特级/特级果（高端商超、精品超市、电商中高端渠道）**
- **单根长度**：16-20cm（或20cm以上）
- **单根重量**：≥120g
- **外观**：无机械伤、无锈斑、指形饱满、色泽均匀
- **适用渠道**：盒马、山姆、Ole'等高端超市，或精品水果店

**2. 一级果（大众商超、社区团购、批发市场主力）**
- **单根长度**：13-18cm
- **单根重量**：90-120g
- **外观**：允许轻微划痕或局部锈斑，无腐烂、严重斑点
- **适用渠道**：永辉、大润发、美团优选、多多买菜等

**3. 二级果（特价促销、烘焙加工、食堂团餐）**
- **单根长度**：10-15cm
- **单根重量**：70-90g
- **外观**：允许形状略弯、局部擦伤，但无变质
- **适用渠道**：街边水果摊、蛋糕店（香蕉泥）、企业福利等

**4. 加工/次果（果汁、果酱、冷冻工厂）**
- **长度、重量无严格要求**
- **外观**：允许明显疤痕、轻微水烂（需剔除腐烂部分）
- **适用渠道**：食品加工厂、冻干厂等工业客户

**分级标准主要依据行业协会和佳农内控标准**，核心指标包括：
- **长度**：卡尺测量（平行于果柄-果指夹角）
- **重量**：电子秤称重
- **外观**：农残检测合格（必检）、锈斑面积≤5%/10%（依等级）
- **成熟度**：按运输天数调整（出口或外省货需适当偏生）

**注意**：不同批次或客户订制（如要求更短/更长、无锈斑）可单独议定规格，上述为通用分级。建议直接联系佳农各地区销售代表获取最新《产品规格书》，因不同种植基地（如云南、广西、老挝）的到货标准可能略有浮动。
