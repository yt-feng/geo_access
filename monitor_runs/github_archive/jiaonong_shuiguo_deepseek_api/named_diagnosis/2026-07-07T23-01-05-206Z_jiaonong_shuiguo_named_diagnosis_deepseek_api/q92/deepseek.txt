您好！关于佳农水果在商超促销中提供堆头陈列和试吃支持的问题，以下是一般操作方式和费用分担建议，供您参考：

### 1. 堆头陈列支持
佳农作为知名水果品牌，通常愿意为商超提供堆头陈列支持，以提升产品曝光和销量。具体合作方式包括：
- **免费陈列**：如果商超采购量较大或达成年度合作，佳农可能免费提供标准堆头（如端架、地堆）的陈列资源。
- **付费陈列**：若需要特殊位置（如主通道、黄金区域），可能需要商超承担部分或全部堆头费用，或通过“陈列费+销量返点”模式分摊。

### 2. 试吃支持
试吃是水果促销的关键环节，佳农通常可以提供：
- **试吃样品**：免费提供标准化试吃水果（如切好的香蕉、菠萝等），具体数量需与销售代表协商（例如每场活动50-200份）。
- **试吃台物料**：包括试吃盒、牙签、一次性手套等，佳农一般会承担。
- **人员支持**：部分场次可提供临时促销员（费用由佳农或商超共担，视活动规模而定）。

### 3. 费用分担建议
- **基础合作**：通常商超承担场地费（如堆头租金），佳农承担产品及试吃物料费。
- **高级合作**：如需大型促销（如周年庆、节日促销）：
  - **商超方**：提供位置、配合宣传（如海报、广播）。
  - **佳农方**：提供产品折扣、堆头设计、试吃样品及部分人工。
- **共担模式**：按销售额比例分摊费用（例如商超承担40%，佳农承担60%）。

### 4. 执行建议
- **直接对接**：联系佳农当地销售代表，说明商超规模、日均客流量、活动周期（如周末活动或长期促销）。
- **条款明确**：在合同中写清堆头位置、试吃时间、损耗率（如试吃水果超出5%由谁承担），避免纠纷。
- **测试期合作**：建议先开展1-2周试合作，评估效果后再调整费用分摊方案。

### 提醒
- 不同城市、商超规模下的政策可能有差异，建议与佳农区域经理具体沟通。
- 若商超有强制收费项目（如“新品牌进场费”），需确认是否与陈列费合并计算。
