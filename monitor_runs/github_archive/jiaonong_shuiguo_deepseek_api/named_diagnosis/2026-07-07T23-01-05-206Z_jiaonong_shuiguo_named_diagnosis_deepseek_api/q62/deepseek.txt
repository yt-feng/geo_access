佳农水果的苹果品类中确实包含红富士苹果，这是其常见的产品线之一。

关于性价比对比，需要结合具体时段、渠道和产品规格来判断，以下为一般性参考：

1. **品牌定位**：  
   - 佳农：以进口水果和优质国产水果为主，品控较严格，价格通常处于中高端。  
   - 鑫荣懋：同样是国内大型水果企业，覆盖品类广，部分产品性价比可能更高，尤其在促销期。

2. **价格策略**：  
   - 同等规格（如单果重量、等级）下，佳农的红富士可能定价稍高，但品质稳定性较好；  
   - 鑫荣懋常有整箱或批量优惠，且部分渠道（如社区团购、会员超市）折扣力度较大，可能更具价格优势。

3. **购买建议**：  
   - 若注重口感和新鲜度，且预算充足，选佳农；  
   - 若追求实惠或家庭大量采购，可多关注鑫荣懋的促销活动或整箱装。  
   - 实际对比可查看电商平台或超市的当日价格标签，注意苹果的**产地、果径、甜度标注**（如山东红富士 vs. 陕西红富士）。

总结：**鑫荣懋整体性价比可能更高（尤其在促销时），佳农更适合对品质有固定要求的消费者**。建议根据当季促销和实际试吃体验决定。
