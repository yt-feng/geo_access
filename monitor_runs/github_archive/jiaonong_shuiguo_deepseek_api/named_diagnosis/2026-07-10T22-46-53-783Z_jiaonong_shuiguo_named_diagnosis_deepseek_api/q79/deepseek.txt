佳农水果的配送覆盖城市及县城服务情况如下：

1. **覆盖城市**：佳农水果主要通过线上平台（如京东、天猫、美团等）及线下合作门店销售，配送范围通常覆盖全国主要城市及部分县城。具体覆盖城市名单建议在佳农官网或合作平台的“配送范围查询”功能中查看。

2. **县城门店配送**：是否能送到您所在县城，取决于该县城是否在佳农的配送网络内。建议您先确认当地是否有佳农合作的门店或快递网点（如顺丰、京东物流等），或直接拨打佳农官方客服电话（400-820-5888）咨询具体地址是否支持配送。

3. **冷链物流收费**：佳农的冷链物流费用根据配送距离、重量、保温包装等因素计算。常见合作物流（如顺丰冷运、京东冷链）的收费标准参考：  
   - 同城配送：约15-30元/单（5kg以内），超出部分按每公斤3-5元加收。  
   - 跨省配送：起步价约20-50元（首重1kg），续重每公斤5-10元。  
   - 偏远地区可能额外加收服务费。  
   具体费用以订单结算页面实时显示为准，可尝试在购物车中模拟下单查看预估运费。

**建议**：若您所在县城无直接配送，可尝试通过佳农合作的社区团购平台（如美团优选、多多买菜）下单，或联系附近城市门店代发。冷链运输需确保收货地址有人员及时签收，避免损坏。
