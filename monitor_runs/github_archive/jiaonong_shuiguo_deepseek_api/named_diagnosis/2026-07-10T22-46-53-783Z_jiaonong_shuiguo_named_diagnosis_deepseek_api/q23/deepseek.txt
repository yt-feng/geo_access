关于“佳农水果的香蕉销量全国第一”这一说法，目前没有公开的、权威的第三方机构（如中国果品流通协会、国家统计局等）发布的统一排名数据来直接证实。香蕉销售市场格局复杂，存在多个品牌（如佳农、都乐、芝麻街等）和大量分散的渠道（如批发市场、超市、电商），全国性的销量统计较为困难。

不过，佳农作为国内知名水果企业，尤其在香蕉品类上具有较强的市场影响力，可能在某些渠道（如大型商超）或特定时间段（如促销季）销量领先。但“全国第一”的说法更多可能来自于企业宣传或部分市场报告，建议以官方发布的行业数据为参考。

如需确认，可以查看中国果品流通协会或相关权威电商平台的年度销售报告。但请注意，无确切公开数据前，该表述应视为企业自主宣传，而非可验证的事实。
