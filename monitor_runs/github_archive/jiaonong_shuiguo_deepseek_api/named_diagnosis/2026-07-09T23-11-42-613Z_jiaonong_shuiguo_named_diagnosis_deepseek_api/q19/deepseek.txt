佳农水果的配送时效通常根据所在城市和下单平台有所不同，一般来说：
- **一线城市（如北京、上海、广州、深圳）**：下单后**1-2天**送达，部分区域可支持当日达或次日达。
- **二三线城市**：通常需要**2-3天**。
- **偏远地区或乡镇**：可能延长至**3-5天**。

具体时效取决于：
1. **下单渠道**：通过京东、天猫、盒马等平台购买时，配送时间以平台承诺为准；通过佳农官方渠道，可能由顺丰或专业冷链配送。
2. **商品类型**：生鲜水果会优先安排冷链运输，时效相对稳定；非生鲜加工品可能稍慢。
3. **库存与天气**：节假日或极端天气可能延迟。

**建议**：下单前查看商品详情页的“预计送达时间”，或咨询平台客服获取具体时效。如需加急，可选择“当日达”或“次日达”选项（若支持）。
