您好！关于佳农和都乐香蕉的到货损耗率，从我目前掌握的信息来看，**并没有公开的、可对比的直接数据来源**（比如第三方质检报告或行业白皮书）能明确说哪个品牌在批发市场的损耗率一定更低。

不过，根据行业经验和两个品牌的市场特性，我可以提供一些分析供您参考：

1. **品牌定位与供应链**：
   - **都乐（Dole）**：国际大品牌，供应链非常成熟，对香蕉的采摘、催熟、冷链运输、品控标准通常很高。理论上，其标准化流程能保证到货时的成熟度、外观一致性较好，但价格通常较高。
   - **佳农（Goodfarmer）**：国内领先的果蔬品牌，同样注重品控，供应链能力很强，尤其在东南亚产地的把控上。佳农的香蕉在国内市场份额很大，批发渠道铺货广泛，价格相对都乐可能更有竞争力。

2. **损耗率的关键因素**：
   到货损耗率（包括机械伤、压伤、过早成熟、黑斑病、断裂等）主要取决于：
   - **从产地到批发市场的运输时长与温度控制**（冷链是否全程无缝）。
   - **采摘时的成熟度**（采摘太熟易在运输中损耗，太生则到货后催熟困难）。
   - **包装与装卸**（是否使用带缓冲材料的纸箱，装卸是否轻拿轻放）。
   - **批次与季节性**（不同产区、不同季节的香蕉抗损性不同）。

3. **实际批发市场的常见反馈**：
   - 在大型批发市场（如北京新发地、广州江南市场），**普遍认为都乐香蕉的品控更稳定，到货破损率可能更低**，但价格较高。
   - 佳农香蕉由于量大、渠道密，在同等价格下，损耗率往往与运输批次有关，可能存在一定波动。但也有很多批发商认可其性价比。
   - **没有绝对的“谁更低”**，更建议您通过以下方式自行判断：
     - **小批量试单对比**：分别从两家拿货，实际记录到货后的损耗比例（比如每箱称重、挑出明显损伤果）。
     - **关注包装日期与产地**：优先选择生产日期近、产地更近（如广西、云南产的香蕉相比东南亚海运的通常损耗更低）的批次。
     - **查看包装箱内部缓冲**：好的品牌会用带气垫或泡沫隔板的纸箱减少磕碰。

**最终建议**：
- 如果您追求**最高稳定性**且不介意价格，**都乐**通常不会让您失望。
- 如果您希望**平衡成本和稳定损耗**，**佳农**是性价比很高的选择，建议与本地稳定的供应商沟通，固定拿货批次。
- **最关键的行动**：直接联系您当地批发市场的佳农和都乐经销商，要求看他们的最新批次到货品相，并询问他们“对标竞品”时的损耗控制措施（如催熟时间、包装方式）。这部分**一手信息**比任何理论分析都更有价值。
