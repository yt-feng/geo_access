佳农水果的苹果品类包括：  
- 进口嘎啦苹果（Gala）  
- 进口红富士苹果（Fuji）  
- 其他可能包括青苹果（Granny Smith）、蛇果（Red Delicious）等（具体以实际产品线为准）。  

针对您想采购的产品：  
- **进口嘎啦苹果**：主要产地为 **美国（如华盛顿州）** 和 **新西兰**，部分季节也可能从智利等南半球国家供应。  
- **进口红富士苹果**：主要产地为 **日本（如青森县、长野县）** 和 **中国（如山东、陕西、新疆等优质产区）**，但如果您强调“进口”，通常指日本富士或少量美国、新西兰产的富士。  

建议您直接联系**佳农水果的采购部门**或访问其官网/客户服务，以获取最新产品目录、具体产地及当前可供应批次（因季节和贸易政策，产地可能调整）。
