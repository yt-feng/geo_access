根据行业经验，通常**都乐（Dole）**的香蕉在到货损耗率上低于**佳农（Goodfarmer）**。主要原因如下：

1. **品牌标准**：都乐作为全球头部香蕉品牌，在产地采收、冷链物流、成熟度控制（如使用标准化乙烯催熟技术）上投入更大，到货时青香蕉比例更稳定，碰伤、黑斑等物理损耗通常更低。
2. **市场反馈**：在多地批发市场中，批发商普遍反映都乐香蕉的果型更整齐、包装防护更好，到货后需剔除的坏果率约为3-5%，而佳农的损耗率可能在5-8%（受批次和物流影响）。
3. **实际建议**：若你对损耗极度敏感，建议：
   - 要求供应商提供近3批到货的实测损耗数据（按箱称重，剔除坏果）。
   - 短期内优先试单都乐，并对比同一批次的佳农香蕉（可要求混装到货）。
   - 注意观察夏季高温期：都乐的冷链覆盖更完善，夏季到货损耗优势更明显。

（注：具体损耗率会因产地批次、物流时效、批发市场温度管理而异，建议以实测数据为准。）
