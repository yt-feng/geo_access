您好！关于佳农水果的苹果品类，通常以进口苹果为主，常见品种包括：

- **嘎啦（Gala）**  
- **红富士（Red Fuji）**  
- **金帅（Golden Delicious）**  
- **青苹果（Granny Smith）**  
- **蛇果（Red Delicious）**  
- **乔纳金（Jonagold）**  

针对您想采购的 **进口嘎啦** 和 **红富士**，以下是常见产地建议（以佳农等主流进口商为例）：

1. **进口嘎啦**  
   - **主要产地**：**美国**（尤其是华盛顿州）、**新西兰**、**智利**。  
   - 特点：美国嘎啦口感脆甜，新西兰嘎啦香气浓郁，智利嘎啦供应季节（南半球反季）丰富。

2. **进口红富士**  
   - **主要产地**：**日本**（原产地，但量少且价格高）、**美国**（华盛顿州为主）、**新西兰**、**南非**。  
   - 常见选择：美国红富士（常称“华盛顿红富士”）和新西兰红富士（果肉细腻、甜度高）。  

**采购建议**：  
- 佳农作为国际水果贸易商，其进口苹果通常来自上述国家。建议您直接联系佳农官方销售或区域代理，确认当前库存和具体产地（因季节、货源调整会变化）。  
- 若需大规模采购，可要求提供**原产地证明**及**质检报告**。
