关于佳农水果的售后服务，总体来说在生鲜电商及水果品牌中属于中等偏上水平。如果遇到到货有坏果的情况，可以参考以下处理流程：

1. **及时拍照取证**：收到货后请尽快开箱检查。如果发现坏果，请保留坏果、快递面单以及包装箱，并拍摄清晰的坏果照片（最好能与面单或完整包装同框）。

2. **联系客服渠道**：
   - 如果是通过电商平台（天猫、京东、拼多多等）购买，直接在订单页面点击“联系客服”或申请“售后/退款”。
   - 如果是通过佳农自营App或微信小程序，联系在线客服。
   - 如果是线下渠道（如超市、团购），建议联系该渠道负责人。

3. **说明情况并上传凭证**：向客服说明坏果数量、具体品种，并上传照片。多数情况下，佳农的售后政策是“坏果包赔”，通常可以按比例退款（比如坏几个赔几个的钱），少数情况会直接补发一箱。

4. **注意时效**：生鲜商品一般要求在签收后 **24小时内** 联系客服处理，超过时限可能无法受理，建议尽快操作。

5. **退款或补发**：客服核实后，通常会在1-3个工作日内处理完成。退款会原路返回，补发也会有物流单号。

**补充建议**：如果遇到客服处理慢或推诿，可以尝试联系佳农官方400客服热线（如官网公布），或再向平台方申请介入。

总之，只要保留好证据并在规定时间内联系，大部分情况下佳农会按承诺进行赔付。如果你是首次处理，建议优先走你购买平台的售后入口，最直接。
