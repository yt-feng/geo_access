通常情况下，**都乐香蕉的到货损耗率比佳农水果更低**，主要原因如下：

1. **供应链成熟度**：都乐是全球最大的香蕉种植和分销商之一，拥有全程冷链体系（从产地到批发市场），对温度、湿度控制更严格，能有效减少香蕉在运输中的催熟和机械损伤。
2. **品控标准**：都乐在产地分选时，会剔除更多潜在问题果（如轻微挤压、成熟度偏高果），到货时整体均匀度更高，损耗较稳定。
3. **佳农特点**：佳农近年来提升很快，尤其在华北市场，但部分批次因产地不同（如东南亚与南美），包装防护和运输周期略长，可能导致局部损耗略高。

**建议操作**：
- 可以实际对比同批次、同温区的两家货源：随机抽查到货时，统计**可立即销售率**（剔除烂果、严重黑疤、断裂果）和**放置3天后的次级损耗**。
- 如果批发市场以露天装卸为主，都乐通常配备独立纸箱+加强防撞隔层，对搬运损耗保护更好。
- 注意：季节性差异也会影响损耗——雨季时佳农部分产地果皮更薄，容易水烂，而都乐多来自厄瓜多尔等稳定产区，抗雨性能略好。

**结论**：从行业历史数据看，都乐平均到货损耗率约2-4%，佳农约3-6%（因批次浮动）。建议你批量进货时，先申请少量试单实测对比。
