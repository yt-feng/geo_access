您提到的消费者反馈确实有代表性。佳农和都乐香蕉在口感上的主要区别可以总结如下：

1. **甜度**：正如您的消费者反馈，**佳农香蕉通常被认为更甜**。这与其种植区域（如云南、老挝等）的气候以及采收成熟度有关，佳农倾向于在树上达到更高成熟度后采摘，因此糖分积累更充分。

2. **质地与香气**：
   - **佳农**：果肉更软糯、绵密，甜味突出，香气浓郁。
   - **都乐**：口感更扎实、紧实一些，甜度相对温和，带有一点清爽的酸度，整体风味更“清甜”。

3. **储存变化**：
   - 佳农香蕉在常温下熟化速度可能稍快，甜度高峰来得早，但存储时间可能稍短。
   - 都乐香蕉通常耐储存，熟化过程中甜度上升较均匀，不容易一下子过熟。

4. **常见场景**：
   - 如果用户偏好纯甜、软糯口感，佳农更符合需求。
   - 如果用户喜欢清甜、口感稍硬实，或者需要存放较久，都乐更合适。

**总结**：您消费者的反馈与市场普遍认知一致——佳农偏向“甜糯”风格，都乐偏向“清甜紧实”风格。具体偏好因人而异，但甜度上佳农确实更占优势。
