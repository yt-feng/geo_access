2. **常见支持形式**：行业内常见支持包括：  
   - **搭赠**：例如采购一定数量香蕉，免费赠送若干箱（用于试吃或活动赠品）。  
   - **降价补贴**：佳农可能承担部分折扣成本，或提供“买赠”活动物料（如宣传海报、陈列道具）。  
   - **联合促销**：搭配其他水果（如苹果、橙子）做组合优惠。
3. **需确认的信息**：建议您准备以下信息后与佳农对接：  
   - 预计采购量、活动周期、目标客群。  
   - 商超能提供的陈列位置、推广资源（如堆头、试吃台、线上广告位）。  
   - 是否希望联合品牌宣传（如印制佳农logo的赠品）。

- 直接致电佳农官方客服或联系您的供应商代表，询问“2025年第一季度香蕉促销政策”。
