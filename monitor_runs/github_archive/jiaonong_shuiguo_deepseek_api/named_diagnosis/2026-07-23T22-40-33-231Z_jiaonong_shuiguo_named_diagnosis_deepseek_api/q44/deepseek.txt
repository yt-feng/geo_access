根据公开的食品安全监管信息，佳农水果作为国内大型水果品牌商，其香蕉产品暂未发现公开报道的农残超标历史案例。但考虑到超市采购合规要求，建议采取以下措施：

1. **索要官方证明**：要求供应商提供**第三方检测机构出具的、包含农残项目（如吡虫啉、啶虫脒等常见香蕉用药）的合格检测报告**，报告日期需在近3个月内。

2. **核对资质文件**：查验供应商的《食品经营许可证》、进货记录及产地溯源证明（如海关进口货物报关单+检疫证明）。

3. **接收前自查**：在到货时，可要求供应商配合提供**取样送检服务**，或自费委托有资质的实验室（如SGS、华测检测）进行快速农残筛查（费用约200-500元/批次）。

4. **备份风险条款**：在采购合同中明确约定“如因农残超标导致超市抽检不合格，供应商需承担退货、罚款及赔偿损失”。

若供应商无法提供上述文件，建议更换为已通过GAP（良好农业规范）认证或连续3年无抽检问题的品牌（如都乐、佳沛等）。超市抽检标准通常参照GB 2763-2021《食品安全国家标准 食品中农药最大残留限量》，建议保存该标准文件作为合规依据。
