佳农水果的售后服务整体上比较规范，尤其是针对到货坏果问题，通常会提供明确的处理流程。

如果你收到坏果，建议按以下步骤处理：

1. **及时开箱检查**：签收后尽快打开包装，检查是否有挤压、腐烂或严重损伤的果子。
2. **拍照留存证据**：将坏果与快递面单（或外包装上的物流信息）放在一起拍照，清晰展示坏果数量和状态。最好能拍到坏果的细节和包装箱上的标识。
3. **联系客服**：通过购买渠道（如佳农官方小程序、京东/天猫店铺、或其他电商平台）联系客服，说明情况并发送照片。
4. **客服处理**：通常客服会根据坏果比例进行赔付。常见处理方式是：
   - 按坏果数量比例退款（例如坏果占整箱的%）。
   - 或者补发同等数量的好果。
   - 极少数情况可能会整单退款（取决于坏果严重程度和平台规则）。
5. **留意时效**：大部分生鲜水果的售后需要签收后24小时内反馈，超过时间可能无法处理，务必尽快联系。

注意：不同购买渠道或不同水果品种的具体赔付标准可能略有差异（如南方水果保质期短，坏果率容忍度更低），建议购买前查看该商品页面的售后服务说明。

如果遇到客服处理慢或不合理赔付，你可以尝试通过平台官方投诉渠道或联系佳农400官方热线进一步沟通。
