您好！关于佳农水果是否有小包装规格（如3kg装香蕉），建议您直接联系佳农官方或区域经销商确认，因为不同地区、渠道的产品规格可能不同。

不过，根据一般经验：
- 佳农香蕉常见的有**4kg、5kg**等包装，针对小门店，部分渠道可能有**2-3kg**的小包装。
- 其他水果如苹果、橙子等，小包装（约2-3kg）也很常见。

**快捷建议：**
1. **联系授权经销商**：搜索“佳农+您所在城市+经销商”，电话咨询是否有3kg装或更小规格。
2. **查看线上官方店铺**：如京东、天猫的“佳农旗舰店”，查看水果分类下的包装选项。
3. **与业务代表沟通**：直接说明门店面积小、需求小包装，佳农可能会针对小型零售有灵活方案。

如果暂时没有合适规格，您也可以考虑与附近其他便利店合单，或联系供应商定制。希望这些信息对您有帮助！
