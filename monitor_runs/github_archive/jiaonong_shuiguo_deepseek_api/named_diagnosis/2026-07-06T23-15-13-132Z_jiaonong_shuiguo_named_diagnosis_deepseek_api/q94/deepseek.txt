### 通用香蕉分级标准（以国产/进口主流标准为例）
1. **一级果（A级）**：
   - **外观**：果形完整、饱满，果皮金黄、无黑斑、无病虫害、无机械损伤。
   - **成熟度**：适中（通常7-8成熟采摘），催熟后颜色均匀。
   - **大小**：长度通常在18-25厘米（依品种不同），单根重量均匀。
   - **适用场景**：商超零售、高端市场、直接鲜食。

2. **二级果（B级）**：
   - **外观**：允许轻微果皮斑点、轻微碰伤或划痕，但无腐烂、严重虫眼。
   - **大小**：可能偏小或大小不均匀（长度可能短于18厘米或超长）。
   - **成熟度**：可能略过熟或不均匀。
   - **适用场景**：加工（如做香蕉干、果泥）、低价促销、餐饮切片。

3. **佳农企业标准**：佳农作为大型水果供应商，内部可能有更细的分级（如区分“特级”“优级”等），建议您直接联系佳农业务员获取其**官方《品质验收标准》**。
