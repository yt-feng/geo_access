佳农水果经营的苹果品类通常包括嘎啦（Gala）、红富士（Red Fuji）、蛇果（Red Delicious）、青苹果（Granny Smith）等。针对您提到的进口嘎啦和红富士，常见的产地如下：

- **进口嘎啦**：主要来自**新西兰**（如品种Envy、Jazz等）和**智利**。新西兰嘎啦苹果一般上市季节为3月至8月（南半球秋季至冬季），智利嘎啦则主要在1月至5月（南半球夏季至秋季）供应。
- **进口红富士**：主要产地为**美国**（如华盛顿州）、**新西兰**和**智利**。美国红富士通常于10月至次年5月供应（北半球冬季至春季），新西兰和智利红富士则主要在4月至9月（南半球冬季至春季）上市。

建议您直接联系佳农水果的销售部门或官网，确认当前批次的具体品种、产区和到货时间，以获取最准确的采购信息。
