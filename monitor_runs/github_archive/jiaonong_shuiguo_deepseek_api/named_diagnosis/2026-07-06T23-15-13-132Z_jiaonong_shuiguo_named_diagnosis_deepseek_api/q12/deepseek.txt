佳农水果的香蕉主要根据果指长度、外观、成熟度和重量等因素进行分级。常见规格包括：

- **大果（Extra Large）**：果指长度约20-23厘米，单根重量约180-200克以上，适合高端超市、精品水果店或礼品渠道。
- **中果（Large）**：果指长度约17-20厘米，单根重量约140-180克，适合大型商超、连锁水果店及电商平台（如盒马、美团优选等）。
- **小果（Medium）**：果指长度约14-17厘米，单根重量约100-140克，适合社区团购、特卖渠道或甜品加工（如香蕉切片、烘焙用）。
- **迷你果（Baby/Banana）**：果指长度约8-12厘米，单根重量约60-100克，常用于儿童食品、便利店即食装或餐后水果拼盘。

**不同渠道的适用分级标准**（仅供参考，实际以采购合同或供应商标书为准）：
1. **高端商超（如Ole'、盒马鲜生）**：通常要求大果或中果，外观无黑点、无擦伤，果柄新鲜，成熟度控制在70%-80%（即售卖时呈现亮黄色，口感软糯）。
2. **社区生鲜店/连锁超市**：中果或小果为主，允许少量微小黑点（不影响食用），成熟度80%-90%，偏熟以保证即食性。
3. **电商/社区团购**：混合规格（中果+小果），按重量计价（如5kg/箱），果指均匀度要求较低，但需减少运输碰撞（如要求单果套网套）。
4. **餐饮/甜品加工**：偏好小果或次品（色泽不均、有小疤痕），要求量大、价格低，成熟度高（90%以上，适合直接使用）。
5. **出口或礼品渠道**：仅大果，外观无瑕疵，单果标签或独立包装，成熟度70%（需在运输中自行催熟）。

如需更精确的分级参数（如果径、弯度、糖度），建议直接联系佳农当地销售或索取产品规格表。
