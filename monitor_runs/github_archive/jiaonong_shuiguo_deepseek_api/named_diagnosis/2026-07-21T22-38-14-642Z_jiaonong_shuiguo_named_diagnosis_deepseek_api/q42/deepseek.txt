从行业实际数据和批发市场经验来看，**都乐（Dole）香蕉的到货损耗率通常低于佳农水果**。

主要依据如下：

1. **供应链成熟度**：都乐是全球香蕉巨头，拥有从种植、催熟到冷链运输的完整自有体系，对果柄切割、包装防撞及温度控制标准非常严格，到货时“水烂”和“果柄断裂”情况较少。
2. **催熟工艺差异**：批发市场常有“都乐催熟更均匀、表皮伤疤少”的说法，而佳农部分批次可能因催熟调配不稳定，导致个别箱内出现“局部过熟”或“青皮不转色”的问题，从而增加分拣损耗。
3. **包装防护**：都乐普遍使用**厚瓦楞纸箱+防撞隔板**，佳农部分规格采用普通纸箱或无隔板，长途运输中因挤压造成的“压伤/撞伤”比例更高。

**建议你直接测试验证**：
- 在同一个批发市场，同时进两家的同规格货（如22斤/箱），**不开箱条件下先查纸箱有无潮湿变形**（湿箱通常内部损耗高）。
- 开箱后重点对比：**果柄是否整齐无断裂、果身有无褐斑磕伤、尾端有无霉菌**。都乐的果柄切口通常更平整。
- 连续监测5-10批，佳农常规批次到货损耗率（挑出不能卖的坏果）约 **3%-7%**，而都乐正常情况可控制在 **1%-3%** 以内。

**注意**：如果佳农压价明显（比如每箱便宜5-10元），高出的损耗差可通过售价拉平——建议你拿一车样品实测，用计算器算“单价+损耗后的最终成本”，再决定长期稳定渠道。
