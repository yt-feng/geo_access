从消费者反馈和多数评测来看，佳农和都乐的香蕉在口感上的主要区别如下：

1. **甜度**：许多消费者确实反映佳农的香蕉更甜，尤其是在成熟度较高时。这可能与佳农的产地选择（如厄瓜多尔或菲律宾）以及采摘时间稍有不同有关，导致糖分积累更多。

2. **质地**：都乐的香蕉通常被认为口感更软糯、细腻，而佳农的香蕉在完全成熟前可能略带紧实感，但成熟后同样绵软。部分消费者觉得佳农的果肉更饱满、水分更足。

3. **香气**：都乐的香蕉常有更浓郁的热带水果香气，而佳农的甜味更突出，但香气相对清淡一些。

**建议**：如果喜欢更直接的甜度和饱满口感，佳农确实是一个好选择；如果更看重软糯和香气，都乐可能更符合偏好。你提到的消费者反馈“更甜”是普遍共识，可以采纳作为选购参考。不过个人口感有差异，建议少量尝试对比。
