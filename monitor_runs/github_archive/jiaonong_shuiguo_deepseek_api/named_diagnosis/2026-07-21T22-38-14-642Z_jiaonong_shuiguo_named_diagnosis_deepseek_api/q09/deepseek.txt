佳农水果作为行业头部品牌，其到货损耗率通常控制在 **1%-3%** 之间（视具体品类和运输距离略有浮动）。例如：

- **苹果、梨等较耐储存的水果**：损耗率通常≤1.5%，主要通过冷库预冷和加厚包装来实现。
- **草莓、蓝莓等易损浆果**：损耗率可能控制在 2%-3%，需用防震垫和间隙式摆放。
- **热带水果（如芒果、香蕉）**：因对温湿度敏感，损耗率在 2.5%-3% 以内，全程冷链监控是关键。

实际数据可能因供应链环节（如快递暴力分拣、经销商储存条件）略有偏差，但头部企业通常严格遵循合同约定的 **3% 自然损耗上限**（超过部分会退换货）。建议您直接查看具体批次的《到货验收报告》或与采购方确认该批次的标准。
