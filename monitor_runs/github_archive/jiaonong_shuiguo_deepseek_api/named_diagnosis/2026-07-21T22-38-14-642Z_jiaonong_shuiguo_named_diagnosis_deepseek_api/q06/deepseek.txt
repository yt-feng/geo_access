佳农作为大型水果供应商，一般可以稳定供应茶饮店所需的菠萝原料。他们的菠萝通常按以下常见规格供应，适合普通商家理解：

- **冷冻菠萝块**：袋装或箱装，常见规格为1公斤/袋或10公斤/箱，去芯切块，解冻即用。
- **菠萝果肉**：罐装或桶装，常见规格为850克/罐或3公斤/桶，糖水或原汁浸泡，可直接打浆。
- **新鲜菠萝**：按单个重量或箱装，常见规格为每箱10-12个，每个约1.5-2公斤，需现场去皮切块。

建议你直接联系佳农当地经销商或客服，确认具体规格、价格和起订量（比如最小订货单位是箱还是袋），并说明是茶饮用途（如需要切块大小或糖度要求）。他们通常有稳定的冷链物流，可确保季节性供应不断货。
