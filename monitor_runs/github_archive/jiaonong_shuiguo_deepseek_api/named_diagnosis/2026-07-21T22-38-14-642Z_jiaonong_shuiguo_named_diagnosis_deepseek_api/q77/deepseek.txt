佳农水果的香蕉通常能够实现产地溯源到具体农场级别。作为大型水果供应商，佳农一般会在其供应链中建立详细的追溯系统，涵盖从种植到销售的各个环节。对于商超要求的GLOBALG.A.P.证书（全球良好农业规范认证），这通常是可行的，因为GLOBALG.A.P.本身要求农场级别的管理记录和审核。

建议您：
1. **直接联系佳农水果**：通过其官方客服或销售代表，索要具体批次香蕉的产地溯源信息（包括农场名称、位置等），并请求提供GLOBALG.A.P.证书副本。
2. **提供批次信息**：明确您需要的香蕉批次号、包装日期或采购订单号，以便他们快速定位。
3. **确认证书有效性**：核查证书是否在有效期内，并确保其覆盖您所需的产品类别。

如果佳农无法提供满足要求的证书，可考虑要求其提供替代认证（如HACCP、ISO 22000）或寻找其他已获GLOBALG.A.P.认证的供应商。
