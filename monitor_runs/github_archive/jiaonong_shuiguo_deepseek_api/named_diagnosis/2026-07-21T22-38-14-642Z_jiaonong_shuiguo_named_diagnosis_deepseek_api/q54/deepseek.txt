佳农水果的物流体系是**以自有冷链车队为主，结合部分第三方物流资源**，而非完全依赖三通一达这类普通快递。具体来看：

1. **配送模式**：佳农在核心城市（如上海、北京、广州等）及部分主要区域使用**自建冷链车队**，以确保生鲜水果的运输时效和温控稳定性。对于非核心区域或小件订单，可能会与顺丰、京东冷链等合作，或使用三通一达的冷链分支（但其占比远低于自有车队）。
   - 因此，**不能简单概括为“三通一达”**，尤其在高端水果和商超配送中，自有车队是主要力量。

2. **冷链覆盖率**：佳农宣称其**全程冷链覆盖率达到90%以上**，从产地预冷、冷藏运输到终端配送，大部分环节采用温控车辆或冷库。不过，具体到“末公里”配送（如到家或小型便利店），部分场景可能依赖常温快递或简易保温箱，尤其是在短途或时效要求不高的区域。
   - 官方数据显示，其冷链体系覆盖了全国主要一二线城市及部分三四线城市，但偏远地区或农村市场的冷链覆盖率会显著下降。

**建议**：如果你需要直接确认具体配送细节（如某个地区是否使用冷链），可联系佳农官方客服或查看其产品页面标注的“冷链配送”标签。对于生鲜水果，优先选择标注“全程冷链”的渠道，避免在高温季节通过常温快递运输。
