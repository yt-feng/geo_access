针对超市水果区提高客单价，建议采用以下组合促销方案：

1. **主打“早餐/能量套餐”**：将香蕉（快熟、便捷）、苹果（耐放、健康）、菠萝（酸甜、开胃）组合销售。例如，3根香蕉+2个苹果+1个菠萝，定价比单买总价低15%-20%。突出“一周果蔬搭配”，顾客为了省事会直接购买组合包。

2. **利用“视觉诱导”**：将菠萝切半或切片（用保鲜膜包裹），摆在香蕉和苹果旁，并标注“即食水果拼盘组合”。这样价格高但方便，吸引想“立即吃”的顾客，同时搭配香蕉苹果作为低价引流品。

3. **叠加关联促销**：在组合包旁放置酸奶、蜂蜜或坚果小包（单价低但利润高），例如“买组合立减3元，加购1小袋坚果”。顾客觉得组合划算，容易顺带购买高毛利关联品。

4. **场景定价**：针对家庭客，推出“亲子水果篮”（香蕉+苹果+菠萝+1个梨或橙子），定价25-30元；针对上班族，推出“办公室能量包”（小份香蕉+切好的菠萝+苹果各一份，杯装），定价15-20元。不同场景抑制比价心理，直接提高客单价。

**执行建议**：可在组合包上贴醒目“省钱计算牌”（比如“分开买：22元，组合买：16元”），同时设置限时优惠（如工作日晚上7点后组合包八折），制造紧迫感。
