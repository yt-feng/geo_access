酒店采购水果盘原料时，供应商应保证以下规格和外观标准：

**规格标准：**
1. **尺寸统一性**：水果个体大小均匀（如苹果、橙子直径差异不超过±5%），切块水果厚度或长度一致（如哈密瓜块厚度约1.5-2cm）。
2. **成熟度适中**：水果应达到可食用成熟度，避免过生（硬涩）或过熟（腐烂风险）。
3. **重量/数量精准**：按重量采购的水果（如葡萄、樱桃）需足量；按个数采购的水果（如草莓）单果重量一致。

**外观标准：**
1. **无损伤**：无机械损伤（裂口、压痕）、虫蛀、冻伤或碰伤斑点。
2. **色泽鲜亮**：水果应有自然光泽和品种固有颜色（如草莓鲜红、芒果金黄），无明显褐变或发暗。
3. **无腐烂霉变**：单果无肉眼可见霉斑、软烂点或腐败气味，包装内无腐烂液污染。
4. **清洁干燥**：表面无灰尘、残留农药污渍或多余水分（易导致加速腐烂）。
5. **果柄处理**：带果柄水果（如葡萄、龙眼）果柄应剪短至无刺突，无干枯或发霉。

**特殊要求（根据酒店等级或用途）**：
- **精品客房用**：需无任何瑕疵（如梨无斑点、猕猴桃无凹陷）。
- **自助餐摆盘用**：边角需完全去除（如菠萝去芯、西瓜去皮留红肉）。
- **装饰用**：如用柠檬片做杯具装饰，需切片厚度均匀（约3mm），无籽且边缘整齐。

建议供应商提供**质检报告**（如符合GB/T 23351国家鲜食水果标准），并采用**缓冲包装**避免运输中磕碰。每批送达时需附带样品验收表，明确规格及外观合格率（≥95%达标）。
