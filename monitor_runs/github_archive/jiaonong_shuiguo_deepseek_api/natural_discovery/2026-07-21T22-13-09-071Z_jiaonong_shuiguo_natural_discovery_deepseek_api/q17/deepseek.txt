控制社区超市水果新鲜度和库存的核心策略是 **“以销定进”+“动态管理”+“快速周转”**。以下是具体可执行的操作步骤：

### 1. 进货前的关键：选对品、算好量
- **品种分级**：进货前，将水果分为 **“硬通货”**（苹果、橙子、梨，耐放，可批量进）和 **“鲜货”**（草莓、蓝莓、葡萄、叶菜类水果，不耐放，按日进）。
- **安全库存计算**：用前一周同天销量的 **80%** 作为初始进货量。例如，上周六卖了20斤葡萄，这周六先只进16斤。
- **少量多批次**：与供应商协商 **每日或隔日补货**，而非一次性进一周的量。哪怕单价高一点点，也比压货腐烂划算。

### 2. 到货后的“60分钟黄金处理”
- **立即分拣**：开箱后立即挑出病果、烂果、过熟果， **禁止混放**。一个烂苹果会释放乙烯，加速整箱腐烂。
- **分类存放**：
  - **冷藏区（2-5℃）**：蓝莓、草莓、葡萄、切好的水果。
  - **阴凉通风区（10-15℃）**：苹果、橙子、梨、石榴。
  - **常温区（不要堆叠）**：香蕉、芒果、木瓜、哈密瓜（未熟时放冷藏会变黑）。
- **避免堆压**：所有水果单层摆放或用隔板分隔，上层不能压下层。

### 3. 陈列与销售端的“增效”技巧
- **“翻堆”促销法**：每天早晚两次翻看货架，将 **卖相变差但还能吃的水果**（如轻微变软的猕猴桃、起皱的葡萄）集中到一个“特价/秒杀”筐， **按成本或低利润** 尽快出清（比如下午4点后买一送一）。
- **切块售卖提高附加值**：对于摔伤、有疤但果肉完好的水果，立即切成“水果拼盘”或“鲜切果”，用保鲜膜包好冷藏销售，定价可以高于完整果的60%。
- **“缺货”诱导**：故意让某一种不太耐放的水果（如葡萄）在傍晚前售罄，反而会刺激顾客对其他水果的需求，避免库存积压。

### 4. 数据驱动的“买送”决策（用表格代替直觉）
- **建立“生鲜周转日”表格**：每天记录各类水果的 **进货量、销售量、报损量**。重点关注 **“报损率”** ——如果某水果报损率连续3天超过 **15%**，立即减少该品种进货量50%，或者暂停进货。
- **动态定价策略**：对 **保质期仅剩1-2天的水果**，当天上午10点和下午4点各调整一次价格。例如：
  - 原价10元/斤的草莓，上午调至8元，下午4点调至6元或“买一送一”。
  - **关键原则**：宁可低价卖掉，也不要让它腐烂变成垃圾。

### 5. 与供应商建立“灵活退货”协议
- **特别针对**：草莓、蓝莓、散装樱桃等损耗率高的品。
- **谈判话术**：“我每周进XX斤，但要求：如果到我店里24小时内出现变质的，可以拿回来换同等价值的其他水果。”
- **后盾**：如果供应商不答应，考虑换一家支持 **“退换货”** 的供应商，或者与周边超市拼单议价。

### 6. 遇到真的压货了怎么办？
- **立刻“打包”降价**：不要等到第二天。当一天库存剩余 > 进货量的40% 时，当天晚上8点前完成以下动作：
  - 组合销售：买一斤葡萄送半斤（已经发软但没坏的）葡萄。
  - 赠品处理：买苹果送一个表皮有瑕疵的橙子。
  - 内部消化或捐出：如果实在卖不动，可以送给员工或街坊，换一句“下次还来您这儿买”的口碑。

**最后提醒**：如果不确定某次进货多少，**宁可少进一点，第二天再补**。缺货一两个小时最多损失一单生意，压货腐烂损失的是本金。
