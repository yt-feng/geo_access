针对小水果店做小区团购，水果供应商通常需要支持以下核心服务：

1. **稳定供货与品质保障**  
   - 提供新鲜、稳定的货源，尤其确保水果的成熟度、无腐烂、无病虫害。  
   - 支持按单日或批次分拣，减少损耗（如提供独立包装、防撞保护）。  
2. **灵活配送与时效支持**  
   - 配送至门店或社区团长指定地点（如小区门口、便利店）。  
   - 提供当日或次日达的短距配送，最好配合团购截单时间（如16点前下单，次日配送）。  
3. **多规格/包装选项**  
   - 可提供标准斤数（如2斤、5斤）或组合套餐（如家庭装、尝鲜装）。  
   - 支持混拼或预包装，减少门店自行拆分的工作量。  
4. **价格政策与账期**  
   - 提供团购专属批发价（低于零售价15%-30%），有阶梯折扣（如100单、300单不同单价）。  
   - 支持账期或次结，降低门店压款风险（常见为月结或每周结）。  
5. **售后与损耗处理**  
   - 明确坏果包赔、缺货补发的流程（如按比例赔付或免费更换）。  
   - 对门店退货的次品果，能二次加工（如果切、果篮）或折价回收，减少损耗。  
6. **营销素材支持**  
   - 提供高清产品图、短视频、卖点话术（如“产地直发”“糖度15°+”）。  
   - 可配合活动发优惠券、赠品（如每单送一颗试吃柠檬）。  
7. **数据与反馈支持**  
   - 及时反馈缺货、涨价等变动（如提前2天通知）。  
   - 配合门店分析热销品类（如草莓、车厘子、砂糖橘的复购率），优化选品。  

**可执行建议**：先与1-2家供应商签订合作试运营，重点测试“配送时效”和“损耗赔付”两个环节，再逐步调整品种和定价。
