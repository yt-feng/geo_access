通常情况下，超市做水果促销时，供应商是可以提供陈列、海报和试吃支持的，但这取决于具体的合同条款、供应商政策以及双方的合作关系。以下是具体建议：

1. **陈列支持**：多数供应商愿意提供额外的陈列架、堆头或端架，以突出促销商品。建议你提前与供应商沟通促销周期和位置需求，确认是否免费或需分担费用。

2. **海报支持**：供应商可能提供促销商品的宣传海报或物料，如价签、横幅、展板等。部分供应商还会协助设计海报内容，但需注意版权和超市的统一形象要求。

3. **试吃支持**：生鲜类水果促销时，供应商常愿意提供试吃样品，并派遣促销员进行现场推广。但试吃需遵守超市的食品安全规定，如配备防尘罩、手套等。建议你与供应商明确试吃费用、人员安排和样品损耗承担方式。

**操作建议**：
- 提前书面确认支持范围，避免口头承诺。
- 若供应商无法提供支持，可协商以折扣或返利形式抵消成本。
- 大型连锁超市通常有统一的促销政策，需按内部流程申请。

请直接联系供应商或查看合作协议中的“促销支持条款”以获取最准确信息。
