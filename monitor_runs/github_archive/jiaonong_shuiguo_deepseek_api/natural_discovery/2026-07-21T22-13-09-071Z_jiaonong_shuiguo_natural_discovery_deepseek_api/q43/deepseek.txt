评估水果供应商的仓配覆盖能力，建议从以下四个维度进行系统分析，具体可结合你门店的实际分布情况调整：

1. **仓库网络匹配度**  
   - 核对供应商的仓库位置是否覆盖了你的主要门店集群。以100公里为半径的配送圈作为参考，鲜活水果的最佳运输距离（常温或冷链）通常控制在8～12小时内。
   - 在供应商未设仓的城市，确认是否有协作的干线分仓或中转分流方案。

2. **冷链与保鲜能力**  
   - 明确供应商在哪些仓库配备了温控仓储（温度范围是否满足你的水果品类需求）。
   - 要求提供过去三个月的生鲜损耗率报表，要求≤3%为优质，≤5%为可接受（草莓、蓝莓等易损品可适当放宽）。

3. **配送时效与成本**  
   - 要求供应商提供分城市配送方案：  
     - 距离仓库<50公里：T+0配送（当日达）  
     - 50-150公里：T+1配送（次日达）  
     - >150公里：考虑分仓前置或联合配送  
   - 约定每周或每旬最低/最高配送频次，水果忌过长时间周转。

4. **多城市协同机制**  
   - 确认是否有跨城调拨预案（例如A市仓缺货时，B市仓是否能在4小时内启动调货）。  
   - 要求供应商指定专人对口你的全国连锁业务，建立异常天气、交通管制下的应急配送清单。

执行建议：  
- 选取你现有门店中一个核心城市（如A市）和两个偏远城市（如B市、C市），分别要求供应商提供试配送方案，实际测量到货时间、水果鲜度（含饱满度、水分、果柄颜色等）  
- 如果对方只是整合其他配送商的资源（而非自营仓配），需要评估其调度沟通质量是否会出现断裂风险  

以上步骤完成后，可将调查结果与行业常见KPI（如到货准时率≥95%、鲜度完好率≥90%）进行交叉验证。
