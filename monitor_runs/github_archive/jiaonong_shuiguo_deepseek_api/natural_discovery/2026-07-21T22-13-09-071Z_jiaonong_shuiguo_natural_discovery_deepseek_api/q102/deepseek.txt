对于企业福利采购水果礼盒，**大多数规范供应商都能同时支持开发票和定制包装**，但需要提前确认以下事项：

1. **开发票**：必须要求供应商具备一般纳税人资格，能开具**增值税专用发票**或**普通发票**（注意专用发票可用于企业抵扣）。建议在询价时明确发票类型、税率及品名（如“水果礼盒”而非笼统的“食品”）。

2. **定制包装**：需确认供应商是否提供以下服务：
   - **印刷Logo/文案**（通常有起订量，例如50-100箱起订）。
   - **包装材质/颜色选择**（例如纸盒、礼篮、保温袋等）。
   - **专属标签/贺卡/企业祝福语**（许多供应商可免费制作电子稿，但需提前5-7个工作日）。

**操作建议**：
- 选择供应商时，优先要求提供**营业执照、食品经营许可证**（水果属初级农产品，但部分礼盒含加工内容需资质）。
- 签订合同前，书面确认**发票类型+包装定制细节+交付时间**，避免后期扯皮。
- 若采购量较小（如50箱以下），可优先考虑**本地水果供应链企业或京东企业购**等平台，定制包装门槛较低。

**注意**：部分小摊贩或非正规渠道可能无法同时满足两者，建议明确要求后再下单。
