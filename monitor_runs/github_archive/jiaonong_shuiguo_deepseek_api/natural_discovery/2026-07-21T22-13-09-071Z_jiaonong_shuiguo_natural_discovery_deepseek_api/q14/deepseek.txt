一般来说，供应商是否给支持，主要取决于**合作基础**和**促销规模**。以下是常见的支持情况供你参考：

1. **价格支持**：多数供应商愿意在周末、节假日给予**临时特价**（比如每箱降5-10元），尤其是当你的订货量达到一定门槛（例如一次订20箱以上）。部分供应商还会支持“买赠”：比如你进30箱苹果，可以额外送1箱，用于引流。

2. **物料支持**：常见的有**宣传物料**（如海报、价签、展架），如果供应商品牌较大，还能提供**试吃装**或**免费小包装样品**，用于现场推广。

3. **关键谈判点**：你需要主动提出“周末集中主推某款产品”，并承诺在店内做显著陈列（如门口堆头）。大部分供应商更看重**销量提升**和**品牌曝光**，只要你能证明周末活动对销量有明显拉动，他们通常愿意投入。

4. **建议话术**：  
   “老板，我打算周末主推你家的（某某水果），预计能走50箱以上。你能不能给这个周末一个特价，或者赞助一些试吃品？我能保证把最好的位置留给你。”

⚠️ **注意**：如果是首次合作的小供应商，物料支持可能较少；而长期合作的、或代理知名品牌的供应商，更容易争取到折扣和赠品。最好提前1-3天沟通，同时说清促销时长和预期销量，对方决策会更快。
