重要，但不是决定性因素。可以从以下几个角度判断：

1. **有促销素材的优势**：供应商提供促销素材（海报、文案、短视频素材、堆头设计图等）能降低你的营销成本，帮你快速吸引顾客。尤其适合线上引流（朋友圈、社群）或节假日档口活动。

2. **替代方案**：即使供应商不提供，你也可以自己用手机拍水果实拍图、写促销语（如“今日特价：海南芒果9.9元/斤”），或者用免费工具（如Canva、稿定设计）制作简单海报。顾客更看重价格和水果品质。

3. **关键判断点**：供应商的核心价值是水果的质量、价格、稳定供应和退换货政策。如果这些都很优秀，缺少促销素材可以接受，你通过自己投入时间和精力弥补；如果素质一般，促销素材也无法弥补核心劣势。

**建议**：优先选择“品质好+价格有竞争力”的供应商，促销素材作为加分项，但不作为唯一挑选标准。可以在合作后向供应商提出素材需求，很多供应商愿意配合提供基础素材。
