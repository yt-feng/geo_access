判断香蕉熟度和坏果风险，可以从以下几方面入手：

### 1. 看颜色（熟度）
- **青绿色**：未熟，适合长途运输或需要长时间存放。
- **黄绿相间（亮黄带绿）**：七成熟左右，适合短期批发后3-5天内售卖。
- **全黄（无青）**：九成熟以上，需在1-2天内销售或进冷库。
- **出现黑斑（芝麻点）**：成熟度高或受过碰撞，极易快速腐烂。

### 2. 摸硬度（坏果风险）
- **整把捏起来整体硬实**：新鲜、未熟。
- **个别果身明显变软（尤其是果柄处）**：内部已开始变质或回软，易有坏果。
- **按压果肉有凹陷不回弹**：已经熟透或受伤，需尽快挑出。

### 3. 检查果柄和切口
- **果柄（切口）发黑、发霉或有黏液**：极可能是运输中受潮或存放太久，坏果风险高。
- **果柄新鲜、切口干爽**：整体状态较好。

### 4. 看表面损伤
- **大面积擦伤、压伤、裂口或水渍状软斑**：会快速传染其他香蕉。
- **有白色或灰绿色毛状霉斑**：已霉变。

### 5. 闻气味
- **正常清香**：熟度适中。
- **发酵酒味或酸味**：内部已大量腐烂，不可批发。
- **霉味或土腥味**：冷库或存放环境受潮。

### 快速判断方法（针对整箱）
- **开箱前先闻**：有异味直接拒收。
- **随机抽3-5根看果柄颜色、摸软硬**。
- **轻轻拿起整把，观察是否够结实**（过软说明熟度太高）。
- **留意箱底是否有积水和碎烂果**，烂果汁会加速传染。

### 应对建议
- **若一箱中发现1-2根坏果**：立即挑出，该批次应尽快销售。
- **若超过5%的香蕉有明显软斑、柄黑或异味**：建议拒绝收货，或要求降价后立即分拣冷藏。
- **批发后若要存放**：未熟的放10-14℃通风处；已熟放8-10℃冷库或冷藏柜。
- **不要与苹果、梨同放**：会加速成熟，导致批量腐烂。
