采购菠萝时判断熟度可从以下几个方面入手，建议优先选择八九成熟的菠萝（即已开始变色但果肉未过度软熟），既能保证运输储存过程中减少损耗，又能确保到店后快速出清：

1. **看表皮颜色**：绿色越多说明越生，适合长途运输；黄色面积达1/3至1/2左右为最佳采购熟度，此时果肉已开始变甜但不会过快腐烂。
2. **轻压果身**：手指轻按菠萝中部，如果感觉坚实有弹性（类似按苹果），说明熟度适中；若明显软塌甚至渗水，说明已过熟易损耗。
3. **闻气味**：靠近底部闻，有淡淡清香最好；无味说明太生，甜味过重或酒味说明过熟。
4. **掂重量**：同等大小选更沉的，说明水分充足、熟度均匀；过轻的可能内部失水或空心，易坏。
5. **查果眼**：果眼（表面凸起的鳞片）如果大而饱满、间距明显，通常熟度较好；不饱满或干瘪的果肉品质差。

**实操建议**：采购时混合拿货，70%选黄绿相间（可以放2-3天成熟的），30%选略更黄（当天或次日能卖出），可有效平衡损耗与销售。另外冬季或运输时长超过3天时，选偏生1-2天更安全。
