避免甜品店香蕉到店后快速变黑，关键在于**控制乙烯气体**和**低温抑制酶活性**。以下是具体可操作的方法：

1. **采购时选择成熟度适中**：不要买全黄或带斑点的。选**青皮带黄、手感紧实**的香蕉，可多留出2-3天自然成熟窗口。

2. **运输与卸货**：避免堆叠挤压，用透气纸箱或竹筐盛装。**不要将香蕉与苹果、梨、牛油果混放**，它们会释放乙烯加速催熟。

3. **到店紧急处理**：
   - 立即拆开塑料袋或保鲜膜，放阴凉通风处散去运输中积蓄的乙烯。
   - 如果短期用不完，**放入冷藏（10℃-12℃）**，但不要低于8℃，否则会变褐且丧失风味。随取随用，冷藏后的香蕉肉可能发暗但皮仍黄。

4. **成熟香蕉的应对**：如果已开始有黑点，尽快做成熟甜品使用（如派馅、蛋糕糊），或冷冻（去皮切段密封）延长至2-3周。

5. **清洗**：到店后**不要用水洗**，水会破坏免疫力屏障，加速霉变。如需清洁，用干布擦去灰尘即可。

销售计划：每天评估剩余香蕉，优先用较熟制作产品，避免累积。
