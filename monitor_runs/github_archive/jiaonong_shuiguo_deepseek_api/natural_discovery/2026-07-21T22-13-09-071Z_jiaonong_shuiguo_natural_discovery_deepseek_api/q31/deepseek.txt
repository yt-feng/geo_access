为连锁商超招标长期水果供应商，评分标准需兼顾品质稳定性、供应能力、成本控制及合作潜力。建议采用“百分制”体系，将评分分为硬性条件（60%）和软实力（40%），具体设置如下：

**一、基础资质与合规（20分）**
1. 营业执照、食品经营许可证、水果产地证明（必备，缺一票否决）。
2. 近2年无重大食安事故记录（需提供第三方检测报告或政府证明，5分）。
3. 年销售额或供应规模（如：年供应量≥XX吨得10分，每少20%扣2分）。

**二、产品质量与标准（30分）**
1. 产品规格：分等级（如特级、一级）对应不同果径、甜度阈值（如草莓糖度≥12度），提供达标承诺书（10分）。
2. 安全检测：需提供每批次农残、重金属、添加剂检测报告（10分），可接受第三方随机复检加5分。
3. 包装标准：统一品牌、规格箱装，防止运输损耗（5分），使用可回收材料额外加5分。

**三、供应稳定性与物流（25分）**
1. 基地规模：自有/合作种植面积、年产量（10分），需提供土地流转合同或合作协议。
2. 物流时效：承诺48小时内送达门店，冷链覆盖率≥90%（10分），可提供备用物流方案加5分。
3. 应急响应：缺货时24小时内可调货或提供替代品（5分）。

**四、成本与商务条款（15分）**
1. 价格竞争力：报价低于市场均价10%以上得10分，低于5%得5分，持平得3分。
2. 付款周期：接受60天账期或更长得5分，月结扣2分。
3. 签署长期协议（1-5年）并有固定折扣阶梯（5分），单年度协议0分。

**五、合作潜力与增值服务（10分）**
1. 季节品种组合能力：如夏季能供应新疆哈密瓜、冬季提供进口车厘子，按种类数量评分（5分）。
2. 售后与技术支持：提供陈列建议、损耗控制方案（5分），每周派人巡查门店加3分。

**评审流程建议：**
- 先对硬性资质打分（低于60分排除）。
- 对入围者进行盲测：随机抽取3款水果（如苹果、蓝莓、猕猴桃）由质检组品鉴，按口感、新鲜度打分（10分）。
- 实地考察产地、分拣线、冷库，确认数据真实性（一票否决权）。
- 最终得分=50%技术标（上述评分）+30%价格标（最低价者满分，其他按比例扣分）+20%商务谈判（付款、合同灵活度）。

**风险控制：**
- 设置“黑名单”：连续2次供货质量不合格或违约的供应商，3年内禁止投标。
- 动态调整：每年复盘评分标准，根据市场变化（如气候对产季影响）优化阈值。

建议邀请第三方食安机构参与评审，并保留“样品签订品质约束条款”在合同中。根据实际需求，可对成熟供应商（如已合作3年以上）设置加分项（提供上年度问题品汇报率）。
