是的，起订量太高会增加损耗风险。原因如下：

1. **保鲜期短**：香蕉和菠萝属于易腐水果，保质期通常只有几天到一周。起订量过大，若销量无法快速匹配，容易因存放过久导致腐烂或品质下降。

2. **库存压力**：大量存货需要足够的冷藏空间，若存储条件不当（如温度、湿度控制失衡），会加速水果变质。

3. **销售周期**：如果客户需求不稳定或季节波动，高起订量可能导致滞销，尤其是菠萝需后熟处理，存储不当易过熟。

**建议**：  
- 根据历史销售数据和季节性需求，设定合理的起订量（如单次订单不超过3-5天销量）。  
- 与供应商协商灵活分批到货，或采用“小批量多频次”采购模式。  
- 对菠萝，可优先采购未完全成熟的果实，延长可控存储期。  

若库存已过高，考虑降价促销、与餐饮合作或加工成果汁/果干以减少损失。
