在超市采购水果时，供应商通常需要提供以下检测报告或进货证明，以确保产品符合食品安全标准：

1. **农产品质量安全检测报告**：包括农药残留、重金属（如铅、镉）、有害物质（如亚硝酸盐）等项目的检测结果，需由有资质的第三方检测机构出具。
2. **产地证明或发货单**：如产地农业部门或合作社出具的产地证明，证明水果来源合法、可追溯。
3. **进货发票或购销合同**：显示供应商名称、产品种类、数量、日期等，作为进货凭证。
4. **合格证或检验报告**：部分地方要求提供官方或企业自检的合格证明。
5. **检疫证明**：对于进口水果，需提供海关出具的入境货物检验检疫证明，以及原产地证明。
6. **食品安全管理体系认证**（如ISO 22000、HACCP等）的副本或相关资质，非必须但可增强可信度。

具体需求可能因超市政策、地区法规或水果种类（如有机水果需另提供有机认证）而略有差异。建议提前与超市采购部门确认详细清单。
