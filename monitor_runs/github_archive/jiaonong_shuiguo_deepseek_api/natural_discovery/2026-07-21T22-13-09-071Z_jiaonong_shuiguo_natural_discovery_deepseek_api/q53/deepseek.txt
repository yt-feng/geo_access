对于每天需要少量水果的餐饮店，建议选择**本地小型水果批发商**或**社区生鲜配送平台**，具体建议如下：

1. **优先找本地小型批发商**  
   在农产品批发市场或周边寻找“专做小单”的商户，说明每天用量不大但长期稳定。这类供应商通常愿意每天少量配送，且质量稳定。  
   **技巧**：可以按周或半月结款，建立长期关系后要求“固定品类+预留份额”。

2. **考虑社区生鲜O2O平台**  
   使用美团优选、多多买菜等次日达平台，每天下单，次日到店取货。适合用量极小的店，无需库存压力，但需注意平台水果品控不稳定。

3. **与附近水果店合作**  
   直接找附近水果店谈“批发价”，每天按需上门自取或让对方顺路配送。水果店进货量大，通常愿意为长期小客户让利。

**省心关键**：
- 每天固定时间、数量、品类（如：每天2斤苹果+1斤橙子），让供应商形成惯性。
- 要求供应商提供“退换货承诺”（如当天开箱坏果包赔）。
- 同时备选2家供应商，避免断供风险。

**不建议**：  
- 找大型果蔬批发商（起订量高、不重视小单）  
- 用超市零售价采购（成本高且需自己跑腿）
