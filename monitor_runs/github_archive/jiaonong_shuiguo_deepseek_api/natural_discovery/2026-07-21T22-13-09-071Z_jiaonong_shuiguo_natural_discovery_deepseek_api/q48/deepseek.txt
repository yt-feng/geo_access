要判断茶饮连锁的菠萝供应商是否能稳定按规格送货，可以采取以下可操作的措施：

1. **明确规格标准**：在合同中写明菠萝的具体要求，如单果重量（例如1.2-1.5斤/个）、成熟度（如7-8成熟）、果形（椭圆形无畸形）、无损伤、切面标准等，并附上图片示例。

2. **要求样品试送**：让供应商提供3-5批次的试送货，每批至少50-100箱，连续试送2-3周。检查每批次的规格一致性，包括重量、外观、成熟度等。

3. **设定验收流程**：在收货时进行抽检，比如每车抽10%的箱数，每箱抽3-5个菠萝。若不符合规格比例超过5%，要求供应商即刻整改或拒收。

4. **考察供应商的生产和冷链能力**：实地查看供应商的种植基地（是否规模化）、分选线（是否有分级设备如重量筛选机）、冷链仓储（是否有冷库和冷藏车，保证菠萝在12-15℃运输）。

5. **检查历史履约记录**：要求供应商提供过去6个月给其他连锁客户（尤其是茶饮或餐饮客户）的送货记录，重点看批次合格率（应≥95%）、准时率（≥98%）。

6. **引入阶梯式处罚与调整机制**：合同中明确约定：若连续2次送货不合格，减少订单量50%；若连续3次不合格，终止合作并索赔。同时，要求供应商提前备货并设安全库存（比如多备20%的货源）。

7. **季节性风险控制**：菠萝有季节性，询问供应商是否有多产地（如广东、海南、云南）轮换供应计划，或在淡季是否有稳定冷库储备，确保全年供应。

8. **定期绩效评估**：每月根据到货记录、退货率、客户反馈（如门店反馈菠萝成熟度、切面质量）评分，低于80分要求整改，低于60分立即替换。

通过以上步骤，你能系统地判断供应商的稳定性。如果供应商无法提供连续试送样品或拒绝接受抽检条款，通常代表其能力或意愿不足。
