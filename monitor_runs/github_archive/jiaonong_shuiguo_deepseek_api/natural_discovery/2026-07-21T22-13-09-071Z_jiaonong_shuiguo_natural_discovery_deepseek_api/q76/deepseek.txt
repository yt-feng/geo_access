针对生鲜电商配送香蕉的压坏和变黑问题，建议从以下环节重点控制：

1. **采收成熟度控制**：选择7-8成熟的香蕉发货（果皮仍为青绿色，果肉未软），避免完全成熟后配送。此时香蕉硬度高，耐碰撞，且乙烯释放较少，可延缓后熟变黑。

2. **包装防护**：
   - 使用**独立瓦楞纸托盘或珍珠棉凹槽**固定每根香蕉，防止相互挤压和滚动碰撞。
   - 果柄朝内放置，减少果皮与包装接触点。
   - 包装内放入**乙烯吸收剂**（如高锰酸钾吸附剂或活性炭包），降低催熟气体浓度。
   - 避免使用塑料袋密封，改用**透气包装**（如带孔的PE袋或纸箱内置通风孔），防止水汽积聚加速霉变。

3. **冷链运输**：
   - 配送全程保持**13-15℃**（香蕉最适储藏温度），温度过高会加速成熟变黑，过低（<10℃）会导致冻伤发黑。
   - 短途配送可用**冰袋+保温箱**，但需用报纸或缓冲材料隔离冰袋与香蕉，避免直接接触冷凝水。

4. **分拣与装箱**：
   - 装箱前剔除已有擦伤、黑斑或受挤压的香蕉。
   - 采用**多层隔板分层放置**，单层高度不超过15cm，避免堆叠过高压坏底部。

5. **配送时效与预警**：
   - 在包装箱外标注“易压伤”“请轻放”“禁颠倒”，并与快递/配送公司签订**防损责任协议**。
   - 对配送员进行培训：香蕉需水平放置，不能竖向堆叠，避免急刹车导致位移碰撞。
   - 设置**签收后2小时以内开箱检查**的提示，如有问题拍图理赔。

6. **售后预判**：发货前在包裹内附一张“食用指南”，说明香蕉未完全成熟时请放置于室温自然催熟，已成熟后立即冷藏并5天内食用，减少因用户误判导致的差评。
