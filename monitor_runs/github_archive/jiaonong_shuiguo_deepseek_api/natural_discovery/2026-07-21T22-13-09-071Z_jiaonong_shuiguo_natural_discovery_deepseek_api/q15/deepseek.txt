水果店遇到坏果的结算或赔付方式，通常取决于合同中或采购时双方约定的条款。以下是一般的行业标准做法，供你参考：

1. **确定责任归属**  
   - **运输途中损坏**：如果是物流破损、挤压或运输时间过长导致，通常由供应商或物流方承担。  
   - **到货后发现坏果**：需在到货后**24-48小时内**（部分品类更短）完成验货并保留证据，提交给供应商。超过时限可能视为货源方接收。

2. **常见赔付方式**  
   - **按比例折价**：根据实际坏果重量/数量，在应付货款中扣除对应金额。例如，坏果占10%，则按货款的90%结算。  
   - **按比例补货**：供应商下次发货时，免费补充同等数量或折合的坏果。  
   - **全赔/换货**：如果整批严重腐坏（如运输冷藏脱落），可要求全额退款或重新发货。

3. **关键操作步骤**  
   - **立即拍照/录像**：记录坏果的完整状态、包装、物流单号及时间。  
   - **保留证据**：保存到货签收单、物流底单、包装箱上的批次信息。  
   - **沟通并书面确认**：立即通知供应商，明确要求其书面回复赔付方案（微信聊天记录、邮件等也可作为证据）。  
   - **考虑合同条款**：如果此前合同明确约定“若有坏果需在X小时内反馈并附照片”，务必遵守时限。

4. **注意事项**  
   - 避免口头承诺，最好书面留痕。  
   - 对于生鲜类商品（如草莓、荔枝），坏果容忍率通常比耐储水果（如苹果、橙子）高。  
   - 如果供应商长期不赔付，可保留证据并终止合作，或向当地市场监督部门投诉。

**建议**：在下次进货前，务必与供应商书面确认坏果赔付标准，明确“验货反馈时限”“赔付比例”等细节。如果没有合同，至少需在微信等渠道留下双方同意的记录。
