建议优先选择**冷冻果丁**或**冷冻果泥**。具体选择取决于以下因素：

1. **鲜果**：适用于高客单价、强调新鲜手作的店铺，但存在季节性、损耗高、切片大小不均、需额外加工（削皮去芯）等问题。若每日销量大且对品控要求极高，可考虑，但成本较高。
2. **冷冻果丁**（速冻菠萝丁）：是最推荐的**中间路线**。标准化程度高（大小均匀）、出杯快、保质期长（-18℃冷冻可存半年），风味接近鲜果（现代速冻技术能较好保留香气），稳定性优于鲜果。适合多数茶饮店。
3. **冷冻果泥**：适合追求**浓郁口感**和**省去过滤步骤**的场景（如果茶中需突出菠萝的纤维感或做挂壁效果）。但果泥可能含更多糖分或添加剂，且口感较糊状，不如果丁清爽。

**采购建议**：
- 若主营果茶（如菠萝百香果、菠萝乌龙），**冷冻果丁**是综合成本、出杯效率、客户满意度的较优选择。
- 若需制作**分层果茶**或**泥状打底**，选用无糖无添加的**冷冻纯果泥**。
- 务必向供应商索要**第三方检测报告**（农残、防腐剂），并**试制样品**，对比真实出品口感与损耗率。
