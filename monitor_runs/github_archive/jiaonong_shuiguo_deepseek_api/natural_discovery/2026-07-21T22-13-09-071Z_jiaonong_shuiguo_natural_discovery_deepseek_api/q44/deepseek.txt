判断商超采购中菠萝和香蕉供应商的售后响应稳定性，建议从以下可执行步骤入手：

1. **明确售后条款**：在合同中写明具体响应时间（如24小时内反馈）、问题处理流程（如退换货、赔偿标准）及联系方式。
2. **获取可追溯的案例**：要求供应商提供近3个月内的售后处理记录，包括问题描述、处理时间、客户反馈。优先选择有公开售后评价的供应商。
3. **模拟测试响应**：在沟通中故意提出一个小问题（如询问票据或规格），记录供应商的回复时长和质量（如是否专业、主动解决）。
4. **实地或远程考察**：检查供应商的仓储或办公环境，确认是否有专职售后人员、是否配备客户系统（如CRM）。
5. **历史合作参考**：询问其他零售商或同行（非竞争对手）的口碑，尤其是关于售后环节的真实体验。如果供应商是知名品牌或长期经营，可靠性通常更高。
6. **小批量试单**：先采购小批量菠萝和香蕉，实际测试售后流程（如运输损耗、品质问题处理），观察响应速度和解决态度。

重点关注：避免依赖口头承诺，所有沟通保留书面记录（邮件、聊天截图）。菠萝和香蕉易损耗，售后响应速度直接影响成本。
