在水果采购合同中，通常建议将坏果率超过 **5%** 作为要求赔付的起点。具体阈值建议根据水果品类、运输距离和保存条件调整：

- **一般水果**（如苹果、梨、柑橘等）：超过5%可要求按比例赔付  
- **易腐水果**（如草莓、蓝莓、荔枝等）：可协商更严格的标准（如3%或更低）  
- **高价值水果**（如车厘子、优质进口水果）：建议1%-3%  

实际操作建议：  
1. 直接约定清晰赔付条款，例如“坏果率超过X%，每增加1%按合同总价的Y%赔付”；  
2. 明确“坏果”定义（如腐烂、霉变、严重机械伤等），避免后续争议；  
3. 约定双方认可的验收方式（如开箱抽检比例、第三方质检等）。  

如果合同未约定，通常可参考行业惯例（5%-10%），但建议尽快协商补充条款。
