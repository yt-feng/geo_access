社区超市销售小包装香蕉时，采购环节需要重点关注以下包装和损耗问题：

1. **包装规格与成熟度匹配**：小包装（如2-3根装）应选择**七至八成熟**的香蕉（果皮微黄、略带青绿），避免过熟导致运输中软烂。包装内建议放**成熟度一致**的香蕉，避免一根过熟引发连锁腐烂。

2. **包装材质与透气性**：使用**透气性好的薄膜或网袋**，避免密封塑料袋导致凝露加速霉变。包装底部可加设**防压泡沫垫**，减少挤压磕伤。

3. **损耗控制核心**：
   - **预冷处理**：采购前要求供应商在4-7℃冷库预冷，降低呼吸热。
   - **运输温度**：全程保持12-14℃（低于10℃易冻伤发黑），到店后立即转入常温（15-25℃）催熟区。
   - **验收标准**：拒收长黑斑、软烂、果柄断裂的香蕉，包装内无冷凝水。

4. **包装标识与货架期**：印刷**建议零售价、保质期（常温2-3天）**，并标注“请勿挤压”。小包装建议**分色分类**（如青皮/微黄/黄熟），便于店员按成熟度梯度上架。

5. **供应商合作**：优先选择提供**带轴整串预切装**（减少人工分装二次损伤）、有**运输保损协议**的供应商，并要求每月抽检包装湿度与透气孔密度。

**关键建议**：到货后**立即拆包装检查**，压痕、划伤的香蕉若未腐烂可打折单独售卖出清，避免混入整包加速变质。
