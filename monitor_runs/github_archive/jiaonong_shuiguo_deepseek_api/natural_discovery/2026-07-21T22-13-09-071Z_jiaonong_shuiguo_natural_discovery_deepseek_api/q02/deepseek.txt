对于小水果店来说，进货香蕉时，建议按以下优先级排序：**熟度 > 坏果率 > 价格 > 送货速度**。具体原因和操作建议如下：

1. **熟度（第一优先级）**  
   - 香蕉是后熟型水果，进货时熟度直接影响销售窗口。建议选择**7-8成熟**的香蕉（表皮青黄、果肉硬实），这样能保留3-5天自然成熟时间，避免当天就烂或未熟难卖。  
   - 小店铺空间小，无法长期囤货，高熟度香蕉（全黄或有黑点）当天必须卖出，否则损耗大。可让供应商提供不同熟度混搭的货。

2. **坏果率（第二优先级）**  
   - 香蕉易受压、碰撞后变黑腐烂，进货时务必开箱抽查。建议将坏果率控制在**5%以下**，高于10%的批次直接淘汰（即使价格低）。  
   - 运输中的“三明治”装法（上下铺报纸）或独立纸箱能减少损耗，优先选择这类包装的供应商。

3. **价格（第三优先级）**  
   - 香蕉进货价通常透明，若熟度、坏果率达标，可对比周边3家批发商报价。但不要为低价接受高熟度或高坏果率——香蕉滞销1天，亏损可能超过5%的价格差价。  
   - 注意：冬季香蕉价格普遍高，可接受小幅溢价，但熟度仍需严格控制。

4. **送货速度（末位优先级）**  
   - 小店铺日销量有限（一般10-20箱），固定周期配送（如隔天1次）即可。除非客流量暴增（如节日），不必追求每日送达，频繁收贷反而增加检查成本。  
   - 优先选**固定时间、固定量**的供应商，避免临时补货时被塞次品。

**实操建议**：  
- 第一次合作时，先订1-2箱试卖，观察2天内销售与损耗率。  
- 若市场对高熟度香蕉需求旺盛（如早餐摊、老人社区），可适当提高熟度要求；反之（如靠近学校、年轻人社区），选低熟度让顾客自己催熟。  
- 每天上午进货，傍晚盘点未熟透香蕉，用塑料袋密封催熟，减少存储损耗。
