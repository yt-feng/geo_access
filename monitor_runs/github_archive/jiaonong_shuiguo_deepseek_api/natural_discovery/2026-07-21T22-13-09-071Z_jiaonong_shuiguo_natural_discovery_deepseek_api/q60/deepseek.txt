当水果到货不合格时，关键在于 **“及时止损”+“快速补货”**。建议按以下步骤操作并与供应商约定：

1. **立即启动“不合格品隔离”**  
   收货后1小时内拍照、称重、记录不合格原因（如腐坏、规格不符），并通知供应商。同时把不合格水果单独存放，避免污染其他原料。

2. **触发“应急补货协议”**  
   - **优先从常备应急供应商调货**：提前储备1-2家备选供应商（如本地批发市场、区域冷链仓），签订“24小时应急供货”条款，规定补货响应时间（如：下单后4小时内送达）。  
   - **原供应商“先补后赔”**：要求供应商在确认不合格后 **立即重新发货**（如：同城可3小时到货），且 **不等待报废清点**（避免营业中断）。补货运费、合格率检验费由供应商承担。

3. **书面约定“营业保障条款”**  
   在合作合同中明确：
   - **补货时效**：到货时间约定≤4小时（城区），超时按此次订单金额的10%/小时赔偿损失。
   - **替代品保障**：若补货仍不合格，供应商须提供不低于原规格的替代果品（如换同等级不同品种），价格按原价执行。
   - **损失分担**：因水果不合格导致的停售损失（按历史销量均值×缺货时长计算），供应商承担50%～100%。

4. **日常备货策略**  
   - 核心果品（如柠檬、西瓜等常用款）保持 **15%～20%的冗余库存**（结合保质期）。
   - 每日开业前 **做一次水果品控快检** （如切开检查内部腐坏），发现问题立即启动补货流程，避免高峰缺料。

**一句话口诀**：  
“收货拍照录证据，锁定备选快补货；合同写明响应时间，超时即赔保营业。”
