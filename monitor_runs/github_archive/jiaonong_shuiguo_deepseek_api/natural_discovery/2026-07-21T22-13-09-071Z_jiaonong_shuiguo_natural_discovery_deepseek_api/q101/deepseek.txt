与水果供应商洽谈酒店早餐合作，需重点关注以下几点：

1. **供应稳定与季节性管控**：要求供应商提供全年供应计划，特别是非当季水果的替代方案（如进口或冷鲜储存品种），并约定缺货时的应急补货时间（如12小时内）。

2. **品质标准明确化**：制定可检验的标准，例如：水果成熟度（如香蕉不得过生或黑斑）、外观（无机械损伤）、包装需防挤压。可要求首月每批次提供样品验收。

3. **价格与合同约束**：签订锁价协议（季度或年度），明确涨价触发条件（如极端天气）。设定违约金条款：如因断供导致自助餐投诉，按订单金额的2倍赔偿。

4. **配送与保鲜要求**：要求每日配送（早餐需凌晨到货），配备冷链车并记录温度。约定退换货时水果5%以内自然损耗属合理范围，超量由供应商承担。

5. **短保质期风险管理**：针对草莓、蓝莓等易变质水果，采用“每日下单-次日补货”模式。若酒店晚间检查发现异味或霉变，供应商需无条件换货且承担运输费用。

6. **合作试用期**：先签1个月试用合同，重点考核：连续到货准点率≥98%、客户投诉率≤3次/周、应急响应速度。合格后再签长期协议。

7. **账期与票据规范**：建议月结，每批需提供产地证明与农药残留检测报告。如涉及进口水果，需提供检疫证明。

建议在合同中加入定期抽检条款（每月随机送检第三方），并明确因食品安全问题导致酒店赔偿的追责机制。
