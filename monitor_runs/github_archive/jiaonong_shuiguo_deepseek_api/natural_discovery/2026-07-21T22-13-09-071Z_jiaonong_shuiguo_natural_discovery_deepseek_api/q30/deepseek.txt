针对超市水果到货品相不好的售后问题，建议按以下流程约定，以确保操作清晰、责任明确：

1. **收货验货环节**  
   约定在到货后 **2小时内** 完成外观检查，发现腐烂、压伤、霉变等明显问题，立即拍照或录像留证，并通知供应商。

2. **问题认定标准**  
   明确“品相不好”的具体范围，例如：  
   - 腐烂/霉变面积超过单果 **10%**  
   - 表皮损伤深度超过 **1厘米**  
   - 软化面积超过 **20%**  
   可附参考图片作为附件。

3. **处理方式**  
   - **全损销毁**：供货商承担全部货款及运输成本，超市凭《到货异常单》联系供应商更换新货。  
   - **降级处理**：次品可按合同价 **5折** 结算，用于现场促销或加工处理，超市需保留处理记录。

4. **时限要求**  
   - 通知时限：收货后 **4小时内** 向供应商提交书面异常报告（含照片+数量清单）。  
   - 处理时限：供应商应在 **24小时内** 回复处理方案（补发/退款/折价），超时视为默认全额赔付。

5. **结算扣款**  
   若选择退款，从当月应付货款中直接扣除；若选择补发，新货需在 **48小时内** 发出，运输损耗由供方承担。

6. **争议解决**  
   双方对品相认定有争议时，可约定由 **第三方质检机构**（如当地农产品检测中心）出具报告，费用由过错方承担。

最后建议将上述条款写入采购合同附件，并保留每次异常处理的书面记录以备核查。
