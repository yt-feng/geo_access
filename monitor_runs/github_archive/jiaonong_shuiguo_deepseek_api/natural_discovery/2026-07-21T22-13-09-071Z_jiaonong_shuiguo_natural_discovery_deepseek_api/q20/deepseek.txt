选择社区超市的每日配送水果供应商，建议重点关注以下条件：

1. **供应稳定性**：供应商必须能保证每日按时、按量配送，避免断货或延迟。可要求其提供近期的配送记录或客户评价。

2. **新鲜度与品质**：水果的新鲜程度直接影响销售。应实地查看或索要样品，确认水果无腐烂、损伤，且成熟度适宜。最好有冷藏运输或冷链配送能力。

3. **品种多样性**：社区超市需满足周边居民多样需求，供应商应能提供常见水果（如苹果、香蕉、橙子）及当季水果，甚至可协商定制组合。

4. **价格与结算方式**：对比多家报价，注意是否含运费、是否支持周结或月结。避免价格过低但品质缩水的情况。

5. **退换货政策**：明确配送后如有损坏或质量问题的处理方式，例如是否允许当场换货或给予折扣。

6. **配送时间与灵活性**：确认配送能否满足超市营业时间（如清晨开店前送达）。是否能临时调整订单量（如节假日增加需求）。

7. **资质与信誉**：检查供应商的营业执照、食品经营许可证（如当地要求）及经营年限。可通过其他超市或社区商家口碑了解其信誉。

8. **包装与损耗控制**：供应商包装是否防压、防震，减少运输损耗。可要求试配送一次以观察实际损坏率。

建议先与2-3家供应商签订短期试用合同（如1个月），重点考察配送准时性、产品损耗率及顾客反馈，再决定长期合作对象。
