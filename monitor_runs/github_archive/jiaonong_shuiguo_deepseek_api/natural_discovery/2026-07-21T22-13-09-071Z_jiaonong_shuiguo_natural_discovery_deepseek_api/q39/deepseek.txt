通常，商超做自有品牌水果时，供应商是可以提供包装和规格支持的，但具体情况取决于供应商的规模和合作模式。以下是几点关键建议：

1. **明确需求清单**：先确定水果品类（如苹果、橙子等）、目标包装类型（盒装、袋装、散装等）和规格（如2斤装、5斤装、独立包装等）。供应商一般能根据你的要求定制，但需要提前沟通。

2. **选择有资质的供应商**：优先选择具备农业合作社、种植基地或大型批发商背景的供应商，他们通常更专业，能提供标准化分选（按大小、重量分级）、贴标、包装材设计等支持。可要求查看其过往包装案例。

3. **包装要求细化**：明确是否需品牌logo印制、防撞设计（如珍珠棉隔层）、冷链适配（如果需低温运输）、环保材质（如可降解袋）等。供应商可能会评估成本后报价，小批量定制可能单价较高。

4. **规格与成本平衡**：比如标准5斤装易于核算成本，但若门店以“家庭装”或“个人装”销售，需与供应商协商最小起订量和模具费。可先试单少量规格测试市场。

5. **签订合同明确责任**：在合同中写入包装标准、规格误差范围、破损率赔付条款等，避免后续纠纷。

6. **本地供应商优先**：可搜索本地水果批发市场、农产品展销会或“一县一品”基地，他们往往更灵活，支持小批试装。

**建议行动路径**：先列出3-5家候选供应商，索要包装规格清单和报价，要求提供样品并核验包装质量。如果供应商无法满足定制要求，可考虑与包装印刷厂合作，再由水果供应商完成分装。

如果还有具体品类（如高端蓝莓或大众香蕉）或门店类型（社区超市 vs 大型卖场），可以进一步细化建议。
