挑选香蕉时，可以参考以下技巧避免买到太生或太熟：

1. **看颜色**：选择整体呈 **亮黄色**、略带 **绿色** 的香蕉（约七分熟）。避免全青（过生）或大面积黑斑、皮色发暗（过熟）。
2. **摸硬度**：轻轻捏一下香蕉中部，**有弹性、微微柔软** 为佳。太硬则生，太软一捏就陷下去为过熟。
3. **查果柄**：果柄（香蕉把）**完整、青绿或浅黄** 更好。若果柄发黑、干枯，说明存放时间较长。
4. **观棱角**：香蕉横截面 **棱角分明** 可能偏生；**棱角圆润、接近光滑** 通常成熟度适中。
5. **闻气味**：凑近闻根部，有淡淡 **香蕉清香** 为正常。无味则可能很生，有发酵酒味或酸味则过熟或变质。

建议去超市时，**先拿较黄的，再补几根略青的**，这样回家后分批食用，避免同时成熟。如果买回家还需要存放2-3天，可以选微带绿色的；如果打算1-2天内吃完，选全黄的更安全。
