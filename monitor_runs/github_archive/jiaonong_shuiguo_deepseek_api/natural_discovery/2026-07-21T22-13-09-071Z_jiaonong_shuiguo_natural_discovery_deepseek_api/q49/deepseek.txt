建议你按以下步骤与供应商沟通并约定临时补货：

1. **提前沟通（最快）**  
   立即联系供应商说明情况：“旺季菠萝用量激增，需要临时补货 XX 斤/件，希望能在 X 月 X 日前到货。” 明确数量、到货时间、是否接受分批到货。

2. **确认价格与结算**  
   询问临时补货的单价是否按原合同价，还是加急/小批量有额外费用（如运费、加急费）。明确付款方式（现结、账期或预付定金）。

3. **约定质量与验收**  
   强调“按原有质量标准验收”，如果临时换品种或产地，需提前告知并确认是否接受。约定验收标准（无霉变、大小均匀、成熟度合适）。

4. **明确违约责任**  
   口头或书面确认：“如无法按时足量到货，供应商需提前 1 天通知，并协助寻找替代货源。” 避免断货。

5. **书面确认（可选但推荐）**  
   通过微信、邮件或订单系统发送“临时补货确认单”，内容包括：品名、数量、单价、到货时间、验收标准、联系人。确保对方回复确认。

**注意事项：**
- 提前确认供应商现有库存量及进货周期（如菠萝产地是否现采）。
- 如果用量长期变大，建议将此次补货作为“旺季临时协议”的起点，协商未来几个月稳定的加量供应方案。
- 如果供应商无法满足，尽快寻找第二备选供应商做应急补充。

**可参考的话术模板：**  
“李经理，我们菠萝用量近期暴涨，需要临时追加 500 斤，能安排这周四（X月X日）到货吗？价格按原合同走，另外如果到货时间紧张，可否接受分批到货？请回复确认，我们马上走审批。”
