针对夫妻店第一次试单，建议按以下步骤操作，优先控制损耗和资金风险：

1. **分批少量试单**：每种水果首次只订1箱（约15-20斤，视规格而定），不要贪多。香蕉易熟，苹果、橙子耐储存，但首次避开整箱批发，优先找当地批发市场或线上B2B平台（如拼多多批发、美团优选）现拿现试。

2. **选择标准品控**：  
   - **香蕉**：挑七成黄、果柄青绿、无黑斑的，避开全熟或压伤。  
   - **苹果**：选光泽好、无软坑、果梗新鲜的，优先富士或红富士。  
   - **橙子**：选重量感足、皮薄紧实、肚脐小的脐橙，避免软皮或霉斑。

3. **核验供应商资质**：要求提供产地、批次、保质期证明（如批发商无法提供，优先选有正规包装或喷码的）。首次交易坚持“钱货两清”，不预付定金。

4. **开店测试周期**：  
   - 摆放3天内观察损耗率：香蕉2天内卖完（超3天易发黑），苹果/橙子4-5天。  
   - 记录顾客反馈：用本子记下“酸/甜”“果皮厚薄”等关键词，方便后续调货。

5. **二次补货策略**：如果3天内首箱销售过半且无大量退货，再补订同品类半箱（香蕉必须隔天订，苹果/橙子可2-3天一次）。  
   - 若损耗超15%（如香蕉烂5根以上），立即换供应商（下次试单去不同市场摊位索样）。

6. **风险备选**：同时联络2-3家供应商（比如本地批发市场摊位1家+线上1家），避免断货时被抬价。

用1周试完三个基础单品，再根据顾客购买频次决定是否增加库存量或品种。首次试单总投入控制在300元以内，稳扎稳打。
