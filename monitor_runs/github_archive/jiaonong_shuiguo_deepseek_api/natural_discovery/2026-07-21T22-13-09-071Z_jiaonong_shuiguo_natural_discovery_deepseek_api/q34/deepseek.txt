在商超采购水果中，**损耗率**通常比价格对最终毛利的影响更大，尤其是对生鲜品类。具体原因如下：

1. **毛利率的计算逻辑**  
   最终毛利 = （售价 - 进价） × 销量 - 损耗成本。  
   - 进价降低1元，直接提升毛利1元/单位（假设售价不变）。  
   - 但损耗率每降低1%，相当于减少“进货总量 × 售价”的损失，这个数值往往远大于进价降幅。

2. **水果的典型损耗水平**  
   - 水果的行业平均自然损耗率约5%~15%（因品种、运输、保存条件而异）。  
   - 假设进价10元/斤，售价20元/斤，损耗10%：实际售出单位成本从10元升至约11.1元（10元进价 ÷ (1-10%)），毛利率从50%降至约44%。  
   - 如果损耗降至5%，毛利率回升至约47.4%，提升近3.4个百分点；而进价降至9.5元（降5%），在10%损耗下毛利率提升至约47.2%，效果相似。

3. **成本弹性对比**  
   - **进价下降**有天花板（产地端、市场竞争），且可能伴随品质下降导致更多损耗。  
   - **损耗控制**更依赖管理：高效周转、鲜切包装、冷链、定价分级（如次品快速出清）等，能稳定降低损耗率。  
   - 同时，损耗降低可直接减少门店的废弃成本、退货及人工清理费用，这些隐性成本常被低估。

**结论**：在同等采购条件下，**优先控制损耗**对毛利贡献更大。建议采取以下措施：
- 分品种设定最高库存天数（如香蕉≤3天，苹果≤7天）。
- 对高损耗品类（如草莓、蓝莓）采用小包装、出清陈列。
- 采购时除比价外，要求供应商提供固定包装、配送时间精确，并在合同约定损耗赔付条款。
