在超市与水果供应商的账期谈判中，常见账期一般为**30天到45天**。具体天数会受到以下因素影响：

1. **超市规模与议价能力**：
   - 大型连锁超市或品牌超市（如永辉、华润等），账期通常为**45-60天**，甚至更长（如90天），因采购量大、信用较好。
   - 中小型超市或社区超市，账期多为**15-30天**，以降低供应商资金压力，获取更灵活的合作。

2. **水果品类与保质期**：
   - 易腐水果（如草莓、荔枝、芒果）保质期短，供应商通常要求更短账期，如**7-15天**，甚至现结，以减少坏账风险。
   - 耐储水果（如苹果、橙子、柚子）账期可适当延长，常见**30-45天**。

3. **合作模式与市场行情**：
   - 首次合作或新供应商，账期可能较短（15天以内），双方建立信任后逐步放宽。
   - 旺季或水果供不应求时，供应商议价能力强，账期可能缩短到**15-20天**；淡季或供大于求时，账期可谈至**45-60天**。

**建议**：
- 如果你是超市方，可先提出30天账期，根据供应商反馈和货源稳定性协商调整；若水果易坏或需求紧急，可缩短至15天。
- 如果你是供应商，可要求首单现结或15天账期，并争取“账期内提前付款折扣”（如10天内付款减免1%），以加快资金回笼。
- 所有账期需明确写入合同，注明逾期付款的违约金（如每日千分之五）。
