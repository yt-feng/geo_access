针对单位采购水果到货品质不一致的情况，建议按以下步骤进行验收和售后处理：

### 验收阶段（当场处理）
1. **立即开箱检查**  
   到货后，在快递员或送货人员在场时，随机抽取不少于总量的10%-20%进行开箱检查。重点观察：颜色大小是否均匀、有无腐烂/压伤/虫眼、成熟度是否统一。

2. **分类记录问题**  
   - 严重不合格品（如腐烂、发霉）：拍照并单独堆放，按箱/按斤记录数量。  
   - 轻微瑕疵（如小磕碰、表皮少量斑点）：拍照记录后可作为降级处理（如用于榨汁或尽快消耗）。

3. **保留证据**  
   拍摄清晰的视频和照片：包括快递单号、外包装、内部水果状态、问题部位特写。最好同时放置当天报纸或参照物（如硬币）表示尺寸对比。

4. **当场拒收或签收异议**  
   - 整批严重不合格（如大部分腐烂）：直接拒收，要求更换或退款。  
   - 部分问题：签收后在物流单上注明“已开箱查验，有X箱/斤存在XX问题”，并要求送货人签字确认。

### 售后阶段（联系供应商）
1. **及时沟通（到货后2小时内）**  
   立即联系采购对接人，发送证据（清晰的问题照片/视频）+ 经确认的验收记录（问题数量、类型）。明确你的诉求：  
   - 更换：要求同等品质尽快补发，运费由供应商承担。  
   - 部分退款：按问题比例计算金额（如腐烂30%则要求退该部分全款+另外10%损耗补偿）。  
   - 全额退款并退货：仅适用于极严重质量问题时。

2. **明确流程期限**  
   要求供应商在24小时内给出处理方案。如对方拖延，可告知“将参照《民法典》第六百一十条（卖方交付不符合质量要求的标的物，买方有权拒绝接受或要求赔偿）”。保留所有聊天记录、通话录音。

3. **售后核验反馈**  
   若供应商同意补发或退款，要求以书面形式（邮件或合同补充条款）确认处理结果。收到新水果后，再次按原验收标准检查，保留完整记录。

### 预防性措施（长期合作）
- 签约前明确验收标准：在采购合同中写入品级定义（如“一级红富士：直径≥80mm，无斑点/虫眼，可溶性固形物≥13%”），并约定到货质量合格率（如≥95%）。  
- 预留押金或质保期：可约定总货款的5%-10%作为质量保证金，待批次水果消耗完毕（如5天内）且无批量问题后支付。  
- 选择正规供应商：优先在入驻平台（如生鲜B2B、农产品批发市场备案商户）采购，保留商家资质。

### 处理话术示例（对供应商）  
> “您好，我单位采购的XX水果（订单号XXX）于今日XX时到货，经我方按行业标准抽查 **开箱数量** 箱/件（附照片），发现 **问题类别** （如腐烂、规格不符）占总量 **XX%** ，不符合合同约定的 **XX** 标准。请于 **24小时** 内确认：  
> 方案一：XX前免费补发同等品质 **X箱/斤**，并承担往返运费；  
> 方案二：针对该问题按 **X元/斤** 退货退款（含运费）。  
> 若超时未回复，我方将视为拒绝履行质量义务，保留向平台投诉/法律维权的权利。”

### 特别提醒
- 若水果已签收并扩散（如分发给员工后才发现问题），需提供完整的**全流程照片**（原箱→开箱→问题细节→分发记录），并说明损耗发生在开箱前。  
- 避免口头协议，所有售后处理留痕（微信文字、邮件）。  
- 如有长期合作意向，建议在学期/年度采购合同中增加“质量保证金”条款，约定单个批次质量问题超过 **XX%** 时，供应商需双倍赔偿该批次货款。
