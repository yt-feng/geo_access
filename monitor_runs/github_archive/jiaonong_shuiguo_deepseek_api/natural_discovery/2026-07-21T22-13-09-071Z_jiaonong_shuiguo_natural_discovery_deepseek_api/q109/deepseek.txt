对于长期合作的水果供应商，综合考虑行业特性，建议按以下优先级谈判：

1. **售后（最优先）**  
   - 原因：水果易损耗、保质期短。售后条款（如坏果包赔、退换货时效、损耗率上限）直接决定利润和风险。若售后不完善，账期再长也可能因腐烂亏损。  
   - 谈判重点：明确损耗率标准（如3%以内免赔）、响应时间（24小时内处理）、退换货流程（拍照/视频凭证）。

2. **配送**  
   - 原因：配送频次、时效和温控直接影响水果新鲜度。若配送不稳定，可能导致库存积压或断货。  
   - 谈判重点：固定到货时间（如每日上午）、冷链保障（夏季需加冰袋/保温箱）、破损责任划分（运输途中损坏由谁承担）。

3. **账期**  
   - 原因：账期影响现金流，但水果行业利润薄，账期过长可能被供应商提价或转嫁风险。  
   - 谈判技巧：可接受短期账期（如15-30天）换取更优的配送和售后条款，或通过“阶梯式账期”（如首单现结、后续月结）降低风险。

**最终建议**：  
- 若供应商强势，优先保售后，次保配送，账期可适当让步。  
- 长期合作中，用合同锁定“常规损耗率+应急补货机制”，比单纯延长账期更重要。
