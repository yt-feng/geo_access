混合采购（即同时采购多种水果）对水果批发商有以下潜在优势：

1. **降低运输成本**：一次运输多种水果，可摊薄单位物流成本，尤其是当供应商集中在同一产区或批发市场时。
2. **增强议价能力**：批量采购多种水果（如香蕉、苹果、橙子）时，供应商可能更愿意给予整体折扣或优先供货。
3. **平衡市场风险**：不同水果价格波动周期不同（如季节性、天气影响），混合采购可对冲单一品种价格暴跌的风险。
4. **满足客户需求**：零售商或终端客户往往需要一站式采购，混合库存能提高客户满意度和复购率。
5. **周转效率优化**：香蕉成熟快、苹果耐储存、橙子适中的特点，可以搭配短、中、长期库存，减少资金积压。

**需要注意的潜在劣势**：
- 若各品种存储条件差异大（如香蕉需控温、苹果需低温），可能增加仓储复杂度。
- 若采购量分散，可能错失单一品种的超大订单折扣。

**建议**：根据自身客户结构、仓储条件、资金周转能力，优先选择需求匹配度高、储存需求兼容的品种组合。如果刚开始混合采购，可先选择2-3种互补性强的水果试运营。
