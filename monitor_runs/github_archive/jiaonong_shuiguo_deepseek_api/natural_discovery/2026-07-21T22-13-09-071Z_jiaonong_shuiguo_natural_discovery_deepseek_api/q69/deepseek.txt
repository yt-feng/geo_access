你可以把香蕉和菠萝按以下方式分级报价，简单清晰，便于执行：

**香蕉分级报价建议：**

1. **特级果（A级）**：果形完整、无斑点、无机械损伤，成熟度7-8成，单根长度≥18cm。  
   - 报价：最高价（如每斤高1-2元）

2. **一级果（B级）**：轻微瑕疵（如少量斑点、轻微擦伤），成熟度6-8成，单根长15-18cm。  
   - 报价：中等价位（如比A级低10-15%）

3. **二级果（C级）**：有疤痕、黑点或部分挤压，成熟度5-9成，长短参差。  
   - 报价：最低价（比A级低20-30%），适合做促销或榨汁用。

4. **次果（处理品）**：有明显腐烂、断裂或过熟。  
   - 单独处理，按堆或按斤低价出清。

**菠萝分级报价建议：**

1. **特级果**：果形端正、鳞片饱满、无褐斑、顶部绿叶新鲜，单果重1.2-1.5kg。  
   - 报价：单价最高（单果加价2-5元）

2. **一级果**：果形稍不规则、轻微褐斑或部分鳞片受损，重1.0-1.2kg。  
   - 报价：中等价（比特级低15-20%）

3. **二级果**：有明显疤痕、碰伤或顶部叶枯，果重0.8-1.0kg。  
   - 报价：低价（比特级低30-40%），适合切块或加工。

4. **处理品**：有霉斑、软烂或果心褐化。  
   - 单独低价出清，或标明“次品特价”。

**操作建议：**
- 为每个等级设定一个固定单价，避免漫无边际的讨价还价。
- 提前给小店老板看样品或等级参照图，现场验货时对号入座。
- 对老客户可以给出等级浮动范围（如：“按一级品供，如果实际品质达到特级，按二级加价”）。
- 用货架标签或发货单明确写明“等级+单价”，避免纠纷。
