大型超市对水果供应商的食品安全和物流能力通常有严格的要求。以下是必备的关键能力：

**一、食品安全能力：**

1.  **资质与认证：**
    -   具备合法有效的营业执照、食品经营许可证（或生产许可证）。
    -   通过相关的食品安全管理体系认证，如ISO 22000、HACCP（危害分析与关键控制点）认证，或GLOBALG.A.P.（全球良好农业规范）认证（针对源头种植）。
2.  **源头控制与追溯：**
    -   建立完善的**农产品追溯体系**，能追溯到具体产地、种植户、采摘日期、批次号等信息。
    -   对供应商（种植基地）进行审核，确保农药、化肥使用符合国家标准，并提供农残检测合格报告。
    -   建立不合格品召回机制，能快速定位并召回问题批次。
3.  **加工与仓储环节：**
    -   加工车间（如分拣、包装）需符合卫生标准，如防虫、防鼠、防尘设施，并具备温湿度控制能力。
    -   仓储环境需根据水果特性分温区管理：冷藏（0-4℃）、冷冻（-18℃以下）或常温，并有实时温湿度监控记录。
    -   建立规范的入库检验制度，包括外观、重量、农残快检等。
4.  **运输支持文件：**
    -   提供每批次的《食用农产品合格证》或第三方检测报告。
    -   提供物流过程中的温控记录、车辆消毒记录等文件。

**二、物流能力：**

1.  **冷链运输能力：**
    -   拥有或合作专业的冷链车队（冷藏车、保温车），车辆需配备温控记录仪（可实时回传数据），确保运输全程温度达标（如香蕉：13-15℃，草莓：0-4℃）。
    -   具备**最后一公里**的冷链配送能力，如从配送中心到超市门店的短途冷藏运输。
2.  **时效与末端配送：**
    -   能根据超市的订货周期和门店分布，制定高效的配送计划，例如每日下午或深夜配送至门店，确保次日上架新鲜。
    -   能应对突发的大额订单或紧急补货需求，具备灵活调配车辆和运力的能力。
3.  **仓储与分拣能力：**
    -   拥有或租赁大型冷库，支持不同品类水果的分温区存储和分级分拣。
    -   具备快速分拣、包装、贴标的能力，能根据超市要求进行定制化包装（如预包装净菜、礼盒装等）。
4.  **信息化管理能力：**
    -   使用TMS（运输管理系统）或WMS（仓储管理系统），实现订单、库存、运输的可视化追踪，并能与超市的ERP（企业资源计划系统）或供应链系统对接。
    -   提供精准的到货时间预测，并能实时反馈运输异常（如拥堵、延误）。

**总结：** 核心要求是**全程冷链可控** + **可追溯的食品安全管理体系** + **高效协同的物流能力**。供应商需要提供完整的资质文件、冷链记录和追溯信息，才能满足大型超市的准入标准。
