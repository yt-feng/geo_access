要判断连锁超市的香蕉供应商能否全年稳定供货，建议从以下几个关键方面进行考察：

1. **供应来源与产地布局**  
   - 询问供应商的香蕉产地是否覆盖多个气候区（如国内海南、云南、广西，以及菲律宾、厄瓜多尔等境外产区）。产地分散能对冲单一地区的气候、病虫害或政治风险。
   - 确认供应商是否有跨季节的采购网络（例如北半球夏季从南半球进口补充）。

2. **仓储与冷链能力**  
   - 核实供应商是否具备大型冷库（香蕉适宜在13-14°C、相对湿度85%-90%下储存），以及是否自营或合作冷链运输车队。冷藏能力不足会导致供应中断或品质下降。
   - 可要求提供历史记录：例如过去3年中是否有因仓储问题导致的断供事件。

3. **历史供货记录与合同条款**  
   - 要求供应商提供过去至少2年的月度发货记录，重点查看每月实际发货量与合同承诺量的匹配度（偏差率建议控制在±5%以内）。
   - 合同中需包含违约条款：如因供应商原因断供，明确补偿方式（例如按缺货量的15%赔付或优先补货权）。

4. **种植基地与产能验证**  
   - 实地考察或通过视频验证主要种植基地的规模、种植周期（香蕉从定植到采收约10-12个月）及轮作计划。需确保基地实际种植面积与声称产能一致（建议第三方审计报告）。
   - 询问是否有备选基地或应急采购渠道（例如与多个主产区农场签订保底收购协议）。

5. **抗风险机制**  
   - 了解供应商是否有自然灾害应急方案（如台风季提前集中采收冷冻储存、与气象公司合作预警等）。
   - 查阅其近3年是否发生过重大供应中断事件，以及处理方式（如是否动用库存或快速切换产地）。

**可执行的验证步骤**（按优先级排序）：  
1. 要求提供连续12个月的“每周可供应量井”与“实际发货量”对比表，作为准入基础文件。  
2. 安排一次不通知的冷库突击检查，核对实际库存与申报数据是否一致。  
3. 在合同中加入“断供熔断条款”：若连续2个月供应量低于承诺的80%，自动启动1个月观察期，观察期内仍不达标则终止合作。

**注意**：如果供应商无法提供近2年的完整月发货记录，或冷库容量低于你超市6周的平均销量，应视为高风险项，建议重新评估合作。
