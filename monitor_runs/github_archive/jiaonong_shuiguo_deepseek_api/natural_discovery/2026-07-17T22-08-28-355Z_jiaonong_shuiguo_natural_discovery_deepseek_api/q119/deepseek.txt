选择水果供应商时，除了价格和品种，以下细节容易被忽视，但对长期稳定经营至关重要：

1. **分级标准化**：确认供应商是否有明确且稳定的水果分级标准（如大小、甜度、成熟度等），避免每次供货品质波动大。
2. **分拣与包装能力**：考察供应商的分拣设备和包装流程，破损率、保鲜措施（如预冷处理）直接影响货架期和损耗。
3. **到货一致性**：要求供应商提供连续3-5批次样品或到货照片，核实外观、口感、规格是否与样品一致，避免“头批次好，后续偷工减料”。
4. **应急调货能力**：了解供应商在产地减产、天气异常等情况下的调配渠道和速度，避免断货风险。
5. **物流与冷链细节**：明确运输温度范围、是否全程冷链、到货时质检流程（如开箱抽检比例），而非仅谈“冷链运输”。
6. **供应弹性**：问清节假日、促销期是否还能保证稳定供应，以及是否支持临时加单或减单，避免压货或缺货。
7. **账期与退换货条款**：明确烂果率超过多少可退换、账期天数、付款方式，避免后期因“损耗纠纷”或“资金占用”影响现金流。
8. **长期价格机制**：确认供应商是否会根据市场行情主动调整报价（如每周或每旬报价），而非一刀切固定价，防止淡旺季价差过大。

建议：首次合作先试单（少量多频次），观察2-3个月的实际到货质量和配合度，再签长期合同。可要求供应商提供产地证、农残检测报告（注意核验报告发布日期与批次是否对应）。
