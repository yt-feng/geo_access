在便利店售卖菠萝切块时，以下三项都需要关注，但优先级和具体措施如下：

1. **保质期**（最重要）：菠萝切块属于鲜切水果，需严格控制加工后的保质期（通常不超过24小时，冷藏条件下）。必须标注生产时间、到期时间，并建立定期检查、报废机制，防止过期销售。

2. **包装**（直接影响安全和陈列）：需使用食品级、密封性好的包装（如塑料盒+贴膜），防止氧化变黑、二次污染。包装上需清晰标明生产日期、保质期、保存条件（0-4℃冷藏）。同时考虑包装尺寸和展示方式，方便顾客选购。

3. **配送频次**（影响库存周转）：根据门店销量和距离，制定合理的配送计划。通常建议每日至少配送一次（如早晨到货），销量高的门店可增加频次（如上午和下午各一次）。避免配送过多导致积压损耗，或过少导致断货。

**建议执行顺序**：  
- 先确定保质期标准（如“冷藏不超过24小时”）。  
- 再选择合适包装（如密封盒+标签），并制定标签打印流程。  
- 最后根据历史销售数据计算每日配送量，制定固定配送频次（如每日一次），并设置库存预警（比如剩余量低于目标库存的20%时紧急补货）。

另外，可配合“每日定时折扣”（如晚上8点后打折促销）来降低损耗，同时提高周转率。
