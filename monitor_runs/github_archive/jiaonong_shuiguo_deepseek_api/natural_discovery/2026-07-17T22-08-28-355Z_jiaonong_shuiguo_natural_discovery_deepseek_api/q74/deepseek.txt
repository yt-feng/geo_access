针对批发档口拓展餐饮客户（如水果店、甜品店、饮品店、餐厅等），菠萝和香蕉的规格建议如下，重点关注**成熟度、包装和细分用途**：

### 一、菠萝规格建议

餐饮客户主要需求是**即切、即用、损耗低**，建议准备两种规格：

1. **全熟鲜菠萝（用于即做、果切）**
   - **成熟度**：表皮金黄、按压微软（8-9成熟），当天或次日可使用。
   - **果重**：1.2-1.5公斤/个（金菠萝或徐闻菠萝品种）。
   - **包装**：单果独立网套+泡沫箱，避免撞伤。
   - **附加服务**：可提供“去头去尾”整果，或切片（1-2cm厚）独立包装，满足甜品店摆盘需求。

2. **半熟菠萝（用于备货、外卖）**
   - **成熟度**：7成熟（表皮青黄相间，果肉硬）。
   - **果重**：1公斤左右（小果更易控制成本）。
   - **包装**：小箱装（6-8个/箱），标注“需存放2-3天自然熟”。
   - **适用客户**：连锁奶茶店、餐厅（用于菠萝炒饭、菠萝咕咾肉）。

### 二、香蕉规格建议

香蕉对成熟度非常敏感，餐饮客户需“即用”或“可催熟”，建议分两类：

1. **即食甜香蕉（用于果切、摆盘、奶昔）**
   - **成熟度**：果皮金黄、无青边、有芝麻黑点（全熟）。
   - **果重**：单根120-150克（细短形，如“巴西蕉”或“小米蕉”）。
   - **包装**：无破损的整挂或切断后泡沫箱分装，避免相互压坏。
   - **注意**：必须标注“开箱即用”，避免客户误以为还需催熟。

2. **供货用绿香蕉（用于连锁店分时段催熟）**
   - **成熟度**：果皮全青、果肉硬（6成熟）。
   - **果重**：单根100-120克（小果更易控制熟度）。
   - **包装**：10公斤/箱，内垫硅藻土或吸潮纸，标注“需乙烯催熟”。
   - **适用客户**：大型酒店、便利店（自主控熟，减少损耗）。

### 三、针对餐饮客户的统一服务建议

1. **按用途分装**：例如菠萝可以提前切成“菱形块”（用于炒饭）或“扇形片”（用于饮品装饰），香蕉可整根或切段真空包装。
2. **提供成熟度分级卡**：用颜色标签区分（红标即食、黄标需催熟），避免客户误判。
3. **建立“坏果包赔”机制**：尤其针对全熟菠萝和香蕉，损耗高，承诺一定比例补偿（如5%以内免费补发）。
4. **推广小份包装**：适合中小型餐饮客户，例如2-3公斤小箱菠萝、5公斤小箱香蕉，降低客户首次尝试门槛。

### 四、备选注意事项（针对热带水果特性）

- **菠萝：** 餐饮客户可能依赖“无纤维丝”品种（如金钻凤梨），需提前确认客户是否接受普通带刺菠萝（如都乐金菠萝）。
- **香蕉：** 夏季（气温＞25℃）推荐直接供全熟果，避免客户在催熟过程中烂果；冬季推荐供绿果（客户可存放更久）。

建议先以“3种规格混装”样品发给重点客户试单，根据反馈快速调整。餐饮客户通常看重**稳定供货（每天相同品质）**和**低损耗（包装避免磕碰）**，这两点比单纯价格更重要。
