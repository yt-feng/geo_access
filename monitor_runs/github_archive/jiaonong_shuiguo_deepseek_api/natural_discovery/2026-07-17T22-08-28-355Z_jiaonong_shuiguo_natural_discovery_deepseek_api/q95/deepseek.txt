选择学校食堂的水果供应商，建议按以下步骤操作：

1. **明确需求**：根据学生人数、食堂用量、预算和季节确定采购量。香蕉、苹果、橙子等常见水果需求量大，需重点考虑耐储存和供应稳定性。

2. **资质与安全**：
   - 要求供应商提供营业执照、食品经营许可证（含水果类）。
   - 查看检疫证明、农药残留检测报告（可要求每批次提供）。
   - 优先选择有“绿色食品”或“无公害农产品”认证的供应商。

3. **实地考察**：
   - 检查仓储条件（低温、通风、无虫害）。
   - 核实货源渠道（是否从产地直采或大型批发市场），避免中间加价。

4. **索样试吃**：
   - 要求提供样品，对比品质（如香蕉的成熟度、苹果的果径和糖度、橙子的新鲜度）。
   - 制定验收标准（如无腐烂、无机械损伤、颜色均匀）。

5. **比价与合同**：
   - 至少对比3家供应商，要求报价单包含单价、配送费、税点。
   - 签订合同明确：供货周期、付款方式、退换货条款（如腐烂率超过5%可拒收）、应急供应方案。

6. **配送与售后**：
   - 要求固定时间配送（如早上开餐前），确保冷藏运输。
   - 保留样品和送货单，后期出现质量问题可追溯。

**常见风险提示**：
- 避免一味追求低价，可能牺牲品质（如催熟水果）。
- 警惕供应商“以次充好”，需定期抽查。
- 苹果和橙子适合长期合作，香蕉易损耗，建议与多家供应商签订短周期合同。

**推荐渠道**：
- 本地大型农产品批发市场（如北京新发地、上海农批）的固定档口。
- 与能追溯产地的农业合作社直接签约（如陕西苹果、江西橙子）。
- 使用政府“食材供应平台”如“食安云”等，信息透明。

如果需要，可向教育局或同业学校索取推荐名单。
