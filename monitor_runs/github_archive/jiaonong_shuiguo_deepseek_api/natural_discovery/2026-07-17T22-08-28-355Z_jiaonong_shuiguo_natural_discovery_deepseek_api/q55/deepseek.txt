建议按以下规格选购更划算：

1. **苹果**：选**中等个头（单果约200-250克）**，性价比最高。太大易浪费，太小削皮后出肉率低。批量购买时，**散装比礼盒装省20%以上**。

2. **橙子**：选**直径7-8厘米（单果约150-200克）**，甜度高、水分足。避免过大的（可能皮厚或空心），小果（<120克）出汁率低。**按箱买（20斤装）比零售便宜30-50%**，但需确认是否适合自家消耗速度。

3. **菠萝**：选**单个1.5-2斤**的。太小的肉少、削皮损耗大；太大的可能粗纤维多、口感差。**带皮整颗买比切好的便宜一半以上**，建议请摊贩代削皮以节省时间。

**综合建议**：
- 按出肉率算：**苹果约85%可食，橙子约70%，菠萝约55%**。同样重量下，苹果最划算。
- 若拼盘要求品种均衡，可优先选当季水果（议价空间大），并**整批采购**（如水果批发市场），避免超市高价单品。
- 注意：**有瑕疵的中果（局部压伤、不套袋）常打折，不影响口感**，可让摊贩按特价处理。
