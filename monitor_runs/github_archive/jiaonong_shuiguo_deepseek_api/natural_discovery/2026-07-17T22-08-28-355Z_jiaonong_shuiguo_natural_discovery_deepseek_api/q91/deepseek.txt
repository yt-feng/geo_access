建议优先选择**小把包装（3-5根）**，原因如下：

1. **售卖灵活性**：单根适合即食场景（如上班族、学生），但损耗可能较高；整袋（如8-10根）适合家庭采购，但客单价较高、可能滞销。小把兼顾单次购买量和价格，符合便利店“少量多次”的消费习惯。
2. **损耗控制**：香蕉易熟、不耐放，小把包装可减少因单根挑选造成的表皮破损，同时避免整袋囤积导致熟度不均后批量报废。
3. **视觉和动线**：小把卷成弧形摆放在货架拐角或收银台附近，易吸引冲动消费；可搭配价格标签（如“3根装 4.5元”或“5根装 6.9元”），并与牛奶、早餐组合陈列。
4. **鲜度管理**：按成熟度分色标记（青→黄→全熟），每天分批次上架小把，优先出熟透的。单根残次品可单独打折或用于试吃。

**执行建议**：
- 启动期每店试销2-3种规格（如3根、5根小把），监控2周内的日销量、损耗率，若某规格周损耗＜5%且日均销量≥20组，则保留。
- 长期看可结合促销：单根或小把搭售酸奶、麦片等；整袋放进礼盒区或周边家庭客群门店。
