建议采用 **“小批量试卖 + 分阶段增量”** 的策略，常见稳妥比例是 **该门店同类水果日均销量的 10%-20%**，或者控制在 **1-3 天的安全周转库存**。

具体数字参考（假设是社区或水果店）：

- **首次试卖量**：20-50 公斤（或 1-2 箱），先铺 1-2 家核心门店或摊位。
- **试卖周期**：3-7 天，观察顾客反馈、复购率、损耗率。
- **二次补货判断**：如果试卖期间日销售能达到预估量的 60% 以上且损耗低于 5%，可考虑加量至 50-100 公斤并铺更多门店。
- **最保守做法**：按最差情况——如果全部滞销，能否在 3 天内处理完（降价、做水果切、赠品）？按此计算上限。

**关键前置动作**：提前要求供应商提供 **1-2 公斤免费样品**，先内部试吃并检查耐储性、口感稳定性。如果样品阶段就不稳定，直接跳过试卖。
