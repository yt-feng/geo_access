小商家采购水果时，**现金结算**和**月结账期**各有以下风险：

1. **现金结算风险**：
   - **资金压力**：需要一次性支付全部货款，可能占用较多流动资金，影响其他采购或应急开支。
   - **验收风险**：若水果品质有问题（如腐烂、缺斤短两），现金支付后追责较难，因为缺少账期缓冲和证据留存。
   - **付款纠纷**：缺乏正式合同或凭证时，容易因口头约定不清产生争议，尤其是小商贩之间。

2. **月结账期风险**：
   - **坏账风险**：如果供应商经营不善或跑路，已售出水果但未收到货款，可能血本无归（尤其对小商家而言）。
   - **账期管理成本**：需要保留所有采购单据、对账记录，若单据丢失或对方不认账，后续维权困难。
   - **利息或滞纳金**：部分合同可能隐含逾期费用，若延迟付款会增加成本。
   - **供需不稳定**：供应商可能因账期长而优先供应现金客户，导致小商家在旺季缺货。

**建议**：
- 优先选择**现金结算**，但需当场验收品质、称重并索取收据（写明品名、数量、金额、日期）。
- 若选择月结，务必签订书面协议（明确账期、违约责任、退货条款），并定期对账（用微信记录或签字单）。
- 对高风险供应商（如新合作、无固定场所的），避免给予月结账期。
- 考虑**混合模式**：小额现金采购日常补货，大额交易用月结加小额保证金。
