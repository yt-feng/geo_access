建议你按以下步骤与供应商沟通并约定临时补货：

1. **紧急沟通**：立即电话或微信联系供应商，说明旺季菠萝用量激增，需要临时补货，明确告知需要的具体数量（精确到公斤或箱数）和希望到货的时间（如“明天下午3点前”）。

2. **确认库存与产能**：询问供应商当前是否有足量库存或能否紧急采购/加工。如对方库存不足，可询问是否有替代品种（如冷冻菠萝块或同等品质的菠萝品种），并确认是否会影响品质。

3. **明确临时条款**：  
   - **价格**：确认是否按原合同价格补货。如临时涨价，要求对方明确涨幅和原因（如运输加急、市场缺货），并明确本次临时补货是否影响后续长期价格。  
   - **数量与有效期**：写明本次补货数量、交付日期、验收标准（如成熟度、切块规格等）。  
   - **退换货**：约定如到货品质不达标（如腐烂、过熟）的处理方式（拒收或折扣处理）。  
   - **结算方式**：确认是否需预付，或按原账期结算（如“货到7天付款”）。

4. **调整后续计划**：同时与供应商协商，是否将本次临时补货纳入未来两周的常规订单（即“提前用掉原有配额”），或作为独立追加订单，避免影响原有合同的稳定性。

5. **书面确认**：无论口头怎么谈，立刻要求对方通过微信、短信或邮件发送确认函，内容包括：补货数量、单价、总价、交货时间、验收标准、退换货规则。保留聊天记录作为凭证。

**关键点**：  
- 如果供应商无法完全满足，分阶段补货（如“先发30%，后天再发剩下70%”）。  
- 若供应商临时大幅涨价，直接说明“本次可接受，但需书面标注清楚，否则下次按原价不再续签”。  
- 要求供应商提供菠萝的产地、批次信息，以便你端备用核验（比如同一批货里是否有过度催熟产品）。

**示例回复模板（发给供应商）**：  
> 老板，现在旺季菠萝用量暴增，预计本周需要额外增加**200公斤**。请问咱库存够吗？如果够，我**明天中午12点前**要这批货送到店里。价格按原来合同价**还是涨**？品质标准不变（7-8成熟、无腐烂、切片规格XX）。请发条确认消息给我，包括数量、总价和到货时间，谢谢。如果库存不够，能否先发100公斤，后天再发剩下100公斤？

完成以上步骤后，确保在到货时当场验货，符合条件的才签收。
