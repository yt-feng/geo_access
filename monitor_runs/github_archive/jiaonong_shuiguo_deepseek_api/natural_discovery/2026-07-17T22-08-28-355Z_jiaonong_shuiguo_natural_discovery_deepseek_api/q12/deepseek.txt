在采购菠萝时，判断熟度主要看外观、手感、气味和声音，建议按以下步骤操作：

1. **看颜色**：成熟的菠萝外壳呈金黄色或橙黄色，绿色部分较少的相对更熟。但注意，不同品种（如台湾金钻菠萝）可能自然偏黄，而云南小菠萝可能外皮偏青但内部已熟。优先选择整体颜色均匀的，避免局部过绿或过褐。

2. **摸硬度**：用手轻按果皮，成熟菠萝应有弹性，稍软但不塌陷。如果按压后凹陷不回弹，说明过熟易烂；如果硬邦邦，则偏生，甜度低。

3. **闻气味**：靠近底部闻，成熟的菠萝有浓郁甜香；生菠萝几乎无味或带青草气；若闻起来有酒糟味或酸腐味，说明内部已变质。

4. **看底部**：检查果眼（表皮的菱形突起），成熟菠萝的果眼较扁宽，颜色亮；如果果眼小而凸起、颜色发暗，可能是采摘过早。

5. **听声音**：轻敲果身，声音发闷略带空鼓感（类似敲西瓜）说明水分和糖分较足；声音清脆则偏生。

**采购实操建议**：
- **根据到货周期定熟度**：若门店3天内能卖完，可选8-9成熟的（外壳金黄偏亮，底部稍软）；若需周转5-7天，选7成熟的（外壳绿黄相间，手感硬）。避免一次采购全熟或全生。
- **整箱检查**：打开箱体看中下层，避免出现“外面熟、里面烂”（果眼间隙有水渍或发黑）。批发市场允许的情况下，随机切一个试甜度（可协商）。
- **控制损耗技巧**：进货后及时剔除有明显软烂或碰伤的菠萝，这类瓜果会加速周围成熟。常温下2天内未售完的，去皮切块冷冻或做促销，避免变质。

**特别提示**：菠萝是后熟型水果，采购时偏生的可以在门店常温放置3-5天自然成熟，但不要冷藏，否则会停止成熟且易变黑。如果当地习惯削皮售卖，可以选择8-9成熟，以减少顾客削皮时发现内部发黑引发的退货。
