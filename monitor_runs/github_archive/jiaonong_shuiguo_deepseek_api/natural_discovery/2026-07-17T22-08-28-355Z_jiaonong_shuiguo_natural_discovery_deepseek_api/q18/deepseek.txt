避免超市香蕉太生或太熟，可以按照以下步骤挑选：

1. **看颜色**：选择**黄色略带青绿**的香蕉，这种是“七分熟”状态，室温下可放2-3天；全黄（无青）的偏熟，需当天吃；有黑斑的过熟，不建议买。

2. **摸硬度**：轻轻捏一下香蕉柄和中部，**有弹性但不过软**的为好。太硬说明生，太软一按就凹进去的熟过头。

3. **查果柄**：**果柄完整、未发黑**的是新鲜采摘的；若果柄干枯或发黑，说明存放过久。

4. **看损伤**：检查有无**碰伤、压痕或霉点**，这些会加速腐烂。

**快速判断公式**：
- 想**放2-3天吃**：选带青斑、硬实、果柄鲜绿的。
- 想**当天或第二天吃**：选金黄色、微软、无黑斑的。

**超市采购小技巧**：
- 选整串挂着的香蕉（比散装更新鲜）。
- 避开冷藏柜里的（低温会催黑但内里生）。
- 如果超市只提供熟透的，可买少量全黄的当天吃，再另买半生不熟的放两天。

**到家后处理**：
- 生香蕉：放室温、用报纸包住苹果同放，可加速催熟。
- 熟香蕉：吃不完放冰箱冷藏（表皮会变黑但果肉正常），或冷冻做冰沙。
