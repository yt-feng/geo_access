在长期合作的水果供应商谈判中，建议优先谈判**账期**。原因如下：

- **账期直接影响现金流**：水果行业利润薄、损耗高，账期长短直接决定你的资金压力。如果账期过长（如60天以上），即使配送和售后再好，也可能因资金周转不灵导致断货或亏损。优先争取合理的账期（如30天内或按批次结算），能保障生意的稳定性。
- **配送和售后可后续优化**：配送可以通过补签合同细化频率、时间窗口；售后（如坏果赔付）通常有行业默认标准。这些条款相对容易调整，且对经营影响不如账期直接。
- **谈判策略**：先获取对方账期底线，再用“缩短账期”作为换取长期合作量的筹码，反之则要求更优配送或售后条件。

**建议行动**：明确自己现金流承受上限（例如最长可接受45天账期），以此为底线，优先谈下级账期方案。如果对方账期过长，再重点谈配送和售后。
