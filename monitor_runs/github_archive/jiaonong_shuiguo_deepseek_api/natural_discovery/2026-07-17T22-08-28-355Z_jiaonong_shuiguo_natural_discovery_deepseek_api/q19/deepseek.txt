关注顺序建议：

1. **外观**（最重要）：优先选择果皮金黄、芽眼浅、叶片翠绿挺拔的菠萝。避免有霉斑、软烂或过度干枯的果实。

2. **甜度**（次重要）：可通过按压果实基部（底部）判断：轻微弹性说明成熟度适中，过硬偏酸，过软则可能过熟发酵。也可闻香气，有浓郁果香为佳。

3. **大小**（第三）：个头适中（约1.5-2斤）通常果肉更饱满。过大可能纤维粗，过小可能发育不足。

4. **切果损耗**（辅助因素）：若现场切割，观察果皮厚度（皮厚损耗大）和果眼深度（浅眼切后出肉率高）。但建议优先保证水果本身质量，再权衡损耗问题。

总结：先看外观质量，再通过手感和香气判断甜度，然后选择合适大小，最后才考虑切果损耗。
