对于水果店销售菠萝，建议提供**三种方式**供顾客选择，根据具体场景灵活调整：

1. **整果（性价比高、适合即买即存）**  
   - 适合：家庭用户、打算自己处理（如冷冻、做果酱）、购买后非立即食用、对新鲜度有掌控欲的顾客。
   - 注意：展示时表明熟度（如“已催熟”或“当日现切”），可附带“免费去皮”提示。

2. **去皮好的（便利性强、适合即食）**  
   - 适合：办公室人群、逛街顾客、无工具或不想费力处理的人。
   - 关键：必须当天现切并冷藏保存（4℃以下），盒子密封且标明“当日食用最佳”；可搭配销售一次性手套和纸杯。

3. **让顾客自己挑（参与感+减少投诉）**  
   - 可提供少数几个“示范品”供顾客按眼缘挑，但需提前挑选成熟度均匀、无软烂的。
   - 适合：喜欢亲手挑选水果的顾客、希望确认新鲜度的人。

**建议执行策略：**
- **选位分区**：整果成列在出入口（吸引冲动消费），去皮成品放在冷柜或收银台旁（促进附加销售）。
- **价格分层**：去皮好的按“整果价格+加工费”定价（例如整果10元/个，去皮12元/个）。
- **引导技巧**：对犹豫的顾客可询问“您是今天吃还是过两天吃？今天吃建议选去皮好的”，或提供“免费削皮服务”吸引整果购买。

**风险控制**：
- 去皮品必须标注“切好后限8小时内食用”，避免投诉。
- 如果店铺流量小，优先提供“整果+免费削皮”选项，避免去皮品滞销变质。
