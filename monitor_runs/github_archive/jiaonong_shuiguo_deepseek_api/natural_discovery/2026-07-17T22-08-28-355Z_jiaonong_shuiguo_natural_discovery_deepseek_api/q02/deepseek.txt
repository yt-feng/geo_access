作为小水果店老板，建议按照以下优先级权衡这四个因素：

1. **熟度（最重要）**  
   香蕉的上市期非常短，熟度直接影响售卖窗口。选择七成熟左右的香蕉（表皮绿中带黄，手感坚硬），这样到店后有2-3天自然成熟时间，能避免快速腐烂。如果进全熟或过熟香蕉，当天卖不完就容易报废。

2. **坏果率（第二重要）**  
   理想情况控制在1%以内。进货时随机抽检底层和中心位置的香蕉（批发商常把坏果藏在中间），注意观察蕉柄是否发霉或断裂。坏果率每增加1%，实际成本可能上升5-8%（因需折价或销毁）。

3. **送货速度（第三重要）**  
   建议选择能做到“当天下单、次日早到”的供应商。香蕉对温湿度敏感，运输超过24小时容易出现“水烂”（果肉变软渗水）。优先选本地批发商（车程2小时内），而非跨省发货。

4. **价格（末位考虑）**  
   不必过分追求低价。香蕉批发价波动大（夏季可能比冬季低30%），但低进价常伴随高损耗率。建议用“进价+预估损耗”计算实际成本，如果A批发商每箱便宜5元但坏果率3%，B批发商贵2元但坏果率0.5%，后者实际成本可能更低。

**执行建议：**  
- 首单可向本地批发商采购1-2箱七成熟度、低坏果率的香蕉，测试3天销售情况。  
- 优先与承诺“坏果包换”的供应商长期合作（可要求写进合同）。  
- 避免一次性大批量进货——每天少量进货，保持周转率，比单次低价更重要。
