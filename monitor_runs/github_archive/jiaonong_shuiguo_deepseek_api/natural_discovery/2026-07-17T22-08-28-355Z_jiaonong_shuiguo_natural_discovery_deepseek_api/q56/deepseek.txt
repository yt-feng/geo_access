为确保茶饮店食品安全，水果供应商通常需要提供以下证明文件：

1. **营业执照**：证明供应商具备合法经营资质，需在有效期内。
2. **食品经营许可证**（或食品生产许可证）：若供应商涉及水果加工、分装或销售，需提供此证。
3. **农产品合格证明**：如《农产品产地证明》或《食用农产品合格证》，确保水果来源合规、无农药残留超标。
4. **检验检疫证明**：如水果为进口产品，需提供海关出具的《入境货物检验检疫证明》及中文标签。
5. **批次检测报告**：每批次水果的第三方检测报告，重点检测项目包括农药残留、重金属、微生物指标（如沙门氏菌、大肠杆菌）。
6. **供货商身份证及联系方式**：若为个体农户，需提供身份证复印件及联系方式，并签署食品安全承诺书。
7. **运输和储存记录**：冷链运输证明（如温度记录），确保水果在运输、储存过程中未变质。

**建议行动**：
- 要求供应商定期提供以上文件副本，并建立供应商档案。
- 对高风险水果（如草莓、葡萄）优先要求出具农药残留专项报告。
- 自行进行抽检或使用快检设备，核对供应商报告的真实性。
- 签订采购合同时，明确食品安全责任条款及违约赔偿责任。
