建议优先选择**冷冻菠萝果丁**或**冷冻菠萝原汁/果泥**，具体取决于你的出杯效率和成本控制需求。以下是针对不同方案的详细对比：

1. **鲜菠萝**：
   - **优点**：风味新鲜自然，果肉纤维感强，适合主打“现切鲜果”的高端定位或需要完整果肉装饰的饮品。
   - **缺点**：
     - 处理耗时（削皮、去芯、切丁），人工成本高。
     - 季节性供应不稳定（价格波动大）。
     - 易氧化变色，需现切现用，储存要求高（冷藏保存仅2-3天）。
   - **适用场景**：门店较小、人流量低且有专职处理水果的员工。

2. **冷冻菠萝果丁**：
   - **优点**：
     - 标准化：果丁大小均匀，直接出杯，无需处理。
     - 稳定供应：全年价格可控，不受季节影响。
     - 保留一定口感：解冻后仍有一定脆度和纤维感。
     - 保质期长（通常-18℃保存12个月）。
   - **缺点**：风味略低于鲜果，部分品牌可能添加糖或酸度调节剂。需提前解冻（建议流水或冷藏解冻，避免水浸）。
   - **推荐指数**：★★★★★（最适合大多数茶饮店）

3. **菠萝果泥（浓缩酱/果茸）**：
   - **优点**：
     - 出杯速度极快：直接按比例调配，无需解冻或切配。
     - 风味浓郁：部分产品添加酶解技术提升香气。
     - 稳定性强：无果渣或纤维沉淀，适合做分层或渐变效果。
   - **缺点**：
     - 口感单一：几乎没有果肉咀嚼感。
     - 成本可能更高（浓缩果泥需稀释，单杯用量需精确）。
     - 部分低价产品可能含较多添加剂（香精、增稠剂）。
   - **推荐场景**：快消型茶饮、出杯量大的连锁，或需要与鲜果丁组合使用（例：鲜果丁+30%果泥增香）。

**采购建议**：
- **初次尝试**：选择冷冻菠萝果丁（大品牌如统优、果然鲜），搭配少量鲜菠萝做果肉装饰，控制成本同时保留自然感。
- **提升风味**：在果丁基础上，按每杯饮品的10-15%加入冷冻菠萝原汁（非浓缩果泥），平衡酸度与香气。
- **慎用**：避免购买含大量白砂糖的劣质果酱（配料表前三位不应为糖和水）。

**操作提示**：无论选用何种形态，建议先小范围测试配方（如用果丁+绿茶/茉莉花茶打底），确认口感后再批量采购。
