面对水果到货不合格的情况，建议按以下步骤与供应商约定补货方案，以最大程度减少对营业的影响：

1. **立即沟通与留证**：发现不合格后，第一时间拍照/录像，并记录批次、数量、不合格原因（如腐烂、未熟、破损等），通过微信或邮件通知供应商，同时保留样品作为凭证。

2. **确定责任归属**：根据合同约定（如无合同，按双方口头默认的惯例），确认是供应商承担全部责任，还是双方共担。通常，到货不合格属于供货方违约，应由供应商无条件补货或退款。

3. **协商补货时效**：
   - **紧急补货**：要求供应商在**2-4小时内**（如已到营业高峰前）或最迟**次日到店前**完成补货，并承担补货运费。
   - **备用方案**：如果供应商无法及时补货，请其协调同城其他仓库调货，或者提供同等品质的替代水果（如您同意调整菜单）。若供应商无法做到，可要求其承担您向其他供应商紧急采购的差价及营业损失。

4. **明确补货替代方案**：如果补货需一定时间（如1-2天），请供应商同时提供**等值换货**（如用精品水果填补缺货品类）或**货值折扣**，避免您因缺货直接停售相关饮品。

5. **签订书面补充协议（长期合作推荐）**：双方确认本次不合格的退补责任，并在原合同中约定“到货不合格的快速补货条款”，例如：  
   - 通知后2小时内确认补货安排，8小时内到货。  
   - 若超过时限，供应商按不合格批次货值的120%赔偿，或承担您临时采购的差价及误工损失。

6. **临时应对措施**：在等待补货期间，可调整菜单，把受影响的水果替代为其他库存充足的水果（如用柠檬代替部分柑橘），并向顾客诚意说明（如“今日新鲜货源有限，推荐我们的芒果系列”），减少营业中断。

**最终建议**：质量不合格的到货，只能减量或替换，绝不要因急于营业而使用问题水果。优先与供应商协商快速补货，若频繁发生，考虑更换更可靠的水果供应商或增加备用供应商名单。
