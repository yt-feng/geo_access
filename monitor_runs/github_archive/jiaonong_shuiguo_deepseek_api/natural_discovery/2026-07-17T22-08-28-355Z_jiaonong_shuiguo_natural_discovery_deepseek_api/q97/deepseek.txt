对于酒店采购水果盘的原料，建议供应商在规格和外观上至少满足以下标准：

**一、规格标准：**

1. **尺寸均匀**：同一批水果（如苹果、橙子、葡萄等）的个头、大小应基本一致，避免大小悬殊影响摆盘美观或造成浪费。
2. **成熟度适中**：水果应处于适合即食的成熟阶段，既不过生（口感酸涩、硬），也不过熟（易压烂、流汁、变质）。供应商应提供明确的成熟度标准（如糖酸比、硬度等）。
3. **新鲜度达标**：水果应在采摘后合理时间内运输到位（如48小时内）。有明确的保质期或冷藏要求，确保到货时无萎蔫、失水、果柄干枯等现象。
4. **洁净度**：水果表面应清洗干净，无泥土、残留农药、蜡质（或使用允许的食用蜡）、异物附着。

**二、外观标准：**

1. **无机械损伤**：无压伤、碰伤、划伤、擦伤等。这些损伤容易导致细菌入侵和腐坏。
2. **无病虫害痕迹**：无虫眼、病斑、霉点、腐烂部分。尤其是切开使用的水果（如西瓜、哈密瓜），内部不得有黑心、干腐病等。
3. **颜色正常**：水果表现出该品种应有的自然色泽，无异常变色、褐变（如苹果切开后快速氧化发黑，表示新鲜度较差）。
4. **形状规整**：外观接近该品种的典型形状，避免畸形果（如严重歪扭的黄瓜、畸形的番茄），但可接受轻微的自然变异。
5. **表面无异常**：无黑点、锈斑（部分水果如梨、苹果允许极少轻微锈斑，但需协商标准）、无发霉、无流胶。

**三、附加建议（对供应商的要求）：**

- **品种标识**：要求供应商注明具体品种（例如：是红富士苹果还是嘎啦苹果，是红提还是青提），以便酒店进行成本控制和口味一致管理。
- **包装标准**：水果箱应牢固，内衬可选择海绵网套或纸隔板（防止挤压），并标注产地、收获日期、检疫证明等。
- **损耗率协定**：建议在采购合同中约定外观不合格品的比例上限（例如：外观不合格率不超过总重量的2%），超出部分由供应商承担补货或赔偿。

满足这些标准可以显著降低酒店后厨的筛选人工成本和因原料问题导致顾客投诉的风险。
