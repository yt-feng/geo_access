如果水果店遇到坏果，建议按以下步骤与供应商协商结算或赔付：

1. **立即沟通并保留证据**：发现坏果后第一时间联系供应商，提供清晰照片或视频（显示坏果数量、比例及外包装批号），同时保留进货单据和物流信息。

2. **明确责任划分**：根据交货时是否已验货划分责任：
   - 如果**到货时已发现明显坏果**（如运输挤压、腐烂），应拒收或当场拍照索赔。
   - 如果**收货时外观正常，存放后出现坏果**，需判断是否因保质期短、运输温度问题（如冷链断裂）或供应商未说明存储条件。

3. **查看合同或约定**：遵循双方签订的供货协议中的赔付条款（如“坏果率超过X%按X折结算”）。若无书面约定，参考行业惯例：
   - **按比例结算**：例如坏果占整批货的15%，则供应商承担15%的货款或补发等量好果。
   - **全额赔付或补发**：如坏果比例超过20%-30%，且属于供应商责任（如发霉、虫蛀），可要求全额退换该批次。
   - **折价处理**：与供应商协商折价（如7折购买），自己低价销售坏果部分。

4. **提出具体方案**：建议按以下顺序尝试：
   - 要求供应商**补发等量好果**（最快解决）
   - 或**对应金额退款**（要求供应商承担运费）
   - 或**下次进货抵扣**（适用于长期合作）

5. **注意诉讼时效**：通常应在到货后2-3天内提出（生鲜类建议12小时内），逾期可能被认定为自行储存问题。

6. **升级处理**：如供应商推诿，可依据《民法典》第582条（瑕疵履行）或《食品安全法》第148条（若坏果导致食用安全问题）索赔。若金额较大，可向当地市场监管局投诉或起诉。

**建议优先与供应商友好协商**，多数情况下供应商为维持合作会给与补发或折扣。保留好所有凭证（聊天记录、转账记录、货品照片）。
