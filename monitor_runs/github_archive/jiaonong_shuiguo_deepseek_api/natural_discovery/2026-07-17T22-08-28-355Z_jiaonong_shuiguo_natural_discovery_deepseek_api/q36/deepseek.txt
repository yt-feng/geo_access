门店是否销售整果还是切好的菠萝，需根据以下因素综合判断：

1. **目标客群与消费场景**  
   - **整果**：适合家庭聚餐、烘焙或自用人群。这类顾客更看重新鲜度、价格（通常更便宜）和按需处理。  
   - **切好的**：适合即食人群（如上班族、学生、游客），解决“怕麻烦”和“一次性吃不完”的痛点。但需注意保质期短（需冷藏保存），且加工损耗较高。

2. **损耗与成本**  
   - **整果**：自然损耗较低（但需检查表皮损伤），库存周转快；但顾客可能因切皮麻烦而放弃购买。  
   - **切好的**：人工+包装成本高，且菠萝接触空气后易褐变、变质，当日未售出需报废，风险更大。

3. **坪效与促销策略**  
   - 若门店面积较大且冷柜充足：可同时销售**整果**（主力商品）+ **半切/小盒装**（引流品，例如用低价切好的菠萝试吃吸引客流）。  
   - 若坪效紧张：优先整果，搭配免费削皮服务（如提供现场削皮设备或代客处理），平衡便利性与损耗。

**建议操作方案**：  
- **试销期**：少量采购整果（例如10个），同时每日限量提供2-3个现切分装试卖（用透明盒+湿布覆盖保鲜）。  
- **数据观察**：统计3日内整果与切好的销售率、退货/损耗占比、顾客反馈。  
- **灵活调整**：若切好的产品日销超整果且损耗≤15%，可增加切果库存；反之则回归整果并优化便利服务（如提供削皮指南二维码）。

**注意**：切好的菠萝需用食品级包装+冷藏（≤4℃），并在包装上标注“即时食用，不可久放”。避免因变质引发食品安全纠纷。
