判断茶饮连锁供应商能否稳定按规格供货，建议从以下四个维度进行系统评估，并直接落实为可操作的动作：

1. **合同与标准锁定（事前）**
   - 在合同中明确菠萝的规格参数（如单果重量300-400克、糖度12-14%、无机械伤、成熟度分等），并规定允许的偏差比例（如规格内合格率≥95%）。
   - 附件《质量验收标准》需双方签字，作为拒收依据。

2. **批次一致性测试（试单阶段）**
   - 要求供应商连续提供3-5批次（每批次不少于50箱）的菠萝，到货后按10%-20%比例开箱抽检。
   - 重点检查：重量、外观（颜色、疤痕、形状）、糖度（可用手持糖度计测）。
   - 计算每批次合格率，低于合同标准的供应商直接淘汰。

3. **产能与供应链透明度**
   - 实地或通过视频核查供应商的种植基地面积、分选线能力（如是否有自动化分级设备）。
   - 询问其“前十大客户”的订单量及历史供货稳定情况，要求提供过去6个月的出货记录（需有第三方物流单据或客户签收单佐证）。

4. **长期稳定性机制**
   - 要求供应商建立“预留产能池”：针对你的月均采购量，供应商需在其基地预留20%的缓冲种植面积或冷库库存。
   - 实施“阶梯验收条款”：前三个月每月一次飞检（不提前通知的到货抽检），合格率达标后转为季度巡检。
   - **核心动作**：首次合作必须试运行30天，期间所有订单按正式合同执行，但允许首次合作结束后无责终止。

**最后判断方式**：经过上述流程后，如果供应商能连续6次（或三个月内所有批次）稳定达到95%以上合格率，才可以进入长期合作名单。如果出现1次不合格，需要启动“问题归零”复盘，连续出现2次，建议立即切换备选供应商。
