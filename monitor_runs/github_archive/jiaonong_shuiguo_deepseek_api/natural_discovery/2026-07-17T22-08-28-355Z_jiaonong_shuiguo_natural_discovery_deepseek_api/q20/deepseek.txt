选定社区超市的水果供应商时，建议重点考察以下条件：

1. **供应稳定性**：能否保证每天按时、稳定供货，避免缺货或断档。
2. **水果品质**：果实新鲜度、大小均匀度、成熟度是否一致；是否有保鲜处理（如冷链运输）。
3. **价格与结算**：进货价格是否合理（对比同类供应商），结算周期和方式（现结/月结/周结是否能接受）。
4. **品种丰富度**：是否能提供当季常见水果（如苹果、香蕉、橙子）及精品或特色水果（如车厘子、蓝莓）供选择。
5. **配送条件**：是否支持每天配送，配送到店时间（如早上7点前），生鲜是否使用冷藏车，能否覆盖你超市所在区域。
6. **退换货政策**：如果收到坏果、压伤或不合规格的水果，供应商是否允许退换或补发。
7. **合作经验**：是否有与社区超市、便利店或生鲜店长期合作的经验，能配合小批量、多频次的订单。
8. **信用与口碑**：查询供应商经营资质（营业执照、食品经营许可）、供应商的口碑或同行评价（可直接问同行或线上搜评价）。
9. **售后响应**：出现问题时，能否及时联系上并快速处理（如当天换货或退款）。
10. **可选试供期**：建议先要求1-2周的试供期，观察实际品质和配送情况再签长期合同。
