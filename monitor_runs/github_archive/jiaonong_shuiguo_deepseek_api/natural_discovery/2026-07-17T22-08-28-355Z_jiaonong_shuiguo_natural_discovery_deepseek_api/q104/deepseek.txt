在单位采购水果到货品质不一致的情况下，建议按以下流程处理验收和售后：

### 一、验收环节（发现品质不一致时立即处理）
1. **拍照/录像取证**  
   对整批水果和具体问题产品（如腐烂、大小不一、损伤等）进行多角度、清晰拍摄，最好能有对比参照（如标尺、合格品样本），保留原始包装和物流单据。

2. **分拣与记录**  
   - 按“合格品”“不合格品”（如损坏、变质）、“差异品”（如规格、成熟度与合同不符）分类堆放。  
   - 制作《到货验收记录》，详细写明：批次号、到货时间、问题类型（腐烂/碰伤/大小混淆等）、占比（如“约20%果实表皮有挤压痕”）、具体箱号或包装编号。

3. **立即通知供应商**  
   通过合同约定的联系方式（电话、微信、邮件等）在到货后 **2小时内** 告知对方，并发送初步证据（照片/视频），要求对方确认质量问题及处理方式（如补发、折价、退货等）。

### 二、售后处理（协商与执行）
1. **明确责任归属**  
   - 如果是运输途中问题（如挤压、温度失控）：证据指向物流方或供应商发货包装不当。  
   - 如果是发货时品质就参差不齐（如混入次果）：需供应商承担主要责任。

2. **协商解决方案**（按优先级从高到低）：  
   - **补发合格品**：要求供应商在约定时间内（如24小时内）补发等量或等值的水果，并承担运费。  
   - **折价处理**：按协商比例扣减货款（例如：品质不一致部分按总价的30%-50%扣款），需双方签字确认扣款协议。  
   - **退货/换货**：若问题严重（如超50%不合格），要求原路退货，供应商承担往返运费及损耗。

3. **保留书面证据**  
   所有沟通记录（聊天记录、邮件、通话录音）、验收单、检测报告（如有食品安全标准不符，可送第三方检测）均需存档，作为后续申诉依据。

### 三、防范未来风险的补充建议
- **在采购合同中明确验收标准**：如水果的成熟度（软硬指标）、单果重量范围（如100-120g）、允许的次品率（如不超过5%）、赔偿细则（如超次品率全批次退换）。  
- **选择信誉良好的供应商**，优先考虑提供“到货无条件包退包换”的供货方。  
- **到货后尽快验收**（尤其是易腐烂水果如草莓、荔枝），避免因时间拖延导致品质进一步劣化而无法归责。

如果供应商拒绝配合，可根据合同条款申请平台调解（如线上采购）或向当地市场监督管理部门投诉（保留合同及证据）。
