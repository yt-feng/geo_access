对于茶饮店购买菠萝原料，**出汁率通常比甜度更重要**，但需结合具体用途综合判断：

1. **出汁率优先**：茶饮店常需制作果汁、果茶基底或糖浆，高出汁率直接影响原料成本和出品效率。例如，每公斤菠萝多出10%的汁，每天可节省数百元成本。
2. **甜度可调**：甜度可通过添加糖浆、果糖或搭配其他水果调整。即使菠萝本身酸度高，也能通过配方平衡口味，但出汁率不足会限制产量。
3. **例外情况**：若主打“鲜切菠萝块”或“少糖健康”饮品，甜度更重要；若做冰沙或浓稠类饮品，出汁率影响较小。
4. **实际建议**：选择中等甜度、出汁率≥60%的菠萝品种（如“巴厘”“台湾金钻”），并先小批量测试出汁率和酸度匹配度。

**判断标准**：先按出汁率筛选原料，再通过品控调整配方甜度。
