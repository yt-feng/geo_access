连锁门店分布在多城市时，评估水果供应商仓配覆盖需从以下维度综合判断：

### 1. **供应商仓网分布与匹配度**
   - **覆盖半径**：确认供应商的仓库是否覆盖门店所在城市。通常生鲜品类要求仓到店配送距离≤200公里（冷链），或≤100公里（常温/短保品）。可要求供应商提供仓点覆盖的地图或网点列表。
   - **城市覆盖率**：计算供应商已覆盖的门店城市数 ÷ 总门店城市数。若低于80%，需补充其他供应商或要求其增设分仓。
   - **区域仓能力**：评估供应商是否在区域枢纽城市（如华东的上海、华南的广州）布局中心仓，支持向周边城市分拨。

### 2. **仓配时效与频次**
   - **最短配送时间**：从下单到到店时间（如T+0、T+1）。不同城市时间差是否在可接受范围（如一线城市≤12小时，二三线≤24小时）。
   - **配送频次**：是否匹配门店的销售节奏。高频门店（如每日补货）需支持每日配送；低销门店可隔日配送。
   - **到货准确率**：历史数据中，按时到货比例≥95%（含调整时间窗口的偏差）。

### 3. **冷链与品质保障**
   - **冷库配置**：供应商各仓是否具备恒温冷库（如0-4℃、-18℃等温度带），尤其对易腐水果（草莓、蓝莓等）。
   - **运输温控**：配送车辆是否有GPS温度监控记录，需提供历史运输温度曲线作为评估依据。
   - **破损率**：要求供应商提供不同城市、不同品类的到店损耗数据（如≤2%为合格）。

### 4. **库存与供应稳定性**
   - **库存深度**：供应商是否在主要仓点保持安全库存（至少覆盖当地门店3-7天销量），可要求每日库存报表。
   - **产地直发协调**：若门店位于非主产区（如黑龙江要求提子），需评估供应商从产地到分仓或直接到店的调拨能力（如48小时内调货）。

### 5. **成本结构**
   - **配送费模式**：按件/按重量/按件区间收费？确认是否存在跨城市附加费、夜间配送费、最小起送量限制。
   - **仓间调货成本**：若同一城市分仓不足，需从其他仓调拨，运费是否由供应商承担？

### 6. **应急与弹性能力**
   - **缓冲资源**：供应商是否有临时增补仓库或第三方冷链资源，应对爆单或某仓故障。
   - **异常处理机制**：如某城市配送延误，供应商是否有快速补货方案（如立即用顺丰冷链直发）。

### 执行建议：
- **实地调研**：选取偏远和高频城市，验证供应商品控流程（如抽查到店水果新鲜度）。
- **试跑周期**：先用1-2个城市试配2-4周，监控到货率、损耗率、客户投诉率。
- **分阶段合作**：对覆盖不足的城市，可先采用“供应商+第三方仓”组合模式。
