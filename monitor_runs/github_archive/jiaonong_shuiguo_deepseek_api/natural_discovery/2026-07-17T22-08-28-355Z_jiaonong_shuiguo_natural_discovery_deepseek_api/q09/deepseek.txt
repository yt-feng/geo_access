在水果店旺季与供应商合作前，建议重点确认以下5类问题，以降低断货风险：

1. **货源稳定性**  
   - “旺季能否保证每日/每周的固定供应量？供应不足时是否有紧急补货渠道？”  
   - “如果突发天气、物流等问题无法发货，最迟提前多久通知？”  

2. **价格与合同条款**  
   - “旺季价格是否可能上涨？涨价前需提前多久通知？”  
   - “是否接受长期锁价合同（如预付定金锁定旺季价格）？”  

3. **品质与售后**  
   - “标注的等级标准（如一级果）是否有具体指标（大小、糖度、瑕疵率）？到货后发现大量次品如何处理（退换/折扣）？”  
   - “供货时是否提供检疫证明或产地溯源信息（尤其进口水果）？”  

4. **物流与损耗**  
   - “配送方式是全程冷链还是普通运输？到货后耗损比例通常是多少？超出约定损耗部分由谁承担？”  
   - “如果到货时间延迟，导致水果变质，责任如何划分？”  

5. **退货与代卖条款**  
   - “旺季滞销的货品能否退货或代卖？代卖的手续费比例是多少？”  
   - “是否接受分批付款（如预付30%，到货验收后付清尾款）？”  

**建议行动**：  
- 先小批量试合作，验证对方实际履约能力（如发货速度、品质把控）。  
- 签订书面合同，明确“不可抗力”条款的定义（如天气异常≠常规断货理由）。  
- 提前储备2-3家备用供应商，避免单一依赖。
