针对超市水果到货品相不好的情况，建议在采购合同或协议中约定以下售后流程：

1. **验收及异议期**：收货时当场验收，如有品相问题（如腐烂、损伤、不新鲜等），应在收货后 **24小时内**（具体时间可根据水果易损程度调整）以书面或拍照方式向供应商提出异议。

2. **判定标准**：明确“品相不好”的具体标准，例如：腐烂面积超过5%、明显机械伤、严重脱水软蔫、颜色异常等，避免主观争议。

3. **处理方式**（按优先级排序）：
   - **全额退货退款**：若整批水果品相都差，要求供应商承担运费。
   - **部分拒收**：对不合格水果按数量扣除货款，由超市自行处理（如折价销售或报废）。
   - **换货补发**：供应商在 **2-3天**内补发合格品，并承担全部运费。
   - **折价补偿**：双方协商按比例降价，例如按原价 60%-80% 结算。

4. **留存证据**：要求超市保留不合格水果的**清晰照片/视频**、物流签收记录及异议通知的**截图或邮件**，作为后续处理依据。

5. **争议解决**：若供应商拒不处理，超市可采取：拒付对应货款、扣减履约保证金、暂停合作或提交合同约定的仲裁/法院管辖。

6. **补充条款**：对于易腐烂水果（如荔枝、草莓），可约定 **“到货后即时沟通”**，并通过微信、电话等快速反馈，避免拖延导致损失扩大。

**提示**：具体时限和补偿比例需根据水果种类、价格及双方协商调整。建议在合同中明确引用《民法典》关于货物检验的条款（第620条起）。
