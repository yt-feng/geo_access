为了减少香蕉在店内放坏，采购时建议按以下要求控制熟度和到货时间：

1. **熟度要求**：要求供应商提供**3号色至4号色**的香蕉（即果皮整体绿色，棱角分明，但果肉已开始转黄。按国际标准：1号全绿，2号微黄边，3号绿多黄少，6号全黄）。**不要超过5号色**（黄多绿少），因为熟度太高容易在2-3天内过熟变黑。

2. **到货时间**：建议**到货后至少保留3-5天的销售期**。根据门店销量，要求供应商在**你计划上架销售的3-5天前**到货（例如：若你周一开始卖，则周四或周五到货）。同时要求**周末避免到货**（若周末销量高，可要求周四周五到货，避免过早到货在周末前变坏）。

3. **包装与运输**：要求供应商采用**透气包装**（如带孔的塑料袋或网袋），运输过程**保持15-18℃**（避免温差过大导致催熟）。到货后立即检查，剔除已有斑点、软烂的香蕉。

4. **备用方案**：与供应商协定**到货后24小时内发现明显过熟（超过5号色，或果皮出现黑斑）可退换**，或要求分批到货（如每周2次，每次量少），降低单次库存压力。

> **补充建议**：若店内销量不稳定，可要求供应商提供**颜色样板卡**，每次到货签字确认熟度等级，避免纠纷。同时，在到货当天将香蕉按熟度分柜摆放：较熟的（4-5号色）放显眼处优先销售，较生的（3号色）放后区或冷柜（13-15℃）延缓后熟。
