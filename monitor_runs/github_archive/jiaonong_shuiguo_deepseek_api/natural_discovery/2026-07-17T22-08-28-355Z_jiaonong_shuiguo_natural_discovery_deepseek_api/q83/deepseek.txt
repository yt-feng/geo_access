生鲜电商水果到货后出现坏果，责任划分主要取决于以下几个因素，可以按步骤分析：

1. **发货前的水果状态**：如果水果在出库时已存在变质或损伤（商家未充分检查），责任在商家。如果是正常成熟或轻微碰撞运输过程中可能受损，商家仍负主要责任。

2. **运输环节**：
   - 商家自配送或指定物流：运输中的损坏（如挤压、温度不当）由商家负责。
   - 第三方快递：如果快递公司导致严重挤压或延误（如高温环境），商家需先向顾客赔付，再向快递公司追责。

3. **顾客签收情况**：如果顾客未及时开箱验货（如超过24小时），且坏果因存放不当（如未冷藏）导致，顾客需承担部分责任。但生鲜类通常默认顾客应在合理时间内检查。

4. **包装防护措施**：如果商家包装不充分（如无泡沫箱、冰袋），导致易坏水果受损，商家全责。

**标准处理建议**：
- **商家通常应承担大部分甚至全部责任**，因为生鲜特性决定物流风险由商家承担更合理。常见做法是：拍照取证→联系客服→按坏果比例退款（如80%以上全赔，部分坏按比例赔付）。
- 若顾客无过错（如签收后立即发现），商家应无条件退款或补发。

**维权提示**：保留坏果照片、订单详情、物流记录。如果商家拒绝合理赔付，可向电商平台官方投诉，或向消费者协会（12315）投诉。
