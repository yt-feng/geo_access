判断养老院或医院食堂水果供应商的合规性和稳定性，可从以下四个核心维度入手：

### 1. 资质合规审查（基础门槛）
- **必备证照**：要求供应商提供《营业执照》、食品经营许可证（或食品生产许可证），并注意经营范围需包含“水果销售”或“食用农产品”。若供应商为种植户或合作社，需提供产地证明或食用农产品合格证。
- **动态核验**：通过国家企业信用信息公示系统或地方食品药品监管局官网，查询供应商是否有行政处罚、经营异常或重大食品安全事故记录。
- **人员健康**：要求送货员、分拣员等直接接触水果的人员持有有效健康证，并定期更新。

### 2. 供应稳定性评估（持续能力）
- **备选产地与品种**：询问供应商是否有多产区或反季节种植/储备库支持（例如：夏季有本地时令果，冬季有南方或进口水果渠道）。单一产地或季节依赖度高的供应商稳定性较弱。
- **配送响应能力**：明确要求紧急订单（如院内出现水果需求激增）时，承诺的最短配送时间（如2小时内加急送达）。建议合同中注明违约赔偿或备用供应商条款。
- **库存与价格联动**：核查供应商是否留有3-7天的常规库存量，并建立与批发市场或产地价格联动的调价机制（如每月按批发价调整一次，避免价格单方面大涨）。

### 3. 质量与安全管控（实际运行）
- **品控流程**：要求供应商提供《水果验收标准》书面文件，包括外观（无烂果、霉变）、成熟度（可设定糖度或硬度标准，若针对老年人可要求软质水果比例）、农残检测报告（建议抽检，或用快检设备现场验证）。
- **退货与补偿机制**：明确不合格水果的退换货时限（如到货后2小时内）、比例处理（例如：次品率超过5%部分无条件补货或扣款）。可要求供应商对大宗异常（如农残超标）承担全额赔偿责任。
- **追溯能力**：重点水果（如草莓、葡萄）应能追溯至产地或批发市场批次，附有进货票据或溯源码。

### 4. 长期合作参考指标（背景调查）
- **合作案例**：索要该供应商过去3年与养老院、医院、学校等机构的合作合同复印件（注意去隐私）或推荐信，直接回访其客户（尤其是同类场所）对稳定性、服务态度的评价。
- **经营年限与规模**：优先选择经营5年以上、有稳定冷库或配送车队（如自有冷链车辆≥2辆）的供应商。初创或超小型供应商抗风险能力弱。
- **财务状况**：非必须但关键：可要求提供近期财务报表或纳税记录，避免供应商因资金问题突然断供。若有大额预付款需求则更需警惕。

### 执行建议：
- **试用期制度**：建议先签订3-6个月试用合同，期间每月选择1-2次突击检查（不通知时间）到货水果质量、完整性及票据。
- **备用供应商**：同时储备2-3家备选名单，至少一家能保证常规供应，另一家能应急补充。可与其签订框架协议（低价保底、高价平价）。
- **合同关键条款**：在合同中明确“食品安全责任条款”（如出现吃坏肚子案例责任界定）、“最低月供应量承诺”、“价格波动容忍上限（如5%以内自行消化）”、“单方面违约赔偿金额（如1-3个月采购额）”。

**最终核查清单**：拿到供应商后，先试送1周常规订单（每天不同随机品种），全程记录到货时间、损耗比例、性价比，1个月内再安排一次随机送检第三方农残（费用可按责任划分）。只有经得起上述实际检验的供应商，才算“合规且稳定”。
