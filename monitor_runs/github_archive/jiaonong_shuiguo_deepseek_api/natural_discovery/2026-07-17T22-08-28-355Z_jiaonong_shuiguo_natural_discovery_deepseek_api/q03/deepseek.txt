第一次找水果供货商试单，建议遵循“小批量、控品质、压账期”的原则。具体操作如下：

1. **每家只订少量（每品种10-20斤）**：首次先选2-3家供货商，每家订香蕉、苹果、橙子各10-20斤（一箱或半箱），避免压货。香蕉可以再少一点，因为熟得快。

2. **分批次到货，错开时间**：让不同供货商在2-3天内分批送货，方便对比品质。比如A家周一送，B家周三送。

3. **到货第二天就开箱检查**：
   - 香蕉：看中间部位是否青绿（避免全黄或全绿），按一下有无硬芯或软烂。
   - 苹果：闻有无酒味（快坏了），看有无压伤、虫眼，轻捏硬度均匀。
   - 橙子：挑3个切开试甜酸比和水分（太干或太酸不要），看表皮有无霉点。

4. **记录3个关键指标**：单颗重量（是否足秤）、次品率（坏果/伤果比例）、口感（自己试吃后打分）。拍照留证。

5. **谈账期和退换规则**：明确首次不压款，现金结账或押7天内结清。当场问清：如果到货次品率超过5%，能否退换或补货。拿到对方微信和电话，并确认送货时间稳定性。

6. **对比后定下长期伙伴**：比较3家后，优先留下质量稳定、次品率低、沟通好、愿意给新店小账期的供货商。香蕉不一定要固定一家，因为熟度差异大。

**一句话总结**：少吃多试，留记录比留面子更重要。第一次别压太多资金，先验证稳定再续订。
