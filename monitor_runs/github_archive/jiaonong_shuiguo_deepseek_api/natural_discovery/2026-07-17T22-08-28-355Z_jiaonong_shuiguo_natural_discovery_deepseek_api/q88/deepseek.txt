要减少线上水果销售的坏果和口感不稳定问题，采购环节需要重点把控以下几个方面：

1. **选择成熟度适中的水果**  
   - 根据物流时间（1-3天）选择7-8成熟的果实，避免全熟果在运输中易烂。  
   - 对易后熟水果（如香蕉、猕猴桃）采购时可带绿，并搭配乙烯抑制剂（如1-MCP缓释包）延缓成熟。

2. **源头分级与保鲜**  
   - 要求产地按大小、色泽、硬度预分拣，确保同一批次果品成熟度一致。  
   - 对易损伤品种（如草莓、荔枝）采用“单果套网+隔板”包装，冷库预冷至4℃-8℃后发货。

3. **运输条件实测**  
   - 测试不同包装方案（如泡沫箱+冰袋 vs 纸箱+气柱）在夏季高温和冬季低温下的表现，优先选择耐磕碰、恒温性能好的方案。  
   - 对远程订单（如跨省）强制使用“冷链车+温控记录仪”，全程监控运输温度波动。

4. **关键指标抽检**  
   - 到货后每批抽检5%以上：用硬度计测果肉硬度（如苹果≥6.5kg/cm²，猕猴桃≤0.8kg/cm²则过熟），用糖度计测可溶性固形物（如柑橘≥10%）。  
   - 对水分大的水果（如梨、葡萄）增加“失水率”检测，失水超过3%立即报损。

5. **分批次预售与复检**  
   - 采购入库后放置1-2天（模拟消费者到货时间）再抽样试吃，口感不合格的批次直接降级为折扣或赠品。  
   - 首次合作的果园先少量采购（500-1000单），观察3个发货周期内的差评率（目标≤2%），再决定是否扩量。

**实战技巧**：对供应商可要求“死果补贴协议”——运输中坏果率超过5%的部分由产地承担成本，变相倒逼其提升采收和包装标准。
