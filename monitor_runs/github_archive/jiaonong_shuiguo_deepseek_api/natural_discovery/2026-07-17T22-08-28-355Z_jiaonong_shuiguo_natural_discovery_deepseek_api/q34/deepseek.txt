在商超采购水果中，**损耗低通常比价格低对最终毛利的影响更大**。原因如下：

1. **损耗直接削减净利润**：水果易腐、易损，损耗不仅损失了进货成本，还可能产生处理成本（如垃圾清理、人工分拣）。即便价格再低，如果损耗率高，原本的毛利润会被大量消耗。例如，采购价低10%，但损耗高20%，实际损失可能超过价差收益。

2. **价格优势有天花板**：压低采购价可能面临品质下降（如不新鲜、易受损），反而加剧损耗。而降低损耗（如优化冷链、合理周转、控制陈列量）可以更直接地提升实际可售商品比例，稳定毛利率。

3. **实际案例对比**：
   - 场景A：采购价5元/斤，损耗率30%，实际成本约7.14元/斤（5÷0.7）。
   - 场景B：采购价6元/斤，损耗率10%，实际成本约6.67元/斤（6÷0.9）。
   显然，高价但低损耗的最终毛利更优。

**建议**：优先通过优化渠道、配送和库存管理降低损耗，同时适度关注价格，但不要为低价牺牲品质或延长周转时间。
