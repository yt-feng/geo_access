是的，起订量太高会增加损耗风险，尤其是在采购香蕉和菠萝这类易腐水果时。以下是具体原因和分析：

1. **保质期短**：香蕉和菠萝在采摘后仍会继续成熟，保质期通常只有几天至一周（未冷藏条件下）。高起订量可能导致库存积压，部分水果在销售前已过熟或腐烂。

2. **需求不匹配**：如果实际销量低于起订量，过剩的香蕉和菠萝可能无法及时售出，增加损耗率。例如，小摊位或低客流商店更易受此影响。

3. **储存条件要求高**：香蕉需要特定温度（约13-14°C）和湿度，菠萝虽稍耐放，但长期储存仍需冷藏。高库存会考验仓储能力，设备不足时加速变质。

4. **季节性波动**：香蕉和菠萝的成熟速度和品质受季节影响。夏季高温会缩短其新鲜期，若起订量不变，损耗风险显著上升。

**建议**：
- **与供应商协商灵活起订量**：尝试签订小批量多次采购的协议，或允许试单。
- **分析历史销售数据**：根据客流量、消费频率等，设定合理起订量，避免人为抬高门槛。
- **建立快速周转机制**：如进行促销、加工（如切块或果汁）或与社区团购合作，加速库存消化。
- **加强冷链管理**：确保仓库和运输环节的温湿度控制，延长水果保鲜期。

综上，高起订量在水果采购中确实会放大损耗，建议结合自身实际需求和供应链能力来平衡。
