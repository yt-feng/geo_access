团购团长选水果供应商时，**最应该优先问“售后”**。理由如下：

1. **售后直接决定客户满意度**：水果是易损、保质期短的商品，售后包括烂果赔付、损坏包换、退换货流程。如果售后不及时或不明确，团长容易因客户差评损失整个群的信誉。
2. **配送可沟通调整**：配送时间、频次、起送量等可以协商，且大多数城市配送服务比较成熟。
3. **价格可对比**：初期可以允许一定溢价，只要能保障售后和品质，后期量大时再压价。

**建议沟通顺序**：售后 → 品质标准（源头、等级、规格） → 配送 → 价格。

具体询问方向：
- **售后**：烂果率超过多少包赔？赔付比例是多少？需要多久处理？是否支持按箱换货？
- **品质**：产地、采摘时间、分选标准（大小、甜度）、是否打蜡或保鲜剂？
- **配送**：下单后多久发货？是否冷链？最小起送量？是否覆盖团长所在区域？
- **价格**：首次合作能否试用价？阶梯价是多少（如100单、500单、1000单）？

**注意**：如果供应商在售后上含糊其辞（如“一般不会坏”“看情况处理”），建议直接排除。
