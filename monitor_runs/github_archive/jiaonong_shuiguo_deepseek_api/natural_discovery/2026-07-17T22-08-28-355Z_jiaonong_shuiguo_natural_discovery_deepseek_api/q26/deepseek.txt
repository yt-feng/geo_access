针对社区超市销售小包装香蕉，采购时需重点关注包装与损耗管理。以下是核心注意事项：

**一、包装问题**
1. **透气与防护**：选用带透气孔的塑料袋或网袋，避免密封导致香蕉加速成熟；包装应有一定缓冲性（如泡沫网套），减少运输磕碰造成的褐变。
2. **规格匹配**：小包装通常建议2-3根/袋（约300-500克），便于即食消费；避免过度挤压，包装尺寸需留有空间。
3. **信息标识**：标注“未熟/可即食”等成熟度提示，以及“避免冷藏”（低于12℃会冻伤）等储存说明。
4. **防漏清洁**：检查包装封口牢固，防止汁液渗出污染货架或顾客购物袋。

**二、损耗控制要点**
1. **成熟度选择**：采购时选购**7-8成熟**的香蕉（果皮黄中带绿，无黑斑），避免全熟或过青（过青需自行催熟，增加损耗风险）。社区超市周转快，可选稍偏黄但未软化的批次。
2. **拒收标准**：拒绝有明显黑斑、开裂、表皮严重擦伤或果柄发霉的香蕉；小包装每根香蕉都应无冻伤痕迹（表皮变暗、凹陷）。
3. **分拣与包装**：到货后立即剔除病、残果，**按成熟度分层包装**—将相同色度的香蕉放入同包装袋，避免混装后“熟果催熟生果，导致部分商品过早变质”。
4. **温度与摆放**：到店后需在**13-15℃**阴凉处暂存，不可直接放进冷柜。上架时避免堆叠，使用平铺或悬挂方式，减少挤压伤。
5. **周转控制**：根据日销量定量采购，小包装香蕉保质期约2-4天（视温度），建议采用**“先进先出”和“每日分批次陈列”**（将即将成熟的批次提前上架促销）。

**三、损耗预警**
- 每日检查小包装中是否有单根变软或出现黑点，及时取出“损耗果”单独低价处理（如作为烘焙原料）。
- 对加工人员培训，避免在包装时用力握捏，造成隐性损伤。

**总结**：采购时严格把控成熟度与表皮完整性，包装采用透气防挤压设计，到店后按色度分装、低温但非冷藏存储，并设置快速周转机制。
