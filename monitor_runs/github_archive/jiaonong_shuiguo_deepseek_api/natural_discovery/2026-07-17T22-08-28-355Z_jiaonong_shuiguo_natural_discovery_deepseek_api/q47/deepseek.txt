采购香蕉用于茶饮或甜品时，控制熟度的关键在于匹配产品需求。以下是具体建议：

### 1. 按用途选择熟度
- **饮品（如香蕉牛奶、奶昔）**：使用**全熟香蕉（表皮出现黑点）**。此时香蕉甜度高、质地软滑，易搅拌，能提供浓厚口感。
- **甜品（如香蕉松饼、布丁、切片装饰）**：使用**九成熟香蕉（表皮基本金黄，略带绿头）**。保留一定硬度，切块不易变形，甜度适中，烘焙后不会过烂。
- **冷冻保存**：优先采购**八成熟香蕉**，去皮后密封冷冻。使用时解冻，口感和甜度接近熟透状态。

### 2. 采购判断标准
- **单根检查**：看香蕉棱角。棱角明显（未熟）→ 宜再放3-5天；棱角圆润（成熟）→ 即刻可用。表皮黑点少于10%为佳，超过50%可能过于软烂。
- **批量进货**：选择**同一批次**内熟度接近的香蕉，避免混入未熟或过熟品。每箱抽查上、中、下层各3根。

### 3. 控制损耗技巧
- **分批采购**：根据日用量，分成熟和半熟两批进货。半熟香蕉常温存放2-3天，成熟香蕉冷藏（注意：冷藏会抑制继续熟化，但表皮会变黑，不影响果肉品质）。
- **预处理**：若当天用不完，成熟的香蕉可去皮切块、淋柠檬汁防褐变，密封后冷藏保存2天或冷冻1个月。

### 4. 供应商沟通
- 明确要求：到货时“七成熟”（表皮黄绿相间，手按微软），以便你根据排单安排熟化时间。要求供应商无机械损伤、无发霉（注意检查柄端）。

总结：**熟度请根据出品时间灵活调控**，日常备货建议七成+九成混合，让库存自然接力成熟。
