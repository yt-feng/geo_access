生鲜平台做一件代发水果，供应商需具备以下核心仓配能力：

1. **产地/仓库分拣能力**：需设有专业分拣场地，能按果径、重量、成熟度、外观分级，并匹配不同包装规格（如家庭装、礼盒装）。

2. **冷链仓储（CSC）**：配备恒温保鲜库（如苹果0-4℃、热带水果10-15℃），并具备气调、除乙烯等保鲜设备，确保水果到仓后预处理、预冷及时。

3. **时效性发货系统**：与快递/冷链物流系统打通，支持当天下单可当日/次日凌晨发货；需能提供揽收（如每日固定截单时间前完成打包贴单）、物流信息实时更新。

4. **包装防护能力**：使用瓦楞纸箱、泡沫箱、珍珠棉隔层、吸水纸、冰袋/冰瓶等，确保抗压、防震、控温（如荔枝需双层冰袋+保温箱）。

5. **库存与批次管理**：能按SKU实时更新库存，处理预售（如按成熟期分批上架），具备批次追溯（如可发货前更换坏果）。

6. **售后与核损能力**：能快速响应平台赔付规则，匹配坏果包赔、按比例退款等；提供坏果照片核损流程（如要求拆箱后6小时内反馈）。

建议供应商还具备小批量常混装、贴面单、快速退换货的工人配置（如日发量1000单以上需5人分拣组），以及对接平台ERP系统的能力（如自动打印订单、同步物流单号）。
