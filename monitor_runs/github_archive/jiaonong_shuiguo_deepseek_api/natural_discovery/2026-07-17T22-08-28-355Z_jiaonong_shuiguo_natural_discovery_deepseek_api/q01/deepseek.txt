判断水果供应商是否靠谱，对于长期进货尤其关键。以下是几个核心判断维度，你可以逐一考察：

1. **考察水果品质与一致性**
   - 要求供应商提供样品（尤其是同一批次多箱的样品），检查成熟度、果形、有无霉变或虫眼。
   - 香蕉：注意看有没有“大把头”（柄部发黑），是否自然成熟（避免使用催熟剂过度催熟，导致肉质软烂无味）。
   - 菠萝：检查果眼是否均匀，底部是否发霉或有异味（可能已过熟或内部变质）。

2. **了解货源与冷链能力**
   - 问清香蕉、菠萝是从哪里来的（比如广西、海南、云南还是进口），不同产地成熟周期不同。
   - 要求提供冷链运输记录：水果在运输中必须保持适宜温度（香蕉13-15℃，菠萝7-10℃），中断冷链会加速腐烂。
   - 查看仓储条件：供应商的仓库是否干净、温度可控、分拣打包是否规范。

3. **试单与售后政策**
   - 先少量试购1-2箱，观察到货后的品质、损耗率（正常损耗应低于5%，旺季或长途运输可接受10%以下）。
   - 约定售后条款：比如“到货后24小时内发现坏果或严重腐烂，按比例退款/补货”。
   - 对于长期合作，要求签订合同明确：结算周期、退货标准、最低送货量。

4. **综合评估口碑与稳定性**
   - 向同行（附近水果店、批发市场常客）打听供应商的行业口碑：“他家长期供应的香蕉、菠萝品质稳不稳？有没有中途涨价或断供的情况？”
   - 观察供应商的出货量：如果同一品种每天供应量波动大，说明其货源不稳定或资金链可能存在问题。
   - 换个时间（比如早晨和傍晚）分别联系同一家供应商，看报价是否一致，服务态度是否随量价而变。

5. **合同与财务注意事项**
   - 首次合作尽量“押一批结一批”或“月结”（而非预付全年货款）。
   - 要求提供营业执照、食品经营许可证（如涉及进口水果需有检疫证明）。
   - 索要送货单据，注明品种、等级、单价、数量、日期，以备后续对账。

**执行建议**：
- 优先选择在水果批发市场（如当地一级批发）有实体摊位且入驻3年以上的供应商（可查工商登记）。
- 如果条件允许，亲自去供应商的仓库或分拣中心看一次，现场了解其分选、包装、库存管理流程。

**行动清单**（按重要性排序）：
1. 试单并验货（最直接有效）
2. 确认冷链与存储条件（防损耗）
3. 谈妥售后条款（防亏损）
4. 签正式合同（防风险）
5. 小额定金或月结（防跑路）

如果供应商对上述大部分问题都能清晰、自信地回答，并且愿意配合试单和提供书面承诺，基本可以判定为靠谱。反之，如果对方含糊其词、拒绝看仓库、没有售后保障，建议果断放弃。
