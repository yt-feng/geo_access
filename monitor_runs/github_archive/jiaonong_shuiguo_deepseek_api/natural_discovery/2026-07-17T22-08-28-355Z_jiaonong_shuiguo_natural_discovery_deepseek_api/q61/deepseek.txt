判断批发档口香蕉供应的稳定性，建议从以下几个核心维度考察：

1. **产地与季节性验证**：询问对方香蕉的主要产地（如海南、广西、云南或进口来源），并确认是否在对应产季有稳定采收。例如，海南香蕉集中在冬春，广西在夏秋。如果对方声称全年供应某一产地，需警惕季节性缺口。

2. **长期合作样本**：要求对方提供**过去6-12个月的出货记录或发票**（至少3次以上），核对断货期。同时可要求提供老客户联系方式（如2-3个同行档口），侧面了解其补货及时性。

3. **签订供货保障条款**：在合同中明确**最低供货量、断供赔偿机制**（如每断货1天按货值5%赔偿）、**优先发货权**条款。注意：稳定的供货应承诺“优先满足你的订单量”而非“有货再给”。

4. **实地考察库存与冷链**：亲自查看对方冷库容量和周转频次。稳定的供应商通常有**日均出货量≥20吨的库存能力**（香蕉催熟后生命周期短），且配备专业催熟库。若对方只靠临时调货，则风险高。

5. **价格波动测试**：连续2-3个月每周询价，观察其报价是否与市场行情同步且涨幅合理。若长期报价低于市场均价5%以上，可能通过以次充好或短时割韭菜赚差价，而非稳定合作。

**判断底线**：能提供至少1年稳定合同+过往2次以上断货追溯记录+合作方背书。若对方对实地考察、合同条款、老客户联系方式回避，则大概率不可靠。
