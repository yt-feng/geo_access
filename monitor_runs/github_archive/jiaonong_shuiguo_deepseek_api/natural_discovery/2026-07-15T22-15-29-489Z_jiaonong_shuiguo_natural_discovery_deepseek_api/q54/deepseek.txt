采购香蕉时，应优先关注**成熟度**，其次才是价格。原因如下：

1. **成熟度直接影响蛋糕品质**：做香蕉蛋糕需要成熟度较高的香蕉（表皮出现黑斑、果肉软糯），这样的香蕉甜度高、香气浓郁，能提供天然甜味和湿润口感。生香蕉（表皮青黄）硬涩、甜度低，烤出的蛋糕质地干硬、风味不足。
2. **节省额外成本**：采用成熟香蕉可减少额外糖分和香精的添加，降低原料成本。同时，过度成熟的香蕉（如果酱化）仍可用于蛋糕，但需调整配方（减少液体）。
3. **价格需结合损耗控制**：若价格极低但成熟度不够，需额外时间催熟或加糖调味，可能增加隐性成本（如催熟剂、存储空间、人工筛选）。而成熟度适中的香蕉即便价格稍高，也能直接使用，减少浪费。

**采购建议**：
- 确认香蕉表皮至少70%以上呈现黄色或出现黑点，捏起来微软但不烂。
- 批量采购时，价格可作为次要考量：若成熟香蕉价格过高（超过市场均价30%以上），可采购略青的香蕉，计划提前2-3天存放于室温催熟。
- 检查香蕉柄部是否有霉菌或潮湿，避免内部变质。

最终决策：**以成熟度为底线，在保证适宜成熟度前提下，追求性价比**。
