要判断养老院或医院食堂水果供应商的合规性和稳定性，建议从以下方面进行考察：

1. **资质审查**：要求供应商提供营业执照、食品经营许可证或农产品经营相关资质，确保其合法经营。对于有机或绿色水果，需提供相应认证证书。

2. **质量与安全**：检查供应商是否具备水果农药残留检测报告或第三方质量检测报告。定期要求提供产地证明、采摘时间等溯源信息。

3. **供货能力**：评估供应商的规模、库存、物流能力。可要求提供近三个月供货记录或与其他机构（如学校、医院）的合作案例。观察其能否满足突发订单或季节性需求。

4. **稳定性指标**：询问供应商是否有长期固定合作基地或自有农场，对比多家供应商的价格波动幅度（如是否随季节合理调整）。要求签订合同，明确供货周期、违约责任和应急预案（如断货时的替代方案）。

5. **实地考察**：对意向供应商的仓库、分拣流程进行现场查看，检查卫生条件、水果周转效率、包装是否符合食安标准。

6. **第三方信息**：查询企业信用信息公示系统或行业黑名单，了解供应商是否存在行政处罚、合同纠纷等记录。参考其他采购单位的口碑评价。

7. **试用考察期**：建议先签订短期试用合同（如1-3个月），重点观察持续供货的稳定性、退换货响应速度、沟通效率等实际表现。

最后，建立供应商评估档案，每季度或半年进行综合评分（质量、时效、配合度、价格等），优先选择评分稳定且无重大问题的供应商。
