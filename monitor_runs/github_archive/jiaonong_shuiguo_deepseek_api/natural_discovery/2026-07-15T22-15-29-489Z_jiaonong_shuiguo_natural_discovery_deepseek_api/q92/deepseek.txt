针对小面积便利店的水果经营，核心策略是 **“高流转、小批量、少品类、精包装”** 。以下是具体的控制方法：

**一、品类控制：选对“小而美”的爆款**

1.  **只做“高频、易吃、耐放”的品类**
    -   **优先选择**：香蕉（走量最大）、苹果/梨（耐放）、柑橘类（橙子、桔子）、圣女果/千禧果（一口一个，损耗低）。
    -   **谨慎尝试**：草莓、蓝莓、榴莲、山竹（太娇贵或单价高，一旦卖不掉就是纯损耗）。
    -   **坚决不碰**：需要现场切开的（如大西瓜、菠萝）、易压坏的超大件（如大哈密瓜）。

2.  **做“现切现卖”的增值服务（可选）**
    -   如果店员有空，把即将过熟或有小瑕疵的芒果、火龙果、菠萝切成果盒（保鲜膜密封），定价低于散称，既去库存又能提高毛利。

3.  **数据化选品**
    -   每天记录前3名畅销品、前3名滞销品。常规保持4-6个核心品类即可，不要超过8个品类。比如：香蕉+苹果+橙子+圣女果+当季特价品（如冬季的砂糖橘）。

**二、损耗控制：从进货到销售的全程管理**

1.  **进货环节：少进快补，宁缺毋滥**
    -   **每日或隔日进货**：保持2-3天的销量即可，不要囤货。
    -   **按“份”订货**：让批发商按“箱”或“件”送货，每次只进1-2箱最常见的品种。试销期只进半箱。
    -   **验收严格**：开箱后立刻检查底部的压伤、腐烂、霉变，不合格的当场拒收或折价。

2.  **陈列环节：只摆“卖相”最好的**
    -   **勤翻勤挑**：每天早晚各挑一次，把有疤痕、发软的挑出来，放入“特价篮”或“试吃盘”（免费试吃能加速消化）。
    -   **小份量包装**：尽量把水果按“一份够一个人吃一天”的量（比如2个苹果/1串香蕉）用保鲜膜或包装盒预包装。这样顾客拿了就走，避免多次翻捡造成碰伤。
    -   **冷柜陈列**：如果门店有冷藏柜，所有易损耗的水果（草莓、葡萄、蓝莓）必须放在冷柜。不要常温摆放超过半天。

3.  **定价与促销：最后一道防线**
    -   **动态调价**：当天下午4点后，未卖完的非耐放水果（如圣女果、切好的果盒）立即打折（7折、5折），甚至买一送一，确保当日清空。
    -   **关联销售**：比如“买燕麦酸奶+送一根香蕉”、“买沙拉+搭配圣女果半价”。
    -   **“残次”处理**：有些水果只是皮外伤，可以削皮后切片卖“混合水果杯”，或者作为门店自制饮品的原料（如水果茶）。

**三、风险提醒：极端情况快速止损**

-   **如果某品类连续2天销量差**：立刻停止进货，现有库存全部特价清仓，哪怕亏本也要快。小面积门店最怕“烂在货架上没人看”。
-   **如果有单日未售完的香蕉**：香蕉皮变黑不代表肉坏了，可以立刻做“特价香蕉，当天必吃”，或者免费赠送给买够一定金额的顾客。

**总结口诀**：  
“少进快卖不断货，品项不杂宁精勿滥；  
勤翻常挑用包装，打折试吃管好坏货。”

只要坚持**“不囤货、不贪全、当日销”**，5-10平米内也能把水果做成稳定小利润点。
