针对茶饮店旺季菠萝用量激增的情况，与供应商约定临时补货时，建议按以下步骤操作，确保供应稳定且控制成本：

1. **立即沟通确认**：联系供应商说明“旺季需求突然增长”，明确需要补货的具体数量（建议根据近几日销量预估，如“每日需增加XX公斤”）、急需的到货期限（如“明天上午10点前”），并确认对方库存是否满足。

2. **协商价格锁定**：若临时补货量较大，可要求对方按原本的供货价格或略高于原价（幅度控制在5%以内），避免因“急单”被大幅提价。明确说“这次急单我承担部分运输成本，但希望单价不要超过XX元”。

3. **约定品质验收标准**：针对菠萝，明确“成熟度在7-8成，无腐烂、无机械损伤”，并要求提供检验或照片确认。可要求到货后现场抽检，不合格品按比例折价或退货。

4. **形成书面记录**：通过微信/短信发送确认文字，含：补货数量、单价、到货时间、验收标准、付款方式（如现结或月结）。回复“收到，确认以上条件”即可作为凭证。

5. **应急预案**：询问供应商“如果本次库存不足，是否能分两批次到货？比如先发50%，第二天再补剩余30%？” 并同步联系1-2家备用供应商（如批发市场或周边水果商），随时可调货。

6. **后续合作优化**：补货结束后，主动沟通“未来旺季预计持续X周，这期间能否每周固定增加XX公斤供应？是否需要预付部分定金？” 避免临时补货重复发生。

**示例话术**：  
“李老板，最近菠萝销量突然翻倍，现在库存只能撑到明天中午。我要紧急再补100公斤，还是按原价6元/斤，明天早上8点前能到货吗？如果不够，至少先到50公斤，剩下的后天补。请确认品质和到货时间，我马上安排付款。”

**注意事项**：  
- 如对方无法满足时效，立即联系本地水果批发市场（如郑州万邦、广州江南等）或社区菜商作为临时渠道，成本可能略高但能解燃眉之急。  
- 本周内跟踪菠萝用量，若持续高位，主动与供应商签订7-15天的临时增量协议，锁定供应与价格。
