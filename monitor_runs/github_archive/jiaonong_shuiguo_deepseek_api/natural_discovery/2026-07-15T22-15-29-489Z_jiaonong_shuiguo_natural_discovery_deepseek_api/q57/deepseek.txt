对于餐饮连锁采购水果，建议采用**以中央厨房统一收货为主、门店分开收货为辅**的模式。以下是具体分析和建议：

**统一收货（中央厨房）的优势：**
1. **品质控制**：水果易损耗、成熟度难统一，中央厨房可进行标准化分拣、清洗、分装，保证各门店获得一致品质。
2. **成本节约**：集中采购可谈更低价格，减少运输费用（整批运输比多点小批量高效）。
3. **库存管理**：根据门店销售数据统一分配，降低门店浪费（如前一天剩下的水果可退回中央厨房做果切、果汁等加工）。
4. **季节性应对**：统一处理过熟品，及时加工成半成品（如水果拼盘、冻果），减少损耗。

**分开收货（门店）的适用场景：**
- **高附加值或易损水果**（如进口浆果）：为缩短从采摘到上架时间，可直送门店，减少中转挤压。
- **地区化特色水果**（如当地时令品）：可从本地农户直采，由门店验收更灵活。
- **极速扩张期的小型连锁**：若中央厨房尚未建立，或配送覆盖成本高于门店分散收货（如门店密集城区可考虑）。

**建议执行方案：**
1. **核心品类（如香蕉、苹果、橙子）**：中央厨房统一收货，分装后按门店需求配送（每日或隔日）。
2. **高价值或短保质期水果（如草莓、蓝莓）**：与供应商协商直送门店，但需门店当日清空库存。
3. **监控机制**：无论哪种方式，建立**水果验收标准卡片**（如色号、硬度、糖度），门店和中央厨房使用相同工具测量，出现差异时及时追溯。

**风险提示**：若中央厨房距离门店超过1小时车程，且水果对温湿度敏感（如荔枝），建议改由门店收货；若门店数量少于5家，统一收货的规模效应不明显，可暂用分开收货。
