判断菠萝哪个规格更容易出货，主要取决于**下游客户类型**（批发市场、水果店、超市、电商/餐饮）和**当地消费习惯**。没有绝对的答案，但可以通过以下步骤和数据维度来快速判断：

### 1. 先问自己：你的主要客户是谁？
不同客户偏好不同：

- **水果店/社区生鲜店**：偏好 **单果重1.5-2.5斤** 的中等规格。太大（>3斤）难一次吃完，太小（<1斤）卖相不好。容易出货。
- **超市/高端精品店**：通常要求 **匀称、卖相好**，重量在 **2-2.5斤**，且包装精致（如切好半成品）。小商贩可能会挑。
- **食堂/餐饮/果汁店**：偏好 **大果（3斤以上）**，因为出肉率高、削皮人工成本相对固定，更划算。如果是做批发给餐饮，大规格易出货。
- **批发市场走量（二批/地摊）**：通常 **价格最敏感**，中低规格（1.5-2斤、有轻微瑕疵）性价比高，走量快。越小越不好卖，除非价格极低。

### 2. 实操数据判断法（最直接）
没有经验时，用 **试销+记录** 方法：

- **新品试销**：同时进一批3个规格（小1-1.5斤、中2-2.5斤、大3-4斤），放到批发档口或给几个核心客户。
- **看3个指标**：
  - **周转速度**：同一批进货，哪个规格最先被客户挑完？记录（例如：中果3天卖空，大果2天卖空）。
  - **客户问询频率**：哪个规格被问价、要求看货的次数最多？（例如，7个客户中有5个问“有没有2斤左右的？”）。
  - **退货/压货率**：哪个规格最后卖得慢、需要降价处理？（例如，小果最后烂了5斤，说明不好出）。

**简单记录表**（用笔在纸箱上标记）：

| 规格 | 到货日期 | 售罄日期 | 客户评价（好卖/压货） |
|------|----------|----------|----------------------|
| 小果 | 3.1      | 3.5      | 只卖了60%，剩40%降价 |
| 中果 | 3.1      | 3.3      | 2天内卖光，客户追问 |
| 大果 | 3.1      | 3.7      | 卖得慢，但餐饮客户要 |

### 3. 观察竞争档口
- **看同行进什么**：去当地最大批发市场看看，同行档口哪个规格堆得最多、补货最快，那个规格就是当下主流。
- **看爆款单品**：如果市场上“黄金菠萝”卖得好，一定是中等规格（约2斤）为主。

### 4. 特殊时期判断
- **节日/送礼季**：大果+靓装（如2个装礼盒）更好出。
- **天气热时（夏季）**：中果或小果更好，因为消费者想一次吃完、不浪费。
- **雨季/价格战**：小果（低价）可能暂时好出，但走量不赚钱。

### 最后建议（直接可执行）
- **新手批发**：**优先主推2-2.5斤（中等规格）**。这个规格兼容水果店、社区团购、散户，出货最均衡。大果备20%货源给餐饮，小果不超过10%。
- **已经压了某个规格**：立刻降价5%-10%促销，或者混拼（买大送小）清掉。
- **终极判断**：问3个老客户，“您一般喜欢多大的？”，大部分回答就是你的主力规格。
