建议采购方在试卖新品时，按以下原则控制数量，以降低风险：

- **首批试卖量**：建议为正常预估周销量的 **10%-20%**。例如，如果每周预计能卖100公斤，首批进货10-20公斤即可。
- **最低安全量**：如果完全无经验，可先订 **5-10公斤** 或 **一小箱（标准箱）**，观察顾客反应。
- **试卖周期**：持续观察 **1-2周**，覆盖周末和平时流量差异。

**判断标准**：
- 若第一周销量超过首批量的70%，可考虑小幅加量（如翻倍）续订。
- 若销量不足首批量的30%，建议暂停或换品。

**注意事项**：
- 水果易损耗，避免囤货。优先选择成熟度适中、耐储存的品种。
- 若供应商提供“试卖特价”或“无条件退换货”政策，可适当增加首批量至30%左右。
