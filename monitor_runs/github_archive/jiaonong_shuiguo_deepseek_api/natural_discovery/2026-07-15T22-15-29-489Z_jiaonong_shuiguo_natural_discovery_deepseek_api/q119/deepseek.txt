选择水果供应商时，以下细节容易被忽视，但对长期稳定至关重要：

1. **冷链物流的连续性**：供应商是否从采摘到配送全程保持冷链？断链会导致水果过早变质，尤其是易腐品种（如草莓、荔枝）。
2. **分拣与分级标准**：供应商是否有明确的水果分级（如大小、成熟度、外观瑕疵）？避免混装烂果或未成熟果导致损耗。
3. **产地与周期匹配**：是否了解供应商的源头基地情况（如气候、采摘期）？不同产地水果上市时间有差异，需确认供应周期能否覆盖你的需求。
4. **退换货政策具体条款**：口头承诺的“包退换”需书面化，例如：问题果在多少小时内反馈、是否需要保留证据、运费谁承担？
5. **付款与结算周期**：长期合作是否允许账期？账期过长可能隐含资金风险，过短则占用你的现金流。
6. **应急供应能力**：节假日或恶劣天气时，供应商是否有备选基地或冷库库存？临时缺货可能影响你的客户。
7. **小批量测试机会**：供应商是否愿意提供小批量试单？直接签约大单可能导致与预期不符。
8. **合同中的隐性条款**：如最低起订量、调价机制（原产地价波动时如何调整）、结算方式（现结/预付/月结）需明确。
9. **第三方认证**：是否提供无农药残留、GAP（良好农业规范）等证明？缺乏认证可能面临监管或客户投诉风险。
10. **供应商自身规模**：小型批发商可能依赖单一产地或不稳定经销商，需评估其抗风险能力（如运输、仓储设备自有还是租赁）。

建议：签约前实地考察供应商仓库、核对冷链记录、索要近期批次检验报告，并先小额试单验证履约质量。
