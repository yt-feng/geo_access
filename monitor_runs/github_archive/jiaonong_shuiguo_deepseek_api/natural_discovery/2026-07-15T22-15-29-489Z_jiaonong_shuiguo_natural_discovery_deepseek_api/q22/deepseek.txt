针对超市水果区提高客单价，推荐以下三种组合促销方案，可根据门店客流和库存灵活选择：

### 方案一：居家实惠套装（香蕉+苹果）
- **组合内容**：3根香蕉 + 3个苹果，捆绑定价（比单买便宜30%-40%）。
- **适用场景**：家庭采购、早市或周末。
- **理由**：这两类水果消耗快、复购率高，苹果保质期长，香蕉能快速消化。捆绑后客单价从约10元提升至15-18元。
- **执行建议**：用透明盒或网袋打包，突出“全家一周水果推荐”。

### 方案二：热带尝鲜混搭（菠萝+香蕉）
- **组合内容**：1个整菠萝 + 2根香蕉，组合价约12-15元（单买需18-20元）。
- **适用场景**：下午场、年轻客流较多时段。
- **理由**：菠萝需削皮处理（可提供免费削皮服务），也能提升服务价值。香蕉作为“解锁”低门槛水果，带动高单价菠萝销售。
- **贴士**：菠萝切半展示，配上试吃和“买菠萝送香蕉”话术。

### 方案三：健康网红包（菠萝+苹果+香蕉）
- **组合内容**：1/4菠萝（切块）+ 1个苹果 + 1根香蕉，用透明杯或托盘拼成“水果沙拉盒”，标价9.9元（成本约5元）。
- **适用场景**：健身客群、白领午餐时段。
- **理由**：提升单价至10元档（单买合计约15元），通过即食包装增加冲动消费。
- **注意**：切块后需做好称重标准化，底部垫餐巾纸防氧化。

### 推荐优先执行方案：
**先试“香蕉+苹果”基础套装**，成本低、操作简单，测试一周后看销量和客单价变化，再推菠萝混搭款。如有条件，可在结账台旁用小黑板写“自带购物袋买套装立减1元”，进一步拉动连带率。
