通常供应商在周末促销时可以给予两种主要支持：

1. **价格支持**：常见的形式包括批量采购折扣（如买满一定数量单价降低5%-15%）、短期降价（针对特价商品由供应商补贴差价）、试吃装免费提供（减少采购成本）等。这些需要提前一周与供应商沟通，并说明促销规模和预期销量。

2. **物料支持**：如免费提供促销海报、价签、展架、试吃台、冷柜贴纸等。部分供应商还会提供统一形象的宣传物料（如当季水果品牌宣传单张）。

关键建议：
- 优先联系长期合作、销量较大的供应商，他们对老客户支持力度更大。
- 明确告知促销时间（周末）、主题（如“周末鲜果特惠”）、预估销量，方便供应商评估投入回报。
- 如果供应商首次合作，可以先用小批量试销，并协商支持条件。

注意：不要虚构供应商的回应或承诺，建议你直接与1-2家主要供应商电话或微信沟通，用“周末促销计划”为切入点，询问他们能提供的具体支持种类和门槛。
