判断水果供应商是否靠谱，主要看**稳定性、诚信度、售后能力**。针对香蕉和菠萝这两种水果，建议从以下五个关键点考察：

1. **资质与经营时长**  
   - 要求查看营业执照、食品经营许可证（尤其是香蕉可能涉及催熟环节）。  
   - 优先选择经营3年以上、有固定档口或冷库的供应商，临时摊点风险高。

2. **果品质量与分级标准**  
   - 香蕉：观察整把果形是否均匀、无机械伤；成熟度（绿转黄程度）需能与你销售节奏匹配；要求提供“无催熟剂”证明或自然催熟方案。  
   - 菠萝：检查冠芽新鲜度、果眼无霉斑、按压有弹性（太软为过熟）；对“黑心”菠萝高发季节（雨季后）要求供应商签署质量担保。

3. **供应链能力**  
   - 香蕉和菠萝是**后熟型水果**，询问供应商是否有独立催熟库和控温运输车。  
   - 测试发货准时率：首次尝试小批量下单（如20箱香蕉+10箱菠萝），记录到货时间、破损率。正常情况下破损率应＜3%。

4. **价格与结算方式**  
   - 对比3家供应商的批发价：香蕉当前产地价+催熟成本约占总价60%，菠萝约50%。低于市场均价20%需警惕（可能是次果或虚报等级）。  
   - 拒绝“押金制”或预存大额货款，要求**周结或现结**，首次合作坚持货到付款。

5. **售后处理案例**  
   - 直接问：“如果到货菠萝切开有黑心，或香蕉到店两天就全熟，怎么处理？”  
   - 靠谱供应商会明确：24小时内反馈视频/照片可退款或补发；或承担损耗（如赔付总货款的5%-10%）。

**具体操作建议**：
- **先用小单验货**：下单时要求供应商随机在每箱/每把上贴标签，到货后抽查50%，检查标签是否对应（防调包）。  
- **建立黑名单**：记录每次交易中供应商是否主动承诺“三天内降价补差价”（香蕉价格波动快，这是判断诚信的关键）。  
- **长期签约条件**：稳定合作1个月后，再谈账期（如压一批货2万以内）或独家供应。
