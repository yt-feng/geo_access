判断苹果和橙子大小规格是否适合门店销售，可以从以下几个具体、可执行的角度来评估：

1. **查阅门店标准手册**：首先确认你所在超市是否有**公司统一制定的水果采购规格标准**（如单果重量、果径范围）。这是最直接的依据，没有手册可向采购部或店长索要。

2. **看果径（最大最直观的指标）**：
   - **苹果**：常规销售规格通常要求果径在 **75mm-85mm** 之间（大约相当于成年人拳头大小）。过大（>90mm）可能不好零售，过小（<70mm）可能口感差或被视为次果。
   - **橙子**：脐橙/甜橙类常规规格在 **70mm-80mm** 之间（女拳大小）。过小（<65mm）酸度高且果肉少，过大（>85mm）可能皮厚或风味变淡。
   - **工具**：携带便携式果径测量尺或卡尺，随机抽取上下层、不同位置的10-20个果子测量。

3. **称单果重（辅助判断）**：
   - 苹果：单果重在 **200g-250g** 左右最合适。过轻可能空心或发育不良，过重价格高消费者犹豫。
   - 橙子：单果重在 **180g-250g** 为佳（大果可到300g，但要结合销售价）。

4. **观察均匀度（稳定预期）**：要求一箱（或一批）中 **80%以上** 的果子大小误差控制在**±10%以内**。例如标称80mm的苹果，多数在72-88mm之间，而不是有的60mm有的100mm。不均匀会给顾客“选大小、挑剩”的感觉。

5. **现场测试售卖包装**：如果是**预包装**（网袋或礼盒），把尺寸规格放进去，看能否**正常封口、不漏果**；如果是**散称**，把果子倒进堆头中，看是否**摆放整齐、大小视觉协调**。歪斜、明显突出或凹陷就是不合格。

6. **类比竞品**：逛一下附近竞争对手的蔬果区，对照他们的苹果、橙子大小。如果你们的目标是平价竞争力，大小应**不低于同行同价位的平均水平**；如果定位高端，则可以考虑更大或更小的特色规格（如小糖心苹果）。

7. **快速抽检表（实操口诀）**：
   - 拿一个苹果/橙子放掌心，**大拇指能完全盖住果柄、另一侧刚好到中指指尖**（成人男手），≈75-80mm，合格。
   - 如果两个放一起，**一个明显大半个拳头的、一个小拇指长短差很多**，直接判定为“规格不统一”，建议拒收或降价。

**总结执行步骤**：
第一步：看标准 → 第二步：测果径（主）和单果重（辅） → 第三步：看均匀度（8成以上一致） → 第四步：试装/试摆 → 第五步：对照竞品。至少满足前三项才能放入门店销售。
