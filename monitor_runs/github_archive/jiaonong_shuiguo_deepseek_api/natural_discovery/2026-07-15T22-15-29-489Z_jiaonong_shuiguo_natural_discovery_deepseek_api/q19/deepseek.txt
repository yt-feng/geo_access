在超市挑选菠萝时，建议按以下优先级综合判断：

1. **甜度（最优先）**  
   直接决定口感。可通过闻基底部香气（有浓郁果香则更甜）、按压表皮（微软且弹性好为佳）、观察底部果眼颜色（金黄偏深通常更甜）等方式判断。若允许试吃或商家标注糖度，可优先选择。

2. **外观（第二优先级）**  
   查看表皮：选择叶片翠绿挺拔、果身完整无霉斑、无软烂或机械损伤的菠萝。但注意：过度金黄可能过熟，带青绿色但香气足也可接受。

3. **大小（第三优先级）**  
   在同等甜度下，稍大的菠萝果肉更厚实，可食率更高。但不必过分追求大果，因其可能纤维更粗或水分不足。

4. **切果损耗（最次要）**  
   若超市提供切果服务，可选择现场切开确认果肉无黑心、无褐变（褐斑可能由冻伤或腐烂引起）。但通常外观正常的菠萝损耗可控，无需为此牺牲前三个指标。

**总结**：先闻、看、摸确保甜度和无腐坏；再优先选外观佳、大小适中（约1-1.5千克）的菠萝；最后检查切面。若为带皮包装菠萝，则直接按前两步选择即可。
