对于小水果店来说，进货香蕉时，建议按以下优先级综合考虑：

1. **熟度（最关键）**  
   香蕉是呼吸跃变型水果，熟度直接影响货架期和销售速度。应选择**7-8成熟**的香蕉（果皮绿中带黄、果肉硬实），避免过熟（已出现黑斑或软烂）或过生（全绿、难催熟）。这样既能给顾客1-2天的自然熟化时间，又不会因过快腐烂造成损耗。

2. **坏果率（直接成本）**  
   检查整箱香蕉时，注意是否有明显霉变、水渍状斑点、机械伤或挤压腐烂。通常允许1-2个坏果，超过5%的坏果率建议拒收或要求折扣。坏果会加速其他香蕉腐烂，并影响顾客口碑。

3. **价格（辅助决策）**  
   在保证熟度适中、坏果率低的前提下，对比不同供应商价格。但不要因低价接受劣质熟度，因为香蕉损耗率高，低价但高损耗的香蕉实际成本更高。

4. **送货速度（灵活性）**  
   小店铺资金和仓储空间有限，不需要极速补货，但需确保供应商能在**24-48小时内**送达，且承诺次日可灵活调整订单。过于频繁的送货会增加配送成本，而太慢则可能断货。

**操作建议**：  
- 首次进货时，先少量试拿（例如1-2箱），核实供应商的信誉和实际品质。  
- 要求供应商提供带标注的熟度等级（如“7成熟”），并约定坏果率补偿机制（如：坏果1%以内正常，超过部分按斤扣减）。  
- 优先选择有稳定青蕉催熟能力的本地或邻近区域的批发商，以缩短运输时间和减少熟化不均。
