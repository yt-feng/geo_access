通常情况下，水果采购与供应商协商的锁价期限主要取决于水果品种（如应季/反季、耐储存性）、供应量稳定性以及双方的议价能力。常见的锁价期限如下：

1. **短期锁价（1-2周）**：适用于当季、产量波动大或易腐坏的水果（如草莓、荔枝、芒果）。供应商一般只愿意锁定短期价格，以降低自身风险。

2. **中期锁价（1-3个月）**：适用于耐储存、供应相对稳定的水果（如苹果、梨、柑橘）。如果采购量较大或签订长期合作合同，供应商可能同意锁定1-3个月。

3. **长期锁价（一个产季或半年以上）**：极少见，通常需要采购方承诺极大采购量（如覆盖供应商全年产量的30%-50%），且供应商本身有较强规模控制能力（如自有农场、冷链稳定）。此时锁价可能覆盖整个产季或半年。

**实际操作建议**：
- 优先锁定短期（1-2周）或中期（1个月），避免因市场剧烈波动（如极端天气、运输中断）导致供应商违约或不发货。
- 在签署锁价协议时，明确违约金、波动调整机制（如价格浮动超过一定范围可重新协商）以及最低/最高采购量。
- 针对易损耗水果，用“批次锁价”替代全期限锁价，例如每批到货前3天锁价。

具体锁价时效需根据您当前合作的水果品类和供应商资质灵活谈判，建议首次合作时以短期锁定+每批核价起手，建立信任后再延长锁价周期。
