针对夫妻店第一次试单，建议采用“小批量、多频次、观察销”的策略，避免压货风险。以下是具体操作步骤：

1. **选品试单量建议**  
   - **香蕉**：易熟，先试1箱（约13公斤）。  
   - **苹果**：耐储存，试1-2箱（常规箱约10公斤/箱），优先选红富士或冰糖心。  
   - **橙子**：选当季品种（如秋冬脐橙、夏季夏橙），试1箱（约15公斤）。  
   **总原则**：每品种首单控制在30-50斤以内，总进货额不超过500元，降低资金压力。

2. **选择试供渠道**  
   - **本地批发市场**：直接去市场看货、对比价格，找3-5家摊位对比品质（如香蕉的成熟度、苹果表皮的亮度）。  
   - **线上B2B平台**（如惠农网、一亩田）：要求供应商邮寄样品（自付邮费），优先找“支持试单”“一件代发”的商户。

3. **验货关键点**  
   - **香蕉**：表皮微黄带绿（7-8成熟），无磕碰黑斑，果柄结实。  
   - **苹果**：果柄鲜绿（新鲜），用手轻弹有清脆回弹感，无苦痘病（表面凹陷黑点）。  
   - **橙子**：手感沉（水分足），肚脐凸起越小越好（皮薄），用指甲轻刮外皮闻香味。

4. **试销期管理**  
   - **陈列**：别全摆出来，留半箱库补。香蕉挂起来（减少挤压发黑），苹果和橙子铺底层防碰撞。  
   - **定价**：比超市低5%-10%，但别亏本。首日可“新品尝鲜价”，带一句“不好吃包退”（建立信任）。  
   - **观察**：记录每天每种水果的损耗率（如香蕉3天内变黑过量则说明进货过熟）、回头客购买比例。

5. **风险兜底措施**  
   - 和供应商约定：首单如出现坏果（如苹果内部褐变、橙子霉变），拍照后按比例退款或补货（提前确认售后政策）。  
   - 香蕉若卖不动，第二天开始降价促销或做“香蕉奶昔”试饮引流。  

**忌**：别跨省找不熟的供应商、别一次订多箱要求包邮（损耗风险高）。先做好本地批发档口，验证1-2周后再考虑其他渠道。
