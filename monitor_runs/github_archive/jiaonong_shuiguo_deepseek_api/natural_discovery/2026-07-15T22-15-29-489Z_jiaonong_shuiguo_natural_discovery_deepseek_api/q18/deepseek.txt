挑选香蕉时，可以按以下方法避免买到太生或太熟：

1. **看颜色**：
   - 全青色：较生，可能需放2-4天才能吃。
   - 黄色为主略带青头：最佳购买时机，1-2天成熟。
   - 黄色均匀无斑点：已完全成熟，适合当天吃。
   - 出现大量黑斑或软烂：过熟，不推荐。

2. **摸硬度**：
   - 太硬且无弹性：过生。
   - 稍软有弹性（轻按能回弹）：适合购买。
   - 过软、凹陷或表皮易破：过熟。

3. **看棱角**：
   - 棱角分明：较生。
   - 棱角圆润饱满：成熟度适中。

4. **首选“全黄带青头”**：这个阶段的香蕉在常温下能放2-3天，且口感好，不会突然变烂。

5. **避免有碰伤或霉点**：这些部位会加速腐烂，导致整根香蕉早熟。

**温馨提示**：如果买回来发现太生，可以和苹果或梨装袋存放1-2天；如果太熟，建议放冰箱冷藏（皮会变黑但果肉不变），并尽快吃掉。
