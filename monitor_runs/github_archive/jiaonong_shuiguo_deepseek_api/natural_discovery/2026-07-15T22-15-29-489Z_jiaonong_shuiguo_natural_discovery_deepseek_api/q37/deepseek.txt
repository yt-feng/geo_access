商超会员日水果活动通常涉及较大客流与促销力度，供应商需提前配合以下准备工作，确保活动顺利执行：

### 1. **货源与库存准备**
   - **备货量预估**：根据历史销售数据、会员日活动规模（如会员数量、往期销量）及促销力度，提前1-2周与商超确认备货计划，建议按预估销量的1.2-1.5倍备货（考虑损耗和峰值需求）。
   - **品种精选**：选择当季爆款、高毛利、易损耗的水果（如车厘子、蓝莓、芒果、西瓜等）作为主推品，避免稀有或运输易损品类（如榴莲需控制数量）。可设置“引流品”（低价走量）和“利润品”（中等价格、高毛利）。
   - **供应稳定性**：与上游产地或批发商签订临时保供协议，预留应急调货渠道，防止断货。

### 2. **库存周转与冷链运输**
   - **分批到货**：建议分2-3批到货（活动前1天、活动当天上午、下午补货），避免一次性大量堆放导致损耗。
   - **冷链合规**：确认运输车辆全程温控（如常温水果控制在15-25°C，冷藏水果0-4°C），运输时间控制在2小时内，到货后立即入库或上架。
   - **库存预警**：协助商超建立实时库存监控，低于安全库存时及时补货（如每小时告知一次剩余量）。

### 3. **分拣与打包标准**
   - **品控分级**：按商超要求筛除次果、烂果，确保外观、大小、成熟度一致。对引流品（如特价橙子）可放宽外观要求，但必须保证口感无问题。
   - **预包装**：提前为常购套餐（如“会员价3斤装苹果”）准备包装袋/盒，标注促销价、会员标识、产地及保质期。散卖水果需备足分开称重的料筐。
   - **损耗品处理**：提前与商超协商，活动期间产生的损耗由谁承担（通常按比例分摊），并准备可二次加工的水果（如切盒、果汁）降低损失。

### 4. **促销物料与陈列协助**
   - **物料准备**：提供或协助制作吊旗、价签、促销台卡、地贴（如“会员专享价”）、试吃台用品（牙签、纸杯、湿巾）。重点标注“会员价”与“非会员价”差异（如普通价：8元/斤，会员价：6元/斤）。
   - **陈列规划**：配合商超安排黄金陈列位（收银台旁、主通道端头、入口处），将引流品放在最外层，利润品放在中间层。高单价水果可带果套/保护膜陈列。
   - **试吃服务**：安排1-2名供应商人员现场试吃引导（如切样、递送），每日至少补切3次，保持水果新鲜度，避免试吃台空置或堆满残渣。

### 5. **人员与资金配合**
   - **人员到位**：活动期间至少派驻1名熟悉业务的负责人（需携带工牌、健康证），协助理货、解答顾客疑问、处理客诉（如水果口感问题、缺斤少两）。高峰期（如超市开门后2小时、晚间6-8点）可增加临时工。
   - **资金垫付**：若涉及“买赠”“满减”等机制，需提前与商超明确费用分摊比例（如供应商承担50%折扣成本或赠送商品），预留活动期间的回款周期（通常活动后7-15天结算）。

### 6. **应急预案**
   - **缺货应对**：如果某款水果销量超预期断货，立即启动第二替代品（如用当季蜜瓜替代哈密瓜），并更新价签、告知商超促销员话术（如“会员特惠升级，现在买蜜瓜同样享8折”）。
   - **质量问题**：准备3-5%的备用损耗品（如200斤活动中预留10斤），若顾客投诉口感、发霉等问题，当场无条件换新或退款，避免差评扩散。

### 7. **数据与后续复盘**
   - **销售数据共享**：活动期间每小时/每两小时向商超同步实时销量、库存、客流数据，帮助商超调整促销力度（如下午降价5%出清滞销品）。
   - **活动后复盘**：活动结束后3天内，提交销售数据表（含各品种销售量、退货率、损耗率、补货次数）、顾客反馈（如“是否认为价格划算”“品类是否满意”），与商超共同分析下期会员日需改进的品类或价格策略。

**关键提醒**：提前3-5天与商超采购、店长、营销部开碰头会，确认所有细节（备货数量、陈列位置、费用分摊、试吃标准、应急流程），并以书面形式（邮件或合同附件）确认，避免现场扯皮。
