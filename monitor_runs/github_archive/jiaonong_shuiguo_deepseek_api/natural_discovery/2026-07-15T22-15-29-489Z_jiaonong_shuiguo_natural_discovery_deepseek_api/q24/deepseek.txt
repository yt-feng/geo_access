在超市采购中，为防范水果农残抽检风险，建议提前向供应商索取以下资料（优先要求提供原件或加盖公章的复印件）：

1. **第三方检测报告**  
   - 必须包含**农残项目**（如有机磷、拟除虫菊酯类等常见农残指标），优先选择有CMA、CNAS认证的检测机构出具的报告（有效期通常为1年，部分品类需随批次更新）。

2. **产地证明或供货产地备案信息**  
   - 明确水果来源地（种植基地名称、地址、规模等），例如《农产品产地证明》《食用农产品合格证》或当地农业部门备案材料。

3. **批次质量保证书/承诺函**  
   - 供应商书面承诺该批次水果符合国家食品安全标准（如《食品安全国家标准 食品中农药最大残留限量》GB 2763-2021），并注明批次号、生产日期、保质期。

4. **供应商资质文件**  
   - 营业执照（经营范围含水果销售）、食品经营许可证（若涉及预包装或分装）、农业合作社备案证明（若为直采种植户）。

5. **特定高风险品类附加材料**  
   - 对易超标品类（如草莓、葡萄、柑橘、苹果等），可要求供应商提供**种植环节用药记录**（如最近一次农药使用日期、名称、休药期执行情况）。

**操作建议**：  
- 要求供应商在供货前3-5天提供以上资料，并在采购合同/订单中明确“因农残不合格导致的退货、处罚及连带损失由供应商承担”。  
- 验收时随机抽样送第三方快速检测作为双重保障（如胶体金法测特定农残，30分钟内出结果）。  

若供应商无法提供核心材料（尤其检测报告），建议暂停采购并更换合规供应商。
