好的，针对水果批发商拿香蕉，判断熟度和坏果风险可以通过以下几步快速评估：

**1. 判断熟度（从青到熟大致分7级）**
- **全青硬邦邦**（1-2级）：适合长距离运输或冷库存储，但批发后需催熟。
- **微黄带青、棱角分明**（3-4级）：适合本地批发并计划2-3天内销售。
- **大面积黄色、棱角模糊、稍软**（5-6级）：已进入最佳食用期，需立即上市，有快速变色风险。
- **出现黑斑、按压过软**（7级）：熟度太高，基本没有批发价值，极易变质。

**实用方法**：随机抽3-5根，轻轻捏香蕉中部（最厚处）。  
- 硬得捏不动 → 按生蕉处理。  
- 有一点弹性的软 → 可放货架3天。  
- 一捏就凹且不回弹 → 今天必须卖掉或降价处理。

**2. 坏果风险判断（重点检查“硬熟伤”和“轴腐”）**
- **看轴（连接整串的梗）**：  
  轴部发黑、发霉或有水渍，基本整串内部已开始腐烂。
- **看香蕉尾部（尖端）**：  
  有黑色霉点或发软流水，这批货极易2天内全烂。
- **捏端头**：  
  轻轻捏香蕉两侧（靠近柄的粗端），如果软塌塌，说明内部已黑，属于“硬熟伤”——外表还黄，内部已烂。  
- **闻冷库或车厢气味**：  
  有霉味、酒精味（发酵味）直接不要，整批货已变质。

**3. 加速判断的小技巧**
- 取3根看起来最青、最硬的香蕉，用刀横切。  
  如果果肉中心发黑或变透明 → 经历过低温冻伤，以后会整批变黑、腐烂快。  
- 手指轻刮表皮，感觉滑爽有细纹 → 新鲜；感觉黏腻、有粘液 → 开始腐烂。

**4. 拿货行动指南**
- **优先选**：整串轴白净、香蕉尾青绿、表面无黑斑、硬度均匀的（摸10根以上）。  
- **重点避开**：一串中混有软和特别青的（暗示催熟不均匀，坏果风险高）。  
- **计算回购率**：如果一箱里坏果超过1-2根，整箱品质较差，不要拿大货。

简单总结：**捏轴、捏尾、闻味、抽切检查**，20秒内可判断80%坏果风险。
