处理生鲜电商水果坏果投诉的责任划分，主要依据**订单签收状态**、**是否经顾客确认收货**及**配送环节**来综合判断。以下是标准划分思路：

1. **顾客在快递员面前当场开箱验货**：如果发现坏果，**责任完全在发货方或快递方**。顾客可直接拒收或拍照并当场拒收，电商应全额退款或补发。

2. **顾客未当场验货、后续拆箱发现**：这是最常见争议场景。
   - **快递已签收**：顾客需在签收后**第一时间（通常建议2小时内）**内拍照并联系客服。如果水果明显**因包装不当、挤压、碰撞剧烈造成破损（如裂口、压烂、汁液渗出）**→ 责任主要在**电商（包装不合格或发货时已坏）**。
   - **快递未签收但已放至代收点/快递柜**：顾客取货后发现坏果，需**立即提供完整清晰的坏果照片及视频**（含快递面单、水果取出状态）。如果距签收时间短（如2小时内）且坏果明显属于**运输途中造成损伤（外皮完好但内部腐烂、碰伤斑点）**→ 电商应承担售后责任。

3. **明显属于品种或采摘不成熟导致的易烂**：例如顾客收到时水果已软烂、表皮起皱、无新鲜感，明显非运输造成。这属于**电商质量控制问题**，责任全在卖家。

4. **顾客存放不当**：如果水果送到时完好，顾客放冰箱后隔天发现坏果（如草莓、蓝莓易受潮腐烂），或未按说明书冷藏保存，则**责任在顾客**，电商可不予赔付。

**标准处理建议**：
- **电商侧**：应有**标准售后政策**（如“坏果包赔、按比例赔付”），建议**无需顾客寄回坏果**（生鲜不适合二次运输），直接根据坏果数量/重量比例退款或补发。
- **顾客侧**：**立即提供清晰坏果照片+快递面单+整箱外观图**。如超过7天不投诉且无证据，通常不再受理。

**如果双方争执不下**：
- 可基于**天猫/京东等平台生鲜售后标准**（坏果率≤5%通常免赔，5%-30%按比例赔，>30%全额退）来参考处理。
- 若走法律途径，需保留完整证据链（聊天记录、照片时间戳、物流轨迹、平台处理记录）。
