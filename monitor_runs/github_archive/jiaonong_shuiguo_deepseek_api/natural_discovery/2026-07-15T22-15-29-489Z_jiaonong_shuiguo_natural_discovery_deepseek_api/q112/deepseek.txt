在水果采购合同中，**通常建议设定坏果率超过 5% 至 10% 时启动赔付机制**。具体标准需结合以下因素确定：

1. **行业惯例**：大部分生鲜水果的合理损耗率在3%-5%以内，超过10%通常属于严重质量问题。
2. **水果品类**：
   - **易腐水果**（如草莓、蓝莓、荔枝）：建议5%左右。
   - **耐储存水果**（如苹果、橙子、柚子）：可放宽至8%-10%。
3. **包装与物流条件**：如使用冷链运输，坏果率标准可适当降低；简易包装则应预留更高容忍度。
4. **明确区分"坏果"定义**：应在合同中注明"坏果"包括**腐烂、霉变、严重机械损伤**等可量化的判断标准，避免争议。

**实操建议**：
- 合同条款中写明：例如 _"单批次坏果率超过5%时，超出部分由供货方按单价2倍赔付；超过15%时，整批次退货且供货方承担运费。"_
- 约定检查方式：到货后24-48小时内，双方共同随机抽样（建议开箱检查3%-10%的货物）。
- 提前预估赔付公式：赔付金额 = (实际坏果率 - 基准坏果率) × 批次总价 × 1.5~2倍。

建议与供应商协商一个双方都能接受的基准比例，并在合同书中明确写入。如需参考行业标准，可查阅《水果等级规格》等国内相关农业标准，但以合同约定为准。
