对于企业食堂水果采购，三者的重要性排序应为：**食品安全 > 配送准时 > 价格**。具体分析如下：

1. **食品安全（最优先）**  
   - 水果易腐烂、可能携带农药残留或微生物，若出现食源性疾病，会导致员工健康受损、企业声誉下降甚至法律风险。  
   - 必须要求供应商提供：农药残留检测报告、产地证明、保质期承诺，并建立到货验收机制（如检查外观、切开抽检）。

2. **配送准时（次优先）**  
   - 食堂需要按计划加工和供应，配送延误会导致断供或浪费（如提前到货可能变质）。  
   - 建议在合同中明确：配送时间窗口（如每日7:00-8:00）、迟到罚款条款、临时补货响应时间（如1小时内）。

3. **价格（可灵活调整）**  
   - 可接受合理溢价换取可靠的品质和履约。例如：比市场均价高10%-15%，但附带无条件退换货服务。  
   - 避免单一低价：低价供应商可能用不新鲜或劣质水果，反而增加挑拣损耗和投诉成本。

**执行建议**：  
- 选择通过ISO22000或HACCP认证的供应商。  
- 试合作期设定为1个月，重点评估退货率（<3%）、准时率（>95%）和客户服务响应速度。  
- 长期采购建议签订框架协议，明确年度供货质量标准、浮动价格机制（如按批发市场周均价±5%）及责任条款。
