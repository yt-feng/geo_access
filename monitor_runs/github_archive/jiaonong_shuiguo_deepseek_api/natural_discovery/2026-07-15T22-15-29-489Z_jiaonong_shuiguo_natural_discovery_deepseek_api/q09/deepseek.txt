在和水果供应商谈判时，为降低断货风险，建议提前明确以下关键问题：

1. **供货稳定性**：供应商的日均/月均产能是多少？极端天气或旺季时，最大供应量能否达到你的目标需求？是否有备用货源（如其他产区或冷库库存）？

2. **品种与品质保障**：他们能稳定提供哪些水果品种？不同批次的水果成熟度、甜度、外观（如无碰伤）有无标准？是否有分级（如一级果、特级果）？

3. **价格与调价机制**：旺季价格是否锁定？若批发价暴涨，是否会临时涨价？合同约定调价需提前几天通知？是否接受长期合作批量折扣？

4. **物流与时效**：配送范围、最低起送量是多少？旺季能否做到“次日达”或“当日达”？运输损耗（如破损、腐烂）由谁承担？是否提供冷链直达？

5. **应急方案**：如果出现断货（如产地歉收），供应商能提供同品类替代品吗？是否有合作果园或二线供应商作为补充资源？

6. **退货与赔偿**：到货后如发现质量问题（如霉烂、虫害），退货比例上限是多少？赔偿标准如何（例如按金额比例或免费补发）？发生断货影响销售时，是否有违约赔偿条款？

7. **账期与资金支持**：旺季期间是否允许延长结账周期（如30天→45天）？能否提供信用额度或分期支付？首次合作是否需要预付款？

8. **售后支持**：是否提供滞销品的换货/调货服务？客服响应时间（如24小时内处理投诉）？是否有驻场或线上对接人员协助促销活动？

建议在签订合同前，先进行小批量试单并签订短期协议，验证对方履约能力。同时，保持与2-3家供应商的合作关系，作为风险分散方案。
