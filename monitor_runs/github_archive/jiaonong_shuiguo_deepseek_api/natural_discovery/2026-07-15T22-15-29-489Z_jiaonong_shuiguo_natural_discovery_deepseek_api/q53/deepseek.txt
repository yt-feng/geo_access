针对餐饮店每天需要少量水果的需求，以下是省心供应商的选择建议：

1. **本地农贸市场/批发市场的固定摊位**  
   - 选择离店近的批发市场，固定一个或几个摊位长期合作，方便每日少量采购。  
   - 可要求摊主预留新鲜水果，次日直接取货或送货（少量可能需加小额配送费）。  
   - 优点：灵活性高、价格低、可实地验货。  

2. **社区生鲜电商平台（如美团优选、多多买菜、盒马集市）**  
   - 每日下单，次日上午到货，适合少量订单。  
   - 注意选择品质稳定的平台，早上收货时检查新鲜度。  
   - 缺点：品种可能有限，需提前对比平台评分。  

3. **本地水果零售店/夫妻店**  
   - 与周边水果店（尤其小规模店铺）达成每日小量供应协议，价格略高但配送及时。  
   - 优点：可协商每日固定品种和数量，店主熟悉本地需求。  

4. **生鲜供应链服务商（如美菜网、快驴进货）**  
   - 多见于大城市，支持每日小单，定期配送。  
   - 需对比起送门槛和配送费，适合需要固定品种和规格的店铺。  

**省心操作建议：**  
- 优先与2-3家供应商同时合作，避免依赖单一渠道。  
- 每日固定时间（如早8点）下单，要求固定包装和重量（如每袋1斤），减少现场挑选时间。  
- 建立简单的验收标准：无腐烂、颜色正常、无机械损伤，每日拍照记录问题批次。  
- 如对方支持批量按周结算，比每日单独结账更省心。  

**风险提示：**  
- 避免与无店面的流动摊贩长期合作，易出现品质不稳定的问题。  
- 少量订单建议先试单1-2周，观察到货时间和品质稳定性后再长期合作。
