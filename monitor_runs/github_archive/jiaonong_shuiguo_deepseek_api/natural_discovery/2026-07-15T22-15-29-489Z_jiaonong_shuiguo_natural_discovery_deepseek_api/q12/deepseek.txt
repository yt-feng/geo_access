采购菠萝时判断熟度主要依靠“看、按、闻”三步，以下为具体可执行的方法：

1. **看颜色**：成熟度适中的菠萝表皮呈**金黄色或橙黄色**，整体色泽均匀，底部微微泛黄。若表皮全绿则生涩，暗褐色或发黑则过熟或已损伤。
2. **按硬度**：用手指轻轻按压菠萝的果身，**果肉有弹性、微微下陷但不会过软**。如果按下去像石头一样硬，说明未熟；如果一按就塌陷或出水，说明过熟易腐坏。
3. **闻气味**：靠近菠萝底部（果柄处）闻一下，**有淡淡的清香**为好。无味可能较生，强烈甜腻味可能过度发酵或内部开始变质。
4. **看叶片**：新鲜菠萝顶部的叶片**绿而直立、易折断**。若叶片枯黄、软塌，可能存放过久。
5. **掂重量**：同等大小下，**越沉的菠萝水分越足**，相对新鲜，损耗风险小。

**补充建议**：
- 选择七到八成熟（果皮金黄占70%以上、果身稍硬）的菠萝，进店后常温放置1-2天即可完全成熟，避免运输中磕碰烂果。
- 进货时可要求供应商提供少量样品切开验证，减少整批不合格的风险。

希望这些标准能帮你降低损耗。
