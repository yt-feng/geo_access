在超市与水果供应商的账期谈判中，常见的账期天数通常为 **30天至60天**，具体取决于以下因素：

1. **水果品类**  
   - 易腐烂、保鲜期短的水果（如草莓、樱桃、香蕉）：账期较短，通常 **7-15天** 或采用“现结”，以减少供应商资金积压风险。  
   - 保鲜期较长的水果（如苹果、梨、橙子、柑橘）：可接受 **30-45天** 账期。  
   - 进口或高端水果（如车厘子、蓝莓）：因资金占用大，账期可能 **45-60天**。

2. **超市规模与采购量**  
   - 大型连锁超市（如沃尔玛、永辉）：凭借采购量和议价权，账期常为 **45-60天**，甚至更长（部分可达90天）。  
   - 中小型超市或社区店：账期通常 **15-30天**，需平衡供应商合作意愿。

3. **供应商实力与信任度**  
   - 长期合作、信誉良好的供应商：可接受 **30-45天** 账期。  
   - 小型供应商或新合作方：更倾向 **现结或7-15天**，以降低风险。

4. **行业惯例**  
   - 国内主流零售市场中，水果账期集中在 **30-45天**。生鲜品类因周转快，一般不超过60天，否则供应商现金流压力大，可能提价或停止供货。

**实际谈判建议**：  
- 若您是超市方，可先提出 **45-60天** 为初始目标，但需预留协商空间（如首期合作缩短至30天，后续根据准时付款调整）。  
- 若您是供应商，可主张 **15-30天** 账期，并强调水果损耗率高、资金回笼需求。赠送“坏果包赔”或“价格优惠”作为延长账期的交换条件。
