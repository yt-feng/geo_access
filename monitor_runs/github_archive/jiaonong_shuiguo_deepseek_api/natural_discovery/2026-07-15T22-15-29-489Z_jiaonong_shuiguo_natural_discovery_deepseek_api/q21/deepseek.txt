在超市采购水果时，供应商通常需要提供以下检测报告和进货证明以确保合规与质量安全：

1. **产地证明**（如产地证明书或产地证明单）：证明水果来源地合法，通常由当地农业部门或相关机构出具。
2. **水果检验报告**：包括农残检测报告（如GB/T 20769等标准，检测有机磷、氯菊酯等常见农药）、重金属检测报告（如铅、镉、砷等）和微生物检测（如细菌总数、大肠杆菌等），需由第三方检测机构出具。
3. **植物检疫证明**：针对进口水果，供应商必须提供《进境动植物检疫许可证》或《植物检疫证书》，证明水果无有害生物。
4. **食品安全管理体系认证**（如ISO 22000、HACCP、Green Food认证等）：证明生产过程符合卫生和质量标准。
5. **批次检测报告**：针对每批次水果，需提供当次采购的抽检报告，确保批次内产品质量稳定。
6. **采购合同或发票**：证明供应链可追溯性，包括供应商基本信息、产品批号、数量等。
7. **有机食品认证**（如适用）：如果水果标称为有机产品，需提供中国有机产品认证证书或国外等效认证。
8. **生产许可或经营许可证**：供应商需持有有效的食品生产许可证（SC编号）或食品经营许可证。
9. **保质期与储存证明**：如冷藏水果需提供冷链运输记录或温度管控证明。

建议超市采购时与供应商签订明确协议，要求对方提供上述文件的原件或复印件，并定期核查报告的有效性（如检测日期、检测机构资质等）。具体需求可根据当地监管政策或超市内部标准调整。
