以下是为小店主设计的香蕉和菠萝分级报价方案（简洁、可执行）：

**1. 香蕉分级报价（按外观与品质）**  
- **一级（精品果）**：  
  颜色：黄绿适中，无黑斑，柄粗壮；  
  尺寸：单根长度≥20cm，单把≥4根；  
  价格：**2.8元/斤**（示例，实际按市场浮动）。  
- **二级（常规果）**：  
  颜色：微黄有少量褐斑，柄较细；  
  尺寸：单根15-20cm，单把2-3根；  
  价格：**2.2元/斤**。  
- **三级（处理果/促销果）**：  
  颜色：全黄或偏青，有破损、黑斑超过20%；  
  尺寸：≤15cm或单果异常；  
  价格：**1.5元/斤**（建议打包卖或当天清货）。

**2. 菠萝分级报价（按成熟度与果形）**  
- **A级（特选/鲜食）**：  
  成熟度：八成黄，散发果香，按压微弹；  
  果形：长椭圆、无畸形，果眼浅；  
  价格：**4.5元/斤**（按个：约7-9元/个，3斤左右）。  
- **B级（标准/加工）**：  
  成熟度：青黄各半，需放2-3天食用；  
  果形：轻微歪斜，果眼较深；  
  价格：**3.2元/斤**（按个：5-6元/个）。  
- **C级（特价/处理）**：  
  成熟度：过熟或偏青，有轻微撞伤；  
  果形：过小（<2斤）或明显变形；  
  价格：**2元/斤**（建议切块试吃或榨汁促销）。

**3. 报价清单模板（给小店老板看的“一张纸”）**  
```
【今日水果批发报价】
香蕉：一级2.8元/斤（成色好，上架首选）
　　　二级2.2元/斤（性价比高，适合正常销售）
　　　三级1.5元/斤（处理快，适合搞特价）

菠萝：A级4.5元/斤（最佳口感，主推）
　　　B级3.2元/斤（放两天刚好熟，耐存放）
　　　C级2元/斤（有磕碰，建议当天切盒卖）
```
**附加建议**：  
- **香蕉**：强调“一级当天上市，三级需当日清货”，写在报价单顶部。  
- **菠萝**：说明“A级根部可切平，B级留一小段杆更耐放”，帮小店降低损耗。  
- **统一单位**：报价单必须注明“元/斤”或“元/个（标重）”，避免结算争议。
