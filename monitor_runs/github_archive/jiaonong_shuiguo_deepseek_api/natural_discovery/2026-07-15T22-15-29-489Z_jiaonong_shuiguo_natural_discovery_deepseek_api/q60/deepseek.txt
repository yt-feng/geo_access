针对茶饮店水果到货不合格的情况，建议从以下环节与供应商约定补货条款，以保障营业连续性：

1. **明确验收标准**：合同中写明水果品种、成熟度（如蕉类7-8成熟）、霉变/损伤比例（如不超过3%）、外观尺寸及重量误差范围（如≤5%）。到货后拍照留证，以电子数据形式同步反馈。

2. **快速补货时限**：约定不合格品需在2-4小时内补发（视距离远近），且补货需同步安排“加急单”到店或直接送达备选门店。在补货到达前，供应商需提供等量替代品（如冷冻果泥、预制果切）供当天使用。

3. **连带配送补偿**：写清补货运费由供应商承担，并附加门店因此产生的处理工时费（如垃圾处理、额外清洗耗时按1小时薪资折算）。

4. **触发备选方案**：对于频繁不合格的品类（如易损耗的草莓、蓝莓），建议签署“双供应商备援”条款——主供应商补货超过2次不合格，则自动启用备选供应商的原价订单，差价由主供应商承担。

5. **动态库存缓冲**：在每日开档前，保留至少2小杯份的替换果酱或浓缩果汁，避免因补货延迟影响现制饮品的出杯。同时，对第三方外卖平台设置“该饮品暂时售罄”的自动标记，防止差评。

6. **优先级配送通道**：要求供应商将补货订单标记为“茶饮VIP紧急单”，并接入其物流系统实时查看到达节点。若补货后再次不合格，供应商需即刻提供货损全赔+该单免运费。

7. **合同备份条款**：约定补货违约阶梯：首次延迟每小时按订单价的5%扣款；二次延迟直接折算为当季免服务费，并限定补货必须在双倍量基础上。

**应急实操**：若当天无法完成补货，可立即调整员工班次转做“物料盘点”，或临时上线“用当日合格水果制作的特调饮品（如单果限定）”并关闭其他饮品供应，减少顾客选择焦虑。同时要求供应商书面承诺24小时内用冷链完成补发，附赠样品水果以表歉意。
