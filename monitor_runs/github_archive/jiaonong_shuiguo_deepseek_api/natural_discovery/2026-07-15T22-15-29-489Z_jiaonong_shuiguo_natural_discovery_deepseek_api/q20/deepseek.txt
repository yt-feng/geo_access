选择社区超市的水果供应商时，建议您优先考虑以下几个条件：

1. **配送稳定性**：确认供应商能否保证每日准时送货，包括数量、品种和到货时间，建议查看过往客户的评价或试用期表现。
2. **水果质量与新鲜度**：要求供应商提供样品或实地考察其分拣、冷藏流程，关注是否有品控标准（如大小、甜度、无腐烂等）。
3. **品种丰富度与季节性**：是否提供当季主流水果（如苹果、香蕉、柑橘等）以及一定量的高端或小众品种（如蓝莓、牛油果等），以满足不同顾客需求。
4. **价格与账期**：对比3-5家供应商的批发价，注意是否包含运费，同时询问是否接受月结或账期（如30天），避免占用过多流动资金。
5. **应急处理能力**：明确退换货政策（如坏果包赔、缺货补发），以及节假日或天气异常时的替代方案。
6. **合规资质**：查验供应商的营业执照、食品经营许可证，以及水果的产地证明或检疫报告（必要时）。
7. **服务响应速度**：测试售后沟通效率，比如下单流程是否便捷、问题反馈后多久回复。

建议先通过本地农产品批发市场、生鲜电商平台或行业协会推荐寻找3-5家候选供应商，短期试合作后再决定长期伙伴。
