茶饮店在控制食品安全风险时，要求水果供应商提供以下证明文件，以确保原料合规、可追溯：

1. **食品经营许可证**（或食品生产许可证）  
   - 证明供应商具备合法经营或生产水果的资质（需在有效期内，且经营范围包含水果销售或鲜切水果加工）。

2. **农产品合格证明**（或产地证明/自检报告）  
   - 对未包装的果蔬，供应商应提供产地证明或相关合格文件，证明水果来自合法、安全的产地。

3. **农药残留检测报告**  
   - 第三方检测机构或供应商自检的最近批次的农残检验报告，重点关注常见高风险农药（如毒死蜱、克百威等），必须符合国家标准（GB 2763-2021）。

4. **新鲜度与感官检验记录**  
   - 供应商应提供当批次的水果新鲜度证明（如采收时间、运输温度记录），以及外观、成熟度、无腐烂霉变的感官检查记录。

5. **配送资质及温控记录**（如果涉及冷藏水果）  
   - 如草莓、荔枝、切块水果等需要冷链运输的，供应商需提供具有冷藏条件的配送车辆证明，并附有运输过程中的温度监控记录。

6. **检疫证明**（如适用进口水果）  
   - 若水果为进口产品，需提供海关出具的入境货物检验检疫证明，以及原产地证书、中文标签。

7. **供应商资质审核表**（企业自行管理用）  
   - 茶饮店可要求供应商填写并提供其生产加工环境、人员健康证（接触食品的员工）、食品安全管理制度等材料，作为准入审核依据。

**建议：**
- 要求供应商每批次提供上述关键文件（特别是农残报告和许可证），并保留存档至少2年。
- 定期（如每季度或每半年）对供应商进行现场审核或资质复核，确保其持续合规。
- 如果水果需要二次加工（如鲜切水果），还需提供供应商的加工场所的食品经营许可证（含制售项目）及相关标识标签。

以上文件应确保真实、清晰、在有效期内，并能在市场监管部门检查时随时调取。
