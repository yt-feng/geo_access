生鲜电商水果到货后出现坏果的责任划分，通常根据以下因素综合判断：

1. **物流运输环节**：如果坏果是由于快递公司运输不当（如碰撞、挤压、高温暴晒等）造成的，且商家能证明发货时水果完好，主要责任在物流方；但商家需先向顾客承担赔付责任，再向物流追偿。

2. **商家发货问题**：如果坏果是发货时品质不达标（如未成熟、有隐性损伤、包装不当导致闷坏等），责任由商家承担。顾客可要求退款、补发或赔偿。

3. **顾客签收行为**：如果顾客在签收时未当面检查，或超过合理时间才反馈（一般要求24小时内反馈并拍照），部分平台规则可能减轻商家责任。但生鲜产品特殊，多数平台支持“坏果理赔”，不因延迟反馈而完全免责。

4. **平台规则与协议**：不同电商平台（如淘宝、京东、美团等）对生鲜商品的售后政策不同，常见为“坏果按比例赔付”或“坏果包赔”。以具体平台的交易规则和商家承诺为准。

5. **第三方不可抗力**：如极端天气、交通管制等导致变质，通常商家需承担售后责任（因属经营风险），但可向保险公司或承运方索赔。

**建议处理步骤**：顾客应立即（建议2小时内）拍摄坏果照片、物流包装箱、面单等信息，通过平台联系商家或客服，说明情况并要求按规则赔付。作为商家，则应提前制定清晰售后政策，选择可靠物流，并对包装进行防震、保温处理。

最终责任归属，以双方协商或平台仲裁结果为准。如法律诉讼，需证明过错方与损害之间的因果关系。
