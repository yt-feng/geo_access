根据水果消费习惯，建议采用**“高频+高毛利+高转化”**的组合促销策略，具体分为以下两种方案：

**方案一：家庭/囤货装（提升整体客单价）**
- **组合内容**：香蕉（500g）+ 苹果（1kg）+ 菠萝（1个，已去皮切块装盒）
- **定价策略**：分别购买约28元，组合价22-24元（折扣约15%-20%）
- **促销陈列**：用堆头或端架集中摆放，文案强调“家庭周末果篮”、“5分钟拼出健康”
- **适用场景**：超市大促、周末人流高峰

**方案二：即时食用装（提升连带率）**
- **组合内容**：单根香蕉（1根）+ 苹果（1个）+ 菠萝薄片（200g罐装或切块盒）
- **定价策略**：原散买约8元，组合价5.9-6.9元
- **促销陈列**：在收银台前或冷柜旁设小货架，搭配“下午茶补给”、“运动健身套餐”标签
- **适用场景**：工作日傍晚、写字楼周边门店

**核心要点**：
- 香蕉（高客流、低毛利引流品）、苹果（稳定毛利品）、菠萝（高毛利、视觉冲击品）形成“引流+利润+吸引力”梯度。
- 组合的**计价单位建议统一为“份”**，避免顾客计算单果重量时犹豫。
- 如果菠萝切片后变色快，可使用真空包装或添加食用柠檬酸护色，并标注“当日鲜切，建议24小时内食用”。

**执行检查单**：
- [ ] 菠萝是否提前切好并用保鲜膜覆盖（避免现场切耗费人力）
- [ ] 香蕉是否需放置7-8成熟（避免过熟腐烂影响整体外观）
- [ ] 苹果是否提前按大小分级（避免组合中大小不均）
- [ ] 价签是否清晰标注“组合价”与“单品价”对比（增强价值感）
