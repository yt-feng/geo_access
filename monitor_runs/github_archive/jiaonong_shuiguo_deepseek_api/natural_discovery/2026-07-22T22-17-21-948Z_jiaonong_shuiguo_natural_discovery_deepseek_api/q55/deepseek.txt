选择水果拼盘的规格时，建议按以下方式考虑：

1. **苹果**：选择**中等大小**（约150-200克/个），单价适中且出肉率高。小果剥皮后损耗大，大果单价可能更贵且不易摆盘。

2. **橙子**：选**大果**（约200-250克/个），皮厚但果肉利用率高，切瓣后外形美观。小橙子皮薄但果肉少，且去皮费工。

3. **菠萝**：优先选**整只**（约1.5-2公斤），比切好的菠萝块便宜30%-50%。注意选成熟度高的，避免硬心造成浪费。

**成本参考**（假设按市价估算）：
- 苹果：3-5元/斤（中等果约0.6-1元/个）
- 橙子：4-6元/斤（大果约1.2-1.8元/个）
- 菠萝：2-4元/只（整只约3-6元，处理后可出果肉800-1200克）

**操作建议**：
- 批量采购时，直接联系批发商，要求按上述规格提供（例如苹果60-70mm直径）。
- 避免购买预处理切好的水果，通常价格翻倍且损耗不透明。
- 如果餐厅客流量不稳定，建议苹果、橙子按箱采购（如苹果18斤/箱），菠萝按周进货，存放不超过3天。

最终选择时，结合自家菜单的摆盘尺寸和顾客偏好微调即可。
