针对长期合作的水果供应商，建议优先谈**账期**。原因如下：

1. **现金流影响最大**：账期直接决定你的资金周转效率。水果行业利润率薄、损耗高，延长账期能显著降低你的短期资金压力，避免因应收回款困难导致经营风险。  
2. **谈判空间相对较大**：供应商通常愿意为长期稳定客户提供更灵活的账期（如从月结延至45天或60天），而配送和售后更多是标准化服务，调整空间较小。  
3. **可博弈的长期利益**：账期条款一旦确定，对双方合作模式影响最持久。售后问题通常可通过合同条款补救，配送频次、时间等也可以通过下订单阶段协调。

**操作建议**：  
- 先确认供应商的资金实力和行业惯例，试探其可接受的账期上限。  
- 同时可附带约定：若账期延长，你需承诺月度最低采购量或提前支付部分预付款（比如首单预付30%）。  
- 配送和售后作为辅助条款，可在账期敲定后再提出调整要求，通常不会成为谈判破裂的障碍。

如果供应商账期本来就极优（比如月结），再重点谈**配送时效**（降低损耗）或**售后条款**（退换货比例）。但一般情况下，账期是首要博弈点。
