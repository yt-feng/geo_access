与供应商沟通临时补货时，建议按以下步骤操作，确保供应稳定、价格合理：

1. **立即确认库存与需求**  
   先统计门店未来3-7天的菠萝实际需求量（可根据历史销量、天气、促销活动预估），明确需要追加的数量、规格（如去皮/带皮、成熟度）和到货时间。

2. **联系供应商备用渠道**  
   优先联系原有供应商，说明“旺季突发需求”，询问其是否有额外库存或调配能力。若无法满足，提前准备1-2家备用供应商联系方式（如批发市场、产地直供）。

3. **约定补货条件**  
   重点确认以下条款并书面记录（微信/邮件/书面协议）：
   - **价格**：明确临时补货是否按原合同价，或允许一定溢价（如当前市价+5%）。要求供应商提供含税、含运费的最终价。
   - **交期**：约定具体到货时间（如“下单后48小时内送达”），并设定延迟补偿（如“延迟1天，单价降5%”）。
   - **质量**：要求供应商提供近期的菠萝样品或质检报告（如糖度、无腐烂），并约定“到货抽检不合格可拒收”。
   - **结算**：临时补货款建议“货到付款”或“周结”，避免预付定金风险。

4. **设定反馈机制**  
   明确首次临时补货为“试单”，要求供应商提供到货后客户（门店）的反馈通道（如24小时内可退换）。若合作顺利，可协商转为长期备货协议（如“旺季每周固定多备50件”）。

5. **备用方案**  
   若供应商无法满足，考虑：  
   - 从附近批发市场当日现采（需提前联系档口预留）。  
   - 与本地其他茶饮店联合拼单，分摊运输成本。  
   - 临时调整菜单（如用罐头菠萝或当季其他水果替代部分用量）。

**话术示例**：  
“李总，我们菠萝用量突然增加30%，需要这周五前加送200斤去皮菠萝。能否按原价再优惠5%？如果能今天下午发货，我们后续旺季订单可以优先考虑您。如果量太多吃不消，可否介绍货源稳定的同行？”

**风险提示**：  
- 临时补货品质易波动，到货开箱时务必抽检（切开观察果肉色泽、气味）。  
- 若对方要求大额定金，建议缩短账期或采用“对公转账+发票”方式，避免纠纷。
