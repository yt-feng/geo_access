在售卖菠萝切块时，**保质期、包装和配送频次都很重要，但关注优先级建议如下**：

1. **优先关注保质期（安全性）**  
   - 切块菠萝因接触空气、刀具、人手，细菌滋生快，常温下保质期通常不超过4-6小时。  
   - 冷藏条件下（0-4℃）一般可保存24-48小时，需标注具体切割时间、保质截止时间。  
   - 若含糖度高（如成熟菠萝），酸度下降后易发酵变质，更需严格管控时效。  

2. **重点优化包装（防止氧化与污染）**  
   - 使用食品级保鲜盒/密封袋，隔绝空气、减少水分流失和微生物污染。  
   - 标签需含：切割日期、保质时间、储存条件、建议“24小时内食用”。  
   - 避免直接接触金属容器（加速氧化变黑），建议底部加吸水垫片防止汁液浸泡软化。  

3. **调整配送频次（减少损耗）**  
   - **高频低量配送**：每天至少配送1-2次（如早上、下午各一次），根据门店销量调整批次。  
   - 配送时使用冷链车/保温箱（保持2-6℃），避免常温运输超过1小时。  
   - 门店可根据前一日销售数据，选择“预售+现切”模式，避免库存积压。  

**执行建议**：  
- 在保质期管理上，对店员进行“先入先出（FIFO）”培训，每2小时检查品质（颜色、气味、汁液是否浑浊）。  
- 遇到未售出的临期切块，可在保质期前1小时降价或转为试吃品。  
- 夏季/高温天气时，保质期缩减30%，配送频次需增加至每2小时一次。
