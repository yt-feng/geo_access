判断茶饮连锁供应商能否稳定按规格送货，建议从以下四个关键环节进行考察：

1. **明确书面规格与验收标准**  
   先与供应商共同签署详细的规格标准书，包括菠萝的成熟度（如7-8分熟）、果径（如12-15cm）、糖度（如12-15Brix）、外观（无霉变、无机械伤）以及包装形式（如单个重量、箱装数量）。标准要可量化、可测量。

2. **实地评估供应商的采收与品控能力**  
   参观供应商的种植基地、采后处理及仓储环节，重点看：
   - 是否按成熟度分级采摘（如色卡对照）；
   - 是否有规范的清洗、分选、冷藏设备；
   - 是否建立批次溯源系统（如每个批次可追溯到产地、采收日期）。

3. **要求提供历史履约记录与抽检数据**  
   索要该供应商过去3-6个月向其他客户送货的规格稳定记录（如每批次抽检合格率、退换货比例）。并要求其提供近期3批次的第三方或内部抽检报告，重点核对规格偏差率（例如果径误差≤1cm的比例是否达标）。

4. **设置小批量试单与履约测试**  
   先下3-5批小额订单（如每次100-200箱），要求供应商按正式标准交货。到货后按以下方法快速评估：
   - 随机抽检20%的包装箱，逐箱称重、测果径、看外观；
   - 检查到货温度（要求冷链运输时果心温度≤10℃）；
   - 记录每次的准时率（如24小时内准时送达比例）和规格偏差率；
   - 若连续2批不合格或准时率低于90%，则淘汰该供应商。

**可执行行动建议：**  
- 要求供应商提供**书面承诺**，明确规格不合格时的退换货条款（如3天内无条件换货）和违约金比例（如单批次不合格超5%则扣除当批次货款的10%）。  
- 在合同中加入**定期复查条款**，例如每季度随机抽检一次，若持续达标则给予更大订单份额。

如果供应商无法提供上述任何一项数据或证明，即视为履约风险较高，建议优先选择有明确质量体系和历史记录的合作方。
