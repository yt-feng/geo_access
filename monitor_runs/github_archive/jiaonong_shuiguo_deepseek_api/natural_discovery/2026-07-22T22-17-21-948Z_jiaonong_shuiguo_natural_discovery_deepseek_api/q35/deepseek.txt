针对连锁超市降低门店坏果率，合同条款建议从**验收标准、责任归属、赔付计算方式、追责时效**四个核心维度设计。以下是可操作的合同条款建议：

---

### 1. 明确“坏果”界定与验收标准  
- **定义条款**：约定“坏果”包括腐烂、明显霉变、虫蛀、无法食用（如严重冻伤/压伤），并附《坏品率图示对照表》作为合同附件。  
- **验收时间**：到店后**2小时内**完成感官抽检（抽样比例不低于到货量的10%），超时视为默认合格。  
- **不合格判定**：抽样坏果率超**3%**（具体比例可根据品类调整），则判定该批次不合格。

### 2. 责任归属与赔付触发条件  
- **签收前**：运输途中产生的坏果（如冷链断链、机械损伤）**全数由供应商承担**，超市可当场拒收。  
- **签收后**：超市需按供应商指定的存储条件（温度、湿度、分类存放）执行，若因超市违规操作导致坏果，供应商免责。  
- **自然损耗**：合同约定一个合理损耗率（如香蕉5%、草莓3%），超出部分按以下公式计算赔偿：  
  **赔付金额** = （实际坏果量 - 合理损耗量）× 该批次单品**进货价** × **1.2~1.5倍**（覆盖超市人工处理成本）

### 3. 赔付执行与争议解决  
- **阶梯赔付**：  
  - 月度坏果率≤5%：不触发额外赔付，仅扣除该批次坏果对应货款  
  - 月度坏果率5%~10%：按超出部分1.5倍进货价赔偿，并书面警告  
  - 月度坏果率>10%：超市有权单方解除合同，且供应商需赔偿当月该品类总销售额的5%~10%作为违约金  
- **结算方式**：赔偿金额直接在下一付款周期货款中扣除，或供应商48小时内现金赔付（可二选一）。  
- **争议解纷**：若坏果率争议，双方可委托第三方质检机构（如SGS、华测）检验，费用由败诉方承担。

### 4. 证据留存与追责时效  
- **留痕要求**：超市验收时需全程录像（覆盖外包装、拆箱、表面抽检），并保存系统后台（WMS）的入库时间记录。  
- **追责期限**：超市应在发现坏果后**48小时内**通过邮件或系统上传《质量问题报告》及录像片段，超时视为放弃追索权。  
- **例外条款**：若超市长期（连续3个月）反馈同一问题但供应商未整改，视为供应商根本违约，超市可提前解约。

---

### 示例条款（可直接加入合同）：

**第X条 坏果率考核与赔偿**  
1. 卖方承诺本批次产品的自然运输损耗率不超过___%（具体见附件二单品损耗表）。  
2. 买方有权在到货2小时内进行抽样检验，若抽样坏果率超过3%，则判定该批次不合格，卖方需在24小时内补发等量合格产品或按不合格数量×进货价×1.5倍赔付。  
3. 月度结算时，若卖方当月供应给买方的同一品类坏果率超过5%，超出部分按进货价×2倍赔偿；若超过10%，买方有权无责解除该品类采购合同，且卖方需赔偿买方当月该品类销售总额的8%。  
4. 所有质检结果以买方留存录像、入库系统记录及双方确认的《质检报告》为准，卖方有异议应在3个工作日内书面提出，否则视为认可。

---

### 补充建议  
- **水果品类分设条款**：高损果（如荔枝、杨梅）单独约定更高赔付倍数和更低可接受坏果率。  
- **连带供应商责任**：若坏果源于供应商上游种植/加工环节，可要求供应商提供其与农户的供货协议，确保追溯赔偿链条清晰。  
- **动态调整机制**：每季度复盘坏果率数据，若连续三个季度改善显著（低于2%），可适当降低赔付倍数，作为供应商激励。
