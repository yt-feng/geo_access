针对批发商在合同中规避坏果损耗风险，建议在合同中明确以下条款：

1. **定义“坏果”标准**：写明坏果的认定依据，如腐烂、霉变、虫蛀、明显机械伤等，并约定参照的国家或行业质量标准（如GB/T 10650-2008等）。

2. **约定到货验收期**：写明收货后**X小时内**（如24小时）为验收期，收货方需在此期限内完成开箱检查并书面提出异议，逾期视为验收合格。

3. **明确责任划分**：
   - **运输途中的损耗**：由**承运方**负责。建议在合同里单独列明“交货条件”（如FOB或CIF），明确风险转移节点（如货物交付承运人后风险转移给买方）。
   - **到货后的自然损耗**：约定合理损耗率（如水果可约定3%-5%的自然失水、轻微碰伤等不计入赔偿）。
   - **超出合理范围的坏果**：写清“若坏果率超过X%（如5%），超出部分由**卖方**按合同单价赔偿或退还相应货款”。

4. **追加违约条款**：例如“若坏果率超过20%，买方有权拒绝收货或要求卖方承担返运、销毁及损失”。

5. **拍照/视频取证条款**：要求收货方在发现坏果时，立即拍照或录制开箱视频（含时间水印），并同步通知卖方确认，作为索赔依据。

**注意事项**：
- 避免笼统写“坏果”，必须用量化指标（%或具体数量）。
- 如果水果易腐，建议明确“卖方需使用符合保鲜要求的包装和运输方式”，并约定因包装不当导致的坏果由卖方承担全责。
- 争议解决条款建议约定快速检验机构（如当地第三方商检机构）。

请根据具体水果品种（如易腐程度、季节温度）和交易习惯，对上述条款进行微调。
