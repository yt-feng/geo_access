在水果采购合同中，**建议将坏果率的赔付标准设定在3%-5%以上**较为常见，具体比例可根据水果品类、运输周期、保存条件等因素协商确定。以下是操作建议：

1. **明确赔付触发阈值**：通常约定坏果率超过**3%**（对易腐类如草莓、荔枝）至**5%**（对较耐贮存类如苹果、柑橘）时启动赔付。若双方未约定，可参照行业惯例或《民法典》关于违约责任的条款（第577条）主张权利。

2. **约定赔偿计算方式**：常见方案包括：
   - **按比例折价**：超出部分的水果按采购价的50%-100%扣减（例如合同总价10万元，若坏果率8%，超出3%部分扣款=超出数量×单价）。
   - **直接替换或退款**：由供应商在约定时间内补发或退款对应金额。

3. **保留关键证据**：
   - 到货时进行**双方共同验收拍照录像**，并在签收单上注明“已开箱查验坏果率”。
   - 若无法当场验收，应在**24小时内**通知对方并留存抽样样品。

4. **补充条款建议**（若合同未涉及）：
   - “因运输或包装问题导致的坏果，责任由供应商承担。”
   - “分批到货时，每次到货分别计算坏果率，累计超出部分合并计算赔付。”

如果合同已签订但未约定，可尝试依据**《农产品质量安全法》第48条**或**《合同法》第111条**（质量不符合约定）请求赔偿，但需提供第三方检测报告（如当地检验检疫中心）。建议优先协商补充协议，避免陷入举证困难。
