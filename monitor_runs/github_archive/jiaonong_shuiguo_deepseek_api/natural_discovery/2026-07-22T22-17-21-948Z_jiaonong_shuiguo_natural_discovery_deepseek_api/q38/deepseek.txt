大型超市的水果供应商需具备以下核心的食品安全和物流能力：

**一、食品安全能力**

1.  **合规与资质：** 持有有效的《食品经营许可证》或相关生产许可；严格遵循《食品安全法》及超市方供应商准入标准。
2.  **源头控制与追溯：** 建立可追溯的供应链系统（如使用二维码或条码），能追溯到具体产地、批次、采摘日期；对自有或合作种植基地进行农残、重金属、微生物等项目的定期第三方检测。
3.  **标准化操作：** 实施良好农业规范（GAP）或ISO 22000/FSSC 22000等食品安全管理体系；建立采摘、分级、包装、存储的标准化流程。
4.  **冷链控制：** 对易腐水果全程冷链监控（采摘后预冷-冷藏运输-超市冷库交接）；配备温度记录仪，确保物流与储存温度符合水果特性（如草莓在0-4℃，香蕉在13-15℃）。
5.  **仓储与卫生：** 仓库保持干燥、通风、防虫鼠设施完备；不同水果分区存放（避免乙烯敏感型水果与产生乙烯水果混放）；定期消毒，包装材料符合食品级要求。

**二、物流与配送能力**

1.  **专用冷链运输：** 自有或长期合作的冷藏/冷冻车队（配备温控器、GPS实时追踪），运输箱体需定期清洁、消毒，确保配送时效（如每日或隔日送达）。
2.  **快速响应与补货：** 支持紧急补货（如促销或销售高峰），能在24-48小时内调配产地资源；具备稳定的产地或批发市场进货渠道。
3.  **精确分拣与包装：** 根据超市订单进行分级（如一级果、二级果）、称重、预包装（符合超市上架规格）；使用透气保鲜膜或气调包装延长货架期。
4.  **订单与信息系统：** 对接超市的订单系统（EOS或ERP），实现扫码出入库；能提供实时库存、到货预测、生鲜损耗分析等数据。
5.  **逆向物流能力：** 建立退换货机制（如临期、损伤水果的回收与处理），与超市共同承担损耗风险。

**三、其他关键要求**

-   **季节性管理：** 能应对不同季节水果（如荔枝、西瓜）的高温易腐特性，调整冷链强度。
-   **应急处理：** 制定食品安全事故预案（如召回流程、消费者投诉响应）。

建议供应商在合作前提供：**第三方检测报告、冷链运输记录、HACCP计划**，并接受超市方的定期审核与抽检。
