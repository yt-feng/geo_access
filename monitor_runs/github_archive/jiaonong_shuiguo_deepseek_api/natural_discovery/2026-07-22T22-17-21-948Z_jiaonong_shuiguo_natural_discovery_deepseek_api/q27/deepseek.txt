判断苹果和橙子大小规格是否适合门店销售，主要依据**目标客群定位**、**陈列需求**以及**行业通用标准**。以下是具体的操作建议：

**1. 明确行业常见分级标准（可直接用于核对）**

-   **苹果**：通常按直径和数量/箱划分。
    -   特级/大果：直径≥80mm
    -   一级/中果：75mm-80mm
    -   二级/小果：70mm-75mm（部分小超市或散货用）
-   **橙子**：
    -   大果：80mm-85mm（精品超市或礼品场景）
    -   中果：75mm-80mm（最常用，性价比高，家庭销售好）
    -   小果：65mm-75mm（适合低价促销或果汁用）

**2. 根据门店类型选择对应规格（可执行判断）**

-   **社区生鲜店/便利超市**：优先选中果（苹果75-80mm，橙子75mm左右）。这种尺寸顾客单次消费无压力，且易堆放。
-   **精品超市/高端会员店**：选大果（苹果80-90mm，橙子80mm以上）。尺寸大观感好，单价高，顾客对价格不敏感。
-   **折扣店/特卖场**：可接受小果或混果。控制成本，用低价吸引客流，但需保证整齐度，避免大小相差太大。

**3. 现场“3秒目测法”**

-   **掌心对比法**：握住一颗水果。苹果若略大于成人的正常拳头（约70mm），属于标准大小；如果小半个指节，偏小；多一个指节，偏大。
-   **果径圈/卡尺（带一把）**：采购时，随机从箱子四个角和中间取10颗，用卡尺量直径，看90%以上是否落在目标范围内。

**4. 关键避坑点**

-   **均匀性比单颗大小更重要**：同一批次内，最大果与最小果直径差异不应超过10mm。否则陈列出来非常难看，顾客会挑挑拣拣，造成损耗。
-   **熟度与色泽**：单纯看大小不准确。苹果要检查蒂部是否发软（开始变质），橙子要捏表皮弹性。大小合适但果体塌软或颜色发青的坚决不拿。
-   **跟供应商约定验收标准**：在采购单上书面注明“要求XXmm以上占比95%以上”，避免货到后才发现不符。

**总结建议**：起步阶段，直接选**75-80mm的苹果**和**75mm左右的橙子**，最容易卖且损耗低。如果条件允许，给苹果配70mm、80mm、90mm三个规格的套筒，橙子配65mm、75mm、85mm三个规格，采购时随机套一下即可快速判断。
