判断菠萝熟度，建议重点关注**外观**和**手感**，结合**香气**，避免依赖单一片面指标。采购时可参考以下方法：

1. **看颜色**：成熟的菠萝外皮呈**橙黄色**或**金黄色**，颜色越深通常越甜。完全青绿的菠萝未熟，口感酸涩；整体发黑或软烂则过熟。
2. **按压果身**：用手指轻轻按压菠萝的**果眼（凸起的小格）**或两侧。如果**稍能按动、有弹性**（类似按压西瓜熟度），说明成熟度适中。如果硬得像石头，尚不熟；按压凹陷且不回弹，则已过熟，易损耗。
3. **闻气味**：成熟菠萝在果柄处或底部会散发**浓郁清甜香气**。无气味代表未熟，有酒味或酸腐味代表过熟或内部变质。
4. **看果眼大小和形状**：成熟菠萝的果眼（叶片基部的凹陷处）通常**更饱满、间距更大**，果眼越宽往往更甜。果眼密集且扁平的多偏生。

**采购策略建议：**
- 根据销售计划决定成熟度搭配。计划**2-3天内销售完**，选**八成熟**（黄中带绿、有弹性、微香）；如果**当天或次日能卖完**，可进**九成熟**（全黄、香气浓）。
- 避免采购**全绿或按压坚硬**的菠萝，这类果放熟时损耗高（易青皮不熟），且费时间催熟。
- 也需注意**果型匀称**，歪斜、有霉斑、底部发黑的菠萝内部可能已坏。
- 门店可设置“今日切好”处理过熟菠萝，减少整果损耗，标价让利销售。

如果采购时不确定，可切开一个样品检查：果肉金黄、纤维细、芯小且软，说明熟度理想；果肉发白或发黑则要回避同等批次。
