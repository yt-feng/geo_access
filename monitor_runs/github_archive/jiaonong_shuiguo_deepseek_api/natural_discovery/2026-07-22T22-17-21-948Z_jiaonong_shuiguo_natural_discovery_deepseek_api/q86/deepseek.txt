生鲜平台做一件代发水果，供应商需具备以下核心仓配能力：

1. **冷链仓储能力**  
   - 配备温控仓库（冷藏0-4℃、冷冻-18℃及以下，视水果种类而定），确保水果入库后迅速预冷，维持新鲜度。  
   - 库房需分区管理，防止不同水果串味或交叉污染（如苹果与香蕉分开存放）。

2. **分级分拣与包装能力**  
   - 具备自动或人工分拣线，按规格、成熟度、品相分级，剔除坏果、烂果。  
   - 采用防震、透气包装（如泡沫网套、珍珠棉、纸箱+气柱袋），减少运输损耗，并提供定制化包装（如礼盒装、试用装）。

3. **快速分拣与发货效率**  
   - 支持订单实时接入（API对接），能在订单下达后2-4小时内完成拣货、包装、出库。  
   - 具备流水线作业和标准化操作流程（SOP），高峰期（如节日、促销）能临时增加人力或自动化设备。

4. **运输与配送网络**  
   - 合作或自建冷链车队，覆盖“最后一公里”配送区域（如社区、驿站、快递柜）。  
   - 能实现产地直发或次日达服务（如顺丰冷运、京东冷链），并对温度和物流轨迹实时监控。

5. **库存与品质管理能力**  
   - 实时更新库存系统，支持多仓、多SKU管理，避免超卖。  
   - 建立抽检制度（如定期检测农残、糖度），并配备快速退货处理机制（如坏果包赔、拍照理赔直接退款）。

6. **应急与售后处理能力**  
   - 有备用发货方案（如预案换仓、提前备货），应对恶劣天气或物流堵塞。  
   - 具备48小时内的售后响应团队，能提供补发、退款或等额代金券等补偿方案。

7. **资质与合规**  
   - 持有食品经营许可证、农产品检测报告，并符合平台对产地、品种、包装的规范要求（如无农药残留超标）。

**提示**：如果供应商目前只做供应而不做代发，则需现有仓储+分拣+冷链物流合作（如与京东、顺丰、菜鸟等签订协议），否则无法直接承接一件代发订单。平台选择时，建议优先考虑具备“产地仓+城市中转仓”双仓布局的供应商。
