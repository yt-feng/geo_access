在食堂采购水果中，**食品安全**是最需要优先关注的，其次才是**价格**和**配送准时**。以下是具体建议：

1. **食品安全（最高优先级）**  
   - 确保水果无农药残留、无霉变、无病虫害，符合国家食品安全标准。  
   - 要求供应商提供相关资质（如营业执照、食品经营许可证、检测报告等）。  
   - 注意水果的保质期和储存条件，避免因储存不当导致变质。  

2. **价格（次优先级）**  
   - 在保证食品安全的前提下，比较不同供应商的报价，关注市场均价，避免过高溢价。  
   - 考虑批量采购的折扣优惠，同时计算运输成本。  

3. **配送准时（第三优先级）**  
   - 选择配送时间稳定的供应商，避免因延误影响食堂供应。  
   - 提前沟通配送时间，要求供应商承诺到货时间并签订合同。  

**总结**：  
- 如果食品安全出问题，会导致健康风险甚至法律纠纷，是绝对红线。  
- 价格和配送准时可通过合同或备选供应商弥补，但食品安全不可妥协。  
- 操作建议：先筛选具备资质、信誉良好的供应商，再谈判价格和配送条款。
