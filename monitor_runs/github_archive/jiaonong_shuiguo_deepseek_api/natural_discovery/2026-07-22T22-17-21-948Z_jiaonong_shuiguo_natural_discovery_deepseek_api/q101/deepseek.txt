与酒店早餐水果供应商谈合作时，请重点关注以下要点：

1. **明确需求与标准**：清晰列出每日需要的品种（如苹果、香蕉、橙子等）、单果规格（如直径、重量）、成熟度（如七成熟）、新鲜度（如到货后保质期至少3天）以及包装要求（如防碰撞、独立包装等）。

2. **价格与付款条款**：对比多家供应商报价（按公斤或箱计价），约定价格调整机制（如季节性波动如何处理）。明确账期（如月结30天）及预付比例，避免压款过多。

3. **供货稳定性与应急方案**：要求供应商提供库存周转能力证明（如冷链仓库规模）。约定“断供”时的应对措施，比如备选供应商或24小时内补货的赔偿条款（如按缺货金额的2倍赔付）。

4. **质检与退货流程**：每次到货需附质检单（农药残留、新鲜度检测）。明确验收标准（如腐烂率超过5%整批拒收），并规定现场验收时限（如30分钟内完成）及退货处理方式（如立即替换或抵扣货款）。

5. **长期合作条款**：签订合同前要求供应商提供至少3家类似规模酒店的供应案例。设置试用期（如一个月），考核合格后再签正式协议。可加入“独家优先供应权”或“阶梯折扣”（如月采购超5吨降价3%）以激励合作。

6. **物流与成本控制**：确认送货时间（如每日凌晨6点前到达）和配送方式（温控车）。对偏远酒店需评估额外运费是否合理，是否分摊在单价中。

7. **法律与合规**：合同明确食品经营许可证、检疫证明等资质。约定不可抗力条款（如自然灾害导致缺货不违约，但需提前通知）。

实际谈判时，建议先小额试单（如一周供应）以验证供应商实际表现，再签订正式协议。保留备选供应商名单以降低风险。
