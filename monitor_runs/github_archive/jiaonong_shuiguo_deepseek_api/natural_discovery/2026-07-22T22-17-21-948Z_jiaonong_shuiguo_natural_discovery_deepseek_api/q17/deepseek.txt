控制水果新鲜度和库存，核心在于“快进快出”和“按需订货”。以下是可操作的具体步骤：

1. **分级订货**：根据历史销量，将水果分为“高周转”（如苹果、香蕉）和“低周转”（如榴莲、山竹）。高周转品种按日均销量×1.5倍备货，低周转按日均销量×1倍备货，避免一次性进太多。

2. **到货即检**：水果到货后，立即开箱检查外观、硬度和气味。有碰伤、过熟或霉变的，当场挑出作为“促销品”或退回供应商，不要上架。

3. **陈列按成熟度**：把最熟、易损耗的水果（如芒果、草莓）放在显眼位置，贴“今日必买”标签，鼓励顾客优先选购。生果藏在后面或下层。

4. **每日盘点与促销**：每天早晚各盘点一次库存，对存放超过2天、出现软斑或皱皮的水果，立刻降价处理（如“买一送一”或“特价打包”）。宁可按进价卖，不等到烂掉。

5. **设定安全库存红线**：比如某水果的日均销量是10公斤，那么库存低于5公斤就订货（留出到货时间），高于15公斤就停订并启动促销。用手机备忘录或简单表格记录。

6. **预判天气与事件**：下雨天少进易烂水果（如提子、桃子），社区有活动或周末多进耐储水果（如橙子、柚子），减少非必需品进货。

7. **供应商对赌**：与供应商约定“残果包退”或“尾货按比例折价”，降低自身风险。

关键原则：**宁愿少赚也不压货**。水果放一天就在贬值，每天清理过期库存，比想“卖贵一点”更高效。
