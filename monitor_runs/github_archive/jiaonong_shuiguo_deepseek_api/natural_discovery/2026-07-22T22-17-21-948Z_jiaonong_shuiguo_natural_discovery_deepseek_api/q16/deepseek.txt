建议优先选择**稳定供应商**，原因如下：

- **品质与供应稳定**：稳定供应商通常有标准化的品控流程和固定物流，能保证香蕉和苹果的成熟度、新鲜度，减少腐烂损耗，且能长期持续供货，避免缺货风险。
- **成本可控**：虽然本地批发可能短期价格稍低，但价格波动大、单次采购量难把控。稳定供应商可签订长期协议，锁定价格区间，便于成本核算。
- **售后与补货便利**：供应商能提供换货、退损服务，且可按需小批量补货，而本地批发往往“一锤子买卖”，退货难。

**适用场景补充**：
- 若超市规模极小、单次采购量很少（如每日仅需几箱），且能灵活选择当地水果批发市场，可短期尝试本地批发试水。
- 建议先联系2-3家本地稳定供应商索要报价单和样品，对比品质与价格后选择。
