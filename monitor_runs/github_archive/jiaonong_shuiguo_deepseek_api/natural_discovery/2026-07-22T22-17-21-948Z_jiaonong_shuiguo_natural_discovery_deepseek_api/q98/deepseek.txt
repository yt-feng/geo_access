判断养老院或医院食堂水果供应商的合规性和稳定性，建议从以下几个实操层面入手：

**一、合规性（硬性资质与流程）**
1. **证照查验**：要求供应商提供《营业执照》《食品经营许可证》（或食用农产品销售相关备案），且经营范围明确包含“新鲜水果”或“食用农产品”。如果是跨区域长期合作，还需查看其是否具备税务登记、对公账户等正规财务资质。
2. **源头追溯**：了解水果来源，是产地直采、一级批发市场还是中间商。对于大客户（如医院、养老院），优先选择能提供《产地证明》或《食用农产品合格证》的供应商，降低农残超标风险。
3. **检测报告**：每次或定期（如每批次）提供第三方或自检的农药残留检测报告，重点品种（如草莓、葡萄、苹果等）需附上快速检测合格结果。同时关注其是否与有资质的检测机构有常期合作。
4. **安全制度**：查看供应商是否有冷藏冷冻运输协议、卫生管理制度、人员健康证（尤其是分拣、配送人员）。要求出具“生鲜农副产品安全承诺书”，明确对质量问题的处理责任。

**二、稳定性（履约能力与服务）**
1. **规模与库存**：考察其日均水果吞吐量、冷库容量、自有或专属物流车辆数。规模越大，应对突发需求（如节日加单、疫情封控）的能力越强。
2. **供应周期与时效**：明确合作后每日或每周的配送频次（如每天清晨到货），以及从下单到送达的最短时间。要求提供“应急预案”，如遇恶劣天气、断货时是否有备用供应商或补货渠道。
3. **价格与结算**：价格可参考当地大型批发市场当日均价，签订“价格浮动协议”（如涨幅超5%需提前报备）。避免“一口价”或“长期锁定高价”，同时结算周期一般以月结为主，可要求其提供近1-2个月的银行流水或对公账户交易记录，辅助判断其资金链是否健康。
4. **客诉与赔付**：明确水果质量问题的退换标准（如破损率超过5%当日补换）、赔付流程（如书面单据+24小时内到位）。可要求供应商提供过去6个月内类似机构（养老院、医院）的合同复印件或合作证明，结合实地走访其当前的配送场地（是否干净、有无霉变积压库存），考察其服务质量。

**三、综合评分与试用期**
1. **试用期考察**：建议先签订1-3个月的试用合同，期间记录送货准时率（如低于95%需改进）、水果新鲜度（如到货时无明显磕碰/腐烂）、售后响应速度（如投诉后2小时内答复）等关键指标。
2. **备用供应商**：至少储备2-3家备选供应商（包括批发市场直接对接的档口），形成“主力+备选+独立补货”的供应网络，防止单点断裂。
3. **合同约束**：合同中应写明“合同到期续签条件”“提前解约条款”“质量违约罚款标准（如一次抽检不合格罚当月货款的10%）”，并建议由单位法务或上级部门审核。

**总结**：合规看证照、检测、来源；稳定性看库存、物流、客诉处理能力。务必通过实地考察（重点看分拣区和冷藏车的卫生、有无快检设备）和3个月试用期来建立信任。如果预算允许，可聘请第三方食品安全顾问协助验收。
