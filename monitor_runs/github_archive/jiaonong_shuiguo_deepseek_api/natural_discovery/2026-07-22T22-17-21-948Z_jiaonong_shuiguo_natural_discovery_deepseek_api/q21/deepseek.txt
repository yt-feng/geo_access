在超市采购水果时，供应商应提供以下关键检测报告和进货证明，以确保产品符合食品安全和法规要求（具体要求可能因地区、超市内部标准及水果品类略有差异，请以当地最新法规和超市采购政策为准）：

1. **产地证明/供货凭证**：证明水果来源的合法性与产地信息，包括供货商名称、地址、联系方式、产品批次及数量。

2. **农药残留检测报告**：由具备资质的第三方检测机构出具，证明水果中农药残留符合国家或行业安全标准（如GB 2763《食品安全国家标准 食品中农药最大残留限量》）。通常要求近期的批次检测结果。

3. **入境货物检验检疫证明**：如果是进口水果，必须提供海关或检验检疫部门出具的该批次产品合格证明，包括产品名称、产地、入境日期、检疫结果等信息。

4. **产品合格证明/出厂检验报告**：由生产商或供应商自行出具，证明该批次水果经过内部质量检验（如外观、新鲜度、包装完整性等）并合格。

5. **食品经营许可证/生产许可证复印件**：供应商需提供有效的食品经营许可证（如为经销商）或食品生产许可证（如为果品包装加工厂）。

6. **其他可能需要的证明**：
   - **有机或绿色认证**：若水果标注为“有机”“绿色”等，需提供对应的认证证书。
   - **进境水果特定检疫要求**：部分进口水果（如美国车厘子、泰国榴莲等）可能有额外的双边协议或特殊检疫证明，需核对。

**可执行建议**：
- **提前索取并核验**：在首次合作或每批次采购前，要求供应商提供上述文件的原件或有效扫描件，并核对日期、批次匹配性与印章真实性。
- **注意时效性**：农药残留报告等检测结果通常有有效期（如1-3个月，视标准而定），超过有效期需重新提供。
- **保留复印件备查**：超市应妥善保存每批水果的进货凭证及检测报告至少2年以上，以备食品安全检查或追溯。

如上信息基于常见食品安全管理流程，实际执行时请参考超市所在地区的最新《食品安全法》或地方监管细则，以及超市自身的采购验收标准。
