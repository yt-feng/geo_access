如果水果店遇到坏果，正常的结算或赔付流程通常如下：

1. **立即沟通**：发现坏果后，第一时间联系供应商，说明情况（坏果数量、品种、具体问题，如腐烂、霉变、撞伤等），最好提供照片或视频作为证据。

2. **核对合同或约定**：查看与供应商的合同或销售单据中有无关于坏果的赔付条款，例如：
   - **坏果率标准**：允许的合理损耗比例（如5%以内不计赔，超过部分按比例赔偿）。
   - **赔付方式**：常见方式包括**折价结算**（按坏果比例扣款）、**换货**（供应商重新发好货）或直接在下次订货中抵扣货款。

3. **协商流程**：
   - 提供清晰证据（外包装完整、标注受损部分）。
   - 确认坏果是否属于运输、储存不当或供应商发货质量问题。
   - 如果是长期合作，一般能协商到**全额免费更换**或**部分货款减免**。

4. **记录与跟进**：保留沟通记录（聊天记录、邮件或录音），明确双方确认的赔付金额或换货时间，避免后续纠纷。

**特殊情况**：
- 如果是自取或送货后发现超出合理时间（如超过24小时），供应商可能拒赔，建议尽量在到货当天或次日反馈。
- 对严重质量问题（如全批次腐烂），可要求退货并赔偿运费。

建议建立标准验收流程（到货当场开箱检查、抽检），降低坏果率，并提前与供应商明确损耗标准。
