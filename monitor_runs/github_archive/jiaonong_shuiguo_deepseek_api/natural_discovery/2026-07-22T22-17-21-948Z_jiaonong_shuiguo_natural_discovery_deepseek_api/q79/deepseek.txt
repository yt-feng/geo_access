最应该优先问**配送**。

原因如下：
1.  **水果是时效性极强的生鲜品**：配送延迟半天，损耗率可能上升10%-20%，直接影响你的客诉和利润。
2.  **售后是补充保障**：如果配送不靠谱，售后工作量会暴增。先确保配送稳定，售后压力自然变小。
3.  **价格是结果**：价格不是单独谈出来的，而是建立在稳定的配送、靠谱的售后基础上的。过低的价格往往伴随配送差、售后无。

**具体问法建议（按优先级排序）：**

1.  **配送（第一问）**：
    -   “你能覆盖到我所有团点的**到货时间**是几点？”（回答含糊/说“不一定”的慎选）
    -   “暴雨或节假日，你的配送是怎么保证的？”
    -   **关键点**：确认是否“定时达”+“区域全覆盖”。（例如：每天10点前必须送到所有小区门口）

2.  **售后（第二问）**：
    -   “到货后**多少小时内**可以报损？”（常见标准：12-24小时）
    -   “损耗超过**百分之几**可以全额退/补货？”（常见标准：5%-10%）

3.  **价格（最后问）**：
    -   在确定前两样靠谱后，再对比价格才有意义。
    -   要求**阶梯报价**（例如：日配50箱和100箱是什么价？）

**一句话建议**：先谈配送时效和补货机制，再谈价格。宁可单价贵5%，也要确保每天能准时、完整到货。
