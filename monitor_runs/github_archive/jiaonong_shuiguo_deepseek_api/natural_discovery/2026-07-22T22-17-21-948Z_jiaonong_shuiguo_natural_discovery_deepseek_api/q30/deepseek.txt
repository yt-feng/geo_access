针对水果品相不好的售后问题，建议按以下流程和约定执行：

**1. 明确品相标准**
- 定义“品相不好”：可量化为“明显腐烂、霉变、严重压伤、变色面积超过10%”等具体指标。
- 示例：单果腐烂面积＞1cm²或斑点超3处视为不合格。

**2. 到货验收时限**
- **现场验收**：收货后30分钟内完成抽检（至少5箱/批次）。
- 异常处理：若不合格率≥5%，当场拒收或协商折扣。

**3. 售后申报流程**
- **责任方**：由收货方在验收后24小时内提出，附照片/视频证据。
- **分级处理**：
  - 轻微瑕疵（如表皮划痕）：折价10%~20%或作赠品。
  - 严重问题（如内部发霉、腐烂）：全额退款或免费补货。

**4. 赔偿与时效**
- **违约金**：若供货方延迟处理超48小时，每日按问题货值0.5%补偿。
- 争议解决：保留第三方检测权利（如中国果品流通协会标准）。

**示例模板**：
```
1. 验收标准：单果腐烂≤1cm²，允许2%的轻微压伤。
2. 售后窗口：到货后24小时内反馈，超时视为验收合格。
3. 处理方式：严重问题48小时内补货或退款；轻微瑕疵按售价30%补偿。
4. 争议仲裁：双方盖章的验货单据作为最终依据。
```

**建议**：在合同中明确写清上述条款，并附图片模板（如“合格品”与“不合格品”对比照），避免歧义。
