针对茶饮店水果到货不合格的问题，建议从以下三个层面与供应商约定补货条款，确保营业不受影响：

1. **明确验收标准与退货时效**：在合同中写明具体水果的验收标准（如色泽、大小、甜度、无腐烂等），并规定到货后1-2小时内为验收期。不合格品需立即拍照留证，要求供应商在2小时内确认并安排补发。

2. **设定应急补货与替代方案**：约定供应商须在确认不合格后4-6小时内配送合格品到店，超出时间需承担临时向本地市场采购的差价。同时要求供应商提供备用品类（如冷冻水果或替代水果）供临时使用。

3. **约定补偿与优先权**：因补货延迟导致的经营损失，按日销售额的5%-10%补偿。并明确下次订单可优先发货或给予折扣作为信用修复。所有条款需以书面形式（如合同或微信记录）确认。

执行时，建议保留每次沟通记录，并定期评估供应商表现。若频繁出现问题，应启动备选供应商清单。
