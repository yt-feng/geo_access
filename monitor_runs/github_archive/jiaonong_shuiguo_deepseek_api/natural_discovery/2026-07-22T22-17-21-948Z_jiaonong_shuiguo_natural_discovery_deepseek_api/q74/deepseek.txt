批发档口拓展餐饮客户时，菠萝和香蕉的规格建议如下：

**菠萝规格准备：**
- **成熟度**：提供半熟（用于后期加工或冷藏）和全熟（即食）两种选择，标注保存温度（如10-15℃）。
- **大小与包装**：
  - 小菠萝（约500g-800g/个），适合制作饮品、沙拉等少量用需。
  - 大菠萝（约1.0-1.5kg/个），适合切块作为水果拼盘或烘焙原料。
  - 按箱或散装供应，标注去皮后净重（如每个去皮后约300-500g）。
- **处理形式**：
  - 去皮切块（真空包装，注明储存期限，如冷藏3-5天）。
  - 整果（注明去冠芽）。

**香蕉规格准备：**
- **成熟度**：提供青熟（青皮，适合存放更久或催熟）和全熟（黄皮带斑点，即食）两种，标注催熟日期或最佳使用期。
- **大小与甜度**：
  - 中小型香蕉（约18-20cm，单根80-120g），适合甜品、沙拉或包装使用。
  - 大号香蕉（20-24cm，单根150-200g），适合作为主水果拼盘或烘焙装饰。
- **包装**：整把（5-12根/把）或散装称重，标注单把重量范围。

**附加建议：**
- 提供样果让餐饮客户品尝，确认成熟度和风味符合需求。
- 标注产地、无催熟剂或有机认证（如有）。
- 对数量大的客户，可提供定期送货服务，并约定保鲜运输条件（如冷链车或加冰袋）。

这些规格能覆盖饮品店、西餐厅、甜品店等典型餐饮场景的常见需求。
