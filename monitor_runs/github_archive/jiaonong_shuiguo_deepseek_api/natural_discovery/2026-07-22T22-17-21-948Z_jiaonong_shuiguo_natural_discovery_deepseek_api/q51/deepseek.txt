建议**不要完全固定产地或供应批次**，而是采用**固定标准 + 动态选择供应商**的策略。主要原因如下：

1. **水果品质受自然条件影响大**：同一产地不同年份、不同季节的水果，甜度、酸度、香气和水分都会波动。固定产地无法规避这种波动。
2. **单一产地风险高**：如果该产地遭遇极端天气、病虫害或物流中断，会导致断货或品质骤降，影响茶饮出品和成本。
3. **供应批次不固定可能带来口感不一致**：如果换产地或批次时没有严格品控，客户可能喝到不同味道的产品。

**建议做法**：
- **制定明确的采购标准**：比如糖度（Brix值）范围、酸度、成熟度、外观大小等。所有供应商需符合标准。
- **选择2-3个主产区**：例如芒果可同时关注海南、广西、越南等地，根据季节和实测品质动态调整采购比例。
- **批量采购前做小样测试**：每批新货到店前，先做一杯成品茶饮进行口味评估，确认符合出品要求。
- **建立品质反馈与供应商评级**：记录每批水果的产地、批次号、到货日期和品质评分，定期淘汰不稳定的供应商。
- **考虑与供应商签订“保品质”协议**：明确约定品质标准和退换货条款，而非单纯固定产地。

**可执行动作**：
1. 立刻制定水果验收标准（可参考行业或自行测试）。
2. 联系现有供应商，要求提供不同产区当季水果样品对比测试。
3. 建立简单的采购档案表，记录每次进货的产地、批次、品质评分和客户反馈。

这样既能稳定出品品质，又能灵活应对水果的自然波动和市场变化。
