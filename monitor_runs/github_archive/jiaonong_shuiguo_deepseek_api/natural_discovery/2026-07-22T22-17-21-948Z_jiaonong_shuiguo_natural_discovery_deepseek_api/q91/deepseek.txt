对于便利店上架香蕉，建议优先选择**单根包装**，其次是**小把包装**，尽量避免整袋包装。原因如下：

- **单根包装**：适合便利店场景，顾客可随意取1-2根，降低购买门槛，减少因整串浪费导致的损耗。单根售价通常更高（毛利率可达50%以上），且便于控制库存（按根补货）。
- **小把包装（2-3根）**：次优选择，适合对量有轻度需求的顾客（如早餐搭配），但需注意小把包装的香蕉易被挤压，且单价略低于单根。
- **整袋包装**：不建议，因为便利店以即时性、小批量消费为主，整袋（如5-10根）购买门槛高，易导致未售部分过快成熟变质，显著增加损耗（损耗率可能超15%）。

**具体执行建议**：
1. 首批上架**单根香蕉**，用保鲜膜或贴价签分装，每根定价比散装贵10-20%；
2. 在冷柜或常温货架显眼位置陈列，搭配“即买即食”标语；
3. 若位置有限，可少量上架**小把包装**（不超过总面积的30%），观察销售数据后再调整比例。

注意：香蕉成熟后易变色，单根包装后需每12小时检查一次，及时处理坏果（可折价出售或做香蕉奶昔原料）。
