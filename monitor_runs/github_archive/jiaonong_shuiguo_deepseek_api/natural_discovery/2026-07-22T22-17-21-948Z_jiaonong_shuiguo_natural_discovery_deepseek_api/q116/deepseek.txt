对于小商家采购水果，现金结算和月结账期各有以下风险：

1. **现金结算的风险**：
   - **资金占用**：需要立即支付全款，可能影响短期现金流，尤其当进货量大或资金紧张时，容易造成周转困难。
   - **缺乏追溯凭证**：若未索要收据或发票，发生水果质量问题（如腐烂、缺斤少两）时，难以维权或要求退款。
   - **安全隐患**：携带大额现金交易，存在丢失或被盗风险。
   - **机会成本**：若市场出现更低价的临时货源，现金已用尽则无法灵活采购。

2. **月结账期的风险**：
   - **信用风险**：供应商可能因经营不善、虚假承诺或恶意拖欠而拒绝供货，或卷款跑路，导致已销水果无法结账。
   - **隐性成本**：供应商可能将账期成本计入单价（实际提价），长期看反而增加总支出。
   - **账目管理压力**：需详细记录每笔交易，若记账不清，可能引发对账纠纷或超付。
   - **依赖风险**：过度依赖单一供应商的账期，若对方突然缩短账期或停止合作，可能面临断货。

**建议**：
- 现金结算时，务必索要正规收据或合同，并保留交易记录（如转账凭证）。
- 月结前，核查供应商资质、过往信誉，并签订书面协议明确账期、违约责任和纠纷处理方式。
- 小商家可尝试“现金折扣”（如付现享5%优惠）或“分批结账”来平衡风险和成本。
