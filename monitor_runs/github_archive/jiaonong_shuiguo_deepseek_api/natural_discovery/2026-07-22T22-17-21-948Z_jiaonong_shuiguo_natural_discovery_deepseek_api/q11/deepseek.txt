会的，会明显影响销售。主要有以下几个原因：

1. **顾客偏好差异**：部分顾客喜欢大香蕉（显眼、觉得划算），部分喜欢小香蕉（适合孩子或一次吃完）。如果一箱里大小混在一起，顾客可能为了挑到满意的而翻动箱子，导致其他香蕉被碰坏或卖相变差。

2. **定价困难**：大小均匀的香蕉可以统一定价（如每斤或每箱固定价），但大小混杂时，顾客会挑剔大小差异，认为大香蕉价格应该更高，小香蕉应该更便宜，从而降低购买意愿或要求打折。

3. **损耗增加**：
   - 小香蕉容易被挤压或埋在箱底，卖不掉容易烂掉。
   - 顾客挑选时频繁翻捡，导致更多香蕉受损。

4. **陈列效果差**：整齐划一规格的香蕉在货架上更有视觉冲击力，显得品质高。大小不一则显得杂乱，降低整体档次感。

**建议**：
- 如果已经采购了大小不一的香蕉，可以按大小分拣，分成大果、中果、小果分别定价或组合销售（如家庭装、儿童装）。
- 如果条件允许，尽量优先采购规格一致的香蕉，尤其是靠近客单价高的区域或精品水果店。
- 如果是批发或走量，大小混杂的香蕉可以降价处理（如“混合装特价”），但需控制风险。
