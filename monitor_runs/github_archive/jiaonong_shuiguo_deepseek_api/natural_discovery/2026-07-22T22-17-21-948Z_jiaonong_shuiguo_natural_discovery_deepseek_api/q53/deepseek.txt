针对餐饮店每天少量水果的采购需求，建议优先寻找**本地小型批发商**、**社区配送平台**或**生鲜供应链服务商**。以下是具体选择标准和建议：

1. **选择本地小型批发商或市场固定摊位**  
   - 与附近农贸市场的核心摊位（如常客口碑好的）建立长期合作，要求每日按需送货，通常无起订量或起订量低。  
   - 优点：灵活调整品种与数量（如每日只送3-5种时令水果），可按实际使用量结算，减少损耗。

2. **使用生鲜配送App或社区团购平台**  
   - 如美团优选、多多买菜、每日优鲜（区域平台），选择“次日达”服务。适合小金额采购（如每日30-50元），平台提供标准化包装，可设定定时配送。  
   - 注意：需确认配送时效是否匹配营业时间，避免断货。

3. **对接本地食材供应链公司（如美菜、快驴进货等）**  
   - 这类平台提供“即订即送”服务，支持每日少量多频次订单，部分平台对餐饮新人提供首单优惠或免配送费。  
   - 省心策略：固定每周菜单，提前沟通让供应商按周打包（如每日2斤苹果+1斤蓝莓+半斤圣女果）。

4. **与其他小型餐饮店拼单**  
   - 联合附近同行（如早餐店、奶茶店）共同向批发商下单，共享起送量，分摊运费。可通过微信群协调需求量。

**关键注意事项**：  
- **验货流程**：每日收货时当场检查新鲜度（如瑕疵率超过5%拒收），要求供应商提前预冷或处理（如清洗、切块）。  
- **结算方式**：争取月结或周结（小额订单避免垫付压力），合同注明“损耗包换”条款（如水果变质，次日补发）。  
- **应急替代**：保留2-3个备选供应商（如1个本地商贩+1个平台），防止单方断供。

**推荐操作步骤**：  
1. 在地图App搜索“水果批发市场”，实地考察并添加3-5个商户微信，发送每日需求清单。  
2. 同时注册2个生鲜平台账号，对比价格和配送速度（优先选择免配送费或低于5元运费的）。  
3. 试用一周后，固定选择响应快、损耗补偿率最高的1-2个渠道长期合作。  

若所在区域有“城市共享仓”模式（如上海“鲜果盒子”），可按周定制每日冷冻份装水果（如切好的芒果丁），减少人力分拣成本。
