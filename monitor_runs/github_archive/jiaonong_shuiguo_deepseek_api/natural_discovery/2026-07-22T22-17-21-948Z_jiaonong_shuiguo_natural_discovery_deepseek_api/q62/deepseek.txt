判断香蕉的熟度和坏果风险，可以从以下几个方面入手：

1. **观察外观颜色**：  
   - 生香蕉：果皮青绿或黄绿色，果肉较硬，适合长途运输或长期存放。  
   - 熟度适中：果皮呈亮黄色，略带小褐点（“梅花点”），果肉软糯、甜度高，适合即时销售。  
   - 过熟或坏果：果皮暗黄、出现大面积黑斑或软烂、霉斑，或手感发软、有异味。

2. **摸硬度与弹性**：  
   - 用手轻捏香蕉中部：过硬则未熟，过软（尤其在棱角处凹陷）可能已变质。  
   - 正常香蕉有弹性，挤压后能回弹；若一捏就破或渗水，说明内部腐烂。

3. **检查果柄与切口**：  
   - 果柄新鲜、干燥，若变黑或发黏，可能已感染细菌。  
   - 切口处有白色乳汁或霉变，说明储存或运输条件不当。

4. **闻气味**：  
   - 正常香蕉有淡淡清香；若有酒精味、酸味或霉味，表明过熟发酵或腐烂。

5. **检查整体批次**：  
   - 随机拆开纸箱内部，留意中部和底部的香蕉（更易受挤压或通风不良）。  
   - 留意是否有“黑心病”（内部腐烂但外表正常）等隐蔽问题，可通过切开样本判断。

6. **结合运输与储存历史**：  
   - 询问运输温度（香蕉适宜13-15℃间，遇低温会冻伤变黑，高温加速成熟）。  
   - 确认是否使用乙烯催熟，或储存在密闭环境中。

**风险控制建议**：  
- 优先选择“七八成熟”的货（颜色转黄、微有绿，但仍较硬），根据销售速度安排存放或催熟。  
- 到货当天随机挑出5-10根纵向切开，观察果肉颜色与质地，确认无内部褐变或异常。  
- 若发现少量坏果，需立即检查相邻香蕉是否受影响，并考虑降低库存批次温度或加快销售。
