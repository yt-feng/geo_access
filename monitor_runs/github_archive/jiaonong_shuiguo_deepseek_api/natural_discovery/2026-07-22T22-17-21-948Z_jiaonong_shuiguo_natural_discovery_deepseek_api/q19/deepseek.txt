建议优先关注**切果损耗**，其次是**甜度**，最后才是大小和外观。具体原因如下：

1. **切果损耗直接影响利润**：菠萝削皮去眼后实际可售果肉通常只有鲜果重量的50%-60%。进货时若选果形不规则、芽眼深或过熟的菠萝，损耗会更高。超市必须计算“切果后每斤成本”来决定定价，因此损耗是关键指标。
2. **甜度决定复购与口碑**：甜度不够会导致顾客投诉或滞销。可通过试吃或看果皮颜色（金黄偏深通常更甜）、底部果眼间距（疏松的更成熟）来初步判断。
3. **大小和外观为次要**：大小主要影响陈列和包装（大果适合整颗卖，小果适合切块），外观有轻微疤痕不影响口感，但霉斑、发软则说明变质，不能进货。

**实操建议**：
- 小批量采购时，切开一个样品称重计算损耗率（剥皮去籽后果肉÷原重），再结合甜度记录，作为选果依据。
- 如果损耗率高于40%，或甜度明显偏淡，即使外观好、个大也应放弃。
