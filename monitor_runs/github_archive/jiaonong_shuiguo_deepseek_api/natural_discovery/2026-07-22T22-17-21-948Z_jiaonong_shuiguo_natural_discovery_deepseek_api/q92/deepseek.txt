针对便利店面积小的特点，水果品类和损耗控制建议如下：  

**1. 品类选择原则**  
- **选“高频刚需”单品**：只保留销量前5-8种水果（如香蕉、苹果、橙子、梨、葡萄、圣女果），避免长尾品类。  
- **优先耐储存品种**：如苹果、橙子、哈密瓜（可放3-7天），减少易损品（草莓、蓝莓、荔枝）的占比（控制在20%以内）。  
- **分切或小包装**：对西瓜、菠萝等大果，提前切块或分装成小盒（200-300g/盒），单价低且损耗可控。  

**2. 损耗预控措施**  
- **“以销定进”原则**：根据前一日销售数据按150%备货（预留50%弹性），避免存货积压。  
- **拆箱验收**：到货时立即检查坏果，按比例扣减货款或要求换货（3%以内自然损耗可接受）。  
- **低价快出**：对表皮微损/不新鲜的果子，立即做成切块果切或果汁，按原价80%售卖。  

**3. 陈列与保鲜**  
- **冷风柜优先**：所有易损水果（如葡萄、草莓）必须放在4-8℃冷柜，至少覆盖20%陈列面。  
- **翻堆频率**：每2小时旋转货物，将底层或外观不佳的果品提前处理（切块/报损）。  
- **包装标识**：每盒水果标注“分切时间”，超过4小时未售出直接下架，避免变质回头客投诉。  

**4. 动态调整**  
- **周度销量排行榜**：删除连续7日销量末尾的2个品类，补入1-2个时令水果试销。  
- **反馈机制**：顾客投诉“酸/不新鲜”的水果，直接收银台退款（不退货，避免二次损耗），同步后台标记问题品类淘汰。  

通过**精准选品-快进快出-拆零销售**三步，可将损耗率控制在5%以内，同时满足顾客基础需求。
