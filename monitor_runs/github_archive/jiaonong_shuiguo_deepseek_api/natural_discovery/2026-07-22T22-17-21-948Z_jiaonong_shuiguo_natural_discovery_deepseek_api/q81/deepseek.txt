支持。做即时零售的水果供应商，应该支持小批量高频补货。原因如下：

1. **匹配平台特性**：即时零售的核心是“快”和“即时”，用户下单后通常1小时内送达。如果供应商只能大批量送货，平台库存压力大，容易缺货或积压变质。
2. **降低损耗**：水果易损耗，小批量高频补货能让库存周转更快，减少因存放过久导致的坏果、损耗率。
3. **提升履约效率**：平台可以根据实时销售数据动态调整库存，供应商配合小批量补货，能更精准匹配需求，避免断货影响销售。
4. **增强合作竞争力**：能灵活配合即时零售平台（如美团闪购、京东到家）补货节奏的供应商，更容易获得平台流量和长期合作机会。

**操作建议**：
- 与平台约定最小起订量（比如每单5-10箱），支持每日或半日补货；
- 优化拣货和配送流程，确保小批量订单能快速响应（如自己安排同城配送或使用平台前置仓）；
- 利用平台数据预测补货频次（比如根据历史销量，工作日和周末的补货周期可以不同）。
