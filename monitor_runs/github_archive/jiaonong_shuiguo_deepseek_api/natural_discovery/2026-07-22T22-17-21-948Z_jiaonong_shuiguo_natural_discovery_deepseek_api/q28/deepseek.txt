建议从**进口香蕉**开始。理由如下：

1. **市场接受度高**：香蕉是日常消费量最大的水果之一，消费者对进口香蕉（如菲律宾、厄瓜多尔等产区）的接受度已较高，无需过多教育成本。
2. **供应链稳定**：进口香蕉的成熟度控制（如催熟工艺）和冷链运输体系成熟，损耗率相对可控。
3. **成本与周转**：相比菠萝需终端处理（去皮、切块等人工成本），以及苹果有国产优质品种替代（如红富士），进口香蕉的货架期更长，单品流转快，适合初涉进口水果的超市试水。
4. **风险最小化**：菠萝对温度敏感易腐，苹果面临国产鲜果价格竞争，而进口香蕉的品牌化程度高（如都乐、佳农等），合作门槛较低。

**操作建议**：先与大型进口商合作小批量试售，选货架期较长的“青黄熟”香蕉，并在门店设置独立陈列区，辅以“产地牌”营销（如“菲律宾直送”），观察1-2周动销率。稳定后，再逐步引入菠萝或苹果。
