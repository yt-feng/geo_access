商超做会员日水果活动，供应商需要提前配合以下关键准备：

**1. 货源与库存准备**
- 确认活动主打品种（如爆款、应季、引流品）和预估销量，提前2-3天备足货，避免断货。
- 备好特价商品（如会员专享价、买赠品）和试用装/试吃品。

**2. 质量与包装**
- 确保水果新鲜度，临期或次果单独处理。提前检查果品外观、甜度、成熟度。
- 提供定制包装（如小份装、礼盒装、贴会员日标签），便于陈列和促销。

**3. 物流与配送**
- 协调配送时间，建议活动前1天或当天凌晨到货，避免积压。
- 准备应急补货方案（如24小时内快速响应）。

**4. 陈列与物料支持**
- 提供专属展架、价签、海报、宣传卡（如“会员专享价”标识）。
- 配合商超安排堆头位置，设计试吃台（需配备牙签、纸巾、垃圾箱）。

**5. 人员与培训**
- 安排熟悉产品的促销员或导购（会员日当天驻场）。
- 提前掌握活动规则：会员折扣流程、买赠规则、兑换方式等。

**6. 定价与活动方案**
- 提前商定会员特价（建议比非会员价低10%-20%）及起购数量限制。
- 准备组合套餐（如“会员买三送一”）或关联商品推荐（如搭配酸奶、果汁）。

**7. 合规与数据**
- 确认水果符合超市准入标准（如产地、农残检测报告）。
- 提供历史会员销售数据，辅助商超预估备货量。

**8. 应急与售后**
- 准备补偿方案：如遇品质问题（磕碰、过期），即时退货或换货。
- 预留部分高毛利商品，应对临时调价或补单需求。

**最后建议**：与商超采购负责人提前3天开对接会，逐一确认以上清单的责任人和时间节点。
