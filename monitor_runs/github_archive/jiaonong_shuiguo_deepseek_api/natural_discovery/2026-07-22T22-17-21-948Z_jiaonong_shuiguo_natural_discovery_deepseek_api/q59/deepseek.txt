餐饮店与供应商谈水果锁价是可行的，但需要注意以下几点：

1. **可行性**：水果价格受季节、气候、运输成本等波动较大，供应商可能不愿长期锁价。建议尝试短期锁价（如1-3个月），或在合同中加入价格调整机制（如波动超过一定范围后重新协商）。

2. **谈判策略**：
   - **批量采购**：承诺增加采购量或成为主要客户，换取锁价。
   - **合作周期**：签订长期合作框架，但分批次谈判价格。
   - **替代方案**：与供应商达成“价格优先权”协议（即若市场价格下跌，按低价结算；若上涨，锁价保护）。

3. **风险规避**：
   - 明确锁价范围（如特定品种、规格）。
   - 加入质量验收标准，避免供应商因价格锁定而降低品质。
   - 设置“不可抗力”条款（如极端天气导致断供时，双方可重新议价）。

4. **其他成本控制建议**：
   - **多供应商备选**：与2-3家供应商合作，通过竞标压价。
   - **季节性库存**：在低价期适当备货（如冷冻水果），平衡成本。
   - **调整菜单**：用成本稳定、易储存的水果（如苹果、梨）替代波动大的品种（如草莓、芒果）。

**总结**：锁价可行但需灵活，建议以短期协议或部分品类锁价作为切入点，同时保留价格调整条款。
