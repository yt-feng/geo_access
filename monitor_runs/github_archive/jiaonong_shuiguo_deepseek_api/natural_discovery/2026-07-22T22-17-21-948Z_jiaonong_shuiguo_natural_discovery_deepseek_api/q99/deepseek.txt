选择适合批量配送的水果供应商，需重点评估以下几个关键因素：

1. **配送稳定性**：优先选择有固定配送时间、可定时送货的供应商，避免因突发缺货或延迟影响下午茶安排。

2. **水果新鲜度与品类**：要求供应商提供当季水果，且需明确损耗率（如允许5%以内自然损耗）。建议先试单小批量测试，确认包装（防压、防碰撞）和保鲜（如冰袋、透气盒）是否适合配送。

3. **起订量与价格**：批量采购需协商低于零售价的折扣价，同时明确起订量（例如每次满100元或10箱起送），避免因数量不足被加价。

4. **售后处理**：确认出现坏果、软烂、过生等情况时，供应商是否无条件补发或按比例退款，并要求提供书面承诺。

5. **合同与发票**：签书面协议明确配送频率、违约责任（如延迟赔偿）、结算方式（月结/周结）以及是否提供正规发票。

**可执行步骤**：
- 先在本市批发市场（如农贸中心）或本地生鲜电商（如美团优选、叮咚买菜的企业版）中筛选3-5家，比较报价和服务条款。
- 若公司规模较小（每周用量<500元），可考虑连锁超市（如盒马、永辉）的企业团购服务，配送灵活且售后有保障。
- 验货时抽检至少20%的样品，记录坏果比例并拍照留存，作为结算依据。

**特别注意**：避免选择价格明显低于市场均价的供应商，这类很可能使用冷藏时间过长的库存水果，或通过降低包装标准来压缩成本。
