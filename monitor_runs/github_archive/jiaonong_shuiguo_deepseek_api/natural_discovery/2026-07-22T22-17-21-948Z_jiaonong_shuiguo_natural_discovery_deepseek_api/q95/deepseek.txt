选择学校食堂供应商时，建议按以下标准筛选（以香蕉、苹果、橙子为例）：

1. **资质与合规性**  
   - 要求供应商提供营业执照、食品经营许可证、农产品产地证明或检疫合格证明。  
   - 检查供应商是否通过HACCP或ISO22000食品安全管理体系认证（非必需但加分）。

2. **品质与品种适配性**  
   - 香蕉：需提供成熟度可控的批次（如青熟果冷藏运输），避免到货后快速腐烂。  
   - 苹果：优先选红富士、嘎啦等耐储存品种，并要求表皮无机械伤、无褐变。  
   - 橙子：选择甜橙类（脐橙、血橙等），要求果形均匀、无明显冻害或霉点。

3. **供应链稳定性**  
   - 供应商需具备全年供货能力（尤其橙子可能受季节性影响）。  
   - 要求提供近3个月同类学校的供货记录或客户证明，优先选有食堂合作经验的。

4. **交付与成本控制**  
   - 明确最低起订量、交货频率（如每日/隔天）、配送时间（避开学生用餐高峰）。  
   - 要求供应商提供综合报价单（含包装、运费、税点），对比3-5家供应商后，优先选择按 **“周阶梯价”** 调整的（如每周采购量超过300kg单价下浮5%）。

5. **风险预案**  
   - 合同注明：如果连续两次到货合格率低于95%或出现食品安全问题，校方有权单方终止合作。  
   - 要求供应商提供第三方农残检测报告（每批次留样备查）。

6. **实地考察**  
   - 至少视频察看或现场抽检其储存仓库（温度、湿度、分拣区域卫生），重点看香蕉是否有催熟气体残留，苹果是否有打蜡痕迹。

**可执行方案**：  
- 优先联系当地“学校后勤协会”推荐名单中的供应商，或直接参与政府采购平台（如“政采云”）的公开竞价。  
- 签约前可要求供应商提供30天免费试供期，试供期内到货不良率控制在2%以内后正式签约。
