在超市与水果供应商的账期谈判中，常见的天数范围主要取决于超市的规模、水果的品类以及双方的合作历史。一般来说：

- **大型连锁超市**：账期通常较长，常见为 **30天至60天**，部分甚至可达90天。这是因为大超市议价能力强，资金周转压力大。
- **中小型超市或社区店**：账期通常较短，常见为 **7天至30天**，尤其是对易腐烂的水果（如草莓、葡萄），可能要求现结或7天内结算。
- **进口或精品水果**：因成本高、货值大，账期可能偏短（15-30天），或需预付部分定金。

**建议**：
- 如果你是供应商，可争取 **15-30天** 作为平衡资金和风险的起步方案，并考虑对易腐品设置更短账期。
- 如果你是超市，可从 **30天** 开始谈判，但需注意水果的保质期和供应商的现金流承受力。
- 最终账期应写入合同，并明确违约金条款。
