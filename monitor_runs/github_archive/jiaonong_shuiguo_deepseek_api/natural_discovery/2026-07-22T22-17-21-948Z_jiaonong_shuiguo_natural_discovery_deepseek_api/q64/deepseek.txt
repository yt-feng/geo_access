为降低退货纠纷，批发档口在收货验货环节应建立以下标准流程：

1. **当面验货，保留证据**  
   - 收货时务必**全程录像**（建议从开箱到逐件检查连续录制）。  
   - 核对**数量、款式、颜色、尺码**是否与单据一致。  
   - 检查外观：面料有无污渍、破损、脱线；金属件（拉链、扣子）是否完好；配件（腰带、吊牌）是否齐全。

2. **关键细节重点标记**  
   - 对**易争议部位**（如领口、袖口、下摆）用手电筒照看瑕疵。  
   - 如发现质量问题，**立即拍照并联系供应商**，保留原始包装和单据。  
   - 使用**记号笔或贴纸**在问题处标记，避免后续混淆。

3. **分类处理与记录**  
   - 轻微瑕疵（如线头）：可自行修补后收货，但需在验收单备注。  
   - 严重瑕疵（破洞、染色）：直接拒收，并发送验收照片与供应商确认后续（补发/退款）。  
   - **制作电子版验收清单**，注明每批货物的验收结果、时间、责任人。

4. **纠纷预防措施**  
   - 与供应商签订协议：**约定验收期限（如到货后4小时内反馈问题）**，超时视为认可。  
   - 拒收不清晰的货物：要求供应商在箱外标注**每箱明细清单**，避免混淆。  
   - 对**易碎品或工艺复杂商品**（如亮片衫、蕾丝裙），拆包后检查所有叠放面。

5. **售后配合**  
   - 向客户说明验收标准，提供**瑕疵定义示例图**（如什么是允许的色差、线头范围）。  
   - 一旦发现大量批次问题，**立即暂停该供应商订单**，重新评估合作。

**关键原则**：证据留底（视频+照片）、责任明确（4小时内反馈）、边界清晰（不接受事后议价）。
