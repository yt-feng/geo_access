对于水果批发商来说，面对价格波动，建议**根据具体情况灵活结合两种方式**，而非单一选择：

1. **临期或易腐水果**：建议**每天议价**。因为这类水果价格波动剧烈，且保质期短，灵活定价能快速应对市场供需变化，避免因固定价格造成更大损失。

2. **耐储存或反季节水果**（如苹果、梨、部分柑橘等）：可以考虑**签短期（如1-2周）价格合同**。这能锁定部分利润，降低频繁议价带来的管理成本，同时留有一定弹性应对突发波动。

3. **核心客户**：对长期合作、有稳定需求的大客户，可提供**有保底机制的浮动价格协议**（如：以某个批发市场均价为基准，给予±5%的稳定区间）。既维持关系，又规避完全固定价格的风险。

**可执行建议**：
- 分品类管理：快腐类当日议，耐储类签短约。
- 每日关注产地、物流、天气等关键信息，以支持灵活决策。
- 避免签超过2周的固定价格合同（除非客户愿意支付保费价差），以控制风险。

最终：没有绝对最优解，关键是根据**品类特性、客户类型、库存能力**动态调整。
