要判断哪种规格更划算，建议按**“单价（每斤或每公斤价格）”**和**“可食用率（去皮去核后的净重）”**综合比较。

1. **苹果**  
   - 通常按斤或按箱（例如5斤、10斤）卖。  
   - 整箱购买（尤其当季或特价）单价更低，但需确认有无坏果。  
   - **一般建议**：按斤买时选中等大小、果型均匀的，避免过大（可能空心或费工）或过小（削皮耗时）。

2. **橙子**  
   - 常见规格：礼盒装、散装成斤、或按个数（如10斤/筐）。  
   - 橙子皮厚，**实际可食用部分约70%左右**。  
   - 买整筐或大包装通常单价更低，但需注意保鲜。如果餐厅用量大，选15-20斤装的批发价更划算。  
   - **关键点**：称重前掂量，手感沉实、果皮光滑的果汁多，性价比更高。

3. **菠萝**  
   - 大多按“个”或“斤”卖。  
   - **注意**：菠萝皮和芯占重比例大（约40%-50%），如果按“个”买，需要对比每个的重量和价格。例如，两个大小差不多的菠萝，一个1千克5元，另一个1.5千克8元，后者的实际果肉单价可能更低（因为核芯占比类似）。  
   - **规格建议**：选中等大小（1-1.5公斤/个），切掉皮和芯后浪费较少。如果餐厅量大，可以找供应商切好果肉（速冻或鲜切），按净重计价，能避免浪费。

**最终建议**：  
- 苹果和橙子都**优先考虑按整箱/整筐购买**，只需核对品质。  
- 菠萝如果用量稳定，找批发商**按去皮净重报价**最划算；否则买中等个头的、重量较大但单价低的。  
- 所有水果都需**注意损耗率**：去皮去核、切块丢弃的边角料越少，实际使用成本越低。
