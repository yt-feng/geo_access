针对超市水果品相不佳的售后问题，建议按以下步骤约定清晰的流程，以保障双方权益并降低纠纷：

**1. 约定“品相标准”定义**
- 在采购合同中明确“品相不好”的具体标准（如：腐烂超过1/3、明显机械损伤、霉变、颜色异常等）。
- 可附图片示例作为合同附件或验收参照依据。

**2. 约定验收与反馈时间**
- **到货即时验收：** 收货后24小时内（或更短，如6小时）完成外观检查，逾期视为品相合格。
- **特殊情形处理：** 如发现内部品质问题（如内部腐坏），可在开售前或售出一定比例后反馈，但需保留证据（如照片、视频）。

**3. 约定处理流程**
- **第一步：举证**  
  要求供货方提供到货后完整开箱视频或照片（含时间戳），证明品相问题。
- **第二步：责任划分**  
  - 运输环节：若物流原因导致，可向物流索赔。
  - 供应商责任：需提供退货、换货、补发或折价方案。
- **第三步：协商方案**  
  例如：  
  - 整批品相不符：全额退货或换货，供应商承担运费。  
  - 部分不符：按问题比例扣款（如30%品相差，则按货值30%赔偿）。  
  - 可销售但品相次：协商折价（如按正常价60%结算）。

**4. 约定免责条款（可选）**
- 如水果因自然成熟、运输轻微挤压导致的外观瑕疵（非变质），可约定不属于“品相不好”范围。
- 明确不可抗力（如极端天气、疫情封控）导致的延迟或损坏处理方式。

**5. 书面记录与时效**
- 所有沟通、证据、处理结果需以书面形式（邮件、盖章凭证、平台聊天记录）留存。
- 建议设定最终解决时限（如：收货后48小时内处理完毕，超时自动按最优方案处理）。

**示例合同条款**  
> “乙方（采购方）在到货后24小时内有权对水果品相提出异议。品相不合格的标准为：单只水果腐烂面积超过1/3、或有可见霉变、严重机械损伤。甲方（供应商）收到异议后需在8小时内提供补发或退款方案。若未按时响应，视为同意全额退款。”

**执行建议：**  
- 要求供应商在发货前发送样品照片或视频确认。  
- 小额高频交易可逐步建立“品相分级”体系（如A级、B级），匹配不同采购价和售后条件。  
- 若供应商配合度差，可在合同中加入“违约金条款”（如按问题货值20%罚款）。
