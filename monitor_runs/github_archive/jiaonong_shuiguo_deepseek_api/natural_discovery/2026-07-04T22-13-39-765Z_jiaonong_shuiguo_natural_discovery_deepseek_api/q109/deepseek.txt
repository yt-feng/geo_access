对于长期合作的水果供应商，建议按以下优先级排序：**账期 > 配送 > 售后**。具体原因如下：

1. **账期（最优先）**  
   - 水果行业现金流周转快（尤其是生鲜），账期直接决定你的资金压力。  
   - 优先争取30-45天账期，能有效避免因库存积压或销售延迟导致的资金断裂。  
   - 可协商阶梯式账期（如前3个月15天，后续延长至45天），降低初期风险。

2. **配送（次优先）**  
   - 水果易损耗，配送稳定性直接影响品质。需重点谈：  
     - **时效**：明确到货时间（如每日上午10点前），避免断货。  
     - **损耗率**：约定运输中正常损耗比例（如5%以内，超出部分供应商承担）。  
     - **最低起送量**：避免因小单导致运费分摊过高。

3. **售后（可灵活调整）**  
   - 水果售后问题（如腐烂、品质不符）往往难量化，优先通过合同明确：  
     - **退换货标准**：如收货后2小时内反馈、损坏率超10%可免费补发。  
     - **责任划分**：配送途中损坏由供应商负责，到店后因储存不当损坏由你方承担。  
   - 如果供应商口碑好（如长期合作、退换率低），可适当降低此优先级，换取账期或配送优势。

**谈判技巧**：  
- 先谈账期，用“长期订单量”作为筹码（如“协议期内每月采购额不低于X万元”）。  
- 如果账期谈不下，可尝试“账期+配送折让”（如账期短但供应商承担更高配送损耗）。  
- 售后条款建议用“阶梯式标准”（如次品率<3%无售后，3%-5%退货，>5%全额赔付）。
