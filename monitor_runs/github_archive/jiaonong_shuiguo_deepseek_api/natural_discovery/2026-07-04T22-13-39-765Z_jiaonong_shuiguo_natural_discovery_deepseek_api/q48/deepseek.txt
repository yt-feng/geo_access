要判断供应商能否稳定按规格送货，可以从以下几个关键维度进行考察和验证：

1. **资质与历史记录**  
   - 要求供应商提供食品经营许可证、菠萝产地证明、农药残留检测报告等资质文件。  
   - 查看其过往合作客户（尤其同类型连锁茶饮品牌）的订单记录，要求提供至少3个月的送货单据，核对规格一致性（如单果重量、成熟度、包装标准）。

2. **实地考察与生产流程**  
   - 前往供应商的种植基地或分拣中心，查看以下流程是否标准化：  
     - 采摘后分拣：是否按客户规格（如直径6-8cm、无机械损伤）进行人工或机器筛选。  
     - 包装流程：是否使用防震托盘、冷鲜运输箱等，减少途中损耗。  
     - 质检环节：是否有专人定期抽检并记录规格偏差率（如≤5%为可接受）。  

3. **合同约束与违约条款**  
   - 在采购合同中明确约定：  
     - 规格标准（如单果重量误差范围±50g）、成熟度（可参考Brix值≥12%）、包装形式（如每箱10kg±0.5kg）。  
     - 送货频次（如每周三、五固定到货）及到货验收规则（如到货后2小时内完成抽样检查）。  
     - 违约条款：若连续2次不合格，供应商需赔偿物流费用并替换货品；累计3次不合格，可终止合作。  

4. **小批量试单与追踪**  
   - 先下3-5次小批量订单（如每次200kg），要求供应商按合同规格送货。每次到货后：  
     - 随机抽取10%的菠萝，测量重量、检查外观、切开测试成熟度（如金菠萝需1/4果肉变黄）。  
     - 记录每次合格率（如≥90%为通过），连续通过3次后方可签订长期合同。

5. **应急与追溯能力**  
   - 询问供应商：若某批次菠萝规格不达标，是否有补货库存或异地调货渠道（需24小时内能补上）。  
   - 要求供应商提供每批菠萝的种植批次号，确保可追溯至具体地块和采摘日期（应对质量投诉时能快速查证）。

**关键可执行动作**：  
- 向供应商索取近期连续3个月的送货单/质检单，核对历史规格偏差率。  
- 在首单合同中加入“验收后先支付80%货款，剩余20%在确认1周内无规格投诉后结清”的条款，降低风险。  
- 建议设立供应商评分卡，将“规格达标率”权重设为30%以上，连续3次低于90%则自动降级为备用供应商。
