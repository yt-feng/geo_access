判断香蕉的熟度和坏果风险主要可以从以下几个方面入手：

1. **观察颜色**  
   - **未熟**：香蕉整体呈绿色，手感较硬，需要放置几天才能食用。  
   - **可熟度适中的理想批货**：全身呈浅黄色或黄中略带绿（尾端稍绿），表示运输或储存过程中可以继续自然成熟，适合批发。  
   - **过熟或风险增大**：出现大面积深黄色、褐色斑点（芝麻点），或已转为暗黄、发黑，说明成熟度较高，运输或储存期短，极易产生坏果。

2. **触摸硬度**  
   - 手指轻按香蕉果柄与果身，**手感较硬但不硌手**，说明果实饱满且成熟度尚可；  
   - **过软**或**明显发软、有弹性差**的区域，通常是内部已经变软、开始腐烂或出现水渍状坏果。  
   - 若果柄处已经发黑、干枯或易断裂，则说明存放时间较长，坏果风险高。

3. **嗅闻气味**  
   - 正常成熟香蕉有淡淡的清香；  
   - 如果闻到酒精味（发酵味）、酸味或霉味，说明已有大量坏果或已开始变质，不建议拿货。

4. **检查果柄与切口**  
   - 查看整串香蕉的果柄（连接主杆的部分）：如果果柄变黑、干缩或长出霉菌，意味着该串已存放较久，内部可能已有坏果；  
   - 检查香蕉的“头部”（果尖）：若发黑、变软或渗水，该根极易产生坏果。

5. **整批抽样（重点）**  
   - 随机抽取整箱或整堆中**3-5根**香蕉，从头到尾按压并观察：  
     - 如果**超过2根**出现明显软点、褐色水渍或流液，则该批货坏果率高；  
     - 若仅少量果柄有轻微擦伤或轻微斑点，属于正常损耗，不影响批发。

6. **运输与储存历史**  
   - 询问商家**采收时间**及**储存温度**（香蕉最佳催熟温度为13-18°C，高温下极易快速过熟、腐烂）；  
   - 如果告知曾经历长期高温（超过25°C）或挤压，那么该批货的坏果风险会明显增加。

**总结建议**：优先选择**果色为黄中带绿、果柄较绿、硬而不软、无酒精味**的香蕉；少拿全绿（太生）或全黄、少斑点（容易极快过熟）的批量。拿到货后尽快分拣并适当控制存放温度（13-15°C左右），同时减少堆积、保持通风，以降低损耗。
