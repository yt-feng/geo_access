与酒店早餐水果供应商合作时，建议重点关注以下事项，确保供应稳定且成本可控：

1. **明确供货标准与验收细则**  
   - 书面约定水果品种（如香蕉、苹果、橙子等）、规格（如单果重量150-200克）、成熟度（如七成熟到货）、包装要求（如防撞泡沫箱）。  
   - 制定验收流程：到货时当场抽检变质率（建议低于3%）、新鲜度、破损率，不合格部分按比例扣款或退货。

2. **锁定价格波动机制**  
   - 避免固定价格，可采用“基准价+浮动系数”，如参照当地批发市场周均价±5%调整。  
   - 明确极端天气/节日期间的涨价上限（如最高不超过合同价的15%）。

3. **设立备用货源与应急条款**  
   - 要求供应商提供至少1-2家备用基地或应急调货渠道，并写入合同。  
   - 约定断供违约责任：如超过2次断供，酒店有权提前终止合作并索赔（参考按日供应价值的30%）。

4. **质量安全与可追溯性**  
   - 索要供应商的《食品经营许可证》、每批次水果的《农药残留检测报告》（重点检测百菌清、毒死蜱等）。  
   - 要求供应商提供产地证明（如来自二级以上产地批发市场），便于追溯问题批次。

5. **配送与存储计划**  
   - 明确每日配送时间（如6:00前到货），避免影响早餐备餐。  
   - 约定低温运输车温度（如苹果、梨0-4℃，芒果、香蕉12-15℃），到货后由酒店冷库直接入库。

6. **账期与结算方式**  
   - 建议试合作期（1-2个月）采用周结，稳定后月结（次月15日前对账）。  
   - 设置对账模板：包含日期、品名、单价、数量、总金额、验收签字栏。

7. **合同关键条款**  
   - 争议解决地约定在酒店所在地法院（便于诉讼）。  
   - 要求供应商缴纳履约保证金（如2万元，用于抵扣违约金或赔偿）。

**行动建议**：  
- 先要求供应商提供3-7天的免费试货（包括配送、包装等），测试其稳定性。  
- 合作初期安排专人每日记录到货延迟率、次品率，形成数据用于谈判议价或淘汰供应商。
