推荐以下两种不同定位的搭配方案，覆盖日常送礼和节日需求：

---

**方案一：实用均衡型（走量、价格适中）**  
- **组合**：苹果2个 + 橙子2个 + 香蕉1把 + 菠萝半个（切盒装）  
- **理由**：  
  - 品种多样，颜色鲜艳（红、黄、橙、绿），视觉丰富。  
  - 香蕉和菠萝易消耗，适合家庭短时间食用；苹果和橙子耐放。  
  - 菠萝切块可提升档次，且避免顾客处理麻烦。  
- **售价建议**：30-50元（根据当地水果成本浮动）。

---

**方案二：高端礼盒型（适合节日、探望长辈）**  
- **组合**：苹果4个（选红富士、糖心） + 橙子4个（选脐橙） + 香蕉0.5把（小把） + 菠萝1个（完整带叶）  
- **理由**：  
  - 数量对称（每种4个/1个），寓意“平安、吉祥、招财”。  
  - 保证水果新鲜度，避免因品种太多导致部分腐烂。  
  - 可附赠果叉、礼袋，提升价值感。  
- **售价建议**：60-100元（搭配精美包装）。

---

**注意事项**：  
1. 香蕉不耐压，需用软垫分隔，避免和硬质水果（苹果、菠萝）直接接触。  
2. 菠萝最好提前去皮切块单独包装，或附赠削皮工具。  
3. 不同季节水果价格波动大，可灵活替换为同色系其他水果（如梨替换苹果）。
