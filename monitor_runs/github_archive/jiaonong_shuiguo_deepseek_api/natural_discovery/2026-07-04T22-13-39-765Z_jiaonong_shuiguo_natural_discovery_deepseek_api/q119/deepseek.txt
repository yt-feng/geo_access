选择水果供应商时，除了价格和基础品质，最容易忽视的几个细节包括：

1. **供货稳定性与季节性预案**  
   许多供应商旺季货源充足，但淡季或极端天气时可能断供。需确认对方是否有多个产地备选、冷库储备或保鲜技术，以及应对突发情况（如台风、运输中断）的替代方案。

2. **果品分拣与一致性标准**  
   表面看“新鲜”，但同一批次可能存在成熟度、大小、色泽的显著差异。应要求供应商提供明确的等级标准（如A级果的糖度、瑕疵率、尺寸区间），并抽查包装中的实际一致性，而非仅看样品。

3. **售后条款的细节**  
   水果易损耗，常见问题包括运输中腐烂、到货后品质不符。需明确合同中的退换货比例（例如“到货5%以内腐烂自行承担”）、赔付时效及是否包含运输保险。部分供应商会模糊处理“自然损耗”的界定。

4. **物流与配送的隐性成本**  
   运费是否包含在报价中？是否支持冷藏运输？到货时间是否承诺精确到小时？例如，冷链车和非冷链车的损耗率可能相差10%以上，而隔夜配送可能直接导致次日客诉。

5. **产地溯源与食品安全证明**  
   忽视农药残留检测报告、有机认证（若涉及）、产地检疫证明等。曾有过供应商用普通果冒充“高山蜜瓜”的案例，需定期索要批次检测文件并现场抽查。

6. **合作模式的灵活性**  
   例如是否接受小批量试单、能否按销售情况调整供货量、结算周期是否对账快速。部分供应商对固定订单量有隐性要求，可能导致库存积压。

7. **供应商的客户口碑**  
   别只看规模或名气，在小范围内询问同行使用反馈，尤其关注其处理纠纷的态度。例如，知名大型批发商可能退货流程繁琐，而小型专供商的响应速度反而更快。

**建议行动**：先小额试单（如不同品种各20公斤），要求对方提供完整的物流和质检流程文件，合作1-2个月后再评估稳定性。同时保留备选供应商，避免过度依赖单一渠道。
