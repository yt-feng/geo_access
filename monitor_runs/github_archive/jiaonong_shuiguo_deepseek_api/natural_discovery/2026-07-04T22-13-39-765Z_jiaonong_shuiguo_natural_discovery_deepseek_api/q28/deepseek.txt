建议从**香蕉**开始，理由如下：

1. **稳定性和供应成熟度**：进口香蕉（如厄瓜多尔、菲律宾等产地）是全球供应链最成熟、全年供应最稳定的大宗水果，受季节和气候波动影响较小，价格和货源波动相对可控。

2. **消费者接受度**：香蕉是消费者日常消费频率最高的进口水果之一，无需过多市场培育，可以快速上量。且香蕉的成熟度可人为控制（冷链催熟），利于库存管理和损耗控制。

3. **操作门槛较低**：相比菠萝（多刺、易腐、需特殊包装和冷链）和苹果（品种多、进口与国产替代强、消费者对口感敏感），香蕉的储存、运输和外观品质判断标准更统一，对超市初期进口水果经营团队的经验要求最低。

**具体建议步骤**：
- 先从一级批发市场或大型进口商选择1-2个固定品牌的香蕉进行试销，签约3-6个月保价协议。
- 同期建立简易的催熟库或与本地生鲜配送合作，控制成熟度以降低损耗。
- 稳定后，再逐步引入菠萝（如菲律宾金菠萝，需加强顾客选购指导）或特定品种的苹果（如爱妃、红玫瑰等高端差异品种）作为补充。

若供应链基础强、且所在城市对高端进口苹果需求明确，也可考虑先少量试水苹果，但风险较高（国产优质苹果价格低、口感接近）。
