会的。大小不一致的香蕉对销售有以下几方面的影响：

1. **顾客心理与购买意愿**：顾客通常会认为大小均匀的水果品质更好、成熟度一致，更愿意按整箱购买。如果箱内大小差异明显，顾客会担心小果不甜或未熟、大果过熟，从而降低购买意愿。

2. **定价与利润**：大小不均匀的香蕉难以统一定价。如果按箱卖，价格过高会让顾客觉得“小果不值”，价格过低又可能让“大果”部分损失利润，整体利润率会受影响。

3. **损耗风险增加**：不同大小的香蕉成熟速度不同（小果可能慢、大果可能快），容易在运输或陈列过程中出现部分过熟腐烂，增加损耗成本。

4. **陈列与美观度**：零售时（如散卖）大小不一，堆在一起显得杂乱，影响货架吸引力，进而影响客流转化率。

**建议做法**：
- 优先采购大小一致的整箱香蕉，或要求供应商分选后再发货。
- 如果已进到大小不一的货，可以按大小分拣后分价格或分区域销售（大果卖出溢价，小果做特价或加工用）。
- 向顾客解释或标明“自然生长，大小不一”，但这类说明效果有限，整体还是以分装为佳。
