可以。餐饮店与供应商协商锁价（即约定一段时期内水果价格固定不变）是常见的做法，尤其是当前水果价格波动较大的市场环境下。以下是具体建议：

### 1. 谈判条件
- **锁定期长度**：建议短期（1-3个月）锁定，避免供应商因市场价格大幅上涨而亏损违约。长期（如半年以上）对方风险高，可能要求更高溢价。
- **采购量保障**：承诺稳定、足量的采购量（例如每月固定采购额或最低采购量），让供应商依靠批量规模盈利，而非单一高单价。
- **价格基准**：以当前市场批发价或历史均价作为参考，约定上浮或下浮幅度在合理区间（例如±5%），避免极端行情造成一方吃亏。

### 2. 合同条款注意
- **附条件调整机制**：例如约定若市场价格涨幅超过10%，双方重新协商，避免供应商因成本失控而违约。
- **质量与规格明确**：水果品种、等级（如一级红富士）、成熟度、大小、包装等需写入合同，防止供应商用次品代替。
- **违约条款**：如果供应商中途涨价或断货，约定赔偿比例（如采购额的20%）。同时你方不能随意退货或降低采购量。

### 3. 具体操作建议
- **小规模试单**：先选择1-2种核心销量大的水果（如苹果、橙子），锁定1个月，观察合作效果。
- **多家供应商比价**：用其他供应商的报价作为谈判筹码，但不要过度压价导致对方拒绝。
- **提前付款激励**：如果现金流允许，预付部分货款或缩短账期，换取对方更低的锁价。

### 4. 风险提示
- **市场下跌时**：如果水果市场价格大幅下跌，你仍按锁定价购买，可能比市价高。可约定保护条款（如“若市场价低于锁定价的5%，则按市场价执行”）。
- **供应商违约**：锁定后若供应商感觉亏本，可能以断货、质量下降等方式变相涨价。建议选择有合作历史、信誉良好的供应商。

**总结**：锁价能降低短期波动风险，适合用量稳定、资金充足的餐饮店。建议优先做短期、小范围试点，并在合同中留上调价余地。如果谈判困难，也可考虑与供应商签订“保底价格”或“浮动区间”协议，不必完全锁定。
