判断水果供应商是否靠谱，建议从以下几个维度进行核查和测试，避免踩坑：

**1. 实地考察与资质核验**
- **看仓储条件**：香蕉是呼吸跃变型水果，要求温控（13-15℃）和乙烯催熟能力；菠萝需阴凉通风。若仓库脏乱、混放或温控设备简陋，风险较高。
- **查经营资质**：要求对方提供营业执照、食品经营许可证。若为一级批发商，可查询其是否与大型种植基地或产区合作社直接签约（减少中间环节）。
- **索要检测报告**：要求提供近期农残检测报告（尤其菠萝易有农药残留），正规供应商会定期送检。

**2. 询价与报价逻辑**
- **警惕异常低价**：如果报价明显低于市场均价（比如最近菠萝批发价约2-3元/斤，香蕉约1.5-2.5元/斤），可能是库存积压或即将腐烂的尾货。
- **要求明码标价**：正规供应商会清晰列出：① 产地、品种（如都乐、海南蕉/进口蕉） ② 规格（如每箱净重、果指数量） ③ 包装费、运费、税点是否包含。模糊报价的要警惕。

**3. 试单与验收标准**
- **先少量试单**：首单不要超过5箱香蕉+5箱菠萝，观察到货时的成熟度。香蕉要求7-8成熟（果皮全绿但果指饱满无黑斑），菠萝需无霉变、果眼干燥。
- **验收流程**：到货后立即开箱检查：
  - 香蕉：随机抽3-5根，捏果柄判断硬实度（太软可能过熟），检查果皮有无黑星病、炭疽病斑点。
  - 菠萝：闻基部有无酒味（发酵变质），按果体看弹性（过软则已熟透易烂）。
  - **记录损耗率**：若单箱损耗超过10%（如香蕉断头、菠萝烂眼超过3个），供应商需承担赔偿或补货。

**4. 长期合作的隐性指标**
- **周转率**：询问供应商的进货周期（例如来自云南的香蕉，正常3-5天到货）。若经常断供或需压货，说明供应链不稳定。
- **售后响应**：试单时故意制造一个合理投诉（如“到货菠萝有2个烂了”），观察24小时内是否处理。靠谱供应商会主动提供退货、换货或抵扣下一次货款。
- **合同条款**：长期合作应签书面协议，明确：① 稳定供应量（如每周固定送一次，临时加单需提前24小时） ② 价格调整机制（如遇产地涨价，提前3天通知） ③ 滞销品处理（允许按70%价格退回部分积压货）。

**5. 行业口碑佐证**
- **线下去问同行**：到附近其他水果店，直接问“你们香蕉找谁拿的？踩过坑吗？” 多数店主会愿意分享。
- **线上查**：在天眼查/企查查搜索供应商公司全名，看是否有劳动纠纷、合同违约等官司记录。避开有“采购合同纠纷”案底的公司。

**决策清单**（满足 ≥4项可长期合作）：
- [ ] 能提供近3个月农残检测报告
- [ ] 试单损耗率 ≤8%
- [ ] 有固定冷库/催熟库（非临时租赁）
- [ ] 接受首单少量采购且支持无条件退货
- [ ] 合同明确价格调整需提前通知

**最后提醒**：香蕉菠萝类易腐水果，第一次合作前务必要求“先货后款”或“立即结算一单”，坚决拒绝预付大额押金。如果供应商要求你支付“保证金”、“风险金”才能开始合作，直接放弃。
