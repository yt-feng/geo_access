当水果店遇到坏果并与供应商结算或赔付时，通常可按以下步骤处理：

1. **确认责任归属**  
   - 检查坏果是否因运输、储存不当、包装破损或自然损耗造成。若为供应商原因（如货物发运前已变质），应由供应商负责；若因店铺储存或操作不当，则需自行承担。

2. **参考合同条款**  
   - 查看与供应商签订的合同或协议，重点关注：
     - 坏果率上限（如允许3%-5%的自然损耗）
     - 赔付标准（如按零售价、进价赔付，或按比例扣减货款）
     - 理赔流程（如48小时内报损、提供证据等）

3. **及时报损并取证**  
   - 发现坏果后，立即拍照或录像，保留坏果、包装、物流单据等证据。按合同要求的时间内通知供应商（通常为收货后24-48小时）。

4. **协商解决方案**  
   - 常见处理方式：
     - **退货退款**：将坏果退回供应商，供应商全额退款或补发新货。
     - **抵扣货款**：在下次结算时直接扣减坏果对应的金额。
     - **折价处理**：若坏果可降价销售，供应商补偿差价。
     - **按比例赔付**：根据坏果数量占总量的比例，按进价或零售价赔付。

5. **书面确认结果**  
   - 无论采用哪种方式，均应通过微信、邮件或合同补充协议书面确认，避免后续争议。

6. **长期预防**  
   - 如果同类问题频发，可考虑：
     - 更换信誉更好的供应商
     - 签订更严格的损耗赔付条款
     - 提高收货时的验货标准，必要时请第三方检测。

**总结**：正常流程是“确认责任→查合同→报损取证→协商赔付→书面确认”。建议保留所有凭证，同时与供应商建立良性沟通，既可减少损失，也能维护合作关系。
