选择学校食堂水果供应商时，建议按以下步骤操作，确保安全、性价比和供应稳定：

### 1. 核心筛选标准
- **资质齐全**：供应商需有营业执照、食品经营许可证、农产品检测报告（尤其关注农药残留）。
- **新鲜度保障**：香蕉通常7分熟运输、苹果耐储存但需无虫蛀、橙子需果皮完整无霉变。
- **价格透明**：建立比价机制，要求供应商提供每周或每日报价单（含规格、产地、单价）。

### 2. 推荐供应商类型
- **本地大型农贸批发市场**：如北京新发地、广州江南市场，可实地看货，议价空间大。
- **区域水果供应链公司**：例如“万纬物流”、“本来生活”等，提供定制化服务（如去核、切块）。
- **产地直供合作社**：通过农业农村厅官网或“农商互联”平台对接，减少中间商（如陕西苹果、广西橙子基地）。

### 3. 风险防控措施
- **合同条款**：明确退换货标准（如香蕉变黑超30%拒收）、违约金（延迟到货扣款10%）。
- **试吃与抽检**：每周随机抽取3箱水果，用糖度计测甜度、农残快检卡测甲醛/敌敌畏。
- **备用供应商**：至少储备2家供应商，避免单一来源断供（如台风季节需提前备货）。

### 4. 长期合作管理
- **月度评分**：按准时率（30%）、品质（40%）、售后（30%）打分，连续3月低于70分解约。
- **账期谈判**：可协商月结（30天账期）换更低价格，但需警惕现金流问题。

**立即行动建议**：
1. 联系当地市监局获取合法供应商名录（如“学校食材配送企业白名单”）。
2. 本周内走访2个批发市场、联系3家供应链公司，横向对比报价单。
3. 先签1个月试用合同，重点观察水果到货损耗率（目标<5%）。

如果需要具体某类水果的产地推荐或比价模板，可以进一步说明！
