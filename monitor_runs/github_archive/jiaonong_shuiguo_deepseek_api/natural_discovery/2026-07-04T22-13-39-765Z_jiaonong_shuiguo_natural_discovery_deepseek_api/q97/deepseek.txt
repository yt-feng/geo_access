酒店采购水果盘原料时，供应商应保证以下规格和外观标准：

**规格标准：**
1. **尺寸与重量**：水果（如苹果、橙子、哈密瓜）需符合约定大小（如直径8-10厘米），重量均匀（单果误差±10%以内），避免过小或过大。
2. **成熟度**：水果成熟度适中，不宜过生（口感涩）或过熟（易烂），如香蕉需八成黄即可。
3. **形态完整**：无歪瓜裂枣、畸形果（如严重变形的草莓），表面清洁无泥土或污渍。

**外观标准：**
1. **色泽**：颜色自然鲜亮（如苹果红润、猕猴桃翠绿），无异常变色或褪色（如黄斑、褐斑）。
2. **无瑕疵**：无机械损伤（如划伤、碰伤）、虫眼、霉变、腐烂（如按压有软烂部分），果梗剪裁平整不尖刺。
3. **新鲜度**：表面无干皱、失水蔫软现象（如葡萄果梗发黑），无农药残留白粉（需清洗后可见）。

**附加要求：**
- 供应商需提供批次检验单（如农残检测报告），包装要求防挤压（如使用分层隔板、气垫膜），运输过程冷链（易腐水果需2-8℃冷链）以保证到货后即用性。

建议采购方在收货时随机抽检（如每箱抽检10%），不符合标准需按合同约定拒收或折价处理。
