建议重点关注以下三个维度，按优先级排序：

1. **保质期（核心）**  
   - 菠萝切块属于鲜切水果，切开后易氧化变质，常温下保质期通常为 **4-6小时**，冷藏（0-4℃）可延长至 **24小时**。  
   - **操作建议**：  
     - 标注“包装时间”和“保质期至”时间（精确到小时）。  
     - 超过保质期的商品需立即下架并记录损耗数据，用于优化进货量。

2. **包装（影响卖相与安全）**  
   - **密封性**：使用食品级保鲜盒或自封袋，减少与空气接触，延缓氧化。  
   - **防渗漏**：菠萝汁液易溢出，需用防漏托盘或吸油纸垫底，避免污染货架。  
   - **标签信息**：包含品名、重量、价格、包装时间、保质期及储存条件（如“需冷藏”）。

3. **配送频次（控制损耗）**  
   - **高周转策略**：  
     - 若日销量>50份，建议 **每天配送1-2次**（上午/下午）。  
     - 若销量低，可改为 **隔日配送**，但需减少单次进货量，避免隔夜库存。  
   - **弹性调整**：根据历史销售数据和天气（高温天需求增加）动态调整频次，设置库存预警线（如低于10份时紧急补货）。

**优先级总结**：保质期 > 包装 > 配送频次。  
**执行checklist**：每日开档检查保质期 → 包装完整性抽查 → 根据前日销量确定次日配送量。
