对于餐饮连锁采购水果，选择中央厨房统一收货还是门店分开收货，取决于企业规模、产品特性及管理需求。以下是两种模式的对比及建议：

### 1. 中央厨房统一收货
- **适用场景**：门店数量多、水果需求标准化、有冷链配送能力的企业。
- **优势**：
  - **质量控制**：集中验收可统一标准，减少门店自行采购的品控差异。
  - **成本效益**：批量采购降低单价，减少物流中间环节。
  - **供应链管理**：简化供应商对接，便于加工（如清洗、分切、包装）。
- **劣势**：
  - **运输损耗**：水果易腐，需冷链运输，增加二次配送的损耗风险。
  - **库存压力**：需要中央仓库和短期库存能力，不适合高损耗或短保质期水果（如草莓、荔枝）。
  - **响应速度**：突发缺货时，补货需经中央仓库，可能延迟。

### 2. 门店分开收货
- **适用场景**：门店数量少、水果需每日鲜采、或门店有独立加工能力。
- **优势**：
  - **新鲜度高**：门店就近采购或直接配送，减少中转时间，适合高损耗水果。
  - **灵活性**：可根据本地需求调整品种和数量，避免统一配货的浪费。
  - **损耗可控**：无中央分拣环节，但需门店自行承担采购和验收责任。
- **劣势**：
  - **成本较高**：单店采购议价能力弱，物流成本分散。
  - **品控不稳**：各店标准不统一，可能出现品质差异或退货纠纷。
  - **管理复杂**：需管理多个供应商，增加采购和合规成本。

### 决策建议：
- **优先中央厨房统一收货**，如果：
  - 水果需加工（如切配、混合装盒）以制作标准产品。
  - 门店超20家或有稳定冷链网络。
  - 水果保质期较长（如苹果、柑橘、冷冻水果）。
- **优先门店分开收货**，如果：
  - 水果需当日鲜采（如草莓、樱桃、叶菜类）。
  - 门店有独立采购团队和严格验收流程。
  - 企业规模较小，中央厨房成本过高。

### 折中方案：
- **混合模式**：部分标准化水果（如柠檬、香蕉）由中央厨房配送，高损耗水果（如蓝莓、芒果）由门店直接供货。
- **区域分仓**：在核心区域设小型分拨中心，减少配送距离，兼顾集中与灵巧。

**最终判断**：建议根据水果的损耗率、加工需求及连锁规模测试。例如，生鲜沙拉连锁更适合中央厨房，而高端水果专卖店则需门店直采。选择前可进行小范围试点，对比损耗率、成本及顾客反馈。
