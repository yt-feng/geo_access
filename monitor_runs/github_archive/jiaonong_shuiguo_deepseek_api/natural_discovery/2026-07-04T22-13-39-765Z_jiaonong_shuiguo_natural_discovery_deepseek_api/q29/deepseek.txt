在超市与水果供应商的账期谈判中，常见的账期长度主要取决于**水果品类**、**合作规模**以及**双方地位**。一般而言：

1. **普遍标准**：**30天至45天**是最常见的账期区间。大多数中小型超市与常规水果供应商（如苹果、柑橘、香蕉等耐储存品类）会采用此账期。
2. **短账期（7-15天）**：常见于**高损耗、易腐烂**的水果（如草莓、蓝莓、荔枝、鲜切水果等），因为供应商需要快速回笼资金以降低风险。超市若要求更长账期，供应商可能要求加价。
3. **长账期（60-90天）**：多见于**大型连锁超市**对**稳定大供应商**（如水果进口商、大型种植基地），尤其是供应包装化、标准化产品的长期合作。部分账期甚至可达120天，但供应商通常会因此提高供货价。

**谈判建议**：
- 如果是超市方，可以**按品类差异化设定账期**：耐储存水果用30-45天，易损耗水果用7-15天。
- 如果是供应商方，可主动提出**“更短账期+略高单价”**方案，或要求超市方提供**授信额度、抵押**作为保障。
- 参考行业数据：根据中国连锁经营协会（CCFA）及部分生鲜供应链报告，水果品类平均账期在**20-45天**之间，但实际中需根据水果损耗率动态调整。
