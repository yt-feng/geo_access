针对批发档口拓展餐饮客户（如奶茶店、水果捞、甜品店、烧烤店等），菠萝和香蕉的规格建议如下：

**一、菠萝**
1. **规格**：建议主推 **2个/箱（约4-5斤/个）** 或 **6个/箱（约3-4斤/个）** 的小包装。餐饮客户通常单次用量不大，小箱更易周转。
2. **成熟度**：提供 **七成熟（可现场催熟）** 和 **九成熟（即食）** 两种选项。奶茶店多需要即食，烧烤店需耐存放。
3. **加工形态**：可额外供应 **去皮切块菠萝（真空包装）**，适合沙拉、水果茶类客户。
4. **品种**：主推 **金钻（台农17号）**（甜度高、纤维少）或 **香水菠萝**（小型，适合单人餐）。

**二、香蕉**
1. **规格**：提供 **1-2把/箱（约6-8根/把）** 或 **5kg/箱（散装）**。餐饮客户常用香蕉做奶昔、烤制或摆盘，小把装更易控制消耗。
2. **成熟度分级**：
   - **青皮（8成熟）**：适合需存放3-5天的烧烤店、快餐店。
   - **黄皮带绿（9成熟）**：适合水果捞、甜品店（需2日内用掉）。
   - **全黄带麻点（即食）**：适合饮品店、立刻使用的客户。
3. **品种**：推荐 **巴西蕉**（性价比高）或 **小米蕉**（适合小吃、装饰）。高端客户可备少量**红皮香蕉**（口感软糯）。

**三、附加服务建议（提升竞争力）**
- **提供“餐饮专用”标签**：注明规格、天数保质期，方便客户库存管理。
- **试吃装**：定期免费送少量不同成熟度样品，帮客户验证配方。
- **订货折扣**：对长期客户：累计订货量满一定数量，赠送去皮菠萝或香蕉托盘。

**四、注意事项**
- 避免主推大包装（如40斤菠萝箱），餐饮客户损耗大。
- 提前沟通物流路线，保证香蕉无机械损伤（易褐变），菠萝避免冷害。

最后，建议先向当地3-5家典型餐饮客户发送《需求调研表》（包含用量、耐储天数、加工需求），根据反馈动态调整库存结构。
