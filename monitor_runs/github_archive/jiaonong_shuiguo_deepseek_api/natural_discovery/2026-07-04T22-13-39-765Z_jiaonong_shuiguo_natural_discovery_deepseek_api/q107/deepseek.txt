起订量太高确实会增加损耗风险，原因如下：

1. **保质期限制**：香蕉和菠萝都是易腐水果，香蕉成熟速度快（通常3-7天），菠萝切开后保存期更短。高起订量意味着库存周转时间更长，容易因自然变质产生损耗。
2. **存储条件**：香蕉对温度敏感（最佳存储13-15℃），菠萝需避光、通风。若起订量超出冷库或常温存放能力，会加速腐烂。
3. **需求波动**：若起订量远高于实际销量，剩余货物可能无法及时售出，尤其是菠萝带皮保存期虽长（约1-2周），但切开后损耗率飙升。
4. **运输与搬运**：高起订量通常需要密集堆放，挤压、碰撞导致机械损伤，增加细菌感染风险。

**建议方案**：
- 评估历史销售数据，将起订量控制在1-3天周转量；
- 与供应商协商分批次到货（如先发80%，售完再补）；
- 尝试“预包装”或“折扣促销”快速消化库存；
- 如无法降低起订量，可开通社区团购或与餐饮企业分货。

⚠️ 注意：起订量需平衡单价优惠和损耗成本。若起订价折扣低于损耗成本，反而得不偿失。
