采购时应当优先关注**成熟度**，再考虑价格。

具体建议如下：
- **成熟度优先**：香蕉的成熟度直接影响蛋糕的甜度、风味和质地。过生的香蕉涩味重、淀粉含量高，甜味不足；过熟的香蕉虽然甜度高，但果肉发软、易出水，可能导致蛋糕口感湿黏或发酵不充分。理想的成熟度是香蕉皮上出现**黑色斑点**（“芝麻蕉”状态），此时甜度最高、香气浓郁，最适合烘焙使用。
- **成本控制**：当成熟度达标的香蕉短缺时，可以考虑少量采购价格更低的生香蕉，但需要在家中放置1-3天催熟（如与苹果同装袋）。如果烘焙店销量大，可以固定与供应商签订“指定成熟度”的采购合同，避免因价格波动影响品质。
- **特殊用途调整**：若制作香蕉泥含量较低的蛋糕（如香蕉马芬），可适当放宽成熟度；若制作以香蕉为主馅的糕点（如香蕉奶油蛋糕），则必须使用最佳成熟度的香蕉。

综上，建议采购时先检查香蕉表皮斑点比例（至少80%的果皮出现黑斑），再与供应商协商价格。长期来看，稳定质量比短期降价更重要。
