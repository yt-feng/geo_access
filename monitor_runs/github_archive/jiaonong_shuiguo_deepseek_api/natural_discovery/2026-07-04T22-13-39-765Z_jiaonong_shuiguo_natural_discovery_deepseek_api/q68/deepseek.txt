判断水果批发商进货时供应商能否按时到车，可从以下几个方面入手：

1. **查看历史合作记录**：优先与长期合作、有过按时交付记录的供应商交易，可比对过往订单的到货准时率。
2. **要求提供运输计划**：让供应商提前出具详细的运输安排（如装车时间、预计出发时间、途经路线、司机联系方式等），并确认其是否具备自有车队或长期合作的物流资源。
3. **设置违约金条款**：在合同中明确迟到或未到车的赔偿标准（如按小时或按货值比例罚款），有约束力时供应商履约会更慎重。
4. **参考第三方评价**：向同行或批发市场管委会了解该供应商的信用口碑，特别是近期香蕉产量旺季或物流紧张时期的交付情况。
5. **实时沟通与监控**：在发车前进行电话或信息确认，要求提供发车照片、GPS定位截图等证明。对高价值订单，可要求装上GPS追踪。
6. **小批量试单**：首次合作时先下小单测试其交付能力，避免大额订单的风险。

判断时需结合供应商规模、当地天气、节假日交通状况等外部因素。如果供应商对以上要求含糊其辞或无法提供稳定运输方案，则应保持警惕。
