第一次与水果供应商合作时，小批量试单是降低风险、验证双方匹配度的关键步骤。以下是具体设计建议，分为流程、合同、执行三个部分：

---

### 一、试单前准备（选择与沟通）
1. **明确试单目标**  
   - 验证品质：口感、外观（果径、色泽）、成熟度（可控催熟或即食）。
   - 验证供应能力：交货准时性、包装牢固度（防止运输损耗）。
   - 验证售后响应：对次果、损耗、迟到等问题的处理速度。

2. **优先选择可追溯的供应商**  
   - 优先选有产地直发能力、自有品牌或提供第三方检测报告（如农残、糖度）的供应商。

3. **沟通试单条款**  
   - 明确告知是试单，为长期合作做准备，降低对方压价或要求起订量过高的可能。

---

### 二、试单合同/协议关键条款
1. **数量与品类**  
   - 控制在常规订单的 **10%-20%**，例如：通常采购批量为500斤，试单选 **50-100斤**。  
   - 品类：只选1-2个核心高毛利单品，避免试错多个品种。

2. **付款方式**  
   - **首付30%-50%**（象征性预付），**尾款验货后付清**。  
   - 如对方坚持全款预付，要求对公账户付款并开票，保留转账凭证。

3. **品质标准**  
   - 明确果径（如苹果80#以上）、成熟度（八成熟可存放3天）、允许破损率（例如≤5%）。  
   - **附照片或标准卡**（如颜色、形状对比图），截图发给对方确认。

4. **售后与赔付**  
   - 明确：收货后**24小时内**反馈问题，超时不赔。  
   - 坏果赔付：分档次（如损坏率5%-8%按斤折算退款，＞8%全单退运或补发）。  
   - 附条款：“如试单品质达标，后续订单按约定折扣上浮5%”作为激励。

5. **交货与物流**  
   - 指定物流公司（如顺丰冷链或京东生鲜），试单建议走 **冷链直达**，避免中间商仓储环节。  
   - 明确运费承担方（建议卖方承担到指定仓库）。

---

### 三、试单执行与复盘清单
1. **收货流程**  
   | 步骤 | 检查内容 |
   |------|----------|
   | 拍照/录视频 | 外包装、内箱摆放、单果状态（至少拍摄3箱开箱全貌） |
   | 称重 | 实际净重是否与单据一致 |
   | 品控检测 | 随机抽取10%样品：切开检测内部品质（空心、霉变）、甜度计测糖度 |
   | 损耗登记 | 统计当日可用的好果比例 |

2. **销售环节监控**  
   - 试单部分建议**单独贴标**或标注“新品试吃”，收集客户反馈（例如：每天统计回购率、差评原因）。  
   - 如果品相不佳，直接作为“特价促销”快速清仓，避免影响常销商品。

3. **试单后决策参考**  
   - **继续合作**：品质稳定、损耗＜8%、售后友好 → 直接升级至常规订单（先按试单量的3-5倍下第二单）。  
   - **改进型合作**：某方面有问题（如成熟度不够）→ 要求对方调整后，再试单一次同规格产品。  
   - **终止合作**：出现腐坏率＞15%、货不对版、临时抬价等严重问题，及时止损。

---

### 四、避坑要点
- **不要预付全款**：水果生鲜非标品，一旦对方履约失败，追款成本高。  
- **不要一次性试多个品类**：增加品控难度，建议单次只试一个品种。  
- **保留所有沟通记录**：微信聊天记录、样品照片、报价单截图作为证据。  
- **小试单不等于零成本**：即便只赔几千元，也要当场止损，比后续大额亏损好。

最后：试单成功后，首次常规订单建议仍保留“验货后付尾款”或“账期5天”的过渡期，等连续2-3次交易稳定后再给长期账期。
