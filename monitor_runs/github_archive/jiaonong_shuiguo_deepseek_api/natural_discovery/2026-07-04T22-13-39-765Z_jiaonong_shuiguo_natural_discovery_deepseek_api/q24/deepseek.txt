向供应商索要以下资料清单，用于核验水果的农残合规性：

1. **产地证明**：明确水果的产地、种植基地信息，便于追溯来源。
2. **农药使用记录**：包括使用农药的名称、有效成分、使用时间、用量、安全间隔期等。
3. **第三方农残检测报告**：由具备资质的检测机构（如SGS、华测等）出具的近期（建议1个月内）检测报告，需涵盖常见农残项目（如有机磷、氨基甲酸酯类等，或按GB 2763标准）。
4. **合格证明文件**：供应商自检报告或出厂检验报告，附带批次号及检验结论。
5. **承诺书**：供应商书面承诺所供水果符合国家食品安全标准（如GB 2763《食品中农药最大残留限量》），并明确责任条款。
6. **绿色/有机认证（如有）**：若宣称绿色、有机等，需提供对应认证证书及检测报告。
7. **出入境检验检疫证明（进口水果）**：若为进口水果，需提供海关出具的卫生证书及入境货物检验检疫证明。

建议：要求供应商在每次供货时提供上述资料，并保留复印件或电子存档；对于新合作供应商，优先选择有标准化种植记录和长期第三方检测历史的。
