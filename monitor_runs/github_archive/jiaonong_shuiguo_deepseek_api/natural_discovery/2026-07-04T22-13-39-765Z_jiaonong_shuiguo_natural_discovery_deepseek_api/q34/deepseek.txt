在商超采购水果时，**损耗低比价格低对最终毛利的影响更大**。

原因在于：
1. **损耗直接侵蚀实际销售利润**：水果易腐烂、碰伤，损耗率常达10%-30%。即使价格低，若损耗高，实际可售数量大幅减少，导致单位成本摊高，最终毛利可能为负。
2. **价格低可能引发损耗加倍**：低价采购的水果往往品质更差（如过熟、病害），损耗率更高，且可能因品相不佳而需降价促销，进一步压缩毛利空间。
3. **实例对比**：假设采购单价5元，售价10元，毛利50%。若通过压价至4元（降20%），但损耗从10%升至30%，实际可售数量减少，最终毛利仅为原毛利的约70%；相反，若通过优化保鲜、控制损耗至5%，即使采购价维持5元，毛利可提升至约90%。

**建议**：优先选择供应稳定、品质好、配送快的供应商（减少运输损耗），并加强冷储、分批上架等仓储管理，即使单价稍高，也能通过低损耗实现更高总毛利。
