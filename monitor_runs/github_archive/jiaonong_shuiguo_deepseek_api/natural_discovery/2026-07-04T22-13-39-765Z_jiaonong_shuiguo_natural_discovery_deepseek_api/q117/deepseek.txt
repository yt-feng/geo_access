作为采购方评估新品时，建议采用“小批量试卖”原则，通常可以先试卖**该品类日均销售量的5%-10%**，或者控制在**一个月进货量的1-2周销量**。

具体操作建议：

1. **按门店类型调整：** 高流量核心店可适当多点（比如日均5-10箱），社区店或低流量店可以再减半。
2. **试卖周期：** 建议首次订货控制在**1-2周销售周期**内，避免库存积压导致损耗。
3. **观察核心指标：** 重点看**周复购率**（≥20%说明接受度不错）和**损耗率**（低于品类平均损耗率即为合格）。
4. **风险更低的做法：** 如果供应商允许，可以**先申请免费试销样品**（比如5-10斤），放在店门口做试吃，收集顾客反馈后再决定进货量。
5. **特殊水果注意：** 如果是从未卖过的稀有品种（如手指柠檬、刺角瓜等），首次试卖量建议控制在**1-2箱**，因为市场教育成本高，需求不确定性大。

总之，“卖完即止，不断货”的理想库存量就是最稳妥的试卖量。订货时一定要和供应商确认**退货或调换政策**，以防新品滞销。
