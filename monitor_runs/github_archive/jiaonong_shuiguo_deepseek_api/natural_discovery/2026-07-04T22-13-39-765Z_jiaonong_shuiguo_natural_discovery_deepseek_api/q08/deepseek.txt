对于进货量不大的小店，建议优先考虑本地批发市场，同时逐步培养1-2个固定供应商。具体分析如下：

1. **批发市场更适合起步阶段**  
   可以少量、多次拿货，降低库存压力，灵活试款。直接看到实物，方便比对质量和价格选择；部分市场（如服装、小商品）允许混批，甚至可和同行拼单降低单价。

2. **固定供应商适合稳定后**  
   当销量稳定、需求明确时，找批发商或厂家建立长期合作，能获得更低价、账期、退换货等支持。但初期量小议价能力弱，容易被动囤货。

3. **混合策略**  
   - 高频、稳定需求的商品（如饮料、日用品）找附近供应商，月结或按需补货  
   - 季节性、潮流性商品（如服饰、节庆用品）主攻批发市场，卖不完的款可快速清退  
   - 通过批发市场接触的商户，订单量提升后可转为固定合作

**执行建议**：  
- 先跑通3次批发市场，记录价格、质量、售后，确定2-3个长期可去的档口  
- 月采购额超过3000元时，主动问供应商是否可申请协议价或包邮  
- 关注线上批发平台（如1688），小额拿样对比线下价差，但注意核算运费和退换成本

最终原则：**小体量重现金流**，批发市场解决“试错”，固定供应商解决“效率”，避免签订长期合同或承诺最低进货量。
