在面积有限的便利店经营水果，核心策略是**“高周转、小批量、精选品”**。以下是具体的控制品类和降低损耗的实操建议：

**一、 品类控制：只选“高频刚需”和“耐存储”的**

1.  **坚决剔除皮厚、个头大、不方便即食的**（如整颗西瓜、菠萝、哈密瓜）。
2.  **主打“包装化”**：所有水果尽量分装成小盒（如：4颗草莓/盒、2个苹果/袋、1-2根香蕉/袋），提升客单价和便携性，同时减少顾客挑拣导致的损耗。
3.  **品类结构推荐（建议占水果区70%）：**
    -   **基础引流款（20%）：** 香蕉、苹果（红富士）、当季最便宜的大路水果（如夏季的西瓜切块、秋冬季的砂糖橘），确保价格有竞争力。
    -   **高毛利利润款（30%）：** 蓝莓、树莓、圣女果、芒果切块、菠萝蜜。这些单价高、易出精品包装。
    -   **网红/应急款（20%）：** 盒切西瓜、杯装水果沙拉、剥好的柚子肉。满足“即买即吃”场景，高毛利。
    -   **少采购或放弃的品类：** 榴莲（太占地方、气味大）、大串葡萄（易压烂）、软柿子/水蜜桃（易碰损）、需要大量存储的整箱装。

**二、 损耗控制：从进货到销售全流程**

1.  **采购原则：**
    -   **以销定采，不囤货。** 观察前一天+当天上午的销售数据，只采购预计**1-1.5天**能卖完的量。
    -   **少量多次，分时段补货。** 早高峰（7-9点）、午休（11-13点）、晚高峰（17-20点）分别小量少量补货，避免一次摆满。
    -   **优先选择常温耐放的品类。** 比如：苹果、梨、橙子、圣女果，比草莓、树莓、葡萄损耗低得多。

2.  **店内陈列与处理：**
    -   **不混放，分层陈列。** 易压坏的草莓、蓝莓放上层，耐压的苹果、橙子放下层。不要堆得高到顾客一碰就掉。
    -   **强制保鲜。** 香蕉、芒果、木瓜不可放冷藏；草莓、蓝莓、葡萄、切块水果必须置于**风幕柜**（温度2-5℃）。
    -   **分级销售，杜绝“边卖边烂”。**
        -   **A品（完好）：** 正常价格摆显眼位置。
        -   **B品（轻微磕碰、熟透）：** 立即移至专区，贴上**“今日特价/最后一批/立即食用”**标签，降价20%-50%处理。不要混入A品堆里。
        -   **C品（即将变质）：** 洗好切好，做成水果沙拉或鲜切盒，作为当日促销品。
    -   **设置“报损/待售”区。** 每天固定时段（如下午4点、晚8点）打折处理当日剩余品。

3.  **日常操作细节：**
    -   **每天早中晚三次翻拣。** 员工看到有明显伤痕、发软、长毛的水果，立即挑出进打折或报损区，避免加速其他水果腐烂。
    -   **勤快包装。** 早上补货时，把大串葡萄剪成小串、单根香蕉剥开装袋，既能提高单价，也减少顾客用手直接翻烂。
    -   **控制订货量。** 门店小，宁可早上10台摆满，晚上8台卖空，也不要剩下5台卖不掉的。

**三、 简易利润模型（验证可行性）**

-   **假设：** 每天水果销售额目标500元。
-   **推荐采购清单（总成本约250-300元）：**
    -   香蕉：1箱（约20斤），成本40元，卖2元/根，预计卖完。
    -   红富士：1箱（约30个），成本60元，卖2.5元/个，预计卖完。
    -   盒装圣女果/蓝莓/草莓：合计3-5盒，成本60元，卖12-18元/盒，高毛利。
    -   切好的西瓜/菠萝/芒果杯：成本40元，卖80-100元，当日清空。
    -   当日特价处理品（B/C品）：成本10-20元，卖15-30元，止损回本。
-   **损耗控制目标：** 总损耗控制在**8%-12%**以内（包括自然损耗、打折、报损）。超过15%就需要反思品类和进货量。

**四、 紧急行动清单**

-   **今天起：** 清点库房，把不耐放的、积压的、不新鲜的水果全部挑出打折处理。
-   **在进货单上：** 砍掉榴莲、大西瓜（可保留小西瓜切块）、大串提子，换成盒装草莓、蓝莓、圣女果、苹果。
-   **购置：** 1-2个风幕柜（或小型冷藏展示柜）专门放切块水果和浆果。
-   **制定：** 每日下午3点自动打折流程（如：所有未售完鲜切水果、B品水果统一贴7折标签）。

**最后提醒：** 便利店水果的本质是“零食化、便捷化”，而不是“卖场化”。让顾客随手拿、立即吃、不浪费，就是最低损耗的商业模式。
