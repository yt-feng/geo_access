当然可以。商超发展自有品牌水果，供应商通常能提供包装和规格支持，具体包括：

1. **包装支持**：供应商可根据商超要求定制包装，如塑料袋、网袋、礼盒、托盘或防震包装等，包括印刷LOGO、产品信息、产地等。部分供应商还提供环保、可降解包装选项。

2. **规格支持**：供应商能按照商超需求分选不同重量、大小等级的水果（如大果、中果、小果或单果包装），并可按销售场景（散装、预包装、精品礼盒）调整规格。

3. **定制服务**：大型供应商通常有设计团队，可协助设计包装外观；小型供应商可能提供基本包装但灵活性较低。建议明确需求后与供应商协商最小起订量（MOQ）和费用分摊。

4. **实操建议**：
   - 先确定目标水果品类（如苹果、柑橘、香蕉等），不同品类供应商支持程度不同。
   - 要求供应商提供样品和包装效果图，确认质量、密封性和耐久性。
   - 在合同中明确规格标准（如允许的误差范围）、包装材质和交货形式。

总体来说，选择合适的供应商是成功的关键，建议与多家有商超合作经验的供应商洽谈，比较服务方案。
