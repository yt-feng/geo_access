第一次做水果供货商试单，建议按“严控风险、小量测试”的原则操作，核心流程如下：

**1. 货量控制（关键）**
- 香蕉：建议每次只试一箱（约13-15公斤），因为香蕉熟得快，到货后3-5天内必须卖掉。不要一次多箱。
- 苹果：可选一箱（约20斤）同型号、同产地的，先试一种产地（如山东栖霞或陕西洛川）。
- 橙子：建议半箱或一箱（约15公斤），优先试单个果径（如70-75mm规格）。

**2. 找供应商渠道**
- 优先去同城/邻近城市的水果批发市场（如北京新发地、广州江南市场等）现场看货，找到在售的固定档口，谈“试单价格”或“样品价”。不要线上直接打全款。
- 要求供应商提供真实的产地证明或检测报告（尤其香蕉常被催熟剂处理）。

**3. 到货检查清单（24小时内完成）**
- 香蕉：开箱闻是否有刺鼻药味，检查整箱是否均匀黄绿（不要头绿尾黄的混货），把受伤、发黑的单独挑出。
- 苹果：捏硬度、看果柄是否新鲜（发褐说明存放久），切开一个看果肉是否发糠。
- 橙子：用手掂重量（同尺寸越沉越好），切开尝酸度，观察皮厚薄（皮厚难卖）。

**4. 试销与回款纪律**
- 这批货单独摆个小堆头或做促销（比如比市场价低10%），看顾客购买频率和反馈。
- 如果3天内非损耗原因（如外观、口感）出现退货或卖不动，立即暂停与该供货商合作，换下一家。
- 所有付款不要预付押金，坚持“到货验收后当场付清”，或者最多给500元以内的定金。

**5. 避坑提示**
- 如果供应商要求“起步50箱”“押金2000”才能试单，直接拒绝。合格供货商会接受1箱测试。
- 香蕉尤其小心“熟得快但芯硬”的问题（化学催熟），如果发现，下次换一个批发档口。

建议你做一张简单的《试单记录表》，记录：供应商名称、档口号、货品批号、到货日期、50%销量完成天数、顾客差评点。一旦表格中“首单亏损”标记2次以上，就换品类或换渠道。
