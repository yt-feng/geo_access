生鲜电商水果到货后出现坏果，责任划分通常遵循以下原则：

1. **运输环节**：如果是快递运输过程中因挤压、温度不当等导致坏果，责任主要在配送方或物流公司。但商家有义务选择可靠的物流并承担赔付责任。
2. **包装环节**：如果包装不当（如防震、保温措施不足）导致坏果，责任在商家。
3. **发货时状态**：若商家发出时已存在坏果或即将变质，责任完全在商家。
4. **签收时效**：顾客收到后未及时开箱检查，或长时间放置（如超过24小时）导致坏果扩大，可能需要承担部分责任，但商家通常应提供售后保障（如坏果包赔）。

**常见处理方式**：
- 商家一般会按“坏果包赔”政策，根据坏果比例退款、补发或赔偿（例如坏几个退几个的钱）。
- 顾客应在签收后尽快开箱拍照，将坏果与快递单或包装一起拍清晰照片，作为证据向商家或平台申诉。

**建议**：
- 如果顾客未及时检查，但商家包装或物流有明显问题，仍可主张商家负主要责任。
- 如果商家承诺“坏果包赔”，应直接根据约定执行，无需过度纠结责任划分。
