水果采购谈锁价的时间长短主要取决于水果的品种、市场供需情况、季节性和供应商的合作意愿。一般来说：

- **非季节性水果**（如苹果、梨等可长期储存的品种）：锁价可能持续1-3个月。
- **季节性水果**（如荔枝、樱桃等应季水果）：锁价通常较短，约1-4周，或仅在特定采摘期内有效。
- **进口水果**：受运输周期影响，锁价范围往往为1-2个月，但也受汇率、国际行情波动限制。
- **大宗或长期合作**：如果采购量大、合作关系稳定，供应商可能同意锁定3-6个月甚至更久，但通常会设置价格调整条款（如市场价波动超过一定比例时重新协商）。

实际中，建议先明确采购品种和预计采购量，与供应商协商时明确锁价期限、数量范围以及解约条件。如果市场波动大，锁价时间会相应缩短。
