食堂采购水果时，推荐采用**按斤结算**作为主要方式，结合具体情况灵活调整。以下是分析：

- **按斤结算**：最公平、精确，能根据实际使用量控制成本，避免浪费。适合日常采购如苹果、橙子等常见水果，管理方便且易于核算。但需注意称重准确和记录。
- **按箱结算**：适合大批量采购或水果种类脆嫩易损（如草莓、葡萄），可减少称重及运输损耗。但可能因包装重量、内部品质不一导致成本不透明，需提前协商标准（如每箱净重、成熟度）。
- **按份结算**：适合小份供应、标准化管理，如食堂提供果切盒、搭配套餐。但需要注意按份结算可能导致单价模糊，且不适用于按重量计价的大宗采购。

**综合建议**：
- 若水果即买即用、种类多且需控制成本，优先**按斤结算**。
- 若采购量大、有固定标准（如每箱10斤苹果），可与供应商协商**按箱结算**，并定期抽检确保净重及品质。
- 若食堂主要提供份装果切或单品水果杯，可部分采用**按份结算**，但需设定固定规格（如每份200克）。

最终应结合食堂规模、供应商诚信度及日常用量决定，可保留多种方式并存以适应不同场景。管理时建议建立采购台账，记录每次重量或箱数，并定期核对实际库存与消耗。
