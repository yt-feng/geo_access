作为采购方，核实水果供应商的全国配送能力，可以采取以下几个具体步骤：

1. **索要并验证配送区域清单**：要求供应商提供一份明确的、可配送的城市或区域列表（精确到地级市或区县）。不要只听“全国”二字。对比你希望覆盖的重点区域（如一二线城市、偏远地区、乡镇等），看列表是否包含。

2. **进行小规模试单或样单测试**：这是最有效的验证方法。选择2-3个不同区域（例如：一线城市、中部省会、偏远县城）的地址，分别下单少量水果。实际追踪配送时效、物流追踪信息、到货时的水果新鲜度和包装完整性。测试结果最真实。

3. **查阅第三方物流合作记录**：询问供应商合作的物流商（如顺丰、京东、中通冷链、德邦等）。如果是长期合作的正规供应商，一般会有物流合作协议或系统对接记录。查看他们是否具备冷链配送能力（水果易损，需冷链），特别是夏季或发往远距离地区。

4. **核实库存与分仓配置**：全国配送能力往往依赖分仓（如华东、华南、华北仓库）。问清供应商的仓库布局。例如，在东北的订单是否从沈阳或哈尔滨的仓库发出？如果没有分仓，完全依赖一个仓库全国发，远距离配送的时效和保鲜会大幅打折。

5. **要求提供历史配送案例**：让供应商提供3-5个不同区域的典型配送案例，包括客户名称、配送量、配送时长、客户反馈等。可以尝试联系其中1-2个客户进行侧面验证。

6. **签订明确违约条款**：在合同中明确约定配送时效、破损率、到货品质等标准，并设置违约金或无条件退货条款。例如：“承诺72小时内送达XX地区，超时或品质不符，采购方有权拒收并索赔。”这会让供应商不敢虚报能力。

7. **通过平台或行业渠道交叉验证**：如果供应商是电商平台的商家，查看其商品的用户评价中关于配送到不同地区的反馈。如果是传统供应商，向同行业采购方（如其他水果店、餐厅）打听其口碑。

**总结：最直接的方法是“试单”。不要听宣传，只看实际配送结果。另外，关注冷链能力（是否是全程冷链车/保温箱）、起送门槛（是否支持小批量单点测试）、以及退换货政策（不良品如何处理）。**
