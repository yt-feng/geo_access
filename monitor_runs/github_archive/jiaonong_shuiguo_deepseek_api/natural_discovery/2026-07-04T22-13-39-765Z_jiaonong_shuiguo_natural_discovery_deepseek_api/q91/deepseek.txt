建议优先选择**单根**和**小把（2-3根）**组合销售，谨慎上架整袋包装。理由如下：

1. **单根**：满足即食、少量购买需求，减少消费者因整把吃不完导致浪费的顾虑；适合搭配早餐、零食场景，周转快。
2. **小把（2-3根）**：兼顾家庭及分享场景，比整袋更灵活；香蕉成熟度易控制，避免整袋中部分过熟催生损耗。
3. **避免整袋**：整袋（如5-6根以上）易因消费者挑选成熟度不均（如部分青、部分熟）导致退货或丢弃；且便利店顾客流量高、单次采购量小，整袋动销慢、损耗风险大。

**执行建议**：  
- 初期测试：70%单根 + 30%小把，根据销量和损耗数据调整比例。  
- 注意香蕉成熟度分级：销售前按青熟、可食用（微黄）、全熟（带芝麻点）分3类陈列，并标记“今日可吃/明天可吃”引导顾客。  
- 损耗预警：单根单价可略高于小把（如单根1元，小把2.5元），避免消费者拆包挑拣导致损耗。
