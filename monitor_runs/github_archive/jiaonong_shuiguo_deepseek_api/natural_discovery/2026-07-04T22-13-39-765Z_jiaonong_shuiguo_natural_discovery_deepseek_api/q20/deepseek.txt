对于社区超市选择每日配送的水果供应商，建议重点考察以下条件：

1. **供货稳定性与新鲜度**：确保供应商能全年稳定供应常见水果（如苹果、香蕉、橙子等），且承诺每日配送（如凌晨或清晨到货），水果需保鲜处理（如冷链运输、预冷包装）。

2. **品种丰富度与季节性**：能根据季节提供应季水果（如夏季的西瓜、荔枝，秋季的猕猴桃），并支持小批量、多品种组合，以适应超市灵活调整货架。

3. **品质分级与检验**：供应商应有明确的水果等级标准（如一级、二级），并提供质检证明（如农药残留检测报告）。可要求每批次抽样验收，确保外观、成熟度、无烂果。

4. **价格与结算方式**：对比多家报价，优先选择长期批发价透明、支持灵活结账（如月结或周结），且有退换货或补货政策的供应商（如坏果按比例赔偿）。

5. **配送时效与应急能力**：需确认每日固定配送时间（如5:00-7:00），以及节假日或突发情况（如部分水果断货）时的临时补货或替代方案备选。

6. **合作经验与口碑**：优先选有社区超市或果蔬店长期合作经验的供应商，可通过询问同行或实地考察其分拣场地了解其信誉。

7. **售后与合同条款**：签订合同前，明确退坏果流程、配送延误补救措施（如减运费）、以及价格调整周期（如波动超过10%需提前通知）。

建议先试合作1-2周，从小量订单测试其稳定性，再逐步扩大订货量。
