小商家采购水果时，现金结算和月结账期各有以下风险：

**现金结算风险：**
- 资金压力大：需即时支付货款，占用流动资金，可能影响其他经营环节。
- 售后风险高：发现水果品质问题（如腐烂、缺斤少两）后，追索退款或换货较困难，因已钱货两清。
- 交易记录弱：若没有规范收据或合同，发生纠纷时缺乏凭证，维权成本高。
- 现金流管理难：若遇到突发情况（如销量下降），无法灵活调整付款节奏。

**月结账期风险：**
- 坏账风险：若供应商经营不善、失联或拖延，可能无法收回货款（即使结算日已到）。
- 信誉依赖性高：需与供应商建立稳定信任，否则对方可能因资金紧张延迟发货或降低质量。
- 利率隐性成本：部分供应商可能通过提高单价等方式变相补偿账期利息，增加采购成本。
- 账期管理复杂：需记账和催收，增加人工成本；若忘记结算可能影响合作关系。

**建议应对策略：**
- 优先选择现金结算时：对小额采购（如补货量少），或与首次合作、信誉未知的供应商交易。
- 适用月结账期时：与长期合作、信誉良好的供应商签约，并明确合同条款（如延期利率、质量不合格处理方式）。
- 混合模式：按比例搭配现金和账期，例如小额日采现金结算，大额周结或月结，并结合少量预付款。
- 强化记录：无论哪种方式，均要求供应商提供盖章收据或电子合同，保留微信/邮件沟通记录。
