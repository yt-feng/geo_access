在水果采购合同中，建议约定**坏果率超过5%** 即触发赔付条款。具体建议如下：

1. **行业常见标准**：新鲜水果的合理坏果率通常控制在1%~5%之间。具体比例需根据水果品种、运输距离、季节气候调整（如草莓、荔枝等易腐水果标准可适当放宽至10%）。
   
2. **合同明确条款**：  
   - 直接写明“若到货坏果率超过X%（建议5%），超出部分需按单价全额赔付”或“损耗超X%时，每超1%扣减总货款Y%”。  
   - 同时注明验收方式（如抽检比例、开箱拍照留证、24小时内反馈等），避免后续争议。

3. **特殊情形处理**：  
   - **到货即损**：若坏果率高于行业常见值（如超10%），可直接拒收并要求全部退货。  
   - **长期合作**：可协商阶梯赔付（如5%~8%部分按半价赔付，超8%部分按全额赔付）。

4. **法律参考**：  
   若合同未约定，司法实践中通常参照《民法典》第583条（质量不符合约定的，可要求赔偿损失），但举证难度较大。建议务必提前书面约定。

**建议**：  
- 合同中写入“验收后24小时内双方共同参与过磅、拍照并留存证据”。  
- 对高风险品种（如进口榴莲、蓝莓）可额外约定“冷链断链”视为全部坏果。

（本建议仅供参考，具体条款需结合交易习惯、运输条件及法律顾问意见确定。）
