要判断苹果、橙子、菠萝这三种水果做拼盘时哪种规格更划算，关键是看**“可食用果肉占比”**和**“单价/克”**，而不是只看总重量或个数。以下是具体建议：

### 1. 苹果：选“小果”更划算
- **原因**：苹果皮和核占比相对固定（约15%-20%），个头越大，果肉占比略高，但价格通常按个数或斤数计算，**小果单价往往更低**（如5-6个/斤的苹果比3-4个/斤的便宜）。拼盘需要切片或切块，大小均匀的小果更方便，损耗也小。
- **建议**：买**中等偏小、个头匀称**的苹果（如70-80mm直径），避免特大果（差价可能达30%以上）。

### 2. 橙子：选“中果”且皮薄品种
- **原因**：橙子皮、核和海绵层占重量的20%-30%，**皮厚品种（如脐橙）** 果肉率低；小果剥皮难、出肉少，大果单价高。中果（如直径8-9cm）往往果肉率最高（约70%-75%）。
- **注意**：优先选**夏橙、血橙等皮薄品种**，比普通脐橙果肉率高5%-10%。
- **建议**：买**中果（约180-220g/个）**，且用手掂量——沉甸甸的说明水分足、果肉多。

### 3. 菠萝：选“小菠萝”或“国产菠萝”
- **原因**：菠萝外皮和果眼占重量的40%-50%，**大菠萝**（>2kg）果肉率未必更高，但价格通常比小菠萝贵；**小菠萝**（如都乐金菠萝，约1kg左右）果肉率相近，单价更低。
- **注意**：进口菠萝（如泰国小菠萝）甜度高但价格贵；**国产徐闻菠萝**性价比最高（约3-5元/斤），但需处理果眼。
- **建议**：买**1-1.5kg的成熟菠萝**（表皮金黄、有清香），避免超2kg的大果。若可用菠萝罐头替代，成本更低（但口感稍差）。

### 总原则：按“每元可得果肉克数”算账
- 去菜市场或超市时，直接问**每斤价格**，然后估算：
  - 苹果：实际果肉≈85%重量  
  - 橙子：果肉≈70%重量（皮薄品种可算75%）  
  - 菠萝：果肉≈55%重量  
- **举例**：
  - 苹果5元/斤 → 每元可得果肉 = 500g×0.85÷5 = **85g/元**
  - 橙子8元/斤 → 500g×0.7÷8 = **43.75g/元**
  - 菠萝4元/斤 → 500g×0.55÷4 = **68.75g/元**
  显然苹果最划算，菠萝次之，橙子最贵。

### 最终操作建议：
- 如果预算有限，**多放苹果**，搭配少量橙子和菠萝提味。
- 如果追求卖相，**橙子选大小均匀的脐橙**（切面好看），菠萝买**去皮的鲜切菠萝**（省时间但单价高）。
- 注意：超市的**切好拼盘**通常比买整果贵2-3倍，除非你用量极少，否则不推荐。
