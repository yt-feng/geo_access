当向普通商家介绍专业农产品时，建议用他们熟悉的生意语言来翻译专业术语。核心思路是：**把“技术参数”换成“卖货优势”**。

以下是一个清晰、具体的说话框架和话术示例，你可以直接套用：

---

### 第一步：用他们关心的“痛点”开头
不要说“我们有新品种凤梨”，而是说：
> “老板，很多客人觉得普通菠萝要泡盐水、削皮麻烦，买个回去好几个人还吃不完。我们这款小凤梨，**切开就能吃，一个人就能吃完一个，复购率特别高**。”

### 第二步：把“产地品种”翻译成“好卖的理由”
| 你说（专业术语） | 商家听到（利益点） |
| :--- | :--- |
| “我们是金枕头榴莲” | “肉厚核小，出肉率高，客人觉得值这个价。” |
| “我们是海南树上熟香蕉” | “不用催熟，拿回去放两天就自然黄了，味道比催熟的甜两倍，回头客多。” |
| “我们是泰国山竹，4A级” | “皮薄肉白，一剥全是肉，基本没坏果，损耗率很低。” |

**具体话术示例：**
> “老板，这个菠萝是‘凤梨’品种，最大的区别是**不用挖眼、不用泡盐水**，客人买回去切了就能吃。很多年轻妈妈懒得处理菠萝，就专门找这种。摆在你水果摊上，**普通菠萝旁边放它，能多卖一份钱**。”

### 第三步：直接用“利润账本”说服
不用讲“冷链运输成本”，直接算：
> “这个芒果进价虽然比普通贵2块一斤，但你卖价可以翻倍卖，而且**坏果率低**。普通芒果进货100斤可能烂10斤，这个最多烂2斤。算下来，卖这个你能**纯赚的利润反而更高**。”

### 第四步：给一个“试卖方案”降低决策门槛
不要催着下单，而是给一个“零风险”方案：
> “老板，您先拿一箱回去试卖，**放在货架最显眼的位置，标价11.8元一斤**（不要跟普通菠萝放在一起）。如果三天内这箱没卖完，剩下的我按进价回收。您没有任何损失，就能测出来好不好卖。”

---

### 一个完整的“下单型”介绍话术案例

**产品：树上熟木瓜**

**不专业的说法（商家听不懂/不想听）：**
“我们的木瓜是红心品种，糖度14度，产自海南，全程冷链运输，果形漂亮。”

**专业的说法（商家秒懂并想下单）：**
> “老板，您是不是经常遇到这种顾客：买木瓜回去切开，**要么是白色的不甜，要么是硬邦邦的没熟**？（痛点）
>
> 我们这个木瓜不一样，**是‘树上熟’的**（优势）。什么意思呢？就是别人地里摘下来运，我们是等它在树上长到8分熟才摘。**拿回去切开就是红的、甜的，不用再等**。你摆一个切开的样品在摊上，客人看到流口水，**转化率能翻倍**。（卖点翻译）
>
> 进价虽然比普通贵1块，但**普通木瓜1斤卖3块，这个能卖5块**，而且客人因为好吃会反复来买。你算算，**卖一斤赚的顶别人两斤**。（利润账）
>
> 这样吧，**你先拿半箱试卖**，我教你怎么切样品展示。如果卖不动，剩下的我回收，**你零风险**。”（方案）

---

### 总结：让供应商立刻记住并下单的3个关键词

1.  **省事**：免削皮、免催熟、免泡盐水。
2.  **少损耗**：坏果率低、耐放、好运输。
3.  **多赚钱**：利润高、复购多、好展示。

**一句话原则：不要讲“它是什么”，要讲“它能让你的顾客多花钱，让你的损耗少花钱。”**
