中央厨房统一收货通常更优，原因如下：

1. **质量控制**：可集中检验水果的新鲜度、规格、成熟度，确保一致性，减少门店因验货差异导致的损耗。
2. **成本优势**：大宗采购能获得更低单价，且减少单次运输费用。中央厨房可采用专业分选、分切设备，降低加工浪费。
3. **标准化流程**：统一清洗、预加工（如去核、切块）、分装，便于门店直接使用，减少操作差异。
4. **库存管理**：集中库存更易控制周转，避免门店各自囤货造成过期或断货。

**注意事项**：
- 需具备冷链运输能力和中央厨房的加工产能（如冷库、分拣线）。
- 对运输距离较远的门店，可考虑"区域性中央厨房+周边门店直配"模式。
- 若门店销量波动大，可门店自行视销量按日补货（如直送新鲜整果），但需统一采购渠道以维持品质。

综合中小型连锁，建议先试点中央厨房统一收、预处理后分送，再根据损耗率、客户反馈调整。
