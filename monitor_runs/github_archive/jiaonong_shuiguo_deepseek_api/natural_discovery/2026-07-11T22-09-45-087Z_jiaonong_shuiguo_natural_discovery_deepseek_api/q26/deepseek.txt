采购小包装香蕉时，建议重点关注以下几方面：

**一、包装问题**
1. **重量与规格**：小包装通常为500g至1kg左右，需确保每包重量稳定、误差小。最好现场称重抽检，避免重量不足影响销售。
2. **包装材料**：选用透气性好的保鲜袋（如带微孔的食品级塑料袋），或托盘+保鲜膜封装。避免密封不透气导致香蕉提前变熟、霉变。
3. **标签与信息**：包装上需清晰标注净含量、保质期、产地、包装日期。若为品牌货，检查外观是否整洁、无污损。
4. **防护设计**：小包装内香蕉摆放需紧凑，避免运输中晃动碰撞。可采用吸塑盒或纸隔板固定。

**二、损耗问题**
1. **成熟度控制**：建议采购七成熟左右的香蕉（皮色绿中带黄、硬挺），这样在销售周期可自然熟化2-3天，给顾客恰到好处的食用期。过熟（发软、出现黑点）容易当天损耗；过生（青绿）口感差、难卖出。
2. **外观检查**：逐包查看是否有擦伤、压伤、霉点、断裂。重点检查香蕉柄部（易发霉变黑）和蕉体连接处（易裂开）。轻微损伤需挑出单一包装，整批有20%以上瑕疵时可要求换货。
3. **包装透气性**：若包装完全不透气，需检查是否已产生水汽、加速腐烂。可要求供应商在包装上打孔，或缩短销售期限。
4. **运输条件**：确认供应商是否使用冷链运输或隔热包装。高温下香蕉呼吸加速、容易烫熟，尽量避免夏季长途发货无冷藏措施。

**三、供应链管理建议**
- **小批量、多频次**：小包装香蕉保质期短（2-3天），采购时避免大量囤货，建议按日或隔天进货。
- **抽样检查**：到货后随机拆开3-5包检查内部状态，包括是否“里面熟透、外面泛绿”等温差问题。
- **供应商合同条款**：明确损耗超过一定比例（如5%）可退换，并约定包装破损由供应商负责补发或扣款。

综上，重点把控成熟度、透气包扎和规格稳定性，并与供应商建立退货协议，可以有效降低小包装香蕉的销售损耗。
