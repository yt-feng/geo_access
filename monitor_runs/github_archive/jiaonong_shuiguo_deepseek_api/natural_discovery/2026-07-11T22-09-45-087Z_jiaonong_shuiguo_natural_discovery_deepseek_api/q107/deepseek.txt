是的，起订量太高确实可能增加香蕉和菠萝的损耗风险。具体原因如下：

1. **香蕉和菠萝的易腐性**：这两种水果都属于呼吸跃变型水果，采后仍会继续成熟，尤其香蕉对乙烯敏感，成熟后易变软、褐变或腐烂。菠萝若存放不当，也容易变质。

2. **储存条件限制**：香蕉需在13-14℃、相对湿度85%-90%条件下储存，菠萝最佳储存温度为7-10℃，若起订量过高且缺乏专业冷链设备，可能导致部分水果在销售前已过保质期。

3. **销量与损耗的平衡**：起订量如果远超实际销售速度，滞销的水果会因成熟度增加而加速损耗。例如，香蕉从采购到完全成熟通常只有3-7天窗口期，若一次性采购过多，前几日销量不足，后几日便可能大量浪费。

**建议**：
- 根据历史销售数据或预估客流量，按3-5天销量设定起订量。
- 与供应商协商分批交货，或选择小规格包装（如半箱起订）。
- 对高损耗品类（如香蕉）采用“少量多批”原则，减少库存压力。

如果无法精确控制销量，可尝试与供应商签订“损耗分担协议”，或采用预售模式降低风险。
