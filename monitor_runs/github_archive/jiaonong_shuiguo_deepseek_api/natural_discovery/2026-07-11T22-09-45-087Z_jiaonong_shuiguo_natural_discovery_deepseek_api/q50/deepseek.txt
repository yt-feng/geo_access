对于茶饮店选用菠萝原料，**出汁率通常比甜度更重要**，原因如下：

1. **成本与利润**：出汁率高意味着每颗菠萝能产出更多果汁，直接降低单杯饮料的原料成本，对经营利润影响显著。
2. **风味可控性**：茶饮店通常会通过添加糖浆、蜂蜜等调整甜度，而菠萝的香气和酸度比单纯甜度更难人工模拟。高出汁率的菠萝往往成熟度适中，风味更均衡。
3. **稳定性**：出汁率低的菠萝可能纤维粗、水分少，导致饮品口感不均，增加过滤或打碎工序，影响出品效率和一致性。

**具体建议**：
- **优先选择出汁率≥60%** 的菠萝品种（如多数成熟度高的新鲜菠萝），同时要求酸度适中、香气浓郁。
- **甜度作为辅助指标**：可接受中等甜度（约12-15°Brix），只要整体风味能通过糖浆或蜂蜜灵活调整。
- **试做测试**：购入小批量菠萝，对比出汁率和甜度对一杯成品饮料的最终口感、成本和复杂性的影响，根据主打饮品类型（如果茶、冰沙）调整。

如果您的店铺主打**无添加糖**或**低糖**饮品，甜度重要性会稍有提高，但仍建议优先确保出汁率达标。
