通常情况下，超市做水果促销时，供应商是可以提供陈列、海报或试吃支持的，但具体需要根据双方的合作协议和供应商的政策来确定。以下是一些常见的情况和建议：

1. **陈列支持**：供应商通常愿意提供特殊陈列（如堆头、端架等）的设计和货架维护，以提高产品曝光度。建议您直接与供应商沟通，确认是否包含在合同中，或者作为促销合作的一部分。

2. **海报支持**：许多供应商会提供宣传海报、价格标签或其他视觉物料，尤其是大型品牌供应商。您需要提前要求，并明确数量、印刷费用承担方式。

3. **试吃支持**：试吃是水果促销的有效手段，供应商可能会提供样品、试吃人员或试吃设备（如切板、牙签等）。但要注意食品安全要求（如切开的瓜果需要冷藏、符合卫生标准），并确认试吃产生的费用（如损耗补偿）由谁承担。

**执行建议**：
- 先检查现有采购合同中的“市场支持条款”，看是否有明确约定。
- 与供应商代表（如业务经理、促销负责人）直接协商，明确支持内容和时间。
- 对于试吃，建议签订补充协议，明确食品安全责任和损耗补偿方案。
- 如果供应商无法提供支持，您可以考虑与超市共享部分促销成本，或申请超市自有推广资源。

**判断依据**：供应商提供促销支持是行业常见做法，尤其是在新品推广或季节性水果（如夏季西瓜、冬季车厘子）竞争激烈时，但支持力度会受供应商规模、利润空间、合作关系影响。如果您无法核验合同细节，请以书面沟通为准，避免口头承诺。
