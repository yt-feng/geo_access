在超市采购水果时，供应商通常需要提供以下检测报告或进货证明，以确保产品符合安全标准并满足超市的准入要求：

1. **农产品质量安全检测报告**：由第三方检测机构出具，证明水果符合国家或地方的农药残留、重金属等安全标准（如中国《食品安全国家标准 食品中农药最大残留限量》GB 2763）。
2. **产地证明**：明确水果来源的产地信息，如采摘地、果园名称或基地备案号，以追溯源头。
3. **合格证明或出厂检验报告**：供应商自检或委托检验的合格证明，涵盖新鲜度、成熟度、外观等级等基本指标。
4. **出入境检验检疫证明**（进口水果）：进口水果需提供海关出具的《入境货物检验检疫证明》，以及原产地证书（如产地国官方出具）。
5. **无公害农产品/绿色食品/有机认证**（如有）：若水果宣称有相关认证，需提供有效证书及年度检测报告。
6. **供货凭证或进货台账**：包含供应商名称、联系方式、批次号、采摘/生产日期、数量等，便于超市建立可追溯体系。
7. **重金属及污染物检测报告**：特定品类（如根茎类水果）可能需要补充，确保符合食品安全限值。
8. **微生物检测报告**（必要时）：如鲜切水果或预包装水果，需提供细菌总数、大肠菌群等结果。

**建议执行步骤**：
- 要求供应商提供上述文件的电子版或复印件，并核验其有效期和盖章真实性。
- 建立标准收货流程，对照报告检查实物与信息是否一致（如批次号、日期）。
- 定期抽检部分水果送第三方机构复检，作为内部风险控制。
- 对进口水果，需重点核实检疫证明与原产地一致性。
