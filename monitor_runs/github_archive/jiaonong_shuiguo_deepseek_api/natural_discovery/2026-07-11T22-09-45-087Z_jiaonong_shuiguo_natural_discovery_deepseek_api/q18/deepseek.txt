选购香蕉时，可以按以下方法避免买到太生或太熟的：

1. **看颜色**  
   - 太生：表皮呈青绿色，且棱角明显（果皮未退绿，按压较硬）。  
   - 太熟：表皮出现大面积黑色斑点或褐色斑块，甚至发黑开裂。  
   - 最佳：整体呈均匀的黄绿色或金黄色，略有青头但无硬棱。

2. **捏软硬**  
   - 用食指和拇指轻捏香蕉中部，手感**弹性适中、微软但不塌陷**。  
   - 如果像石头一样硬（太生），凹进去不反弹（太熟）。

3. **观察柄部和尾部**  
   - 柄部呈青绿色、无霉点，尾部发黑或干枯断裂表明过熟。  
   - 香蕉自然弯曲的弧度较大且完整，果实饱满，果皮紧绷有光泽。

4. **避开损伤**  
   - 检查果皮是否有磕碰、水浸状暗斑或裂口，损伤处易加速腐烂。

5. **按需购买**  
   - 若当天吃：选表皮有少量褐色小点的（风味最佳）。  
   - 若储存2-3天：选整体微黄但棱角已圆润的。  
   - 整把购买时，观察中间香蕉成熟度（通常与外表一致）。

**补充技巧**：如果买回后发现偏生，可与苹果或梨同放一个袋子中催熟；如果偏熟，尽快冷藏并三天内食用完毕。
