建议优先选择**本地批发**，原因如下：

1. **新鲜度**：香蕉和苹果对新鲜度要求高，本地批发可减少运输时间，确保水果口感。
2. **成本控制**：本地批发免去长途运输费，价格通常更低，且可根据销量灵活采购，避免库存积压。
3. **试错灵活**：初期可小批量试销不同品种（如本地苹果、进口香蕉），根据顾客反馈调整。

**注意**：如果社区超市客流量大且稳定，可同时搭配**稳定供应商**保障长期供应，比如每周固定配送国产红富士苹果和优质香蕉。

**执行建议**：
- 先走访本地水果批发市场，对比3-4家商家的价格和品质。
- 前2周每次采购香蕉和苹果各10-20斤，观察销售速度和损耗率。
- 若日销量超过50斤，再与供应商签订周供协议（含次品退换条款）。
