商超会员日水果活动需要供应商在以下几个关键环节提前配合准备，以保证活动顺利、库存充足、品质稳定：

1. **备货规划与预报**  
   - 提前与商超采购沟通活动销量预估，按“预估销量×1.2-1.5”申请备货（留安全余量）。  
   - 确认活动主打水果品类（如爆款引流品、高毛利品），安排相应品种的优先采收/入库。  
   - 提供采摘成熟度精确控制（如预计到店后1-2天为最佳食用期）。

2. **品质与包装预案**  
   - 制定严于常规的收货品质标准（如糖度、硬度、外观尺寸），并附书面说明给采购。  
   - 提供包装升级方案：小份装（如3-4个苹果一盒）、预包装鲜切混合果盒（高客单引流）、环保礼盒装（适合送礼）。  
   - 配备防震、防碰伤的内衬（如珍珠棉、气柱袋），尤其是易损水果（葡萄、草莓、芒果）。

3. **物流与时效安排**  
   - 确认活动前48小时完成首批到仓（试卖品），活动前24小时全部到齐（避免断货）。  
   - 安排冷链运输（温度控制在2~8℃，特殊水果除外），每车配温度记录仪。  
   - 明确到店时间节点（如开业前2小时送达），避免早高峰卸货拥堵。

4. **营销物料与现场支持**  
   - 提供试吃品（按每店每30分钟消耗量×活动时长×分店数备货）。  
   - 准备统一促销牌、价格标签、堆头陈列设计图（含主推品位置、高度、装饰道具）。  
   - 活动期间安排专职导购员（每店至少1人，高峰时段加派），负责理货、切样、叫卖和客诉处理。

5. **应急与补货预案**  
   - 与物流方签署“3小时内紧急补货”协议，准备小型周转车辆应对突发缺货。  
   - 预留每日可支配库存（如活动总备货量的10%），用于门店间调拨或替代品补充。  
   - 设立24小时品控热线，到店后如有坏果，按合同条款快速启动退换或折算。

6. **结算与数据对接**  
   - 配合商超IT系统同步活动条码、价格调整（避免收银差异）。  
   - 确认结算周期与返利方式（如活动期间让利点数、尾款支付期限）。  
   - 活动结束后3天内提供销售汇总、损耗率、顾客反馈等数据，供复盘参考。

**简单检查清单**：  
- [ ] 销量预估与备货量已确认  
- [ ] 包装与品质标准书面送达  
- [ ] 冷链物流路线与时间表已定  
- [ ] 试吃品、促销物料充足  
- [ ] 应急补货机制已启动  
- [ ] 导购人员排班表及培训完成  

按此提前准备，能有效降低现场风险，让水果活动利润最大化。
