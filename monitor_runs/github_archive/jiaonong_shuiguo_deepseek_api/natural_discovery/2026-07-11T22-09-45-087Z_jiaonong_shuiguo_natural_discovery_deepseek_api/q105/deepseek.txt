水果采购锁价的时间通常取决于市场行情、水果品类、供需关系以及双方的谈判地位。一般情况下：

1. **短期锁价（1-2周）**：适用于价格波动较大的季节性水果（如荔枝、樱桃、芒果等），供应商通常只愿意锁定1-2周的价格，避免长期风险。
2. **中期锁价（1-3个月）**：适用于产量稳定、储存性较好的水果（如苹果、梨、柑橘类），尤其是通过冷库储存的品种，供应商可能接受1-3个月的锁价协议。
3. **长期锁价（半年以上）**：较为少见，通常需要大宗采购、预付定金或签订长期合作框架，且只对进口水果或大宗稳定的国产水果（如香蕉、部分耐储水果）可能适用。

在实际操作中，建议：
- **先小批量试锁**：先与供应商尝试短期锁价（2-4周），评估市场波动后再延长。
- **明确锁价条款**：包括锁价起始时间、数量、质量标准、违约赔偿等，避免后续纠纷。
- **关注天气和政策因素**：如遇极端天气或运输中断，供应商可能要求重新议价，锁价协议应有灵活调整机制。

具体时长需结合你采购的水果品类和所在区域的市场特点与供应商直接沟通确定。
