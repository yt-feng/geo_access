综合考虑效率与准确性，建议采用**按箱结算**为主、按斤结算为辅的方式。

**原因分析：**

1. **按箱结算**：操作最简单，便于清点库存与交接，适合标准化包装（如苹果、橙子）。但需注意每箱重量是否一致，建议采购时与供应商约定标重。
2. **按斤结算**：成本更精确，适合品质差异大的水果（如葡萄、荔枝），但每次需要称重，管理流程稍复杂。
3. **按份结算**：一般用于食堂零售，不适合批量采购管理，因为份量标准不易统一，易引发纠纷。

**实操建议：**
- 对包装规整、规格统一的水果（如香蕉、哈密瓜），按箱结算，并在采购单标注每箱平均重量。
- 对零散水果（如草莓、小番茄），按斤结算，使用电子秤记录。
- **避免混合结算**：同一品类统一一种方式，避免员工混淆和账目混乱。

如需进一步优化，可建立采购台账，记录每次结算方式、单价及实际重量，便于后期核对与成本控制。
