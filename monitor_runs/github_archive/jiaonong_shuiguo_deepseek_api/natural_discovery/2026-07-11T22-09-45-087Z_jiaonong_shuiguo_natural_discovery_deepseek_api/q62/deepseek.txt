判断香蕉批发的熟度和坏果风险，主要依赖**感官检查**和**存储条件评估**。以下是具体可执行的步骤：

1. **观察果皮颜色（熟度判断）**
   - **全青**：未熟，运输后需人工催熟或自然存放3-5天。
   - **黄绿或黄中带绿柄**：最理想的批发状态，耐储存，适合短期分销。
   - **全黄**：已熟透，需1-2天内售出，风险高。
   - **出现黑色斑点/大片褐变**：过熟或受过冷害（低于13℃），口感可能软烂，不能留货。

2. **检查柄部与果实连接处**
   - **柄部干燥、有轻微裂痕**：正常采收表现。
   - **柄部发黑、湿软**：已开始腐烂，整把香蕉可能被感染。
   - **轻轻捏果实中部**：应**有弹性**（能回弹），若**发软**则内部可能已坏或开始发酵。
   - **碰触尖部**：**发黑或软烂**代表霉斑起点，若尖部新鲜紧实则风险低。

3. **检查关键坏果迹象**
   - **捏到局部发软或流水**：可能有**炭疽病**（果皮黑斑向内扩散）或**细菌性软腐**，全批应拒收。
   - **闻果柄或根部**：若有**酸味、酒味、霉味**，说明在运输中受热或受损，已内腐。
   - **取3-5根拿掉包装袋**，观察是否出现**白色或灰色毛状霉**（通常发生在尾部或伤口处），有则坏果率高。

4. **评估存储与运输历史**
   - **摸包装袋内部**：若**潮湿或滴水**，说明已在高温高湿下存放过久，即使外观暂时新鲜，也极容易在1-2天内爆发软烂。
   - **询问运输温度**：香蕉最适温度为13-16℃，若货车温度高于18℃持续超2天，坏果率会急剧上升；低于11℃会出现冷害（果皮变灰、不能正常转黄）。
   - **检查果箱堆叠方式**：若箱底被压凹陷或有液体渗出，底层香蕉很可能已经烂了。

**快速决策表（可当场判断）：**
- **可收**：柄干、果身青绿有弹性、无霉斑、无酸味。
- **需打折或只收部分**：有少量局部软点（<5%），或柄部微潮但果身硬。
- **直接拒收**：多处发软、酸酒味、果柄黑湿、内部有霉。

**行动建议**：拿货后，立刻开箱随机抽检**上中下层各2把**。若抽检合格率低于90%，建议整批退货或要求降价至少30%覆盖风险。
