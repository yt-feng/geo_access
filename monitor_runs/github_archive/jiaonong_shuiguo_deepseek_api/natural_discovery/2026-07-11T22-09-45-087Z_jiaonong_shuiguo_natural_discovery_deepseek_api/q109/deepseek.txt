针对长期合作的水果供应商，**配送条款**是最值得优先谈判的重点，其次是**账期**，最后是**售后**。具体原因和谈判要点如下：

**1. 优先谈判：配送（直接决定营业额与损耗）**
- **原因**：水果保质期短、易损耗，配送的及时性、频率和稳定性直接影响你的库存损耗、销售周转和顾客满意度。延迟或损坏可能导致无法挽回的损失。
- **谈判要点**：
  - 明确**配送时间窗口**（如固定每天清晨或提前预约），并约定延迟的违约金（如延迟超过1小时可扣减货款）。
  - 要求**免费补货或换货**：对运输途中造成的磕碰、腐烂（非你方存放导致），供应商需无条件补发。
  - 争取**多批次小批量配送**：尤其在旺季，避免一次性大量压货。

**2. 次优先：账期（影响现金流）**
- **原因**：账期关乎你的资金压力，但水果生意利润薄、周转快。若对方要求现结或短账期，可能加大你的现金流风险；反之过长账期可能让供应商提高单价（羊毛出在羊身上）。
- **谈判要点**：
  - 争取**7-15天账期**（行业常见），避免周结或月结（除非对方主动提出更低价格）。
  - 若对方坚持现结，可要求**价格折扣**（如现结价低于月结价5%）。
  - 绑定销售对赌：承诺稳定采购量（如每月固定金额），换取更长账期。

**3. 最后考虑：售后（可量化但优先级低）**
- **原因**：售后条款如“烂果包赔”属于行业底线，但通常采购方比供应商更在意验收环节。若配送和账期谈定，售后只需确保基本条款即可。
- **谈判要点**：
  - 明确**验收规则**：到货后多久内可反馈问题（如2小时内），并约定赔偿比例（如超过5%的损耗，全数免费补发）。
  - 要求供应商提供**检测报告**（尤其进口水果），避免农药残留或品种不符的争议。

**总结谈判策略**：
- **第一步**：用配送条款锁定“库存周转零风险”，如“每日配送+免费换损”。
- **第二步**：用账期锁定“现金流缓冲”，如“15天账期+最低起订量”。
- **第三步**：用售后条款兜底，但不要为此牺牲前两者。

执行时，可先试探供应商对配送的接受度（如“您能每天上午10点前送到吗？”，若对方犹豫，则用其他条款让步）。同时，注意核查对方是否已有现成的配送和售后制度，避免花时间谈判对方本已具备的标准流程。
