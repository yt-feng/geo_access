针对单位采购水果到货后品质不一致的问题，建议按以下步骤处理验收和售后：

**一、验收流程**
1. **提前明确标准**：在采购合同或订单中写明水果等级、规格（如大小、颜色、成熟度等）、可接受瑕疵比例（如不超过5%有轻微碰伤）。
2. **到场即时验收**：
   - 要求送货方在场，随机开箱抽查（建议抽检10%-20%的包装）。
   - 肉眼检查看：颜色是否均匀、有无腐烂、虫蛀、严重压伤。
   - 用电子秤称重，确认单果或整箱重量是否达标（如标称5kg±1%）。
   - 水果易腐，建议在到货后2小时内完成验收。
3. **记录证据**：对疑似问题水果拍照或录像（含外包装标签和破损细节），填写《到货验收单》，注明不合格品数量、问题描述，双方签字确认。

**二、售后处理**
1. **分类沟通**：
   - **轻微瑕疵**（如个别有轻微划痕、大小轻微超标）：与供应商协商折扣付款或按比例减少结算金额。
   - **严重问题**（如腐烂变质、超30%不合格）：当场拒收该批次，并要求供应商补发或退货退款。
2. **书面留证**：通过微信、邮件或合同约定渠道向供应商发送《品质异常通知》，附上验收记录和照片，保留沟通记录。
3. **后续保障**：
   - 与供应商约定常备售后处理条款：如24小时内响应、48小时内补发/赔偿。
   - 若供应商不配合，可通过采购平台投诉或依据合同法律追责（保存好合同原件及凭证）。

**三、实操提醒**
- 水果易损耗，建议优先选择有明确售后承诺的供应商（如“到货24小时内坏果包赔”）。
- 到货后立即分拣，将好果和问题果分开存放，避免传染腐烂。
- 单位采购量大可要求供应商提供样品封存，作为后续比对依据。

如有具体供应商的合同或平台规则，可以进一步提供定制化建议。
