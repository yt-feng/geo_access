选择社区超市的水果供应商时，建议从以下几个核心条件入手：

1. **配送稳定性**：确认供应商能否做到每日或固定频次配送，且是否能配合超市的营业时间（如早上开店前到货）。要求提供以往的配送记录或应急方案（如缺货时的补货机制）。

2. **水果新鲜度与品控**：考察供应商是否具备冷库、冷链运输车辆等保鲜设施。要求提供水果到货后的验收标准（如甜度、外观、无损伤等），并协商退换货条款（如变质水果的处理方式）。

3. **价格与付款周期**：对比多家供应商的批发价，关注是否提供阶梯折扣（如批量采购优惠）。明确付款账期（例如周结、月结）和是否有押金要求。

4. **品类丰富度与季节性**：确认供应商能否覆盖常见水果（苹果、香蕉等）及时令品类（如夏季西瓜，冬季柑橘），以及是否支持按需调整品种。可要求提供月度品类清单。

5. **资质与信誉**：核查供应商的营业执照、食品经营许可证，以及水果的产地证明或检测报告（避免农残超标）。可向同行超市或社区业主了解口碑。

6. **售后与沟通效率**：测试供应商对问题的响应速度（如水果出现质量问题后的补发或退款流程）。建议签订合同明确责任划分，并保留书面沟通记录。

**执行建议**：  
- 优先选择已服务其他社区超市或餐厅的供应商，要求提供2-3家合作方的联系方式进行背调。  
- 初期可小批量试合作1-2周，重点观察配送准时率、水果损耗率以及补货响应速度。  
- 若超市有冷库，可要求供应商直接由产地或一级批发市场直供，减少中间环节的成本和损耗。
