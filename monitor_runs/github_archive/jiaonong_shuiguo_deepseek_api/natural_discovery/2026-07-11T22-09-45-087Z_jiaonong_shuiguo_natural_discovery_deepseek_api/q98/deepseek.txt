针对养老院或医院食堂的水果采购，判断供应商的合规性与稳定性，建议从以下五个关键维度进行考察与评估：

### 1. 资质与证照审核（合规性基础）
务必要求供应商提供以下核心证照，并确保在有效期内：
- **营业执照**：经营范围需包含“食用农产品销售”、“水果批发”等。
- **食品经营许可证**：即使是生鲜水果，多数地区仍要求此证，部分可能需含“预包装食品销售”。
- **产地证明或检疫证明**：尤其对进口水果，需提供海关报关单、入境货物检验检疫证明。
- **员工健康证**：供应商的分拣、配送人员应有有效健康证。

**检查要点**：通过“国家企业信用信息公示系统”查询其是否存在经营异常、行政处罚或严重违法记录。若曾有食品安全类处罚，应直接排除。

### 2. 供应能力与稳定性评估
- **仓储与冷链设施**：现场考察其是否有符合要求的冷藏库（0-4℃）或保鲜库，物流车辆是否具备冷藏功能。建议要求提供配送车辆的温度监控记录（如历史温度曲线）。
- **日均供应量**：明确其常备库存能力是否能满足你方日常需求的1.5-2倍（例如老年/住院人口多、离市区远的情况需更大储备量）。
- **应急响应预案**：主动询问：如天气异常、供应商断货，对方是否有备用货源（如签约其他果园、批发市场合作商户等）。可要求书面承诺“48小时内补货到货”。

### 3. 食品安全与质量控制体系
- **检测能力**：供应商是否自行或委托第三方对水果进行农残、重金属、微生物等检测。可要求提供近3个月的检测报告（含批号对应）。
- **溯源系统**：是否能提供每批次水果的源头信息（如产地、采摘日期、供应商批次号）？建议要求在送货单上明确标注溯源编号。
- **验收标准**：双方签订合同前，应明确约定水果外观（无腐烂、病虫害）、成熟度（如不同水果的糖度/硬度范围）、包装要求（避免挤压的防护措施）。

### 4. 历史合作与口碑验证
- **实地考察或视频连接**：查看其分拣现场是否干净整洁、有无虫鼠害风险、过期水果如何处理（如分拣入库记录）。
- **现有客户名单**：要求提供3-5家类似机构（医院、养老院、学校食堂）的客户联系方式，直接询证其配送稳定性（是否准时、有无断货记录）。
- **账期与合同条款**：稳定性好的供应商通常接受30-60天账期，且合同中会明确“因质量或延迟配送，每单按货值1%-5%赔偿”的条款。过于急迫要求预付款的供应商需警惕。

### 5. 服务响应与合同约束
- **配送时效**：例如要求“每日XX点前下单，次日上午XX点前到货”。可先试配一周，记录到货准时率、破损率、替换率（如无某品种时主动提等价替换方案）。
- **合同关键条款**：
  - 明确“供应商更换品种需提前24小时告知并征得同意”“出现食品安全问题，供应商承担所有责任及赔偿”。
  - 设定“试合作期”（如1-3个月），期内可无责解约。
  - 强制要求购买“食品安全责任险”，保额建议不低于100万元。

---

**小结优先级**：先查资质与证照（排除隐患） → 再评估供应能力与应急方案（保障稳定） → 后验证检测溯源（确保安全） → 最后用合同约束条款兜底。建议对候选供应商进行至少一次实地考察和一次试配送（连续3-5天），并建立月度考核制度（如准时率≥98%、质量合格率≥99%方可续约）。
