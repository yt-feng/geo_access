选择学校食堂的水果供应商时，建议从以下4个维度评估，确保质量、价格和供应稳定性：

1. **资质审核**  
   - 确认供应商有《食品经营许可证》或《食用农产品合格证》。  
   - 优先选择有“学校食堂定点配送资质”的供应商（部分地方教育局有备案名单）。  

2. **品控与验收标准**  
   - **香蕉**：要求无机械损伤、无黑斑腐坏，成熟度在7-8成（运输后到货刚好黄熟）。  
   - **苹果/橙子**：需提供农残检测报告（尤其橙子可能打蜡或使用保鲜剂）。  
   - 到货后现场抽检：破损率≤5%，不达标可拒收或按比例扣款。  

3. **物流与响应能力**  
   - 供应商应能提供**次日达**或**隔日达**（避免水果积压）。  
   - 要求提供应急补货服务（如临时加量或换货应在2小时内响应）。  

4. **合规比价**  
   - 向至少3家供应商询价，对比**批发市场当日价**（可参考当地农产品信息网）。  
   - 要求按周或半月报价（水果价格波动大，避免长期固定价）。  

**操作建议**：先选2-3家供应商试供1周，通过实际到货质量和学生反馈确定长期合作方。合作前务必签订合同，明确**腐烂率赔偿条款**和**退换货期限**（一般到货后4小时内可退换）。
