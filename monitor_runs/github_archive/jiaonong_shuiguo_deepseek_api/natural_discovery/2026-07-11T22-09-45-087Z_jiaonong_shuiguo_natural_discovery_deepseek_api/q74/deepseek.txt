针对批发档口拓展餐饮客户（如奶茶店、果汁店、甜品店、烧烤店等），**菠萝和香蕉**的规格建议如下，重点考虑**出品损耗、成本控制和使用便利性**：

### 🍍 菠萝规格建议
餐饮店通常需要标准化、易加工的果肉或切片，因此**定制化半成品**比整果更受欢迎。

1.  **带皮整果**
    -   **规格**：单个重约 **1.5-2斤**（太小的果肉少，太大的运输易损）。
    -   **适用客户**：中高端果汁店、创意菜品餐饮（需现切展示）。
    -   **注意**：提供去除头尾、根部处理服务（减少餐厅后厨工作量）。

2.  **去冠/切块（核心竞争力）**
    -   **规格**：去掉叶子与硬芯，直接切成 **2cm×2cm 方形果肉块**（标准果汁果肉大小）或 **长条型**（用于配菜/烧烤）。
    -   **包装**：真空袋（防氧化）或 **1.5-2公斤/袋** 的冷冻包装（保存期长至数月）。
    -   **利润点**：加工后单价可提高 20-30%，且客户粘性高。

3.  **特殊场景**
    -   **烧烤/锅物料理**：准备 **去皮厚片（0.8-1cm厚）**，适合炙烤。
    -   **饮品店**：可提供 **1公斤装冷冻菠萝丁**（糖水浸泡或未调味），用于气泡水、冰沙。

---

### 🍌 香蕉规格建议
香蕉极易熟化、碰伤，所以**按成熟度分级供货**是核心。

1.  **熟度明确的整把**
    -   **符合标准**：**7-8成熟**（青皮微黄，果肉硬实，室温3-5天可自然成熟）。
    -   **适用客户**：奶茶店（需现打泥）、甜品店（需摆盘）、高端烘焙。
    -   **包装**：纸箱或网袋包装（不可密封，需透气孔），**净重净重6-9公斤/箱**（行业标准进货量）。

2.  **催熟服务（增值服务）**
    -   **规格**：提供 **快速催熟服务**，按“当天出/明天出”分装。
    -   **适用客户**：街边小摊、快餐店（急需使用）。
    -   **注意**：催熟后存放时间极短（2-3天），需客户承诺次日取货，否则退换货风险高。

3.  **冷冻半成品（利润区）**
    -   **规格**：**去皮冷冻香蕉段**（每段约50g），真空冷冻包装，**1kg/袋**。
    -   **优点**：甜度锁定（无需糖浆），用于奶昔、冰沙、冷冻甜品。
    -   **价格**：毛利较高，适合高端连锁。

---

### 📦 针对餐饮客户的核心经营建议

1.  **提供“免洗免切”选项**
    -   餐饮店后厨用工成本高，**切片/切块/真空包装**的菠萝和去皮的冷冻香蕉，远比整果好卖。建议准备不同规格的 **“净菜式水果”**。

2.  **按“斤”不按“个”报价**
    -   整果按 **斤**定价（更透明），切块/冷冻品按 **公斤** 报价。避免客户因单果重量差异投诉。

3.  **建立“两档定价”体系**
    -   **基础档**：整果（批发价，引流）。
    -   **加工档**：切块/去皮代客加工（增收，锁定大客户）。

4.  **测试“小规格”样品**
    -   给潜在餐饮客户（特别是小店）送 **0.5-1斤** 的小样，加上成品照片（茶饮、甜品示例），比纯报价有效10倍。

**关键总结**：
-   **菠萝**：主推 **1.5斤/个** 的整果，增值款做 **2cm果肉块真空装**。
-   **香蕉**：主推 **8成熟整把（6-9kg箱）**，重点推 **冷冻去皮香蕉段（1kg/袋）** 作为差异化爆品。

（注：以上规格基于当前一线餐饮水果供应链标准，可根据你所在城市及客户反馈微调包装量。）
