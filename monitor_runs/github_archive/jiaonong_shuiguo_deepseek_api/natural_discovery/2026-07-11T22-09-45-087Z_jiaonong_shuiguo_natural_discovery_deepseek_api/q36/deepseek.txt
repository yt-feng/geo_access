商超采购菠萝时，选择卖整果还是切好的，需根据门店实际情况、目标客群和损耗控制综合决定。以下是具体分析供参考：

### 1. **优先推荐：整果与切果组合销售**
   - **整果优势**：适合家庭采购、看重性价比的顾客（通常单价更低）；保存时间较长，门店操作简单，损耗易控制。
   - **切果优势**：满足即食需求（年轻上班族、单人消费）；避免顾客处理菠萝的麻烦（削皮、去眼），提升购买体验；单价更高，可增加客单价。

### 2. **关键决策因素**
   - **门店位置与客群**：
     - **社区店/家庭客群为主**：整果占比可更高（60%-70%），搭配少量切果。
     - **商圈/写字楼/交通枢纽**：切果占比提升（50%以上），尤其是小份装（半果或果切盒）。
   - **门店加工能力**：
     - 有后场或熟练员工：可每日现切，保证新鲜度和卖相。
     - 无加工条件：优先销售整果，或选择预包装切果（如冷链配送）。
   - **损耗控制**：
     - 整果损耗约5%-10%（表皮损伤、过熟）。
     - 切果损耗可达20%-30%（氧化变色、出水、剩余果皮处理）。建议切果当天或半天内售完，避免隔夜。

### 3. **实操建议**
   - **整果销售**：
     - 标注新鲜度和成熟度（如“现摘/甜度达标”），提供削皮服务（额外收费或搭配简易工具）。
     - 按大小分级定价（大果适合送礼，小果家庭尝鲜）。
   - **切果销售**：
     - 用透明盒封装，搭配冷藏展示（温度4-6℃），标明当天日期。
     - 推出组合装（如半果+整果切块），或“买整果免费削皮”引流到切果区域。
   - **数据试跑**：先试卖一周切果（如100份/天），记录销量、损耗和顾客反馈；计算切果毛利率是否高于整果（通常切果毛利高3%-5%）。

### 4. **注意事项**
   - 切果不可过度浸泡盐水（会变软），建议销售时附赠盐包说明。
   - 整果需定期翻动检查，及时处理烂果（避免霉变传染）。
   - 若当地季节性菠萝上市（如4-6月），可主推整果；非旺季时用切果维持周转。

**结论**：建议整果与切果按6:4或5:5比例搭配，优先保证切果当日现切、冷藏销售，整果作为基础款。灵活根据每日销量调整比例（如周一至周五切果需求高）。
