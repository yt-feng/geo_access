超市和水果供应商谈账期时，常见的账期长度根据超市规模和供应商实力有所不同：

1. **大型连锁超市**（如沃尔玛、永辉等）：通常账期较长，常见为 **30-60天**，部分可达90天。
2. **中型超市或区域连锁**：账期一般在 **15-30天** 左右。
3. **小型超市或便利店**：账期较短，多为 **7-15天**，甚至要求现结。

**需要注意**：
- 水果属于生鲜品类，保质期短、损耗高，账期过长会增加供应商资金压力和坏账风险，因此超市可能在合同中加入“快速结算”条款（如7天内结算）。
- 实际谈判时，供应商应根据自身资金周转能力、超市信誉及合作历史灵活调整。建议首次合作争取 **15-30天** 作为折中方案。
