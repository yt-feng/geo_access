针对批发商担心货到后损耗高的问题，建议在合同中明确以下条款以界定坏果责任：

1. **损耗率标准**：明确约定运输和到货后的合理损耗率（例如“到货时坏果率不超过3%视为正常损耗，由买方承担；超过部分由卖方负责补发或退款”）。

2. **验货时限与方式**：规定买方在收到货物后24小时内（或合理时间内）完成验收，并采用拍照、视频等方式记录坏果情况，逾期视为默认合格。

3. **责任归属**：明确运输途中损坏的认定标准（如因包装不当或虚假保鲜导致，由卖方担责；因物流不当导致，由买方与物流方协商或约定卖方代发货需承担物流选任责任）。

4. **赔偿方式**：约定超标的处理方案，例如“按实际坏果金额的110%退款”或“等值补发新鲜水果”。

5. **争议举证**：约定以买方留存证据（如开箱视频、第三方质检报告）为准，同时要求卖方提供发货前的质检或录像记录。

6. **特殊说明**：若为长距离运输，可增加“若因不可抗力导致延迟到货并引发变质，双方按比例分担损失”等条款。

建议所有条款需具体化、可量化，避免模糊表述。若金额较大，可考虑请律师审核合同草案。
