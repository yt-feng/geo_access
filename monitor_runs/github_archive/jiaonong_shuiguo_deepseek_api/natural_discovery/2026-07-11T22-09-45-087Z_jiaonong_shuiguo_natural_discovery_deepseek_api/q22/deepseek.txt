可以针对不同消费心理设计以下几组**高客单价组合促销方案**：

### 方案一：健康轻食组合（提升整体单价）
- **组合内容**：**1串香蕉 + 1个菠萝 + 3个苹果**（或其他固定规格）
- **定价逻辑**：三者单价通常较低，组合后总价可能约20-35元。直接定一个**比单买总和低10%-15%**的价格，比如单买需30元，组合价25.8元。
- **适用场景**：标注“轻食能量包”、“早餐水果组”。菠萝可切好装盒，香蕉选熟度适中的，苹果选脆甜多汁的。
- **操作要点**：用透明托盘或专用手提纸袋打包，放在水果区主通道端头。

### 方案二：自制饮品组合（拉高毛利）
- **组合内容**：**2根成熟香蕉 + 半个菠萝 + 2个红富士苹果** → 顾客购买后搭配**即食燕麦片**或**酸奶**（放在旁边关联陈列）。
- **定价逻辑**：组合价可能为28-38元，比单买水果本身贵一些，但附赠一包10元级燕麦片或一盒酸奶，相当于把饮品原料组合成“自制奶昔套餐”。
- **适用对象**：有自制思慕雪、果汁需求的年轻顾客、家庭。苹果打底、香蕉增稠、菠萝提香。
- **操作要点**：组合内水果选大果（提升单价），旁边立牌写“+5元可换购1L鲜牛奶”，进一步拉升客单价。

### 方案三：多口味囤货组合（深度承包需求）
- **组合内容**：**1根香蕉（一次性品尝装）+ 1个菠萝 + 2斤苹果**（按斤称重），或者**大包装**。
- **定价逻辑**：用“单斤价”或“封顶价”拉高直觉。比如香蕉单根1.5元、菠萝15元、苹果每斤8元，组合标价“总价不超过40元”，实际按称重结算。可以利用 **“满30元减5元”** 或 **“加1元多1个苹果”** 等手段。
- **适用场景**：面向家庭用户。顾客原本可能只买一两样，组合装提醒他“今天买齐三样，未来3天水果不用愁”。
- **操作要点**：用大号网袋或统一规格的盒子，组合名称叫“全家福水果篮”，价格写在显眼绿色价签上。

### 方案四：场景化“今日特惠盘”（体验式）
- **组合内容**：**已切好的菠萝块 + 苹果片 + 香蕉段**（泡过淡盐水防变色，装在透明杯中）→ 加上一根竹签或小叉子。
- **定价逻辑**：单买整果需30-40元，而这种混合切盒标价**18-22元**——看似单价低，但实际剔除了皮、核、损耗，顾客直接拿到的是可食用部分。同时可搭配**“加5元换购2元酸奶”**，总消费从几元推高到20-30元。
- **适用对象**：街边、写字楼附近门店；午休时段或下班高峰。
- **操作要点**：杯子容量300-400ml，水果按“黄（香蕉）+金（菠萝）+红（苹果）”搭配，视觉鲜艳，用冷柜陈列。

### 核心销售话术（员工用）：
- 对犹豫顾客：“现在买这三个一起，比单买省6块9，够再做一次奶昔了。”
- 对只看香蕉的顾客：“带个菠萝和苹果回去，放一起能互相催熟，香蕉也不易坏。”
- 对只看贵菠萝的顾客：“今天香蕉和苹果做超低价活动，凑单买这个菠萝能再减3元。”

### 风险控制（必备）：
- 香蕉极易碰伤、压坏，组合包装内使用**防压纸格**或放在最上层。
- 苹果表皮脆、易磕碰，与菠萝同装时要隔开（菠萝带刺）。
- 菠萝或香蕉未熟透时，**不**放进切盒类组合，避免顾客差评。

选择**方案一或方案三**起步成功率较高，先用固定组合拉高平均客单，后续可切换到方案二做关联和方案四做高毛利试吃转化。
