针对公司下午茶水果批量配送的供应商选择，建议从以下几个维度进行考察和筛选：

1. **配送能力与范围**  
   - 确认供应商是否覆盖贵司所在区域及写字楼，能否保证指定时段（如下午2-3点）送达。  
   - 询问最低起送量、配送频率（如每日/隔日），以及是否支持分开配送（如不同楼层、不同工区）。

2. **水果品质与供应稳定性**  
   - 要求供应商提供《采购清单》及近期样品照片或小样试吃，重点检查水果新鲜度、成熟度、大小均匀度。  
   - 了解其货源渠道（如产地直采/批发市场/品牌代理商），是否能提供时令品种替换方案（如根据季节主动建议当季水果）。

3. **分拣与包装标准**  
   - 确认是否提供“免洗、即食”服务（如切好的西瓜、剥好的柚子等）。  
   - 要求使用防震包装（如独立网套、泡沫隔板），避免配送过程中碰撞损伤。

4. **价格与账期**  
   - 要求提供《批量报价表》，对比批发市场价格（如当地果品市场官网价格）。  
   - 明确是否包邮、是否含税，账期是否支持月结（通常30-45天），避免频繁对账打扰。

5. **售后与应急响应**  
   - 确认水果破损、腐烂或品质不达标的处理方式（如24小时内免费补发/退款）。  
   - 考察供应商是否提供“当天加急补送”服务，应对临时增加人数或替换水果的需求。

**实操建议**  
- **试单测试**：先下单1-2次正式配送，观察水果品质、配送时间、包装结实程度。  
- **签约前核查**：要求供应商提供《食品经营许可证》及最近一期第三方检测报告（重点关注农药残留检测）。  
- **备用方案**：同时联系2-3家供应商（如本地知名果品公司、生鲜电商企业客户部），避免唯一依赖。

**注意**：避免单纯以“价格最低”作为唯一标准，重点考察配送准时率（若超时影响下午茶正常发放）和售后响应速度。
