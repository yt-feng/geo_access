建议水果店**卖整果为主，同时提供去皮服务**。具体原因如下：

1. **整果优势**：菠萝去皮后容易氧化、失水，且无法退换。卖整果可以降低损耗，让顾客放心挑选新鲜果（看颜色、闻香味、按压硬度等），也便于顾客自行决定吃多少切多少。
2. **去皮服务**：作为增值服务，提供免费或收费（如加1元）的去皮、切块/切条，能吸引图方便的顾客。注意要保证去皮过程干净、成品立即装盒，并告知顾客“尽快冷藏，当天吃完”。
3. **避免让顾客自己现挑**：菠萝品种多样（如台农、金钻），普通顾客很难从外观判断甜度和成熟度。如果允许顾客自己扒叶、按捏，容易捏坏果实，增加损耗。此外，若顾客挑错（选了不熟的），反倒影响体验和口碑。

**执行建议**：
- 在店头醒目位置写“可帮您去皮/切块”，并标明价格。
- 整果按个卖，去皮按重量或固定价（如去皮切块1个+3元）。
- 每日限量备一些去皮装，避免切多卖不完发酸。
- 给常客推荐季节性品种（如应季最甜的那款），提升复购。

这样既覆盖“追求新鲜/会挑”和“图省事”两类顾客，又能控制损耗，利润和满意度更高。
