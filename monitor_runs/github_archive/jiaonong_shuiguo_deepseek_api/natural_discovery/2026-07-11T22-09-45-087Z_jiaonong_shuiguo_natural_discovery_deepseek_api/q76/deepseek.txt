针对生鲜电商配送香蕉时出现的压坏和变黑问题，以下是具体可执行的控制方案：

**1. 采收与预处理阶段（源头控制）**
- **成熟度选择**：只发7-8成熟的青香蕉（果棱明显，表面青绿），避免发黄熟果。到消费者手中时正好转为可食用状态。
- **清洗与抑菌**：用0.5%-1%的柠檬酸或抗坏血酸溶液浸泡1-2分钟，抑制氧化酶活性，减少配送中切口和摩擦处变黑。
- **冷激处理**：包装前用10-12℃冷水浸泡2-3分钟，降低果温和代谢速率。

**2. 包装防护（防压与防撞）**
- **内包装**：每根香蕉单独套入**珍珠棉网套**或**特制气柱袋**，减少果间硬接触。
- **隔层设计**：在包装盒底铺一层**瓦楞纸波浪隔板**或**EVA泡沫垫孔板**，将香蕉按弓背朝上放入卡槽中，果柄朝同侧避免碰撞。
- **固定填充**：用充气袋、碎纸团或气泡膜填满盒内空隙，防止运输中晃动。香蕉顶部加一层薄海绵盖，避免上层堆积压力。
- **推荐方案**：使用**标准7-8根香蕉专用瓦楞纸盒**（内有珍珠棉凹槽），每根独立固定。

**3. 温控与气体管理（防变黑）**
- **冷链运输**：全程保持12-15℃（不可低于10℃，否则冷害导致果皮发暗；不可高于20℃）。使用蓄冷冰袋（隔层包裹，不直接接触果皮）或冷藏车。
- **气调包装**：包装内放入**乙烯吸收剂小包**或**活性炭包**，吸收香蕉释放的催熟气体（乙烯），减慢后熟速度。或用**微孔保鲜膜**（如PE膜扎6-8个针孔）替代完全密封袋，保持气体交换。

**4. 分拣与装箱操作**
- **包装前分拣**：剔除已经出现斑点、软腐或果柄断裂的个体。
- **单层摆放**：禁止叠放两层香蕉，必须单层平铺，且果柄不能压在果身上。
- **标签避位**：贴标签或价格贴在果柄或包装外壳，避免直接贴在果皮上，防止胶带导致接触变黑。

**5. 配送端管理**
- **轻拿轻放提示**：在包装盒外部显著位置标注：“易损水果，请勿倒置、叠压”并印上香蕉形象标识。
- **分时段配送**：将香蕉安排在早上或傍晚温度较低的时段配送，避免中午高温。
- **配送工具缓冲**：电动自行车或货车配送箱内垫5cm厚珍珠棉垫，减少路面颠簸传递。

**6. 消费者收果建议（附带卡片）**
- 在包裹内附提示卡：“香蕉收到后请立即打开包装，常温存放（18-22℃最佳），不要与苹果、梨放在一起，食用前勿清洗。如果发现个别黄透，请优先吃掉。”并给万一压坏或变黑的售后处理通道。

- [x] 使用防撞网套+固定凹槽纸盒

- [x] 包裹外贴易损提示
