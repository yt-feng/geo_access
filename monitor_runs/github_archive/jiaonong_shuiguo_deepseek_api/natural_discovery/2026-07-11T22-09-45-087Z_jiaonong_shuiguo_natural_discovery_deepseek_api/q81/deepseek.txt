应该支持。

理由是：

1. **用户需求驱动**：即时零售主打“半小时到一小时达”，用户下单通常按“个”“份”买，而非成箱购买。供应商若能支持小批量高频补货，可以更快响应前端订单，减少缺货。

2. **降低库存风险**：水果属于生鲜，保质期短、损耗率高。小批量高频补货可以让供应商和平台都减少积压库存，减少因卖不完造成的腐烂损失。

3. **提升资金周转**：对供应商来说，虽然单次订单金额变小，但补货频率更高，总销售额可能并不下降，且回款周期更短，有助于现金流健康。

4. **竞争需要**：即时零售平台上，其他供应商如果支持灵活补货，而你只做大批量低频率，会导致平台更倾向选择配合度高的供应商，你的曝光和订单会被挤压。

建议的操作方式：
- 与平台沟通设定**最低起送量门槛**，比如最低两箱或10公斤，保证物流成本可控。
- 约定**补货前置时间**，例如提前4小时或6小时下单，确保仓库配货顺畅。
- 对高频补货订单可以适当**加收小额物流服务费**，或设定阶梯价格（量大价优，小量适当加价），在灵活与利润之间取得平衡。
