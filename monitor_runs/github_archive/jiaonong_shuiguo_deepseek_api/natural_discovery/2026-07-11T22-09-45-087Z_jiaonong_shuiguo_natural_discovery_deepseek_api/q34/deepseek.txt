在商超采购水果时，**损耗低通常比价格低对最终毛利的影响更大**。原因如下：

1. **损耗直接减少可售商品**：水果易腐烂、碰伤，损耗率可能高达5%-20%。即使进价低，若损耗高，实际可售数量减少，单位成本会上升，毛利被吞噬。

2. **毛利公式拆解**：
   - 毛利 = 销售额 - 成本。
   - 销售额 = 销量 × 售价（售价通常基于进价和竞争定价，折扣空间有限）。
   - 成本 = 采购量 × 进价 + 损耗成本（损耗等于采购成本的一部分）。
   - 若损耗率从5%升至10%，相当于成本增加5%以上，而进价降低5%可能难以抵消。

3. **运营现实**：商超常通过促销消化高损耗商品（如打折、加工），但会拉低平均售价。低损耗水果可保持正常售价，形成持续毛利。

4. **调节空间**：优秀供应链管理（如冷链、短周转、精准订货）能显著降低损耗，而价格谈判受市场行情限制，且低价可能导致品质下降，反增加损耗。

**结论**：优先关注损耗控制（如优化供应链、快速周转），而非单纯追求低价。当然，在两者平衡中，需根据具体果品（如易损的草莓 vs 耐存的苹果）动态调整，但大体上，**降低损耗是提升毛利的更关键杠杆**。
