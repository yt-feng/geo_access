会。香蕉大小不一通常会影响销售，主要体现在以下几个方面：

1. **顾客感知与购买意愿**：消费者普遍偏好外观整齐、大小均匀的商品，不均匀的香蕉可能被看作“品质差”或“不够新鲜”，降低购买欲。
2. **定价困难**：大小混杂时，难以统一定价。按高价卖可能让买到大果的顾客觉得值，但买到小果的顾客觉得亏；按低价卖则会损失利润。
3. **损耗增加**：小果或过熟的香蕉容易先被挑剩，导致后期只能打折处理甚至报损，增加损耗成本。
4. **陈列效果差**：大小不一的香蕉摆在一起显得杂乱，影响整体货架吸引力。

**建议**：
- 采购时尽量要求供应商按规格分级（如大果、中果、小果分开装箱）。
- 如果已购入混装香蕉，可以分拣后按不同价格或用途销售（如大果卖高价，小果做特价或用于熟食、饮品加工）。
- 短期可搭配销售或提示“家庭装，大小不同更实惠”，但效果不如分级明显。
