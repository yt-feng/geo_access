小商家采购水果时，现金结算和月结账期各有以下主要风险：

**现金结算风险**
1. **资金压力**：需要立即支付全款，可能导致短期现金流紧张，影响其他运营支出。
2. **质量隐患**：钱货两清后，若水果出现腐烂、变质等问题，供应商可能拒绝退换或补偿。
3. **缺少账期缓冲**：没有容错空间，一旦遇到销售不佳或水果滞销，损失直接承担。
4. **潜在价格不透明**：部分供应商可能利用现金结算压低成本，但实际水果品质与样品不一致。

**月结账期风险**
1. **信用依赖**：依赖供应商的信誉，如果供应商跑路或断供，可能导致货款损失或货源中断。
2. **坏账风险**：若商家自身销售回款慢，长期欠款可能累积成难以偿还的债务。
3. **被动议价权弱**：供应商可能因账期长而提高单价，或在供应紧张时优先满足现金客户。
4. **管理成本**：需记录对账单、发票等凭证，月末核对时易出错，增加人工负担。

**建议**：
- 根据自身现金流稳定性、供应商信誉、水果保质期长短（如耐储存的苹果适合分期，易烂的草莓首选现金）权衡。
- 对月结供应商，要求签订书面合同明确账期、违约责任；对现金结算，保留付款凭证并当场验收品质。
- 小商家可尝试混合结算：如首次小额现金试合作，后期逐步转为月结或分期付款。
