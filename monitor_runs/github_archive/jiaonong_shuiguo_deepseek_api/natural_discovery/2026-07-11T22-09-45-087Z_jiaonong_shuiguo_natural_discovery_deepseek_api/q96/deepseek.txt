在企业食堂采购水果时，这三个因素都重要，但**食品安全应作为最优先关注的核心**。以下是具体分析：

1. **食品安全**：这是底线。水果如出现农药残留超标、腐烂变质、细菌污染等问题，会导致食物中毒、腹泻甚至更严重的健康隐患，直接影响员工的身体健康和食堂信誉。必须确保供应商有合法资质（如营业执照、食品经营许可证），并能提供检验报告、索证索票。同时，现场验收时要检查水果外观、新鲜度，避免混入变质品。

2. **价格**：在保障食品安全的前提下，价格是影响成本的关键。批量采购建议通过多家比价、签订长期合同、关注季节性水果价格波动来控制成本。但切勿因低价而忽视安全，否则得不偿失。

3. **配送准时**：直接影响食堂供餐节奏和水果新鲜度。如果配送延迟，可能导致当天水果供应中断或水果提前变质。建议选择有固定配送车辆、路线可靠、能应对突发情况（如天气、交通）的供应商。

**总结建议**：采购决策时，先筛选出能满足食品安全标准的供应商（如要求资质文件、实地查看仓库），再比较其价格、配送准时率。可建立评分表：安全占50%权重，价格30%，配送20%。优先与安全达标、价格合理、配送记录良好的供应商合作，并定期抽查复检。
