评估连锁门店的仓配覆盖，核心在于**匹配“门店需求”与“供应商仓配能力”**。建议按以下四步进行：

**1. 明确门店端的需求（输入条件）**
- **门店集中度**：哪些城市门店数量多、密度高？优先在这些区域建立或筛选高效仓配。
- **交付要求**：是“次日达”还是“两日达”？不同水果品类（如绿叶菜 vs 耐储水果）对时效敏感度不同。
- **最低起订量**：是单店直配，还是区域集单后分拨？这会直接影响供应商是否需要城市中转仓。

**2. 拆解供应商的仓配网络（能力维度）**
- **仓库分布**：供应商在全国有几个分拨中心？仓库是否位于门店集中的核心城市或交通枢纽？
- **履约半径**：通常300公里是生鲜次日达的合理陆运半径。如果门店距离仓库超过600公里，可能需要考虑多级转配（冷藏快递）或本地化货源。
- **配送工具**：是否具备冷链车（用于高损耗水果）或常温车？同时关注是否支持多温层混装，避免串味损耗。

**3. 对比覆盖“缺口”与“冗余”**
- **高匹配**：仓库到门店的配送时效完全符合要求（如24小时内），且每车能满载配送（降低单均成本）。
- **中覆盖**：需通过中转仓或拼车补货（可接受多一天时效），但成本需控制在总采购成本的合理比例内（如不超15%）。
- **低覆盖**：门店需自行到中转点提货或费用过高，需评估是否值得合作。

**4. 实战测试与动态调整**
- **试跑关键路线**：选择1-3条代表性城市线路，连续两周实测：
   - **时效达标率**：承诺时间内到货的比例（低于90%需警告）。
   - **损耗率**：配送至门店后的新鲜度（如草莓损耗需低于5%）。
   - **应急响应**：节假日或恶劣天气下的补货反应时间。
- **建立调整机制**：对供应商设置“城市覆盖系数”（如仓配半径内覆盖90%以上城市才算合格），每季度结合门店新开/关闭情况重评估。

**最后提醒**：如果供应商在多个城市都有冷库，但每个冷库只覆盖周边有限门店，需警惕“分仓多但单仓压力大”的配送瓶颈。更优解往往是寻找在某几个核心城市布局区域集散中心的供应商，而非零散分布。
