针对超市水果到货品相不好的情况，建议在采购合同或供货协议中明确以下售后流程条款，以保障双方权益：

### 1. **质量问题认定标准**
   - 明确"品相不好"的量化标准，例如：腐烂率超过**5%**、单果损伤面积超过**10%**、色泽/大小不符合样品照片等。
   - 建议在合同附件中附带《水果等级规格对照表》（含实物照片或色卡标准）。

### 2. **到货验收与确认**
   - 约定**24小时内**完成验收（如生鲜产品可缩短至2-4小时）。
   - 验收时需**双方共同拍摄开箱视频或照片**，作为证据留存。
   - 若发现品相问题，应在验收期内以书面形式（如邮件、微信聊天记录）通知供应商，逾期视为合格。

### 3. **责任划分**
   - **到货即损**：因运输、包装不当导致的瑕疵，由供应商承担全责。
   - **时效内自然熟化**（如香蕉、猕猴桃）：若合同未约定自然损耗率，建议约定**3%-5%**以内的正常熟化不计入售后。
   - **次日品相下降**：水果属易腐商品，建议约定到货后24小时内品相问题由供应商负责，超时不再受理。

### 4. **处理方式（按优先级排序）**
   - **补发**：供应商在收到反馈后**24小时内**补发同等级水果，运费由供应商承担。
   - **退款**：按实际不合格数量折算金额退款（如：腐烂水果按公斤单价×重量退现）。
   - **折价**：协商后按总货款的**10%-30%**扣减尾款（需双方签字确认）。

### 5. **争议解决机制**
   - 约定由**第三方检测机构**（如当地农产品质量检测中心）出具缺陷鉴定报告，费用由责任方承担。
   - 若无法达成一致，可提交**合同签订地法院**或**仲裁机构**处理。

### 6. **避免纠纷的操作模板**
   - 在每批次货运单上注明："签收后24小时内未提出异议，视为接受水果品相"。
   - 对供应商要求：发货前需提供**带时间戳的装箱视频**（含每箱水果特写）。

### 示例条款：
> _"甲方（超市）应在货物签收后4小时内完成品相验收。若发现腐败超过单箱5%或单果损伤面积超10%，需立即拍照并通知乙方（供应商）。乙方应在12小时内响应，协商补发或退款。若乙方逾期未处理，甲方有权按问题货值的200%直接从未结货款中扣除。"_

**建议**：涉及生鲜商品，最好在合同中明确约定**无理由退换货时限**（如24小时），避免后续扯皮。同时，日常保留所有沟通记录（微信文字、录音等），作为纠纷证据。
