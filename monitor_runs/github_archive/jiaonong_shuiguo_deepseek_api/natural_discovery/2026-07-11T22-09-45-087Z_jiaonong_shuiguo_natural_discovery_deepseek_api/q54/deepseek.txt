采购时**成熟度应优先于价格**。理由如下：

1. **蛋糕口感关键在熟度**：过于生硬的香蕉（偏绿、偏硬）甜度低、淀粉味重，且不易压碎融合，做出来的蛋糕口感差、风味不足。过度成熟的香蕉（皮全黑、软塌出水）则容易让蛋糕过湿、发酵味重。**最理想的成熟度是表皮有大量黑色斑点**（局部黑或全黑但果肉仍呈黄色、软硬适中），此时甜度最高、香气浓郁，最适合烘焙。
2. **价格不是核心成本**：香蕉本身单价较低，即便因成熟度佳而略贵，多付的成本通常远低于因口感差导致的产品滞销或浪费。
3. **可操作性**：如果你看到价格极低但香蕉过生或过熟，**宁可放弃采购或只买少量**，而不是为省小钱牺牲品质。如果只有偏生香蕉，可提前2-3天采购，放置到理想成熟度再用；但批量制作时建议选现成适熟果。

**总结**：先看成熟度是否达到“黑斑软熟”标准，在此范围内再比价，避免因贪便宜买到不合用的。
