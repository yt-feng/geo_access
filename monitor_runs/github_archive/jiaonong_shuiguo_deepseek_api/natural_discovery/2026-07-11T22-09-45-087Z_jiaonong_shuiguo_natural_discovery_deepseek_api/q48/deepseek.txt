要判断茶饮连锁供应商能否稳定按规格送货，建议从以下几个实操维度进行考察和测试：

1. **考察历史合作记录**：要求供应商提供过去半年到一年的送货记录，重点看订单履行率（是否按时发货）、规格匹配率（重量、大小、熟度是否达标）、次品率。如果对方不太愿意提供，可以小范围要求他们提供第三方仓库或物流方的对账单。

2. **进行小批量试单测试**：先下2-3次小批量订单，每次规格相同（如“单个菠萝重量1.2-1.5斤，成熟度7-8成”）。观察每次到货的规格一致性，以及错发、漏发、延迟的情况。连续3次都达标，才算初步通过。

3. **要求提供品控流程文档**：好的供应商应该有明确的采摘标准、分选流程、包装和冷链运输规范。要求他们提供书面的品控SOP（标准操作程序），并确认是否会对每批次菠萝进行抽检。

4. **现场/视频验厂**：建议实地或通过视频检查供应商的仓库、分选设备、冷藏条件。重点看分选线上是否有专人剔除不达标果实，储存温度是否稳定在7-13℃（菠萝最佳储存温度），是否配备有专业硬度或甜度检测设备。

5. **明确合同条款与违约惩罚**：在采购合同中写入“规格不符时的处理方式”，例如：每批次抽检，不合格率超过5%时，供应商需无条件换货或按比例扣款；延迟交货超过24小时需支付违约金。条款越明确，供应商履约压力越大。

6. **建立长期库存和安全库存机制**：要求供应商承诺每周可调货的弹性量（比如你能接受20%的浮动），并保留至少2家备选供应商。这能让你在供应商出现问题时快速切换，同时也给对方压力。

如果以上测试和条款能顺利执行，供应商通常具备稳定按规格送货的能力。关键还是“多测几次，看结果说话”，而不是只看报价和口头承诺。
