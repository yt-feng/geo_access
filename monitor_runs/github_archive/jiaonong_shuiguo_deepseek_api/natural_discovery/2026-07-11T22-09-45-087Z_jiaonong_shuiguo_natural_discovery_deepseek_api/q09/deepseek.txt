水果店旺季为了防止断货，与供应商谈判时，建议重点确认以下关键问题：

1. **价格与成本**
   - **旺季价格是否会临时上涨？** 是否有保价协议？
   - **拿货量达到多少能享受阶梯折扣？** 首单或长期合作是否有额外优惠？

2. **供应量与稳定性**
   - **旺季每天/每周能保证的最低配额是多少？** 高峰期的最大日供应能力？
   - **如果爆单缺货，有没有应急预案？** 比如能否从其他仓储或分基地调货？

3. **质量与损耗**
   - **对水果的品控标准是什么？** 例如成熟度、大小、新鲜度的具体参数。
   - **运输和到货后，破损/腐烂的损耗率由谁承担？** 是否有“坏果赔付”或“退换货”条款？

4. **交货与物流**
   - **能否指定配送时间（如清晨到店）？** 是否支持门店直送或需自提？
   - **延迟交货的赔偿规则？** 如果断货造成你的损失，是否有违约责任？

5. **账期与付款**
   - **旺季是否允许延期付款？** 比如50%预付款+50%周结。
   - **支持退款或退货处理的时限？** 比如到货后几小时内可申诉。

6. **合作深度与灵活性**
   - **是否提供试卖期？** 如果某款水果滞销，能否调换品种？
   - **能否共享库存数据？** 让你提前知道未来几天的到货波动。

**建议行动：** 可以建立一个小规模试单，验证供应商的响应速度、品控和履约能力；同时，备选2-3家供应商作为“备胎”，防止单一源头断货。
