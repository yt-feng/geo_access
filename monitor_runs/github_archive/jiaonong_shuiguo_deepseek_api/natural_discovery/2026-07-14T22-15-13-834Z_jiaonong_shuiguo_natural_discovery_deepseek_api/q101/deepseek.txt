和酒店早餐供应商谈水果供应合作时，建议重点关注以下几个方面，确保稳定、优质且成本可控：

1. **签订长期框架协议，明确供应周期与最低保供量**  
   - 要求供应商承诺每日/每周最低供应数量，并写明若因供应商原因断货的违约责任（如扣款、赔付）。
   - 约定极端天气、季节性短缺等特殊情况下的备选方案（如启用二级供应商）。

2. **制定分级验收标准与退换货流程**  
   - 明确水果品种、成熟度、大小、外观（无斑点/损伤）等具体标准，必要时提供图文示例。
   - 约定到货后验收时限（如1小时内），不合格品退换货流程及运输损耗承担比例（建议要求供应商承担80%以上损耗）。

3. **价格锁定与调价机制**  
   - 谈判时争取3-6个月价格稳定期，避免频繁调价。
   - 若因市场价大幅波动需调价，要求提前30天书面通知，并约定涨幅上限（如不超过批发均价15%），或参考指定的批发市场价格指数。

4. **冷链物流与时效约束**  
   - 要求供应商使用冷链车配送，运输温度需符合水果保鲜要求（如苹果0-4℃、芒果10-13℃），并在合同中明确温控记录查验方式。
   - 约定每日配送时间（如早餐食材需在前一晚20:00前到货），超时罚则（如按迟到比例扣减当日货款）。

5. **食品安全与溯源要求**  
   - 要求供应商提供每批次水果的农药残留检测报告（建议参照国家标准GB 2763），以及供货资质（营业执照、食品经营许可证）。
   - 明确可追溯至产地、批次号，并保留样品至少48小时以备抽检。

6. **结算与账期谈判**  
   - 通常采用月结，可争取先试用2-3天再签长约，观察其供货稳定性。
   - 要求供应商提供增值税专用发票，并明确账单核对周期（如每月5日前对账，15日前付款）。

7. **合同附加条款**  
   - **不可抗力条款**：明确极端天气、疫情等突发情况下双方沟通义务与减责条件。
   - **独家供应权**：若对方要求独家供应，需同时要求其保证价格竞争力与优先满足权。

**验证建议**：首次合作时，要求供应商提供3-5家其他酒店客户的联系方式进行背景核实；试合作期间每日记录到货合格率、时效及响应速度，作为续约依据。
