采购菠萝时判断熟度，核心目标是**确保到店后能在1-3天内自然熟化，减少因过熟或过生导致的损耗**。以下是可直接操作的判断方法：

1. **看外观颜色**：选择**青黄相间（约70%绿、30%黄）** 的菠萝。全绿太生，需长时间催熟且容易烂心；全黄过熟，到店当天或次日就可能变质。避免有明显**黑斑、软塌或渗出水渍**的。

2. **摸硬度与手感**：轻轻按压菠萝**果眼之间的果肉**，应**坚实有弹性**（像按新鲜苹果）。如果按压处**发软、凹陷**，说明内部可能已开始腐烂；如果**硬如石头**，则太生。

3. **闻底部气味**：将菠萝翻转，闻**底部切口**。应有**清淡的草香**或**几乎没有气味**。如果闻到**明显的甜香、酒味**，说明已经过熟，易损耗；如果有**酸腐味**，绝对不采购。

4. **掂重量与听声音**：同等大小选**手感更沉**的，说明水分充足、内部紧实。轻轻敲击菠萝**声音清脆**、类似拍打西瓜“咚咚”声，代表成熟度适中；**声音发闷、空响**可能内部干瘪或已变质。

5. **检查果眼与刺**：观察菠萝的“眼睛”（果眼），成熟的果眼**相对平坦、凹陷不深**，呈黄褐色；如果果眼**高耸、刺扎手**，说明生涩。

**总结建议**：采购时以**青黄相间、底部硬实、无甜酒味、手感沉重**的菠萝为优选。若必须选择全熟（黄色多）的，仅当日快速出清的少量进货，并优先安排促销或切块处理。
