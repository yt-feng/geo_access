针对茶饮店旺季菠萝用量突然增大的临时补货需求，建议按以下步骤与供应商沟通并约定：

1. **提前沟通与确认**  
   第一时间联系供应商，说明旺季需求突增的具体情况，明确需要补货的**品种、规格（如去皮/整果、大小分级）和数量**。建议提供近期的日均消耗量数据，让供应商理解需求的合理性。

2. **约定价格与结算**  
   - 临时补货价格可能高于原合同价，需提前协商：可接受小幅上浮，或要求供应商提供“旺季特批价”。  
   - 明确结算方式（现结、周结或月结），避免资金压力。

3. **签订临时补充协议**  
   书面（或微信/邮件确认）补充以下条款：  
   - **交付时间**：例如“48小时内到货”“指定日期前分批次到货”。  
   - **质量验收标准**：明确新鲜度、损伤率（如≤5%）、成熟度（如7-8成熟）等。  
   - **违约责任**：如延迟到货按日扣款，质量不合格可拒收并要求补货。  
   - **优先权**：要求供应商在产能紧张时优先保障你的订单。

4. **分批到货与备选方案**  
   - 若一次补货量过大，可要求分2-3天到货，避免存储压力。  
   - 同时联系1-2家备用供应商，以防主供应商无法满足全部需求（但需注意质量统一）。

5. **后续合作优化**  
   - 要求供应商提供下一周的菠萝到货预期，提前锁定下一轮订单。  
   - 与供应商协商“旺季浮动配额”，如允许按历史用量上浮20%自动补货。

**示例沟通话术**：  
“李总，我们门店最近客流爆炸，菠萝用量涨了3倍，需要紧急补200斤去皮菠萝。这批次价格可以按市场行情上浮5%，但希望明天下午4点前到货，品质按上周标准（损伤≤3%）。如果做不到，我这边需要找另一家备选，您看能协调吗？”

**注意**：若临时补货金额较大，建议让供应商提供**近期仓库实拍照片**或**批次检验报告**，防止以次充好。
