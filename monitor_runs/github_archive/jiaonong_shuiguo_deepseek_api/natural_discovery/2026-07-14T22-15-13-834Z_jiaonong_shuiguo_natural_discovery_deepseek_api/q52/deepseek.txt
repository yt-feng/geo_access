为避免甜品店采购香蕉后快速变黑，建议采取以下措施：

1. **采购时选择适当成熟度的香蕉**：挑选颜色青黄、表皮紧实、无黑斑或软烂的香蕉。避免选择全黄或已出现褐斑的，因为后者在运输后可能很快过熟。
2. **运输中注意保护**：轻拿轻放，避免挤压和碰伤，可装纸箱内并填充缓冲物。伤处会加速氧化变黑。
3. **到店后立即处理**：
   - **分开存放**：将香蕉与其他水果（特别是苹果、梨、芒果等释放乙烯的水果）隔离，减少催熟气体影响。
   - **挂置或平放**：使用挂钩悬挂，或凸面朝上平放，减少与台面的接触面积。
   - **温度控制**：未熟的香蕉在室温（约12-16℃）通风处存放。如果香蕉已熟但暂时不用，可放入冰箱冷藏（注意：冷藏会减慢变黑但表皮会变褐，但果肉品质可保持1-2天）。
   - **避免密封或高温**：不要用塑料袋密封，否则湿气会加速霉变。远离炉灶、烤箱等热源。
4. **使用酸性溶液处理**：采购后，用柠檬汁、醋或盐水轻轻擦拭香蕉柄和轻微破损处（浓度不要高，以免影响甜点风味），再擦干。酸性环境可抑制氧化酶活性。
5. **尽快使用**：根据甜品需求，规划每日消耗量。当天或次日使用的香蕉可先剥皮切块后冷冻（适合做冰沙、松饼等需煮熟的甜品），冻香蕉不会快速变黑。

**总结关键点**：选青黄未熟的、轻拿轻放、隔离储藏、低温通风、使用前再清洗。
