采购香蕉时，控制熟度需根据最终用途、储存条件和制作周期决定。建议按以下方法操作：

1. **明确用途对熟度的要求**：
   - **直接做饮品（如香蕉奶昔、冰沙）**：选择**全熟但未过软**的香蕉（表面有少量黑点，手感有弹性但不软烂），甜度高、风味足，且打碎后口感顺滑。过生则涩、淀粉味重；过熟则易氧化变黑、口感发黏。
   - **做蛋糕、松饼等烘焙品**：需要**极熟甚至表皮全黑**的香蕉，内部软烂，甜度最高，适合捣碎与面粉混合。
   - **做装饰或切片（如香蕉船、分层甜品）**：需要**刚熟透但质地结实**（皮全黄无斑），便于切片不塌，甜度足够。

2. **按销售周期分批采购**：
   - 若当天或次日即可用完，采购**七到八成熟**（表皮黄绿相间，手感微硬），室温放置1-2天即可成熟。
   - 若需存储3-5天，采购**六到七成熟**（表皮黄绿、较硬），分批取出催熟。
   - 避免采购全熟（全黄带小黑点）的香蕉，除非当天即用尽，否则易过熟浪费。

3. **分批催熟技巧**：
   - 常温（20-25℃）自然成熟：按使用计划将香蕉分开存放，加快或延迟成熟速度。成熟速度可通过是否放入密封纸袋、是否与其他水果（如苹果、梨）同放调节。
   - 避免冷藏：香蕉在12℃以下会受冷害，表皮变黑、风味变差，不适合长期冷藏。若需临时保存，可剥皮后冷冻，用于奶昔等饮品。

4. **采购时现场检查**：
   - **皮色**：全绿太生，黄绿转黄中可根据到货后使用时间调整。
   - **触感**：轻微按压果柄处，有弹性为合适；过硬或凹陷（软烂）则否。
   - **果柄**：新鲜香蕉果柄青绿或浅黄，若发黑、干枯则已过熟或受损。

5. **应急处理**：
   - 若采购到稍过熟的香蕉，立即剥皮切块冷冻（用于制作冰沙、奶昔时效果佳），或尽快制作熟香蕉酱（加糖熬煮冷藏），延长使用期。
   - 若到手偏生，可将香蕉与成熟苹果、番茄一起放纸袋内，24-48小时可催熟至理想状态。

**总结**：按“用途定熟度、按物流定批次、到货靠手感”原则。对饮品/甜品店，建议熟度范围在“全黄微带斑”至“刚全熟”间，采购时多买微生一批，熟度可通过室温存放调节，避免因一次买太多过熟导致浪费。
