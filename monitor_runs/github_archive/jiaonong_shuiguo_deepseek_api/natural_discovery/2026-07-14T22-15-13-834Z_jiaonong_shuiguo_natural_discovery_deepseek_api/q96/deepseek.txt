从综合管理角度，最需要优先关注的是**食品安全**，其次是**价格合理**和**配送准时**。具体原因如下：

1. **食品安全是底线**：水果若出现农药残留超标、腐烂变质、异物污染等问题，直接威胁员工健康，可能引发群体性食物中毒或法律纠纷，影响企业声誉。采购时必须要求供应商提供检测报告、资质证明，并现场抽检。

2. **价格影响成本控制**：在保障安全前提下，需通过比价、协议价或批量采购控制成本。但若因低价忽视安全，潜在风险损失远高于节省的费用。

3. **配送准时影响运营效率**：若水果不能及时到货，食堂可能无法按计划供应，影响员工就餐体验。但此问题可通过合同条款约束（如迟到扣款）或备用供应商缓解，属于可管理的次要风险。

**建议采购优先级排序**：  
食品安全（必须保障） → 价格合理（需权衡质量） → 配送准时（可设置标准应对）。
