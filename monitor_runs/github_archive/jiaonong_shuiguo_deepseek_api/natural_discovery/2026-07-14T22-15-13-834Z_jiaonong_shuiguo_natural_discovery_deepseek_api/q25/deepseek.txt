针对超市门店多但单店量小的特点，建议采用“**集采分单+区域性灵活补货**”模式。具体方案如下：

1. **联合采购平台**  
   与同规模连锁超市或小商超联盟，通过第三方采购平台（如鲜丰、鑫荣懋等B2B平台）集中下单，降低单价。单店按需分货，避免库存积压。

2. **区域分仓+高频小批量配送**  
   在门店密集区域设一个中心分仓或与本地大型批发商合作，由仓库每天/隔天小批量配送到各门店。例如：当天订单汇总后，次日凌晨分拣派送，确保水果新鲜。

3. **源头直采与灵活拼车**  
   对高周转品类（如苹果、香蕉），与产地合作社签订短周期直采协议，但仅采购单店一周左右的量；同时与其他生鲜店（如社区菜店）拼车运输，分摊运费。

4. **利用前置仓或第三方代发**  
   对非标、易损水果（如草莓、蓝莓），由供应商直接代发到各门店（如京东冷链、顺丰冷运），门店无需大量囤货，按需下单。

5. **动态分拣与损耗共享**  
   与供货商约定：部分损耗可由供应商承担（如每箱预留1-2个损耗指标），或按实际销售结算部分尾款，降低单店风险。

**执行建议**：先从2-3家核心门店试点，选择2-3款标准化水果（如柑橘、猕猴桃）测试集采模式，稳定后再扩展品类。同时，利用门店POS数据预测单店需求量，逐步优化分货比例。
