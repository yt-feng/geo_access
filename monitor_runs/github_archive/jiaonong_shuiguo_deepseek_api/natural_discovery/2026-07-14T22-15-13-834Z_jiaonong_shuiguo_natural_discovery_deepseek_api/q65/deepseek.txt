对于水果批发商而言，面对价格波动，建议优先采用**短期定价为主、长协为辅**的策略，具体取决于以下因素：

1. **日常高频交易品种**：建议**每天议价**。水果价格通常受天气、供需、运输等短期因素影响较大，每日议价能及时反映市场行情，避免因固定价格导致亏损或错失利润。例如，苹果、香蕉等上市集中、易腐性强的水果，适合随行就市。

2. **稳定供应或大客户订单**：可签订**短期固定价格协议**（如1周至1个月）。前提是双方能合理分担风险，例如设定价格调整触发条件（如涨幅超过10%重新议价），或约定最低价、最高价保护条款。适用于有一定冷藏能力的品种（如柑橘、进口水果）。

3. **季节性高峰或预期大涨品种**：建议采用**“基准价+浮动”**模式，如每3天调整一次，结合产地到货价格和仓储成本。避免长期锁定价格，特别是反季节水果。

4. **实操建议**：  
   - 建立**每日价格监测机制**，关注主要产地批发价、天气预报、物流动态。  
   - 对易腐商品（草莓、荔枝等）必须日议；对耐存储商品（土豆、苹果）可签周价。  
   - 与供应商签订框架合同，约定定价频率和参考基准（如当地市场批发均价），保留灵活调整权。

总之，**不建议长期固定价格**，但可适度利用短期协议锁定部分订单，同时保留对市场变化的快速响应能力。
