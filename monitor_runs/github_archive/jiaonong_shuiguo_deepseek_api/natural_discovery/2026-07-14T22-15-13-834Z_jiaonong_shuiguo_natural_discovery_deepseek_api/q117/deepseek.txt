对于新品水果，建议按以下策略控制试卖数量：

1. **最小可行性试卖**：建议首批进货量控制在 **10-20公斤** 或 **1-2箱**（视水果单价和体积而定）。这个量既能满足小范围测试需求，又不会造成严重积压损失。

2. **试卖周期**：设 **3-5天** 为试卖窗口。新鲜水果保质期短，需快速评估周转速度。

3. **评估维度**：
   - 顾客复购率（购买后一周内是否有人回头买）
   - 每日自然损耗率（如第一天剩余30%，第二天剩余50%需警惕）
   - 目标人群反馈（可让员工试吃并记录优缺点）

4. **安全边界**：如果试卖3天后：
   - 日销≥80%首批量 → 可考虑追加订货（增加至30-50公斤）
   - 日销≤40% → 立即打折清货，避免库存腐烂

5. **特殊节点**：
   - 如果是高价进口水果（如日本晴王葡萄），建议首批量减半，先做样品展示+预购登记
   - 如果是季节性爆款（如中秋石榴），可适当提至30公斤，但需提前和供应商确认退换条款

**核心原则**：宁可错过首波销售高峰，也不要因囤货导致亏损。新品试卖的本质是用最小成本验证市场反应。
