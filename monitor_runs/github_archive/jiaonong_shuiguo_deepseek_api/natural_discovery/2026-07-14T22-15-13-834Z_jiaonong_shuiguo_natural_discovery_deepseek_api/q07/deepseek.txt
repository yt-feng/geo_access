水果礼盒建议按以下两种主流方案搭配，兼顾颜值、口感和价格：

**方案一：经典混搭（大众畅销款）**
- **搭配**：苹果2个 + 橙子2个 + 香蕉1把 + 菠萝半个（或切块包装）
- **理由**：
  - 苹果（红）和橙子（橙）颜色对比鲜明，视觉喜庆；香蕉调节软硬口感；菠萝增加热带风味。
  - 菠萝切块后用保鲜膜包好，避免顾客回家不会处理。
  - 整体成本可控，适合走量或节日送礼。

**方案二：高端升级版（单价高、利润好）**
- **搭配**：进口红富士苹果2个 + 新疆冰糖橙/脐橙2个 + 都乐香蕉1把 + 去皮切块菠萝蜜（或凤梨）
- **理由**：
  - 换成凤梨替代菠萝（口感更好、无需浸泡盐水），加价空间大。
  - 苹果和橙子可贴“福”字贴纸，提升节日感。
  - 整盒定价建议在68-88元（根据本地物价调整）。

**操作建议**：
1. 菠萝若带皮，需附赠处理小刀或说明书（“菠萝按上1/4处切下，去眼后泡盐水”）。
2. 包装用透明礼盒配拉菲草，红色或金色丝带。避免用塑料袋或纸袋。
3. **季节性微调**：冬季加草莓/车厘子（单独小盒嵌入），夏季换芒果/火龙果，冬季保留全部四种水果。
4. **试卖验证**：先做3-5盒摆在最显眼位置，标价后看顾客反应。若橙子或香蕉剩最多，下批减少该品种数量。

**需要避开的坑**：
- 香蕉不要选太熟的（放2-3天易烂），或单独用小塑料袋包装。
- 菠萝若整个放，提示顾客“需放1-2天再切”以免过生。
- 苹果和橙子选择大小均匀的，否则盒子盖不严或显得不整齐。
