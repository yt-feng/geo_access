商超做会员日水果活动时，供应商需要提前从货品、物流、人员、促销与售后四个维度配合准备：

1. **货品与备货**
   - 按活动预估销量提前1-3天备足货品，确保主力品种（如苹果、香蕉、橙子）库存充足。
   - 分批到货：活动前一天送达大部分，当天清晨补送小部分，保证新鲜度。
   - 提前做好分拣、预包装（如按斤或按份打包），减少现场称重排队时间。
   - 准备3-5款超低价引流品（如0.99元/斤的应季水果），以及中高端利润品（如进口提子、车厘子）。

2. **物流与配送**
   - 确认活动当日最早到货时间（建议商超开门前1-2小时），并安排备用运输车辆应对突发。
   - 使用冷链车或保温箱运输，避免水果在途中受损或变质。
   - 随货附带完整质检报告、产地证明及合格证。

3. **现场人员与陈列**
   - 活动当天安排专职促销员（至少1-2名）到场，负责试吃推广、补货、叫卖。
   - 提前与商超沟通陈列位置（主通道端架或水果区黄金位），准备统一价签、爆炸贴、围挡等物料。
   - 准备试吃品（切块装盘、牙签），试吃水果需单独备货，不可用等售品代替。

4. **价格与售后**
   - 提前在系统中维护好会员专享价（建议低于日常价20%-30%），并确认价签与系统价格一致。
   - 明确会员限购数量（如每人限购3斤），防止黄牛扫货。
   - 准备备用库存及退换货预案：对软烂、磕碰水果提供现场更换或退款服务，提升会员口碑。

**关键检查清单**（提前1天确认）：
- [ ] 库存数与系统数据一致
- [ ] 促销员排班及到岗时间
- [ ] 冷链车/保温箱就位
- [ ] 价签、宣传物料已送达
- [ ] 与商超采购确认活动细则（如营业前30分钟内上货）

以上准备可以确保活动当天货品充足、人流不堵、投诉减少。
