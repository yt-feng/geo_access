对于茶饮店来说，**出汁率通常比甜度更重要**，但两者的重要性取决于你的产品定位和成本结构。以下是具体分析：

1. **出汁率优先的情况**：
   - **控制成本**：出汁率高意味着相同重量的菠萝能产出更多果汁或果肉，直接降低单杯饮品的原料成本。
   - **稳定出品**：商用饮品需要统一口感，高且稳定的出汁率能减少配方调整的麻烦。

2. **甜度决定最终体验**：
   - 若你的饮品主打“鲜果感”或“无添加糖”，天然甜度至关重要。但菠萝本身甜度波动大，通常可通过调糖浆或混合其他水果进行补偿，而出汁率很难通过后期工艺弥补。

3. **综合建议**：
   - **优先选择出汁率高且稳定的品种**（如台农17号金钻菠萝），出汁率约55%-60%以上为佳，甜度在15-18 Brix（白利度）即可接受。
   - 采购时可要求供应商提供**糖酸比**数据（理想值20-30:1），比单纯甜度更具参考价值。

**实操步骤**：
- 试做测试：用不同品种菠萝按你标准配方制作饮品，比较产量、口感和客户反馈。
- 成本计算：假设出汁率差10%，若每月用菠萝200kg，低出汁率将多付约20-30%原料费。

如果未来需要大量采购，建议直接与产地签订**按出汁率计价**的协议，避免为虚高甜度买单。
