酒店在采购水果盘原料时，供应商应确保以下规格和外观标准：

1. **规格标准**：
   - **尺寸统一**：水果大小、形状需基本一致（例如苹果直径5-8cm、葡萄每串长度相近），便于摆盘美观。
   - **重量标准**：按单果或包装重量要求（如每箱标准果重10kg±0.5kg），避免缩水。
   - **成熟度**：按酒店要求控制在7-8成熟（如芒果、牛油果）或全熟（如草莓），确保口感并减少损耗。
   - **品种与产地**：符合合同约定品种（如红富士、夏黑葡萄），并提供可追溯的产地证明。

2. **外观标准**：
   - **无缺陷**：无碰伤、压痕、虫蛀、腐烂、霉斑、异色斑点。
   - **色泽新鲜**：表面光泽自然，无氧化、褐变（如苹果、梨切开后需避免氧化处理）。
   - **无杂质**：无泥沙、叶片、茎部（如草莓去蒂）、虫卵或化学残留。
   - **清洁度**：可清洗后直接摆盘，表面无明胶、蜡质或多余水分（避免出水影响卖相）。
   - **包装要求**：使用食品级容器（如塑料筐、纸箱），衬垫防震材料，标注明确的生产日期和冷藏温控要求（如2-7℃控温运输）。

**额外注意**：供应商应提供质量检测报告（如农残、重金属），并承诺在酒店验货时对不符合标准的批次进行退换货（如8%以内瑕疵率可接收，超出则全退）。
