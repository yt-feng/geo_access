在超市与水果供应商的账期谈判中，常见账期通常为 **30天至60天**。具体天数和行业惯例如下：

1. **常规水平**：  
   - 对于中小超市或非连锁店铺，账期多为 **15-30天**。  
   - 大型连锁超市（如沃尔玛、永辉）通常要求 **45-60天**，部分甚至可达 **90天**（多见于风险较低的供应商）。

2. **水果行业特性**：  
   - 水果易腐、保质期短，供应商常希望缩短账期以降低资金压力和损耗风险。实际中，**7-15天**的短期结算也常见于生鲜供应商与超市的合作。  
   - 若超市对账期要求较长，供应商可能要求提高供货价格或采用“赊销+返点”模式。

3. **影响因素**：  
   - **超市规模**：大型超市议价能力强，账期长；小型超市或个体店账期短。  
   - **水果品类**：高单价、耐储存品类（如苹果、柑橘）账期可较长；叶菜、浆果等易腐品类账期更短。  
   - **合作历史**：长期稳定供应商可能获得略长账期。  
   - **付款方式**：部分超市采用“月结”或“滚动结算”（如每周结算前一周销量）。

**建议谈判策略**：  
- 作为超市方，可提出 **30天账期** 作为基准，并观察供应商反馈。  
- 若供应商要求缩短账期，可尝试用“增加采购量”或“支付部分预付款”来交换更长账期。  
- 明确合同中的滞纳金条款（如逾期付款按日收0.1%违约金），避免纠纷。

（注：以上均为行业常见惯例，具体以实际合同约定为准。）
