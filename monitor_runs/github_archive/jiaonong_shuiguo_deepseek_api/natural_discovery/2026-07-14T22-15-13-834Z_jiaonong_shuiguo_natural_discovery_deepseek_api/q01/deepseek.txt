选择水果供应商时，需要从以下多个维度判断其可靠性，尤其针对香蕉和菠萝这类易损耗水果：

1. **资质与信誉**  
   - 要求供应商提供营业执照、食品经营许可证、产地检疫证明。  
   - 查询企业信用信息（如天眼查）是否有经营异常或处罚记录。  
   - 向同行打听口碑，或要求提供以往合作的零售商联系方式（可暗中核实）。  

2. **香蕉专项考察**  
   - **成熟度控制**：香蕉不耐储存，应选择能提供“青-黄”梯度熟度（例如七成青、三成黄）的供应商，避免一次性进全熟果。  
   - **冷链能力**：查看是否有恒温车（13-15℃冷链运输），到货时香蕉表皮应无大量黑斑、果柄不发霉。  
   - **催熟技术**：确认供应商是否使用规范催熟方法（如乙烯利），避免使用不合法催熟剂导致果肉未熟。  

3. **菠萝专项考察**  
   - **品相与重量**：要求每个菠萝重量≥1.2公斤（标准果），叶片青绿无干枯，底部切口新鲜不干燥。  
   - **糖度与酸度**：可要求供应商提供近期批次糖度检测报告（菠萝糖度多≥12°），或随机切果试吃。  
   - **抗病性**：关注是否经过“黑心病”预防处理（如适时保鲜剂浸泡），到货后切开检查果心无褐变。  

4. **服务与合同**  
   - **拼货灵活性**：询问能否批发量的香蕉配同车菠萝（降低运费成本）。  
   - **退换规则**：明确签收后48小时内发现腐烂、裂纹（菠萝）、损伤可退货/换货，并写入合同。  
   - **账期与价格**：观察是否给出低于市场均价15%以上（警惕低价陷阱，可能用次果冲量），建议首单小量测试。  

5. **实地抽检步骤**  
   - 先下小额试单（比如50箱香蕉+20箱菠萝），收货后：  
     - 按10%比例抽检，测量香蕉单根长度（≥18cm为优级）和菠萝单果周长（需≥35cm）。  
     - 压箱底层果是否有水渍、异味。  
     - 连续两次试单合格再签长期合同。  

**简单总结**：优先选本地有冷库、能配合阶梯交货（比如每周二送青香蕉，周四送半熟香蕉）的供应商，并要求每次到货附带《出库检疫记录》。
