在商超水果采购中，**损耗低对最终毛利的影响通常大于价格低**。原因如下：

1. **损耗直接侵蚀利润**：水果是易损耗商品，运输、储存、上架过程中可能因磕碰、腐烂、过熟等产生损失。假设进货价降低10%，但损耗率从5%上升到15%，多出的10%损耗会直接抵消甚至超过价格优势带来的利润。
2. **毛利公式的杠杆效应**：最终毛利 = (售价 - 进货成本) × 销量 - 损耗成本。损耗成本不仅包含进货价，还包括人工、场地等分摊成本。降低损耗可减少无效成本，而单纯压低进价可能牺牲品质，导致售价上不去或销量下降。
3. **实际案例**：例如进价5元/斤的水果，售价10元，正常损耗5%时毛利约4.5元/斤。若采购价降至4元，但损耗升至15%，实际毛利约3.4元/斤。可见损耗增加对利润的破坏性远大于价格优惠。

**建议策略**：优先选择品质稳定、新鲜度高、耐储存的货源，通过优化供应链（如冷链、快速周转）降低损耗。在保证低损耗的前提下，再争取价格优势。同时，关注门店陈列、库存管理和促销节奏，从销售端减少损耗。
