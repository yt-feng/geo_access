对于每天需要少量水果的餐饮店，建议选择以下类型的供应商，比较省心且稳定：

1. **本地生鲜配送平台（如美团快驴、美菜、多多买菜）**  
   这类平台支持次日达、按需采购（最小起订量低），可避开批发市场的高门槛，价格透明、配送稳定。适合长期小批量采购，客服售后相对规范。

2. **社区/农贸市场周边的水果摊主**  
   与固定摊位建立长期合作，约定每日送货（通常可免费配送少量）。可现场看货，确保品质，沟通灵活。适合店面附近3公里内的商铺。

3. **小型水果分拣仓或批发市场里的“精品档口”**  
   找那些专门做餐饮配送的档口，说明自己“每日小量、稳定拿货”，他们通常愿意接单并安排小货车或电动车顺路配送，价格略高于大量批发但远低于零售。

4. **线上本地社区团购水果仓（如多多买菜、美团优选等）**  
   适合用量极少的时段，作为临时补货渠道。次日达，无需囤货，但需提前一小时下单。

**省心要点**：
- 选择“免费配送、无起送量或低起送量、支持退货/换货”的供应商。
- 尽量找有两三个备选供应商，预防某家断货或质量差时应急。
- 协议约定“长期固定时间送货、支持次日调整数量”（比如今天报明天要的品种和斤数），减少临时采购压力。
- 如果是餐厅连锁店，可考虑与本地水果配送公司签约，他们常具备小单配送能力。

**不推荐**：
- 大型批发市场（必须整车购买、无售后、需自行拉货，对小店不友好）。
- 大型超市（价格高、品种固定、且要求量大或自提）。
