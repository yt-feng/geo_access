控制社区超市水果的新鲜度和库存，核心是“高频少量、动态调整”。以下是具体可落地的建议：

1. **按进销存数据倒推进货量**：参考上周同一天、同天气下的销售数据，按 **1.2-1.5倍** 的安全系数进货（例如上周卖出50斤苹果，本周进60-75斤）。避免凭感觉进货。

2. **分级采购与周转策略**：
   - 将水果分 **高周转**（苹果、梨、香蕉）和 **易损耗**（草莓、芒果、葡萄）两类。
   - 高周转：可3天进一次，保持货架丰满。
   - 易损耗：**每日进、下午5点前**（参考社区客流高峰）少量补货，避免隔夜。

3. **物理控鲜方法**：
   - 陈列区加装 **冷雾系统** 或使用冰袋覆盖（如叶菜类水果）。
   - 早市后（如10点前）将未卖完的易损水果 **回冷库**，下午再少量出样。
   - 设置 **“当日鲜”专区**，明确标签提醒顾客优先选购。

4. **动态处理临期库存**：
   - 设定 **“半价清货时间”**：如下午6点后对剩余易损耗商品打折（提前画好区域，避免混放）。
   - 对轻微瑕疵（如碰撞斑）的水果，切盒装盘 **当天处理**，定价为原价的60%。
   - 与附近餐厅、奶茶店合作 **临期直供**（如熟透的香蕉可做甜品）。

5. **最小化库存天数**：
   - 水果平均库存天数控制在 **1.5-2天** 以内。
   - 如果某种水果连续2天有剩货，立即 **暂停进货1-2天**，消化后再恢复，避免堆货导致黑斑损耗。

6. **工具辅助**：
   - 手工记录：用白板每日更新“进货量-销量-剩余量”，重点标注损耗高的单品。
   - 如果熟练后，可以尝试用Excel或手机记账APP（如随手记）记录每天数据。

**最后提醒**：观察社区顾客的购买时间（例如上班前、晚饭后），将补货时间集中在这两个时段前，能有效减少在库时间。如果某类水果连续3天有15%以上损耗，建议直接更换供应商或品种。
