可以。餐饮店通常能与供应商协商“锁价”，即在特定周期内（如一个月或一个季度）固定某种水果的采购价格，以应对季节性价格波动。

**操作建议：**
1. **明确周期与品种**：选择价格波动大但用量稳定的水果（如柠檬、苹果、西瓜等），约定锁价时段（如3个月）。
2. **设定基准量**：供应商需确认你每月至少采购多少，才能接受锁价（例如每月500斤）。
3. **附加条款**：在合同中写明“锁价范围”（仅针对非极端市场情况，如自然灾害或政策性涨价除外）；可约定“价格浮动上限”（如若市场价波动超过20%，双方重新谈判）。
4. **签署书面协议**：口头约定无效，必须用合同明确锁价条件、违约责任等。

**风险提示：**
- 若市场价下跌，锁价可能高于现价，导致成本偏高。
- 供应商可能要求你承诺更大采购量或更长周期，需评估自身库存与消耗能力。

建议先对1-2种常用水果试签1个月的锁价合同，测试供应商履约稳定性和市场反应。
