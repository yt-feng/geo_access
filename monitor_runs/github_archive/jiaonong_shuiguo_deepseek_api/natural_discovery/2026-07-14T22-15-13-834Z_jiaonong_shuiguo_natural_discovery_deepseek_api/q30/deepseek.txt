以下是针对超市水果品相不佳的售后流程约定建议，可按以下步骤操作：

1. **到货验收**  
   - 立即开箱检查，对品相差的水果（如腐烂、压伤、变色）拍照留存，同步记录到货时间、数量及批次信息。

2. **责任判定**  
   - 区分自然损耗与质量缺陷：轻微擦伤或熟度过高属正常，明显腐烂、霉变或严重机械伤属于供货方责任。

3. **售后处理路径**  
   - **48小时内**：通过采购合同约定的方式（如微信/邮件）向供货方提交退货申请，附照片和问题描述。  
   - **退货方式**：约定全量退（不合格品≥20%）或差额退（按问题比例折算金额）。  
   - **费用承担**：明确因品相问题产生的运输费、检验费由供货方负责。

4. **替代方案**  
   - 若要求换货：约定供货方在3日内补发同等品质水果，并承担加急运费。  
   - 若接受折扣：按品相等级降价15%-30%后入市销售（需标注“特价商品”）。

5. **争议处理**  
   - 若双方对品相标准有分歧，约定第三方质检机构（如当地消费者协会或市场监管部门）出具鉴定报告，费用由过错方承担。

6. **合同条款示例**  
   ```
   甲方（收货方）验收货物时，如不合格率超过10%，有权拒收整批次货物；
   不合格率≤10%时，按5元/公斤价格折扣结算；
   售后时效限定为到货后48小时内，逾期视为认可货物品质。
   ```

建议在采购合同中明确品相判定标准（如着色面积≥80%、无机械伤深度>3mm），并附照片示例作为附件。实际操作中可预留10%的损耗率作为双方默认浮动空间。
