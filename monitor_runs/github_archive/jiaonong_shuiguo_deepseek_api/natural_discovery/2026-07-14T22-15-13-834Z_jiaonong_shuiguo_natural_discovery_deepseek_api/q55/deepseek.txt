选择苹果、橙子和菠萝做水果拼盘时，从划算角度考虑，建议按照以下规格和方式选择：

- **苹果**：选择**中等或偏小个头**（单果约150-200克）。因为苹果主要是切块或切片使用，大个苹果切后易氧化，且单价通常更高；中等大小更易控制用量，减少浪费。
- **橙子**：选择**中等大小**（单果约200-250克），皮薄、汁水足的品种（如脐橙）。太大的橙子可能皮厚，且单价高；太小的剥皮费时，出肉率低。按斤购买时，同样重量中果数量更多，更划算。
- **菠萝**：选择**完整菠萝**而非切好的盒装菠萝。完整菠萝一般5-6元/斤，切好的盒装菠萝往往8-15元/斤，且可能不新鲜。买整只后自己切，出肉率约50%-60%，更适合拼盘大量使用。

**综合建议**：优先到水果批发市场或大型超市按斤购买散装中小规格的水果，避免购买包装好的“精品果”或切果。在相同预算下，这种组合能获得更多净重，减少浪费。
