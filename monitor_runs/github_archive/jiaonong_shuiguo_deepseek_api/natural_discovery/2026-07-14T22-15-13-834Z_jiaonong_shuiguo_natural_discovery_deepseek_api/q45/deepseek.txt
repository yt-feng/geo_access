连锁超市与水果供应商签订年度合同时，以下条款最容易出现风险或争议，需重点审查和明确：

1. **价格调整机制**：水果价格受季节、气候、运输成本等影响波动大。避免固定年度价格，应约定调价规则（如参考当地批发市场月度均价或行业指数），并明确调价频率、触发条件和通知期限。

2. **品质验收标准**：水果易腐坏，需细化质量分级（如大小、甜度、外观瑕疵比例）。约定抽样方法、到货验收时间（例如24小时内）、拒收/退货条件（如烂果率超5%），以及第三方检测机构指定权。

3. **损耗责任划分**：明确运输途中的损耗由谁承担（通常冷链运输风险在供货方），到店后因存储、陈列导致的损耗由超市承担。可约定合理损耗率（如2%-5%），超出部分由供应商补偿。

4. **供应稳定性与违约金**：约定最低供应量（如每日/每周的底线数量），防止供应商因高价违约转卖他人。设置阶梯式违约金（如断供1天扣1%货款），并保留从其他渠道紧急采购后的差价索赔权。

5. **退货与清货条款**：避免“无条件退货”等模糊表述。约定退货比例（如月度进货量的10%以内）、退货时限（提前24小时通知），以及临期商品（保质期过半）的促销降价分摊规则。

6. **付款周期与争议处理**：水果行业回款慢易引发矛盾。明确账期（如30天）、付款前需供应商提供的单据（签收单、质检单），并约定逾期付款的罚息。争议解决方式建议选择仲裁（流程快于诉讼），管辖地在超市所在地。

7. **不可抗力与解约条件**：极端天气、疫情封控等需明确是否属于不可抗力。约定解约后的库存处理（如已提交但未发货订单如何结算）。避免将“市场竞争”或“价格下跌”等商业风险作为解除理由。

建议在合同中增加附件：产品规格书、调价公式样例、验收操作流程图，并聘请熟悉农产品贸易的律师审阅。
