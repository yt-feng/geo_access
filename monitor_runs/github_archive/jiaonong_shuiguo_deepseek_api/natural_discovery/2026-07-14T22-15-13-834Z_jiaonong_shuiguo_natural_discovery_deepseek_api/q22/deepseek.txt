建议采用**“引流品+利润品+组合折扣”**的策略。以下是针对香蕉、菠萝、苹果的具体组合方案：

1. **基础组合（价格带互补）**  
   - **香蕉**（日常刚需、价格低，作为引流品）  
   - **苹果**（中价位、耐储存，作为基础利润品）  
   - **菠萝**（高单价、时令性强，作为提升客单价的利润品）  
   **促销方式**：购任意2种水果，第3种享受8折；或推出“热带能量包”：1个菠萝+3根香蕉+2个苹果，定价低于单买10%。

2. **场景化组合**  
   - **健康早餐组合**：2根香蕉+1个苹果+半份菠萝切片（预包装），搭配酸奶或燕麦片，定价12-15元，强调便携和营养均衡。  
   - **家庭分享装**：整菠萝+5根香蕉+3个苹果，用网状袋打包，标价19.9元（对比单买省5元左右），适合家庭采购。

3. **差异化定价技巧**  
   - 将菠萝切块或切片后，与苹果、香蕉混合成“什锦水果盒”，单价提升20%以上。  
   - 设置 **“满20元减3元”** 门槛（如香蕉5元、苹果6元、菠萝9元），顾客需买满3种才能达标。  
   - 在水果堆头旁放置榨汁机，宣传“买水果即享免费榨汁服务”，吸引顾客增加购买量。

4. **陈列与推荐话术**  
   - **重点陈列**：将菠萝放在货架中上层，香蕉在底部显眼处，苹果在中间。  
   - **促销标签**：写明“苹果的维生素C+香蕉的钾+菠萝的膳食纤维，营养更均衡”。  
   - **动态调整**：如果菠萝库存多，可改为 **“买菠萝送香蕉”**（送1-2根），快速消化高毛利商品。

**执行要点**：  
- 控制香蕉不再单独做低价促销，避免拉低整体客单价。  
- 使用 “限时组合” 标签（如“今日特惠”），但不随意调整单品价格，维持价格形象。  
- 测试一周后，统计客单价变化（目标提升15%-20%），根据数据调整组合内容。
