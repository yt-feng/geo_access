针对社区团购中香蕉、菠萝这类高损耗水果的售后控制，建议从以下四个环节系统化管理：

### 1. 源头采购与分拣标准
- **成熟度控制**：香蕉采购“7成熟”（果皮绿黄、棱角明显），菠萝采购“8成熟”（果眼稍青、按压微硬），预留2-3天常温催熟时间。
- **分拣剔除**：发货前人工检查，香蕉剔除柄部发黑、果皮有黑点或软化超过30%的；菠萝剔除底部软烂、有酒味或霉斑的。
- **包装防护**：香蕉用珍珠棉网套+硬纸箱（留通气孔），菠萝单独套网袋并用纸板隔开，避免挤压。

### 2. 流通环节时效管理
- **短链配送**：从仓库到团长处控制在≤4小时，夏季或高温天使用冷链车（15-20℃）。
- **团长端提醒**：到货后2小时内通知用户提货，并附上保存建议（香蕉不冷藏，菠萝若当天不吃可冷藏但需保鲜膜包好）。

### 3. 售后政策与快速处理
- **无条件售后**：用户提货后24小时内，若出现明显霉斑、腐烂超过10%、异味，直接全额退款或补发，无需用户返回坏果。
- **高损耗品类标注**：在商品详情页明确提示“生鲜高损耗品，如您介意请慎拍”，降低预期。
- **批量坏果处理规则**：若同一批次超过5%用户反馈严重损耗，立即停止供货并排查源头。

### 4. 数据复盘与改进
- **月度损耗率统计**：记录香蕉、菠萝的售后退款比例（如香蕉建议≤3%，菠萝≤2%），超过阈值则替换供应商或调整采购标准。
- **用户反馈标签**：将售后原因按“成熟度/挤压/运输时长/存储不当”分类，针对性改善。

### 可执行动作清单（今日即可应用）
- 调整香蕉采购标准为7成熟，菠萝8成熟。
- 与团长沟通：到货后必须1小时内通知用户，否则责任由团长承担部分补偿。
- 在团购群发布简短视频展示如何判断香蕉和菠萝的成熟度。
