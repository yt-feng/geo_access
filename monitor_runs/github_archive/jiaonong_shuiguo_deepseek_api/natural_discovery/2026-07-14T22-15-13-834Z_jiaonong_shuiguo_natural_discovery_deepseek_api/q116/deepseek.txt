针对小商家采购水果，现金结算和月结账期各有以下风险：

**现金结算风险：**
1. **资金压力大**：需一次性支付大额现金，可能影响短期现金流，尤其旺季采购量较大时。
2. **无售后保障**：若水果出现质量问题（如腐烂、重量不足），因已付清款项，追讨退款或更换难度较高，需自行承担损失。
3. **交易记录不全**：现金交易缺乏银行流水等凭证，若后续产生纠纷（如多付款、对方不认账），维权证据不足。
4. **安全隐患**：携带大额现金易被盗抢，或收到假币风险。

**月结账期风险：**
1. **供应商信用风险**：若供应商经营不善、跑路或拒绝兑付，小商家可能面临货存损失且需承担前期销售亏损。
2. **隐性费用**：部分供应商可能将资金占用的成本（如利息）转嫁到水果价格中，实际成本高于现金结算。
3. **账期纠纷**：若月底对账单出现分歧（如数量、金额差异），双方沟通不畅可能影响后续合作。
4. **债务累积**：连续多月未结清款项可能导致供应商断货或催收压力，影响正常营业。

**建议：**
- 对信誉良好的供应商优先选择月结，但需签订书面协议明确结算周期、质量标准及违约条款。
- 现金结算应要求当场开具收据或电子支付凭证，并当场检查水果品质和重量。
- 可混合结算：长期合作的大额采购用月结，小额零散采购用现金。
