对普通商家来说，介绍产品时不要用太专业的农业术语，而是要用**卖货的语言**和**算账的逻辑**。以下是几条可操作的建议：

1. **直接说“卖点”而不是“品种名”**  
   比如，不要只说“这是金钻凤梨”，要换成“这个切开不用挖眼、不泡盐水，客人直接啃，省事又好卖”。把品种特性翻译成商家关心的**损耗低、操作快、回头客多**。

2. **对比常见水果，量化好处**  
   比如介绍“牛奶西瓜”时，可以说“和普通西瓜比，这个皮更薄，出肉率多15%，批发价只贵5毛，但零售能多卖两块”。直接给**利润差**，商家自己会算账。

3. **用场景告诉对方“怎么卖”**  
   例如介绍“手撕小菠萝”时，说“不用刀削皮，直接手撕就能吃，适合放奶茶店、甜品铺或者摆摊，拍照也好玩，年轻人抢着买”。让商家立刻想象出售卖环节。

4. **讲“抗折腾”的能力**  
   商家最怕水果不耐放。可以说“这个香蕉青皮发货，到店能放5天不黑，就算皮皱了，里面肉还是甜的，损耗比普通香蕉低一半”。用**天数+状态变化**证明耐久性。

5. **给具体销售建议提升信任**  
   比如“熟透的芒果可以改做切块盒装，卖不完的还能做果酱”，或“这个小番茄摆摊可以按杯装，一杯卖10元，成本不到3元”。直接帮对方**降风险、增利润**。

一句话总结：**把水果讲成“赚钱工具”，而不是“农作物”**。用数字、场景和损耗控制来打动商家。
