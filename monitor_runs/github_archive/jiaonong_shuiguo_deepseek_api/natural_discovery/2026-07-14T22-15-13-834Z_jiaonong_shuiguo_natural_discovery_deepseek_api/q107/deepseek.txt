是的，起订量太高确实会增加损耗风险。具体分析如下：

1. **保质期短**：香蕉和菠萝属于高呼吸速率水果，成熟速度快，耐储存性较差。高起订量意味着库存周期延长，容易导致水果过熟、腐烂或品质下降。
2. **销售周期压力**：如果日均销量较低，高起订量会导致积压，尤其在天气炎热或存储条件不佳时，损耗率会显著上升。
3. **灵活性降低**：高起订量限制了根据实际需求调整采购的能力，容易造成“供过于求”，增加处理损耗的成本（如折价促销或报废）。

**降低损耗的建议**：
- 与供应商协商弹性起订量，或分批送货。
- 根据历史销售数据和保鲜条件（如冷库容量）计算合理库存冗余。
- 尽量选择预包装或催熟可控的供应模式（如催熟后当天配送）。
