针对生鲜电商，建议优先选择 **整果** 和 **冷冻原料**，切配菠萝需谨慎。具体分析如下：

1. **整果**：最适合线上销售。因为菠萝有天然外壳保护，不易在运输中破损；而且消费者能直观判断新鲜度（外观、颜色、香气），退货风险低；存储时间长，适合常规快递。
   - **注意**：需控制成熟度（7-8分熟发货），避免过熟变质。

2. **冷冻原料**：适合追求复购率和低损耗的品类。冷冻菠萝块或果肉可直接用于做菜、烘焙或冰沙，保质期长（6-12个月），且无需担心运输变质。适合有冷冻条件的商家。

3. **切配菠萝**：不推荐。因为切配后氧化变色快（需真空包装+冷链），且菠萝中的菠萝蛋白酶会在切割后加速组织软化，24小时内口感变差；另外，切配品容易出现重量缩水、卫生投诉等问题，退货率较高。

**最终建议**：  
- 如果主打鲜食市场，主推 **整果**，搭配其他“方便处理”的促销文案（如附赠水果刀、开菠萝教程视频）。  
- 如果想拓展即时消费或家庭烹饪场景，上线 **冷冻菠萝块**，标注“去芯处理”“免切速冻”等卖点。  
- 若非要卖切配品，必须使用气调包装（MAP）和全程冷链配送，且限法定配送区域（同城2小时达），且单价需高于整果30%以上才可能盈利。
