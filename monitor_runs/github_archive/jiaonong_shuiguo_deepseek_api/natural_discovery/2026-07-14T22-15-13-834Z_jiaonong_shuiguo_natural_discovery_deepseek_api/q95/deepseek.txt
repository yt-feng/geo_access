选择学校食堂水果供应商时，建议按以下步骤操作：

1. **明确采购核心要求**  
   - **食品安全**：供应商需提供营业执照、食品经营许可证，水果需有农残检测报告或产地证明。  
   - **新鲜度**：要求供应商承诺每日配送，且卸货前现场验收（如香蕉表皮青黄适宜、苹果无压伤、橙子果蒂新鲜）。  
   - **供应稳定性**：考察供应商是否有稳定的种植基地或长期合作果园，能保证全年供应（尤其规避反季节水果断货风险）。  

2. **筛选本地供应商优先**  
   - **本地果蔬批发市场**：直接联系市场里的大型批发商，可要求试送少量货品（例如2箱香蕉）测试品质和配送效率。  
   - **附近果农合作社**：若学校周边有产区（如南方橙子产地、北方苹果基地），直接合作可减少中间环节，2-3家比价。  

3. **关键资质审核点**  
   - 要求提供《食用农产品合格证》或第三方检测（至少检测“农药残留”“重金属”等基础项）。  
   - 确认供应商是否有冷链运输能力（尤其香蕉需5-8℃储存运输，否则易过早成熟腐烂）。  

4. **合同与验收条款**  
   - 合同明确：  
     - **品质标准**：苹果直径≥7cm、无黑斑；橙子糖度≥12%（可带糖度计抽查）；香蕉按“黄绿比例≤1:5”验收（黄果比例过高易烂）。  
     - **退换货规则**：24小时内不合格品（如烂果率＞5%）由供应商无条件换货。  
   - 每日现场验收：抽检至少10%的数量，烂果率超过约定阀值则整批拒收。  

5. **价格对比与成本控制**  
   - 要求报价含增值税发票和运费（通常本地供应商报价比市场价低10%-15%）。  
   - 建议按**周比价**，锁定2-3家稳定供应商签订短期合同（如1-3个月），避免长期独家导致品控下降。  

6. **试运营期**  
   - 新供应商前2周小批量（如每周采购100kg测试品种），观察配送准时性、水果消耗率（学生反馈丢弃率是否高于20%）。  

**注意事项**：  
- 避免接受果农“先赊后付”的现金结算方式，坚持对公转账并索要正规发票。  
- 若校方要求低价，可引导优先采购应季水果（如秋季苹果、冬季橙子，夏季西瓜），反季水果可能因存储问题产生隐形损耗成本。
