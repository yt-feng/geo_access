选择适合批量配送的水果供应商，可从以下方面考量：

1. **规模与稳定供应能力**：优先选择有固定果园合作或大型批发市场直供的供应商，确保能按需批量发货，避免出现缺货或断供。可要求对方提供过往类似企业合作案例。

2. **配送与时效服务**：确认供应商是否有冷链配送能力，能否按约定时间（如下午茶前1-2小时）送达。重点关注运输过程中对水果的防护（如防撞包装、温控），避免损耗。

3. **水果品质与损耗率**：要求提供样品试用，检查新鲜度、成熟度、大小均匀度。了解售后政策（如对明显变质腐烂的赔付比例或补发），避免批量到货后出现品控问题。

4. **价格与成本**：比较2-3家供应商的报价，关注综合单价（含配送费、包装费）。批量采购可协商阶梯价或月结优惠，但避免为低价牺牲品质。

5. **灵活性与定制服务**：看对方能否提供水果切盘、拼盘或应季水果组合（如夏季选西瓜、葡萄，限定选蜜桔等），适应下午茶场景需求。

6. **口碑与合规**：通过大众点评、企业评价或同行推荐筛选供应商，优先选择有营业执照、食品经营许可证的单位。可要求提供产品检疫或合格证明。

**行动建议**：先小批量试单（如50人份），测试配送时间、品控和售后响应。再根据实测结果决定长期合作对象。若预算有限，可考虑本地批发市场+第三方配送组合。
