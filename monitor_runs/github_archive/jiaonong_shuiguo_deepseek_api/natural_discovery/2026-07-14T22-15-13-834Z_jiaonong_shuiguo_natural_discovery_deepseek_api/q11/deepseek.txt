会影响的。水果店销售香蕉时，如果同一箱内大小差异明显，通常会产生以下问题：

1. **影响定价和标价**：顾客通常期望同等价格下商品品质一致。大小不一容易让顾客觉得“花同样的钱买到了大小不等的商品”，降低购买意愿。
2. **增加挑拣成本**：顾客可能会着手挑拣大香蕉，剩下小香蕉难以卖出，导致损耗增加。店员也可能需要提前分拣，增加人工成本。
3. **降低整体卖相**：大小混杂的香蕉放在一起显得杂乱，不如整齐划一美观，影响陈列效果。
4. **打乱利润结构**：如果按整箱定价，可能大香蕉卖便宜了，小香蕉卖贵了（或不好卖），影响利润。

**建议做法**：
- 进货时优先选择大小均匀的整箱，或向供应商明确要求“同箱大小相近”。
- 如果已进到大小不一的香蕉，可以在上架前进行人工分拣，按大小分成两个区或两个价格档位（如大香蕉贵一点，小香蕉便宜一点或作为促销品）。
- 也可以将小香蕉作为“烘烤/榨汁/切块”用途提前处理，避免占用正常销售位。

简单判断：一箱香蕉大小差异超过20%（比如一根比另一根长1/3以上），就很可能影响销售。
