为确保采购水果符合农残安全要求，建议向供应商索要以下关键资料：

1. **第三方农残检测报告**：要求提供近期（建议1-3个月内）由具有资质的检测机构（如SGS、华测、农业农村部下属检测中心等）出具的农残检测报告，需包含具体检测项目（如国家强制性标准GB 2763中的常见农药）、检测结果及判定结论。

2. **产地证明或追溯码**：获取明确的水果产地信息（如具体产区、种植户/合作社名称），或带有追溯码的包装标签，以验证产品来源可追溯性。

3. **供应商资质文件**：营业执照、食品经营许可证（如适用）、生产/种植许可证（如有机认证、绿色认证等），确认其合法经营资质。

4. **批次合格证明**：针对该批水果的出厂检验报告或进货查验记录，证明该批次已自查农残符合要求。

5. **农产品合格证或承诺书**：部分产地政府要求开具的准出证明，或供应商签署的“农残安全承诺书”（需包含违约责任条款）。

**操作建议**：
- 要求供应商在送货时同步提供以上资料纸质版或电子版，并留档备查。
- 每批次水果到货后，建议超市自行随机抽样进行快速农残检测（如试纸法），并与供应商报告比对。
- 若供应商无法提供有效报告，建议建立黑名单或降低采购优先级。
