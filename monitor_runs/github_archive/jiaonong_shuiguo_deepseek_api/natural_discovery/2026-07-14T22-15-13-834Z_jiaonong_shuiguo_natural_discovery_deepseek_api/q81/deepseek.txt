建议支持。原因如下：

1. **匹配消费者需求**：即时零售用户通常购买少量水果（如1-2个苹果、半斤草莓），一次性备足大货容易导致库存积压和损耗。小批量高频补货能灵活匹配实际订单。

2. **降低损耗风险**：水果易腐烂，批次越少、库存周期越长，损耗率可能超过15%。高频补货可减少单次库存量，保持新鲜度。

3. **提升平台竞争力**：平台倾向选择能快速响应补货的供应商。支持小批量补货意味着配送更快（如30分钟达），有利于获得更多推荐和订单。

4. **验证市场**：初期可先试点1-2款高周转水果（如香蕉、苹果），观察补货频率和实际销量，再逐步扩展到其他品类。

操作建议：
- **设定最小起订量**：例如每单不少于2公斤或5份，避免零散配送成本过高。
- **合理定价**：小批量补货单价可略高于大单，覆盖分拣和配送成本。
- **提供次日达或定时补货**：与平台协商每日固定时段（如早上、下午）集中配送，提高效率。

如果担心利润，可对比现有补货模式（如每周1-2次大批量）下的损耗成本与补货频次增加后的物流成本，通常高频补货反而能降低总成本。
