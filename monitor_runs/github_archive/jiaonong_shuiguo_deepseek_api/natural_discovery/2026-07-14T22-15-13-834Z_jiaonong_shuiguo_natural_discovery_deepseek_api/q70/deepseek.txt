在合同中约定坏果责任时，建议明确以下几点以降低批发商风险：

1. **定义损耗标准**：明确“坏果”的具体标准（如腐烂、霉变、虫蛀等），并约定可接受的损耗率（例如“运输途中坏果率不超过5%”或“超过2%即视为超标”）。

2. **责任划分**：明确坏果由谁承担。建议写“运输途中的坏果风险由卖方/承运方承担，但买方需在收货后24小时内提供坏果照片及数量证明”。或约定“货到后由双方共同检验，坏果数量超出约定比例的部分由卖方按单价赔偿”。

3. **检验与索赔程序**：写明收货后检验时限、方式（如抽样检查、拍照留存）。例如：“买方应在到货后2小时内开箱验货，超过该时间视为默认接受货物质量；对坏果索赔需提供清晰照片及第三方证明”。

4. **损耗扣款计算方法**：明确计算方式，如“超出约定损耗率的部分，直接按合同单价的100%从应付货款中扣除；低于约定损耗率的，卖方无需补偿”。

5. **免责条款**：注明不可抗力（如极端天气、道路事故等）导致的坏果可部分免责，但需提供证据。

**示例条款**：
> “货物到达目的地后，双方应在2小时内共同开箱验收。若坏果率超过合同总箱数的3%，超出部分由卖方按该批货物单价全额赔偿。买方需在验收后立即拍照留存并提供电子凭证，否则视为货物完好。因不可抗力导致的坏果，卖方免责但需提供有效证明。”

建议签约前与卖方协商，明确双方责任比例，避免模糊表述（如“合理损耗”）。最后确认合同是否需经法律人士审核，尤其是大额订单。
