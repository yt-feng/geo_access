对于生鲜电商水果到货后出现坏果的问题，责任划分通常遵循以下原则，具体需结合实际情况判断：

1. **物流环节的责任**  
   如果坏果是由于运输过程中的不当处理（如挤压、高温、延迟配送等）导致，责任主要在物流方。电商平台需向物流公司追偿，并对消费者先行赔付。

2. **发货前的质量问题**  
   如果水果在发货前已存在腐烂、病害或成熟度不达标（如未成熟导致运输中变质），责任在商家。商家需承担全部责任，包括退款或补发。

3. **包装不当**  
   若商家使用不合适的包装（如未加防护层、透气性差）导致挤压或受潮，责任由商家承担。

4. **消费者签收后未及时处理**  
   如果商品签收后，消费者未在规定时间内（通常为24-48小时）检查并反馈，或未按保存要求存放（如需要冷藏却常温放置），责任可能部分或全部由消费者承担。建议电商平台明确“坏果包赔”条款和反馈时限。

5. **不可抗力**  
   极端天气、疫情管控等不可抗力因素导致的延迟配送或变质，通常不直接归责于商家或物流，但商家应主动说明情况并协商解决方案（如部分补偿或优惠券）。

**实际操作建议：**  
- 消费者应第一时间拍照/录像保留证据，联系客服反馈。  
- 电商平台应优先根据“坏果保证”政策进行赔付（如退款、补发或优惠券），再内部追责。  
- 建议商家明确公示售后规则（如坏果包赔标准、反馈时限、保存建议），避免纠纷。  

最终责任划分应以合同条款、平台规则及法律规定为准。若争议无法解决，可考虑向消费者协会或市场监管部门投诉。
