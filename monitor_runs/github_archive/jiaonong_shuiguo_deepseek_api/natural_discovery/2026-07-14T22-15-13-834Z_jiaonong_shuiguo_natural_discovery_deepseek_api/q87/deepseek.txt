旺季爆单前锁定库存，建议分三步走：

1. **签订保量协议**：直接与供应商签订旺季“保量不涨价”或“优先级供货”协议，明确锁定**最低起订量**和**最大供应量**。重点写明若供应商断供，需承担相应赔偿（如按订单金额的20%赔偿），避免临时变卦。

2. **预付定金+分批结算**：支付一笔定金（如总货款的30%-50%），作为“排他性锁定费”，要求供应商将指定批次的水果优先发给你。后续按实际交货量分批结算余款，既保证库存，又分摊你的资金压力。

3. **建立动态补货机制**：提前2周发一份“预估爆单爆品清单”（含品名、日销量、周期），要求供应商按清单预留产能或冷库容量；同时约定每3天一次小批量补货，避免一次性囤货过多导致损耗。

**重要提醒**：旺季水果价格波动大，最好在合同里加一句“若批发市场价上涨超20%，双方协商调价不超过10%”，平衡双方风险。同时备选2-3家供应商以防意外断供。
