为确保茶饮店食品安全，水果供应商通常需要提供以下关键证明文件：

1. **营业执照**：证明供应商具有合法经营资质。
2. **食品经营许可证**：或食品生产许可证（若供应商为生产商），确保其具备食品销售或生产的许可。
3. **水果产地证明**：明确水果的来源地，尤其是对进口水果需提供海关检疫证明及入境货物检验检疫证明。
4. **产品检验合格报告**：由第三方检测机构出具，涵盖农药残留、重金属、微生物等指标，需定期更新（如每批次或每月）。
5. **承诺书/背书**：供应商书面承诺遵守食品安全法规，并对其所供水果的安全性负责。
6. **运输与仓储证明**：如冷链运输记录、温度控制证明，确保水果在运输和储存过程中未变质。

此外，建议茶饮店要求供应商提供追溯信息（如批次号），并建立进货台账，记录供应商名称、联系方式、来源地及证明文件有效期。定期现场审核供应商也属间接控制风险的有效措施。
