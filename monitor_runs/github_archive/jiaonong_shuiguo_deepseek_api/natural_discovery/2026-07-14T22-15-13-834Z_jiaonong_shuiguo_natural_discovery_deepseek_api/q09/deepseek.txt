在水果店旺季与供应商合作前，建议重点确认以下问题：

1. **供应能力与周期**  
   - 旺季日均最大供货量是多少？能否按订单优先补货？  
   - 从下单到到货通常需要几天？遇到极端天气或运输问题时如何调整？

2. **价格与结算机制**  
   - 淡季和旺季的单价是否有浮动空间？长期合作有无阶梯价格优惠？  
   - 货款支付方式是现结、周结还是月结？是否接受部分账期？

3. **品质与退换货规则**  
   - 能否提供符合国家标准的果品等级（如大小、甜度、外观）？  
   - 到货后出现腐烂、破损或品质不符时，如何退换？损耗率超过多少可索赔？

4. **物流与损耗责任**  
   - 运输过程中损坏、失水、变质由谁承担？是否提供冷链车直达？  
   - 若因配送延迟导致断货，是否有赔偿或补发机制？

5. **独家合作与违约条款**  
   - 能否承诺不优先向周边竞争门店供货？  
   - 若供应方擅自涨价或断供，违约金如何计算？

6. **备选方案与应急措施**  
   - 如果主供品种断货，能否协调其他地区货源或替代品？  
   - 是否提供紧急调货通道？最快处理时效是多久？

7. **合作保障**  
   - 能否签订书面合同并注明上述条款？  
   - 是否有长期合作案例或历史供货数据可供参考？

---
**建议**：首次合作可先小批量试单，验证品质与流程；旺季前1-2个月敲定合同细节，并保留至少2-3家备用供应商。
