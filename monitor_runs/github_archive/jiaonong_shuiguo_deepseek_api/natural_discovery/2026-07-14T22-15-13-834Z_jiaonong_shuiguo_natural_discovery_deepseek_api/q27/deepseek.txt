在门店销售中，判断苹果和橙子的大小规格是否适合，建议参考以下标准：

1. **苹果**  
   - **大果**：直径约8-9厘米（适合高端零售或整箱销售）。  
   - **中果**：直径约7-8厘米（最常见，适合家庭日常消费）。  
   - **小果**：直径约6-7厘米（适合促销、切盘或低单价销售）。  

2. **橙子**  
   - **大果**：直径约7-8.5厘米（适合礼品装或高端渠道）。  
   - **中果**：直径约6-7厘米（主流规格，口感与性价比平衡）。  
   - **小果**：直径约5-6厘米（定价偏低，适合散称或特价）。  

**实际操作建议**：  
- 用**水果专用测径环**或**卡尺**现场抽样测量（每箱测5-10个）。  
- 对比**同类竞品**或根据**历年销售数据**，优先选择该位置门店的主流规格（如社区店中果，商业区大果）。  
- 注意**外观统一**：同规格误差不超过0.5厘米，避免大小混装影响陈列效果。  

**示例**：如果门店用户多为家庭主妇，橙子选6-7厘米的中果更易动销；若主打节日礼盒，则选8厘米以上大果配高档包装。
