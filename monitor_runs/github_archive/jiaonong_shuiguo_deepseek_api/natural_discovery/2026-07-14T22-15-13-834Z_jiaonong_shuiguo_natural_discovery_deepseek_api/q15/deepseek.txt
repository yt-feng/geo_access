遇到水果店进货出现坏果的情况，与供应商结算或赔付的标准流程通常取决于双方事先的约定。以下是常见且合理的处理方式，建议按以下步骤操作：

1. **区分坏果类型**：首先确认坏果是运输途中造成的（如挤压、碰伤、高温腐烂），还是水果本身的质量问题（如内部霉变、虫蛀、成熟度不够）。不同原因责任方可能不同。

2. **查验合同与单据**：查看进货合同或单据中是否有关于坏果率的免责条款（例如“允许5%以下自然损耗”），以及具体的赔付标准（如“超过5%部分按进货价全额赔偿”）。如无书面合同，应按行业默认习惯处理。

3. **现场取证与沟通**：当场拍摄坏果照片或视频，保留原始包装和运输凭证。立即联系供应商，说明情况，要求对方承认事实或提供解决方案。一般建议24小时内沟通（不同水果保质期短，逾期可能被拒）。

4. **常见的结算/赔付方式**（按优先级排序）：
   - 直接扣除：在当次货款中扣除全部坏果金额（按进货价）。
   - 补货替代：供应商免费补发等量好果，或下次进货时按坏果数量冲减货款。
   - 比例赔付：如果坏果比例较小（如3-5%），且供应商信誉好，可协商由供应商承担一半损失，双方各认一部分。
   - 部分折扣：供应商对整批水果给予一定折扣（例如折扣5%），不再单独处理坏果。

5. **关键操作建议**：
   - 优先采取**拍照+视频+称重**的方式留存证据。
   - 最好通过微信、短信等留下文字沟通记录，避免口头承诺。
   - 如果供应商拒绝或拖延，可要求第三方（市场管理方、冷链物流公司）介入协调。

6. **法律底线**：如果坏果比例很高（如超过10%且属于质量隐患），且供应商拒绝赔付，可以考虑根据《民法典》合同编向当地市场监督管理局投诉，或依据合同约定申请仲裁。但日常小规模水果店建议优先协商解决，避免成本过高。

**最终建议**：在未特别约定时，商业惯例是**供应商对运输和收货时发现的明显坏果承担100%责任**；对于收货后1-2天内出现的腐败变质，需由供应商承诺退货或赔偿。建议你从**首次合作就确认好赔付规则**，并写入供货单。
