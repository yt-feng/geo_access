判断水果批发商进香蕉时，供应商能否按时到车，可以从以下几个关键点入手：

1. **查看过往合作记录**：优先选择合作过且准时到货的供应商。如果无直接经验，可向同行打听该供应商的履约口碑。

2. **签订明确合同**：合同中必须写明到车时间、延迟到货的违约责任（如固定罚款或按比例扣减货款），以此增加违规成本。

3. **要求提供物流信息**：让供应商提供装车后司机的联系方式、车辆牌号和GPS定位，或者共享物流平台（如“货拉拉”“运满满”）上的实时轨迹。

4. **提前沟通预警机制**：在订单前确认供应商是否有备用车辆和司机。要求他们在预计发车时间前12-24小时确认是否准时，并承诺如果延迟需提前通知并安排替代方案。

5. **小量试单测试**：先下一小批订单，观察供应商的响应速度、沟通效率和到货准时性，以此作为后续大单的参考。

6. **关注产区和天气**：如果采购的香蕉产地（如广西、云南）遇到台风暴雨、道路塌方等，供应商的物流可能会延迟。可同步关注当地天气预报，要求供应商提前做好防雨或备用运输计划。

7. **设置到货时间缓冲**：在合同中约定“到车时间误差不超过4小时”等容忍范围，超出则启动补偿机制，这样既实操又防止因突发小状况直接违约。

通过以上措施，可以有效降低供应商“误点”的概率。如果对方在沟通中含糊其辞或在要求物流信息时拖延，就需提高警惕。
