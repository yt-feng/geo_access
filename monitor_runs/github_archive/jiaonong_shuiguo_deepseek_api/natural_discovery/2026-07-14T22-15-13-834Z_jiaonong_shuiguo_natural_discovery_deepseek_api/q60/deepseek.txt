针对茶饮店水果到货不合格的情况，为了不影响营业，建议采取以下步骤与供应商约定补货：

1. **立即沟通并留证**：发现不合格后，第一时间拍照、视频保留证据（如腐烂、破损、规格不符等），并立即通知供应商确认问题。明确要求供应商在**4-6小时内**给出补货方案（如换货或退款），避免影响次日营业。

2. **约定补货时效**：与供应商签订补充协议，明确“不合格批次需**当日或次日上午**前完成补货”，例如“上午10点前确认问题，下午3点前送达”。如果距离近，可要求2小时内配送。

3. **设定应急替代方案**：提前约定“若补货延迟超2小时，供应商需承担紧急采购差价”（如从周边市场临时购买）。同时，让自有配送或员工应急去批发市场采购少量合格水果（成本计入供应商违约赔偿）。

4. **明确验收标准**：签订合同时细化“重量误差不超过5%、无霉变、成熟度达标”等量化指标。例如“青柠直径≥5cm，柠檬表面无褐斑”，避免模糊表述。

5. **建立动态库存预警**：备足1-2天最低使用量，遇到不合格时优先消耗库存。比如将不合格部分用于调制果酱（若仅外观瑕疵），或调整当天菜单（如减少相关饮品主推）。

6. **设定快速补货流程**：与供应商协商“直送门店”模式（不经中间分拣），并指定门店专人验收。例如：供应商备好备用批次，收到退货通知后1小时直达门店。

7. **风险分摊条款**：在供应商合同中加入“每月可允许2次不合格补货，超过则按货值20%罚款”，并约定“补货运费由供应商承担”。补货到店后需再次现场验货，避免二次不合格。

**示例操作**：发现芒果有磕伤后，立即通知供应商，并启动备用：从冰箱取出预留的冷冻芒果块（用于做沙冰），同时要求供应商在下午2点前补送一箱合格品。若延误，则从该批次货款中扣除紧急采购的差价+10%管理费。

做到以上，既能保证营业连续性，又能约束供应商质量，且所有约定需书面补充到采购合同或协议中。
