在超市采购水果时，为确保食品安全并满足国家法规要求，供应商通常需要提供以下检测报告或进货证明：

1. **产地证明**：证明水果原产地或来源地，例如由当地农业部门、合作社或种植户出具。

2. **农药残留检测报告**：提供第三方检测机构出具的农药残留检测合格报告，符合国家《食品安全国家标准 食品中农药最大残留限量》（GB 2763）的要求。建议每批次或定期提供。

3. **产地检疫报告（针对进口水果）**：如果是进口水果，需提供输出国官方检疫部门出具的原产地检疫证书，以及中国海关的入境检验检疫证明（如《入境货物检验检疫证明》）。

4. **产品合格证或出厂检验报告**：由供应商（如批发商、加工企业）出具，证明水果符合国家或行业相关质量等级标准（如外观、成熟度等）。

5. **进货单据或购销合同**：包括发票、收据或合同，证明水果来自合法、可追溯的供应链。

6. **其他特殊检测（如重金属、添加剂）**：根据超市所在地区的要求或水果品种（如易受污染的水果），可能需提供重金属、微生物或防腐剂残留的检测报告。

建议超市采购部门在合同中明确要求供应商在供货时附带上述报告，并保留复印件或电子备案。对于高风险水果（如草莓、葡萄等），可要求更频繁的检测抽查。同时，需注意报告有效期，一般不超过一年或一个收获季。对于进口水果，务必核对检疫证明的真实性，可在中国海关网站查询。
