与酒店早餐水果供应商合作时，需重点关注以下几点：

1. **品质标准明确化**  
   - 明确水果新鲜度、成熟度、外观规格（如大小、色泽、无损伤腐烂）。  
   - 要求供应商提供农残检测报告、产地证明等资质文件。  
   - 对易损耗水果（如草莓、蓝莓）提前约定可接受损耗率（如不超过5%）。

2. **供应稳定性与应急预案**  
   - 要求供应商提交备选货源方案，确保极端天气或季节短缺时仍可供应。  
   - 在合同中签订“缺货补偿条款”（如缺货时按货值双倍赔付）。  
   - 设定最低及最高库存预警线，供应商需提前24小时通知价格波动或断供风险。

3. **价格与结算模式**  
   - 签订浮动价格协议，参考当地批发市场均价调整（如每周/每月更新）。  
   - 约定账期（如月结30天）及现金折扣（如提前付款享2%优惠）。  
   - 避免固定长期合同，建议按季度或半年重新议价，保留竞争性。

4. **物流与交付管理**  
   - 要求供应商每日凌晨配送至酒店，确保早餐前到货。  
   - 约定冷链运输温控（如冷藏车温度2-8℃），并在收到货时抽检温度。  
   - 明确送货单需与实物完全一致，验收时按批次开箱检查。

5. **长期合作机制**  
   - 设立月度/季度质量评分制度（如分数低于90分需整改）。  
   - 定期安排供应商参与早餐菜单设计，提供季节性新品建议。  
   - 优先选择可提供A级超市或高端酒店合作案例的供应商。

**示例条款**（可要求供应商签署）：  
- “每日配送时间不晚于凌晨5:30，延迟30分钟以上扣除当日总价8%。”  
- “同一批次瑕疵水果占比超过5%时，整批拒收且供应商需在2小时内补货。”  

**额外提醒**：合作前实地考察供应商的果园或分装中心，关注仓储卫生、分拣流水线及紧急调度能力。
