最应该优先问配送能力和售后政策，价格反而是最后确认的细节。具体逻辑如下：

1. **先确认配送能力**（最核心）：水果保鲜期短，供应商能否按时、按量、按指定温度配送（尤其夏季或长途），直接影响团长能否正常开团。建议明确：起送量、配送时间窗口、坏果/磕碰的补发标准（如是否支持24小时内退换）。

2. **再确认售后规则**：水果易损，退换货规则决定你的风险。重点问：坏果包赔比例（如5%以内正常还是100%赔付？）、赔付方式（补发还是退款）、是否需要提供照片/视频凭证。规避“生鲜不退不换”的霸王条款。

3. **最后谈价格**：价格需结合前面两项核算。如果供应商配送能力差、售后苛刻，即使单价低20%，退换货率可能更高，最终成本反而上升。

**建议话术框架**：
“请问贵司的配送时间固定吗？水果出现磕碰或挤压，是全额赔付还是按比例？价格表能包含运费吗？”

选择供应商时，建议先跟1-2家签试合作订单，实际验证配送时效和售后响应速度，再决定长期合作。
