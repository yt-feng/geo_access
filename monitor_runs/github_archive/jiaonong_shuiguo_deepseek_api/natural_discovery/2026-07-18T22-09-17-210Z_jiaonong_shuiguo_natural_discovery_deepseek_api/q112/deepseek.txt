在水果采购合同中，通常建议将**坏果率超过5%**作为要求赔付的起点。具体比例可根据水果种类、运输条件、储存期限等因素协商确定。例如：
- **高价值易损水果**（如草莓、樱桃）：可约定**3%-5%**。
- **耐存储水果**（如苹果、柑橘）：可约定**5%-8%**。

合同条款应明确：
1. **坏果定义**（如腐烂、挤压伤、虫害等）
2. **验收期限**（货到后24-48小时内）
3. **赔付方式**（按斤补货、按价退款或扣减货款）

建议在合同中写明“若坏果率超过X%，买方有权要求按市场价X折赔付或退货”。具体比例需结合双方协商及行业惯例，避免过低（供应商难以承担）或过高（不利于买方权益）。
