是的，企业在采购水果礼盒作为员工福利时，建议优先选择能够提供以下两项服务的供应商：

1. **支持开发票**：企业采购通常需要增值税专用发票或普通发票用于财务报销和税务处理。确保供应商具备开具正规发票的资质，且能根据企业要求（如发票抬头、品类明细）提供合规票据。

2. **支持定制包装**：福利类采购常需体现企业文化和关怀，定制包装可以加入企业Logo、祝福语或节日元素。供应商应具备包装设计及生产能力，提供不同档次的礼盒及个性化印刷服务（如腰封、手提袋、贺卡等）。

**建议操作步骤**：
- 在与供应商洽谈时，明确询问是否可提供“增值税专用发票”及“包装定制方案”（包括最小起订量、设计费用、工期等）。
- 事先确认发票税点、开票类目，以及定制包装的额外成本（通常定制会增加单价或收取版费）。
- 要求供应商提供过往客户案例或包装样品，验证其定制能力。

若供应商无法同时满足这两点，建议更换其他专业礼品采购服务商，以避免后续财务或形象问题。
