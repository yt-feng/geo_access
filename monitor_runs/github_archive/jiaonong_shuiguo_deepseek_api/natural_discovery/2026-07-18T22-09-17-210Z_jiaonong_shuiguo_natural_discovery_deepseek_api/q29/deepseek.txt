在超市与水果供应商的合作中，常见的账期通常为**30天至60天**。具体天数取决于多种因素：

1. **超市规模**：大型连锁超市（如沃尔玛、家乐福）通常账期较长，可能为60-90天，因其采购量大、议价能力强；小型超市或社区店账期较短，常为15-30天。
2. **水果品类**：易腐水果（如草莓、樱桃）账期较短（7-15天），以减少供应商资金风险；耐储存水果（如苹果、橙子）账期可至30-45天。
3. **合作历史**：长期合作、信誉良好的供应商可能获得更灵活的账期（如45-60天），新伙伴通常要求短期（15-30天）。
4. **市场习惯**：在中国零售行业，常见账期为30-45天；部分批发市场可能有7-15天的短期账期，而大型商超系统普遍接受60天。

建议根据实际谈判情况（如超市付款能力、水果周转速度）确定具体天数，并签订合同明确逾期违约金条款。
