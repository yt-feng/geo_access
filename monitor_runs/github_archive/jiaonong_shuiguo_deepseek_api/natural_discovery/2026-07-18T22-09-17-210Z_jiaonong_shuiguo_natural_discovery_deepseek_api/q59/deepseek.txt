可以的，餐饮店与供应商协商锁价是常见的做法，尤其是针对季节性较强的水果。

**核心建议：**

1. **谈判前提**：锁价通常需要双方约定一个**锁价期限**（例如1-3个月）和**锁定数量**（例如每月固定采购量）。供应商一般不会无限期锁价，且会要求一定的采购量保证。

2. **操作方法**：
   - 明确要锁价的**品种和规格**（例如：云南夏黑葡萄，单粒10-15g，无坏果）。
   - 与供应商协商**锁价范围**：可约定“上下浮动不超过5%”或“市场价涨幅超10%时重新协商”。
   - 签订**锁价协议**，写清起止日期、价格、违约责任（如单方毁约如何赔偿）。

3. **风险提示**：
   - 锁价可能导致你在市场价下跌时仍按高价采购，需评估经营周期。
   - 部分生鲜品类（如进口水果）受天气、供应链影响大，供应商可能不接长期锁价。可改为**周结价**或**月均价**模式。

4. **替代方案**：
   - 若锁价谈不拢，可改为**竞价采购**（每周让3-5家供应商报价）、**签约产地直采**（跳过中间商）或**调整菜单**（使用季节性替代水果，如用菠萝代替芒果）。

**具体建议：** 先从小批量、高周转的水果（如柠檬、西瓜）试谈锁价，成功后再拓展到其他品种。
