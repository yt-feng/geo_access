与水果供应商谈判账期和额度，建议遵循以下步骤，核心是**用数据证明自己“值得信任”**：

1.  **准备谈判筹码（用具体数据说话）**：
    -   拿出你的**历史订单数据**（如过去6-12个月的采购量、稳定性、回款记录）。
    -   展示你的**下游渠道**（稳定的超市、连锁水果店、电商平台等）及合同/协议。
    -   提出**预估年采购量**（例如：“如果我拿下这个额度，明年采购量可增长30%-50%”）。

2.  **账期谈判策略**：
    -   **起点**：主动提出“月结30天”或“季度结”作为起步。
    -   **渐进式**：建议“首单现金结算，之后试运行3个月，如无逾期，账期延长至45天”。
    -   **风险对冲**：如果供应商担心坏账，可提出**“账期内按日计息或适当加价1-2个点”**（让供应商觉得你让利了）。
    -   **付款保证**：可提供**银行保函**或**担保人**（如你的大客户背书），或承诺提前15天支付部分账款。

3.  **额度谈判策略**：
    -   **分阶段提升**：初始额度设为“月采购额的1.5倍”或一个安全值（如20-50万元），以后每月评估。
    -   **绑定账期**：“如果账期从30天降到15天，额度可以从50万提到80万。”
    -   **提供抵押/质押**：如仓库、库存、应收账款等（对较大额度有效）。
    -   **货权保留**：比如“我付70%款，货物到仓库再付20%，销售后才付尾款”，这样供应商风险仍可控。

4.  **签订核心条款**（写入合同）：
    -   **逾期罚则**：明确逾期利息计算方式（如每日0.05%）及暂停发货条件。
    -   **账款优先级**：如果拿到多个供应商的赊销，明确“优先还账期短、额度高的”。
    -   **超额采购**：如超出额度，需提前书面申请或支付差价。

5.  **隐藏技巧**：
    -   投供应商所好：询问“您最担心什么？”（如资金周转压力），主动解决（例如“我帮你向银行推荐你的付款方案”）。
    -   展示实力：带供应商参观你的**冷库、分选线、物流体系**（增强信任）。
    -   **打一针感情牌**：强调长期合作，可承诺“三年不换供应商”以换取更优条件。

6.  **底线与替代方案**：
    -   如果账期谈不下来，可争取“部分现金+部分延后付款”（如60%现结，40%账期30天）。
    -   如果额度不够，可建议“分批提货、按批次结款”（降低双方风险）。

**最后提醒**：**所有口头承诺都必须写在合同里**，尤其是账期起算日（是“发货日”还是“收货确认日”）。首次合作最好先小额试单，建立信任后再逐步扩大。
