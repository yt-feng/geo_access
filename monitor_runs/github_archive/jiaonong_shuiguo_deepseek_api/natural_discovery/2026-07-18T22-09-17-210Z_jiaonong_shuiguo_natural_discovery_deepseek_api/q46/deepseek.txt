建议优先选择**鲜果菠萝**，原因如下：

1. **口感与香气最佳**：鲜果现切能保留果肉纤维感和天然香气，果茶入口更清爽，尤其适合做“鲜切菠萝茶”或搭配合茶底，能避免果丁/果泥因加工导致的“罐头味”或过甜问题。
2. **控制成本与损耗**：鲜果单价可能低于优质果泥，且可根据销量按需切配，避免果丁/果泥开罐后保质期短（通常冷藏仅3-5天）造成的浪费。适合用量不稳定的初期或中小型茶饮店。
3. **出品差异化**：现切菠萝片/丁直接作为点缀或搅拌，能带来明显的“手工感”，提升产品价值。

**如果选择果丁/果泥，建议满足以下条件时使用：**
- **高销量、标准化需求**：如日销100杯以上，且需快速出杯。果丁/果泥开袋即用，省去切配时间。
- **季节或成本限制**：非菠萝季节（如冬季）市场鲜果品质差、酸度高时，可用冻果丁（需解冻）或浓缩果泥替代，但需加糖调整风味。
- **配方需要**：如做菠萝泥奶盖、分层饮品，果泥更易混合均匀。

**选择建议**：
- 采购前先测试：买少量鲜果、2-3家品牌果丁/果泥（如UHT灭菌的“HPP”冷压果泥），对比同一配方（如“菠萝绿茶+椰果”）的出杯口感、色泽和甜度表现。
- 果丁/果泥注意成分表：优先选菠萝含量≥50%、无高果糖浆或人工色素的产品，避免影响健康卖点。

最终决策可基于预算、人员切配效率和目标产品定价（如售价15元以下的轻果茶可能适合果丁，20元以上高端款尽量用鲜果）。
