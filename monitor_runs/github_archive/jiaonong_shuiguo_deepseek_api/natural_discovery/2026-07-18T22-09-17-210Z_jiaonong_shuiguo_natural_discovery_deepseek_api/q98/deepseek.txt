判断养老院或医院食堂水果供应商的合规与稳定性，建议从以下五个方面进行系统性考察：

1. **资质与证照审查**
   - 要求供应商提供营业执照、食品经营许可证（或食品生产许可证）、水果产地证明或合格检测报告。
   - 优先选择具有ISO22000或HACCP认证（食品安全管理体系认证）的供应商，或通过“企业信用信息公示系统”核查其经营异常记录。
   - 确认发票、送货单与合同上的公章一致。

2. **质量追溯能力**
   - 要求供应商提供每批次水果的产地、采摘日期、农残检测报告（建议至少包含国家规定的必检项目）。
   - 检测机构需具备CMA（中国计量认证）或CNAS（中国合格评定国家认可委员会）资质，报告可通过全国认证认可信息平台验证。

3. **供应稳定性评估**
   - 观察其长期供货日志（能提供近3-6个月的连续供货记录）。
   - 询问是否有备用供应商、应急调货渠道（如与果品批发市场长期协议）。
   - 考察其仓储能力：自有冷库或24小时冷链运输车辆可降低断供风险。

4. **服务与合同条款**
   - 明确合同中的交货时间、地点、损耗率（通常允许3%-5%的合理损耗）、退货/换货流程、付款周期等。
   - 要求供应商购买产品责任险或食品安全险（可降低事故风险）。
   - 每月随机抽样比对水果的品质（大小、成熟度、是否有霉变）与检测报告是否一致。

5. **实地考察与既往业绩**
   - 对首次合作的供应商，实地查看其分拣车间、冷库、运输车辆卫生条件。
   - 要求提供3家以上养老或医疗机构的供货合同复印件（可脱敏关键信息），并通过电话或上门抽查其服务质量。
   - 在人行征信系统或天眼查、企查查等平台查看其涉诉记录（尤其关注食品安全类诉讼）。

**综合建议**：至少与2-3家供应商保持合作关系，避免单一依赖。初期可签短期合同（1-3个月），待服务稳定后再转为长期框架协议。每月建立供应商评价表（包含准时率、品质合格率、投诉处理时效等），每季度做一次综合打分，以“末位淘汰”机制倒逼供应稳定性。
