选择社区超市的水果供应商，建议重点关注以下几个条件：

1. **供应稳定性**：确保对方能每天按时配送，并保证不间断供应（特别是节假日或天气异常时）。
2. **水果品质与品种**：要求供应商提供新鲜、无损伤、符合成熟度的水果，品种最好多样化（如应季果品+常青款），且能根据社区需求灵活调整。
3. **价格与结算方式**：对比多家报价，确认是否包含配送费，价格是否随市场波动调价，结算周期（如日结、周结）是否合理。
4. **配送时效与范围**：明确每日最晚到货时间（建议早7-9点前），能否覆盖您的超市营业时间。了解配送覆盖区域，确保准时送达。
5. **售后与退换服务**：确认变质、破损水果的退换货规则（如到货后几小时内可退换），以及出现缺货时的补发方案。
6. **资质与证件**：核实供应商的营业执照、食品经营许可证、产地证明或检疫检验报告（尤其进口水果需相关报关单）。
7. **运输与存储条件**：了解供应商是否有冷链运输设备，能否保证水果在运输和存储中保持新鲜度（如温度控制、防挤压包装）。

**建议行动**：  
- 先选择3-5家本地或区域内有口碑的供应商（如批发市场、大型果园直供、网络生鲜平台等），要求试送3-5天，重点测评送货准时率、水果损耗率、售后处理速度。
- 签订合同时明确以上条款（每日最低订货量、罚款机制等），可避免后续纠纷。
