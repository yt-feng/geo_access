遇到水果到货不合格时，可通过以下步骤约定补货，尽量减少对营业的影响：

1. **立即检查与记录**：到货后第一时间开箱检查，对不合格水果拍照、录像，并记录数量、问题类型（如腐烂、未熟、破损等）。最好有第三方在场（如另一位店员或供应商送货员）共同确认。

2. **明确合同或原有约定**：查看采购合同中关于补货、退换货的条款。若无明确约定，尽快与供应商沟通，确认补货标准，例如：  
   - **时限**：要求供应商在X小时内（如4小时）补送合格水果，避免长时间缺货影响当日营业。  
   - **数量**：约定补货数量应为不合格数量的1.2倍（如部分水果在运输中进一步损耗，多备10%-20%作为缓冲）。  
   - **质量**：强调补货水果须达到签样标准或同批次中合格品的质量等级。

3. **分步到货预案**：如果当天客流高峰临近，可要求供应商先补急用品种（如畅销饮品主料），剩余品种次日补足。同时，与供应商商议**“借调”方案**：从邻近门店或其他供应商处临时调拨少量水果应急，后续补货抵扣。

4. **备选供应商清单**：提前准备1-2家备用供应商。一旦主供应商无法在约定时限内补货，立即启动备用供应商送货，同时按合同对主供应商追责（如扣款、违约金）。

5. **当日临时调整**：若补货时间确实赶不上营业高峰，可临时调整饮品单，下架依赖该水果的饮品，或推出替代饮品（如用库存充足的其他水果制作特调），并在线下、线上渠道（小程序、外卖平台）及时更新菜单，避免顾客点单后无法制作。

6. **书面确认与记账**：无论口头如何约定，务必通过微信、邮件或书面单据确认补货条款（包括补货时间、数量、质量标准、违约条款）。保留好原始不合格记录，作为未来协商或追责的依据。

**总结一句话**：发现不合格后马上拍照沟通，要求4小时内补足1.2倍合格水果，同时启动备用供应商，并临时调整菜单应急。
