在门店销售中，判断苹果和橙子的大小规格是否适合，主要依据以下标准化方法：

1. **规格对照**：根据行业标准，苹果和橙子通常按重量或直径分级（如：大号≥80mm/200g、中号70-80mm/150-200g、小号60-70mm/100-150g）。采购前确认门店目标客群需求（如家庭客户偏好中号，高端客户偏好大号）。

2. **肉眼筛选**：同一批次内，果实摆放时直径差异应控制在5mm以内，避免大小混乱。可用标准环形尺（卡尺）快速抽检10%果实。

3. **触觉校验**：苹果需手感硬实，橙子按压有微微弹性（避免过软或过硬）。大小不宜超过成人拳头2/3，避免单果过重在称重时影响毛利。

4. **视觉统一**：优先选择外形圆整、无畸形果实，过大可能内质疏松，过小可能酸涩。建议采购单果在120-200g的橙子、150-250g的苹果作为基础。

5. **实验方法**：每日开箱时随机抽取10个果实，使用厨房秤称重，若超过40%的果实偏离目标规格范围（如目标中号150g，实际偏差≥30g），则退回该批次。

最终判断需结合门店历史销售数据（如3个月的各规格销量占比），优先采购月销量占比超过60%的规格区间。同时关注季节性：夏季应选稍小规格（避免运输损耗），冬季可适当选取大果。
