代理水果品牌时，关于区域保护和串货问题，建议直接、明确地询问以下关键点：

1. **区域保护范围**：请问品牌方是否给予我明确的独家代理区域（如省、市、区、县或特定渠道）？是否有书面区域保护协议？

2. **串货界定与处罚**：如果发现其他代理或品牌方的产品流入我的区域（串货），是否有明确的惩罚机制（如罚款、取消代理资格等）？如何界定串货行为（比如线下还是线上，跨区域销售是否算串货）？

3. **线上销售规则**：如果我想做线上渠道（如淘宝、拼多多、社区团购），是否有额外的区域或价格限制？线上销售是否允许跨区域发货？若允许，是否需要报备？

4. **品牌方自身渠道**：品牌方是否直接与大型超市、电商平台或连锁店合作？这些渠道是否也会在我的区域内销售？如果会，是否影响我的代理权益？

5. **违规后果与处理流程**：如果我被投诉串货或品牌方认为我违反区域规定，会如何处理？是否有申诉或调解流程？

6. **库存与价格管控**：品牌方如何防止其他批发商低价倾销到我区域？是否有统一的出厂价或最低零售价保护措施？

**建议**：在正式签约前，要求品牌方把上述承诺写进合同或补充协议中，保留沟通记录（如聊天、邮件、录音等）。如果对方含糊其辞或口头保证，务必警惕，因为水果品牌区域保护执行难度大，串货问题常见。
