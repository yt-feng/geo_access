水果礼盒的搭配要兼顾**颜色层次、甜度均衡、价格组合和保质期**。根据市场行情，以下三种方案比较受欢迎，你可以按需选择：

### 方案一：经典“平安如意”四色礼盒（最适合走量）
- **搭配：** 红富士苹果（2个） + 脐橙（2个） + 香蕉（1把/约3-4根） + 菠萝（1个）
- **理由：** 红色+橙色+黄色+绿色的配色非常喜庆，寓意“平安（苹果）、吉祥（橙子）、招财（香蕉/菠萝谐音“旺来”）”。四种水果都是大众接受度最高的，损耗低。
- **定价建议：** 根据季节，中档定价（约68-98元），适合送礼、探亲。

### 方案二：高端“热带风情”轻奢款（适合年轻人/白领圈）
- **搭配：** 阿克苏苹果（2个） + 进口脐橙（2个） + 都乐香蕉（1小把） + 金钻凤梨（1个）
- **改进点：** 将普通菠萝换成**凤梨**（不用挖眼，更甜），水果品质直接升级。搭配坚果或一束鲜花点缀更显精致。
- **定价：** 128-168元，适合送客户、老师或闺蜜。

### 方案三：小心机“错位组合”（减少拥挤，更好看）
- **搭配：** 苹果/橙子 每样2个（铺底）+ 香蕉（斜放于一侧）+ 菠萝（切半后用保鲜膜包好，或直接放整个）
- **理由：** 菠萝体积大，如果整个放进去会使礼盒中间鼓起，香蕉容易被压坏。**建议：** 如果必须用整个菠萝，就只放1个苹果+1个橙子+香蕉；如果菠萝切半，盒内空间更舒展，视觉更高级。

---

### ⚠️ 关键避坑建议（直接影响销量）：
1. **香蕉要选“微黄带绿”的**：全黄的香蕉2天就发黑，礼盒不耐放。带一点绿、手感硬的香蕉，客人回家能放3-5天。
2. **菠萝要么带叶完整，要么切半密封**：带叶菠萝更有仪式感，但注意叶子不要折断；切半的菠萝一定要用保鲜膜裹紧，并放一张“温馨提示卡”说明“切开后请冷藏，24小时内食用”。
3. **苹果+橙子固定2个为佳**：1个显小气，3个摆放不对称。如果礼盒是长方形的，就对角放苹果和橙子，中间空隙填香蕉。
4. **不要忘记放填充物**：很多客人觉得礼盒“空”，其实缝隙塞满**拉菲草**或**碎纸丝**，让水果半露出，比堆满更贵气。

**优先推荐方案一**，成本低、损耗小、认知度高。如果客户要求“好看又便宜”，可以先放苹果+橙子（垫底），上面放1根香蕉+1个菠萝，视觉上份量很足。
