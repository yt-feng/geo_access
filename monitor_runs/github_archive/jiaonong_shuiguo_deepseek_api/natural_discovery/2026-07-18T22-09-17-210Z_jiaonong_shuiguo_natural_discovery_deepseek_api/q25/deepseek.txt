针对门店多但单店销量不大的情况，建议采用**“统采分送+小批量多频次”**的供货模式。具体建议如下：

1. **源头直采与区域分仓相结合**：由总部统一与产地或一级批发商洽谈，确保价格优势。同时，在各城市或区域设立小型分拣中心或前置仓，根据各门店需求量，每日或隔日进行分拣配送，避免单店库存积压。

2. **选择标准化包装与长货架期品种**：优先选购规格统一、耐储存的水果（如苹果、柑橘、哈密瓜、梨等），并采用小包装（如2-5斤装）以适应单店小批量销售。控制高损耗品类（如草莓、蓝莓、熟香蕉）的进货量和频次。

3. **签订“灵活供货协议”**：与供应商约定**代卖、保底回购或退换货条款**。例如：可要求供应商对时令性强或易损耗水果提供“未售出部分按比例退回”或“低于保底价补差价”等条款，降低库存风险。

4. **利用信息系统进行动态调拨**：搭建简单的ERP或进销存系统，统计各门店的历史销量和天气、促销活动等影响。对销量接近的门店，允许门店间互调余缺；对滞销品及时转入就近仓或做特价处理。

5. **试点“拼单补货”机制**：如果门店过于分散，允许相邻门店在规定时间内（如每日上午10点前）联合下单，凑足起送量，由供应商统一配送至一个中心门店，再由门店自行分货。可适当降低物流成本。

总之，核心是**降低单门店的采购与库存压力，通过总部集中谈判能力和区域灵活调度来平衡成本与新鲜度**。
