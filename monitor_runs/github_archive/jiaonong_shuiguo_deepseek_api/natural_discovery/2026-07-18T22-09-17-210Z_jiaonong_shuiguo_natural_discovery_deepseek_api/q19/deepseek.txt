如果要在超市卖菠萝，建议按以下顺序关注：

1. **甜度**（最关键）  
   消费者核心诉求是“好吃”，甜度高直接决定复购率。优先选择糖度在14%以上的品种（如金菠萝），并标注“树上熟”或“自然成熟”作为卖点。

2. **切果损耗**（影响利润）  
   菠萝去皮后损耗约30%-50%。可提供“现场切果”服务，将损耗部分转化为加工果切商品（如菠萝杯、混合果盒），同时用“免费削皮”吸引客流。

3. **外观**（第一印象）  
   避免有明显碰伤、褐斑或腐烂的果子，否则会被直接淘汰。但不必过度追求“完美外形”，轻微瑕疵可通过低价促销处理。

4. **大小**（次要因素）  
   在甜度与外观达标的前提下，优先选择**中等规格**（1.2-1.8kg/个），大小易拆分、适合一餐消耗。过大的菠萝（>2kg）可切开销售或定位为“家庭装”。

**执行建议**：  
- 首次进货可小批量试销（如20箱），对比不同产地、品种的甜度与损耗数据。  
- 在摊位设置“试吃区”，让顾客通过甜度直接决策，降低外观挑剔度。  
- 每日晚间对剩余菠萝进行切果加工，控制次日损耗成本。
