在合同中明确坏果责任，建议重点写清以下条款：

1. **验收期限**：写明“收货方应在货到后【X】小时内（建议12-24小时）完成开箱验收，超时未提出书面异议视为货物合格。”
2. **分级责任**：约定“自然损耗率”范围（如水果类通常约定3-5%以内由买方承担），超出部分由卖方负责。
3. **举证方式**：要求“收货方需提供清晰照片/视频证明坏果比例及数量，作为索赔依据。”
4. **赔偿标准**：具体写明“坏果按单价【X元/斤】从总货款中扣除，或直接退还坏果对应货款。”
5. **争议处理**：约定“如双方对损耗率有争议，可委托【第三方检测机构名称】鉴定，费用由责任方承担。”

建议在合同正文或附件中保留实物样品照片/录像，并明确运输保险是否覆盖此类风险。
