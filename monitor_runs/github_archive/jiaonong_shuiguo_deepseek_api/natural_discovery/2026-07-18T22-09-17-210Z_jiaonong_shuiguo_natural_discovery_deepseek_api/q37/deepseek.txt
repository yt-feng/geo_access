供应商配合商超会员日水果活动，需要从**货源、物流、陈列、售后、数据**五个维度提前准备，具体如下：

1. **货源保障**：提前2-3天与商超确认活动单品、预估销量（参考同期数据），备足安全库存（建议1.5倍以上）。优先对接产地直发或囤货，避免断货。对易损耗水果（如草莓、蓝莓）要控制到货批次，分时段补货。

2. **包装与规格优化**：根据活动定价调整包装形式（如小份装、家庭装、混合礼盒），明确标出“会员价”或“特惠”标签。包装需结实透气，避免运输挤压。准备少量试吃装（密封独立包装）。

3. **物流与保鲜**：安排冷链运输（温度：热带水果12-15℃，温带水果0-4℃），提前规划到货时间（建议活动前1天到店，避开高峰期）。每批次附加“到货时间-数量-批次”单据，配合商超验收抽检（糖度、外观、重量）。

4. **陈列与物料支持**：提供定制化陈列道具（如水果托盘、分层展架）、价格牌、爆炸贴、试用装。准备促销员手举牌、试吃车、现场切配工具。培训促销员掌握会员日活动话术（如“会员专享价”“满减叠加”）。

5. **数据与应急预案**：活动前与商超共享历史会员消费数据（如偏好芒果、葡萄的客群），用于定向推送。建立24小时应急响应机制：配备2名以上售后对接人，针对**水果品质客诉**（霉变、过熟）承诺“1小时内换新或退差”。提前备好替换用存货（至少10%比例备用货）。

6. **定价与促销规则**：确认会员价与非会员价差额（建议10%-20%），核对是否叠加会员积分、满减券。禁止出现“活动价高于平日日常促销价”的情况，需保留调价记录备查。

7. **品牌与宣传配合**：提供品牌授权书、质检报告（重点农残检测）供商超合规审查。设计线上推广素材（海报、短视频、社群文案），提前3天发至商超运营方。鼓励供应商在自有直播间、公众号同步导流。

**关键检查清单**：  
- [ ] 活动前3天：签订保量协议（注明赔付条款）  
- [ ] 活动前1天：到货100%质检（目测+糖度仪+称重）  
- [ ] 活动当天早7点：完成陈列与试吃车就位  
- [ ] 活动期间：每小时巡查一次水果新鲜度，挑拣瑕疵品。

以上准备需与商超采购、门店店长、活动策划三方最终确认时间节点。若有特殊要求（如产地现切现卖），需额外提供操作台、保鲜膜、一次性手套等物资。
