为了更划算地选择苹果、橙子和菠萝做水果拼盘，建议从**单价、出肉率、损耗**和**应季性**几个角度综合考虑。以下是具体建议：

1. **苹果**：选择**中等大小（约180-250克/个）**、果形端正的。  
   - **原因**：小苹果去皮去核后损耗大，大苹果容易出现空心或口感不均。中等个头出肉率高（约70-75%），且单位价格通常适中。  
   - **购买技巧**：若批发，选按箱卖的，计算每公斤单价；日常可买散装，优先选本地应季品种（如红富士、嘎啦）价格更低。

2. **橙子**：推荐**中大型果（直径约70-80毫米，单果重250-350克）**。  
   - **原因**：小橙子皮厚肉少，极小型（如脐橙小果）出肉率可能低于50%；大型果（如夏橙、血橙）果肉饱满，出肉率可达65-70%。  
   - **省钱技巧**：购买时按压果皮，选软硬适中、沉甸甸的，这种水分足、果肉比例高。避免过熟或干瘪的。

3. **菠萝**：选择**中等重量（1.5-2公斤/个）**、底部有清香且叶片新鲜的。  
   - **原因**：太大可能纤维粗、中心硬，太小出肉率低（削皮去芯后约40-50%）。中等菠萝果肉比例高、甜度均衡。  
   - **购买技巧**：按个卖时，用手掂量，同等大小下更重的水分多；按斤卖时，计算每斤可食用部分的成本（比如削皮后剩余1公斤，实际成本增加）。

**综合划算策略**：  
- **比价**：计算每种水果的**可食用部分单价**：  
  可食用部分净重 = 总重 × 出肉率（苹果70%、橙子65%、菠萝50%为例）。  
  用该净重除以总价，比较真实成本。  
- **应季采购**：本地当季水果通常最便宜（如秋季苹果、冬季橙子、春季菠萝），反季的采购价可能高30-50%。  
- **拼盘搭配**：三种水果按成本比例分配，如果某一类特别贵，可减少用量，用价格低的（如时令苹果）填充总量。

**注意事项**：  
- 不要只按个头大小选，要计算实际出肉率（例如：小橙子每斤可剥出果肉可能比大橙子少30%）。  
- 选购时注意检查有无霉斑、虫眼，避免损耗。

最终判断标准：**在保证新鲜度的前提下，选单价最低、出肉率最高、损耗最小的规格**。
