混合采购（即同时采购香蕉、苹果、橙子）对水果批发商通常有以下优势：

1. **降低物流成本**：同一产地或供应商可能同时供应多种水果，一次性采购可减少运输次数、降低单位运输成本。
2. **提高议价能力**：采购量更大，与供应商谈判时有更高话语权，可能获得更优惠的价格、账期或服务条件。
3. **丰富产品组合**：满足零售商或客户对多种水果的需求，提供一站式采购体验，增加客户粘性，减少客户流失。
4. **分散风险**：单一水果价格受季节、产量、天气等因素波动影响大，混合采购可平衡不同水果的盈亏，稳定整体利润。
5. **库存与销售协同**：根据市场需求灵活调整各品类比例，例如夏季橙子需求减少时可增加香蕉、苹果采购，优化库存周转。

**注意事项**：  
- 需考虑不同水果的存储条件（如香蕉需催熟，苹果耐存储）和损耗率差异，合理规划库存管理。  
- 应确保供应链稳定，避免同时依赖单一供应商导致风险集中。  

因此，在合理规划的前提下，混合采购通常具有显著优势。
