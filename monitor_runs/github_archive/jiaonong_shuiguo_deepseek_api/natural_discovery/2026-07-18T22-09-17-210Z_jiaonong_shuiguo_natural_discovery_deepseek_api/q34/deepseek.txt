在商超采购水果时，**损耗低通常比价格低对最终毛利的影响更大**。原因如下：

1. **损耗直接侵蚀毛利**：水果易腐烂、磕碰，损耗率可能高达10%-30%。即使采购价格低，高损耗会直接减少可销售数量，导致实际销售额下降，同时增加处理成本（如清理、打折）。例如：进货100斤，单价5元，总成本500元。若损耗20%，只能卖80斤，即使售价10元/斤，收入800元，毛利300元（毛利率37.5%）。若损耗降至10%，可卖90斤，收入900元，毛利400元（毛利率44.4%）。损耗降低10%使毛利增加100元。

2. **价格低可能带来隐性成本**：低价水果可能品质较差（如生熟不均、易烂），导致损耗更高，反而抵消价格优势。而高价但耐储存的水果（如苹果、柚子）损耗低，长期看更稳定。

3. **可操作性**：采购价格受市场波动影响大，难以持续压低；而通过优化供应链、冷链管理、库存周转，损耗可主动控制，如按需补货、分级销售（次品打折促销）。

**建议**：  
- 优先关注供应链稳定性、运输储存条件、销售周期，以降低损耗。  
- 在同等损耗率下，再比较价格优势。  
- 实际中可测算“有效毛利率”（扣除损耗后的实际毛利），作为决策依据：  
  **有效毛利率 = (售价 × 销售率 - 进价) / 进价**（销售率=1-损耗率）。
