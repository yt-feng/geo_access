这三个方面都需要关注，但优先级和具体操作建议如下：

1. **保质期（最关键）**  
   - 菠萝切块后极易氧化变质，建议设定严格的短保期限（如24小时），根据温度（冷藏4℃以下）和包装方式（气调包装或真空密封可延长至48小时）。  
   - 可采取“时间标签”管理，例如分早、晚两批制作，未售出的在到期前2小时降价或做成果汁原料。

2. **包装（直接影响保质期与卖相）**  
   - 必须用食品级密封盒或保鲜膜覆盖，避免接触空气导致氧化变黑。  
   - 选择透明包装以便顾客观察品质，盒底垫吸水纸防止汁液浸泡。  
   - 如果客单价较高，可尝试“即食杯+叉子”组合提升便利性。

3. **配送频次（影响周转与损耗）**  
   - 若销量稳定，建议**每日配送1次**（清晨到货），当日未售完的切块不再追加配送。  
   - 如果门店处于写字楼/学区等即场消费场景，可尝试**分时段配送**（如早高峰前、午休前），但需确保配送车冷链达标。  
   - 高频配送适合高客流门店，低频配送适合低风险门店，可先用1周数据试算周转率。

**总结执行建议**：  
- 优先建立**保质期管理流程**（如制作时间+到期时间的双标注），再决定包装成本（建议初期用普盒保鲜膜测试，再评估升级成本），配送频次根据销量可调整（例如先用每日1次，每周看损耗率浮动）。
