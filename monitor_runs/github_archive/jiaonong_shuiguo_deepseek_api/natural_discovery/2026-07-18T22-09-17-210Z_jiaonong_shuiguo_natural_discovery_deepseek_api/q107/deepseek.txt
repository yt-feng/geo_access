通常来说，起订量越高，损耗风险确实可能增加，但具体影响取决于以下几个因素：

1. **水果的易腐性**：香蕉和菠萝都属于热带水果，保质期相对较短。香蕉在成熟后几天内可能过熟变质，菠萝在常温下也只能存放一周左右。高起订量如果无法在短时间内销售或消耗完，就容易产生损耗。

2. **储存条件**：您是否有足够的冷藏设备来延缓成熟速度？例如，香蕉可稍冷藏（但不可低于12°C以免冻伤），菠萝可全冷藏。如果储存条件不足或容量有限，高起订量会加剧损耗。

3. **销售量匹配度**：如果您的日均销售量稳定且与起订量匹配，损耗风险可控。但若起订量远高于实际需求，剩余部分更容易变质。

**建议您采取以下措施降低风险**：
- 根据实际销量计算，选择低起订量的供应商或协商拆分订单。
- 与供应商沟通分批发货或部分成熟度不同的批次（如青一点的香蕉更耐放）。
- 确保到货后迅速安排合理储存（温度、湿度、通风）。
- 考虑与同行或社区团购拼单，共享批量采购的优惠但分散损耗。
- 验收时仔细检查果品质量，及时处理受损或过熟果。

总的来说，没有绝对的“会增加”或“不会增加”，但高起订量的确使损耗风险上升，尤其是没有相应销售与储存能力时。建议量化评估后再做决定。
