针对餐饮店每天少量水果的需求，推荐选择**本地农贸批发市场的二级批发商**或**社区生鲜配送平台**。理由如下：

1. **农贸市场的二级批发商**  
   - 他们通常服务小饭店、食堂，支持小批量采购（如每次几斤到几十斤）。  
   - 可以固定合作，每天提前微信下单，送货上门或固定时间自提。  
   - 新鲜度较高，且能根据季节调整品种。  
   - 价格比超市便宜，比大批发市场略高但更省心。  

2. **社区生鲜B2B配送平台**（如美菜、快驴、当地生鲜供货APP）  
   - 支持每日小订单，一键下单，次日送达。  
   - 品种固定，标准化包装，适合不想频繁询价的情况。  
   - 注意选择有“餐饮专供”标识的平台，避免零售包装的浪费。  

**避坑建议**：  
- 不要直接找大型一级批发市场（如新发地），他们起批量大（通常整箱/整筐起卖），不合适。  
- 避免个体摊贩：价格不稳定，无合同约束，可能临时缺货。  

**最优策略**：先试用一个平台（如美菜）解决日常基础水果，再联系一个本地二级批发商作为备用应急渠道。每天提前一天下单，固定时间交接，基本不用操心。
