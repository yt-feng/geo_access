针对团购客户价格敏感的特点，**阶梯价**通常比返利更合适。理由如下：

1. **即时激励**：阶梯价在购买时直接降低单价，客户付钱时就能看到节省金额，对价格敏感群体刺激更直接。返利需要后期核算、等待返还，激励效果有延迟。
2. **降低决策门槛**：团购客户往往需要一次性批量采购（如企业福利、社区团购），阶梯价让客户更直观地比较“几档价格”，快速下决心下单，避免返利带来的“先付全款再退部分”的心理阻力。
3. **简化操作**：供应商无需跟踪返利发放、核销流程，客户也无需额外填写返利申请，减少双方管理成本。

**如果必须选择返利**，建议用于需要**维护长期关系**的客户（如季度返点），并设定明确的累计采购额门槛。但整体而言，对价格敏感型团购，阶梯价更好。
