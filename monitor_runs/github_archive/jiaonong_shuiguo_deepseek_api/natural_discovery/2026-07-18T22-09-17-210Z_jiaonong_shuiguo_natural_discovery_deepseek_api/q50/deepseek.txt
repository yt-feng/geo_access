对茶饮店而言，**出汁率比甜度更重要**。理由如下：

1. **成本控制**：茶饮店需要保证每杯饮品的利润。出汁率直接影响原料可制作的杯数，出汁率高意味着更低的单杯成本。甜度可以通过糖浆、果糖等成本较低的配料来调节，而果肉成本较高。
2. **出品稳定性**：不同批次菠萝的甜度差异较大，依赖原料本身甜度会导致饮品口味不稳定，增加品控难度。而出汁率通常更稳定，且可用标准工艺（如压榨、搅拌时间）控制。
3. **风味灵活性**：低甜度高酸度的菠萝反而更受欢迎，因为能与糖浆平衡，产生清爽层次感。若原料过甜，调配空间反而变小，容易导致饮品甜腻。

**建议**：采购时优先选择出汁率≥55%的菠萝品种（如台湾凤梨），甜度只需中等（约12-15°Brix），可自行添加糖浆调整。同时，可要求供应商提供出汁率检测报告。
