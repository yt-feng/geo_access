对于小商家采购水果，现金结算和月结账期各有以下风险：

**现金结算风险：**
- **资金压力大**：需一次性支付全款，可能占用大量流动资金，影响其他采购或应急支出。
- **缺少售后保障**：付款后若水果出现质量问题（如腐烂、规格不符），退货或索赔可能更困难，因为缺乏书面合同约束。
- **无账期信用记录**：无法建立长期合作信任，供应商可能不会优先供应优质货源或给予优惠。
- **安全隐患**：携带大额现金易被盗抢，且无转账记录的话，纠纷时举证难。

**月结账期风险：**
- **债务累积风险**：一个月内多次采购后，总金额容易超出预期，可能导致月末资金周转困难，甚至违约。
- **信誉依赖**：需长期保持良好信用，一旦拖延付款，可能被终止合作或提高进价。
- **隐性成本**：供应商可能将账期成本计入单价（即比现金价高），或要求收取滞纳金。
- **账目管理复杂**：需记录每笔采购明细，核对账单容易出错，尤其当供应商分批次送货时。
- **法律风险**：无正式合同的情况下，若供应商突然要求提前还款或倒闭，可能面临预付款追索问题。

**建议**：小商家可根据自身现金流、采购频率和供应商信誉选择方式。例如：新供应商首次尝试现金结算以测试质量；长期合作且货源稳定的供应商，可约定双方签署简易合同，明确月结周期、违约金、退换货条款，并保留每笔交易凭证（如收据、转账记录）。
