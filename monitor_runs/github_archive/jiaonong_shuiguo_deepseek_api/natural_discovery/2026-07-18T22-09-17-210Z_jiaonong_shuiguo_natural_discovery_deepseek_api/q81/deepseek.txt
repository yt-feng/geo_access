对于即时零售平台卖水果，**供应商支持小批量高频补货是非常必要的**。主要理由如下：

1. **匹配平台履约模式**：即时零售的核心是“30分钟-1小时送达”，订单量分散且波动大。小批量高频补货能快速响应订单，避免缺货导致的订单取消或差评。

2. **减少损耗**：水果保质期短、易损耗。大批量囤货容易造成库存积压和腐烂，小批量补货能保持新鲜周转，降低售后成本。

3. **提升现金流效率**：高频补货意味着供应商按实际销售回补库存，资金压在大仓库的金额少，资金周转更快。尤其对中小供应商来说，压力更小。

4. **适应平台算法**：即时零售平台（如美团闪购、饿了么）的搜索排名、流量分配会参考“在售SKU数”和“履约率”。小批量补货能保证关键SKU不断货，稳定曝光。

**不过，这需要供应商具备：**
- 分拆打包能力（如按5斤/10斤小份包装）
- 相对稳定的短途冷链配送
- 与平台数据系统对接，降低补货决策成本

**建议**：即使无法做到“次日达”级的高频，也可以采用**1天1补**或**按门店销量分时段补货**的折中方案。初期可以选2-3个热销品试点，再逐步推广。
