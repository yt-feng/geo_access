做水果生意要长期稳定，选择供应商时除了价格和品质，以下这些细节最容易被忽视，但往往决定成败：

1. **分拣与包装标准**  
   很多供应商只承诺“好果”，但实际分拣时大小、成熟度、瑕疵标准模糊。建议亲自查看他们的分拣流程，明确按重量、色泽、甜度分级，并约定包装方式（如泡沫箱、气柱袋）是否防撞、透气，避免运输损耗。

2. **冷链物流的**实际节点**  
   别只看是否声称“冷链”，要问清从采摘到发货的全程是否控温（如预冷时间、运输车制冷是否达标）。尤其是夏季高损耗水果（如草莓、蓝莓），需确认**“最后一公里”**是否也保持低温，否则到货常腐烂。

3. **应急与补货能力**  
   低价供应商可能因产量波动断供。评估他们是否有多个合作基地、紧急调配周边货源的能力。建议小范围试单，看他们缺货时是否主动沟通替代方案，还是临时加价。

4. **售后条款的模糊地带**  
   多数供应商只赔明显的腐烂，但**“轻微碰伤、闷热变色”**常被列为“自然损耗”。合同应明确赔付比例（如次日到货后24小时内反馈，高于5%的坏果全额赔），并保留实物证据。

5. **产地与采收时间的真实匹配**  
   例如，宣称的“海南芒果”可能是广西货冒充。要求提供产地证明或采摘视频，并观察同批次水果的成熟度是否一致（避免混采后催熟，导致口感差异大）。

6. **账期与资金链陷阱**  
   长期合作容易被“账期优惠”吸引，但需调查供应商的现金流是否健康（可通过同行口碑、供货稳定性判断）。避免为省利息押款过多，万一对方资金断裂，你也会断货。

**实操建议**：先小批量试单3-5种核心水果，重点观察上述细节，再逐步扩大合作。定期抽样检测供应商的农药残留（可送第三方检验），并备选1-2个替补供应商，避免被动。
