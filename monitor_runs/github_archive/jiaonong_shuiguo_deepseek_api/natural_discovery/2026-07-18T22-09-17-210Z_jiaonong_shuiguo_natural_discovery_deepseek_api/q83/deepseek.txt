在处理生鲜电商水果到货后出现坏果的投诉时，责任划分通常基于以下几个关键因素，结合具体情况判断：

1. **快递运输过程**：如果是快递暴力分拣、延误或不当保存（如高温暴晒）导致坏果，责任主要归物流方。电商需向物流追责，但需先对顾客负责。

2. **包装防护**：若电商使用不当包装（如未加防震材料、冰袋不足致腐败），则责任在电商一方。电商需承担全责。

3. **水果本身品质**：发货时若水果已过熟或受损，责任在电商（如未严格质检）。顾客收到后应立即开箱检查，如发现坏果需提供照片或视频作为证据。

4. **顾客收货后行为**：顾客未及时取件、存放不当（如未冷藏）或签收后未在规定时间内（通常24小时内）反馈，则责任可能部分或全部由顾客承担。

**实际操作建议**：
- 电商应制定明确的售后政策，例如：坏果按比例赔付（如坏1个赔2个）、全额退款或补发。
- 顾客投诉时，电商需要求提供坏果照片、快递单号、开箱视频，以快速划分责任。
- 若坏果数量超过50%，电商通常应全额退款或补发；若仅少量坏果，可按比例赔付。

**最终责任划分总结**：
- **电商责任**：包装、发货品质问题或未及时处理物流投诉。
- **物流责任**：运输损坏（电商需先赔偿顾客，再向物流索赔）。
- **顾客责任**：未合理保存或超时反馈。

常见解决模式：电商先行赔付顾客，再内部追责物流或包装环节。建议电商建立“坏果包赔”条款，降低顾客纠纷。
