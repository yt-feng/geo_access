判断香蕉熟度和坏果风险，可以按以下步骤操作：

1. **观察果皮颜色**：青绿色表示未熟，黄绿色表示半熟，黄色均匀表示可立即销售；出现黑斑或大面积发黑说明过熟，需检查果肉是否腐烂。

2. **触摸硬度和弹性**：轻轻捏香蕉中部及两端。手感硬实、有韧性的较新鲜；局部发软、按压凹陷处可能已软烂；过软则加速腐烂风险。

3. **检查果柄（蕉柄）**：果柄呈绿色、紧实为新鲜；发黑、干枯或松动说明存放时间久，坏果率升高。

4. **闻气味**：有甜香、青草味正常；若有酸味、酒味或霉味，内部已开始腐坏，应整批拒收或折价处理。

5. **抽查果肉**：随机掰开一两个香蕉（尤其两端和中间位置），果肉洁白、有轻微淀粉味为正常；发褐、发黑或有汁液渗出表示内部已坏。

6. **看存放环境**：如果货箱密封、通风差、温度高（超过30℃），或袋内有大量凝结水珠，香蕉会快速成熟并腐坏。建议打开包装让空气流通后再评估。

7. **打听来源与时间**：向供货商问明采摘后放了几天、是否经过冷库贮藏。青蕉一般需要3-7天自然成熟，若商家说“已熟”但实际全青，则可能使用了催熟剂但未到位，后续转熟不均衡，坏果风险高。

**紧急处理原则**：  
- 如果整批中超过10%有明显黑斑或软烂点，建议拒收或要求降价，并优先销售品质好的部分。  
- 当天用不掉的熟香蕉（黄色均匀且无黑斑）可冷藏（低于13℃会皮变黑但果肉保持）或通风阴凉处存放，但冷藏过的香蕉应尽快在1-2天内卖出。  
- 对已出现少量黑点或软点但未腐烂的，可以切掉坏果并单独包装降价卖，或做特价组合，避免混入好货影响整体品质。
