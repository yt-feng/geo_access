采购菠萝时，判断熟度主要靠“看、闻、按”三个步骤，可减少损耗：

1. **看外观和颜色**  
   - 选择表皮**金黄均匀**、不带大面积青绿的菠萝。青绿色未熟，口感酸涩；全黄可能过熟。  
   - 避免果眼（小黑点）处有**发黑、软烂或渗水**的菠萝，这通常是内部腐烂的前兆。  
   - 顶部叶子应**翠绿、挺拔**，若叶子枯黄或易折断，表明存放时间较长。

2. **闻气味**  
   - 靠近底部（果蒂处）能闻到**浓郁甜香**的菠萝最好吃；若气味清淡，可能未熟。  
   - 闻到**酒味或酸败味**，说明内部已发酵或变质，不应采购。

3. **按硬度**  
   - **轻按果皮**，感觉**微微有弹性、不硬不软**最合适。按下去非常硬则未熟；按着发软或凹陷则过熟。  
   - 避免采购**头部或尾部明显软塌**的菠萝，此处易先腐烂。

**采购建议**：  
- 若需短时间销售，选七八成熟（微黄带一点青、按着稍硬）的菠萝，在店内自然催熟。  
- 收货后当天或次日卖不完，可分批成熟供应：少量金黄色当天卖，稍青的放一两天再卖，减少一次性全熟导致的损耗。
