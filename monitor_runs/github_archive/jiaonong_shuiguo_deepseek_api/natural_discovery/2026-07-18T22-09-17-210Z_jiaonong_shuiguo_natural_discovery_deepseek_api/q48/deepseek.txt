要判断茶饮连锁的菠萝供应商能否稳定按规格送货，可以采取以下具体步骤和评估方法：

1. **明确规格要求**：在合作前，清晰列出菠萝的规格标准（如单果重量、成熟度、品种、直径、糖度、无病虫害、外观完整等），并形成书面文件。例如：“单果重量1.2-1.8kg，表皮无黑斑、无腐烂，糖度不低于14°Bx”。

2. **实地考察供应商**：优先选择有种植基地或长期稳定合作果园的供应商。考察其生产规模、采收流程、分级筛选设备（如自动称重、分拣机）、冷链运输能力。观察其库存管理是否有序，仓库温湿度控制是否达标。

3. **要求提供样品并测试**：要求供应商连续提供3-5批次样品（间隔1-2周），自己或委托第三方检测重量、成熟度、糖度、外观等，检查批次间一致性。抽查合格率需≥95%以上方可考虑合作。

4. **审查供应商历史记录**：要求提供过去6-12个月的供货记录（包括退货率、逾期率、规格达标率），或向其他客户（如商超、餐饮连锁）侧面打听其信誉。若无法提供，可要求其开具银行或行业协会的信用证明。

5. **签订合同并设置约束条款**：合同中明确：
   - 每次供货的规格标准及验收方法（如到货后24小时内验收，不合格品退货或折价）。
   - 首次供应批次合格率≥95%，否则供应商需承担换货、补货或赔偿损失。
   - 连续两次批次合格率低于95%，甲方有权单方面终止合同并索赔。

6. **建立渐进式合作机制**：先小批量试单（如每周100-500公斤），观察其1-2个月的送货准时率、规格达标率。达标后逐步增加订单量，并记录每次交货数据，形成台账。

7. **考察应急能力**：询问供应商在菠萝采收旺季、雨季或极端天气时的应对措施（如是否有备用果园、冷链车数量、应急预案）。可模拟一次临时追加订单，观察其响应速度。

8. **利用第三方验证**：委托第三方检测机构（如SGS、中国检科院）对供应商的种植基地、分拣中心、物流环节进行突击检查，或将供应商纳入行业黑名单查询（如餐饮连锁供应链违规记录）。

通过以上方法，优先选择有稳定种植基地、成熟分拣流程、可提供历史数据、能接受严格合同条款的供应商。如果供应商无法提供任何可验证的证据或应急方案，应谨慎合作。
