建议按以下步骤与供应商沟通，确保快速、稳定地补货：

1. **提前沟通**：立即联系供应商，说明旺季菠萝用量突然增大，确认对方是否有现货、能提供的最大数量以及最短交货时间。

2. **明确临时补货条件**：协商临时补货的价格（可能略高于常规价）、最低起订量、付款方式（如预付部分定金）以及是否包含运费。

3. **签订短期补充协议**：以书面形式（邮件或微信确认）约定：补货数量、单价、交货日期、验收标准（外观、成熟度、重量等）以及违约责任（如延迟交货的赔偿条款）。

4. **协商弹性条款**：请求供应商允许“分批送货”或“预留存量”，例如每周固定送一次，同时根据销售情况提前2天通知调整数量。

5. **保留备选方案**：同步联系1-2家其他供应商作为应急储备，并告知主供应商，适当增加谈判筹码。

6. **注意合规**：确保供应商资质齐全（如食品经营许可证），并要求提供批次质检报告，尤其注意菠萝农药残留和新鲜度。

7. **长期建议**：旺季结束后，与供应商复盘本次临时补货流程，考虑签订包含“旺季增量优先供应权”的长期框架协议。
