签订年度合同时，以下条款最容易出现风险，建议重点审查并明确约定：

1. **价格与调价机制**  
   - 陷阱：固定价格未考虑市场波动，或调价条款模糊（如“随行就市”无具体参照标准）。  
   - 建议：明确基准价、调价触发条件（如CPI、产地行情）、调价周期及通知方式。

2. **质量标准与验收**  
   - 陷阱：标准笼统（如“新鲜”无量化指标），或验收权责不清（如到货后仅外观检验）。  
   - 建议：附详细质量清单（甜度、大小、损伤比例），约定第三方检测或到库抽检流程，及不合格品的退货/折扣规则。

3. **交付与物流责任**  
   - 陷阱：未明确运输风险承担（如冷链断裂），或损耗率模糊（如“合理损耗”无比例）。  
   - 建议：约定交货地点（如供应商仓库或超市仓库），物流方责任归属（如冷链车温控标准），以及自然损耗率上限（如3%以内）。

4. **付款与账期**  
   - 陷阱：长账期（如90天）导致供应商资金压力，或激励条款不明确（如“提前付款折扣”未标明具体比例）。  
   - 建议：设置阶梯账期（如15天付款可享折扣），延期付款的违约金比例，及发票开具时间。

5. **违约责任**  
   - 陷阱：仅约定单方责任，或赔偿上限过低（如“仅供货总价10%”）。  
   - 建议：双向约定（如超市拒绝收货的补偿），明确质量问题的赔偿计算方式（如按回收成本+滞销损失）。

6. **续约与终止**  
   - 陷阱：自动续约陷阱，或提前终止条件苛刻（如“无故终止需付全年利润”）。  
   - 建议：明确续约需双方书面确认，提前终止的通知期（如30天）及合理补偿（如按已交货比例结算）。

**操作建议**：
- 保留书面沟通记录（如微信、邮件）作为合同附件。
- 对季节性水果（如草莓、荔枝），补充“尾期降级条款”（如产量不足时允许替换同价值品种）。
- 争议解决条款避免“协商不成再仲裁”的模糊表述，直接约定具体仲裁机构或法院。
