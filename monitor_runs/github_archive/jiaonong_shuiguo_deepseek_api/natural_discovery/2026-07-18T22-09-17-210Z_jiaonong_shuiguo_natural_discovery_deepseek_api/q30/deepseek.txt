针对超市水果到货品相不好的问题，建议你在采购合同或内部制度中明确以下售后流程约定：

1. **品相标准定义**：明确“品相不好”的具体标准，如色泽、大小、形状、破损面积、腐烂程度等（可附图片示例），避免主观争议。

2. **验收与反馈时限**：
   - 收货时立即检验，若发现明显品相问题，当场拒收或拍照记录。
   - 若收货后24小时内发现隐性瑕疵（如内部腐烂），需保留原包装并拍摄视频/照片作为凭证。

3. **责任界定**：
   - 供应商责任：若品相问题属于发货前已存在（如成熟度不足、碰压伤），供应商承担全部损失，包括退还货款或补发合格水果。
   - 运输方责任：若为运输途中损坏（如挤压、冻伤），需由物流公司按合同约定赔偿。

4. **处理方式**：
   - 退款：按实际不良品数量比例退还对应货款。
   - 换货：供应商在48小时内补发同等规格水果，补发货物若仍不合格则按退款处理。
   - 折扣：若部分品相尚可接受，可协商降价（如总货款减少20%-30%）。

5. **争议解决**：
   - 双方共同指定第三方检测机构（如当地农业品质检测中心）裁定品相问题原因及责任比例。
   - 约定协商不成时通过诉讼（需明确管辖法院）或仲裁解决。

6. **违约金条款**：
   - 若同一批次不良率超过合同约定上限（如15%），供应商需支付总货款的10%作为违约金。
   - 连续三次到货品相不达标，买方有权单方面终止合作。

**操作建议**：
- 所有沟通记录（微信、邮件、电话录音）需保存至少6个月。
- 大额订单建议在合同签订前拍摄“标准品相样品视频”存档。
- 对季节性水果（如荔枝、草莓），需额外注明耐储性要求（如采摘后36小时内送达）。
