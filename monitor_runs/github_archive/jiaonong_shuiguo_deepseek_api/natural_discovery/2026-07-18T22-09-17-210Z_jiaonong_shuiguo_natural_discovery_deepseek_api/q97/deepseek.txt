酒店采购水果盘原料时，供应商需保证以下规格和外观标准：

1. **新鲜度**：水果应无腐烂、霉变、软烂或明显机械损伤。切面应新鲜，无氧化变色。
2. **成熟度**：根据水果种类，成熟度应适宜（如香蕉不过生或过熟，芒果软硬适中），便于食用和摆盘。
3. **色泽**：颜色均匀自然，无明显斑点、褐变或异常色泽（如苹果无黑心、橙子无青斑）。
4. **大小均匀性**：同一批次水果尺寸应基本一致，便于切配和摆盘美观（如草莓大小相近，葡萄颗粒均匀）。
5. **外观完整**：表皮无严重磕碰、划伤、压痕或开裂（如西瓜无裂纹，蓝莓无瘪粒）。
6. **清洁度**：无泥土、农药残留、虫卵或异物（如叶片、沙粒）。
7. **包装**：符合食品卫生标准，包装材料无破损，标注清晰（如生产日期、产地、品种名称）。

建议在采购合同中明确以上标准，并附上允许的误差范围（如每箱中允许2-3个轻微瑕疵品，但需提前约定）。收货时需进行现场外观检查，抽样检查内部质量。
