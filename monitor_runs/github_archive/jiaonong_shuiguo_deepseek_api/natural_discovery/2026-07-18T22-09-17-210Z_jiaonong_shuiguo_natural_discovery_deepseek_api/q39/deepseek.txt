通常情况下，供应商能够为商超自有品牌水果提供包装和规格支持，但具体支持程度取决于供应商的类型和合作模式。以下是关键信息：

1. **包装支持**：许多供应商（尤其是大型水果种植企业、合作社或专业包装厂）可以定制包装，包括设计、材质（如纸箱、塑料盒、网袋等）、品牌标识（商超自有品牌Logo）和规格（如不同斤数、组合装等）。但小规模供应商可能仅提供基础包装。

2. **规格支持**：供应商通常能根据商超需求调整水果品级、大小、成熟度、包装数量等规格。例如，可以按标准重量（如5斤/箱）、单果规格（如苹果直径80mm）或混装组合（如“家庭混合装”）供货。

3. **注意事项**：
   - **起订量**：定制包装和特殊规格通常有最低起订量要求，需提前协商。
   - **附加成本**：设计费、打样费、模具费可能由商超承担或分摊。
   - **合同条款**：明确包装材料标准、规格误差范围、交货时间等细节。

建议行动：
- 直接联系意向供应商，列出具体需求（如包装材质、规格尺寸、预计销量），要求提供样品和报价。
- 可优先选择具备水果产销一体化能力的供应商（如大型农场或专业水果供应链公司），因其包装和规格调整能力更强。

如果供应商无法完全满足要求，可考虑追加一家专业包装服务商协助。
