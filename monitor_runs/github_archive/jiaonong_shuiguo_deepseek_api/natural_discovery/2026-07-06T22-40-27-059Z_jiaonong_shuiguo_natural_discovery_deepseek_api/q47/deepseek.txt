为茶饮店采购香蕉时，控制熟度非常重要，因为这直接影响饮品的口感、甜度和出杯稳定性。以下是具体建议：

### 1. 按用途区分熟度标准
- **做奶昔、冰沙、香蕉拿铁类饮品**：需要**完全成熟**（表皮出现少量黑点，手感软但不变形）。此时果肉甜度最高、香气最浓，打碎后细腻无涩味。
- **做香蕉蛋糕、布丁等烘焙甜品**：使用**过熟香蕉**（表皮大面积黑斑，手感极软）。天然糖分和香气更突出，能减少额外加糖。
- **做切片装饰、果昔碗顶料**：选**八成熟**（表皮金黄无斑，手感稍硬）。切片后不易氧化变黑，保持形状。
- **做冷藏保存的配料（如香蕉泥）**：可采购**七成熟**（表皮青黄色偏绿），常温放置1-2天后使用，确保出餐期间不腐烂。

### 2. 采购时的操作要点
- **与供应商约定规格**：明确要求“七成熟（青黄皮）可放3天”或“全熟（有斑点）需当天用”，并在合同或订单备注。
- **分批次采购**：每周2-3次少量进货，避免单一批次熟度集中。常温库房按成熟度分区域摆放：青的放通风处、熟的放冷藏（12℃以下抑制进一步成熟）。
- **用感官检查**：到货时随机掰开两根：
  - 果肉紧实、颜色乳白→偏生
  - 果肉绵软、颜色淡黄→刚好
  - 果肉透明或发黑→过熟或冻伤（拒收）
- **冷链运输注意事项**：香蕉对低温敏感，运输温度应在13-15℃以上。若收到时果皮变黑但果肉正常，是运输受寒，需冷藏观察，当天内用。

### 3. 内部品控标准（示例）
| 品类 | 采购熟度 | 适宜使用期 | 储存方式 |
|------|----------|------------|----------|
| 冰沙类 | 微有斑点的全熟 | 到货后24小时内 | 常温阴凉处 |
| 装饰切片 | 金黄无斑的八成熟 | 到货后2天内 | 冷藏（10-12℃） |
| 甜品馅料 | 大面积黑斑的过熟 | 到货当天用 | 急冻（可保存1周） |

### 4. 紧急处理方案
- **到货偏生（硬且涩）**：放入纸箱，放入一根红苹果或熟香蕉，密闭催熟1-2天。
- **到货过熟（出水或发黑）**：冷冻保存作香蕉泥，或立即做成熟香蕉酱（加糖熬煮后冷藏）。
- **短期用不完**：剥皮切段真空冷冻（避免氧化），解冻后只适合做泥类。

最后，建议每周盘点一次香蕉的熟度分布，对常用品种（如巴西蕉、小米蕉）的成熟速度做记录，逐步精准调整采购量。
