小商家采购水果时，现金结算和月结账期各自的风险如下：

**现金结算风险：**
- **资金占用压力大**：需即时支付全款，可能影响流动资金周转，尤其水果易损耗、价格波动大，若销售不畅可能导致亏损。
- **无账期缓冲**：无法通过延迟付款应对突发问题（如水果变质、滞销），需自行承担全部库存风险。
- **交易凭证缺失**：现金交易若未规范记录（如收据、合同），纠纷时难举证（如质量不符、数量短缺）。

**月结账期风险：**
- **供应商信用风险**：若供应商履约能力差或倒闭，可能无法按时收款，甚至出现“跑路”导致货款全损。
- **账期管理成本**：需人工核对账单、催收，增加时间和管理成本；可能因对账误差引发矛盾。
- **资金链风险**：若自身回款不及时（如客户拖欠），可能无法按期结账，影响合作关系或触发违约金。
- **价格波动风险**：采购时约定价格，但账期内水果价格下跌，可能面临“高价买、低价卖”的亏损。

**建议：**
- 对信誉好的长期供应商，可尝试“部分现金+短期账期”组合（如预付30%，余款7日内结清）。
- 无论哪种方式，务必保留书面合同、送货单、转账记录等凭证，明确数量、质量标准和违约责任。
- 小商家初期建议以现金结算为主，优先保证资金安全；随着规模扩大，再逐步引入账期并控制比例（如不超过20%）。
