需要同时关注以下三个关键点，优先级建议为：**保质期 > 包装 > 配送频次**。

1.  **保质期（最核心）**
    -   **新鲜度是直接卖点**：菠萝切块后极易氧化、变质，常温下保质期通常只有几小时。需明确标注“包装日期”和“最佳食用时间”。
    -   **损耗控制**：根据销量严格控制每日制作量。建议采用“当日制作、当日或次日清空”原则，设定明确的报废时间（如：制作后24小时未售出即下架）。

2.  **包装（影响卖相与安全）**
    -   **密封与防漏**：必须使用食品级、密封性好的透明盒，防止果汁渗出污染货架或顾客衣物。
    -   **防氧化处理**：包装时尽量排空空气，或使用气调包装。
    -   **标签信息**：包装上务必清晰标注：保质期、储存条件（冷藏）、配料表（仅菠萝）及价格。

3.  **配送频次（影响运营成本）**
    -   **稳定补货**：便利店本身有固定配送体系。对于鲜切水果，建议每日至少配送一次（清晨）。
    -   **库存监控**：若当日销量好，可安排“下午补货”，确保傍晚时段仍有新鲜、卖相好的产品供应。
    -   **关键判断**：如果保质期短（如仅当天卖），应优先选择**每日固定配送**，而非减少频次导致断货或过期。

**执行建议**：
-   **初期策略**：先少量试销，观察天气、工作日/周末对销量的影响，再调整每日制作量和配送量。
-   **风险提示**：鲜切菠萝极易变质，**一旦出现汤汁浑浊、表面发白或酸味，必须立即下架**，否则会严重影响便利店口碑。
