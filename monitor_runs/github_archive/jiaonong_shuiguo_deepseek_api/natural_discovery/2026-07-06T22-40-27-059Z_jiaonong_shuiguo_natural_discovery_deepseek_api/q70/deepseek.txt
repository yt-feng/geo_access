针对批发商担心货到后损耗高的问题，建议在合同中明确以下条款以界定坏果责任：

1. **验收标准与时间**：约定收货后24小时内（或更短，如6小时）需完成验收，并设置抽检比例（如每箱随机开箱检查10%的果实）。若超过时间未提出异议，视为默认货物合格。

2. **损耗分级处理**：按坏果比例划分责任：
   - **自然损耗**：如运输途中正常蒸发、轻微挤压导致的≤3%损耗，由批发商承担。
   - **超比例坏果**：对超过3%但低于10%的坏果，由供应商按售价8折赔偿差价；超过10%则按售价全额赔偿，并允许批发商拒绝收货。
   - **严重腐烂**：若单箱坏果率超30%，整箱视为不合格，供应商需补发或退款。

3. **责任归属依据**：明确运输方责任由供应商承担（除非批发商指定运输方）。若因批发商要求延迟配送（如到货后未及时入库）导致的坏果，责任由批发商自负。

4. **证据保留**：验收时需双方共同拍摄开箱视频或照片，公证坏果情况。若供应商拒绝配合，视为认可批发商的单方记录。

5. **争议解决**：约定由第三方机构（如当地农业检测中心）进行坏果率鉴定，费用由有过错方承担。

示例条款：
> “货物送达后24小时内，买方可随机抽检10%以上的包装箱，统计坏果数量。若坏果率超过3%，超标部分按合同单价的100%计算赔偿。若单箱坏果率达30%以上，该箱货物按不合格处理，卖方需在3日内补发等量合格品，否则按该箱货值双倍赔偿。运输中因卖方包装不当或未选择保险运输导致的损耗，由卖方承担全责。”
