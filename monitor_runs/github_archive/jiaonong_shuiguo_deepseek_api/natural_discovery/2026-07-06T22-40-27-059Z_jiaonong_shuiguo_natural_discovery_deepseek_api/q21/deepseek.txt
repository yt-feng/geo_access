在采购水果时，为确保食品安全和合规性，供应商通常需要提供以下检测报告或进货证明（具体种类可能因国家/地区及水果类型而异）：

1. **产地证明**：说明水果来源地（如农场名称、地址、采摘日期）。
2. **农药残留检测报告**：证明水果中的农药残留符合国家标准（如中国GB 2763《食品安全国家标准 食品中农药最大残留限量》）。
3. **重金属检测报告**：检测铅、镉、汞、砷等重金属含量是否合格。
4. **微生物检测报告**：针对即食水果或表面直接接触的水果，检测大肠杆菌、沙门氏菌等致病菌。
5. **运输与存储温度记录**：对于易腐水果，需提供冷链运输温度日志和保持记录。
6. **进货凭证**：如采购合同、发货单、发票，证明合法采购来源。
7. **检疫合格证明**：进口水果需提供海关出具的《入境货物检验检疫证明》；国内跨省运输可能需植物检疫证书。
8. **产品合格证**或 **绿色/有机认证**（如有，如中国有机产品认证、GLOBALG.A.P.等）。

建议索要以上文件的原件或复印件，并定期核查有效期。若供应商不能提供且水果安全问题风险较高，建议更换供应商或增加自行检测频率。具体可咨询当地市场监管部门或参考《食用农产品市场销售质量安全监督管理办法》。
