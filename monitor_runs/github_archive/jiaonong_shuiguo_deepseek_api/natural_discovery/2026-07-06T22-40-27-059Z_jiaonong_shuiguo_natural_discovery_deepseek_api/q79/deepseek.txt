最应该优先问**配送**。原因如下：

1. **配送决定水果品质**：水果是生鲜品，配送时间长、制冷不足或搬运不当会直接影响新鲜度和损耗率。即使价格再便宜、售后承诺再好，频繁出现烂果、失水，团长的口碑会立刻受损。
2. **影响交付体验**：团购通常有固定的提货时间，供应商能否准时、按量、规范包装送达，决定了团长能否顺利完成分拣和分发。
3. **价格和售后可在后期调整**：价格可以谈判，售后条款可以补充（如坏果包赔、比例限制），但配送能力差往往很难短期改善，且成本高。

**建议顺序**：  
先谈**配送**（时效、冷链、破损控制） → 再谈**售后**（坏果赔付比例、处理流程） → 最后谈**价格**（协商账期、满减、阶梯价）。
