判断苹果和橙子是否适合门店销售，可以遵循以下具体标准：

1. **看果径大小**  
   - 苹果：一般要求果径在70mm-85mm之间（中小型门店可接受65mm起步），太小卖相差、太大易压坏。  
   - 橙子：常见果径在70mm-80mm，精品店可选75mm以上，太小的（<65mm）酸度高且顾客接受度低。  
   - **判断方法**：用水果尺（果径环）快速测量，或目测比上图的“圈圈”模板（比如门市常备的3-5个标准尺寸孔板）。

2. **看外观均匀度**  
   - 同一筐内大小差异不超过10mm（如75mm±5mm），否则混合摆放影响视觉感。  
   - 形状要端正，畸形果（如严重偏斜、棱纹、凹凸不平）剔除；苹果要无“倒三角”或严重高低肩，橙子要圆度好、无硬梗。

3. **看颜色与成熟度**  
   - 苹果：底色金黄或淡黄，红色面积占50%以上（红富士等品种）；绿底多或全青未熟的不推荐。  
   - 橙子：果皮橘黄至橙红色，光滑有光泽；暗黄或带青斑多为酸涩果。  
   - 判断：轻按果皮有弹性的为成熟，过软（可能内部变质）或过硬（未熟）的剔除。

4. **看重量与手感**  
   - 同个头下，苹果应150-250g，橙子200-300g。太轻（水分流失或不饱满）和太重（含水率异常）都不行。  
   - 拿在手里有沉实感、果柄鲜绿不干枯。

5. **现场快速筛选步骤**  
   - 第一步：用果径环挑出小于下限和大于上限的。  
   - 第二步：目测颜色和形状，剔除不佳的。  
   - 第三步：掂重量，抓出过轻的。  
   - 第四步：随机抽样切开，确认内部无黑心、木质化、冻伤等。

**最终标准**：符合以上80%以上条件的果品即可上架，但需结合门店定位（社区店可稍放宽品质、精品店严控果径≥75mm且色泽一致）。
