为提高超市水果区客单价，建议采用以下组合促销策略：

1. **“热带风情组合”**：将香蕉和菠萝打包销售，利用它们互补的颜色和口感（香蕉的软糯+菠萝的酸甜），定价比单买低10%-15%。单价较高的菠萝能拉高整体客单价。

2. **“全家营养套装”**：苹果（耐储存、日常消耗快）+香蕉（易消化）+菠萝（高纤维），按家庭3-5天份量组合，定价为三者单买总价的9折，并附赠保鲜袋或水果刀等小赠品。

3. **“果篮升级”策略**：将香蕉作为基础，苹果作为中间填充物，菠萝作为视觉亮点，直接组装成简易果篮（若原本只卖单果）。按份标价，单价可提升20%-30%。

4. **“买贵送便”**：购买菠萝或香蕉（单价更高任一），可加0.5元换购苹果，利用高价品带动低价品，且不显著降低整体客单价。

5. **“混合拼装秤”**：将香蕉、苹果、菠萝分别切块（需备相应包装），按“满一盒”（如500g）统一标准价促销，便于顾客一次性购买三种水果，且比单独购买平均单价更高。

请确保促销信息明确、标签醒目，并配合试吃或现场展示，提升购买冲动。注意检查菠萝处理是否安全（去皮工具）及香蕉保存时间（避免过熟导致损耗）。
