选择社区超市的水果供应商时，建议从以下几个关键条件衡量：

1. **配送稳定性**：确认供应商承诺每日固定时间送达，避免断货。可要求提供过往配送记录或合同约束。

2. **水果新鲜度与质量**：要求供应商提供源头信息（如产地、采摘时间），并制定验收标准（如外观、成熟度、有无损伤）。建议先试送一批检验。

3. **品种多样性**：确保供应商能提供日常常见水果（如苹果、香蕉、柑橘）以及季节性品种，满足社区多样化需求。

4. **价格与结算方式**：比价至少3家供应商，明确批发价、折扣政策、付款周期（如月结、现结），避免模糊条款。

5. **退换货保障**：确认售后政策，例如烂果、坏果的包退包换比例（如5%以内不计，超出部分补偿）。

6. **资质与信誉**：核查营业执照、食品经营许可证，并查询过往客户评价或与其他超市的合作案例。

7. **物流与仓储条件**：询问供应商是否有冷链运输或每日分拣能力，确保到货商品状态良好。

**建议行动**：优先从本地大型水果批发市场或专做社区配送的供应商中筛选，要求对方提供样品测试一周，再签约合作。
