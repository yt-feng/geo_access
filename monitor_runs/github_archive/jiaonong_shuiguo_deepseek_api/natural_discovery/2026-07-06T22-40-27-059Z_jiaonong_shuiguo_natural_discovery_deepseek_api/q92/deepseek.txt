对于面积小的便利店，水果品类的管理与损耗控制确实更具挑战。核心策略是**“少而精、快周转”**。

以下是具体的控制方法：

**一、 品类控制（选对品，少即是多）**

1.  **主打“高频刚需+方便吃”的品种**：
    *   **香蕉**：全年都有，转化率高，是公认的引流品。
    *   **苹果/梨**：耐储存，家家都需要，但不要上太多高价品种，主推红富士、皇冠梨即可。
    *   **应季爆款**：比如夏天的西瓜（切半或切块卖）、秋天的橘子/蜜柚。应季水果价格低、味道好、需求旺。
    *   **预包装/盒装水果**：如切好的菠萝、哈密瓜、西瓜、草莓盒。**这是小便利店的核心**，减少顾客挑选和称重时间，也降低被顾客捏坏的风险。

2.  **坚决放弃的品类**：
    *   **整颗大西瓜**：除非你有位置能切开卖，否则别进整颗，大且占地。
    *   **易挤压/怕碰的品种**：如草莓、蓝莓。若进，只能进小盒装，且摆放在最安全、显眼的位置。
    *   **小众水果**：如牛油果、百香果、山竹。购买人群窄，容易死库存，小面积门店不碰。

3.  **单品精简原则**：
    *   把水果货架控制在 **5-8个单品**。
    *   其中：1个引流品（香蕉/应季低价品）+ 2个常规品（苹果、梨）+ 2个应季品（橘子/葡萄等）+ 2个高毛利品（预包装果切或进口小果）。

**二、 损耗控制（动作要快，习惯要改）**

1.  **源头控货：只进“硬七分熟”**。
    *   宁可多跑一趟，也不要进太熟、太软的货。例如香蕉，进颜色微青、皮紧绷的，能在店里放2-3天。熟透的坚决不要。
    *   小门店没有冷库，每次进货量要算好**“半天”**的量，而不是“一天”的量。如果下午客流少，上午少进。

2.  **陈列技巧：改变摆法**。
    *   **不堆不叠**：香蕉、芒果、柠檬不建议堆在一起，要平铺单层或两层，避免底层被压烂。
    *   **带把朝外**：葡萄、提子，把柄朝向顾客，减少顾客捏拽。
    *   **上小下大**：上层放最熟、最该出清的；下层放稍硬的。顾客习惯拿上面的。

3.  **出清机制：建立“鲜度时钟”**。
    *   **黄金出清时间**：下午3-5点、晚上7-8点。在这个时间点前，对品相变差、卖相变软的水果进行打折。
    *   **分级定价**：设3个价格区：
        *   **精品区**（最漂亮，正常价或略高）
        *   **通货区**（稍瑕疵，便宜10%-20%）
        *   **出清区**（明显软或小瑕疵，**现做现卖**，如做成一杯杯的果切/果汁，定价在成本价附近快速走掉）。
    *   **绝对原则**：**“有烂即扔”**。只要有一个霉点，立即剔除，防止感染整箱。

4.  **包装改量**：
    *   **用袋装代替散装**：对于苹果、橙子、柠檬，建议一袋3个或一袋5个，用保鲜膜或网兜封好，顾客直接拿，无需挑选，减少损耗。
    *   **盒装果切**：把卖相不好的但口感好的西瓜、凤梨、哈密瓜去皮切块，用透明盒装好，单价低、走量快。

5.  **营业后复盘**：
    *   每天记录：今天哪种水果剩得最多？哪种全卖光了？
    *   连续3天都剩最多的品类，直接砍掉，换成另一种。
    *   连续7天都卖光的品类，可以适当增加进货量10%。

**三、 一句话总结**

> **香蕉引客流，果切赚毛利，应季刷存在，腐果立刻清。**
> **宁可卖断货，不要卖剩货。**
