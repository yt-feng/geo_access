针对团购客户价格敏感的特点，建议优先选择**阶梯价**，将返利作为辅助激励。具体分析如下：

1. **阶梯价更直接匹配价格敏感心理**  
   阶梯价根据采购量直接降低单价，客户能立即感知到“买得多省得多”，降低决策门槛。例如：100件单价100元，300件单价85元。这种机制简单透明，客户无需计算复杂返利条件，更符合团购客户对“即时优惠”的需求。

2. **返利适用于长期合作或附加条件场景**  
   若客户需后续复购（如月度团购）或配合推广（如门店展示、推荐新客），返利可设置在销量达标后额外返还（如年采购量超5000件返3%）。但需注意：  
   - **明确前置条件**：告知客户达到多少量或完成何种行为能获得返利。  
   - **降低兑付门槛**：避免设置“年度总量”等复杂标准，建议使用季度或单次阶梯返利。

3. **结合策略：阶梯价为主+返利为激励**  
   - 对首次合作或临时团购：直接使用阶梯价锁定订单。  
   - 对长期客户：在阶梯价基础上，额外设置“年度返利”，激励持续合作。例如：阶梯价已享受批量优惠，若全年采购额超过20万元，再返2%。  

**建议执行步骤**：  
1. 根据成本数据，划分2-3个清晰的价格阶梯（如5%-15%折扣梯度）。  
2. 对老客户或大客户可附加简易返利规则（如“单次超万元订单，下期额外返5%代金券”）。  
3. 避免使用“满减券”“复杂积分兑换”等需要客户计算的返利形式。  

总结：**团购客户更倾向“所见即所得”的优惠，阶梯价能快速促进成交；返利更适合锁客，但需降低理解成本**。
