水果采购与供应商谈锁价（价格锁定）的期限通常取决于多种因素，包括水果品类、市场供需状况、合作信任度以及合同条款。一般来说，锁价期限有以下常见情况：

1. **短期锁价（1-2周）**：适用于鲜果、叶菜等易腐或季节性强的水果（如草莓、荔枝、葡萄）。这类水果受天气、运输影响大，供应商通常只愿意短期锁定价格，以避免市场波动风险。

2. **中期锁价（1-3个月）**：适用于保质期较长、产量相对稳定的水果（如苹果、橙子、梨、香蕉）。若采购量较大且双方有长期合作基础，可协商锁定1-3个月，但需约定“触发条件”（如价格波动超过10%-20%时重新协商）。

3. **长期锁价（半年至1年）**：极少数情况，主要适用于反季水果（如冷冻水果）或大宗进口水果（如进口车厘子、奇异果）。这需要签订长协合同，通常附带“价格调整机制”（如参照批发市场指数或基差定价）。

**实操建议**：
- 对热门、易变质水果，优先锁价1个月以内，并加入“按质论价”条款。
- 对库存能力较强的水果（如苹果），可谈3个月锁价，但要求供应商提供“最低供货量”作为条件。
- 谈判时明确“锁价”是否包含包装、运输等成本，避免后期加价。
- 若市场看涨，可主动提出“预付定金”换取更长期锁价。

最终期限需结合具体品类和双方议价能力，建议先小量试锁1个月，根据合作效果逐步延长。
