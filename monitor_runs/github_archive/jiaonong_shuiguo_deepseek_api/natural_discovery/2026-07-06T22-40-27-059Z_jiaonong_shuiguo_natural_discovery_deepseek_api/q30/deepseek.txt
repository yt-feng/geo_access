针对超市水果到货后品相不好的情况，建议在采购合同或供货协议中明确以下售后流程约定，以保障双方权益：

1. **验收时效与标准**  
   - 约定**收货后24小时内**为验收期，买方需在此期限内对水果品相（如色泽、大小、损伤、腐烂等）进行书面或拍照记录。
   - 明确“品相不好”的具体标准（例如：单果腐烂面积超过5%、碰伤超过2处、色泽不达标等）。

2. **处理方式**  
   - **部分不良品**：按“协商折价”处理（如总价扣减10%-30%），或要求卖方在下一批供货中补发同等级水果。
   - **大量不良品（超过到货量10%）**：买方有权拒收整批货物，并要求卖方承担退货、补发或调换的运输费用。

3. **索赔流程**  
   - 买方在验收期内通过**书面通知**（含照片/视频证据）提出售后申请。
   - 卖方需在收到通知后**24小时内**确认处理方案（退款、补发或折价），逾期未回复视为默认买方提出的处理方式。

4. **责任划分**  
   - 明确运输损耗（如挤压、高温导致的损伤）归责方。若为物流问题，卖方需先向买方赔付，再自行与物流协商。
   - 约定“非验收期内发现的问题”，除非能证明运输或包装隐性缺陷，否则由买方承担。

5. **争议解决**  
   - 若双方对品相标准存在分歧，可约定由**第三方质检机构**（如当地农业检测中心）出具报告作为最终依据，费用由责任方承担。

6. **记录留存**  
   - 要求双方保留物流单号、验收照片、沟通记录（如微信/邮件截图）至少6个月，以备核查。

**示例条款**：  
“乙方（卖方）应确保交付水果品相符合约定标准。甲方（买方）收货后24小时内如发现品相不良，有权按以下方式处理：  
（1）不良比例≤10%时，按总价10%折价；  
（2）不良比例＞10%时，甲方可要求乙方在48小时内免费补发同等数量货物，或全额退款并承担运费。  
逾期未处理的，视为乙方同意甲方提出的处理方案。”
