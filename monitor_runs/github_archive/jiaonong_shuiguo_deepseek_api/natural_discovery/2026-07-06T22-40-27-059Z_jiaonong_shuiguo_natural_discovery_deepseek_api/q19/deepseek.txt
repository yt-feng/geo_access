在购买超市菠萝时，建议综合关注以下四个维度，按优先级排序：

1. **核心指标：甜度与成熟度（最重要）**  
   - **外观判断**：表皮呈金黄色或橙黄色（非全绿），叶片翠绿易拔落，底部有淡淡果香。  
   - **触感**：按压菠萝中部有轻微弹性（太硬生涩，太软过熟）。  
   - **避免陷阱**：超市切好的菠萝可能为了掩盖未成熟果肉，建议选完整水果。

2. **实用考量：切果损耗（间接经济因素）**  
   - 超市经常削皮切块的菠萝，如果果肉发白、硬芯、边缘干褐，可能是次品损耗转化的。  
   - **优先选整果**：自己切虽然麻烦，但能避免买到霉变或半腐烂的切块（损耗率高）。

3. **辅助参考：大小与外观（中低优先级）**  
   - **大小**：同等成熟度下，选中等偏大（约1.5-2kg）的菠萝，果肉比例较高；极小的菠萝常因生长期不足而酸涩。  
   - **外观**：避免有明显黑斑、软烂、虫眼或形状畸形的菠萝，但轻微刺手是正常现象（品种差异）。

**总结建议**：先通过颜色和香味判断甜度，再选整果（避免切果损耗陷阱），不必过度纠结大小和完美外观。如果超市提供试吃，可结合口感决定。
