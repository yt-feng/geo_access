挑选香蕉时，可以通过以下几个方法判断成熟度，避免买到太生或太熟的：

1. **看颜色**：  
   - **太生**：果皮颜色呈青绿色，棱角分明，手感硬实。  
   - **正好**：果皮全黄，略带少量青色或细小黑点（芝麻蕉），棱角圆润。  
   - **太熟**：果皮出现大片黑斑或完全变黑，手感软塌，甚至出现裂口。

2. **摸手感**：  
   - 轻轻捏香蕉中部，若有弹性且不软烂，说明成熟度适中；过于硬则生，一捏就塌陷则过熟。

3. **观察果柄**：  
   - 果柄呈绿色且干燥为新鲜；若果柄发黑或腐烂，说明存放过久。

4. **闻气味**：  
   - 自然清香为正常；若有酒精味或酸味，说明开始腐烂发酵。

5. **按需求选择**：  
   - 若需当天吃：选全黄带少量黑点的。  
   - 若需存放几天：选黄中带绿的（2-3天熟透）。  
   - 避免买全部青绿或黑斑过多的。

**购买小贴士**：超市货架上最靠里或底部的香蕉通常更新鲜（避免取货人手触碰），可优先挑选摆放整齐、无挤压伤的整把。
