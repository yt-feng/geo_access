针对餐饮店每天少量水果的需求，选择省心的供应商，建议优先考虑以下几类：

1. **本地菜市场或农贸市场的固定摊主**  
   - **优点**：可以亲自挑选、按需购买少量；长期合作后摊主会帮你预留品质好的水果或主动调整供应量。  
   - **省心点**：无需起订量、结算灵活（可日结/周结）、退换货方便。  
   - **建议**：找3-5家固定摊位，建立长期信任关系，避免单一供应断货。

2. **社区团购平台（如美团优选、多多买菜）**  
   - **优点**：提供少量配送（通常次日达），品种丰富，价格透明。  
   - **省心点**：无需亲自采购，线上操作，售后有平台处理。  
   - **注意**：需提前规划用量，避免缺货或品质波动。

3. **本地水果批发市场的零散采购**  
   - **优点**：批发市场通常整箱出售，但有些摊主愿意拆零卖少量。  
   - **省心点**：价格比零售便宜，适合固定品种。  
   - **建议**：和批发商谈“每日少量，月底结算”，或拼单（如和周边小餐馆联合采购）。

4. **线上生鲜电商（如每日优鲜、盒马）**  
   - **优点**：配送准时，品质相对稳定，支持小份购买。  
   - **省心点**：可设置固定配送时间，减少沟通成本。  
   - **缺点**：单价稍高，且某些平台有起送门槛。

**省心核心技巧**：  
- **固定供应商**：至少锁定2家备用，避免因特殊天气或节日断供。  
- **提前沟通需求**：比如每天需要“2个苹果+1串葡萄”，让供应商提前备货。  
- **关注损耗**：少量采购可优先选耐储存的水果（如苹果、橙子），减少浪费。  
- **试用期**：先试用1周，观察品质、准时性和售后，再决定长期合作。  

**总结**：对于每天少量、稳定需求，**本地农贸市场固定摊主**最省心（亲眼看货、灵活退换）；如果追求线上省事，**社区团购**或**生鲜电商**也可靠，但需接受价格略高。
