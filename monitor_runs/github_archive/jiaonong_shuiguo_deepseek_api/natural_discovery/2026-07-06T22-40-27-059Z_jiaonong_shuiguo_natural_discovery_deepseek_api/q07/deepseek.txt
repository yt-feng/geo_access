根据市场需求和大众口味，以下是4种水果的推荐搭配方案，兼顾颜值、口感和实用性：

**方案一：经典喜庆款（适合送礼、探病）**
- 搭配：**苹果（红富士） + 橙子（脐橙）**
- 理由：苹果谐音“平安”，橙子寓意“心想事成”，红黄配色喜庆。这两种水果耐压、耐储存，不易损坏，适合长途携带或长久摆放。
- 附加建议：可加一两个石榴（多子多福）或火龙果（红红火火）。

**方案二：清爽健康款（适合年轻人、家庭自用）**
- 搭配：**橙子 + 香蕉 + 菠萝**
- 理由：橙子补充维C，香蕉提供饱腹感，菠萝开胃解腻。整体口感层次丰富（酸甜+软糯+脆甜），价格适中。
- 注意：香蕉易熟易压坏，需单独用泡沫网包装，底部放碎纸条缓冲。

**方案三：高颜值水果篮（适合节日、见朋友）**
- 搭配：**苹果 + 橙子 + 菠萝 + 少量香蕉**
- 理由：苹果垫底，橙子穿插空隙，菠萝放中间显大（用叶柄装饰），香蕉弯放边缘。整体红、橙、黄、绿四色自然分层。
- 成本控制：菠萝可用青柠或柠檬替代（成本更低），菠萝头可保留直接装饰。

**方案四：本地特色款（做差异化，适合社区/老客户）**
- 搭配：**上海/陕西苹果 + 赣南/湖南脐橙 + 都乐香蕉（带贴纸） + 徐闻/海南菠萝**
- 理由：突出产地背书（如“井冈山红米苹果”“柯基香蕉”），增强可信度。价格可上浮15%-20%。
- 附加品：附赠小卡片写明产地故事（手写更佳）。

**通用销售建议：**
1. **包装**：推荐竹编/藤编篮子（成本低、显档次），纸袋或礼盒选亮黄色/红色，印上“平安喜乐”“橙心橙意”等短语。
2. **定价**：礼盒价建议比散称总价上浮25%-40%（例如散卖合计30元，礼盒可标价42-50元）。
3. **针对易损水果**：菠萝用防撞网+保鲜膜单独包；香蕉不要和苹果放同一层（苹果会催熟香蕉）。
4. **小贴士**：如果本地菠萝/香蕉消费者接受度较低，可用**哈密瓜**或**圣女果**替换（更耐储存）。
