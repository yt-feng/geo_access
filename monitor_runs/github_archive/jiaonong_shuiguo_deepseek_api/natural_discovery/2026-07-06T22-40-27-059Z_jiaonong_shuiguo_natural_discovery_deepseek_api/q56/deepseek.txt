为了保障茶饮店的食品安全，水果供应商通常需要提供以下几类证明文件：

1. **营业执照**：供应商的合法经营资质证明，确保其为正规注册企业。
2. **食品经营许可证**：若供应商涉及食品销售（如鲜切水果、预包装水果），需持有此证；若仅为初级农产品（如整果），可能仅需营业执照。
3. **产品检验报告**：每批次水果的第三方检测报告或供应商自检报告，重点检测农药残留、重金属、微生物等指标（如GB 2763、GB 2762标准）。
4. **产地证明或进货凭证**：明确水果来源（如产地、采摘日期、供应商信息），确保可追溯。
5. **合格证或质量承诺书**：部分供应商可提供质量安全承诺书，或需符合当地市场监管要求。
6. **运输及储存资质**：若涉及冷链运输，需提供冷链物流资质或温度记录证明（如车辆消毒记录、温控设备校准证书）。
7. **行业特定认证**：如有机、绿色、无公害认证等（非强制，但可增强信任）。

**实用建议**：
- **索证存档**：建立供应商档案，每批次留存上述文件副本（电子或纸质）。
- **现场审核**：如条件允许，定期到供应商基地或仓库检查清洗、分拣、储存条件。
- **抽检制度**：茶饮店自购快速检测试剂（如农残速测卡）对高风险水果（如草莓、葡萄）进行抽检。
- **合同条款**：明确在采购合同中注明供应商需持续提供合格证明，并约定违约责任。

需注意：不同地区（如中国、东南亚）要求可能略有差异，建议同步咨询当地市场监管部门。
