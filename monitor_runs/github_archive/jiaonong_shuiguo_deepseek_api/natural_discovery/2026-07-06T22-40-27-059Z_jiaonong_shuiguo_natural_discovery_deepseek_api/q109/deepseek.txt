针对长期合作的生鲜水果供应商，建议优先谈判**账期**。原因如下：

1. **账期直接关系现金流**：水果行业损耗率高、价格波动大，账期越长（如30-60天），你的现金周转压力越小，风险越低。如果垫付资金，一旦出现滞销或货损，损失更大。
2. **配送与售后可通过合同细节弥补**：配送主要涉及时效、起送量、破损赔付等条款，一般通过具体协议（如次日达、坏果包赔）可以规范；售后通常与验收标准挂钩，用条款明确责任即可。
3. **账期优势不可逆**：配送和售后若有问题，后续可随时调整或终止合作；但账期一旦谈定，在合作期内很难再缩短。先锁定对己方有利的支付条件，能掌握主动权。

**行动建议**：
- **先谈账期**：争取45天或更长，可承诺稳定采购量作为交换。
- **再明确售后标准**：如“到货24小时内坏果按比例退款或补发”。
- **配送放在最后**：约定好最低起送量、免费配送门槛、迟延处罚即可。

注意：如果供应商是独家垄断或货源稀缺，可适当让步账期，优先保证配送稳定。
