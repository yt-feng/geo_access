判断菠萝批发哪个规格更容易出货，核心看**目标客户的消费场景**。通常建议优先关注**中果（单果重1.2-1.5斤）**，具体分析如下：

**1. 对比常见规格的优劣势（假设零售端）**

-   **大果（单果1.5斤以上）**：
    -   **出货难度**：中等偏高。
    -   **原因**：家庭购买通常一次吃不完，切口易污染、不好保存；对于水果店和超市，客单价高（总价高），消费者决策门槛提高。
    -   **适用场景**：主要面向工厂、食堂、宴会或果汁店等加工批发，而非C端散户。

-   **中果（单果1.2-1.5斤左右）**：
    -   **出货难度**：低（最畅销）。
    -   **原因**：大小适中，方便一次食用或两人分享；零售价相对亲民（单果价格约在10-15元区间，消费者无压力）。**这是大多数社区水果店、生鲜超市和线上团购的首选。**

-   **小果（单果1.2斤以下）**：
    -   **出货难度**：中低（但单价低）。
    -   **原因**：适合一个人吃或做果切，价格低吸引对价格敏感的人群。但批发毛利薄，且“卖相”不如中果，如果整体市场缺货或消费降级明显时，小果出货快。

**2. 直接判断“更容易出货”的方法（实操建议）**

-   **看周边场景**：如果主销超市、住宅区周边水果店，**80%会选1.2-1.5斤规格**。如果周围有大量工地、工厂或高校食堂，**大果**反而好出货。
-   **盯商圈价格带**：中果的零售单价通常可以控制在“10元一个”或“15元2个”的关键价格锚点，这最容易让消费者无脑下单。
-   **做小范围试销**：进货时，拿**5箱中果、2箱大果、1箱小果**混合试销。如果2-3天内中果最先卖完，且客诉少（没有抱怨太大吃不完或太小不好削），那它就是你选定的主力。

**3. 额外提醒（防止踩坑）**

-   **成熟度比规格更重要**：相同大小的菠萝，**金黄色的要比青绿色的**更容易出货（除非区域偏好青皮酸脆口感），很多人不买是因为看起来没熟。
-   **带叶状态**：带冠叶的菠萝卖相好（更“鲜”），但会明显增加运输成本和摆放空间；去冠叶的口感一样，但部分消费者会误以为不新鲜。如果走超市/连锁水果店，**务必保持带叶**，这是“新鲜感”的直观标识。

**结论：** 如果不确定，优先订**单果重1.3斤左右的“中果”**，搭配少量小果拉低价格带。如果后续发现大果需求起飞，再调整比例。
