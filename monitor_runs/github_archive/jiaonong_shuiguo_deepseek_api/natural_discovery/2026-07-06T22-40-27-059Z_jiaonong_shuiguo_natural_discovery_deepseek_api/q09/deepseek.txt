在水果店旺季与供应商谈合作时，为确保货源稳定并规避风险，建议提前问清以下关键问题：

1. **供货稳定性**  
   - 旺季时你们的最大日供应量是多少？  
   - 如果我们紧急加单（如提前1-2天通知），能否保证供应？  
   - 如果其他渠道抢货，我们的优先级如何？

2. **品质与溯源**  
   - 水果的产地、采摘时间、保鲜期是多久？  
   - 是否有稳定的质检流程？如农药残留检测报告？  
   - 到货后若出现品质问题（如品相、成熟度不符），如何处理？

3. **价格与结算**  
   - 旺季价格是否浮动？具体调价机制如何（如按市场价、周价还是合约价）？  
   - 是否有最低起订量或阶梯价格？  
   - 结账周期是多久？支持预付款还是分期付款？

4. **物流与损耗**  
   - 你们用哪种运输方式（冷链车/常温车）？到店时间通常为几点？  
   - 运输途中产生的损耗（如压伤、腐烂）由谁承担？赔付标准是什么？  
   - 退货或换货流程是什么？需要多久处理完成？

5. **合作机制**  
   - 是否接受长期合作合同（如签署旺季保供协议）？  
   - 是否有独家供应条款？若有，对销量有何要求？  
   - 能否提供试销期（如1-2周）以测试品质和配合度？

6. **应急方案**  
   - 如果遭遇自然灾害（如台风、霜冻）导致减产，你们如何分配有限的货源？  
   - 是否有备用供应商或渠道能紧急补货？  
   - 如果连续到货延误，是否有处罚或补偿机制？

**建议**：首次合作可先签短期合同（1-2个月），观察供应稳定性与配合度。同时，与至少1-2家其他供应商保持联系，作为应急备选。
