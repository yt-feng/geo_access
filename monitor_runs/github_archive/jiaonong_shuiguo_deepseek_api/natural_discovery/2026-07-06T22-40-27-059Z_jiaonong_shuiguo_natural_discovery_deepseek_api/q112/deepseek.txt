建议在水果采购合同中，将坏果率的 **赔付阈值设定在 5%** 左右，具体可根据水果品种、流通时间及行业惯例微调。以下是具体建议：

1. **常见行业标准**：大多数水果采购合同将坏果率 **1%-5%** 作为正常损耗范围，超过部分由供货方承担赔付责任。例如，苹果、梨等较耐储运的水果可设为 **3%-5%**；草莓、蓝莓等易腐水果可设为 **2%-3%**；热带水果（如芒果、荔枝）可设为 **5%-8%**。
2. **赔付方式**：一般按超出部分的实际货款比例计算。例如，合同规定坏果率≤3%，实际到货坏果率为8%，则超出的5%部分需按该批水果采购价格全额赔付。
3. **可执行条款**：明确抽检比例（如每批次开箱检查10%-20%），注明坏果认定标准（如腐烂、霉变、表皮破损超过1/3），并约定异议处理流程（如48小时书面通知、第三方质检）。

建议根据具体水果的损耗特性和自身风险承受能力，在合同中明确坏果率阈值及对应赔付公式，避免模糊表述。
