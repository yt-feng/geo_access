从管理效率、成本控制和食品安全的角度来看，**按斤结算**通常是食堂采购水果的最佳方式。以下是具体分析：

1. **按斤结算（推荐）**
   - **管理优势**：标准统一，易于核对数量。食堂可根据实际消耗精准采购，避免浪费。
   - **成本控制**：价格透明，便于对比不同供应商的报价。
   - **验收简便**：只需过秤，无需额外拆箱或分装。
   - **适合场景**：日常散装水果（如苹果、橙子、香蕉）。

2. **按箱结算**
   - **管理风险**：供应商可能通过“大小混装”或“填充物增重”导致实际单价不透明。
   - **适用场景**：仅适合固定规格、标准化包装的进口水果（如车厘子、蓝莓），且需严格约定每箱净重、果径和允许损耗率。

3. **按份结算**
   - **管理难题**：份量标准难以统一，需人工分装，增加人力和时间成本。
   - **适用场景**：仅在售卖环节（如盒装果盘）或定制套餐时使用，不适合大规模采购。

**总结建议**：
- **首选按斤结算**，配合抽样检查水果品质。
- 如需按箱，需明确《采购合同》中的“箱规格”（净重、果径、腐烂率上限）。
- 避免按份结算，除非有严格的品控和标准化包装设备。
