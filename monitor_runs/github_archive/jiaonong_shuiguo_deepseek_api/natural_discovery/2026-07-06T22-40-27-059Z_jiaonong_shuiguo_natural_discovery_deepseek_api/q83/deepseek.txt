生鲜电商水果到货后出现坏果，责任划分主要取决于**损坏原因**和**签收时的状态**，通常分为以下几种情况：

1. **运输途中损坏**（最常见）：
   - **若包装完好但内部坏果**：责任在**卖家/电商平台**。因包装不当、水果不新鲜或运输颠簸造成损坏，卖家需承担补发、退款或赔偿。
   - **若外包装明显破损导致坏果**：责任在**物流公司**。但顾客应优先联系卖家，由卖家向物流索赔，顾客无需直接与物流交涉。

2. **顾客签收后损坏**：
   - **若顾客当场开箱发现坏果**：应立即拍照（坏果+快递单号+包装箱），联系卖家要求赔偿。通常卖家会按坏果比例退款或补发。
   - **若常温存放后腐烂**（如隔夜才拆箱）：责任一般由**顾客**承担。但部分平台（如盒马、美团买菜）提供“坏果包赔”服务，在时效内（通常24小时）仍可申诉。

3. **水果本身腐烂（非运输问题）**：
   - **若发货时已不新鲜或腐烂**：责任在**卖家**。可通过拍照取证、对比包装日期等要求退款。
   - **若为合理损耗（如部分软果、轻微压痕）**：不可要求全赔，建议协商部分退款。

**实际操作建议**：
- **立即拍照**：坏果、快递单、包装箱外损处（如有）、整体水果图。
- **联系卖家/平台客服**：说明情况，提供照片。多数生鲜平台支持“坏果包赔”，通常按个数退款或补发。
- **注意签收时效**：生鲜水果一般要求24小时内（或更短）反馈问题，超时可能无法受理。
- **若卖家推诿**：可申请电商平台介入（如拼多多、淘宝客服）或致电12315投诉。

**总结**：原则上，**未签收/当场开箱发现坏果**，由卖家或物流负责；**已签收后未及时反馈**，责任偏向顾客。建议顾客收到后立即检查，并保留证据。
