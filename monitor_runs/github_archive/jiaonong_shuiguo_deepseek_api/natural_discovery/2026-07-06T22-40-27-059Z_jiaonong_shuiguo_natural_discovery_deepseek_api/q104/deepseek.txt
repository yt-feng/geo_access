针对单位采购水果时出现到货品质不一致的情况，建议按以下步骤进行验收和售后处理：

### 一、验收流程（尽快完成，一般在到货2小时内）
1. **提前准备**：采购合同中应明确品质标准（如无腐烂、无机械伤、果径/颜色/成熟度分级等），提前告知供应商验收时间。
2. **全员开箱抽检**：
   - 随机抽取整箱的10%-20%（至少3箱）。
   - 按“优、良、差”三级直观分类：优（符合合同品质）、良（轻微瑕疵，如小斑、擦伤）、差（腐烂、严重外伤、发霉、虫蛀）。
   - 拍照或录像记录每箱内容及分级情况。
3. **重量与数量核查**：
   - 所有箱子称毛重（与合同标注单箱重量对比，误差<2%）。
   - 清点总箱数，如有短缺现场记录。
4. **填写验收单**：列出“合格品”“次品”“不合格品”的数量及比例，双方签字确认。

### 二、售后处理方案
如果次品率超过约定标准（例如合同要求≤5%），或差品超出合理范围（如腐烂>2%），立即启动以下程序：

1. **立即联系供应商**：
   - 告知不合格品比例，发送照片/视频证据。
   - 参考处理方式（按严重程度从轻到重）：
     - **轻微**：次品≤10% → 要求补发差品（如要求次日补同等重量合格品）。
     - **中到严重**：次品≥10%，或腐烂/虫蛀≥5% → 要求换货（整批次全部退回并换新批次），或折扣（如差品部分按单价50%付款，差品免费）。
   - **特别严重**：腐烂/虫蛀≥20%，或整体无法使用 → 整批次拒收并要求全额退款及赔偿（若已影响其他物资）。

2. **协商期限**：
   - 要求供应商在24-48小时内给出处理方案，并安排后续物流（如需换货或补发）。

3. **合同追溯**：
   - 如果供应商未在约定时间内妥善处理，依据合同条款（一般含“品质不符，退货/换货/折扣”条款），暂停后续采购或启动扣款机制。

### 三、中长期优化建议
- **采购合同加强约定**：明确“不合格品”定义（如“单个果实直径<5cm，或腐烂面积>2cm²即为差品”），并注明“售后处理流程：签收后2小时内提出，48小时内补发/换货”。
- **样品比对**：下次采购前，要求供应商先寄送1-2箱样板，单位验收确认品质后再批量发货。
- **供应商管理**：同一供应商出现2次以上品质不一致，更换至备用供应商。

### 示例快速响应流程
1. 到货开箱→发现大批量腐烂→立即拍照（含箱号、日期）→电话通知供应商→要求次日换货。
2. 供应商若承诺换货→确认物流单号→保留原货至换货运达（不混合回用，否则无法索赔）。
3. 若供应商拖延→按合同扣款（如总金额的10%），并纳入未来采购黑名单。

如果你能提供具体的次品率（如“差品占比多少”），可以根据数字给出更精确的建议。
