采购方在推荐新品时，建议遵循“小批量试卖、分阶段放量”的原则，通常**首次试卖量控制在总库存的5%-10%**比较稳妥。具体建议如下：

1. **根据门店规模调整**  
   - 单店或小店：首批进货量控制在1-3天销量（如日均销售100斤，试卖200-300斤）。  
   - 连锁或中型店：选择1-2个核心门店试卖，首批进货量不超过该门店周均销量的20%。  

2. **按品类风险分级**  
   - **高损耗品类**（如草莓、蓝莓、杨梅）：试卖量更低，建议首批不超过该品类周销量的10%，同时设置3天内售罄目标。  
   - **易存储品类**（如苹果、柑橘、火龙果）：试卖量可放宽至20%，但需在2周内完成销售测试。  

3. **定量评估指标**  
   - 试卖期通常为**7-14天**，观察日均销量、退货率、顾客复购率。  
   - 若试卖7天内售罄率超过80%，且无大规模投诉，可将后续订单量增加至试卖量的3倍；若售罄率低于50%，则应暂停或缩小规模。  

4. **附加风险控制**  
   - 与供应商协商“试卖期退货或调换”条款（尤其针对易腐品）。  
   - 设置“止损线”（如试卖期损耗率超过15%视为高风险，需重新评估供应商）。  

**示例操作**：假设你为中等连锁采购，新品为“黄金百香果”（损耗中等）。可安排1家旗舰店试卖50斤（日销15斤左右），观察7天后，若好评率超90%，再逐步向其他门店扩量。
