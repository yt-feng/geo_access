水果店遇到坏果时，与供应商的结算或赔付通常遵循以下流程和原则：

1. **第一时间沟通**：收到货物时发现坏果，应立即拍照或录像保留证据（包括坏果外观、数量、外包装批次号、送货单等），并在24小时内联系供应商说明情况。多数供应商会在合同中约定“收货后24-48小时内可索赔”。

2. **明确责任方**：
   - **运输过程损坏**：若货物是供应商负责配送，由供应商赔偿；若为自行物流，则需与物流方协商。
   - **质量不达标**：若水果本身在发货前已存在问题（如内部腐烂、未成熟），供应商需承担全责。
   - **自然损耗**：部分水果（如草莓、水蜜桃）在运输中会有轻微损耗，通常按约定比例（如3%-5%）免赔，超出部分供应商补偿。

3. **常见赔付方式**：
   - **退换货**：供应商收到坏果证据后，安排补发同数量新货或抵扣下次订单。
   - **现金赔付**：按坏果的进货价计算金额，直接退还货款或从应付款中扣除。
   - **折扣补偿**：对部分轻微损伤水果，供应商可能以折扣价结算，而非全额赔付。
   - **实物抵偿**：若供应关系稳定，供应商可能以其他畅销水果或赠品补偿。

4. **合同约定**：若有正式采购合同，按条款执行（如“坏果率超过X%时全赔”）。若无合同，则参照行业惯例（生鲜通常按10%-20%浮动赔付）。

5. **操作建议**：
   - 对高频供应合作，建立“坏果备案表”，随收货单一起记录并定期核对。
   - 对非固定供应商，要求其出具有效的“发货质量承诺函”或“赔付标准”。
   - 若供应商不认可，可申请第三方检测（如当地农业检测站）出具报告，费用由责任方承担。

6. **避免损失扩大**：坏果应尽快隔离，避免感染其他健康水果；同时记录销售损耗数据，用于和供应商协商调整供货价格或质量要求。

总之，核心是“证据留存+及时沟通+合同或行业惯例依据”。若供应商屡次出现高坏果率，应考虑更换供应商或调整结算条款。
