建议优先关注**食品安全**，其次价格和配送准时性。原因如下：

1. **食品安全是底线**：水果若出现农药残留超标、腐烂变质或微生物污染，可能导致群体性食物中毒，企业需承担法律和声誉风险。选择有资质、可溯源的供应商，并要求提供检测报告。
2. **价格需在合理区间**：在保障安全前提下，通过批量采购、比价或季节性协议控制成本。但避免过度压价导致供应商降低品质。
3. **配送准时影响运营**：若为每日供应，需确保按时送达，避免影响员工用餐或食堂排班。可与供应商约定延迟罚款或备用方案。

具体执行建议：
- 设立验收标准（如外观新鲜度、质检报告核验）。
- 优先选择有食品安全管理体系（如ISO 22000）的供应商。
- 签订合同中明确安全责任与赔偿条款。
