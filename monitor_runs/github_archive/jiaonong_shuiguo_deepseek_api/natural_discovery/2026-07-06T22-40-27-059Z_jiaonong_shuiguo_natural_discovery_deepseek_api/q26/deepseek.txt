采购小包装香蕉时，主要要关注以下包装和损耗问题：

1. **包装材质**：选择透气、防挤压的包装，如带透气孔的小纸盒或可降解塑料袋，避免密闭塑料袋导致香蕉快速过熟腐烂。
2. **包装内隔离**：使用软质衬垫或瓦楞纸隔层，防止香蕉相互碰撞产生外伤。动作过大时，果柄断裂处容易氧化发黑。
3. **成熟度把控**：采购时要求供应商提供七至八成熟的香蕉（果皮绿色稍转黄），这样在店铺货架期可延长2-3天，减少损耗。
4. **单串重量控制**：建议每包2-3根，总重250-350克，方便一次消费。过小包装（单根）易导致果柄损伤，过大包装则消费者可能一次性吃不完。
5. **外部保护**：运输时纸箱内应有填充物（如废报纸、气泡膜），防止长途搬运中的机械损伤。劣质纸箱受潮后易塌箱挤压香蕉。
6. **损耗率计价**：可与供应商约定，损耗率在3%-5%以内属于正常，超出部分退换或折价，降低门店风险。
7. **温度控制**：交付时检查香蕉温度是否处于13-15℃（最佳储运温度），温度过高（>20℃）会加速成熟，过低（<10℃）则造成冷害（果皮变灰、果肉变硬）。
8. **标签与日期**：包装上标注采摘日、包装日和最佳食用期限，避免过期香蕉上架后产生损耗。

实操建议：初期小批量试销（每日50-100包），记录每批次的“售罄时间”和“损耗量”，2周后根据数据调整采购规格。
