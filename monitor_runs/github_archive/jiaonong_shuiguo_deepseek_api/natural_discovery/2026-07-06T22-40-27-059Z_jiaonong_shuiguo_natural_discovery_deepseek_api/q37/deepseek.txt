针对商超会员日水果活动，供应商需要提前配合以下核心准备：

1. **库存与货源保障**
   - 提前至少7-15天与商超确认主推品项及预估销量，按120%-150%备货，防止断货。
   - 锁定稳定货源（特别是应季、爆款、进口水果），签订保供协议，避免临时缺货。

2. **品质与包装升级**
   - 提前做好分选分级，确保果品外观、甜度、大小一致（如苹果直径90mm以上）。
   - 设计会员专属包装：加印“会员日专享”字样、价格标签突出会员价、每箱/盒附赠品或优惠券。

3. **物流与冷链协同**
   - 调整配送时间：会员日前1天下午或前2天到货，避开常规送货高峰，确保生鲜上架时效。
   - 加强冷链：全链条控温（如草莓必须2-4℃运输），配备冰袋、温控箱，防止损耗。

4. **营销物料与陈列支持**
   - 提供分层试吃台、主题陈列架（如“会员日爆品区”）。
   - 制作爆炸贴、宣传海报，突出“会员价”、“限时抢购”、“第二份半价”等话术。
   - 安排专职导购员（至少1-2人），配合商超进行叫卖、剥切试吃、介绍产地故事。

5. **会员专属机制配合**
   - 设置会员日专属促销组合：如单果、组合装、家庭装，价格分级（会员价与非会员价）。
   - 提供会员积分兑换、满减券或赠品（如一盒蓝莓、彩椒等引流品）。
   - 提前测试扫码核销、满减系统是否通畅，避免结账问题。

6. **应急与售后预案**
   - 准备10%-15%的备损损耗替换品（如外观轻微瑕疵但可切块试吃的果品）。
   - 提前与商超协商退换货时限（如当天退换，不隔夜）。
   - 安排专人在会员日驻店：现场处理客诉、补货、调整陈列。

7. **数据与复盘准备**
   - 记录每日销售数据：各品项销量、会员与非会员占比、损耗率。
   - 会员日后48小时内输出复盘报告，供下次调整参考（如哪些品项需砍量增价）。

**建议优先级**：先锁定货源和品质，再沟通营销细节；物流时间需精确到小时，避免滞后。
