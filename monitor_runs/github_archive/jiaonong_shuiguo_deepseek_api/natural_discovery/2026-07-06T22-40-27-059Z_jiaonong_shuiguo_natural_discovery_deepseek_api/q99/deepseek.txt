针对公司下午茶水果批量配送，建议按以下步骤选择供应商，并重点关注配送效率、品质稳定性和性价比：

**1. 明确核心需求**
- **每日/每周配送量**（如：30人份、每次5-8种水果）
- **配送时间**（如：周一、周四上午10点前固定送达）
- **品质要求**（如：不可有腐烂、需洗净切盒或整果）
- **预算范围**（如：人均15-25元/次）

**2. 寻找备选供应商**
- **本地大型超市/生鲜平台**：如永辉、盒马、山姆等，支持企业团购，有稳定供应链和冷藏配送。
- **专业水果批发商**：联系本地水果批发市场（如北京新发地、广州江南市场）的档口，可定制装箱。
- **线上B2B平台**：美团快驴、美菜网等，专注企业采购，支持次日达。
- **社区团购渠道**：如当地有规模的社区团购团长，可协商单次大批量配送。

**3. 关键考察项**
- **样品试吃**：索要一份标准配送清单，检查水果新鲜度、成熟度和大小一致性。
- **配送保障**：
   - 是否有冷链车/保温箱（夏季易腐坏水果如草莓、蓝莓需特别注意）。
   - 是否提供“次日达”或“准时达”服务，延迟配送如何处理。
- **售后政策**：不良品（如压坏、变质）是否包退换、比例上限（如5%以内免费补发）。
- **价格透明性**：要求按品种、重量明确报价（如：进口红提18元/斤，国产巨峰12元/斤），避免混合计价。

**4. 拟定试运行订单**
- 首单建议选择少量（如50份以内），观察配送准时性、包装防护（如有无防撞泡沫）及实际损耗。
- 试运行后对比2-3家供应商的性价比、沟通效率。

**5. 长期合作建议**
- 签订框架协议：明确最低起送量、价格锁定期（如每季度更新）、违约赔偿条款。
- 要求提供每日/每周《水果品质报告》或到货双方签收确认单。
- 可推荐一种“拼搭方案”：每周固定品种（如苹果、香蕉）+ 当季时令水果（如夏季西瓜、荔枝），降低采购谈判成本。

**注意事项**：避免依赖单一供应商，保留1-2家备选；若公司人数超过100人、日均配送量大，可优先考虑与批发市场档口或农场直接对接。
