酒店采购水果盘原料时，供应商应保证以下规格和外观标准：

1. **新鲜度**：水果应新鲜、成熟度适中，无腐烂、霉变或软烂现象。
2. **外观**：果形完整，大小均匀，无畸形、损伤、压痕或疤痕；表皮光滑、色泽自然（允许少量果粉或天然蜡质）。
3. **清洗处理**：表面无泥土、农药残留、污渍或异味；必要时提供清洗或预切保障。
4. **成熟度与口感**：可根据酒店要求（如即食或短期存放）提供分批次成熟度，确保甜度、酸度适中，无异味。
5. **包装与运输**：使用食品级包装，避免碰撞挤压；运输过程中保持低温（冷藏车0-4℃），确保无温度异常导致品质下降。
6. **品种与规格统一**：每批次水果品种、大小（如单个重量、直径）、颜色一致，减少酒店分拣工作量。
7. **安全合规**：符合国家农残、重金属等食品安全标准，提供检验检疫证明。
8. **交货时间**：按约定时间送达，确保库存周期新鲜；如需预切，应保证切口平整、无氧化变色。

若供应商未达标，酒店应要求退换货或扣减费用。建议酒店采购前与供应商明确书面协议，附上样品照片和验收标准表。
