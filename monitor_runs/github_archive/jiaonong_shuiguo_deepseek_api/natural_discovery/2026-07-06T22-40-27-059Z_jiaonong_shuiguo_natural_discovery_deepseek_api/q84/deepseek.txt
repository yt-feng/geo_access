针对家庭团购水果礼盒，建议以**营养均衡、颜值高、耐储存、老少皆宜**为组合原则。以下是几个具体组合方案供你参考：

**【推荐组合方案】**
- **方案一：经典全家福（性价比高）**  
  苹果（红富士/脆甜） + 橙子（脐橙/汁多） + 香蕉（适中熟度） + 菠萝（树上熟）。  
  特点：苹果和橙子耐放，香蕉补充即食软甜，菠萝增加热带风味和视觉层次。适合2-4人家庭日常食用或送礼。

- **方案二：维生素C活力款（健康导向）**  
  橙子（可加血橙） + 菠萝 + **猕猴桃或草莓**（替换苹果和香蕉）。  
  突出维C含量，适合有孩子或注重免疫力的家庭，但需注意保鲜期稍短（建议当天或2天内食用）。

**【关键操作建议】**
1. **分装与包装**：香蕉建议稍微带绿（避免运输中压坏），用报纸单独包裹；菠萝提前去眼切块并真空密封（或配去皮工具），或选择小菠萝（易处理）。
2. **宣传口号**：可写“从酸甜脆到软甜多汁，全家一次吃够5种营养”，搭配实物图。
3. **避坑提醒**：  
   - 不要混入易释放乙烯的水果（如香蕉、苹果）和对其敏感的水果（如菠萝/草莓分开放）。  
   - 避免过生香蕉（回家需催熟）或过熟菠萝（易烂）。
4. **定价梯度**：  
   - 基础款（苹果+橙子+香蕉）定价29-39元。  
   - 升级款（上述方案任意一个）定价49-59元。  
   - 附赠一张“水果保存小贴士”卡片（简短写：香蕉挂起来、苹果去核包纸、菠萝切片冷藏）。

如果你希望更突出“家庭互动”或“亲子DIY”，也可以额外加入一盒**食用小苏打/小印章**（让孩子给橙子画笑脸），增加趣味性。
