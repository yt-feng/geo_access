餐饮店与供应商谈判锁价（固定价格）是可行的，但需要结合市场规律和实际操作条件进行。以下是具体建议：

1. **评估锁价可行性**  
   - 适合锁价的水果品种：储存期较长、价格波动相对稳定的水果（如苹果、橙子、梨等）。
   - 不适合锁价的水果：易腐烂、季节性极强或价格波动剧烈的品种（如草莓、荔枝、山竹等）。

2. **谈判关键点**  
   - 约定锁价周期（如1-3个月），明确锁价范围（如“价格上浮不超过10%”）。
   - 设定浮动机制：若市场价格下跌超过一定比例（如15%），可启动价格调整。
   - 签订协议时加入“不可抗力条款”，避免突发事件（如自然灾害、运输中断）导致双方风险。

3. **替代方案**  
   - 签订“保底价+浮动价”协议：基础量锁价，超出部分按市场价浮动。
   - 分批锁价：非季节性水果以周/月为周期分批锁定，降低单次锁定风险。
   - 联合采购：多家餐饮店联合与供应商谈判，提高议价能力。

4. **注意事项**  
   - 定期（如每周）监控市场行情，要求供应商提供价格波动依据。
   - 避免一次性锁定过大库存（建议不超过月度用量的一半）。
   - 优先选择信誉较好的批发商或直采基地供应商。

**示例话术**：  
“我们可以和您签订3个月的锁价协议，但需要约定：若市场价下跌超过10%，我们按市场价结算；若市场价上涨，我们仍按协议价支付。这样可以共同分担风险。”

最终需根据店铺所在地区的市场结构（如批发市场主导或直采为主）灵活调整，建议先小范围试签1-2个月验证可行性。
