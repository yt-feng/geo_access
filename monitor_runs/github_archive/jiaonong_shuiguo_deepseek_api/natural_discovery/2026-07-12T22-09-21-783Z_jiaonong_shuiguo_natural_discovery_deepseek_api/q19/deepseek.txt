如果是超市卖菠萝，**优先级建议：甜度 > 外观 > 大小 > 切果损耗**。

具体分析和操作建议如下：

1.  **甜度（最关键）**：这是顾客重复购买的核心。超市可以：
    -   在进货时要求供应商提供甜度检测数据（如糖度12以上）。
    -   **做“试吃”**：这是最直接的销售方式，让顾客尝到甜味，转化率会大幅提升。
    -   在菠萝旁张贴提示：“已检测，甜度XX”，建立信任。

2.  **外观（影响购买欲）**：成熟的菠萝应呈现金黄色，果眼饱满、无霉斑。
    -   如果外表带青，未熟透，顾客不敢买。如果外表有磕碰或开始腐烂，更会直接放弃。
    -   操作：每天上架前，挑出软烂、发霉的菠萝。

3.  **大小（影响客单价和损耗）**：
    -   大菠萝单价高，容易让顾客犹豫。建议**大小分开定价**，或者只进中等个头（如1.2-1.5斤/个）的，适合多数家庭。
    -   不必过分追求“越大越好”，容易造成剩下的大果卖不出去而损耗。

4.  **切果损耗（最次要）**：
    -   **切果服务是引流手段，不是增加利润的手段**。千万不要为了减少损耗而削皮很薄，导致切出来的果肉变色、有黑眼，反而增加退货和差评。
    -   策略：用标准化削皮刀法，保证果肉完整；削好的切果**定价稍高于**带皮果，让顾客自己决定（很多顾客宁愿多花几块钱享受服务）。

**实操建议：**
-   进一批“熟度适中”的菠萝，分成两堆：
    -   一堆**带皮**摆放，重点展示外观和甜度试吃。
    -   一堆**去皮切块**放在冷藏柜，方便即买即食。
-   **每日复盘**：早上多摆整果，下午如果剩的整果较多，傍晚再大量切成果切盒或做特价促销。这样能平衡外观展示和最终损耗。
