从损耗角度来看，**直接采购成品果丁**的损耗通常更低。

原因如下：
1. **原料损耗**：鲜菠萝去皮、去眼、去芯后，可食用部分的出成率大约在50%-60%（依据菠萝品种和成熟度可能有差异）。这意味着购入1公斤鲜菠萝，切配后实际可用的果肉大概只有500-600克，其余部分（外皮、果眼、硬芯）均为损耗。
2. **加工损耗**：鲜果切配过程中，操作不当（如削皮过厚、切丁大小不均、丢弃边角料）会进一步增加损耗。此外，切开后的果肉接触空气后易氧化变色、失水变干，若未及时使用或保存不当，可能产生二次浪费。
3. **成品果丁优势**：直接采购的成品果丁通常经过标准化处理，果肉利用率高（供应商已剔除不可食部分），且包装密封、配送及时，减少了门店切配环节中的人为损耗和储存损耗。

**但需注意**：
- 成品果丁的单位价格通常高于鲜果（因为包含了加工和包装成本），如果采购量较小或即使用不完，整体成本可能更高。
- 鲜果切配能更好地控制果丁大小、成熟度和新鲜度，对于强调品质和新鲜感的甜品店，鲜切可能更合适。

**建议**：
- 如果甜品店日用量稳定且较大（比如每天需要加工10公斤以上菠萝），直接买鲜果自己切可能综合成本更低（单价优势可覆盖损耗）。此时需优化切配流程：使用专用工具（如菠萝去眼器）、安排专人负责、控制切丁后存放时间（建议冷藏、密封、2小时内用完）。
- 如果用量较小、不稳定，或人力紧张、操作经验不足，建议优先采购成品果丁，以减少损耗和人工成本。

**总结**：从损耗率直接对比，成品果丁更低；但综合成本需结合采购量、人工成本、品质要求综合评估。
