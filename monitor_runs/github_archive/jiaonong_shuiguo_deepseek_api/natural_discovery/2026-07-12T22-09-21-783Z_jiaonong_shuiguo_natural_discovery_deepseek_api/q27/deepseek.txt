判断苹果和橙子的大小规格是否适合门店销售，主要看以下几点：

1. **参考标准规格**  
   超市通常有统一的规格要求：  
   - 苹果：常见为 **70#、75#、80#、85#**（指果实横径，单位毫米）。门店销售以 **75-80#** 为主流，偏大（85#以上）可做礼品果，偏小（70#以下）可能适合特价或包装。  
   - 橙子：常见为 **60#、65#、70#、75#**。门店销售以 **65-70#** 为主，过小（60#以下）口感可能不佳，过大（75#以上）容易有皮厚问题。

2. **看均匀度**  
   同一批次中，果实大小差异不应超过 **5毫米**。混入过大或过小的果子，会被顾客挑拣，增加损耗。

3. **试销售反馈**  
   如果没有历史数据，可以分两档采购（如苹果：75# 和 85# 各一小批），试卖1-2天，观察哪个规格的**售罄率更高、客单价更理想**。

4. **比对库存周转**  
   如果门店面积小、客流快，优先选偏小规格（苹果75#、橙子65#），方便散称、减少损耗；如果主打高端或礼品场景，选偏大规格。

5. **手工快速判断**  
   采购时会带一个**果径圈**（用硬纸板挖圆孔）。如果门店暂无果径圈，可目测对比：  
   - 苹果：比成年**女性拳头小一圈（约70#）**，与男性拳头接近（约80#）。  
   - 橙子：与**网球**近似（约65#），比**垒球**稍小（约70#）。

**最终选择标准**：优先选门店常销的规格，并且果实横径差异小、果形端正、无畸形。不确定时，参考同一城市同类超市（如社区型或精品型）的主流规格。
