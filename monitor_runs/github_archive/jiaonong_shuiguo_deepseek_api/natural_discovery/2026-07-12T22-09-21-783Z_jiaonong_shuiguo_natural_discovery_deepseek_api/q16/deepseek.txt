建议优先选择**稳定供应商**，理由如下：

1. **货源稳定**：本地批发可能受季节、天气影响大，而稳定供应商有长期合作协议和备货计划，能保证香蕉和苹果的持续供应，避免断货。
2. **品质统一**：稳定供应商通常有标准化分选和储存流程，香蕉的成熟度、苹果的规格更一致，减少顾客投诉。
3. **成本可控**：短期看本地批发可能单价低，但长期合作供应商往往能提供批量折扣、固定运费和账期支持，整体成本更优。
4. **售后保障**：稳定供应商会提供退换货服务（如香蕉碰伤、苹果腐烂），本地批发商通常“出门不认”。

**操作建议**：
- 先与当地2-3家稳定供应商谈判，要求提供样品、报价单及供货协议（明确最低起订量、配送频率、退换条款）。
- 对比供应商与批发市场的价格差异（计算运输、损耗、人工等隐性成本）。
- 初期可少量从本地批发补货，观察销售周期后，再签订长期合同。

如果社区超市销量波动大（如节日暴增），可保留本地批发作为临时补充，但主力依赖稳定供应商。
