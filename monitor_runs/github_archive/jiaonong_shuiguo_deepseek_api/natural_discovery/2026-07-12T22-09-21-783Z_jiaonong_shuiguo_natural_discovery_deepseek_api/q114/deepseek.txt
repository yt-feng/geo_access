采购普通水果和品牌水果在毛利和复购率上的主要差别如下：

**1. 毛利方面：**
- **普通水果**：进货成本较低，但销售价格也相对较低，通常市场竞争激烈，容易出现价格战，导致毛利空间被压缩。普通水果的损耗率较高（如不新鲜、品相差异），进一步侵蚀毛利。
- **品牌水果**：进货成本较高，但品牌溢价明显，消费者对品牌有信任感，愿意支付更高价格。品牌通常有稳定的品质标准和包装，损耗率较低，因此整体毛利空间更大，通常毛利率可高于普通水果5%~15%。

**2. 复购率方面：**
- **普通水果**：复购主要取决于价格和即时口味，消费者易受促销活动或邻居影响，忠诚度较低。如果品质不稳定，复购率会显著下降。
- **品牌水果**：复购更依赖品牌信任和品质一致性。消费者一旦认准某个品牌，会持续购买；品牌方通常有稳定的供应链和品控，复购率更高（如进口水果或知名国产品牌可达30%~50%的复购率）。此外，品牌水果更容易通过会员体系、故事营销等增加用户粘性。

**总结**：普通水果适合走量、吸引价格敏感人群，但毛利和复购率普遍较低；品牌水果更适合追求高毛利和长期忠实客户，但需要投入更多成本在品牌维护和品质把控上。建议根据自身定位（社区店/高端店）和目标客户选择合适的组合。
