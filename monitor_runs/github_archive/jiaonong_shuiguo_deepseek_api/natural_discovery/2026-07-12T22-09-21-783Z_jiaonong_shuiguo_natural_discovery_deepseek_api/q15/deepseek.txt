遇到坏果需要和供应商结算或赔付时，通常建议遵循以下流程和原则：

1. **保留证据**：第一时间拍照或录像记录坏果的具体情况（如腐烂程度、数量、包装箱号等），并保留送货单或采购记录。
2. **确认责任归属**：判断坏果原因：
   - 运输途中损坏（如挤压、冻伤）→ 可能由物流方负责；
   - 发货时已变质（内部腐烂、霉变）→ 由供应商负责；
   - 储存不当（温度、湿度问题）→ 可能需自行承担。
3. **联系供应商**：在发现坏果后（通常建议24小时内）联系供应商，说明情况并提供证据。多数正规供应商有“到货损耗”（如3%-5%）和“坏果赔付”条款。
4. **遵循合同或惯例**：
   - 若有书面合同，按合同中的损耗标准或赔付比例处理（如全额赔付、折价补偿）；
   - 若无合同，常见做法是：坏果率在约定范围内（如5%）由买家承担，超出部分供应商按实际数量赔付或补发。
5. **协商解决**：根据坏果比例可提出：
   - 全额退款或换货（坏果较多时）；
   - 按坏果数量打折赔付（如按进货价*坏果斤数）；
   - 或从下次货款中抵扣。
6. **避免争议**：如果供应商不认可，可协商双方共同认定的检测机构（如市场质检部门）出具鉴定报告。
7. **预防为主**：在后续采购中，可要求供应商在发货前提供批次质检证明，或约定明确运输和仓储条件。

**示例沟通话术**：
“张经理，今天到的XX水果有XX斤烂果（占总数约X%），照片已发您。按我们之前的口头约定，超过5%的部分应该由你们赔付，这周能否补发或抵扣下次货款？”

如果供应商不配合，可考虑通过市场管理方或行业组织调解。
