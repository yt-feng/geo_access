招标评分标准宜围绕**品质稳定、供应能力、价格竞争力、服务与合规**四大维度设计。以下是一个可参考的评分框架（总分100分），建议根据商超定位和预算灵活调整权重：

### 1. 品质指标（建议占比30-35分）
- **产品合格率（15分）**：承诺批次合格率≥98%得满分，每降1%扣2分（需提供第三方检测报告或自行送检计划）。
- **口感/外观一致性（10分）**：该供应商的果实甜度、大小、色泽等关键指标的方差系数（越小越优），可要求提供近3个月供货记录或现场抽样检测。
- **标准化包装（5分）**：散装、预包装、贴标、保鲜措施是否适配商超货架销售。

### 2. 供应能力（建议占比25-30分）
- **日均/周供应量（10分）**：需明确供应商能按商超各门店峰值销量提供多少吨水果，并留出20%余量。
- **产地与物流（8分）**：是否靠近商超所在区域？有无自有冷库/冷链车？距离≤200km且物流时效≤24h可加分。
- **全年稳定度（7分）**：反季节水果是否有储备基地或与多个产区签约？能抵消气候风险的可加分。

### 3. 价格与结算（建议占比20-25分）
- **报价合理性（10分）**：以**当季市场批发均价**为基准，低于基准价10%以内得满分，波动过大或显著低于行业平均需提供成本证明（防恶性比价）。
- **调价机制（5分）**：当批发价波动超过15%时，供应商愿按固定比例（如5:5）或阶梯化调整，而非仅单方涨价。
- **账期优惠（5分）**：给予45-60天账期得基础分，账期更长或主动提供现金折扣（如“2/10，Net 30”）可加分。

### 4. 服务与合规（建议占比15-20分）
- **售后响应（5分）**：因包装破损、变质等问题，承诺24小时内无条件换货或全额赔付的可得满分。
- **资质证明（5分）**：需提供营业执照、食品经营许可证、绿色/有机认证（如有）、农药残留检测报告（农残≤国标GB 2763的50%更优）。
- **可持续性（3-5分）**：是否通过GLOBALG.A.P.、FSA等认证？有不使用违禁保鲜剂书面承诺的可加分。

### 5. 附加项（建议5-10分，可选）
- **合作经验（3分）**：过去3年有至少2家同体量商超合作案例及好评证明。
- **创新服务（2分）**：提供水果陈列方案、促销人员培训、节气营销活动支持等。

### 执行建议：
1. **硬性门槛**：未通过农残检测或曾有重大违约记录的供应商直接淘汰。
2. **现场考察**：对得分前3名的供应商，建议到产地和分拣中心实地检查。
3. **分档评分**：例如品质指标的“合格率”分为三档——≥98%得满分、95%-97%得80%权重分、＜95%得50%分或淘汰。
4. **历史数据核查**：要求供应商提交过去12个月的供货记录，计算实际合格率和按时交货率，而非仅口头承诺。

如果商超主打高端水果，可提高品质和可持续性的权重（各提升5分），适当降低价格权重；如果是平价社区店，可优先保证价格稳定、供应充足，增加价格和账期权重。
