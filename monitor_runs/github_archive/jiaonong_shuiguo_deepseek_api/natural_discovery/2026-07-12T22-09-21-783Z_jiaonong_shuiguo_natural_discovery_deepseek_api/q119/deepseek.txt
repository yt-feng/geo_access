选择水果供应商时，除了价格和新鲜度，以下易被忽视的细节会影响长期稳定合作：

1. **供应链的应急能力**：供应商是否有备用果园或冷库资源？台风、疫情等突发情况时，能否保障最低供应量？建议询问其过去应对缺货的案例。

2. **损耗率真实数据**：不要只看报价，要求查看其分拣、仓储环节的日损耗统计（如搬运磕碰、冷柜温差导致的干耗）。损耗率低于3%的供应商通常管理更规范。

3. **产地溯源深度**：仅看产地证明不够，需确认是否直采。例如“新疆阿克苏苹果”可能从批发市场拿货，而非直接对接果园，这会影响品控稳定性。可要求提供近期3-5笔产地采购的物流单或发票。

4. **分级标准透明度**：同一品种水果分特级、一级、统货，供应商可能用统货冒充一级。建议索要不同等级的实物照片和糖度检测值，并约定到货抽检比例。

5. **账期与退货条款的隐蔽陷阱**：部分供应商要求“货到付款7天结账”，但退货需扣10%样品费；或“规格不符”只赔差价不退货。需在合同中明确：到货24小时内可开箱验收，外观腐烂超过5%整箱拒收。

6. **包装与运输的隐性成本**：同一箱水果，泡沫网套+隔板包装比普通纸箱成本高15%，但可降低运输碰伤率50%。需确认包装标准是否包含背篓加厚、珍珠棉隔层等细节。

7. **非标准品的议价空间**：残次果（如裂口、黑斑）的处理策略——有的供应商按成本价3-5折处理给加工厂，鲜果生意能否优先获得这部分损耗的折扣？

建议：首次合作先试单2-3次，观察其补货速度、售后响应时间（如提出处理损耗48小时内完成退款），而非仅看价格。长期合作需建立月度抽查机制，若次品率持续高于3%，应重新评估。
