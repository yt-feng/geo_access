建议**根据店铺定位和顾客需求灵活组合销售**，以下是具体分析：

**1. 三种方式的优劣对比**
- **整果销售**：适合追求新鲜、喜欢DIY的顾客；需提供削皮服务（额外收费或免费）避免售后纠纷；库存损耗较低。
- **去皮包装**：适合追求便捷的顾客（如白领、游客）；需注意现切现卖以避免氧化变色，可搭配小份量包装；售价可适当上浮（人工成本+损耗）。
- **顾客自选+代削**：平衡体验与便利；增加互动感，但需要规范削皮流程（如收费、工具准备、卫生标准）。可能增加现场等候时间。

**2. 推荐策略**
- **主推整果**，同时提供**免费削皮服务**（可配套收费“去芯切片”等增值服务），并备少量去皮盒装（保质期标注2小时内，降价处理临期品）。
- 在高峰时段（如傍晚）削好一批成品菠萝，减少顾客等待和员工压力。
- **培训员工**：指导顾客如何挑选甜菠萝（闻香气、看颜色、按硬度），减少退货。

**3. 注意事项**
- 去皮后的菠萝容易滋生细菌，务必冷藏（2-6℃），不可常温过夜。
- 标注去皮时间，明确“建议2小时内食用”，避免顾客投诉变质。
- 根据当地消费习惯和竞争对手调整：若周围居民多为家庭主妇，可侧重整果；若靠近写字楼，加大去皮产品比例。
