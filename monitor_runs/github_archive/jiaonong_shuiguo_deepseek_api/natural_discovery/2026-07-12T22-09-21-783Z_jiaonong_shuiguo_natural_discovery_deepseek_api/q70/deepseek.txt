在合同中明确坏果责任的条款，建议从以下几个方面入手，以降低批发商的损耗风险：

1. **约定验收期限与方式**  
   明确到货后需在**X小时内**（如24小时）完成验收，并拍照或视频记录坏果情况，作为责任认定的依据。

2. **区分运输与仓储责任**  
   - **运输途中损耗**：由承运方或卖方承担，约定损耗率上限（如不超过1%），超出部分由卖方赔偿。  
   - **收货后保管损耗**：由买方承担，需明确仓储条件（如温度、湿度要求）。

3. **设定“坏果率”的量化标准**  
   例如：  
   - “坏果率超过3%时，超出部分按批发价全额赔偿。”  
   - “若坏果率≥10%，买方有权拒收整批货，或要求卖方重新发货并承担运费。”

4. **明确赔偿/补救方式**  
   - **现金赔付**：按实际坏果数量乘以单价计算。  
   - **补货**：卖方在X天内补发等量好果，运费由卖方承担。  
   - **折扣**：按坏果比例减免总货款。

5. **证据保留与争议处理**  
   - 要求双方在发货前共同封存样品，或留存第三方（如物流方）签字确认的货物状态记录。  
   - 争议时，以双方认可的第三方检测机构（如海关商检）报告为准。

**示例条款**：
> 货物送达后24小时内，买方应进行开箱验收并拍摄含时间戳的坏果照片。若坏果率超过3%，卖方应在48小时内按超出部分的批发价赔偿；若坏果率超过10%，买方有权拒收并解除合同，卖方承担全部退货运费及货款退款。验收后发生的仓储损耗由买方负责。

**关键提醒**：建议在合同中加入“不可抗力”除外条款（如极端天气导致的运输延迟），同时保留发货前的农药残留、新鲜度等检测报告作为辅助证据。
