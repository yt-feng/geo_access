要在旺季避免断货，和供应商谈合作时，建议提前确认以下关键问题：

1. **库存与供应能力**：询问供应商在旺季的预估库存量、日供货上限，以及能否应对突发性补货需求（比如加急订单或临时增量）。
2. **订货与补货周期**：明确提前几天需要下订单、从下单到到货需要多久，以及是否有紧急补货通道（比如加急订单能否在24小时内发货）。
3. **产能与来源**：了解果品的主要产地（是否受气候/运输影响）、采摘和加工能力，以及是否有备用供应商以防主渠道出问题。
4. **价格与协议保障**：询问旺季价格是否会浮动（比如按单量阶梯定价），是否有锁价协议（如提前预付订金锁定已谈好的价格），以及物流/损耗成本由谁承担。
5. **品质与责任条款**：确认到货后的验收标准（如果重量/大小/成熟度不达标如何处理），以及因供应商原因导致断货时，是否有违约补偿（比如部分退款或优先补货权）。
6. **信息同步机制**：最好是供应商能定期提供库存预警或到货时间表，并约定双方的信息沟通频率（如每日一次进度通报）。

建议在正式合作前先试订单小批量测试，并把这些条款落实到书面合同中。
