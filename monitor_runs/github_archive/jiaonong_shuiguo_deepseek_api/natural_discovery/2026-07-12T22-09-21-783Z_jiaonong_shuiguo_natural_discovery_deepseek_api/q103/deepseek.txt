建议优先采用**按斤结算**，原因如下：

- **成本透明**：水果常见按重量计价，能精准反映实际采购量，避免“按箱”可能出现的大小果差异或包装过重问题。
- **库存可控**：便于根据实际消耗（如每餐用量）灵活调整采购量，减少浪费。
- **核算简单**：出库、盘点时重量数据容易标准化，避免“按份”带来的规格不一致（如份量大小不同）导致的财务混乱。

**操作建议**：  
1. 与供应商约定按斤入账，收货时现场称重、双方签字确认。  
2. 如果确实需要按箱采购（如批发），务必事先约定单箱净重范围，并随机开箱抽检。  
3. 避免完全“按份”结算，除非食堂的每份标准完全固定且能统一供应。
