水果采购的锁价时间通常取决于以下几个因素，一般常见情况如下：

1. **市场供需稳定性**：如果该水果正值产季且供应充足，锁价时间可能较短，通常为**1-2周**；若供需紧张（如反季节或产地受灾），供应商可能只愿锁价**3-7天**。

2. **价格波动风险**：对于价格波动较大的水果（如荔枝、榴莲等），供应商通常只接受**1周内**的锁价；而对于价格相对平稳的品类（如苹果、梨等），锁价可能长达**1个月**。

3. **采购量和合作周期**：长期大批量采购的客户，供应商可能同意**1-3个月**的锁价，但需附加条款（如价格高于市场时不再调整）。单次小批量采购一般仅锁**3-5天**。

4. **合同约定方式**：部分供应商会提供“**阶梯式锁价**”，即设定不同时间段的价格区间（如第1-7天固定价，第8-15天浮动价）。大型连锁商超或协议合作中，锁价期可达**整个产季**（3-6个月），但需提前预付部分款项或承担行情下跌风险。

**操作建议**：
- 优先与供应商明确**锁价条件**（如最低采购量、市场价涨跌超过多少时有权调整）。
- 对于季节性强、价格波动大的水果，建议**分批签订**（每次锁7-10天），避免一次性锁定过长导致亏损。
- 可协商“**双向锁价**”（即价格涨跌超出5%时双方重新商议），平衡风险。

锁价时间本质上反映双方的博弈结果，若你作为采购方体量较大或合作稳定，可尝试争取更长期限（如30天左右）。建议结合具体水果品类的历史价格波动周期和当前市场行情，与供应商具体沟通。
