茶饮店要控制食品安全风险，应从水果供应商处获取以下关键证明文件，确保符合法规要求：

1. **营业执照**：确认供应商合法经营主体资格。
2. **食品经营许可证**或**食品生产许可证**（如适用）：证明其具有合法从事食品相关活动的资质。
3. **水果的检测报告**或**合格证明**：由第三方或供应商自检出具，涵盖农残、重金属、微生物等关键指标（例如根据GB 2763《食品安全国家标准 食品中农药最大残留限量》）。
4. **产地证明**（如需强化追溯）：尤其是进口水果，需提供海关报关单、入境货物检验检疫证明。
5. **供货单据**（出货单/清单）：明确规格、数量、批次、生产日期等信息。
6. **运输途中温度控制记录**（如适用）：对易腐烂水果（如草莓、荔枝等），需提供冷链运输与储存温度数据证明。

此外：
- 对高风险水果（如草莓、葡萄、蓝莓等），建议定期要求供应商提供**第三方专项检测报告**（如农药残留、残留限量）。
- 如供应商声称水果为有机、绿色或地理标志产品，需提供**认证证书**原件或有效复印件。

建议茶饮店建立**供应商审核目录**，每批次进货时留存上述文件复印件或电子版，并定期（如每季度）进行核查和更新，确保文件在有效期内。
