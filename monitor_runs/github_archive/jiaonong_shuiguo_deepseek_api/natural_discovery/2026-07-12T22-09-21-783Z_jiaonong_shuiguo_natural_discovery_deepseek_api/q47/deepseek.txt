采购香蕉用于茶饮店时，控制熟度是关键，直接影响饮品/甜品口感、甜度和外观。以下是针对性建议：

### 1. 明确用途，匹配熟度
- **奶昔、冰沙类（需浓郁甜味和顺滑口感）**：选**全熟**香蕉（皮色金黄密布小黑斑，手感微软、有弹性）。此时糖分高、淀粉转化充分，能自然增甜，避免额外加糖。
- **切片装饰、果茶、搅打泥**：选**八分熟**（皮色金黄，两端稍绿，果肉微硬）。易切出规整薄片，不易氧化变黑，且保留一定酸度平衡甜腻。
- **烘焙甜品（如香蕉蛋糕、布丁）**：选**过熟**（皮大面积黑斑甚至全黑，果肉软糯）。成熟度最高，甜度巅峰且果香浓，但需快速使用或冷冻。

### 2. 采购验收标准
- **视觉**：避免青涩（皮全绿）或黑斑过多（可能内部开始腐烂）。优先表面光洁、无机械损伤的香蕉。
- **触觉**：轻捏柄尾和中部，按需要软硬度。注意：若整串香蕉极软，可能储存过久或冷藏受损。
- **气味**：八分熟应有淡淡果香；全熟香气明显；青涩或无味则需常温催熟2-3天。

### 3. 库存与周转策略
- **分批次采购**：每次进两种熟度（如八分熟和全熟），前者备用，后者先加工。每天检查香蕉状态，优先使用易过熟的。
- **催熟/延缓成熟**：
  - 加速：放入纸袋，加一个苹果或未熟梨，室温25℃左右放1-2天。
  - 延缓：冷藏（低温会抑制果皮变黑但果肉可能变软），仅适合两天内要用完的香蕉；或悬挂通风处，避免堆叠压塌。
- **处理多余香蕉**：过熟立刻去皮冷冻（密封袋，可保存1个月），用于奶昔或烘焙，避免浪费。

### 4. 常见问题规避
- **黑皮但果肉正常**：通常为冷藏后表皮氧化，内部可能仍够甜，但切出后易氧化发褐。可用于搅打泥，不宜装饰。
- **涩味**：青香蕉含单宁高，可直接微波50-60秒（千万不能过熟），或油炸做脆片，不适合做饮品。

### 实操建议
制定每日用蕉量，例如“周一进半熟，周三再用全熟”，根据本周饮品销量微调。每天早班检查库存，先加工熟度最高的。若遇大批量，可用香蕉杯测（打泥试糖度），确定是否需要调整当日配方。
