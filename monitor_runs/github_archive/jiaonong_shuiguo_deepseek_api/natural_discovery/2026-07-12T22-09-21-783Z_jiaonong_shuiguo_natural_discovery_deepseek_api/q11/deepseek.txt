会的。大小不一的香蕉确实可能对销售产生影响，主要体现在以下方面：

1. **顾客体验不一致**：顾客购买时可能期望同一箱香蕉大小、成熟度相对均匀。大小差异明显时，容易让顾客担心“小的太生或不好吃，大的熟过头”，降低整体满意度。

2. **定价和分拣困难**：零售时需要额外分拣、定价。大小不一可能导致部分香蕉被挑剩，增加损耗。

3. **包装和陈列效果**：整齐大小更美观，大小不一可能使堆头不均匀，影响视觉吸引力。

**建议做法**：
- 如果批次中大小差异在可接受范围内（如仅普通自然差异），可以标明按整箱或混装出售，并适当折扣。
- 如果差异过大（如明显不同成熟度、畸形的混在一起），最好先分拣，按大小/品相分级销售，劣势部分可做特价或加工。
- 进货时与供应商沟通，提出对香蕉大小、成熟度的标准要求，避免采购大小混杂的批次。

简单来说，统一大小能让顾客更放心选购，价格和损耗也更容易管理。
