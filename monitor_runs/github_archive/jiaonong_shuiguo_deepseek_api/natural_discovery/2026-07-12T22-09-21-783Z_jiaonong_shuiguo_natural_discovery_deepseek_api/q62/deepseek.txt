判断香蕉熟度和坏果风险，按以下步骤操作：

1. **观察颜色**：  
   - 青皮（7成熟以下）：耐存放，适合长途运输或慢销。  
   - 黄绿相间（7-8成熟）：2-3天内会全黄，适合近期销售。  
   - 全黄（9成熟）：1-2天内会变软，需快速分销。  
   - 出现黑斑或褐色斑点：熟度偏高，可能已有软烂，需逐根检查。

2. **手感测试**：  
   - 取一根香蕉轻捏中部：  
     - 硬实且弹性好：未熟，风险低。  
     - 略有弹性、不粘手：适熟。  
     - 软糯、表面粘手：过熟，可能内部腐烂，批次若有5%以上此类即风险高。

3. **闻气味**：  
   - 浓烈甜香：正常成熟。  
   - 闻到发酵酸味或酒精味：已有内部腐坏，易扩散，应拒收或隔离。

4. **检查果柄和切口**：  
   - 果柄干燥、无霉斑：正常。  
   - 果柄潮湿、有灰黑色霉层：易引发真菌感染（如炭疽病），需抽样切开看果肉是否变色。

5. **抽样剖检**：  
   - 随机取5%的货箱，每箱拿1-2根纵向切开：  
     - 果肉中心白色、无褐变：健康。  
     - 果肉褐变或出现黑色条纹：已感染坏死，全批风险高（可能运输中受低温伤害或机械伤）。

6. **检查储运环境**：  
   - 果柄朝上摆放（防挤压伤）、未见明显水渍或包装破损——水渍会加速细菌繁殖。  
   - 若用乙烯催熟箱，需确认是否已使用催熟剂；未催熟的生蕉可能无法自然转黄，需补充处理。

**快速判断风险等级**：
- 低风险：颜色青/黄绿、手感硬、无酸味、抽样无褐变。
- 中风险：全黄但无斑点、手感略软、气味正常——需优先卖，避免积压。
- 高风险：超过10%表面有黑斑/软点、闻到酸味、抽检内部褐变——整批应降价速销或退货，避免损耗扩大。
