酒店在采购水果盘原料时，供应商应至少保证以下规格和外观标准：

**一、规格标准**
1. **新鲜度**：水果应无腐烂、无变质、无病虫害，采摘时间控制在合理范围内（通常不超过48小时）。
2. **成熟度**：根据水果种类，成熟度应适中（如香蕉不宜过青或过黑，芒果、猕猴桃等需可即食或短时后熟）。
3. **尺寸统一**：同一批次水果的直径、重量或大小应基本一致（如苹果直径差不超过1厘米），以利于摆盘美观。
4. **单品均匀度**：同一种水果的色泽、形状、硬度应无明显差异。

**二、外观标准**
1. **清洁度**：无泥土、灰尘、农药残留痕迹，表面无异常黏液或霉斑。
2. **完整性**：无机械损伤（如压伤、划伤、摔伤）、无裂口、无冻伤、无局部发软或凹陷，果梗完整或修剪平整。
3. **色泽**：符合该品种自然成熟颜色（如草莓鲜红、葡萄紫黑或青绿分明），无异常变色或黑斑。
4. **气味**：具有品种特有的清香，无发酵味、酸败味或化学药品味。
5. **包装**：如为预包装原料，需密封完好，标签标注产地、采摘/包装日期、保质期等。

**三、附加建议**
- 供应商应提供第三方或自我质检报告，确保农药残留、重金属、微生物等符合食品安全标准（如GB 2762、GB 2763）。
- 对于高价或易损果品（如蓝莓、树莓、樱桃），需额外要求单粒完整度及防震包装。
- 可要求供应商提供样品验收，且到货时货物温度、湿度符合冷藏要求（多数水果需0-8℃冷链运输）。

如果不能确认供应商是否达标，现场拆包检查并核对以上标准即可判断。
