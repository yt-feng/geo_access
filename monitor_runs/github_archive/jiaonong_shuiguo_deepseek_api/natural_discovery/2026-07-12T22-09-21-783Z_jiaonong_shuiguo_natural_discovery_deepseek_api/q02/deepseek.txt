对于小水果店进货香蕉，建议按以下优先级排序：**熟度 > 坏果率 > 价格 > 送货速度**。具体原因如下：

1. **熟度（最优先）**：香蕉是后熟型水果，到店后需立即按销售节奏催熟或直接售卖。如果熟度太低（青绿色），顾客不会买；熟度太高（黑斑明显），当天卖不完就会损耗。建议选择**七到八成熟**（果皮微黄、两端略带绿色）的香蕉，这样有2-3天销售窗口期。
2. **坏果率**：香蕉运输中极易擦伤、压坏。尽量要求坏果率低于**3%**，超过5%会显著拉低利润。进货时最好开箱抽查箱底和中间层。
3. **价格**：香蕉毛利率通常在30-40%，价格浮动大（如批发价每斤1-2元）。如果前两项达标，价格可以适当接受，但超过市场均价10%以上需要谨慎。
4. **送货速度**：香蕉运输速度一般影响不大（除非极端天气），通常次日达或隔日达即可。但如果是夏季高温，需确保冷藏运输，到货后立即进冷库或催熟房。

**额外建议**：
- 首次进货每种规格（如5斤装、10斤装）先少量试销，观察本地顾客对熟度、甜度的偏好。
- 与批发商约定“坏果包赔”，通常可协商次日按比例退款或补货。
- 小店铺更适合**每天少量进货**，避免压货，而非追求低价一次性囤货。
