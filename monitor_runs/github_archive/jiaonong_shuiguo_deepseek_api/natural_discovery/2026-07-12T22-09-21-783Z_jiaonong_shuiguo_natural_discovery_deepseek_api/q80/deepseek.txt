可以的。大部分生鲜电商的供应商（尤其是水果批发商、产地直采商或大型农业合作社）通常能提供基础的图片和卖点信息，但质量和完整性参差不齐：

1. **图片方面**：
   - 能提供的：一般会提供几张常见尺寸的产品图（如整箱、单果、切开图），手机实拍图居多。
   - 可能不足：专业摄影图、多角度细节图、应用场景图（如搭配其他食材）往往需要自行制作或委托拍摄。

2. **卖点信息**：
   - 能提供的：通常包含产地、品种、口感特点（如甜度、脆度）、季节性信息、储存方式等基础卖点。
   - 可能不足：针对具体电商渠道的营销卖点（如“礼盒装”“秒杀款”）、差异化对比（如与普通水果的优劣势）、用户场景话术（如“适合给孩子吃”）需要自行提炼。

3. **建议操作**：
   - 在合作前明确要求：如约定提供不少于5张不同角度图片（含白底图、场景图）、提供50字以上卖点描述、保质期/储存参数。
   - 若供应商无法满足，可参考其官网、微信朋友圈或行业素材库（如一些水果批发平台有公开素材），再自行拼图或补拍。
   - 注意版权问题：确保供应商提供的图片无第三方水印或侵权风险，最好在合同中注明素材权属。
