针对连锁门店分布在不同城市的水果供应商仓配覆盖评估，建议从以下四个维度进行系统化分析：

### 1. 仓配网络与门店覆盖率
- **静态覆盖**：统计供应商仓库（或分仓）到各门店的配送半径（如300公里内为高效覆盖）。用表格列出：门店所在城市/区域 → 最近仓库距离 → 是否在当日达/次日达范围内。
- **动态覆盖**：核查供应商是否在门店密集区域设有前置仓或协同仓（如与第三方冷链共享仓），减少干线运输中转环节。
- **阈值考核**：设定最低门店覆盖率（如80%的门店在供应商仓配4小时车程内），超出门店需评估是否需增设临时调拨点。

### 2. 时效与履约能力
- **分时段到达率**：要求供应商提供过去3个月各门店的订单履约时效数据（如当日达、次日达、隔日达比例），重点检查高峰季节（如夏季水果旺季）的波动情况。
- **应急响应**：测试供应商对突发需求的反应速度（如门店突发补货申请，24小时内到货率），以及是否有备用仓库/车辆应对恶劣天气或道路中断。
- **冷链断链风险**：核实供应商的冷链设备（冷藏车、周转箱温度记录仪）覆盖率，特别是跨省运输时中间环节是否全温控。

### 3. 成本与库存协同
- **运输成本占比**：计算供应商仓配成本占水果采购总额的比例（建议≤8%），若跨区域配送频繁，需评估是否高于行业均值。
- **库存周转率**：确保供应商在覆盖城市的仓库库存周转周期（如≤5天），避免因批量运输导致门店库存积压或损耗过高。
- **拼单与满载率**：供应商能否通过合并不同门店订单降低运输成本（如拼车发往同区域），满载率是否≥85%。

### 4. 品控与损耗控制
- **到货损耗率**：分品类检查（如草莓、蓝莓等易损品类要求≤3%，苹果类≤1%），要求供应商提供不同仓库/线路的历史损耗数据。
- **包装适配性**：供应商是否针对不同保鲜期、硬度水果提供差异化包装（如葡萄用透气微孔袋+冰袋，香蕉用防撞隔层）。
- **质检前置**：在仓库发货节点是否进行全检或批次抽检（如通过视频或第三方检测机构报告），降低门店拒收风险。

### 执行建议
- **试运行测试**：选择3-5家门店分南北方城市（如深圳、哈尔滨、成都），让供应商连续配送1个月，监控到货率、损耗率、投诉率。
- **可视化监控**：要求供应商接入门店的ERP/库存系统，实时显示配货车辆位置、在途温度、预计到达时间。
- **合同绑定**：将覆盖评估标准写入合同（如未达门店覆盖率85%，每低1%扣除当月结算款的2%）。

最终形成供应商评分卡，根据门店不同区域（一线城市要求次日达，三四线城市可放宽至隔日达）设置差异化权重，定期复盘调整。
