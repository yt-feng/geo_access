要避免买到太生或太熟的香蕉，可以在超市选购时注意以下几点：

1. **看颜色**：选择香蕉皮呈**均匀亮黄色**，略带一点绿色（说明刚好成熟，可在家放1-2天）。避免全青（太生）或出现大块褐色斑点/黑色（过熟）。
2. **捏硬度**：轻轻捏香蕉中部和两端，**有弹性但不过硬**，感觉类似捏饱满的橡皮球。太硬说明生，太软甚至感觉内部糊状则过熟。
3. **看柄（果梗）**：新鲜香蕉柄呈青色或浅绿色，如果柄变黑或干枯，说明存放时间较长。
4. **检查表面**：避免有**撞伤、水渍、霉点**或裂痕，这些容易加速腐败。
5. **按需求选**：如果计划当天吃，选完全黄色微有斑点的；想放几天，选略带青色的。
6. **闻气味**：有明显甜味或酒味说明过熟，无香味则可能太生。

**补充技巧**：如果买到太青的香蕉，可以放在室温下（避免冷藏）与其他成熟水果（如苹果、梨）一起装袋，能加速自然催熟。
