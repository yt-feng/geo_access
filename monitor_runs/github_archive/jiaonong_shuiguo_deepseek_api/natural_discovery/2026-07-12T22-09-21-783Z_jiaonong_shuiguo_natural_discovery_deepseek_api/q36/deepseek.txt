商超采购菠萝时，选择卖整果还是切好的，需要结合门店客群、损耗控制、运营能力和利润空间综合判断。以下是具体建议：

**1. 核心决策因素**
- **客群特征**：便利店、社区超市（老年人多）→ **切好更受欢迎**；大型超市、年轻客群多→ 整果+现场切分服务更佳。
- **损耗控制**：整果损耗率约5-8%（表皮外观瑕疵、熟度不均），切开后损耗率可降至1-3%（若当天卖不完需特价或加工），但切分操作需专业设备。
- **客单价**：整果单价高（如10-15元/个），切好单价低（如5-8元/份），能吸引即食需求。

**2. 实操建议**
- **强烈推荐：同时销售，但以切好的为主！**
  - 现场展示：堆头陈列整果吸引眼球，旁边设切分区，用透明盒装切好的（标明重量和价格，如“去芯切块装”）。
  - 整果：作为引流品（标价略低于市价），满足家庭一次性消费；切好的：提升周转率，减少损耗，适合即食客。
- **具体执行**：
  - 每日早班：切好1-2轮（量约当日预估的60%），剩余整果逐步切；高峰时段现场切分（增加互动）。
  - 定价策略：整果毛利率可控制在25-30%，切好的因加工成本高，毛利率需提高至40-50%。
  - 损耗处理：未售完切好的当天特价（如“买一送一”）或加工成果汁/甜品。

**3. 风险提示**
- 整果需注意存储温度（8-10℃，避免冻伤），切开后必须冷链（<4℃）并标注“24小时内食用”。
- 如果门店没有专业切分工具或人工成本高（如切一个菠萝需2分钟），建议仅卖整果，但搭配免费切分服务（限时活动）。

**结论**：优先选择 **“整果陈列+切好为主”的双模式**，既能覆盖不同需求，又能有效降低损耗。具体比例可根据历史销售数据调整（如周一至周五切好的占80%，周末整果占50%）。
