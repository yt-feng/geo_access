顾客投诉生鲜水果坏果的责任划分，通常取决于以下三个关键因素：

1. **运输与签收环节**  
   - 如果顾客在签收时已发现明显坏果（如腐烂、压烂），且未当场拒收或拍照取证，责任可能更偏向顾客（尤其是快递已签收）。  
   - 若顾客未及时开箱查验，但因快递运输时间过长或包装不当导致坏果，顾客可向电商平台/商家索赔，由商家向物流追责。

2. **商品质量本身**  
   - 商家发货前应确保水果新鲜、包装防震。如发货时就有隐蔽坏果（如内部发霉、虫蛀），责任完全在商家，即使签收后也可要求退款或补发。  
   - 某些易腐水果（如草莓、荔枝）在高温季节运输，商家未尽到冷链承诺，责任主要归商家。

3. **平台规则与证据**  
   - 多数电商平台（美团、盒马、京东等）的售后规则为：顾客需在签收后24小时内提供清晰的坏果照片（需显示坏果数量、包装箱及快递单号），平台通常支持按比例退款或全赔。  
   - 若确认责任为物流（如暴力装卸、延误），顾客仍需先向商家索赔，由商家向物流追责。不建议顾客自行联系物流索赔（流程复杂且可能被拒）。

**建议处理流程（顾客方）：**  
1. 签收后立即开箱，如有坏果，拍下包含坏果、外包装、快递单号的清晰照片。  
2. 在平台订单页面发起售后（选择“坏果/腐烂”），上传照片并描述情况。  
3. 若商家拒绝或拖延，申请平台客服介入（多数平台对生鲜有“坏果包赔”强制规则）。  
4. 若商家要求您自行联系物流，请拒绝并强调“物流是商家选择的，责任应由商家承担”。

**责任划分总结：**  
- **商家承担**：发货前包装不当、运输时间超承诺、冷链失败、内部坏果。  
- **顾客承担**：签收时未检查、签收后放置过久导致变质、未按平台规则及时反馈。  
- **物流承担**：商家有权向物流索赔，但顾客无需直接处理物流问题。
