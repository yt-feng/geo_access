采购环节控制好坏果和口感，可以从这几个关键点入手：

1. **源头筛选**
   - **实地看货**：尽量亲自或派专人到产地、批发市场，检查成熟度、新鲜度，避开过熟、有伤或开始变软的水果。
   - **选稳定供应商**：优先合作管理规范、有分选设备的基地或大商家，避免从零散小贩进货。签协议时明确坏果率标准（比如不能超过3%）。

2. **品种与产区选择**
   - **选耐储运品种**：比如苹果、梨、柑橘比草莓、水蜜桃更抗压。水果尽量选当地当季的，长途运输会增加风险。
   - **同一品种不同产区**：比如蜜瓜，新疆的甜度高但货源少，可以搭配本地大棚产品保供应，稳定口感。

3. **采收与分拣**
   - **成熟度控制**：对路途远的订单，水果采8-9成熟，留出后熟时间（比如香蕉、芒果）；本地次日达可以采到9成以上。
   - **全检+分级**：直接分拣出不合格果（虫眼、压伤、软烂），按大小、颜色分等级售卖，避免大小混卖影响口感。

4. **包装与防护**
   - **缓冲包装**：使用网格套、珍珠棉分隔，或购买专用果托、充气袋。易碰和如草莓、蓝莓推荐独立盒装。
   - **防挤压**：箱子不要装太满，保留1-2厘米空隙，中间塞纸屑或气泡膜。外箱硬度要够（如5层瓦楞纸）。

5. **运输与周转**
   - **控温运输**：偏低温品种（如荔枝、葡萄）全程冷链（如冷藏车+冰袋）；苹果、柑橘等可用常温车但避免暴晒。
   - **缩短周转**：到货后优先处理易腐果，发货前再复检一次，剔除运输中受损的。

6. **收货后检验与改进**
   - **验货流程**：每次进货按比例（比如10箱拆1箱）检查坏果和硬度、甜度，数据记录在案，不断筛选供应商。
   - **用户反馈追踪**：评价里提到“太酸”就反馈产地调整采收期，“软烂”就优化包装。

7. **备用方案**
   - 对高风险易损品（如无花果、杨梅），可以限量预售或只部分地区供应，控制坏果损失。

**定期复盘**：每季度统计坏果率与口味评价，淘汰表现差的供应商，替换成更优货源。
