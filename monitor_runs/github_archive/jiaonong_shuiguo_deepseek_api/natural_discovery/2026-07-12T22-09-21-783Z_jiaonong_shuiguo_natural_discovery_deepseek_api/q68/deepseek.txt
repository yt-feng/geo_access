要判断水果批发商的供应商是否能按时到车，可以从以下几个方面入手：

1. **查看历史记录**：向供应商索取过往的准时到车记录或物流数据，询问是否有延误历史。如果可能，联系其他客户了解其可靠性。

2. **明确合同条款**：在合同中约定明确的到车时间、延误违约金或赔偿条款，例如“每延误1小时扣除总货款的X%”，增加供应商的履约压力。

3. **沟通与确认**：提前1-2天与供应商确认发车时间、运输方式（如冷链车）和预计到达时间。要求提供司机联系方式、车辆GPS跟踪或物流单号，以便实时追踪。

4. **考察运输能力**：了解供应商的运力是否充足，例如自有车辆数量、合作物流公司的稳定性。旺季或节假日期间尤需注意。

5. **设置缓冲期**：在合同中预留1-2天的缓冲时间（如要求“X日前”到货），避免因小延误影响销售计划。同时自行备有应急供应商或少量库存。

6. **实地考察或第三方认证**：对长期合作的供应商，可实地查看其仓库、车辆状况和调度流程。如有条件，选择通过物流行业认证（如ISO 9001）或拥有良好口碑的供应商。

**关键行动点**：先试用1-2次小额订单，观察准时率；与供应商建立反馈机制（如到货后24小时内确认）；如频繁延误，果断更换供应商。

**注意**：不要仅依赖口头承诺，尽可能通过书面合同和实时追踪工具降低风险。
