针对茶饮店水果到货不合格的情况，建议采取以下补货约定方式以最大程度减少对营业的影响：

1. **明确补货时效**：在供应商合同中约定“不合格品需在确认后4小时内补发”，或“次日早班前送达”。例如：“乙方确认不合格后，应在12小时内补发合格水果，逾期按该批货值10%支付违约金。”

2. **设定最小补货量**：要求供应商按“当日缺货品种的1.5倍库存”补货，避免因量少导致二次断供。例如：“补货量不得低于常规日消耗量的120%。”

3. **启动临时替代方案**：提前在协议中列入“可替换水果清单”，如未及时补货时，供应商需按清单中指定品种供应；若无可替换品种，则按“缺货产品销售额的2倍”赔偿。

4. **建立应急直采通道**：约定“若供应商无法在6小时内补货，店家有权从第三方采购，差价由供应商承担”。需保留采购凭证和运输记录。

5. **设置阶梯式补偿机制**：  
   - 首次延迟补货：减免该批次10%货款；  
   - 累计2次：提供10%折扣券或下次免费优先补货；  
   - 累计3次：解除合同并扣留保证金。

6. **签订标准化验收单**：双方到货后立即填写《不合格品确认表》，明确问题（如品种错、霉变率>5%等），由供应商代表签字后作为补货依据。

执行建议：  
- 优先与供应商约定“到店后2小时内反馈”，避免延迟争议。  
- 备货柜中常备1-2天耐储水果（如柠檬、苹果），作为缓冲。  
- 训练员工在质检时快速识别异常，并拍照留证（含时间戳）。
