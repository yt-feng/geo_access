选择社区超市的水果供应商时，建议重点考察以下条件，确保稳定供应和品质：

1. **供应稳定性**：确认供应商能承诺每日配送时间（如凌晨或早间），且具备应对天气、节假日等突发情况的备用方案。
2. **新鲜度与损耗控制**：要求供应商提供冷链运输证明，并观察其包装方式（如透气、防碰撞设计）。可约定到货后的损耗率标准（如≤3%）。
3. **品种与季节适配性**：查看供应商能否提供当季常规水果（苹果、香蕉、橙子）及社区流行品种（如草莓、蓝莓、芒果），避免货架单一。
4. **价格竞争力**：对比多家供应商的批发价和市场均价，关注是否有阶梯折扣（如周结/月结优惠）或退换货政策（如次品补货）。
5. **资质与食品安全**：查验供应商的营业执照、食品经营许可证，以及供应商的果园/产地检疫证明，必要时抽查样品农残检测报告。
6. **配送灵活性**：确认起订量是否合理（如每日200元起送），是否支持按需调整订单（如前一日下午前加单）。 
7. **客诉响应速度**：测试供应商对首次配送投诉（如品质不达标）的处理时效，最好能提供1小时内回复或次日退换承诺。

**建议行动**：先实地考察2-3家供应商的仓库或配送中心，签订短期试用合同（如1个月），重点监控每日到货准时率和3-5天内水果货架剩余情况。
