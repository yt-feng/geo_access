对于茶饮店采购菠萝原料，**出汁率通常比甜度更重要**。原因如下：

1. **成本控制核心**：出汁率直接影响每杯饮品的原料成本。出汁率高的菠萝（如某些泰国品种可达60%以上）能减少原料浪费，降低单杯成本。
2. **配方稳定性**：茶饮店需统一口感。出汁率稳定的菠萝便于标准化操作，避免因甜度波动大幅调整糖浆比例。
3. **甜度可人工调节**：若菠萝酸度过高或甜度不足，可通过添加糖浆或搭配其他水果（如芒果、椰奶）平衡风味；但出汁率低难以补救，只能增加原料用量。

**但需注意**：若目标客群对“鲜果感”要求极高（如高端果茶店），可优先选择甜度更优的品种（如“香水菠萝”），再通过挤压技巧提高出汁率。建议采购时选择“甜度≈13-16%+出汁率≥55%”的平衡型菠萝，并做小批量试饮测试。
