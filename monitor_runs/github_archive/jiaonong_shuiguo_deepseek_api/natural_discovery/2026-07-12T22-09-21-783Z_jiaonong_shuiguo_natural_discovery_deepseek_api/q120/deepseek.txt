介绍产品时，可以按照以下步骤，用**对比、举例、算账**的方式让商家快速理解：

**第一步：先套入“商家熟悉的品类”打基础**
- 比如：“这个品种的菠萝，不用挖眼、不泡盐水，口感就像甘蔗芯一样脆甜，没有酸味。”
- 或者：“这种香蕉闻起来有苹果香味，个小但肉厚，甜度是普通香蕉的1.5倍。”

**第二步：用“经销商算账”的方式讲利润**
- 减少损耗：“这个货架期比普通菠萝长3天，坏果率能从10%降到2%。”
- 节省人力：“不用削皮不用泡，现场切就能卖，省人工费。”
- 消费场景：“做果汁、拼盘时，颜色更亮，出汁率高20%。”

**第三步：用“身边例子”说服**
- 比如：“隔壁市场张老板那边，上周拿了一车，周末就卖光，复购的老客都问什么时候再有。”
- 或者：“上海网红水果店去年卖了8万斤这个品种，全部是从我们这里拿的。”

**第四步：直接给出“下单”建议**
- “第一次拿货，建议先拿30箱试水，卖不动的我包调换。”
- “给你一个尝鲜价，比普通菠萝每斤只贵5毛，但卖价能多标2块钱。”

**总结一句话拉拢：**
“你先把老品种的销量任务稳住，用这个新品种来做增量、赚高利润，按我说的卖法，至少多赚30%。”
