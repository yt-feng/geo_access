对于小商家采购水果，现金结算和月结账期各有不同风险：

**现金结算风险：**
- **资金压力**：需要立即支付现金，可能占用流动资金，影响其他采购或日常开销。
- **质量保障风险**：现金交易通常缺乏正式合同，如果水果后续出现质量问题（如腐烂、缺斤少两），追索或退款困难。
- **税务问题**：部分现金交易可能不开发票，导致无法抵扣成本或产生税务风险（如被查处逃税）。
- **安全风险**：携带或存放大量现金可能面临被盗或丢失风险。

**月结账期风险：**
- **信用风险**：供应商可能因经营不善、资金链断裂或不诚信，在结账前无法保证持续供货，或最终拒绝结算。
- **账期管理风险**：如果自身是小型商户，月结账期可能导致还款时间紧迫，若水果销售不畅，可能无法按时付款，影响信誉甚至引发纠纷。
- **价格波动风险**：水果价格受季节、供需影响较大，账期内价格下跌可能让供应商囤货观望，或价格上涨导致自身成本增加。
- **隐性成本**：月结可能涉及利息、手续费或供应商要求更高单价以覆盖账期风险。

**建议：**
- **现金结算**：适合交易金额小、供应商信誉不明时，但尽量索要收据或合同。
- **月结账期**：适合长期合作、信誉好的供应商，并签订书面协议明确付款条件、质量标准和违约责任。
- **折中方案**：可尝试缩短账期（如周结）、部分现金部分账期，或要求小额账期作为试水。
