如果担心水果农残超标，建议在采购前向供应商索要以下关键资料：

1. **第三方检测报告**  
   要求提供近3个月内、由具备CMA/CNAS资质的检测机构出具的《农残检测报告》，重点关注是否涵盖采购水果的完整农残项目（如有机磷、拟除虫菊酯类等常见农残检测指标）。

2. **产地证明与产地合格证**  
   索要《产地证明》或《食用农产品合格证》，确保供应商能追溯种植地信息。部分地区要求合作社或种植户开具该证明。

3. **进货台账复印件**  
   查看供应商以往检测记录与进货批次对应情况，确保每批次产品有清晰溯源链条和检测覆盖记录。

4. **供应商资质文件**  
   提供营业执照、食品经营许可证（如适用）等基础资质，并确认其经营范围涵盖水果销售。

5. **企业自检记录**  
   部分大型供应商会设有内部快检实验室，可要求提供其自检记录（如胶体金法快速检测卡结果），作为辅助参考。

**建议行动**：  
- 优先选择有“绿色食品”“有机认证”或“GAP（良好农业规范）”认证的供应商，这类产品农残风险更低。  
- 签订合同时明确约定：**如抽检不合格，供应商需承担全部退货、销毁及违约责任**，并保留随机抽检第三方复检的条款。  
- 收到货物后，建议自留样品并定期委托官方检测机构抽检，形成双重保障。
