针对批发档口拓展餐饮客户（如奶茶店、果汁店、甜品店、宴席备餐等），**菠萝和香蕉**的建议规格如下：

**一、菠萝：优先准备“成熟度可控、易加工”的规格**

餐饮客户通常需要直接使用果肉，对出成率、甜度、酸度一致性要求高。

1. **推荐品种与规格**
   - **品种**：首选**“金钻菠萝”（无眼菠萝）**或**“香水菠萝”**。避免传统有眼菠萝（去皮麻烦，损耗大）。
   - **单果重量**：**1.25-1.5公斤/个**（中等大小）。这个大小单果去皮后净果肉约700-900克，适合单份或小批量使用。
   - **包装**：**10公斤/箱**（约6-8个）或**20公斤/箱**。建议提供“带箱发货”的标准纸箱。
   - **成熟度**：
     - **八成熟（黄绿色带香）**：供**奶茶店、果汁店**（需冷藏后1-3天内用完，风味最佳）。
     - **六至七成熟（果皮青绿色）**：供**食堂、宴会切配**（需要常温后熟，方便提前备货）。

2. **增值加工形式（可提升客单价）**
   - **去皮整果**：已削皮去眼，真空包装，单果约1公斤净重，适合高端茶饮店（减少人工成本）。
   - **分级规格**：
     - **大果（>1.5公斤）**: 适合甜品店做菠萝船、菠萝饭摆盘。
     - **小果（0.8-1公斤）**: 适合果汁店快消量，不占库存。

> **关键提醒**：餐饮客户对“成熟度一致性”非常敏感。如果提供青果，务必明确告知“常温下放置X天后使用”，避免客户因催熟时间不足导致投诉。

**二、香蕉：优先准备“耐储存、易熟化”的规格**

餐饮客户使用香蕉主要分两类：现切装饰（如甜品、沙拉）和搅打化（奶昔、芭菲）。

1. **基础规格**
   - **品种**：**“巴西香蕉”**（最常用，口感软糯甜度高）或**“小米蕉”**（主打小份量、造型精致）。
   - **单果重量**：**80-120克/根**（一根果肉约60-90克）。
   - **包装**：**13-15公斤/箱**（约140-180根）。
   - **成熟度**：
     - **七成黄（果柄青、果身黄）**：适合**现切水果店、生食餐厅**（室温可放2-3天）。
     - **全熟（果皮有少量褐斑）**：适合**奶茶店、烘焙店**（直接出浆，无需额外催熟，需当天或隔天用完）。

2. **专业规格推荐**
   - **“果梳”分级**：
     - **大果（100-120克/根）**：适合切盘、拼盘。
     - **中果（80-100克/根）**：适合单份甜点搭配。
   - **熟化速冻香蕉肉**：如果客户用量大（如大型连锁奶茶店），可提供**去皮速冻香蕉段（每段约50克）**，常保鲜期6-12个月，方便按次解冻。

> **关键提醒**：香蕉**极容易碰伤**。在批发场景中，如果客户是餐厅，可额外提供**“定制催熟服务”**（如按客户要求控制在3天或7天后成熟），这对大型客户有很强吸引力。

**三、针对餐饮客户的具体议价/沟通建议**

1. **批量阶梯价**：
   - 菠萝：单次订购5箱以上可提供“定制切块服务”（如切2cm丁或3mm片）。
   - 香蕉：单次订购10箱以上可提供“预熟化按批发货”（如可要求按周五到货、周一全熟）。

2. **损耗承诺**：
   - 菠萝：如果客户开箱后发现“黑心病”超过5%，可以按比例折价或补货。
   - 香蕉：如果客户存放得当但“突然全部同时变黑”，可协商处理——这类售后最容易建立长期信任。

3. **季节性提示**：
   - 菠萝3-6月口感最佳，冬春果偏酸，可建议客户搭配“糖浆”或“酸度调节”。
   - 香蕉全年供应稳定，但夏秋季香蕉皮色泽更鲜亮，适合摆盘；冬季皮色易暗，适合搅打使用。

**总结建议**：
- **菠萝**：主推 **1.25-1.5公斤/个、八成熟、10公斤/箱** 的“金钻菠萝”。
- **香蕉**：主推 **80-120克/根、七成黄、13公斤/箱** 的“巴西香蕉”，并额外提供“速冻去皮果肉”选项。
- **增值点**：为餐饮客户提供“免费切块/去皮样品试做”和“损耗包赔”政策。
