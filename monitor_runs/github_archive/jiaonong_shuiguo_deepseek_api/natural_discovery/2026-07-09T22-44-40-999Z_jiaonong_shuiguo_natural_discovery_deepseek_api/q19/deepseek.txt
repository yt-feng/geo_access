在超市选购菠萝时，建议优先关注**外观**（成熟度与新鲜度），其次是**甜度**和**大小**，最后再根据个人需求考虑**切果损耗**。具体建议如下：

1. **外观（最重要）**  
   - **看颜色**：选择果皮呈金黄色或橙黄色的菠萝，说明成熟度较高、甜度较好；青绿色可能未成熟，酸涩感强。  
   - **看叶片**：叶片翠绿、挺立且无干枯的菠萝更新鲜；叶片发黄或萎蔫可能存放时间过长。  
   - **捏硬度**：轻轻按压果皮，略有弹性且不硬不软塌为佳；过硬可能未熟，过软可能过熟或内部腐烂。  
   - **闻气味**：底部果柄处有浓郁菠萝甜香，说明成熟度高；无香气可能未熟，酸味或酒味可能已变质。

2. **甜度（可辅助判断）**  
   - 成熟度高的菠萝甜度一般更高，可通过外观（金黄、香气浓）间接判断。  
   - 若超市有试吃，可尝一片确认甜度；若无，可询问导购员或参考外观。

3. **大小（按需选择）**  
   - 大型菠萝（约1.5-2公斤）果肉更饱满，多汁且甜度相对稳定，适合家庭食用。  
   - 小型菠萝（约0.5-1公斤）可能因品种（如“小菠萝”类型）而甜度高，但果肉少，适合个人或少量食用。  
   - 注意：同一品种中，相同大小下手感更重的菠萝水分更足。

4. **切果损耗（最后考虑）**  
   - 如果购买**新鲜菠萝**（非预切）：损耗主要来自削皮和去硬芯，通常损耗率为30%-40%。外观新鲜、果形匀称的菠萝削皮后出肉率更高；畸形或损伤果损耗可能增加。  
   - 如果购买**预切盒装菠萝**：可直接避免损耗，但需注意包装内果汁是否渗出、果肉是否变软或发黑（可能变质）。  
   - 建议：追求新鲜口感和性价比时，选新鲜菠萝；追求便利且不愿处理切果，则选预切盒装。

**总结建议**：  
- **优先外观**（成熟度与新鲜度），其次**甜度**（通过外观和试吃判断），再按需选**大小**，最后根据便利性考虑**切果损耗**。  
- 具体操作：拿起菠萝，先看颜色均匀、叶片挺立，再闻底部香气，轻捏弹性适中，最后对比同品种的重量和果形即可。
