采购菠萝时判断熟度，建议从以下5个维度入手，确保到店后既能满足即时销售需求，又能减少损耗：

1. **看颜色（最直观）**  
   - **青绿带黄**：约七成熟，适合运输和短期储存，可存放2~3天。  
   - **金黄明显**：约九成熟，适合当天或次日销售，需尽快处理。  
   - **全黄或出现褐斑**：过熟，易变质腐烂，避免采购（除非特价急售）。

2. **摸硬度（关键）**  
   - **按压果身**：整体硬实但有轻微弹性（像捏橡皮球）最佳。  
   - **过硬**：未熟，口感酸涩。  
   - **过软或局部软塌**：已过度熟或内部开始腐烂，损耗高。

3. **闻气味（区分生熟）**  
   - **基部（底部）有淡淡清香**：自然熟度适中。  
   - **无味或青草味**：未熟。  
   - **浓烈酒味或酸味**：已发酵变质，不可采。

4. **查果眼（根部细节）**  
   - **果眼（表皮凹点）颜色深、眼距宽**：成熟度高。  
   - **果眼紧密、颜色浅**：偏生。  

5. **掂重量（鲜度指标）**  
   - **同体积选重的**：水分足、果肉饱满；**手感轻**可能干货、失水，口感差且易干枯。

**采购建议**：  
- **按销售节奏分批次**：到货后即刻销售的选金黄带弹性（约八成熟）；计划隔天上架的选青黄各半、稍硬（七成熟）。  
- **避免“全熟”或“全生”集中采购**：全熟当天卖不掉损耗大，全生客户试吃后差评多。  
- **保鲜处理**：到货后立即挑出过熟果，未熟者放在阴凉通风处，避免低温冷藏（易损伤果肉）。  

**注意**：不要仅凭“香气浓郁”就判定熟度，部分催熟果气味重但内部未熟透（可切开一个验证）。
