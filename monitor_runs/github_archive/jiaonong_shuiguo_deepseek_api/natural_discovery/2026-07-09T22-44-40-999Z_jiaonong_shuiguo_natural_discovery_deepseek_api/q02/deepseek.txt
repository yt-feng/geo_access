在小水果店进香蕉时，**最应该优先看重的是“熟度”**，其次才是“坏果率”，然后综合考虑“价格”和“送货速度”。具体原因和操作建议如下：

1. **熟度（首要）**：香蕉是典型的“后熟型”水果，进店后需要自然成熟。过生（全绿）的香蕉顾客不会买，过熟（发黑、软烂）的香蕉损耗极高。建议选择七到八成成熟度的香蕉（果面微黄、棱角分明），这样有3-5天的销售期，既能及时卖出，又不会快速腐烂。

2. **坏果率（第二）**：坏果直接影响直接成本。可以要求供应商在装箱时确保坏果率低于5%（如每箱40斤，坏果不超过2斤）。进店后立即开箱检查，发现明显腐烂、水渍或断裂的，可以当场扣除或要求补货。

3. **价格（第三）**：在保证熟度和坏果率达标的前提下，比较不同供应商的价格。但不要为了低价接受生果或高坏果率，否则损失更大（比如低价生果需时间催熟，期间可能赔本）。

4. **送货速度（第四）**：对于小水果店，如果附近有稳定的批发市场，可以当天自提。如果依赖配送，建议选择能次日达或隔日达的供应商，避免夏季高温导致香蕉在运输中加速成熟。紧急补货时，可优先考虑配送快的商家。

**实操建议**：
- 初次与合作供应商签订小批量试单（如10箱），重点观察熟度一致性和坏果率。
- 如果资金允许，可以备少量“全绿”香蕉作为库存，用乙烯利（按说明稀释）催熟控制销售节奏。
- 每天到货后，用电子秤随机抽检5%的香蕉称重，对比标重，防止水分蒸发造成的缺斤少两。

综上，**熟度 > 坏果率 > 价格 ≈ 送货速度**，按此顺序权衡最有利润。
