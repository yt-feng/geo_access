建议优先购买**鲜果**或**冷冻菠萝丁**，具体选择取决于您的门店规模与出杯效率需求。以下是分析：

1. **鲜果**：适合少量、定制化高或对口感新鲜度要求严格的场景。需自行削皮、切丁，人工与时间成本高，但风味最自然，适合突出“现做”概念。适合小型精品店。

2. **冷冻菠萝丁**：成熟度、切丁规格标准化，省去预处理环节，适合中大型门店。建议选加糖冷冻产品（注意标注“糖渍”与否），直接解冻或随用随取，出杯稳定。

3. **果泥（浓浆）**：风味最稳定，但一般用于饮品调色、增稠或提供基础果味，缺少果肉纤维感。如果做“菠萝果茶”而非“菠萝汁茶”，果泥需搭配鲜果丁或果冻丰富口感；单一用果泥口感偏单一。

**总结建议**：
- 若主打“真果肉”卖点 → 选**冷冻菠萝丁**（如供应稳定可换鲜果）。
- 若求省钱、简化操作 → 可**冷冻丁为主 + 少量果泥调色**。
- 避免只用果泥做果茶，除非是清透果茶（保留汁水感）。

**注意事项**：菠萝酸性强，选择果泥时注意总酸和糖度，建议搭配茶底（如四季春、绿茶）中和涩感。采购前要求供应商提供**Brix值（含糖量）** 与**果肉含量**数据。
