为确保茶饮店在采购水果时控制食品安全风险，供应商应提供以下关键证明文件，建议您逐一核实并留存原件或清晰的复印件备查：

1. **《食品经营许可证》或《食品生产许可证》**  
   证明供应商具备合法从事食品/农产品经营的资质。需注意许可证的经营范围明细中是否涵盖水果或初级农产品。

2. **水果的合格证明文件**  
   - **产地证明**：由生产地村委会、农业部门或批发市场出具的产地来源证明。
   - **农产品合格证**：符合《农产品质量安全法》要求的《食用农产品合格证》，须明确标注产品名称、产地、生产者、检测结果及承诺等内容。
   - **第三方检测报告**：针对高风险的品种（如草莓、葡萄、蓝莓等易超标水果），建议检测项目包括：农药残留（如有机磷、拟除虫菊酯类）、重金属、致病菌等。检测报告需由具备CMA/CNAS资质的第三方机构出具，且出具日期应在近3个月内。

3. **进货与经销记录**  
   供应商需提供**完整可追溯的销售台账**，包括：进货日期、水果品种、数量、产地、批次号、供货下游客户名称等信息，以便出现问题时能快速溯源。

4. **运输及储存环境证明**（如需冷链运输）  
   若水果需冷藏（如鲜切水果、杨梅、石榴等），供应商应提供**冷链运输温度记录**或**冷藏设备校准证明**，确保运输及储存温度符合要求（如0-4℃或-18℃）。

5. **供货合同（含质量条款）**  
   合同中应明确食品安全责任条款，包括但不限于：  
   - 水果应符合国家强制性标准（如GB 2762《食品安全国家标准 食品中污染物限量》、GB 2763《食品中农药最大残留限量》）；  
   - 供应商对每一批次水果的质量负责，并向茶饮店提供真实的批次抽检或全检结果。

6. **发票或合规票据**  
   每次采购均应附正规发票或专用收据，作为采购与税务合规的凭证，同时便于追溯交易过程。

**实操提醒**：  
- 建议您与供应商签订**长期供货协议**，约定每批次供货时随货提供上述证明文件的扫描件或复印件（加盖供应公章），或由供应商主动上传至茶饮店指定的管理系统。  
- 对于季节性、高单价水果（如进口榴莲、车厘子），务必额外索要**入境货物检验检疫证明**。  
- 对于小农户直供的散装水果，可要求提供村委会或合作社出具的**产地承诺书**，但尽量通过有许可证的批发商或经销公司采购，以降低风险。

通过上述文件和记录体系，您能最大程度证明“履行了进货查验义务”，在食药监部门检查或有消费者投诉时获得免责依据。
