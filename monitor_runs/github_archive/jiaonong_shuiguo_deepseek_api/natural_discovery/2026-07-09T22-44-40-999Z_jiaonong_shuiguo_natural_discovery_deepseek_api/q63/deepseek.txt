判断批发商哪个规格的菠萝更容易出货，可以从以下几个具体维度分析，并直接给出执行建议：

**1. 分析近期市场销售数据（最核心）**
- **动作：** 查看过去1-2个月、甚至上年同期的销售台账，统计不同规格（如单果重1-1.5斤、1.5-2斤、2-2.5斤）的进货量、实际出货量和退货率。
- **判断标准：** 哪个规格的**月均出货量最大、库存周转最快（到货后3天内出货比例高），哪个就是易出货规格**。如果某规格退货率高于5%，则要警惕。

**2. 调研终端渠道的偏好（不同渠道要求不同）**
- **电商/社区团购：** 通常偏好**中小果（1.2-1.8斤）**，因为单价低、易邮寄、适合家庭一次性吃完。如果批发客户以这类为主，中小果易出货。
- **水果店/超市：** 偏好**中果（1.5-2.2斤）**，外观漂亮、卖相好，顾客喜欢拎着有分量。中果是传统零售主力。
- **餐饮/加工厂（如做菠萝炒饭、罐头）：** 偏好**大果（2.5斤以上）**，出肉率高、处理效率高。如果批发客户中有这类大客户，大果规格的批量出货更稳。
- **行动：** 直接问老客户“你们最近哪种果最好卖”，或观察同行发货包装箱上的规格标签。

**3. 观察货架和线上价格差异**
- **简单方法：** 到批发市场看竞品同行哪些规格堆头最大、搬运工最忙。**堆头大+搬运频次高 = 当前主流规格**。
- **线上（如鲜丰、美团）：** 搜索菠萝，看哪个规格的销量排序靠前、评价多。通常**单品折扣力度大的规格往往是为了清库存，反而不一定是最易出货的**；常年价格稳定、销量高的才是刚需规格。

**4. 季节性快速判断**
- **旺季（3-5月）：** 菠萝大量上市，**中果（1.5-2斤）最易出货**，因为走量最大，消费者尝鲜为主。
- **淡季（其他月份）：** 菠萝价格贵，**中小果（1-1.5斤）更易出货**，因为顾客买得少、买小的不心疼。
- **特殊节点（如婚庆、节前）：** 大果（2.5斤以上）可能因为摆盘好看而短暂热销，但通常占比小。

**5. 风险校验：避免压错规格**
- **立即执行：** 控制试错成本。首批进货时，**中小果、中果、大果各进10%-20%的量**，用7天观察：哪个规格先被买完、哪个规格库存剩最多？**最先卖光的规格就是最优规格**。
- **警惕**：不要只看单价利润。比如大果单价高、毛利高，但如果周转慢（库存3天以上），资金占用大、损耗高（菠萝易烂），反而容易赔钱。小果毛利低但周转快，往往利润总额更高。

**总结：最直接、可操作的判断法——**
1. **查自己店过去1个月的销售排名**（中小果通常最稳）。
2. **问主要客户要什么**（终端反馈永远最准）。
3. **小批量试错试卖**（用最小成本验证）。

如果没有历史数据，优先试**1.5-2斤的中果**，这是零售和批发市场最通用的“黄金规格”。
