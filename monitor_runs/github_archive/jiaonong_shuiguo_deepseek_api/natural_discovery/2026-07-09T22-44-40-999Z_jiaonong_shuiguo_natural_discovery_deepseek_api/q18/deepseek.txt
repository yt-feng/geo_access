选购香蕉时，建议按以下步骤操作，可以尽量避开过生或过熟的香蕉：

1. **看颜色**  
   - 选**整体黄色、略带青头**的：这种香蕉在室温下2~3天会自然熟透，不会立刻软烂。  
   - 避免全青（太生，难催熟）或已有大片黑斑、果柄发黑的（太熟，可能内部软烂）。

2. **摸硬度**  
   - 轻捏香蕉中部：**有弹性、不软塌**为佳。  
   - 太硬（生涩）、太软（可能过熟或碰伤）都容易踩雷。

3. **观察棱角和果柄**  
   - 棱角模糊、圆润饱满的通常成熟度刚好；棱角明显的还比较生。  
   - 果柄**干燥、无霉斑**：如果果柄发黑或潮湿，香蕉可能已开始变质。

4. **留意存放时间**  
   - 如果计划**当天或第二天吃**，选全黄但无斑点的。  
   - 如果打算**放几天**，则选稍带青头的。

5. **避开缺陷**  
   - 拒绝有明显撞伤、压痕、裂口的香蕉（易腐烂）。  
   - 如果超市允许，可以轻轻拿香蕉串，感受整体硬度是否均匀。

**总结一句话**：选“金黄微青、捏起来有弹性”的香蕉最保险。
