为了减少香蕉在水果店因过熟而损耗，建议在采购时从以下两个方面提出明确要求：

**1. 熟度要求：**
- **要求到货熟度在“绿熟期”或“硬熟期”（即香蕉果皮整体呈绿色，仅尖端略带黄色，果肉坚实）。**
- **避免采购“催熟过度”的香蕉：** 果皮大范围变黄甚至出现黑斑、果皮变软的香蕉，库存寿命极短（通常1-2天就会腐烂）。
- **具体标准：** 建议和供应商约定“色卡等级”，例如要求到货时果皮颜色指数在2-3级（全绿至微黄），或要求果实硬度达到可弯曲但不易折断的“七成硬”。

**2. 到货时间要求：**
- **要求“每天固定时间到货”，最好是早上开门前（如7:00-8:00）到货**，这样当天可以及时上架并控制展示数量。
- **要求“少量多次”配送：** 如果销量不大，尽量隔天到货或一周2-3次，每次到货量控制在2-3天内能售罄的范围内。
- **避免“周末/节假日前大批量到货”：** 除非有明确促销计划，否则要求供应商不要一次性送来超过5天销量的库存。

**补充建议（自行把控）：**
- 到货后立即打开包装箱通风，避免香蕉释放的乙烯气体在箱内积聚加速成熟。
- 如果到货发现部分已变黄，单独挑出来放在显眼位置促销，尽快处理。
- 可以要求供应商提供“低温运输证明”（如在13-14℃冷链下运输），避免到货后因高温导致快速变熟。
