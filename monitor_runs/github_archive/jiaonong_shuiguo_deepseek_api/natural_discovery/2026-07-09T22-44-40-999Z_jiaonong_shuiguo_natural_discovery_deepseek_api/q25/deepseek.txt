针对超市门店多但单店销量不大的情况，建议采用**“集采分配+灵活加单”**的混合供货模式，兼顾成本与灵活性。具体可参考以下几个方向：

1. **区域型一级批发商合作（集采分货）**  
   - **操作**：与当地或邻近大城市的一级批发市场、产地直采商签订协议，由总部统一采购核心高流转水果（如苹果、香蕉、橙子等），再按各店预估销量分货到店。  
   - **优势**：能拿到比单店采更低的价格，降低总成本，同时避免因一次性到货量过大导致单店滞销。  
   - **适用**：常规、耐储运的水果。

2. **“产地直发+预售/快消”模式（减少库存压力）**  
   - **操作**：针对单价高、易损耗的精品水果（如草莓、蓝莓、进口鲜果），与产地或供应链平台合作，采用 **“门店预售+产地直发至店”** 模式。顾客线上下单后，系统汇总订单，由产地直接按店打包发货，门店仅作为提货点或短期中转。  
   - **优势**：单店无库存压力，损耗低，适合高价值但销量不高的品类。  
   - **注意**：需要系统支持订单汇总和分单逻辑，适合有线上平台的超市。

3. **生鲜配送平台/本地供应商的“次日达+T+1”配送**  
   - **操作**：与覆盖本地的生鲜供应链平台（如美菜、快驴等）或本地多家小型供应商签约，每日或隔日由平台按各自门店的小额订单统一分拣配送。  
   - **优势**：按需下单，灵活应对单店波动，单店量小也能享受配送服务；供应商多，可相互比价。  
   - **劣势**：单均配送成本可能略高于直采，但相比单店自行去批发市场更低。  
   - **注意**：需至少维护2-3家供应商，防止供应链中断。

4. **错位采购：核心单品统一采购+非标品属地分散**  
   - **操作**：  
     - **核心大单品**（如红富士、香梨、柑橘）：总部集中采购或与产地代采商签约，统一配送至区域分仓或直达核心门店，其他门店由分仓配送。  
     - **本地化/时令小众水果**（如当地杨梅、特产瓜类）：由各门店自主联系本地小型批发商或农户，甚至共享供应商资源（A店主推、B店可少量拼单）。  
   - **优势**：核心品控严、成本优；非核心品保持灵活性，避免“一刀切”导致过度损耗。

5. **线上拼单+门店分拣（提升单店平效）**  
   - **操作**：针对周末或节日，在超市自有线上社群/小程序上做拼团（如“3家门店凑100箱芒果”），由总部集中采购，按拼单量分单到各店。  
   - **优势**：通过“预售”提前锁定客流和销量，单店即使量小，也能通过总量提升议价权。  
   - **适用**：爆品或应季水果。

**总结建议**：  
- 控制损耗：单店量小则尽量避免“整箱起批、到店再分”，优先选择按实际销售预估的单品采购。  
- 工具化：使用简单的进销存系统，跟踪每家门店的日均销量和退货率，辅助采购决策。  
- 试点先行：从一个品类（如精品水果或香梨）切换到新模式，跑通流程后再扩展。  

**风险提示**：避免与单一大型供应商签订长期锁量协议（以免滞销），优先选择“按需供货、可调单量”的合作伙伴。
