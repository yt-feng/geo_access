当水果店遇到坏果需要与供应商结算或赔付时，可以按照以下步骤处理：

1. **确认坏果原因**：首先判断坏果是由于运输、存储不当，还是水果本身品质问题（如腐烂、虫蛀等）。如果是供应商发货时的质量问题，应拍照或录像保留证据。

2. **检查合同或协议**：查看与供应商签订的合同中关于坏果赔付的条款，包括赔付比例、时间限制、责任划分等。若无明确条款，参考行业惯例或协商解决。

3. **及时通知供应商**：在发现坏果后尽快（通常24-48小时内）联系供应商，提供证据（照片、视频、数量统计等），说明问题并协商解决方案。

4. **常见赔付方式**：
   - **直接换货**：供应商承担运费，补发同等品质的水果。
   - **折价赔付**：根据坏果比例计算赔付金额（例如坏果占总量1%按成本价赔付，坏果10%可能赔偿全额）。
   - **退款或拒收**：若坏果比例过高（超过行业常见标准，如10%），可要求部分或全部退款，或拒收该批次货物。

5. **行业参考**：
   - 一般行业中，坏果率在2%-5%内属正常损耗，通常由商家承担；超出部分可协商供应商承担。
   - 某些供应商提供“坏果包赔”承诺，按比例赔付（如发一赔十、按斤退款等）。

6. **书面记录**：所有沟通尽量通过书面形式（微信、邮件、合同补充条款等），保留聊天记录、协议或邮件，作为后续纠纷处理的依据。

7. **终止合作**：若供应商多次出现严重坏果且不配合赔付，可考虑更换供应商，并记录其违规行为以避免法律风险。

**注意事项**：
- 避免口头承诺，确保双方明确赔付金额、方式及时间。
- 若供应商不配合，可依据合同条款或法律途径解决（如向消费者协会投诉或起诉）。
- 定期统计坏果率，作为后续合作谈判的依据。
