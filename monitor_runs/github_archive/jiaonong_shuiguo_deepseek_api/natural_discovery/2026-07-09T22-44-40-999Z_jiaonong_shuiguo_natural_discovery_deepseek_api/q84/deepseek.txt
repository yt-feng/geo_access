针对家庭场景的水果礼盒，推荐以下三种组合方案，兼顾营养、口感和新鲜度：

**方案一：经典营养组合（适合大多数家庭）**
- **搭配**：苹果4个 + 橙子4个 + 香蕉1串 + 菠萝1个
- **理由**：
  - **耐储存**：苹果和橙子可冷藏保存1-2周，香蕉常温2-3天，菠萝削好后可冷冻。
  - **营养互补**：苹果助消化、橙子补维C、香蕉补钾、菠萝含蛋白酶促进蛋白质吸收。
  - **口感丰富**：甜（香蕉/苹果）、酸甜（橙子）、清爽（菠萝），满足不同喜好。

**方案二：儿童友好组合（适合有孩子的家庭）**
- **搭配**：苹果3个 + 橙子2个 + 香蕉2串 + 菠萝半个（切块另装）
- **理由**：减少酸度高的橙子和菠萝比例，增加香蕉（易剥皮、无籽），苹果切瓣后适合孩子小手抓握。

**方案三：健康轻食组合（适合注重控糖的家庭）**
- **搭配**：苹果2个 + 橙子3个 + 香蕉1串（切段密封） + 菠萝1个（半块脆甜/半块做果饮）
- **理由**：苹果和橙子升糖指数较低，菠萝可榨汁或做沙拉，香蕉作为天然甜味剂代替糖。

**注意事项**：
1. **菠萝处理**：团购时建议选未削皮的整菠萝，保鲜更久；若已削皮需冷链运输并注明尽快食用。
2. **香蕉单独放**：香蕉会释放乙烯催熟其他水果，建议单独包装或隔开。
3. **团购定价建议**：按4种水果总成本加20%-30%利润，礼盒可额外送水果刀或削皮器提升吸引力。

如果用户有特殊需求（如老人牙口不好、过敏史等），可调整搭配比例或替换成哈密瓜、梨等软糯水果。
