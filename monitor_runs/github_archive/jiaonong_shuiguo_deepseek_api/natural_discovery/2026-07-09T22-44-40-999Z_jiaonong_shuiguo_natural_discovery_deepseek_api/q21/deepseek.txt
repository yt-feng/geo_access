在超市采购水果时，供应商通常需要提供以下检测报告或进货证明，以确保产品符合食品安全和质量标准。具体要求可能因地区、超市类型及水果种类（如进口或国产）有所不同，但以下清单涵盖常见要求：

1. **水果产地证明**：证明水果来自合法种植基地，可能包括产地检疫合格证或产地证明文件。
2. **农产品质量安全检测报告**：由有资质的第三方检测机构出具，检测项目包括农药残留、重金属、微生物等，需符合当地（如中国GB 2763-2021《食品安全国家标准 食品中农药最大残留限量》）或国际标准。
3. **植物检疫证明**：特别是跨省或进口水果，需提供由农业农村部门或海关出具的植物检疫证书，确保无病虫害。
4. **进口水果报关单及入境货物检验检疫证明**：如水果为进口，需提供海关签发的报关单、入境货物检验检疫证明（包括产地、种类、数量、检疫结果等）。
5. **生产或加工企业资质证明**：如水果经过初加工（例如清洗、分拣），需提供加工商的食品生产许可证或营业执照副本。
6. **批次合格证明**：每批水果附带的生产批次号、采摘日期、保质期等信息，以及供应商内部检测报告（如糖度、成熟度等）。
7. **运输与储藏记录**：证明水果在运输和储存过程中符合温控、卫生标准，可能包括温度记录、运输车辆卫生证明。
8. **绿色/有机/无公害认证（如适用）**：如果水果标称为有机或绿色产品，需提供相应的认证证书及年度检测报告。

**建议**：
- 向供应商索取上述文件的原件或清晰复印件，并定期核对有效期。
- 保留采购合同、发票、送货单等进货凭证，建立档案以备食品安全检查。
- 对于高风险水果（如草莓、葡萄等易残留农药种类），可要求附带更详细的检测报告。
- 不同超市可能要求额外条款（如供应商质量保证函），建议提前与采购部门确认具体清单。

如果你需要特定国家或地区的详细标准，可进一步咨询当地市场监管或农业部门。
