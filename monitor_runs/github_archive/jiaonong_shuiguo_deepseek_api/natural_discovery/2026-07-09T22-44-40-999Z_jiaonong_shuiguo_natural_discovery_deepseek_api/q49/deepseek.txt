针对茶饮店旺季菠萝用量突然增加，建议按以下步骤与供应商高效沟通并签订临时补货协议：

1. **明确需求与时间**  
   立即统计所需菠萝的**具体数量**（如：每日额外需求200斤）及**补货周期**（如：未来3周每周一、四到货）。明确是持续性补货还是单次应急。

2. **紧急联系供应商**  
   - 通过电话/微信快速沟通，说明“旺季销量激增，需临时补货，价格和配送可协商”。  
   - 优先联系长期合作供应商，强调“后续旺季订单继续合作”。

3. **协商关键条款**  
   - **价格条款**：要求维持原价或适当上浮（如+5%~10%），避免对方大幅提价。可提出“本次补货按原价，但单量超过1000斤部分给予98折”。  
   - **品质标准**：明确菠萝的新鲜度（如：无腐烂、无霉变，果体成熟度70%-80%）、规格（如：单个重1.2-1.5斤）。  
   - **交货与验收**：约定交货地点（如：门店或中央仓库）、验收时段（到货后1小时内检查），以及**不合格品退换**处理方案（如：2小时内补货或立即退款）。  
   - **付款方式**：紧急补货可先付30%定金，尾款验收后3日内结清；或争取月结30天。  

4. **书面确认（重要）**  
   口头约定后，立即通过微信/邮件发送正式补充协议，内容包括：  
   - 补货数量、单价、总金额（含税）  
   - 到货时间表和违约责任（如：延迟到货赔偿5%货款）  
   - 双方签字确认（可先电子版回复“同意”记录）  

5. **风险管控**  
   - 询问供应商的库存深度：是否需从外地调货、能否确保充足供应。  
   - 备选方案：同步联系1-2家备选供应商作为“期货备份”，约定若主供中断，24小时内启动备用供货。  

**示例沟通话术**：  
“王总，菠萝最近日均销量翻倍，紧急需要每周额外200斤，想确认您这边是否能按原价加供？如果本次补货稳定，后续可优先长期合作。我们将在1小时内发邮件确认具体条款（数量、到货时间、验收标准），请查收后回复确认。”

**关键提醒**：立即行动！旺季补货时间紧迫，需在1个工作日内完成谈判和书面确认，避免断货。
