为批量配送选择水果供应商时，建议优先考虑以下标准：

1. **资质与口碑**：选择有营业执照、食品流通许可证的供应商，查看企业客户案例或线上评价，优先考虑长期为企业提供服务的供应商（如美菜、快驴、每日优鲜等B端平台）。

2. **水果品种与新鲜度**：选择时令水果（如夏季的西瓜、葡萄，冬季的柑橘、苹果），避免易损伤品种（如草莓、芒果）。要求供应商提供当日或前日采摘证明，并支持到货抽检。

3. **配送能力**：确认是否能按指定时间（如下午茶前）送达，覆盖公司区域，并提供冷链运输。可要求试用一次小批量配送，测试准时率和水果损耗率（正常损耗应低于5%）。

4. **包装与售后**：批量配送需用防撞泡沫箱或分隔纸箱，每份独立包装。明确破损、腐烂水果的退换方案（如24小时内拍照补发）。

5. **价格与结算**：批量采购单价通常比零售低15%-30%，但需对比3家以上报价。选择支持月结或按需结算，避免预充值。

建议您先列出公司人数、预算和常用水果清单，联系2-3家本地供应商进行试送，对比后确认长期合作方。
