从损耗角度来看，**直接买成品果丁**通常损耗更低，但需综合考虑成本、新鲜度及使用场景。具体分析如下：

1. **鲜果切配的损耗**  
   - **去皮去芯**：菠萝可食部分一般为果重的55%-65%（具体因品种和成熟度而异），即1斤鲜果切配后约剩0.55-0.65斤。  
   - **切配损耗**：若切配技术不熟练或储存不当（如切后氧化、水分流失），可能额外增加5%-10%损耗。  
   - **时间成本**：切配耗时，且需专人操作。

2. **成品果丁的损耗**  
   - **包装损耗**：成品果丁已去皮去芯，直接使用，无额外切配损耗。  
   - **储存损耗**：若未及时用完或冷藏条件差，可能因出水、变质产生3%-5%损耗。  
   - **品质稳定性**：大品牌成品果丁一般经过标准化处理，成熟度均匀，可减少因果品好坏不一导致的浪费。

**结论**：如果没有高技能切配人员且果品采购不稳定，建议优先选成品果丁；若甜品店日均用量大、有专业切配经验且果品采购价格低廉，鲜果切配可能更经济，但需严格监控损耗率。
