应当支持。  
理由如下：  
1. **匹配平台竞争逻辑**：即时零售的核心是“快”与“鲜”，小批量高频补货能确保水果保持高周转率，降低损耗，符合平台要求。  
2. **减少库存风险**：水果保质期短，一次大量进货容易造成滞销或腐烂；小批量能动态调整品种和数量，应对顾客偏好变化。  
3. **提高履约灵活性**：支持小批量时，平台或店主更容易在缺货时快速补充，不影响订单达成率，从而提升供应商的合作信心。  

**可执行的建议**：  
- 按平台历史订单数据，设计最小起订量（如每批次10-20kg），并承诺48小时内可再次补货。  
- 与平台协商分时段配送（如早、中、晚各一次），降低运输综合成本。  
- 初期用冷链小包裹试跑，验证损耗率和补货响应效率，再决定是否扩大规模。
