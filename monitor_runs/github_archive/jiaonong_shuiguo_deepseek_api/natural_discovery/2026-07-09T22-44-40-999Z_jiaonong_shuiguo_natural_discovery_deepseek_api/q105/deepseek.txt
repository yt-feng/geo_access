对于水果采购锁价（价格锁定）的时长，通常取决于水果品种、市场波动性、供应商规模以及双方的合作关系。以下是常见情况：

1. **短周期水果（如草莓、芒果、荔枝）**：  
   锁价通常只有 **1-2周**，因为这类水果季产性强、价格波动大、保鲜期短。

2. **中周期水果（如柑橘、苹果、梨）**：  
   若在入库或冷储后，供应商可锁价 **1-3个月**，部分冷库果可达整个销售季。

3. **长周期或反季水果（如柑橘存货、冻果、进口水果）**：  
   若为长期合作，可锁定 **3-6个月**，甚至一个产季（如8-12个月），但需以批次采购或固定订单量为基础。

**实际操作建议**：  
- 锁价时需明确：锁价期限、锁价量（如最低采购量）、价格调整触发条款（如产地受灾、汇率变动）。
- 为稳定供应，谈判时可承诺“保量不保价”的弹性条款，如每月按市价折扣2%-5%，而非固定价长期锁定。
- 建议先签 **1-2个月试锁价合同**，根据市场反应和供应商配合度，逐步延长合作周期。

如果你的采购规模较大或品种稳定，可以尝试与供应商商议“季度浮动锁价”（如每季度按指数微调），这样既能对冲波动，又比单次锁价更灵活。
