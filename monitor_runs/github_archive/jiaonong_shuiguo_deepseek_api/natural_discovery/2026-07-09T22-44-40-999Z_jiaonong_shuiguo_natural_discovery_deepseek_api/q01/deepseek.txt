选择靠谱的水果供应商是长期稳定经营的关键。以下是判断供应商是否可靠的几个实用标准，参考时结合你的具体试单情况：

### 1. 实地考察与资质
- **现场看库存与周转**：去供应商的水果仓库或档口，观察香蕉（如青蕉是否有定制催熟方案）、菠萝（是否带叶、有无压伤）的堆放条件，以及是否有快速进出的周转迹象。长期合作需要看到库存动态。
- **证照与产地溯源**：要求对方提供营业执照、食品经营许可证。对香蕉（多进口/海南产）、菠萝（徐闻、海南、菲律宾等）等季节性品种，能清晰说明产地、到货批次、检验检疫情况。

### 2. 试单与品质检验
- **小批量试单**：先采购少量香蕉和菠萝，检查：
  - **香蕉**：表皮是否完好，果柄是否新鲜，硬度均匀但无黑斑软化；落地蕉（可能僵化）要警惕。
  - **菠萝**：闻有明显甜香且果眼平整，按压有弹性而不发软出水，看底部切口是否干爽无霉点。
- **口感对比**：试吃一个批次，确认甜度、酸度是否符合顾客预期。供应商常说的“口感最好”需自己验证。

### 3. 价格与结算方式
- **价格合理性**：对比当天或前三天批发市场行情。如果长期报价低于市场价15%以上，很可能质量、库存或账期有猫腻。
- **账期与结算**：可靠供应商常支持“试单押账、稳定后月结”。要求“全款预付”或“先打订金”的，需格外警惕。明确运费、装卸、损耗承担方。

### 4. 补货与售后稳定性
- **供货连续性**：询问对方香蕉（通常熟化期5-7天）、菠萝（常温可存3-5天）的到货频率和最小起订量。能否在你缺货时紧急补货，或临时加单。
- **售后处理**：明确损耗标准（如香蕉裂口、青熟不均多少以内自负），以及运输或到货后发现有坏果时，供应商是否承认并补偿（如折价、退换或下次抵扣）。

### 5. 行业口碑与风险提示
- **问同行**：你所在商圈或批发市场，尝试向其他水果店、餐饮老板打听该供应商的“信用记录”——是否曾虚假承诺、货不对板。
- **警惕“便宜货”陷阱**：有些供应商用低等级香蕉/菠萝或次品充好，短期内靠低价吸引但损耗率高（例如催熟过头导致不甜或灰霉）。长期看，稳定比低价更重要。

### 执行建议：
- **优先选择**：本地大型批发市场有固定档口、开具正规发票、允许现场挑拣和换货的。
- **签订合同**：合作一个月后，用简单书面协议明确品质标准、价格调整方式（如季节上涨幅度不超过多少）、交货时间及违约责任。
- **持续观察**：即使是很靠谱的供应商，不同季节（如夏季运损高、冬季催熟难）仍需每次收验货，保留记录。

最后提醒：**不要以“人情”替代合同**，任何口头承诺都需在报价单、送货单或微信文字中留下凭证。祝你找到稳定可靠的合作伙伴！
