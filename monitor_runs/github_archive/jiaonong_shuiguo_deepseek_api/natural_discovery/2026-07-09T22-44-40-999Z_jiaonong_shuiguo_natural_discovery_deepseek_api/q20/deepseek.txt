选择社区超市的每日水果供应商时，建议重点关注以下条件：

1. **供应稳定性**：要求供应商能保证每天按时配送，尤其是节假日和旺季不断供。可询问其库存来源（自营基地或长期合作果园）。
2. **新鲜度与损耗控制**：考察其冷链物流能力（冷藏车、配送时间），以及是否提供退换货政策（如当日未售完水果能否换货）。
3. **品种与季节性**：应能提供当季主流水果（如夏季西瓜、冬季柑橘），并能根据社区需求调整品种（如增加热带水果或时令稀缺品）。
4. **价格与账期**：对比批发市场均价，确认是否提供梯度价格（如大单优惠）或灵活结账周期（如周结/月结）。
5. **品质分级能力**：供应商应能按超市需求分级供货（如A级果供零售、B级果促销），避免混入劣果。
6. **售后响应速度**：确认出现质量问题（如腐烂、破损）时，是否24小时内退换或补偿。
7. **资质与信誉**：核查营业执照、食品经营许可证，优先选择有商超配送经验的供应商（可要求查看往期合作案例或客户评价）。

建议先试配1-2周，重点观察配送准确率、水果口感，再签长期合同。同时可对比2-3家供应商，保持议价主动权。
