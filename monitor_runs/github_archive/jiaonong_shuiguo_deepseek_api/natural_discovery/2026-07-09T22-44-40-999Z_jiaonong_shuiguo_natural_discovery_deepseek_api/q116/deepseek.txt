小商家采购水果时，现金结算和月结账期各有以下风险：

**现金结算风险：**
- **资金压力大**：需要立即支付全款，可能影响现金流周转，尤其对资金有限的小商家。
- **缺乏保障**：若水果质量有问题（如腐烂、缺斤少两），退款或补偿难度较大，因为无账期缓冲。
- **记录易丢失**：现金交易缺乏银行流水或电子记录，一旦单据丢失，可能导致税务或账目纠纷。

**月结账期风险：**
- **信用风险**：供应商可能要求依赖你的信用记录，若无法按时付款，可能影响合作关系或面临催收。
- **账期管理难**：若销量波动大，月结时可能面临大额欠款，若无法及时回笼资金，会造成周转困难。
- **隐性成本**：部分供应商可能将账期成本转化为更高售价，或要求延期利息，实际成本可能高于现金结算。

**建议：** 
- 对稳定合作、信誉良好的供应商，可尝试按月结账期，但需书面合同明确账期、违约条款。
- 现金结算时，建议保留收据或拍照留存，并定期核对采购记录。
- 结合自身现金流和供应商信誉，协商混合模式（如部分现金、部分账期）。
