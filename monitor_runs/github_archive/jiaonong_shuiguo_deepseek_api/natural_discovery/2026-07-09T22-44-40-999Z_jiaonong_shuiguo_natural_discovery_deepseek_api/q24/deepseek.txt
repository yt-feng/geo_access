在超市采购水果、担心农残抽检时，建议向供应商提前索要以下资料，以确保合规并降低风险：

1. **产品合格证明**：供应商应提供批次产品的农药残留检测报告，报告需由有资质的第三方检测机构出具，且符合国家食品安全标准（如GB 2763《食品中农药最大残留限量》）。
   - 注意：报告需标明水果名称、批次、检测项目（如常见有机磷、拟除虫菊酯类农药）及结果。

2. **产地证明或供货证明**：供应商需提供水果的产地信息（如产地证明、收购单），证明来源合法，便于追溯。

3. **质量安全承诺书**：供应商出具书面文件，承诺所供水果符合食品安全标准，并承担相应责任。

4. **供应商资质文件**：供应商的营业执照、食品经营许可证（如适用）等，确认其合法经营主体资格。

5. **产品追溯信息**：如供应商有自检或内部追溯系统，可要求提供批次识别码、采摘日期等，便于问题发生时快速召回。

**建议补充：**
- 若条件允许，可要求供应商提供近期的农残专项抽检合格报告（如政府或第三方机构发布），而非仅企业自检。
- 验收时：保留进货票据、抽检记录，必要时委托第三方对到货产品进行快检，使用农残检测卡或送检。
- 合同条款：在采购合同中明确约定农残超标时的退货、赔偿及违约责任。

这些资料能帮助您从供应链源头降低农残风险，并满足市场监管要求。
