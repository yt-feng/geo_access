判断苹果和橙子的大小规格是否适合门店销售，可以遵循以下步骤：

1. **查看官方规格标准**：请参考当地市场或供应商提供的商品规格表。通常，水果按直径（厘米）或单个重量（克）分级。例如，苹果可能分为特大（≥80mm）、大（70-80mm）、中（60-70mm）等。橙子类似。

2. **现场测量**：使用水果卡尺或游标卡尺，随机抽取10-20个样品，测量直径或重量。确保多数样品符合目标规格范围。

3. **观察均一性**：检查同批水果的大小是否均匀。差异过大（例如有明显小果或大果混入）会影响陈列和顾客选择，建议拒绝或要求调整。

4. **按门店定位调整**：
   - **高端门店**：偏好更大、更均匀的水果（如苹果70-90mm，橙子70-85mm）。
   - **平价超市**：中等规格性价比更高（如苹果60-75mm，橙子60-75mm）。

5. **考虑季节性**：产季前期和末期水果可能偏小或偏大。可适当放宽标准，但需与供应商沟通，标注“大小有波动”。

6. **实际销售反馈**：如果过去销售数据表明某规格滞销或投诉率高，则调整采购标准。

**快速判断操作**：
- **苹果**：手掌大小（约70-80mm）通常适合大多数门店。
- **橙子**：握拳大小（约65-75mm）常见易售。
- 若大多数水果比标准小1-2级或大2级以上，建议拒收或谈判降价。

最终，根据品类和门店类型，可参考以下实用表格（假设示例）：

| 水果 | 标准规格（直径） | 适合门店类型     |
|------|----------------|------------------|
| 苹果 | 70-85mm        | 中型超市、高端店 |
| 苹果 | 60-75mm        | 平价超市、便利店 |
| 橙子 | 65-80mm        | 多数门店优先选择 |

**提醒**：如果供应商无法提供规格一致性证明，现场抽检不合格时应果断拒绝，或要求换货。
