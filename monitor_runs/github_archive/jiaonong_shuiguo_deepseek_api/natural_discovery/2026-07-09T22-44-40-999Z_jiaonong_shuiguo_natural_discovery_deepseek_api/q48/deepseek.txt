判断茶饮连锁的菠萝供应商能否稳定按规格送货，可以从以下几个关键环节进行考察和验证：

1. **明确并量化规格标准**：
   - 先制定清晰的采购规格书，包含：成熟度（如七成熟）、大小（单果重量或直径范围）、糖度（Brix值）、色泽、果形、无病斑/机械伤等。要求供应商明确标注其能达到的批次内果径、重量公差（如±5%）。
   - 可要求提供样品批次内，各规格菠萝的实际分布数据，评估批量一致性。

2. **实地考察与过程审核**：
   - 实地查看供应商的种植基地、分选线、冷库和包装环节。尤其关注是否有分级设备（如滚轴式大小分级机）和自动称量设备，这是稳定按规格分拣的基础。
   - 询问其是否建立田间采摘标准、分级操作流程（SOP）以及品质检验记录。要求查看过去3-6个月的发货规格统计（如批次内大小分布、次品率）。

3. **小批量试单与连续跟踪**：
   - 先下达3-5个小批量试单（每次10-20箱），覆盖不同时间段（如月头、月中、月末）。记录每次到货的规格合格率（合格箱数/总箱数）、果径波动范围、果形均匀度、运输损耗率。
   - 若连续3次试单合格率在95%以上，则初步稳定。重点观察供应商是否随季节有规格波动（菠萝产季中后期大小差异可能加大）。

4. **合同条款的约束与验证**：
   - 在合同中明确规格验收标准（如：单果重在1.2-1.5kg占≥90%），并约定“未达标”的处理方式：如退货补货、扣款、扣减信用分等。优先选择承诺“超规比例可接受”的供应商。
   - 要求供应商提供批次随货单（包含采摘日期、分选结果、冷链温度记录），以便追溯。

5. **气候与产量风险预判**：
   - 询问供应商是否建立“多产区备货”机制（如国内海南/云南/广东轮换，或进口补充）。遭遇台风、霜冻等天气时，能否快速切换产地保证规格。
   - 要求提供过去2年内因气候原因导致规格波动的历史报告，评估其抗风险能力。

6. **反向验证与长期合作**：
   - 抽测2-3次大货到货，与试单数据进行对比。若发现大货规格明显劣于试单，说明其存在“样品优化”问题。
   - 长期合作时，建议双方定期（如每季度）共同复盘规格数据，建立“合格率阶梯奖励”（如合格率>98%给予+2%采购量奖励），强化供应商的稳定性动机。

**总结**：稳定性的核心在于供应商是否有系统化的分选设备和流程。优先选择拥有自动化分级线（而非依赖人工目测）的供应商，同时通过多批次试单和合同约束锁定其行为。如果供应商只提供口头承诺，没有分级数据和设备，则风险较高，建议谨慎合作。
