社区超市卖小包装香蕉，主要需注意以下几点：

**一、包装问题**
1. **包装材料**：优先选择透气性好的食品级塑料袋或带孔薄膜，避免香蕉快速熟化或霉变。如果有条件，可用可降解纸袋搭配塑料托盘，既美观又减少乙烯堆积。
2. **包装规格**：通常2-4根/包（0.6-1斤），适合即食或短期储存。太大会导致顾客嫌多，太小则增加包装成本。
3. **标签设计**：清晰注明“此份可放2-3天”“轻拿轻放”等提示，避免顾客因挤压导致损耗。
4. **包装时机**：建议在香蕉七八成黄（带微绿）时包装，完全成熟后包装易在运输中擦伤。

**二、损耗控制**
1. **成熟度筛选**：现场挑出表皮有黑点、软塌的香蕉做成“特价半熟”或“鲜切果杯”，主货架只放七至九成熟的。未完全成熟的香蕉（微青）可单独作为“回家放1天吃”的选项。
2. **包装内防碰**：在包装底部垫一层软纸或泡沫网，尤其是多列包装时。香蕉弯面朝上放置，避免果指互相压伤。
3. **存储与周转**：到货后立即拆箱通风，去除乙烯释放过快的微熟或损伤果。小包装香蕉货架期通常只有1-2天，建议每天早晚各检查一次，变软的立刻下架或折价处理。
4. **陈列方式**：尽量使用悬挂式陈列（挂钩或开放篮），减少顾客翻捡导致触伤。若平铺，每层只放单排包装，严禁叠压。

**三、采购谈判时注意**
- 要求供应商配备标准纸箱（每层带隔板），避免长途颠簸产生压伤。
- 明确损耗补偿：可协商如到货实际损耗超过3%，供应商承担部分或补充免费替换包。
