可以谈。水果锁价在餐饮行业中是常见的操作方式，但需要注意以下几点来实现可行性：

1. **锁定价格周期**：通常可以和供应商约定一个月、一季度或更短周期的固定价格（如每周更新）。太长周期（如全年）难度大，因为水果价格受季节性、天气、运输等因素影响较大。

2. **锁定品种和规格**：明确具体是哪种水果（如红富士苹果、海南金钻凤梨），以及果径、糖度、新鲜度等质量标准。避免供应商用次果替代。

3. **确定锁价面积/用量**：和供应商协商一个最低采购量，方便对方备货和承担价格风险。例如：每周固定采购200斤，如果实际采购不足，可能需要补差价。

4. **附加条款**：
   - **价格调整机制**：约定当市场行情波动超过10%-15%时，双方可重新协商价格，避免供应商因亏损过度而断供。
   - **质保条款**：锁价期间供应商需保证水果品质，如果出现腐烂、规格不符等情况，可要求退换货或降价。
   - **免责条款**：如遇极端天气、产地减产等不可抗力，双方可免责并终止锁价。

5. **实际操作建议**：
   - **与规模大的供应商谈**：大型批发商或基地直采供应商更有能力接受锁价。
   - **明确合作期限**：先尝试1-2个月的短锁定期，验证双方合作效果。
   - **书面合同**：一定要签订正式协议，注明有效期、价格、数量、违约责任、争议解决方式。

如果供应商不愿意接受锁价，可以退一步谈**价格参考机制**，比如约定“按批发市场均价下浮5%”，这样也能降低成本波动风险。

总结：可以谈锁价，但建议控制锁定期、明确标准、提前设定调整规则，同时保留随时协商的权利。
