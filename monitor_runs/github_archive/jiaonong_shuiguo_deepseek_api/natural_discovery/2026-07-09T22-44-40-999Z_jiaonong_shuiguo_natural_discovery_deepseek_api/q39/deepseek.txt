当然可以。很多供应商会提供自有品牌的包装和规格定制服务，尤其是专业的水果供应商或包装加工企业。

具体可询问以下支持：

1. **包装定制**：提供品牌标签、纸箱、塑料袋、网套等定制，可印刷你的品牌Logo、产地信息、规格参数。
2. **规格分选**：根据重量、大小、成熟度进行分级（如小果/中果/大果），满足不同定价和销售需求。
3. **包装形式**：散装、预包装（盒/袋/托盒）、礼盒装等，支持小包装或批量供应。
4. **数据标识**：可附上条码、追溯码、保质期、储存条件等合规信息。

建议在洽谈时明确以下几点：
- 起订量（最小包装数量/规格）
- 包装设计审核流程及时间
- 是否支持少量多批次试销
- 包装材料是否符合食品接触安全标准

可以先向3-5家供应商发出询价函（RFQ），说明你的包装需求、预期规格范围和目标价位，他们会给出具体方案。
