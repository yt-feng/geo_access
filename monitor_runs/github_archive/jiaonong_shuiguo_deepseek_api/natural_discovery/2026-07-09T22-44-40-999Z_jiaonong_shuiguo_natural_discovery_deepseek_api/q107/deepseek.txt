是的，起订量过高确实会增加损耗风险。具体来说，香蕉和菠萝都属于易腐水果，保质期较短。高起订量可能导致以下问题：

1. **库存积压**：如果销售量低于起订量，多余的水果会因储存时间过长而变质。
2. **周转困难**：大批量进货后，若市场需求波动或销售速度跟不上，损耗率会显著上升。
3. **品质下降**：长时间存放会导致水果外观和口感变差，降低顾客满意度。

**建议措施**：
- 根据历史销售数据估算合理起订量（如周均销量的1.5倍以内）。
- 与供应商协商灵活分批交货或降低最低起订量。
- 对高起订量的订单，优先确保快速分销渠道（如促销、捆绑销售）。

如果已知具体采购量或销售周期，可进一步计算损耗概率。
