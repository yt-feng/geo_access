首次采购水果，建议采用“小批量、多品种、快验证”的方式。以下是具体步骤：

1. **确定试单用量**  
   每种水果先订 **1-2箱**（约20-30斤），不要超过3箱。这样既能检验品质，又不会积压资金。

2. **寻找2-3家供货商对比**  
   - 去当地 **一级水果批发市场**（如北京新发地、广州江南市场），直接找档口老板。
   - 优先找 **产地直发**或 **一级批发商**，成本更低。注意问清楚是否包运费、是否提供破损赔付。

3. **试单关键检查点**  
   - **香蕉**：选七成熟（青皮但果身饱满），避免催熟过度的。到货后放常温1-2天自然转黄，观察是否均匀、有无黑斑。
   - **苹果**：选果面光洁、无苦痘病（黑点）。切开看果肉是否紧实、有无霉心。不同品种（如红富士、嘎啦）耐储存性不同，先试一个品种。
   - **橙子**：选皮薄、有光泽的。切开尝酸甜度和汁水，注意是否有枯水（干瘪无汁）。脐橙、血橙试一箱即可。

4. **记录并评估**  
   - 到货后检查：损伤率（低于5%为优）、颜色、大小一致性。
   - 摆放2-3天后观察：香蕉是否快速变软、苹果是否失水起皱、橙子是否出水发霉。
   - 记下供货商报价、到货时间、包装情况（如竹筐或纸箱），方便下次比对。

5. **风险控制**  
   - 一定要 **当场开箱验货**，有问题拍照留证，当场协商退换或扣款。
   - 首次交易用现金或微信转账，避免预付大额定金。
   - 如果试单后觉得品质一般，别急着下大单，换下一家继续试。

最后提醒：水果保鲜期短，第一次进货建议少量多频，等摸清催熟和储存规律后再逐步加量。
