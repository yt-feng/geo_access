生鲜平台开展水果一件代发业务，对供应商的仓配能力要求较高。以下是核心能力要求，按优先级排序：

1. **产地/销地预冷与分拣能力**  
   - 必须具备**自有或合作的冷库**，能够对采摘后的水果进行快速预冷（如遇冷、压差预冷），延长保鲜期。
   - 具备按规格（重量、大小、成熟度）、品质（无坏果、无伤疤）进行分级分拣的设备或流程，减少平台品控退货率。

2. **包装与保鲜材料能力**  
   - 需使用**抗压、透气、防震**的专业包装（如珍珠棉网袋、泡沫箱、纸箱+冰袋组合），确保运输中减少挤压和温度波动。
   - 应根据不同水果特性（如草莓需防震、芒果需防冻、荔枝需冰袋）提供差异化的内包装方案。

3. **冷运/冷链配送到仓或中转站的能力**  
   - 必须与**稳定冷链物流商**合作（如顺丰冷运、京东冷链），或自有冷链车辆，确保从仓库到快递网点/平台中心仓全程温控（如0-4℃或按品类设定）。
   - 能实现**T+1或T+2发运**，即用户下单后次日或第三日完成采摘、打包并交付快递揽收。

4. **库存管理与批次追溯能力**  
   - 需具备**WMS系统或简单库存台账**，能实时监控库存量、库存天数（水果保质期短，避免超卖）。
   - 能提供**批次码或生产日期标识**，方便平台或用户追溯对应产地、采摘日期，支持售后维权。

5. **异常处理与快速补发能力**  
   - 需有**售后响应机制**：如果运输中出现腐烂、损坏，供应商能在24小时内响应补发或赔付。
   - 建议预备**5%-10%的备用货源**，应对突发爆单或坏果率过高情况。

6. **符合平台标准的订单处理能力**  
   - 必须支持**电子面单打印、批量打印**，并能按平台要求（如贴对应商品条码、大货标签）准确发货。
   - 能够对接平台的API接口（如推送发货单号、反馈物流状态），或至少有高效的手动操作流程（如ERP软件辅助）。

**补充建议**：
- 如果是首次合作，可以从**少品类、小范围测试**开始（如先试发单一水果，如脆桃），验证其包装稳性、时效和坏果率。
- 可要求供应商提供**仓储基地实地视频**或**第三方质检报告**（如冷库温度达标证明），降低合作风险。
