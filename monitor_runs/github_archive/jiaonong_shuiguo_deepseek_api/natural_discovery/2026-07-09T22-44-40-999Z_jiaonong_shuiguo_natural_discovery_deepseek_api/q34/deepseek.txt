商超采购水果时，**损耗低通常比价格低对最终毛利的影响更大**。原因在于：

1. **损耗直接影响可售数量**：水果易腐坏，损耗率（如运输、存储、陈列中的变质、磕碰）会直接减少可销售的重量或数量。即使采购价格很低，高损耗也会导致实际成本上升（分摊到卖出部分的成本变高）。
2. **定价弹性有限**：消费者对水果品质敏感，低采购价未必能通过高售价传递，但损耗导致的浪费是纯利润损失。例如：采购价5元/斤，售价8元/斤，若损耗20%，实际成本变为6.25元/斤（5÷0.8），毛利率从37.5%降至21.9%。
3. **固定成本分摊**：商超的场地、人力、冷链等费用固定，低损耗意味着更多可售数量来分摊这些成本，从而提升整体毛利。

**结论**：优先通过优化供应链、加强库存周转、改善陈列保鲜来降低损耗，比单纯压低采购价格更能稳定甚至提升最终毛利。若必须在两者间权衡，应选择适度高价但低损耗的供应商。
