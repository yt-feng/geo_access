选择学校食堂水果供应商时，建议重点关注以下几点：

1. **资质与安全**：优先选择具有食品经营许可证、卫生许可证的供应商，要求提供水果检测报告（农残、重金属等），确保符合《学校食品安全与营养健康管理规定》。

2. **供应稳定性**：考察供应商是否有稳定的产地或合作果园，能否保证每日配送新鲜水果（尤其是香蕉易熟烂，需要控制成熟度）。

3. **价格与成本**：对比多家供应商报价，同时关注：起送量、运费、退换货政策。建议使用“统采分送”模式（统一采购、分校区配送）降低物流成本。

4. **品类与品质**：常见水果（香蕉、苹果、橙子）需确认：  
   - 香蕉：7-8成熟带青皮（适合校内周转3-5天）  
   - 苹果：优选冰糖心、红富士，大小均匀（建议80#以上）  
   - 橙子：当季，果肉紧实无干水（避免夏橙偏酸）

5. **试吃与合同**：先小批量试订（如100公斤），检查实际配送品质。合同中明确：  
   - 退换货：出现腐烂、畸形可当场退换  
   - 付款周期：可月结或分批付款，避免预付款风险

6. **参考渠道**：通过当地农产品批发市场（如北京新发地、上海江桥）或“阳光采购平台”（如教育局推荐的供应商名录）筛选。

建议优先选择学校周边3小时物流圈内的供应商，便于快速处理突发问题。
