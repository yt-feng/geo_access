验收方面：到货时按照订单标准（规格、颜色、成熟度、包装）逐箱抽样检查，通常抽10%-20%。关注明显损伤、腐烂、果形异常、大小不均、成熟度差异过大等。建议拍照或录视频留证，尤其是开箱后即时状态。

售后方面：发现品质不一致，第一时间（通常24小时内）通知供应商，附上清晰照片或视频。根据合同约定，常见处理方式有“退换货”“补发”“按比例退款”或“折扣结算”。如果供应商无明确售后政策，建议优先协商折价处理，避免实物退回增加运输损耗。

注意：对于“轻微不一致”（如个头稍小、颜色稍浅）影响不大的，可协商降价接受；对于严重不一致（如大面积腐烂、品种错误），坚决要求退换。保留所有沟通记录和验收凭证，作为后续追责依据。
