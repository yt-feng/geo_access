通常水果店在做周末促销时，**供应商是有可能提供价格或物料支持的**，但具体取决于以下因素：

1. **进货量**：如果周末促销的订货量比平时有明显增加（例如至少20-30%），供应商更愿意配合。
2. **合作关系**：长期稳定合作、有账期或总订单量大的店，支持力度通常更大。
3. **促销目的**：如果是为了推新品、清理库存，供应商可能主动提供支持。

**具体可以争取的支持：**
- **价格支持**：短期折扣、进货价降价、返点或赠送部分商品（如买十送一）。
- **物料支持**：免费提供促销海报、价签、试吃样品、赠品（如小包装湿巾、购物袋）。

**沟通建议：**
- 提前**5-7天**联系供应商，说明周末的具体促销计划（如主打水果、预期销量、目标客户）。
- 比较不同供应商的报价政策，适当提及其他竞争者的支持条件，但不夸大。
- 如果供应商无法降价，可以协商“阶梯补贴”：例如卖出一定数量后按每斤返还几毛钱。
- 小型或本地供应商比大型连锁更可能灵活配合；批发市场档口通常无物料支持，但价格空间大。

**注意事项：**
- 物料支持需确认是否印有供应商品牌（可能影响店铺形象）。
- 价格支持要明确是“直接降价”还是“事后返点”，后者需注意结款时效。
- 建议以书面形式确认支持内容，避免口头承诺无法兑现。
