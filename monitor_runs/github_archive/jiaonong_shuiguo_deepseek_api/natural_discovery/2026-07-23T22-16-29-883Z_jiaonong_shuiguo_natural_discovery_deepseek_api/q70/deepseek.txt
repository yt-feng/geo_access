针对批发商担心的坏果损耗责任，建议在合同中明确以下关键条款，以防范风险：

1. **验收标准与期限**：  
   - 约定收货后**24-48小时内**（具体视运输距离调整）为异议期，需开箱查验。  
   - 明确坏果定义：如“单个坏果面积超过30%”、“霉变、腐烂、压伤”等具体情形。

2. **损耗比例责任划分**：  
   - 规定**自然损耗率**（例：≤3%），超出部分由供货方承担。  
   - 明确责任主体：例如“运输途中损耗由物流方/供货方负责”或“因包装不当导致的损耗由供货方负责”。

3. **证据留存义务**：  
   - 要求收货方在发现坏果时，**拍照（含货物批次、日期）、保留原包装**，并同步通知供货方。  
   - 约定“未在期限内提供证据的视为默认验收合格”。

4. **赔偿与结算方式**：  
   - 写明赔偿方案：如“按实际损失数量×单价扣除货款”或“折价部分在尾款中扣除”。  
   - 明确坏果证明或报废后的**样品保留义务**（例如双方共封留样备查）。

5. **特殊免责条款**：  
   - 若因**不可抗力**（如台风导致运输延误）或**批发商自身储存不当**（如未按温度要求存放）导致的损耗，供货方不承担责任。

建议示例条款（可根据实际修改）：  
> **第三条 验收与责任**  
> 1. 乙方（批发商）应在货物到达后24小时内完成验收，若发现坏果率超过3%，须在期限内以照片形式通知甲方，并保留坏果样品。  
> 2. 超出3%的坏果部分，甲方按该批次对应单价从总货款中全额扣除；若未及时通知，视为验收合格。  
> 3. 因物流运输所致损耗，由甲方负责向承运方追偿，但甲方需先行按本条第2款对乙方承担责任。

**补充建议**：  
- 发送前请供货方签字确认上述条款，口头承诺缺乏法律效力。  
- 如货值较高，可考虑购买**货运险**（建议明确保费和受益人）。
