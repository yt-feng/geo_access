挑选香蕉时，可以按以下步骤操作，避免太生或太熟：

1. **观察颜色**  
   - **太生**：表皮整体呈青绿色，尤其是果柄和顶端。  
   - **太熟**：表皮出现大面积黑斑（芝麻点）或发黑、果柄发黑发软。  
   - **理想状态**：表皮鲜黄、略带青边，或有少量棕色斑点（此时口感最佳）。

2. **触摸硬度**  
   - **太生**：果皮硬实，捏不动；果柄硬且与果体连接紧密。  
   - **太熟**：果皮发软，甚至一捏就凹陷；果柄易断裂。  
   - **理想状态**：果肉有弹性，轻捏能感受到轻微软感但不会塌陷。

3. **看果柄**  
   - 果柄呈绿色、粗壮、与果体连接无裂口。  
   - 果柄发黑、干枯或松动，通常意味着储存时间过长。

4. **闻气味**  
   - 有浓郁香蕉甜味（尤其是果柄处）≈ 熟透。  
   - 无味或只有青草味 ≈ 偏生。  
   - 有酒精味或酸味 ≈ 过熟变质。

**采购小建议**：  
- 根据食用速度选择：  
  - 未来1-2天吃 → 选黄色略带青边、有少量斑点的（中性）。  
  - 未来3-5天吃 → 选黄色带较多青绿（偏生一些，室温可自然熟）。  
  - 当天或明天吃 → 选全黄、有少量芝麻斑的（熟度刚好好）。  
- 尽量避开堆在底层或角落里、被其他水果压到的香蕉，这些容易碰伤腐烂。

**到家后处理方法**：  
- 若买回来较生，用报纸包好放在室温下（不要放冰箱），2-3天即可变熟。  
- 若买回来已经偏熟，立即放入冰箱冷藏（表皮会变黑但果肉可多保存1-2天）。
