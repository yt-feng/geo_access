为社区超市挑选每日配送的水果供应商，建议重点考察以下条件：

1. **供应稳定性**：要求供应商能确保全年每日稳定供货，特别是非应季水果的供应能力。
2. **水果品质与新鲜度**：确认有无冷库、冷链车，能否提供次日凌晨采摘、当天送达的“鲜果”服务，以及退换货（如损耗）政策是否合理。
3. **品种丰富度**：需覆盖常备基础款（苹果、香蕉、橙子等）及应季热销款，并可根据社区客户需求灵活调整。
4. **价格与结算**：对比多家批发价或集采价，明确有无阶梯折扣，结算账期是否支持月结或周结。
5. **配送能力**：核实每日是否固定时间配送、节假日前能否加急，以及自提或送货到店的具体范围（通常3-5km内优先）。
6. **资质与检验**：要求提供营业执照、食品经营许可及水果农残检测报告，尤其注意进口水果是否有检疫单。
7. **客诉响应速度**：明确因物流延误、品质问题或数量短缺的补偿方案，以及售后判断时效（如12小时内反馈）。
8. **试合作期**：建议先短期试订1-2周，观察其实际履约准时率、水果损耗率及服务配合度。

可通过附近农贸批发市场、本地农产品对接平台或超市商圈实地走访筛选，优先选择资质齐全、有社区配送经验或提供“分拣称重贴标”一体化服务的供应商。
