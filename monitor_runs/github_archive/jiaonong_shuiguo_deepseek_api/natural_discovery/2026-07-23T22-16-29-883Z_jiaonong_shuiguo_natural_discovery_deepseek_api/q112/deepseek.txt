在水果采购合同中，通常建议将**坏果率超过5%**设定为赔付起始线。具体建议如下：

1. **常规标准**：新鲜水果的行业习惯是坏果率（腐烂、严重机械伤、变质）超过**3%-5%** 时，买方有权要求降价、换货或按比例赔偿。合同可明确约定：  
   - 3%以内：买方自行承担（视为正常损耗）。  
   - 3%-5%：按超出部分金额的1-2倍扣减货款。  
   - 超过5%：可要求整批退货或按比例全额赔付。

2. **易腐水果**（如草莓、蓝莓、荔枝）：建议将赔付线设为**3%**，因为这类水果损耗率更高。  
3. **耐储存水果**（如苹果、橙子、柚子）：可适当放宽至**5%-8%**，但需在合同中明确“坏果”定义（如腐烂、霉变、严重压伤等）。  

**关键动作**：  
- 必须在合同中写明“坏果率超过X%时，卖方需按超出部分货值的X倍赔付”。  
- 约定**验收方式**（如到货24小时内抽样开箱检查，留存照片/视频证据）。  

**注意**：若合同未约定，法律上通常以**行业惯例**为准，但纠纷时举证困难，务必提前书面明确。
