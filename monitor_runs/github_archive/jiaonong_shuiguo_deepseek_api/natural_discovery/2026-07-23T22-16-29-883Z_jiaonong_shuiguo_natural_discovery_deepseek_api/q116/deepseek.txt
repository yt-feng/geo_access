小商家采购水果时，现金结算和月结账期各有以下风险：

- **现金结算风险**：
  - 资金占用压力大：当场支付全款，若采购量较大或季节性资金紧张，可能影响其他周转。
  - 缺少账期缓冲：若水果出现质量问题或损耗（如运输中损坏），需立即协商退款或换货，若对方不配合，损失较难追回。
  - 发票和凭证易缺失：现金交易可能无正式单据，日后出现纠纷时举证困难。

- **月结账期风险**：
  - 供应商信用风险：若供应商资金链断裂或经营不善，可能拖延发货或突然要求提前结款，导致供应中断。
  - 账期成本隐性增加：月结通常隐含利息成本（如供应商会提高单价），实际总支出可能高于现金价。
  - 赊销依赖风险：若长期依赖月结，一旦供应商收紧账期，可能打乱库存计划；同时，需自行垫付损耗、退货等差额，若管理不善易形成坏账。

**建议**：  
- 对信誉良好、合作稳定的供应商，可少量采用月结（如30天），但需签订合同明确账期、违约责任、退货条款。
- 现金结算适合小额、高频或季节性采购，建议保留收据或转账记录，并提前约定品控标准。
- 可混合使用：大宗常用水果用月结，应急或小量用现金，平衡风险与现金流。
