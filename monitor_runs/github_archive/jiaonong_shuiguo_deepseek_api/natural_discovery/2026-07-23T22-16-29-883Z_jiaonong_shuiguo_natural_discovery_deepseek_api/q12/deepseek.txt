采购菠萝时判断熟度，建议通过以下方法综合评估，既能保证顾客买到即食果，又能减少因过熟导致的损耗：

1. **看颜色**：选择**底部和中部为金黄色、顶部略带青绿**的菠萝。全青的未熟，酸涩；全黄或带黑斑的过熟，极易腐烂。

2. **闻气味**：靠近菠萝底部（切口处），**有浓郁果香**为最佳。无气味说明未熟，有酒味或酸味代表过熟或已变质。

3. **按硬度**：用手轻压菠萝**中下部两侧**，**果肉有弹性、稍软但能回弹**为合适。过硬则生，过软或按压处凹陷不回弹则过熟。

4. **看叶子**：挑选**叶子翠绿、易拔落**的菠萝。叶子发黄枯萎说明存放时间久，太紧不易拔落则过生。

**采购策略建议**：  
- 根据销售速度：若两天内能卖完，可拿**八成黄**的；若需存放3-5天，选**六成黄、底部略青**的。
- 避开**表皮有软斑、渗水或裂口**的菠萝，这类极易在运输中进一步腐烂。
- 夏季高温时，选择**稍青一些**的菠萝（如七成熟），到店后常温摆放1-2天即可自然成熟，减少损耗。
