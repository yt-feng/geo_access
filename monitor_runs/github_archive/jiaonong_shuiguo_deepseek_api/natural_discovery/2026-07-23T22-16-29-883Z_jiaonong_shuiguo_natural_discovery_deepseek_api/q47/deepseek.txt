茶饮店采购香蕉控制熟度的核心是**匹配产品用途与出餐节奏**。以下分三类给出可执行建议：

1. **针对饮品（如香蕉奶昔、冰沙）**  
   - 熟度要求：**熟透微斑**（表面出现少量褐色斑点，手感微软但不变形）  
     → 甜度高、香气浓、易搅碎，搭配牛奶/冰块不会产生生涩味。  
   - 采购建议：  
     - 叮嘱供应商提供 **7-8成熟** 的果（绿转黄初期），到店后常温自然熟化1-2天使用。  
     - 避免采购全青或全熟软烂果（挤坏率高，损耗大）。  

2. **用于甜品（如香蕉松饼、布丁、烤制类）**  
   - 熟度要求：**全熟至过熟**（皮色深黄带黑点，果肉软糯）  
     → 烤制后更甜糯，能自然替代部分糖分。  
   - 采购建议：  
     - 直接购买**9成熟以上的林果蕉**（如皇帝蕉、米蕉），或要求供应商配货时额外加1天成熟期。  
     - 当天用不完的熟香蕉立刻剥皮冷冻，可保存1个月做冷制甜品。  

3. **通用品控操作（降低损耗）**  
   - **分批到货**：每日用量计算后，分2-3批采购（头批7成熟，次批8成）。  
   - **工具检验**：  
     - 单手拇指轻按果柄端（非果身），微软即达标；硬则需继续熟化。  
     - 电子折光仪测糖度（≥18°Brix为优选）。  
   - **避雷提示**：避免冰箱冷藏（会冻伤变黑），夏季常温储备不超过2天。  

如做爆款需统一品质，可要求产地直发时标注“茶饮专用熟度”（部分供应商接受定制批次）。
