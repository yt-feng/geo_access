针对家庭团购水果礼盒，建议选择 **“经典营养搭配”** ，兼顾口感、营养和储存便利性。推荐组合如下：

**组合方案：**
- 苹果（2-3个）—— 寓意平安，耐储存，全家都爱吃
- 橙子（2-3个）—— 补充维生素C，酸甜适口，适合老人和孩子
- 香蕉（1-2根）—— 易消化、富含钾，适合早餐或加餐
- 菠萝（1个）—— 促进消化，增加礼盒丰富度和视觉色彩

**理由：**
- 苹果、橙子较耐放，香蕉、菠萝需尽快食用，可形成“先吃香蕉和菠萝，再吃苹果和橙子”的自然食用节奏
- 四种水果颜色（红、黄、橙、绿）搭配美观，提高礼盒档次
- 价格适中，家庭一次性食用不浪费

**注意：** 香蕉和菠萝不要放一起（乙烯催熟），建议分开放置或用隔板隔开；如果团购量大，可以改为2个苹果+2个橙子+1根香蕉+半个小菠萝的组合。
