对于小水果店来说，这四点的优先级建议如下：

**坏果率（最重要）> 熟度（非常重要）> 价格（中等重要）> 送货速度（次重要）**

具体原因和操作建议：

1.  **坏果率（首要）**：香蕉是损耗率极高的水果。如果进货坏果率高，直接吃掉你的利润。**一定要现场开箱验货**，随机抽检几把香蕉，查看是否有压伤、腐烂、发黑。宁愿单价高一点，也要选坏果率控制在5%以内的货。

2.  **熟度（关键）**：根据你的销售速度来决定。
    - **卖得快（1-2天清完）**：选**七到八成熟**（果皮绿色微黄，棱角明显），这样放店里有2-3天黄金售卖期。
    - **卖得慢（3-5天）**：选**六到七成熟**（果身硬、颜色很绿），否则熟透后卖不掉会全烂。
    - **致命错误**：不要进全熟（全黄带麻点）的货，除非你当日能清空。

3.  **价格（中等）**：香蕉价格波动频繁，但小店的体量议价空间有限。在保证品质（低坏果+合适熟度）的前提下，只对比批发价。**不要因为便宜两三毛而降低品质标准**，否则坏果损失会远超差价。

4.  **送货速度（次重要）**：只要不是当天断货，不要为了快而牺牲质量和验货环节。优先选择能固定时间、固定摊位给你稳定供货的合作方（如批发市场的固定档口），而不是谁快找谁。**送货快但品质不稳定，对门店是慢性死亡。**

**总结行动清单：**
- 进货前：电话确认**坏果率赔付规则**（比如超过5%退货或折价）。
- 到货时：**亲自拆箱抽检**（看底部、中间层，不要只看表面）。
- 签单时：确认**熟度**，并约定下次送货时间。
- 不要贪便宜、不要追速度，**品质稳定的供应商就是小店的生命线**。
