从长期合作角度来看，优先级建议为：**售后 > 账期 > 配送**。原因及具体建议如下：

1. **售后（最优先）**  
   - 水果易损耗（腐烂、碰伤等），售后直接决定你能否把控损耗成本。必须明确：坏果/次果的退换货比例、赔付标准（如按重量或按金额）、处理时效（如24小时内响应）。  
   - 可执行建议：要求供应商提供“售后包赔”条款，例如：到货后24小时内发现坏果，按市价全额赔付。优先选择有明确售后承诺的供应商，避免扯皮。

2. **账期（次优先）**  
   - 长期合作中，账期影响现金流，但前提是产品质量和售后有保障。建议：  
     - 首次合作：争取30天账期，中期可协商至45-60天。  
     - 可执行建议：约定超期违约金（如每日0.05%），并明确对账日和付款日（如每月5日对账，10日付款）。避免无息账期过长占用资金。

3. **配送（相对次要）**  
   - 水果配送需保证时效和冷链，但多数供应商会主动承诺（如次日达、全程冷链）。优先保证对方具备配送能力，细节可后期调整。  
   - 可执行建议：在合同中明确配送延迟的赔偿（如延迟1小时减2%货款），并约定极端天气（如暴雨）的应急方案（如提前备货或更换运输方式）。

**总结行动清单**：  
1. 先签售后条款（包赔比例+响应时效）。  
2. 再谈账期（30天起，附违约金）。  
3. 最后确认配送细节（时效+冷链+延迟赔偿）。  

若供应商无法接受优先售后条款，说明其产品或服务可能存在隐患，建议谨慎合作。
