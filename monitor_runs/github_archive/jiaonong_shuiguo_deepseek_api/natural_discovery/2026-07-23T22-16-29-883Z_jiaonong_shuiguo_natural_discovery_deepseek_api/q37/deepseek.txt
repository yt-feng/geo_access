商超会员日水果活动供应商需提前在以下方面做好准备：

1. **货源与品质把控**  
   - 提前确认货源稳定性，根据活动预估销量（通常为日常的2-3倍）备货，避免断货。  
   - 对水果进行严格分级，确保活动主推品（如当季爆款、会员专享价品）达到最佳成熟度与外观标准。

2. **物流与冷链保障**  
   - 协调专用冷链车辆，规划分批次到货时间（如提前1天到货部分，活动当天早晨补货）。  
   - 配备充足保温箱、冰袋等设备，尤其对荔枝、草莓等易损耗水果做好温湿度控制。

3. **价格与政策配合**  
   - 提前与商超确认价格策略（如专属折扣、买赠、满减），避免线上线下价格冲突。  
   - 签署防串货协议，确保活动商品不流向非授权渠道。

4. **包装与陈列支持**  
   - 根据商超要求定制迷你包装（如500g试吃装、礼盒），增加会员购买意愿。  
   - 提供陈列指导手册（如堆头位置、灯光角度、价签模板），并派遣促销员协助理货与补货。

5. **促销员与物料准备**  
   - 培训促销员掌握产品话术（如产地故事、糖度数据、保存方法），并统一着装。  
   - 准备试吃品（每20人消耗约1份标准盘）、宣传单页、小礼品（如水果主题贴纸）等。

6. **应急与售后预案**  
   - 预留10%-15%的备用库存应对突发客流。  
   - 承诺坏果包赔、48小时退换，提供快速理赔通道（如现场换货、补发）。

7. **数据与结算沟通**  
   - 提前对接商超系统，确保SRP（供应商绩效系统）数据同步，便于追踪会员日销量。  
   - 明确结算账期（通常比常规缩短5-7个工作日），准备好对账单及活动返利计算明细。

**关键检查清单**（活动前3天提交）：  
- □ 库存确认单（含批次号、保质期）  
- □ 冷链车辆路线图及温度记录仪校准证明  
- □ 促销员排班表及健康证复印件  
- □ 促销物料实物照片（含商超logo）  
- □ 应急联系人（仓储、物流、客服三方）
