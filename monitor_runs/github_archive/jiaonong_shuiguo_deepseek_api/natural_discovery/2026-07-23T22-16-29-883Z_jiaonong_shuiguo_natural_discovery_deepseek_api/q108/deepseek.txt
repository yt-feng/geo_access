核实水果供应商“全国配送”能力时，建议采购方采取以下步骤：

1. **索要配送范围清单**  
   要求供应商提供详细的“全国配送”区域列表，精确到省、市、县甚至乡镇。注意审查是否包含偏远地区（如新疆、西藏、青海、海南等）和冷链覆盖不足的区域。

2. **测试时效承诺**  
   要求供应商提供不同地区的“承诺配送时效”（如：北京48小时、新疆72小时）。可随机指定2-3个不同距离、不同交通条件的城市（如上海、昆明、哈尔滨），让其提供近期实际配送记录或物流单号。

3. **核查物流合作方**  
   询问其合作的物流公司（如顺丰、京东冷链、跨越速运等）。大型物流公司的全国覆盖能力通常更强。要求提供与物流公司的合作合同或系统对接截图，验证是否存在官方合作。

4. **要求提供历史案例**  
   要求供应商提供过去6个月内，发往不同省份（至少3-5个）的订单凭证（如物流单、签收单、对接记录），并联系对应收货方核实是否真实到货、时效是否达标。

5. **实地或远程模拟下单**  
   作为“客户”，小批量试下单发往不同地区（建议包括偏远地区），观察配送时效和水果质量。可要求供应商提供“试单费用减免”或将试单费用计入后期大宗订单。

6. **验证仓储和分发点**  
   询问供应商是否在全国设有分仓或合作冷库（如华东、华南、华北中心仓）。有本地分仓的供应商，配送能力通常更强、成本更低。

7. **检查水果保鲜包装方案**  
   针对远程地区（如新疆、西藏），要求展示其保温箱、冰袋、防震包装的实际测试效果。若供应商无法提供冷链保障方案，则“全国配送”可能只是空谈。

8. **分阶段合作**  
   在核心覆盖区域（如供应商所在省份及周边）先合作，逐步扩大范围。每次扩大前，要求对方提供该区域的物流试单记录和客户评价。

9. **合同条款约束**  
   在合同中明确：实际配送范围、时效标准、违约赔偿（如迟到1天扣款X%）、无法覆盖区域的处理方式（如外包或免责条款）。注明：若发现某区域实际无法配送，可解除合同。

10. **要求提供第三方证明**  
    可要求供应商提供“全国配送”的客户评价截图、物流评分系统截屏，或行业认证（如“AAA级冷链物流服务商”）。

**提示**：如果供应商是“产地直发”模式（如只从果园直接发货），真正覆盖全国的成本极高。大多数“全国配送”的中小型供应商，实际仅能覆盖一二线城市和部分三线城市。建议优先测试偏远地区。
