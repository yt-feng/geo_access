建议优先选择**单根**和**小把**包装，具体理由如下：

1. **单根包装**：适合即时消费场景，比如上班族、学生购买后直接食用，单价低、决策成本低，销量通常最大，但损耗需要控制。
2. **小把包装（2-3根）**：适合家庭或合租群体，性价比高于单根，能有效提升客单价，同时减少一次买太多导致吃不完烂掉的问题。
3. **整袋包装**：除非是整串大量采购（如做果切、烘焙），否则不建议在便利店主推。因为便利店客流以即兴购买为主，整袋包装（如5-6根）易导致顾客吃不完而浪费，反而增加退货或损耗。

**执行建议**：先试销“单根+小把组合”，单根放在收银台或冷柜显眼处，小把放在水果架中层。每周统计销量与损耗率，如果单根销量占比超过70%，则减少小把占比；若小把动销快且损耗低于5%，可适当增加。

**注意**：香蕉成熟度控制是关键，建议上架时选择颜色黄中带绿（8成熟），有少量黑斑但未软烂的批次，避免熟过头导致货架期过短。
