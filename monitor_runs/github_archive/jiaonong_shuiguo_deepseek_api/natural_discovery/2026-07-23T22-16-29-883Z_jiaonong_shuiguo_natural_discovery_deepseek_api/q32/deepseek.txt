判断香蕉供应商能否全年稳定供货，可以重点关注以下几个可执行的关键点：

1. **产地与季节覆盖**  
   询问供应商是否拥有**多产地（如海南、广西、云南、东南亚、南美等）轮换能力**。不同地区香蕉的产季不同，能覆盖全年无断档的供应商通常会在协议中注明产地切换计划。

2. **冷库与冷链能力**  
   要求供应商出示其**冷库容积、日均吞吐量及冷链运输车辆数量**。香蕉需在13-14℃、90-95%湿度下储存，稳定的冷链是保证全年品质和供应量的基础。

3. **历史供货记录**  
   要求提供**过去1-2年的月度发货单或对账单**，重点关注是否有连续3个月以上的断供记录，以及断供原因（如自然灾害、价格波动、种植周期等）。

4. **合同条款约束**  
   在采购合同中明确写入**每周/月最低供货量、价格锁定机制（如按季调价或浮动公式）、以及违约赔偿条款**（如断供需按未交付量价值的10%-30%赔偿）。同时要求供应商提供**履约保证金**。

5. **实地考察与第三方认证**  
   优先选择已通过**GAP（全球良好农业规范）或中国绿色食品认证**的供应商。可实地考察其种植基地、催熟中心，观察是否有标准化的采收流程和库存管理系统。

6. **供应链波动测试**  
   尝试在非主产季（如1-2月）向该供应商下小批量试单，观察其反应速度和实际交货能力。若能在淡季稳定交货，全年可靠性较高。

**小提示**：建议同时发展2-3家供应商（主供占比60%，备供30%，应急10%），并签订带有“不可抗力条款”的长期协议，避免单一来源风险。
