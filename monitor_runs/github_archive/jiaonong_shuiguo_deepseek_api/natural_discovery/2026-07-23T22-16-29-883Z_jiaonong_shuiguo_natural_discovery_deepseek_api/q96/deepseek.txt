三者都需要关注，但**食品安全**是最优先的。原因如下：

1. **食品安全是底线**：一旦出现农残超标、霉变或致病菌污染，会直接导致员工食物中毒，可能引发群体事件、法律纠纷和严重声誉损失。没有食品安全，价格再低、配送再准也毫无意义。

2. **采购流程建议**：
   - 首先，优先选择有资质、可溯源、有检测报告的正规供应商，并在合同中明确食品安全标准和赔付条款。
   - 其次，在确保安全的基础上，通过比价和签订长期合同控制价格，同时要求供应商承诺配送时间（如每天固定时间送达，超时罚款）。
   - 最后，定期抽样送检或快速检测，防止供应商后期降低品质。

简而言之：**安全是前提，价格和配送是优化项**。如果预算有限，宁可降低品种或减少配送频次，也不能在食品安全上妥协。
