酒店采购水果盘原料时，供应商应确保以下规格和外观标准：

1. **规格标准**：
   - 水果大小均匀，符合约定重量或数量（如每盘苹果直径8-10cm，每颗葡萄粒重8-12g）。
   - 无腐烂、变质或机械损伤（如压痕、切口）。
   - 成熟度一致，避免过生或过熟（如香蕉表皮无青绿或黑斑）。

2. **外观标准**：
   - 表面清洁，无污渍、虫斑或农药残留。
   - 色泽自然鲜艳，无异常变色（如芒果表皮无褐变）。
   - 形状规整，无畸形或萎缩（如草莓无干瘪、蓝莓无瘪粒）。

3. **包装与标识**：
   - 使用食品级包装材料，防止运输中挤压或污染。
   - 标注产地、采摘/生产日期、保质期及储存条件（如2-4°C冷藏）。

建议在采购合同中明确上述标准，并抽查到货样品。供应商需提供可追溯的产地证明及检测报告。
