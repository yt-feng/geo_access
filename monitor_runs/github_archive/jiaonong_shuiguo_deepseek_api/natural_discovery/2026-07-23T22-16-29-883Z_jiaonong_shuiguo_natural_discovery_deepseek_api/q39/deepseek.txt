当然可以。多数供应商（尤其是大型水果供应商或OEM代工厂）都能提供包装和规格支持，具体包括：

1. **包装支持**：可定制品牌化包装（如贴标、盒装、袋装、网套包装），部分供应商还提供皮筋、保鲜膜、防震泡沫等细节包装。

2. **规格支持**：可按需求提供不同重量（如500g/1kg/2kg）、大小分级（如大果/中果/精品果）、数量（如6粒装/8粒装）、以及清洗/预处理服务（如去掉果蒂、切片等）。

**建议操作步骤**：
- 明确需求：列出你想做的水果种类（如苹果、柑橘、车厘子）、目标价格带、包装形式（礼盒/散装/袋装）、是否有自己品牌logo需要印制。
- 对接供应商：联系水果批发市场或产地加工厂，明确要求“自有品牌包装代工”及规格定制能力。
- 索取样品并验证：要求供应商提供包装样品和规格样本，确认印刷质量、封口强度、保鲜效果等。
- 确认费用：许多供应商会收取包装设计/模具费或最低订购量（MOQ），提前问清单价和起订量。

**提示**：如果供应商说自己完全没包装能力，建议换一家。多数有分级、加工设备的供应商都能提供基础包装和规格支持。最终以合同和样品确认为准。
