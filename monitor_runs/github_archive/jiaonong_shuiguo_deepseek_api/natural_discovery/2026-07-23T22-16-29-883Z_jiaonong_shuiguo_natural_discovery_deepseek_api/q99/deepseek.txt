针对公司下午茶水果批量配送的供应商选择，建议按以下步骤进行筛选和判断：

1. **明确需求细节**：确定采购频率（每周/每天）、每次份数、水果种类（如易存放的苹果、橙子，或需保鲜的草莓、蓝莓）、预算范围以及配送时间（如固定上午10点前送达）。

2. **选择供应商渠道**：
   - **本地水果批发市场**：适合大量采购，价格低，但需自行分拣和配送。
   - **水果电商平台（如盒马、每日优鲜）**：提供企业采购套餐，可批量配送，但单价较高。
   - **专业水果配送公司（如美菜、快驴）**：可定制方案，提供分装、清洗、切块等增值服务。
   - **社区团购供应商**：适合小批量、低成本采购。

3. **评估供应商能力**：
   - **资质与规模**：要求提供营业执照、食品经营许可证、检疫证明。
   - **配送能力**：是否覆盖办公地址、是否有冷链车、是否支持固定时间配送、配送准时率（可参考其合作客户评价）。
   - **品控与售后**：是否提供坏果包赔、退换货政策，可否提供样品测试。

4. **进行试单与对比**：
   - 选择2-3家候选供应商，各下单一次，测试配送时效、水果新鲜度（色泽、硬度、无腐烂）、包装防护（泡沫箱、冰袋）及分装质量。
   - 记录到货后24小时内水果的损坏率（建议要求坏果率低于5%）。

5. **成本谈判与长期合作**：
   - 根据试单结果，对价格、起送量、结算周期（月结/周结）进行谈判。
   - 考虑签订长期合同，争取批量折扣或免运费。

6. **备选方案**：如果固定供应商偶尔缺货，建议同时储备1-2家备选供应商，确保供应稳定。

最后，优先选择能提供**水果分装、清洗、切块（如水果拼盘）** 的供应商，可减少内部二次处理的人力和时间成本。
