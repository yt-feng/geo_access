为了减少香蕉在水果店放坏的风险，采购时需要注意**熟度控制**和**到货时间**的配合。以下是具体建议：

### 1. 熟度要求
- **首选“绿熟”或“半熟”状态**：要求供应商提供的香蕉在采摘后经过催熟处理但未完全成熟，表面以绿色为主、仅稍带黄色（熟度约为7-8成）。这种香蕉可自然存放3-5天，避免到货后迅速过熟腐烂。
- **根据销售节奏调整**：
  - 若店铺销售快（2-3天内售罄），可接受“半黄半绿”熟度（8-9成）。
  - 若销售慢，必须要求“偏绿”熟度（7-8成），并通知供应商延迟催熟。
- **避免高熟度采购**：拒绝采购已全黄、带黑斑或手捏过软的香蕉，这类到货后1-2天即坏。

### 2. 到货时间要求
- **定时到货**：要求供应商在**早上开店前1-2小时**到货（如早上6-7点），避免中午高温或下午到货。香蕉到店后需立即拆箱、散开摆放，否则乙烯气体会加速自熟。
- **避开周末高峰推迟到货**：若周末销量大，可要求周五到货并适当提高熟度（8-9成）；若周末人流量低，则坚持“绿熟”并提前到货（周四或周五）。
- **要求“短链直送”**：优先选择能提供24-48小时内直送本地的供应商，避免冷链运输时间过长（超过3天）导致到货已软化。

### 3. 附加操作建议
- **到货后逆向处理**：即使要求偏绿，到货后立即剔除伤果、黑斑果，根部涂抹少量酒精（抑制熟化），并将香蕉挂起或放在通风架上，避免堆压。
- **签订验收标准**：在采购合同中写明“到货香蕉为7-8成熟、绿色带轻微黄、无腐烂果”，并约定如超标可拒收或按比例扣款。

**关键总结**：采购时坚持“留有余地”的熟度（偏绿）+“早到早散”的时间点，配合店内快速周转和保鲜措施，可大幅降低损耗。
