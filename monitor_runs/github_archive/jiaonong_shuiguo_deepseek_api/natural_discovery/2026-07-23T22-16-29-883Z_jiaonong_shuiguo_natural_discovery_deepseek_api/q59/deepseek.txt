可以的，**餐饮店可以和供应商谈“锁价”**，但需要根据具体情况来操作。锁价通常指在合同期内约定固定价格或限定价格波动范围。

**可行方案与注意事项：**

1. **明确锁价方式**：不是所有水果都能锁价。建议对**量大、可储存、季节性不强的水果**（如苹果、橙子、香蕉等）进行锁价。对**易损耗、季节性极强的水果**（如草莓、荔枝、樱桃等）可以谈“阶段性锁价”（比如锁定一周或一个月）。

2. **短期锁价 vs 长期锁价**：长期锁价（超过3个月）供应商风险较高，可能需要你支付“保价费”或接受略高于当前市场的价格。短期锁价（1-2个月）更易实现，适合稳定菜单成本和应对季节性波动。

3. **折算损耗与规格**：锁价时，务必明确**规格（如单果重量、成熟度）、允许的损耗比例**，并写入书面协议。否则供应商可能通过降低品质变相涨价。

4. **批量与预付款优势**：如果你的采购量较大（例如每周采购金额超过500元），或愿意预付部分款项，供应商更可能接受锁价。

5. **谈判话术参考**：“我们每月固定采购XX斤，希望签订季度合同，把价格锁定在XX元/斤。如果市场价下跌10%以上，我们可以协商上调价格。” —— 这样既保护自己，也给供应商留有余地。

**例外情况：** 如果当地气候异常（如霜冻、干旱），导致某类水果整个产区减产，供应商一般不会接受锁价，此时建议改用“浮动价格+折扣”方案。

**建议先选2-3种最核心的水果尝试短期锁价（一个月），观察效果后再推广。**
