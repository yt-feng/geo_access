针对水果礼盒的搭配，建议遵循“色彩对比+口感互补+寓意吉祥”的原则。以下是几个好卖的搭配方案：

1. **经典红黄搭配（寓意健康、吉祥）**  
   - 苹果（红色/绿色，象征平安）  
   - 橙子（橙色，象征心想事成）  
   - 香蕉（黄色，象征招财）  
   - 菠萝（金黄，象征旺财）  
   优点：颜色鲜艳（红+黄+金），视觉冲击强；苹果脆甜、橙子酸甜、香蕉软糯、菠萝香甜，口感层次丰富。适合春节、中秋等送礼场景。

2. **轻奢清爽混搭（更受年轻人欢迎）**  
   - 苹果（选红富士）  
   - 橙子（选脐橙）  
   - 菠萝（切块装盒，或选迷你菠萝）  
   - 香蕉（避免熟透，选微青的，保持硬挺）  
   - 可以加少量薄荷叶或小卡片装饰  
   优点：去除香蕉（怕压坏）后可改用阳光玫瑰葡萄或蓝莓，更显高档；菠萝切块后更易食用，适合公司福利或朋友拜访。

3. **健康低糖组合（适合健身或长辈人群）**  
   - 苹果（绿苹果，酸度低）  
   - 橙子（选甜橙）  
   - 香蕉（选择口未完全熟透的）  
   - 菠萝（去芯切块，含菠萝蛋白酶助消化）  
   注意：香蕉和菠萝都易软，建议用切片分隔或独立包装，避免磕碰。

**避坑建议**：
- **香蕉最易烂**：如果礼盒是预包装且需要运输或冷藏，建议用纸袋单独包裹，或替换成猕猴桃、蜜瓜等硬果。
- **菠萝处理**：若提供切好的菠萝，需用密封盒并标明“尽快食用”，或用整只菠萝（配削皮器）更显专业。
- **规格建议**：每个水果数量取双数（如苹果6个、橙子6个、菠萝1个、香蕉1串），寓意“六六大顺”“一帆风顺”。

**推荐组合定价**：
- 入门款（包装简单，主打性价比）：苹果+橙子+香蕉（总价约30-50元）  
- 畅销款（配色统一，适合送礼）：苹果+橙子+菠萝（切块）+少量蓝莓（总价约60-90元）  
- 尊享款（加葡萄、石榴等）：苹果+橙子+菠萝+香蕉+阳光玫瑰（总价约100-150元）

建议先小批量试卖，记录反馈。如果顾客反馈想要更硬挺的水果，可把香蕉换成梨或哈密瓜。
