针对超市门店多但单店销量不大的情况，建议采用以下水果供货模式：

1. **“统采+分仓”模式**：由总部统一向产地或一级批发商大规模采购，降低采购成本。然后根据各门店需求，通过区域分仓或直配物流进行分拨。这样既保证了供应链的稳定性，又避免了单店库存积压。

2. **“小批量多批次”配送**：与供应商签订框架协议，约定按固定周期（如2-3天）或根据销售数据触发补货。这种方式适合门店销量波动不大的场景，能减少损耗并保持新鲜度。

3. **“线上订货+次日达”模式**：采用类似社区团购的机制，允许顾客当日下单，次日到店自提。超市作为提货点，无需提前囤货，只需处理订单及最后一公里配送，适合当前门店密度高但单店量小的状态。

4. **“短供应链+核心品种优先”**：重点供应当地当季、销量稳定、耐储存的水果（如苹果、橙子、香蕉），集中精力做少数精品SKU。避免引入过多小众或易损水果，降低管理和损耗成本。

5. **合作分账模式**：与小型水果配送商或合作农户采用“销售后结算”方式，减少自身资金占用，同时供应商更有动力提供新鲜水果。

可根据实际门店分布、消费习惯及冷库条件选择适用模式，初期建议从“统采+分仓”与“核心品种优先”组合入手。
