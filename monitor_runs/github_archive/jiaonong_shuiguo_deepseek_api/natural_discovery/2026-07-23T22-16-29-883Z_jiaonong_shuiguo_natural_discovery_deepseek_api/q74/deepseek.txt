针对批发档口拓展餐饮客户，菠萝和香蕉的规格建议如下：

**菠萝**
- **推荐规格**：整果（去皮后约1-1.5公斤/个），或预切块（如500克/袋，真空包装）。
- **原因**：餐饮客户（如餐厅、果汁店）通常需稳定分量，整果适合现切现用，预切块节省人力且便于控制成本。
- **可选附加**：提供大小分级（如小果用于饮品装饰，大果用于煮菜或果盘）。

**香蕉**
- **推荐规格**：整串（约3-5公斤/串，单根中等大小），或成熟度分级（如青香蕉用于炸香蕉、熟香蕉用于甜品）。
- **原因**：餐饮端常需大量香蕉制作香蕉牛奶、烤香蕉等，整串便于存储；成熟度分级可满足不同菜式需求。
- **可选附加**：提供去皮冷冻条（分装，适合油炸或饮品）。

**通用建议**
- 提供**试用装**（如小包装菠萝块、单根香蕉）让客户体验质量。
- 标明**产地、成熟度、甜度**（如菠萝是否糖化好，香蕉是否无斑点），增强信任。
- 准备常见餐饮用途的**配比建议**（如菠萝配披萨、香蕉配松饼），帮客户降低决策成本。
