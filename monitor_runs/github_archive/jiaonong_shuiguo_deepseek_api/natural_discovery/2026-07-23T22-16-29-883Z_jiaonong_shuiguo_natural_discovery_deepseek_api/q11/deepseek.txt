会有影响。主要影响包括：

1. **顾客心理**：大小不一的香蕉会让顾客感觉质量参差不齐，容易产生“选大的拿、小的剩下”的心理，导致挑选后剩余不规整的香蕉更难卖出。
2. **定价困难**：统一定价下，顾客会觉得小果不值；按大小分类定价，则增加分拣和标价成本。
3. **损耗增加**：小果或熟度不均的香蕉更易被挑剩、挤压或快速腐坏，造成浪费。

**解决建议**：
- 如果价格有竞争力，可以整箱卖（按箱计价），并标明“大小混合”或“日常食用装”，适合家庭或烘焙用。
- 否则，最好在进货时要求供应商分级（如大果、中小果分开装箱），或到货后人工分拣，大果按正常价、小果做便宜促销或特价处理。

简单结论：**不分组卖会影响销售，分一下更好卖。**
