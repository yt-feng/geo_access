针对超市水果区提高客单价，建议采取**“引流爆款+利润搭配+附带提篮”**的组合策略。以香蕉、菠萝、苹果为例，具体方案如下：

**促销组合：热带风情家庭装**

- **组合内容：** 1份香蕉（约500g）+ 1个菠萝 + 2个苹果（中等大小，约300g）
- **定价策略：**
  - 单买价格：香蕉3元（按6元/kg计）、菠萝10元/个、苹果4元（按13元/kg计），合计17元。
  - **组合售价：13.8元**（相当于原价~8折，但实际利润不低）。
- **理由：** 香蕉是低单价高频带客，菠萝单价高且能带动季节感，苹果作为常规水果保障价值感。组合后顾客觉得“省了3块多”，但超市通过批量进货和损耗控制，毛利反而比单卖高5-8%。

**提高客单价的关键细节：**

1. **视觉陈列：** 用藤编篮子或带把手的小纸箱装好，贴上“限时组合特价”标签，放在主通道端架。
2. **连带劝导：** 在菠萝区附近设置“做水果沙拉/菠萝饭”的简笔画提示，暗示苹果也是配料，鼓励一次拿满。
3. **梯度升级：** 在组合旁边放置“加1元换购薄荷叶/酸奶”的小筐，将客单价从13.8元再拉至15元以上。
4. **限时+限量：** 每天下午4-6点（家庭采购高峰）主推，用“今日专供50份”提升紧迫感。

**注意避开以下陷阱：**
- 三样水果不要全部是易损耗品种（香蕉和菠萝均已较易熟，苹果是耐储的，这样平衡）。
- 不要直接打“5折”或“买一送一”，会降低水果区形象。用“组合价”比“折上折”更利于后续恢复原价。
- 所有宣传单上必须用**实物照片**，而非“示意图”，反映真实大小和成色，避免客诉。

**实施后评估：** 观察一周，如果组合日均销量超过三种水果单卖日均总量的70%，则可固定为每周三、五的“水果日”活动，并尝试将苹果换成当季高利润水果（如梨、奇异果）。
