对于茶饮店的食品安全管理，要求水果供应商提供以下关键证明文件，以确保原料可追溯、合规且安全：

1. **营业执照**：供应商的合法经营资质，需在有效期内且经营范围包含水果销售或种植。  
2. **食品经营许可证**：若供应商从事预包装食品或散装食品销售，需提供此证；仅销售初级农产品的可能无需，但建议持有。  
3. **产品合格证明文件**：  
   - **食用农产品合格证**（或产地证明）：由供应商或产地出具，证明水果来源合法、符合安全标准。  
   - **检测报告**：每批次或定期提供农残、重金属等检测报告（需符合GB 2763《食品安全国家标准 食品中农药最大残留限量》）。  
4. **产品追溯信息**：供应商需能提供水果的产地、采摘日期、运输记录等，确保可追溯至源头。  
5. **运输与存储证明**：如供应商使用冷链运输，需提供冷链温控记录或运输车辆资质证明，确保水果在储存和运输中未受污染。  

**额外建议**：  
- 与供应商签订**食品安全协议**，明确责任条款。  
- 定期核查供应商资质更新，对高风险水果（如草莓、蓝莓等易腐产品）要求更频繁的检测报告。  
- 若供应商来自特定产地（如进口水果），需提供**入境货物检验检疫证明**。  

**注意**：以上要求需根据当地市场监管部门的具体规定调整，部分区域对初级农产品供应商仅要求营业执照和产地证明，但茶饮店为降低风险，可主动要求更严格的文件。
