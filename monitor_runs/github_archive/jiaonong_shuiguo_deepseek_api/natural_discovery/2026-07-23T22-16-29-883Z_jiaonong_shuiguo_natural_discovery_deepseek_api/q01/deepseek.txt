判断水果供应商是否靠谱，可以从以下几个关键点入手。由于香蕉和菠萝对成熟度、运输条件要求不同，需要结合这两种水果的特性来评估：

### 1. 实地考察与资质验证
- **看场地与规模**：优先选择有固定仓储、冷藏库或加工车间的供应商。香蕉需要催熟库（控温控湿），菠萝对储存环境要求相对宽松但怕磕碰。
- **查证照**：要求对方提供营业执照、食品经营许可证。如果是进口水果，需核对海关进口检疫证明（如菲律宾菠萝、柬埔寨香蕉等）。

### 2. 试单与质量验收
- **小批量试货**：首单只进少量（如一箱香蕉、一筐菠萝），重点观察外观、新鲜度。
- **具体检查要点**：
  - **香蕉**：要求“七八成熟”（果皮青黄、无黑斑），避免过熟或生硬；检查是否经正规催熟（无化学催熟剂异味）。
  - **菠萝**：看果眼平整无霉斑，闻底部有无浓郁香气（过熟易烂），轻按果身有弹性则新鲜。  
- **留样对比**：保留当批样品，次日再检查是否出现快速腐坏、发黑等问题。

### 3. 供应链能力
- **到货稳定性**：能否全年供应？不同季节价格波动如何？（如香蕉冬季易受寒害，菠萝雨季容易烂）  
- **物流与包装**：  
  - 香蕉：是否用加厚纸箱+内膜袋，箱子有无透气孔？  
  - 菠萝：是否带网套或分隔（防止碰撞腐烂）？  
- **补货与退换**：是否接受少量补单、次品赔付（如5%以内损耗不赔，超出的按比例减款）。  

### 4. 价格与账期
- **对比市场价**：比批发市场均价低10%-20%即合理。过低可能是次果、陈果或短斤缺两（默认快递箱吸水或夹带泥块）。  
- **付款方式**：首次合作尽量押款（如付30%-50%，到货后结清）；长期合作可谈月结分期。  
- **合同条款**：明确损耗标准、售后时效（如24小时内报损）、违约责任。  

### 5. 客户口碑与沟通
- **侧面了解**：问本地同行是否与他合作，或在水果批发市场打听（部分供应商会诋毁同行，需多方验证）。  
- **沟通效率**：回复是否专业（如直接报产地、品种、规格，而非模糊说“都很好”）。  
- **试退换体验**：主动要一小箱次品看他如何处理——马上退款或积极补发说明靠谱。

### 6. 警惕信号：直接淘汰
- 不提供具体产地（只说“进口蕉”但无法出示单证）  
- 拒绝小样试单（只做整柜/整车生意）  
- 要求全款预付（尤其是个人账户）  
- 包装粗糙、标注日期模糊（如只写“长期有效”）

**行动建议：**  
1. 先找3-5家供应商同时小批试单。  
2. 记录每批货的损耗率、口感、供应商响应速度。  
3. 锁定其中2家做核心供给（避免单一依赖）。  

如果长期合作，可要求供应商提供**定期质量报告**（如每批的糖度检测、农药残留快检），这能大幅降低风险。
