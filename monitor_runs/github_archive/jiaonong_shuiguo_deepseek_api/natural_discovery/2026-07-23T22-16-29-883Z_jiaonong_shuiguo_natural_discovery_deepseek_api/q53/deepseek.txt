根据你“每天少量”且“省心”的需求，以下是适合的供应商选择思路：

1. **本地农贸市场或批发市场的小档口**：找固定摊位、营业时间稳定、送货意愿强的小批发商。这类供应商通常愿意为长期稳定的小订单提供每日配送，且价格灵活，可要求其按你要求的规格（如切好、装盒）备货。

2. **社区团购或生鲜电商平台的“B端配送”服务**：如美团优选、多多买菜、每日优鲜等，有固定配送时间、无需起订量，且商品标准化（如预包装水果）。但需注意到货时间和品质稳定性，建议先试单。

3. **周边小型水果店或夫妻店**：与附近一家或两家口碑好的水果店达成长期协议，要求其每日按你的清单备货并送货上门。这类店铺灵活、服务好，但需确认其每日供应能力和价格。

4. **食材供应链公司的“小微订单”服务**：部分区域型食材配送公司可承接每日少量订单，通常需要签订合同、预付押金，但能保证稳定供货、有发票（如果需报销）。

**省心关键点**：  
- 优先选择可以 **每日定时配送** 的供应商，避免你自己取货。  
- 先要求 **试送3-5天**，确认品质（如新鲜度、重量是否足）、配送时间和售后服务（如坏果包换）。  
- **固定1-2家供应商**，不要频繁更换，便于对方建立对你的供货习惯。  
- 沟通时明确： **“每天XX元以内的少量水果，品种随机或按你们当天新鲜的来配，按周结账”**，这种模糊需求反而容易达成长期合作。

**不建议**：  
- 一开始就找大型水果批发商（起订量大、价格高、不送小单）。  
- 用超市或外卖平台的“次日达”服务（价格高、品种固定、无法灵活调整）。
