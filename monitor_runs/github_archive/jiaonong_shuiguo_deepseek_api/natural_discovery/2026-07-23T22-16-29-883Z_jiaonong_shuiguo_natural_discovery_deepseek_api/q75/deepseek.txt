针对第一次合作的水果供应商，小批量试单的目的是验证品质、稳定性与服务水平，同时控制风险。以下是具体的设计步骤与建议：

### 1. 明确试单目标与品种
-   **选品**：选择1-2个核心品种，避开高损耗或价格波动剧烈的品类（如特殊进口水果）。优先选该供应商口碑较好、有稳定货源的单品。
-   **数量**：控制在正常采购量的10%-20%，例如常规量100箱，试单量10-20箱。金额建议在500-3000元区间（根据品种调整）。
-   **时间**：避开销售淡季或极端天气前后。

### 2. 设计合同与付款条件
-   **合同要点**：即使小单，也要有书面合同或订单确认记录。条款包括：
    -   **验收标准**：明确果径、成熟度（如八成、九成）、无腐烂/冻伤、包装（净重、瓦楞纸箱规格）。
    -   **付款方式**：优先“货到验收后结款”或“30%预付+70%尾款”。避免全款预付。
    -   **违约处理**：若品质不符，退货或按比例降价（如次果率超过5%则扣除差价）。
-   **风险控制**：用第三方支付平台（如微信转账备注订单号）或对公账户，保留转账截图。

### 3. 物流与验收流程
-   **运输要求**：要求供应商用冷链车（若为生鲜），或至少加冰袋+泡沫箱。明确到货时间（如“下单后48小时内到”）。
-   **现场验收**：
    -   **抽样**：随机开箱检查，至少抽检3-5箱。
    -   **测量工具**：带糖度计（可选）、电子秤、卡尺。
    -   **记录**：拍照（整箱外观、内部摆放、缺陷个体）、填写《到货验收表》（满列品种、果径、重量、坏果数、包装破损）。
-   **异常处理**：若坏果率≥10%，直接拒收并要求供应商补发或退款；若≤5%，可接受但拍照留档作为后续议价依据。

### 4. 销售与反馈机制
-   **测试渠道**：优先用于稳定客群（如老客户、社区团购团长、线下熟客），明确告知“新货源试单，品质可能有波动”。
-   **跟踪指标**：
    -   客户复购率（≥60%可考虑深度合作）
    -   客诉率（低于3%）
    -   损耗速度（如统计“到货后72小时内的溃烂率”上限：普通水果≤5%，易损水果如草莓≤10%）。
-   **记录数据**：制作简单的表格，包含到货日期、售价、损耗、退货率，与同类型老供应商对比。

### 5. 试单后的决策
-   **大规模采购条件**：
    -   连续3次试单（每次间隔1-2周）均达到：品质稳定、客诉率≤2%、物流准点率≥95%。
    -   供应商愿意提供小量试销支持（如免费样品、账期）。
-   **合作终止信号**：
    -   一次试单出现批量腐烂（超过15%）。
    -   供应商对合理反馈推诿或延迟处理。

### 示例试单流程表
| 步骤 | 内容 | 风险控制点 |
|------|------|------------|
| 1.选品 | 确认品种、数量（如10箱哈密瓜）、报价单 | 避免特殊定制或非标果实 |
| 2.下单 | 发送正式订单（包含货期、验收标准） | 保留聊天/邮件记录 |
| 3.到货 | 立即开箱拍照、称重 | 货到6小时内完成验收 |
| 4.销售 | 限定区域/客户群体测试 | 注意控制当前库存周转 |
| 5.评估 | 整理客诉数据，与供应商开复盘会 | 形成书面总结 |

### 关键提醒
-   **避免踩坑**：警惕价格过低（可能因次果或存货）、要求全额预付、无法提供产地证明（如需溯源的品类）。
-   **工具推荐**：使用“微信小程序+Excel”记录试单数据，或尝试免费的供应链管理工具（如“采销宝”轻量版）。

按照以上设计，你可以在控制风险的前提下，快速判断该供应商是否值得长期合作。
