建议参考以下步骤和话术模板，确保补货高效且减少后续争议：

**1. 立即评估需求并主动沟通**
   - **明确数量与时间**：先内部确定当前库存、每日消耗量及预计持续天数，计算出紧缺量。
   - **立即联系供应商**：优先电话或微信语音沟通，避免文字延误。说明“旺季销量激增，菠萝库存本周将断供，需要紧急补货X件，最快何时能到？”

**2. 临时补货约定要点（建议口头确认后补文字）**
   - **价格与附加费**：询问临时补货是否按原价，或是否需要承担加急物流/分拣费。可商议：“这次旺季补货能否按原合同价，或者我们分担部分物流差价？”
   - **品质与规格**：明确菠萝的成熟度（如“七成熟，便于短期催熟”）和去留要求（如“带冠直切，单果重量≥1.2斤”）。
   - **补货批次**：若需求量极大，可要求分批次到货（如“先发50%救急，3天后补齐”，避免一次性积压变质）。
   - **具体节点**：约定最晚到货时间（如“6月5日下午18:00前送至门店”），并说明超期影响（如“超期24小时将影响开店，需协商处理方案”）。

**3. 提出诚意支持方案（降低供应商顾虑）**
   - 主动表示可适当提高单价（如“在原价上浮5%”），或承诺此笔补货付款周期缩短（如“船到30天内付款改为15天内”）。
   - 如果长期合作，可承诺未来旺季优先采购其菠萝。

**4. 书面确认（防纠纷）**
   沟通后立即发送微信或邮件，内容包括：
   - 补货数量、单价、总金额
   - 到货时间及地点
   - 双方确认收货前质检标准（如“腐烂率超过3%部分由供方承担”）
   - 因突发情况延迟的补救措施（如“超时自动按订单金额日息0.5%赔付损耗补偿金”）

**5. 风险备选方案**
   - 同时联系1-2家备选供应商（果农商或批发市场）确认应急供应。
   - 如果原供应商无法保证时效，可备用部分冷冻菠萝（需测试口感），或临时压缩菠萝饮品的菜单占比。

**附：临时补货话术模板**  
“张总/李经理，我们是XX茶饮店，最近菠萝旺季销量暴涨，目前库存只能撑到下周。您看能否这周五前先发200件中等成熟度的菠萝救急？我们可以在原价基础上每件加5元承担物流调度成本，这趟货的货款我们押后15天付。如果行，我马上把加急订单发您，到货我们专人验果。”
