针对水果到货不合格影响营业的问题，建议采取以下措施约定补货，确保快速恢复运营：

1. **明确验收标准**：与供应商提前在合同中约定水果的品级（如大小、色泽、成熟度）、新鲜度（无腐烂、损伤）、及可接受的损耗比例（如每箱不超过5%）。到货后立即抽检，不合格品拍照留证并拒收。

2. **设定应急补货时间**：约定不合格品需在**2小时内**启动补货流程，**4-6小时内**送达（视距离而定）。超时则按延迟扣款或要求供应商承担临时采购差价。

3. **分段补货机制**：如整批次不合格，要求供应商先调拨“应急库存”（如从其他门店或仓库），同时启动新货补发；部分不合格品，仅要求更换劣质部分，而非全退，减少流程耗时。

4. **短期替代方案**：在补货到店前，用同店现有其他水果调整饮品配方（如用柠檬替代部分青柠，或推出限定款消耗库存），同时通知客户暂时缺货款，预留退款/换购券避免客诉。

5. **合同约束违约成本**：约定不合格率超过10%时，供应商需支付**订单金额的20%作为违约金**，并免除本次不合格品费用；累计三次违约可解约。

6. **本地应急渠道**：提前储备2-3家本地小型供应商或批发市场联系方式，一旦补货延迟，立即采购少量应急（成本可能略高，但保销售优先）。

**执行示例**：
- 发现到货芒果有黑心，立即拍照发供应商要求2小时内补发；同时通知后厨用库存香蕉调整制作“香蕉拿铁”，并让前台告知今日芒果系列暂停。
- 若供应商超时，扣减货款同时联系本地市场采购10斤芒果（成本计入供应商违约金）。

这样既能保障营业，又能推动供应商提高质量意识。
