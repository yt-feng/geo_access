拿香蕉时判断熟度与坏果风险，建议按以下步骤操作：

1. **观察果皮颜色**  
   - 全绿且棱角分明：尚未成熟，适合长期贮存。  
   - 黄中带绿：开始转熟，1-2天内可食用。  
   - 全黄带少量褐斑：最佳食用期，需尽快销售或冷藏。  
   - 黄色深、出现大片黑斑或软塌：过熟，坏果风险高，不建议批发。

2. **触摸果实质感**  
   - 用拇指轻压果身中部：  
     - 硬实、有弹性：熟度适中。  
     - 局部发软、无回弹：内部可能已经变质。  
   - 检查果柄（香蕉把）：若发黑或发霉，容易蔓延至果肉。

3. **闻气味**  
   - 正常：带有清甜的果香。  
   - 异常：出现酸味、酒味或腐败味，表明已过熟或霉变。

4. **检查果梗和果尖**  
   - 果梗（连接处）干枯变黑：代表采摘后时间较长。  
   - 果尖（顶部）发黑、凹陷：易从该处开始腐烂。

5. **批量抽检**  
   - 随机拆开1-2箱，翻看底层与中间：最易出现闷坏、挤压伤。  
   - 观察包装箱内：有液体积存或霉斑，代表已有坏果流出汁液。

6. **判断环境与运输**  
   - 如果货箱温度较高（手感温热），且密闭不透气，整体加速成熟，坏果率会上升。  
   - 注意运输时间：超过7天未进冷库的香蕉，风险显著增高。

**高风险信号**：单箱内超过10%的香蕉出现局部软烂、果皮渗水、白霉斑点，建议拒绝收购或要求降价并立即低温处理。
