建议优先考虑**整果+切果组合销售**，重点分析以下几点：

1. **客流与场景**：
   - 人流量大、家庭客群多的门店（如社区店），整果更受欢迎，因为顾客可自行处理、保存时间长。
   - 如果门店位于写字楼、商圈周边（单人/白领客群多），切好的更易售卖，满足即食需求。

2. **损耗控制**：
   - 整果耐储存（常温3-5天），但顾客买回家可能一次吃不完，导致复购降低或浪费。
   - 切好的菠萝保质期短（冷藏约24小时），但能高价销售剩余果肉，减少损耗。建议每日分早/晚两次鲜切，傍晚打折清货。

3. **利润与定价**：
   - 整果单价低，但毛利稳（约20%-30%）。
   - 切盒通常可加价50%以上（如整果10元，切盒15元），且提升客单价。

4. **操作建议**：
   - **备选方案**：货架同时摆放整果和切盒，用价签标明“整果10元”与“鲜切盒15元（含加工费）”。
   - **促销策略**：整果可附赠削皮器或演示视频，切盒可搭配竹签进套餐。
   - **试错法**：先试一个月，统计每日两种品类销量、损耗率、投诉量（如整果腐烂投诉或切盒不新鲜），再调整比例。

**结论**：如果门店日均客流500人以上且附近有居民区，建议整果占60%、切盒占40%；若为外卖/办公区为主，可整果30%、切盒70%。
