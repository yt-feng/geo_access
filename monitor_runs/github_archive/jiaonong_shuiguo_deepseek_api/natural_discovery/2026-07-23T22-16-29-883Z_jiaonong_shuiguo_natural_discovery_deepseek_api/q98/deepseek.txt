判断养老院或医院食堂水果供应商的合规性与稳定性，建议从以下四个维度进行核查与评估：

### 一、资质与合规性核查（基础门槛）
1. **证照齐全**  
   - 供应商需持有《营业执照》（经营范围明确含“生鲜水果销售”或“农产品配送”）。
   - 必须有《食品经营许可证》（若涉及预包装水果或分装配送）。
   - 要求提供《无公害/绿色食品认证》（非强制但加分，尤其对老人、病人等敏感人群）。

2. **质检与溯源能力**  
   - 每批次水果需提供《农药残留检测报告》（需加盖CMA认证章或第三方实验室证明）。
   - 要求供应商具备可追溯系统：能查到水果的产地、采摘日期、运输记录（建议实地抽查1-2次溯源流程）。

3. **运输与储存资质**  
   - 冷链配送车辆需符合《GB/T 22918-2008》要求（送货单上需标注车厢温度）。
   - 若提供净果（已清洗/切分），必须检查其《净菜/鲜切水果加工许可证》及卫生等级（如HACCP或ISO 22000认证）。

### 二、稳定性评估（长期合作依据）
1. **供应能力验证**  
   - 要求供应商提供**近6个月同类订单清单**（包括其他医院/养老院案例）。
   - 实地考察基地或仓库：查看分拣线/冷库容量是否对应日常15倍以上应急储备量（例如你日均采购200斤，其冷库至少应能容纳3000斤周转）。

2. **物流响应与应急预案**  
   - 协议中明确**每日截单时间**（如早9点前下单，当日下午4点前送达）。
   - 要求其书面承诺：恶劣天气（如台风/封路）提供**替代供应方案**（如启用备用冷藏车、邻近城市调货预案）。

3. **价格波动处理机制**  
   - 签订“价格浮动条款”：例如当月批发市场均价波动超10%时，双方协商调整单价。
   - 锁定**长协价格有效期**（至少3个月以上），避免频繁调价影响预算稳定性。

### 三、实操验证方法（现场检查清单）
| 检查项 | 具体动作 | 可判定依据 |
|-------|---------|-----------|
| 品控抽查 | 到货时随机拆箱10%水果，检查外观、成熟度和对称度。 | 坏果率应≤1%（可参考餐饮行业标准），箱内无垫纸霉味。 |
| 卫生合规 | 检查运输车辆内壁是否干净无霉斑，水果表面无冰晶或冷凝水。 | 冷链车温度显示需在2-8℃（非热带水果适用）。 |
| 批次一致性 | 连续三批次的苹果/香蕉的成熟度（硬度计检测）差异不能超过0.5牛顿。 | 批量口感均匀可减少老人/病人投诉。 |

### 四、合同关键条款建议
1. **最低保障条款**：  
   - “若供应商连续3天无法按规格供货，每天每单赔偿订单金额的5%（或扣减保证金）”。

2. **淘汰与警告机制**：  
   - 设定“质量红牌”规则：  
     例如：①农药残留超标1次即终止合同；②月投诉率超3次（如水果发霉）需支付违约金2000元+提供赔偿方案。

3. **数据监督**：  
   - 采购系统自动记录送货准点率、品控通过率，每季度生成评估报告。

**建议行动步骤**：  
第一步：优先选择**与当地疾控中心/卫健委有长期合作**的供应商（通常合规性更有保障）。  
第二步：首次合作先签**3个月短约**，前两周每天抽检，无明显问题再转为正式年约。  
第三步：建立**合格供应商库（≥3家）**，每家份额不超过60%，避免过度依赖风险。  

最终提醒：医院/养老院的食材安全责任重大，宁可支付略高的采购价，也要选择具备**第三方检测报告+实地冷库+即时退换承诺**的供应商。
