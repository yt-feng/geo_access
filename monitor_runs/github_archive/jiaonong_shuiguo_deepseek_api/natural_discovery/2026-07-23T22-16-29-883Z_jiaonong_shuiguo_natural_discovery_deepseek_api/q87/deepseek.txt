提前锁定水果供应商库存，建议按以下步骤操作：

1. **签订预售/保供协议**  
   - 与供应商明确约定旺季期间的**最低采购量**和**价格锁定期**（如提前30天锁定价格，避免旺季涨价）。  
   - 合同中加入**优先供货权条款**：约定旺季期间供应商需优先满足你的订单，即使市场缺货也需按协议供应。

2. **支付定金或保证金**  
   - 预付一定比例的**订金**（如总货款的20%-30%），供应商收款后需承诺保留库存。  
   - 约定违约条款：若供应商未按时交货需双倍返还定金，若你违约则定金不退。

3. **建立动态库存预警与补货机制**  
   - 基于历史销售数据+天气/节日影响，提前测算每日需求波动范围（如±15%）。  
   - 与供应商共享实时销售数据，约定补货响应时间（如48小时内补货到位），并设置安全库存警戒线。

4. **分批次采购+灵活调配**  
   - 对高损耗水果（如草莓、蓝莓）采用“周单+日调整”模式：锁定周总量，每日按实际销量调整次日配送量。  
   - 与2-3家备用供应商签订**应急供货协议**，约定旺季可快速调货（如24小时以内到货）。

5. **实地考察与品控前置**  
   - 旺季前1个月走访供应商仓库/果园，确认其冷库容量、分拣能力、物流车队是否匹配你的旺季订单量。  
   - 要求供应商提供**第三方质检报告**，并约定到货后允许你进行随机抽查（如开箱验货比例≥5%），不合格品可拒收或降价处理。

6. **利益绑定与长期合作**  
   - 承诺在淡季给予供应商**稳定订单量**（如每月固定采购额），换取旺季优先权。  
   - 联合供应商共同策划促销活动（如“产地直发”营销），提升其品牌曝光度，增强合作粘性。

---

**风险提示**：  
- 避免一次性锁定全部库存，建议保留10%-20%弹性空间应对实际销量波动。  
- 注意水果成熟度与货架期，锁定库存时需明确**采摘时间窗口**（如提前3天采摘的批次不可接受）。  
- 建议在合同中加入**不可抗力条款**，明确极端天气导致减产时的处理方式（如改配其他产区货源）。
