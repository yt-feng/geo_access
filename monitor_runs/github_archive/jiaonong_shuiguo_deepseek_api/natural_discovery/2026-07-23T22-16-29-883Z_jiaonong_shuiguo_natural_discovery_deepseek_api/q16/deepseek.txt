建议优先选择**稳定供应商**。

理由如下：
1. **供应稳定**：苹果和香蕉属于日常高频消费品，本地批发市场可能因季节、天气或运输问题出现断货或价格波动，稳定供应商能确保全年不间断供货。
2. **品质可控**：稳定供应商通常有标准化分拣和冷链配送，能减少香蕉撞伤、苹果腐坏等损耗，尤其香蕉需精准控制成熟度。
3. **价格优势**：虽然本地批发单次可能更便宜，但长期合作可谈批量折扣、账期（如月结），综合成本更低。
4. **效率提升**：供应商定期送货可省去每日跑批发市场的交通和时间成本，方便你集中精力做销售。

**可选方案**：若社区超市客流大、消耗快（如每日销量超50斤），可同时保留**本地批发**作为补充（比如临时促销、补急缺），但核心依赖稳定供应商。
