在超市采购水果时，为应对农残抽检风险，建议提前向供应商索要以下资料：

1. **产品检测报告**（农残检测报告）：  
   必须提供近期（一般要求1年内）由具备CMA或CNAS资质的第三方检测机构出具的农残检测报告，明确列出各项农残指标（如有机磷、拟除虫菊酯等）的检测结果，且符合国家强制性标准（如GB 2763《食品安全国家标准 食品中农药最大残留限量》）。

2. **产地证明或产地来源说明**：  
   包括水果的具体产地（种植基地地址）、采摘日期、批次号等信息，便于溯源和跟踪。

3. **供应商资质文件**：  
   - 供应商的营业执照、食品经营许可证（或相关食品流通许可）；  
   - 如为农业合作社或农户，需提供身份证明、产地备案证明等。

4. **种植环节相关资料**（可选但建议）：  
   - 农药使用记录（包括使用农药名称、剂型、使用时间、安全间隔期等）；  
   - 是否使用违禁农药的自我声明或承诺书。

5. **产品合格证明**（如无检测报告）：  
   部分市场或地方要求提供备案的“食用农产品合格证”或“产地准出证明”。

6. **既往抽检记录**（如有）：  
   同批次或供应商以往供货的抽检合格记录，以降低风险。

**建议操作流程**：  
- 在采购合同中明确要求供应商提供上述资料，并将每一项资料作为合同的附件或验收必备文件。  
- 对供应商定期进行资质审核，确保检测报告在有效期内。  
- 到货后，可自行或委托第三方进行快速农残抽检（如酶抑制法速测），作为双重保障。

这样既能提前规避风险，也便于在发现问题时快速溯源和追责。
