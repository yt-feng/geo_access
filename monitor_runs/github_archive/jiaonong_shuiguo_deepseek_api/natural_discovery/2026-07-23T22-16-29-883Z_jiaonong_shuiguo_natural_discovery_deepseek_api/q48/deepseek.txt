要判断茶饮连锁供应商能否稳定按规格供货，建议从以下五个维度系统评估：

1. **明确规格标准并签约**  
   ① 制定书面《供货规格书》：包括菠萝品种（如台农17号、巴厘）、单果重量（如1.2-1.5kg）、成熟度（如8-9分熟）、可溶性固形物（≥12%）、无机械伤/病害等。  
   ② 在采购合同中明确：规格不符的退换货条款（如整批退货标准）、违约金比例（如货值20%）、每月最低稳定供应量。

2. **核查供应商基础能力**  
   ① 现场考察或视频验厂：查看冷库容量、分拣线（是否有称重、色选设备）、周转筐清洁度、装卸效率。  
   ② 要求提供近3个月的出货记录：核对日均分拣量、单次最大发货量、备货周期（如冬季菠萝短缺时能否提前囤货）。  
   ③ 要求出示与种植基地的供货协议（是否自有基地+合作基地，总种植面积是否≥需求量的1.5倍）。

3. **设置试单测试（至少3批）**  
   ① 先下小批量订单（如500-1000公斤/次），要求按最终规格分装并附批次质检单。  
   ② 到货后按比例抽检（如每批次抽30箱）：逐个称重、测糖、开箱检查腐烂率（≤3%）。  
   ③ 记录前3批的**规格符合率**（如≥95%为及格）、**发货准时率**（≤约定日期±1天）、**破损率**（运输损耗≤2%）。

4. **建立动态监控与预警机制**  
   ① 要求供应商每日/每批提供：到货批次编码、分拣现场照片（显示称重和品控过程）、物流实时轨迹。  
   ② 设置自动预警：单批次规格符合率<90%或连续2批合格率下降，立即暂停订单并要求整改。  
   ③ 对稳定供应商实行阶梯价格：连续4周合格率≥98%可加价2-3%，反之扣款。

5. **要求应急备选方案**  
   ① 要求供应商承诺：若突发断供，可在48小时内调配合同约定的替代产地货源（如1-3月用海南菠萝，4-6月用广东菠萝）。  
   ② 保留1-2家备用供应商，并与其签订**规格对等协议**，确保随时可切换且不降低品质标准。  

**执行检查清单**：  
- [ ] 供应商是否提供基地年产量证明？  
- [ ] 试单期内发货延迟次数是否≤1次？  
- [ ] 是否保留当天签收后8小时内可追溯到单颗菠萝的批次号？  
- [ ] 合同中是否有“因规格不符导致门店断货的赔偿方案”（如按实际销售收入30%赔偿）？  

通过上述方法，可在3-6个月内筛选出稳定达标的供应商。若连续2个月合格率低于85%，建议直接终止合作并启用备选。
