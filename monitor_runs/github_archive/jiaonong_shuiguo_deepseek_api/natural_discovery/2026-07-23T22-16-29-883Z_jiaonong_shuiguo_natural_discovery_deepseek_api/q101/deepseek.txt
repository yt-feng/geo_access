和供应商谈酒店早餐水果供应合作时，建议重点关注以下几点：

1. **供应稳定性与季节性调整**：明确供应商能否全年稳定供应（尤其反季水果），以及是否具备根据季节灵活替换品种的方案（如夏季西瓜、冬季柑橘）。

2. **品质标准与验收规则**：提前约定水果的新鲜度（如成熟度、无损伤）、规格（大小、重量）和包装（防压、冷链），并制定退换货标准（如发现腐烂退货时效）。

3. **价格条款与账期**：对比多家供应商的报价（是否含税、运费），约定调价机制（如超市统一调价可协商调整）。账期建议控制在30-45天，避免垫资压力大。

4. **配送频次与响应时间**：根据早餐用量确定每日或隔日配送；要求供应商提供紧急补货措施（如当日断货，需2小时内到货）。

5. **食品安全资质**：要求供应商提供食品经营许可证、水果检测报告（农残、微生物），以及冷链运输记录，确保合规。

6. **合同关键条款**：建议明确违约责任（如断供按订单金额3倍赔偿）、争议解决方式，并加入“环保包装”等附加要求（避免过度塑料）。

可以先从少量试合作开始（如1个月），再根据实际表现签订长期合同。
