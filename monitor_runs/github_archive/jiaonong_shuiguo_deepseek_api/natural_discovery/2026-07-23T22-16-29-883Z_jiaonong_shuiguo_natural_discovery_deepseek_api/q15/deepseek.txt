水果店遇到坏果时，与供应商的结算或赔付方式通常取决于**双方合同约定、行业惯例以及坏果的具体原因**。以下是一般处理流程和常见做法：

### 1. 首先要确定责任归属
- **供应商责任**：如到货时发现坏果（外包装破损、内部腐烂、明显品质不符），属于供货质量问题。
- **运输方责任**：如果是物流导致挤压、磕碰、运输时间过长。
- **店内责任**：如果货物已验收、储存不当或上架时间过长导致腐烂。

**核心原则**：**发现问题越早，越可能得到供应商全额或部分赔付。**

### 2. 常见结算/赔付方式（按严重程度排序）

#### A. 到货后发现明显问题（最理想情况）
- **当场拒收**：如果到货时坏果比例很高（例如超过5%-10%），直接拒绝签收，由供应商承担全部损失。
- **扣除对应重量/金额**：签收前核实坏果总量，在结算单上注明“扣除XX公斤坏果”，只按好果重量付款。
- **按折扣价结算**：协商将整批货物折扣价处理（例如按原价6-7折），但需明确坏果区分责任。

#### B. 入库后或销售前期发现（常见情况）
- **比例赔付**：供应商通常接受按坏果重量退还货款。例如：100公斤苹果，坏果20公斤，直接扣除20公斤货款。
- **折让积分/下次抵扣**：供应商有时会用“赠送等价好果”“下次订货抵扣货款”等替代现金退款。
- **拍照留证**：务必拍下坏果特写、整箱腐烂分布照片，作为举证材料发给供应商。

#### C. 销售后期或已经上架（较难说明）
- **按比例分摊**：如果坏果在销售过程中出现，一般需要判断是否为保鲜期、储存条件所致。部分供应商会承认部分责任（例如坏果数量超过正常损耗率），但力度会降低。
- **店内承担**：如果已超过供应商承诺的保质期或表明储存不当，通常由店家自行承担。

### 3. 需要提前做好的准备工作
- **建立明确的合同或口头约定**：新供应商合作前，建议明确“坏果赔付标准”（如到货24小时内可赔付、超过多少比例可拒收、按箱/按重量核算等）。
- **每次到货检查并拍照**：尤其是易烂水果（草莓、荔枝、桃子、芒果等），到货时打开随机抽检，发现异常立即联系供应商。
- **保存物流单和票据**：用以判断运输途中可能造成的损坏。

### 4. 遇到供应商不认账时的应对
- **提交照片与记录**：清晰展示坏果数量、品牌包装、到货时间。
- **逐步升级沟通**：先找售后或业务员 → 区域经理 → 公司负责人。
- **如长期合作且已大量坏果**：可要求**延长结款周期**或**部分赔钱+赠送补货**。
- **必要时中止合作**：作为商业决策，如果多次出现坏果且不赔付，终止合作比持续损失更好。

### 5. 特殊水果的常用标准参考（行业惯例）
| 水果类型 | 常见坏果赔付标准 |
|---------|----------------|
| 草莓/蓝莓 | 按盒起赔，整盒腐烂全赔；部分发霉按比例 |
| 芒果/香蕉 | 按根/按斤赔，运输中挤压导致爆裂、黑斑基本全赔 |
| 桃子/李类 | 软烂、发霉、水渍按重量扣除 |
| 苹果/梨 | 碰撞伤、腐坏按个/斤赔，表面轻微不影响通常归零售商 |
| 整箱包装有内包装的 | 通常整车/整箱全赔或部分按比例 |

### 行动建议
1. **到货时**：立即检查，现场拍照，视频留底，直接联系供应商。
2. **结算时**：主动提交坏果证据，要求按实际好果重量结算，或扣除相应金额。
3. **常态下**：与供应商签署明确的果蔬品质条款（包含赔付比例、时限）。没有合同的情况下，按行业惯例处理。

如果有具体的坏果照片或采购合同条款，可以进一步帮助你分析最可能的处理方式。
