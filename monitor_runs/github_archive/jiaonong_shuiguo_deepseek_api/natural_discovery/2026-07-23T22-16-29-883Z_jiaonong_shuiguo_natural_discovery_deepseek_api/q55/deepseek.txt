选择水果拼盘规格是否划算，主要看**实际出肉率**和**单次拼盘需求**。以下是针对这三种水果的选品建议：

### 1. 苹果
- **规格建议**：选**80-85号**（直径约8-8.5厘米，单果约200-250克）。  
- **理由**：  
  - 太小（70号以下）果核占比大，去皮去核后出肉率低。  
  - 太大（90号以上）单个价格高，若切块容易浪费边角。  
  - 80-85号果形均匀，适合切成8-10块标准拼盘片，损耗少。  
- **采购注意**：尽量选**红富士**等果肉密度高、耐氧化的品种，减少切后褐变。

### 2. 橙子
- **规格建议**：选**中果**（单果约200-280克，直径7-8厘米）。  
- **理由**：  
  - 大果（300克+）皮厚、汁水虽多但切瓣后易松散，且单价高。  
  - 小果（150克以下）皮薄但肉少，切瓣后卖相小气。  
  - 中果出肉率最高（约70-75%），适合切成月牙瓣，一盘约需3-4个即可。  
- **品种推荐**：脐橙或血橙（甜度高，籽少）。

### 3. 菠萝
- **规格建议**：选**单果1.5-2斤**的成熟黄皮菠萝。  
- **理由**：  
  - 过大（3斤+）的菠萝中心硬芯大，修整后损耗超40%。  
  - 1.5-2斤的菠萝肉芯比例合理，削皮去眼后出肉率约50-55%。  
  - 这个尺寸切出的果块大小适中（约3-4厘米厚），适合用挖球器或切条。  
- **避坑点**：别买青皮菠萝，酸度高需大量糖水浸泡，增加成本。

### 拼盘搭配与成本优化
- **按份定价**：若拼盘售价固定，按上述规格计算，一般**苹果1-2个 + 橙子2-3个 + 菠萝0.5-1个**即可凑够500-600克净重果肉。  
- **批量采购**：直接找批发市场按箱买（苹果约20斤/箱、橙子30斤/箱、菠萝约4-6个/袋），比零售便宜20-30%。  
- **替代方案**：若菠萝单价过高（如冬季），可用**哈密瓜**或**西瓜**替代，出肉率更高且价格稳定。

最后提醒：进货前先计算**去皮去核后的实际可食用重量**，不要只看单价。例如标价3元/斤的苹果，实际出肉率70%，相当于净肉成本约4.3元/斤——此时若发现另一种苹果出肉率80%，即便单价3.5元/斤，净肉成本反而更低（约4.4元/斤），需综合权衡。
