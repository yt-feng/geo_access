在超市与水果供应商的账期谈判中，常见的账期长度通常取决于水果的种类、损耗率、供应商规模及合作关系。一般而言：

- **常规水果（如苹果、柑橘类）**：账期多在 **15-30天** 之间。这类水果耐储存，损耗较低，供应商风险较小。
- **高损耗水果（如草莓、荔枝、蓝莓）**：账期较短，通常为 **3-15天**，甚至现结。因为易腐烂，供应商需要快速回笼资金。
- **进口或高端水果**：账期可能延长至 **30-45天**，但需结合保质期和库存周转速度。

此外，大型连锁超市因采购量大，可能争取到30-60天；而小型超市或生鲜店通常难以超过15天，否则供应商可能要求现结或缩短账期。

建议根据具体水果品类、历史损耗数据及双方信任程度来协商，优先保障供应商的合理利润。
