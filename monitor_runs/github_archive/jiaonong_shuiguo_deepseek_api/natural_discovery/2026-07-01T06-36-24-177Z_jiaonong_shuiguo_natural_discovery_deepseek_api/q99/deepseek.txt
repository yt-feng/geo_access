选择适合批量配送的水果供应商，可参考以下步骤：

1. **明确需求**：先确定每周/次采购的水果种类、数量、配送频率及预算。常见批量水果如苹果、橙子、香蕉、西瓜等耐储存品种。

2. **筛选供应商**：
   - **本地批发市场**：实地考察，价格低但需自行运输或小额配送。
   - **线上生鲜平台（如美菜、快驴）**：专业针对企业，支持批量下单、定时配送，但起送价较高。
   - **果园/合作社直供**：适合大量长期采购，需签订合同，注意起订量。
   - **综合食品批发商**：提供多品类水果，可议价。

3. **关键考察点**：
   - **资质**：营业执照、食品经营许可证（如需）。
   - **配送能力**：是否覆盖公司地址，能否按时送达（如上午下单下午到）。
   - **品质保障**：退货/换货政策、坏果处理（如按比例赔偿）。
   - **价格与结算**：明确单价（含税/不含税）、是否包邮、付款周期（月结/现结）。

4. **试单验证**：先小额采购试用，评估水果新鲜度、包装完整度及物流时效。

5. **建立长期合作**：根据试单结果，选择2-3家备选供应商，签署协议明确服务条款。

**注意**：避免依赖单一供应商，保持备选；优先选择有企业供货经验、支持定期对账的供应商。
