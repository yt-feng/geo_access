在合同中明确坏果责任，可以按以下要点写：

1. **损耗标准**：明确约定“合理损耗率”（例如运输途中自然损耗不超过总货量的3%）。超过标准的坏果，责任由供货方承担。

2. **验收方式**：规定收货时进行现场抽检或全检，并在[XX小时/日内]提出异议（如48小时内）。过期未提出，视为默认合格。

3. **责任划分**：运输途中（物流方责任）、到货后因包装/品质问题（供货方责任）、到货后存放不当（收货方责任）需分开写。例如：“因供货方包装不当或货物本身质量问题导致的腐烂变质，由供货方按实际损失金额赔偿。”

4. **赔偿条款**：写明赔偿计算方式，如“按坏果数量×单价×[1.2~2倍]”或“更换等量合格货物”。明确是否包含运费、人工等间接损失。

5. **举证要求**：要求投保货运险，并在出险时配合提供照片、视频等证据。

示例条款（可直接修改使用）：
> “本合同货物自然损耗率不得高于3%。收货方应于到货后48小时内完成质量检验，如有异议需留存证据并书面通知供货方。因供货方贮藏或包装不当导致的腐烂变质，超出损耗率部分由供货方在7日内按零售价1.5倍赔偿或补发等量货物，货运险理赔款归收货方所有。因收货方存放不当或未及时检验造成的损失，供货方不承担责任。”

建议合同签署前咨询律师，根据实际货物类型（如易腐性、运输周期）调整比例和时间。
