对于小水果店来说，进香蕉时最应优先看重的因素是**熟度**，其次是**坏果率**，然后是**价格**和**送货速度**。具体分析如下：

1. **熟度（最关键）**  
   - 香蕉是后熟型水果，到店后需要时间自然成熟。如果进货时熟度高（例如表皮全黄、有黑点），当天卖不掉就会迅速变软腐烂；如果熟度太低（全青），顾客不会购买，且成熟不均匀会增加损耗。  
   - **建议**：选择大多数为**七成熟（果皮带微黄、果肉硬实）** 的香蕉，这样常温下可保存2-3天，且能按销售节奏自然转熟。

2. **坏果率（直接影响利润）**  
   - 香蕉在运输和批发过程中容易碰伤、压坏或出现“柄裂”和“水烂”。小店铺货量少，进货时若坏果率超过5%，就会明显挤占利润空间。  
   - **建议**：进货前仔细抽检整箱内部（尤其是底部），要求坏果率低于5%-10%，并当场协商扣除或拒收。

3. **价格（与成本相关）**  
   - 香蕉价格波动大（受产地、季节、运输影响），但小店铺规模小，议价能力有限。如果批发价过低，往往意味着果品质量（如熟度不均、有小果或残次）或储存时间偏长。  
   - **策略**：在熟度、坏果率合格的前提下，选择中等偏低价位，避免因极端低价导致高损耗。

4. **送货速度（次关键）**  
   - 香蕉对时间敏感：进货后若搁置过久（超过半天），熟度会进一步增加，影响销售窗口。但一天内送达通常足以应对。  
   - **注意**：如果供应商承诺“上午定下午到”，实际却隔天到货，可能导致熟度超标，因此需确保供应商有稳定、准时的配送能力。

**总结排序**：熟度 > 坏果率 > 价格 > 送货速度。  
建议小店主：**先固定1-2家能提供稳定七成熟香蕉的供应商**，并在每次进货时现场开箱抽检，用“熟度+烂果率”两把尺子筛选，价格和速度作为次要谈判点。
