是的，企业在采购水果礼盒作为员工福利时，通常需要供应商同时支持以下两项服务：

1. **开发票**：企业采购一般需要进行财务入账和税务抵扣，因此建议优先确认供应商能否开具**增值税专用发票**或普通发票（根据企业财务要求选择），且发票内容需明确为“水果礼盒”或相关食品类别。

2. **定制包装**：如果需要体现企业文化、节日主题或员工个性化需求（如增加企业Logo、贺卡等），建议明确要求供应商支持包装定制。部分供应商可能会收取额外费用，需要提前确认数量和设计成本。

**补充建议**：
- 提前与供应商沟通开票资质（如是否具备一般纳税人资格），以及可开具的发票类型和税率。
- 定制包装时，确认设计周期和最低起订量，避免影响采购进度。
- 若企业有预算限制，可先询问供应商是否提供“基础包装”和“定制包装”两种方案，按需选择。

如果供应商无法同时满足这两项需求，建议更换供应商或分步操作（如分开采购水果和包装服务）。
