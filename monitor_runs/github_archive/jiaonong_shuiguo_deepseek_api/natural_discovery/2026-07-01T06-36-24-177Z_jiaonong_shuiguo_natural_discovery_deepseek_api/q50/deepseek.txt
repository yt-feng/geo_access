对于茶饮店选择菠萝原料，通常**出汁率比甜度更重要**，但两者的优先级需结合具体用途和成本平衡。以下是详细分析：

1. **出汁率优先的场景**  
   - **制作果汁基底/果茶**：出汁率直接决定原料成本，高出汁率能减少菠萝用量，降低每杯饮品的原料成本。  
   - **需大量果肉的果茶**：出汁率低可能导致果肉浪费（例如，果渣中残留果汁多），增加操作复杂度。  
   - **标准化出品**：出汁率稳定可让配方汁量更可控，避免饮品口感波动。

2. **甜度优先的场景**  
   - **直接做水果茶/冰沙**：菠萝本身甜度影响是否需要添加额外糖浆，高甜度菠萝可减少糖的使用，符合健康趋势。  
   - **与低甜水果搭配**（如柠檬、青柠）：高甜菠萝可平衡酸感，简化调味流程。  
   - **主打“自然甜”概念**：若饮品强调无添加糖，菠萝的甜度直接影响成品表现。

3. **实用决策建议**  
   - **测试出汁率**：购买前要求供应商提供样本，用厨房秤称重并榨汁，计算果肉与果汁的比例。若出汁率低于40%，需谨慎。  
   - **甜度参考**：对于茶饮，菠萝折光甜度（白利度）在12-14°Brix足够，过高可能腻口，过低则需加糖。  
   - **平衡选择**：优先选**出汁率≥50%、甜度≥12°Brix**的菠萝品种（如台湾金钻、泰国小菠萝），易电商平台订购可备注“茶饮专用”。  
   - **成本核算**：假设200g果肉做成1杯饮品，出汁率50%可得100ml果汁，若出汁率降至40%，需用250g果肉——额外增加25%成本。甜度差1°Brix≈每杯节省3-5g糖（成本约0.1元），而出汁率差异的成本影响可达0.5-1元/杯。

**总结**：如果预算是首要限制，**先用出汁率筛选供应商**（淘汰低出汁率品种），再在同档次中选甜度较高的。同时注意，菠萝的品种、成熟度（过熟出汁率下降）、新鲜度（储存时间长出汁率降低）都会影响这两个指标，建议到批发市场现场测试后再批量订购。
