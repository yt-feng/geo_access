判断水果批发商进香蕉时供应商能否按时到车，建议从以下几个关键点入手：

1. **查看历史信誉与记录**：询问同行或行业内是否有合作过该供应商的反馈，特别是准时到货的记录。可以要求供应商提供近期几笔交易的物流记录（如运单号、发车时间截图）作为参考。

2. **签订合同并明确条款**：在合同中明确约定到车时间、超时罚款（如“超过XX小时未到，每日按货值的X%赔偿”），并注明不可抗力除外。有约束力的合同能降低违约概率。

3. **要求实时物流追踪**：要求供应商提供车辆GPS定位或物流平台（如货拉拉、运满满）的实时轨迹，或者每天发送一次车辆位置照片。对于长途运输，可以要求分段确认（如“装车后发车图、中途特定节点确认”）。

4. **观察供应商的应答与沟通效率**：提前联系时，如果供应商对车辆状态、发车时间、司机联系方式等回复及时且具体（如能说出具体车牌号、司机姓名、预计到达时间），说明调度能力较强；反之，含糊或拖延答复的应警惕。

5. **试用小批量订单测试**：初次合作或对供应商有疑虑时，先下小额订单（如1车香蕉），观察其是否按约定时间到车。若有延迟，则需重新评估。

6. **考虑季节性因素与备选方案**：香蕉运输易受影响（如雨天路滑、产区拥堵）。可要求供应商准备1-2辆备用车辆，或自己联系1-2家备用物流公司作为补充。

如果供应商多次无法提供上述任何一项信息，或有明显推诿，建议更换供应商。始终优先选择有长期合作记录且信誉良好的对象。
