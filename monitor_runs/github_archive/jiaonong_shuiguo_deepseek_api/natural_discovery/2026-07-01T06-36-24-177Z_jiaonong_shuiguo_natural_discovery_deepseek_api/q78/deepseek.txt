针对社区团购中香蕉和菠萝等易损耗水果的售后控制，建议从以下四个环节入手，形成闭环管理：

1. **采购与分拣**  
   - 选7-8成熟度的青香蕉（避免全熟后运输擦伤），菠萝选硬度适中、无黑心的品种  
   - 到货后立即逐颗检查：香蕉捏柄部判断软硬度，菠萝按压表皮无软斑  
   - 分装时使用带气孔的瓦楞纸箱/泡沫套袋，避免挤压

2. **物流与存储**  
   - 香蕉要求12-14℃、菠萝8-10℃冷链运输（避免温度剧烈波动）  
   - 到仓后立即散开摆放，严禁堆叠（香蕉用保鲜膜包裹根部，菠萝切口涂盐防褐变）  
   - 建议凌晨到货、早晨配送，缩短中间停留时间

3. **销售与交付**  
   - 页面标注「生鲜类产品时效」：如「收到后当天食用最佳」  
   - 设置套餐搭配：香蕉+其他耐储水果（如苹果），菠萝建议切块配送  
   - 团长收货时需检查外观，发现腐烂立即拍照退单（可设置1小时内无理由退换）

4. **售后处理**  
   - 明确「腐烂3颗以上全额退款」「单颗腐烂按比例补发」的标准  
   - 建立「主动跟单」机制：开团后第2天团购群发送「反馈提醒」，附简易退赔二维码  
   - 统计高频问题（如香蕉折断、菠萝酸涩），每月调整选品策略和分拣标准

**关键指标参考**：  
- 损耗率控制目标：香蕉≤5%，菠萝≤3%  
- 售后响应时效：5分钟/15分钟解决  
- 用户复购率反馈：对高质量售后用户，可赠送下次订单的「水果保护剂」（小份盐/保鲜袋）提升粘性
