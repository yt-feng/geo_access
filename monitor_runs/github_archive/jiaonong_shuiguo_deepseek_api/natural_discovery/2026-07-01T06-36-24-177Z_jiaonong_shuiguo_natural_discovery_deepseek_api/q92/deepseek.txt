针对小面积便利店的实际情况，控制水果品类和损耗可以从以下几个可落地的方法入手：

**1. 品类选择：做“减法”与“复购”**

-   **严禁铺全品**：不要卖像西瓜、菠萝、柚子这样体积大、需切开的“大果型”。
-   **主攻“高频硬通货”**：只选**3-5种**日常需求大、周转快的品类。推荐组合：香蕉（带话题度、易撑门面）、苹果/梨（耐储存）、当季小果（圣女果/砂糖橘）、提子/蓝莓（高毛利）。
-   **拒绝非标品**：避免采购容易压伤、成熟度参差不齐的桃子、草莓等。选择本身就有自然包装（如网袋装橙子、盒装蓝莓）的品种。

**2. 采购与到货管理**

-   **定量采购**：每日到货量严格控制在“当日预估销量的1.2-1.5倍”以内。宁可下午卖完断货，也不要留库存过夜。
-   **验收标准**：到货时必须逐箱打开检查。拒收任何有腐烂、发霉、机械伤（碰伤、压伤）的水果。注意香蕉不能有青头、苹果不能有明显的磕碰。
-   **优先短保质期**：选择产地近、成熟度适中的批次，避免采购需要长时间储存才能转熟的（如生硬的猕猴桃）。

**3. 陈列与保鲜**

-   **少摆勤补**：台面只摆放**2-3层**，约半小时可售完的量。货架下面用库存箱存放，人流少时再补货。
-   **坏果即时隔离**：设立“坏果/次品区”。一旦发现单个腐烂水果，立刻移除并清理周边接触过的水果（霉菌会扩散）。部分微损伤但可食用的，可打折或加工（切成果切）。
-   **冷柜+常温混合**：香蕉、芒果、哈密瓜不要进冷柜（会冻伤变黑）。苹果、梨、提子、圣女果直接放进**风幕柜**。夏天所有易熟水果（牛油果、芒果）必须进冷藏。

**4. 损耗控制与止损**

-   **每日销售倒计时**：下午4点后，对当天没卖完的易损品（如草莓、蓝莓、葡萄）直接进行**9折-7折-5折**阶梯折扣。晚上7点后，将不新鲜的水果加工成**果切杯**（混合装）或**水果沙拉**，按份销售。
-   **处理“坏果”**：无法食用的烂果必须及时称重记录损耗量，并直接丢弃，不要留在店内散味或引诱虫蝇。
-   **库存周转**：所有水果严格遵循**先进先出**原则。在冷库或台面，新货放在底部/后面，旧货放在顶部/前面。

**总结口诀**：
**除了香蕉和苹果，其它能盒装别散装；每天只进一天量，下午促销清完场。**
