社区超市控制水果新鲜度与库存的关键在于“以销定进、动态周转”。以下是可执行的策略：

1. **分类分级采购**  
   - **高周转品**（如香蕉、叶菜、草莓）：每日少量多次进货，重点放在早晨和傍晚补货。  
   - **耐储品**（如苹果、橙子、哈密瓜）：每周进2-3次，根据前7天销量×1.2倍订货。  
   - **应季品**（如荔枝、芒果）：按天预估销量，首日进预估量的80%，第二日根据实际销售补货。

2. **陈列与保鲜**  
   - **先进先出**（FIFO）：新货放下层/后侧，旧货摆上层/前侧。  
   - **水果分类温度**：  
     - 常温货架（香蕉、芒果、木瓜）：15-25℃，注意避免阳光直射。  
     - 冷藏展示柜（蓝莓、草莓、葡萄）：0-4℃，用透气保鲜盒装。  
     - 冷库（苹果、梨）：-1~1℃。  
   - **每日翻检**：营业前清理软烂、发霉水果，立即打折或加工成果切、果汁。

3. **库存预警与促销**  
   - **设定安全库存**：每个品种保留1-2天的日常销量库存（如日销20斤的苹果，库存保持在30-40斤）。  
   - **实时监控动销**：  
     - 上午10点：销量未达当天目标50% → 启动“限时特价”“买一送一”。  
     - 下午5点：剩余库存超过20% → 包装成果切盒或搭配鲜奶销售。  
   - **数据复盘**：每周统计损耗率（正常应<3%），淘汰连续3周库存积压或损耗超10%的品种。

4. **到货验收标准**  
   - 验收时抽检10%：  
     - 硬果（苹果、梨）无软斑、压痕  
     - 浆果（草莓、蓝莓）无霉变、渗汁  
     - 热带水果（芒果、木瓜）轻按微弹为佳  
   - 拒收预成熟度过高或运输破损的货品。

5. **应急处理流程**  
   - 极端天气：提前3天减少订货量20%-30%。  
   - 节假日：提前5天根据去年同期销量×1.5备货，但预留“临时追加订单”渠道。  
   - 突发损耗：立即分拣出次品，按“买水果赠次品果切”或“成本价促销”处理。

**示例操作**：  
- 假设日销香蕉20斤，每次进货40斤最佳（可卖2天），存到冰箱的货提前1天取出自然成熟。  
- 每天早晨营业前，检查香蕉是否有黑点：长黑点但果肉无恙 → 立即调价至9折先卖；出现水渍状软斑 → 制成香蕉牛奶（搭配牛奶销售）。  

**关键指标**：  
- 库存周转天数：建议水果控制在1.5-3天。  
- 损耗率：控制在2%以内，超出需回溯原因。  

通过以上措施，既能减少压货，又能维持新鲜度，同时保有一定的利润空间。
