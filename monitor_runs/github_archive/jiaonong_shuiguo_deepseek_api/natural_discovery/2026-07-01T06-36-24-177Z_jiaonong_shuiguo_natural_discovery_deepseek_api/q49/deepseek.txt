针对茶饮店旺季菠萝用量突然增大的情况，建议按以下步骤与供应商沟通临时补货：

1. **提前联系**：不要等到库存耗尽再联系，要主动告知供应商“旺季销量激增，菠萝库存紧张”，并确认对方目前的供应能力。
2. **明确需求量**：给出具体的补货量（如“本周需要额外100公斤”），并说明预计补货周期（如“希望分3次到货，每次30-35公斤”），避免笼统说“多要一些”。
3. **协商价格与品质**：
   - 如果是临时追加，可询问供应商是否能维持原价；若对方因缺货或加急要提价，可以协商“能否优惠一些，或本次按原价，下次大宗采购再调整”。
   - 约定品质标准，例如“要求成熟度7成、无腐烂、单果重量不低于300克”，避免以次充好。
4. **确认交付细节**：明确时间、地点、运输方式（如是否需加急配送，加急费谁承担）。最好让对方书面或文字确认。
5. **建立预警机制**：同步告知供应商“后续可能持续增加”，并请对方定期反馈库存和价格波动，方便你提前计划。
6. **保留留底**：所有沟通结果（微信、邮件、订单截图）保存，作为后续核对或纠纷凭证。

示例话术：“王经理，最近店里菠萝销量突然翻倍，想临时追加100公斤，分3次送到，第一次明天到货。价格能不能按之前的合同价？我们会优先和你合作长期订单。”
