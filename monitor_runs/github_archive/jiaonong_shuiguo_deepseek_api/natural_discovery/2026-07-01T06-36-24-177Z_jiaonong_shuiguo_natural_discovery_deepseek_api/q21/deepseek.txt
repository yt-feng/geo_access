在超市采购水果时，供应商通常需要提供以下检测报告或进货证明，以确保水果的安全性和可追溯性：

1. **产地证明**：显示水果的来源地（如种植基地、产地合作社等），证明水果的合法来源和产地信息。
2. **农药残留检测报告**：由具备资质的第三方检测机构出具，证明水果的农药残留符合国家食品安全标准（如GB 2763《食品中农药最大残留限量》）。
3. **重量或规格证明**：部分超市要求提供水果的规格（如重量、大小、成熟度等）符合合同约定的证明。
4. **检疫证明**：对于进口水果，需提供中国海关出具的入境货物检验检疫证明，或出口国官方出具的检疫证书，证明水果无检疫性有害生物。
5. **运输和储存记录**：包括冷链运输的温度记录、储存条件证明等，确保水果在运输和储存过程中未变质或污染。
6. **供应商资质文件**：如营业执照、食品经营许可证或食品生产许可证（若供应商是种植基地或加工企业）。
7. **销售发票或合同**：证明供应商与超市之间的交易合法、合规。

具体所需文件可能因水果类型（如鲜果、冷冻水果、加工水果）、产地（国内或进口）及超市自身采购政策而略有差异。建议与超市采购部门确认详细要求，并优先选择有良好信誉的供应商。
