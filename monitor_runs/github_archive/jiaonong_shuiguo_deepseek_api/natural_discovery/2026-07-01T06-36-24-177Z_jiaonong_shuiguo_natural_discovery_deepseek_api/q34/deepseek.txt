在商超采购水果时，**损耗对最终毛利的影响通常大于价格低带来的影响**。原因如下：

1. **损耗直接削减毛利**：水果属于易腐商品，损耗（如腐烂、磕碰、过熟）会导致实际可售数量减少。即使采购价格低，若损耗率过高（例如达到15%-20%），已售商品的利润会被未售出商品的成本摊薄，甚至导致整体亏损。

2. **价格优势有限**：采购价格低通常只节省单次进货成本，但水果毛利率本身较低（一般20%-40%），而损耗率每增加1个百分点，毛利可能下降1.5-2个百分点（取决于定价和促销策略）。例如，若采购价降低10%，但损耗从10%上升到20%，实际毛利可能反而不如高价低损耗的方案。

3. **运营建议**：
   - **优先控制损耗**：优化冷链运输、分拣标准、库存周转率（如小批量多频次进货）、适时打折处理临期商品。
   - **平衡价格与损耗**：选择供应商时，不要单纯追求最低价，需综合评估其品控、成熟度（如青果或半成熟果损耗低但影响口感）和配送时效。
   - **数据监控**：建立损耗率与单品毛利的联动分析表，明确不同水果的“可接受损耗阈值”（通常叶菜类<10%，核果类<5%）。

综上，**损耗控制是毛利管理的核心杠杆**，而价格优势需在保证损耗率不超预期的前提下才能有效提升利润。
