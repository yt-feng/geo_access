选择社区超市的每日水果供应商时，建议重点关注以下条件：

1. **供应稳定性**：供应商能否保证每日按时、足量配送，避免断货或延迟。
2. **水果品质**：新鲜度、外观、口感是否符合要求，建议实地查看或要求样品。
3. **品种多样性与季节性**：能否根据季节调整水果种类，提供常见及应季品种。
4. **价格与结算方式**：批发价是否合理，是否有账期或折扣政策（如批量优惠）。
5. **配送范围与时效**：是否覆盖你所在的社区，配送时间是否灵活（如早间或晚间）。
6. **包装与损耗控制**：水果包装是否防碰撞，损耗是否由供应商承担。
7. **资质与卫生**：是否有正规营业执照、食品流通许可证，仓储和运输是否符合卫生标准。
8. **退换货政策**：如遇质量问题，供应商是否支持退换或补偿。
9. **长期合作关系**：是否愿意签订长期合同，提供稳定价格和优先供货权。
10. **沟通与反馈机制**：是否有专属客服或对接人员，能否及时处理问题和调整需求。

建议先筛选2-3家备选供应商，进行试配送或实地考察，评估实际表现后再签订合同。
