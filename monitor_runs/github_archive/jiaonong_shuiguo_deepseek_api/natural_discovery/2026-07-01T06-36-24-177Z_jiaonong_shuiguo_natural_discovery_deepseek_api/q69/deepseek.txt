为批发商向小店供货时，香蕉和菠萝的分级报价通常按照**果品外观、成熟度、重量/尺寸**等核心维度来划分。以下是清晰、可执行的分级与报价方案，供你直接使用或调整：

### 一、香蕉分级标准与报价逻辑

香蕉主要按**果把大小、果指长度、成熟度、外观瑕疵**分级。

| 等级 | 核心标准 | 常见报价策略（参考） |
|------|----------|---------------------|
| **特级** | - 果指长度 ≥ 20厘米<br>- 果把整齐，无擦伤、黑斑<br>- 成熟度7-8成（便于运输和催熟） | 价格最高，如 **5元/斤** 或 **XX元/箱**，附加催熟服务可加价 |
| **一级** | - 果指长度 16-20厘米<br>- 少量轻微机械伤，无明显黑斑<br>- 成熟度8-9成 | 较特级低 **10%-15%**，常见的备货级别 |
| **二级** | - 果指长度 ≤ 15厘米<br>- 有少量黑斑、果把松散<br>- 成熟度较高（接近全熟） | 价格接近成本价或低于一级 **20%-30%**，适合小店特价促销 |
| **次品/处理品** | - 严重碰伤、发黑、腐烂、果头断裂 | 单独定价，通常1-2元/斤，或按件低价整包出售 |

> **报价示例**：建议对小店报价时，直接说：“特级香蕉20cm以上，今日XX元/箱（约18斤净果）；一级18cm左右，XX元/箱；二级次品处理价XX元/斤，不保品质。”

### 二、菠萝分级标准与报价逻辑

菠萝主要看**果型、重量/大小、芽眼深度、成熟度**。

| 等级 | 核心标准 | 常见报价策略（参考） |
|------|----------|---------------------|
| **特级** | - 单果重量 ≥ 1.5公斤（约3斤）<br>- 果型端正，芽眼浅，无机械伤<br>- 成熟度适中（表皮金黄带绿，手弹声音清脆） | 价格最高，如 **6-8元/个**（或按斤3-4元/斤），标注“精装、大果” |
| **一级** | - 单果重量 1.0-1.5公斤（2-3斤）<br>- 果型基本端正，少量轻微擦伤<br>- 成熟度OK | 比特级低 **15%-20%**，如 **4-6元/个** |
| **二级** | - 单果重量 0.8-1.0公斤（1.5-2斤）<br>- 果型不圆、芽眼稍深<br>- 表皮可有轻微青斑（可放置后熟） | 性价比最高，如 **3-4元/个**，适合小店走量或特价 |
| **三级/处理果** | - 霉变、严重压伤、裂果、过熟有酒味 | 按斤或论堆卖，**1-2元/个** 或更低价 |

> **报价示例**：统一按个报价更直观：“特级菠萝3斤以上，6元/个；一级2.5斤左右，5元/个；二级2斤左右，3.5元/个；裂果次品1.5元/个，当日售出不退。”

### 三、给小店报价时的实用建议

1. **统一计价单位**：
   - **香蕉**：建议按“斤”或“箱（净重）”报价，因为重量波动大。如果按箱，必须标注“一只箱约XX斤净果”。
   - **菠萝**：推荐按“个”报价（分级时已固定单果重量区间），小店和顾客都容易理解。

2. **自动分级术语**：直接使用“大果”“中果”“小果”或“精品”“普通”“特惠”等通俗词，比“特级”“一级”更易被小店老板记住。

3. **设置梯度价格**：如果小店要货量大（如香蕉一次20箱以上），可以在分级报价基础上再额外给予**5%-10%的批量折扣**，写在报价单上。

4. **必须包含说明**：每次报价附带一句话写明“分级依据”（如“所有菠萝单果重量误差在±0.2斤内”），减少纠纷。

5. **灵活处理次品**：可主动提出“买一批一级/特级香蕉，附赠一箱处理品做试吃或促销”，帮助小店消耗次品。

**最终报价单样例（可直接发给小店）**：
```
【今日香蕉报价】
- 精品大果（20cm+）：95元/箱（18斤净果）  
- 标准中果（16-20cm）：80元/箱  
- 特价小果（15cm左右）：60元/箱  
- 处理果（黑斑碰伤）：30元/箱，离场不保

【今日菠萝报价】
- 大果（3斤+）：6元/个  
- 中果（2.5斤左右）：4.5元/个  
- 小果（2斤左右）：3元/个  
- 裂果次品：1.5元/个，当天售完

注：以上价格均为离场价，整批取货再享每箱/每个立减0.3元。
```
