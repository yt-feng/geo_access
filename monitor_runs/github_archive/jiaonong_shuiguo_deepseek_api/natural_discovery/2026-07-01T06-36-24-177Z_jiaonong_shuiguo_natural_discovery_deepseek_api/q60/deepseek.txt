遇到水果到货不合格，建议你与供应商明确以下补货约定，以保障营业不受影响：

1. **预先约定质量标准**：在合同中写明水果的验收标准（如成熟度、大小、无腐烂、无损等），并附上可量化的指标或样品照片。

2. **立即反馈与应急方案**：规定到货后1-2小时内完成验收，发现不合格立即拍照、录像取证，并通知供应商。同时要求供应商在**4-6小时内**提供替换品或调货，避免断货。

3. **临时调货机制**：约定如果快速补货无法实现，供应商须提供附近合作商户或批发市场的调货渠道，保证当日能用上合格水果。

4. **补货期限与惩罚**：明确补货必须在**当天或次日营业前**完成。若超时，按缺货比例扣减货款或由供应商承担营业损失（如预估的销售额损失）。

5. **备用供应商**：建议至少储备一个备用水果供应商的合同，一旦主供方频繁出问题，可立即切换。

6. **合同条款示例**：可以在协议中写“如单次不合格率超过10%，供方须在X小时内补送合格水果，否则按该批次货款的XX%作为违约金，并赔偿因此造成的营业额损失（按前一周日均销售额计算）”。

优先选择有现货周转能力、响应快的供应商，减少对库存的依赖。
