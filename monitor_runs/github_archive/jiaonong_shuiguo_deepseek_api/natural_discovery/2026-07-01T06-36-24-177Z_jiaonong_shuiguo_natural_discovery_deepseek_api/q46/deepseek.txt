建议优先采购**菠萝果泥**，其次为**冷冻菠萝果丁**，鲜果不建议直接用于批量果茶制作。原因如下：

1. **果泥** — 工厂已去除菠萝芯和纤维，质地均匀，便于直接加入果糖或茶汤中，出杯快、口味稳定，不易氧化变黑。适合快节奏门店。
2. **冷冻果丁** — 保留果肉嚼感，但需提前解冻或熬煮，去芯和切丁工序会延长备料时间。适合追求颗粒感的中高端产品线。
3. **鲜果** — 鲜菠萝含有菠萝蛋白酶，直接打汁或切丁容易使茶汤产生浑浊、苦味，且成熟度不一，品质难控。仅建议用于现场切配展示或极少量现制产品。

购买时请确认所购产品为**无糖或低糖基础果泥/果丁**，避免成品含添加糖影响你自身配方。
