在中国超市行业中，与水果供应商谈账期，**常见的天数通常在30天到60天之间**。具体天数往往取决于以下几个关键因素：

1. **超市规模与渠道话语权**  
   - 大型连锁超市（如永辉、沃尔玛）通常要求账期较长，常见为**45-60天**，甚至部分会谈到**90天**。  
   - 中小型超市、社区生鲜店账期较短，多在**15-30天**，因为供应商更依赖快速回款。

2. **水果品类特性**  
   - **易损耗、保质期短**（如草莓、蓝莓、热带水果）：
     - 供应商通常希望账期**7-15天**，甚至现结（T+0或T+1），以避免坏果风险与资金压力。  
   - **耐储存、周转快**（如苹果、柑橘、西瓜）：
     - 超市可能争取**30-45天**，供应商也愿意接受，因为损耗率较低。

3. **供应商实力与供货关系**  
   - **品牌大供应商或独家代理**：可能争取到**30天以内**，因为超市依赖其稳定品质与货源。  
   - **小型批发商或农户直供**：通常接受**7-15天**，否则可能放弃合作。

**实操建议（按优先级排序）：**  
- **初期谈判：** 先从**30天**起步，观察供应商反应后再调整。  
- **博弈技巧：**  
  - 若供应商要求短账期，可承诺**缩短结款时间但提高进货量**（如量价挂钩）。  
  - 对易坏水果，建议要求**售出后结算（滚结）**，而非固定账期，例如：每周结算上周售出部分。  
- **行业参考：** 根据中国连锁经营协会数据，2023年生鲜类平均账期约为**35天**，超市可参考此底线。

**提醒：** 账期过长（如超过60天）可能导致供应商抬高报价或降低供货优先级，最终反而不利于超市获得性价比高的商品，需平衡现金流与供应链稳定性。
