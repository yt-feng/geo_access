选择学校食堂水果供应商时，建议按以下步骤操作：

1. **资质审核**：供应商必须具备营业执照、食品经营许可证、农产品合格证明（如检测报告），确保符合学校采购合规要求。

2. **考察源头与配送**：
   - 优先选择本地或周边产区的供应商，减少运输时间，保证水果新鲜度（如苹果、橙子可耐储存，香蕉需控熟度）。
   - 确认供应商是否具备冷链运输能力，尤其是夏季高温时。

3. **对比样品与报价**：
   - 要求供应商提供2-3次样品，检查香蕉的成熟度（无过量药水催熟）、苹果表皮自然光泽（无打蜡）、橙子果肉水分及酸度。
   - 按“周度或月度”询价，关注季节性价格波动（如香蕉冬季价格波动大，橙子秋冬季价格低），避免长期固定价。

4. **注意品控要求**：
   - 要求供应商按规格分级（如苹果直径8-10cm、香蕉7-9成熟），避免掺入次果或伤果。
   - 约定退换货条款：如损坏率超过5%可无条件补发或退款。

5. **参考其他学校案例**：联系本地类似规模学校的总务处，获取其供应商名单及合作评价。

**示例操作**：可实地走访当地最大的农产品批发市场（如北京新发地、上海江桥市场），筛选3家有长期学校供应经验的档口，对比报价、配送频率（每周2-3次）及售后响应时间，最终签订短周期试用合同（如1个月）。
