销售小包装香蕉时，采购环节注意以下几点，可以有效降低损耗、提升销售：

1. **成熟度控制（最关键）**  
   应选择 **七成熟**（果皮绿中带微黄，果肉硬实）的香蕉进行包装。避免采购全青（难催熟、消费者等不及）或过熟（出现黑斑、易压烂）的香蕉。建议与供应商约定“到店后2-3天可自然转黄”的标准。

2. **包装材料与方式**  
   - **透气性**：必须使用带透气孔的食品级塑料袋或盒装（如打孔袋、带透气孔的PP盒），避免密封导致“焗熟”加速腐烂。  
   - **单串重量**：建议每包 **300-500克**（约2-3根），符合单人食用量。过重容易吃不完放坏，过轻利润低。  
   - **保护措施**：盒装需用珍珠棉或纸托固定蕉柄，防止搬运中晃动碰伤；袋装尽量直放（不横躺），避免受压。

3. **损伤筛选**  
   每根检查：  
   - **无机械伤**：擦伤、压痕、划伤会迅速发黑腐烂。  
   - **无冻伤/热伤**：表皮暗黑但果肉未软的“冷害”或表皮发灰的“热伤”不可买。  
   - **果柄无断裂**：断柄处易滋生霉菌。

4. **贮藏与物流要求**  
   - **恒温运输**：要求13-14℃冷链到店，避免高温加速变熟或低温（低于11℃）导致“冷害”——果皮无法正常变黄，发灰变黑。  
   - **到货时效**：从采摘到包装到店，最好不超过48小时。

5. **损耗预判与拒收标准**  
   - **拒收**：包装内已有超过10%的香蕉出现黑斑、霉点、软烂。  
   - **接受微绿**：全部发黄的香蕉意味着2天内可能集体过熟，反而需要购入一部分微青的，分批次自然成熟销售。

6. **建议测试成本**  
   初期少量采购，观察3天：  
   - 记录每天“正常销售”“打折处理”“丢弃”的占比。  
   - 调整成熟度：若第二天仍全青无人买，下次选更黄一些；若第三天大量变黑，则下次选更青一些。

总结口诀：**七分熟、透气包、单串小、伤疤无、冷链到、试错卖**。
