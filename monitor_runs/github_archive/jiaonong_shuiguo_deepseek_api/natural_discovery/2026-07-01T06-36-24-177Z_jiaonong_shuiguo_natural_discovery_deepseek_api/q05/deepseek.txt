建议您采用**让顾客自己挑，现场削皮**的模式。这是目前水果店最主流、最平衡的做法，原因如下：

1.  **提升信任与复购**：顾客自己挑选，能直观检查菠萝的成熟度、新鲜度以及是否有坏果，这比直接买剥好的或预包装的更能建立对您店铺的信任。
2.  **个性化选择**：顾客可以根据自己吃水果的时间（现吃、囤货）来选择不同成熟度的菠萝，满足不同场景需求。
3.  **降低损耗与人工成本**：
    -   **整果销售**：无需提前削皮，避免了削皮后卖不掉导致的变质损耗。
    -   **现场削皮**：只在顾客确定购买时再操作，避免大量预削皮卖不完。但需要配备一个熟练的削皮师傅，并提前准备好削皮工具和打包袋。
4.  **避免库存风险**：不会出现“顾客挑了A，但把削好的B剩在冷柜里”的退货/损耗问题（除非您有专门的无理由退换货承诺）。

**具体执行建议：**

-   **核心流程**：在店内显著位置陈列出整果，旁边立一个小牌子或由店员口头告知：“**我们是现买现削，您自己挑好菠萝，我们免费帮您削好皮。**”
-   **卫生与差异化**：如果您的成本允许，想提升档次，可以推出“**无菌包装的现削菠萝切块/菠萝条**”（用食品级透明盒包装，标明削皮时间），放在冷柜里供有急事或怕麻烦的客人直接买。但这一部分量不宜太多，建议占总库存的20%以内。
-   **展示与引导**：将整果按大小、新鲜度分层堆头，用价格签清晰标注品种（如：都乐金菠萝、普通菠萝等）。在整果旁边可以放一块切开的菠萝样品，供顾客闻香和触摸。
-   **售后保障**：如果顾客对削好的菠萝品质（如内里有烂心、发霉）不满，不论是否他们自己挑的，都应立即无条件更换或退款。

**避免的做法：**

-   只卖**整果**（不提供削皮服务）——这样会把绝大多数想现场吃、怕麻烦的顾客推给旁边的竞争对手（比如超市或水果批发市场）。
-   只卖**预先削好的**——损耗极高，且顾客无法确认品质（除非您的店铺非常有品牌信誉，且价格特别低廉）。

**总结：**
以“**整果陈列 + 顾客自挑 + 免费现场削皮**”为主，可辅以少量“预削包装”服务满足不同人群。这是目前市场上验证下来效益最好、顾客满意度最高的做法。
