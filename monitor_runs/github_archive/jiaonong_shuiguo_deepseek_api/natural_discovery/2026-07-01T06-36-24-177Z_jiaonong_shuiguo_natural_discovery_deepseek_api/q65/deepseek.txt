对于水果批发商来说，建议采用**短期浮动议价与中长期合同相结合**的策略，具体根据以下情况选择：

1. **议价（适合以下场景）**：  
   - **高波动品种**（如荔枝、樱桃等时令强、耐储性差的水果）：每日或隔日根据市场供需、运输成本、天气变化议价，可灵活应对短期风险。  
   - **库存周转快、资金压力大**：快速调整价格避免积压，减少亏损。  
   - **客户稳定、信息渠道灵通**：能快速获取行情并沟通调整，避免僵化合同造成的损失。

2. **签一段时间价格（适合以下场景）**：  
   - **耐储性强、供需相对稳定**（如苹果、柑橘等冷库储存期长的水果）：签订1-2周或月度合同，减少每日决策压力，锁定基础利润。  
   - **与大型客户/长期合作方**：通过长期合同保障基本订单量，同时约定价格调整条款（如“价格上涨超过15%可重新协商”）。  
   - **需稳定供应链**：若自身采购量较大，与种植户或一级批发商签订远期合同，对冲价格波动。

**执行建议**：  
- **核心部分签合同**：将60%-70%的常用库存（如苹果、梨）签短期合同（1-2周），其余高波动品种每日议价。  
- **设置价格波动缓冲**：合同中加入价格调整触发条件（如“若某水果价格周涨幅>10%，双方重新议价”）。  
- **关注数据**：每日查看全国批发市场均价、产地天气、节日需求变化（如春节前价格普遍上涨）。  

**简单判断方法**：  
- 如果某水果产地到销售地运输需3天以上、且不耐存放（如草莓、枇杷）→ **每日议价**  
- 如果该水果可冷藏1个月以上（如西瓜、哈密瓜）→ **签5-7天合同**  

最终建议：**避免极端两极化**，用“浮动价格框架”代替固定价或纯每日议价，例如签订“基准价+每周调整浮动比例”的协议。
