判断苹果和橙子的大小规格是否适合门店销售，可参考以下具体步骤和标准：

1. **明确门店目标客户群体**：如果是社区超市（家庭客户为主），中等大小（如苹果直径7-8厘米，橙子直径6.5-7.5厘米）最受欢迎；如果是高端精品店，可考虑大果（如苹果直径8.5厘米以上，橙子8厘米以上）；如果是学校或写字楼周边，小果（适合单次食用）可能更好。

2. **使用专业测量工具**：使用水果分级卡或卡尺，测量果实横径。苹果和橙子的商品果通常要求横径在65-85毫米之间，具体可参考国家标准（如苹果GB/T 10651-2008中的规格要求）。超市常分为L（70-75mm）、M（65-70mm）、XL（75-80mm）等。

3. **目测评估均匀度**：同一批次中，个头差异不应超过10%（例如直径70-77mm的苹果混卖均匀）。随机抽取10个样品，如果7个以上大小相近，说明规格合理。

4. **考虑品种特性**：例如红富士苹果通常比国光苹果大，脐橙比血橙大。确认该品种的常见规格范围，避免将品种特性误判为超标。

5. **观察包装标识**：如已预包装，检查标签上的“规格”栏（如“8粒装苹果，直径80mm以上”），并与实际果实对比，误差大于5%可要求供应商调换。

6. **简单手感判断**：以手掌做参考（普通成年人手掌宽度约8-9厘米）。苹果或橙子直径小于手掌宽度约1-2厘米（即6-7厘米）为中小果，接近或略大于手掌宽度（8-9厘米）为大果。太小的果实（<5.5厘米）口感可能酸涩，太大的（>9厘米）可能果皮厚或水分不足。

7. **参考客流和动销数据**：如果近期销售记录中，某种规格的日销量低于同类商品30%，可调整下一批采购规格。例如，家庭装中号苹果日销100斤，大号苹果日销仅60斤，应优先采购中号。

**执行建议**：在收货区设立“规格样品板”，将合格和不合格的果实各贴1-2个在透明盒中，便于采购员和理货员快速比照。每次到货后，随机抽取10%的果箱进行检查，不合格率超过5%则整批退货。
