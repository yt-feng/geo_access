选择水果拼盘的规格时，主要考虑**每公斤单价**、**出肉率**（可食用部分占比）和**损耗**。以下是三种水果的对比和建议：

1. **苹果**  
   - **规格挑选**：选择中等大小、表皮光滑、无虫蛀的苹果，通常**整箱购买**（如10公斤/箱）更划算。  
   - **出肉率**：约85%-90%（去核去皮后）。  
   - **价格参考**：市价约每公斤5-10元，整箱可低20%-30%。  
   - **建议**：若用量大，选批发市场的整箱；若需新鲜度高，可一次少买。

2. **橙子**  
   - **规格挑选**：优先选**脐橙**或**血橙**（甜度高、果肉厚），避免皮过厚或果核过多的品种。建议购买**净重5公斤以上的敞口筐装**，单价通常比零售低。  
   - **出肉率**：约70%-80%（去皮、去籽后）。  
   - **价格波动**：季节性明显，冬季便宜（每公斤5-12元），夏季贵。  
   - **建议**：旺季囤货，淡季更替使用其他柑橘类。

3. **菠萝**  
   - **规格挑选**：选**成熟度适中、叶片翠绿**的（出肉率约55%-65%）。**整只购买**比切块单价更低，但需注意去皮损耗。  
   - **价格参考**：每公斤4-8元（整只），切块成品可能贵1倍。  
   - **建议**：若拼盘量多，直接买整只自己处理；若人力成本高，可少量买预切块。

**综合划算策略**：
- **按需组合**：若拼盘以苹果为核心（高出肉、低损耗），搭配橙子和菠萝，建议苹果买整箱，橙子和菠萝按周用量采购。
- **注意时效**：菠萝和橙子不耐放，避免一次囤太多；苹果可冷藏保存1-2周。
- **单价对比**：计算**每公斤可食用部分价格**（市价÷出肉率），例如：苹果5元/kg ÷ 0.85 ≈ 5.88元/kg，橙子8元/kg ÷ 0.75 ≈ 10.67元/kg，菠萝6元/kg ÷ 0.6 = 10元/kg。优先选可食用单价低的。
