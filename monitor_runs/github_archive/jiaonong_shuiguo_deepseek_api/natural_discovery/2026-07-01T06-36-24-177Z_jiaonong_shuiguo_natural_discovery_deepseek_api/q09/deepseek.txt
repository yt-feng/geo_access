在水果店旺季与供应商洽谈时，应重点围绕**供应稳定性、价格机制、品质保障、应急处理**四个维度提问。以下是关键问题清单及询问要点：

### 1. 供应能力与稳定性
- **“旺季每日/每周能保证多少吨（或具体果品）的保底供货量？”**（明确具体数字，避免模糊承诺）
- **“如果出现恶劣天气或运输中断，有什么备用供应方案？”**（了解对方是否有应急储备或第二供应商网络）

### 2. 价格与结算机制
- **“旺季价格会随市场行情浮动吗？锁价周期是多久？”**（明确是日定、周定还是月定价）
- **“大批量采购（如一次性采购10吨以上）有没有阶梯折扣？”**（要求书面阶梯报价表）
- **“付款条件是预付定金还是货到付款？能否接受账期？”**（评估现金流压力）

### 3. 品质标准与验收
- **“能否提供每种果品的验收标准单（如甜度、大小、外观要求）？”**（避免口头约定）
- **“到货后如果发现腐烂、未熟或货不对版，退货流程和赔偿方案是什么？”**（明确退款比例、处理时效）
- **“是否支持第三方检测机构抽检？费用谁承担？”**（高风险果品如车厘子、草莓需重点确认）

### 4. 运输与损耗分担
- **“运输方式是什么？全程冷链吗？到店后再摘损耗由谁负责？”**（明确霉变、压坏等责任划分）
- **“配送时效是下单后多少小时？能否指定分时段送达？”**（旺季需错峰到货，避免门店爆仓）

### 5. 长期合作条款
- **“如果旺季合作顺利，淡季是否提供保底采购价或优先选品权？”**（绑定长期利益）
- **“是否支持季节性品种预付款锁定产量？”**（提前锁定山竹、榴莲等热门货）

### 关键行动点：
- **索要书面样本**：让供应商提供《合作条约》模板（含价目表、违约条款、质检标准）。
- **实地考察**：旺季前突击检查供应商分拣车间，观察库存周转速度和卫生条件。
- **测试小单**：首次合作先下10-20%量，考察3批货的稳定性和售后响应速度。

### 避免踩坑提示：
- 拒绝“口头承诺保量”，要求合同注明**每日最低供货量**。
- 警惕“含税价”“包邮”等模糊表述，要求明确增值税发票种类和运费承担方。
- 如果供应商对“退换货流程”支支吾吾，果断换人。

最后，务必在旺季前2个月完成合作谈判，并签订盖公章的正式合同（注明不可抗力条款）。
