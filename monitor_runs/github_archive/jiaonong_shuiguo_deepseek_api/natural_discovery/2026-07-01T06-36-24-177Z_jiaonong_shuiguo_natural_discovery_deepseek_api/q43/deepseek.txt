评估水果供应商的仓配覆盖能力，核心是看其**仓储网络**能否匹配你门店的**分布密度**和**时效要求**，以及其**配送能力**能否应对生鲜损耗。以下是具体的分步评估方法：

### 第一步：地理盘点与匹配
1.  **制作门店热力图**：将全部连锁门店按城市或300-500公里半径（水果保鲜运输的经济半径）分组，标记出**高密度区域**（如华东、华南、华中）。
2.  **列出供应商的仓储节点**：供应商的**产地仓（原产地）、区域分选中心、城市前置仓**分别位于哪里。
3.  **关键匹配度判断**：
    -   **最优**：供应商在你门店密集的**每个核心城市圈**都有城市仓或区域仓。
    -   **及格**：在省级核心城市（如省会）有区域仓，能覆盖省内其他城市。
    -   **危险**：供应商只有一个总仓（如主产区），所有跨省门店都需要长距离干线运输。

### 第二步：评估配送时效与损耗控制
1.  **确认“最后一公里”配送方式**：
    -   供应商是用**自己的冷链车**直接配送门店，还是外包给第三方物流？
    -   **自有车队**：可控性高，适合每日补货（如精品水果店）。
    -   **第三方物流**：需确认其是否提供**恒温（0-4℃）**运输，以及是否能做到**每日+次日达**（生鲜标准）。
2.  **测算覆盖时间窗口**：
    -   假设所有门店均在**早上6-8点**收货，供应商从仓库发车到门店的**最晚发车时间**能否满足？（例如：城市仓深夜发车，区域仓则需凌晨发车）。
    -   **测试极端门店**：选一个最偏远、最小的县城门店，询问供应商该店可支持什么配送频次（每日/隔日/周2次），以及预计送达时间。
3.  **损耗承诺与理赔**：
    -   不只看口号，问清楚：**卸货后2小时内**发现的损果、烂果，供应商是否接受**无条件退换或补货**？流程图是什么？

### 第三步：验证弹性与备用方案
1.  **突发爆量测试**：假设你突然加开50家门店（如在某个城市大幅扩张），供应商的**总仓库存**和**分拣人力**是否能在一周内扩容？
2.  **极端物流中断**：如果供应商的**主力城市仓**因天气或疫情停摆，其是否有**备用仓**或**从其他区域仓紧急调货**的方案？时效是X天还是Y天？
3.  **多品种组合能力**：如果门店同时需要易损的**草莓、蓝莓**和耐储运的**苹果、哈密瓜**，供应商能否通过**同一车次、不同温区**完成配载？还是需要分两单？

### 第四步：定量指标对比（可以做一张评分表）
| 评分维度 | 具体指标 | 优秀（10分） | 及格（5分） | 不合格（0分） |
| :--- | :--- | :--- | :--- | :--- |
| **覆盖率** | 门店所在城市/区域90%以上可次日达 | 同一城市圈仓满足 | 省会能次日达，地市需隔日 | 多数城市需3天以上或无法达 |
| **稳定性** | 过去半年平均到货准时率（精确到小时） | >98% | 90%-98% | <90% |
| **温控** | 运输全程温度记录是否提供 | 每日发货带温度监控报告 | 无报告但无投诉 | 经常出现冻伤/高温烂果 |
| **紧急响应** | 因损耗或漏发的补货时长 | <12小时补到店 | 24-48小时 | >48小时或推诿 |
| **成本与经济性** | 起订量、配送费、溢价 | 按件计费，无最低起订 | 有起订但结合损耗后划算 | 起订量极高且运费单独计算 |

### 最后一步：实战测试（最重要）
在没有正式签约前，要求供应商**试运行一个月**：
1.  选取**3-5个典型门店**（1个市中心旗舰店、2个社区店、1个偏远县城店）。
2.  按正式订单流程（提前24小时订货）履行。
3.  跟踪**每一单的到货时间、温度、损耗率**，并做留证拍照。
4.  一个月后计算：
    -   整体到货及时率（按小时算）
    -   综合损耗率（包含运输、分拣、装卸）
    -   客服响应效率（处理损耗申请的时间）

**结论线**：
-   **评估通过标准**：试运行期间，**偏远门店**损耗率不超过5%，**响应时效**（从提交补货到货到门店）不超过24小时，且**物流费用**可控（不高于你原来采购模式总成本的15%）。
-   **坚决否决不通过的信号**：供应商无法提供明确的**仓配路径图**，或对**生鲜损耗没有明确的退换原则**。
