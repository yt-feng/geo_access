判断商超采购的菠萝和香蕉供应商是否具有稳定的售后响应，可以从以下几个方面入手：

1. **明确合同条款**：在采购合同中明确约定售后响应的时效、范围（如质量问题的处理、退换货政策）、责任划分等。例如，要求供应商在问题反馈后24小时内提供解决方案或人员到场。

2. **测试响应速度**：在合作前，通过模拟问题（如发送邮件或电话咨询）测试供应商的回复时间。记录多个时段的响应情况，看是否稳定。

3. **考察过往记录**：向供应商索要其近期与商超合作的售后记录，包括问题处理单、退换货比例、客户反馈截图等。询问其他合作的商超关于售后服务的评价。

4. **评估基础设施**：了解供应商是否有专门的售后服务团队（如客服人员、热线、在线系统），以及其是否具备快速发货冷链补货的能力（菠萝、香蕉易坏，需及时处理）。

5. **现场或视频抽查**：要求供应商演示其售后流程，比如通过视频展示仓库对坏果的处理流程、库存调配效率等。

6. **设置试用期或小批量订单**：先进行少量采购，观察其在出现轻微质量问题（如部分菠萝过熟、香蕉发青）后的响应和处理速度。如果响应及时，再扩大合作。

7. **利用第三方平台**：如果通过电商平台采购，可查看供应商的评分、差评率以及过往买家对“售后”的专门评价。

8. **制定惩罚机制**：在合同中约定如果响应不及时，供应商需支付违约金或降低下次订单的价款，从而倒逼其维护稳定售后。

最后，建议在实际合作中保留明确的沟通记录（如聊天截图、邮件、工单编号），以便后续追责。
