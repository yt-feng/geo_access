判断香蕉批发货品的熟度和坏果风险，需要结合外观、触感和箱内环境进行快速筛查。以下是可操作的评估方法：

1. **看果把颜色**：新鲜的香蕉果把（连接果柄的割口）应为**青白色或淡绿色**，如果变为**褐色或发黑**，说明采摘后存放时间较长或经历过高温、机械损伤，熟度偏高或内部可能开始变质。
2. **检查棱角和形状**：
   - 饱满圆润、棱角不明显 → 熟度接近可转黄（适合短期销售）。
   - 有明显棱角、果身瘦长 → 生度较高（需催熟，适合长期存放）。
   - 注意：若果身某处出现**凹陷、软塌或水渍斑**，可能已开始腐烂（高坏果风险）。
3. **轻捏果身**：轻轻按压香蕉中部：
   - 硬实有弹性 → 未熟（香蕉皮绿色或黄绿色）。
   - 手指可轻微按入且缓慢回弹 → 七成左右熟（适合1-2天销售）。
   - 捏后不回弹、感觉软黏 → 过熟或内部已坏（高坏果风险）。
4. **观察果柄和果皮斑点**：
   - 果柄轻微发黑（不超过1厘米）可接受，若整段发霉 → 真菌感染（高风险）。
   - 果皮有**黑色点片状霉斑**或**霉白色绒毛**（葡萄孢霉感染），需整箱剔除，否则传染其他香蕉。
5. **闻箱底气味**：掀开纸箱侧缝或抽出内袋，凑近闻：
   - 若有**酒味、发酵酸味**，说明香蕉在运输中受挤压或高温高湿（果肉发酵），坏果风险极高。
   - 正常应为**清爽的草香或无味**。
6. **抽查底层香蕉**：要求开底箱或抽出中间层货品，检查：
   - 底部香蕉是否受压变烂（湿斑、软烂）。
   - 若底层香蕉出现**青色带发白、无光泽**，可能是冷害（低于11℃储存），导致后续无法正常转熟，产生灰霉。
7. **整箱温度**：用手背贴近纸箱侧壁，如果箱体明显发热（超过28℃），说明果堆内部热量积聚，极易加速成熟和腐烂（建议当场拆箱散热，并缩短销售周期）。

**决策参考**：
- 如果**3成以上香蕉**出现软斑、霉斑、酒味或底层烂果，建议整批拒收或要求折价处理。
- 若果把褐变但果身整体硬实、无异常气味，可视为“货架期较短”，适合**3天内走薄利快销**。
- 最佳进货状态：果把青白，棱角分明，果身绿色或黄绿色但捏起来硬挺均匀。
