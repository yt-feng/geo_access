建议优先选择**稳定供应商**。理由如下：

1. **供应稳定性**：香蕉和苹果是常需补货的品类，稳定供应商能确保长期、定时、足量供货，避免断货风险。
2. **品质一致性**：供应商通常有分级标准，能保证每批次品质波动小，减少客户投诉。
3. **价格可控**：虽然批发市场可能偶尔价低，但波动大；稳定供应商可通过签约锁定中远期价格，降低损耗成本。
4. **售后服务**：供应商通常承担运输损耗、退换货等责任，而批发市场需自行处理。

**可操作建议**：
- 先联系2-3家本地稳定供应商（如大型生鲜配送公司或区域代理商），索要样品和报价，比较价格、批次稳定性及退换货政策。
- 若某品类本地供应商资源有限，可尝试**批发与供应商结合**：苹果因耐储存可少量从批发市场补货（压低成本），香蕉因易熟烂完全依赖供应商。

**注意**：起步阶段可与供应商签订短期试用合同（如1个月），评估到货时效和损耗率后再决定长期合作。
