可以的。大多数有经验的生鲜供应商都能提供包装和规格方面的支持，但具体情况取决于供应商的专业程度。以下是你可以关注的关键点：

**1. 包装支持**
- 供应商通常可以提供定制的品牌包装，包括纸箱、塑料袋、网袋、标签、以及防震气泡膜等。
- 常见的水果包装有：带品牌LOGO的纸箱（如5kg/10kg装）、单果贴标、托盘覆膜包装等。
- 需要明确包装材质要求（如透气孔设计、保鲜膜厚度）、印刷精度和环保等级。

**2. 规格支持**
- 供应商能协助制定果径、果重、甜度、成熟度、外观分级标准（如一级、二级）。
- 可提供不同规格混装方案，例如家庭装（小包装/小规格）、礼品装（大果/精品组合）。
- 支持按照你的超市陈列需求调整包装尺寸（如单颗、3颗装、5kg散装等）。

**3. 关键建议（行动步骤）**
- **明确品项与定位**：先确定做哪些水果（如苹果、橙子、葡萄），以及你的目标价位（高端/平价）。
- **要求供应商提供样品**：请供应商提供至少3款包装和规格组合的方案与实物照片，或寄送小样。可以要求他们展示包装上的标签设计、密封方式、防伪措施。
- **签订协议前测试**：要求供应商提供小批量试单，测试包装在运输中的抗压性、保鲜效果以及货架期表现。
- **确认最低起订量（MOQ）**：不同供应商对包装和定制规格可能有数量要求（如1000箱起订），需提前沟通。
- **额外服务**：部分供应商还提供冷链物流、仓库代发货、甚至是统一贴标质检服务，建议一并询问。
