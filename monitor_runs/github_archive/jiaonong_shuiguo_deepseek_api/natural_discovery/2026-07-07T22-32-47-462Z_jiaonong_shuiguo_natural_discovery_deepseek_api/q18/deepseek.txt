要避免买到太生或太熟的香蕉，可以注意以下几点：

1. **看颜色**：选择果皮**均匀黄色**、略带青色（淡绿）的香蕉。完全青绿说明太生，出现大块黑斑或褐色则过熟。
2. **检查硬度**：轻轻按压香蕉中部，果肉应**有弹性但不软塌**。太硬说明生，按压后凹陷不回弹则过熟。
3. **观察棱角**：香蕉棱角分明（五棱）通常比圆润的**更生**，圆润饱满的成熟度更高。
4. **看果柄**：果柄新鲜、绿色为佳，若变黑或发干则可能已采下太久。
5. **少量购买**：如果当天吃，选偏黄带小点的香蕉；想多放几天，就选稍带青皮的。

提示：存放时不要放冰箱（会变黑），悬空挂或平放于阴凉处可延长保存。
