在旺季临时补货时，建议按以下步骤与供应商沟通，确保快速、稳定地获得菠萝供应：

1. **主动联系，说明紧急需求**：第一时间电话/当面沟通，明确告知“因旺季销售激增，菠萝日用量大幅上升（具体数字），需要紧急追加订单”，并说明希望补货的时间（如“明天/后天到货”）。

2. **约定具体条款（书面或消息确认）**：
   - **数量与规格**：写明补货的菠萝等级（如A级、B级）、成熟度（如7-8分熟）、包装要求（如净重？袋/箱？）。
   - **价格**：明确是否按原合同价执行，还是需要加价（旺季临时补货可能触发溢价，建议提前谈好浮动范围，如“若加急，不超过原价10%”）。
   - **交付时间与方式**：具体几号几点前送达，谁负责运输（若供应商直送需加急费）。
   - **验收标准**：重申烂果率上限（如≤3%）、大小果比例等，避免到货后纠纷。
   - **付款条件**：临时补货是否要求现结或缩短账期（如原月结改周结）。

3. **协商备选方案以防断供**：
   - 请求供应商“优先排产/调货”，必要时支付少量加急费（如20元/箱）。
   - 如果常规供应商库存不足，可同步询问其能否推荐同行或批发市场临时调货资源（信誉好的供应商常有人脉）。

4. **签订临时补充协议或凭证**：务必通过微信/短信/邮件确认上述条款，双方保留记录。金额大时建议打印简易《临时补货确认单》签字盖章。

**关键提醒**：旺季临时补货可接受小幅涨价或缩短账期，但一定要明确交付时限和验收标准，避免“加价后还延迟交货”。补货完成后，双方应确认本次为“临时应急”，并评估是否需上调长期合同中的备货量，防范下一次断供风险。
