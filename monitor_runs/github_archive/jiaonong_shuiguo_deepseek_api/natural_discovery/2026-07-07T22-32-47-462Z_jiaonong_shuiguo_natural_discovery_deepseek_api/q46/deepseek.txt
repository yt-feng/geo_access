建议优先选择**菠萝果丁**，其次是**菠萝果泥**。具体原因如下：

1. **鲜果**：成本较低，但加工繁琐（去皮、去眼、切块），果肉硬度和甜度不稳定，需要额外储存和人工，且容易氧化变色。适合少量、现做现用，但大批量采购效率低。
2. **果丁**：冻品或罐头形式，果肉大小均匀，耐储存，开袋即用，能保持果肉口感。适合追求效率、需稳定出品的大多数茶饮店。
3. **果泥**：风味浓郁，适合打底或调味，但缺少果肉颗粒感。若果茶主打清爽口感和咀嚼感，则不如果丁。

**操作建议**：
- 若果茶需体现真实果肉，选**果丁**（优选冷冻果丁，风味损失少）。
- 若主做菠萝味基底或冰沙类，可搭配**果泥**（减少用量即可节省成本）。
- 不建议大量采购鲜果，除非有专门切配人员且销量稳定。
