判断商超采购菠萝和香蕉的供应商是否有稳定的售后响应，可以采取以下具体措施：

1. **要求书面承诺**：明确要求供应商在合同中写入售后响应时间、处理流程及违约条款（例如：24小时内响应退货/换货需求，48小时内处理完毕）。

2. **查看历史记录**：要求供应商提供近3个月与其它商超的售后沟通记录或处理单，重点看平均响应时间和问题解决率。

3. **随机测试响应**：模拟小问题（如：对一批菠萝提出轻微外观疑问），通过邮件、电话或即时通讯工具联系供应商，记录从发起提问到首次回复的时间。重复测试2-3次看稳定性。

4. **走访现有客户**：向供应商提供的其他商超客户（尤其是类似规模的）进行匿名电话确认，询问“遇到烂果时对方多久回复？是否主动提供补发方案？”。

5. **检查库存与物流配合**：询问供应商是否常备替换库存或合作物流资源。如果对方能快速调换，说明售后支持能力强。

6. **关注行业评价**：在生鲜供应链平台（如“鲜桥”、“美菜”）或本地水果批发市场口碑中，搜索该供应商的评价，尤其注意“售后”“坏果率”“退换”等关键词。

7. **小额试单验证**：先采购少量菠萝和香蕉（如半车），故意观察到货是否包含明显劣果（例如挤压、黑斑），记录对方处理态度和时间。如果实际响应慢或不负责，则停止合作。

8. **定期考核指标**：合作后，设立月度售后响应时效、问题解决率、客户满意度等量化指标，每月评分，低于标准则触发警告或解约条款。

最后，注意优先选择有独立售后团队、提供专用24小时响应热线、支持到货48小时内无损包换的供应商。对于香蕉（易熟）和菠萝（易损伤），这类生鲜品售后响应要求较高，建议选择本地或有冷链运输经验的供应商以降低风险。
