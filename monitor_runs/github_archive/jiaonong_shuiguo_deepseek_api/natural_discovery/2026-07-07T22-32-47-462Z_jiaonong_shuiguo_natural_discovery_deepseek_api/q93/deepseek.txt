针对便利店售卖菠萝切块，需要重点关注以下三个维度，按优先级排序和建议如下：

**1. 保质期（最重要）**
- **关注点**：菠萝切块后氧化快，需明确适用保质期。通常4℃冷藏条件下不超过48小时（具体视切块前成熟度和包装方式）。
- **建议执行**：
  - 使用**生产日期+到期时间（精确到小时）标签**，如“切块时间：10月1日 09:00，到期：10月3日 09:00”。
  - 严格遵循“先进先出”原则进行货架补货。
  - 非当天切块的菠萝，每日至少检查一次色泽、气味、出水情况，出现褐变或软烂立即下架。

**2. 包装（影响保质期和卖相）**
- **核心问题**：隔绝空气和细菌接触，延缓氧化。
- **建议做法**：
  - 使用**食品级PET密封盒或带气调包装**（如充氮气或二氧化碳的气调盒可延长保质期到3-4天）。
  - 避免使用普通保鲜膜（易破损、密封性差）。
  - 每盒标注果块重量（如200g/盒），统一售价。

**3. 配送频次（影响库存和新鲜度）**
- **关键逻辑**：配送频次需与门店销量和保质期挂钩。
- **建议操作**：
  - 按**每日至少1次配送**（最好早间配送，确保全天新鲜）。
  - 销量稳定店铺可调整为**每日2次配送**（早、午各一次），减少库存积压。
  - 与供应商约定：剩余未售商品的回收或换货规则（如超过24小时未售出可部分折价回收）。

**总结优先顺序**：保质期 > 包装 > 配送频次  
**实操提醒**：如果便利店不具备中央厨房标准化切割能力，优先选择有成熟气调包装和每天下午补货的供应商，而非追求高频配送但包装简陋的渠道。
