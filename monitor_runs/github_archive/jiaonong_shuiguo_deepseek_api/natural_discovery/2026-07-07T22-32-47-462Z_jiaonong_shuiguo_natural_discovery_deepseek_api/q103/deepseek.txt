按斤结算、按箱结算或按份结算各有优劣，具体选择取决于食堂的规模、管理能力和实际需求。以下是对三种方式的简要分析：

1. **按斤结算**：适用于大批量采购，便于精确控制成本，但需要称重设备，可能增加操作时间。适合对重量要求高的水果（如苹果、橙子），且与供应商容易达成交易。

2. **按箱结算**：适合标准包装的水果（如香蕉箱、葡萄箱），方便验收和清点，但需确保箱内重量或数量统一，否则易出现成本偏差。适合采购量大的食堂，能简化流程。

3. **按份结算**：更适合小规模或即食场景，如切好的水果拼盘，便于直接销售或分配，但单价可能较高，且对供应商依赖性强。

**建议**：若食堂规模较大、采购频繁，且水果规格统一（如常见水果每箱5公斤），可优先**按箱结算**，配合定期抽查重量来避免误差；若需精细控制成本（如营养餐要求），则**按斤结算**更合适；若主要为零售或套餐供应，**按份结算**更省事。实际操作中可混合使用，例如按箱主采、按斤补缺。
