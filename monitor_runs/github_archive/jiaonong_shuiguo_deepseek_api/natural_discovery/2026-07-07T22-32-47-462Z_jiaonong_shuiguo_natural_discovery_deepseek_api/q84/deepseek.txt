根据水果特性和家庭需求，以下是推荐的4种组合方式：

**组合一：家庭健康基础款（适合日常、送长辈）**
- **苹果×4个**：寓意平安，耐储存，适合全家。
- **橙子×4个**：补充维C，酸甜开胃，寓意心想事成。
- **香蕉×1把**：进口方便，老人小孩易消化，提供即时能量。
- **菠萝×1个**：切块后搭配其他水果，增加热带风味和膳食纤维。
- **推荐理由**：覆盖维生素C、钾、膳食纤维，性价比较高，不用额外搭配其他水果。

**组合二：无籽易食款（适合有幼童或老人家庭）**
- **苹果×3个** + **橙子×3个** + **香蕉×2根** + **菠萝（已削皮切块）**
- **关键调整**：菠萝可提前削皮切块，避免家人自己处理刺口；香蕉选8-9成熟（避免青涩或过软）。
- **注意**：菠萝切开后需冷藏，建议购买时让店家代切。

**组合三：清热润喉款（适合换季、上火人群）**
- **苹果×2个** + **橙子×3个** + **香蕉×1把** + **菠萝（选较熟金黄品种）**
- **搭配思路**：苹果和梨同属润燥水果，但这里用菠萝替代梨，因为菠萝含菠萝蛋白酶，对咽肿有缓解。

**组合四：视觉缤纷款（适合拍照、节日送人）**
- **红苹果×3个** + **绿橙/血橙×2个** + **皇帝蕉/小米蕉×1小把** + **凤梨型品种×1个**
- **包装建议**：苹果放底层，橙子居中，香蕉和菠萝竖插（间隔摆放）。加一张彩纸或麻绳手提盒更美观。

**避免踩坑建议**：
- **不推荐混合催熟**：香蕉释放乙烯会加速其他水果成熟，如想一次吃完可忽略，但如需存放2天以上，建议将香蕉与苹果、橙子分开放。（菠萝是唯一和香蕉共存影响较小的）
- **价格参考**：如果团购价低于市场价30%，需确认是否包含重包装费；如包含运费，菠萝最好确认是“整果”还是“切块”，切块菠萝需当天送当天吃。

根据家庭人数可调整数量：2-3人家庭用组合一按上述量，4-5人家庭可各加50%。
