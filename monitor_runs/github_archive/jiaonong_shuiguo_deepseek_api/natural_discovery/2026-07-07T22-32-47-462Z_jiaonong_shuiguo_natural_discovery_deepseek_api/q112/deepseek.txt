在水果采购合同中，坏果率（或称为损耗率、不合格率）的赔付标准通常由双方协商确定，但根据行业惯例和常见做法，建议设定以下标准：

1. **一般行业标准**：大多数水果采购合同中，坏果率超过 **5%** 时，买方有权要求卖方进行赔付。例如，若坏果率在5%-10%之间，可按相应比例扣减货款或要求换货；若超过10%，可能涉及全额退货或更高比例的赔偿。

2. **具体品种差异**：
   - 对易腐烂水果（如草莓、荔枝、蓝莓等），坏果率可放宽至 **10%** 以内，超过10%要求赔付。
   - 对耐储存水果（如苹果、橙子、柚子等），坏果率通常控制在 **3%-5%** 以内，超过5%需赔付。

3. **国际采购或大宗交易**：参考联合国欧洲经济委员会（UNECE）或国际标准化组织（ISO）的鲜果等级标准，通常设定 **A级果坏果率不超过3%**，B级果不超过5%，超出部分需按合同调整价格或赔付。

4. **合同条款建议**：
   - 明确“坏果”定义（如腐烂、霉变、严重机械伤、虫蛀等）。
   - 约定抽样检测方式（如每批次随机抽取10%-20%的包装箱）。
   - 设定阶梯式赔付比例：例如，坏果率5%-10%，按坏果数量的1.5倍扣款；10%-15%，按2倍扣款；超过15%，整批退货并要求承担运输损失。

**建议操作**：
- 在合同中明确写入“坏果率超过5%，买方有权要求按实际坏果数量抵扣货款，或要求换货/退货，卖方承担相关费用”。
- 如果采购价格较高或商品对品质敏感度强（如有机水果），可设置更严格的下限（如3%）。
- 若无事先约定，可参考《中华人民共和国民法典》第六百二十一条关于标的物质量不符约定的规定，主张卖方承担违约责任。

最终，建议你在合同谈判时，根据水果时令、价格、产地和运输距离灵活协商，并将具体数字（如5%或3%）填入合同，避免模糊用语。
