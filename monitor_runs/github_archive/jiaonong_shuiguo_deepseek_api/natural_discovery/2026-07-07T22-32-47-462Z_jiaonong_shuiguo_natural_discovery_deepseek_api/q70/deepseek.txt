针对批发商担心货到后损耗高的问题，合同中建议明确约定以下条款以界定坏果责任：

1. **损耗标准与责任划分**：
   - 明确约定“自然损耗率”（如在运输或仓储中允许的合理损耗比例，约定由谁承担）。
   - 超出自然损耗率的坏果，明确由**供货方**承担（如因包装不当、货品质量不合格导致），或由**批发商**承担（如因存储、运输不当导致）。

2. **检验与异议期**：
   - 约定货到后的**检验期限**（如48小时内），批发商需在此期限内完成验收并记录坏果数量、拍照或视频留存。
   - 若未在规定期限内提出异议，视为供货方对货物质量无异议，坏果风险由批发商承担。

3. **坏果处理与赔偿方式**：
   - 约定具体赔偿方案：如坏果按比例折价、退货退款、替换补发等。
   - 明确坏果鉴定方式（如需第三方机构介入），以及处理时限（如供货方在24小时内回应）。

4. **证据与争议解决**：
   - 要求双方保存运输单据、验货记录、沟通记录等。
   - 指定争议解决方式（如仲裁或法院，需明确地点）。

**示例条款（可参考）**：
> **第X条 验收与损耗责任**
> 1. 货物运输至双方约定地点后，批发商应在**48小时**内完成验收。验收时，若发现货物存在**超过自然损耗率**（自然损耗率由双方协商约定，如5%）的坏果，应拍照留存并书面通知供货方。
> 2. 供货方应在收到通知后**24小时**内确认坏果情况，并选择以下之一处理：  
>    （a）按坏果数量**折价**（如原价×0.7）；  
>    （b）在**3个工作日**内补发等量合格商品；  
>    （c）对坏果部分**退款**。  
> 3. 若批发商未在规定期限内提出异议，视为货物全部合格，后续损耗由批发商自行承担。  
> 4. 因包装不当、运输不当等导致的异常损耗，由责任方承担全部损失。

**注意事项**：
- 清晰记录水果品种、规格、成熟度等，避免质量争议。
- 若运输由物流方负责，建议同时约定物流方的责任（如破损赔偿）。
- 实际执行时，确保双方签字确认验收单，留存证据。
