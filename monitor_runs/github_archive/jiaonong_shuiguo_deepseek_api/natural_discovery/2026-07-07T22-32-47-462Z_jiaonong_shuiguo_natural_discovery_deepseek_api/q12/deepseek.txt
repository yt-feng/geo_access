采购菠萝时判断熟度，主要为了平衡“到店后能尽快销售”和“避免因过熟导致损耗”两个目标。如果是水果店，建议优先选购**七八分熟（微黄带青、硬挺有弹性）**的菠萝，这样能留出3-5天的销售期。

以下是具体的判断方法：

1. **看颜色（最直观）**：
   - **青绿色为主（七成熟）**：适合长途运输或需存放3天以上再售卖。口感较酸，但耐放。
   - **底部微黄、中上部带青（八成熟）**：最理想的进货成熟度。到店后1-2天会完全变色变甜，适合快速销售。
   - **通体金黄/橙黄（九成熟以上）**：已经熟透，糖分最高、最甜，但**损耗风险大**。如果当天卖不掉，隔夜就容易产生酒味或腐烂，建议谨慎采购，或仅少量作为展示品。

2. **闻气味（辅助判断）**：
   - **无明显香气**：偏生，需继续放。
   - **底部和果眼处有轻微清香**：七八成，适合进货。
   - **浓郁甜腻香气（甚至带酒味）**：过熟，当天不卖完就会坏。

3. **摸手感与按硬度**：
   - 优质七八成熟菠萝：**整体硬实，但用手指轻压底部（果脐处）有轻微弹性**（不是硬如石头，也不会一按凹陷）。如果按下去空洞、发软或渗汁，说明内部已经开始腐烂。
   - 太硬且手感沉（果眼很紧、不平坦）：可能未全熟，糖分累积不足。

4. **看叶片**：
   - 顶部叶片**翠绿、挺立**，说明新鲜度高。
   - 叶片枯黄、萎蔫或轻易一拔就脱落：存放时间较长，成熟度可能已过高或开始变质。

5. **看底部切口**：
   - 切口**新鲜、干燥、颜色白或微黄**：好。
   - 切口发黑、发黏或有霉点：已经开始腐败。

**总结实操要点**：
- **日常采购**：优先选“底部微黄微香、整体青绿、按压硬中带弹”的。这种到店后容易统一成熟度，方便管理。
- **降低损耗技巧**：不要一次进太多全熟的金黄色菠萝，尤其是夏季高温期。如果必须进熟果，当天以“现切试吃”或“搭配盐、酸梅粉”促销，当日清空。
- **采购后处理**：回来后不要捂在箱里，应分开摆放（果眼朝上通风），根据成熟度分开定价或做二级处理（如做菠萝汁、菠萝冰）及时消化较熟的部分。
