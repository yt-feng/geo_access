采购时**应优先关注成熟度**，理由如下：

1. **直接影响产品品质**：香蕉蛋糕对香蕉的熟度要求较高。过生的香蕉淀粉味重、甜度低、质地硬，难融入面糊；过熟的香蕉（表皮完全变黑）可能带有发酵味、果肉过于稀软。理想的成熟度是**表皮金黄带少量黑点**（“芝麻斑”状态），这时的香蕉甜度最高、香气浓郁、质地软糯，最契合焙烤品控。

2. **价格需结合实际损耗**：即使遇到非常便宜但过生或过熟的香蕉，可能导致后续人工催熟（耗时间、占空间）、过量损耗或成品口味不佳。因此应在**保证成熟度达标的前提下**，对比多家供应商价格；也可以和供应商约定“指定熟度”的供应周期，通过固定渠道降低成本波动。

3. **操作建议**：  
   - **若当天或次日便用**：选80-90%成熟度（表皮60%带黑点）。  
   - **若计划3-5天内陆续使用**：选70%成熟度（整体偏黄）自行存放催熟。  
   - **常见陷阱**：避开表皮大面积破损或出水（内部可能已过度发酵）。

**结论**：成熟度是品质底线，价格是成本优化——先锁定成熟度，再在达标货源中谈价格。
