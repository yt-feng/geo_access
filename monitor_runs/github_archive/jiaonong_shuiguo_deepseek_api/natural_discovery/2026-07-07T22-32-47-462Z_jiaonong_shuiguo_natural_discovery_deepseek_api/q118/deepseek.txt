要从源头减少中间环节，建议按以下优先级考虑：

1. **首选：直接对接产地（合作社/大型种植基地）**  
   - 优势：价格最低、品质可控、可定制规格（如分拣标准、包装），适合大批量稳定采购。
   - 路径：查询国家或省级“一村一品”目录、农业展会（如中国国际农产品交易会）、或通过农业农村部门推荐。

2. **次选：品牌方（有自建种植基地的品牌）**  
   - 如果目标水果有成熟品牌（如佳沃蓝莓、褚橙），可联系其基地直供。品牌方通常注重新鲜度和品控，但可能要求整车起订（如20吨以上），且价格略高于产地。

3. **限制性选择：区域经销商（一级代理）**  
   - 仅适合小批量、多品种采购。区域经销商库存稳定、能拆分规格，但通常加价15%-30%，且无法跳过仓储物流环节。

**操作建议**：  
- 先通过“中国农产品市场协会”或“全国农产品商务信息公共服务平台”获取产区上市高峰期信息，直接联系当地农业局对接合作社。  
- 若量小，可与同类商家拼单，降低分摊成本。  
- 注意：避开“转手超过2级”的经销商（如省级→市级→县级），直接找“产地代办”或“一级批发商”更有效（可询问对方是否持有“产地直供证明”）。
