建议你重点关注**切果损耗**，其次是**甜度**，外观和大小相对次要。原因如下：

1. **切果损耗直接影响利润**：菠萝通常需要削皮、去眼。如果果形不规则、眼深或熟度过高，损耗率可能超过30%，严重压缩利润。采购时应优先选择果形匀称、果眼浅、适合标准化切割的批次。
2. **甜度决定复购率**：如果消费者买到的菠萝酸涩，会影响口碑。建议抽样测糖度（一般13°以上口感较好），或观察果皮色泽与果香。
3. **外观是初印象**：无黑斑、无霉变、无机械损伤是基本要求，但不必追求完美外形，因为这往往对应更高进价和更大损耗。
4. **大小按市场定位选**：小果适合个人消费，大果适合家庭或餐饮，但大小与甜度、损耗没有直接关联。

**补充操作建议**：
- 现场切开一个样品，估算可食用部分的实际出品率（比如带皮重量与去皮去眼后的净重比值）。
- 如果超市已有切片或切块销售，直接将损耗和包装成本算入定价。
