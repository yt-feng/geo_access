针对连锁商超水果供应商招标，建议从**资质、供应能力、品控、价格、服务**五个维度设置评分标准，具体参考如下：

1. **基础资质（10-15分）**
   - 企业注册年限、经营范围、行业经验（如5年以上水果供应经验）。
   - 相关认证（GAP、有机、ISO22000等）。
   - 无重大食品安全事故记录（需提供证明）。

2. **供应能力与资源（25-30分）**
   - **产地资源**：是否有自有或稳定合作的种植基地，产地直采比例。
   - **品类覆盖**：能稳定供应常见水果（苹果、柑橘、香蕉等）及季节性水果（荔枝、榴莲等）。
   - **运输与仓储**：冷链物流能力、配送范围覆盖门店数、响应速度（如24小时内补货）。
   - **供应量保障**：高峰期（如春节、中秋）及灾期（如台风季）的供应方案。

3. **品质管控（25-30分）**
   - **品控标准**：明确水果分级标准（如规格、糖度、外观瑕疵率、成熟度）。
   - **检测能力**：自检实验室或第三方检测报告（农残、重金属等）。
   - **退换货机制**：到货次品率处理方案（如当日分拣后按比例退换）。
   - **溯源体系**：是否支持扫码查询产地、种植户、采摘时间等全流程信息。

4. **价格与财务（15-20分）**
   - **报价合理性**：需对应市场行情（可设置基准价，超基准价扣分）。
   - **价格调整机制**：季节性波动、长期锁价比例或调价公式。
   - **付款条件**：账期（如月结）、付款灵活性。

5. **服务与合規（10-15分）**
   - **售后服务**：质量问题处理时效、投诉响应速度。
   - **合作历史**：与同类商超合作案例、用户评价。
   - **违诺惩罚**：违约（如断供、品质不符）的扣分或保证金条款。
   - **供应链应急预案**：如自然灾害导致断货的替代方案。

**建议加分项**：
- 季节性新品推荐能力（选品建议）。
- 提供门店陈列、营销支持（如水果试吃、创意包装）。
- 库存数据同步系统（实时更新到货量）。

**注意事项**：
- 建议设置**最低分数线**（如75分），低于线的供应商直接淘汰。
- 评分权重可根据商超定位调整：若定位高端，可提升品控和资质权重；若注重低价，价格权重可适当增加。
- 需准备样品盲评（如随机抽取批次进行口感、外观测试），并加入实地考察环节（考察供应商仓储、分拣中心）。
