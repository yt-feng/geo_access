选择水果供应商时，以下细节容易被忽视，但对长期稳定经营很关键：

1. **分拣与包装标准**：供应商是否按规格（大小、成熟度）分拣，包装是否防挤压、透气。不统一会导致损耗增加，影响客户体验。
2. **冷链物流能力**：是否全程冷链运输？断链会加速变质。尤其高价值水果如草莓、蓝莓，需核实冷藏车温度记录。
3. **应急处理方案**：遇到下雨导致品质下降、爆单缺货时，供应商有无快速补货或替换品机制？现金赔款还是换货？
4. **产地与中转环节**：是否直接自产地直采？中转次数越多，损耗越大。可要求提供产地证明和运输时间表。
5. **结算与账期弹性**：是否允许部分赊账？结算周期是否灵活？现金流紧张时能否协商延后？这影响你自身的资金链。
6. **样品与实际一致性**：是否支持小批量试单？避免大货到货质量与样品天差地别。
7. **退货与赔付政策**：如果到货有烂果，是否当天报损即赔？有些供应商仅提供下次补货，不扣减费用，但需明确标准（如超过X%烂果全赔）。
8. **季节性风险分担**：是否愿意在淡季帮你消化少量尾货？或在新品上市时首单打样成本共担？

建议：可以要求供应商提供之前合作客户的联系方式（征得同意），直接了解实际服务水平。同时，**签订合同时标注以上细节**，而非仅口头承诺。
