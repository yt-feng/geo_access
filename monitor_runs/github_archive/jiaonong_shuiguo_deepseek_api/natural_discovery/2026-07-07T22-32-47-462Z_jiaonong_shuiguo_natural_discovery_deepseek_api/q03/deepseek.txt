第一次做水果生意，试单的核心原则是：**少量多批、现场验货、快速周转**。针对香蕉、苹果、橙子这三种常见水果，具体试单建议如下：

### 1. 香蕉：风险最高，必须谨慎
- **特点**：香蕉是呼吸跃变型水果，到店后仍在后熟，极易批量腐烂。卖相与温度、挤压直接相关。
- **试单方法**：
  - **只拿1~2箱**（约30~40斤）。哪怕供应商说“一板（24箱）起批”也别信，可以加钱让供应商拆件。
  - **选8成熟（皮绿、棱角明显）**。全黄熟果到你手最多撑1天，绿果风险大。8成熟能卖3~4天。
  - **检查时重点看“把手”**：果柄发黑、发软，或箱底有大量水渍、酒精味，直接拒收。
- **止损判断**：如果3天内底部发黑变软面积超过20%，这个供货商的香蕉不能长期合作。

### 2. 苹果：问题在“库存果”和“糖心”
- **特点**：苹果耐储存，但新手常被忽悠拿冷库旧货或打蜡果。
- **试单方法**：
  - **每样只拿半箱**（约15~20斤）。苹果不像香蕉容易烂，但品类多（红富士、嘎啦、阿克苏等），先试2~3个走量品种。
  - **现场必须切开看**：看果心是否发褐、变色（陈年冷库果特征），尝一口是否有“糠心”或异味。
  - **询问“下树日期”**：如果供应商含糊其辞说“今年的”，可以直接问“是2024年10月还是11月入库的？”越新越好。
- **止损判断**：放3天后出现明显皱皮或内部发绵，说明货源偏老，换供应商。

### 3. 橙子：注意“防腐”和“汁水”
- **特点**：橙子表皮厚，但中间易空心，且部分供货商会打保鲜剂让果皮永远光亮。
- **试单方法**：
  - **拿2~3个品种各1小筐**（约10斤/筐），如赣南脐橙、血橙、冰糖橙各一筐对比。
  - **重点看“肚脐”和“重量”**：脐橙肚脐极小或没有且手感轻飘飘的，可能空心。拿在手上有“沉甸甸”感的最好。
  - **切开检查内膜**：白色内膜明显、切面立刻出水的是好货；如果切后15分钟才出水，或者果肉白瓤厚且发干，可能是“打过头了”的次果。
- **止损判断**：室温放3天果皮变软但果肉干瘪，说明水分保持技术差，不适合长期供货。

### 4. 统一操作建议（试单期必做）
- **付款策略**：所有试单定为“**现结**”（当场付清），但告诉老板“如果质量稳定，下次可以押3天账期”。这样不会因为试单损失信誉，也保留谈判空间。
- **记录与反馈**：用手机拍下每个批次的开箱视频（重点拍：果柄颜色、果皮损伤、箱底卫生），并标注 **“到货日**→开始出售日→下架日**”**。如果同一箱果子第四天还有超过60%完好的，才值得长期拿货。
- **不要因为便宜而多拿**：试单阶段，价格贵20%~30%都可以接受，重点是看“稳定出货期”（从到货到卖完/损耗超过30%之间的天数）。稳定期超过5天才是好货源。

**最终原则**：第一次试单总货值不要超过 **300~500元**。哪怕全部卖不掉亏了，也不如长期跟差供应商合作损失大。如果发现某个品种品质不稳定，果断放弃那个供应商，第一次小亏比长期亏更好。
