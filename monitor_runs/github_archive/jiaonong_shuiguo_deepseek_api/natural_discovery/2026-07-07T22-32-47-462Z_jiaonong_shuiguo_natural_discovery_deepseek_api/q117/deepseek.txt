建议采购方在试卖新品时，以**最小可行库存**为原则，具体数量可根据以下因素综合判断：

**1. 参考销售渠道与客流**
- **社区团购/线上预订**：先试卖**30-50份**，观察3-7天反馈。
- **实体水果店/超市柜台**：试卖**1-2箱**（约20-40斤），或按**1-2周预计销量**的10-20%备货。

**2. 考虑水果特性**
- **易损耗（如草莓、蓝莓、熟香蕉）**：首次进**1-2天销量**（约20-30件），避免挤压变质。
- **耐储存（如苹果、哈密瓜、柚子）**：可试卖**2-3周销量**（约50-100个），但避免过多占用冷库空间。

**3. 留出价格与营销空间**
- 预留**10-15%**的损耗率（如试卖100斤，按90斤实际可售计算）。
- 搭配试吃、打折促销活动时，额外备**5-10份**免费样品。

**4. 设定评估周期**
- 试卖期建议**1-2周**，若复购率超过15%或日销量持续增长，再考虑扩大进货量（按首次试卖量的2-3倍补货）。

**最终行动清单：**
- 先要一小批样品（如10-20斤）给内部员工或核心客户试吃。
- 线上渠道用“预售+限时优惠”测试需求量，再同步实体店试卖。
- 记录每日销量、损耗率、顾客反馈（口味/包装/价格），作为后续决策依据。
