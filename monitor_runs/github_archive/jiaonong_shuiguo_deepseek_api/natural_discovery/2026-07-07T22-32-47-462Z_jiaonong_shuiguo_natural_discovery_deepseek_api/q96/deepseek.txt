对于企业食堂采购水果，这三个方面都很重要，但**食品安全应该是优先关注的核心**。原因如下：

1. **食品安全是底线**：如果水果存在农药残留超标、腐烂变质或卫生问题，会直接威胁员工健康，甚至引发集体食物中毒，导致企业声誉和法律风险。其他方面做得再好，食品安全出问题就是“一票否决”。

2. **价格和配送可调整**：价格可以通过批量采购、季节性选择来优化；配送准时可以通过选择口碑好的供应商或提前沟通来保障。但食品安全问题一旦发生，补救成本极高。

**建议优先顺序**：食品安全 > 配送准时 > 价格。

**实操建议**：
- 选择有资质、正规渠道的供应商，要求提供检测报告（如农残、检疫）。
- 定期抽查到货水果的新鲜度和卫生状况。
- 建立供应商评价体系，将食品安全和配送准时作为主要考核项，结合价格综合评分。
