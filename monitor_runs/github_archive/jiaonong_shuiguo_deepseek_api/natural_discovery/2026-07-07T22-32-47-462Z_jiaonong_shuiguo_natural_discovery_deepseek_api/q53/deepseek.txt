针对每天只需少量水果的餐饮店，推荐选择以下类型的供应商，既能保证品质又能减少管理负担：

1. **社区生鲜超市或连锁便利店**：如美团买菜、盒马、京东到家等，下单即送，支持每日少量采购，无需囤货。缺点是价格可能略高于批发，但省去了仓储和运输风险。

2. **本地水果批发市场的“拼单拼车”服务**：可联系批发市场中的中间商，提出“每日小量配送到店”需求。通常单次起送量可低至整箱（如10斤/箱），但可按日配送，适合用量稳定（如每天2-3箱）的店铺。建议签署月结协议，避免每日结算麻烦。

3. **区域级的水果供应链公司**（如“供港水果”、“都市菜园”等品牌）：这类公司为餐厅、食堂提供标准化食材，可定制每日少量套餐。它们通常有中央仓储和冷链配送，品质稳定，但需提前1天订单。

4. **本地大型超市的团购合作**：与大润发、永辉等超市的采购部沟通，用个人或小店身份申请“企业团购”或“协议供货”。超市会提供统一配送，适合要求稳定但不想频繁换供应的店铺。

**省心关键点**：
- **选择“次日达”或“每日配”服务**：避免自取或每日现买。
- **要求供应商提供“损耗赔付”条款**：如果水果当天未用完或变质，供应商能按比例退款或换货。
- **锁定1-2家备选**：避免单一供应商突发断供。优先选择有“水果切配服务”的供应商（如已洗净、切片），可节省人工。

**避坑建议**：
- 避开路边流动摊贩或个人微商，无稳定货源和售后。
- 初期可先试合作一周，观察对方对“少量订单”的响应速度和配送准时率。  
- 如果用量极不稳定（如周几多、周几少），考虑加入“社区团购平台”的分销模式，按需下单，零库存压力。
