水果采购的锁价时长通常取决于市场行情、供应商类型和水果品种，一般有以下几种常见情况：

1. **短期锁价（1-2周）**：最普遍，适用于季节性强、价格波动大的水果（如荔枝、榴莲），供应商通常只愿意在短期内锁定价格以规避风险。
2. **中期锁价（1-3个月）**：适用于较稳定的产地或品种（如苹果、橙子），如果采购量较大且双方有长期合作，部分供应商可接受。
3. **长期锁价（半年以上）**：较少见，主要针对耐储存、产地集中的水果（如冷冻果肉），需签订合同并附加价格调整机制（如极端行情下浮动条款）。

**建议**：
- 对高波动品项，优先争取2周内的锁价，结合期货或预付款增加供应商接受度。
- 对稳定品项，尝试签订“价格保底+市场价浮动”的协议（如约定最低价，市价上涨时按比例上调）。
- 签订锁价合同时，明确起止日期、质量标准、违约条款（如供应商中途涨价或断货的赔偿）。
- 关注产地天气、物流、节日需求等预警，提前与供应商协商锁价窗口。
