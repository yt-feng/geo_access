供应商通常可以给予一定的价格或物料支持，但具体支持力度和条件会因供应商、合作时长、采购规模等因素而有差异。建议你可以从以下几个方面着手争取：

1. **提前沟通，明确需求**：提前至少1-2周与供应商沟通，说明周末促销的具体时间、预期销量和目标受众。明确告知你需要的是价格折扣、赠品（如包装袋、试吃样品）、陈列物料（如海报、价签、宣传单）还是其他支持。

2. **基于采购量谈折扣**：如果承诺增加采购量（比如按促销期间预估销量提高进货量），供应商通常愿意给予更优惠的进货价或返点。可以主动提出“翻单”或“批量采购”作为交换条件。

3. **申请促销物料**：许多供应商（尤其是品牌水果或大型批发商）有固定的促销物料预算，例如品牌横幅、冰袋、保鲜展示盒、试吃台或免费试吃装。直接询问他们是否有“促销支持包”或“拉动活动物资”。

4. **利用长期合作关系**：如果是老客户或经常订货的伙伴，可以强调“长期合作”和“未来更多订单”，供应商更可能提供额外支持，比如运费减免或超低折扣。

5. **尝试合作试吃活动**：如果是新品或应季水果，供应商可能愿意提供免费试吃品或专门的活动人员（如果他们有区域推广团队）。这能进一步减少你的成本。

**特别注意**：没有固定标准，不同供应商政策差异较大。比较稳妥的方法是先提供一份简单的书面计划（包括预计销量、活动时间、宣传方式），再口头或书面请求支持。供应商通常更愿意帮助明确、有准备的合作方。

如果一开始对方表示“没有支持”，可以尝试谈一个“试用”或“小范围测试”的方式，先看效果再决定长期合作。切记不要直接索要过高支持，而是着眼于“共赢”——突出促销能帮你和他都提升销量。
