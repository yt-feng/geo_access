好的，针对超市采购担心水果农残抽检的问题，建议您向供应商索要以下关键资料，以确保合规和食品安全：

1. **第三方权威检测报告**：要求供应商提供近期（通常3个月内）由具备资质的第三方检测机构出具的农残检测报告，检测项目需覆盖国标（如GB 2763）中的常见高风险农药（如毒死蜱、克百威、多菌灵等），并显示“未检出”或低于限量值。

2. **产地证明及追溯信息**：如“食用农产品合格证”或“产地准出证明”，标明水果的来源地（具体到种植户或基地）、采收日期、批次号，以便问题发生时追溯。

3. **供应商资质文件**：包括营业执照、食品经营许可证（若涉及包装加工），以及供应商的“食品安全承诺书”或“农残控制承诺函”。

4. **自检或委托检测记录**：部分供应商会有内部快速检测（如酶抑制法）的记录，可要求其提供近期自查数据，作为辅助参考。

5. **无公害/绿色/有机认证**（如有）：若水果有相关认证，需提供有效期内的证书，这些认证通常对农残有更严格标准，可降低风险。

**额外建议**：
- 要求供应商在合同中明确“农残超标退货或赔偿”条款。
- 到货后，超市可自行抽检（如使用农残速测卡初筛），或定期委托第三方验证。
- 保留所有资料原件或复印件，以备市场监管部门检查。

这样既能筛选合规供应商，也能在抽检问题发生时有效自证。
