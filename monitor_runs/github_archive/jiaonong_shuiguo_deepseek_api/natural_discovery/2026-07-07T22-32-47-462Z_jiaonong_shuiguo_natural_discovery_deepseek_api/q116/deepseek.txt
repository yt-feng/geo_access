对于小商家来说，现金结算和月结账期是两种常见的结算方式，但各有不同的风险：

**现金结算的风险：**
1. **资金压力大**：需要一次性支付货款，可能占用大量流动资金，影响其他经营支出（如备货、人工）。  
2. **缺少保障**：如果水果有质量问题（如腐烂、不新鲜）或数量短缺，现金交易后难以追溯或索赔（特别是无合同或收据时）。  
3. **丢失或假币风险**：需要携带大量现金，可能被盗窃、遗失，或收到假币。  
4. **不便于财务记录**：现金交易容易遗忘或记错账，不利于核算成本和利润。

**月结账期的风险：**
1. **卖方（供应商）信任要求高**：供应商可能不愿意给小商家长期账期，除非有良好信用或预付定金。  
2. **拖账或坏账风险**：如果自家资金周转出现问题，无法按时付清货款，可能导致供应商断货或起诉。  
3. **利息或附加成本**：部分供应商会对月结加收手续费或提高单价，变相增加采购成本。  
4. **水果损耗问题**：如果月结期内水果降价或变质（如季节水果），你需要自行承担这部分损失。

**建议：**
- 若现金结算，务必当面清点数量、检查质量，并索要正规收据或发票。  
- 若选择月结，先与供应商签简单合同（注明账期、违约金、质量责任），并评估自己是否有稳定回款能力。  
- 对小商家而言，可以尝试“混合支付”：部分现金+部分短期账期（如15天），以平衡风险和资金。
