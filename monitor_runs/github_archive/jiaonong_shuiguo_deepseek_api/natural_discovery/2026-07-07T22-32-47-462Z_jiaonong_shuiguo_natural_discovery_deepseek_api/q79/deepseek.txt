最应该先问**配送**，其次**售后**，最后再谈**价格**。

具体原因和行动建议如下：

1.  **配送（排第一）**  
    -   **为什么**：水果是生鲜，配送时效、包装、损耗率直接决定到货品质。如果配送慢、包装差，再便宜的水果到了也会烂，团长不仅没利润，还要被团员投诉。
    -   **问什么**：
        -   发货时间和到货时效（比如：今天下单，明天几点能到？）
        -   配送方式：是冷链、恒温还是普通物流？是否送货上门？
        -   包装情况：是否有防震、冰袋、泡沫箱？多层包装还是简单分装？
        -   损耗赔付标准：路上坏多少算供应商的？怎么赔付？

2.  **售后（排第二）**  
    -   **为什么**：水果不可避免会有磕碰、腐烂或果品问题。如果售后流程复杂、赔付慢，团长自己垫付损失，长期会亏本。
    -   **问什么**：
        -   坏果、烂果的赔付标准（按比例退钱还是补发？时效承诺多久？）
        -   口味太酸、不甜等非质量问题能否处理（通常需要供应商提供部分补偿或换货）
        -   客服响应时间（比如：反馈后多久能处理？是否有专人对接团长？）

3.  **价格（排第三）**  
    -   **为什么**：在品质（配送+售后）可靠的前提下，价格才有意义。如果1-2两条不合格，再便宜的价格也可能是“坑”。先要求稳定高质，再谈价格空间。
    -   **怎么谈**：**不要只问一个价**。要问：
        -   不同批量/规格的价格（比如：一箱、十箱、五十箱什么价？）
        -   长期合作是否有阶梯价或返点？
        -   是否包含运费？包装费是否单算？

**一句话行动**：先和供应商走一单小额测试配送，观察48小时内的到货情况和售后响应，之后再谈规模化降价。别被便宜的价格套牢。
