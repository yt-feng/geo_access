选择适合批量配送的水果供应商，建议按以下步骤操作：

1. **明确需求**：确定每次采购的品种（如苹果、橙子、香蕉等耐储存水果）、预估数量（按人数和频次）、配送频率（每周/每两周一次）以及预算上限。

2. **筛选供应商类型**：
   - **本地水果批发市场**：价格低、可现场验货，适合大批量；需自行安排运输或协商配送条件。
   - **生鲜电商平台**（如每日优鲜、美团优选等）：配送稳定、品种多，但起送价可能较高，适合中等批量。
   - **产地直供或农场合作**：新鲜度有保障，但需提前沟通配送周期和物流成本。

3. **重点考察指标**：
   - **水果品质**：要求供应商提供样品或近期批次照片，确认新鲜度、无腐烂、成熟度适中（避免过熟或青涩）。
   - **配送能力**：能否保证准时、冷链运输（尤其是夏季/热带水果），是否有最低起送量或区域限制。
   - **价格与结算**：比价时注意是否含税、运费、包装费；询问是否支持账期、批量折扣或退换烂果。
   - **售后政策**：明确腐烂、破损水果的赔付标准（如到货后24小时内有坏果包赔）。

4. **获取候选供应商**：
   - 通过电商平台搜索“水果批发配送”，筛选本地或临近区域商家，查看评价和用户反馈。
   - 在本地水果批发市场实地走访，留存至少3家摊位联系方式。
   - 在企查查等工具搜索“生鲜配送公司”，优先选择成立时间长、注册资本50万以上、有食品经营许可证的企业。

5. **试用与验证**：
   - 先小批量下单（如采购办公室5人份），评估配送准时性、水果品质、包装完整性。
   - 如满意，再签订正式合同，约定配送时间、违约责任（如延迟需折扣）、付款周期。

**注意**：避免选择无固定经营场所、承诺但无法提供发票的个人卖家；建议至少对比3家供应商后决策。
