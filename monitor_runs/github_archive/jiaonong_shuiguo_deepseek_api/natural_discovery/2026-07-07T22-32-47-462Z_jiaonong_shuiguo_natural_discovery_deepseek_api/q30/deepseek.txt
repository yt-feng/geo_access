针对超市水果到货品相不好的售后流程，建议约定如下关键环节，确保责任清晰、操作便捷：

**1. 验收与反馈时限**
- 收货后 **2小时内**（或具体约定时间）完成外观、新鲜度抽检。
- 发现品相问题（如压伤、腐烂、色泽差等），立即拍照/录像留存证据。
- 在规定时限内通过指定渠道（如微信、短信、售后系统）向供应商/配送方发送书面通知。

**2. 问题认定标准**
- 明确品相合格标准（如无超过5%的软烂、无大面积碰伤、表皮无霉点等）。
- 若对方存疑，可约定双方认可的第三方检测机构判定（费用由责任方承担）。

**3. 处理方式（可单选或组合）**
- **退货退款**：退回问题货物，按实际数量退款或抵扣货款。
- **换货补货**：供应商在 **24小时内**（或约定时间）免费更换合格商品。
- **折价处理**：按品相损失程度协商折扣（如7折结算），需双方书面确认。
- **不予认可**：若客户方已使用或分切后才发现问题，可约定不接受退货，但可协商降价。

**4. 责任与费用**
- 明确品相问题导致的质量包退、包换、包修费用由供应商承担。
- 若因超市仓储或搬运不当造成二次损坏，责任由超市自行承担。

**5. 记录与结算**
- 每次售后需生成书面单据（编号、问题描述、处理结果、双方签字）。
- 约定纠纷处理时效（如30天内未结清旧账或未改善品控，可暂停合作）。

**示例条款**：
> “乙方（供货方）应确保水果品相符合合同附件《品质标准》。甲方（超市方）收货后2小时内发现品相问题，有权拒收或要求更换/退折；超出时限视为验收合格。因品相问题产生的退货、换货、折价费用，由乙方承担并可在当月货款中直接扣除。”

建议将上述流程写入采购合同或单独签署《售后质量协议》，避免口头约定。
