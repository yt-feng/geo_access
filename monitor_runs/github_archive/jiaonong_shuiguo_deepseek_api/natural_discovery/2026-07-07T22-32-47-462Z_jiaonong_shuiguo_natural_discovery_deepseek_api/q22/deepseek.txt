针对超市水果区提高客单价，建议采用**“引流爆款 + 高毛利搭配 + 场景化升级”**的组合策略。

以下是针对香蕉、菠萝、苹果的具体促销方案：

**核心组合策略：**

1.  **香蕉（低价引流品）**：作为日常刚需，价格敏感度高，用来吸引顾客走到水果区。
2.  **菠萝（中价/高毛利品）**：具有季节性、需要现场加工（削皮），能增加顾客停留时间和二次购买机会。
3.  **苹果（高复购/稳定毛利品）**：作为搭配销售的主力，适合成袋/成箱销售，提升单次消费金额。

**推荐三种具体促销组合方案：**

### 方案一：健康活力套餐（推荐，最稳妥）

-   **组合形式**：**“香蕉 + 苹果”** 绑定销售，**菠萝作为加价购**。
-   **促销话术**：*“每日能量套餐：香蕉苹果各1斤，仅需X元（比单买便宜15%）。加3元换购现削菠萝半个！”*
-   **定价逻辑**：
    -   香蕉：维持市场价或微亏引流（如2元/斤）
    -   苹果：选取中等价位品种（如红富士），与香蕉打包后，苹果相当于享受了折扣，但整体客单价从单买香蕉的5元提升到15-20元。
    -   菠萝：作为加价购，毛利空间大，提升附加价值。
-   **执行动作**：将香蕉和苹果用网袋装好，放在收银台附近。菠萝现削，用保鲜盒装好放在旁边。

### 方案二：热带风情派对（高客单价，适合周末）

-   **组合形式**：**“菠萝 + 苹果”** 组合成**水果礼盒或果切拼盘**，香蕉作为凑单补贴。
-   **促销话术**：*“整颗菠萝+3颗苹果，礼盒装仅需XX元。买即送2根香蕉！”*
-   **定价逻辑**：
    -   核心是菠萝（视觉冲击力强）和苹果（易储存、耐摆放）。
    -   赠送的香蕉看似免费，实际成本已摊入礼盒售价中。这比直接降价更能维持商品档次。
-   **执行动作**：在水果区显眼位置设置“派对精选”堆头，用保鲜膜包好礼盒，并搭配一次性水果叉、一次性手套。

### 方案三：自制果茶/健康轻食（吸引年轻人，提升连带率）

-   **组合形式**：**“香蕉 + 苹果 + 菠萝”** 三品**指定数量**后，赠送**蜂蜜或酸奶包**。
-   **促销话术**：*“自制高纤活力果饮：买2根香蕉+1个菠萝+3个苹果，送1盒酸奶/1小瓶蜂蜜酱，回家打果汁或做沙拉！”*
-   **定价逻辑**：
    -   利用“简单食谱”降低决策门槛。顾客为了得到赠品，会一次性购买三种水果，并可能额外购买其他调味品区的商品（超市可联动促销）。
    -   香蕉和苹果提供膳食纤维和量感，菠萝提供风味，赠品提升复购冲动。
-   **执行动作**：在旁边张贴一张海报：*“推荐搭配：酸奶菠萝苹果杯 或 香蕉菠萝牛奶昔”*。

**关键执行提醒：**

-   **位置**：组合产品放置于**动线末端**或**收银台前**，而不是门口。避免顾客刚进店就拎着水果逛。
-   **陈列**：菠萝要**提前削好、切片装盒**，配上竹签，纳入“即食区”，这样客单价提升最明显。
-   **时机**：菠萝价格波动大，建议在菠萝上市中期开始此活动，因为此时价格稳定且口感最佳。

**总结建议：**
优先采用**方案一**，因为它**确定性最高**、**风险最小**。如果周边有办公楼或社区，可增加方案三的食谱卡片。
