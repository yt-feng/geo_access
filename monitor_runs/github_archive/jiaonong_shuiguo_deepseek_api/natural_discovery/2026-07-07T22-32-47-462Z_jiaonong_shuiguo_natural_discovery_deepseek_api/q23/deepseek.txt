通常情况下，超市做水果促销时，供应商是**能够提供陈列、海报或试吃支持**的，因为这是供应商与超市合作的常见市场推广方式。

**具体支持内容及建议：**
1. **陈列支持**：供应商可以协助设计或提供特殊的陈列架、堆头、端架，以突出促销水果，提高视觉吸引力。
2. **海报支持**：供应商可提供促销海报、POP广告、价格标签或其他宣传物料，帮助吸引顾客注意。
3. **试吃支持**：通过提供试吃样品、人员和相关设备，来促进顾客对水果的尝试和购买。

**实际操作建议：**
- 建议你与供应商**提前沟通**，明确促销时间、水果品种、促销目标等，以便供应商准备相应资源。
- 如果是长期或大型促销，可以考虑签订**书面协议**，明确费用分摊、物料归还等责任。
- 注意确认超市自身是否有品牌或陈列标准，避免物料不符合要求。
- 对于试吃，需遵守超市的食品安全和卫生规定，提前办理相关手续。

**结论**：供应商通常愿意提供这些支持，尤其是为了促进销量和品牌曝光。但具体落实情况取决于供应商的能力、合作深度以及促销活动的规模。建议你主动联系供应商洽谈细节。
