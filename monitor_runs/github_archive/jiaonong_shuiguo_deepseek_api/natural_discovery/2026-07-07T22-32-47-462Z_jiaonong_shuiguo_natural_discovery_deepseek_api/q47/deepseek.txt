在茶饮店采购香蕉用于饮品或甜品时，控制熟度的关键是明确**使用场景**，然后选择对应成熟阶段的香蕉。以下是具体建议：

### 1. 根据用途确定最佳熟度
- **做奶昔、冰沙、果泥（需香蕉味浓郁、口感绵密）**  
  → **全熟阶段**：香蕉皮出现黑色斑点（糖斑），果肉变软、甜度高。此时香蕉风味最浓、易打碎，无需额外加糖。  
  ✅ 判断标准：手捏有轻微凹陷，但内部无腐烂。

- **做甜品装饰、果切、分层饮品**  
  → **半熟偏熟**：香蕉皮全黄，无黑斑，果肉稍硬。切块成型好，不易氧化变黑，口感微甜带韧性。  
  ✅ 判断标准：果皮平滑无黑点，用力捏能感觉内部坚实。

- **做烘焙（如松饼、蛋糕）或长期冷冻保存**  
  → **极熟（甚至带软烂感）**：香蕉皮变黑、果肉极软。甜度最高，适合冷冻后打成泥。  
  ⚠ 注意：需去除任何发酸或发酵部分。

### 2. 采购验收清单
- **看颜色**：  
  - 黄色为主（无绿皮） → 可1-2天内使用  
  - 出现黑斑（5-10个针尖大小） → 当天可使用  
  - 全皮变黑 → 只适合做冷冻果泥  
- **摸硬度**：  
  - 按下能回弹 → 适合切片  
  - 按下不回弹 → 适合捣泥  
- **闻气味**：  
  - 无发酵酒精味 → 安全  
  - 若有酒味 → 适合烘焙，但售价应降低

### 3. 供应链管理技巧
- **批次分开成熟**：采购时配比
  - 当日需用：全熟（70%）
  - 次日需用：中熟（30%）
- **催熟/延迟方案**：
  - 催熟：未熟香蕉与苹果/梨放纸袋，室温放置1-2天
  - 延迟：冷鲜保存（13-15℃），不要放入冰箱（易冻伤变黑）
- **统一标准**：与供应商约定  
  - 明确要求“茶饮店规格”（如：全黄略带斑点，无机械伤）  
  - 每周要求供应商更新到店香蕉的“中心硬度值”（可用硬度计抽检）

### 4. 常见风险规避
| 问题 | 原因 | 对策 |
|------|------|------|
| 发苦、涩感 | 未熟 | 加50℃温水浸泡15分钟可部分脱涩，但风味会降低 |
| 果肉发黑 | 冻伤或碰撞 | 拒收有内部瘀伤的香蕉 |
| 次日开始腐烂 | 成熟度过高且收到后未及时处理 | 当天立即分切成块/泥冷冻 |

### 5. 操作建议
- **采购量预测**：  
  香蕉熟化速度一般为3-5天（室温）。按“当日使用量×1.5倍”采购，避免因温度波动导致浪费。
- **设立“香蕉熟度图”**：  
  在操作区贴不同用途对应的香蕉颜色/硬度照片，培训员工在接货时按图分类。
- **高性价比方案**：  
  如果用量大，直接联系批发商专供“8成熟青黄香蕉”（弹性好、运输损耗低），回店后用乙烯利溶液（按说明稀释）均匀喷涂，可精准控制到果皮变色。
