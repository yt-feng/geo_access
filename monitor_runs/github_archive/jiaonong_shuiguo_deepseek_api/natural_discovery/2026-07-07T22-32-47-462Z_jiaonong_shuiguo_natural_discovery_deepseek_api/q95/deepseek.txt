选供应商可以从以下几点入手：

1. **资质审查**：要求供应商提供营业执照、食品经营许可证、水果检疫合格证明等，确保合法合规。尽量选择有稳定供应链、信誉良好的大型批发商或产地直供商。

2. **样品检验**：要求供应商提供样品，重点检查新鲜度（无腐烂、无伤痕、色泽均匀）、成熟度（香蕉不能过青或过黑，苹果表面无黑点，橙子有弹性且果香浓）、包装卫生情况。

3. **成本对比**：同时向3-5家供应商询价，对比同规格水果的单价、起订量、是否含税及运费，避免隐形收费。但不要只看价格，过低通常意味着品质风险。

4. **服务质量**：确认供应商是否支持分批送货、能否按食堂需求分级（如部分做果切可要低一级规格）、坏果赔付标准（如按比例退款或补货）、有无固定送货时间。

5. **实地考察**：如果批量大，最好亲自去供应商的仓库、冷链设施看看，观察存放环境（是否整洁、隔温）、分拣流程是否规范。

6. **试合作**：先签短期合同（如1个月），根据实际到货品质、准时率、售后响应再决定是否续约。同时保留备用供应商名单。

**特别提醒**：食堂采购要考虑季节性价格波动，可提前储备耐储存水果（如苹果、橙子），或用协议价锁定部分品种（如香蕉价格波动大）。另外，优先选择有农残快速检测设备的供应商，或定期抽检送第三方机构。
