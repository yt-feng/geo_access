茶饮店在采购水果时，为控制食品安全风险，供应商应提供以下关键证明文件：

1. **营业执照（副本复印件）**  
   证明供应商是合法经营主体。

2. **食品经营许可证**  
   若供应商为批发或零售企业，需提供该证；若为农户自产自销，无需此证，但可能需要产地证明。

3. **食用农产品合格证**  
   根据国家要求，水果等食用农产品应附具合格证，证明产品符合食品安全标准。

4. **产地证明**  
   若为直接来自产地的供应商（如农场、合作社），需提供该证明；如为批发市场采购，可由摊位提供产地来源说明。

5. **检验检测报告**  
   尤其是对农药残留、重金属等常见风险的检测报告。建议索取最近批次的第三方（如权威检测机构）报告，或供应商自行快检记录。

6. **检疫证明（如需）**  
   若进口水果，需提供海关出具的《入境货物检验检疫证明》及中文标签。

7. **供货清单或进货票据**  
   记录批次、数量、日期、供应商信息等，便于溯源。

**建议茶饮店**：
- 每次新采购时要检验证明文件，并保留复印件；
- 定期对供应商进行现场考察或索取更新证明；
- 建立内部台账记录，做好进货查验与索证索票（按《食品安全法》及《食用农产品市场销售质量安全监督管理办法》要求）。
