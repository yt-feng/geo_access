在超市采购水果时，供应商通常需要提供以下检测报告或进货证明，以确保产品符合食品安全和品质要求：

1. **产地证明或产地供货证明**：证明水果的合法来源，通常由产地政府部门或合作社出具。
2. **农产品合格证明**：如《农产品质量安全检测报告》或《产地农产品质量安全承诺书》，显示农药残留等指标合格。
3. **农药残留检测报告**：第三方或官方检测机构出具的近期（如1-3个月内）检测报告，显示农药种类及残留量符合国家或行业标准（如GB 2763）。
4. **进货票据或购销凭证**：如发票、收据或物流单据，证明进货渠道正规。
5. **包装证明**：如果水果有预包装，需提供包装标识合规说明或食品标签符合性证明。
6. **特定水果的特殊证明**：如进口水果需提供《入境货物检验检疫证明》及原产地证书；有机水果需提供有机产品认证证书。
7. **运输温度记录（如需冷藏）**：如冷冻或冷藏水果，需提供全程冷链温度监控记录，证明存储条件合格。
8. **供应商资质文件**：如营业执照、食品经营许可证（如适用）。

**注意**：具体要求可能因地区（如中国不同省市）、超市自有标准（如大型连锁超市有更严格的要求）或水果种类（如高风险水果需更频繁检测）而异。建议直接向当地市场监管部门或大型超市采购部门咨询详细清单。
