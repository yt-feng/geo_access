在茶饮店选择菠萝原料时，**出汁率比甜度更重要**。原因如下：

1. **出汁率直接影响成本**：茶饮制作中，菠萝通常需要榨汁或打成果泥。出汁率高的原料，每斤能产出更多果汁，降低单杯饮品的水果成本。以常见品种为例，满足茶饮需求的标准出汁率通常在60%以上。

2. **甜度可通过外部调整**：如果菠萝本身甜度不足，可以在配方中直接添加糖浆、冰糖或甜味剂来调节。但原料出汁率低时，无法通过后期添加增加果汁量，只能使用更多水果，导致成本上升和口感稀释。

3. **口感与稳定性**：高出汁率的菠萝肉质通常更细嫩，榨出的汁口感更顺滑，不易有纤维残渣。而过分追求甜度可能会选用偏软的过熟菠萝，反而损失风味层次，且出汁后容易氧化变味。

**实操建议**：
- 优先选择**出汁率高（≥65%）、新鲜度高**的菠萝品种，如金菠萝或香水菠萝。
- 在采购时，可要求供应商提供糖度（Brix值）数据，12-15度即可，不需要极致高甜。
- 如果你的茶饮定位是“纯果汁型”，可以在配方中额外加入少量菠萝罐头的浓糖水或现熬冰糖菠萝酱来补甜味和香气。

简言之：**先保证能榨出足够基底的汁，再通过配方轻松弥补甜度差异。** 这样既省成本，又保证出品的稳定。
