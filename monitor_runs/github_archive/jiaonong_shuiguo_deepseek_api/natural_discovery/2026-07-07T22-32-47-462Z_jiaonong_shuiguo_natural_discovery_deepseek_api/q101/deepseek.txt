与酒店早餐水果供应商合作时，需重点关注以下要点：

1. **质量与品种稳定性**  
   - 明确早餐所需水果的品种（如香蕉、苹果、橙子等）、成熟度、大小、色泽标准。  
   - 要求供应商提供定期质量报告，避免出现腐烂、过熟或未熟等问题。  
   - 建议签订合同时附上《水果品质验收标准清单》，包含可接受瑕疵比例（如≤5%）。

2. **供应能力与时效性**  
   - 确保供应商有能力保证每日/每周稳定供货，尤其节假日或旺季需求增加时。  
   - 明确交货时间（如每日6:00前送达）、配送频次（如每日1次）及应急补货机制（2小时内响应）。  
   - 要求供应商提供库存预警方案，避免断供。

3. **价格与结算条款**  
   - 明确单价是否包含送货费和税点，约定价格调整机制（如因季节或市场波动，每季度协商一次）。  
   - 付款方式建议月结（如每月15日结清上月款项），预留10%-15%质保金至收货确认无误后支付。

4. **卫生与冷链运输**  
   - 核查供应商是否具备食品经营许可证、冷链物流资质。  
   - 要求所有水果运输全程使用清洁冷藏车辆，温度控制在0-10℃（视品类调整）。  
   - 要求每日提供消毒记录和运输温度监测数据。

5. **合作风险控制**  
   - 签订合同明确违约责任，如供应商连续3次不合格可单方解约并索赔。  
   - 建议同时储备1-2家备选供应商，避免单一依赖。  
   - 定期抽检水果农残检测报告（建议每季度一次），保留送检权利。

**实操建议**：  
- 小批量试合作1个月后，再签长约。  
- 要求供应商提供5-8种当季水果的备选清单，提升早餐菜单灵活性。  
- 在合同中加入“明示条款”：若因水果质量问题导致顾客投诉，供应商需承担全部直接损失。
