判断香蕉熟度和坏果风险，建议按以下步骤操作：

1. **观察果皮颜色**  
   - 全绿：未熟，需催熟，风险较低但运输中易擦伤。  
   - 黄绿相间（果柄绿、中部黄）：最佳拿货期，可存放3-5天。  
   - 全黄且出现少量褐斑：基本成熟，需尽快销售，24小时内易变软。  
   - 大面积黑斑/水渍状：过熟或已开始腐烂，风险高，不宜拿货。

2. **触摸果实硬度**  
   - 硬挺有弹性：正常，尚可存放。  
   - 明显软糯：内部可能已发酵，或受冻伤（冷害导致内部软烂但外观尚可）。  
   - 局部凹陷、渗水：挤压伤或真菌感染，坏果风险高。

3. **检查果柄与切口**  
   - 果柄发黑、发霉：暗示储存期间湿度高，内部可能已感染。  
   - 切口处有白色/绿色霉斑：运输过程中已有霉变，应拒收或扣除比例。

4. **闻气味**  
   - 正常香蕉清香：安全。  
   - 酒精味或酸腐味：果肉已发酵，无法长期保存，必须当天处理或退货。

5. **抽查开箱**  
   - 随机取3-5箱各中部和底部位置，每箱翻查10根。  
   - 若单箱坏果率超过5%（如整箱25根中有2根以上黑烂），整个批次风险高，需协商降价或退货。

6. **评估运输与包装**  
   - 包装箱干燥、有透气孔：减少闷坏。  
   - 箱内壁有水珠或潮湿：可能被泡过水，后续会加速腐烂。  
   - 堆压严重（下层箱体变形）：底层香蕉必然有压伤，坏果率上升。

**紧急建议**：  
- 若需要快速判断整批货，优先抽查**箱子中部**（温度最高、易闷熟）和**底部**（受压最大）。  
- 拿货后24小时内拆箱并放置通风处，如未及时销售，可放入10-12℃冷库（低于8℃会冻伤发黑）。  
- 具体坏果率标准可参考合同约定（通常允许3%-5%自然损耗），但遇霉斑或水渍应直接拒收。
