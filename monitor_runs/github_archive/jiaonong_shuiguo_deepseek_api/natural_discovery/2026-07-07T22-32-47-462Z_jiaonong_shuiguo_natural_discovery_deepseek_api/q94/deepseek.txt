是的，供应商能否支持小批量试卖非常重要。原因如下：

1. **降低库存风险**：水果保质期短，销量不稳定容易造成损耗。小批量试卖能减少滞销和腐烂损失。
2. **灵活调整品类**：可以测试不同水果的受欢迎程度，根据销售数据快速调整进货种类和数量。
3. **减少资金压力**：不需要一次性投入大量资金押在库存上，现金流更健康。
4. **控制试错成本**：如果某款水果不适合门店周边客群，小批量试卖损失的金额和精力都更可控。

**建议**：
- 优先选择提供“最低起订量低”或“支持样品/小批次”的供应商，即使单价稍高也更划算。
- 合作前明确询问是否允许“补货周期短、拆包发货”等灵活性条款。
- 如供应商不支持小批量，可考虑联合附近其他小店集中采购，或先用微信群等线上渠道预售测试需求。

这样能帮你更稳妥地管理水果品类，减少因销量波动带来的损失。
