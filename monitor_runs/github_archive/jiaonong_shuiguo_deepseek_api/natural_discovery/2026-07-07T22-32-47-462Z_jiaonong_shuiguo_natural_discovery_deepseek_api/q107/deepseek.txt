是的，起订量太高确实会增加损耗风险。具体原因和应对建议如下：

1. **原因分析**：香蕉和菠萝属于易腐水果，成熟后保鲜期短（香蕉约3-7天，菠萝约5-7天）。如果单次采购量超过销售速度，未及时售出的部分容易腐烂变质，导致直接损失。

2. **风险评估**：高起订量还会带来储存压力——香蕉需避免低温（12-15℃最佳），菠萝需避光通风。库存积压时若仓储条件不达标（如温度、湿度控制不当），损耗率会显著上升。

3. **建议措施**：
   - **协商灵活供应**：与供应商沟通，尝试分批次发货（如每周两次），或接受起订量但按实际需求分期配送。
   - **精准预测需求**：根据历史销售数据、节假日、天气等因素，计算每日/每周合理销售量，反推采购量。例如，日均销20串香蕉，起订量设为100串时，可分摊到5天内售完。
   - **设立周转库存**：优先采购成熟度较低的水果（如青香蕉、未转黄的菠萝），延长货架期；同时制定促销策略（如组合折扣）加速清货。
   - **关注损耗指标**：记录每次采购的损耗比例，若超过5%-10%，需重新评估起订量或供应商选择。

总结：起订量过高会直接放大损耗风险，尤其对保鲜期短的水果。通过分批次采购、精确预测和库存管理，可在控制成本的同时降低损失。
