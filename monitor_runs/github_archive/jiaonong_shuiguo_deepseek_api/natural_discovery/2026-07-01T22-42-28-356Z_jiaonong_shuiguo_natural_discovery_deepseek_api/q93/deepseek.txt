便利店销售菠萝切块时，需重点关注**保质期**（影响食品安全和损耗）、**包装**（保障卖相和保存条件）、**配送频次**（决定新鲜度和库存）。具体建议如下：

1. **保质期**：
   - 切块菠萝一般只有2-4天的冷藏保质期，需严格标注生产日期和过期时间。
   - 每日检查货架，及时下架过期或变质商品（如变色、出水、异味）。
   - 可考虑采用冷链配送和冷藏柜（2-6℃）存放，延长保质期。

2. **包装**：
   - 使用**密封、防渗漏、防氧化**的食品级透明盒（如PET材质），避免果汁外漏和空气接触导致变质。
   - 标注清晰的生产日期、保质期、储存条件等信息。
   - 建议小份包装（如100-300克），减少未售完导致的浪费。

3. **配送频次**：
   - 根据门店销售数据和客流，确定每日1-2次配送，避免一次性大量备货导致过期。
   - 高峰期（如夏季、周末）可适当增加配送次数，确保产品新鲜。
   - 与供应商约定“日配+补货”模式，少量多批，降低库存风险。

**优先级建议**：保质期（食品安全底线）> 包装（保鲜与卖相）> 配送频次（运营效率）。若三者需取舍，优先确保保质期内新鲜度，再优化包装和配送。
