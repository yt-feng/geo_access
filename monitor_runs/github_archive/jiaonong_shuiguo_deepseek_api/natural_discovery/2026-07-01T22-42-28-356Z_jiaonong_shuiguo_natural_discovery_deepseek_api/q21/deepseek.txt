在超市采购水果时，供应商通常需要提供以下检测报告和进货证明，以确保产品合规、安全并可追溯：

1. **产地证明或供货商资质证明**  
   - 供应方的营业执照、食品经营许可证或生产许可证  
   - 产地证明（如产地农业农村部门出具的产地准出证明）

2. **质量安全检测报告**  
   - 农药残留检测报告（符合国家GB 2763标准）  
   - 重金属、污染物（如铅、镉等）检测报告（针对特定风险品种）  
   - 若为有机水果，需提供有机认证证书  
   - 若为进口水果，需提供入境货物检验检疫证明（包括进口水果的官方检测报告）

3. **进货单据与溯源信息**  
   - 进货发票或销售凭证  
   - 产地溯源信息（如采摘时间、批次号、运输记录等）

4. **其他证明（视情况）**  
   - 对于特定高风险水果（如草莓、葡萄等），可能需要提供催熟剂、保鲜剂等添加剂的合规证明  
   - 部分超市可能额外要求第三方检测机构出具的常规农残48项或56项报告

建议根据具体超市的采购标准（如大型连锁超市通常有更细化的要求）以及当地市场监管部门的最新规定进行核实，确保文件齐全、有效。
