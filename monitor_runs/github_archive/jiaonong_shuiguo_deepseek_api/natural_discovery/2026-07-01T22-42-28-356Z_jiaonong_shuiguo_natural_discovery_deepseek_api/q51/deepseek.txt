建议固定产地并尽量稳定供应批次，但也要保留一定的灵活性。具体分析如下：

1. **固定产地的好处**：不同产地的同一种水果（如芒果、柠檬、草莓）在甜度、酸度、香气和含水量上差异较大。固定产地有助于保持饮品口味的一致性，减少季节变化带来的波动，也能简化供应链管理。

2. **批次稳定的重要性**：同一产地不同批次的成熟度、采摘时间、存储条件也可能导致风味差异。与供应商协商定期供货（如每周固定批次），并建立验收标准（如糖度、颜色、硬度等），能进一步控制波动。

3. **灵活应对的措施**：水果本身是农业产品，受气候、自然灾害等影响，即使固定产地也可能出现质量下降。建议：
   - **备选产地清单**：与2-3个优质产地的供应商签约，当主产地品质不稳定时迅速切换。
   - **分级采购**：指定主、副批次，主批次用于核心饮品，副批次可用于特调或季节限定款，减少整体风险。
   - **多点尝样**：每次到货前要求供应商寄送小样测试，通过盲测确认风味符合标准后再批量采购。

4. **实际操作建议**：在合同中明确“风味一致性要求”，比如规定酸度范围、糖度波动不超过±1度等，并要求供应商提供每批次的农残检测和风味报告。同时，店内可保存少量上一批次样品，用于对比新批次。

总之：固定产地和批次是降低波动的基础，但一定要建立弹性供应体系和验收标准。这样既能稳定品质，又能应对突发情况，避免饮品口味严重偏离顾客预期。
