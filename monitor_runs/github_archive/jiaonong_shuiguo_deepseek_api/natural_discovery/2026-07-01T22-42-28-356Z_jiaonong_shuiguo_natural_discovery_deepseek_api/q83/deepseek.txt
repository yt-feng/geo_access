对于生鲜电商水果到货后出现坏果的情况，责任划分通常需要结合**运输环节**、**包装方式**、**商品本身质量**以及**签收状态**来综合判断。以下是一些常见的责任划分原则：

1. **运输途中损坏**：  
   - 如果坏果是由于快递运输过程中挤压、碰撞、温度不当等外部因素导致，**责任一般由物流方或卖家承担**。电商平台或卖家有义务选择合适包装和运输方式，如使用泡沫箱、冰袋、防震材料等。  
   - 如果物流环节有明显暴力装卸或延迟，电商可向物流追责，但顾客应当直接向卖家反馈。

2. **商品本身质量问题**：  
   - 如果到货时水果已经霉变、腐烂、虫蛀等，属于**发货前质量不合格**，责任在于卖家。顾客有权要求退款、补发或赔偿。  
   - 部分电商会承诺如“坏果包赔”“按比例赔付”，此时按约定处理。

3. **包装不当**：  
   - 若包装过于简陋、没有缓冲或保温措施，导致水果在正常运输中受损，责任归**卖家**。顾客拍下包装照片可作为证据。

4. **顾客签收后未及时检查**：  
   - 一般情况下，生鲜产品要求顾客**签收后尽快开箱检查**（如24小时内）。若顾客延迟开箱或未及时反馈，可能被认定为未尽到注意义务，责任可能部分或全部由顾客承担（除非有明显发货时已坏的证据）。

5. **不可抗力或天气因素**：  
   - 高温、暴雨等极端天气导致水果加速腐坏，通常卖家应提前告知风险，并采取保护措施（如冰袋、冷链）。若已尽力仍无法避免，双方可协商分摊责任或由平台介入判定。

**建议您作为顾客的优先操作步骤：**
- **立即拍照/录像**：坏果、包装、快递单、整体状态，清晰反映问题。
- **联系卖家**：提供证据，说明情况，要求按赔付规则处理。多数电商平台支持“坏果包赔”。
- **若卖家拒绝**：可申请平台客服介入（如淘宝、京东、美团等），或向市场监督管理局投诉（12315）。

**总结责任归属表**（供快速判断）：

| 情况 | 责任方 |
|------|--------|
| 包装完好、发货时坏 | 卖家 |
| 包装破损、运输压坏 | 物流/卖家（卖家先负责，再追物流） |
| 正常天气、正常包装、顾客延迟开箱 | 顾客需承担部分或全部 |
| 极端天气且卖家已采取合理措施 | 双方协商或平台判定 |
