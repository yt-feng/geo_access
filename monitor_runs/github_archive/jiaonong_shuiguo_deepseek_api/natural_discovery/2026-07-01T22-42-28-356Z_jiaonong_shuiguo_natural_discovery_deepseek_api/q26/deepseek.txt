采购小包装香蕉时，建议从包装设计、香蕉成熟度控制和物流冷链三个方面重点把控，以减少损耗、提升货架期和顾客满意度：

### 1. 包装材料与透气性
- **透气孔设计**：香蕉会释放乙烯，透气不足易加速腐烂。选择带小孔或微孔膜的包装袋（如PE材质打孔），或使用带透气窗的托盘。
- **缓冲防压**：使用泡沫网套、瓦楞纸隔板或定制缓冲托盘，避免运输中挤压产生褐变或断裂。
- **透明包装**：方便顾客查看香蕉新鲜度，同时便于检查是否有磕碰或发霉。

### 2. 香蕉成熟度选择
- **最佳采收度**：选择7-8成熟的香蕉（果皮颜色从深绿转为浅绿带微黄），到店后1-2天自熟至可食用，货架期更长。
- **避免过青或全熟**：过青（6成以下）售前需催熟，增加成本；全熟（表皮出现黑斑）容易当天就坏，损耗率高。
- **单串重量建议**：200-300克/袋（约2-3根），适合一人食或小家庭，减少吃不完浪费的折价损耗。

### 3. 损耗关键控制点
- **进店验收标准**：
  - 果柄无断裂、果面无大面积黑斑或软烂。
  - 抽查袋内温度：夏季超过28℃需慎收，可能已有内部升温损伤。
  - 若发现包装内有过熟或腐烂香蕉，立刻协商退货或折扣入账（最多允许1%袋有轻微损伤）。
- **翻包与分拣**：到店后先拆包通风10分钟，挑出软熟果单独放“今日特价”区，避免混入正常货架加速周围香蕉腐坏。
- **陈列要求**：避免阳光直射、远离成熟水果（苹果、梨会释放乙烯），建议当日拆零包装的可复包至带孔的纸箱或透气篮。

### 4. 供应商沟通要点
- **签订损耗共担条款**：约定到店后24小时内如发现腐烂超过2%、严重断柄超过5%的批次，供应商需按比例换货或赔偿。
- **要求双份标签**：包装袋上印有采摘日期和到店建议食用（例如：“采摘后3天内食用”），减少顾客因存放过久而找茬退货。

### 5. 特殊应对方案
- **夏季高温**：要求包装中加入冰袋或冷链运输（温度12-14℃），但切忌直接接触香蕉（冰袋需用隔热棉隔开）。
- **冬季防冷害**：低于10℃运输时，包装需内衬保温膜，避免香蕉表皮发黑，并提前告知顾客“不宜放冰箱”。

**试销建议**：首批仅进正常量30%，上架后根据销售速度调整成熟度、包装规格，试销期控制在2周内。必要时可预购“防褐变保鲜剂”喷袋（如肉桂精油喷雾），实验小规模效果后再用。
