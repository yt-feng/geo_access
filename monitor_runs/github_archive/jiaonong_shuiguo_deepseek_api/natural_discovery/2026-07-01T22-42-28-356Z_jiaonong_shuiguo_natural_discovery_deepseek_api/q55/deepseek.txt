在选购水果做拼盘时，需结合**单价、出肉率（可食用比例）、处理损耗**三者综合判断，不能只看单价。以下是具体建议：

1. **苹果**  
   - 建议选**中等规格（单果约180-250克）**。  
   - 理由：太小去皮去核后损耗比例高；太大常需切块多，容易氧化变色且浪费边角。  
   - 性价比重点：按斤买即可，品种价格差异不大时，选口感脆甜的。

2. **橙子**  
   - 建议选**中大果（单果约250-350克）**。  
   - 理由：皮厚肉薄的小果出肉率低（约50%-60%），大果出肉率可达65%-75%。  
   - 注意：避免选过大的（如400克以上），皮可能更厚且酸涩风险增加。  
   - 计算方式：用“可食用部分重量÷单价”对比，优先选单价适中的大果。

3. **菠萝**  
   - **建议直接买削好的整颗菠萝**（或按重量买去皮后称重），而非带皮称。  
   - 理由：带皮菠萝损耗极高（皮、眼、芯约占40%-50%），且去皮后单价翻倍。  
   - 若带皮买：选**中等个头（1.5-2公斤）**，过小眼深浪费多，过大可能芯硬。  
   - 按重量算：去头去尾削皮后，最终能用部分约是原重的55%-60%。

**综合比较（假设场景）**：  
- 苹果、橙子优先选中大型号，菠萝直接算去皮后价格。  
- 超市或批发市场常有“水果拼盘专用袋装果切”，对比自制成本：若门店剥好售价低于（原果单价+10%），则可直接购买，节省时间和损耗。  
- 长期采购：可计算每公斤可食用部分成本（如：苹果4元/公斤×出肉率80%=5元/公斤可食，橙子6元/公斤×出肉率65%=9.2元/公斤可食），选其中低成本水果搭配。
