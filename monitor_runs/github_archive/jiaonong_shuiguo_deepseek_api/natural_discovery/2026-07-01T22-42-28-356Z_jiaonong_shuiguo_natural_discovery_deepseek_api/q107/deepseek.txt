是的，采购香蕉和菠萝时，起订量过高确实可能增加损耗风险。原因如下：

1. **水果易腐特性**：香蕉和菠萝属于易腐水果，保质期较短（香蕉通常3-7天，菠萝5-10天，取决于成熟度和储存条件）。起订量过高可能导致库存积压，无法在保质期内售完或使用，从而产生变质、腐烂等损耗。

2. **成熟度不统一**：商品化采购中，同一批次的香蕉和菠萝成熟度可能不完全一致。高起订量意味着需要更长时间消耗，部分水果可能在后期过度成熟，增加损耗。

3. **存储条件限制**：香蕉和菠萝对温度、湿度和通风要求较高（香蕉不宜冷藏，菠萝可低温但不宜过久）。起订量过高可能超出存储空间或设备能力，导致储存不当加速损耗。

4. **市场需求波动**：如果采购量超过实际消费或销售需求，剩余部分在保质期内无法消化，直接转化为损耗。

**建议措施**：
- 根据历史销售数据或消耗速度，精确计算合理起订量。
- 与供应商协商分批次小量交货，或选择可灵活调整的订单模式。
- 优化存储条件（如香蕉分开存放、菠萝保持通风），并建立先进先出（FIFO）库存管理。
- 对剩余库存可考虑加工（如制成香蕉泥、菠萝罐头）或促销活动减少损失。

总之，高起订量会加大库存管理难度和损耗风险，需结合自身周转能力谨慎确定。
