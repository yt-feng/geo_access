判断苹果和橙子是否适合门店销售，主要看**外观、重量和新鲜度**。以下是具体标准和操作建议：

### 1. 明确门店定位和目标客户
- **高端/精品店**：选**大果**（苹果直径≥75mm，橙子≥70mm），外观均匀、无瑕疵。
- **社区/平价店**：选**中果**（苹果直径65-75mm，橙子60-70mm），性价比优先。

### 2. 现场检查关键指标
| 指标        | 苹果                         | 橙子                         | 操作方法                     |
|-------------|------------------------------|------------------------------|------------------------------|
| **大小均匀** | 单果重量差异不超过20%        | 单果直径差异不超过10%        | 随机取10个，用卡尺或电子秤对比 |
| **外观无伤** | 无虫眼、碰伤、腐烂、锈斑     | 无干疤、霉变、软斑、油胞破裂 | 翻看果蒂、底部、果面         |
| **果柄完整** | 保留绿色果柄，断口不腐烂     | 保留果蒂，无干枯脱落         | 轻捏果柄，无松动感           |
| **硬度良好** | 按压不软、不凹陷，有弹性     | 按压有紧实感，不浮皮         | 拇指轻压果肉（避开果梗）     |
| **色泽鲜亮** | 底色淡黄或淡绿，红晕均匀    | 颜色橙黄或橙红，有光泽        | 自然光下观察，避免青绿果     |

### 3. 额外筛选技巧
- **声音法**：轻轻敲击橙子，**声音清脆**表示水分足；**声音沉闷**可能过熟或空心。
- **掂重量法**：同等大小下，**越重**水分越足，口感越好。
- **批次检查**：打开至少3箱（上、中、下层），避免底层混入小果或次果。

### 4. 快速淘汰标准
出现以下任一情况，直接退回或下架：
- 单个重量低于标准下限10%以上
- 果皮有超过1cm的伤口或腐烂点
- 果肉已软化、有异味
- 果梗处发霉或出水

### 5. 门店陈列建议
- **大小分级**：建议分为**大果（精品装）**和**中果（散称装）**，用不同价格带区分。
- **定期巡查**：每天挑出软果、坏果，避免影响周边商品。

这样既能保证顾客满意度，也能减少损耗。如果需要更具体的尺寸标准，可以结合你所在地区主流品种（如富士苹果、脐橙）的常规大小来调整。
