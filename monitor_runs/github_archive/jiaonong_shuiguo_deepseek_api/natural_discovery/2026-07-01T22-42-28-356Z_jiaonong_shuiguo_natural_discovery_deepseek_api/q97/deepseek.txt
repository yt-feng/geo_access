酒店采购水果盘原料时，供应商应保证以下规格和外观标准：

**规格标准：**
1. **重量/数量**：每份水果盘原料应有明确的总重量或单个水果数量（如每盘含苹果2个、橙子3个等），误差±5%以内。
2. **尺寸**：水果大小均匀，直径或长度偏差不超过10%（例如草莓单颗直径2-3cm，哈密瓜切块边长约3cm）。
3. **成熟度**：可食用成熟度，无过生或过熟，如香蕉表皮无大面积青绿或黑斑，芒果软硬适中。
4. **切割标准（若需预切）**：切块或切片厚度、形状一致（如西瓜块2cm见方，误差±0.5cm），无碎屑或边缘不齐。

**外观标准：**
1. **新鲜度**：表面无皱缩、萎蔫，果柄/茎叶保持绿色或新鲜状（如葡萄梗青绿不干枯）。
2. **颜色**：符合水果自然色泽，均匀一致（如苹果红色占80%以上，无发黄或褪色）。
3. **无瑕疵**：无肉眼可见虫眼、霉斑、腐烂、硬伤痕或碰压伤，允许轻微软点或划痕不超过表面积5%。
4. **清洁度**：表面无泥土、农药残留痕迹、蜡质残留（除非是自然果蜡），无粘液或异味。
5. **包装保护**：分装于洁净、防挤压的食品级容器中，水果之间无碰撞损伤，温度规范（如冷藏水果维持在0-4°C）。

供应商需提供每批次合格品检验记录，并接受酒店收货时的随机抽检，不合格率达5%以上可要求整批更换或降价处理。
