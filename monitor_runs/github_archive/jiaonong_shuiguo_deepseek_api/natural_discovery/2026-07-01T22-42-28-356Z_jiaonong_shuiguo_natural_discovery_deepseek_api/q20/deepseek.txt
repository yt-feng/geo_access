选择社区超市的水果供应商时，建议重点考察以下条件：

1. **配送稳定性**：确认是否能保证每日固定时间送货，能否应对节假日或突发天气的延迟。

2. **水果新鲜度与品种**：关注其采购渠道（如产地直采或一级批发市场），每日到货日期是否为当日/前一日采摘，是否提供应季和常青品种。

3. **库存与补货灵活性**：是否支持小批量、多频次补货，缺货时能否快速协调替代品。

4. **品质把控与退换政策**：是否提供开箱验货、次品现场调换或售后补偿（如打折、退款），有无明确的分级标准（如特级/一级果）。

5. **价格与账期**：报价是否包含配送费，是否与市场行情挂钩（如按日/周更新），账期是否灵活（如周结或月结）。

6. **冷链与包装**：运输车辆是否为冷藏车，包装是否防挤压（如使用气柱袋或分层泡沫箱），夏季有无冰袋措施。

7. **应急响应能力**：是否有备用配送方案（如合作其他车队或自提点），天气预报异常时是否主动调整配送时间。

建议先试订一周，重点监控**到货准时率、次品率、损耗率**三项指标，同时对比2-3家供应商，并索要食品经营许可证和产地证明。
