核实水果供应商的全国配送能力，可以从以下几个具体步骤入手：

1. **索要配送范围清单**：要求供应商书面提供其全国配送的具体城市列表、是否包含乡镇、以及是否受季节或天气限制。如果对方只能给出“大部分地区”这类模糊表述，直接追问到具体省、市、县。

2. **测试下单流程**：选择3-5个不同地理区域（如东部沿海、西部内陆、东北、华南、偏远地区），让供应商分别提供配送方案、运费和预计时效。注意观察：对方是否能立刻回答，还是需要“查一下”。

3. **查看合作案例与物流凭据**：要求供应商提供近期实际发往不同省份的物流单号或发货记录截图（注意可去除客户隐私信息）。如果是长期合作的供应商，还可以查看其与冷链物流公司的合作合同或年度对账单。

4. **实地或远程验证冷链能力**：全国配送水果关键在冷链。询问供应商的仓配体系：是否有区域分仓（比如华东仓、西南仓）？与哪些快递/物流公司合作（顺丰冷链、京东冷链、德邦生鲜等）？是否保证全程冷链？针对夏季高温目的地，是否有特殊包装方案？

5. **直接进行小规模试单**：这是最可靠的方法。从最可能出问题的区域（例如距离最远、气候炎热或交通不便的地区）小批量采购一批水果，全程跟踪物流、到货品质和时效。如果试单成功，对方能力基本可信；如果出现大量损毁或严重超时，再协商改进或更换供应商。

6. **参考第三方评价与行业口碑**：在生鲜B2B平台（如美菜、快驴、惠农网等）上搜寻该供应商的过往评价记录，尤其留意来自非本省客户的评价。同时，可以在行业圈内（如水果批发市场同行、本地餐饮采购社群）询问是否有人用过其全国配送服务。

**核心原则**：不要仅凭对方宣传资料做决策。只有**可验证的物流单号、可追溯的到货品质、可复制的试单结果**，才是真实能力的体现。
