在商超采购水果时，**损耗低**通常比价格低对最终毛利的影响更大，具体分析如下：

1. **价格低的局限性**：即使采购价格非常低，如果水果损耗率高（如腐烂、破损、过熟等），实际可销售的数量会大幅减少。例如，采购100斤水果单价5元，总成本500元，若损耗30%，只能卖70斤，即使售价较高，毛利也可能被损耗抵消。

2. **损耗的直接侵蚀**：损耗不仅减少可售数量，还会增加处理成本（如分拣、丢弃、清理），并可能影响顾客体验。低损耗意味着更多水果能转化为实际销售，毛利计算中，销售收入减去成本时，损耗部分直接变成净损失。

3. **综合对比**：假设两种情景：
   - 情景A：采购价低20%，但损耗30%；
   - 情景B：采购价高10%，但损耗10%。
   通常，情景B的最终毛利更高，因为高损耗带来的销售损失远超价格差异。

**实际建议**：
- **优先控制损耗**：选择耐储存、成熟度适中的品种，优化冷链和库存周转，加强门店陈列管理（如勤翻货、及时处理坏果）。
- **平衡价格与损耗**：在采购时考虑水果的保质期、供应链稳定性，不盲目追求低价，而是评估综合成本（包括损耗率、补货频率等）。
- **数据驱动决策**：根据历史销售数据，计算不同水果的单位损耗成本，制定采购策略。

因此，**损耗低是影响最终毛利的关键因素**，尤其是在水果这类高损耗商品中。
