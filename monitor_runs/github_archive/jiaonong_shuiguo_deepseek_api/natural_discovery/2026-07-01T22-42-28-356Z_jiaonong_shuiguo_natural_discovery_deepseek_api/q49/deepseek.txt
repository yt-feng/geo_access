针对茶饮店旺季菠萝用量激增的情况，与供应商临时补货需注意以下几点，确保供应稳定且成本可控：

1. **提前沟通，明确需求**：立即联系供应商，说明突发需求及具体增幅（如“本周需增加200公斤菠萝”），确认其库存和供应能力。
2. **签订书面补充协议**：明确临时补货的条款，包括：
   - **数量与时间**：额外采购量、分批次送货的时间表。
   - **价格**：是否执行原价、临时调价或批量折扣（旺季可能涨价，需提前锁定）。
   - **质量标准**：重申菠萝的成熟度、大小、无腐烂等要求，防止以次充好。
   - **违约责任**：如供应商无法按时交货，需明确赔偿方案（如折扣、违约金）。
3. **备用供应商清单**：若主要供应商无法满足全部需求，快速联系备选供应商，避免断货。
4. **协商灵活结算**：可提议适当预付部分款项或缩短账期，换取供应商优先供应。
5. **跟踪抽样检查**：每批次到货时，随机抽检菠萝品质，确保符合茶饮出杯标准（如甜度、酸度、色泽）。
6. **短期备货方案**：若供应商响应慢，可考虑少量从批发市场或大型商超紧急采购，但需控制成本。

**示例话术**：“李总，本周末菠萝销量突然翻倍，我们需要紧急追加200公斤，分两天送达。价格能否维持原合同？若您库存紧张，可否协调其他渠道？我们愿意预付30%定金。”

建议优先与长期合作供应商协商，以避免旺季被临时加价。如果对方无法保证质量，果断启动备选方案。
