通常供应商是愿意配合周末促销的，但具体支持力度取决于你和供应商的合作关系、采购量以及促销形式。

你可以从以下几个方面争取支持：

1. **价格支持**：如果你能承诺在周末促销期间增加进货量（比如比平时多30%~50%），大多数供应商会愿意给予**临时折扣**或**特价商品**。建议你明确告知促销档期和预期销量，让供应商看到利益。

2. **物料支持**：供应商经常会提供**促销物料**，例如：宣传海报、价签、堆头设计、试吃台、小赠品等。如果你店里有重点推荐的应季水果或新品，可以直接向供应商申请物料赞助。

3. **联合促销**：部分供应商（尤其大品牌或果园直供）会派出**促销员**协助，或提供**试吃品**来吸引客流。你可以提出“周末活动期间主推他们的产品”，换取他们的支持。

4. **谈判技巧**：你可以说：“周末人流比平时多20%，我想做你们的爆款水果专场，但利润很薄，如果你们能给我5%的价格支持和一套促销物料，我可以把这个产品摆在门口最显眼的位置。” 供应商通常愿意为**曝光和销量**投入。

建议你提前1~2周联系供应商，明确告知你的促销方案（如“本周末9:00-18:00做特价水果专区”），这样他们有足够时间申请内部支持。如果供应商表示“没有预算”，可以主动提出“分摊成本”——比如他们出物料你出场地，或者他们打折你让利。
