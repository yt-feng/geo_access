针对连锁商超长期水果供应商的招标，评分标准应兼顾**供应稳定性、品质一致性、价格竞争力**以及**合作潜力**。建议采用综合评分法，总分100分，按权重分配如下：

### 1. 资质与基础实力（15分）
-   **企业资质**（5分）：营业执照、食品经营许可证、水果类相关认证（如GAP、有机认证等）。提供海关进口检疫证明（如涉及进口水果）。
-   **经营规模**（5分）：近三年年均营业额、仓储面积和冷链物流车辆数量。需提供审计报告或税单。
-   **行业经验**（5分）：服务过的连锁商超、餐饮连锁或大型客户数量及合同复印件（隐去价格）。

### 2. 产品与品质（30分）
-   **品类覆盖与供应链**（10分）：能稳定供应苹果、柑橘、香蕉等基础品类（5分），以及应季高端水果（如车厘子、榴莲）的直采或一手货源能力（5分）。
-   **品质标准**（10分）：需提供详细品控手册（如单果重量、甜度、成熟度、无腐烂率等量化指标）。现场提供各品类样品，由评审小组盲测打分（外观、口感、新鲜度）。
-   **产地优势**（10分）：自有或长期合作的规模化种植基地、核心产区直采占比（如新疆阿克苏苹果、海南芒果）。提供基地实地照片、合作协议或采摘记录。

### 3. 配送与售后服务（25分）
-   **配送能力**（15分）：
    -   覆盖门店数量与频率：是否支持单品每3天至少配送一次（或按订单）。
    -   冷链物流：运输车辆温控能力（0-4℃冷藏/ -18℃冷冻）、到货温度记录可追溯（5分）。
    -   到货时间：规定时间内（如下单后24小时内）送达率的承诺与历史数据（5分）。
    -   损耗率控制：到货时外观完好率≥98%，次品率低于约定比例（5分）。
-   **退换货与应急**（10分）：
    -   退换货政策：变质、压伤、规格不符的迅速处理机制（如48小时内退换）（5分）。
    -   应急方案：节假日、极端天气下的保供预案，以及备用供应商名单（5分）。

### 4. 价格与商务条款（20分）
-   **报价合理性**（10分）：对基础品类（如红富士苹果、香蕉）进行三年固定价或浮动公式（如“大宗批发价 + 固定加价率”）的比较。避免最低价中标；可参考市场均价，偏离超过15%者酌情扣分。
-   **付款与账期**（5分）：接受6个月或以上的账期（得分较高），或提供灵活的付款方案。
-   **合作专业性**（5分）：提供长期合作框架（如可签订3年合同）、季度采购预估、质量保证金比例（建议不超过货款的5%）。

### 5. 履约能力与合同响应（10分）
-   **信用记录**（5分）：通过企业征信系统查询，无重大合同违约、诉讼记录。
-   **合同条款响应**（5分）：对供应短缺时的违约金、断货替补机制、商业机密保护条款等无争议性修改，或提出合理补充方案。

---

### 评审流程建议
1.  **预审**：先排除资质不全、信誉不良的企业。
2.  **盲测**：对样品进行编号，由采购、品控、门店代表匿名评分。
3.  **实地考察**：对高分供应商进行仓储、基地、冷链系统的实地抽查。
4.  **模拟配送**：要求供应商在指定时间向3家门店配送指定水果，考核实际效率。

### 注意事项
-   **避免低价陷阱**：建议设定“价格得分”不超过总分的20%，避免供应商用低品质清库存。
-   **设置阈值项**：如“到货损耗率超过5%则一票否决”。
-   **动态评估**：中标后按季度考核配送准时率、退货率、客户投诉，作为续约依据。
