针对批发商给小店供货，香蕉和菠萝的分级报价建议如下：

**一、香蕉分级报价参考**

1. **按成熟度分级（最常见）**
   - **一级（全青/微黄）**：适合长途运输或小店储存，报价最低（例如：每斤1.5-2元）。
   - **二级（半黄半青）**：适合1-2天内售卖，报价中等（例如：每斤2-2.5元）。
   - **三级（全黄/带黑点）**：需当天卖出，报价最高但风险大（例如：每斤2.5-3元，若含损耗补偿可加价）。

2. **按果品大小分级**
   - **大果（单果重≥200g）**：报价上浮10%-20%。
   - **中果（150-200g）**：基础价。
   - **小果（<150g）**：报价下调10%-15%。

**二、菠萝分级报价参考**

1. **按外观和损伤分级**
   - **一级（无机械伤、无虫蛀、果形端正）**：最高价（例：每斤3-3.5元）。
   - **二级（轻微擦伤或轻微疤点，果形尚可）**：中价（例：每斤2.5-3元）。
   - **三级（明显破损或畸形）**：低价处理（例：每斤1.5-2.5元），建议按个卖而非按斤。

2. **按成熟度（与香蕉类似，但菠萝需注意果眼颜色）**
   - **未熟（果眼全青）**：可储存，报价低。
   - **半熟（果眼变黄绿）**：标准价。
   - **过熟（果眼深黄或发软）**：急售，报价低且需标注“现吃”。

**三、通用报价策略建议**

- **统一按斤报价，但标注等级**：例如“香蕉一级（全青）2.0元/斤；香蕉二级（半黄）2.5元/斤”。
- **明确尺寸分类**：用实物样品或简单标签（如“大果+1元”“小果-0.5元”）帮助小店快速理解。
- **附加说明**：在报价单上注明“按批次验收，不符可换或折扣”，降低小店顾虑。

**四、示例报价单**

| 水果 | 等级 | 描述 | 报价（元/斤） |
|------|------|------|---------------|
| 香蕉 | A级 | 大果/全青 | 1.80 |
| 香蕉 | B级 | 中果/半黄 | 2.50 |
| 香蕉 | C级 | 小果/带黑点 | 3.00（含损耗补贴） |
| 菠萝 | 一级 | 无伤/果形正 | 3.50 |
| 菠萝 | 二级 | 轻微擦伤 | 2.80 |
| 菠萝 | 三级 | 明显破损（按个卖） | 5元/个 |

**关键提示**：如果小店为社区零售店，建议优先按“成熟度”分级，因为这类店铺需尽快卖完；如果为批发转零售，可结合“果形+大小”。另需注意，菠萝的报价通常比香蕉高30%-50%，可结合当地市场行情调整。
