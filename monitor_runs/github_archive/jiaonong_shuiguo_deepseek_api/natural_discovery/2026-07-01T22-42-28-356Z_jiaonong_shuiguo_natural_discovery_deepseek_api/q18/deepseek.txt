挑选香蕉时，可以通过以下方法避免买到太生或太熟：

1. **看颜色**：
   - 挑选表皮呈**均匀黄色**、略带青头（但不多）的香蕉，这种通常是七八成熟，买回家放1-2天即可食用。
   - 完全青绿的太生，需要较长时间催熟；出现大量黑斑或变黑、变软的太熟，可能内部已变质。

2. **摸硬度**：
   - 用手轻轻按压香蕉柄和果身，**稍微有弹性、不软烂**的为佳。
   - 过硬（像石头）说明生，按压后凹陷不回弹或果肉软塌说明过熟。

3. **观察果柄和棱角**：
   - 果柄干燥、颜色正常（非发黑或发霉），且香蕉棱角**较圆润**（无尖锐棱角），说明成熟度适中。
   - 棱角明显尖锐的通常较生；棱角消失、果身软塌的过熟。

4. **根据食用计划选择**：
   - **当天或次日吃**：选表皮金黄、轻微黑点（芝麻点）但果肉仍硬的，口感最佳。
   - **存放几天再吃**：选带青皮、果柄新鲜的，回家置阴凉处自然成熟。

5. **避开异常特征**：
   - 避免购买果皮有**开裂、软塌、霉斑**或散发**酒精味**的香蕉，可能已过熟或腐烂。

**小贴士**：超市整把香蕉中，中间位置的往往比两端成熟度更均匀。如果买回后发现略生，可和苹果或梨一起装入纸袋，苹果释放的乙烯能加速催熟。
