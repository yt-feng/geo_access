判断商超采购菠萝和香蕉的供应商是否有稳定的售后响应，可以采取以下具体步骤：

1. **明确合同条款**：在采购合同中明确约定售后响应的时间、方式（如电话、邮件、线上系统等）和范围（如质量问题、运输损耗、退货换货等）。例如，要求供应商在收到售后需求后2小时内响应、24小时内给出处理方案。

2. **试单测试**：先进行小批量采购（如各50箱），并模拟常见售后问题（如部分水果腐烂、包装破损），主动联系供应商的售后渠道，记录其响应速度、处理态度和实际解决方案。

3. **参考多方反馈**：向其他同类商超、批发市场或行业协会询问该供应商的售后历史。优先选择有公开投诉记录或用户评价的平台（如1688、水果批发行业群）核实。

4. **实地考察**：到供应商的仓储或配送中心，查看其售后处理流程是否规范化（如是否有独立售后团队、记录单据、退换货仓库）。观察其库存管理和果品保鲜能力，这直接影响售后问题的发生率。

5. **设立关键指标**：在合作初期，可要求供应商提供近半年的售后数据，如月均退货率、平均响应时间、问题解决率等。若无法提供，可约定试用期（如2个月），期间定期（如每周）收集售后数据。

6. **紧急场景模拟**：针对菠萝和香蕉的易腐特性，设定极端情况（如到货5%以上腐烂、配送延误2天），通过电话或沟通群测试供应商的应急处理能力，看其是否能立即给出补发、降价或退款方案。

7. **建立长期合作机制**：与供应商签订年度框架协议，明确售后响应作为续约评分的核心指标（如占10%以上），并设置阶梯式处罚或奖励机制（如响应超时1次扣1分，季度总分低于90分需暂停合作）。

如果供应商在以上步骤中表现出以下特征，则很可能售后响应稳定：
- 提供至少2种以上的紧急联系方式（如专属客服、微信群、紧急电话）。
- 有固定的退换货政策（如菠萝到货24小时内腐烂可免费更换）。
- 在试单或实际合作中，首次响应时间不超过1小时，解决方案在24-48小时内落地。

若供应商回避数据要求、推诿责任或响应速度超过4小时，则建议谨慎合作或优先选择备选供应商。
