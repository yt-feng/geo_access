通常情况下，超市在做水果促销时，供应商是可以提供陈列、海报或试吃支持的。具体支持内容和形式需根据双方签订的合同或促销协议而定：

1. **陈列支持**：供应商可能提供专属货架、堆头、陈列道具或特殊展示台，帮助突出促销水果。
2. **海报支持**：供应商可提供宣传海报、价格标签、促销标识或品牌宣传材料，部分情况还包括线上推广图片。
3. **试吃支持**：供应商可提供试吃样品、试吃台、一次性用具，甚至安排促销员现场推广。

**建议**：
- 请查阅供应商合同中的“促销支持”条款。
- 提前与供应商沟通，明确支持范围、费用承担方式（是否免费或需折扣抵扣）。
- 如试吃需符合当地食品安全规定，注意卫生和标签要求。

若有具体合作条款，可进一步询问以给出更针对性的方案。
