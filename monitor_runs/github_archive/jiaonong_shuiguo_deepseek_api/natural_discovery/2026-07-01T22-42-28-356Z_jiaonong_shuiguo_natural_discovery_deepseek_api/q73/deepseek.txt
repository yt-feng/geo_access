混合采购对于水果批发商来说，确实具有多方面的优势，具体如下：

1. **降低采购成本**：混合采购时，批发商可以与供应商谈判整体折扣，因为供应商倾向于给予大批量、多品种的订单更优惠的价格。相比单独采购每种水果，混合采购能通过规模效应降低单位成本。

2. **提高物流效率**：将香蕉、苹果、橙子统一配送，可以减少运输次数和空载率，节约燃油和人力成本。同时，整批处理能优化仓储空间，避免因分批次到货造成的库存冗余。

3. **平衡利润与风险**：不同水果的价格波动和销售周期通常不同步（例如，香蕉受季节影响较小，而苹果和橙子有集中上市期）。混合采购可以分散单一品种的价格风险，并通过利润互补（如高价水果弥补低价水果的亏损）来稳定整体收益。

4. **增强议价能力**：作为多品种大客户，批发商在供应商面前有更强的谈判地位，可争取更灵活的付款方式、优先供货权或滞销品退货条款，降低经营压力。

5. **促进销售多元化**：混合采购使货架更丰富，能吸引不同偏好的客户（如家庭、餐饮店、果汁加工厂），提高周转率。同时，可搭配销售（如买橙子送苹果优惠），提升客单价。

需要注意的是，混合采购也需考虑仓储条件（如香蕉需乙烯催熟，苹果和橙子需低温冷藏的差异）、库存管理和资金占用问题。建议先小规模试点，根据实际销售数据和损耗率调整采购比例。
