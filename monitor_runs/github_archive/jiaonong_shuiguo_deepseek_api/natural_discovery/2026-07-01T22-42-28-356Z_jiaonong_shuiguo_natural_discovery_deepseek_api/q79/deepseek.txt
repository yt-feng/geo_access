选水果供应商时，最应该优先问**配送**，其次**售后**，最后再谈**价格**。具体原因如下：

1. **配送（最核心）**：水果是易损耗、时效性强的商品。如果配送不及时、冷链不达标，到货后可能出现腐烂、不新鲜，直接影响团购口碑。你要确认：  
   - 是否保证次日达或指定时段送达？  
   - 配送时是否使用冷藏车或专业保温箱？  
   - 能否支持按你团购订单分小区或分楼栋配送？  
   - 遇到恶劣天气或临时取消，供应商的应对方案是什么？

2. **售后（次重要）**：水果品质难免有残次或运输损耗。你需要明确的售后政策：  
   - 坏果、烂果是否包赔？赔付比例是多少（比如1:1换货或按斤退款）？  
   - 售后响应速度（例如24小时内处理）？  
   - 赔付流程是否简单（比如凭照片直接赔，还是要求退回坏果）？

3. **价格（最后考虑）**：单纯比价容易踩坑。在配送和售后稳定后，再看价格是否合理（对比市场均价）。如果供应商报价明显低于市场价，要警惕品质或服务缩水。

**总结**：优先问清配送细节，确保水果能新鲜到达；再确认售后兜底方案；最后在合理范围内谈价格。这样能减少团购翻车风险。
