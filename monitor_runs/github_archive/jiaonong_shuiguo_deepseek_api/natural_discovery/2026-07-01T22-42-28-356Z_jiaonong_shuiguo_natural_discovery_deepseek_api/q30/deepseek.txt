如果超市水果到货后品相不好，售后流程应明确约定以下关键环节，以保障双方权益并高效处理问题：

**1. 验收与反馈时限**
- **到货后检查时限**：约定收货方（超市）在水果到货后**30分钟至1小时内**完成开箱验货，超过时限可能视为默认品相合格。
- **反馈问题渠道**：指定唯一联系方式（如微信/电话/邮件），要求收货方在时限内拍摄清晰照片/视频，并附上到货单号、问题描述。

**2. 品相问题的判定标准**
- **明确分级**：列出可接受与不可接受的品相问题，例如：
  - 轻微磕碰（不影响果肉）：可不退货但协商折扣（如按原价80%结算）。
  - 腐烂、霉变、压坏严重：直接拒收或按原价全额退款。
  - 未成熟或过度软烂：根据数量折算（如1箱中5个坏果，按坏果比例退款）。
- **争议解决**：若双方对品相认定有分歧，可约定由第三方质检（如当地批发市场专业人员）判定，费用由过错方承担。

**3. 处理方式与时间**
- **退货/退款**：确认问题后，收货方需在**2小时内**将问题水果密封保存并联系退还；发货方在收到退货后**24小时内**完成退款（原路返回或转账）。
- **补发/换货**：若需换货，发货方应在**48小时内**安排新货发出，并承担所有运费；补发数量需与退货数量一致。
- **折扣结算**：如需协商折价，双方应**当天内**确认折扣比例并更新账单，超时视为自动接受原价。

**4. 责任与异常处理**
- **运输责任**：明确谁承担运输风险（通常发货方负责直到签收）。若品相问题由快递暴力运输导致，收货方需保留原包装及快递单据，协助发货方向物流索赔。
- **恶意拒收**：若收货方无正当理由反复提出品相问题，发货方可保留证据暂停合作，并需提供此前其他批次正常签收记录作为对比。

**5. 书面记录**
- 所有沟通过程（照片、聊天记录、视频）需保存至少30天，作为售后凭证。建议使用微信文件传输助手或专用群聊，避免使用语音（口语化）记录。

**建议流程模板：**
1. 收货时拍照开箱，30分钟内通过【指定渠道】发送照片/视频。
2. 发货方2小时内确认是否属于品相问题。
3. 确认后，收货方选择：退货退款 / 补发 / 折扣。
4. 选择退货的，当日内退回；发货方收到后24小时内退款。
5. 选择补发的，48小时内发新货；选择折扣的，当天内按比例结算。

总之，核心是**时限明确、标准清晰、证据留存**，避免模糊措辞（如“尽快”“酌情处理”）。
