茶饮店要控制食品安全风险，水果供应商通常需要提供以下关键证明文件：

1. **营业执照**：供应商的合法经营资质证明，确保其具备合法主体资格。
2. **食品经营许可证**：若供应商销售的是预包装水果或需特定存储条件的水果，需提供此证。
3. **水果检验报告**：包括农药残留、重金属、微生物等检测项目，证明水果符合国家标准（如GB 2763、GB 2762等）。最好提供近期（如3个月内）的第三方检测报告。
4. **进货凭证**：如销售单、发票或收据，显示水果来源、批次、数量等信息，便于追溯。
5. **产地证明**：如水果来自特定产区（如有机认证地），需提供产地证明或相关认证文件（如绿色食品认证）。
6. **冷藏/冷链运输证明**：若水果需冷链运输（如草莓、蓝莓等），需提供运输温度记录或冷链资质证明。
7. **其他特殊证明**：
   - 进口水果：需提供入境货物检验检疫证明及海关报关单。
   - 有机水果：需提供有机产品认证证书。
   - 无公害或绿色食品：需提供相应认证证书。

**建议**：与供应商签订合同时，明确要求提供并定期更新上述文件（如每季度或每批次），并定期抽查验证其真实性。同时，建立供应商评估和淘汰机制，优先选择资质齐全、信誉良好的供应商。
