从损耗角度来看，**一般情况下，直接采购成品果丁的损耗更低**。具体分析如下：

1. **鲜果切配的损耗**  
   - **去皮去芯损耗**：菠萝去皮、去硬芯后，通常损耗率在**30%-40%**（取决于菠萝品质和切配技巧）。  
   - **储存损耗**：切好的果丁若未及时使用或冷藏不当，容易氧化变色、出水，进一步增加损耗。  
   - **时间与人力成本**：切配需要人工和时间，操作不当也可能造成额外浪费。

2. **直接买成品果丁的损耗**  
   - **几乎无切配损耗**：工厂已处理掉果皮、果芯，你购得的果丁重量即为可用的净重。  
   - **保存更稳定**：成品果丁通常采用真空包装或冷冻保鲜，氧化和水分流失较少，储存损耗低。  
   - **但需注意包装余量**：部分供应商可能会在包装中添加糖水或防腐液，实际果丁净重可能低于标称重量，需确认产品规格。

**综合建议**：  
- 如果**销量大且切配成本高**（如连锁甜品店），选成品果丁更划算，可节省人工和减少即时的损耗。  
- 如果**对新鲜度要求极高**，且能保证快速消耗（如小众精品店），鲜果切配虽损耗大，但口感更好。  
- **折中方案**：部分菠萝品种（如“香水菠萝”）皮薄、芯软，切配损耗可降至25%左右，适合自切。  

**最终判断**：从纯损耗率（净重/原料重）出发，成品果丁通常优于鲜果切配。但需结合成本、口味和供应商信誉做综合决策。
