在进香蕉时，建议按以下优先级权衡，核心是**降低损耗、控制周转**：

1. **首要看重：熟度（成熟度）**  
   - 香蕉是后熟型水果，到店后需要自然催熟或人为控制熟度。  
   - **进货建议**：选择**青硬（7-8成熟）**的香蕉，给自己预留3-5天的货架期。太熟（表皮发黄、有芝麻点）可能当天卖不掉就亏损；太青（5成以下）可能无法自然成熟或口感差。  
   - 实操小技巧：同批次香蕉，挑颜色均匀、手感坚实的；避免单根过黄或发黑的。

2. **其次看重：坏果率（即“净果率”）**  
   - 成本直接损失，且坏果会加速周围香蕉腐烂。  
   - **标准**：整件（通常18-20斤）中坏果（黑斑、发软、断裂超过1/3）控制在**2根以内**，超过4根就得压价或拒收。  
   - 散装进：逐把检查柄部有无霉变，捏一下两端是否过硬（可能冻伤）。

3. **再次看重：价格**  
   - 香蕉是大众低价水果，利润微薄（毛利通常15%-25%）。  
   - 但**不要贪便宜而忽视前两条**：特别低价的香蕉往往是产区尾货或冻伤果，坏果率可能翻倍，综合成本反而更高。  
   - 价格参考：比批发均价略低5%-10%是合理空间。

4. **最后看重：送货速度**  
   - 香蕉不是急需的应急商品，除非你处于**气温超30℃或空调损耗严重**的夏季。  
   - 重点要求：**当天订货最晚次日中午前到**，避免香蕉在仓库多放24小时因温差变坏。  
   - 速度优先级：本地供应商（半天内）> 就近批发市场（半天到1天）> 长途产地直发（1-2天）。

**最终建议**：
- **中小型水果店**：优先找本地批发商（熟度可控→少量多次进货），宁贵5-10元/箱，也要确保熟度差和低坏果率。  
- **如果不确定供应商**：第一二次进货时，尽量按“熟度70%青硬度+挑干净果”的标准少量试单，成交后再谈价格和送货频率。
