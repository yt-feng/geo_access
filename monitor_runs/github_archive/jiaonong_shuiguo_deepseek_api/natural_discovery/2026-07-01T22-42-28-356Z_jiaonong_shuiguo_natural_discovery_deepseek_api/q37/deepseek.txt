商超会员日水果活动通常客流较大、促销力度强，供应商需提前做好以下配合准备以确保活动顺利进行：

1. **提前确认库存与备货计划**  
   - 根据商超提供的预估销量，提前1-2周锁定充足货源，避免断货。  
   - 备货量建议按历史数据上浮20%-30%（尤其是爆品如车厘子、西瓜、蓝莓等）。  
   - 确认水果成熟度（避免过熟或生硬），预留部分预熟产品。

2. **物流与到货节奏配合**  
   - 活动前1-2天集中到货（避免过早导致损耗、过晚影响陈列）。  
   - 安排专车或冷链配送（尤其高温季节），确保到货时新鲜度和外观完好。  
   - 预留紧急补货通道（如会员日当天临时加单，需能限时送达）。

3. **商品包装与分级准备**  
   - 根据商超要求提供适合会员日销售规格（如小包装、引流款、礼盒装）。  
   - 提前完成分级筛选（如按大小、甜度、瑕疵率分级，避免现场返工）。  
   - 按要求准备促销贴纸、价签或定制包装（如“会员专享价”标识）。

4. **价格与促销政策支持**  
   - 确认供货底价，配合商超会员日价格策略（如超低折扣、买送、换购）。  
   - 提供部分特价引流品或赠品（如小份试吃装、搭配赠果）。  
   - 明确退换货规则（如活动后剩余库存如何处理：退货、折价、寄售等）。

5. **现场人员与培训支持**  
   - 安排驻场人员负责陈列、补货、试吃推广（可提前1天进店培训）。  
   - 准备话术：如推荐搭配（苹果+橙子礼盒）、功效介绍（如草莓补维C）。  
   - 提供切果工具、试吃盘、湿纸巾等物料。

6. **品质控制与应急处理**  
   - 提前派驻品控人员在收货验收、开箱、上架等多个节点监控品质。  
   - 准备备用批次的应急替换方案（如部分水果损坏严重需临时更换）。  
   - 制定损耗分担机制（明确活动后剩余或报损水果是否由供应商承担）。

7. **活动后反馈与结算配合**  
   - 活动结束后3天内提供销售数据、损耗报表、顾客反馈。  
   - 配合商超完成核对、对账和正常结算流程。

**一句话总结**：核心是“量要足、货要鲜、价要优、人要到、赔要明”。提前确认每一环节的职责和节点，避免临时抱佛脚。
