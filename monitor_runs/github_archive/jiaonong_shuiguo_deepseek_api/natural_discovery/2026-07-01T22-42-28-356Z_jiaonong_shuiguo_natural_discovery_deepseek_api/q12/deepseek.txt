采购菠萝时判断熟度可以从以下几个关键点入手，既能保证口感，又能减少因过熟或未熟导致的损耗：

1. **看颜色**：选择表皮**黄中带绿**或**2/3以上变黄**的菠萝。完全青绿色的通常未熟，口感酸涩；全黄或带黑斑的已过熟，易腐烂。

2. **闻气味**：凑近菠萝底部（果蒂处），有**淡淡清香**为最佳。无味说明未熟，浓烈甜味或酒味代表过熟（需尽快出售）。

3. **摸硬度**：用手指轻压菠萝外壳，**微微弹性、不软塌**为宜。过硬说明未熟；软塌或有凹陷处则内部可能已坏。

4. **看果眼**：果眼（表面的螺旋状凸起）**大而饱满、分布均匀**，且底部果眼呈**浅褐色**的通常成熟度较好；过小的果眼可能未熟。

5. **听声音**：用手指弹菠萝中部，**声音清脆（类似敲木头）**为成熟适中；声音闷实可能过熟；声音太响可能水分不足。

**采购建议**：优先选择**约7-8成熟**的菠萝（颜色黄中带绿，无明显黑斑）。这样的菠萝在运输和储存期间可自然熟透，减少损耗。到店后，按成熟度分类：全黄的当天或隔天促销；青绿色的存放1-2天再出售。
