食堂采购水果时，**三者都重要，但食品安全是最高优先级的底线**。以下是具体建议：

1. **食品安全（必须首位）**  
   - 选择有资质、能提供水果产地证明、检疫报告或合格证的供应商，避免采购腐烂、农药残留超标或来源不明的水果。
   - 优先采购当季、新鲜、外观完好的水果，并现场验收（查看颜色、硬度、有无霉变）。

2. **价格（可量化比较）**  
   - 在食品安全达标的前提下，通过询价、比价控制成本，但不要为了低价接受劣质货源。
   - 可签订长期合同或批量采购，争取折扣；定期对比市场均价。

3. **配送准时（影响运营）**  
   - 明确要求供应商按指定时间送达（如开餐前1小时到货），避免影响食堂正常供应。
   - 可通过合同约定延迟配送的违约金，或预留应急补货渠道。

**综合建议**：  
先列出3-5家符合食品安全要求的供应商，对比报价和配送服务；首次合作可少量试单，评估品质和响应速度；后续根据记录（如损坏率、投诉次数）动态调整供应商。
