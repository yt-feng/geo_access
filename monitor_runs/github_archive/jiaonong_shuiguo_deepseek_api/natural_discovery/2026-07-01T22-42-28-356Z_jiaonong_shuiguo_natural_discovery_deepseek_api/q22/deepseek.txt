针对超市水果区提升客单价的促销需求，以下是针对香蕉、菠萝、苹果三款水果的组合促销方案，兼顾购买冲动、健康搭配和单价提升：

---

**1. 根据水果特性制定“家常健康组合”**
- **组合建议：** 香蕉（3根）+ 苹果（3个）+ 菠萝（半个或切块装）
- **单价提升逻辑：** 香蕉单价低，苹果适中，菠萝单价较高。打包后整体客单价会明显提升（例如香蕉6元+苹果9元+菠萝15元，组合定价28元，让利2元，但客单价从单买香蕉的6元跃升至28元）。
- **话术引导：** 可搭配“一周水果计划”牌子，强调三样水果搭配营养均衡（钾/维生素C/膳食纤维）。

**2. 打造“热带风情+日常”创意套餐**
- **组合建议：** 菠萝1个 + 苹果2个 + 香蕉2根，赠一个小袋装沙拉酱或酸奶
- **单价吸引点：** 菠萝通常最大，定价权高。配赠低成本酸奶或沙拉汁（成本0.3-0.5元），会让顾客感觉“多花一点，获得额外用途”，实际客单价可从菠萝的10元提升至组合的35元以上。
- **陈列：** 将菠萝切开一半展示果肉，旁边放切好的苹果块、香蕉片，现场试吃，激发“买回去做水果拼盘”欲望。

**3. 梯度定价法——以菠萝为“高价锚点”**
- **组合1（入门提升）：** 香蕉+苹果 两样组合价（如原价15元，组合价13元），客单价从5元（单买香蕉）提到13元。
- **组合2（核心提升）：** 香蕉+苹果+菠萝 三样组合价（原价30元，组合价26元），比两样组合多出13元——但顾客会对比发现“加13元就能获得菠萝”，更容易接受。
- **效果：** 顾客更可能选择三样组合，客单价直达26元。

**4. 场景捆绑销售：切块即食“水果碗”**
- 如果超市有现场切水果能力：将香蕉、菠萝、苹果切块，装塑料碗+叉，定价12-15元/碗（利润高、单价低？不对，目的是提升客单价，可让价格比单买稍贵——但更便利。或用这样方式引导顾客买整个水果自行回家切）。
- 实际可推：**周末特惠碗**：装满切好的水果（菠萝占体积），配一小盒酸奶，定价9.9元，吸引一个人购买，再引导附近的整果：选这个9.9还是选一整份自己切（多花20元更多量）？用对比促使更多人选购整果组合。

**5. 与奶制品联动：香蕉苹果菠萝+牛奶/酸奶**
- 组合包装成“全家营养早餐包”：香蕉3根+苹果2个+半个菠萝（去硬心切好）+ 1升牛奶或大盒酸奶。定价比分开买便宜10%，如原价42元，组合价37元。客单价从9元（单卖香蕉）升至37元。
- 陈列位置：水果区相邻奶制品区，立牌上写“早餐不重样，这样搭更划算”。

**执行要点：**
- **锁定时间：** 周五下午到周一上午，惯常家庭采购高峰。
- **醒目标签：** 用大红底白字标签写“组合价省X元”，并单独写“如果分开买，要多花X元”。
- **试吃：** 在组合陈列点旁边放一碟切好的香蕉苹果菠萝块，浇些酸奶，立刻提升成交率。
- **库存管理：** 菠萝切好半卖，记得当日处理掉，香蕉和苹果可多备货2-3天。如果菠萝销售慢，可转为切片单卖。

---

以上方案均围绕“顾客感知利益”（省事、省心、划算）设计，实际提价幅度控制在10-20%，组合后客单价预计可从单买香蕉的6-8元，提升至25-40元。
