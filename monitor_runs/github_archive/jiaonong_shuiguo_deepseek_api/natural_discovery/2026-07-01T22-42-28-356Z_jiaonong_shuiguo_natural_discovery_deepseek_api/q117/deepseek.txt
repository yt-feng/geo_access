建议采购方试卖时控制**首次进货量在日均预估销量的3-7天**范围内，这样既能测试市场反应，又不会积压过多库存。具体操作如下：

1. **初步预测日均销量**  
   参考同类水果（如相似品种、价格区间）的日常销量，设定一个保守的日销预估值。例如，如果类似商品日销50斤，新品首次试卖日销预估可设为30-40斤。

2. **试卖周期与进货量**  
   - **快消类新品（如草莓、车厘子）**：建议3天试卖量，约日均预估销量×3。这类水果损耗较高，快速测试反馈。  
   - **耐储存类新品（如苹果、柚子）**：可延长至7天，约日均预估销量×7，但需观察前5天的复购率。  

3. **风险控制**  
   - 首次试卖量不要超过总预算的10%-15%。如果亏损，便于止损。  
   - 注意保持**SKU（单品品种）**与现有商品比例平衡，避免新品挤压畅销品库存。  

4. **动态调整**  
   - 试卖结束后，若复购率高于30%，可逐步加量至2-3倍；若低于10%，立即暂停或更换供应商。  
   - 记录用户反馈（口味、大小、价格接受度），作为后续订货依据。  

如果条件允许，也可以采用**“小包装+样品试吃”**模式，先少量铺货并根据消费者直接反馈调整。
