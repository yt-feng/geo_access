针对茶饮店水果到货不合格的情况，建议在合同中或后续协商时明确以下几点补货约定，以减少对营业的影响：

1. **明确不合格标准**：提前约定水果的规格（如大小、成熟度、颜色、有无损伤）和验收时限（如到货后2小时内验收）。一旦发现不合格，立即拍照留证并通知供应商。

2. **补货时限**：约定供应商在接到通知后**4-8小时内**完成补货，或由店方先紧急调货（比如从附近超市或批发市场临时采购），供应商承担差价。

3. **补货数量与费用**：要求供应商补足合格品，所有补货运费、加急费由供应商承担；若因补货不及时导致店面缺料停售，供应商需按停售损失（如当日该饮品预估销售额）或合同罚则赔偿。

4. **备选应急方案**：在合同中加入“如超过约定时间无法补货，店方有权从第三方采购，费用由供应商承担”，并保留追索权利。

5. **动态库存预警**：日常保持核心水果品类有1-2天安全库存（比如多用冷冻果、浓缩汁等半成品替代），与供应商建立“高优补货通道”，要求对方承诺优先满足你的订单。

最后，所有沟通和约定尽量落实到书面（合同或聊天记录截屏），避免纠纷时无依据。保持与供应商的定期沟通，例如每周复盘到货质量，逐步筛选稳定供应商。
