选择学校食堂的水果供应商时，建议优先考虑以下维度：

1. **资质与安全**：供应商需具备食品经营许可证、农产品合格证明，并可提供每批次的农残检测报告。
2. **供应稳定**：选择能长期稳定供货（尤其苹果、橙子这类耐储水果）、且能处理季节性断档（如香蕉成熟周期短）的供应商。
3. **品质把控**：要求供应商有严格的分拣标准，例如香蕉无严重压伤、苹果蜡质正常、橙子无霉变；建议实地考察仓库、运输冷链条件。
4. **成本透明**：要求报价单明确品种、规格、单价，并与市场批发价对比；避免仅看低价，需考虑损耗率（香蕉易损耗）。
5. **售后保障**：明确退换货条款，例如坏果率超过X%可退换、到货后24小时内处理质量问题。
6. **配送服务**：确认配送时间是否匹配食堂开餐节奏（如每日早餐前送达）、能否分校区或分批次配送。

**可操作步骤**：
- 向当地农产品批发市场、大型果品公司或教育局推荐清单中索取至少3家供应商报价。
- 索要样品（按食堂3天用量），试吃并记录果径、甜度、新鲜度（从表皮、硬度、气味等判断）。
- 实地考察其仓库、分拣流程，并查看近期供货学校的评价（可电话咨询该校食堂管理员）。
- 签约试行期（如1个月），期间记录损耗率、学生反馈，再决定长期合作。

需注意：避免与无实体仓库的中间商合作（易出现断供或转包劣质品），优先选择有政府采购协议的供应商。
