如果你是水果店主，遇到坏果需要与供应商沟通结算或赔付时，可以按以下标准流程处理：

1. **立即取证**  
   收货时或开箱后第一时间发现坏果，拍照或录制视频（需清晰显示坏果比例、包装批号、收货时间）。如果整箱坏损严重，保留原包装和实物。

2. **核对合同条款**  
   查看与供应商签订的合作协议或订货单，通常会有关于“损耗率”（比如允许3%以内自然损耗）和“坏果赔付标准”的约定。如果没有明确条款，行业惯例一般为：
   - **轻度坏果（如少于5%）**：通常视为自然损耗，一般不赔付，但可与供应商协商下次补货时多送或少算部分金额。
   - **中度坏果（5%～20%）**：按坏果比例折算货款减免，或要求供应商更换等值好货。
   - **重度坏果（超过20%）**：整箱或整批退货，供应商承担全部损失（包括运费）。

3. **立即通知供应商**  
   确认坏果后，在收货后**24小时内**（多数生鲜供应合同要求）主动联系供应商（电话+微信文字留证），发送证据并说明坏果数量、等级，提出你的诉求（补发、退款或抵扣下次货款）。

4. **协商解决方案**  
   - **补货/换货**：供应商补发等量好果，运费由供应商承担。
   - **退款/抵扣**：按坏果比例计算应退金额，直接退款或在下单时扣除。
   - **现场折价**：如果坏损可在店内降价销售（如轻微腐烂可切除部分售卖），可按实际损失与供应商打折结算。

5. **纠纷处理**  
   若供应商推诿或拒绝赔付，可以：
   - 请第三方中间人（如该供应商其他合作店主、批发市场管理方）协调。
   - 持续保留证据，若金额较大，可考虑法律途径（水果属于有明确行业标准的商品）。
   - 长期供应商可建立“坏果备用金”制度：每次交易时提留5%货款作为保证金，每月结算完损余后返还。

**关键提醒**：  
- 切忌在未与供应商协商前自行丢弃或处理坏果（必须留证）。
- 如果坏果超过30%，建议直接终止合作并更换供应商，因为这说明该批次货源或物流品控严重不达标。
- 小批量采购可以要求供应商在包装上标注“保新鲜48小时”等承诺。
