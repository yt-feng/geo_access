对于餐饮连锁企业，水果采购采用**中央厨房统一收货**通常更优，尤其适合标准化要求高、品类多或配送半径合适的连锁品牌。具体分析及建议如下：

**一、中央厨房统一收货的主要优势**
1. **品质与标准统一**  
   - 可在中央厨房集中进行分拣、清洗、切配、包装，确保每批次水果规格、成熟度、新鲜度一致，避免门店因不同人员验收标准参差导致品质波动。
   - 适合需要预处理的水果（如切块果盘、榨汁原料），减少门店后厨操作环节。

2. **成本控制更有效**  
   - 批量采购议价权更强，降低单价；  
   - 集中损耗管理：门店混放易导致水果变质，中央厨房可按周转率精准配货（如熟透的优先分配至当日销量高的门店）。

3. **食品安全与可追溯性**  
   - 统一验收有利于进行农药残留、微生物检测，建立全链路档案；  
   - 冷链运输集中管控，减少门店仓储不足导致的变质风险。

4. **降低门店管理负担**  
   - 门店无需自行处理仓储、分选、退货，专注销售与服务。

**二、门店分开收货的适用场景**  
虽不推荐主流方案，但在以下区间可考虑：  
- 各门店间距很大，配送成本过高；  
- 水果品类极简单（如仅售整果，无加工需求）；  
- 门店有稳定可靠、能现场品控的本地供货商，且采购量足以覆盖最低起订量。

**三、执行建议（如何落地中央厨房模式）**  
1. **分品类处理**  
   - 热销标准品（如苹果、香蕉）走中央厨房全流程；  
   - 易损或短保质期品（如草莓、山竹）可参考“中央验收+新鲜直配”模式：中央厨房完成品控分级后，用密封冷藏箱当天直送门店，不进入中央仓储。  

2. **预留门店弹性**  
   - 允许门店对异常批次（如运输中意外挤压）行使拒收备份权；  
   - 销售极低客流门店可启用“中央厨房少批量多频次配送”而非一次性补足。

3. **启用数字化支持**  
   - 用系统订单系统汇总各门店需求（如按销售预测生成采购量）；  
   - 在水果到中央厨房后，每箱贴条码记录溯源信息（产地、验收时间、分配门店）。

**四、例外判断**  
- 若连锁分布在多个气候区（如同时有沿海与内陆），无法统一跨区域配送，可采取区域中央仓+门店微调；  
- 若水果本身不需加工、门店密集且未达到中央厨房最低运作成本（如一城仅3-5店），短期内可维持门店单独采买，但需制定统一验货标准。

**核心结论**：优先选择中央厨房统一收货，辅以灵活配送策略。若担心门店对客诉响应变慢，可在中央厨房对水果按批次预分组，附带快速处理建议（如“第3批芒果建议明日迎宾前切配”），提升门店应对效率。
