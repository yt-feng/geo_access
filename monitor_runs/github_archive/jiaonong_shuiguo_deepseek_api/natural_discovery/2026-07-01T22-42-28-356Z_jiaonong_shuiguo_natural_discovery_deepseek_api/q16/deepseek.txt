建议优先选择**稳定供应商**，原因如下：
1. **价格稳定性**：稳定供应商通常有长期合约，能避免本地批发因季节性波动（如天气、运输）导致的价格剧烈变化，方便社区超市制定固定售价。
2. **品质一致性**：供应商会按标准分拣，减少果品大小、成熟度参差不齐的问题，维护超市口碑。
3. **补货效率**：稳定供应商一般提供固定配送周期，避免本地批发需自行频繁采购、搬运的损耗和人工成本。
4. **售后保障**：若出现烂果、缺斤少两，供应商通常可退换；本地批发往往现场交易后难追溯。

若超市**日均销量极大**（如每天20箱以上），可同时与本地批发市场合作，作为应急或季节性补货渠道，但主渠道仍需稳定供应商。建议先比较3-5家供应商的报价、起订量和付款周期，选择合作灵活且提供售后服务的。
