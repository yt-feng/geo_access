选购菠萝原料时，**出汁率通常比甜度更为重要**，但具体取决于你的茶饮定位。以下是具体分析：

1. **出汁率直接影响成本和口感**：
   - 高出汁率的菠萝能产出更多果汁，降低单杯饮品的原料成本。
   - 大部分茶饮（如菠萝冰沙、果茶）依赖果汁作为基底，出汁率低会导致饮品稀薄、口感寡淡，甚至需要额外添加糖或水来调整。
   - 例如：制作一杯500ml的菠萝饮品，可能需要约150g菠萝果肉。若出汁率从60%下降到40%，相当于每杯多消耗30g果肉，长期成本差异显著。

2. **甜度可后期调节**：
   - 茶饮店通常用糖浆、果糖或添加蜂蜜来统一甜度标准。即使菠萝本身酸度高，可以通过加糖平衡。
   - 但出汁率不足时，无法通过后期添加物质弥补果肉本身的含水量和风味密度。

3. **特殊场景下的优先级调整**：
   - 如果你做**纯天然/低糖线**产品（如强调“无添加糖”），那么甜度更重要，因为酸度过高会直接影响消费者口感。
   - 若以**菠萝为主打小料**（如切块加在饮品顶部），则甜度比出汁率更重要，因为顾客会直接咀嚼果肉，酸度过高会降低体验。
   - 若用于**菠萝糖浆/浓缩液**制作，优先出汁率，因为浓缩后甜度会自然提升。

**实操建议**：
- 优先选择**出汁率≥65%**（可通过挤压测试或向供应商索要数据）且**甜度≥12°Bx**（白利糖度）的原料，两者均衡。
- 与供应商明确要求：提供样品后，分别测试冷饮和热饮场景下的出汁率，因为热处理会导致部分果汁挥发。
- 如果预算有限，可搭配**出汁率高+酸度略高**的菠萝（省成本），并通过红糖或蔗糖调节口感，避免高价购买“极甜但出汁低”的品种。
