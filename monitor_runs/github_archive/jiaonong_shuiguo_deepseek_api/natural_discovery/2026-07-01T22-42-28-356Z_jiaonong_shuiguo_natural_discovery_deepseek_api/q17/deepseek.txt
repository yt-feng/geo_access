控制社区超市水果新鲜度和库存，核心是**“小批量、高频次、快周转”**。以下是可执行的具体建议：

### 1. 进货策略：以销定进
-   **拒绝“整箱盲进”**：不要只看整箱便宜。根据前一周的销售数据，预估每天每种水果的销量，按**“单日销量 × 1.5倍（安全库存）”**来进货。
-   **分批次、分时段**：热销水果（如苹果、香蕉）可分2-3天进一次；高损耗水果（如草莓、荔枝、葡萄）尽量每天进，甚至只进半天的量。
-   **预留“试卖量”**：对新品种水果，第一次只进最低起订量（如5斤），用一周时间验证周边顾客的接受度。

### 2. 到货验收：设立“止损红线”
-   **当场验货**：到货后20分钟内，开箱检查果柄是否新鲜（青绿为佳）、果面有无伤痕/褐变、有无软烂或发霉。**发现品质不达标，当场拒收或按比例扣减货款**。
-   **测量硬度**：用指尖轻压苹果、梨、桃等，手感硬挺为佳；过硬或过软都不可取。
-   **关注隐蔽问题**：检查箱底和中间层，防止“面儿上光鲜，底下烂果”。

### 3. 店内储存与分拣
-   **绿色分级**：将最漂亮的果子摆在最上层/前台吸引顾客；有轻微瑕疵（如小划痕）的立即降价处理（如“特价区”）；有腐烂迹象的**第一时间挑出扔掉**，避免“一个坏果传染整筐”。
-   **温控管理**：
    -   **常温货架**：香蕉、芒果、木瓜、牛油果（未熟时放常温）。
    -   **冷藏库/冰箱**：草莓、蓝莓、葡萄、荔枝、樱桃（0-4℃）。
    -   **避光/不可挤压**：大部分水果避免阳光直射；洋葱、土豆（虽不是水果但同理）不可与苹果同放（苹果释放乙烯催熟）。
-   **催熟小技巧**：未熟的水果（如猕猴桃、硬桃）可单独放入盒中，加一个熟透的香蕉/苹果，密封1-2天催熟，控制上架节奏。

### 4. 上架与销售：动态调整
-   **陈列原则**：**少摆勤补**。每个品种只摆出预计2-3小时能卖完的量，卖完再从库房补；库房中按“先进先出”码放。
-   **价格阶梯**：
    -   **黄金位置**（入口、第一层）：新鲜度最高、卖相最好的，正常价。
    -   **促销区**（收银台附近、货架底层）：当日临期（如晚7点后）或轻微瑕疵品，直接标“今日特价”“买二送一”，**宁可低价卖出，不要压到第二天**。
-   **销售波动应对**：天气变冷或下雨前，减少热带水果进货量；节气或节日（如中秋）前适量增加礼盒装储备。

### 5. 滞销处理：必须“快刀斩乱麻”
-   **设定“最后期限”**：一种水果上架超过2天仍卖不动（非高价品），立即降价10-20%促销；超过3天，降价至成本价以下（如“一口价”），或做成“果切拼盘”“果茶”等加工品。
-   **捆绑销售**：买樱桃送一小袋柠檬片（去库存）；买苹果凑单送一根蔫了的香蕉（提醒顾客尽快吃）。
-   **反向激励员工**：对员工设定“损耗率考核”，如果某种水果损耗率超过15%，扣减奖金，倒逼他们积极推销、勤于理货。

### 6. 数据跟踪（简单有效）
-   **每日记录**：用Excel或纸笔记录每天4个关键数据：① 今日进货量；② 昨日库存量；③ 今日销量；④ 今日损耗量（扔掉、喂狗、送人）。坚持1个月，就能摸清规律。
-   **计算“周转天数”**：库存金额 / 日均销售金额。当周转天数超过3天时，立即预警并停止该品类进货。

### 执行口诀：
**“进场严、摆得少、卖得快、剩的切、烂的丢、价勤调、账日结。”**

---

**风险提示**：水果损耗是天然存在的（通常社区超市合理损耗率在5%-12%），不要追求“零损耗”。允许少量正常损耗，但要坚决杜绝“过夜烂货再卖”导致的客诉和口碑坍塌。
