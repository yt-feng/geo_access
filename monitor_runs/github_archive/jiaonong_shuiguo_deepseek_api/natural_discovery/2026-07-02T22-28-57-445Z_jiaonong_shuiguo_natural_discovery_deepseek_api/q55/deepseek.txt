为餐厅采购水果拼盘原料时，选择规格的关键在于**出肉率（可食用部分比例）**、**损耗率**和**切配时间成本**。以下是针对苹果、橙子、菠萝的具体规格建议：

### 1. 苹果（选择小果或中果，价格敏感型选三级果）
- **优选规格**：直径7-7.5厘米（约180-220克/个）。
  - **理由**：小果或中果单价更低，切片后数量足够拼盘（一个中果可切成8-10片）。大果虽吃相好看，但成本高20%-30%，且单价通常更高。
- **节省技巧**：购买时注意“糖心”或“花牛”等品种可能更贵，普通红富士是性价比之选。如果用于拼盘，允许表面轻微瑕疵的“三级果”价格可低30%。

### 2. 橙子（选大果，但避免超大型）
- **优选规格**：直径8-9厘米（约250-300克/个）。
  - **理由**：橙子皮占比较重（约20%-25%），小果（150克）出肉率低，去皮后所剩无几。中等偏大的橙子皮更薄、果肉饱满，单价相近时更划算。超大果（>400克）可能因皮厚、风味淡而不值。
- **节省技巧**：采购脐橙或夏橙，选择新鲜度高的（表皮紧实、有光泽）。如果允许，可直接购买切好的橙子瓣（冷冻或新鲜），但需核对重量是否含水或冰。

### 3. 菠萝（选带皮整果，规格中等偏大）
- **优选规格**：整果重量1.2-1.5公斤（去皮后约700-900克）。
  - **理由**：菠萝去皮损耗约40%（外皮+硬芯），小果（<1公斤）处理麻烦，出肉率更低。大果（>2公斤）可能因过熟而纤维变粗。中等规格的菠萝单价通常最低，且芯部较细，可减少丢弃。
- **节省技巧**：**不要买已切好的菠萝**（价格可能贵2-3倍，且可能用过熟果）。如果用量大，直接买整箱绿熟果（室温放2-3天即可），成本可再降15%。

### 综合性价比建议
- **苹果**：每天按拼盘需要切3-4个中果，直接去批发市场买20斤一箱的**三级红富士**，单价约3-4元/斤（超市价6-8元）。
- **橙子**：选**脐橙**大果（单果约300克），按拼盘需要每天煮8-10个，批发价约4-5元/斤（超市7-9元）。
- **菠萝**：若一天用3-5个，找**水果批发商**拿整箱菠萝（约15-20个/箱），每个约3-4元（超市6-8元）。

### 关键提醒
- **避免过度包装**：预切好的水果（如切块苹果、菠萝）通常质次价高，且因氧化变色快。
- **按需采购**：苹果和橙子可冰箱冷藏保存2-3周，但菠萝建议5-7天内用完，否则糖分下降。
- **对比“出肉率”**：例如，一个2元的苹果（200克去皮可食约170克）比一个5元的芒果（去皮核后可能仅150克）更划算——但这取决于你的拼盘定价。对于餐厅，选**出肉率高的低价水果**（如苹果、橙子）作为基座，附加少量高价水果（如草莓、蓝莓）提升档次，这样成本更低且易控。
