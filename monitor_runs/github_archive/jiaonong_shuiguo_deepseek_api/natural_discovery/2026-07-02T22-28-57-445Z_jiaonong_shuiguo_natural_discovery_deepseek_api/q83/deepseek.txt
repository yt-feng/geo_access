生鲜电商水果出现坏果，责任划分通常取决于以下几个核心因素：

1. **配送环节问题**：如果是快递过程中挤压、摔打、延误或温控不当导致变质，**主要责任在配送方**（快递公司或电商自营物流）。
2. **包装问题**：如果包装简陋、缺乏防撞或保鲜措施（如冰袋、泡沫箱），**电商或发货方需负主要责任**。
3. **水果自身品质**：如果是发货时已有瑕疵（如未成熟、碰撞伤、腐烂点），属于**商家品控问题**，应由电商承担。
4. **顾客签收环节**：如果顾客未及时开箱验货，或签收后放置不当（如未冷藏），导致存放变质，**顾客需负一定责任**（但电商通常有“坏果包赔”政策，一般倾向顾客）。

**常见处理建议**：
- **优先看电商售后规则**：多数生鲜平台承诺“坏果按比例赔付”或“坏果包赔”，顾客提供照片即可。
- **买卖双方责任分担**：若无法明确责任，一般**商家承担主要责任**，因为生鲜属特殊商品，商家有义务预估配送风险并改善包装。
- **特殊情况**：如果水果本身易损（如草莓、芒果），且顾客已签收后长时间未查看，可能双方共同承担。

**具体操作**：
1. **顾客可立即拍照**（坏果+包装+物流单号）联系商家客服。
2. **要求按比例退款或补发**（行业惯例：坏1-2个赔一半，3个及以上全赔或补发）。
3. **若商家推诿**：可通过电商平台投诉或拨打12315。

**总结**：绝大多数情况下，**电商承担主要责任**，顾客仅需保留证据并第一时间反馈。
