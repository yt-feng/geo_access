酒店采购水果盘原料时，供应商应保证以下规格和外观标准：

**规格标准：**
1. **新鲜度**：水果应新鲜无腐烂、无霉变、无虫蛀，采摘后及时处理并冷链运输。
2. **大小均匀**：每种水果的个头、重量应基本一致（如苹果直径7-8厘米，橙子150-180克/个），避免大小参差影响摆盘美观。
3. **成熟度适宜**：水果成熟度适中，如香蕉无青涩且无黑斑，芒果、猕猴桃等柔软但不软烂，甜瓜类微硬可放熟。
4. **无损伤**：表面无磕碰、挤压、擦伤、裂口或其他机械损伤。
5. **无病虫害及药残**：符合食品安全标准，无农药残留超标，无黑点、病斑等。

**外观标准：**
1. **色泽自然**：水果颜色应饱满、均匀，如草莓鲜红、红提紫黑、火龙果鲜亮，无褪色、发黄或过度着色。
2. **外形完整**：果形端正，无畸形（如草莓过大变形、柠檬奇形等），无凹陷或扁平。
3. **表面清洁**：无泥土、灰尘、汁液或化学物质残留，无蜡质异常堆积。
4. **无瑕疵**：无锈斑、褐变（如苹果切开后防褐变处理需合规）、冻伤或水渍，无籽粒脱落（如石榴、蓝莓）。
5. **包装规范**：水果应使用符合食品级标准的包装材料，分层隔开，避免挤压，有防撞、防潮、防串味措施。

供应商应提供批次检测报告或产地证明，并保证冷链运输，确保到货时水果温度在3-7℃（视品种调整），且开封后外观无异常。酒店可到货抽检，不合格则退回。
