处理水果店坏果的结算或赔付，一般遵循以下流程和原则，建议根据实际合同或约定灵活操作：

1. **确认坏果标准与责任方**  
   - **自然损耗**：如运输途中轻微碰伤、少量腐烂（通常约定在合同损耗率内），一般由水果店自行承担，不列入赔付范围。  
   - **品质问题**：如大面积腐烂、虫害、成熟度过高等，需判断是否因供应商采摘、包装、储存不当造成。需提供照片、视频或第三方检测证明（如坏果比例超3%-5%）。

2. **启动赔付流程**  
   - **A. 当场验收或到货24小时内发现问题**：立即通知供应商，保留证据（称重、拍照、视频），要求退货或换货。  
   - **B. 到货后24小时以上至保质期内**：需按合同约定的“理货期”处理。常见做法：  
     - **按比例折算**：例如坏果占10%，则结算时扣减对应货款（如总价1000元，扣100元）。  
     - **全额赔付**：若坏果比例极高（如超过50%），可要求供应商全额补偿该批次货款或换货。  
     - **阶梯式定价**：部分合同约定，坏果率在5%-10%内，供应商赔付50%货款；超10%则全额赔付。

3. **结算方式**  
   - **直接扣款**：在原有应付货款中扣除坏果对应金额。  
   - **补发冲抵**：供应商补发等量新鲜水果，当期结算时扣除已补发部分。  
   - **延迟结算**：若坏果量大，双方协商将结算周期延长至换货后。

4. **注意事项**  
   - **明确合同条款**：签署供货协议时，需约定“退换货标准”“赔付比例”“理货期（通常为24-48小时）”“举证方式（如除照片外需保留坏果实物）”。  
   - **分级赔付**：对次果（可降价销售）和烂果（需销毁）分开处理，次果可要求打折结算。  
   - **供应商信用**：长期合作供应商可协商“月结赔付额度”，超出部分当月扣清。  
   - **法律依据**：若供应商拒不处理，可依据《合同法》主张违约责任，或向市场监督管理局投诉。

**示例操作**：  
- 某批苹果到货后，抽样发现约15%碰伤腐烂。你立即拍照、称重坏果（5公斤），通知供应商。  
- 若合同约定“坏果率超10%全额赔付”：要求供应商换货5公斤好果，或扣除该批次15%货款（假设总成本500元，扣75元）。  
- 若合同无效：协商扣款30-50元作为补偿，或补发等价水果。

建议优先通过书面或可留存证据的即时通讯（微信、邮件）沟通，避免口头承诺。如为长期合作，可约定每月汇总赔付。
