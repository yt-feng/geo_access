建议你按照以下步骤与供应商沟通并约定临时补货：

1. **及时沟通**：第一时间联系供应商，说明菠萝用量突然增加的原因（如旺季、订单激增等），并表达紧急补货需求。
2. **明确需求**：告知具体的补充数量、期望到货时间、品质标准（如成熟度、大小、无损伤等）以及包装要求。
3. **协商价格与条款**：询问临时补货的价格是否与之前合同一致（提前确认是否有旺季加价），并明确付款方式、结算周期。
4. **确保可用库存**：请供应商确认现有库存能否满足你的补货量，如果不足，协商是否分批次交付。
5. **约定交货与验收**：明确发货时间、运输方式（如冷链）、到货日期，并约定验收流程（如到货后24小时内检查，不合格品扣款或退换）。
6. **签订补充协议**（视情况）：如果补货量大或价格变动，建议用微信文字、邮件或简单书面协议确认，内容包括数量、单价、总金额、交货期、违约责任等。
7. **建立备用方案**：为防止再次出现用量波动，可询问供应商能否提前备货（如提前采购菠萝在冷库储存），或与第二家供应商建立联系作为应急渠道。

**示例话术**（电话或微信沟通）：
“张总，近期我们门店生意特别好，菠萝用量比平时多了将近一倍，之前合同的数量不够用了。能否紧急帮我追加**300公斤**菠萝？最好**后天中午前**能送到，品质还是按上次的B级偏大果，价格是否维持原来合同价？如果不加价，我这边可以立即下单；如果需要调整，我也好做预算。您看库存和排期是否支持？”

**关键点**：保持态度诚恳但紧急，确认信息落实到书面记录，明确验收标准，避免后期扯皮。
