作为采购方，推荐新品时建议采用“少量试卖、快速验证”的策略，具体数量取决于店铺类型和流量情况：

1. **线上店铺/社区团购**：首批试卖建议控制在 **50-100份**（如单箱水果或小包装）。这样既能测试市场反应，又不会积压库存。
2. **线下实体店（中小型）**：准备 **1-2箱（约20-40斤）**，放在显眼位置试卖。注意观察顾客试吃和购买情况。
3. **批发/供应渠道**：用于二次分销时，建议先订 **5-10箱**，优先在核心客户或老客户中推广。

**判断标准**：如果1-2周内，回头客或复购率超过30%，且损耗率控制在5%以下，再考虑扩大3-5倍进货量。试卖期间要重点记录退货、投诉和客单价变化。
