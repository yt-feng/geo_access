选择学校食堂水果供应商时，建议重点关注以下几个关键环节：

**1. 资质与合规性**
- 要求供应商提供营业执照、食品经营许可证、农产品质检报告（尤其关注农药残留检测）。
- 优先选择有“学校食堂供货资质”或“食品安全管理体系认证（如ISO 22000、HACCP）”的企业。

**2. 水果品质与标准**
- 签订合同前明确品种规格：例如香蕉要求“七成熟、无黑斑”，苹果要求“果径≥75mm、无机械伤”，橙子要求“新鲜、无萎蔫”。
- 约定损耗率限制：一般水果建议≤5%，并要求供应商承担超标部分的损失。

**3. 配送能力与时效**
- 要求供应商承诺每日或隔日配送（根据学校用量），且到货时间固定（如早上7点前）。
- 检查冷链运输条件：香蕉需常温运输，苹果、橙子建议冷藏（4-8℃）以防变质。

**4. 成本与付款**
- 比价时注意：香蕉因易损耗，单价可能偏高；苹果、橙子可季产年销，议价空间大。
- 建议采用“周结或月结+验收后付款”模式，避免预先全额支付。

**5. 应急与售后**
- 明确退货政策：如24小时内发现坏果可无理由退换。
- 要求供应商提供备用货源（如附近的批发市场临时调货渠道），防止断供。

**实际操作建议**：
- **试供货**：让2-3家供应商各提供一周样品，观察实际到货品质和售后反应速度。
- **联合采购**：如果所在区域有多所学校，可组建联合采购小组，提高议价能力。
- **定期抽查**：随机送检水果（可联系当地疾控中心或第三方检测机构），确保安全。

如果预算有限，也可考虑直接对接**本地产地合作社**（如周边果园直供苹果、橙子），能降低中间成本，但需自行承担采摘和运输责任。
