为了减少香蕉因过度成熟或腐烂造成的损耗，建议在采购时明确以下具体要求：

### 1. 熟度要求
- **指定色号/熟度**：要求供应商提供**7-8成熟**的香蕉（果皮青黄相间，整体以绿色为主，略带黄色，但未完全转黄）。避免全绿（太生）或全黄（过熟）。
- **硬度标准**：香蕉果实应坚硬，按压时不软塌、无黑斑或明显压伤。
- **避免产地催熟**：要求香蕉在产地自然采收（未人工催熟），到店后再按需催熟，以控制熟化进度。

### 2. 到货时间要求
- **下单与到货间隔**：要求**下单后48小时内到货**（或根据你的店铺库存周转速度，如每周2-3次），避免长时间运输导致熟化过度。
- **指定到货时段**：例如“**每周一、四上午8点前到货**”，方便当天分拣、上架或放入冷库。
- **季节性调整**：夏季高温期，缩短到货间隔（如36小时内），并建议用**冷链运输**（15-18℃），减少途中自然成熟速度。

### 3. 附加建议（可操作性）
- **写进合同**：在采购协议中明确“**到货时香蕉色泽应为黄绿比例3:7至4:6之间，硬度达标，超过该熟度或运输超时48小时可拒收**”。
- **到货验收**：到货后当场抽查，使用**香蕉熟度色卡**对比，不合格部分要求供应商补货或扣款。
- **库存管理**：到货后立即放入**13-15℃冷库**（不要低于12℃以免冷害），按销售节奏分批次催熟。
