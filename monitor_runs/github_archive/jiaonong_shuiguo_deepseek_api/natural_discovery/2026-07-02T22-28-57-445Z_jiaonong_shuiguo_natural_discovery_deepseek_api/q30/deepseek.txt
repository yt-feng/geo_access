针对超市水果到货品相不好的售后流程，建议按照以下步骤约定，确保权责清晰、操作简便：

1. **到货验收与反馈时限**  
   - 收货方在签收后立即开箱验收，发现品相问题（如腐烂、压伤、霉变等），**需在签收后2小时内**拍照（包含外包装、整体、问题产品细节）并通知供货方。
   - 逾期视为品相合格，供货方不再受理。

2. **问题判定标准**  
   - 明确品相“不好”的具体范围（如：腐烂面积超过5%、压伤导致无法售卖、霉变斑点超过2处）。可参考行业通用等级，或双方随货附上“品相验收参考图”。

3. **售后处理方式**（选择其一或组合）  
   - **补货/换货**：供货方需在接到通知后**24小时内**按问题数量补发同规格好品，运费由供货方承担。  
   - **退款/扣款**：按问题产品占比扣除对应货款（如每箱单价200元，问题产品占30%，则扣除60元）。  
   - **折扣结算**：如整体品相勉强可接受，双方协商按原价**70%-90%** 结算，并备注“品相折扣”。

4. **争议解决**  
   - 双方对品相判断有争议时，**保留原产品及包装**，在12小时内由第三方见证（如市场管理员或共同指定人员）拍照存档，按“半数以上坏品”原则判断。  
   - 若仍无法解决，可约定“以最终实际售价低于成本价的部分”为赔偿依据（需提供销售记录）。

5. **执行层面建议**  
   - **书面合同**：在采购单或合同中明确上述条款，并用加粗字体标注“到货2小时内反馈”。  
   - **定期复盘**：若连续3次出现品相问题，自动触发供货商调整机制（如降低采购量、增加赔付比例）。  

**关键点**：时限越短（如1-2小时）、标准越具体（附照片或样板），售后纠纷越少。若供货方为非长期合作方，建议先试用小额订单测试流程。
