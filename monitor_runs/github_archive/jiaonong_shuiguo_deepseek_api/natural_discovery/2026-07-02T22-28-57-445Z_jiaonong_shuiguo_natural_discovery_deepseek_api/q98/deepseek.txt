针对养老院或医院食堂采购水果，判断供应商的合规与稳定性，建议从以下维度进行系统性评估和筛选：

### 一、合规性核查（硬性条件）

1. **资质证照**
   - 营业执照（经营范围含“食品销售、农产品批发”）
   - 食品经营许可证或食品生产许可证（若涉及分装）
   - 员工健康证（配送人员、分拣人员）
   - 农业订单或产地证明（体现可追溯性）
   - 符合当地市场监管局要求的“浙食链”“沪冷链”等电子追溯平台接入证明（适用地区）

2. **标准文件**
   - 每批次农残检测报告（需有CMA或CNAS标识，或政府快检点报告）
   - 供应商内部质量控制制度（如HACCP、ISO22000认证更优）
   - 产品标准号（如GB/T 26150等）

3. **配送环节合规**
   - 冷链配送车辆合格证明（冷藏车温度应能保持在0-8℃）
   - 配送全程温控记录（需可追溯）

### 二、稳定性评估（动态能力）

1. **供应保障能力**
   - 近期（如3-6个月）历史供货记录：每日/每周到货准点率、缺货率
   - 多产地备用渠道：主要单品是否有3个以上稳定产地
   - 应急响应能力：突发缺货后多少小时内能补货（建议≤4小时）

2. **质量稳定性**
   - 每批次验收合格率：如90%以上不出现腐烂、机械损伤、农残超标
   - 退换货率与处理时效：如低于2%，24小时内完成退换
   - 季节性品种替代能力：例如6-8月西瓜供应能否用蜜瓜、甜瓜替代

3. **价格稳定性**
   - 签订价格锁定条款：如月度单价波动超过5%，提供调价预警
   - 公开参考市场：如对接农产品批发市场（如北京新发地、上海江桥）官方报价

### 三、实地与文件核查步骤

1. **现场考察（建议每季度一次）**
   - 仓库：检查分拣环境、杀虫剂使用规范、水果冷藏温度
   - 运输车：查看是否混装有毒有害物质
   - 追溯体系：随机抽取3-5个批次，检查从产地到入库的完整单据

2. **第三方验证**
   - 通过“国家企业信用信息公示系统”核查有无行政处罚或失信记录
   - 查询该供应商是否被当地市场监管局通报过农残超标案例

### 四、合同关键条款设计

| 条款类型 | 具体内容 |
|---------|---------|
| 违约责任 | 约定不合规行为（如伪造检测报告）一次即解除合同，并索赔 |
| 保证金 | 建议收取合同金额5-10%作为履约保证金 |
| 定期评估 | 每季度按以上指标打分，低于70分可启动替换机制 |

### 五、替代方案

- **优先使用国有或政府集采平台**（如部分城市由区级卫监局统一招标）
- 若供应商数量有限，与至少3家供应商签订短期轮换合同，降低单一依赖风险
- 建立“主要 + 备用”供应商并行机制（主供80% + 备供20%以维持合作）

### 六、风险预警信号

- 无法提供带追溯码的电子票据
- 连续2个月到货批次农残快检阳性率超过5%
- 突击检查时仓库内水果堆叠超高、混存其他化工产品

如需更具体的操作模板（如供应商评分表、应急补货预案），可进一步提供。
