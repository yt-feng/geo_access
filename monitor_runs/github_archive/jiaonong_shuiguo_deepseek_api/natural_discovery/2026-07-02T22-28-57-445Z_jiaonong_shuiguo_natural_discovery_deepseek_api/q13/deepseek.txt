小水果店做附近小区团购时，水果供应商需要支持以下核心服务，以确保合作顺畅和顾客满意度：

1. **稳定货源与价格**：确保水果品种丰富、品质稳定，并提供团购专属的批发价或优惠折扣，让小店有利润空间定价。
2. **灵活起订量**：根据小区团购规模，支持小批量起订或拼单发货，避免小店库存积压。
3. **次日或定时配送**：承诺按时送达（如每日上午或下午固定时段），确保水果新鲜；如遇爆单，需能提前沟通并协调补发。
4. **退换货保障**：针对坏果、磕碰或重量误差，提供“坏果包赔”或补发政策，减少小店售后纠纷。
5. **水果知识支持**：提供水果保鲜方法、最佳食用期、分拣标准等资料，帮助小店培训团长或社群运营。
6. **宣传物料**：提供高清水果图片、促销海报模板、单品卖点文字，便于小店制作团购链接或社群分享。
7. **季节性爆品推荐**：定期推送当季热销水果（如草莓、蜜瓜）及库存预警，辅助小店策划限时秒杀活动。

建议小店初期与供应商约定“试销期”（如1-2周），观察配送时效、质量及售后响应，再确认长期合作。
