通常情况下，企业福利采购水果礼盒时，**建议优先选择支持开发票和定制包装的供应商**，以确保合规性与企业品牌形象。以下是具体说明：

1. **开发票**：企业采购需进行财务入账、报销或税务抵扣，供应商应能提供正规增值税发票（普通或专用发票）。如为一般纳税人，专票可抵扣进项税。建议在询价时明确发票类型与税率。

2. **定制包装**：企业福利场景中，统一、美观的定制包装（如印有企业LOGO、祝福语或节日主题）能提升员工归属感或客户体验，体现企业形象。需确认供应商是否支持包装设计、起订量及加印费用。

**建议优先筛选同时满足这两点的供应商**：部分供应商可能仅提供基础包装或不含定制服务，或无法开具正规发票。若遇到此类情况，可：
- 明确告知供应商需求，询问是否可额外收费定制包装或提供发票；
- 要求供应商在报价单或合同中注明“含发票及包装定制服务”；
- 若对方明确无法满足，则需更换备选供应商，避免后续财务或品牌风险。

最终选择前，建议要求供应商提供包装设计样图及发票样本（隐去敏感信息），并书面确认服务条款。
