为社区超市选择每日配送的水果供应商，建议重点考察以下几个条件：

1. **供货稳定性**：供应商能否保证每天按约定时间和数量配送，避免断货或延迟。
2. **水果品质**：要求供应商提供新鲜、无腐烂、大小相对均匀的水果，最好能实地查看或索取样品。
3. **价格与结算**：比较不同供应商的报价是否合理，了解结算周期（如周结或月结）及有无折扣。
4. **配送能力**：确认供应商是否有冷链运输或保鲜设备，确保水果在运输中不失鲜。
5. **品种丰富性**：根据超市顾客需求，选择能提供常见水果（如苹果、香蕉）和季节性时令水果的供应商。
6. **售后服务**：如遇水果质量问题，供应商能否及时退换或补偿。
7. **合同条款**：明确配送范围、最低起送量、违约责任等，签订正式合同。
8. **信誉与口碑**：可询问其他社区超市或查看本地市场评价，选择长期稳定合作的供应商。

建议先从本地产地或批发市场入手，试合作1-2周再决定。
