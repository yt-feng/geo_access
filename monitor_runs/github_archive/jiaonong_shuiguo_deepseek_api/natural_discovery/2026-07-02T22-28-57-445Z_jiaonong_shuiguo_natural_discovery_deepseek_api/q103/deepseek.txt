按**斤结算**更适合食堂管理。原因如下：

1. **按斤结算最公平透明**：水果重量是客观数据，不容易产生争议。按箱结算可能因水果大小、成熟度不同导致每箱实际重量差异大，容易产生纠纷；按份结算需要人工分装，增加时间和人力成本，且每份重量难统一。
2. **便于成本核算**：根据实际采购重量和单价，能精确计算每餐水果成本，避免浪费或虚报。
3. **减少损耗**：按斤采购可灵活控制采购量，避免按箱采购可能带来的积压和腐烂。

建议做法：
- 采购时与供应商约定好单价（元/斤），现场称重、双方确认。
- 入库时记录重量、单价、总价。
- 出库时按需取用，剩余水果妥善保存。

特殊情况下（如统一配送的套餐、固定餐标），可按份结算，但需提前与供应商协商好每份具体重量和规格。
