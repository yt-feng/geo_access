针对家庭团购场景，建议组合应**兼顾营养均衡、口感互补、耐储存**和**性价比**。以下推荐四款组合，每款适合不同家庭需求：

**1. “健康常备”组合（适合有老人、孩子家庭）**  
- **苹果（3-4个）**：酸甜适口，富含果胶和维生素，耐储存（常温1-2周）。  
- **橙子（3-4个）**：维C丰富，酸甜解腻，全家通用。  
- **香蕉（3-4根）**：快速补充能量，软糯易消化，适合早餐或加餐。  
- **菠萝（1个）**：开胃助消化，可做菜或甜品，提升风味层次。  
→ **搭配建议**：苹果/橙子隔开摆放防压，香蕉单独放置避免催熟。

**2. “高性价比”组合（适合预算敏感、多口之家）**  
- **苹果（4-5个）**：价格亲民，耐放，可替代零食。  
- **橙子（3个）**：冬季时令水果，易保存。  
- **香蕉（4-5根）**：走量实惠，快速补充热量。  
- **菠萝（1个）**：单价高但用量灵活，1个足够全家吃2-3天。  
→ **搭配技巧**：团购时主打“水果篮”概念，混搭填满空间减少空隙。

**3. “均衡营养”组合（适合注重多样性的家庭）**  
- **苹果（2个）** + **橙子（2个）** + **香蕉（2根）** + **菠萝（1/2个切块）**  
→ 每种水果份量减少，避免浪费，适合2-3人家庭。  
→ 菠萝提前处理好（去芯切块），方便即食。

**4. “升级送礼”组合（适合探亲或节假日）**  
- **苹果（4个：红富士或蛇果）** + **橙子（4-6个：脐橙或冰糖橙）** + **香蕉（小把）** + **菠萝（1个：找金钻菠萝，口感更甜）**  
→ 装盒时苹果橙子对半堆叠，菠萝立放或斜靠，香蕉放空隙处，用丝带装饰。

**关键注意事项**：  
1. **成熟度控制**：香蕉选7-8成熟（香蕉两端略绿），橙子/苹果需完全成熟，菠萝选叶根有清香、果眼凸起的。  
2. **防压处理**：装箱时用软纸或泡沫隔开，菠萝的硬皮刺可削掉锐利凸起。  
3. **团购话术建议**：  
   “这款礼盒4种水果搭配，维C+膳食纤维+天然糖分，老人孩子都爱吃。开盒就能吃，菠萝可以鲜吃或做菠萝饭，香蕉和苹果适合带出门。价格比单买省钱约15%。”  

**最终推荐**：如果只选一种，首选“健康常备组合”（苹果+橙子+香蕉+菠萝），因为覆盖面最广，家庭接受度最高。
