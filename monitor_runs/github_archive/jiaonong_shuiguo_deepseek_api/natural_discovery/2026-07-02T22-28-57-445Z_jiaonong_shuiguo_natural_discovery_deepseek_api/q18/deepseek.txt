挑选香蕉时，可以按以下步骤操作，避免买到过生或过熟的果实：

1. **看颜色**：
   - **金黄略带青头**：最适合立即食用或1-2天内吃完。  
   - **全青**：明显过生，需放置2-3天才能熟透。  
   - **出现黑色斑点**：熟度已高，需当天食用，否则易变软烂。  
   - **整体变黑或软塌**：过熟，不建议购买。

2. **摸硬度**：
   - 轻轻按压香蕉中部，**有弹性、不硬不软**为最佳。  
   - 按下去**非常硬**：生。  
   - 按下去**明显凹陷或发软**：熟过头。

3. **看形状与棱角**：
   - 选择**饱满、棱角不明显**的香蕉（说明水分足、成熟度均匀）。  
   - 棱角尖锐、瘦长：通常较生。

4. **闻气味**：
   - 成熟香蕉有淡淡清甜味，**过熟**会有酒精味，**过生**则基本无味。

5. **注意“催熟”店铺**：超市冷柜或通风差的香蕉常被催熟，表面金黄但内部可能生硬。可挑**单根而非整把**，更容易判断成熟度。

**总结**：买金黄带微青头、饱满微弹的香蕉；避开全青、全黄发黑或软塌的。如果打算几天内吃完，选稍带青头的；如需立即吃，选全黄且无黑斑的。
