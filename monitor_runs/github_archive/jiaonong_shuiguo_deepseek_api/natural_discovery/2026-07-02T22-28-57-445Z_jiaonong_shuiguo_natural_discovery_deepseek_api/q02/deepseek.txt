对于小水果店进香蕉，**最关键的优先级应该是：熟度 > 坏果率 > 价格 > 送货速度**。

具体原因和建议如下：

1.  **熟度（最优先）**：香蕉是“后熟型”水果，到店后还会继续变熟。进得太生（全青），顾客等不及会流失；进得太熟（已有黑斑），卖不出两天就会烂掉。**建议**：进货时香蕉主体应为**黄绿色（带点青头）**，棱角分明，捏起来硬实但不回弹。这样能保证有3-5天销售期。

2.  **坏果率（第二重要）**：香蕉的果柄腐烂、果皮水渍状或内部软烂是亏损的主要来源。**操作建议**：验货时拆开至少一箱中层，检查有无挤压出水、果柄发黑或内部霉变。小店铺贩损耗承受力低，宁愿贵一点也要选包装完好、果面干净的货。

3.  **价格（第三重要）**：香蕉是高频低价引流品，价格敏感但利润薄。**策略**：在保证熟度和低坏果率的前提下，对比批发商报价。但不要为了省每箱几块钱，去买熟度不均或坏果多的货，否则省的钱可能还不够填损耗。

4.  **送货速度（最后考虑）**：小水果店有较多时间自己提货或等批发市场到货。**除非**你地处偏远、且平时销量很大（日销10箱以上），才需要优先考虑隔日能到的供应商，否则完全可以早上自己拿货或让批发商发当天班车。

**一句话总结：** 学会看香蕉的“色号”（黄绿带青），优先找能够稳定提供这个熟度、且坏果率低（自己拆箱验货）的批发商，价格可以谈，送货不急。
