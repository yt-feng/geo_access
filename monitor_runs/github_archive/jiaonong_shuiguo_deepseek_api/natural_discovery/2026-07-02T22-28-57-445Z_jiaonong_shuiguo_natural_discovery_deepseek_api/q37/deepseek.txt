商超会员日水果活动要顺利落地，供应商需提前重点配合以下准备：

1. **备货计划与库存确认**
   - 提前7-10天与商超采购确认活动主推品项（如引流款、爆款、利润款），按预估销量1.2-1.5倍备货，避免断货。
   - 明确供货周期：常温水果提前3天到货，冷链水果（如蓝莓、草莓）活动前1天或当天早晨送达。

2. **品质把控与分拣**
   - 活动前2天对果品逐箱抽查，剔除坏果、压伤、不熟果实，确保上架即卖相佳。
   - 针对活动价品项，可接受一定规格调整（如小果包装），但需明确告知商超，避免客诉。

3. **包装设计与标识**
   - 按商超要求定制统一活动标签（如“会员专享价”“限时特惠”）、价签底座、堆头围板。
   - 溢价品（如进口车厘子）需额外附加“开箱验证”或“果径标注”标签。

4. **物流与配送协调**
   - 规划配送时间：避开超市早高峰收货时段，安排专人在收货区对接，确保快速卸货。
   - 对娇嫩水果（如芒果、荔枝）需使用防震垫、冰袋或气柱包装。

5. **促销物料与试吃支持**
   - 提供活动期间的试吃样品（建议提前1天送样，当天补充），并配备切块刀、湿巾、牙签等工具。
   - 协助商超布置主题海报、吊旗、地贴等视觉物料。

6. **人员配合与现场培训**
   - 活动当天安排至少1名供应商促销员，负责理货、补货、称重解答。
   - 提前培训商超员工：活动果品知识（口味、储存、催熟方法）、退换货规则。

7. **应急预案**
   - 备好2-3款替补果品（如当季热销桃/梨），若主推品缺货可快速替换。
   - 准备退货/换货流程：残次品现场拍照确认后，24小时内完成补货或抵扣。

8. **售后数据反馈**
   - 活动结束后48小时内，向商超提交售罄率、退货率、客诉类型等数据，为下次活动优化提供依据。

**重点提示**：所有书面沟通（邮件/钉钉）确认备货量、到货时段、突发处理标准，避免口头约定导致扯皮。
