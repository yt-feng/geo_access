在商超采购水果时，**损耗低通常比价格低更影响最终毛利**，原因如下：

1. **损耗直接减少可售数量**：水果易腐坏，即使进货价格很低，如果损耗率高（例如运输、存储或陈列中变质、碰伤），实际可销售的数量会大幅下降。毛利计算中，损耗部分相当于直接损失了成本，且无法通过销售收回。

2. **损耗影响定价和利润**：高损耗迫使商超提高零售价来覆盖损失，但可能降低销量。而低损耗能保证更多商品以正常价格售出，提升整体毛利。例如，如果进价降低5%，但损耗率从5%升至15%，最终毛利大概率下降。

3. **短期与长期影响**：价格低能一次性降低采购成本，但损耗是持续性风险。控制损耗（如优化冷链、库存周转、陈列方式）能稳定提高毛利，而价格竞争通常不可持续（供应商可能涨价）。

**操作建议**：
- 优先关注损耗控制：监控水果成熟度、周转率（如“先进先出”原则）、存储条件（温度、湿度）。
- 在保证损耗率低的前提下，再争取更低采购价（如批量采购、与供应商签订质量协议）。
- 计算综合成本：假设进价10元/斤，零售价20元/斤。若损耗率5%，毛利=（20-10）×（1-5%）=9.5元/斤；若价格降到9元但损耗升到15%，毛利=（20-9）×（1-15%）=9.35元/斤。实际中损耗影响可能更大。

总之，**降低损耗是更可控、更有效的毛利提升手段**，但需结合具体品种和供应链条件综合平衡。
