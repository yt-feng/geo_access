判断茶饮连锁供应商能否稳定按规格送货，建议从以下四个维度进行系统性评估和验证：

### 1. 资质与生产能力的硬性核查
- **现场审计**：实地考察供应商的种植基地、分选包装线、冷藏仓储。重点看分选设备（如光学分选机、重量分级机）和冷链周转能力。如果对方连分选机都没有，规格一致性很难保证。
- **合同条款**：在合同中明确约定“规格误差范围”（如单果重量±10g、直径±5mm、色泽等级），并加上“连续三次不合格，甲方有权终止合同并索赔”的违约条款。

### 2. 历史发货数据与样品验证
- **索要连续3-6个月的出货记录**：要求提供过去一个产季（如去年9-12月）的每批次检验报告，看规格达标率（≥95%为优，≤90%需警惕）。
- **试单测试**：先下3-5次小批量试单（每次1吨左右），在收货时严格按你设定的规格抽检（如每箱开检20%）。统计合格率、变异系数（CV值），如果CV＞8%（即大小差异大），说明分选能力不稳定。

### 3. 供应韧性与备选方案
- **产季稳定性**：确认菠萝产地是否为多产区（如海南、云南、广西轮换或南半球补季），单一产地受气候影响大。要求供应商提供“应急调配预案”：当主产区出问题时，从备选基地或冷库调货的响应时间（建议≤3天）。
- **产能锁定量**：签订年度框架协议时，要求供应商预留月度产能的20%作为你的安全库存（尤其在台风季、寒潮季），并定期（如每周）汇报库存与在田情况。

### 4. 物流环节的规格保持能力
- **包装与运输标准**：要求使用“防撞泡沫网套+抗压纸箱+冷板控温车”。可先发一条干线试运，到货后检测碰伤率（≤3%为优）和失水率（≤5%）。如果碰伤率过高，说明包装或装车方式需改进。
- **到货时效**：明确“从采摘到送货≤72小时”，超过时间即可拒收或折扣结算。频繁延期说明冷链衔接有漏洞。

### 执行建议：
- **建立动态供应商评分卡**：每月根据抽检合格率、准时率、配合度打分，连续两个月低于85分触发降级或淘汰。
- **设置最低安全库存**：假设你每周稳定需要10吨规格菠萝，建议备20吨的冷藏库存（约一周周转量），并找两家供应商互为备份（主供占70%，备份30%）。

最后，可以要求供应商提供“批次溯源码”，每箱喷码标明采摘日期、产地、分选员，这样能精准定位问题环节。如果对方做不到，说明内部管理粗放，规格稳定性存疑。
