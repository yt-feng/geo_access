和酒店早餐水果供应商谈合作时，建议从以下方面重点核实和约定，以确保持续稳定的供应：

1. **供应能力与稳定性验证**：
   - 先要求供应商提供其自有或稳定合作的果园/种植基地证明（如产地、面积、年产量）
   - 明确其补货周期（如每日/隔日配送）、**应急响应时间**（如突发断货时，几小时内可补货？）
   - 要求提供近3-6个月对同类客户的供货记录（证明其旺季和淡季的应对能力）

2. **合同核心条款**：
   - **最低保障量**：约定每天/每周的最低供货量（如每日至少供应10箱鲜切水果）
   - **品种与品质标准**：明确水果品种清单及等级（如香蕉需4-5成熟度、切块大小统一）；禁用催熟剂、保鲜剂的具体要求
   - **损耗责任划分**：规定收货时的验收标准（如表面无霉斑、成熟度达标），验收后损耗由酒店承担；运输中损耗由供应商承担
   - **价格锁定与调整机制**：签订季度/半年度锁定价，同时约定“行情波动超过XX%”时的调价方式（如参考产地批发价指数调价+提前7天书面通知）
   - **违约条款**：明确断供罚款（如单次断供罚当日货值200%）、长期合作保证金金额

3. **实地考察与试供**：
   - 要求提供最近一次“食品安全抽检合格报告”或ISO22000认证
   - 先进行1-2周试供（注明合同中的“试供期条款”，无责终止），每天记录准时率、果品新鲜度、实际损耗率

4. **支付与结算安全**：
   - 避免预付款，争取月结或周结；要求提供正规增值税发票
   - 可设置延期付款条款：供应稳定性达标（如月到货率≥98%）则按时付，否则顺延

5. **应急预案与备份机制**：
   - 明确供应商面对台风、节假日爆单等情况的**替代供应方案**（如是否可使用本地的合同储备库）
   - 要求提供至少两家备用供应商的联系方式（对方知情可紧急调货）

**立即执行项**：先拿到供应商近期的供货明细表和检验报告，现场查看其冷库冷藏条件（温度是否波动、水果堆叠是否压坏）。

**风险预警**：若供应商无法提供任何产地证明或基地照片，且拒绝试供，建议直接淘汰。水果供应受季节、天气影响极大，务必在合同中把“不可抗力范围”缩窄至仅限政府强制令/自然灾害，不含常规台风、高温等。
