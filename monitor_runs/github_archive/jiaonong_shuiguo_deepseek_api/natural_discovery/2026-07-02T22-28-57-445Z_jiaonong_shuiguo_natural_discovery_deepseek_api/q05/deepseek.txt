建议根据店铺定位、客流量和损耗情况灵活搭配，具体分析如下：

1. **整果**：适合有耐心、喜欢挑选新鲜度的顾客，且能减少人工成本。建议作为主要陈列，让顾客自行挑选（可提供现场削皮服务）。
2. **去皮好的**：适合追求方便、赶时间的顾客，或客流高峰时段。可切块装盒或保鲜膜包裹，定价稍高，但需注意控制削皮时间和冷藏条件，防止氧化变色。
3. **让顾客自己挑**：这是最传统做法，能提升顾客参与感和信任度。但需配备削皮工具和指导，避免因削皮困难导致投诉。

**实操建议**：
- 同时提供整果（自行挑选）和去皮切块装盒（明码标价）两种选择，覆盖不同需求。
- 整果区域放削皮刀或免费削皮服务提示，去皮区域用透明盒展示新鲜度。
- 根据销量动态调整：如果去皮产品一天内未售完，可降价促销或做成果汁，减少损耗。

最终目标：减少损耗、提高周转率，同时让顾客感觉物有所值。
