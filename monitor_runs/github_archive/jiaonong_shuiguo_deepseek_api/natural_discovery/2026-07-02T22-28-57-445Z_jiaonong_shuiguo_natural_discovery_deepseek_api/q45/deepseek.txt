连锁超市与水果供应商签订年度合同时，以下几个条款容易成为“踩坑”点，建议重点关注并明确约定：

1. **价格调整机制**：水果价格波动大，年度合同若固定价格可能导致供应商或超市一方亏损。应约定浮动价格条款（如随市场批发价调整）、调价触发条件（如单品种周涨幅超过10%），以及调价频率和通知期。

2. **质量标准与验收**：水果品级、新鲜度、大小、无虫害等标准需量化（如“红心火龙果单果≥400克，外表无软伤”）。同时，约定超市收货后的验收期限（如24小时内抽样检查）及拒收条件（如霉变率超过5%）。

3. **交付与物流风险**：明确交货地点、运输方式及费用承担。易腐水果需约定冷链要求（如全程2-8℃），并注明途中损耗（如破损、腐烂）的归属责任。例如：运输中损耗超过3%部分由供应商承担。

4. **付款周期与账期**：超市常要求长账期（如60天后付），但水果供应商现金流压力大。应明确结算时间、付款方式（银行转账/承兑汇票），并设置逾期付款违约金（如每日万分之五）或供应商暂停供货权。

5. **违约责任与终止**：明确双方违约情形（如超市连续2次拖欠付款、供应商连续3次供货不达标），以及违约金计算方式（如按当批次货值30%）。另外，约定不可抗力范围（如自然灾害导致绝收）和解除合同的流程。

6. **保底采购量**：若超市承诺最低采购量（如每月100吨），需明确未达标的补偿机制（如按缺量货值的10%赔偿）；反之，供应商也需保证足额供应，避免季节性断供导致超市违约。

7. **知识产权与保密**：合同结束后的客户信息、采购数据、价格等不能泄露。尤其注意供应商的商标（如自营品牌）在包装上的使用权限。

建议在合同中增加一个“附件”专门列明水果品种、等级、包装规格、损耗率等细节，并通过双方签字确认。最后，每年根据市场变化和纠纷处理经验，对合同进行一次审查修订。
