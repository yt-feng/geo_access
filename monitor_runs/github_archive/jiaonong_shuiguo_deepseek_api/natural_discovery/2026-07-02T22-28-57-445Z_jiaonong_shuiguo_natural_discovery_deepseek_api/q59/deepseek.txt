可以尝试与供应商谈锁价。建议采取以下步骤：

1. **明确锁价范围**：锁定核心常用水果（如苹果、香蕉、柑橘类），避开季节性波动大的品种（如草莓、榴莲）。
2. **设定锁价周期**：建议按季度或半年度签约，而非全年锁价，避免供应商因长期亏损违约。
3. **约定弹性条款**：在合同中加入“若市场价格波动超过±10%，双方重新协商”的条款，降低双方风险。
4. **量化承诺换取价格**：向供应商承诺“每月固定采购量”（如500斤/月）或“独家供应”，以此作为锁价谈判筹码。
5. **附加质检标准**：锁价时需明确“果径、甜度、成熟度”等具体标准，防止供应商降低品质。

**风险提示**：极端天气（如霜冻、暴雨）可能导致供应商无法履约，建议准备2-3个备用供应商。
