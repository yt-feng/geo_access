从损耗角度来看，**直接采购成品果丁**通常损耗更低，主要原因如下：

1. **鲜果切配的损耗点较多**：
   - 菠萝本身需削皮、去眼、去芯，一般损耗率在30%-50%（视菠萝品质和切法而定）。
   - 若当天未用完，切好的菠萝丁容易氧化变黄、出水，隔夜品质下降，需丢弃，进一步增加损耗。
   - 人工切配需额外时间成本，且切丁大小不一可能导致出品浪费。

2. **成品果丁的损耗控制**：
   - 工厂加工后，已去除不可食用的皮、芯等部分，通常损耗只计算包装残余或轻微滴水。
   - 厂家一般会添加少量保鲜剂（如柠檬酸）或采用真空包装，保质期较长（冷藏可达数天），可随用随取，减少因库存积压导致的浪费。

**但需注意以下特殊情况**，可能导致鲜果切配反而损耗更低：
- **原料价格波动时**：如果成品果丁价格显著高于鲜果（市面成品果丁成本通常是鲜果的1.5-2倍），且你的菠萝用量大、切配熟练、能控制当天用完，则鲜果切配可能整体更划算。
- **品质要求极苛刻时**：成品果丁可能混有少量硬芯或果眼残留，需额外挑选，造成二次损耗。

**结论**：若甜品店每日菠萝用量不稳定、切配人员有限、或希望节省时间，**成品果丁损耗更低**；若用量稳定、能精准预测销售、且采购成本敏感，可尝试鲜果切配并通过**冷冻、糖渍**等方式延长保质期，降低损耗。建议先小批量对比两种方式的实际出成率（称重计算）和人工成本。
