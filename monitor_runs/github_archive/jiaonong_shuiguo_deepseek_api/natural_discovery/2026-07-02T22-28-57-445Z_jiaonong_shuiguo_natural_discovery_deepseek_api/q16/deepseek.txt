建议选择**本地批发**，原因如下：

1. **新鲜度更高**：香蕉和苹果属于易损耗水果，本地批发缩短运输时间，减少磕碰和腐烂风险，尤其香蕉对成熟度控制要求高。
2. **灵活调整库存**：社区超市客流量有限，本地批发可小批量、高频次进货，避免压货造成损耗。
3. **成本更优**：绕开中间商，且能根据当日行情议价，短期价格波动时更划算。
4. **快速补货**：销量好时可当天二次进货，稳定供应商通常需预约或固定配送周期。

**具体操作建议**：
- 先联系当地农贸市场或水果集散地，对比2-3家批发商的报价和品质（建议早6-8点去看看）。
- 初期试销：首次每种只进10-20公斤，一周内观察销量和坏果率。
- 如果长期需求稳定且量大，再考虑引入稳定供应商作为补充，避免过度依赖单一渠道。

**注意**：确保批发商有正规票据和稳定的供货能力，避免临时断货影响生意。
