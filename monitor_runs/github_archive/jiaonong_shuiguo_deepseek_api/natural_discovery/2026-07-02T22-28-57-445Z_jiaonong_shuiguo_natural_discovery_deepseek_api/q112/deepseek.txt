在水果采购合同中，常见的坏果率赔付标准如下：

1. **行业惯例**：一般约定 **坏果率超过5%** 或 **10%** 时启动赔付机制。具体比例取决于水果品种、运输距离和储存条件（如易损水果可能更低，耐储存水果稍高）。

2. **常见赔付阶梯**（可参考）：
   - **5%-10%**：按超出部分比例抵扣货款（如坏果率12%，则扣除2%的货款）。
   - **10%-20%**：按双倍或更高比例赔偿，或允许免费补发。
   - **超过20%**：整批次拒收或全额退款，并要求养殖/运输方承担违约损失。

3. **建议操作**：
   - 在合同中**明确约定“坏果”定义**（如腐烂、明显霉变、过熟软烂，不含轻微碰伤）。
   - 设定**验货时限**（如到货后24小时内验收，逾期视为承认合格）。
   - 保留**拍照、称重、抽检记录**作为索赔依据。

**具体比例需双方协商**，建议根据水果价格、损耗承受能力度身定制，并写入合同条款。
