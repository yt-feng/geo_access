判断香蕉的熟度和坏果风险主要依靠感官检查和经验。以下是具体步骤：

**一、判断熟度**

1. **颜色**：青绿色表示未成熟；黄绿色为半熟；全黄说明已成熟或接近食用期；黄带少量褐斑为最佳食用期；大面积褐斑或变黑则过熟或开始腐烂。
2. **硬度**：用手轻捏，注意力度不要用力按压。果身坚实、有弹性说明未熟；稍软且能回弹则为成熟；过软或发黏则过熟或已坏。
3. **棱角**：香蕉有棱角时说明未熟；棱角不明显或变圆润，表示成熟。
4. **果梗**：如果果梗青绿，代表新鲜且后熟不足；如果变黑或干枯，可能存放时间过长。

**二、判断坏果风险**

1. **外表伤痕**：观察是否有挤压伤、擦伤、虫蛀、裂口。破损处易吸引霉菌，是感染腐烂的隐患。
2. **霉斑或白色菌丝**：尤其是果柄连接处、蕉体表面出现水渍状斑点或白色/灰色霉斑，说明已开始腐烂。
3. **异味**：闻一下是否有发酵味、酸味或苦涩味，若有，说明内部已坏。
4. **触摸凹陷**：如果手指轻按某处感觉软烂、凹陷、出液，必定内部腐烂。
5. **挤压与堆压**：成箱香蕉若内部压力过大，容易在后熟过程中一同加快腐烂，需翻看每一层。

**三、实用建议**

- **分批检查**：抽取底层、中间、上层的几串/几根作为样本。
- **注意果柄**：批发时如果果柄已发黑、干枯，可能已经熟过头。
- **关注整体气味**：打开包装袋立即闻到酒味或甜酸味，大概率处于坏果爆发期。

如果以上指标都正常，那么这批货的坏果风险较低。如果发现少量个体有异常，应尽快快速售出或剔除，避免蔓延到整批。
