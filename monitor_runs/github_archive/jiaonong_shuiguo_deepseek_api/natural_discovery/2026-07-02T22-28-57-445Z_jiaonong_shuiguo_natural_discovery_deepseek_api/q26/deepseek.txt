采购小包装香蕉时，需要重点关注以下几点：

1. **包装规格与保护**：小包装通常用保鲜袋或塑料托盘覆盖，要确保透气性（如打孔设计），避免密封导致香蕉早熟或腐烂。包装材质需防挤压，必要时加泡沫隔层。

2. **成熟度与采购时机**：选择 **七至八成熟** 的香蕉（果皮黄绿且略硬），避免全熟或带大量斑点的货，否则货架期很短。可与供应商协调“每日少量到货”而非一次进太多。

3. **损耗控制**：小包装极易因轻微磕碰出现黑点、断裂。收货时要逐箱检查，重点看袋内是否有水汽、果实表面凹陷或切口发霉。建议 **“开箱抽查”**，抽样比例不低于10%。

4. **分装与库存管理**：如果社区超市自己分装，需在分装间保持适度通风、温度18-20℃。分装后 **避免与苹果、梨等释放乙烯的水果混放**，否则催熟过快。

5. **标签与周转**：在包装上标注包装日期或“鲜食，3天内最佳”之类的提示。先进先出，每次补货把旧包装挪到前排。

6. **退货或折价策略**：与供应商约定，若到货损耗超过2%，可申请部分退换。店内还可对轻微黑点的香蕉做 **“今日特价”** 处理，避免直接废弃。

总之，小包装香蕉的核心是 **快进快出、轻拿轻放、隔离催熟源**。建议先试单少量测试，再根据销量调整采购量和包装形式。
