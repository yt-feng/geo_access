可以尝试以下三种组合促销方案，供参考：

**方案一：水果搭配合购（侧重增加购买件数）**  
- **组合方式**：香蕉+菠萝+苹果三样组合，统一标价X元（比单买总和低15%-20%）。  
- **宣传点**：“一周水果包”或“家庭水果篮”，适合一次购买一周量。  
- **货架陈列**：将三样水果放在同一区域，用展示篮或托盘绑定销售。  

**方案二：热带水果冲量组合（侧重提升菠萝/香蕉销量）**  
- **组合方式**：买菠萝送香蕉（比如买1个菠萝送2根香蕉），苹果作为标品维持原价。  
- **宣传点**：“买大送小品热带”或“清爽特惠”。  
- **操作提示**：赠送的香蕉可以选稍有熟斑的，既减少损耗又提升吸引力。  

**方案三：价格阶梯组合（侧重引导高客单）**  
- **组合方式**：购买任意水果满20元，加5元换购菠萝；或购买菠萝+苹果满30元，香蕉半价。  
- **宣传点**：“凑单更划算”。  
- **货架动线**：将组合信息制作成竖版POP（卖点展示牌），放在苹果区，利用苹果作为价格锚点。  

**执行小贴士**：  
- 提前计算三种水果成本与毛利，确保组合价总毛利不低于单卖的80%。  
- 在组合区设置试吃（切好菠萝块+香蕉切片），提升连带购买率。  
- 收银台附近可再放一组小份组合装（苹果2个+香蕉3根+菠萝小半个），针对快速结账顾客。
