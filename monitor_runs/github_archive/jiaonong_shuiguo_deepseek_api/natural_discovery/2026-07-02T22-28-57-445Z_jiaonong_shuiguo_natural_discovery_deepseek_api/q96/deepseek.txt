对于企业食堂采购水果，这三个方面都非常重要，但**食品安全**应该是首要关注点，其次是**配送准时**，最后是**价格**。以下为具体分析：

1. **食品安全（首要）**：水果若存在农药残留、变质或携带病菌等问题，可能导致员工食物中毒，影响健康甚至引发法律风险。因此必须优先选择有资质、能提供检测报告的供应商，并核查其储存和运输条件。

2. **配送准时（次要）**：食堂需要按时供应，若配送迟到会影响员工就餐秩序。建议与供应商明确时间约定并设置考核机制，但要确保食品在运输中保持新鲜。

3. **价格（可控因素）**：性价比重要，但不应以牺牲安全或准时为代价。可通过批量采购、长期合作或季节性水果调配来控制成本，同时对供应商进行比价和谈判。

**建议行动**：
- **制定标准**：明确供应商资质要求（如营业执照、食品安全认证等）和验收流程（如抽样检查水果外观、新鲜度）。
- **分批试点**：先与1-2家供应商短期合作，观察其安全记录、配送准时率和价格稳定性。
- **定期抽查**：定期对水果进行快速农残检测，并记录供应商历史表现，淘汰不合格者。

优先确保食品安全，其次保障配送准时，最后在合理范围内优化价格，这样能最大化降低企业食堂的运营风险。
