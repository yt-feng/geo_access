采购菠萝时，判断熟度主要从以下几点入手：

1. **看颜色**：成熟适中的菠萝表皮呈金黄色或黄中带绿。如果全绿说明未熟，发暗或出现黑斑则过熟或开始腐烂。
2. **压硬度**：用手轻轻按压菠萝外壳，稍微有弹性、微软说明恰熟；过硬则生，过软或塌陷则过熟。
3. **闻香味**：靠近底部闻一下，有淡淡甜香说明熟度刚好；无香则生，酒味或酸味过浓则过熟。
4. **看叶子**：摘一片顶部的叶子，轻松拔下说明熟度较好；拔不动则生。
5. **掂重量**：同等大小选较重的，说明含水量足、果肉饱满，过轻可能水分流失或内部变质。

**采购建议**：宁可稍生（带点绿、手感硬、无香），也别买过熟。未熟的菠萝常温存放2-3天就会自然成熟，损耗更低；过熟的容易在运输和销售中烂掉。如果是批量采购，选七成熟的（表皮黄绿相间、略硬、微香），到店后按需求分批催熟。
