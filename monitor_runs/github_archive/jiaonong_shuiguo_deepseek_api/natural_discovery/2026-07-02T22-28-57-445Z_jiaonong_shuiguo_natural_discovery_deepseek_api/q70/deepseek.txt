针对批发商担心的运输损耗问题，建议在合同中明确以下条款，以降低风险：

1. **损耗标准条款**：约定“运输途中自然损耗率”（如水果类产品常见损耗率为3%-5%），超过该比例的损失由责任方承担。

2. **责任分界点**：明确“货到收货方指定地点且签收前”的坏果责任由发货方/承运方承担，签收后如未及时检验则由收货方承担。可约定“收货方应在到货后XX小时内完成检验，逾期视为货物合格”。

3. **检验与举证**：要求收货方提供坏果照片、视频或第三方检测报告作为凭证，并约定“因包装不当导致的质量问题由发货方负责，运输造成的损坏由承运方负责”。

4. **补偿机制**：明确“超出约定损耗率的坏果，按单价XX元/斤或总价的X%进行折价补偿”，或直接约定“由发货方在下一批次中补发同等价值货物”。

5. **争议解决**：建议优先约定“由发货方所在地法院管辖”，避免异地诉讼成本。

**示例条款**：
> “乙方（收货方）应在货物到达后2小时内完成清点，运输途中自然损耗率不超过4%。若损耗超过4%，甲方（发货方）按超出部分货值的80%补偿乙方。因乙方延迟验货或存放不当导致的额外损耗，甲方不承担责任。”

如需进一步细化，可结合具体货物类型（如易腐生鲜）、运输方式（冷链/非冷链）等调整。
