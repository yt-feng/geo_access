是的，起订量太高确实可能增加损耗风险。具体分析如下：

1. **水果易腐特性**：香蕉和菠萝都属于后熟型水果，保质期较短（尤其香蕉在常温下通常只能保存3-5天）。如果起订量超过实际销售或消耗速度，未及时处理的剩余部分容易过熟、腐烂，造成损耗。

2. **储存条件限制**：这两类水果对温湿度敏感，不当储存（如香蕉不宜冷藏、菠萝需阴凉通风）会加速变质。高起订量可能导致库存积压，进一步增加因储存不当带来的损失。

3. **分拣与处理难度**：批量采购时，水果成熟度可能不均匀（例如整箱香蕉中既有青熟也有过熟的），需要人工分拣。若数量过大，分捡不及时或操作不当，易造成连带损耗。

**建议**：
- 根据实际销量或消耗量设定起订量，预留10%-20%的缓冲余量。
- 优先选择可分批到货或灵活调整的采购条款。
- 对采购批次标注到货日期，按“先进先出”原则管理库存。
- 若必须大量采购，可考虑与供应商协商部分加工（如香蕉切段冷冻、菠萝去皮切块）以延长保质期。

总之，合理控制起订量与库存周转周期是降低损耗的关键。
