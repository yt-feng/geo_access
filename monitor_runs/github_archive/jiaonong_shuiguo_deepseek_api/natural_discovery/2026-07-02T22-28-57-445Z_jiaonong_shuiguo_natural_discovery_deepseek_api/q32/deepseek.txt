判断连锁超市供应商能否全年稳定供货香蕉，建议从以下维度进行考察和验证：

1. **产地与供应周期**：询问供应商的香蕉产区分布（如国内海南、广西、云南，以及菲律宾、厄瓜多尔等进口产区），确认是否有跨季节、跨地区的产地轮换方案。若仅有单一产区（如只依赖海南），则全年供应风险较高。

2. **仓储与冷链能力**：实地考察或调取供应商的冷库容量（香蕉需12-14℃恒温）和冷链运输车队规模。可要求查看近一年的冷库温度记录和运输损耗率（通常在5%以内为佳），判断是否有能力应对产量波动。

3. **历史供货记录**：要求提供过去2-3年（涵盖完整自然年）的月度供货量数据，重点查看是否有连续3个月以上断货或发货量骤降（如下降30%以上）的情况。可随机抽取3个月份，要求提供对应的物流单据或采购合同作为佐证。

4. **合同与保障条款**：在采购合同中明确最低年供应量、延迟交货处罚条款（如每日按货值0.5%赔偿）以及不可抗力（如台风、疫情）下的替代方案。例如，约定若连续2周无法足量供货，需由指定备选供应商补货并承担差价。

5. **财务与抗风险能力**：通过企业公开信息（如天眼查、企查查）查看供应商的注册资本、有无法律纠纷，或要求提供优质金融机构的授信额度证明。资金链稳健的供应商更可能应对价格波动（如香蕉收购价暴涨20%时仍履约）。

6. **田间与加工端考察**：实地走访合作种植基地（至少3个）、分拣包装车间。若供应商自有或长期签约基地面积≥500亩，并配备催熟库（可控制成熟度，延长货架期至7-10天），则稳定性更强。

7. **异常情况测试**：在淡季（如北方冬季）试要求连续4周小批量（如每周100箱）准时交货，观察其响应速度和品质稳定性（成熟度、果柄长度是否一致）。同时，调取该供应商在其他零售商（如地方生鲜店）的供货记录（建议匿名联系咨询）。

若供应商在以上7点中有至少5点达标，可初步判断为稳定供应商。建议先以3年或以上长期合同锁定，并保留每月评估、季度复盘的权利，以防中期风险。
