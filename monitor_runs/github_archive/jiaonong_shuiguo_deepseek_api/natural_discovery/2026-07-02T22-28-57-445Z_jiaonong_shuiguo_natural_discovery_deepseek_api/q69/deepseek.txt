针对批发商给小店供货的香蕉和菠萝分级报价，以下是一套清晰、可操作的建议，结合市场常见标准，帮助您高效沟通并减少纠纷：

### 1. 香蕉分级报价
**主要依据：成熟度、外观、大小**

| 等级 | 名称 | 关键特征 | 参考报价方式 |
| :--- | :--- | :--- | :--- |
| **A级** | 精品/一级果 | 果皮金黄、无黑斑、无机械损伤，果形直，大小均匀（单根重约150-200g） | 按**斤**报价（如：4.5元/斤），或按**箱**报价（注明箱毛重/净重） |
| **B级** | 二级/通货果 | 允许少量轻微擦伤、少量褐斑，果形稍有弯曲，大小不均 | 按斤报价（如：2.8-3.2元/斤），或按**箱**报价（提示“可能有1-2根瑕疵”） |
| **C级** | 次果/特价果 | 较多黑斑、轻微腐烂（切掉可售）、果皮严重擦伤 | 按**斤**或**筐/袋**报价（如：1.5元/斤，或30元/筐约20斤），需明确“当日销售” |
| **特殊类** | 熟度区分 | 如青蕉（待催熟）、黄蕉（可即售）、过熟蕉（需特价） | 必须注明**成熟度天数**（如：青蕉3天后熟，黄蕉今天可上架） |

**话术示例**：
> “A级香蕉4块5一斤，一箱大概28斤，单根个头匀称，适合放前台卖。B级3块2一斤，可能有几根带伤，但切掉不影响，适合做特价。”

### 2. 菠萝分级报价
**主要依据：成熟度、果形、剥皮处理**

| 等级 | 名称 | 关键特征 | 参考报价方式 |
| :--- | :--- | :--- | :--- |
| **A级** | 精品果 | 果形端正（圆润或长筒形），无裂纹、无腐烂，果眼平滑，重量≥2斤/个，成熟度8-9成（香味浓） | 按**个**报价（如：12元/个），或按**斤**报价（如：5元/斤） |
| **B级** | 通货果 | 果形略微不正，果眼较深，有轻微划痕或日灼斑，重量1.5-2斤/个 | 按个报价（如：8元/个），或按**筐**报价（注明筐内个数与总重） |
| **C级** | 处理果/特价果 | 有小裂纹、局部软烂（可切掉）、有虫眼，适合做“切块销售”或熟食加工 | 按**斤**报价（如：2元/斤），或按**筐**报价（如：40元/筐约30斤），明确“需当天处理” |
| **剥皮服务** | - | 可提供去皮、切块（如半成品） | 额外加收**1-2元/斤**（视损耗率），并注明“当日必售，不可隔夜” |

**话术示例**：
> “A级菠萝12块一个，果形漂亮，保证个个香味足，适合礼盒或高价区。B级8块一个，有些果眼深，但削皮不影响口感，适合散卖。C级的2块一斤，可以推荐给做果汁或切盘的小店。”

### 3. 通用报价技巧（避免纠纷）
- **明确计量单位**：永远写清楚“斤/公斤/个/箱”，且附上**净重**（如：箱约28斤，非毛重）。
- **标注损耗率**：对B/C级果，主动说明“有5%-10%的损耗需剔除”。
- **到店验收标准**：口头告知“到货后2小时内验货，有质量问题拍图退差”。
- **临时调价预警**：如遇天气、运输影响，可提前说明“今日价格浮动不超过5%”。
- **拍照留档**：给每个等级拍1-2个实物照片，发到微信群或报价单中，避免口头描述差异。

### 4. 简易报价单模板（发给小店）
> **今日香蕉/菠萝报价（202X年X月X日）**
> 
> **香蕉**：  
> - A级：4.5元/斤（精品箱，净重28斤/箱）  
> - B级：3.2元/斤（通货箱，允许少量伤）  
> - C级：1.5元/斤（特价果，当日售完）
> 
> **菠萝**：  
> - A级：12元/个（精选大果，≥2斤）  
> - B级：8元/个（中果，有轻微划痕）  
> - C级：2元/斤（瑕疵果，适合切售卖）
> 
> *注：所有价格含运费，到店后2小时内验货，有问题拍图退差。*

通过**分级、计量、时效**三要素明确报价，小店能快速匹配自身需求（高价区、特价区、加工区），减少退货和沟通成本。建议首次合作时，专程送一个等级实物到店示范，后续微信群报价即可。
