在超市选购菠萝时，建议按以下优先级综合考虑：

1. **甜度（最核心）**  
   - 可通过观察颜色（金黄色通常更甜）、按压弹性（稍软有弹性说明熟透）、闻底部香气（浓郁甜香为佳）判断。  
   - 避免青绿色或过硬、无香味的菠萝，通常酸涩。

2. **外观（判断新鲜度）**  
   - 选择果眼大而平、叶片翠绿挺拔、无霉斑或软烂的菠萝。  
   - 避免表皮有黑斑、裂口或挤压痕迹。

3. **大小（影响性价比）**  
   - 同等甜度下，大菠萝果肉占比更高，但需结合价格判断，中小果可能更实惠。

4. **切果损耗（针对预切菠萝）**  
   - 若购买切好的，注意看果肉是否发暗、出水，边缘干枯或变色说明不新鲜。  
   - 整果自己切更可控，损耗主要在果皮和果眼，熟练后约损失30%。

**执行建议**：优先用手感、颜色和气味判断甜度与新鲜度，再结合价格与切果损耗做最终选择。
