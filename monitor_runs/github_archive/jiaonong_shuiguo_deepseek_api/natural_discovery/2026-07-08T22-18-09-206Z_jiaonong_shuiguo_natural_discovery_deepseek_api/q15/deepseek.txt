当水果店遇到坏果需要与供应商结算或赔付时，通常遵循以下步骤与原则。请根据具体情况（如合同约定、行业惯例、原因归属）灵活处理：

1. **确认坏果原因与责任归属**
   - 运输途中损坏：通常由物流方或发货方承担（需有照片、签收记录等证据）。
   - 到货后储存不当：如店家未按标准冷藏，责任在自身。
   - 供应商发货时变质：供应商需承担主要责任。

2. **检查合同或采购协议**
   - 是否有明确的“质量验收标准”“赔付比例”“损耗率”等条款。
   - 常见约定：坏果率超过5%-10%时，供应商按比例折价或全额赔付。

3. **处理流程**
   - **立即拍照/录像**：清晰显示坏果、包装、批次、时间。
   - **保存证据**：保留发货单、运输单据、验收记录。
   - **通知供应商**：在约定时限内（如24-48小时）联系，说明数量与程度。
   - **协商方案**：
     - **补货**：供应商在下次发货时补足同等品质数量。
     - **抵扣货款**：下次结算时直接从应付货款中扣除对应金额。
     - **退款**：针对该批次坏果部分退还现金。
     - **按比例结算**：例如坏果30%，则只支付70%货款。

4. **具体赔付计算参考（无合同约定时）**
   - 坏果率 ≤ 5%：一般视为正常损耗，供应商不赔，但可友好协商。
   - 坏果率 > 5%：可按以下方式计算：
     - 赔付金额 = （坏果数量 / 总数量）× 该批次总货款 × (1+运输损耗系数，如0.1)。
     - 或直接要求退还坏果价值部分。

5. **常见注意事项**
   - 不要擅自丢弃坏果，需保留至争议解决后或按供应商要求处理。
   - 如供应商不认责，可要求第三方（如批发市场管理方、电商平台客服）介入调解。
   - 对反复出现质量问题的供应商，考虑终止合作。

**总结步骤**：拍照确认 → 通知供应商 → 核对合同/行业惯例 → 选择补货、抵扣或退款 → 保留书面记录（聊天/邮件/单据）。
