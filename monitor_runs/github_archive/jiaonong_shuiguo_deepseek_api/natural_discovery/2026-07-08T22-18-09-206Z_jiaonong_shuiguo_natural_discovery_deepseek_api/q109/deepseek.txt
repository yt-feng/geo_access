针对长期合作的水果供应商，建议优先谈判的顺序为：**售后 > 配送 > 账期**。理由如下：

1. **售后（最优先）**  
   水果易腐烂、损耗率高，且品质波动大。如果售后条款不清（如坏果率、退换货标准、赔付时效），可能导致长期合作中频繁出现争议和隐性成本。优先明确：  
   - 损耗率上限（如超过5%由供应商承担）；  
   - 退换货流程（物流损坏、品质不符的处理时限）；  
   - 赔付方式（退款、补货或下次抵扣）。  
   *好售后能直接降低你的风险，避免“便宜没好货”。*

2. **配送（次优先）**  
   水果对时效和储存条件敏感。配送条款决定能否保证新鲜度和减少二次损耗：  
   - 配送频次（是否按需灵活调整）；  
   - 冷链标准（温度、包装要求）；  
   - 延迟交货的补偿机制（如减免部分运费或免费退货）。  
   *稳定配送能提升你的库存周转和客户满意度。*

3. **账期（可最后灵活调整）**  
   初创期或现金流紧张时，账期重要；但若供应商信誉好，账期可通过长期信任逐步协商：  
   - 首次合作月结，中期视合作量延长至45-60天；  
   - 避免一次性压款过大（如初期试单采用固定结算周期）。  
   *如果售后和配送不牢，账期长反而成为“坏账风险”。*

**总结**：先保障售后和配送的质量，再谈资金流便利。如果供应商无法在核心条款上让步，建议优先考虑备选供应商。
