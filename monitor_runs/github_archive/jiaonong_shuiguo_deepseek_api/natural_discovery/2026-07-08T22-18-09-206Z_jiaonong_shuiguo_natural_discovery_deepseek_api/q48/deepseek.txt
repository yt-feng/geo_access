判断茶饮连锁供应商能否稳定按规格送货，建议重点考察以下环节：

1. **实地审查供应链能力**：要求供应商提供最近6个月的**批次质量检测报告**和**出库记录**，核对规格（如菠萝成熟度、重量、糖度、切口处理等）的稳定性。建议安排品控人员突击检查其分选线，看是否具备自动化分级、温控仓库等保障规格一致的设施。

2. **设定合同约束**：在采购合同中明确“规格达标率≥98%”的条款，约定每次到货随机抽检15%以上，若不合格率超标，设置阶梯式扣款或退货权利。要求供应商缴纳履约保证金（建议为月采购额的10%~20%）。

3. **小批量试单验证**：先下3~5次完整订货周期的试单（如每次200公斤），重点检查：到货时间偏差（建议±1天为可接受）、包装破损率（≤2%）、单果重量波动范围（如±10克内）。试单期间要求供应商提供每个批次的采摘和运输记录。

4. **建立动态评分系统**：对每次到货按“规格符合度、准时率、破损率、反馈响应速度”四项打分（满分100分），连续3次低于80分则直接淘汰。每月将评分结果与供应商同步，要求整改计划。

5. **储备备选供应商**：至少开发2家通过类似规格测试的备用供应商，并签订“应急供应协议”（如48小时内调货500公斤）。主供应商库存周转天数应大于茶饮店安全库存的1.5倍（如茶饮店库存可维持3天，则供应商需保证≥5天发货能力）。

6. **提前锁定产区资源**：要求供应商提供菠萝产地（如海南、徐闻等）的**果园承包合同**或**合作社直采证明**，确保上游源头稳定。大型连锁可考虑与供应商合资建立定制种植基地，按规格参数种植并全产包销。

操作建议：先用Excel或专业采购系统（如准时达、采仓通）建立供应商履约台账，每批次自动生成达标率。当连续3个试单批次达标率≥98%时，再签订长期框架协议。
