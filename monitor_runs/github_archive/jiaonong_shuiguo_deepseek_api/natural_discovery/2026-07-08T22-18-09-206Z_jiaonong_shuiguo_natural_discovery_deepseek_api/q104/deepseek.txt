针对单位采购水果到货品质不一致的情况，建议按以下步骤进行验收和售后处理：

### 一、验收环节（关键）
1. **收货时立即检查**：在签收前，由至少两人在场，随机拆箱抽样（建议抽检比例不低于10%）。重点检查颜色、大小、新鲜度、表面损伤、腐烂率等，与采购合同或样品对比。
2. **拍照/录像取证**：对整体包装、问题水果特写、货单信息、抽检过程进行清晰记录，保留原始包装和物流单据。
3. **区分问题类型**：  
   - **腐烂/霉变** → 判定为不合格品，不可食用。  
   - **擦伤/碰伤**（短时间内可处理的） → 可与供应商协商部分折价。  
   - **规格不符**（如大小、成熟度） → 记录差异比例，留存证据。
4. **出具验收单**：详细填写到货数量、不合格数量、问题描述，并由验收人、送货方共同签字确认。

### 二、售后处理（行动顺序）
1. **立即联系供应商**：在24小时内（尤其对易腐水果）将证据发给对方，明确提出诉求（如补货、退款、折价）。
2. **根据合同条款**：若采购合同中有质量标准和赔付条款，以此为准；若无，可参考《民法典》中关于买卖合同质量不符合约定的处理方式。
3. **协商方案**：  
   - **不合格部分**：要求供应商全额退款或按比例补发等值水果。  
   - **轻微品质问题**：协商按比例折价（如打8折）。  
   - **全部不合格**：有权拒收整个批次，要求重发或终止合同。
4. **书面记录**：通过邮件、微信聊天记录或有签字的书面文件确认协商结果，避免口头约定。

### 三、注意事项
- **不要拖延处理**：水果变质快，超过48小时容易推诿责任。
- **保留剩余样品**：直到问题解决，问题水果不要自行处理或丢弃。
- **建立供应商评价机制**：对多次出问题的供应商，应列入黑名单或减少合作。
- **紧急情况**：若水果已用于单位活动或会议，先拍照取证，再与供应商协商补偿方案。

如果需要更进一步，可让供应商提供该批水果的产地、采摘日期、质检报告等文件，辅助判断是否为合规批次。
