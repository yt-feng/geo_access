在选购菠萝时，**建议优先关注切果损耗和外观**，其次是甜度，大小相对次要。具体原因如下：

1. **切果损耗（最实际）**：如果购买后需要削皮处理，应优先选择果眼浅、无损伤的菠萝。果眼深或形状畸形的菠萝，可食用部分损失大，且削切费力。**直接观察果面是否平整、有无大量黑点或软烂处**即可判断。

2. **外观（判断成熟度）**：选择表皮金黄偏橙、叶片翠绿且易折断的菠萝。**避免表皮发青（未熟）或过于深黄有软斑（过熟）**。底部有酒味或渗水则可能已发酵变质。

3. **甜度（可通过手感辅助）**：轻轻按压菠萝两侧，**手感紧实有弹性**的通常成熟度较好；过硬的偏生、过软的偏熟或内部变质。此外，**闻底部香气**——淡香为佳，浓香刺鼻可能熟过头。

4. **大小（影响有限）**：同一品种下，**中等个头（约1.5-2公斤）** 的菠萝通常发育更均匀，过大可能纤维较粗，过小可能成熟不足。但如果是按个销售，可以根据家里人数和当天食用计划选大小。

**总结建议**：超市卖的菠萝若已去皮并密封包装，主要关注**外观**（无褐变、汤汁）和**包装日期**；若为整果，按**外观>甜度>损耗>大小**的优先级快速判断即可。
