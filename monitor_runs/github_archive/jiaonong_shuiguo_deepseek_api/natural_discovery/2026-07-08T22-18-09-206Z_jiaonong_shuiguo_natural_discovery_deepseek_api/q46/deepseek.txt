建议优先选择**冷冻菠萝果丁**，其次是**鲜果**，最后是**果泥**。原因如下：

- **冷冻菠萝果丁**：开袋即用，无需削皮切块，大大节省人工和备料时间；风味和色泽稳定，适合标准化出品；口感保留较好，能提供咀嚼感，适合果茶。注意选择无添加糖或低糖的品种。
- **鲜果**：风味最自然，但成本波动大，需要每日处理（削皮、去芯、切块），且容易氧化变黑，影响卖相。如果门店出品量大且能快速消耗，可以用鲜果。
- **果泥**：一般含糖和添加剂，用于调节风味和口感，但无法提供果肉颗粒感。如果做“不加小料”的纯茶饮，果泥可选；若强调“有料果茶”，需搭配其他固态果粒。

**执行建议**：先采购冷冻菠萝果丁做测试，对比成本和顾客反馈。若追求高端鲜果概念，可在鲜果稳定供应时推出限时款。果泥更适合作为补味、调色的辅料。
