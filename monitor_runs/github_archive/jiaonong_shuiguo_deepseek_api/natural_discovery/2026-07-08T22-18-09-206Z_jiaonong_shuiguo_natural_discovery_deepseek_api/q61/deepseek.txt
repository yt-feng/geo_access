要判断香蕉一手货源是否稳定供货，建议从以下5个可执行维度考察，优先通过**实地+数据+合同组合验证**：

**1. 要求查验「产地直采证明」与「种植基地合同」**  
- 让对方提供近3个月的**产地采购单据**（包含车牌号、采摘日期、出货量），或与农场/合作社签订的**长期包销协议**。  
- 如果是进口香蕉（如菲律宾、厄瓜多尔），需提供**报关单、检验检疫证明、船运提单**，且关注货柜到港时间是否稳定。

**2. 追踪连续3批货的「到货时效与损耗率」**  
- 要求**试单3批**（每批至少1-2吨），记录：  
   - 从下单到到货的时间（国产香蕉一般2-5天，进口需7-14天）  
   - 到货时**青香蕉的瑕疵率**（机械伤、黑星病、腐烂率是否低于5%）  
   - 催熟后是否出现**炸把、果肉变软**等稳定性问题  
- 稳定货源通常能承诺**到货后7天内损耗≤3%**（自然成熟损耗不计入）。

**3. 考察「冷库与冷链配套」**  
- 稳定供货商必须有**自建冷库**（至少能存放30吨以上），且配备**冷链车**（非保温车）。  
- 要求每周提供**库存周转表**，看是否持续有2-3天的安全库存（避免断货风险）。  
- 雨天/季节交替期，可突击要求对方提供**冷库实时温度记录**（标准青香蕉13-14℃）。

**4. 索要「客户复购记录」与「违约赔偿条款」**  
- 请对方提供3家以上批发档口（特别是你的同类渠道，如农批市场、商超配送）的**连续6个月采购记录**（可隐去姓名，但需显示采购量和日期）。  
- 合同必须包含：  
   - **缺货违约金**：如未按时到货，按货值10%-20%/天赔偿  
   - **质量异常处理**：到货后48小时内可退货或换货  
   - **保供承诺**：约定最低月供应量（如20吨/月），并明确断供后的替代方案

**5. 实地暗访与行业交叉验证**  
- 直接去对方广东、广西、海南等产区基地看：  
   - 蕉园是否连片（稳定供货需200亩以上）  
   - 是否有标准化采收流程（如工具消毒、落梳后立刻进冷库）  
- 向当地果业协会或批发市场“扛杆队”（资深搬运工）打听该供货商的口碑，尤其是**恶意压价或突然断货**的历史记录。

**警惕以下危险信号**：  
- 声称“做一手”但无法提供任何产地出库单  
- 报价长期低于市场均价15%以上（大概率是次品或即将断货的清仓货）  
- 催货时频繁以“台风/船期延误”为借口（全年最多容忍2次）

**最终判断：完成试单+签完含保供条款的合同+现场看过冷库后，如果连续3个月到货准时、损耗可控，则可视为稳定货源。** 首年建议保持2家备选供货商，降低单一依赖风险。
