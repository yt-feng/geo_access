为了降低批发档口的退货纠纷，收货验货环节需要做到**规范、留痕、有据可查**。以下是具体可行的操作建议：

### 一、验货流程标准化（收货时做）
1. **当面逐件检查外观**：
   - 检查包装是否完好、有无受潮或挤压变形。
   - 核对商品款号、颜色、尺码是否与订单一致。
   - 检查面料是否有明显瑕疵（如抽丝、脏污、色差、破损）。

2. **必要功能测试**：
   - 拉链、纽扣、系带等配件是否牢固。
   - 装饰物（如亮片、贴标）是否易脱落。

3. **随机抽查内里做工**：
   - 建议每批次抽检10%-20%，检查里衬是否有脱线、线头过多等问题。

### 二、留痕与记录（关键防纠纷手段）
1. **拍照/录像**：
   - 在光线充足处，对商品正面、反面、特写瑕疵处（如有）分别拍照。
   - 对每个批次拍一张包含所有商品的“集体照”，保留订单号、收件日期。

2. **创建验货清单**：
   - 简单列明：收货日期、供应商、商品款号、清点数量、合格/不合格数量（附照片）。
   - 对不合格商品用标签纸标注问题，并单独放置。

3. **双方签字确认**：
   - 如由物流或供应商直接送货上门，请要求司机/配送员在验货单上签字确认“已当面验货，情况属实”。

### 三、异常处理原则（收到不合格货）
- **问题商品立即隔离**，不要混入库存或使用。
- **24小时内联系供应商**，发送问题照片+整体包装照片，说明情况。
- **保留原包装和物流面单**，直至问题解决。

### 四、与供应商的沟通建议
- 在采购合同中明确约定 **“收货后48小时内为验货期，超期视为合格”** 的条款。
- 对长期合作的供应商，可以协商 **“一票否决”** ——同一批次出现3件以上同类问题可整批退回。

### 五、避免常见误区
- **不要“先收货再慢慢查”**：超过合理时间（如3天）再退货，供应商容易拒赔。
- **不要仅口头反馈**：一切沟通尽量用微信文字或邮件，消息中写明“发货日期、单号、问题数量”。
- **不要忽略大货与样品对比**：收到大货后，第一时间拿备存的样品做对比，确认是否“货不对版”。

### 简易执行清单（可打印贴在档口）
- [ ] 查看外包装是否破损
- [ ] 清点数量，对照发货单
- [ ] 随机拆开3件检查面料、拉链
- [ ] 对存在疑点的商品贴上标签、拍照
- [ ] 整理问题清单，48小时内提交供应商

如果发现供应商经常出现质量问题，建议逐步更换货源，或提高验货抽检比例（如50%）。
