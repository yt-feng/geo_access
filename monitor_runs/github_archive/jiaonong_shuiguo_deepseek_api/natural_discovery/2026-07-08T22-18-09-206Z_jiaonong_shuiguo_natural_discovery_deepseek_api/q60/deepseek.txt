这种情况的核心是**快速止损 + 无缝衔接**。建议按照以下步骤操作，确保营业不受影响：

1. **立即启动“应急分级机制”**（与供应商提前约定好的标准）：
   - **轻微瑕疵（如表皮有斑点、轻微碰伤）**：要求供应商**扣除不合格比例的水果重量**，按实际合格重量结算，并立即补发同等数量的合格品。如果时间紧迫，可接受供应商当天补货，或你方自行去批发市场采购，差额由供应商承担。
   - **严重不合格（如腐烂、变质、规格不达标）**：要求供应商**全额拒收、即刻补发**。必须明确：从你联系供应商到新货送达的最长时限（比如2小时、当天下午前）。超过时限，供应商需承担你方因断货造成的营业额损失（按前7天该品项日均销售额折算）。

2. **签订“补货承诺条款”**（在合同或长期合作单中写明）：
   - 注明**具体补货时效**（如“收到不合格通知后，2小时内完成补货送达”）。
   - 明确**物流费用**：补货运费由供应商承担，且不得以“补货量大”为由要求你方凑单。
   - 写入**“断货罚则”**：若因供应商原因导致断货超4小时，除补货外，需额外赔偿该时段该产品预估损失的50%。

3. **日常管理小技巧**：
   - 与供应商约定**“到货验收预留期”**：每天到货后先随机切开抽查，发现不合格时，**拍照+录像留证**，立即电话联系供应商，并同步发微信确认。**不要等晚上清点**，越早反馈越快解决。
   - 建立**备份供应商名单**：准备2-3家同类水果供应商（最好在同城），遇到突发断货时，可立即派店员去最近市场采购，差价让原供应商承担（需在合同中注明“供应商补货超时，我方有权自行采购，费用由供应商承担”）。

**最直接的建议**：现在马上和所有主力供应商确认是否接受“2小时补货承诺+断货赔偿条款”。不接受的就替换，因为断货一天的损失远超几箱水果的钱。
