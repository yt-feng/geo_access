建议采购方先采取**小批量、多平台试卖**的方式，具体数量可根据店铺日均销量和品类特性参考以下原则：

1. **基础规则**：试卖量 = 该品类日均销量 × 3~5天（常规商品） 或 1~2天（易损耗/高单价水果）。
   - 例如：店铺日均卖出100斤葡萄，新品葡萄可先进货300~500斤试卖。

2. **分品类建议**：
   - **高单价/易损耗**（如车厘子、草莓）：首次进20~50斤，用于线上社群/小程序限量预售，或线下试吃引流。
   - **常规走量果**（如苹果、橙子）：进200~500斤，分到1-2家门店或1个线上渠道测试，观察复购率。
   - **小众/新奇特**（如释迦、黄金百香果）：进50~100斤，以“会员体验装”或“买赠”形式送出，收集反馈再决定是否续订。

3. **关键动作**：
   - 设置**7天试销期**：每天记录销量、损耗率、客户评价，若7天内复购率＞15%可逐步加量。
   - 与供应商协商**滞销退货或补货折扣**（例如：首批试卖损耗超20%可部分退款）。

**示例**：某社区水果店日均卖西瓜30个，夏天试卖新品种西瓜，先进50个（约2天销量），搭配试吃活动，3天后数据显示复购率20%，则快速追加200个订单。
