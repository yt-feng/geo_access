建议向供应商索要以下关键资料，以确保水果符合农残安全标准：

1. **第三方检测报告**：要求提供由具备资质的检测机构（如SGS、华测、谱尼等）出具的近期农残检测报告，重点关注是否包含《食品安全国家标准 食品中农药最大残留限量》（GB 2763）中涵盖的常见水果品种相关指标。

2. **产地证明及追溯信息**：包括明确标注水果产地、种植户或合作社、采收日期、运输记录的证明文件，便于源头追溯。

3. **供应商资质文件**：供应商的营业执照、食品经营许可证或生产许可证，确认其合法经营资格。

4. **生产管理记录**（如可能）：如果是来自规模基地，可要求提供用药记录、施肥记录、病虫害防治记录（包括农药名称、用量、使用日期、安全间隔期），以验证是否合规用药。

5. **合格供应商评估报告**：内部或第三方的供应商评估记录，证明供应商通过农残风险管控审核。

建议在采购合同中明确约定农残超标的法律责任，并保留对每批次抽检的权利。
