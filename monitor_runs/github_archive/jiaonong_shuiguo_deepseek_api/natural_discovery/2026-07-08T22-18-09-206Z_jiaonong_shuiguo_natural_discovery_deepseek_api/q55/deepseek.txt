选购水果拼盘食材时，划算与否取决于**重量单价**和**可利用率**。以下是每款水果的选规格建议：

1. **苹果**  
   - **选规格**：中等个头（约200-250克/个），或按整箱（10-15斤装）购买。  
   - **理由**：大苹果单价通常比小苹果高，但切块后浪费多（核大、皮厚）。中小苹果去核去皮后净果率更高，且单价更低。整箱买比散装便宜10%-20%。  
   - **注意**：挑口感脆、果肉紧实的（如红富士），耐切、不易氧化变色。

2. **橙子**  
   - **选规格**：大果（约250-300克/个）或按整箱（20-30斤装）购买。  
   - **理由**：橙子可利用率高（果皮薄、果肉饱满），大果剥皮后出肉量多，且单价通常比小果只贵5%-10%，但出肉率高出15%-20%。整箱（如脐橙）比散装便宜15%-30%。  
   - **注意**：避免皮厚汁少的品种，选皮薄、沉手的（汁多）。

3. **菠萝**  
   - **选规格**：中等个头（约1.5-2斤/个），或整箱（20-30斤）购买，批量时可考虑去皮切好的冷冻菠萝块。  
   - **理由**：菠萝去皮后净果率约50%-60%（去皮去芯后），小菠萝（1斤以下）去损耗后单价高；超大菠萝（3斤以上）芯大、浪费多。中等大小性价比最高。整箱买比散装便宜30%-40%。  
   - **注意**：选表皮金黄、按压微弹、香气浓的（成熟度高，水分足）。

**综合划算建议**：
- **批量采购**：所有水果整箱买（批发市场或小区拼单），比零售便宜20%-40%。  
- **按拼盘用量计算**：预估每份拼盘需苹果80克、橙子100克、菠萝120克，换算单个水果可切份数来选大小。  
- **避免浪费**：苹果用“切块+泡盐水”防氧化；橙子切瓣或切丁（不用剥皮）；菠萝提前去芯切块。
