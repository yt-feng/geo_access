判断养老院或医院食堂的水果供应商是否合规且稳定，建议从以下4个核心方面入手，每一步都有可操作的检查方法：

### 1. 查验资质合规性（硬门槛）
- **基础证件**：要求供应商提供《营业执照》、《食品经营许可证》（许可范围需包含“水果销售”或“食用农产品”），并核查有效期。
- **健康证明**：所有配送人员应持有有效期内的健康证。
- **产地/检测报告**：每次送货时，供应商应能提供当批次水果的《产地证明》或《食用农产品合格证》。对于高风险水果（如易残留的葡萄、草莓等），可要求定期提供第三方农残检测报告（建议每年至少2次）。
- **合同条款**：合同必须明确食品安全责任，约定如出现质量问题（如腐烂、农药超标等）的退货、赔偿条款及违约责任。

### 2. 评估供货稳定性（数据与能力）
- **日均库存/供应量**：明确供应商每日能稳定供应的水果种类和数量（例如：苹果/香蕉各50公斤/天）。需考察其是否有自建有冷库或合作种植基地，而非仅靠临时批发市场采购。
- **历史配送记录**：要求供应商提供过去6个月为类似机构（养老院、医院、学校等）配送的清单或发票，重点核对配送时间是否准时、品种是否齐全。
- **应急响应机制**：向供应商提出一个“紧急加单”测试（如：今日下午3点前增加50箱当季水果），观察其能否在2小时内回复可执行方案，48小时内完成配送。无法响应或推诿的，说明稳定性不足。
- **运输条件**：配送车辆应为冷链或常温果蔬专用车（非敞篷货车），并配备温湿度记录仪。可要求提供车辆消毒记录。

### 3. 现场与样品考察（实地验证）
- **现场看库房与分拣线**：考察其分拣中心是否有独立的水果仓库，是否做到生熟分开、无鼠迹蟑迹。观察其分拣时是否剔除烂果、清洗包装。
- **随机抽检**：未通知情况下，要求供应商提供3种不同水果（如柑橘、叶菜类除外，因非水果）各1公斤，现场切开检查是否内部腐烂、虫蛀、黑心。同时称重，检查是否有“冰冻增重”或“注水”现象。
- **参考同行口碑**：联系该供应商正在服务的其他养老院或医院食堂，询问其“是否经常缺货、临时涨价、不按约定时间配送”。

### 4. 长期管理机制（持续跟踪）
- **建立动态评分表**：每月对供应商进行打分（例如：重量误差±2%、外观合格率≥95%、准时率100%），连续2个月低于80分的，启动整改或更换流程。
- **预留备选库**：至少储备2-3家通过资质核验的候选供应商，定期（如每季度）更新其资质与报价，防止单一供应商断供时无法衔接。
- **保险保障**：要求供应商购买“食品安全责任险”（额度建议不低于200万元），并在协议中注明。

### 特别提醒：
- **避免仅看报价**：低价供应商往往在水果品质（如不熟、打蜡水、以次充好）或配送稳定性上妥协，建议将“性价比”与“合规稳定性”结合评估。
- **警惕空壳公司**：核实供应商的实际经营地是否与营业执照一致，可使用天眼查等工具查看其法律诉讼记录（尤其是食安纠纷）。

操作建议：先进行一轮书面资质核查，再组织食堂管理、采购、护士长（或营养师）共同进行一次实地考察，最后签订包含试配送期（如1个月）的合同，试配送期内重点观察日常表现。
