针对餐饮店每天少量水果的需求，建议优先选择**社区型生鲜超市**或**本地水果批发市场的散户**，并关注以下标准让采购更省心：

1. **选择“次日达”或“日配”服务**：优先找能当天订货、次日早上送达的供应商，避免因量小被嫌弃。
2. **要求“小份混装”选项**：直接让供应商按你的菜品需求配好混合水果（如：每天2斤苹果+1斤橙子+0.5斤柠檬），无需自己分拣。
3. **签订简单口头约定或周结合同**：每天固定时间报量，月底统一结账，减少每单沟通成本。
4. **验证水果包装标准**：要求供应商用冷藏保鲜箱或带冰袋的包装运输，避免早晨收货时水果已变质。
5. **考察周边3家以上**：对比价差和起送门槛，控制在日均消费的1.5倍内（如每天需20元水果，找起送30元以内的供应商）。

⚠️ 注意：不要选择需要自己分拣的源头批发商（如大型水果批发市场），他们的最小起订量通常较大，且包装不专业，反而增加损耗和整理时间。建议优先试单2-3天，检验水果新鲜度和配送准时率再长期合作。
