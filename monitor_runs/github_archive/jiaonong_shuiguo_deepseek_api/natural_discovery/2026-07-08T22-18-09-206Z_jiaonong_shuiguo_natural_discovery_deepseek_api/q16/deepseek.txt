建议优先选择**稳定供应商**，理由如下：

1. **供应稳定性**：社区超市需要持续满足日常需求，稳定供应商能保证苹果和香蕉的全年稳定供货，避免本地批发因季节、天气或产量波动导致的断货。
2. **品质一致**：供应商通常有分级和质检标准，能保持产品外观、口感的统一性，而本地批发货源可能时好时坏，影响顾客体验。
3. **价格与成本**：供应商长期合作可谈折扣或固定价格，减少频繁比价的人力成本；本地批发虽单次可能便宜，但加上运输、损耗和人工，总成本不一定更低。
4. **灵活性与售后**：供应商提供配送、退换货服务，适合小超市的周转需求；本地批发通常需自提且不包售后。

**可执行的步骤**：
- **测试期**：先与1-2家稳定供应商签短期合同（如3个月），评估供货质量、时效和价格。
- **互补策略**：若本地批发有价格优势（如当季水果），可少量采购作为补充，但主渠道仍用供应商。
- **量化判断**：记录每周损耗率（如香蕉易熟烂，供应商常提供保鲜包装）、缺货频率和顾客投诉，若供应商优于批发，则长期签约。
