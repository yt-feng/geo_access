搭配水果礼盒建议兼顾**色彩搭配、口感互补、寓意吉利和保鲜时长**，以下是四款好卖的搭配方案（按推荐顺序）：

**1. 【经典红橙组合】（苹果+橙子）**
- **搭配**：红富士苹果4个 + 脐橙3个 + 点缀薄荷叶
- **优势**：红橙色系喜庆，苹果寓意"平安"，橙子寓意"心想事成"，耐储存，适合春节、送礼、探病
- **建议售价区间**：35-55元

**2. 【热带风情套餐】（菠萝+香蕉+橙子）**
- **搭配**：金菠萝1个（切半装饰）+ 香蕉2把（青黄各1） + 橙子3个
- **优势**：黄色系鲜明，菠萝象征"旺来"，香蕉寓意"招财"，口感酸甜搭配
- **保鲜注意**：香蕉需单独隔层放置（释放乙烯催熟其他水果）
- **建议售价**：45-65元

**3. 【四喜平安篮】（全品种）**
- **搭配**：苹果2个 + 橙子2个 + 香蕉1把 + 菠萝半颗（切块包装）
- **优势**：品种齐全，适合尝鲜、多口味需求，可打出"四季平安"的宣传
- **关键**：菠萝需用保鲜膜包切口，香蕉用防碰网格包裹，避免挤压

**4. 【高端轻奢款】（苹果+橙子+装饰香蕉）**
- **搭配**：精选糖心苹果3个 + 血橙2个 + 小米蕉/皇帝蕉1串 + 迷迭香插签
- **优势**：差异化品种提升档次，适合高端微商、节日定制
- **建议售价**：80-120元

**销售实操建议**：
- **不要混放香蕉与菠萝**：两者放一起会互相催熟，容易提前变质
- **苹果去蒂**：苹果蒂容易划伤香蕉皮，导致黑斑
- **菠萝预处理**：切块装盒时用盐水浸泡半分钟再沥干，减少酸涩感
- **试销策略**：先做10-15份小范围售卖，根据顾客反馈调整组合比例
- **礼盒包装**：选用红色/金色底衬+提手丝带，可增加10-20元溢价空间

重点优先推荐**方案1（苹果+橙子）**和**方案3（四喜拼盘）**，这两类顾客接受度最高且损耗最低。
