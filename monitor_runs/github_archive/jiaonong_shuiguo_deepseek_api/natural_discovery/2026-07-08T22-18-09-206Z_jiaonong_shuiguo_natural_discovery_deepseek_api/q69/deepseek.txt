为小店供货时，报价清晰、分级简单、执行方便是关键。以下是一套针对香蕉和菠萝的分级报价建议框架，重点强调“可直观判断”和“常见货损”。

**一、香蕉分级报价（建议按“外观+成熟度”分级）**

1.  **A级：精品/即食果**
    -   **标准：** 果指饱满，无严重青头，无烂疤、无黑斑（轻微芝麻点可接受），果身硬挺，七成熟左右。
    -   **报价法：** 报“**元/斤”**或“**元/箱（约XX斤）**”，并注明“保证3天内不烂”。
    -   **适用：** 水果店、精品摊。

2.  **B级：通货/普通果**
    -   **标准：** 允许少量青头（占比不超10%），允许轻微擦伤、裂口（不深入果肉），部分斑点。
    -   **报价法：** 报“**元/斤”**，与A级价差约20%-30%。注明“需消耗快，可能有少量黄蛆或撞伤”。
    -   **适用：** 超市促销、路边摊。

3.  **C级：处理货/熟透果**
    -   **标准：** 全黄或发软，有明显黑斑、裂口、果柄发霉，但果肉未烂。
    -   **报价法：** 报“**元/把**”（或按筐计价）。明确告知“需做促销或榨汁，不退货”。
    -   **适用：** 奶茶店、果汁店。

**标注关键信息：**
-   **净重（去柄）**：避免用“每箱”模糊，必须写清“净重XX斤”。
-   **成熟度**：标注“青黄各半”、“全黄次日吃”、“带绿需放3天”。

**二、菠萝分级报价（建议按“重量+果型+损伤”分级）**

1.  **一级：** 果个均匀（每斤2-2.5斤），果眼浅，叶色鲜绿，无碰伤、无削皮痕迹。
    -   **报价：** **元/斤**（整箱代卖），或“**元/个**”（约X斤/个）。
    -   **卖点：** 现削现卖，损耗低。

2.  **二级：** 果形稍差（不等、歪形），允许少量干花、轻微碰伤（不烂）。
    -   **报价：** **元/斤**（价格低20%-30%）。可标注“需削皮卖，介意勿拍”。
    -   **适用：** 特价篮子。

3.  **三级/切块货：** 发泡果、底部霉烂、机械伤严重、有出水的。
    -   **报价：** 按个计价（**元/个**），或按切块果肉重量（**元/斤**）。明确“切块处理，不保证块数”。

**特别注意（菠萝损耗）：**
-   **“黑心”风险：** 冬季/过熟菠萝易黑心。必须在报价单上注明：“若切开黑心，请当天拍照，认定后按比例退差价（一般不退全款，因可切掉黑心部分）”。
-   **“空心”处理：** 空心菠萝算二级货，一般少扣点秤或直接打8折。

**四、给小店老板的沟通话术（说清楚）**

-   **说分级：** “我给您分A、B、C三档。A档您卖精品，B档您做促销走量，C档您切块或榨汁。每一档我都明码标价，进货时您看货选档。”
-   **说损耗：** “菠萝黑心、香蕉断指这些我按规矩退差价，但您要在到货后**2小时内**验货，拍照片发给我。临时退货我不认。”
-   **说结款：** “这批货，您先付30%定金，卖完结算尾款。或一次性付清，我多给您配**XX斤包损额度**（比如100斤里包3斤烂的）。”

**核心原则：**
-   **能看、能摸、能对比**（实物货与标注等级对应）。
-   **常见问题提前说清**（黑心、熟烂、压伤、果型差异）。
-   **零售店看的是“到手净价”，而非低价**——你报价高一些，但承诺“包切片、包挑拣”，小店老板更愿意拿。
