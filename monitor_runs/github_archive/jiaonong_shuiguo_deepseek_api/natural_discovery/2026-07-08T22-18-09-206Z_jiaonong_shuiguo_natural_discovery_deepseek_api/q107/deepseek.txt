是的，起订量过高可能会显著增加损耗风险，主要体现在以下几个方面：

1. **保质期限制**：香蕉和菠萝属于高呼吸跃变型水果，成熟速度快。香蕉在室温下通常只能存放3-5天，菠萝虽然略长（约1周），但剖开后易变质。高起订量可能导致无法在保质期内完全售出或消耗，尤其是需求不稳定时。

2. **存储与处理要求**：香蕉需避免低温（<13°C）导致冷害，菠萝则需阴凉通风。若库存量超出存储能力，可能导致温湿度控制不足，加速腐烂。例如，香蕉堆叠过密会释放乙烯气体，促使周围果实提前成熟。

3. **销售节奏不平衡**：如果起订量远超过日常或季节性需求量（如夏季销售高峰但实际客流有限），积压的库存会直接转化为损耗。例如，单次采购100箱香蕉但日均销量仅10箱，第5天后剩余部分可能已过熟。

**建议措施**：
- **计算安全系数**：根据历史销售数据或预估需求，设置起订量不超过3-5天实际消耗量的上限，并加入20%-30%的缓冲余量。
- **分级采购**：与供应商协商“阶梯起订量”，例如首次采购低量测试，后续稳定后逐步增加。
- **动态调整**：利用时效性强的促销手段（如“当季特惠”）加速消化库存，或提前与餐饮/加工合作方预定部分果品。
- **包装与配送优化**：要求供应商提供预熟处理（如箱内催熟或延迟发货），分批次到货以延长可用期限。

通过控制起订量在合理范围内，并配套灵活的库存管理策略，可有效降低因过度采购造成的浪费。
