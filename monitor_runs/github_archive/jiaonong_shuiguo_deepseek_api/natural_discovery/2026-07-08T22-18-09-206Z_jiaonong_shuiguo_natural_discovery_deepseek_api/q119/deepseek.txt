选择水果供应商时，最容易忽视的细节包括以下几个方面：

1. **季节性供应能力**：部分供应商在旺季表现良好，但淡季可能断货或品质下降。需确认其全年供应稳定性和缓冲方案（如库存、替代产地）。

2. **分拣与包装标准**：水果在运输中易损，需明确供应商的分级标准（如大小、成熟度）、包装材料（如防撞网、冰袋）和冷链衔接细节，避免到货后损耗过高。

3. **售后条约的模糊地带**：如“坏果包赔”的具体标准（赔付比例、认定时效、是否包括运输延迟导致的变质），以及退货/换货的物流成本由谁承担。

4. **农残与认证的真实性**：要求提供第三方检测报告（注意日期和检测项目），而非仅凭宣传。有机或绿色认证需核对发证机构及有效期。

5. **灵活性与账期**：能否接受小批量试单？紧急补货的响应时间？对账期（如月结、现结）的违约惩罚是否明确？

6. **供应商的产地直采比例**：许多中间商混杂多地货源，口感不一。直接对接产地的供应商通常更稳定，但需核实其自有基地或长期合作协议。

**建议措施**：  
- 索要近3-6个月的发货记录及客户评价（最好非熟人提供）。  
- 首次合作时，要求小范围试单（如1-2箱不同品种），并自行记录物流时长、到货损耗率。  
- 合同中明确“保鲜期”“坏果率上限”“退换货流程”，避免口头承诺。  

稳定生意依赖于长期信任，但初期用细节约束会更可靠。
