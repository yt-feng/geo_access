针对便利店面积小、水果经营的核心痛点，建议采用 **“精选快周转、包装控损耗、动态调库存”** 的精细化策略。具体操作如下：

### 一、品类控制：做减法、选爆品
1.  **选品原则：人气+高效**
    -   **锁定大单品**：只选 **3-5种** 最常见、不需要教育市场、复购率高的水果。如：**香蕉**（走量锚定）、**苹果/梨**（耐储存）、**橘子/橙子**（应季）、**小番茄/黄瓜**（可当蔬菜的低糖选择）。
    -   **放弃“高维护”品类**：不卖榴莲（异味、需现场开）、不卖整箱西瓜（太占地方）、不卖草莓/蓝莓（损耗极高）。
2.  **按SKU优化位置**
    -   将一个水果品种视为一个SKU（库存单位）。建议货架面宽不超过2个标准货架层。采用 **“端架+收银台小篮子”** 组合：爆款放端架，高利润小包装放收银台。

### 二、损耗控制：源头与销售双控
1.  **改变进货模式（最关键）**
    -   **每日小批量**：不要一次进一周的量。与批发商约定“日配”，每天清晨送到。宁可断货1-2小时，也不要积压到第二天打折。
    -   **预包装策略（核心）**：绝大部分水果采用 **分装盒/袋**：
        -   **为什么？** 1. 避免顾客挑拣捏坏（捏坏是损耗大头）；2. 方便定价（统一售价）；3. 便于快速扫码结账。
        -   **包装规格**：苹果、梨2-3个/袋；香蕉2-3根/盒；葡萄/小番茄1盒装。成本约0.1-0.3元/盒，但可减少30%以上损耗。
2.  **动态定价与“鲜度标签”**
    -   **早7点-午2点**：正常价。**午2点后**：对部分品相微瑕的水果（如碰伤、略软）贴 **“今日特价”** 标签（7折）。
    -   **最后防线**：闭店前2小时，对未售完的鲜切水果盘、现榨果汁进行 **买一送一** 或 **5折清仓**。**注意：** 禁止将变质水果混入，宁可报损也不影响口碑。
3.  **报损与统计**
    -   每天记录 **“报损品种和金额”**，每周复盘。如果某个品种连续2周报损率超过8%（便利店建议控制在5%以内），果断下架替换。

### 三、运营辅助技巧
-   **位置与灯光**：将水果区靠近 **收银台** 或 **门口**，提升冲动消费。使用 **暖色射灯**（显色指数Ra>90），让苹果等红黄色水果看起来更鲜亮。
-   **关联陈列**：在水果旁放 **湿巾、纸巾**，方便顾客擦手；在收银台旁放 **切好的水果拼盘**（10-15元/份），适合即食场景。

### 四、避坑清单
1.  **不碰**：进口高价水果（单价高、客群窄）、过度成熟易烂水果（如释迦）。
2.  **必须做**：每天 **早中晚三次** 翻检库存，挑出有瑕疵的立即处理（特价或制作成果切）。
3.  **牢记**：**宁可因缺货少赚2单，也不要多进10斤烂在店里。**

**总结建议：** 从“香蕉+苹果+应季橙”3个品开始，全部采用预包装，观察一周销量与损耗，再决定是否增加1-2个品种。小面积便利店的生存法则不是“全”，而是 **“即买即走、不放坏”**。
