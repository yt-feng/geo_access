与普通商家沟通时，可以把复杂的品种、产地、技术术语替换成他们熟悉的“生活语言”，重点突出“好卖、好做、损耗低”。以下是几个实用技巧：

1. **用“通俗代称”解释品种**  
   直接说“菠萝”太笼统，可以这样拆解：  
   - 不要只说“都乐金菠萝”，而是说“这个**不用挖眼、不用泡盐水**，切开就能吃，新手店员两秒切好。”  
   - 不要只说“徐闻菠萝”，而是说“这个**香味特别冲**，顾客一进店就能闻到，摆摊效果比旁边水果店好三倍。”

2. **用“消费场景”替代产地数据**  
   - 不要只说“产自云南哀牢山”，而是说“**摆摊、食堂、学校小卖部**卖得最火，学生爱买小箱装，一次吃不完不浪费。”  
   - 不要只说“糖度18-20”，而是说“**比你们现在卖的香蕉甜一倍**，下午3点切好试吃，家长接小孩时特别愿意买。”

3. **用“豆腐白菜”比喻操作难度**  
   - 生鲜类：强调“**成熟度看颜色就行**，和挑香蕉一样简单，变黄就熟，不用拿仪器测。”  
   - 加工类：强调“**做奶茶、做拼盘直接切块**，不需要去核削皮，比处理芒果还省事。”

4. **用“老顾客反馈”代替参数说明**  
   - 不要说“抗病性强”，可以说“**隔壁老王卖三年这种货，从来没人说烂果退货**。”  
   - 不要说“保鲜期7天”，可以说“**你们普通冰箱放一周，颜色还和新到的一样**，不怕卖不掉压货。”

5. **给“已验证的利润话术”**  
   直接给定价建议：  
   - “这个菠萝**进价12元，你卖18元**，旁边超市卖22元，顾客绝对抢着买。”（用具体数字降低犹豫）  
   - “香蕉利润低，**这个品种毛利润能到40%**，比香蕉多赚一倍。”（用对比打消换品顾虑）

**关键操作**：  
- 说话时手边放个**普通芒果或苹果**，比划着说“大小就和这个差不多”，更直观。  
- 随身准备几颗样品，直接说“**你切开尝，不好吃不用进货**”，用实物替代信任成本。  

最后一句收尾：“这个星期**隔壁街水果店第一次只订30件，今天回头追了80件**。”——用真实订单代替所有技术参数。
