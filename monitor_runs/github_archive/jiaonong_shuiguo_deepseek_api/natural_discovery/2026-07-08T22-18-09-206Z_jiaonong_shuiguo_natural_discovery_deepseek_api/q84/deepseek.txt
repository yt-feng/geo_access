针对家庭团购水果礼盒，建议以“老少皆宜、颜色搭配、口感互补、不易损伤”为原则。以下是一个推荐组合：

**推荐组合：**  
- **苹果** 4-5个（寓意平安，耐储存）  
- **橙子** 4-5个（颜色鲜艳，富含维C）  
- **香蕉** 1把（约4-5根，需单层放置，避免挤压）  
- **菠萝** 1个（个头较大，需用泡沫网套保护）  

**组合理由：**  
1. **颜色搭配**：红（苹果）、黄（香蕉/橙子）、橙（橙子）、绿（菠萝）交错，视觉喜庆，适合家庭送礼或自用。  
2. **口感互补**：苹果脆甜、橙子酸甜多汁、香蕉软糯、菠萝酸甜爽口，满足不同年龄层需求。  
3. **实用性**：苹果和橙子耐放，香蕉和菠萝需尽快食用，避免一次性囤积过多易腐水果。  

**包装建议：**  
- 用分隔泡沫箱或纸箱，香蕉平放最上层，菠萝单独用厚网套，苹果橙子垫缓冲物。  
- 附上温馨提示卡，标注香蕉、菠萝的成熟度和储存方法（如香蕉勿冷藏）。  

**价格参考**（日常价）：  
苹果约6元/斤、橙子约5元/斤、香蕉约3.5元/斤、菠萝约8元/个（约2斤），总成本约50-60元，团购价可压至40-50元。  

如果需要更低价或更高端组合，可以替换部分水果（如用哈密瓜替代菠萝，用巨峰葡萄替代香蕉），但注意保鲜难度。
