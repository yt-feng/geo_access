建议先从**香蕉**开始。原因如下：

1. **消费者接受度最高**：进口香蕉（如菲律宾、厄瓜多尔产）口感稳定、价格透明，国内消费者熟悉度高，几乎无需教育和推广。
2. **供应链最成熟**：香蕉的冷链要求适中，运输和储存技术成熟，损耗率相对可控。
3. **操作门槛最低**：进口香蕉采购渠道广、品牌多，容易找到稳定供应商；且无需特殊催熟设备（只需常温放置即可自然成熟）。
4. **风险可控**：相比菠萝（易腐烂、需专业催熟）和苹果（品种多、竞争激烈、须快速周转），香蕉的初期资金占用和损耗风险均更低。

**具体步骤建议**：
- 先联系1-2家有资质的进口水果经销商，试销一个货柜（约25吨）的优质品牌蕉（如都乐、佳农）。
- 对店内其他水果做批次销售测试（建议试销2-4周），观察客流、复购率和反馈。
- 若稳定获利，再逐步拓展至菠萝（需增加催熟区）或苹果（需增加仓储和分类能力）。
