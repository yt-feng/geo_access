在水果采购合同中，关于坏果率的赔付标准通常没有统一的法律规定，主要取决于合同双方的约定。但可以参考行业常见做法：

1. **行业参考标准**：一般水果采购合同中，坏果率超过**5%-10%**（视水果品类和运输条件而定）时，通常可以协商或要求赔付。
   - 易腐水果（如草莓、荔枝等）：1%-3%为正常损耗，超过5%可要求赔付。
   - 耐储存水果（如苹果、橙子等）：3%-5%为正常损耗，超过10%可要求赔付。

2. **合同条款建议**：在合同中应明确：
   - **基准比例**：约定一个基准坏果率（如3%），超出部分按比例赔付。
   - **赔付计算方式**：例如，超出部分按单价乘以超出数量赔偿，或者按货值的1.5倍赔付。
   - **检验时限**：约定收货后24-48小时内验收，超出时间可能被视为默认接受。

3. **法律依据**：如果合同未明确，可依据《民法典》第577条（违约责任）或第583条（损害赔偿），以实际损失主张权利。但自行主张时需有详细记录（如照片、视频、第三方检测报告）。

**行动建议**：
- 签合同前：明确写入“坏果率超过X%时，卖方需按Y%比例赔付”的条款。
- 验货时：当场拍照、称重、记录，最好邀请第三方质检机构或保留物流方证明。
- 若已发生纠纷：先协商；协商不成可向当地市场监督管理局投诉或起诉到法院，但需确认合同中没有“排除合理损耗”等不公平条款。

请根据具体水果品类和交易习惯，用上述参考标准调整。如有合同范本或具体数据，可进一步分析。
