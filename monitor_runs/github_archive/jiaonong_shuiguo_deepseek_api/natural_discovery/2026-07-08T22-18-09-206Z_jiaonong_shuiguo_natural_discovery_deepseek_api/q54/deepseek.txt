采购时**应优先关注香蕉的成熟度**，价格作为辅助考量。具体建议如下：

1. **成熟度标准**：选择表皮出现黑色斑点（芝麻斑）或整体呈深黄色、手感微软但未软烂的香蕉。这种成熟度下的香蕉甜度高、香气浓郁，且含水量适中，容易压泥并融入蛋糕糊，能提升蛋糕的风味和湿润度。
2. **价格权衡**：如果成熟度不够（如仍带绿色或过硬），即使价格低，烘焙后也容易导致蛋糕发涩、甜味不足或口感干燥。建议在成熟度达标的前提下，通过批量采购、关注当季价格或与供应商协商来降低成本。
3. **操作建议**：可提前沟通供应商，指定到货时的成熟度要求。如果采购的香蕉过于生涩，可常温放置1-2天催熟后再使用。

优先级：成熟度＞价格，因为熟度直接影响蛋糕品质，而品质不佳会导致顾客流失或原料浪费，长期成本更高。
