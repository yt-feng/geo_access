对于茶饮店来说，**甜度比出汁率更重要**。以下是具体建议：

1. **优先选择甜度高的菠萝品种**（如金钻凤梨、香水菠萝）。高甜度菠萝可以直接减少额外糖浆添加，降低成本，同时提升饮品口感稳定性。出汁率虽影响产量，但可通过调整配方（如增加菠萝果肉占比）弥补。

2. **出汁率作为辅助筛选指标**：如果供货商提供甜度达标的两款菠萝，优先选出汁率适中的（约55%-65%），避免因纤维过多影响口感。极高出汁率（＞70%）可能伴随果肉过软，切块后易塌陷。

3. **实际使用建议**：先试做3-5杯对比测试，记录不同批次菠萝的甜度口感、果肉色泽和耐存放性。通常甜度高的菠萝出成率稳定，饮品复刻率高，比单纯追求出汁率更重要。
