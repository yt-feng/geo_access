建议固定产地和供应批次。具体理由如下：

1. **品质稳定性**：固定产地能确保水果品种、成熟度和风味习惯的稳定（例如台农芒果与海南芒果口感差异明显）。固定批次（如同一果园、同采摘期）可减少因季节、雨水、土壤差异导致的酸甜波动。

2. **可追溯性**：一旦出现口味问题（如过酸、发苦），能迅速锁定批次和产地，调整配料比例或更换供应商，避免影响所有门店。

3. **供应链管控**：与稳定产地供应商签订长期合同，可提前锁定定价和供应量，降低因市场波动导致的断货或涨价风险。同时，可要求供应商提供每批次的糖度、酸度等检测数据。

**实际操作建议**：
- 对核心原料（如柠檬、芒果、草莓）执行“主产地+备选产地”策略（如80%固定产地，20%从其他产地试采备用），既保稳定，又可应对异常天气减产。
- 每次到货做感官测试（糖度、酸度记录），建立批次档案，并与研发团队定期调整配比（例如某批次柠檬更酸时，配方中减少5%的浓缩柠檬汁）。

**注意**：完全固定可能面临单一产地自然灾害或病虫害风险，建议按季度与备选产地做小批量测试，并保存样本数据。
