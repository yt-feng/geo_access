判断香蕉的熟度和坏果风险，批发商可以从以下关键点入手（包含可操作的具体方法）：

1. **看颜色和棱角（熟度判断）**
   - **青绿、棱角分明**：未熟，适合长期运输或储存，但需催熟才可出售。
   - **黄绿相间、棱角稍钝**：7-8成熟，适合一周内销售，是批发最佳状态。
   - **全黄、无棱角**：已熟，2-3天内需售罄，否则过熟。
   - **出现芝麻点（褐斑）或大面积褐斑**：过熟，需当天处理，否则高度易腐。

2. **手感和硬度（坏果风险预判）**
   - 用手轻捏香蕉中部（避开两端）：**过硬**可能未熟；**稍软有弹性**是好果；**局部软塌、渗水**则已内部腐烂。
   - 检查整把香蕉的“底把”（连接处）：如果底把发黑、湿软，往往内部已开始霉变。

3. **切开或碰伤检查（抽样法）**
   - 随机抽取不同位置（箱子中间、底层）的几根香蕉，切开或剥开查看：
     - **果肉白净**：正常。
     - **果肉出现褐色丝条或黑心**：冷害或真菌感染，易快速腐烂。
     - **果皮有凹陷水渍、擦伤**：受伤后果肉会先变黑，是高风险点。

4. **气味嗅觉**
   - 正常香蕉有淡淡清香；**有酒味、酸味**预示发酵（过熟变质）；**霉味**说明已有霉菌，不适合批发。

5. **检查箱体与摆放（物流风险）**
   - 用手机手电筒照箱子缝隙：如果箱底有水、黏湿或泛黄汁液，该箱货坏果率较高。
   - 如果包装内有干燥剂/保鲜袋破裂，且香蕉表面有凝结水珠，加速腐烂。

6. **整体分级建议（可执行阈值）**
   - **高风险拒收**：同一箱中超过5%的香蕉明显变软、渗出液体或发霉、酒味。
   - **中等风险需降价/快速处理**：棱角已无、颜色全黄但没斑点，宜2天内卖完。
   - **低风险可存放**：香蕉青色、棱角分明、无碰伤、箱底干燥。

7. **简记口诀**  
   “看棱看色摸弹性，底把干硬是安心；  
   开箱闻味查水迹，五成软烂立刻弃。”

实际操作建议：不要只检查箱子表层好果，一定要随机抽底层、中层的香蕉剥皮检查，避免卖家“外好里烂”。
