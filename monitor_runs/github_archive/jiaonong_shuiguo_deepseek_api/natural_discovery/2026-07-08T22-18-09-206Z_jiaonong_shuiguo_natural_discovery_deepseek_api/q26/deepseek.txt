采购小包装香蕉时，主要需关注以下包装和损耗问题：

1. **包装防护性**：选择透气性好的包装（如带孔的塑料袋或保鲜膜），避免密封导致香蕉呼吸作用产生的水汽加速腐烂。包装内可加垫薄纸或泡沫垫，减少运输挤压。

2. **包装重量与规格**：确保每包重量统一（如每包500克或1斤），方便标价。香蕉易断，建议按2-3根一组，用胶带固定基部，减少晃动断裂。

3. **成熟度控制**：采购时选择七成熟（果皮黄中带绿、果柄未全黑）的香蕉，避免完全成熟的货品。过熟香蕉在流通中易产生黑斑，损耗率可达20%以上。

4. **损耗预警**：检查果穗切口是否干燥（若渗液说明不新鲜），剔除有机械伤、虫眼的果实。可设置“破损包/特价包”专柜，对轻微碰伤但果肉完好的商品降价处理。

5. **储存与上架**：包装后需在10-12℃下冷藏运输，避免温度波动。上架时避免堆叠超过3层，可用斜坡陈列降低挤压。

简言之，核心是控制成熟度在七成左右、用带孔轻包装减少挤压、对次品降价处理。
