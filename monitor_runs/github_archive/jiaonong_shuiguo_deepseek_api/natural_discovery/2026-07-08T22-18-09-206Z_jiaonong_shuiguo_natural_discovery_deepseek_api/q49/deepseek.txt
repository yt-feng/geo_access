针对菠萝用量突然增大的情况，与供应商临时补货时，建议按以下步骤操作，确保快速、稳定且价格合理：

1. **立即联系确认供应能力**  
   第一时间电话或微信联系供应商，说明需求量（如“本周需追加500斤菠萝”），询问对方现有库存是否足够在1-2天内交货。如果对方库存不足，可要求其协调其他渠道或产地调货，同时明确到货时间（如“最晚后天上午到店”）。

2. **明确临时补货价格**  
   临时加单可能涉及价格浮动，需现场确认是否按原合同价执行。可谈判策略：
   - 如果合作稳定，尝试争取按原价或小幅上浮（如+5%）。
   - 如果对方要求涨价，可回应：“这次量突然增大，希望长期合作优先，后续旺季期间会稳定订单量。” 平衡双方利益。

3. **书面确认订单细节**  
   通过微信或邮件发送补货订单，明确：  
   - 数量（如“追加300斤，分3天每天100斤到货”）
   - 规格（如“约5斤/个，成熟度7-8成，无腐烂”）
   - 交货时间地点（如“每天早8点前送到茶饮店仓库”）
   - 单价及总价（含税、运费），并请供应商回复确认。

4. **约定质量验收标准**  
   临时补货易混入次品，需明确：  
   - “到货后2小时内验收，若发现超过5%的烂果、青果或机械损伤，整批退回或按比例扣款。”
   - 可拍照留证，避免后续纠纷。

5. **协商灵活付款**  
   为降低供应商风险，可提议：  
   - 货到验收后72小时付款（缩短账期）。
   - 或者本批补货先付50%定金，剩余尾款到货后付清。

6. **设定后续补货预案**  
   跟供应商说明：“这波旺季预计持续3周，未来可能每周额外再补1-2次，能否固定每天预留300斤菠萝给我店？” 提前锁定部分产能，避免再次紧急调货。

7. **保留备胎供应商信息**  
   如果当前供应商无法满足需求，立即联系备选供应商（如水果批发市场、其他产区供货商），并告知当前供应商：“另外一家能明天到货，但我优先和您合作。” 给对方压力也留后路。

**关键提醒**：所有沟通尽量留下文字记录（微信/邮件），价格、数量、质量要求、到货时间等核心条款不要仅凭口头约定。同时，旺季期间可提前要求供应商每天预留固定数量，避免下次被动。
