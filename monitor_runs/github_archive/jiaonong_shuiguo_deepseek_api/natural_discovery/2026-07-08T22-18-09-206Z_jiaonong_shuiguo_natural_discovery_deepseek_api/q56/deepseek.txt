为确保茶饮店的食品安全，水果供应商应提供以下证明文件：

1. **营业执照**：证明供应商为合法注册的经营者。
2. **食品经营许可证**：若供应商涉及预包装水果或加工环节，需具备该证。
3. **产品合格证明**：具体到每批次水果的检疫检验报告，证明无农药残留超标、重金属污染等。
4. **产地证明**：明确水果来源地，便于追溯。
5. **绿色食品/有机认证**（如适用）：若水果标称绿色或有机，需提供认证证书。
6. **运输与储存证明**：如冷藏/冷链运输记录，确保水果在适宜条件下运输和储存。
7. **发票或购货凭证**：作为采购凭证，记录交易日期、品种、数量等信息。

建议与供应商签订质量保证协议，明确责任和退换货条款。日常收货时，还需检查水果的外观、新鲜度及保质期，并保留相关票据至少2年以备查验。
