对于企业食堂水果采购，这三个方面都很重要，但**优先级建议为：食品安全 > 价格 > 配送准时**。具体分析如下：

1. **食品安全（最优先）**  
   水果作为直接入口的食品，安全问题直接影响员工健康和食堂声誉。重点关注：农药残留（建议要求供应商提供检测报告）、腐烂变质（采购时检查外观及成熟度）、产地溯源（避免来源不明的水果）。如果食安出问题，可能引发集体事件、法律风险，且补救成本远高于差价。

2. **价格（核心成本控制）**  
   在保证食品安全的前提下，价格影响食堂运营预算。建议：  
   - 根据季节选择当季水果（成本低且口感好）；  
   - 与2-3家供应商比价，但避免单纯压价导致品质下降；  
   - 批量采购可谈折扣，但注意存储周期（易腐水果不宜囤积）。

3. **配送准时（影响服务体验）**  
   虽然重要，但可通过合同约束和备选方案管理。例如：  
   - 约定配送时间窗口（如每日7:00前到达）；  
   - 要求供应商提供备用分拣或运输方案；  
   - 签订违约条款（如延迟1小时扣减费用）。  
   实际中大部分供应商可做到按时配送，但食安风险一旦发生难以补救。

**建议操作顺序**：  
- 先审核供应商资质（食品经营许可证、检测报告、下游客户评价）；  
- 再比较价格（索取报价单并实地查看仓库卫生）；  
- 最后确认配送条款（合同中明确时间、异常处理、赔偿方式）。  

若条件允许，可对同批水果进行快速抽检（如农残试纸），作为长期采购依据。
