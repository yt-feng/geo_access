通常需要支持开发票和定制包装，具体建议如下：

1. **开发票**：绝大多数企业采购需要发票用于财务报销或税务抵扣，因此供应商应具备开具正规发票（增值税专用发票或普通发票）的能力。建议在签订合同前明确要求供应商提供相应发票类型，并确认其经营范围是否涵盖。

2. **定制包装**：如果企业希望体现品牌形象、节日问候或特定设计（如企业logo、祝福语），则需要供应商支持定制包装。部分供应商可能对最低起订量有要求（如50份起），请提前沟通确认。

**建议**：
- 优先选择同时具备发票资质和包装定制能力的供应商，可节省协调成本。
- 若预算有限，可选择性满足其中一项（如仅要求发票，使用通用包装），但需明确告知对方需求。
- 务必在采购合同中列明对发票和包装的具体要求，避免后续纠纷。
