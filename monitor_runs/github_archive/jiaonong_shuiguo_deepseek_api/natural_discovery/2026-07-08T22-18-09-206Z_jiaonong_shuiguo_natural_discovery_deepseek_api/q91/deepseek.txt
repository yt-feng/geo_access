建议选择**小把包装**（通常为3-5根），具体分析如下：

### 1. 核心原因：
- **便于控制购买量**：单根易被挤压碰伤，且消费者购买习惯多为一次吃1-2根；整袋（如5斤装）对即食型便利店来说量太大，容易产生损耗。
- **视觉诱人、周转快**：小把可通过绿色/黄色混搭提升陈列美感，同时每把的重量（约400-600g）刚好匹配单人1-2天的食用量，有利于缩短货架期。
- **适配便利店动销**：相比超市，便利店客单价低，小把单价（约5-8元）符合即时购买心理，而整袋可能因价格偏高导致滞销。

### 2. 操作建议：
- **包装形式**：用带透气孔的保鲜袋或橡皮筋扎成小把，不要过分密封（香蕉需要释放乙烯）。
- **陈列位置**：放在收银台旁或水果货架中层，温度控制在12-15℃（避免冷藏柜低温导致表皮发黑）。
- **库存管理**：每日到货仅上架1-2个批次（共约10-15把），剩余存放在后场阴凉处，避免一次性大量陈列导致严重损耗。

### 3. 替代方案：
- 若上架**单根**：必须用防震托盘独立包装，并标注“特价”，适合早期引流或促销（但损耗率会高30%以上）。
- 若上架**整袋**：只建议在门店入口设堆头，搭配“家庭装”标签，并在袋内垫吸水纸，适合周末或节日。
