水果采购的价格锁定周期通常取决于市场供需状况、水果品种、季节因素以及双方的议价能力。一般来说，常见的锁价期限如下：

1. **短期锁价（1-2周）**：适用于时令性极强、价格波动大的水果（如当季草莓、荔枝、车厘子等）。供应商通常只愿意锁定较短时间，以规避市场风险。

2. **中期锁价（1-3个月）**：适用于常规大宗水果（如苹果、梨、柑橘、香蕉等），尤其在供需相对稳定的季节。批发商或大型采购商可能通过合同锁定2-3个月的价格。

3. **长期锁价（半年至一年）**：极少见，只有在特定合同（如大型连锁超市与大型种植基地或进口商长期合作）中才会出现。这类锁价通常需预设价格调整机制（如随市场指数小幅浮动或固定涨幅），以保障双方利益。

**实操建议**：
- **短保期水果**（如芒果、葡萄）：建议锁价不超过1个月，避免因保质期短导致积压损失。
- **耐储存水果**（如苹果、柚子）：可尝试锁价3个月，但需明确质量标准和付款条件。
- **谈判要点**：锁价时约定“价格包含运输、包装费用”和“如遇不可抗力可协商调整”条款，降低风险。

最终锁价时长需结合您的采购量、付款方式以及对供应商的依赖度灵活协商。建议从小批量短期锁价开始试水，逐步建立信任。
