在超市与水果供应商的常规合作中，常见的账期通常为**30天至60天**。具体天数取决于双方的谈判地位、水果品类、供货规模以及行业惯例：

1. **普遍情况**：对于中小型超市或常规水果供应商，账期一般在**30-45天**。这是行业内最常见的区间，既能保证超市的资金周转，也缓解了供应商的回款压力。
2. **大型连锁超市**：账期可能延长至**60天甚至90天**，尤其是当供应商依赖该超市的销售渠道时。超市会以“占用资金”为筹码争取更长账期，但需注意水果易腐性较强，过长的账期会增加供应商的坏账风险。
3. **时令水果或高周转品类**：例如草莓、樱桃等短保水果，供应商可能要求**15-30天**的较短账期，甚至现金结算，以减少自身资金占压和损耗损失。
4. **首次合作/小供应商**：一般会争取**30天以内**的账期，如果超市强势，可能被迫接受60天，但需在合同中明确逾期付款的违约责任。

**建议**：如果是超市方，可先以近两年的行业均价（如45天）作为谈判起点；如果是供应商，应尽量缩短账期，并提供“现金折扣”作为让步（例如“30天内付款可打98折”）。双方均应关注合同中的“约定账期起算时间”（从收货日还是对账日开始算），这往往比天数本身更关键。
