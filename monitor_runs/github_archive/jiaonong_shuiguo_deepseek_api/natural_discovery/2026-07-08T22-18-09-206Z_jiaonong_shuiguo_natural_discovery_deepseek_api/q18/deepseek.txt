选购香蕉时，可以通过以下方法避免买到太生或太熟的：

1. **观察颜色**  
   - **太生**：表皮青绿色，无光泽，手感坚硬。这种香蕉通常较涩，需放置几天才能吃。  
   - **理想状态**：表皮呈亮黄色，带有少量绿色斑点（“熟度适中”的标志），手感稍软但仍有弹性。  
   - **太熟**：表皮出现大面积黑色斑点或全身发黑，手感软烂，甚至渗出汁水。这类香蕉口感软糯，但需当天吃掉。

2. **触摸手感**  
   - 轻轻捏香蕉中部：  
     - 过硬（无弹性）→ 太生  
     - 软硬适中（轻微回弹）→ 刚好可以吃  
     - 过软（按下去不回弹）→ 太熟或已过熟。

3. **检查果柄**  
   - 果柄（连接香蕉串的茎部）颜色：  
     - 浅绿或带褐色 → 可能刚成熟，适合存放几天  
     - 深褐色或发黑 → 可能熟过头。

4. **看数量与储存需求**  
   - 如果**短时间吃**：选金黄带绿斑、手感偏软的，买回去可直接吃或存放1-2天。  
   - 如果**需要慢慢吃**：选青黄相间的，放室温下2-3天会自然催熟。  
   - **避坑**：不要买表皮有破损、霉斑或水渍的，这类香蕉易腐败。

5. **灵活挑选**  
   - 超市常把不同熟度的香蕉分开放，建议**混合买**（如2根熟度高、3根偏生），分批吃。  
   - 使用**催熟技巧**：如果买到了太生香蕉，可和苹果或梨一起装进纸袋，室温放置2-3天催熟。

**总结**：按颜色、触感、果柄判断；根据自己食用时间选合适的熟度；混合购买更灵活。如果以上方法还是拿不准，直接问超市理货员：“这批香蕉今天熟度如何？”通常能得到参考建议。
