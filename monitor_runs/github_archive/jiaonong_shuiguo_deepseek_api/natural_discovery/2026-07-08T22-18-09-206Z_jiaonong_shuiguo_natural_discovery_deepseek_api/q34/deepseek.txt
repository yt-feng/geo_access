在水果采购中，**损耗对最终毛利的影响通常比价格低更大**。具体原因如下：

1. **损耗的乘数效应**  
   水果的毛利计算公式为：（售价 - 采购成本 - 损耗成本）÷ 售价。损耗不仅直接减少可销售数量，还会增加分摊到每件商品上的固定成本（如仓储、搬运费）。例如：  
   - 采购价降低10%，但损耗从5%升至15%，实际可售商品减少，可能抵消甚至反超降价带来的利润。  
   - 反之，即使采购价稍高，若损耗控制在2%以内，总毛利通常更优。

2. **损耗的隐性成本**  
   - 处理高损耗水果需额外投入人工（分拣、去皮）、保鲜设备（冷链）、促销折扣（临期降价），这些会间接拉低毛利。  
   - 消费者对不新鲜水果产生负面印象，可能影响复购率，长期损害毛利率。

3. **价格低的风险**  
   - 低价原料往往对应更低的品质（如未成熟果实、运输途中易损伤），隐含更高的后续损耗。  
   - 与供应商长期合作时，单纯压价可能降低供货稳定性（如优先供应给高价客户），导致断货或品质波动。

**实际操作建议**：  
- **优先控制关键损耗点**：生鲜水果的损耗是行业核心痛点，建议将70%的精力用于优化运输时效、冷库管理、上架周期（如缩短从采购到销售的时间）。  
- **兼顾价格与品质**：选择性价比高的供应商，而非单纯追求最低价。例如，可接受采购价高3%-5%，但要求供应商提供更好的包装（如防撞海绵垫）或更快的配送服务，以此降低损耗。  
- **利用数据监控**：每日统计不同品类的损耗率（如草莓>10%、苹果<3%），对高损耗产品优先调整采购策略（如增加固态包装、降低单次采购量）。

综上，若只能优化一项，**降低损耗比压低价格更能显著提升最终毛利**。
