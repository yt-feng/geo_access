和酒店早餐供应商谈水果供应合作时，为确保稳定供应和品质，建议重点关注以下几点：

1. **供应稳定性与履约能力**  
   - 明确要求供应商提供**每日/每周最低供货量**及**应急补货方案**（如突发缺货时，24小时内可调配的替代货源）。  
   - 考察供应商的**冷链物流能力**（是否拥有冷藏车、恒温仓库）、**种植基地或上游渠道**（避免依赖单一产地导致季节性断供）。  
   - 约定**特殊天气/突发事件**（如台风、疫情）的响应机制，要求提前2-3天预警缺货。

2. **品质标准与损耗控制**  
   - 制定**分级验收标准**（例如：草莓直径≥2cm、无磕碰斑；香蕉表皮黄绿无黑点），附照片或样品作为参考。  
   - 明确**损耗责任划分**：例如运输途中损坏由供应商承担，酒店储存不当损耗自负；早餐高峰期前建议允许现场退换部分不合格品。  
   - 要求供应商提供**检验检疫证明**（特别是香蕉、芒果等进口水果）以及农残检测报告（建议每月抽检一次）。

3. **定价与结算方式**  
   - 签订**浮动定价协议**：按市场行情（如批发市场月度均价）浮动10%内不变，超出部分双方协商分担，避免单方承担价格波动风险。  
   - 明确**付款周期**（如货到15天后结算），避免账期过长导致供应商积极性下降。  
   - 约定**试用期条款**：前2个月为磨合期，若月退货率超过5%可单方面解约。

4. **包装与配送细节**  
   - 要求使用**可循环周转箱**而非一次性纸箱（减少破损及垃圾处理成本），并标注生产日期、净重。  
   - 明确规定到货时间（如每日早6:00前送达），超时按货值5%扣款，连续3次超时酒店有权终止合同。

5. **长期合作保障措施**  
   - **独家供应协议**可换取更优价格，但需要求供应商优先保障酒店的需求（如旅游旺季时预留20%库存）。  
   - 保留**至少1家备选供应商**，每季度测试其供货能力，避免单一依赖。  
   - 合同中加入**保密条款**（价格、配方、客户信息），避免商业信息外泄。

**建议行动**：首次合作建议签订3个月短期合同，重点观察“到货准时率”和“水果报损率”（控制在3%以下为佳），到期后双方根据数据调整条款。
