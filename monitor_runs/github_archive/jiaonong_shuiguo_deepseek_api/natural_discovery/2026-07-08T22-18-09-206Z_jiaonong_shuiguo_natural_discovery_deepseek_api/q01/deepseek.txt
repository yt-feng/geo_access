判断一个水果供应商是否靠谱，核心在于考察其**供应链稳定性、品质把控能力、售后诚信度**以及**价格透明度**。对于香蕉和菠萝这类热带水果，尤其需注意成熟度、损耗率和冷链运输。

你可以从以下4个维度进行判断，并直接询问对方或现场考察：

### 1. 看货源与库存稳定性
- **香蕉**：是否来自海南、广西、云南或进口（如菲律宾、厄瓜多尔）？能否保证每周稳定供货？是否有自己的催熟库或稳定的冷库？
- **菠萝**：是否掌握季节性产区的切换（如春季海南、夏季广东、秋冬云南/海南）？淡旺季价格波动时能否及时调货？
- **直接问**：“你们在非主产季（如冬天）的香蕉/菠萝从哪里来？库存能维持几天？有没有断货记录？”

### 2. 检验品控与损耗处理
- **香蕉**：要问清到货成熟度（通常按“3-7成熟”分级）。作为水果店，建议要“黄熟度2-3成”的绿色硬果（即耐放、催熟可控）。供应商如果只提供高熟果（易烂）或不给选择，需谨慎。
- **菠萝**：看是否带冠、大小是否均匀。重点问“到货后几天内会变质？是否提供切开检查的样品？” 菠萝的“黑心”和“过熟软烂”是常见问题。
- **直接看**：现场随机打开一箱，检查底下及中间的果品是否与表面一致。如果发现发霉、腐烂、压伤比例超过5%，就是不靠谱信号。

### 3. 评估售后与翻仓机制
- **合同条款**：是否明确“到货后24小时内可验货、有问题的部分按比例退款或补发”？例如香蕉若有10%断裂或黑皮，能否按比例扣款？
- **损耗承担**：保质期短的水果（如香蕉），供应商若完全不承担运输损耗（仅负责到货时外观），风险较高。靠谱供应商会明确“到货时外观腐烂或缺损超过X%负责”。
- **直接问**：“如果一箱香蕉里有3根已经发黑或断裂，你们怎么处理？菠萝切开展示有黑心，退吗？”

### 4. 核查价格与账期
- **价格透明度**：对方是否能清楚列出“到货价x元/斤（含运费和催熟吗）”？如果只报模糊总价、不分解成本，可能隐藏加价或损耗陷阱。
- **账期**：长期合作通常支持“周结”或“月结”，对初次合作可要求“先货后款”或“小额试单”。要求全款预付的供应商，风险较高。
- **避坑**：明显低于市场价（超过10%）的供应商要警惕——可能使用次品果、缺斤少两，或以低价吸引你下大单后转卖劣质货。

### 实操建议：
1. **先试单**：不要一开始就签长期合同。先订同类中最小起订量（比如3箱香蕉、1箱菠萝），检查运输时间、到货品质、售后反应。
2. **实地考察**（如果对方在本地）：看仓库是否有温控（香蕉需要13-15℃冷库，菠萝需7-13℃）、周转是否迅速（堆积过久易变质）。
3. **索要资质**：要求对方提供营业执照、食品经营许可证，以及菠萝的产地证明（尤其是进口菠萝需检疫证明）。
4. **对比3家**：同一周内询价、试单3家供应商，综合对比后，选择有稳定库存、售后主动（非推诿）、且愿意在试买后给你固定优惠价的。

**一句话总结**：靠谱供应商 = 能说明产区+提供多种熟度选择+明确售后标准+支持小额试单+库存动态清晰。先小量验证再长期合作，永远不要轻信口头承诺。
