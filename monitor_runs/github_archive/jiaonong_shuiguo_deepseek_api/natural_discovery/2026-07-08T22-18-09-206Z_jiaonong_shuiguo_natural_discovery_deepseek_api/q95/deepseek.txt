选择学校食堂水果供应商时，建议按以下步骤操作，确保安全、稳定、性价比：

1. **资质审查**：要求供应商提供完整的营业执照、食品经营许可证、农产品检测报告（至少农残检测），优先选择有学校或大型单位供应经验的商家。

2. **实地考察**：检查供应商的仓库或分拣中心，查看储存环境是否整洁、温控是否达标（香蕉需18-25℃催熟，苹果需0-4℃冷藏，橙子需5-10℃），避免长期积压变质。

3. **索要批次检测**：每批水果到货时，要求供应商提供近3天的农残检测报告（可要求自检或第三方检测），重点查验苹果常用农药（如毒死蜱）、橙子保鲜剂（如抑霉唑）。

4. **签订明确合同**：
   - 到货时间：约定每周固定配送时间（如周一、周五上午9点前）。
   - 验收标准：明确水果成熟度（如香蕉青皮留2-3天熟化）、外观无腐烂、直径大小（如苹果直径≥7cm）。
   - 退换条款：24小时内可退换软烂、破损水果，损耗率控制在3%以内。

5. **价格比选**：建议至少找3家报价，要求按“当季+耐储存”品种报价（如苹果选红富士、橙子选赣南脐橙），避开反季节高价品。

6. **运输条件**：确认供应商使用护栏式堆叠框（防挤压），冷藏车配送（夏季橙子、苹果需6小时内降温至10℃以下）。

7. **应急预案**：要求供应商提供备用替换方案（如遇断供，2小时内能调拨附近市场库存）。

最后提醒：建议优先选择当地有长期口碑、同行推荐的供应商，初次合作可先小额采购测试（如500斤），观察一周的损耗率和学生接受度后再签约。
