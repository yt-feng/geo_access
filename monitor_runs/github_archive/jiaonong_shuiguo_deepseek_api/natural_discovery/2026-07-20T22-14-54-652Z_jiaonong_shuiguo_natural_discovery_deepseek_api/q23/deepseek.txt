通常情况下，供应商可以配合超市的促销活动提供支持，具体包括：
1. **陈列支持**：供应商可能提供货架装饰、堆头设计或端架陈列物料（如挂钩、展示盒等），甚至安排专人协助调整布局。  
2. **海报支持**：供应商可提供促销海报、跳跳卡、价格标签或品牌宣传物料，部分供应商还会承担设计或印刷费用。  
3. **试吃支持**：供应商常提供试吃品（如小份水果样品）或试吃台设备，并派出促销员进行现场推广。  

但支持力度通常取决于以下几点：  
- **合作条款**：需查看合同中是否有明确约定。  
- **促销规模**：大型促销活动（如周年庆）更易获得支持。  
- **供应商能力**：品牌商或经销商资源更充足，中小供应商需提前沟通。  

**建议操作**：  
1. 提前与供应商对接，明确所需支持（如陈列面积、海报数量、试吃时段）。  
2. 签订促销协议，以书面形式确认费用分担（如供应商承担物料费，超市承担场地费）。  
3. 对试吃品要求新鲜度，避免质量纠纷。  

若供应商拒绝支持，可考虑：  
- 用销售数据或到店客流说服对方（如“去年同期促销提升30%销量”）。  
- 调整合作模式，比如缩短促销期或分担部分成本。
