针对茶饮店采购香蕉用于饮品或甜品时，控制熟度的核心原则是：**采购时选择7-8成熟（表皮黄中带绿，手感坚实但稍软），并预留1-2天自然成熟时间**。具体操作建议如下：

### 1. 明确具体用途的熟度标准
- **做冰沙/奶昔/果茶**：需高成熟度（表皮全黄且有少量芝麻点），甜度高、软糯易打碎。建议采购后放置1-2天再使用。
- **做蛋糕布丁/烘焙**：选用全熟甚至略过熟（表皮全黑斑，但果肉未出水变稀），甜度高、易压泥。
- **做装饰切片/凉拌**：7-8成熟（表皮黄绿、果肉稍硬），不易氧化变色，口感脆甜。
- **冷冻保存**：选择成熟度偏低（7-8成熟），去皮切块后冷冻，避免解冻后过熟出水。

### 2. 采购时的挑选技巧
- **看颜色**：避免全青或全熟（全青难熟，全熟易变质）。7-8成熟时表皮呈均匀黄色，带少许青色。
- **摸手感**：轻微按压两端，有弹性但不塌陷。手感硬说明未熟，发软且有凹痕可能过熟。
- **闻气味**：有天然香蕉香，无酒精味（发酵变质信号）。
- **查柄部**：果柄新鲜、无褐变断裂，说明采摘时间短。

### 3. 到货后的库存管理
- **仓储**：常温下（18-25℃）挂放或铺放，避免挤压；**不可冷藏**（低于10℃会冻伤导致发黑），急需缓慢成熟时可短暂冷藏（2-3天）。
- **成熟度调控**：
  - 加快成熟：与苹果或熟番茄同袋密封（释放乙烯），室温放置1-2天。
  - 延缓成熟：单独存放通风处，或放入冰箱冷藏室（5-8℃）短期保存。
- **每日检查**：分“次日使用”“3日内使用”“紧急消耗”三类标记，按批次先进先出。

### 4. 如何应对批量采购的不确定性？
- **分批次采购**：若日均用量大，可采用“早上送熟度偏高、晚上送偏低”模式，交替使用。
- **与供应商约定**：明确要求“偏生”（7成熟）或“可调范围”，避免一次送全熟导致损耗。
- **灵活加工**：若到货过熟，立即做成果泥冷冻；若偏生，转化为冷藏储存或快速催熟。

### 关键提醒
- **避免混合成熟度**：同一批次建议统一熟度，否则可能出现部分过软部分过硬，影响成品口感与出品稳定性。
- **注意食品安全**：表皮破损或流汁先处理，不可混放。

按此标准，采购时以“黄中带青、捏动不含软”为基准，结合门店周转速度（快则高熟度，慢则低熟度），一般能控制在3天内自然成熟至最佳使用状态。
