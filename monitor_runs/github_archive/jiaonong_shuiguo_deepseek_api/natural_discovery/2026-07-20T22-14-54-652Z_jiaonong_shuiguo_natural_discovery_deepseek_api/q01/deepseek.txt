判断水果供应商是否靠谱，可以从以下几个关键点入手，结合实际考察和日常管理：

1. **资质与经营年限**：查看供应商的营业执照、食品经营许可证。优先选择经营3年以上的供应商，可以要求提供其他门店的参考信息。

2. **实地考察与试单**：亲自或派人去供应商的仓库、分拣场地查看。首次合作先试小额订单（比如1-2箱），重点观察香蕉的成熟度、菠萝的新鲜度和包装质量。

3. **水果品质与一致性**：
   - **香蕉**：要求供应未催熟或可控催熟的生蕉，避免到货时已经全熟。看表皮是否完整、有无黑斑或冻伤。
   - **菠萝**：检查顶部叶片是否青绿、果身是否坚硬且无软烂，闻气味是否正常。两者都需要看净重是否足称。

4. **运输与到货时效**：询问发货地和到货时间。靠谱的供应商通常有冷链或稳定的货车直送，能保证到货时香蕉不硬心、菠萝不发热。要求“到货不合格可退货或换货”的书面承诺。

5. **价格与账期**：对比市场平均价，过低的价格往往伴随品质问题。了解是否支持月结或分期付款，以及有无长期合作折扣。

6. **沟通与售后**：合作前测试对方响应速度（如晚上或周末能否联系）。明确坏果（如香蕉撞伤、菠萝霉变）的赔付标准，比如“到货24小时内拍照可补发或退款”。

**日常管理建议**：
- 每次到货后立即记录批次、数量、品质（拍照存档）。
- 建立供应商评分表，按月考核到货准时率、坏果率、售后速度。
- 至少储备2-3个备选供应商，避免过度依赖一家。

如果供应商能稳定提供上述保障，且愿意接受定期考核，通常比较靠谱。
