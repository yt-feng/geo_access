要避免买到太生或太熟的香蕉，可以按照以下方法在采购时挑选：

1. **看颜色**：选择**黄色为主、略带青绿**的香蕉，这种熟度适中，适合当天食用或存放1-2天。完全青绿（太生）或出现大面积黑斑（太熟）的不建议选。
2. **摸硬度**：轻捏香蕉身，**有弹性但不软塌**为佳。太硬（生）或按下有明显凹陷、过软（过熟）的都不要。
3. **观察柄和尖端**：香蕉柄部**完整、干爽**，且**尖端未发黑、未开裂**，说明新鲜度较高。若柄部发黑或尖端有霉点，说明已熟过或存放过久。
4. **看果肉（如果可能）**：超市有时会展示切开的香蕉，正常果肉应呈**白色或浅黄**，无黑化或软烂现象。太生的果肉偏青，过熟的果肉偏透明或褐色。
5. **整串检查**：同一串香蕉中，若**多数香蕉处于偏熟状态**，建议整串不要买，因为易同时熟透。选择整串颜色和硬度比较接近的。
6. **预期食用时间**：若打算**当天吃**，选完全黄色、轻微有芝麻点的；若打算**存放3-4天**，选带青绿、稍硬一些的。

在实际超市采购时，可以结合以上几点逐一观察，大概率能选到熟度合适的香蕉。
