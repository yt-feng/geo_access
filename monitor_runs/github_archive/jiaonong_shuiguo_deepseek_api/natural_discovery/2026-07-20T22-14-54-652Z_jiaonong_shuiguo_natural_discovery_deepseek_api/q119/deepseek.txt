选择水果供应商时，除了价格和新鲜度，以下细节常被忽视但影响长期稳定性：

1. **供应稳定性与季节适配性**  
   需明确供应商的产地覆盖范围、不同季节的补货能力。优先选择多产地（如南北产区互补）或具备冷链仓储的供应商，避免断供风险。

2. **品质分级与售后标准**  
   确认供应商是否有统一的分级标准（如大小、成熟度、损伤率），并明确售后规则（如运输中出现损坏的赔付比例、退货条件）。口头承诺容易产生纠纷。

3. **隐性成本**  
   注意包装费用（如泡沫箱、冰袋）、运输损耗率（通常水果运输损耗3%-5%为合理）、结算账期（压款过多影响现金流）。要求供应商提供总成本明细而非单纯报价。

4. **溯源能力与合规证明**  
   检查供应商是否提供产地证明、农残检测报告（重点关注超标高风险品类如草莓、葡萄）。选择能追溯种植环节的供应商，应对食品安全检查时更有保障。

5. **危机处理机制**  
   询问突发情况（如极端天气、物流延误）时的应急预案。可索要至少2次上月订单延迟或品质问题的处理案例作为参考。

6. **长期合作灵活性**  
   测试供应商对缺货预警、定制化产品（如独立小包装、特殊成熟度）的响应速度。愿意配合试销、小批量调整的供应商更值得长期合作。

建议：首次合作从3-5家供应商小批量试单，对比上述细节的落实情况，建立包含“响应时效、损耗率、投诉处理”的评分表，3个月后淘汰末位。
