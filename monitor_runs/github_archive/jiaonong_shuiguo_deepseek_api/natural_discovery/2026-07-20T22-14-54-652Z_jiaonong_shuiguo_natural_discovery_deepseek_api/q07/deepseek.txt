根据水果店常见的消费习惯和礼盒需求，建议以下3种适合走量的搭配方案：

**1. 经典“平安吉祥”礼盒（60-80元价位）**
- **内容**：6个红富士苹果（红色）+ 6个脐橙（橙色）+ 2根香蕉 + 1个菠萝。
- **理由**：红色苹果代表“平安”，橙色橙子代表“吉祥”，香蕉易损耗但能增加份量感，菠萝提供视觉焦点和香味。整体色彩丰富，覆盖送礼、探病、日常消费需求。

**2. 精致“健康养生”小礼盒（40-50元价位，适合办公室或单人）**
- **内容**：4个苹果（选小果型）+ 4个橙子 + 1根香蕉（切半后用保鲜膜包好）+ 半个菠萝（去皮切块装盒）。
- **理由**：香蕉整根易压坏，切半或去尾能减少损耗；菠萝去皮切块可提高单价和便利性，避免顾客自己处理麻烦。适合不想处理大个水果的上班族。

**3. 豪华“五福临门”大礼盒（100-120元）**
- **内容**：8个大苹果 + 8个大橙子 + 1把香蕉（约6根）+ 1个完整菠萝 + 1个火龙果或猕猴桃（补充颜色）。
- **理由**：增加1-2种高价果（如台湾凤梨或澳芒），提升客单价，并且菠萝和火龙果搭配产生红黄绿对比，陈列吸引力强。

**关键操作建议：**
- **菠萝处理**：如果是带皮整果，建议在顶部切一个“十字”开口并绑上红丝带（增加仪式感）；若去皮，必须用保鲜膜紧贴果肉冷藏，避免氧化。
- **香蕉保护**：香蕉只能放最上层，且用软纸或气泡膜隔开，哪怕多包一层，防止挤压变黑。
- **包装优先选**：带透明开窗的手提纸盒（比塑料篮显高档）或红色竹编篮（节日款）。避免全密封盒，因菠萝和香蕉会散发乙烯，若密封易让苹果、橙子加速腐烂。
- **库存控制**：先测试第一天最多做20盒，观察剩盒情况。香蕉和菠萝易损耗，建议礼盒中只放当天切好的，避免提前备货。
