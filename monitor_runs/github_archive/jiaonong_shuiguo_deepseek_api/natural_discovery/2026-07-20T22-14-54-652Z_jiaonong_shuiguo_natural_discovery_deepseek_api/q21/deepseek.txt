在超市采购水果时，供应商通常需要提供以下检测报告或进货证明，以确保产品符合食品安全和品质要求：

1. **产地证明或供货凭证**：如产地证明、供货商资质证明（营业执照复印件）、进货发票或送货单，以证明水果来源合法、可追溯。
2. **农产品合格证或食用农产品合格证**：部分地区要求供应商出具“食用农产品合格证”，证明水果符合国家食品安全标准。
3. **农药残留检测报告**：常见要求包括有机磷、氨基甲酸酯类等农药残留的检测结果，需由有资质的第三方检测机构出具，且结果须符合国家标准（如GB 2763）。
4. **重金属及污染物检测报告**：如铅、镉、砷等重金属含量检测，依据GB 2762执行。
5. **检疫证明**：如从外省或进口水果，需提供海关出具的《出入境检验检疫证明》或《动植物检疫证书》。
6. **质量等级证明（如需）**：如超市要求水果分级（如A级、B级），供应商需提供产品符合采购规格的证明。
7. **包装标识符合性声明**：如预包装水果，需提供标签符合GB 7718等法规的证明。
8. **冷链运输记录（如需）**：若水果属易腐品类，可要求提供运输温度监控记录，证明全程冷链条件达标。

**注意事项**：具体清单最好与超市的采购质量部门确认，因为不同超市对水果品类的风险控制要求（如有机、绿色食品认证）和频率（如每批次或每季度）可能不同。进口水果还需重点关注报关单、原产地证明及中文标签备案情况。
