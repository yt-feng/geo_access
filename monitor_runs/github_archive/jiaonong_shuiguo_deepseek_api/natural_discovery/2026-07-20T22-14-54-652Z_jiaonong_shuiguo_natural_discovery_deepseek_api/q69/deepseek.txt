为了让批发商给小店报价更清楚，建议按以下方式分级和报价：

**一、香蕉分级与报价**
- **一级（精品）**：外观无黑斑、无磕碰、色泽金黄均匀，成熟度7-8成，适合直接零售或高价销售。报价：**XX元/斤**（如2.5元）。
- **二级（普通）**：有少量黑斑或轻微磕碰，不影响口感，成熟度7-9成，适合尽快销售或促销。报价：**XX元/斤**（如2.0元）。
- **三级（次品/处理）**：黑斑较多、软烂或轻微损伤，需降价或用于榨汁。报价：**XX元/斤**（如1.2元）。

**二、菠萝分级与报价**
- **一级（鲜切/礼盒）**：果形端正、无软烂、叶子翠绿，成熟度8-9成，适合直接零售或加工。报价：**XX元/个**（按个头单重，如大果7元/个，中果5元/个）。
- **二级（走量级）**：果形一般或有轻微伤痕，但内部无腐烂，适合低价促销或批量销售。报价：**XX元/个**（如大果4元/个）。
- **三级（尾货/处理）**：果形差、有轻微软腐或严重疤痕，需去皮切块销售。报价：**XX元/斤**（按去皮后重量，如1.5元/斤）。

**三、提示**
- 分级时明确“成熟度”“外观标准”“包装方式”（如散装、袋装、纸箱）。
- 报价中可注明“不含运费”或“起订量”，避免争议。
- 建议口头或微信沟通时直接说：**“香蕉一级2.5，二级2.0，三级1.2；菠萝一级6元/个，二级4元/个，三级去皮1.5元/斤。”**
