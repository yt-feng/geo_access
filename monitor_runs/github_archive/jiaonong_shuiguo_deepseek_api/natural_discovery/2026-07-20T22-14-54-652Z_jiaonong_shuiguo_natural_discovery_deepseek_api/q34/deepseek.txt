在商超水果采购中，**损耗低对最终毛利的影响通常大于价格低**。以下是具体分析：

1. **毛利计算公式**：毛利 = （售价 - 采购成本） × 销售量 - 损耗成本。价格低主要降低采购成本，而损耗低直接影响实际可销售数量（减少浪费）。
   - 例如：采购100斤水果，进价3元/斤，售价6元/斤。若损耗20%，实际销售80斤，毛利 = （6-3）×80 = 240元。若损耗降至10%（销售90斤），毛利 = 270元，提升12.5%。
   - 若进价降低10%（至2.7元/斤），但损耗20%，毛利 = （6-2.7）×80 = 264元，仅提升10%。可见，降低损耗的边际效益更高。

2. **水果特性**：水果易腐、保质期短，高损耗会直接减少可售卖数量，且损耗成本（如丢弃、打折处理）会侵蚀利润。而价格低可能需牺牲品质，反而加剧损耗风险（如消费者因品质差而放弃购买）。

3. **实际案例**：超市中，优质但价高的水果（如进口车厘子）通过低损耗（合理库存、冷链管理）往往比低价但易损耗的普通水果（如香蕉）贡献更高毛利。

**建议**：采购时优先选择品质稳定、耐储存的水果（控制损耗），其次才是谈价。同时，优化库存周转和保鲜措施，比单纯压价更有效。
