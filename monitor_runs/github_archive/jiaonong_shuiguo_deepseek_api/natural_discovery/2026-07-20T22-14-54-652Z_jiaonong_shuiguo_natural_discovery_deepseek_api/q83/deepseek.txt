对于生鲜电商水果到货后出现坏果的责任划分，通常依据以下原则和步骤进行判断：

1. **合同责任方**：根据《电子商务法》和《消费者权益保护法》，生鲜电商作为销售方，对交付的商品质量负有首要责任。如果坏果属于发货前已存在的质量问题（如不新鲜、有虫眼、腐烂），责任在电商。如果运输过程中因包装不当导致损坏，同样由电商承担责任。

2. **物流风险转移**：在商品交付给顾客前（例如电商自营物流或合作快递签收前），商品风险通常由电商承担。一旦顾客签收，风险转移至顾客。但如果是快递员未经本人允许而放入快递柜或代收点，导致顾客长时间未取到货而坏果，责任归属需协商，一般仍偏向电商或物流方。

3. **顾客责任**：
   - 如果顾客在签收后未及时取出、未按保存条件存放（如温度、湿度不当）导致坏果，或超时未投诉视为验收合格（常见自营电商规定24-72小时内可处理），责任在顾客。
   - 恶意退货或虚构坏果情况，电商可以拒绝。

4. **常见操作建议**：
   - **电商自查**：发货前是否检查果实品质、包装是否防碰撞、是否跟踪温度链（如冷运）。
   - **顾客行动**：拿到货后立即开箱检查，发现坏果第一时间拍照（含运单号、坏果特写、外包装破损情况），并在规定时限内联系客服。
   - **责任划分**：若坏果率超过合理范围（比如超过5%），且顾客在规定时限内投诉，电商通常应承担全部责任并提供补发、退款或赔偿。

5. **争议解决**：
   - 优先与客服协商，提供证据。
   - 协商不成，可向电商平台投诉（如淘宝、京东等），平台可依据规则判定。
   - 金额较大或涉及食品安全，可向市场监管部门投诉（12315）或依法诉讼。

总结：在没有特殊免责条款（如“不包熟度”等）的情况下，**常见的坏果责任首先落于销售方（电商）**，除非顾客能证明自己有过错（如签收后不当保管、超时未投诉）。建议顾客保留证据并及时维权，电商应优化品控和物流机制。
