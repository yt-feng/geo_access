针对茶饮店旺季菠萝用量激增的情况，建议按以下步骤与供应商协商临时补货，确保及时稳定供应：

1. **预估缺口并评估可行性**  
   先核算当前库存、日均消耗量（如旺季销量×配方用量），明确需要补充的菠萝数量（例如：未来一周需多进200公斤）。同时确认供应商的产能、库存及物流能力，判断是否能在1-2天内补足。

2. **主动沟通，强调紧迫性与合作诚意**  
   直接联系供应商负责人，说明旺季销量突增，语气诚恳但明确表达需求时间（如“需在48小时内到货”）。若为长期合作方，可提及“后续旺季订单优先考虑贵方”等长期利好。

3. **协商具体条款**  
   - **价格**：若对方提价，可争取‘临时订单享原价’或‘分阶段提价’（例如前500公斤按原价，后续协商）。  
   - **运输**：若对方承担运费，可协商加急配送（如要求“24小时内加急配送”）。  
   - **质量**：约定临时补货的菠萝质量不低于正常订单标准（如“无腐烂、成熟度达标”），并约定退货/换货条件。  
   - **结算**：临时补货款可争取“到货后30天结清”，或预付30%定金。

4. **签订临时补充协议**  
   通过微信、邮件或合同确认补货的品名、数量、单价、总价、最晚到货时间、验收标准及违约责任（如“逾期按总价1%/天赔偿”）。保留签字或聊天记录作为凭证。

5. **同步准备备用方案**  
   要求供应商提供2-3个备选补货时段（如“若周一无法到货，周三能否补足”），同时询问其他供应商或批发市场是否可紧急调货（例如联系2-3家同行拆借或从本地农批市场加急采购）。

**示例沟通话术**：  
“李总，旺季菠萝用量突然涨了40%，目前库存只够两天。能否在48小时内紧急补货200公斤？按原价结算，我们可以在到货后3天内付款。若时间紧张，可安排部分次日下午送达。这次帮我们应急，后续旺季订单会优先维持合作。”  

**风险提示**：  
若供应商无法满足，立即启动备用渠道（如从附近水果市场采购，或让员工分多次少量采购），避免断货影响核心产品。
