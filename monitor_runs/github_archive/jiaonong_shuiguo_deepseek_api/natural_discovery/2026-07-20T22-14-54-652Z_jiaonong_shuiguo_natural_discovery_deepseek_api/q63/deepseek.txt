要判断菠萝哪个规格更容易出货，建议从以下几个角度分析：

1. **观察目标客户偏好**  
   - 如果你是批发给**水果店/超市**：优先选用**标准尺寸（如1.5-2斤/个）**，因为消费者日常购买多为中等大小，便于即买即食，且陈列美观。  
   - 如果是批发给**餐饮/加工厂**：大规格（2.5斤以上）更易被选择，因为做成果切或榨汁时，单颗产量高、成本更低。

2. **参考历史出货数据**  
   - 统计过去1-2周不同规格菠萝的出货量、退货率、补货频次。**出货量最大、退货率最低的规格**即为当前市场易出货的规格。

3. **对比当前市场行情**  
   - 调查周边批发市场（可询问同行或实地走访）同品质菠萝的畅销规格。如果大规格普遍缺货，说明需求大，可侧重；如果小规格供过于求，则需谨慎。

4. **利用价格弹性测试**  
   - 对不同规格设定接近的价格（例如小规格贵5%-10%，大规格便宜5%），观察销量变化。客户更易接受**低价偏大**的规格，因为性价比高（比如同样20元，选2斤的比1.5斤的常见）。

5. **快速实践验证**  
   - 在初期少量进货时，每种规格试铺1-2天，**重点记录“到货后24小时内被询价/出单量”**。哪种规格回复快、客户直接下单，就是潜在易出货规格。

**核心判断指标**：单位成本下，**每日平均出货量最大、客户补货频率最高**的规格。通常推荐优先试水中等偏大规格（1.8-2.3斤/个），除非有明确数据反向。
