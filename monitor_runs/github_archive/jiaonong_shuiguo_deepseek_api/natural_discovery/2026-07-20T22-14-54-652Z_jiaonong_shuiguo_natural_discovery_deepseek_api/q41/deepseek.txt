采购香蕉时判断催熟能力是否稳定，可从以下方面进行：

1. **观察果实成熟度一致性**：同一批次香蕉的果皮颜色、硬度应接近，避免出现部分黄熟、部分全绿的情况，这反映催熟过程中乙烯浓度或温度控制不均。

2. **检查果柄与果肉状态**：轻折果柄，若脆断且伤口干燥无汁液渗出，说明催熟后熟化均匀；若果柄发软、渗液或霉变，可能催熟过度或储存条件不稳。

3. **测试催熟后口感变化**：取少量样品在常温下放置24小时，品尝甜度与涩味，稳定催熟的香蕉甜度均匀、无涩味或硬心，且果肉软糯不水渍。

4. **查看催熟环境记录**：向供应商索取催熟库温度（通常18-22℃）、湿度（85-95%）及乙烯浓度记录，若数据波动大（如温差超过3℃），则催熟能力不稳定。

5. **做“试催”小样**：从同一批次取样5-10根，放入密闭箱中施加标准浓度乙烯（约10ppm），48小时后检查颜色转变率。若转色不一致（部分绿斑或局部过黄），则催熟工艺或香蕉自身成熟度基础不达标。

6. **观察催熟后货架期**：稳定催熟的香蕉在常温下可存放3-5天。如果24小时内快速长黑点或3天后仍坚硬无变化，说明催熟能力不可控。

建议优先选择有冷链追溯、提供催熟批次记录的供应商，并针对不同季节（夏季催熟快、冬季慢）调整验收标准。
