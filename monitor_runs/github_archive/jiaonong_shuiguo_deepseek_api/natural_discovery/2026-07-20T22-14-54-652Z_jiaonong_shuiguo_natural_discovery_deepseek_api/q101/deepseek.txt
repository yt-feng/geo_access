与酒店早餐供应商谈水果合作时，建议重点关注以下方面：

1. **品质标准**：明确水果新鲜度（如无腐烂、无机械损伤）、成熟度（如香蕉、牛油果等需供应适熟期）、规格（如苹果直径、草莓克重）及外观（颜色均匀、无斑点）。

2. **稳定供应**：要求供应商提供全年供应计划（尤其应对反季节水果），并确认补货响应时间（如紧急订单2小时内到货）及不可抗力（如台风、疫情）的替代方案。

3. **价格与合同**：约定市场波动价格调整机制（如月度报价基准、调价幅度上限），确保透明无隐形费用（如冷链费、分拣费）。合同需明确账期、违约责任（如断货赔偿条款）。

4. **物流与仓储**：确认配送时间（如每日/隔日）、运输温控标准（如冷链车温度范围），供应商需提供到货前水果状态说明（如冷藏还是常温）。若酒店无大型冷库，可要求供应商分批次小批量送货。

5. **食品安全**：核查供应商的资质（如ISO 22000、HACCP认证）、农残检测报告（每批次需提供），以及追溯体系（能否快速定位问题批次产地）。

6. **服务与异常处理**：明确退换货条款（如拒收腐烂或未达标的5%外观问题水果），要求供应商提供促销期或换季前滞销品的灵活处理方案（如更换品种）。

7. **试单与考核**：先进行1-2周小批量试单，评估供应稳定性、品控一致性及应急能力。设定月度KPI（如缺货率低于3%、客户满意度>90%）。

建议前期准备：根据酒店日均早餐客流（如50-200人）测算水果用量（如苹果、橙子、西瓜等基础品种），列出最低起订量，并对比2-3家供应商的农场或产地直供资质（产地直采和批发商报价差异可达20-30%）。谈判时优先锁定主供应商（如供应80%份额），保留一家备选以降低风险。
