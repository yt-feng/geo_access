小商家采购水果时，现金结算和月结账期各有不同风险，具体如下：

**现金结算的风险：**
1. **资金占用**：需要立即支付全款，可能影响短期现金流，尤其对于资金周转紧张的小商家。
2. **质量无保障**：若水果到货后发现质量问题（如腐烂、不新鲜），已付现金后追索困难，只能自行承担损失。
3. **缺少交易记录**：现金交易可能没有正式收据或合同，后续维权无凭证，容易产生纠纷。

**月结账期的风险：**
1. **应收款风险**：供应商可能延迟发货或提供劣质水果，但因已赊账，小商家仍需按时结清，否则影响信用。
2. **信用压力**：若小商家自身经营不善或顾客提货延期，月结可能造成还账困难，进而影响与供应商关系。
3. **隐含成本**：部分供应商会为月结提高单价或收取滞纳金，实际采购成本可能高于现金结算。
4. **合同陷阱**：月结往往需签协议，若条款不清（如账期计算方式、退货条件），容易出现额外费用或纠纷。

**建议：**
- 现金结算：适用于初次合作、供应商信誉不明、水果易损且需快速验货的情况，但应要求当场开具收据并保留证据。
- 月结账期：适合与长期合作、信誉良好的供应商，且最好签订书面合同，明确账期、质量标准和退货条款，同时控制账期时长（如30天内），避免过长风险。
