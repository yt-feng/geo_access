作为商超采购菠萝的策略分析，以下是一些关键考量因素，供您参考：

1. **目标客群与消费场景**  
   - **整果适合**：注重家庭消费的社区超市、周末采购主导的门店，或顾客对新鲜度要求高、愿意自行处理的情况。整果损耗可控，且单价较低。  
   - **切好适合**：商务区、交通枢纽或午间客流集中的门店，顾客追求便利、即时食用。包装好的切果（如盒装菠萝/菠萝切片）能节省时间，但较难控制果品成熟度不一致导致的损耗。

2. **损耗与利润平衡**  
   - **整果优势**：损耗主要来自表皮和内核（约30%），但可转化为榨汁/果酱原料进一步消化。若门店有独立切配区，人工成本可控。  
   - **切好风险**：需快速销售（通常24小时内），未售出需报废。若渠道稳定、客流密集，可通过溢价弥补损耗（通常单价整果的1.5~2倍）。建议测试单店数据。

3. **供应链与操作能力**  
   - 若有冷链分装能力且客流稳定，可引入预切包装产品（如带托盘/气调包装），能延长货架期至3~5天。  
   - 若无专业切配设施，建议只卖整果，避免卫生风险（如微生物污染）与客户投诉。

**建议**：  
- **混合策略**：主销整果（陈列在生鲜区），同时在靠近收银台/即食区设置少量当日切好的包装盒（标注加工时间并搭配试吃）。  
- **优先级**：  
  - 单店日均客流>500人次且周边办公区占比高 → 优先上架切果。  
  - 社区店或客流波动大 → 仅供应整果，搭配一次性削皮工具促销。  

最终可对比试点门店的毛利额和损耗率（整果：切果建议控制在3:1以内）确定最适合的比例。
