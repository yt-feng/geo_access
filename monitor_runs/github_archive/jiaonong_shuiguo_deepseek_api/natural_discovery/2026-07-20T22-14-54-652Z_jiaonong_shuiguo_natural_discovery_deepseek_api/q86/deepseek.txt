生鲜平台做水果一件代发，供应商需要具备以下核心仓配能力：

1. **产地/中心仓能力**：拥有温控或冷藏仓库（根据水果种类，如冷链存储），能进行初步分拣、分级（大小、成熟度、品相）、预冷处理（去除田间热），降低损耗。

2. **包装能力**：具备抗压、防震、透气的包装方案（如珍珠棉、泡沫箱、气柱袋），能按订单进行标准化打包（重量、数量一致），并符合快递运输要求。

3. **订单处理系统**：与平台系统对接，能实时接收电子订单，支持一键打印快递单、出库单，避免错单漏单。

4. **一件代发履约能力**：建立小型分拨线或人工流水线，能按单个订单（非批发）快速拣货、打包、贴单，处理每日几百到上万单的并发需求（如大促期间）。

5. **冷链物流合作**：与顺丰、京东冷链、跨越速运等公司签有协议，能实现水果全程冷链（冰袋、干冰、保温箱）发出，确保新鲜度。

6. **库存与保鲜管理**：按水果货架期（如草莓2-3天，苹果7-14天）动态控制库存，做到先进先出，及时预警临期或腐烂产品，避免劣品流入。

7. **退货与赔付流程**：设立专门的退货处理和售后响应机制，对运输途中损坏、变质的产品能快速核实、补发或退款，减少平台投诉。

建议供应商从前期“选品预冷”和“包装防护”两个短板能力入手，这两条是决定烂果率和客户体验的关键。
