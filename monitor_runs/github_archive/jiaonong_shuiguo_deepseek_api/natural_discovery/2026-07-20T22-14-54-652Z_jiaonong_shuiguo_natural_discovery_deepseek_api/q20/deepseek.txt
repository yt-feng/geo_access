选择社区超市的水果供应商时，建议重点考察以下条件：

1. **供货稳定性**：供应商能否保证全年每天稳定供货，避免断货影响门店运营。尤其关注本地应季水果和长期可供应品种。

2. **品控标准**：明确水果的成熟度、规格、新鲜度要求（如无腐烂、无机械损伤），供应商是否有分级筛选流程（如按大小、甜度分拣）。

3. **价格与结算**：对比批发价是否低于市场均价20%-30%，是否有阶梯价优惠（如量大价低），结算周期是日结/周结/月结。

4. **配送时效**：是否具备冷链物流（夏季防高温、冬季防冻伤），能否在规定时间（如早6点前）送达，运输损耗率控制在3%以内。

5. **售后响应**：如果出现坏果、缺斤少两，供应商是否能2小时内补货或赔付，退换货流程是否清晰。

6. **品种多样性**：是否覆盖常见水果（苹果、香蕉、柑橘）与网红品种（阳光玫瑰、牛油果），能否根据节气推出组合搭配（如夏季西瓜+荔枝）。

7. **资质与合规**：查看供应商的营业执照、食品经营许可证，确认水果是否有产地合格证明或农残检测报告。

**实操建议**：
- 先小批量试订1周，观察实际品质和配送稳定性。
- 要求供应商提供日常巡检服务（如每周到店检查库存损耗并调换临期水果）。
- 签订合同明确违约责任（如连续3天断货可解约并赔偿损失）。
