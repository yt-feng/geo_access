建议采用 **小批量试卖 + 梯度加单** 的策略，具体数字参考如下：

1. **最小试卖量**：建议按 **1-3天** 的销量来备货。如果是社区水果店或小型便利店，首次进货量控制在 **5-10公斤**（或 1-2箱）；中型超市或连锁店建议 **15-30公斤**（或 3-5箱）。
2. **核心原则**：确保试卖期（通常3-7天）内不会因库存过多而变质，同时又能铺满陈列面让顾客看到。
3. **操作步骤**：
   - **首周**：按最小量上架，记录每日销售、损耗、顾客反馈。
   - **次周**：如果日均销售稳定且反馈良好（重复购买率 >20%），可翻倍加单；如果滞销或损耗 >15%，立即停货或调整定价/陈列。
4. **风险底线**：无论预算如何，首次备货金额不应超过你 **日均水果采购预算的30%**，避免资金被新品挤占。

这样既能试探市场接受度，又不会造成大量损耗或压货。
