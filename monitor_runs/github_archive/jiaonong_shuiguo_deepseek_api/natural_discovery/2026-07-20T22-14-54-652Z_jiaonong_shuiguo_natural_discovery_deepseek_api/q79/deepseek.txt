选水果供应商时，最应该优先问**配送**，其次是**售后**，最后才是**价格**。具体原因如下：

1. **配送（最关键）**  
   - 水果时效性极强，配送是否准时、冷链是否到位直接影响品质（如腐烂、磕碰）。  
   - 问清配送范围、时间、频率（如是否每日/隔日到）、最低起送量。  

2. **售后（次关键）**  
   - 水果易损耗，必须有明确的坏果包赔或退换标准（如“48小时内坏果按比例赔付”）。  
   - 确认客服响应速度、售后处理流程（如是否需要拍照留证）。  

3. **价格（最后考虑）**  
   - 只有在配送和售后有保障的前提下，才比较价格。低于市场均价过多的供应商需警惕品质或服务隐患。  

**执行建议**：  
- 直接要求供应商提供至少2个现有团长的联系方式，私下核实配送和售后口碑。  
- 初期小批量试单，观察配送时效和售后处理实际情况。
