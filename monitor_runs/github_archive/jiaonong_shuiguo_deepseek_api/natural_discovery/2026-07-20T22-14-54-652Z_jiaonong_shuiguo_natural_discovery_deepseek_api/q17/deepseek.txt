针对社区超市水果控制新鲜度和库存，核心思路是：**“小批量、高频次、快周转、重展示”**。以下是具体可执行的操作建议：

**1. 精准进货：用数据代替感觉**
   - **按历史销量预估**：统计过去1-2周每天各类水果的销量（比如周一到周五卖多少，周末卖多少），按“预期销量+10%-20%的缓冲量”订货。避免一次进整箱，尤其是易损耗的草莓、蓝莓、葡萄。
   - **小包装采购**：优先选择供应商能提供“小包装、多品类”的进货方式，或与周边2-3家小超市拼单，降低单次进货量，提升周转频率（比如从每周2次进货改为每周3-4次）。

**2. 严格分级：按新鲜度分区定价**
   - **“鲜品区”优先展示**：每天到货后，将最新鲜的一批放在门口、收银台旁或冷柜最外层，标正常价。
   - **“快销区”每日清货**：将存放超过1-2天、品相稍有瑕疵（如苹果有轻微压痕、香蕉表皮发黑斑）的水果，单独设一个“当日特价/打折区”，在晚间客流高峰（如下班时间段）集中打折处理，目标：**当天必须卖掉**。
   - **阶梯式降价**：例如：上午新鲜品原价，下午4点后原价8折，晚上7点后5折或买一送一。设置明确的时间节点，店员按指令操作。

**3. 库存周转：遵循“先进先出”原则**
   - 每批到货贴标签：在包装箱或货架卡上写上到货日期（如“3/15到”）。每次上货时，**必须把旧批次的剩余水果挪到外面**，新货补在后面或下面。
   - 区分耐放与易坏：苹果、橙子、柚子等可放3-7天，进货量可稍大；葡萄、草莓、芒果、香蕉等只能放1-2天，进货量严格控制在1-2天内能卖完的数量。

**4. 保鲜手段：延长窗口期**
   - **温湿度控制**：香蕉、芒果（未熟）不要放冷柜，常温存放；苹果、梨、葡萄、草莓必须冷藏（0-4℃）。可以用超市随手可得的**冰袋+保温袋**给水果底下垫冰（如草莓盒下放冰袋）。
   - **包装调整**：对高损耗品（如散称樱桃、蓝莓）预先用保鲜盒或透孔塑料袋分装成小份（半斤或1斤），减少顾客翻拿造成的损耗。
   - **定期巡检**：每天开门后、下午2点、关门前各检查一次，**发现坏果立即挑出**，避免传染给其他水果（尤其是一袋葡萄里烂一颗会迅速烂其他颗）。

**5. 销售刺激：主动引导消费**
   - **组合推荐**：比如买草莓+蓝莓送芒果，或者“水果拼盘”（把各种剩一点的水果切好混装），用10元/盒的价格处理尾部库存。
   - **会员/社区预售**：通过微信群预告“明天到货新鲜樱桃，预定满5份有折扣”，先锁定部分销量再进货，降低风险。

**6. 库存预警硬指标：**
   - **当日售罄率**：设置核心水果（如香蕉、苹果）目标“当天进货量至少卖出80%”，如果累计3天达不到，则下周减少该品进货量30%。
   - **损耗容忍上限**：规定每种水果的月度损耗率不超过5%-8%（例如苹果卖100斤，烂掉丢掉的不能超过5斤）。超过后要复盘原因（是进太多、放太久还是不够保鲜）。

**一句话总结：** 小批量高频进货，按新鲜度分层定价，当天清掉旧货，坏果随时剔除，用预售和组合降低库存压力。
