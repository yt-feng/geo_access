是的，供应商通常需要支持开具发票和定制包装，具体建议如下：

1. **开发票**：确认供应商能开具正规增值税发票（普通或专用发票），并提前沟通开票信息（企业名称、税号等），确保符合企业财务要求。

2. **定制包装**：如需定制礼盒包装（如印有公司logo、祝福语等），需与供应商确认：
   - 起订量：定制包装通常有最低数量限制。
   - 费用：是否包含在单价内，或额外收取设计/制作费。
   - 工期：预留足够时间避免延误。

3. **其他注意事项**：
   - **水果品质**：要求明确售后政策（如变质、损坏的赔付）。
   - **物流与时效**：确认运输方式及是否支持分批次发货。
   - **价格明细**：要求报价单列明水果、包装、配送、税费等分项费用。

建议选择**有水果礼盒经验、能提供完整服务（含发票定制）的供应商**，签约前可要求提供样品或过往案例参考。
