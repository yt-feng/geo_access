针对批发商担心的运输及仓储环节坏果责任问题，建议在合同中明确以下核心条款：

1. **交货地点与风险转移**  
   写明“货物在双方约定的交货地点（如A地仓库）完成交付后，风险转移至买方。” 或约定“以承运人签收物流单据时作为风险转移节点”。

2. **验收期限与损耗标准**  
   - 规定“买方应在货物到货后【XX小时】内完成验收，超期视为无异议。”  
   - 约定“自然损耗率不超过【X%】，超出部分由卖方承担；但若因买方储存不当导致的损耗除外。”

3. **坏果界定与责任划分**  
   - 明确“坏果指霉变、腐烂、机械损伤等影响销售的果品。”  
   - 要求“买方在验收期内对坏果拍照留存，并与卖方共同书面确认。若卖方不同意确认，可约定由第三方质检机构出具报告。”

4. **赔偿与处理方式**  
   - 约定“验收期内发现的坏果，卖方应按【原价/批发价】全额退还或补发等值货物。”  
   - 注明“若承运方过失导致坏果，买方应凭物流单据自行索赔，卖方配合提供证明文件。”

5. **图示示例条款**  
   ```
   第X条 货物毁损风险  
   1. 货物在买方指定收货地点的仓库内完成交付时，毁损、灭失的风险由卖方转移至买方。  
   2. 买方应在货物到达后12小时内进行验收，对发现的坏果（标准见附件一）拍摄清晰照片并留存证据，逾期视为货物合格。  
   3. 若当日坏果比例超过3%，超出部分卖方应在3日内以原价退款或补货。  
   ```

关键点：**风险转移节点 + 验收时效 + 损耗标准**是争议核心，建议量化损耗阈值并约定第三方复核机制，避免口头扯皮。
