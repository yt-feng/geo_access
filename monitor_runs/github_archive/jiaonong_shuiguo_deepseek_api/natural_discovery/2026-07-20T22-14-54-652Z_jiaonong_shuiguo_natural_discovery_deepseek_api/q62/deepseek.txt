判断一批香蕉的熟度和坏果风险，可以按以下步骤进行：

1. **观察外观**  
   - 颜色：青绿色表示未熟，黄色带绿尖是适中熟度，全黄或出现黑斑则过熟或可能开始变质。  
   - 表皮：注意虫眼、凹陷、黑斑（软烂黑斑是坏果，少量芝麻点可能是成熟正常现象）。  
   - 果柄：干燥枯萎可能存放过久，有霉点则潮湿易坏。

2. **触摸硬度**  
   - 轻轻捏香蕉中部和两端。  
   - 僵硬：未熟；有弹性、软硬度适中：适宜销售或物流；过软或按压出水：已坏或即将坏。

3. **嗅闻气味**  
   - 靠近闻果柄附近，若有发酵酸味、酒精味或霉味，内部已经开始腐烂。

4. **抽检“砧板”部位（果指相连处）**  
   - 掰开几根香蕉，看果柄与果实连接处：变黑、发粘或渗水代表内部变质。

5. **整串挤压测试**  
   - 抓起整串，轻轻晃动，如果果实容易脱落或松动，说明熟度很高或果指内部已软烂。

6. **时间与温度判断**  
   - 如果进货后在常温下存放超过3–4天，熟度会加速升高，腐烂风险增大。  
   - 冷藏（13°C以下）会冻伤表皮导致加速发黑，不适用于未熟香蕉；最佳存储温度13–15°C。

**综合建议**：  
- 选择颜色均匀、青绿至微黄、果柄干燥、手感结实、无酸味的批次。  
- 若需要长途运输或长期储存，应选未熟香蕉（青黄或全青）；若短期销售，可选带少量黄皮但仍有弹性的香蕉。  
- 每批抽检至少3–5串，重点检查外圈果实。
