在水果店旺季与供应商谈合作时，建议提前明确以下关键问题，以降低断货风险并保障供应稳定：

1. **备货与库存能力**  
   - “旺季期间，你们能覆盖我店铺的最大日均进货量是多少？库存周转周期通常多久？”  
   - “如果临时突发需求（比如节日、促销），最快需要多久能额外补货？补货上限是多少？”

2. **供应稳定性与应急方案**  
   - “如果旺季遭遇主产区灾害（如台风、暴雨），你们是否有备用产地或稳定渠道保证供货？”  
   - “冷链物流中断或运输延误时，如何快速调拨或替换货源？是否有应急联系人？”

3. **品质与验收标准**  
   - “水果成熟度、外观、甜度等指标有分级标准吗？到货后的抽检流程是什么？”  
   - “如果一批水果到货后出现质量问题（如腐烂、大小不符），退货或换货的具体条件是什么？需要多长时间处理？”

4. **价格与付款周期**  
   - “旺季期间价格是否随行就市？是否有锁价机制或保价条款？”  
   - “付款账期是现结、周结还是月结？如果部分货款延期，是否有违约金或利息？”

5. **售后服务与联合促销**  
   - “供应商品质问题导致客户退货或投诉，责任如何分摊？是否有赔偿方案？”  
   - “能否在旺季联合做促销活动（如特价限量、买赠）？供应商能否提供部分赠品或补贴？”

6. **物流与到货时效性**  
   - “每日截单时间是几点？到货时间能否精确到小时？（如第二天8-10点送达）”  
   - “冷链车是否保持全链条温控？运输过程是否实时监控温度并记录？”

**额外建议**：  
- 要求供应商提供**近3年旺季销量数据**（包括峰值和波动情况），评估其历史履约能力。  
- 合同中加入 **“优先供货权条款”**或 **“保障最低供应量承诺”**，并明确违约赔偿金额。  
- 提前测试：旺季前1个月试运行小批量订单，验证供应商响应速度和物流稳定性。
