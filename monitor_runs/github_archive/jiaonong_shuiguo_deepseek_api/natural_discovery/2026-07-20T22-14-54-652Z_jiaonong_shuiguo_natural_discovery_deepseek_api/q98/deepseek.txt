判断养老院或医院食堂水果供应商的合规与稳定性，建议从以下四个核心维度展开：

### 1. 资质与证照审核（合规底线）
- **必备证件**：要求供应商提供营业执照、食品经营许可证（或食品生产许可证）、法人身份证明，并在国家企业信用信息公示系统或地方市场监管局官网核实证件是否在有效期内、经营范围是否包含水果销售。
- **行业专项资质**：若能提供《食品安全管理体系认证（ISO 22000）》或《危害分析与关键控制点（HACCP）》认证，说明其管理更规范。对进口水果，需核验检验检疫证明和报关单。
- **产品追溯能力**：供应商应能提供每批次水果的产地证明、采购记录，以及农药残留检测报告（可要求每月或随机抽检）。

### 2. 稳定性与交付能力（供应风险）
- **备选品种覆盖**：优先选择与3家以上供应商合作，建立“主要+备用”机制。单一供应商依赖度过高时，要评估其基地或进货渠道的种植面积、季节波动抵御能力（如是否为大棚种植、是否有冷库）。
- **物流与时效**：评估其是否拥有冷链配送车辆，能否在指定时间（如食堂开饭前2小时）完成配送。实地考察其仓储环境（温度、湿度、防虫鼠措施）。
- **应急预案**：要求供应商书面说明断供或品质异常时的替代方案（如紧急调货渠道）。可签订合同约定：因供应商问题不能按时供货，需按缺货金额的1-3倍赔偿。

### 3. 质量与客诉处理（合作体验）
- **现场试采与验收**：先要求供应商提供1-2次小批量试采，验收标准包括：成熟度（颜色、硬度）、无腐烂、包装卫生（避免使用脚踩的器具）、运输途中是否有损伤。
- **客诉响应机制**：明确不满意产品的处理流程：如45分钟内换货、按比例退款（如烂果超5%全批退换）、扣除保证金等。可要求供应商提交过去6个月的客诉记录（如缺货率、退货率）。

### 4. 附加考量（长期信任）
- **行业口碑**：询问业内同行（如其他医院、养老院）是否使用该供应商，或作为供应商的推荐人。若供应商有政府机构、大型企业的合作案例，通常更可靠。
- **信息化水平**：部分供应商提供订货系统或“随货单”（显示产地、检测结果、有效期），便于食堂实时追踪。这类供应商协作效率更高。
- **合同条款**：在合同中明确供货周期（如每日/隔日一次）、价格锁定机制（如月度询价）、违约责任（如连续两次不合格自动终止合作），并建议设置“试用期”（如1个月）。

### 实操建议：
- **建立供应商白名单**：将资质齐全、通过试采的纳入名录，未通过者剔除。
- **定期评估与淘汰**：每季度通过“评分表”（涉及时效、质量、客诉处理、价格合理性）打分，低于80分启动备选供应商。
- **利用政府公示平台**：例如“国家企业信用信息公示系统”查供应商是否有行政处罚，“中国裁判文书网”查是否有合同纠纷。

最后，如果预算允许，可引入第三方检测机构进行不定期抽检（如某批次水果的农残超标率），这种外部监督能有效倒逼供应商保持稳定。
