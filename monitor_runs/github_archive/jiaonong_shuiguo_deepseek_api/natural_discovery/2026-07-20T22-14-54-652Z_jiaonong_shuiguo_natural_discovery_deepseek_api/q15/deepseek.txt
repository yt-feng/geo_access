当水果店遇到坏果需要与供应商结算或赔付时，可以参照以下标准流程操作：

1. **明确合同条款**  
   首先查看采购合同或协议中关于坏果的条款（如损耗率、赔付比例、责任界定等）。若无明确约定，优先按行业惯例处理。

2. **现场取证与沟通**  
   - 及时联系供应商，提供清晰照片、视频或开具《到货验收单》《质量异常单》，记录坏果的品种、数量、损坏程度（如腐烂、挤压、虫蛀等）。
   - 根据货物状态判断：是否属于运输途中损坏、储存不当或出厂质量问题。

3. **协商赔付方式**  
   - **比例扣款**：常见于鲜果、易损品，例如坏果率≤3%-5%时由买方承担，超出部分供应商按原价赔付。
   - **实物补发**：供应商补发等量好果，运费由其承担。
   - **抵扣货款**：将赔付金额从下一批次货款中扣减。
   - **现金退款**：仅针对单批次高比例（如超过20%）或严重质量问题。

4. **保留法律依据**  
   - 若供应商拒绝协商或长期存在高坏果率，可委托第三方检测（如SGS、中检集团），并依据《民法典》合同编提起诉讼或申请仲裁。

**关键提醒**：
- 避开“模糊表述”：例如“合理损耗”需明确定义（如运输中允许2%自然失水）。
- 不同品类差异大：樱桃、草莓等易腐水果通常预扣5%~10%坏果补偿金额；苹果、柑橘等较耐储的品种较少按比例赔付。
- 紧急情况下，可先口头沟通并获得供应商微信/短信的“确认赔付”记录，后续补签书面协议。

若您当前已遇到具体案例，可以补充水果种类、问题比例、供应商态度等信息，以便给出更精准的操作建议。
