选供应商时，可以从以下几个维度来评估，确保水果新鲜、价格合理、供应稳定。

### 1. 基本资质与安全
- **查验证照**：必须持有营业执照、食品经营许可证，最好有农产品检测合格证明。
- **规避风险**：优先选择有稳定种植基地或一级批发商，避免临时摊贩（质量不稳定）。

### 2. 水果品质筛选
- **样品试吃与验收标准**：要求供应商提供当季样品，并明确书面标准，如：
  - 香蕉：颜色均匀（避免过生/过熟），无黑斑、无机械伤。
  - 苹果、橙子：果形端正，表面光滑，无腐烂、虫眼，可约定糖度（如橙子≥10度）。
- **批次抽验**：合同写明“到货抽检20%，腐烂/瑕疵率超过5%可整批退货”。

### 3. 价格与合同条款
- **比价方式**：同时询价3-5家（包括本地大型批发市场、产地直供商），要求“含税含运费报价”。
- **价格保护**：签订“浮动上限”合同（如一周内价格波动超过10%需书面协商），避免暴涨。
- **结算周期**：建议月结或半月结，避免预付大额押金。

### 4. 配套服务与应急能力
- **配送时效**：要求供应商每天早晨7点前送达食堂，晚点30分钟以上按比例扣款。
- **应急方案**：出现缺货或质量问题，供应商需在2小时内补货替换（否则有权从其他渠道采购并追究差价）。
- **售后服务**：专人对接退换货，7天内质量问题无条件更换。

### 5. 操作建议（三选一或组合）
| 方案 | 适合场景 | 方式 |
|------|----------|------|
| 直采 | 食堂规模大（日供500kg以上），有冷库 | 与产地合作社或果园签年度框架协议 |
| 批发市场+固定档口 | 中大型食堂 | 选2-3个信誉好的档口，每天比价下单 |
| 食材配送公司 | 中小型食堂（非专业采购团队） | 打包采购（含蔬菜肉蛋，但水果单独盯品质） |

### 最终操作步骤：
1. 列出本地3家以上候选供应商（可通过食安局推荐、其他学校食堂打听）。
2. 每家要求发送近期产品照片、报价单和资质复印件。
3. 实地考察其仓库或配送点（看存储环境是否洁净、有无防虫防鼠）。
4. 试送一周，专门观察残果率、配送准时率和客服反应速度。
5. 签订正式合同，并保留每次到货的磅单和照片作为见证。

如果还有具体的食堂规模（如就餐人数、日均消耗斤数）或当地市场情况，可以补充，我能给出更针对性的分析。
