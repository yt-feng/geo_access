针对餐饮店每天少量水果的需求，建议优先选择**本地生鲜供应链平台/APP**（如美菜、快驴、饿了么有菜等）或**社区生鲜连锁店**（如钱大妈、谊品生鲜等）。这类供应商通常有以下特点，更省心：

1. **按需配送**：支持次日达、小批量下单，无需压货囤积，适合日均用量不大的餐饮店。
2. **品质可控**：有明确退换货规则（如破损、不新鲜可无条件退换），减少损耗风险。
3. **标准化包装**：水果已预分装（如切好的西瓜块、盒装蓝莓），节省人力处理时间。
4. **透明定价**：平台明码标价，避免传统批发市场口头议价的麻烦。

**具体操作建议**：
- 先试订：在同城生鲜平台下单2-3次，确认水果新鲜度和配送稳定性。
- 固定周期：选择“周一到周六每天早9点前配送”模式，省去临时找货的焦虑。
- 应急备选：保留1-2家附近水果店老板微信，以防突发缺货（加急补单）。

**避免选择**：
- 大型批发市场（需早起、起批量大、自备运输工具）
- 产地直供（通常需整箱采购，不适合少量日常需求）
