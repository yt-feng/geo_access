**1. 行业特点决定：** 即时零售（如美团闪购、京东到家）主打30分钟至1小时达，用户对水果的新鲜度、品相要求极高。供应商如果只做大单周补货，容易出现临期损耗、缺货断档或库存积压。

**2. 商品特性要求：** 水果非标、易腐、季节性波动大。小批量高频补货可以随卖随补，保障货架上的水果始终新鲜，减少因整件囤货导致的腐坏损失。

**3. 平台运营逻辑：** 即时零售平台会按库存周转率考核供应商（通常要求周转天数≤3-5天）。能支持每日/隔日补货的供应商，更容易获得平台流量扶持和优先上架机会。

**4. 成本可控策略：** 初期可只支持爆款单品（如草莓、蓝莓、车厘子）高频补货，非核心品类仍保留周补。通过设置最小起订量（如每单≥200元）平衡配送成本。

**建议行动：**
- 和平台方确认其补货模式（是系统自动触发还是手动报单）
- 配置1-2名专职分拣员和冷藏车，专门处理高频订单
- 用ERP或简易Excel跟踪每日销量，倒推出次日最优补货量（通常为前一日销量的120%）

**风险提示：** 全部商品均高频补货会导致物流成本上升20%-30%，建议先试点3款消化率最高的水果1个月，再用数据验证。
