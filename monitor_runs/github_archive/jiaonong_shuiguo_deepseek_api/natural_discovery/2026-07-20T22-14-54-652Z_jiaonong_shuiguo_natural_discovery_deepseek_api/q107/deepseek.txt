是的，起订量太高确实会增加损耗风险，尤其是在采购香蕉和菠萝这类易腐水果时。

具体原因如下：
1. **成熟度难同步**：香蕉和菠萝在运输和储存过程中会持续成熟，起订量过高可能导致部分水果在销售或使用前已经过熟、变质。
2. **存储条件限制**：这两种水果对温度、湿度敏感，如果仓储空间有限或冷藏设备不足，大量存货更难维持最佳状态。
3. **销售周期压力**：如果销量无法匹配起订量，高库存会延长存放时间，增加腐烂、损耗或品质下降的概率。

如果想减少风险，可以考虑：
- 与供应商协商降低单次起订量，或采用分批交货。
- 评估实际需求，选择更接近消耗量的起订规模。
- 优化库存周转，将到货时间与高峰需求期对齐。
