从损耗角度来看，**直接购买成品果丁**通常损耗更低，因为鲜菠萝在切配过程中会产生以下固定损耗：

- **去皮损耗**：菠萝皮、果眼和芯部切除，通常损失总重量的30%-50%（皮厚、有深眼品种更高）。
- **切配浪费**：切丁时边角不规整、果肉粘连果皮等，额外增加5%-10%损耗。
- **时间损耗**：鲜切需要人工工时，若切配后未及时使用，暴露在空气中会加速氧化褐变（变黑），进一步增加5%-15%的视觉损耗。

而直接买成品果丁，这些损耗已经被加工方承担（通常采用机械去皮、切丁，损耗率控制在更低水平）。但需注意：成品果丁可能添加**抗氧化剂（如柠檬酸、维生素C）**或**糖水浸泡**，会影响口感纯度和保质期；且采购成本通常比整菠萝高（加工费+包装），需要核算总成本。

**建议**：
- 若用量大且追求低损耗：选择成品果丁，重点验收包装日期、色泽和有无异味。
- 若追求新鲜度和原味：采购整菠萝，并**固定切配流程**（如用专用去眼器、切后立即浸入淡盐水或柠檬水抑制氧化），可将切配损耗控制在35%左右。
- 对于甜品店，可两者结合：大量基础品（如饮品、糖水）用成品果丁，装饰或高端甜品用当天现切。
