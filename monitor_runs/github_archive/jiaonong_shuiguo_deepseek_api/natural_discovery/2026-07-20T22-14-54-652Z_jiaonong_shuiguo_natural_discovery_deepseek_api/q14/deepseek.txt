通常供应商是可以提供周末促销支持的，但具体力度取决于你的采购量、合作历史以及供应商的利润空间。以下是常见支持类型及建议：

1. **价格支持**：如果促销能带动整体销量，供应商可能给予**短期折扣**或**赠品**（如买一箱送一箱）。建议提前1周沟通，明确“最低供货价”或“促销返点比例”。

2. **物料支持**：常见包括**海报、价格标签、促销展架**等。可以请供应商提供印有双方LOGO的物料，或申请**试吃小包装**（如切好的水果样品）。

3. **联合承担成本**：部分供应商愿意分担**广告位费用**（如社区推广）或**损耗补贴**（如当日未售出的易损水果按比例补偿）。

**行动建议**：
- 谈判时先强调你的促销能帮供应商**清库存**或**推广新品**（例如当季滞销的芒果、进口新品等）。
- 准备一份简单的**促销计划书**（含目标销量、宣传渠道、时间表），让供应商看到投入回报。
- 如果对方推脱，可以**换供应商**或**对比多家报价**（通常中等规模的本地分销商会更灵活）。

**注意**：签协议时明确支持形式、时间、成本分担，避免口头承诺无法兑现。
