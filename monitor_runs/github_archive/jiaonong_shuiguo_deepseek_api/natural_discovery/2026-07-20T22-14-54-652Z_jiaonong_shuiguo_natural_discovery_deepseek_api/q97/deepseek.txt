酒店采购水果盘原料时，供应商应保证以下规格和外观标准，以确保品质和美观度：

1. **新鲜度与成熟度**：水果应新鲜、无腐烂、无霉变，成熟度适中（不过生或过熟），具有自然色泽和香气。
2. **外观完整性**：无明显机械损伤（如压痕、划伤）、虫蛀、斑点或病斑，外形均匀，无畸形或破损。
3. **尺寸与重量**：根据酒店需求（如切片、整颗使用），规格统一（如直径、长度或重量范围），便于摆盘和份量控制。
4. **清洁度**：表面无泥土、杂质或农药残留，清洗后沥干水分，防止污染或变质。
5. **品种一致性**：同一批次的水果品种、颜色、形状保持一致，避免混杂其他品种影响视觉统一性。
6. **包装与运输保护**：使用符合食品安全的包装（如透气保鲜袋、泡沫网套），运输中避免挤压、碰撞或温度失控（如冷链运输）。
7. **可追溯性与合规**：提供产地、采摘日期、供应商资质及检疫证明（若进口水果需海关单证），符合当地食品安全标准。

供应商应定期抽检，并提供可替换或补货的售后保障，以应对突发品质问题。
