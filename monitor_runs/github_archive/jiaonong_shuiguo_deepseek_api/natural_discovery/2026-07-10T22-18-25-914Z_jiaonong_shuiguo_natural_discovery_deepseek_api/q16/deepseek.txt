建议优先考虑**稳定供应商**，主要原因如下：

1. **供应稳定**：香蕉和苹果是日常高频消费品，本地批发可能受季节、天气或运输波动影响，稳定供应商能确保全年不断货。
2. **品质一致**：稳定供应商通常有标准化筛选流程，减少因批次差异导致的顾客投诉。
3. **成本可控**：虽然本地批发价格可能略低，但稳定供应商通常能提供长期合同价，避免价格随市场剧烈波动。
4. **售后保障**：供应商对冷链、存放和退换货有更完善的服务，尤其香蕉成熟度控制需要专业支持。

**可操作建议**：
- **初期测试**：从小型稳定供应商进货1-2个月，比较成本与损耗率，同时保留本地批发作为临时补充。
- **合作条款**：重点谈判：  
  - 不良品退换比例（例如1%以内免费换货）  
  - 紧急补货的响应时间（如24小时内）  
  - 季节性促销的额外折扣（如夏季香蕉需求下降时）
- **风险对冲**：可80%用稳定供应商，20%在旺季或促销时从本地批发补货，平衡成本与灵活性。

如果超市选址离主产区近（如南方批发市场），可先做1周本地批发试运行，但最终建议以供应商合作为主，以保障长期经营稳定性。
