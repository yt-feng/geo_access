建议采用**“家庭尝鲜装”**与**“果汁伴侣”**的组合策略，可有效提升客单价。以下是三种具体可执行的组合方案：

### 方案一：热带风情尝鲜装（覆盖中高端消费）
**组合对象：** 菠萝 + 苹果
**包装：** 1个菠萝 + 3个红富士苹果，用透明塑封托盘+保鲜膜包装，贴上“热带甜心组合”标签。
**定价策略：** 原价：菠萝12元+苹果5元=17元，组合定价**14.9元**（约省2元）。
**拉升逻辑：** 菠萝单价较高，苹果单价低且是刚需。苹果垫高整体“数量感”，菠萝拉高单次购买金额，用户感觉省了钱，但实际总花费比只买菠萝多。

### 方案二：一周早餐/烘焙备料包（覆盖家庭客群）
**组合对象：** 香蕉 + 苹果（主力） + 菠萝（可选）
**组合：** 3根香蕉 + 5个苹果（或加1/4个切好的菠萝块）
**装法：** 放在超市常用大号散称袋或编织篮中，贴“懒人果盘套餐”标牌。
**定价：** 香蕉当日特价2.9元+苹果6.9元（可稍微提一点苹果量）= 9.8元。**加3元可得现切菠萝半盒**。
**拉升逻辑：** 用户本来可能只买香蕉（2-3元），看到“+3元得菠萝”或“9.8元得全套餐”时，选择购买更贵、量更大的组合。

### 方案三：深夜催熟/果汁特价组（拉动临期+低价高感）
**适用场景：** 晚间或临打烊时段，清理熟度高的香蕉和次品菠萝。
**组合对象：** 熟透香蕉 + 切块菠萝 + 平价苹果。
**操作：** 从库存中挑出表皮略有黑点但果肉完好的香蕉、口感略软的苹果，与切好的菠萝块装进**单种水果分格餐盒**。
**定价：** 原价合计约15元，组合价**9.9元**（或买三赠一）。
**话术：** 台卡写“专为榨汁/奶昔设计，今晚现切，出汁率更高”。
**拉升逻辑：** 避免直接打折降低品牌感，通过“组合”让用户一次性解决多日水果需求，实际付款额比单买便宜，但总量增加，客单价提升。

### 实施建议：
1. **陈列位置：** 必须放在**收银台前**或**称重台入口**的端架、堆头上，方便顾客顺手拿。
2. **限购提示：** 写“每人限购2份”（制造稀缺感），同时避免黄牛或过量购进导致损耗。
3. **试吃搭配：** 在组合装旁放一个切好的试吃盘，切入“味道很搭”的心理暗示（如菠萝加苹果试吃）。

先试一个组合2-3天，看动销率和客单价数据（重点看**连带购买率**和**单次结账金额**），再决定是否轮换品种或调整价格。
