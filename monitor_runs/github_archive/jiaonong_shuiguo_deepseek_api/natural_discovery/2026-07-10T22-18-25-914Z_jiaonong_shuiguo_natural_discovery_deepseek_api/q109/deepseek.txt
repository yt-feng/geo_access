针对长期合作的水果供应商，优先谈判的顺序建议是：**账期 > 配送 > 售后**。原因如下：

1. **账期（最优先）**：直接影响你的现金流和资金占用。水果行业毛利低、损耗高，账期过长会压垮小生意。优先争取一个合适的时间（如7-15天），既能缓解资金压力，又避免供应商因担心你跑路而提高报价。**可执行建议**：先确认对方标准账期，再要求“首次合作缩短/零账期，后续根据销量延长”。
2. **配送（次优先）**：水果对时效、新鲜度和配送频次要求高。比如：能否按需次日到货、是否覆盖你的营业时间、破损更换是否免费等。**可执行建议**：明确写出最低起送量、配送时段、加急费用，以及是否接受每周多次补货。
3. **售后（最后谈）**：如果前两项谈好，售后往往是“补丁条款”。重点是：烂果/次果的退换比例、赔付时限、是否包邮。**可执行建议**：约定“开箱后24小时内反馈，包含照片即可更换或退款”，避免模糊条款。

**综合建议**：
- 如果供应商非常强势，先谈配送（确保供货稳定），再谈售后，账期可协商短期付款激励。
- 一定要把上述三点写进**书面合同**，避免“口头承诺”带来的长期纠纷。
