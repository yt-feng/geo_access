针对普通商家，可以用他们熟悉的品类做对标和类比，从“好卖、好赚、好运输、不容易坏”这几个角度切入。具体建议如下：

**1. 直接对标“大品类”，降低认知门槛**
- 用熟悉的水果来做新品的“风味锚点”。例如：“这个‘黄金火龙果’甜度像菠萝，但口感更软糯，比火龙果更甜，没有酸味。”  
- 对比日常成本和风险：  
  “这个‘紫心番薯’和香蕉一样好卖，但损耗比绿叶菜低一半，放三五天没问题。”  
  “这款‘甜脆枣’和菠萝一样耐运输，但不需要泡盐水，顾客直接买走更方便。”

**2. 用“省心、多赚”两句话说明白痛点**
- 聚焦利润和流程：  
  “这个水果看外观和普通芒果差不多，但个头均匀、大小一致，省去分拣人工，你每单可以多赚2毛。”  
  “和香蕉一样剥皮就能吃，但比香蕉热度高20%，因为电商平台上这类小众水果搜索量每周涨30%。”

**3. 模拟常用场景，让商家感知真实需求**
- 举例说明消费场景：  
  “比如你现在的客户觉得菠萝吃多了‘扎嘴’，那这个‘无刺菠萝’就是升级产品。老客户复购率能提两成。”  
- 直接报“线下常见组货方案”：  
  “你搭配一个‘国产香蕉+泰国小芭蕉’组合，冷柜里一个货架搞定，客人拿一盒香蕉，顺手就能带走。”

**4. 用数据说话，但要简单直接**
- 直接量化的“卖点公式”：  
  “价格是阳光玫瑰的60%，但糖度多2度，门店当天复购率就会翻倍。”  
- 标注关键转折：  
  “卖菠萝你一天要修皮5次，这个新品种整果上架、不用处理，每天省30分钟人工，等于多赚50元。”

**5. 给“试用动作”降低试错成本**  
- “您先拿一箱回去挂样板，和我这盒‘标准货样’比对，卖不动退货，运费我出。”  
- “要是有老客户投诉口感，这批货我补差价。”

**最关键的一句话**：  
“您不用猜能不能卖，就像您卖香蕉一样简单：颜色对、价格好、坏果退款，卖一单算一单。”
