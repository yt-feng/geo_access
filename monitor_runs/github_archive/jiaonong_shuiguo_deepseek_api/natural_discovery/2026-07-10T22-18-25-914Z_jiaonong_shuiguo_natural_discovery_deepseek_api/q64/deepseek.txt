针对批发档口的退货纠纷，以下是一套清晰、可执行的收货验货操作流程，可有效降低纠纷：

**1. 收货前，做好预案（防患于未然）**
- **明确标准**：提前以书面/电子形式告知下游客户（或档口对上家）的验货标准，例如：货物瑕疵范围（如3%以内）、尺码误差允许值、颜色是否以实拍为准、其他特殊要求。
- **约定异议时效**：在订单中写明“买方收货后24小时内进行外观检验，超时视为认可货物状况”（法律允许的合理时间内）。

**2. 卸货时，快速初筛（针对外部包装）**
- **检查外包装**：有无严重变形、破损、水渍或二次封箱痕迹。
- **称重核对**：如果每件货物重量差异较大，可抽查单箱毛重是否与发货单/常规定量一致（防止短装或夹带异物）。
- **保留原始物流面单和录像**（如有监控）。

**3. 拆包后，逐批检验（核心环节）**
- **按比例抽查**：若货物量大（如数百件），按总件数10%-20%的比例随机抽检；如果首次抽检合格率低于95%，应加大比例（如30%）。
- **按重点检查**：
  - **外观瑕疵**：污渍、破损、线头、色差、变形、不对称（针对服装/鞋包）。
  - **尺寸差异**：用标准量具测量关键部位（如胸围、裤长），与报价单/标签对比。
  - **功能性**：拉链、纽扣、配件是否可正常使用。
  - **材质/手感**：与谈判时的样品是否一致（可备一小块样品原件对比）。
- **使用“分级记录单”**：将抽检到的合格、可修复（如可自行熨烫、剪线头）、不合格（如严重染色、断裂）三类分开记录，并拍照留证。不合格品立刻单独存放、贴标签。

**4. 关键争议点 —— 当面或限时视频确认**
- **要求“拆包视频”**：在快递员在场时，或打开手机全程录像（从拆封到验完），一镜到底，不要剪辑。视频中要清晰显示运单号、货物全貌。
- **特别提醒**：对于易损品（如陶瓷、玻璃制品、精密仪器），尽量当面开箱，有问题当场拒收并让快递员出具破损证明。

**5. 验后，留痕归档**
- **填写《验货单》**：包括：收货日期、货物批次/订单号、抽检数量、合格率、具体问题描述、处理意见（退换？扣款？让步接收？）。
- **各方签字**：如果涉及送货方、档口员工、客户代理人，当场签字确认；或者通过微信发送电子表单要求对方回复确认。
- **无争议的情况下，立即发版结算**：避免后来翻旧账。

**6. 特殊情况处理**
- **量少易碎品**：100%拆除所有包装检查。
- **仓库留存异议品**：将不合格品保留原状，建议保留30天（超过常规退货期），以防对方要求退回重新鉴定。
- **如使用“样品封存”制度**：与供货方各保留一份密封样品（附双方签字），作为后续争议时的实物参照。

**最后提醒**：**所有沟通、验收过程养成“文字留底”习惯**（微信群、公函等），而不是仅靠口头。退货纠纷往往始于“心证”（“我记得当时…”），终结于“物证”（照片、视频、单据）。
