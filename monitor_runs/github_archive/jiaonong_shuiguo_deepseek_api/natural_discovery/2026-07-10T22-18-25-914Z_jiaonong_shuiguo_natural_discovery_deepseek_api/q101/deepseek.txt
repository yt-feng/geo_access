与酒店早餐水果供应商合作时，为确保稳定供应和质量，建议关注以下几个核心方面：

1. **供应能力与稳定性**  
   - 确认供应商的种植基地、冷库容量、运输车队等硬实力，要求提供过往大型酒店的供货记录。  
   - 明确供应周期（如每日/隔日配送）和应急方案（如台风、疫情时的替代渠道）。

2. **品质与品控标准**  
   - 约定具体规格：水果大小（如苹果直径≥8cm）、成熟度（如香蕉表皮绿色≤10%）、包装方式（防碰伤托盘）。  
   - 要求每日配送前提供抽样质检图片或第三方检测报告（农残、甜度等）。

3. **价格与结算方式**  
   - 明确报价含税、含运费，签订**季度价格锁定协议**（避免季节性剧烈波动）。  
   - 结算周期建议30-60天，预留5%-10%质量保证金。

4. **合同关键条款**  
   - 违约条款：注明断供3次以上自动解约并承担酒店营业额损失。  
   - 退换货机制：如次日开箱发现腐烂率＞5%，供应商需2小时内补送或扣除当日款项。

5. **沟通与长期合作**  
   - 建立微信群或系统对接，每日18点前提交次日配送清单。  
   - 每季度联合抽查仓库库存，提前1个月协商节假日（如春节）备货量。

**建议行动步骤**：  
1. 优先选择有**ISO22000食品安全认证**、服务过同星级酒店的供应商。  
2. 试供期不少于1个月，期间随机抽查3批次以上。  
3. 在合同中加入**浮动调价机制**（如当月水果批发价波动超10%时重新议价）。
