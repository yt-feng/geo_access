选择学校食堂水果供应商时，建议从以下4个核心环节入手，确保安全、稳定和性价比：

### 1. 资质审核（必须门槛）
- **证件齐全**：要求供应商提供《营业执照》《食品经营许可证》或《食用农产品合格证》，进货单据需保留完整。
- **追溯能力**：供应商能提供每批次水果的产地证明或检测报告（尤其关注农药残留）。
- **保险与赔付**：确认供应商是否购买食品安全责任险；合同中明确因质量问题的退换货、赔偿条款。

### 2. 实体考察（现场评估）
- **仓储条件**：查看仓库是否保持通风、温湿度适宜（苹果/橙子一般需0-4℃冷藏，香蕉需12-14℃避免冻伤）。
- **周转速度**：观察库存是否新鲜（无霉变、软塌、碰伤）；配送频率建议至少2次/周（长期储存的水果品质易下降）。
- **样品品鉴**：随机抽样检查甜度、大小均匀度，同时让食堂厨师试做（如香蕉是否适合切块沙拉，橙子果肉紧实度）。

### 3. 品质与成本平衡
- **分级采购**：
  - 香蕉：选“七成熟”可自然催熟，避免一次性全成熟后损耗。
  - 苹果/橙子：选择直径>7cm的一级果，次果可用于果汁或点心。
- **询价对比**：至少对接3家供应商进行比价（分日常价和批量价）；要求提供当季时令水果清单，避免反季节高价品种。
- **损耗分担**：约定“送货后24小时内可退换明显损坏品”；长期合作可要求每月保底采购量，换取价格折扣。

### 4. 常态化监控
- **留样制度**：每批水果抽检留样至少48小时，附进货记录（名称、批次、供应商、进货时间）。
- **反馈机制**：每月与供应商开一次品质评估会；设立食堂员工匿名评分（如香蕉是否易氧化、果径是否达标）。
- **备选方案**：保留1家备用供应商，避免因天气或物流中断导致断货。

**注意**：学校食堂采购需优先考虑本地或周边产地（降低运输损耗），避免选择仅凭“低价”但无可靠来源的供应商。如预算允许，建议加入校园食品安全联盟统一集采，或通过省级“菜篮子”工程目录中的企业进行采购。
