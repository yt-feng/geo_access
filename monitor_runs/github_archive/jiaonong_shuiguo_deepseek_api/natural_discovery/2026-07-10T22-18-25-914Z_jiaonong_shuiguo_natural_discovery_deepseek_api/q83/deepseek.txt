在生鲜电商水果到货后出现坏果的情况下，责任划分通常基于以下几个关键因素：

1. **发货前的品质**：如果水果在发货时就是坏的（如内部腐烂、明显破损），则责任在商家。
2. **运输过程中的问题**：若因包装不到位（如缺乏防撞材料、冷链不达标）或物流运输延误、暴力装卸导致水果损坏，责任由商家或物流方根据合同划分。通常商家需对运输环节负责。
3. **顾客签收后的处理**：如果顾客签收后长时间未及时取件、存放不当（如未冷藏），或者签收时未检查，则顾客可能需要承担部分责任。
4. **不可抗力因素**：极端天气等不可抗力导致的损坏，一般双方协商分担。

**建议处理流程**：
- 顾客应在收到货后**立即开箱检查**，拍照或视频记录坏果情况（包括快递单号、包装状态、坏果特写）。
- 原则上，生鲜水果“坏果包赔”是常见规则，顾客可直接联系商家客服，提供证据要求退款或补发。
- 如果是物流问题，商家通常会先行赔付顾客，再向物流公司追责。

**常见责任划分**：
- **商家全责**：外包装完好但内部坏果较多（说明发货或包装问题）。
- **商家与物流共同责任**：外包装破损导致的损坏。
- **顾客无责**：签收时立即发现并反馈，商家应无条件处理。

如果商家推诿，可保留证据向电商平台投诉或拨打12315维权。
