在超市采购水果时，供应商通常需要提供以下检测报告或进货证明，以确保合规性和食品安全：

1. **产地证明**：注明水果的产地（如农场、区域），以追溯来源。
2. **农药残留检测报告**：由第三方机构出具，证明水果符合国家或行业农药残留限量标准（如中国的GB 2763）。
3. **检疫证明**：对于进口水果，需提供海关出具的《入境货物检验检疫证明》，确认无有害生物。
4. **合格证或质量认证**：如绿色食品、有机认证或GAP（良好农业规范）证书（如有特殊标准）。
5. **物流与冷链记录**：对于易腐水果，提供运输和储存温度记录，证明新鲜度。
6. **生产日期与保质期证明**：清晰标注批次、采摘日期等。

建议在实际采购前，与供应商明确具体清单（可参考国内《食品安全法》或超市内部验收标准），并在合同或协议中列明要求。如有不确定，可向当地市场监管部门或采购部门咨询。
