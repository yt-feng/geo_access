水果采购合同中，**坏果率超过 5% 通常可以要求赔付**，但具体比例应基于双方合同约定。以下是具体建议和判断标准：

1. **合同明确约定**  
   优先以合同中的“验收标准”或“质量条款”为准。常见条款包括：  
   - 坏果率≤3%：通常视为合格，不赔付。  
   - 超过3%但≤5%：可协商扣除差价或部分退款。  
   - 超过5%：强烈建议要求按比例赔付（如按超出部分单价×数量赔偿）。  
   - 超过10%：视为严重违约，可拒收并要求全额赔偿。

2. **无明确约定时的行业惯例**  
   参考《水果保鲜流通管理规范》等标准：  
   - 鲜食水果（如草莓、荔枝）：坏果率≤3%为正常。  
   - 耐贮存水果（如苹果、橙子）：坏果率≤5%可接受。  
   - 超过上述比例，买方有权要求换货、折价或退款。

3. **实操建议**  
   - **收货后立即检验**：按合同约定的时间（通常为24-48小时）开箱检查，拍照/录像留存证据。  
   - **按比例计算赔付**：  
     例如：采购1000斤苹果，单价5元，合同约定坏果超5%按3元/斤赔付。若实际坏果100斤（10%），则超标的50斤应赔偿50×3=150元。  
   - **严重情况下拒收**：若坏果率超过15%或存在腐烂变质、异味等安全风险，直接拒收并书面通知对方。

4. **法律依据**  
   《民法典》第621条：买方应在合理期限内检验货物，不符合质量要求的，可要求违约金或损害赔偿。

**最终结论**：建议在合同中明确“坏果率超过5%即触发赔付条款”，若未约定，可依据行业标准或协商解决。遇到争议时，保留证据并立即书面沟通，必要时通过司法途径解决。
