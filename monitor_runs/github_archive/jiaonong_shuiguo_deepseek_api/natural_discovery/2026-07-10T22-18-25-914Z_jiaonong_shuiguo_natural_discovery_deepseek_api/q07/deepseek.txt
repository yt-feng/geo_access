根据水果礼盒的常见需求和市场反馈，以下是几种搭配方案，兼顾视觉美观、口感和实用性，供您参考：

**1. 经典红火搭配（送礼首选）**
- **搭配**：苹果 + 橙子 + 火龙果（红心）
- **卖点**：颜色红火，寓意平安（苹果）、吉祥（橙子）、红红火火（火龙果）。火龙果增加高级感，耐放且切面美观。
- **适合场景**：春节、中秋、拜访亲友。

**2. 清新健康搭配（女性/家庭自用）**
- **搭配**：橙子 + 香蕉 + 菠萝
- **卖点**：橙子富含VC，香蕉饱腹，菠萝酸甜解腻。塑料盒装时需注意菠萝切小块并密封，防止串味。
- **适合场景**：日常探病、下午茶、轻食爱好者。

**3. 高性价比搭配（走量款）**
- **搭配**：苹果 + 橙子 + 梨 + 少量菠萝丁
- **卖点**：苹果和橙子耐放，梨清甜，菠萝丁提色。成本低，适合30元左右的平价礼盒。
- **提醒**：香蕉易压坏和褐变，平价款可减少香蕉比例或替换为柚子。

**4. 热带风情搭配（差异化）**
- **搭配**：菠萝 + 香蕉 + 百香果（或芒果）
- **卖点**：热带水果组合新颖，适合年轻客户。香蕉要选全熟但未软烂的，百香果用独立小盒装。
- **适合场景**：夏季促销、网红打卡、咖啡馆伴手礼。

**注意事项**：
- **处理香蕉**：香蕉需放在最上层或单独隔开，避免压在硬水果下。如果盒装，可用纸巾垫底防褐变。
- **菠萝处理**：建议提前去皮切块（放盐腌几分钟去酸涩），或保留全果加赠削皮刀，避免顾客嫌麻烦。
- **配色原则**：避免全绿或全黄。红色（苹果/石榴） + 金黄（橙子/菠萝） + 绿色（网套/绿叶点缀）最保险。
- **成本控制**：如果香蕉价格波动大，可替换为柚子（更耐放）、猕猴桃（显高端但需软熟期）。

**试卖建议**：先做3-5种组合样品，放在店里试卖观察销量，主推销量最好的一种。同时注意避开不耐放的水果（如榴莲、山竹）以免顾客投诉。
