在面对超市水果到货品相不好的情况时，建议在采购合同或售后协议中明确以下关键流程和约定，以有效保障双方权益：

1. **验收标准与时限**  
   - 明确约定“品相良好”的具体标准（如：无腐烂、无明显机械损伤、色泽均匀、大小符合规格等），可附照片或行业参考样图。  
   - 规定验收时限：到货后 **X小时内**（如4-6小时）内，买方需完成外观验收并书面通知卖方问题，逾期视为合格。

2. **不合格品处理程序**  
   - **拍照留证**：发现品相问题后，立刻对不合格水果进行多角度拍照（包括外包装、整体、局部特写），保留原件证据。  
   - **通知方式**：在约定时限内通过书面形式（微信/邮件/系统记录）向卖方发送问题清单，内容包括：到货时间、品名、数量、问题照片、延误影响说明。  
   - **隔离存放**：将问题品与合格品物理隔离，避免混放导致责任不清。

3. **退换货与补偿方案**  
   - **全部退换**：若品相问题比例超过 **X%（如15%-30%）** 且严重影响销售，买方有权要求整批退货，卖方承担运费及差价损失。  
   - **部分退款/换货**：对问题率低于约定比率的，可按问题数量 x 售价进行折扣退款，或由卖方在 **Y小时内**（如24小时内）补发同品质水果。  
   - **紧急替代**：若问题严重导致缺货，卖方需承担买方临时采购替代品（溢价部分）的差额。

4. **争议处理机制**  
   - 保留共同认可的第三方鉴定机构（如当地质检/农科院）作为争议仲裁方。  
   - 明确“不可抗力”条款：如因极端天气导致运输中品相传变，需由卖方提供运输时间线及温度记录等凭证方可免责。

5. **执行建议**  
   - 在合同中加入“品相不合格视为卖方违约”条款，并约定每日千分之几的违约金。  
   - 长期合作商可建立“信誉分”制度，多次问题则降低合作等级或终止合作。

**示例条款**：  
> “水果到货后，买方应在4小时内根据附件一《品质标准》完成外观验收。凡出现腐烂、严重挤压、变色占本次总量的10%以上时，买方有权拒收整批货物，卖方需在48小时内全额退款并承担买方因此产生的仓储及误工费；对少量不合格品，可按单件零售价的80%扣减货款。”

建议将上述条款写入合同，并提前与卖方确认执行细节。如已发生问题但无明确约定，尽量通过协商共同确认损失，避免纠纷升级。
