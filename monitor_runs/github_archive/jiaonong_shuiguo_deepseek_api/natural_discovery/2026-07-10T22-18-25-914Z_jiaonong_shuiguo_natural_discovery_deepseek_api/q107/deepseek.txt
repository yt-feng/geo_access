是的，起订量过高确实会增加损耗风险。主要原因包括：

1. **库存积压**：起订量大意味着进货数量多，如果销售速度跟不上，香蕉（成熟快，易变黑软化）和菠萝（易腐烂、碰伤）容易在储存过程中变质，导致损耗。

2. **储存条件限制**：香蕉和菠萝对温度、湿度敏感，需要特定冷链或通风环境。起订量过高可能超出储存空间或设备能力，加速品质下降。

3. **市场需求波动**：客流或消费习惯不稳定时，起订量高会加大滞销风险，尤其短保质期水果。

**建议**：
- **阶梯式采购**：尝试与供应商协商降低首单起订量，或分批到货（如每周2~3次少量配送）。
- **预估销量**：根据历史销售数据、促销计划、季节等因素，精准计算日均需求，再确定起订量上限（例如不超过3~5天预期销量）。
- **强化管理**：优化冷藏设备，按“先进先出”原则摆放，对菠萝可适当提前切皮分装（延长货架期）。

通过控制起订量平衡库存周转与损耗，是降低经营风险的有效方式。
