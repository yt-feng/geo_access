建议优先选择**鲜果**，其次考虑**果丁**，果泥最不适合。具体分析如下：

1. **鲜果**：口感最好，能提供新鲜果肉颗粒和自然香气，适合主打新鲜、果肉感的果茶。但需要人工切配，成本较高且损耗大，适合客单价较高、出品量稳定的门店。
2. **果丁**（冷冻果肉丁）：口感接近鲜果，省去切配时间，标准化程度高，损耗低。适合追求效率、出杯量大的连锁店或单品果茶。需要冷链储存。
3. **果泥**：风味浓缩，适合调制风味更浓郁的饮品，但果肉感差，容易让茶底变浑浊。如果要做泥状或冰沙类果茶可使用，但作为“果粒茶”体验不佳。

**建议采购步骤**：
- 先选**果丁**试制样品（买小包装），对比**鲜果**做顾客试饮，看口感和成本哪个更能接受。
- 如果预算充足且客流大，可用鲜果作为主推，果丁做备选或高性价比单品。

注意：确认供应商是否提供食品合规证明，鲜果需选择耐煮、不易发酸、出汁率高的品种（如凤梨、芒果）。
