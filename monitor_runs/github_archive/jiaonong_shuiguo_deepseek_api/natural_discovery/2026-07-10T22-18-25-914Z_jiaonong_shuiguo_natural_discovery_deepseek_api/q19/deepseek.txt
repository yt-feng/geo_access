在超市购买菠萝时，优先级建议如下：

1. **甜度**（最核心）  
   甜度直接影响口感。可通过以下方式判断：  
   - 闻基部：有浓郁甜香味的通常较甜。  
   - 看颜色：成熟菠萝金黄或橙黄色更甜，青绿色可能酸涩。  
   - 按果皮：稍软且有弹性的菠萝成熟度更高（过硬则未熟）。

2. **外观**（辅助判断成熟与新鲜度）  
   - 果皮无黑斑、霉点或腐烂。  
   - 叶片翠绿、无枯萎（若叶片易拔落可能过熟）。  
   - 形状匀称（无明显凹陷或畸形）。

3. **大小**（非关键因素）  
   大小不直接关联甜度，但过大的菠萝可能纤维较粗。选择中等大小（约1.5-2.5斤）通常肉质更细腻。

4. **切果损耗**（仅限超市预切果盒）  
   若购买已切好的菠萝，需注意：  
   - 果肉无褐变或黏液，颜色鲜亮。  
   - 检查包装日期，避免隔夜储存导致的酸化。  
   - 优先选整果自己切，可避免损耗和卫生风险。

**总结**：优先用闻香、看色判断甜度，再检查外观无瑕疵；大小和切果损耗在超市场景中次要考虑。
