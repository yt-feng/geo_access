建议采用**小批量试卖**策略，初期采购量控制在**常规周销量的10%-20%**，或者**单店首次进货5-10箱**（根据店铺日均客流调整）。

**具体操作建议：**

1. **最低试卖量**：根据品类风险
   - 高损耗易腐水果（如荔枝、杨梅）：单店1-2箱（约20-50斤）
   - 常温耐存水果（如苹果、梨）：单店3-5箱（约50-100斤）
   - 进口高价水果（如车厘子）：单店0.5-1箱，先测价格敏感度

2. **试卖周期**：控制在**3-7天**内，观察每日出货量、损耗率、顾客反馈

3. **数据指标**：
   - 日周转率 > 60%（当天进货能卖出6成以上，可加量）
   - 损耗率 < 15%（否则说明选品或定价有问题）
   - 复购率 > 20%（回头客占比高，新品值得推）

4. **特殊保护**：要求供应商提供**试卖期售后保障**（如滞销包退、损耗补贴），降低你的囤货风险。

**所以最稳的做法是**：先要1-2周的试卖配额（如每周到货2箱），若数据达标再签正式订单。切勿一次性大量订入，除非供应商支持滞销包退。
