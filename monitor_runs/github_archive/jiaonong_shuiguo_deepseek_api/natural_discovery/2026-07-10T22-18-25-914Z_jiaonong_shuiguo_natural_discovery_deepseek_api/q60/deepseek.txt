核心建议：**设定“紧急补货+阶梯时间承诺+免责置换条款”，而不是只笼统要求“按质换货”**。

具体执行方案如下：

**1. 立即使用“保量应急条款”**
在合同中明确约定：**若到货合格率低于95%，供货方必须在X小时内（如：市区2小时，郊区4小时）完成免费紧急补货或调货，且补货品必须达到门店验收标准。** 这能保障你当天的营业不中断。

**2. 阶梯式数量与损耗约定**
不要笼统说“补货”，要区分情况：
- **轻微瑕疵（如轻微碰伤但可切用）：** 按实际损失重量打折结算（如8折），不强制补整箱。
- **严重不合格（如腐烂、变质、口感严重偏差）：** 供货方需在约定时间内补回合格品，且**补货运费、挑拣人工费由对方承担**。

**3. 设置“差额置换权”**
约定：**当不合格品影响当周出杯量时，门店有权要求乙方在下次送货时，以多送10%-15%合格品的方式，一次性补偿上周的营业损失。** 这样避免了频繁小额补货、人工成本过高的问题。

**4. 每日用量与动态补货机制**
- 要求供货方提前报备“可能存在的品质波动”，并约定**门店有权在到货后2小时内做全检**，超时视为合格。
- 长期合作下，可约定备用库存量：**门店保持1.5天的安全库存**，一旦发现不合格，立即启动安全库存，同时要求供货方在6小时内补货到位，确保安全库存不断。

**5. 断货违约金**
明确约定：**若因补货不及时导致门店停售某单品超过半天，供货方需按该单品当日预估营收的2倍进行赔偿**（该金额有据可查，可依据历史销售数据或门店营收流水）。这一条是让对方不敢拖延的核心。

**总结话术模板（与供应商沟通时用）：**
> “这次这批水果次品率太高，影响我今晚晚高峰出杯了。按照我们约定，你必须在2小时内给我补一箱同规格、同级别的合格货过来。如果补不到，今天这单品停售的损失按营收的2倍计算。另外，这批不合格果我按6折结算，你补货的运费你承担，行不行？”

**实操小贴士：**
- 建议每周固定两天收果，避开供应商集中出货日（如周一、周五），这时候货往往最差。
- 到货后立刻拍照、视频取证发到群里（最好有带时间水印的软件），对方难以推诿。
- 如果对方经常补货不及时，尽快寻找第二家供应商作为备选，不要完全依赖一家。
