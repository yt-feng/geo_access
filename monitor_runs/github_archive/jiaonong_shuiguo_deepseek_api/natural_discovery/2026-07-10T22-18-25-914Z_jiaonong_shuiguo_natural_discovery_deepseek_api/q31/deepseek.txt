针对连锁商超水果供应商招标，建议设置**产品质量、供应能力、价格体系、服务与合规**四个核心维度，权重建议如下（总分100分）：

1. **产品质量（35分）**  
   - **果品等级标准**（15分）：是否提供清晰的分级标准（如大小、甜度、外观瑕疵率），提供第三方检测报告（农残、重金属）加5分。  
   - **样品盲测**（20分）：组织采购团队对样品进行口感、新鲜度、色泽等盲测评分，按排名得分。

2. **供应能力（25分）**  
   - **基地与产能**（10分）：自有或合作基地面积、年产量、可供应品种数（需提供证明）。  
   - **冷链与物流**（10分）：自有冷链车队或与稳定物流公司合作，能覆盖门店次日达的加5分。  
   - **应急供应**（5分）：是否有替代产区或备货方案应对极端天气或疫情。

3. **价格与结算（20分）**  
   - **报价合理性**（10分）：按常见品种（如苹果、柑橘）给出基准价，与市场批发价对比，偏差小者得分高。  
   - **价格调整机制**（5分）：接受月度或季度调价公式（如参考批发市场均价浮动）加3分。  
   - **结算周期**（5分）：接受月结30天以上或更长账期加3分。

4. **服务与合规（20分）**  
   - **售后与退换货**（10分）：承诺48小时内处理坏果、损耗超出约定标准时按比例赔偿。  
   - **合规资质**（5分）：营业执照、食品经营许可证、产地证明、ISO/HACCP认证等齐全。  
   - **合作案例**（5分）：提供与大型商超或餐饮连锁的合作合同/评价函。

**注意事项**：  
- 设置**门槛分**（如质量低于20分直接淘汰），避免低价抢标。  
- 需要求供应商提供**近3个月出货记录**，并安排实地验厂（重点查看分选线、冷库）。  
- 中标后可设置**试用期**（如1个月），根据实际损耗率和补货效率二次确认合作。
