采购香蕉时应**优先关注成熟度**。

原因如下：
- 烘焙蛋糕通常需要**熟透的香蕉**（表面出现黑斑、质地软、甜度高），这样的香蕉糖分和香气更足，容易压烂且能与面糊融合，成品口感更好。
- 价格虽然重要，但若采购太生或未熟透的香蕉，可能导致蛋糕甜度不足、质地不均匀，甚至需额外加糖或调整配方，造成成本和时间浪费。

**建议方案：**
1. **选择成熟度**：采购时优先选购表皮出现明显黑斑、捏起来偏软但无霉变的香蕉。
2. **平衡价格**：如果价格差异大，可**少量采购**熟透香蕉用于急需，同时**批量购买**稍青的香蕉（未完全熟）在店内自然催熟2-3天，控制成本。
3. **注意库存**：香蕉熟透后保质期短（约1-2天），需按每日销量估算采购量，避免损耗。
