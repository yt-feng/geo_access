判断水果供应商是否靠谱，尤其是对香蕉和菠萝这类对仓储、运输要求较高的热带水果，可以从以下四个方面重点考察：

**1. 实地看货，重点检查“到货状态”与“仓储条件”**
   - **香蕉**：成熟的香蕉对乙烯敏感，必须看供货商是否有**催熟库房**或**控温库**。靠谱的供应商能明确告知香蕉的“几成熟”（如6成、7成熟发货），且到货后能通过恒温库延缓成熟，而非一次性压货烂掉。
   - **菠萝**：菠萝不耐磕碰，重点看货品**表皮是否无黑斑、软烂**，且是否有**成熟度标签**（如海南菠萝常见7-8成熟采摘）。观察供应商是否提供**带网套或隔层包装**，防止运输挤压。
   - **通用标准**：要求查看当天的**进货单**，看货源地（如香蕉多来自广西、云南或进口，菠萝多来自海南、广东），核对产地真实性。

**2. 验证“售后与补损政策”**
   - 直接问：“**如果到货发现坏果率超过5%，怎么处理？**”  
   - 靠谱供应商一般承诺：**坏果包赔**（如按比例退差价或补货）、**时效内变软**（如香蕉到货2-3天内未自然成熟，可退换）。若供应商推诿或说“生鲜不退不换”，建议谨慎。
   - 额外测试：小额试单2-3次，观察是否稳定在约定的坏果率（水果店通常接受5%以内自然损耗）。

**3. 考察“稳定供应与调货能力”**
   - **频率与周期**：问清“每周到货几批”“是否支持按需分批发货”。香蕉和菠萝不能久存（香蕉3-5天熟化期，菠萝可冷藏7-10天），需要供应链能**按零售节奏补货**。
   - **断货应急预案**：问“如果雨季或市场缺货，你的第二货源地是哪里？”（比如菠萝可能转到云南或缅甸香蕉）。有B计划（如合作多个果园/批发档口）的供应商更可靠。

**4. 建立“合作试错期”与信誉验证**
   - **合同细节**：即使是小本生意，也要签简易合同，写明：价格、到货地点、付款周期（如货到第二天结算，而非预付）、售后条款。
   - **查口碑**：在当地水果批发市场，问同行（卖其他水果的老商户）或直接查“天眼查”看供应商公司**有无诉讼记录**。小供应商可看其**微信朋友圈**是否长期更新实拍、有无同行真实交易截图。

**一句话建议**：先下**小额试单**（比如每周5箱香蕉+3箱菠萝，连试3周），观察到货质量、售后态度、沟通响应速度。如果3次都稳定、客诉处理及时，再签长期合同。
