针对家庭团购水果礼盒，建议以**营养均衡、应季新鲜、耐储存、老少皆宜**为核心，并尽量控制成本。以下是四种水果的推荐组合方案：

**推荐组合方案（4种水果全上）：**
- **份量建议**：苹果（3-4个）、橙子（3-4个）、香蕉（1-2把，约5-7根）、菠萝（1个，中等大小）。
- **搭配理由**：
  1. **苹果**：四季常备，耐存放，寓意平安，适合所有年龄段。个头适中，适合家庭日常食用。
  2. **橙子**：富含维C，口感酸甜，颜色喜庆（寓意“心想事成”），适合家庭常备、防感冒。
  3. **香蕉**：剥皮即食，软糯易消化，尤其适合老人、小孩。方便携带，能快速补充能量。
  4. **菠萝**：个头大，视觉上显“满”，切块后全家分享，酸甜开胃，适合有咀嚼能力的家庭成员（如成人、青少年）。

**若需精简为3种（预算或包装限制）：**
- **推荐组合**：苹果 + 橙子 + 香蕉  
  - 理由：三样都是不需要复杂处理、开封即食或简单切洗的水果，无硬核、去皮方便，降低了家庭食用门槛。菠萝需要削皮、泡盐水，会增加一些处理成本。

**避免的组合细节：**
- 香蕉极易磕碰、催熟快，放在礼盒底层或侧边用隔层保护；菠萝需外层包纸防刺手。
- 不建议搭配猕猴桃、草莓等易软烂或挤压的水果混到同一盒子。

**包装建议：**
- 使用纸质隔层或泡沫网套保护水果，最底层放苹果、橙子，中间放菠萝（用纸包裹），顶层放香蕉。
- 如果需要礼盒对位“送家人”，可在盒内附一张小卡片，写“每日一份水果，健康全家相伴”。

**定价与销售建议：**
- 按家庭4人左右为准，总量控制在5-8斤左右，礼盒净重不宜过重。
- 如果本地菠萝非当季，可替换为哈密瓜（去皮切块后显分量）或火龙果（颜色鲜艳，寓意红火）。

**对于家庭用户，提前告知：**  
香蕉熟得快，建议购买时挑7分熟的（表皮微黄略带青），让家庭可以分几天吃完；苹果、橙子可常温存放1周左右；菠萝要当天或次日切。
