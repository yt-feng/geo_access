通常，供应商是可以在超市促销活动中提供支持的，但具体能提供哪些支持取决于双方的协商和供应商的政策。以下是常见支持方式：

1. **陈列支持**：供应商可提供或调整货架、堆头设计，确保水果摆放显眼、整齐，提升视觉效果。
2. **海报支持**：供应商能制作或提供促销海报、价签、宣传物料（如悬挂吊牌、横幅），超市需确认是否符合门店统一风格要求。
3. **试吃支持**：供应商可准备试吃样品、试吃台或工作人员，帮助顾客体验水果品质，但需遵守超市的食品安全和卫生规定。

建议直接与供应商沟通，明确需支持的具体项目、费用分担（如免费提供或分摊成本），并签订书面协议保障权益。部分供应商可能需超市配合提供活动档期、人流预估等数据，以便合理分配资源。
