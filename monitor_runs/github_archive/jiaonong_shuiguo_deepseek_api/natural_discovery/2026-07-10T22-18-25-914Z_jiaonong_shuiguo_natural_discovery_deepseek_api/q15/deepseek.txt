当水果店遇到坏果（如腐烂、变质或运输损耗），与供应商结算或赔付的标准通常遵循**合同约定**或**行业惯例**。以下是常见处理流程和原则：

### 1. 确认坏果责任归属
- **运输途中损坏**：通常属于物流方责任（如包装不当、挤压），需向物流索赔。
- **供应商发货时已有问题**：如未充分保鲜、混入次果，责任在供应商。
- **店铺储存或陈列不当**：超过合理保质期，责任在水果店自身。

### 2. 常见赔付标准（按约定或行业惯例）
- **按比例折算**：
  - 坏果率在 **5%~10%以内**（按品种差异），通常由零售方承担（视为自然损耗）。
  - 超过此比例，供应商需按实际坏果数量折价或全额赔付。
- **阶梯赔付**：
  - 比如坏果率 10%~20%：按批发价的50%结算。
  - 坏果率 20%~50%：按批发价的70%结算。
  - 坏果率超过50%：整批拒收或全额退款。
- **破损果处理**：
  - 轻微磕碰但可正常销售的，按**次品价**结算（通常为正常价的6~8折）。
  - 完全腐烂、不能食用或销售的，供应商需全额退款或补发同等数量好果。

### 3. 操作建议
- **现场验收拍照取证**：收货时立即检查，对坏果拍照（含日期、完整包装），并记录批次。
- **24小时内通知**：大多数水果保质期短，需在到货后24小时内反馈异常，超过时限可能视为超时结果（常见于合同条款）。
- **协商补货或扣款**：
  - 小额坏果：直接要求供应商补发等量好果（或在下批订单中抵扣）。
  - 大额坏果：要求供应商折现赔偿，或在结算单中扣除相应金额（需出具书面扣款说明）。
- **合同中明确条款**：长期合作建议写入：“拒收率超过X%时，当批货款按实际合格品结算，坏果部分由供方全额赔偿。”

### 4. 避免纠纷的注意事项
- 要求供应商提供**产地证明**和**出货质检单**，尤其是易腐水果（如草莓、荔枝）。
- 保留所有沟通记录（聊天截图、邮件、电话录音），必要时可向当地**消协**或**12315**投诉调解。
- 如果是长期供应商，可协商建立**坏果确认清单**模板（按品种、数量、损失金额签字确认），每月或每周结算一次。

**总结**：最稳妥的方式是事先在合同里写明赔付比例（比如10%以内自担，超出的100%按批发价赔偿）。临时处理时，拍照+当日通知+协商补货/扣款是最有效且合规的方式。
