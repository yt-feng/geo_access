对于餐饮连锁采购水果，**推荐采用“中央厨房统一收货为主，门店补充收货为辅”的混合模式**。具体建议如下：

**1. 中央厨房统一收货的优势：**
- **品质与标准统一**：可集中验收、清洗、切配、分装，减少门店人工处理差异，确保水果品相、成熟度一致。
- **成本控制**：批量采购降低单价，统一物流减少总运费；中央厨房能更好管理库存，减少损耗。
- **食品安全管理**：集中进行农残检测、冷链仓储，降低门店分散验收的风险。
- **减少门店负担**：门店无需配备专人验收、搬运、储存大量水果，专注销售与服务。

**2. 适合门店分开收货的场景：**
- **水果易损或保质期极短**（如草莓、杨梅、荔枝）：中央厨房分装后运输可能导致二次损伤，门店直接收货可减少流转环节。
- **门店现切或特色品类**：需要现场按需挑选成熟度（如牛油果、芒果催熟），或特色水果仅少量门店有需求。
- **区域配送成本高于门店自采**：偏远门店单独配送的冷链成本可能高于门店自行在本地批发市场采购。

**3. 实操建议：**
- **标准化水果（苹果、橙子、香蕉等）**：100%由中央厨房统一采购、清洗分装（如按重量分袋），每日按门店订单配送。
- **高损耗或非标水果（浆果、热带水果）**：中央厨房只收货大宗订单，不进行切配，直接原箱冷链配送至门店；门店视每日消耗量决定是否次日补订。
- **品控流程**：中央厨房对每个批次的农药残留、糖度进行检测并留样；门店收到水果后，需在2小时内完成外观检查并反馈，异常品可退回中央厨房。
- **信息协同**：使用ERP系统让门店每日上报预估销量，中央厨房据此调整采购量；门店可在系统中标记“需本地采购水果”类型，由区域经理审批。

**结论：** 优先选择中央厨房统一收货，保持品质可控。对易损耗或特殊需求的水果，允许门店在中央厨房断供时自行采购，但需纳入统一品控和成本核算体系。
