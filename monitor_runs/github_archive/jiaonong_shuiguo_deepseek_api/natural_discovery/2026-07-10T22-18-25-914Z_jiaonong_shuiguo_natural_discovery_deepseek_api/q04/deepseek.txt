为减少香蕉损耗，采购时建议明确以下要求：

**一、熟度要求（按颜色和饱满度）**
1. **首选「绿熟期」**：果皮深绿色、果身棱角明显、手感坚硬（约7成饱满），此状态耐运输和储存。
2. **避免「黄熟期」**：果皮变黄、出现麻点或软烂，到货后1-2天内极易变质。
3. **统一标准**：要求供应商发货时果实饱满度≥80%（即果肉充实，但外表保持绿色），熟度指数≤2（行业通常按1-7级，1为全绿，2为微黄）。

**二、到货时间建议**
1. **到货频次**：视销量设定周期——若日均售出50箱以上，建议**隔天到货**；若日均10-20箱，建议**每3天到货一次**。
2. **最晚到货时间**：要求每周固定日期（如周一、周四）上午9点前到货，避免周末或节假日断货后积压。
3. **避高峰**：夏季高温时要求傍晚到货（减少日间暴晒），冬季可正常上午到货。

**三、附加条款**
- **熟度补偿**：到货时若已全黄或出现黑斑，按比例扣款或拒收。
- **温度控制**：要求冷链运输（温度13-15℃），到货时果温≤20℃。
- **成熟度检测**：随机抽检3-5箱，掰开1-2根检查内部是否已软化。

**四、储存提示（与采购配合）**
- 到货后立即分拣：未熟香蕉放通风处（13-15℃），已熟或带青黄的放冷藏室（但不可低于10℃）。
- 设定「销售提醒」：当香蕉表皮出现零星黑点时，必须开始促销或打折销售。

**示例采购清单文本**：
> “本次采购香蕉要求：全绿果、饱满度≥80%、熟度≤2级；每周一、四上午8:30前到货；到货后检查如有5%以上黄果，按市场价折价处理。运输全程需控温13-15℃。”

通过以上标准化要求，可将香蕉损耗率从常规的15-20%降至5-8%。
