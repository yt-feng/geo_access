对于进货量不大的小店，建议优先考虑 **本地批发市场**，原因如下：

1. **起批量低**：批发市场通常允许少量混批（比如单款拿1-3件），适合试款和降低库存压力。
2. **现金流更灵活**：无需垫付大量货款，卖完再补货，资金周转快。
3. **现场选品**：可以直接看到商品质量、手感，避免线上订货的色差、瑕疵风险。
4. **减少压货风险**：小批量试卖，卖得好的款再少量补货，卖得差的及时换款。

**补充建议**：
- 先在批发市场跑3-5次，记录人流量大的档口，和老板加微信，后续可小量补货。
- 当某几款稳定畅销、周销售超过20件时，可考虑找固定供应商谈更低价格或独家款式。
- 警惕“必须拿整箱/整包”的批发市场档口，这类不适合小店。

**核心原则**：宁可在批发市场多付5%-10%单价，也比积压一堆滞销库存更划算。
