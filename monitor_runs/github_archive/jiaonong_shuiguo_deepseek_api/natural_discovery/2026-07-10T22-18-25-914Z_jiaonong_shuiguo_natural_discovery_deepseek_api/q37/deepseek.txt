商超做会员日水果活动时，供应商需要提前配合以下关键准备工作，以确保活动顺利执行和效果最大化：

1. **库存与供应准备**  
   - 根据商超会员日预估销量，提前备货（尤其是爆品水果），避免断货。  
   - 协调物流，确保活动前夜或当日清晨保证新鲜到货，并预留补货通道。

2. **品项与价格策略**  
   - 与商超确认活动专享价、会员折扣或满减规则（如“会员专享5折”）。  
   - 提供高性价比的引流款（如时令水果）和利润款（如进口水果），配合试吃或组合装。

3. **陈列与道具支持**  
   - 准备专用陈列道具：如堆头底座、宣传海报、LED价签、试吃台。  
   - 设计醒目的会员日标识（如“仅限会员日价”），并配合商超动线调整位置。

4. **人员与现场服务**  
   - 安排驻场促销员（数量根据门店规模），负责试吃、导购、补货及剥切服务（如切水果试吃）。  
   - 培训促销员话术，突出“会员专属优惠”和水果新鲜度。

5. **质量与损耗管控**  
   - 提供品质稳定的货品，并预留3%-5%的损耗余量（如坏果换新）。  
   - 与商超约定损耗承担比例（如活动结束后未售出水果的退换方案）。

6. **宣传与引流配合**  
   - 提前向商超提交活动素材（如海报、短视频、宣传口号）。  
   - 支持商超的社群预热：提供优惠券、限时秒杀链接或会员生日礼券。

7. **应急预案**  
   - 备好冷柜或冰块，应对高温天气防止水果腐坏。  
   - 遇突发缺货时，24小时内协调周边门店调货或紧急补单。

**建议节点**：活动前5天提交备货计划，前3天完成门店陈列布局确认，前1天进行人员培训及试吃样品准备。
