判断批发档口的香蕉供应商是否稳定，可以从以下几个方面入手：

1. **实地考察与仓储**  
   - 要求参观对方的仓库或分拣场地，观察库存是否持续有货、保鲜设备（如冷库）是否完善。  
   - 看是否有长期合作的种植基地或固定的进口渠道（如菲律宾、厄瓜多尔等主产区的一级代理权）。

2. **资质与历史交易**  
   - 查验营业执照、食品经营许可证，确认经营年限（3年以上相对可靠）。  
   - 要求提供近半年的采购订单、海关报关单（进口香蕉需检验检疫证明）或产地收购凭证。

3. **供货稳定性测试**  
   - 先下小批量试单（如1-2吨），观察发货时间、到货品质（成熟度、有无黑斑）、尾款支付流程是否顺畅。  
   - 连续下单3-5次，看能否在约定时间内（如24小时内）稳定补货。

4. **合同关键条款**  
   - 要求签订书面合同，明确：  
     - **保供条款**：如每月最低供应量、缺货赔付标准（例如每吨按市场价20%违约金）。  
     - **品质标准**：成熟度（如7-8成熟）、单果重量（如150g以上）、无机械损伤等。  
     - **不可抗力**：自然灾害、港口封锁等突发情况下的免责或补偿方案。

5. **行业口碑验证**  
   - 联系当地批发市场管理方（如嘉兴水果市场、广州江南市场）或同品类商户，侧面询问该供应商的信用记录。  
   - 查看电商平台（如惠农网、一亩田）的实名评价，重点看差评是否集中在“断货”“货不对板”。

6. **供应链韧性**  
   - 问清对方货源多样性和替代方案：例如是否同时对接广西、云南及东南亚多个产地，是否自备冷链物流车。  
   - 测试应急能力：比如突然追加50%订单，对方能否在2天内调货到位。

**快速判断清单（直接可执行）**  
- ✅ 检查：营业执照注册时间 ≥ 3年  
- ✅ 试单：连续3次小批量订单无断货  
- ✅ 合同：有量化缺货违约金条款  
- ✅ 备份：对方至少提供1个备用产地联系人  

**反例警示**：如果有供应商宣称“每天无限量供应”却无法出示任何产地证明，或要求预付总货款80%以上，需高度警惕。
