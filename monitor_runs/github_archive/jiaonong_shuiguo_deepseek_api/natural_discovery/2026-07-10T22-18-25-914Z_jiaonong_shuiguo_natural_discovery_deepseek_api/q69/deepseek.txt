针对批发商给小店供货，香蕉和菠萝的分级报价可以遵循以下清晰、可操作的标准，便于双方快速达成交易：

### 一、香蕉分级报价建议

**1. 常见分级标准（按成熟度、外观、大小）：**

- **特级果（A级）：**
  - 条件：果实饱满、无机械伤、无黑斑、无腐烂，果柄完整，成熟度均匀（如六至七成熟），长度在18cm以上。
  - 报价：按“箱”或“斤”报价，价格通常比普通果高15%-20%。

- **一级果（B级）：**
  - 条件：允许有轻微擦伤或少量黑点，果形基本均匀，长度15-18cm，成熟度稍有不一。
  - 报价：价格比特级果低10%-15%。

- **次品/处理果：**
  - 条件：有明显疤痕、部分腐烂（需处理）、过熟或未熟，长度不均。
  - 报价：按“筐”或“堆”报价，价格约为特级的50%-70%，需注明“需当天售出”。

**2. 报价技巧：**

- 按**重量**报价（如3元/斤）更常见，但需标注成熟度等级（如“六成熟黄香蕉”）。
- 按**箱**报价（如40元/箱/20斤）适合标准包装，需注明每箱净重。
- 提示小店：如果接受混装（如特级+一级混批），可给一个“混合价”。

### 二、菠萝分级报价建议

**1. 常见分级标准（按重量、外观、成熟度）：**

- **特级果（A级）：**
  - 条件：果实圆润、芽眼平整、无腐烂、无裂口，重1.2-1.5斤/个，果皮金黄（七成熟），顶端绿叶新鲜。
  - 报价：按“个”或“斤”报价，价格比普通高20%-30%。

- **一级果（B级）：**
  - 条件：果形稍偏长或偏小（重0.8-1.2斤/个），允许有轻微日灼斑或少量软点，果柄略干。
  - 报价：价格比特级低10%-15%。

- **次品/特价果：**
  - 条件：果身有裂口（需切割）、有腐烂点、过熟或未熟、重量不足0.8斤。
  - 报价：按“筐”或“个”低价处理，建议标注“需当天加工或促销”。

**2. 报价技巧：**

- 按**个**报价（如5元/个）更直观，适合小店零售；按**重量**报价（如4元/斤）需标注平均单果重量。
- 报价时注明“包甜”或“不熟包换”，可提高交易信任。
- 对于混级果（如大小不均但无瑕疵），可以给“整筐价”或“打包价”。

### 四、实用建议（给批发商）

1. **统一包装标准**：使用相同规格的纸箱或塑料筐，并标明净重、等级、产地。
2. **定期更新报价**：每周或每批次更新一次，避免因市场波动产生争议。
3. **提供样品展示**：让小店店主现场看样，口头说明分级依据（如“这种黑点是正常的，不影响口感”）。
4. **注意保鲜提示**：在报价单上注明“建议冷藏”或“保存温度”，减少水果损耗。

通过以上分级和报价方式，小店店主能快速判断成本与售价空间，有助于长期合作。
