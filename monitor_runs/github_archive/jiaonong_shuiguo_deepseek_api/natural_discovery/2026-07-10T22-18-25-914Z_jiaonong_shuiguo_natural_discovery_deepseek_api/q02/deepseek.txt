进香蕉时，优先级建议是：**熟度 > 坏果率 > 价格 > 送货速度**。

具体原因和操作建议如下：

1. **熟度（最关键）**  
   - 香蕉是后熟水果，进货时的熟度直接决定售卖窗口期。  
   - **建议**：进“黄中带青”或“青头绿把”的香蕉（约七成熟）。太熟（全黄）当天卖不完会损耗，太青则顾客很难接受。  
   - 操作：根据你的日销量和冷柜条件，选择不同熟度的批次。比如销量大的店可以少量进全黄熟果搭配青黄果。

2. **坏果率**  
   - 香蕉在运输中极易因挤压、撞伤产生黑斑或腐烂。  
   - **建议**：开箱时逐把检查，重点关注“扇蕉”（受损导致的畸形果）和“指根断裂”处。坏果率超过3%可拒收或要求扣款。  
   - 经验：同一批次不同箱可能质量差异大，最好随机抽检2-3箱。

3. **价格**  
   - 香蕉是民生水果，价格敏感但利润薄，但不宜为了便宜进劣质货。  
   - **建议**：在确保前两项的前提下，对比价格。优先选择“产地直发”或“一级批发商”，避开多次倒手的中间商（可能混入老货）。

4. **送货速度**  
   - 香蕉需要恒温储运（13-15℃），过慢或非冷链运输会导致提前成熟或冻伤。  
   - **建议**：要求头天下单、次日到货。如果供应商无法保证36小时内送达，或经常延迟，考虑换人——因为这会打乱你控制熟度的节奏。

**总结**：  
- **最怕的情况**：进了一车熟透/坏果多的香蕉，即便便宜、送得快，也会烂在店里。  
- **可执行的检查清单**：到货后按“熟度→坏果率→称重验价→拆箱入库”的顺序操作，先否决不合格的批次，再谈价格。
