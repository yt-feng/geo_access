对于企业食堂采购水果，这三个方面都很重要，但建议优先关注的顺序是：**食品安全 > 价格 > 配送准时**。

原因如下：

1. **食品安全是底线**：一旦出现食品安全问题（如农药残留超标、水果变质、异物混入等），可能导致员工食物中毒、企业声誉受损，甚至面临法律风险。因此，必须优先确保供应商有相关资质（如食品经营许可证、水果农残检测报告等），最好能实地考察或要求提供批次检测证明。

2. **价格影响成本控制**：在保证安全的前提下，价格直接影响食堂运营成本或员工福利。建议通过比价、长期合作合同或季节性采购（如当季水果通常更便宜）来控制成本，同时注意避免选择价格过低的产品，以免牺牲品质。

3. **配送准时影响运营效率**：若水果配送延迟，可能导致食堂无法按计划供应，打乱备餐流程。但通常可以通过与供应商约定明确的时间窗口、设置应急备用供应商等方式来管理，相比安全和价格，其风险可控性更高。

**综合建议**：筛选供应商时，优先审核其食品安全资质和抽检记录；其次对比多家报价，选择性价比合理的；最后通过合同明确配送时间及延误处罚条款。
