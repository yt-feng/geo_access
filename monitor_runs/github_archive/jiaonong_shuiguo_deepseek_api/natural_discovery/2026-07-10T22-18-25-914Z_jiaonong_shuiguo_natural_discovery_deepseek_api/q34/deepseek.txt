在商超采购水果时，**损耗低对最终毛利的影响通常大于价格低**。原因如下：

1. **损耗直接侵蚀毛利**：水果易腐坏，损耗率（如破损、变质、顾客挑拣造成的损失）可能高达10%-30%。即使采购价格低，若损耗过高，实际销售成本会大幅上升，导致毛利下滑甚至亏损。例如，采购价降低5%，但损耗增加10%，最终毛利反而可能减少。

2. **价格优势有限**：水果市场价格相对透明，压低采购价的空间有限，且过低价格可能影响品质，进一步增加损耗风险。而通过优化供应链、选择耐储存品种、改进陈列和保鲜措施，可以将损耗控制在更低水平。

3. **综合效益考量**：低损耗能保证更多可销售商品，提升周转率，减少管理成本（如频繁补货、处理报废品）。同时，稳定的品质还能增强顾客满意度，促进复购。

**建议**：优先关注损耗控制（如采摘后快速冷链、分类包装、合理陈列），在保证品质的前提下再争取价格优势。例如，采购成熟度适中的水果（减少运输损耗），并采用小包装销售（降低顾客挑拣导致的损耗）。
