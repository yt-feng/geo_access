小商家采购水果时，现金结算和月结账期各有以下风险，建议根据实际情况权衡：

**现金结算风险：**
1. **资金压力**：需一次性付清全款，可能占用大量流动资金，影响其他经营环节。
2. **无账期缓冲**：若后续发现水果质量问题（如腐烂、缺斤少两），追讨已付现金较困难。
3. **交易记录依赖自身**：缺少正式发票或合同保护，发生纠纷时举证较难。

**月结账期风险：**
1. **拖欠风险**：供应商可能因经营不善或恶意行为，拖延退款或缺货，您可能需要承担前期垫付的损失。
2. **信誉依赖性**：若您自身资金周转慢，月结可能影响供应商关系，甚至被停止供货。
3. **隐性成本**：供应商可能为弥补账期延长，提高单价或降低水果品质（如混入次品）。

**建议**：
- **小批量试合作**：对首次交易的供应商，优先现金结算少量货物以验证品质和信誉。
- **分阶段付款**：月结可协商按“签收后付50%、月底清款”等方式降低风险。
- **书面协议**：无论哪种方式，均要求供应商出具明确数量、单价、验收标准及违约条款的书面凭证。
- **备选供应商**：培养2-3个固定合作商，避免单一依赖月结或现金结算。
