大部分水果供应商是可以提供价格或物料支持的，特别是针对周末促销这种有明确时间节点和销售潜力的活动。具体支持方式和可能性包括：

1. **价格支持**：
   - **进货折扣**：如果销量大，供应商可能会给短期降价或满减，比如拿货满一定金额打9折。
   - **斤两补贴**：部分供应商愿意额外赠送几箱货，用来算作促销时的损耗或赠品。

2. **物料支持**：
   - **宣传物料**：如海报、折页、展架，甚至印有品牌logo的包装袋。
   - **陈列物料**：如水果筐、展台、保鲜设备（部分会借出）。
   - **试吃品**：如果周末主推某一款新品或应季水果，供应商有时会免费提供试吃装。

3. **如何争取**：
   - **主动提前沟通**：至少提前1周联系供应商，说明促销时间、预计订单量、重点推广的产品，提供来客数或历史销售数据会更有说服力。
   - **用“长期合作”做筹码**：明确表示此促销会增加出货量，并承诺后续会持续采购，很多供应商会愿意让利支持。

**实操建议**：  
可以先列一个清单，写明你计划促销的水果、预估销量和需要的物料类型，发给平时合作较多、出货量大的供应商，问“这次周末促销你们能提供哪种支持？”多数会给出积极回应。

没有拿到确认前，不要提前承诺给顾客的价格或赠品，以防供应商临时变卦。
