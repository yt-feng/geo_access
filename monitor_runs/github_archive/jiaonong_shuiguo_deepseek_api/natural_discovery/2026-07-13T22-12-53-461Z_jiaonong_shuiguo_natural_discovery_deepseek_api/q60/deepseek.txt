针对茶饮店水果到货不合格的情况，建议按以下步骤处理补货，最大限度减少对营业的影响：

1. **立即检查并分类**：到货后2小时内完成外观、新鲜度、口感检查。对不合格水果（如腐烂、过熟、虫蛀）拍照留证，按严重程度分为“可筛选使用”（如轻微碰伤可切除部分）和“完全不可用”。

2. **紧急沟通供应商**：第一时间通知供应商，要求：
   - **当日或次日上午补发**：明确补货时间，要求优先配送（可支付加急费）。
   - **补货标准**：要求新批次水果符合合同约定的规格（如甜度、大小、成熟度）。
   - **责任划分**：明确不合格造成的损耗由供应商承担（扣减货款或补货免费）。

3. **启动应急预案**：
   - **调整菜单**：暂时下架使用该水果的饮品，换用库存充足的备选水果（如柠檬换青柠、草莓换芒果）或推出“季节限定特调”。
   - **采购替代品**：如果供应商无法快速补货，立即从本地批发市场或生鲜平台采购少量应急，确保核心饮品不断供。
   - **限量供应**：对热门饮品实施“每日限量”或“售完即止”，集中使用合格批次。

4. **建立补货SOP**：与供应商签订《补货保障协议》，约定：
   - 送货前24小时发送货物照片/视频确认质量。
   - 不合格品触发“24小时补货免责权”（即无需重复支付运费）。
   - 每月累计不合格率超过5%时，启动违约条款（如价格折扣或更换供应商）。

5. **备选方案**：与2-3家供应商保持“周结月结”的临时小额采购关系，作为应急补充。同时培养员工快速处理品控问题的能力（如分拣标准培训）。

最后，所有沟通记录（微信/邮件/电话录音）、照片、补货单据均需留档，以便后续索赔或协商。
