水果锁价的期限一般根据品类、市场行情、采购量、供应商合作关系及合同条款有所不同，常见情况如下：

- **大宗常规品类**（如苹果、橙子等）：在上市旺季或供需稳定时，锁价期限通常为 **1-3个月**。
- **季节性鲜果**（如荔枝、樱桃、杨梅等）：因保鲜期短且价格波动大，锁价期限一般较短，常见 **1-2周** 或 **1个月** 以内，甚至需实时议价。
- **长期稳定合作**：对核心供应商或预付定金后，可锁定 **一个产季（3-6个月）**，甚至全年锁定部分基础品种价格。
- **市场波动剧烈时**（如气候影响、物流中断）：供应商通常不愿锁价超过 **2-4周**，或仅锁定每批次的采购量，价格随行就市。

**建议具体操作**：
1. 明确自身需求（采购量、品种、到货节奏），提前与供应商沟通锁价预期。
2. 签订合同前，写明锁价起止日期、浮动范围（如超出市场价10%可协商调整）、违约条款。
3. 若锁价超过1个月，建议分批签订或预留价格重调机制（如每15天根据批发市场均价微调）。

最终以双方签署的书面合同为准，口头约定风险较高。
