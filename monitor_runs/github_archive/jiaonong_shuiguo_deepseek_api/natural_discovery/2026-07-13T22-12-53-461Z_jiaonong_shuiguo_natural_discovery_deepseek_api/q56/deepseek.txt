茶饮店在与水果供应商合作时，为确保食品安全，应要求供应商提供以下关键证明文件（建议保留复印件或电子存档）：

1. **营业执照**：确认供应商为合法注册经营主体，经营范围包含“水果销售”或“农副产品”等。
2. **食品经营许可证**（如适用）：若供应商从事预包装水果分装或配送，需提供此证（部分地方对生鲜销售是否需此证有不同规定，建议核实当地法规）。
3. **农产品质量检测报告**：包括农药残留（重点检测项目）、重金属、微生物等指标，且报告需由具备CMA或CNAS资质的第三方检测机构出具，并在有效期内（通常每批或每季度更新）。
4. **产地证明或供货凭证**：记录水果来源（如种植基地名称、地址）、采摘日期、批次信息，便于追溯。
5. **承诺书/质量保证协议**：供应商书面承诺所供水果符合国家食品安全标准（如GB 2763-2021《食品安全国家标准 食品中农药最大残留限量》）。
6. **运输与仓储证明**：冷链运输记录或温度监控数据（如鲜切水果或易腐品种）、仓储环境符合卫生要求的证明（如消毒记录）。

**额外建议**：
- 定期核对供应商的证照有效期，并建立“索证索票档案”，每批水果到货时检查随附的送货单与检测报告是否一致。
- 若使用进口水果，还需索要海关报关单、入境货物检验检疫证明（标明品种、原产地、生产日期）。
- 对供应商进行现场审核，重点关注水果分拣、包装、运输环节的卫生状况。

**法律法规依据**（供内部参考）：
- 《食品安全法》第50条：食品经营者采购食品需查验供货者许可证和合格证明。
- 《农产品质量安全法》第38条：农产品批发市场、销售企业应查验产地证明或检测报告。

通过严格审查供应商资质，可有效降低因原料问题引发的食品安全风险，建议结合门店自检（如快速检测卡）形成双重保障。
