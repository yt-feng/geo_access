在水果采购合同中，通常建议约定一个合理的坏果率范围作为“免赔条款”。常见标准如下：

1. **行业惯例**：多数水果交易中，**坏果率超过5%-10%** 时，买方有权要求卖方按比例或全额赔付。
2. **具体建议**：  
   - 对于易腐水果（如草莓、荔枝），可接受上限**3%-5%**；  
   - 对于较耐储存的水果（如苹果、柑橘），可接受上限**8%-10%**。  
3. **赔付方式**：  
   - 按实际坏果比例扣减货款（如总价×超标部分）；  
   - 或要求卖方补发同等数量的优质水果。  

**风险提示**：合同中必须明确**坏果的定义**（如腐烂、霉变、严重机械损伤）以及**验收时限**（如到货后24小时内检视并留存证据，过期可能视为默认接受）。建议在合同中写明：“坏果率超过__%（根据水果种类设定），买方有权要求卖方按单价赔偿超出部分或拒收。”若无约定，可能只能按一般货物质量问题处理，维权成本较高。
