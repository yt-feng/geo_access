采购时应该**优先关注成熟度**，兼顾价格。原因如下：

1. **产品品质决定口感**：香蕉蛋糕需要熟透的香蕉（表皮出现黑斑、手感软糯）来提供天然甜味和湿润质地。太生的香蕉甜度低、质地硬，做出来的蛋糕口感差；过于腐烂的香蕉可能影响风味和食品安全。
2. **成本换算**：成熟度过高的香蕉通常价格更低（甚至处理价），但必须确保没有局部发霉或异味。如果为了低价购入未熟香蕉，需要额外催熟时间（可能导致使用不及时），整体成本可能更高。
3. **采购建议**：
   - 设定**最低成熟度标准**：香蕉皮至少70%转黄或出现少量黑点，果肉软而不烂。
   - 根据使用计划采购：若未来3天内要用，选成熟度80-90%的香蕉；若需存放，选成熟度50-60%的（自然催熟）。
   - 价格应在成熟度达标后进行对比，优先级为：达标成熟度 > 价格合理 > 供应商稳定性。

最后，可以优先找能稳定供应“熟透但未烂”香蕉的供应商（如面包店、甜品店常用渠道），长期合作常能拿到性价比更高的批次。
