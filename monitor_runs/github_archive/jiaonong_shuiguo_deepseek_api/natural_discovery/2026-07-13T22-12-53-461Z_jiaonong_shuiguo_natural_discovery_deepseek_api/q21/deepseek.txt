在超市采购水果时，供应商通常需要提供以下检测报告或进货证明，以确保水果符合食品安全和品质要求：

1. **产地证明或进货凭证**：证明水果来源合法，例如由产地或批发市场出具的产地证、销售凭证、运输单据等。

2. **农药残留检测报告**：需由具备资质的第三方检测机构出具，显示农药残留量符合国家或地区标准（如中国GB 2763标准）。

3. **质量检验报告**：检测项目可能包括感官指标（颜色、大小、成熟度）、理化指标（糖度、酸度）等，由专业机构出具。

4. **入境货物检验检疫证明**（如为进口水果）：需提供海关出具的证明，确认水果已通过检疫并符合进口要求。

5. **特殊认证或文件**：若水果为有机、绿色食品等，需提供相应认证证书；对于鲜切水果，可能需要提供保质期测试报告等。

6. **发票或合同**：作为交易证明，通常要求与检测报告对应，确保信息一致。

**建议**：在采购前与供应商明确要求，并索要上述文件原件或复印件。定期更换或更新报告（如每批次或每月），以便留存备查。根据当地法规（如中国食品安全法）要求，超市应执行进货查验记录制度，保存相关证明至少2年。
