在商超采购香蕉时，判断其催熟能力是否稳定，可以从以下几个关键方面入手：

1. **观察香蕉的成熟度一致性**：
   - 同一批次的香蕉果皮颜色、硬度、果柄状态应基本相近。如果颜色差异大（如有的青绿、有的已出现黑斑），说明催熟过程不均匀或陈果混入。
   - 用手轻轻捏果体，成熟的香蕉应软硬适中；若同一箱中有的过硬、有的过软，说明催熟不稳定。

2. **检查果柄和果皮状态**：
   - 果柄应为新鲜绿色，无发黑、干枯或霉变。果柄发黑可能意味着催熟剂过量或存放时间过长。
   - 果皮无大面积黑斑、水渍状或皱缩。少量芝麻状斑点属正常，但若出现大片黑色斑点或果皮发暗，可能是催熟过度或冷害。

3. **闻气味**：
   - 打开包装箱或近距离闻香蕉，应散发天然香蕉香气，无刺鼻的化学药剂味（如氨水、酒精或乙醛味）。若有异味，可能催熟剂使用不当。

4. **看催熟方式（可询问供应商）**：
   - 了解是否采用**乙烯利**（常用且安全）催熟。规范的催熟会控制浓度（通常500-1000ppm）、温度（18-22℃）和湿度。
   - 非正规催熟可能使用过量乙烯利或未控温，导致香蕉“外黄内硬”或易腐烂。

5. **做小批量试熟测试**：
   - 少量购买后放置于室温（20-25℃）观察：催熟稳定的香蕉应2-3天内均匀变黄，果肉变软，甜度上升；若出现果皮变黄快但果肉生硬、或很快腐烂，说明催熟调节不当。

6. **查验供应商资质与历史记录**：
   - 选择有冷藏链、恒温催熟库的供应商，并查看其过往供货的稳定性（如退货率、客诉记录）。催熟稳定往往与存储、运输、库存周转能力相关。

**总结：优先选择颜色一致、果柄青绿、气味清爽的香蕉，并做好小范围试熟。若经常出现“外表金黄内部硬”或“迅速发黑腐烂”，应更换供应商或要求调整催熟工艺。**
