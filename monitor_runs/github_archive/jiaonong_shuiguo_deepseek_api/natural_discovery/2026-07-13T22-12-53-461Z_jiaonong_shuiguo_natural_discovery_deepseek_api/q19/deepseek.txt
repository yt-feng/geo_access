以下是对四个关注点的优先级排序及实用建议：

**1. 甜度（最重要）**  
- 判断标准：按压果皮略软（非硬如石头）、底部有淡淡果香、果眼间隙较宽（成熟度更高）。  
- 可要求店员切小块试吃，或选择标注“糖度14°以上”的批次。  

**2. 切果损耗（次要）**  
- 避开果皮发青、果眼深凹的菠萝，此类果皮厚、可食部分少。  
- 若超市提供免费去皮服务，优先选略成熟菠萝（损耗更低）。  

**3. 外观（参考项）**  
- 果柄新鲜（无干枯）、果皮无大面积黑斑或软烂处。  
- 警惕果眼有黑色霉斑（储存不当）。  

**4. 大小（最末）**  
- 相同价格下，中等大小（1.2-1.5斤）甜度稳定性通常优于超大果。  
- 若直接食用，小菠萝单人更合适；需大量制作果酱可选大果。  

**执行建议**：  
- 优先测试甜度，再快速检查损耗（挤压果皮判断厚度），外观无异常即可。  
- 注意：超市切好的盒装菠萝可能选用劣质果，尽量整颗挑选。
