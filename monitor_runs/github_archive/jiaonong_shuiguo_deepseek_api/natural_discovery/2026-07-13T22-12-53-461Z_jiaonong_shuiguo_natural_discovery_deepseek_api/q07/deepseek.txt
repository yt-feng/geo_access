根据水果店畅销经验及视觉搭配原则，推荐以下3种高性价比、高颜值的组合方案：

**方案一：经典红黄配（主打喜庆、送礼）**
- **搭配**：苹果（红富士/嘎啦）4个 + 橙子（脐橙/血橙）3个 + 香蕉2串（小芭蕉更精致） + 菠萝1个（小菠萝切半或整颗）
- **亮点**：苹果寓意“平安”，橙子代表“心想事成”，整体节日感强，适合春节、中秋、探病送礼。
- **提示**：香蕉易熟易碰伤，建议用气泡膜单独包裹后放顶层；菠萝切去刺头、顶部保留绿叶提升卖相。

**方案二：清爽热带风（主打颜值、年轻客群）**
- **搭配**：青苹果（青森/金冠）4个 + 橙子（甜橙）2个 + 香蕉2根 + 菠萝（加1个百香果/青柠点缀）
- **亮点**：颜色层次（青、橙、黄、绿），适合夏季、下午茶场景，可在盒内放一张“异域风情”卡片。
- **成本优化**：青苹果通常比红富士便宜，菠萝选中等个头（约1.5斤）控制总价。

**方案三：健康轻食款（主打减脂、办公场景）**
- **搭配**：橙子3个 + 香蕉2根 + 小菠萝1个（切片/切块）+ 柠檬/猕猴桃（替换苹果）
- **说明**：剔除苹果（糖分稍高），用柠檬片装饰提亮，香蕉建议选未全熟（青黄表皮）可存放更久。
- **包装建议**：用竹编篮+牛皮纸垫底，突出天然感，定价可上浮20%-30%。

**通用销售技巧**：
1. **控制总重**：3-4种水果礼盒建议总重4-5斤（含包装），价格区间在38-68元（视本地消费水平调整）。
2. **品质优先**：香蕉选无黑斑、苹果无磕碰，橙子表皮光滑有香气，菠萝底部硬实（不要闻起来有酒味）。
3. **增值服务**：附赠一张小卡片（写祝福语）、食用说明（如菠萝浸泡盐水后吃）、可搭配几颗薄荷叶提升质感。
4. **试水步骤**：先小批量组合3-5盒放店门口测试，观察顾客询问频率和时间段（如下午3-5点下班高峰），根据反馈调整搭配。
