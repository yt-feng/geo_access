选择适合批量配送的供应商，可以按以下步骤操作：

1. **明确需求**  
   - 确定预算、采购量（如每周/每月的件数）、配送频率（如每周一次）和配送时间（如固定周三上午）。
   - 列出常用水果清单（如苹果、橙子、香蕉等耐储品种，避免高损耗的草莓、蓝莓等）。

2. **筛选供应商类型**  
   - **批发市场**：当地大型水果批发市场（如北京新发地、上海辉展等），可现场比价、看货，但需自行取货或协商配送。
   - **生鲜电商B2B平台**：如美菜网、快驴（美团旗下）、永辉彩食鲜等，支持线上选品、批量配送，通常起送量较低。
   - **本地果品配送公司**：通过58同城、饿了么企业购等平台搜索“水果配送公司”，优先选有企业客户案例的。

3. **考察关键指标**  
   - **品质保障**：要求供应商提供样品试吃，或查看配送实拍图。了解其是否提供“损耗包赔”（如收到坏果可退换）。
   - **配送能力**：确认是否覆盖公司地址、能否满足配送时间窗口（如上午9点前送达）。建议要求试配送一次。
   - **价格透明度**：对比单斤价、起送价和配送费（部分供应商有免配送额度）。要求提供含税票报价。
   - **售后响应**：询问退换货流程和时效（如坏果24小时内补发）。

4. **签订合同前**  
   - 索要供应商营业执照、食品经营许可证（若配送切好的果盘）。
   - 明确合同条款：收货标准（如重量误差≤5%）、付款周期（月结/周结）、违约责任（如配送延迟扣款）。

5. **试点合作**  
   - 先尝试1-2次小批量采购，观察水果新鲜度、配送准时性和售后处理效率。若稳定，再转为长期合作。

**避免踩坑**：  
- 不要只看最低价（可能牺牲品质），优先选有企业客户合同的供应商。  
- 如公司人数超50人，建议至少对比2-3家供应商，并索要报价单。  
- 保留每次收货记录和沟通截图，用于后续纠纷处理。
