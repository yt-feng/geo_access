在合同中明确约定坏果责任，可以有效规避批发商的损耗风险。建议在合同中加入以下条款：

1. **验收标准与期限**  
   明确约定：收货方应在货到后【XX小时/日内】完成验收，如发现坏果（腐烂、霉变、严重压伤等），需在验收期内书面或拍照记录通知发货方。

2. **坏果界定标准**  
   列明坏果的具体标准，例如：  
   - 单果腐烂面积超过【XX%】视为坏果；  
   - 霉变、流汁、虫蛀、严重碰压伤等外观缺陷；  
   - 可双方共同委托第三方检测机构认定。

3. **责任承担与赔偿比例**  
   - 若坏果率在【X%以内】，由买（批发）方承担；  
   - 若坏果率超出【X%】，超出部分由卖（发货）方按【X倍/原价】赔偿；  
   - 可约定最高赔付比例（如不超过总货款的【X%】）。

4. **举证与处理流程**  
   约定：批发商发现坏果后，应在【XX小时内】提供带时间戳的影像证据（如照片、视频），并通知发货方；发货方有权在【XX小时内】确认或要求复核。逾期未回应视为认可。

5. **物流责任划分**  
   若运输方为第三方，需明确：  
   - 坏果因包装不当或发货时存在问题，由发货方负责；  
   - 坏果因运输途中挤压、超温等，由物流方负责（批发商应保留物流单据并索赔）。

6. **违约金与争议解决**  
   明确发货方未按约定赔偿时，需额外支付【XX%】的违约金；争议可协商解决或提交【XX法院/仲裁机构】。

**示例条款样本**（直接可用）：  
> “在货到后24小时内，买方对货物进行验收。坏果指腐烂面积超过10%或出现霉变、流汁的果实。坏果率在5%以内由买方承担，超过部分由卖方按坏果对应货款总额的1.2倍赔偿。买方需在验收后2小时内提供清晰影像证据，卖方应在12小时内确认或提出异议，逾期视为同意。运输途中造成的损坏由物流方负责，买方应配合向物流方索赔。若卖方未在3日内支付赔偿，每日加收应赔款0.5%的违约金。”

建议在签订前，双方协商确定具体的百分比和时限，并保留影像、运输单据作为证据。
