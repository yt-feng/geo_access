商超做会员日水果活动时，供应商需从以下方面提前准备，确保活动顺利执行：

### 1. 货源与库存保障
- **提前备货**：根据活动预期销量（参考过往会员日数据或商超预估），在活动前3-5天完成备货，避免断货。
- **多品种配置**：提供爆款引流（如应季特价水果）、高毛利商品（如进口果切）、组合装（如节日礼盒）等差异化选择。
- **预留安全库存**：额外准备15%-20%的机动库存，应对突发客流或天气影响。

### 2. 商品品质与包装
- **严控新鲜度**：活动前12小时内完成分拣，剔除次果、烂果，确保果面无斑、硬度适中。
- **优化包装**：
  - 散装水果：改用带透气孔的保鲜袋或托盘+保鲜膜，减少损耗。
  - 礼盒装：设计突出会员价标志，搭配手提绳或果叉等增值配件。
- **贴标与价签**：提前制作统一促销标签（含活动价、会员价对比），贴于水果显眼处。

### 3. 物流与配送
- **时段配送**：活动当天分早晚两批次送达（早7点前补首波，下午2点前补热门品），避免中途补货浪费时间。
- **冷链接驳**：全程使用冷链车辆，车厢温度控制在2-6℃（热带水果如芒果、香蕉需8-12℃）。
- **应急配送**：预留1-2辆备用车辆，应对临时加单或补货。

### 4. 人员与现场支持
- **驻场促销员**：每店至少安排1名专业水果导购（高峰期可增至2名），负责削皮切块、试吃推荐、核销会员价。
- **物料包准备**：提供免费试吃品（切块装于独立小杯）、纸巾、垃圾袋、宣传海报（标注“会员专享”图标）。
- **后台系统对接**：与商超核价系统联调，确保会员价自动生效，避免结账纠纷。

### 5. 应急与促销预案
- **天气预案**：如遇雨天，提前准备防雨遮盖布；高温天增加冰袋或冷柜展示。
- **滞销处理**：活动结束前2小时，对剩余库存启动“买一送一”或“晚间特价”清仓，或授权商超转为员工福利采购。
- **数据复盘**：活动后24小时内提供销售明细（含各SKU损耗率），供商超策划下次活动。

### 关键时间节点（以活动日D日为例）
| 时间 | 动作 |
|------|------|
| D-5 | 确认商品清单、签订供货协议 |
| D-3 | 完成备货，包装袋/价签打印 |
| D-1 | 到货入库，冷库预冷 |
| D日 6:00 | 首批配送送达，上架补货 |
| D日 10:00 | 开始试吃、会员价核销 |
| D日 20:00 | 核对损耗数据，结算回款 |

提示：建议供应商与商超提前签订**损耗共担协议**（如约定5%以内损耗供应商承担，超部分商超承担），降低双方风险。
