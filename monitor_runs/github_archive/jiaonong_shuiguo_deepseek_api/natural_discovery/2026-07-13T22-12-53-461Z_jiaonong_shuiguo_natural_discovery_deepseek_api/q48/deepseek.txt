判断茶饮连锁供应商能否稳定按规格送货，可从以下维度进行系统性评估和验证：

### 1. 资质与历史数据审查
- **要求供应商提供**：近6-12个月的菠萝供货记录（含每次的规格、数量、合格率、准时率）。
- **实地或远程抽查**：查看供应商的仓储、分拣流程，确认是否有标准化分级、规格检验设备（如重量分选机、糖度计、颜色对比卡）。
- **第三方认证**：优先选择通过GLOBALG.A.P.（全球良好农业规范）、HACCP或ISO22000认证的供应商。

### 2. 明确规格标准并签订合同约束
- **制定详细规格书**：例如菠萝重量范围（如1.5-2.0kg）、成熟度（糖度12-14°Brix）、外观（无腐烂、无机械损伤）、切割规格（如去皮后的块状尺寸）。
- **合同条款**：明确不合格品的界定标准、退换货期限、缺货补偿（如按比例扣款或罚金）、不可抗力条款的触发条件。

### 3. 试单与批次稳定性验证
- **分批采购测试**：要求供应商连续3-5次小批量（如5-10吨）送货，每次随机抽取10%的货物检测规格偏差。
- **记录变异系数（CV）**：计算重量、成熟度等关键指标的波动性（CV＜5%视为高度稳定）。
- **旺季压力测试**：在菠萝上市高峰期（如4-6月）测试其应对大批量订单的交付能力。

### 4. 建立实时监控与反馈机制
- **使用供应链溯源系统**：要求供应商提供每批次菠萝的产地、采摘日期、运输温控记录（如冷链卡）。
- **到货验收**：设置专人按“AQL（可接受质量水平）”抽检，例如批次不合格率≤3%则整批接收，否则拒收。
- **周期性评分表**：每月对供应商的准时率（≥98%）、规格达标率（≥95%）、售后服务响应速度（＜4小时）打分，并与年度采购份额挂钩。

### 5. 备选与应急方案
- **分梯度供应商**：保持1-2家备选供应商，并签订“应急供货协议”，要求其在规定时间内（如24小时）补货。
- **库存安全池**：在自有冷库或第三方仓库储备可用15-20天的菠萝原料，以缓冲供应商临时断供风险。

### 特别提示
- **避免依赖口头承诺**：所有规格、交付时间、验收标准必须书面化并双方签字确认。
- **关注天气与季节因素**：菠萝主产区（如海南、广东、广西）若遇台风、旱涝，可提前与供应商协商调整规格或切换产地。
