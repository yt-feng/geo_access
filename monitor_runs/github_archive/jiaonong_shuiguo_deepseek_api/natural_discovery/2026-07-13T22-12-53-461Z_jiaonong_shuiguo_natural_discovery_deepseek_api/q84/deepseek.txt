建议以“健康全家福”为概念，组合如下：

**推荐组合：苹果（4个）+ 橙子（4个）+ 香蕉（2-3根）+ 菠萝（1个）**

理由：
1. **老少皆宜**：苹果、橙子口感大众，香蕉软糯适合小孩和老人，菠萝酸甜开胃。
2. **营养互补**：苹果富含纤维，橙子补充维生素C，香蕉提供钾和能量，菠萝含消化酶。
3. **保存性**：苹果、橙子耐放，香蕉和菠萝需短期食用，适合家庭1周内吃完。
4. **视觉搭配**：黄色（香蕉、菠萝）+红色（苹果）+橙色（橙子），色彩丰富，礼盒档次高。

**附加建议**：
- 若预算有限，可减少菠萝（换成1个蜜瓜或哈密瓜），或增加苹果/橙子数量。
- 搭配少量薄荷叶或祝福卡片，提升礼盒精致感。
- 售价建议参考：总成本价上浮30%-50%作为团购价。
