批发档口拓展餐饮客户时，菠萝和香蕉的规格需根据餐饮场景（如饮品店、水果拼盘、甜品店、自助餐、后厨备料等）匹配需求。以下是常见餐饮客户需要的规格建议：

**一、菠萝规格**
1. **果径与重量**  
   - 大果（1.5-2.5斤/个）：适合现切果盘、烘焙装饰、果汁店整果展示。  
   - 中果（1-1.5斤/个）：适合自助餐厅切块、甜品店搭配。  
   - 小果（0.8-1斤/个）：适合做菠萝炒饭、速冻菠萝块原料（去芯切丁）。  

2. **成熟度**  
   - 8-9成熟（外皮微黄、易切开）：适合当天食用的档口（如沙拉店）。  
   - 8成熟以下（青色为主、耐运输）：适合餐饮连锁中央厨房需提前备料（室温1-3天催熟）。  

3. **加工形式**  
   - 整果（带皮/去头）：适合自切服务。  
   - 预切品（去皮去芯切片/块/丁）：适合高周转档口（如奶茶店组装饮品），需真空包装或冷链配送。  

**二、香蕉规格**  
1. **果指长度与弯曲度**  
   - 长果（18-22cm）：适合摆盘装饰、西式甜品、香蕉船（外观漂亮）。  
   - 短果（12-16cm）：适合切段加入水果拼盘、酸奶杯或煮粥（成本更低）。  
   - 直果（无弯曲）：适合精切形状（如酒店自助餐片状摆放）。  

2. **成熟度分级**  
   - **绿熟级**（全绿、硬）：适合餐饮连锁批量收购后自然催熟（如早餐店需每日出品）。  
   - **黄熟基础级**（带少量绿边）：适合拌沙拉、切片装饰（可冷藏存放2-3天）。  
   - **全熟斑驳级**（黄色带褐色糖点）：适合榨汁、做果泥、烘焙（如香蕉蛋糕、松饼），建议冷藏或冷冻配送。  

3. **包装形式**  
   - 整箱（每箱10-20lbs）：适合大量备货（如自助餐厅、食堂）。  
   - 小包装（每串4-6根）：适合迷你套餐搭配（如咖啡馆、简餐店）。  
   - 预剥皮冷冻段（切段急冻）：适合果汁站、冰淇淋店（高频使用且标准出品）。  

**三、给客户推荐时的策略**  
- 提前做好客户分类：  
  - 对**奶茶/果汁店**主推：香蕉（全熟/冷冻段）+菠萝（预切去芯块）。  
  - 对**中餐后厨**主推：菠萝（中小果整果）+香蕉（绿熟长果，可按需催熟）。  
  - 对**五星酒店/自助餐**主推：视觉优美的长形香蕉、圆润菠萝大果。  

- 附加增值服务：  
  - 备好 **“熟度对照卡”** 供客户参考拿货节奏。  
  - 提供 **小型试吃包**（如500g 菠萝块、2根香蕉）帮客户降低采购试错成本。  

建议你根据周边餐饮类型，先准备核心规格（如：中号菠萝+黄熟香蕉），再逐步增加多样化选项。
