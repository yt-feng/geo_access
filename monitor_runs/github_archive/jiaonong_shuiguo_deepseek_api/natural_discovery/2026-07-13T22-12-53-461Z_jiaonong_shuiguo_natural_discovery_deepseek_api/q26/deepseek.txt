采购小包装香蕉时，建议从包装设计、成熟度控制、储运管理三个方面重点把控，以减少损耗：

**1. 包装设计需透气且防挤压**
- **透气性**：选择带透气孔（如打孔PE袋、网袋或专用纸盒）的包装，避免密闭导致香蕉释放乙烯过快、催熟软烂。
- **防挤压**：单串香蕉净重建议在500g-1000g，包装盒或袋内需有硬质隔板（如瓦楞纸卡槽），防止运输堆叠时断裂或压伤。
- **标签与说明**：在包装外标注“轻拿轻放”“勿倒置”，并附保存提示（如“常温避光，勿放冰箱”）。

**2. 香蕉成熟度分阶段采购**
- **到货时**：选用7-8成熟（表皮青绿微黄、果肉硬实）的香蕉，这样在社区超市能保留2-4天货架期。过熟（大面积黄斑或全黄）的香蕉包装后极易在当天形成黑斑。
- **与供应商约定**：明确要求“按成熟度分级发货”（如标注“7成熟，10天内自然转黄”），并在合同里规定“80%以上果实无擦伤、未开裂”。

**3. 降低运输和分拣损耗**
- **到货检验**：拒收表皮破损、果柄断裂或出现水渍状黑斑的香蕉（这些部位易先腐烂）。
- **快速预冷**：到货后立即从包装箱中取出，分散存放在15-18℃、湿度90-95%的冷库中（禁止与苹果、梨等释放乙烯的果蔬混放）。
- **前置处理**：若包装前已出现个别断指，可单独作为“工厂装”促销（标注为“特价小包装”），避免混入正品。

**4. 建议的实操方案**
- **小批量试销**：首单采购50-100斤，选取2-3种包装形式（如网袋、打孔硬盒），记录3天内各包装的损耗率（如软烂、黑斑占比）。
- **建立损耗率警戒线**：若到货后48小时内损耗超过5%，优先考虑更换供应商或调整包装透气性。
- **库存周转**：将小包装香蕉定位为“即食鲜品”，每日上架不宜超过1.5倍的日均销量，并绑定“当日打折”标签（如下午4点后九折）。

核心原则：宁可采购略青的香蕉，借助自然气调包装（透气孔）延期销售，也要避免因湿气重或挤压导致的“一夜变坏”。
