通常供应商可以提供包装与规格支持，但具体支持程度取决于供应商类型和合作方式。建议按以下步骤确认：

1. **明确需求**：先确定自有品牌的水果种类、目标客群（如家庭装、精品装）、规格（如单果重量、包装箱式/袋式/礼盒）及包装设计（定制logo、图案等）。

2. **筛选供应商**：选择有包装定制能力的供应商，如大型果园、专业水果批发商或品牌代加工厂。优先询问他们是否配备包装车间或合作印刷厂。

3. **沟通支持范围**：
   - 包装材质（纸箱、泡沫箱、塑料盒、环保袋等）及起订量（通常500-1000套起订）。
   - 规格定制（如每箱净重5kg/10kg，或按果径分级）。
   - 贴标服务（可代贴你品牌标签）。
   - 是否提供成本价包装，或需另外付费。

4. **索取样品**：要求供应商提供样品（含包装实样），测试果品质量、包装牢固度及视觉效果。

5. **签订协议**：明确包装规格、验收标准、破损率及退货条款，避免货不对版。

若供应商本身不做包装，可协调他们与包装厂绑定采购，或由你方（商超）统一采购后发给供应商包装。建议多对比2-3家，了解行业常规的包装起订量及定制费用分摊方式。
