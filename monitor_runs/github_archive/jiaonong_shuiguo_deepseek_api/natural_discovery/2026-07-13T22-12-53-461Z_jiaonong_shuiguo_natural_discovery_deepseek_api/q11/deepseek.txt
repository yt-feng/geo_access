大小不一致可能产生以下影响，需根据销售策略具体判断：

1. **影响定价与利润**  
   - 若按统一售价，小香蕉会拉低整体性价比，大香蕉显得价高，可能导致顾客倾向挑选大果，小果滞销，增加损耗。
   - 若按大小分级定价，操作成本增加。

2. **影响顾客体验**  
   - 顾客期望一致性（如家庭采购、拼盘制作），大小混杂可能引起不满，认为质量不齐。
   - 部分顾客喜欢大果，部分偏好小果，混装难以满足精准需求。

3. **损耗与库存管理**  
   - 大小不一的香蕉成熟进度可能不同，小香蕉易熟透，大香蕉可能后熟，导致批次内部分过早腐烂。

**建议调整方式**：  
- **分级销售**：将大小果分开，分别定不同价格（如大果精品价、小果促销价）。  
- **组合打包**：按重量或数量混装，注明“混合规格”，低价吸引对价格敏感的顾客。  
- **定制品类**：小果可用于烘焙、果汁或特价区，大果陈列为主打商品。
