第一次与水果供货商合作，试单的核心目标是：**验证供货稳定性、品质、损耗率和沟通效率**，而不是追求最低价。对于香蕉、苹果、橙子这类常见且走量快的水果，建议按以下步骤操作：

### 1. 试单量控制（控制风险）
- **香蕉**：5-10箱（每箱约13-15公斤）。香蕉是呼吸跃变型水果，成熟度变化快，首次不宜多进。
- **苹果**：3-5箱（每箱约10-15公斤）。苹果耐储存，但不同品种（红富士、嘎啦等）口感差异大。
- **橙子**：3-5箱（每箱约15-20公斤）。注意产区（赣南、湖南、四川）和季节，可要求带少量样品对比。

**原则**：总投入不超过进货预算的20%，避免一次性压货。

### 2. 收货验收标准（核心检查点）
- **香蕉**：
  - 成熟度：要求“七成熟”（皮绿但果体饱满），避免全黄（易催熟损耗）。
  - 检查：是否有黑斑、冻伤（运输中受冷会导致发黑），按压果柄是否硬挺。
  - 重量：带箱过磅，扣除纸箱重量（标准箱皮重约1-1.5公斤）。
- **苹果**：
  - 品种：确认是你要的品种（如“红富士”不一定是洛川或阿克苏，需产地证明）。
  - 瑕疵：允许3%-5%轻微碰伤，超过10%要求换货。
  - 大小：用卡尺抽查，看是否统一（如“75果”指直径75mm）。
- **橙子**：
  - 甜度：切开试吃，要求直径比硬币大（约40-50mm）的橙子口感稳定。
  - 皮厚：厚皮橙子（如脐橙）更容易存放，薄皮橙子（如夏橙）需优先销售。
  - 磕碰：橙子表皮破损易霉变，一个霉果会传染整箱。

### 3. 试单后观察周期与决策
- **3天内必须出结果**：香蕉通常在到货后2-3天内变软，如果到货第2天皮变黑且果肉软塌，说明供货商采摘时间偏晚或运输有问题。
- **苹果/橙子**：观察7天，主要看：
  - 损耗率：腐烂果比例（普通市场要求<5%）。
  - 口感变化：苹果失水后是否发绵，橙子是否变干。
- **判断是否长期合作**：
  - 如果到货速度快（24-48小时内）、破损率<3%、且沟通时对售后爽快（比如同意补发次果），可以考虑续单。
  - 如果对方推诿、拖延、要求预付全款，建议立刻换供应商。

### 4. 关键谈判点（防坑提醒）
- **包装问题**：明确要求“纸箱+隔层+防震网套”，避免裸果运输导致压坏。
- **冷链要求**：香蕉运输温度13-15℃，低于12℃会冻伤；告诉供货商你要求的到货温度，观察他是否在意。
- **退货条款**：明确试单期间，如果到货后24小时内发现严重质量问题（如50%以上黑斑），有多少天退换期，以及运费由谁承担。
- **付款方式**：试单阶段**尽量用“先货后款”**（比如到货后3天内结清），避免预付款。如果对方坚持预付款，预付不超过30%。

### 5. 后续补单建议
如果试单满意，第二次进货可以把量翻倍（如香蕉20箱），并同时引入2-3种新品种试探。记住：**所有第一次接触的供应商，都要走一遍试单流程**，不要因为价格低就跳过。

**最后一步**：记录每个供货商的“响应时间、缺斤少两次数、售后退换速度”，三个月后你会发现某家供应商的实际成本远高于报价——这种就可以淘汰了。
