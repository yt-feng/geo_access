可以谈，但需要结合具体情况和谈判策略。以下是具体建议：

1. **适合谈锁价的条件**：
   - 常见、需求量大的水果（如苹果、香蕉、橙子等），且供应商供货稳定。
   - 季节性波动不极端的水果，或你有能力在波动期调整菜单（如用冷冻、罐装代替）。
   - 能承诺一定采购量（如月度、季度固定金额或件数），以换取供应商的“量价联动”协议。

2. **谈判要点**：
   - **明确锁价范围**：不要锁全部品种，可对核心水果锁定6-12个月价格，但允许供应商在成本暴涨时（如减产30%以上）重新协商。
   - **加“价格浮动条款”**：例如约定价格参考某批发市场均价，上下浮动5%内自己承担，超出部分双方按比例承担。
   - **附加服务置换**：如提前预付部分货款、帮供应商减少配送批次，来换取更稳定的锁价。

3. **替代方案**（如果供应商不愿锁价）：
   - **签订“优先供应协议”**：通过承诺长期合作，获得该供应商的稳定优惠价（可能低于市场价10%左右），但不完全锁死。
   - **建立多供应商备选**：与2-3家供应商分别谈“市场价格基础上打固定折扣”（如批发价的9折），降低被单方面涨价的风险。
   - **自控源头**：如果用量极大，可考虑联合其他餐饮店直接对接产地合作社，锁定订单价。

4. **实际案例参考**：
   - 连锁茶饮店常对核心水果（如柠檬、芒果）与供应商签年度锁价合同，但会加入“用冷冻水果替代鲜果”时的价格调整条款。
   - 小型餐饮店可通过“微信采购群”获取临时报价，与2家供应商同时合作，利用比价倒逼价格稳定。

**最终建议**：先以“保证采购量+减少退换货率”为筹码，尝试锁定3-6个月的部分品种价格。如果对方拒绝，转而要求“月度报价不高于当月批发均价90%”的浮动优惠，成功率更高。
