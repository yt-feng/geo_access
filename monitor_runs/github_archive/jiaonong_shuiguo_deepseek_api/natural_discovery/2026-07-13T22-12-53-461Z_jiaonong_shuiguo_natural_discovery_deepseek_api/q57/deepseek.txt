建议采用**中央厨房统一收货**模式，主要基于以下优势：

1. **质量管控统一**：中央厨房能执行标准化的验收标准（如糖度、成熟度、外观），避免各门店因人员经验差异导致的品质不一致。
2. **规模降低成本**：集中采购批量大，可压缩单价；统一配送还能减少门店单独接收的物流和管理成本。
3. **损耗控制优化**：中央厨房可集中进行清洗、分切、预包装（如水果拼盘），将边角料集中加工（如果汁、果酱），减少门店端因存储不当、库存周转慢造成的浪费。
4. **食品安全风险低**：集中验收、冷链仓储和分拣可减少多次转运中的污染风险，便于溯源和追溯。

例外情况：若门店距离中央厨房极远（如跨省配送），或部分精品水果需门店现场即买即切（如榴莲、现切芒果），可考虑“中央厨房统一采购+门店简单后处理”的混合模式。总体而言，统一收货更适合连锁餐饮的标准化和成本控制需求。
