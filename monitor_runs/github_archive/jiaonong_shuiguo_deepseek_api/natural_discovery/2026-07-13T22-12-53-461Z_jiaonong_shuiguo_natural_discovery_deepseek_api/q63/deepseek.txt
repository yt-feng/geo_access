判断菠萝哪个规格更容易出货，建议从以下几个核心维度进行分析，结合市场调研和数据验证：

1. **目标客户群体定位**
   - 如果是面向**餐饮/果汁店**：通常需要成熟度高、个头偏大（1.2-1.5公斤/个）的菠萝，因为出肉率高、成本更划算。
   - 如果是面向**家庭/散客**：中等个头（0.8-1公斤/个）更受欢迎，易于一次消费完；小菠萝适合单人食用或零食。
   - 如果是**高端市场/礼品**：要求外观匀称、无瑕疵的特定品种（如凤梨），规格需统一。

2. **地区消费习惯**
   - 北方市场通常偏好较大个头（1公斤以上），认为“实惠”。
   - 南方和沿海地区可能更接受小规格或即食现切菠萝。
   - 调研本地超市、农贸市场、地摊的畅销规格（可通过实地观察或批发市场数据获取）。

3. **价格与利润空间**
   - 小规格菠萝单价低，易走量，但单件利润薄。
   - 大规格菠萝单品利润高，但若市场需求小易滞销。
   - **最佳策略**：同时备货1-2种主流规格，用数据测试1-2周，看哪种周转率更高（进货后最快卖完的规格）。

4. **季节与品种影响**
   - **早期菠萝**（如3月）个头偏小但甜度高，小规格容易清货。
   - **中后期菠萝**（如5-6月）个头普遍变大，大规格价格可能更低，反而好走。
   - 不同产区（如海南、广东、云南）品种天然大小不同，需按批次品控。

5. **实操建议**
   - **小批量试卖**：各进几箱不同规格（比如：0.7-0.9kg、1.0-1.2kg、1.3kg以上），标记进货价和卖出价，记录每天卖出数量、滞销数量。
   - **关注复购率**：老客户反馈的规格需求（如“上次的小菠萝好卖，这次多进点”）。
   - **利用电商/社区团购**：如果线上销售，通常小规格、去皮切好包装的更易出货（适合即时消费）。

**最终判断**：在没有任何数据前，优先选择**中上规格**（约0.9-1.1公斤）作为主力型号，同时搭配30%的小规格和20%的大规格试探市场。如果批发市场小贩采购为主，则主推1-1.2公斤；如果周边有大量加工厂或果汁店，则主推1.3公斤以上品种。
