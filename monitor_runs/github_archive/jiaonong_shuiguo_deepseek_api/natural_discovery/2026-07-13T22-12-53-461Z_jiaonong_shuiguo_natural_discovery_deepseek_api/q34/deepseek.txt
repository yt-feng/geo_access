在商超采购水果时，**损耗低通常比价格低对最终毛利的影响更大**，原因如下：

1. **损耗直接侵蚀毛利**：水果易腐，若损耗率过高（如运输、存储、售卖过程中腐烂或变质），即使采购价格再低，实际可售数量减少，单价成本上升，最终毛利可能为负。例如：进价10元/公斤，售价20元/公斤，若损耗30%，实际成本为14.3元/公斤，毛利仅5.7元，远低于预期。

2. **价格低有局限性**：低价采购可能意味着品质差（如不新鲜、易损坏），反而加剧损耗；或促销门槛降低，影响整体客单价和品牌形象。若市场价波动，低价优势可能被损耗抵消。

3. **综合计算更关键**：比较两种策略带来的净利润：
   - 价格低（假设进价降低10%，但损耗率维持20%）：实际毛利变化有限。
   - 损耗低（通过优化供应链、保鲜技术降低损耗5%）：可售量增加，毛利提升更显著。

**建议**：优先控制损耗（如选择可靠供应商、缩短运输时间、合理库存管理），再追求价格优势。两者需平衡，但损耗管理是水果品类盈利的核心。
