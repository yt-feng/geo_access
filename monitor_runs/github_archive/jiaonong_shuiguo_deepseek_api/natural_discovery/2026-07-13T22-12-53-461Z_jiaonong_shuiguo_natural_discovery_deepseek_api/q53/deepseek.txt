针对餐饮店每天少量水果的需求，建议选择以下类型的供应商，以省心为主要目标：

1. **本地农贸市场的固定摊位**：优先选择农贸市场中稳定、口碑好的水果摊主。长期合作可以要求每日定时配送或送货上门，且可小批量采购。对方熟悉需求后，能帮您挑选、搭配当季水果，价格也更灵活。

2. **社区生鲜超市或连锁便利店**：这类供应商有稳定的日配系统，支持少量多次购买。可以联系店长协商固定时间取货或配送，部分超市甚至提供次日达服务，适合临时补货。

3. **线上生鲜平台的B端服务**：如美菜、美团优选（企业版）、每日优鲜等。APP下单便捷，可精确选择数量（比如半斤、一斤），平台会统一配送。部分平台有“企业专享”渠道，按周结算更省心。

4. **与周边酒店、烘焙店拼单**：主动联系附近用量较大的商户，如甜品店、咖啡厅，与他们合找一个供应商（如果品批发商或水果批发市场经销商）。拼单能降低起订量要求，同时享受批发价和定点配送。

5. **果园直供或合作社**：搜索本地果蔬合作社或小型农场，询问是否能提供“小批量、常客价”配送。这类供应商通常品质稳定、季节性适配，且愿意为固定客户提供灵活服务。

**省心操作建议**：
- 与供应商签订简单协议，明确日常需求种类、规格、最低配送量、付款周期（如周结或月结）。
- 要求对方每日提供简易报价单或清单，便于核对。
- 选择至少有3家备选供应商，避免因意外断货。初期可以先用线上平台测试品质效率，再转为长期合作。

这样做能最大程度减少采购时间、库存压力，同时保证水果新鲜度和价格合理。
