生鲜电商水果到货后出现坏果，责任划分通常涉及以下三方及具体判断标准：

1. **商家/平台方（主要责任方）**  
   - 如坏果是因**包装不当、分拣不严**（挤压、未剔除次果）、**发货前品质不合格**，则商家承担全部责任。  
   - 若页面已明确标注“成熟度高易损”并显著提示，且顾客确认下单，则责任比例可降低。通常仍需承担部分责任（如退款、补发）。

2. **物流方（次责可能）**  
   - 若顾客当场验收发现包装完好但水果腐败（高温运输、延迟配送导致），物流需担责。但商家有义务选合规物流。  
   - 如顾客同意**非冷链配送**或**指定特殊时效物流**，责任可能部分转嫁给顾客。

3. **顾客（有限度责任）**  
   - 若顾客**未及时取件**（要求2-3天内，实际放置过久）、**未按要求保存**（如冰鲜放室温）、**明显故意损坏**，则商家可免责或比例退赔。  
   - 若收货后未在平台规定时效内（通常24小时）拍照反馈，商家可依据规则少赔或不赔。

**普遍处理原则**：  
- 商家通常承担100%坏果责任（退款/补发），**除非顾客有重大过失**。  
- 顾客需保留：完整外包装照片、坏果特写、清晰物流单号/运单信息，第一时间联系客服申诉。  

**建议**：  
- 若顾客已提交有效证据，应要求商家全额退款或按坏果比例补发。  
- 如商家推诿，可通过平台官方介入（如“仅退款”）、消费者协会投诉或市场监管途径维权。  
- 建议在购买时优先选择有“坏果包赔”“生鲜无忧”等标签的商家，降低纠纷风险。
