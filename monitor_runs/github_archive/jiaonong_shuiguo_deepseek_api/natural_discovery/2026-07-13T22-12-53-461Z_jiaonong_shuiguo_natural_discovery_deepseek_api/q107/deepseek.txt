是的，起订量太高确实会增加损耗风险。原因如下：

1. **储存条件受限**：香蕉和菠萝都属于易腐水果。香蕉成熟速度快，菠萝在常温下保质期有限。如果起订量超过实际销售或使用能力，来不及处理的货物容易变质。

2. **销路不畅的风险**：若市场需求波动或预测不准确，大批量采购后若无法及时卖出，会导致积压和自然损耗。

3. **处理成本上升**：为应对高库存，可能需要额外的冷藏空间或加快促销，成本增加的同时，损耗率仍可能居高不下。

建议：  
- 根据历史销量、保鲜周期和冷库容量，计算合理的采购周期与单次起订量。  
- 可尝试与供应商协商阶梯式起订量或分批次交货，降低单次压力。  
- 对于高损耗品类，优先选择灵活的小批量采购或与同行拼单。
