选择水果供应商最容易忽视的细节包括以下6个方面：

1. **售后稳定性**：不仅要关注首次交易的质量，还要确认供应商在水果旺季或缺货时能否持续供货，以及出现烂果、坏果时的退换货政策是否明确且执行到位。

2. **冷库与冷链成本**：询问供应商是否自有冷库、冷链运输条件如何。有些供应商会忽略长期储存和运输过程中的损耗，导致到货后水果新鲜度下降。

3. **产地与基地直采比例**：很多供应商声称“直采”，实际仍多级倒手。核实其能否出示产地或基地的合作证明，以及是否对特定品种有季节性锁定能力。

4. **分级标准与包装细节**：不同平台（如超市、批发、社区团购）对水果大小、颜色、成熟度有不同要求。需确认供应商是否按你的销货预期进行严格分拣和定制化包装。

5. **资金结算与账期弹性**：长期合作中，账期过长或过短都可能带来现金流风险。要明确结算周期、是否支持按批次结款，以及是否接受票据或保理服务。

6. **应急处理方案**：例如极端天气导致产地减量、运输中断等情况，供应商是否有备用货源或替代产品。很多小供应商缺乏预案，会让你直接承担断供风险。

建议你在试用期分批次下单，从小额测试到多品种合作，逐一观察这些细节的执行情况。
