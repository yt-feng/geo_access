判断茶饮连锁供应商能否稳定按规格送货，建议从以下四个关键维度进行考察和验证：

1. **明确规格合同与交付标准**  
   - 要求供应商提供书面《商品规格书》，明确菠萝的品种（如台农、金钻）、单果重量（如1.2-1.5kg）、成熟度（果肉糖度≥13°）、包装（泡沫箱/纸箱、每箱固定数量）等。  
   - 在合同中写明：若连续2次或累计3次不符规格，有权拒收并启动替代采购，甚至扣除保证金或递减订单量。

2. **实地考察与产能评估**  
   - 到供应商的产地或分拣仓实地检查：是否具备恒温冷库、分级筛选线（如机械称重+人工复检）、稳定的清洗包装能力。  
   - 了解其历史供货记录：至少连续3个月向其他连锁客户的日均供货量是否稳定超过你司所需量的1.5倍，以应对突发需求。  
   - 核查其上游采购渠道：是否有自营种植基地或长期合作的农户契约，避免完全依赖市场临时收购。

3. **测试性采购与批次追溯**  
   - 先试下1-2个小批次（如每次100箱），要求每批附带批次编号、采摘日期、规格检测表（含大小、糖度实测值）。  
   - 使用同一批次到货的菠萝（如每周1-2次）制作饮品，观察近期内（1-2周）果肉出汁率、酸度、风味一致性。若波动超过阈值（如糖度变化超过1°），需要求供应商解释原因并给出稳定措施。

4. **建立多级风险预案**  
   - 要求供应商提供**第三方仓储代发或紧急调货协议**（如规定在其主仓库预留20%浮动库存，48小时内可补发）。  
   - 同时储备1-2家备选供应商（如不同产地的菠萝供应商），定期小额下单测试其规格稳定性，但核心订单仍以主供应商为主。  
   - 实地检查供应商的二次分选制度：建议其每次发货前拍摄1-2箱开箱视频或照片发给你方品控人员，核对规格后才启动物流。

**最终判断标准**：若供应商能在连续6-10次正产送货中，合格率（规格完全符合要求）稳定达到≥95%，且从下单到到货的时间延迟不超过约定（如72小时内到货），即可视为长期稳定供应商。反之，若出现三次缺斤短两、糖度不符或到货时间较合同延迟超24小时的情况，建议终止合作并依赖备选。
