控制社区超市水果新鲜度和库存的核心在于 **“以销定采、快进快出”**。可以参考以下实操策略：

### 1. 进货端：只有“预售”才能压库存
- **预售+隔日达**：微信群/小程序做“今晚8点前下单，明早到店自提”。只有确定卖得出，才进货。
- **缩放进货品类**：不要买全品类水果。只进“当地应季”+“2-3种高频需求”（如西瓜、苹果、香蕉），限量进货。
- **少量多频**：香蕉、草莓等易损耗品种，每次进货量控制在**当日预估销量的70-80%**。

### 2. 库存管理：最高效的方法是用“颜色分级”
- **红色区（当日必清）**：软烂、损伤、成熟度过高的。直接打折（如“今晚8点后8折”或“买1送1”）。
- **黄色区（1-2日内售出）**：表皮微瑕但内部完好。可以切成果盒（10元/盒），或搭配赠品。
- **绿色区（可存3-4日）**：未全熟、耐放的（苹果、橙子、柚子）。摆在最里面，避免顾客频繁触碰。

### 3. 陈列技巧：每天只放“半架货”
- **“少摆勤补”**：货架上只放**平时量的60%-70%**（看起来新鲜、满，但实际库存不多）。补货时把旧货朝外，新货朝里。
- **夏季/热带水果**：必须冷藏陈列，标签写“请拿取下方未切水果”（减少顾客反复翻捡导致机械损伤）。

### 4. 销售端：快消策略
- **“当日鲜”专区**：当天到货的、最漂亮的水果摆在门口，用明亮灯光配合“这批货仅今天到”的标签。
- **“熟果特价”**：对过度成熟（但未坏）的水果，直接做“今日特价”、“现做果切”或者“搭配酸奶（免费送1杯）”。
- **会员预购**：对老顾客，可以每周固定时间发“本周水果清单”，按需预交定金，到货后通知取货（锁定需求）。

### 5. 数据工具：用本子或手机记好“三个数”
- **进货量**：记得记下每天每种水果卖了几个（或几斤）。
- **损耗量**：每天关店前记下扔掉/打折处理的斤数。
- **预警线**：如果某种水果连续2天售出率低于50%，下周直接**暂停进货**。

### 6. 终极止损：跟周边小贩或饭店合作
- **协议价包销**：跟旁边小吃店、奶茶店、食堂谈，把当天卖不完的草莓、芒果按低价包给他们做原料。
- **免费试吃变赠品**：熟透的水果切小块，放在收银台旁边，买满10元送一份（变成营销工具）。

**一句话口诀**：**少进勤补预售挂前，熟货特价当日清掉，每次只摆半架货，压货别进下次不订。**
