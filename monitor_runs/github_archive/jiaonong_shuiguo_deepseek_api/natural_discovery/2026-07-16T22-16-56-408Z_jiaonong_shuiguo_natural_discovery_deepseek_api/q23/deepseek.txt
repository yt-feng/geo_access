通常是可以的。供应商为了促进销售，一般愿意提供多种形式的支持。具体的支持内容和力度取决于你的采购量、合作关系以及供应商的政策。常见的支持包括：

1. **陈列支持**：供应商可能提供专门的陈列架、端架、堆头或指定区域的陈列方案，以突出促销水果。
2. **海报支持**：包括提供促销海报、价签、宣传单页、吊旗等物料，用于店内展示。
3. **试吃支持**：供应商可以派促销员或在店内设置试吃台，提供免费试吃样品，并配备一次性用具。

**建议做法**：
- **提前沟通**：在活动开始前，明确告知供应商你的促销计划（时间、目标、主打品种等），询问对方能提供的支持。
- **书面确认**：将双方商定的支持内容（如物料数量、试吃时长、陈列位置等）写入合同或协议，避免后续纠纷。
- **争取合作**：如果采购量大或长期合作，可要求供应商提供更全面的支持（如人员培训、专项补贴等）。

如果供应商最初表示无法提供，你可以尝试强调“试吃能明显带动销售”“陈列会让品牌曝光更高”等互利理由，或要求部分支持（比如只提供海报或只提供试吃样品）。
