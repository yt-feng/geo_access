选择水果供应商时，长期稳定的合作需要关注以下容易忽视的细节：

1. **产地源头与季节性波动**：确认供应商是否有稳定的产地直采渠道，而非仅依赖批发市场。关注其是否对不同季节的水果有对应的供应方案，避免旺季断货或淡季品质下降。

2. **分拣与包装标准**：不同水果（如草莓、芒果、葡萄）对成熟度、包装缓冲、冷链要求差异大。询问供应商是否按下游渠道（商超、餐饮、社区团购）定制分拣标准，而非统一混装。

3. **检测与溯源文件**：常规农药残留检测报告很基础，但需注意是否覆盖重金属、保鲜剂（如柑橘的抑霉唑）等隐性指标。要求提供每批次的可追溯信息（产地、采摘时间、运输记录）。

4. **验收流程与损耗分担**：很多合同只写“按合同验收”，但具体验收标准（如外观瑕疵比例、糖度值）需量化。明确到货后几小时内可提出异议，以及超5%损耗的赔偿方式（退款、补货或抵扣下次货款）。

5. **价格调节机制**：水果价格波动大，优质供应商会提供“锁价+浮动”模式（如提前预付锁定部分平价，其余随行就市但有上限）。避免对方单方面涨价或断供。

6. **应急与补货能力**：测试供应商在极端天气（如台风导致产区减产）或物流受阻时的应对方案，包括备选果园、替代品种及补货响应时间（如4小时内是否能协调发货）。

7. **长期合作意愿**：考察供应商是否愿意共享库存数据、联合做库存优化，或允许小额试单（如首次合作只下一半订单）。警惕只做“一锤子买卖”的供应商。

**实操建议**：
- 首次合作前，要求供应商寄送样品（非仅看图片），并自己模拟长途运输（存放24-48小时）检查损耗率。
- 签订合同时，明确“无条件退换货权利”（如甜度不达标时），并保留10%-20%尾款作为质量保证金。
- 至少储备2-3家不同区域的供应商，避免单一依赖。
