建议您按以下步骤与供应商沟通临时补货，并约定关键条款，确保供应稳定、价格合理：

1. **立即联系供应商**：先通过电话或即时通讯工具紧急沟通，说明旺季菠萝用量激增的情况，确认供应商是否有现货或能否紧急调配。

2. **明确需求量与时间**：提供您需要的具体数量（如“未来一周需增加200公斤”）、补货时间节点（如“三天内到货”），以及预估的持续周期（如“大概持续一个月”）。

3. **商讨价格与结算**：临时补货通常需承担额外成本（如运输、仓储）。您可以：
   - 询问是否按原价执行，或要求不超过原价10%的上浮。
   - 明确是现结、预付定金（如30%）还是月结。

4. **约定品控标准**：要求菠萝符合一贯的品质（如成熟度、无腐烂、规格），可附上样品照片或标准描述，并约定“到货验收，不合格包退换”。

5. **签书面确认**：即使紧急，最好通过微信、短信或简短邮件确认以下内容：
   - 补货数量、单价、总金额
   - 交货日期与地点
   - 验收期限（如“到货后24小时内反馈问题”）
   - 违约责任（如“延迟到货每日扣1%货款”）

6. **预留备用方案**：若供应商无法完全满足，可协商部分补货，同时询问其是否推荐其他上游渠道，或能否接受少量替代品种（如品质相近的凤梨）。

**参考话术**：“王老板，现在旺季菠萝爆单了，我需要紧急加订200公斤，能不能三天内送到？价格能不能保持之前的6元/斤？如果加运费，最多加多少？我先发个微信确认单给你，你回复‘同意’就行。”

**提醒**：及时记录所有沟通截图和确认信息，避免后续纠纷。
