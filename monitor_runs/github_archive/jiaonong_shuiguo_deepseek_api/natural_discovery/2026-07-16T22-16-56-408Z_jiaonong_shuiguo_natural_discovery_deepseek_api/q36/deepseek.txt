针对商超采购菠萝的销售策略，建议根据门店类型、客流特征和损耗控制综合考虑，以下为具体分析：

1. **整果销售更适合**：
   - 生鲜超市或社区店：顾客希望自选并感知水果新鲜度，整果可直观展示品相，适合家庭批量购买。
   - 低客流/冷门时段门店：整果保质期更长（常温可存放2-3天，冷藏5-7天），减少加工造成的损耗和人工成本。
   - 需注意：提供免费削皮服务可提升整果销售吸引力。

2. **切好销售更适合**：
   - 高客流商圈/写字楼周边门店：白领、学生等群体追求便捷，切好包装的菠萝（如盒装、切片）可直接食用，客单价更高。
   - 高峰时段：预切盒装可缩短顾客排队时间，并降低现场削皮的卫生风险。
   - 注意：切好的菠萝需当天完成，剩余果肉可用于制作菠萝汁、冰品或果干以降低损耗。

3. **组合策略建议**：
   - 同步陈列整果与切盒：整果堆头旁设切盒展区，用爆品价引导尝鲜（例如整果9.9元/个，切盒15元/2盒）。
   - 动态调整比例：周末或节假日增加切盒供应（可占总量的40%），工作日以整果为主（80%）。

**风险提示**：
- 切果务必标注加工时间，冷藏保存不超过24小时，制作果切时使用洁净刀具和手套，防止交叉污染。
- 若人力不足，优先保证整果销售；可通过扫码活动推广“现切服务”限时满减，反向测试客群偏好。
