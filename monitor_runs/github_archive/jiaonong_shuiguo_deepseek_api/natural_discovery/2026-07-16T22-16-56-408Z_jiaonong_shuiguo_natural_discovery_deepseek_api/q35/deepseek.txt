针对连锁超市降低门店坏果率，合同中可重点约定以下赔付条款：

1. **明确坏果标准**：定义“坏果”为腐烂、霉变、明显机械伤或不可售果品（如超过5%表皮破损、硬度异常等），避免争议。

2. **约定到货验收与统计**：  
   - 现场验收：设置到货后2小时内开箱抽检，如坏果率超5%，超市有权拒收整批或要求按比例折价。  
   - 库存入仓后72小时内，发现坏果的，超市可拍照留存并通知供应商，按实际损耗量扣款。

3. **分级赔付比例**：  
   - 月累计坏果率≤3%，不赔付。  
   - 3%＜坏果率≤8%，供应商按该部分销售额的120%赔付（如货值1万元，坏果5%，赔付600元）。  
   - 坏果率＞8%，超市有权退回整批货物，且供应商需支付货值50%的违约金，并承担超市因此产生的分拣、处理费用（按人工成本计算）。

4. **连续违规惩罚**：同一供应商连续3个月坏果率＞5%，超市可单方解除合同，并要求供应商支付解除前三个月总销售额10%的违约金。

5. **举证与流程**：超市需保留坏果照片、入库记录和报损单，供应商可申请24小时内派员核验，逾期视为认可。

建议同时约定协商机制，如每月对账后再结算赔付款。以上条款需与具体商品品类（如叶菜、水果）的行业损耗标准适配。
