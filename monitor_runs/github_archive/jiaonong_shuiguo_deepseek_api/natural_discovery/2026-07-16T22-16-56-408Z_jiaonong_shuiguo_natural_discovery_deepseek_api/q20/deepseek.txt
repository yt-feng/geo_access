选择每日配送的水果供应商时，建议重点考察以下条件：

1. **配送能力与稳定性**：确保供应商能每天准时送达，尤其在节假日或天气恶劣时是否保障供应。可要求查看其物流团队或配送记录。

2. **水果品质与新鲜度**：要求供应商提供货源产地、采摘时间、冷库条件等信息。可要求试送几批，快速检查是否有烂果、虫害或过度成熟。

3. **品类与季节性供应**：确认供应商能否提供社区常需的品类（如苹果、香蕉、柑橘等），并能根据季节调整品种（如夏季的西瓜、冬季的草莓）。

4. **价格与账期**：对比市场均价，注意报价是否包含包装、运输、税点等隐藏成本。小超市可争取7-15天账期或阶梯定价（如量大有折扣）。

5. **售后服务**：明确退换货条款，例如到货几小时内可申请退换因配送导致的碰伤或变质水果。建议保留小样或拍照留证。

6. **卫生与资质**：供应商应有食品经营许可证、工人健康证，以及符合食品安全标准的仓储和运输记录（如冷链车温度记录单）。

7. **口碑与合作案例**：优先选择已为周边超市、餐馆等长期供货的供应商，可向其现有客户打听实际配送效率、态度等。

**建议操作步骤**：  
- 筛选3-5家供应商，要求提供报价单、货品图和近期合作案例。  
- 安排试配送2-3天，重点检查到货时间、损耗率和补货响应速度。  
- 首次合同先签1-3个月，设置试用期，后续根据表现调整合作。

如果供应商无法提供明确的产地或质检信息（如非近期采摘、无冷链记录），建议直接排除，避免后续投诉影响社区口碑。
