选水果供应商时，**最优先问的应该是「售后」**，其次是「配送」，最后是「价格」。原因如下：

1. **售后（最核心）**：水果易损耗、变质，团长最怕用户收到烂果、坏果。必须问清：坏果/烂果包赔比例（例如24小时内按比例退赔）、退款流程是否顺畅、是否支持无理由退货。售后没保障，口碑会迅速崩塌。
2. **配送（次重要）**：水果对时效要求高。需要确认：是否按约定时间到货（避开用户午休/深夜）、能否小批量多次配送（减少损耗）、配送破损由谁承担（漏发、压坏算谁的）。
3. **价格（最后考虑）**：比价当然重要，但如果售后和配送不靠谱，再便宜也等于白扔钱——用户坏果投诉、延迟配送导致退货，最终成本反而更高。

**快速判断技巧**：先亮出你的团购量（例如“我每周至少500斤”），然后直接问售后标准。如果供应商支支吾吾或标准模糊，直接跳过；如果响应快、承诺清晰（比如“坏果包赔+48小时赔付”），再问配送细节和价格梯次。
