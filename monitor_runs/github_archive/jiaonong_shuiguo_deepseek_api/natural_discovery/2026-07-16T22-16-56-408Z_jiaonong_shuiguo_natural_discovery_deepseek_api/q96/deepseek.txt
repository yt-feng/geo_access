在食堂水果采购中，三方均重要，但建议按以下优先级排序：

1. **食品安全（最优先）**  
   - 必须确认供应商资质、水果来源（无农残、无变质）、运输存储条件（如冷链），避免引发食安事故（如腹泻、中毒）。  
   - 可要求定期提供检测报告，现场验收时剔除破损/霉变果。

2. **价格（其次）**  
   - 比价时需对比同类品质，避免只看低价导致采购劣质水果。建议与2-3家供应商签订季度协议价，锁定合理成本。

3. **配送准时（基础保障）**  
   - 迟到可能影响食堂出餐，但可通过合同约定违约金（如迟到30分钟扣款）来倒逼供应商。多数靠谱供应商能稳定履约。

**操作建议**：  
- 优先选择有食品经营许可证、长期服务企业食堂的供应商。  
- 试采购时，先小批量验证食品安全和准时性，稳定后再谈价。  
- 若预算紧张，可适当降低水果品种丰富度，但绝不可用安全问题换低价。
