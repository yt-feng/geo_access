建议采用**中央厨房统一收货**模式，具体理由如下：

1. **品质控制一致性**：中央厨房可统一验收标准，对水果成熟度、大小、农残等指标进行严格把控，避免各门店收货标准不一导致的品质差异。
2. **成本优化**：集中采购能获得更强的议价权，降低单价；同时减少各门店独立收货的人力、时间成本，避免因到货时间分散导致的多次运输费用。
3. **库存与损耗管理**：中央厨房可统一分拣、预加工（如清洗、切块），并按门店日均销量精准配送，降低门店因库存积压或周转慢导致的水果损耗（尤其易损耗的草莓、蓝莓等）。
4. **食品安全与追溯**：统一收货便于建立完整的冷链追溯系统，一旦出现质量问题，可快速定位批次并召回，降低食品安全风险。

**适用条件**：
- 各门店距中央厨房配送半径合理（建议≤200公里，且冷链物流完备）。
- 中央厨房具备分拣、短时仓储和冷链配送能力。
- 水果品类以耐储型为主（苹果、柑橘等）或可预加工的（如鲜切水果）。

**例外情况**：
- 若门店为现切水果主打（如高端沙拉店），且距中央厨房过远（>3小时车程），可考虑门店单独收货，但需自建验收标准和库存周转体系，成本会明显上升。

最终建议：优先测试中央厨房模式，先选取3-5家门店试运行1个月，对比水果损耗率、客户投诉率、综合成本，再决定是否全面推广。
