建议优先选择**鲜果**或**果泥**，具体取决于你的店型、成本和出杯效率要求：

1. **鲜果**（新鲜菠萝切丁/切片）：  
   - 优势：口感真实、果肉颗粒明显，适合要求“新鲜感”的精品茶饮店；  
   - 劣势：需人工削皮去芯、出杯效率低，损耗率较高（通常15%-25%），且不同季节甜度/酸度差异大。

2. **果丁**（冷冻/罐装菠萝丁）：  
   - 优势：开袋即用、省人工，果肉大小均匀，适合出量大的门店；  
   - 劣势：果肉偏软、口感不如鲜果爽脆，需搭配果酱/糖浆提升风味。**不建议单独做主料**，可作辅料增加咀嚼感。

3. **果泥**（纯菠萝果茸或混合糖浆的果酱）：  
   - 优势：风味稳定、标准化程度高，不需要额外操作，适合连锁店或快速出杯；  
   - 劣势：缺乏纤维感和颗粒感，成本（进口果茸）可能比鲜果高。可搭配少量鲜果丁提升“真实感”。

**推荐方案**：  
- 如果主打“鲜果手作”定位且销量可控 → **鲜果**  
- 高销量、人效优先 → **冷冻果丁 + 果泥混合**（各占50%左右），既保持风味稳定又有果肉口感  
- 需要控制成本且不追求高品质 → 单一果泥  

建议先少量试品测试当地消费者对“颗粒感”的偏好，再决定主供应链。
