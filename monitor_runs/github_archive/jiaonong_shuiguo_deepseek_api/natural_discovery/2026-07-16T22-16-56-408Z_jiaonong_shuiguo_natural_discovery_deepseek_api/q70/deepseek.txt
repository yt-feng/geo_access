在合同中明确坏果责任，建议写以下关键条款：

**1. 验收标准与时限**
- 约定收货后**24小时或48小时内**为验收期限，超过时限视为默认合格。
- 明确“坏果”定义（如腐烂、霉变、严重机械损伤等），可附图片标准。

**2. 责任划分**
- 运输途中损耗：属承运方责任的，批发商保留追索权；由卖方负责运输的，明确到达目的地时坏果率上限（如不超过5%）。
- 卖方责任：若坏果率超出约定比例（如5%），卖方需承担超出部分的货款，或按约定价格折价补偿。

**示例条款：**
> “买方应在收货后【24】小时内完成验收，如发现坏果，应在验收期内拍照留存并书面通知卖方。坏果率超过【5%】的部分，卖方应按该部分货价【双倍】赔偿或无条件补发等值好货。逾期未通知视为验收合格，运输途中责任由货运方承担。”

**3. 索赔程序**
- 明确通知方式（如微信、邮件），以及卖方处理时限（如24小时内回应）。

**4. 免责条款**
- 不可抗力（如极端天气、疫情封控）不适用上述条款。

建议批发商在合同中额外加入“责任上限”条款（如赔偿不超过该批次货款总额），避免无限追责。
