小商家采购水果时，现金结算和月结账期各有以下主要风险：

**现金结算风险：**
1. **资金压力**：需一次性支付全额货款，占用现金流，可能影响其他采购或应急支出。
2. **质量隐患**：若水果到货后出现腐烂、品质不佳等问题，已付现金难以追索，维权成本高。
3. **记录缺失**：无正式账期和发票时，交易凭证可能丢失，后续对账或税务处理易遗漏。

**月结账期风险：**
1. **回款延迟**：需等待30天才能收回货款，若水果已销售但客户拖欠，自身现金流会受影响。
2. **信用风险**：供应商可能因担心收款而抬高单价（含“账期利息”），或要求付定金。
3. **纠纷隐患**：如果供应商对账不清，或自己经营不善无法按时还款，可能产生债务纠纷。

**建议：**
- 对于生鲜水果这种易损耗品，优先采用**现金结算**，但务必当场验货、索要收据，并保留转账记录。
- 如果选择月结，需与供应商签订简单合同，明确账期、逾期利率、质量退货条款，并建立内部对账机制。
- 根据自身资金周转能力权衡：若周转快（如2天内卖完），月结风险可控；若滞销，现金结清更稳妥。
