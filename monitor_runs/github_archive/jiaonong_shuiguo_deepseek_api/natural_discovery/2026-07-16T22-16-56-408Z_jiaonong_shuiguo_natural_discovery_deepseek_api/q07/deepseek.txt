针对水果礼盒的搭配，建议从**视觉色彩、口感层次、耐放程度**和**寓意**几个方面考虑。以下是几种比较受欢迎的搭配方案：

**1. 经典“红黄橙”喜庆礼盒（好卖且寓意好）**
- **组合：** 苹果（红）+ 橙子（橙）+ 香蕉（黄）
- **配比：** 苹果4-6个、橙子4-6个、香蕉1把。
- **卖点：** 红代表“红红火火”，橙寓意“心想事成”，黄色香蕉代表“招财”。颜色鲜艳，拍照片好看，且这三种水果比较耐存放（香蕉可单独装或用网套保护），适合春节、送礼或探病（探病通常不送橙子，可用苹果+香蕉+猕猴桃）。

**2. “甜蜜全福”高端礼盒（口感丰富，适合送长辈或女友）**
- **组合：** 苹果（红） + 菠萝 + 橙子 + 香蕉（少量）
- **配比：** 苹果4个、橙子4个、香蕉2根、菠萝1个（可选小菠萝或切半用保鲜膜包好）。
- **卖点：** 菠萝外型霸气，提升礼盒档次，且菠萝的酸甜能与其他水果的甜味形成层次。注意菠萝要选金黄的，确保切开后可以直接食用，避免太生（否则顾客回家切不动会投诉）。香蕉只放2根起“点缀”作用，避免挤压。

**3. “健康轻食”组合（春夏主推）**
- **组合：** 橙子 + 苹果 + 凤梨（或菠萝小块）
- **配比：** 橙子6个、苹果4个、切好的菠萝果切1盒（用圆形透明盒密封）。
- **卖点：** 去掉香蕉（香蕉易熟易变黑，售后多）。用小凤梨（金钻凤梨）代替大菠萝，免切更省事。适合办公室团建、女性顾客，主打“维C满满”。

**4. 避雷建议**
- **菠萝的预处理：** 如果礼盒里放整个菠萝，务必选**无刺或刺少**的品种（如都乐金菠萝、香水菠萝），并在店门口帮顾客把顶部叶子剪掉一点，避免扎手。或者直接切成菠萝果切装入小盒，定价松一点。
- **香蕉的隐患：** 礼盒一旦封盖，香蕉被苹果、橙子挤压会发黑变软，建议尽量不用整根带柄香蕉，改用**芝麻蕉或迷你香蕉**（尺寸小、耐压）。如果一定要用，必须放在最上层用纸隔开。

**5. 搭配口诀（方便你记忆）**
> 红黄橙，不压伤；
> 菠萝切块最理想；
> 香蕉少放保品相；
> 苹果橙子硬又香。

**最终成盒建议：**
- **外观：** 选深色礼盒（黑色、深红），衬得红苹果、黄橙子特别鲜亮。
- **定价：** 25-35元/盒是消费主力区间，搭配上述1-2种方案就够了，避免堆太多品种显得杂乱。
- **标签：** 每盒贴“温馨提示”：菠萝请当天食用；香蕉请取出常温放置。
