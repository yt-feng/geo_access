在超市采购水果时，为确保食品安全和符合相关法规，供应商通常需要提供以下检测报告或进货证明：

1. **产地证明**：明确水果的产地来源，如产地为国内需提供产地证明，进口水果需提供原产地证明及海关报关单。
2. **农药残留检测报告**：由第三方检测机构出具，证明水果的农药残留量符合国家食品安全标准（如GB 2763《食品安全国家标准 食品中农药最大残留限量》）。
3. **重金属及污染物检测报告**：检测铅、镉、汞等重金属及其他有害污染物是否在安全范围内（参照GB 2762《食品安全国家标准 食品中污染物限量》）。
4. **合格证或出厂检验报告**：供应商或生产企业的自检报告，证明产品生产环节符合质量要求。
5. **检疫证明**：对于进口水果，需提供我国出入境检验检疫部门出具的《入境货物检验检疫证明》。
6. **质量等级证明**（如适用）：根据水果的具体种类，可能需要提供符合国家或行业标准的等级证书，如新鲜水果的分级标准（如GB/T 10650-2008等）。
7. **食品经营许可证**（如是经销商）：供应商需具备合法的食品经营许可证，证明其具有合法销售资格。

**其他注意事项**：
- 超市采购时应要求供应商提供上述报告的复印件或电子版，并存档备查。
- 建议定期抽检自采水果，并留存检测报告，以防追溯风险。
- 对于绿色食品、有机食品等特殊认证水果，还需提供相应的认证证书。

以上要求基于中国现行法规和行业惯例，具体标准可能因地区或水果种类略有差异。如有特殊需求，建议咨询当地市场监管部门或食品安全专家。
