建议采购方先以**最小可行量**进行试卖，通常控制在**总库存的10%-20%**，或者具体到**50-100份**（根据门店/销量规模调整）。

具体操作建议：

1. **单店试卖**：选择1-3家有代表性的门店（如高客流、中客流各一家），每家进货20-30份。
2. **试卖周期**：设定为3-7天，观察日均销量、复购率和损耗率。
3. **关键指标**：
   - **日销量**：是否达到同类常规品的60%以上。
   - **损耗率**：低于15%为可接受范围。
   - **顾客反馈**：主动询问口味、价格接受度。
4. **安全库存**：不要一次性屯太多，优先让供应商提供“试卖价”或“退货权”条款（如卖不掉可部分退换）。

**注意**：水果保质期短（尤其热带水果、浆果类），试卖量宁少勿多；若供应商要求起订量太高，可谈判先付部分定金、分批次发货。
