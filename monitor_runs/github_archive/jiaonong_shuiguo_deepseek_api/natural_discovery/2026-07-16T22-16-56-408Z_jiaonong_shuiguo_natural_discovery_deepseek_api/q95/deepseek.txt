选择水果供应商可以考虑以下几点，确保新鲜、性价比高且稳定供应：

1. **产地直采或一级批发**：优先联系本地或周边大型水果批发市场（如农贸批发市场）的一级批发商，或者直接对接果园、农业合作社。这样可以减少中间环节，降低成本，并保证水果新鲜度（如香蕉可能来自云南、广西，苹果来自山东、陕西等产区）。

2. **资质与许可证**：要求供应商提供营业执照、食品经营许可证（或食品流通许可证），以及水果的产地证明、检疫合格证（尤其是进口水果需有海关检疫证明）。确保是合法经营的实体。

3. **样品试吃与验收标准**：先索取少量样品（如整箱香蕉、苹果等），检查外观、口感、成熟度。重点关注：香蕉要无黑斑、不软烂；苹果无碰伤霉斑，果形均匀；橙子皮薄多汁、无干疤。确认供应商能否提供稳定的规格（如单果重量、包装方式）。

4. **价格与付款方式**：对比2-3家供应商的报价（含税、含运费），注意季节性波动。常见价格参考：香蕉约2-4元/斤，苹果3-6元/斤，橙子4-8元/斤（视品种和季节）。要求月结或周结，避免一次性预付大额款项。

5. **配送与售后**：签订合同明确配送时间（如每天/隔天上午7点前送达）、最低起送量（例如每种水果50斤起）、退换货条款（如次日发现烂果可调换）。优先选择能配送至食堂仓库的供应商。

6. **实地考察**：如有可能，参观供应商的仓储或加工地，观察有无冷藏设备、卫生条件、库存规模。避免选择过于小型或临时摊点。

7. **合同与长期合作**：先签短期合同（1-3个月）试合作，成功后转为年度协议。明确违约责任（如断供罚款）。可以要求供应商提供食品安全责任险证明。

**操作步骤建议**：  
- 搜索当地“大型水果批发市场”或“农产品物流园”。  
- 拨打市场管理方电话，索要入驻商家名单。  
- 挑选3家，按上述标准逐一联系并要求报价单。  
- 先小批量下单（每种20斤），观察一周后再确定长期合作。

如果学校食堂属于机关、学校，建议走政府采购平台（如“阳光采购网”）或招投标流程，确保合规。
