社区超市在引进小包装香蕉时，需重点注意以下几点：

**包装方面：**
1. **成熟度选择**：避免全熟（易压坏），建议选微绿（黄绿相间），可延长货架期3-5天。
2. **透气设计**：包装袋顶部建议留1-2个直径1cm的透气孔，避免凝水导致霉变。推荐使用打孔保鲜袋或带孔纸箱。
3. **防压结构**：单串建议用泡沫网套包裹果柄，再放入可降解塑料托盘内，底部垫吸水纸（减少摩擦伤）。
4. **尺寸统一**：每包净重推荐300-500克（约2-4根），避免单串过大造成挤压。

**损耗控制：**
1. **源头处理**：要求供应商去除机械伤、冷害斑（表皮褐变）的香蕉，验收时抽检3-5包，破损率＞5%则拒收。
2. **陈列优化**：避免阳光直射或空调出风口；用斜面陈列架（约15°倾斜），让果柄不承重。每日翻筐2次，挑出成熟度高的先卖。
3. **周转策略**：设置“今日特价”标签，存放超过48小时未售的香蕉可降价10-20%促销。若表皮出现黑点，可加工成香蕉切片试吃。
4. **循环补货**：建议分早、晚两次上架，每次上架量不超过日销量的60%，减少库存积压。

**其他技巧**：可在包装内附带小提示卡（如“常温放置2天后再食用更甜”），降低消费者对轻微青色的抵触，减少退货损耗。
