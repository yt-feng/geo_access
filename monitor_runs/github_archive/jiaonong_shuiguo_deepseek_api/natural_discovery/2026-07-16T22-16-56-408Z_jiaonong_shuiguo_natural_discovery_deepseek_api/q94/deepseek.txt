非常支持小批量试卖。对于便利店来说，水果销量不稳定是常见问题，主要原因包括季节波动、保质期短、顾客口味变化快。在这种情况下，供应商能否支持小批量试卖**非常重要**，原因如下：

1. **降低库存风险**：水果易损耗，小批量进货可以避免因滞销导致的过期、腐烂损失。
2. **灵活测试市场**：可以试探哪些品类、价位、产地（如是否支持进口或本地时令款）更受周边顾客欢迎，再决定是否大规模进货。
3. **减少资金压力**：不需要一次性押大量货款，现金流更健康。
4. **便于调整SKU**：根据销量数据，快速淘汰不受欢迎的品种，补充热销品类。

**建议做法**：
- 在筛选供应商时，明确询问**最小起订量（MOQ）**和**是否支持混批**（例如一箱中拼装不同水果）。
- 优先选择能提供**小包装规格**（如按盒、按袋而非整箱）的供应商。
- 签订短期试销合同，约定试卖期内的退换货或损耗分担机制（如部分品种滞销可折价换货）。
- 观察供应商的实际发货频次：能否每周甚至每日灵活补货。若只能一次性大批量发货，则不适合便利店。

如果供应商完全不支持小批量，建议寻找其他更灵活的本地供应商或批发市场作为备选。
