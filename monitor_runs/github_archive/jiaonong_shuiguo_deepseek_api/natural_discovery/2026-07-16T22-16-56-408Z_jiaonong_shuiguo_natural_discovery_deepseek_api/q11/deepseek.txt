会。水果店销售香蕉时，如果一箱内大小差异明显，通常会对销售产生以下影响：

1. **定价困难**：大小混杂导致无法统一按单根或按重量合理定价，顾客容易对比后觉得“小的不值这个价”或“大的没有单独选的价值”。
2. **顾客挑选问题**：顾客会挑走大的、品相好的，剩下小的、差的不良库存增加，影响整体利润。
3. **损耗率上升**：小香蕉成熟速度可能不同，易被挤压或更快腐烂，增加损耗。
4. **可能降低顾客信任**：顾客看到大小参差不齐，可能认为该店选品不严格、品质不稳定，影响复购。

**建议：**
- 优先采购**分级明确**的香蕉（如“特级果”或“一级果”，大小均匀）。
- 如果整箱进货已有大小不均，可以采取**分装或分价出售**：大香蕉可单独按量或按根高价卖；小香蕉可作特价促销、用于切盘或制作饮品、蛋糕等。
- 如果售价已定，建议在陈列前**挑出大小差异过大的**，分开放置或单独处理，不要直接整箱上架。
