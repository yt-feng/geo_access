连锁超市与水果供应商签订年度合同时，以下条款最容易出现风险或纠纷，需重点审查：

### 1. **定价与调价机制**
- **“一口价”陷阱**：若固定全年价格，未约定市场波动（如季节性涨价、天气灾害导致减产）的调整机制，供应商可能亏损或单方面断供。
- **建议**：明确价格联动公式（如参考批发市场均价或指数），保留每季度/半年协商调价的权利，并约定调价触发条件（如价格波动超10%）。

### 2. **质量验收标准**
- **模糊表述**：仅写“符合国家标准”或“新鲜无腐烂”，未细化品种、成熟度、大小、糖酸比等指标。不同水果差异大（如芒果需注明熟度控制，草莓需明确包装防损要求）。
- **建议**：附录具体技术标准（如苹果直径≥80mm，柑橘糖度≥12°），约定验收方式（抽检比例、不符合标准的退换货流程）。

### 3. **交货条款与损耗责任**
- **“到店交付”风险**：若约定超市仓库交货，运输损耗（如挤压、冷害）由供应商承担，但超市需承担到店后未及时上架导致的损耗。
- **建议**：明确交货地点（如供应商仓库或超市物流中心），约定自然损耗比例（如≤3%），超出部分由责任方承担。对易腐水果（如荔枝、蓝莓），需细化冷链运输要求及时效。

### 4. **付款周期**
- **“先货后款”过长期限**：供应商可能面临资金压力，尤其是账期超过60天时，可能默认低价倾销次品或断供。
- **建议**：约定阶梯式付款（如到货后30天支付80%，剩余20%在质检确认后支付），或采用保理、信用保险等金融工具分担风险。

### 5. **排他性与责任条款**
- **“独家供货”无例外**：若供应商要求独家供货，但超市无法保障最低采购量，供应商可能指控违约索赔。
- **建议**：若约定独家协议，需补充最低采购量（如全年采购量的70%）、除外情形（如自有品牌或进口水果），并明确违约责任上限（如赔偿额不超过合同年度总额的20%）。

### 6. **不可抗力与天气免责**
- **“天灾”范围过宽**：供应商可能将正常波动（如病虫害、台风）认定为不可抗力，要求免责。
- **建议**：明确不可抗力清单（仅限政府禁令、极端自然灾害），并约定供应商需提供保险理赔证明或第三方检测报告。

### 7. **终止与退出机制**
- **“任意解约”无补偿**：超市可能因经营调整突然终止合同，供应商备货损失无法追偿。
- **建议**：约定提前通知期（如90天），并赔偿供应链投入（如定制包装、预付定金）。

### 8. **包装与标识合规**
- **环保与标签风险**：若未明确包装材料（如禁用泡沫箱）、产地标签格式，可能面临行政处罚。
- **建议**：要求供应商提供包装符合国家标准（GB/T 31268）、标签含“水果名称+产地+采摘日期”等，并保留检测记录。

### 避坑摘要：
- **价格**：必须有浮动调节条款，避免固定价。
- **质量**：细化到品类标准，附验收流程。
- **损耗**：明确运输、库存、上架各环节责任。
- **账期**：避免过长周期，设置付款保障。
- **终止**：保留合理退场路径，避免无补偿断约。

建议双方在合同签署前，对以上条款逐项进行风险对账，并引入第三方质检机构或公证。
