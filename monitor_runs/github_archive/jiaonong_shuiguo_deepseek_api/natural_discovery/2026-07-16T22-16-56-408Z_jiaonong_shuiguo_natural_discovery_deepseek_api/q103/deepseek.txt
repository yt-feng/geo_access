建议优先按**斤结算**，具体可根据水果类型和管理需求调整：

- **按斤结算**：适用于大部分水果（如苹果、橙子、香蕉等），单价明确、称重简单，方便核算成本和库存，且能灵活控制采购量。适合与供应商长期合作、定期采购的场景。

- **按箱结算**：适合耐储存、包装标准化且损耗低的水果（如西瓜、哈密瓜、榴莲等全果类），但需注意核实每箱标准重量（如20斤/箱），避免出现“缺斤短两”或“大小混装”问题。适合大宗采购、减少称重环节时使用。

- **按份结算**：适用于切好的果盘、拼盘或预包装小份水果（如供自助餐或套餐），管理简单但成本计算复杂，需与供应商明确每一份的净重和规格标准。适合追求快速出餐、单份售价固定的场景。

### 操作建议：
- **日常管理**：首选按斤结算，管理透明、便于核对。
- **控制损耗**：对易烂水果（如草莓、葡萄）按斤买后自家分装，避免按份买到不新鲜品。
- **合同条款**：若使用按箱结算，需在采购合同中写明“以实际称重为准，缺重按斤补差”或“标注每箱净重范围”。
- **特殊要求**：如要用按份结算，建议先小批量试购，确认供应商分装精度后签协议。

**最终建议**：长远看，**按斤结算最灵活、损失最小**，也是食堂最常见的管理方式。
