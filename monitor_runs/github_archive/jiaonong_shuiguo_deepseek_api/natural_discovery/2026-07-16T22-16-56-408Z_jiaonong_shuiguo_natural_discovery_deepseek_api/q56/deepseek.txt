为确保茶饮店的水果原料安全合规，供应商通常需提供以下证明文件（具体要求可能因地区法规略有差异）：

1. **营业执照**  
   证明供应商具备合法经营资质，需在有效期内并包含食品销售相关范围。

2. **食品经营许可证**（或食品生产许可证）  
   若供应商涉及水果分装、初加工或销售，需持有对应许可证；若仅为产地直发，可能需提供产地证明或相关备案。

3. **水果产品合格证明**  
   - **农产品检测报告**：第三方机构出具的重金属、农药残留等指标检测报告（建议批次对应，非长期通用报告）。  
   - **产地证明或产地准出证明**：标注产地、采摘时间等信息，部分地区采用“食用农产品合格证”制度。

4. **检疫/检验证明**  
   - 进口水果需提供《入境货物检验检疫证明》及报关单。  
   - 国产水果需符合当地监管部门要求，部分区域可能要求动物检疫合格证明（如涉及有机水果认证的附加证明）。

5. **批次记录与追溯信息**  
   - 供货台账（含品种、数量、进货日期、来源地等）。  
   - 冷链运输温度记录（若涉及冷藏/冷冻水果）。

6. **相关认证文件（非强制但建议）**  
   - **有机认证**、**绿色食品认证**等，可提升风险管控等级。  
   - **无公害农产品证书**或地理标志产品证明。

**操作建议**：  
- 要求供应商每批次提供随货检测报告或合格证，并定期抽查第三方检测报告（建议每季度更新）。  
- 建立供应商档案，核实其单位是否曾被市场监督管理局列入黑名单或处罚记录。  
- 关注地方食药监最新政策（如“证照分离”后部分地区取消部分证书，改为备案制）。  

若无法从供应商处获取完整证明，建议立即调整进货渠道，并保留进货票据和沟通记录，以备后续核查。
