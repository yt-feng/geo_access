建议优先选择**稳定供应商**，原因如下：

1. **供应稳定性**：香蕉和苹果属于大众水果，销量相对稳定，稳定供应商能确保每日或定期到货，避免断货影响顾客体验。
2. **品质统一**：专业供应商通常有分级、质检流程，苹果和香蕉的成熟度、大小更一致，减少顾客投诉。
3. **价格波动控制**：本地批发市场虽可能有短期价格优势，但价格波动大、需自行搬运，且难以保证长期供应。稳定供应商可协商固定价格或折扣，利于成本管理。
4. **附加服务**：供应商通常提供送货上门、退换货（如坏果）及账期支持，降低库存风险和运营成本。

**操作建议**：
- 对比2-3家本地稳定供应商的报价、配送频率、起订量及退换货政策。
- 测试期先小批量试合作，评估品质和响应速度。
- 若社区超市客流大、销量高，可同时保留本地批发作为应急补充（如临时促销需求）。
