核实水果供应商全国配送能力，建议从以下步骤入手：

1. **查看资质与案例**：要求供应商提供营业执照、食品经营许可证等基础资质，并询问其过往服务过的客户（如连锁超市、电商平台等），检查是否有跨省配送的成功案例。

2. **验证物流资源**：索要供应商合作的物流公司名称（如顺丰、京东物流、德邦等冷链专线），并直接联系物流方确认其与供应商的合作关系及覆盖范围。若供应商自建物流，可要求查看车队或冷库分布。

3. **要求试点配送**：选择与采购方地理位置偏远（如新疆、西藏、内蒙古等）的地址，用100-500公斤小批量订单进行试送，记录从下单到实际签收的时效、水果品质（如新鲜度、损耗率）、物流追踪信息是否详细。

4. **确认冷藏条件**：询问并核查供应商在配送过程中是否具备全程冷链能力（如温度记录仪、GPS监控），要求提供温湿度记录。

5. **查询行业评价**：在企查查、天眼查等平台核实供应商有无“物流纠纷”记录；联系同行或行业协会（如中国果品流通协会）询问其口碑。

6. **合同明确条款**：在采购合同中约定配送范围（列举可覆盖省份）、超时或破损的赔偿标准（如2小时内延迟赔偿货值10%）、签收环节的验收流程。

注意：若供应商无法提供任何物流单号或合作物流商名称，或要求采购方全款预付后才发货，则存在较大风险。
