是的，起订量过高确实会增加损耗风险，尤其在水果采购中。原因包括：

1. **保质期短**：香蕉和菠萝都属于易腐水果，成熟速度快，存放时间有限。起订量高可能导致库存积压，来不及销售或使用而过期、腐烂。

2. **储存条件要求高**：香蕉需要特定温湿度控制（如不宜冷藏），菠萝虽稍耐放但长时间存放也容易变质。若储存空间或设备不足，损耗会加剧。

3. **需求波动难预测**：如果顾客或订单需求不稳定，高起订量可能造成实际消耗低于采购量，剩余库存更难及时处理。

**建议**：
- 尽量与供应商协商更灵活的起订量，或尝试多家供应商对比。
- 可采用分批采购、小批量多频次的模式，降低库存压力。
- 若必须接受高起订量，优先确保有足够的销售渠道、存储条件或再加工能力（如制作果切、果汁等）。

总体而言，控制起订量在合理范围内是降低损耗的关键策略。
