为减少香蕉损耗，建议在采购时明确以下要求：

**1. 熟度要求：**
- **到货熟度：** 要求供应商提供**七成熟（颜色青绿或微黄，果皮硬实，无黑点）**的香蕉。
- **原因：** 七成熟香蕉在常温下可存放3-5天，柜台售卖期长。过熟（全黄或带黑点）易在店内快速变质。
- **细化标准：** 可要求整箱香蕉中成熟度均匀，80%以上为青绿或黄绿相间（颜色指数≤4），无果皮软烂或明显伤痕。

**2. 到货时间要求：**
- **时间节点：** 要求**到货时间避开周末和节假日高峰前48小时**（如周末生意好，则周五到货的香蕉需控制熟度更低）。**最佳到货时段为每周一或周二上午**，便于在整周内分批次上架。
- **分批到货：** 如果销量变化大，可要求供应商分两次到货（如周一和周四），每次送七成熟香蕉，避免一次性囤积过熟。
- **配送环节：** 要求使用**冷链车或通风箱运输**，配送途中温度保持13-16°C，避免暴晒或挤压。

**3. 附加要求（降低损耗）：**
- 要求每箱香蕉带**成熟度标识卡**（如1-7等级），并注明采摘和装箱日期（一般要求到货时间距离采摘不超过5天）。
- **退货/调换条款：** 合同中写明：到货后24小时内开箱检查，发现有超过10%的香蕉出现明显黑斑、软烂或过熟，供应商必须无条件调换或赔偿。

**4. 到店后实操提示：**
- 到货后立即拆箱散热，避免堆叠闷热。
- 按熟度分层上架：青绿香蕉放冷柜或阴凉处（13°C），微黄的放室温区，已有黄斑的立即打折销售或做甜品原料。
- 每日检查，提前取出变软或变黑的香蕉单独处理。

**总结标准话术给供应商：**  
“请务必保证到货香蕉为青绿色（七成熟），周一或周四上午10点前送到并用冷链运输。到货如有超过10%变形或过熟，我方有权拒收或退货。”
