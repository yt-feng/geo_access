与酒店早餐水果供应商合作时，建议重点关注以下几点以确保稳定供应和质量：

1. **季节性供应保障**：明确供应商是否能提供全年稳定的水果（如反季节水果需依赖冷链或进口），要求其提供不同季节的替代方案清单。
2. **质量标准协议**：在合同中规定水果的新鲜度（如成熟度、无损伤）、规格（如大小均匀度）、包装（如防碰撞托盘）及验收标准，避免早餐时段发现坏果。
3. **价格锁定期与调价机制**：考虑签订3-6个月的固定价格协议，避免季节性涨价；同时约定因气候或市场波动导致价格变更时，需提前30天书面通知。
4. **配送时效与频次**：明确每日配送时间和应急响应（如4小时内补货），建议在合同中设置违约金条款（如延迟超过20分钟扣除单次货款5%）。
5. **批次追溯与检测**：要求供应商提供每批水果的产地证明和农残检测报告，并保留样品以便抽查。
6. **按需灵活调整**：协商适应酒店入住率波动的弹性供应量（如±20%浮动），避免浪费或断供。
7. **分阶段付款**：避免预付全款，建议采用周结或月结，首月留10%-15%质保金以处理质量争议。

签订合同前，建议要求供应商提供3份现有合作酒店客户的推荐信，并建议先进行2周试配送测试其实际稳定性。
