判断水果供应商是否靠谱，建议从以下5个核心环节入手，结合具体操作标准进行验证：

1. **资质与实体考察**  
   - **必查证件**：营业执照、食品经营许可证（注意经营范围是否包含水果）、法人身份证复印件。  
   - **实地验厂**：到供应商仓库/基地查看，重点关注：  
     * 香蕉催熟库是否规范（温湿度控制、乙烯气体使用记录）  
     * 菠萝存储区是否通风避光、堆放高度是否合理（防止压伤）  
     * 是否有独立分拣区和包装区（避免二次污染）

2. **货品质量验收标准**  
   长期合作前，要求供应商提供 **具体品控标准书面文件**，并逐条验证：  
   - **香蕉**：成熟度控制在7-8成（果皮泛黄但两端仍带微绿），无冻伤黑斑、无机械压伤。  
   - **菠萝**：果眼饱满干燥、无腐烂软斑、保留1-2片新鲜顶叶（可检查采摘时间）。  
   - 要求供应商按 **“单箱称重±0.5kg”** 作为基础计量规范，避免缺斤少两。

3. **供应链稳定性测试**  
   - **小批量试单**：先订1-2箱，观察到货时间（按小时计，如承诺24小时到货是否达标）、包装是否抗运输损耗（香蕉用气泡袋+纸箱、菠萝用泡沫网套+隔板）。  
   - **应急测试**：主动提出“某天临时加单50%”，看对方能否在6小时内响应调货（优秀供应商通常有3-5个备用基地）。  

4. **价格与账期逻辑**  
   - **价格透明度**：要求供应商提供 **“基础价+浮动因素清单”**（如产地采购价、冷链运输费、催熟加工费、包装费），能清晰列出每项成本。  
   - **账期设计**：建议采用 **“首月周结+次月月结”** 模式（首月每周结算一次，建立信任后转月结）。警惕“30天以上账期”或“需预付半年保证金”的条款。  
   - **比价公式**：同等品质下，批发价格= [“路边水果批发市场同期价”]×90%~95%（供应商应有规模优势）。  

5. **售后与风险共担机制**  
   - **退货条款**：明确“到货后24小时内可挑出坏果（香蕉单箱坏果率≤3%、菠萝≤2%），供应商免费补换”。  
   - **赔付保底**：要求合同中注明“因供应商未及时发货导致店面断货，按日均销售额的1.5倍赔偿”。  
   - **定期对账**：至少每季度要求供应商提供 **产地批次检测报告**（尤其关注农药残留、乙烯催熟剂残留量）。  

**执行建议**：  
- 优先找有 **“基地直采+自有冷链车+本市分仓”** 的供应商（可避免长途转手的损耗）。  
- 先合作3个月，每月对比 **断货次数、坏果率、响应时效** 三项数据，再决定是否签订长期协议。  
- 第一次签合同时，保留押金（比如相当于两周货款的金额），等验收三次无误后再放弃押金条款。  

如果供应商能做到上述5点的80%以上，基本就是靠谱的长期合作伙伴。遇到任何一条无法验证（如拒绝提供品控文件、不签赔付条款），建议立即淘汰。
