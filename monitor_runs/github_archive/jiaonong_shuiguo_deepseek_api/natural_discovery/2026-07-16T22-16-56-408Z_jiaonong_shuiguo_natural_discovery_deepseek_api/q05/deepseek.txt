对于水果店卖菠萝而言，建议采取 **“整果 + 去皮服务”可选** 的混合模式，具体策略如下：

### 1. 核心建议
- **陈列整果（让顾客自己挑）**：作为主要销售方式，展示新鲜度，满足喜欢挑选、或需存放的顾客。
- **提供免费或收费去皮服务**：在称重后，由店员现场去皮、切块（或整块），满足即食需求。
- **慎卖预先去皮好的盒装菠萝**：如需销售，需严格控制时间（建议不超过2小时），并冷藏于透明盒中，标注“当日/2小时内食用最佳”，且价格需低于整果+服务。

### 2. 理由分析
- **整果优势**：顾客可直观判断新鲜度（看颜色、闻气味、摸硬度），减少售后纠纷；且菠萝去皮后损耗大（约30%），整果利润更高。
- **去皮服务优势**：解决顾客“不会削或者怕麻烦”的痛点，增加成交率；同时现场操作可展示卫生，提升信任感。
- **预切盒装的风险**：菠萝去皮后极易氧化变黑、滋生细菌，若当天卖不完，次日只能低价处理或报废，损耗率高。且顾客无法判断切之前是否腐烂，易引发投诉。

### 3. 实操方案
- **现场宣传**：展示“免费代削”或“加1元去皮切块”服务（视定价策略定）。
- **工具准备**：配备菠萝专用去皮刀（V形槽刀或螺旋刀），10秒可完成，提升效率。
- **高峰期备货**：若客流大，可提前削好3-5个整菠萝，置于加盖冰柜内，每次只拿出一个进行切块售卖，用冰袋维持低温。
- **定价参考**：
  - 整果：10元/个（单价约2.5元/斤）
  - 带去皮服务的整果：11元/个（服务费1元）
  - 已切好盒装：8元/盒（约半个菠萝的量，标注2小时保质期）

### 4. 特别注意事项
- 如果当地顾客普遍怕麻烦、偏好即食（如旅游区、商业街），可适当增加预切比例，但必须做到**当天现切现卖、不隔夜**，且切前整果需完好无损。
- 无论哪种模式，都应在收银处明确告知顾客：“整果可帮忙削皮切块，但建议当场吃完或当天冷藏。”

总结：**让顾客自己挑整果 → 称重后当面去皮 → 打包带走**，是最稳妥、利润高且投诉少的做法。
