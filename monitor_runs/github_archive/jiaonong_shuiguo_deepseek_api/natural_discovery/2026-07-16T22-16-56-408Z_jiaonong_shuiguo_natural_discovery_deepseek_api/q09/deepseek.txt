在水果店旺季与供应商谈合作时，建议提前明确以下几个关键问题：

1. **货源稳定性**：询问供应商在旺季的产能、供应渠道是否可靠，是否有备选方案（比如第二基地或应急调货渠道）。  
2. **价格与结算方式**：明确旺季时的价格机制（是否随行就市、有无保价协议）、账期、押金或预付款要求。  
3. **品质标准与验货流程**：要求对方提供分级标准（如大小、甜度、瑕疵率），并确认到货后是否有当场验货、退换货的流程和时限。  
4. **物流与配送**：问清楚配送范围、运费承担方、配送时效以及旺季能否保证按时送达（比如增加次日达服务）。  
5. **售后与损耗处理**：确认到货后多大比例的损耗可由供应商承担（如运输碰伤、霉变），是否支持部分拒收或补发。  
6. **应急机制**：问对方遇到极端天气、运输延误等紧急情况时，是否有快速补货或替代商品的方案。  
7. **长期合作条件**：如果销量稳定，对方是否能给予优先供货、阶梯折扣或独家品种优势。  

建议在谈判前尽可能了解该供应商在行业内的口碑（尤其旺季履约记录），并将关键承诺写入合同或补充协议。
