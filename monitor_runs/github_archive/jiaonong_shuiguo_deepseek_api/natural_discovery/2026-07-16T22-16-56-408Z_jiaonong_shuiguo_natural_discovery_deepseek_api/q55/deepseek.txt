选水果拼盘的规格时，关键是价格、出肉率、损耗和甜度。以下是对苹果、橙子、菠萝的具体建议：

- **苹果**：选**中等个头（约180-250克/个）**。太大虽出肉多，但拼盘切片容易不均匀，且单价通常更高；太小成本省但切片少，失去视觉价值。建议选表面光滑、无斑点（少损耗）的**红富士或嘎啦**，现阶段性价比高。**切时去核损耗约10-15%**。

- **橙子**：优先**中大果（约250-350克/个）**，但不是越大越好。过大的橙子皮厚、海绵层多，出肉率反而低；同等重量下，中等偏大果实肉紧实。建议选**脐橙或冰糖橙**，果形圆润、皮薄更划算。**剥皮去籽损耗约20-30%**。

- **菠萝**：直接买**整颗菠萝（约1-1.5公斤/颗）**，避免切好的（单价贵30-50%且不新鲜）。选果眼浅、叶片挺拔、底部发香且按压有弹性的，可确保成熟度高。**去皮去芯损耗约40-45%**，所以1颗1.2公斤的菠萝，实际出净肉约650-720克。**如果用量大，可对比超市里标“原只菠萝”和“去皮净肉”的价格，算每百克价格更准**。

**综合操作建议**：
- 先计划需要多少净果肉（比如1份拼盘需300克果肉）。
- 按损耗率推算原料量：苹果需约350克原果，橙子约400克，菠萝约540克（1200克整颗能满足2人份）。
- 如果超市有**家庭装混合水果切盒**（如1公斤混合切配），单价通常低于单独购买，且省去处理损耗，但注意新鲜度和是否预拌糖水（增加重量）。
