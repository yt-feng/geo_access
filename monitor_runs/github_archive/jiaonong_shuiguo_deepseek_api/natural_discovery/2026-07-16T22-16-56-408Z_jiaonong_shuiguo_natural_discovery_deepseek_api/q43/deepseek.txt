评估水果供应商的仓配覆盖能力时，需重点考察以下维度，并结合连锁门店的实际分布进行匹配分析：

### 1. 核心评估指标
- **配送时效**：供应商能否在**24小时内**（或门店要求的保鲜期内）将水果从仓库送达门店？需明确是“周/日配”还是“一日两配”。
- **仓储布局**：供应商的仓库是否位于**门店城市或邻近城市**（如300公里范围内）？可要求提供仓库坐标和覆盖半径（如每仓覆盖3小时车程）。
- **温控能力**：是否具备**冷链设备**（冷藏车、冷库），能否分温区存储草莓、香蕉等不同温度要求的水果？检查历史温度记录数据。
- **缺货率承诺**：历史订单中**缺货次数/总订单数**是否低于3%？可要求供应商提供前6个月数据。
- **柔性补货**：当门店临时加单（如周末促销）时，是否支持**48小时内二次配送**？需明确加单的起送量和响应流程。

### 2. 数据化验证方法
- **地图覆盖测试**：用门店地址反向标注供应商仓库位置，计算每个门店到最近仓库的**运输距离**。若存在单侧路程超过4小时（非冷链）或5小时（冷链）的门店，要求供应商在该城市设前置仓或改用其他供应商。
- **历史订单复盘**：调取供应商过去6个月对类似连锁客户的配送记录，统计**准时率**（≥95%）、**破损率**（≤2%）、**到货新鲜度**（符合门店收货标准）。
- **极端场景压力测试**：模拟一次台风天气或节日爆单，要求供应商48小时内向3家分散的门店配送5吨不同品类水果，观察其调配能力。

### 3. 供应商能力分层判断
| 能力层级 | 表现特征 | 参考动作 |
|----------|----------|----------|
| 优秀 | 仓库覆盖所有门店，支持夜间配送，可提供实时库存看板与有效期预警 | 优先合作，签年度协议锁定价格 |
| 及格 | 覆盖80%以上门店，剩余门店通过干线中转+同城配送（总时效≤36小时） | 签季度协议，要求补货点设在中间城市 |
| 不及格 | 需门店自提或直接依赖物流公司拼车，无法承诺时效 | 寻找替代供应商，或将水果改为本地采购 |

### 4. 建议调查清单
向供应商索取下列文件（要求盖章或系统截图）：
- 仓库分布图（含地址、建仓年份、冷库容积）
- 过去3个月配送地图（标注各门店的配送路线和实际用时）
- 合作连锁品牌的推荐信或验收记录
- 水果损耗率数据（分品类，如叶菜类5%以内，核果类2%以内）

**执行建议**：先选2个门店最集中的城市（如上海、广州）让供应商试配送2周，记录**实际到货时间**和**水果外观温度**，再决定是否全国推广。
