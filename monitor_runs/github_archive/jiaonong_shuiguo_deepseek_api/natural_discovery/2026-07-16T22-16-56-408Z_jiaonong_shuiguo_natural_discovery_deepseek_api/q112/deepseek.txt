在水果采购合同中，坏果率的赔付标准没有统一法律强制规定，主要取决于双方协商。通常建议：

1. **常见标准**：坏果率超过 **5%-10%** 时，可要求赔付。具体比例应写在合同中。
2. **行业惯例**：高端水果（如进口车厘子）标准更严，可能要求坏果率不超过3%-5%；普通水果可放宽至8%-10%。
3. **合同条款**：明确约定：
   - 坏果率的定义（如腐烂、磕碰、霉变等）。
   - 验收方法（抽样比例、第三方检测机构）。
   - 赔付方式（按单价折算、按重量扣除、无条件退货或换货）。
4. **实际操作**：若合同未约定，可参照《民法典》中关于质量不符合约定的规定，与供应商协商补偿。建议首次合作时，直接约定“坏果率超过5%按单价1.5倍赔偿”等明确条款。

请务必将具体比例写入合同，避免口头约定。如已有纠纷，保留好现场照片、验收单等证据。
