在超市与水果供应商的账期谈判中，常见的账期长度通常在**30天到60天**之间，具体取决于以下几个因素：

1. **超市规模和议价能力**：大型连锁超市（如沃尔玛、永辉等）因采购量大，通常要求较长的账期，常见为**45-60天**；中小型超市或社区店可能被要求缩短至**30天**左右。
2. **水果种类和保质期**：保鲜期短的水果（如草莓、荔枝、蓝莓）周转快，账期可能较短（**7-15天**）；耐储存的水果（如苹果、柑橘、柚子）账期可更长，甚至达到**45-60天**。
3. **供应商实力**：大型或品牌供应商可能更强势，账期较短（**15-30天**）；小型供应商为争取合作，可能接受**30-45天**账期。
4. **行业惯例**：中国零售行业生鲜品普遍账期在**30-45天**，部分便利店或生鲜电商可能压缩到**15-30天**。
5. **合作深度**：首次合作往往要求现结或短账期（如**7天**试用）；长期稳定合作后才可能延至**45-60天**。

**建议**：谈判时可先提出**45天**作为中点，并解释水果周转特点。如对方坚持，可折中至**30天**，并约定按时支付以避免影响生鲜供应。如果是冷链或高端水果（高损耗率），建议账期不超**30天**。
