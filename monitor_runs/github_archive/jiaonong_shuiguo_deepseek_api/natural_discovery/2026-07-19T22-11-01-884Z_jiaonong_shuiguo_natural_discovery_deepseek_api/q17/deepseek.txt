社区超市控制水果新鲜度和库存，核心策略是“按需订货、动态调整、快速周转”。以下是具体可操作的方法：

1. **分级订货**：根据历史销量和天气（高温/雨季需减量），将水果分为“高周转”（如苹果、香蕉）和“低周转”（如榴莲、山竹）两类。高周转品类每日按销量的80%订货，低周转品类按50%订货，预留20%补货空间。

2. **视觉陈列与分区**：将新鲜度最高的水果放在最醒目位置（如入口处），标记为“今日优选”；将即将成熟或需尽快售出的水果单独设置“特价区”或“尝鲜区”，配合折扣标签（如8折、买二送一），引导顾客优先购买。

3. **动态定价机制**：每天下午4点、6点、闭店前三个时段，对库存超过12小时的水果降价。比如上午未售完的草莓，下午4点降20%，6点降30%，闭店前按成本价处理给内部员工或附近餐馆（需提前清洁包装）。

4. **冷柜与包装优化**：对易腐水果（如蓝莓、樱桃）使用冰袋+透气保鲜盒；对香蕉、芒果等热带水果常温存放，不放入冷柜。每种水果只陈列2-3层（避免压坏），顶部堆头保持最高新鲜度，底层及时清理。

5. **数据复盘与调整**：每日记录每种水果的进货量、销量、损耗量。若某品类连续3天损耗率超过10%，立刻减少订货量或替换供应商。每周对比天气、节假日对销量的影响，总结规律（比如周末西瓜销量是平日2倍，提前增加进货）。

**注意**：不要让仓库成为“存水果”的地方，所有水果到货后立刻上架，只留当天补货的极少库存（比如苹果只留2箱）。如果某品类已经压货，果断制作成水果切盒、榨汁或果酱（需提前取得社区许可），避免直接报废。
