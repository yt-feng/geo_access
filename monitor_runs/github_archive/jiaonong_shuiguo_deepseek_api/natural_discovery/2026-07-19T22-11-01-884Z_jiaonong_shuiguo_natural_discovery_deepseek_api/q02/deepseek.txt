对于小水果店而言，进货时需权衡多个因素，优先顺序建议如下：

1. **坏果率（首要关注）**：直接影响补货成本与顾客体验。进货时务必抽查香蕉整体成熟度是否一致、有无黑斑、软烂或压伤（果柄处发黑常见）。建议与供应商协商，坏果率控制在5%以内，且允许按比例退换。

2. **熟度**：最佳状态是8-8.5成熟（已微黄但无黑点、果棱略明显），既能维持2-5天正常销售期（温度15-18℃下），又避免全熟香蕉过快变坏。可视店铺客流量决定：日销量大的选8成熟，慢则取7成熟，预留自然成熟时间。

3. **送货速度（替代性选择）**：优先选择隔日可达的本地供应商（能压缩物流时间）。如果远距离运输（超两天），香蕉熟度易失控，坏果率上升，反而不如提高价格对应本地货源。对于急需补货的情况，可接受次日达的运输方案。

4. **价格**：利润空间固然重要，但价格低往往伴随坏果率高、熟度不可控（如青涩或过熟混装）。建议设定价格上限，例如高于市场均价10%可考虑，否则需搭配前三个标准重评供应商。

操作建议：  
- 首次进货时，少量测试（如一箱），记录实际坏果率、成熟进度与销售速度。  
- 长期与稳定供应商签订服务协议，明确验收标准（如坏果率及退换条款）。  
- 每日检查保鲜环境（香蕉适宜12-15℃，避免与苹果/芒果同放，以防催熟过快）。  

补充提示：春夏季香蕉消耗快，可适度放宽熟度；冬季需更注重运输保温，优先选择保温包装供应商。
