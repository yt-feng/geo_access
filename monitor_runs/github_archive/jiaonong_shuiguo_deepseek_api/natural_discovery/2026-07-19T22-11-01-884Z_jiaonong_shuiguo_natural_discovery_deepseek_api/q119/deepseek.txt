在水果生意中，选择供应商时容易被忽视但影响长期稳定的细节包括：

1. **供应稳定性与波动预案**：除了价格和品质，需关注供应商在极端天气、节假日或产地歉收时的供货能力。是否有多产地备选或应急调货机制，避免断供风险。

2. **分拣与包装标准**：水果损耗率高，需明确供应商的分拣等级（如大小、成熟度、瑕疵率）和包装方式（如透气性、防震措施）。忽视这些可能导致运输中损耗超预期。

3. **冷链物流细节**：是否全程温控？冷藏车温度记录是否可追溯？部分供应商为节省成本会在中途关闭冷机，影响新鲜度，尤其是高货值水果（如樱桃、蓝莓）。

4. **售后与退换货条款**：明确烂果、压坏、规格不符等问题的赔付比例和时效。有些供应商只愿补货不退现金，或需要顾客提交证据流程繁琐，需提前在合同中约定。

5. **溯源与合规性**：水果可能涉及农药残留、进口检疫等环节。需检查供应商是否提供产地证明、检测报告，以及是否符合当地销售资质（如需开具发票、食品经营许可证等）。

6. **付款与账期灵活性**：长期合作中，付款周期是否可按销量浮动？一些供应商要求预付全款会占用资金，而优质供应商可能接受“部分预付+售后结算”模式。

7. **隐性成本**：如最低起订量、运输距离导致的额外运费、是否需要自己租冷库。小批量采购时，单价可能远高于大客户报价。

建议：首次合作先小批量试单，观察实际到货品质、到货时间和售后处理效率；定期对比2-3家供应商的类似条款，防止长期依赖单一渠道。
