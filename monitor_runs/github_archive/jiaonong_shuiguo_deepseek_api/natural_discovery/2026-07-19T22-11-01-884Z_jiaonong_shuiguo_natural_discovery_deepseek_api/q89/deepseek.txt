针对团购客户价格敏感的特点，建议优先采用**阶梯价**，而非返利。原因如下：

1. **直接满足价格敏感需求**：阶梯价（如买越多单价越低）能让客户在购买时就明确看到优惠，降低决策门槛，符合他们对低价、透明定价的期待。返利通常需要达到一定条件后返还，客户有滞后感，对即时价格敏感群体吸引力不足。

2. **激励批量购买**：阶梯价直接与采购量挂钩，能自然引导客户增加订单规模，尤其适合团购场景中不同规模的客户（如10人、50人、100人团分别对应不同单价）。

3. **返利的潜在问题**：返利（如年底返点、下次抵扣）对价格敏感型客户可能不够直接，且后续结算复杂，有时更适用于长期合作或复购率高的企业客户，而非单次或低频团购。

**具体建议：**
- 设计2-3档阶梯价（如5人团、20人团、50人团），突出“买得越多省得越多”。
- 保留少量返利作为额外激励：例如阶梯价基础上，对达到极量的订单（如100人以上）额外给1%金额的购物券，但核心以阶梯价为主。
- 如果客户对返利有偏好，可将其转化为“即时折扣”：如“满1000元减50”的变相返利，效果优于事后返还。

实施时，明确告知客户阶梯价规则，并考虑试用小范围测试两种策略的转化率。数据会帮你验证哪种更适合你的具体客户群。
