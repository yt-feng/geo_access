选择苹果、橙子和菠萝的规格，建议从**单位价格**和**出肉率（可食用部分比例）**两方面衡量。以下提供一些通用建议：

1. **苹果**  
   - 选择**中等大小（单果约150-200克）**的规格，因为大小差异不大，但果核占比稳定。  
   - 若按重量买，大果和小果去皮去核后的出肉率相近（约85%-90%），但小果需更多处理时间。  
   - 建议按**整箱或散称比单价**：大果通常单价更低，但拼盘对切后美观度可能不如均匀的中果。

2. **橙子**  
   - 选择**单果约200-250克**的中大果，出肉率高（约70%-75%），果皮厚薄适中。  
   - 特例：若用于切片拼盘，**大果切片更大更美观**，但单位重量价格可能略高；若榨汁或切瓣，中果更划算。  
   - **避免过于小果**，因皮占比例大，实际果肉少。

3. **菠萝**  
   - **选择整只带皮（约1.5-2千克）**比自己买切好的划算很多。  
   - 出肉率约55%-65%，大规模商用常选**大果**，因去皮、芯后的剩余部分比例更高。  
   - 注意：若餐厅用量大，可找批发商按**箱/件购买**，单价更低；避免买已去皮切块的（溢价约30%-40%）。

**总体建议**：  
- 先计算**“可食用部分单价”**：即可食用重量 ÷ 总价。例如带皮菠萝2元/500克，去皮后实际200克果肉成本约1元，而预切块可能2-3元。  
- 若用量稳定，直接与供应商联系按箱采购，并要求试算样品出肉率。  
- 拼盘成本控制关键是把**果皮、果核等损耗成本摊入售价**，而非单纯看斤价。
