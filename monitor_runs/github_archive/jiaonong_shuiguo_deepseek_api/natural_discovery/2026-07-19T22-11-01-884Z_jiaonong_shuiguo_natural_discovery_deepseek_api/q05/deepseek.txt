在水果店卖菠萝，建议结合以下因素综合考虑，优先推荐 **“提供整果 + 现场去皮服务”** 的模式：

1. **顾客偏好差异大**：
   - 部分顾客喜欢自己挑选整果（看重新鲜度、外观、大小），愿意自己处理。
   - 另一部分顾客更看重便利，希望直接买到去皮好的果肉（节省时间、避免处理麻烦）。
   - 如果只卖一种，会流失另一类客群。

2. **利润与损耗平衡**：
   - 卖整果损耗低（顾客自行承担坏果风险），但单价可能较低。
   - 卖去皮果能提高单价（附加服务费），但需自行承担削皮损耗（约30%-40%重量损失）和变质风险（需当日售完）。
   - 提供现场去皮服务可灵活切换：顾客选整果后付费去皮，或直接买已去皮果。

3. **实际操作建议**（可选）：
   - **常规方案**：同时陈列整果和已去皮果肉（用保鲜膜包装），标注不同价格（去皮果每斤略贵），并允许顾客现场选整果后由店员免费去皮（需提前说明是否收费）。
   - **试水方案**：先以整果为主，在高峰期（如周末）提前削好3-5个去皮果试销，观察顾客反应再调整。
   - **注意**：避免提前削太多，防止卖不完变质；去皮果需冷藏保存并贴标签提醒当天食用。

**推荐优先级**：提供整果 + 现场去皮服务（免费或低价） > 单售整果 > 只售去皮果。
