采购方核实水果供应商全国配送能力的步骤如下：

1. **要求提供配送范围列表**：直接请对方提供详细的配送城市/区域清单，包括是否覆盖乡镇级、是否区分冷运与普运区域。
2. **测试小批量订单**：选择3-5个不同距离、不同省份的代表性地址（如一线城市、偏远县城、季节性封路地区），下单小额常规水果，验证配送时效、到货损耗与物流商。
3. **核查物流合作方**：要求供应商提供合作的物流公司名称及协议，并致电物流方（如顺丰、京东、德邦冷运等）确认是否接受该供应商的常年委托。
4. **索要历史配送记录**：请对方提供近1-3个月发往不同省份的快递单号或物流轨迹截图（注意去掉关键隐私信息），通过快递官网查询是否真实可查。
5. **询问特殊区域案例**：主动提出新疆、西藏、海南、县级以下等特殊区域，看对方是否能具体回应配送条件（如时效延长、额外包装费、不可保证的品类）。
6. **实地或远程调取仓库能力**：若为大型供应商，视频查看或委托第三方验货其分仓布局、冷库容量、分拣设备，如果对方只有单一产地直发，则“全国配送”可能仅限少数城市。
7. **签约约定与违约金**：在合同中明确“配送延误责任”、“覆盖不力时的退款/赔偿条款”，保留实际未覆盖时追究的依据。

注意：如果对方只说“几乎都能发”而拒绝给具体清单或测试单，很可能能力有限。小额测试是最直接的验证方式。
