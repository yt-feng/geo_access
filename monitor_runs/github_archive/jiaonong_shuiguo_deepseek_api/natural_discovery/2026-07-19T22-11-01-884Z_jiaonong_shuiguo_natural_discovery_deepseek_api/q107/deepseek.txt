是的，起订量太高确实会增加损耗风险。具体来说：

1. **保质期限制**：香蕉和菠萝属于易腐水果，成熟速度快，尤其香蕉在采摘后还会继续释放乙烯加速熟化。如果起订量超出短期销售或消耗能力，未及时处理的部分容易过熟、变质。

2. **储存条件要求**：香蕉和菠萝对温度、湿度敏感（如香蕉不宜低于12°C冷藏，菠萝成熟后最好通风冷藏）。大批量囤货若仓储设备不足或条件不当，会加速损耗。

3. **销售与需求匹配**：如果客户群体消费量有限，或市场需求波动较大（如季节性因素），高起订量可能导致库存积压。同时，水果外观损伤也会随储存时间延长而增加。

建议根据以下方法降低风险：
- **与供应商协商**：尝试压低起订量，或要求小批量多次发货以匹配实际需求。
- **优化库存管理**：采用“先进先出”原则，并监控品控状态，及时将临期或受损果品降价促销（如果条件允许）。
- **明确合作协议**：在合同中约定损耗承担方式（如允许一定比例退货或调换），避免全责自担。

如果当前起订量已无法调整，可增加分销渠道（如合作餐饮、加工企业）以加速消化库存。
