针对商超采购菠萝的销售方式，建议优先选择**切好的（即去皮、切块、盒装）**，具体原因如下：

1. **符合消费趋势**：当前消费者更追求便捷性，切好的菠萝可直接食用，节省处理时间，尤其适合上班族、年轻人和家庭用户。
2. **减少损耗**：整果销售时，顾客需自行削皮、去眼，可能因操作不熟练导致浪费或口感不佳；切好的产品可统一处理，降低因个人操作不当引发的投诉和退货。
3. **标准化定价**：切好后的菠萝可按重量或盒装统一定价，避免因整果大小、外观差异导致的定价困难，更易形成稳定的毛利。
4. **视觉吸引**：切好的菠萝通常呈金黄色，摆放在冷藏柜中视觉效果更佳，易激发购买欲望。

**如果门店具备以下条件，可考虑整果与切好并行销售**：
- 有专门的水果切割服务区，能现场加工并展示。
- 目标客户群体中老年人比例较高（部分习惯整果采购）。
- 整果主要用于节日礼盒或批发渠道。

**操作建议**：
- **切好的为主**：将80%的库存加工为切块/盒装，剩余20%作为整果陈列。
- **加工时间**：根据客流高峰（如上午10点、下午4-6点）分批切割，避免氧化导致颜色变深。
- **包装**：使用透明盒+封膜，标注保质期（建议24小时），搭配叉子或削皮工具（如含牙签）。
- **促销**：切好的可设置“试吃装”，整果可捆绑售价略高的“菠萝+削皮器”套餐。

总之，如果追求**高流转、低损耗、标准化**，建议以切好的为核心销售形式；若门店客流结构复杂，可小范围保留整果作为补充。
