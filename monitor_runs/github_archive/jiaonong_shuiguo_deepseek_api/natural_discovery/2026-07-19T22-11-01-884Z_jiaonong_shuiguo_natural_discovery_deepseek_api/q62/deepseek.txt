判断香蕉批发的熟度与坏果风险，主要依靠**外观、触感、颜色和气味**这四类指标。以下是一些实操性较强的判断方法：

1. **看颜色与光泽**  
   - **理想成熟度**：果皮呈均匀的亮黄色（带少量绿尖），光泽自然，没有过多黑斑或暗沉。  
   - **过熟/风险信号**：果皮出现大面积黑色斑点或褐化，尤其集中在茎部或棱角处，说明呼吸高峰已过，易快速腐烂。  
   - **未熟**：整体青绿、硬实，这种香蕉将需要额外催熟。

2. **摸硬度与弹性**  
   - 轻轻按压香蕉中部（不要用力捏两头）。  
   - **合适**：稍有弹性，指压后略微回弹，不软塌。  
   - **过熟/坏果风险**：触感软烂、发黏或汁水渗出，表明果肉已开始发酵或受损。  
   - **硬邦邦**：完全没熟，需要时间催熟，尤其注意冬季低温下易青硬不转色。

3. **检查果柄（茎）与果梗**  
   - 新鲜香蕉**果柄**应为绿色带脆性，折断时有较清晰的断口。  
   - 若果柄变黑、皱缩或发霉，说明货架期有限，或已经受到微生物侵蚀。  
   - 果梗（香蕉顶部）若有水渍状或变软、变黑，意味着果肉可能已腐坏。

4. **嗅闻气味**  
   - 正常成熟香蕉有轻微甜香。  
   - **坏果风险信号**：闻到酒糟味、酸味或霉味，说明果肉已过度发酵或感染病菌，这类香蕉不宜长期储存。

5. **观察包装与储存环境（尤其是冷链）**  
   - 询问供应商采摘到货时间，理想情况下3-5天内完成流通。  
   - 检查包装内是否有潮湿积水、发霉或大量“水锈”（果皮上灰褐色斑），这些会加速腐烂。  
   - 若拿货时香蕉温度明显高于常温，且堆码过密、无通风，内部坏果风险较高。

6. **抽样检查**  
   - 从不同位置（上层、中层、底层）各取2-3根掰开检验。  
   - 果肉颜色应为**淡黄至乳白色**，中心略微发白。若果心发黑、褐色或有水渍状，说明内部已坏，整箱风险大。

**简化的现场判断口诀**：  
- **颜色均匀亮、果柄绿且脆、按压有弹性、无酒霉味** → 货好。  
- **大面积黑斑、果柄干黑、按压软烂、箱底有汁水** → 立即拒绝或降价处理。  

如果是为了长期销售，优先选择**青黄色、果柄鲜绿**的“八成熟”香蕉，视市场需求短期催熟；如果是为了当天卖完，则可放心拿亮黄色、略有弹性的“九成货”。
