酒店采购水果盘原料时，供应商应确保以下规格和外观标准：

1. **新鲜度**：水果应无腐烂、无霉变、无软烂或过度成熟迹象，表面无异常斑点或变色。
2. **外观完整性**：果皮无机械损伤（如划痕、挤压伤）、无虫蛀孔洞、无疤痕或冻伤痕迹。
3. **大小均匀性**：同一批水果的直径或重量应基本一致（例如苹果直径7-8厘米，葡萄粒大小均匀），避免大小差异过大。
4. **色泽达标**：颜色应符合品种特性（如红富士苹果红色面积≥80%，柠檬呈亮黄色），无过度发暗或发青。
5. **清洁度**：表面无泥土、农药残留、蜡质涂层（除非是食用蜡）、灰尘或包装材料碎屑。
6. **成熟度适中**：根据不同水果要求（香蕉不可全青或全熟、梨注意花萼处是否塌软），需可食用但不过于软或生涩。
7. **无异常气味**：无发酵味、酸败味、化学药剂味或其他腐臭味。
8. **包装规范**：使用食品级托盘或纸箱包装，每层分隔避免挤压，外包装标明品种、产地、采摘日期及产地信息。

建议采购方与供应商签订验收标准清单，并设随机抽检比例。若发现不合格品，应要求退货或按比例扣款。
