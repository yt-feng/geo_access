选择学校食堂水果供应商时，建议按以下步骤操作，确保质量、价格和供货稳定：

1. **明确采购要求**  
   - 确定每日/每周采购量、品种（香蕉、苹果、橙子等）、规格（如苹果直径70mm以上）及成熟度（如香蕉不要过生或过熟）。  
   - 制定验收标准（如无腐烂、无机械损伤、农药残留检测报告等）。

2. **筛选供应商渠道**  
   - **本地批发市场**：直接接触多家批发商，可现场看货、议价，适合中小规模采购。  
   - **区域水果专业合作社或种植基地**：与山东、陕西（苹果）、广西（香蕉）、江西（橙子）等产地的合作社直签，减少中间环节。  
   - **大型农产品电商平台**：如美菜、快驴等，提供标准化配送和发票，适合需要稳定供应的学校。  
   - **本地生鲜配送公司**：优先选有教育系统服务经验的（查其学校供应案例）。

3. **比价与试供**  
   - 要求3-5家供应商提供报价单（含包装费、运费、税点），对比同品种、同规格的价格。  
   - 安排1-2周试供期，每日随机抽检水果品质（成熟度、重量、有无腐烂），留存照片记录。  
   - 重点关注香蕉：易熟烂，要求供应商分批配送（如半熟蕉+催熟蕉混合），避免单次过量。

4. **核查资质与风险**  
   - 必须提供营业执照、食品经营许可证（或产地证明）、员工健康证。  
   - 要求每批水果附农药残留快速检测报告或产地合格证。  
   - 签订合同明确：退换货条款（如2小时内腐烂免费退）、最低供应价保护（如涨价需提前通知）、违约责任。

5. **长期评估机制**  
   - 建立供应商评分表：价格（30%）、品质合格率（40%）、送货准时率（20%）、售后响应（10%），每月打分淘汰末位。  
   - 对核心品种（如苹果）备选2家供应商，防止断供。

**总结**：优先选择本地有资质、能提供检测报告的规模供应商，通过试供期验证再签长期合同。避免直接选最便宜的一家，重点关注品质稳定性（尤其香蕉的成熟度控制）。
