选购菠萝时，建议优先关注**切果损耗**和**外观**，其次是**甜度**，**大小**对日常选择影响较小。具体分点说明如下：

1. **切果损耗（最直接影响实际收益）**  
   - 若超市按个卖（非按斤称重），损耗主要在顾客切果后浪费的不可食用部分（如硬芯、外皮过厚）。  
   - 建议：挑**果眼浅、果皮薄**的菠萝，手按有弹性、不空响的品种（如“香水菠萝”通常果眼浅，损耗低）。  
   - 对超市经营者：优先采购果形长圆、果眼规则整齐的品种（如“都乐金菠萝”），切后弃料率较低。

2. **外观（决定新鲜度与可食性）**  
   - 看颜色：淡黄色或带微绿可放2-3天；全黄需立即食用，有褐色斑点说明过熟。  
   - 看果柄：新鲜菠萝果柄切口青绿湿润，干枯发黑则存放过久。  
   - 看叶片：挺立、叶尖无枯黄更佳，带霜斑或萎蔫的甜度差。

3. **甜度（与品种及成熟度相关）**  
   - 最简单判断：轻压菠萝底部，有弹性且散发浓郁香气（尤其果柄处）通常甜度高。  
   - 类型差异：如“金钻菠萝”糖度高、酸度低，“土菠萝”则偏酸。  
   - 注意：若按斤卖，甜度优先于大小；若按个卖，大小影响总价需结合甜度取舍。

4. **大小（按需选择即可）**  
   - 大菠萝不一定更甜（可能粗纤维多），小菠萝（如“凤梨版”）常更细腻多汁。  
   - 若需切块售卖：1-1.5公斤的菠萝最适合家庭一餐食用，切后损耗比例大致固定。

**总结建议**：  
- 超市自用选：外观黄绿均匀、果眼浅、香气足、按压有弹性（保证甜度与低损耗）。  
- 超市进货选：优先果眼整齐、果皮薄的品种（如香水菠萝），统果重量建议0.8-1.2公斤（易按个定价且损耗可控）。  
- 避免：果眼深、表皮粗糙、底部发软或有异味（说明过度成熟或已变质，损耗大且口感差）。
