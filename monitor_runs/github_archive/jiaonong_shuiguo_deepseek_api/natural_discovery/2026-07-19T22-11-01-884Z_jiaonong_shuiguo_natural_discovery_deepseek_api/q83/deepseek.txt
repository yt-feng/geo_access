生鲜电商的水果出现坏果后，责任划分主要取决于**坏果产生的时间点及原因**。一般遵循以下原则：

1. **发货前已损坏**（如采摘、包装、储存环节）：责任方为**商家/平台**。例如水果本身不新鲜、包装不当导致挤压、发霉等。
2. **运输途中损坏**（如快递暴力分拣、延误、高温）：责任方通常为**物流方**，但实际赔付责任由**商家/平台**先向顾客承担（因顾客与商家有买卖合同），再由商家向物流追偿。
3. **签收后短时间损坏**：如果顾客签收后发现坏果（尤其是24小时内），且能证明非人为（如切开后内部已腐烂），通常视为商家未尽到充分告知或运输条件不当，由商家负责。
4. **顾客保存不当或超时未取**：责任由**顾客自行承担**。

**常见处理方式**：
- 若商家在页面明示“坏果包赔”“按比例赔付”等承诺，则按承诺执行（如坏几个赔几个，甚至全额退款）。
- 若无明确承诺，顾客可依据《消费者权益保护法》，要求赔偿。通常顾客只需提供**坏果照片+快递单号+开箱视频**作为证据。

**实际建议**：
- 顾客发现后立即拍照（坏果细节、快递单号、包装），24小时内联系商家或平台客服。
- 商家应主动承担，避免纠纷负面评价。责任在物流时，商家先赔顾客再追偿物流；责任在自身时，直接退款或补发。
- 物流方可根据签收记录（是否当验货）来划定责任：若签收后未当面开箱且无保价，赔付可能较低。
