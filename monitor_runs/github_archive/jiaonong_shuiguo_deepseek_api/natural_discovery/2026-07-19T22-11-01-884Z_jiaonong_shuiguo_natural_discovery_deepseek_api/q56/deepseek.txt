茶饮店为了控制食品安全风险，从水果供应商处采购时，应要求其提供以下核心证明文件（依据中国现行食品安全法规）：

1. **食品经营许可证**（或生产许可证）：供应商必须持有有效的《食品经营许可证》，且经营范围需包含“水果销售”或相关类别（如预包装食品销售、散装食品销售等）。如果是直接从产地收购的大宗水果，需提供产地证明或食用农产品合格证。

2. **营业执照**：供应商合法的经营主体证明，需在有效期内。

3. **产品检验合格证明**：
   - 每批次水果的**出厂检验报告**（常见项目包括农药残留、重金属、微生物等）。
   - 如果水果是预包装的，需有**第三方检测报告**（通常要求一年内有效，重点检测农残、防腐剂等）。
   - 如果是进口水果，需提供**出入境检验检疫证明**（入境货物检验检疫证明）及海关报关单。

4. **农产品产地证明或合格证**：对于食用农产品（如直接采收的水果），供应商应出具产地证明（可由产地村委会、农业合作社或农业部门开具）或《农产品质量安全合格证》，标明产地、生产日期、生产者信息等。

5. **供货商资质文件的复印件**：供应商提供的以上所有证件的复印件（加盖公章），以及供应商的联系方式、地址等基础信息。

**额外建议：**
- 对于高风险水果（如草莓、蓝莓、葡萄等易腐且农残风险较高的），建议要求供应商提供**当批次或近3个月内的农残专项检测报告**。
- 建立**进货查验记录**，如实记录供应商名称、地址、联系方式，以及水果名称、数量、进货日期、检验报告编号等信息，保存期限不少于2年。
- 建议与供应商签订**食品安全责任协议**，明确不合格产品的召回和赔偿责任。

所有纸质证明文件需妥善存档，以备市场监管部门检查。
