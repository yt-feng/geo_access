针对批发档口拓展餐饮客户，菠萝和香蕉的规格建议如下（注重实用性和餐饮业常见需求）：

**一、菠萝规格建议**
1. **品种**：优先选择**都乐金菠萝**（甜度高、无需挖眼）或**徐闻菠萝**（香味浓郁、性价比高）；若客户做饮品或果切，可准备**无眼菠萝**以降低损耗。
2. **重量**： 
   - **小规格**：单果500-700g（适合饮品店、沙拉店，果芯细、出肉率高）。
   - **大规格**：单果1.2-1.5kg（适合酒店自助餐、果切拼盘，需要提前熟化）。
3. **成熟度**：提供**半熟果**（按需催熟，适合长期备货）和**即食果**（约8成熟，表皮微黄，适合现切）。
4. **包装**：独立网袋+泡沫箱（防止运输磕碰），标注成熟度及是否需要去皮服务。

**二、香蕉规格建议**
1. **品种**：主推**巴西蕉**（果肉紧实、耐运输）或**小米蕉**（小巧、适合儿童餐）；避免软皮过熟品种。
2. **重量与长度**：
   - **餐饮标准**：单根长度15-18cm（装盘美观），整箱净重12-13kg（行业标准尺寸）。
   - **特殊需求**：果切类可提供单根较短的“断指蕉”（约10cm，直接切片）。
3. **成熟度**：按客户用途分：
   - **熟化标准金黄**（表皮无黑斑，适合当天使用）。
   - **七成熟绿皮**（可冷藏储存3天，适合日常补货）。
   - **急用催熟**：建议备注“已达6成熟，勿与苹果同放”。
4. **包装**：瓦楞纸箱内垫防潮纸，**绝不加聚乙烯袋**（会导致褐变）。标注“不可与洋葱、土豆混放”（乙烯敏感）。

**三、附加服务（提升竞争力）**
- **样品试用**：按客户需求免费提供5根或单果样品（需提前付费退回）。
- **定制配送**：提供按周/月的时间段配送（如：每周二、四早6点前送达）。
- **损耗担保**：承诺首批订单如因规格导致退货，全额退款（增强小客户信任）。

**四、对话模板（接单时使用）**
> 客户问：“你们香蕉切开后一天就变黑了，能抗褐变吗？”  
> 回答：“香蕉切开后接触氧气会变黑。如果您需要长保鲜，我可以按规格批发**半熟绿香蕉**，配您专用催熟架。更有**真空包装切片**，冷藏能保存3天。”

**五、风险提示**
- 菠萝如有黑心、水眼，需提前用刀划开检查（餐饮客户多为切片食用，对内部品质要求高）。
- 避免同时售卖青皮和金黄香蕉（不同成熟度的混装容易引起客户投诉）。

根据目标客户的菜单类型（自助餐/茶饮/果切），可灵活调整规格。例如：茶饮店通常需要**500g小菠萝+绿皮香蕉**用于果茶装饰；而学校食堂更需要**大菠萝熟化果**。
