从管理角度分析，**按斤结算**通常更科学、灵活且透明，但具体需结合食堂规模、水果种类和人力成本综合判断。以下对比供参考：

1. **按斤结算**：  
   - **优点**：精确控制成本，避免浪费；适合品种多、价格波动大的水果（如车厘子、草莓）；便于统计人均用量。  
   - **缺点**：需配备电子秤，增加称重和记录时间；部分水果（如西瓜）需提前处理（去皮去籽）。  

2. **按箱结算**：  
   - **优点**：批量采购价格低，搬运和清点效率高；适合标准包装、耐储存水果（如苹果、橙子）。  
   - **缺点**：易出现箱内质量不均；若单箱重量差异大（如芒果箱），易产生隐性成本；小食堂可能库存积压。  

3. **按份结算**：  
   - **优点**：直接配送到餐桌，减少分拣人力；适合即食水果（如切好的西瓜或果盘）。  
   - **缺点**：单价较高，成本控制难；供应商可能调整份量或质量；需明确份量标准（如每份200g）。  

**建议**：  
- **大食堂或成本敏感型**：主选按斤+辅选按箱（核心品类如香蕉、苹果选箱装，减少称重次数）。  
- **小型食堂或注重效率**：按份与按箱结合（如香蕉按箱进，西瓜按份售）。  
- **核心管理指标**：必须要求供应商提供计价依据（如每斤单价、每箱净重）、定期抽检重量或份量，并保留台账备查。
