避免买到太生或太熟的香蕉，可以记住以下几个实用技巧：

1. **看颜色**：选择通体呈**鲜黄色**、两端略带微绿的香蕉，这样的通常最适合当天或第二天食用。如果全黄且带黑色斑点，说明过熟；如果全绿且硬，说明还没熟透。

2. **摸手感**：轻轻捏一下香蕉柄或果身，**有弹性但不软塌**的最好。太硬则生，一捏就凹陷则太熟。

3. **观察棱角**：香蕉的棱角越明显，说明越生；棱角圆润平滑的，说明熟度刚好。

4. **闻气味**：熟透的香蕉会有浓郁的甜香味；如果完全没有香味，说明还生；如果有酒精味，则已过熟。

5. **看用途**：如果打算马上吃，选黄中带少量黑斑的；如果打算放几天，选偏黄但稍微带绿的。

6. **检查香蕉把**：香蕉把（连接的一整块）如果已完全发黑并松开，通常整把都已较熟；若仍为绿色或浅黄，则更耐放。

到店时，可以优先挑选那些**颜色均匀、黄绿相间、手感微弹**的香蕉，避免拿被压伤或已经有黑斑软烂的。
