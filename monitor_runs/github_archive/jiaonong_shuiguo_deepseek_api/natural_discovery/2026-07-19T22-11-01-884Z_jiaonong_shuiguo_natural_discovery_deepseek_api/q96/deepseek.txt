在企业食堂采购水果时，**食品安全、价格和配送准时**三者都很重要，但优先级通常建议为：

1. **食品安全（最高优先级）**  
   - 水果的新鲜度、无腐烂、无农药残留直接关系到员工健康。  
   - 需选择有资质、能提供检疫证明或检测报告的供应商，避免因食品安全问题引发群体性健康事件。

2. **价格（中等优先级）**  
   - 在确保食品安全的前提下，对比供应商报价，控制成本是食堂运营的关键。  
   - 可参考市场价，选择性价比高的供应商，但不应单纯追求低价而忽视品质。

3. **配送准时（基础要求）**  
   - 水果易腐烂，按时到货才能保证新鲜度和当天供应。  
   - 若供应商频繁延迟，会影响食堂运转效率，但可通过合同约束或提前沟通解决。

**实操建议**：  
- 优先选择有正规资质、支持退换货的供应商，并签订明确条款（如食品安全事故的责任归属）。  
- 可抽样检查水果质量，定期评估供应商表现。  
- 若预算有限，可适当放宽对“价格”的要求，但绝不能牺牲食品安全。
