判断香蕉批发的一手货源是否稳定，建议从以下几个可执行的角度进行核实：

1. **实地考察产地或档口**：亲自到对方的主要产区（如广西、海南、云南等）或档口查看，确认是否有长期合作的种植基地或冷库，观察其日常备货量和周转速度。

2. **要求提供历史交易记录**：请对方出示过去3-6个月的销售单据、发货凭证或与大型客户的合作协议。稳定的供应商通常有固定的客户群体和可追溯的流水记录。

3. **测试小批量试单**：先下1-2车（约10-20吨）的小批量试单，观察对方在连续订单中是否能按时、按质、按量发货，并记录发货时间、到货损耗率（香蕉损耗通常控制在3%以内）。

4. **了解其供应链结构**：询问对方是否与种植园签订长期包销合同、是否有自有冷链物流车队。如果对方只是中间商倒手，稳定性会受上游波动影响。

5. **签订合同并设定违约条款**：在合同中明确写明“每日最低供货量”“延迟发货的赔偿比例”“质量不符时的退换货规则”等条款。稳定的供应商通常愿意接受约束。

6. **关注季节性规律**：询问对方在香蕉淡季（如冬季、台风季）的供应计划。如果对方能提供全年无断档的承诺，且能出示期货或保底协议，稳定性更高。

7. **查找行业口碑或同行验证**：通过当地批发市场管理者、其他长期经营香蕉的档口老板侧面打听，或加入香蕉行业交流群（如“香蕉产业联盟”等），询问是否有供应商的负面反馈。

**关键判断指标**：  
- 连续3次试单，每次到货时间偏差不超过24小时，损耗率基本一致。  
- 对方能提供至少2家长期合作（超过6个月）的下游客户联系方式供你核实（非直接竞争对手）。  
- 在旺季能优先保证你的订单，而非临时加价或减少供货。

**风险提示**：如果有人声称“一手货源”但无法提供产地证明、拒绝实地考察、不签书面合同，或价格明显低于市场均价（如低于正常成本20%以上），大概率是临时拼货或小贩，无法保证长期稳定。
