对于餐饮店每天少量水果的需求，建议选择以下类型的供应商，能兼顾稳定性和省心程度：

1. **本地社区生鲜平台（如美团优选、多多买菜）**  
   - **优点**：每日可下单，少量起订（通常1斤起），次日统一配送至门店附近提货点，无需额外运输成本。  
   - **操作**：每晚10点前下单，次日早10点前提取，适合固定量小的需求。  
   - **风险**：水果品质不稳定，可能缺货，需提前囤货备用。

2. **小型批发市场或农贸市场的固定档口**  
   - **优点**：可协商每日少量配送，或自己就近快速提货，价格低于零售价。  
   - **操作**：连续固定1-2家档口合作，提前一天告知需求，对方会预留。  
   - **沟通技巧**：强调“每天都要，量不大但稳定”，通常能拿到批发价的8-9折。

3. **本地水果合作社或农场直供**  
   - **优点**：品质可控，可定制每天少量（如切好、搭配好），无中间商。  
   - **操作**：通过当地农业局网站或抖音搜索“同城水果基地”，联系对方谈“试合作”，先自取2-3次，再要求配送。  
   - **省心点**：对方通常愿意按天送，但需付少量配送费（如10-20元/次）。

4. **综合生鲜B2B平台（如美菜、蜀海）**  
   - **优点**：支持很小订单（2-3斤起订），平台统一配送，有售后赔付。  
   - **操作**：注册后设置“每日自动复购”，选择常用水果，系统自动补货。  
   - **适合**：如果每天固定使用（如柠檬、圣女果、苹果），可设周计划。

**最终建议方案（最省心）**：
- **首选**：本地社区生鲜平台（如美团优选） + 每周1-2次顺路去批发市场备货。  
- **风险对冲**：与1位批发市场档口老板加微信，急了可以跑腿叫外卖补充（如美团跑腿代购）。  
- **关键操作**：每天固定时间（如晚8点）查看次日订单，避免遗漏。  

**提醒**：每天少量时，切勿与大型配送商签长期合同，否则可能被要求最低起送量（如500元/次）。先试用1周，观察品质和稳定性后再定方案。
