建议按以下步骤约定补货流程，以最大限度减少对营业的影响：

1. **立即沟通并要求补货凭证**  
   一旦发现到货不合格（如腐烂、损伤、规格不符），立刻拍照/录像留证，并通过书面（微信或邮件）通知供货商，要求其提供“同等品质补货”及具体到货时间。明确要求补货时间不得晚于原订单送达后的**24小时内**（或双方约定的合理时长），否则视为违约。

2. **设定分批次补货或紧急备用方案**  
   与供货商约定：若无法整批补货，**优先补足核心畅销款**（如柠檬、草莓等常用水果），其余品种分批次在2天内补齐。同时，要求供货商提供**应急供应商名单或备选渠道**（例如同一市场内其他认证供应商），一旦补货延误可立即启动备用采购，差额由原供货商承担。

3. **明确赔偿责任与接货时间窗口**  
   在协议中写明：因补货延迟造成的营业额损失（可参照同日平均流水），由供货商按订单金额的**10%-30%**进行补偿（或具体约定）。补货到店后，设定**1小时验收时限**，超时未接货视为自动签收，避免双方扯皮。

4. **建立长期合同条款**  
   在下次签约时加入“到货合格率≥95%”的考核条款，并约定：“不合格到货情况下，首次警告并补货，第二次起每天按订单额5%扣罚，第三次有权终止合作”。这能倒逼供应商重视品质。

**执行要点**：在当天营业前（如上午8点前）完成验收和补货通知，确保下午高峰前到货；若无法当天补足，可暂时调换饮品菜单（如用芒果替代草莓），并提前在吧台或线上告知顾客“因食材调整，暂时更换水果品种”，避免影响顾客体验。
