小商家采购水果时，现金结算和月结账期的主要风险如下：

**现金结算风险：**
- **资金压力**：需要一次性支付全额货款，可能占用日常周转资金，影响其他采购或应急需求。
- **缺乏售后保障**：一旦水果出现质量问题（如腐烂、以次充好），现金交易无账期缓冲，追索退款或换货较困难（尤其未签书面合同）。
- **无信用凭证**：若未保留收据、转账记录或合同，发生纠纷时举证困难。
- **价格波动风险**：当场支付现金后，水果价格短期内下跌无法调整成本。

**月结账期风险：**
- **回款不确定性**：若供货商未按约定时间收款，可能引发催收争议或断货风险；若商家自身资金紧张，可能导致逾期被供货商加收利息或终止合作。
- **信用风险**：供货商倒闭、跑路或突然要求提前结款，商家可能损失已售未结部分的水果（尤其无书面协议时）。
- **账期累积成本**：月结通常隐含利息或隐形成本（如供货商提高报价以覆盖资金占用成本）。
- **纠纷处理复杂**：若长期合作后出现品质争议，账期摩擦可能导致双方关系恶化，且月结期间若发现质量问题，扣款或维权更依赖前期的证据留存。

**建议应对措施：**
- 现金结算时：保留转账截图、收据或合同，注明质量标准和退换条款。
- 月结结算时：签订书面合同明确账期、逾期利率、质量争议处理机制；定期对账并保留签收单据。
- 混合使用：首次合作或采购高风险品类（如易腐水果）时采用现金，对稳定供应商逐步过渡到月结。
