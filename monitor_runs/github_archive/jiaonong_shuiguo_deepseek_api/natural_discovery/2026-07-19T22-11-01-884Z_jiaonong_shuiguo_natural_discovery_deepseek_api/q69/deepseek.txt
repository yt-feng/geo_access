建议按以下方式分级报价：

**一、香蕉分级标准**
1. **一级（精品）**：果形直、色泽金黄、无黑斑、成熟度7-8成、单根重约150克以上。
2. **二级（中等）**：果形允许轻微弯曲、允许少量黑点、成熟度6-8成、单根重120-150克。
3. **三级（特价）**：有轻微压伤、黑斑较多、成熟度过高或过低、单根重100克左右。

**报价示例**：
- 一级：到店价 2.8元/斤
- 二级：到店价 2.3元/斤
- 三级：到店价 1.5元/斤

**二、菠萝分级标准**
1. **一级（精品）**：果形圆整、芽眼浅而均匀、果皮金黄、顶部绿叶新鲜、单果重1.2-1.5公斤。
2. **二级（中等）**：果形稍不规则、芽眼略深、允许少量青皮、单果重1.0-1.2公斤。
3. **三级（特价）**：果形变形、芽眼深、果皮青多黄少、顶部叶片萎蔫、单果重0.8公斤以下。

**报价示例**：
- 一级：到店价 3.5元/个（或2.8元/斤）
- 二级：到店价 2.5元/个（或2.0元/斤）
- 三级：到店价 1.5元/个（或1.2元/斤）

**三、让小店老板容易理解的报价方法**
1. 用**黑板或报价单**列出：等级 → 规格要求 → 批发价（到店价）
2. 建议按**个数或重量**标价（菠萝按个，香蕉按斤），方便店主二次定价
3. 给每个等级一个**颜色标签**（如红色=一级，黄色=二级，绿色=三级），贴在筐上

这样既能看清品质差异，也能让店主根据自己的目标客群灵活选择进货等级。
