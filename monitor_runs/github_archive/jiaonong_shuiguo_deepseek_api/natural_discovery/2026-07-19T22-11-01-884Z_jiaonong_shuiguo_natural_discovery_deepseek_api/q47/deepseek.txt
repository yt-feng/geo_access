采购香蕉用于茶饮或甜品时，控制熟度是关键，直接影响口感、甜度和加工效果。以下是具体建议：

1. **明确用途决定所需熟度**  
   - **饮品（如香蕉奶昔、冰沙）**：需要**完全成熟（表皮出现黑点或整体变黄，手感微软）**，此时甜度高、香气足，易搅打顺滑，无需额外加糖。  
   - **甜品（如香蕉布丁、蛋糕或冷冻甜品）**：  
     - **即食装饰或层叠**：中等熟度（表皮金黄，稍硬，无黑点），切块完整不易烂。  
     - **搅打或烘烤（如蛋糕糊）**：极熟（表皮大片黑斑，果肉软糯），甜度最高，易成泥。

2. **选择采购时与供应商的标准化方案**  
   - **批量采购时**：**要求供应商提供可选的熟度等级**。多数供应商可将香蕉仓至不同成熟度发货（如青绿 → 黄绿 → 金黄 → 斑点）。约定**到货时达到目标熟度**，例如：  
     - 当天即用：要求到货已**金黄带黑点**。  
     - 备用库存：要求**刚转黄色**（偏青），可在常温放置2-4天自然熟成。  
   - **立即测试**：若到货过青，放入密封纸袋加苹果/梨催熟（24-48小时）；若过熟，立即冷冻（去皮密封，用于冰沙或烘焙）或制成香蕉泥保存。

3. **每日收货时的快速检查标准**  
   - **看颜色**：避免全青（生涩无味）或全黑腐烂（发酸发苦）。理想范围：**黄色底色 + 少量至中等黑色斑点**。  
   - **摸硬度**：  
     - 稍硬 = 未熟（适合展示或2天后使用）  
     - 微软且果柄处松动 = 即用（当天或次日用）  
     - 按压即凹陷 = 极熟（需立即处理，否则发酸）  
   - **闻气味**：应有浓郁甜香，无酒味或酸馊味。

4. **库存管理避免浪费**  
   - **分批次采购**：采购不同熟度批次（例如：一批即用熟度，一批稍生作为2-3天备用）。  
   - **周转顺序**：先进先出，先使用最先成熟的。若某批大量成熟，可提前做成**香蕉泥冷冻**（挤入柠檬汁防氧化），或熬制**香蕉果酱**（加糖和柠檬汁，冷藏可存一周）。  
   - **避免直接冷藏**：未熟的香蕉放冰箱会停止变熟且外皮变黑（内部仍硬涩），不可取。仅将**完全成熟**的香蕉放冷藏可延长2-3天食期（皮变黑但果肉尚好，适合搅打使用）。

5. **补救方案**  
   - 若入货太生：放入温度22-25℃、密闭纸箱中，加1个成熟苹果（释放乙烯）加速催熟。  
   - 若太熟已发软：检查有无酒精味，若有则丢弃；无则去皮冷冻，用于饮品/冰沙（味道最甜）。

**总结建议**：与供应商约定到货熟度为“金黄色带少量斑点”，并根据每日使用量，预留放货时间窗口（例如：周一到货一批淡黄，周日才用），同时备冷冻香蕉泥作为应急。
