针对连锁门店分布在不同城市的场景，评估水果供应商的仓配覆盖能力，建议从以下四个维度入手，并结合量化指标进行判断：

### 1. 配送网络节点覆盖
- **核心指标**：供应商在各城市或周边是否有自建/合作的**前置仓、区域分拨中心**。
- **评估方法**：
  - 列出所有门店所在城市，要求供应商提供其对应的**最近发货仓**位置及距离。
  - 优先选择“仓库与门店同城”或“车程在2小时内”的供应商（水果易损耗，时效要求高）。
  - 注意事项：如果是跨区域供应，需确认中转仓的冷链配置是否齐全。

### 2. 配送时效与频率
- **核心指标**：承诺的**订单截止时间**和**次日达/当日达比例**。
- **评估方法**：
  - 要求供应商提供分城市、分门店的**最晚配送时间表**（例如：上海门店17:00前下单，次日早8:00到达）。
  - 测试其**补货周期**：小型连锁能否做到每日配送？超大型门店能否支持一周两配或全天多频次补货？
  - 询问**紧急补货的响应时间**（如突发促销或断货时）。

### 3. 冷链与品控能力
- **核心指标**：全程冷链覆盖率、损耗率数据、供应商的***分拣与保鲜设备**。
- **评估方法**：
  - 要求提供其**冷链运输车辆数量**及覆盖城市，重点关注“最后一公里”是否有冷藏配送车。
  - 随机抽查过往同类水果（如草莓、荔枝）的**平均到货损耗率**（建议低于3%）及**客诉退换率**。
  - 实地考察仓库：是否配备预冷、控温、气调保鲜等设备。

### 4. 多城协调与应急能力
- **核心指标**：门店分布越分散，越需要评估供应商的**跨城调拨能力**和**淡季/旺季灵活性**。
- **评估方法**：
  - 要求供应商提供一份覆盖其所有仓库、配送半径的**城市清单**，并标注哪些城市可以“直发”，哪些需要“中转”。
  - 测试其**分拨效率**：如杭州仓缺货时，能否从上海仓在 2 小时内调配补货？
  - 考察其**淡旺季预案**：夏季水果需求激增时，能否临时增加配送频次或增设临时冷藏点。

### 快速评估清单（可直接向供应商提问）
1. **“请列明每家门店最近的一个仓库，并说明从该仓库到门店的运输时间（非距离）。”**
2. **“你们可以提供一份过去3个月该门店城市的水果平均配送准时率及损耗率吗？”**
3. **“如果某家门店突发断货，最快要多久能补货到位？补货路径是什么？”**
4. **“你们在不同城市间的库存是否能快速调拨？调拨需要额外费用吗？”**

### 辅助判断工具
- **试用期测试**：先选3-5家不同区域（如一线城市、二三线城市、偏远城市）的门店试配送1-2周，记录以下数据：
  - 到货时间误差（±0.5小时内为合格）
  - 水果新鲜度（如外观、软硬度、汁水流失率）
  - 碎果/烂果比例（超过5%则严重不合格）
- **对比供应商报价**：如果某供应商报价显著低于同行，需警惕其仓配网络可能依托外包，稳定性存疑（如临时租用物流车导致错发）。

### 关键风险提示
- **慎选“全网通吃”的供应商**：部分供应商宣传“全国配送”，实际只有1-2个核心仓，依赖快递/零担中转，不适合生鲜门店。
- **关注“最后一公里”的冷链**：很多供应商在干路上有冷链车，但末端使用常温电动车配送（如面包车），这类情况需重点要求贴标或增加冰袋。
- **签署服务协议**：明确写入“各城市具体配送时效”“损耗赔付标准”（如超时补损、烂果按倍赔）。

最后，建议优先选择**门店所在城市有履约记录、且能提供近3个月各城市配送数据**的供应商。如果某城市门店量少（如仅1-2家），可要求供应商提供**拼单方案**或评估本地小型供应商的分区补货能力。
