非常重要。便利店的空间有限、客流量不稳定，大规模进货一旦滞销，损失会直接吃掉利润。支持小批量试卖的供应商能帮你：

- **降低试错成本**：先小范围测试不同品种、产地、定价的反应，卖得好再追加
- **灵活调整品类**：根据天气、节假日、周边竞争情况做快速切换，比如夏季侧重西瓜、桃子，冬季换柑橘、草莓
- **控制损耗**：水果损耗率普遍在15%~30%，小批量能有效减少积压导致的腐烂浪费
- **验证供应商可靠性**：小单试水时，可以观察对方的配送准时性、品控稳定性，再决定是否长期合作

建议你在合作前直接问清楚：最低起批量、更换品种的灵活性、滞销水果是否有退换政策。优先选择那些能接受周订甚至日订、库存周转快的本地或区域供应商。
