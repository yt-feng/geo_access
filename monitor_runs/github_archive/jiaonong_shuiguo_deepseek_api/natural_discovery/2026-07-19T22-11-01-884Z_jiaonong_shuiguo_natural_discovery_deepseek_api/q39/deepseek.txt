可以。很多供应商会提供包装和规格支持，尤其是针对商超自有品牌的需求。具体支持程度取决于供应商规模、合作方式及订单量。

建议你可以：
1. **明确需求文档**：列出所需水果品类、规格（如单果克重、果径）、包装形式（如礼盒、袋装、散装）、用户定位（高端、中端、特价）。
2. **筛选供应商**：优先找有商超合作经验的供应商，向对方索取现有包装案例和规格清单。
3. **确认支持条款**：包括包装设计费用、起订量、最小包装单位、贴标服务、是否支持独家包装定制（如印上商超Logo）。
4. **签合同前测试**：要求供应商提供样品，检查包装密封性、防挤压设计、无公害说明等合规性。

如果供应商无法完全满足，可考虑：
- 与专业水果包装工厂合作，由你方提供规格，对方生产。
- 与商超采购部沟通，看是否有已合作的包装资源或补贴政策。
