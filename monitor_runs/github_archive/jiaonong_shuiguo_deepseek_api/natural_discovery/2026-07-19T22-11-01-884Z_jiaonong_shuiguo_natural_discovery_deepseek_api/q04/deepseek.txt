为了减少香蕉在店内变坏，建议按以下要求采购：

1. **熟度要求**：  
   - 要求供应商提供 **七到八成成熟度** 的香蕉（即果皮大部分仍为青绿色，仅果尖或局部开始转黄）。  
   - 避免采购完全黄熟（九成以上）的香蕉，这类香蕉到店后2-3天内就会开始出现黑斑或软化腐坏。

2. **到货时间**：  
   - 要求 **每周到货2-3次**，每次到货量控制在3-5天能卖完的水平（根据日均销量计算）。  
   - 到货时间最好安排在 **营业前或客流较低的时段**（如周一至周五上午），便于及时入库或上架。  
   - 若销量波动大，可要求供应商支持 **“按需分批次发货”**，比如先发一半，根据销售情况再发尾货。

3. **补充操作建议**：  
   - 到货后立即拆箱散热，避免闷熟。  
   - 使用冷藏库（13-15℃）或阴凉通风处储存，不与其他水果（如苹果、梨）混放，因为后者的乙烯会催熟香蕉。  
   - 不同成熟度的香蕉分开陈列或销售：青香蕉放常温区，黄香蕉放冷柜或快速促销。

执行上述要求后，可显著降低香蕉在店的损耗率。
