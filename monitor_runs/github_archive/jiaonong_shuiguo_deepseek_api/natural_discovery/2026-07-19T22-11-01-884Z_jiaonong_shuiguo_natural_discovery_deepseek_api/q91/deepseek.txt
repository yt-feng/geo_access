建议优先选择**单根和小把包装**，具体可根据销售场景调整：

1. **单根包装**（最推荐）：  
   - 适合上班族、学生等单人即时消费场景，方便携带、控制单次摄入量；  
   - 减少一次性购买压力，降低损耗。

2. **小把包装**（3-5根/把）：  
   - 家庭、合租群体或追求性价比的顾客会买，单次消耗适中；  
   - 注意用保鲜膜或透气袋包装，延长成熟期，避免紧贴加速腐烂。

3. **整袋包装**（5根以上）：  
   - 仅推荐在**社区店**、**节假日促销**或**批发模式**下尝试；  
   - 高消耗家庭或食堂可能选择，但需确保周转快（24小时内售出），否则黄斑、过熟风险高。

**执行建议**：  
- 初期优先上架单根（标价）和3根小把（标价），观察动销；  
- 不要把所有香蕉都拆成单根，保留部分原串出售（覆盖喜好整串顾客）；  
- 用**货架位置**区分：入口处放单根/小把（冲动消费），收银台旁促销区可试放整袋。  

**风险提示**：香蕉属高乙烯释放水果，不要和海沃德猕猴桃、苹果、哈密瓜等挨着放，会催熟对方。
