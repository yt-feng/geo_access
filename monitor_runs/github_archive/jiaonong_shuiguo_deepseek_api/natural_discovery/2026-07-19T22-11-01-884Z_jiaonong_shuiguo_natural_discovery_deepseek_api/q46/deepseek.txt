建议优先采购**冷冻菠萝丁**或**冷冻菠萝泥**（视具体用途而定），原因如下：

1. **新鲜菠萝**：风味最佳，但需人工削皮、去芯、切丁，耗时耗力；且成熟度不均，容易出现部分过酸或过生，影响成品稳定性；还需冷藏保存，损耗较高。适合门店用量大、有专业后厨团队且能保证快速消耗的情况。
2. **冷冻菠萝丁**：工厂已去皮切块，大小均匀，免切；解冻即可使用，损耗低；口感接近鲜果，但建议用于**果茶中加料**（如嚼得到果肉），或搭配其他果汁调味。
3. **冷冻菠萝泥**：质地细腻，无纤维，能完全融入茶汤，风味稳定，方便按标准量出；适合做**清爽型果茶**或**基底调味**，但对需要果肉咀嚼感的饮品效果不佳。

**综合建议**：
- 如果要做**含果肉颗粒的爆款果茶**，选冷冻菠萝丁（建议搭配柠檬汁或苹果醋平衡酸甜）。
- 如果要做**顺滑无渣的高颜值茶饮**，选冷冻菠萝泥（还可混合芒果泥或百香果提升层次）。
- 尽量避免鲜果，除非你门店日销量极大且能快速处理。

**采购注意**：确认供应商是否为“速冻单冻”工艺，避免粘连成块；开袋后密封冷藏，建议48小时内用完。
