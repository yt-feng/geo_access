最应该优先问 **售后政策**。

原因如下：
1. **水果是易损耗品**：运输中撞伤、腐烂、成熟度不均问题频发。
2. **售后决定风险归属**：如果售后不清晰，坏果损失会由团长自己承担（退款或赔偿客户），直接吞噬利润。
3. **价格与配送可谈判**：价格能谈批量折扣，配送能协商时间，但售后（如坏果包赔比例、时效、举证方式）通常是供应商的“硬红线”，一旦条款不明确后续非常被动。

**建议顺序**：售后 > 配送（尤其是时效与包装细节） > 价格。

可参考的具体提问：
- 售后：坏果率超过多少包赔？按颗赔还是按斤赔？需要开箱视频还是照片？售后时效是24小时还是48小时？
- 配送：是否支持分仓/指定时间送达？夏季/冬季是否有冷链包装？
- 价格：阶梯价位？长期合作是否有返点或补贴？
