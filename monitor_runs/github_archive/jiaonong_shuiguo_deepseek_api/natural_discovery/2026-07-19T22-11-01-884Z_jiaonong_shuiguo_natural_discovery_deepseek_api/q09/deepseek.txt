在水果店旺季与供应商谈合作时，建议提前询问以下关键问题，以降低断货风险、确保品质和利润：

1. **供应稳定性**  
   - 旺季期间，每天/每周最大供应量是多少？  
   - 是否有备用货源（如多个产地）？如果主产地受灾或断供，如何保障发货？

2. **价格与条款**  
   - 旺季价格是否会随行就市波动？能否锁定阶梯价格（如采购量大给予折扣）？  
   - 付款周期是现结、周结还是月结？是否有押金或预付款要求？

3. **品质与验收标准**  
   - 能否提供果品分级标准（如大小、糖度、外观瑕疵比例）？到货后如何抽检？  
   - 如果出现烂果、坏果，如何赔付（按比例退款、补发还是下次抵扣）？

4. **物流与时效**  
   - 从下单到送达需要多久？是否支持加急补货（如突发断货）？  
   - 运输方式是什么？冷链车是否全程温控？到货后是否有冷库暂存服务？

5. **应急与退货机制**  
   - 如果到货品质不符，退货流程和赔偿时间表是什么？  
   - 旺季是否接受部分产品换购（如某种水果滞销，能否换成当季畅销品）？

6. **季节性支持**  
   - 供应商能否提供促销方案（如联合推广、买家秀素材、试吃装）？  
   - 旺季结束后，剩余库存是否可以退货或折价代销？

7. **合作历史与客户评价**  
   - 能否提供过去两年旺季的供货案例？是否有稳定大客户（如连锁超市）的推荐？  
   - 是否有因断货或品质问题被投诉的记录？

**附加建议**：  
- 先在淡季小批量试合作，观察响应速度和品质把控能力。  
- 要求供应商提供产地证明、农残检测报告等资质文件。  
- 在合同中明确“优先供应权”条款，避免热门水果被高价截留。
