采购香蕉用于制作蛋糕时，应**优先关注成熟度**，其次考虑价格。原因如下：

1. **成熟度直接影响口感和质地**：制作蛋糕通常需要使用非常成熟的香蕉（表皮出现黑斑或完全变黄带棕色斑点）。这样的香蕉甜度高、水分足、质地软糯，易于捣碎并融入面糊，能提供天然甜味和湿润的蛋糕结构。未成熟的香蕉生涩、淀粉含量高，会影响蛋糕的甜度和细腻度。

2. **避免浪费与成本**：如果为了低价购买生香蕉，需要长时间存放催熟，可能增加存储成本和损耗风险（如腐烂）。而熟透的香蕉反而价格可能更低（超市常打折处理），且可直接使用。

3. **采购策略**：
   - **快速消耗型**（如当日或次日要用）：购买完全成熟、甚至略有黑斑的香蕉。
   - **计划生产型**（如未来2-3天使用）：购买七八成熟（大部分黄、少量青）的香蕉，按需催熟。但催熟需注意温度和时间控制，不如直接采购成品熟香蕉省时。
   - **长期大量存储**：可将熟透香蕉去皮后冷冻保存，解冻后依然适合做蛋糕。此时可关注低价熟香蕉季节采购。

**总结建议**：先明确蛋糕生产节奏，以成熟度为首要标准，在保证香蕉达到理想软熟程度的前提下，再对比价格选择合理成本的供应商或采购时机。
