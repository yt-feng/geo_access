建议你和供应商按以下步骤沟通和协商，确保补货顺畅且风险可控：

1. **立即沟通**：第一时间联系供应商，说明“旺季销售激增，菠萝库存不足，需要紧急补货”，明确告知需要的**数量**（如“本周额外需要200箱”）和**到货时间要求**（例如“最迟本周五前到货”）。

2. **确认库存与产能**：询问供应商现有库存和近期能提供的最大供应量。如果对方库存不足，可探讨是否可以**临时调配其他渠道货源**，或**协商分批次到货**。

3. **商定临时条款**：
   - **价格**：说明这是临时增量，询问是否维持原价（如因采购成本上升需加价，可协商一个合理的临时涨幅，如5%-8%）。
   - **付款**：临时补货通常需预付或缩短账期（例如“本批次到货后3天内结算”），避免供应商因资金问题延迟发货。
   - **质量与验收**：强调“临时补货需保证与常规批次同等质量”，可约定到货后按样品抽检，如不合格可退货或要求补发。
   - **违约责任**：明确“如因供应商原因延迟到货，导致门店断货损失，需协商补偿”，但语气宜为合作性而非惩罚性。

4. **签订书面确认**：通过微信/邮件发送《临时补货确认函》，写清楚：补货品种（菠萝）、数量、单价、总金额、到货时间、验收标准、付款条件。双方确认后截图留存。

5. **备选方案**：如果供应商无法满足，同时联系1-2家备用供应商（如水果批发市场、邻市供应商），说明“紧急试单，可长期合作”，尽快签订短期采购协议。

**沟通话术参考**：“张经理，最近客流比预计多了30%，菠萝已经断货了。我们需要紧急加订200箱，最晚周五到货，价格能按原合同执行吗？如果不行，咱们协商一个临时价，但质量不能降。我会先预付50%款，到货验收后再付尾款。您看是否可行？”

**关键提醒**：临时补货不要马上承诺与供应商签订长期合同，可先按“单批次”约定，避免未来淡季时被要求按高峰量采购。
