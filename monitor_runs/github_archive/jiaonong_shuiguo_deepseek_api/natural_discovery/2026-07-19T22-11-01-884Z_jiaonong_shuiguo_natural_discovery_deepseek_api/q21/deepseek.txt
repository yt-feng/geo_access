在超市采购水果时，为了确保食品安全和可追溯性，供应商通常需要提供以下检测报告或进货证明：

1. **产地证明或种植基地证明**：证明水果来源的合法性和种植环境的基本信息。

2. **农药残留检测报告**：根据国家或地方标准，提供第三方或自检的农药残留合格报告。

3. **重金属及污染物检测报告**（如铅、镉、砷等）：特别是针对土壤或水源可能有污染风险的区域。

4. **产品合格证或出厂检验报告**：由供应商或生产厂家出具，证明产品符合质量标准。

5. **检疫证明或植物检疫证书**：如果是跨境或跨省（市）运输的水果，需提供检疫部门出具的无病虫害证明。

6. **进货票据或采购合同**：包括供货方名称、品名、数量、日期等信息，作为进货凭证。

7. **特殊认证证书**（如有机食品认证、绿色食品认证、GAP良好农业规范认证等）：如有相关认证，需提供证书复印件。

8. **批次追溯信息**：如生产日期、采摘日期、保质期、批次号等。

9. **运输温度记录**（针对需要冷链的水果）：证明运输过程中温度控制符合要求。

10. **进口水果的报关单、入境货物检验检疫证明**（如适用）。

建议在采购前与供应商明确要求，并保留所有文件复印件或电子版，以备市场监管部门检查。具体报告类型可能因当地法规或超市内部品控标准而有所差异。
