酒店采购水果盘原料时，供应商应保证以下规格和外观标准：

**规格标准：**
1. **水果种类与大小**：明确列出所需水果种类（如西瓜、哈密瓜、葡萄等），并要求单个水果的重量、直径或长度符合统一范围（例如苹果直径80-85mm）。
2. **成熟度**：供应商需确保水果成熟度适中，不宜过生或过熟，避免影响口感和保质期。
3. **净含量**：每份水果盘的净重或单个水果数量应满足合同约定（如每盘净重500g±5%）。
4. **包装规格**：水果需按酒店要求进行分装或独立包装，防止碰撞损伤，且包装材料需符合食品安全标准（如食品级塑料袋或保鲜膜）。

**外观标准：**
1. **新鲜度**：无斑点、霉变、腐烂或软烂现象，表面无明显机械损伤（如划痕、压痕）。
2. **色泽**：颜色自然均匀，无异常变色（如氧化褐变或药渍痕迹）。
3. **清洁度**：无泥土、虫卵、农药残留或明显灰尘，清洗后表面干燥无水渍。
4. **完整性**：水果无裂口、变形或畸形，果柄需修剪平整（如葡萄枝保留不超过2cm）。
5. **统一性**：同一批次的果型、颜色、大小尽量一致，无明显视觉差异（如部分发青部分过熟）。

**额外建议：**
- 要求供应商提供每批次的《水果质量检测报告》（如农残、重金属检测）。
- 对于切好的水果盘（如拼盘），需保证切面平滑、无组织液渗出过度，且冷藏运输（0-4℃）。
- 协商明确的退换货标准（如外观不合格率超过5%需整单更换）。
