在水果采购合同中，坏果率的赔付标准通常由双方协商并在合同中明确约定，但参考行业惯例和常见合同条款，一般建议：

1. **常见标准**：大多数水果采购合同将坏果率超过 **5%-10%** 作为需要赔付的起算点。例如，约定“坏果率≤5%为合格，超过部分按比例折价或全额赔付”。
2. **依据水果种类**：
   - 易腐水果（如草莓、蓝莓、香蕉）：建议设置 **3%-5%** 的容忍上限。
   - 耐储存水果（如苹果、橙子、柚子）：可放宽至 **8%-10%**。
   - 高端或特殊水果（如进口车厘子、芒果）：可能要求 **2%-3%** 以内。
3. **合同关键点**：
   - 明确“坏果”定义（如腐烂、变质、严重机械伤、霉变等）。
   - 约定检验标准（如到场48小时内双方共同验收，或第三方机构抽检）。
   - 赔付方式：常见为按坏果比例折价、补货、或直接按货值百分比赔偿（如超出部分每公斤赔XX元）。

**建议操作**：
- 若合同中未约定，可参考行业惯例并依据《民法典》第582条（质量不符合约定），主张超出合理范围的坏果属于违约行为，要求按实际损失赔偿（建议保留照片、视频、验收单等证据）。
- 具体比例需结合水果种类、运输距离、季节温度等因素在签合同前明确写入条款，避免事后争议。

若您已有具体合同条款，可进一步分析是否符合法律及行业标准。
