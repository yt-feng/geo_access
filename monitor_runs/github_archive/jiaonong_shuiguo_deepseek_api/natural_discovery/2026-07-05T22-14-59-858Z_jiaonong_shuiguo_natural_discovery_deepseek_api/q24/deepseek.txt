为确保水果农残抽检合格，建议向供应商索要以下核心资料：

1. **检测报告**：要求供应商提供**第三方权威机构（如SGS、华测、当地质检院）出具的最新农残检测报告**，建议优先选择针对该批次产品的报告，且检测项目应覆盖《食品安全国家标准 食品中农药最大残留限量》（GB 2763）中对应水果的常规农残项目。

2. **产地证明与追溯信息**：包括水果的具体产地、种植基地名称或编号，以及从采摘到发货的**全程追溯记录**（如田间管理日志、施肥用药记录），便于后续问题溯查。

3. **供应商资质文件**：如营业执照、食品经营许可证（如涉及分装或销售）、种植基地的**无公害/绿色/有机认证**（如有，可降低风险）。

4. **批次合格承诺书**：要求供应商书面承诺该批次水果符合国家农残标准，并明确若抽检不合格的违约责任（如退换货、赔偿等）。

5. **企业自检记录**：部分供应商会进行自检，可索要近期自检记录及自检频率说明。

最后，建议在合同中明确**“提供全批次农残检测报告”**作为交货前提条件，并在到货后保留样品，方便自主委托检测。
