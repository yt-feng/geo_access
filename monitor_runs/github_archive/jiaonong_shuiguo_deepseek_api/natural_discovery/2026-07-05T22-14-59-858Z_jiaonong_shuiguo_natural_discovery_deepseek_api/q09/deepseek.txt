在水果店旺季与供应商合作时，为避免断货风险，建议提前确认以下关键问题：

1. **货源稳定性**  
   - 旺季期间是否保证足量供应？若遇极端天气或产量波动，有无备选货源？  
   - 是否有长期稳定合作的种植基地或一级批发渠道？  

2. **供应周期与交付**  
   - 下单后最快多久发货？配送频次（如每日/隔日）能否配合门店销售节奏？  
   - 配送时效是否覆盖旺季高峰期（如节假日、促销日）？  

3. **品质保障机制**  
   - 如何确保到货水果的新鲜度（如冷链运输、包装方式）？  
   - 出现品质问题（如腐烂、规格不符）的退换货或赔偿标准是什么？  

4. **价格与结算条款**  
   - 旺季价格是否随市场浮动？是否有最低供应量或阶梯定价（如量大优惠）？  
   - 结算周期（如周结/月结）和付款方式是否灵活？  

5. **应急与风险应对**  
   - 若临时加单或紧急补货，能否快速响应？额外费用如何计算？  
   - 是否提供滞销果品的退货或调换服务（避免库存积压）？  

6. **合作案例与历史数据**  
   - 能否提供上一年同期高峰期的供货记录？是否有同类店铺合作经验？  
   - 是否熟悉本地市场偏好（如哪些品种在旺季更畅销）？  

**建议**：  
- 优先选择“多基地+多品种”的供应商，分散断货风险。  
- 合同中明确旺季最低供货量、品质罚款条款及突发缺货时的赔偿方案。  
- 试合作1-2周，观察其实际履约能力，再签订长期协议。
