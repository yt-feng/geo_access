要判断茶饮连锁的菠萝供应商是否能稳定按规格送货，可以从以下几个方面进行评估和验证：

1. **查看供应商资质与历史记录**：要求供应商提供营业执照、食品经营许可证、产地证明等文件，并查询其过往的合作记录（尤其是同类型连锁客户的合作时长、续约率）。可要求提供近半年或一年的出货记录，对比规格达标率（如重量、成熟度、切割标准等）。

2. **签订明确合同与验收标准**：在合同中详细列明菠萝的规格要求（例如：单果重量400-500g、甜度14-18度、无机械伤、成熟度七至八成、切割后误差不超过±5%等），并约定不合格品的处理方式（如退货、换货、折价）。同时明确违约责任（如延迟或规格不符的赔偿条款）。

3. **执行小批量试单**：先与供应商合作1-2个月的小批量订单（如每周500公斤），要求每次送货提供规格检测报告（如糖度计测量结果、称重记录）。同时安排专人随机抽检（抽检比例不低于20%），记录合格率。若连续3次合格率低于95%，则重新评估其能力。

4. **考察供应链与产地源头**：实地走访种植基地或加工分选中心，确认其是否具备分级筛选设备（如称重台、糖度仪、视觉筛检仪等）、冷链运输能力（如冷藏车温度控制在8-12℃）。了解其产地是否可稳定供应全年（如使用多产区轮换：海南、广东、广西及海外补充），避免断供风险。

5. **建立动态评估机制**：要求供应商每月提供规格符合率统计表，并与自己的验收记录比对。设置关键指标（如准时到货率≥98%、规格合格率≥95%），低于标准的给予整改期，连续不达标则启动替补供应商。

6. **风险评估与备选方案**：提前储备1-2家备用供应商（可测试其样品和配送频率），确保在主供应商出现问题时能快速切换。同时与主供应商约定应急协议（如提前48小时通知可加急补货）。

**直接可操作的建议**：要求供应商提供一份**连续6周的每周规格合格率报告**，并附上随机批次照片或视频；同时自己派人在收货时进行**3轮盲测抽检**（每轮随机抽取50个菠萝），若合格率均>95%，则证明其具备稳定能力。
