社区超市控制水果新鲜度和库存，核心原则是 **“少量多批、动态定价、快进快出”**。以下是具体可落地的操作建议：

### 1. 进货端：源头控制和数据驱动
- **优先本地和当季水果：** 本地水果运输时间短，成熟度高，更容易控制新鲜度。当季水果价格低、需求大，流转快。
- **按历史销售数据订货：** 记录过去1-2周每天每种水果的销量，尤其是周末和节假日。初期可以保守，按照**日均销量的1.2倍**订货，防止压货。
- **分批次采购：** 不要一周进一次货。改为**每周2-3次**小批量进货（如周一、周三、周五），确保货架上的水果始终新鲜。
- **合作与承诺：** 和供应商约定：只进**七八成熟**的水果（可自行催熟或顾客购买后在家放熟），并要求供应商提供**无条件退换**次果的服务。

### 2. 店内管理：陈列、保鲜与分拣
- **“先进先出”陈列法：** 新到的货放在旧货下面或后面，旧货放在最显眼、伸手可及的位置。使用**标签标注入库日期**（或批次号）。
- **分类温差管理：**
  - **冷藏区**：草莓、蓝莓、葡萄、樱桃等浆果类，温度控制在0-4℃。
  - **常温区**：香蕉、芒果、木瓜、哈密瓜（未切开）等热带水果，不用进冷柜，但避开阳光直射。
  - **冷藏后回温**：苹果、梨、橙子等可以暂放冷库，但上架前需要在常温回温1-2小时，避免消费者摸到冰手。
- **每日三次理货：** 早、中、晚各一次挑出有碰伤、腐烂的水果，即时打折处理（如“今晚8点后5折”）。
- **“眼见为鲜”陈列：** 水果堆头上可以放一个**切开的样品**（如西瓜、菠萝），吸引顾客购买，也能快速判断内部质量。

### 3. 销售与促销：动态清库存
- **立即打折，不拖延：** 一旦发现某款水果开始出现1-2个斑点或发软，立刻**降价20%** 处理；如果当天未售完，次日再降30%做成“超值盲盒”或组合装。
- **组合销售：** 将销量慢的（如释迦、莲雾）和新到的热门水果（如草莓）组合成“尝鲜礼盒”或“每日水果拼盘”。
- **分时段促销：** 下午4-5点开始针对**易损耗品类**（如切好的菠萝、木瓜、西瓜）做现切果盒，标价按“买一送一”或“晚上8点后7折”来快速出清。
- **预售或小包装：** 对易损耗的水果（如小番茄、蓝莓等），提供**小份包装（200克或300克）**，降低顾客试错成本。

### 4. 库存周转监控：设立警戒线
- **设定安全周转天数：** 所有水果的**平均库存天数**不要超过4天。比如，某水果进了10箱，预计4天售完，那如果过了2天还剩下6箱，就是“警戒信号”，需要立即促销。
- **“零库存”意识：** 每天晚上闭店前，**剩货＜10%的品种**视为健康；剩货超过20%的品种，第二天必须打折或做特价活动。
- **使用库存周转表：** 简单用Excel或手写记录：日期、进货量、销量、剩下量、报损量。重点监控**“损耗率”**（每批烂掉的比例），如果超过5%就要调整进货或存储方式。

### 5. 特殊技巧
- **冷冻+深加工：** 非常容易坏的水果（如熟过头的香蕉、草莓、芒果），可以**去皮切块冻起来**，做成“冷冻水果包”销售（适合于打奶昔或做甜品），变废为宝。
- **顾客互动：** 在群里或店门口发“今日特价”和“明天到货预告”，让顾客有心理预期和抢购动力。

**一句话总结：** 每天只进“够卖2天的量”，上午主推精品，下午主推折扣，晚上8点前把当天的积压货卖完或做成果切。这样既能保持新鲜感，又不会积压库存。
