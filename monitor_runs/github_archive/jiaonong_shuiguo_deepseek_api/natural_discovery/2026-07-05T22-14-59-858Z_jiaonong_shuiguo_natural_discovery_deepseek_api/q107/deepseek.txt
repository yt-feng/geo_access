是的，采购香蕉和菠萝时，起订量过高确实会增加损耗风险。

具体原因如下：
1. **易腐性**：香蕉和菠萝都是高呼吸作用的水果，采摘后仍会持续成熟。香蕉尤其对乙烯敏感，成熟后极易变质，起订量过高容易导致库存积压，在保质期内无法销售完毕。
2. **储存条件要求高**：菠萝需在7-13℃冷藏，香蕉不宜低于12℃，且需避免与乙烯释放水果共存。若仓库空间或温控能力不足，高起订量会加大储存难度，加速腐烂。
3. **市场需求波动**：这两类水果消费受季节、节假日、价格影响较大。起订量过高可能使滞销风险放大，特别是小规模商户或特定渠道（如社区团购、餐饮）。

**建议操作**：
- 评估实际销售数据（如日均销量、周转天数），将起订量控制在安全库存的1.5倍以内。
- 与供应商协商分批到货或灵活起订量条款，例如允许按周分批发货。
- 若必须高起订量，考虑提前加工或促销（如切块销售、制作果汁），缩短库存时间。
- 优先选择成熟度可调节的香蕉（如七分熟）和耐储菠萝（如带冠芽、未完全黄透的），延长售卖期。

总结：高起订量在缺乏精准需求预测和良好仓储条件下，会显著加大损耗。建议根据自身供应链能力谨慎设定。
