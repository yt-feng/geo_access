针对单位采购水果到货品质不一致的情况，建议按以下步骤进行验收和售后处理：

### 一、验收阶段（关键环节）
1. **立即开箱检查**  
   到货后，在送货人员在场的情况下，随机抽取至少20%的包装开箱，检查水果的**大小、颜色、成熟度、有无损伤/腐烂**等。拍照/录像留存证据。

2. **使用标准比对**  
   对照采购合同或商品详情中约定的**规格参数**（如“单果重200-250g”“无机械伤”等），记录不合格水果的比例。对明显不符合标准的批次，当场拒收并注明原因。

3. **称重与数量核对**  
   对整批水果称重，确认毛重/净重是否达标。若合同约定“净重”，需扣除包装箱、填充物等皮重。

4. **签署临时接收单**  
   若部分不合格，可签署“临时接收单”，注明“已发现X%品质问题，待处理售后”，避免后续争议。

### 二、售后处理（分情况操作）
- **情况1：轻微差异（如10%以内的个头稍小、轻微擦伤）**  
  拍照反馈给供应商，要求**按比例折价**（例如：不合格部分按单价的50%扣款）或**下次补发等量合格水果**。

- **情况2：严重质量问题（如腐烂、发霉、变质超过30%）**  
  立即停止使用，保留完整包装和样品。通过书面方式（邮件/平台客服）要求**整批退货退款**，并承担运费。若涉及食品安全，需保留证据并向监管部门报告。

- **情况3：批次混入异物或变质**  
  立即封存该批次，拍照并书面通知供应商。要求赔偿直接损失（如已支付货款）及间接损失（如员工误食需就医的费用）。

### 三、规避后续风险
- **采购合同中明确条款**：约定验收标准（如“单果重量浮动≤5%”“无腐烂”“24小时内验收”）、售后时效（例如“到货48小时内提出异议”）及赔付方式（退货/补发/折扣）。
- **建立验收台账**：记录每次采购的批次号、供应商、验收照片、异常比例及处理结果，便于后续追溯。
- **选择可靠供应商**：优先采购有“有机认证”“绿色食品”标志或长期合作的水果商，并要求提供产地证明和检测报告。

建议在验收时使用**“标准比色卡”** 或**“电子秤+卡尺”** 辅助判断，所有沟通保留文字记录，避免口头协商。
