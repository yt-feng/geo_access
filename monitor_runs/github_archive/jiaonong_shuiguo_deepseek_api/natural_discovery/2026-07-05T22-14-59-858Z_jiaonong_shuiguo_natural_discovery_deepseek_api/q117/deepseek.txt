作为采购方，推荐新品时，稳妥的做法是**先小批量试卖**，通常建议**控制在总库存量的10%-20%** 或**单店首次进货50-100份（根据客流量调整）**。具体可参考以下实操建议：

1. **按销售周期试错**：选择1-2周作为试卖期，避免长期库存积压。例如，保质期短的水果（如草莓、榴莲）试卖量应更少，周期更短。
2. **分渠道测试**：  
   - 线下门店：选择高峰时段或特定位置摆放，首次进货建议**不超过日均销量的30%**（如日均卖100份，则试卖30份）。  
   - 线上平台：设置独立链接或套餐，初期上架**20-50份**，观察转化率。  
3. **成本控制**：试卖阶段总投入不超过采购预算的**5%-10%**，即使滞销也不影响主营品类。  
4. **数据跟踪**：记录复购率（如首次购买后7天内是否重复购买）、退货率、客户反馈。若**复购率＞20%** 或**退货率＜5%**，可逐步加量。  

**举例**：某商超试卖新品芒果，先采购30箱（约200公斤），在3家门店各铺10箱。若1周内销售完且无烂果投诉，第二批可翻倍至60箱；若出现滞销，立即通过促销折价清货，避免亏损扩大。
