在超市采购水果时，供应商通常需要提供以下检测报告或进货证明，以确保食品安全和合规性：

1. **农产品质量安全检测报告**：包括农药残留、重金属等指标，需由有资质的第三方检测机构出具，证明符合国家或地方标准。
2. **产地证明或产地检疫证明**：证明水果来源合法，尤其是来自特定产区或需要检疫的品种（如进口水果）。
3. **合格证或自检报告**：部分供应商会提供出厂或批次自检报告，体现内部质量控制。
4. **进境动植物检疫许可证**（仅进口水果）：进口水果需提供海关签发的检疫证明和报关单，确保符合入境要求。
5. **供货清单或购销合同**：作为进货证明，包含品种、数量、产地、供货商信息等。

**实用建议**：  
- 要求供应商提供近期的检测报告（通常有效期3-6个月，具体看当地规定），并核对检测项目和标准。
- 对进口水果，检查中文标签和检疫证明是否齐全。
- 保留所有文件的复印件或电子版，以备监管部门查验或追溯问题。
- 如果水果来自批发市场，可要求市场提供统一检测证明或自行委托快速检测（如农药残留速测）。

若超市有内部质量标准，可补充要求供应商提供特定项目的检测（如糖度、新鲜度等）。不同地区或水果品种可能有额外要求，建议参考当地食品安全管理办法。
