针对小面积便利店的生鲜经营，核心策略是：**“高周转、短陈列、精品类、不压货”**。以下是针对品类控制和损耗管理的具体建议：

### 一、品类控制：四步选品法

1.  **“爆品+引流”策略（适用季节性强、认知度高的水果）**
    -   只聚焦 **2-3种** 当季最热门的水果（如：夏天的西瓜/芒果，秋冬的柑橘/柚子）。
    -   **理由**：大众认知度高，不需要教育用户，自然走量快，不需要太多陈列面。

2.  **“即食替换”策略（适用高价/后熟型水果）**
    -   少选需整箱购买或未熟水果（如：进口大柠檬、整串香蕉），优先选**即食、小包装、切盒装**。
    -   **推荐品类**：
        -   **切盒水果**：西瓜、哈密瓜、火龙果、凤梨（现场切或冷链配送，半份装卖）。
        -   **小包装浆果**：蓝莓、圣女果、葡萄（独立小盒，单价低，易翻新）。

3.  **“硬通货”常备（适用耐存储、复购高的水果）**
    -   苹果、梨、橙子（这类水果货架期长，不容易烂，适合周末囤货场景）。
    -   **操作建议**：只放 **2-3个精选品种**（如冰糖心苹果、水晶梨、爱媛果冻橙），避免十几个品种挤在一起。

4.  **“爆品轮换”策略（利用周末和节假日）**
    -   每周主推1个特价引流品，比如：周四、周五上架“红心火龙果”，下架上周的“金枕头榴莲”，通过券、海报在门口标注，吸引进店。

### 二、损耗控制：实测有效的六种方法

1.  **动态进货法（解决库存积压）**
    -   **不要按整箱进**：便利店面积小，进货箱码体积大。跟供应商谈“按斤批发”或“日日配”。
    -   **参考数据**：每天傍晚核算当天销售情况，第二天的进货量 = （前一天销量 × 1.1）- 当天库存余量（根据天气、周末调整）。

2.  **“二八陈列”与“翻面”管理（解决撞伤/腐烂）**
    -   将损耗风险最高的商品（如草莓、蓝莓）放在**黄金视线**（成人腰部以上、眼睛以下），要求店员每隔1小时**翻面一次**（把下面的、颜色好看的翻出来，把已有瑕疵的挑出）。
    -   每个单品只陈列**前后两排**（避免堆多层压坏）。

3.  **“烂果即时处理”止损法**
    -   **发现一个，立刻移除**：一旦发现单个水果腐烂（如苹果有黑斑），立刻拿掉，并用湿布擦拭附近的水果（避免霉菌扩散）。
    -   **设立“当日特卖”区**：对轻微碰伤、过熟的水果（如软香蕉、表面有黑点的芒果），用塑料袋统装，贴上“今日特价5元/份”标签，限当天售罄，**严禁和正常水果混堆**。

4.  **控制上货节奏（解决早高峰vs晚高峰）**
    -   **早上8点前**：上齐当日最全品种（保持2种左右主推，其他少量）。
    -   **下午3-5点**：补充**切盒水果**（利用午休后的倦怠期，切盒容易产生食欲）。
    -   **晚上7点后**：不再大量补货，只补空位（避免夜间损耗）。

5.  **“5元/份”统一价止损（解决滞销）**
    -   对当天卖不完、但尚未损坏的尾货（如：3个香蕉+2个橘子+1盒蓝莓），直接混装成“每日混合水果盒”，统一定价**5元**或**9.9元**（便利店场景下，这种“盲盒”形式反而容易清仓）。

6.  **“温度分区”微调（针对易腐品种）**
    -   香蕉、芒果、木瓜**不要放冷柜**（冷害导致发黑），常温阴凉处即可。
    -   草莓、蓝莓、车厘子、葡萄**必须放4℃冷柜**，且在底部垫吸水纸（防止湿气积聚烂底）。
    -   西瓜切割后**立刻覆保鲜膜**竖放（切口伤，损耗最高）。

### 三、具体执行清单（模板，可直接贴在收银台背后）

-   **每日必做**：
    -   [ ] 早上补货前先检查昨日库存，检查是否有烂果（剔除）。
    -   [ ] 挑出3个以下轻微瑕疵水果，放入“今日特价”筐。
    -   [ ] 进店第一眼水果区域必须是新鲜、果柄完整、没有水汽的（象征“干净”）。
-   **每周淘汰**：连续3天销售倒数第一的果品，立刻下架，换新。
-   **每周保留**：苹果、梨、橙子各1个品种常驻（耐放）。

### 总结公式：
**小面积便利店水果 = 爆品（20%sku，贡献60%销售额） + 即食小包装（30%sku，满足冲动消费） + 硬通货（20%sku，兜底） + 当日清理（30%sku，止损）**

如果便利店单日水果营业额低于300元，建议只保留3种果品（切盒混合装+当前热销1种+苹果/梨1种），等客流稳定后逐步扩品。
