针对超市水果区提升客单价，建议采用 **“引流爆品+高价值搭配 + 消耗品组合”** 的策略，利用不同水果的特性来刺激连带购买。以下是针对香蕉、菠萝、苹果的具体促销方案：

**方案一：早餐/果汁组合装（提高单次客单价）**
- **组合内容**：2根香蕉（易熟、快消）+ 1个菠萝（高价值）+ 3个苹果（耐储存）
- **促销定价**：原单独购买约25-30元，组合售卖 **19.9元-22.9元**（约8-7折），刺激一次性大包装购买。
- **话术**：“鲜切菠萝+现做香蕉奶昔+苹果早餐包，为家人备齐3天气色果盘。”
- **好处**：香蕉加速菠萝消化（通便），苹果中和甜度，让客户觉得“搭配值，省每天跑超市”。

**方案二：健康礼盒/零食盒（提升价值感）**
- **组合内容**：1个菠萝（切好装盒）+ 2个苹果（贴笑脸贴纸）+ 1小串香蕉（赠送果叉/小刀）
- **定价**：统一做成 **29.9元“快乐果篮”** （相较于单买3种水果约35-40元）。
- **操作**：放在出口/收银台旁，包装用透明手提盒，标注“切好即食，无需处理”。
- **效果**：方便不愿处理菠萝的人群，连带香蕉作为甜点、苹果作为零食，整体代入“招待访客”或“办公室拼单”场景。

**方案三：楼梯式满减（递增购买动机）**
- **阶梯**：
  - 买2种（比如香蕉+苹果） → 总价打9折
  - **重点推荐**：3种一起买（香蕉+菠萝+苹果） → 直接减5元/套，并**额外赠送1个菠萝冷藏切块杯（成本约0.5元）**
- **包装**：将菠萝提前去皮切块，和苹果、香蕉分盒装在同个大托盘，贴“促销拼盘”标签。
- **效果**：客户本来只想买香蕉（2-3元），看到“凑菠萝和苹果省5元且送切块杯”，容易被引导买下原本不会买的高价菠萝。

**执行要点**：
1.  **摆放位置**：香蕉（最显眼，易冲动购） → 隔开1米放苹果（眼能看到香蕉）→ 4米外处理好的菠萝（突出“切半价”等字眼，吸引闻味下单）。
2.  **定价策略**：**菠萝不单独打折**，作为组合中的核心利润品；香蕉作为吸附品；苹果作为“保证折扣感”的中等品。
3.  **备货**：多进小规格苹果（如4-5个装），减少卖相劣势，配合香蕉的快速腐化周转。

此方案下，单客从购买单一水果（客单价约5-8元）可提升至18-30元，同时解决菠萝滞销和香蕉库存压力。
