评估连锁门店的水果供应商仓配覆盖，需从以下几个核心维度进行系统性检查：

**1. 基础数据对齐**
- **门店清单与需求**：列出所有门店的详细地址、日均销量（按SKU）、配送时间窗口（如早晨6点前）、特殊要求（如避震包装）。
- **供应商仓网**：获取供应商所有仓库（RDC/DC/FDC）的位置、覆盖半径、库存SKU清单及鲜度承诺（如“24小时从树到仓”）。
- **关键指标**：各仓距你门店的运输时长（非直线距离，需考虑交通及限行）。

**2. 覆盖评估四步法**

**第一步：门店分群**
将门店按城市/区域分组，优先评估同一城市/经济圈内是否至少有1个供应商仓库。如果单一仓库无法覆盖所有门店，需考量多仓组合的覆盖面。

**第二步：时效匹配**
- 计算从每个仓库到对应门店群的**单程运输时间**（含卸货）。对于生鲜水果，通常**24小时内完成“从出库到上架”**是基础要求，部分高损耗水果（草莓、蓝莓）要求更短。
- 检查供应商是否具备**多点配送能力**（一辆冷链车配送多家门店），这会显著降低成本与时效。

**第三步：库存承压测试**
- 模拟你的**促销节（如中秋节、年货季）**销量高峰。供应商在主要仓库的**日处理订单能力**（件数/重量）是否足够？是否能调用邻近城市仓库补货？
- 检查**冷库容量**：当大量水果集中到货（如整柜进口车厘子）时，供应商库内暂存和分拣能力是否匹配你的上架节奏。

**第四步：应急冗余**
- 了解供应商的**替代仓库**或**应急配送机制**。例如，某城市仓库突发设备故障，能否在2小时内从邻市仓库调货或通过第三方干线转拨？
- 询问其对**天气/交通管制**的预案（例如：暴雨天是否启用备用路线或提前发车）。

**3. 核心验证与风险点**

*   **水果损耗兜底**：重点关注供应商在“到店验收”环节的责任划分。覆盖能力再强，如果损耗超过3%且不赔付，成本会失控。
*   **单品集中风险**：如果某品种水果（如新西兰奇异果）由单一海外产区或特定仓库供应，需评估该仓库甚至该航线中断的替代方案。
*   **城市末端能力**：一线城市可能有“入城证”“限行”问题，需确认供应商仓库是否处于可正常通行区域，或已备有合规的城配车辆。

**4. 评估工具与行动清单**
- **制作热力图**：将门店位置与供应商仓库位置绘在地图上，直观查看覆盖盲区。若超过10%门店距离最近仓库车程**超过8小时**（水果通常需日配），需重新谈判或补充供应商。
- **分批次试跑**：选择3-5个具有代表性的门店（市中心、远郊、居民区），进行为期2周的小批量订单测试。重点记录：
  - 到货时间是否在承诺窗口内
  - 水果新鲜度（外观、硬度、口味）
  - 破损率与拒收率
- **要求供应商提供盖公章的《仓配服务承诺书》**：明确各仓库覆盖半径、配送频次、响应时效、损赔付标准。

**最终判断标准**：
如果一个供应商能覆盖**80%以上门店**且**旺季高峰履约率≥95%**（以实际到货时间±30分钟为准），且**单店平均到店损耗低于3%**，该仓配网络即可视为合格。剩余不足的门店，可考虑补充1-2个区域型供应商或自建末端中转站。
