小商家采购水果时，**现金结算**和**月结账期**各有以下风险：

**1. 现金结算的风险：**
- **资金压力大**：需要一次性支付全额货款，可能占用周转资金，影响购买其他商品或应对突发开销。
- **质量保障难**：付款后若水果出现腐烂、缺斤短两等问题，追回款项或索赔较困难，尤其是无法提供正规票据时。
- **交易无记录**：现金支付缺乏合同或电子凭证，若发生纠纷（如商家不认账、货物未送达），维权证据薄弱。
- **安全隐患**：携带大量现金交易易遭遇盗窃或丢失。

**2. 月结账期的风险：**
- **坏账风险**：若采购方（你）经营不善或资金周转出现问题，可能无法按时付款，导致供应商拒绝后续供货或产生违约金。
- **供应商信任门槛高**：月结通常要求长期合作或信用背书，新商家很难获得，且供应商可能提高单价来对冲风险。
- **账期管理繁琐**：每月需核对账单、提醒付款，若忘记或延迟，可能影响信誉并导致断货。
- **法律风险**：若没有正式合同约定付款期限和违约金，一旦欠款，供应商可能诉诸法律，增加处理成本。

**建议**：
- 对**小额、高频、易损耗**水果（如易变质水果），优先用现金结算，但务必索要收据并当面验货。
- 对**稳定供应商或大宗订单**，可尝试月结，但需签订书面合同，明确账期、付款条件、违约责任，并保留对账单和转账记录。
- 可结合使用：现金购买少量高波动品种，月结用于长期合作的大额订单，平衡资金流与风险。
