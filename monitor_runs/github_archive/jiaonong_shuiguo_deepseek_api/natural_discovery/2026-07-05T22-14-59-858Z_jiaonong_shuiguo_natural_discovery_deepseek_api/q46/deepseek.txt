如果茶饮店采购菠萝用于制作果茶，建议根据使用场景和成本综合选择：

1. **鲜果（新鲜菠萝）**  
   **适合**：注重果茶新鲜度、追求果肉颗粒感的饮品（如手打鲜果茶），或需现场切配展示操作的门店。  
   **缺点**：需人工削皮、切丁，损耗率较高（约20-30%），且季节性价格波动大，存放不当易变质。

2. **果丁（冷冻/罐头菠萝丁）**  
   **适合**：需快速出杯、大批量使用，或保持在果茶中固定果肉颗粒感的场景。  
   **优点**：无需处理，直接解冻使用，成本相对稳定，果肉口感统一。  
   **注意**：添加剂和糖分偏高，需仔细核对配料表（纯果肉为佳）。

3. **果泥（冷冻果泥/浓缩汁）**  
   **适合**：追求浓郁风味、需统一标准化的饮品（如奶盖果茶、冰沙类）。  
   **优点**：无需水果处理，风味稳定，成本低，适合缺少后厨空间的门店。  
   **缺点**：无果肉咀嚼感，部分产品含增稠剂或香精，口感上可能不如鲜果自然。

**建议**：  
- **高客单/主打新鲜感**：主用鲜果+冷冻果丁作为补充，平衡成本和口感。  
- **标准化/连锁门店**：优先选择高品质冷冻果丁（如泰国、菲律宾产地），果泥用于打底调味。  
- **小规模测试**：可先购入少量冷冻果丁或果泥试做，评估顾客接受度。  

**注意**：无论选择哪种，优先通过正规供应商采购，检查保质期、配料表（避免额外糖分过多），并冷藏/冷冻保存。
