针对连锁商超长期水果供应商招标，评分标准应兼顾**价格竞争力、品质稳定性、供应保障能力、服务配合度**及**长期合作潜力**。建议按以下维度设置评分权重（总分100分）：

### 1. 价格与成本竞争力（30-35分）
- **报价水平（15-20分）**：对比基准价为市场均价或历史采购价，低于均价5%以内得满分，每低5%加1-2分（防止恶意低价），高于均价递减。
- **价格调整机制（5-8分）**：是否接受阶段性调价（如按消费指数或批发价波动浮动），或提供锁价优惠（如季付/年付折扣）。
- **附加成本透明（5-7分）**：是否包含运输、损耗补贴、包装费用（若包含，加分；若需单独计费，需明确标准）。

### 2. 品质与溯源能力（25-30分）
- **品控标准（10-12分）**：提供明确的新鲜度指标（如硬度、甜度、外观分级）、农残检测报告（定期且第三方机构认证）、采摘/冷链时间说明。
- **源头可追溯（8-10分）**：是否实现全程溯源（如二维码追溯果园、采摘、分拣信息），或提供实地考察记录（如基地签约、实地照片）。
- **样品审核（5-8分）**：投标时提供代表性样品，按视觉、口味、储存耐性（如3-5天货架期表现）打分。

### 3. 供应稳定性与物流能力（20-25分）
- **日均/月均供应容量（8-10分）**：明确能稳定供应的品种（如苹果、柑橘等常规品）、月最大供货量（需匹配超市客流峰值，如节假日、促销季）。
- **应急预案（5-7分）**：恶劣天气、产区欠收等场景下，有无备选产区或替代品种方案（需提供具体合作基地信息）。
- **物流时效与冷链（5-8分）**：到货时间（如24-48小时直达门店）、冷链车覆盖率（如有，加分）、运输损耗率（如<3%得满分）。

### 4. 服务与配合度（10-15分）
- **退换货政策（4-6分）**：高损耗品（如草莓、葡萄）是否接受按比例退换（如3%-5%正常损耗免费补货），响应时间（如48小时内处理）。
- **促销与新品支持（3-5分）**：是否配合超市季节性陈列（如车厘子节）、提供试吃品或买赠方案。
- **账期与结算（3-4分）**：是否接受60-90天结账周期（加分），或提供更灵活的月结+承兑汇票形式。

### 5. 长期合作潜力（10-15分）
- **过往合作案例（5-8分）**：提供与其他连锁商超的合作年限、年供应额、客户评价（如若合同期>1年、无重大违约记录，此项满分）。
- **可持续发展认证（2-4分）**：如GAP、有机认证、绿色食品认证（加分项）。
- **违约条款明确性（3分）**：合同是否详细列明质量违约、供应不足的罚则（如按缺货量30%-50%扣款）。

### 评分规则建议：
- **总分设门槛**：低于80分（满分100分）的直接淘汰，确保基础供应能力。
- **加权后再议价**：例如，中标后可根据最终报价微调1-2分，但需提供成本明细（如种植成本、物流费）以防价格虚高。
- **多轮评分**：第一轮按纸质材料打分；第二轮现场随机抽检仓库库存与冷链设备，结合样品盲测结果加权。

**其他注意事项**：  
- 若关注品种时效性，可设置季节性加分（如草莓、荔枝的产区温度控制方案）。  
- 建议引入第三方质检机构（如SGS）定期抽检，并在评分中预留5-10分作为“签约后履约监督项”，按月度得分浮动调价。
