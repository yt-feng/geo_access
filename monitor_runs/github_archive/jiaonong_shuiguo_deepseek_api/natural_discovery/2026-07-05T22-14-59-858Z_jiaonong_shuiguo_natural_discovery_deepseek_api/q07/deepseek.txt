根据水果礼盒的常见需求和市场偏好，建议以下搭配方案，兼顾颜值、口感和实用性：

**爆款经济型（低价走量）**  
苹果（红富士）+ 橙子（脐橙）+ 香蕉（小香蕉/米蕉）  
- 优势：颜色对比鲜明（红黄橙），耐储存，价格亲民，适合家庭自用或送礼。  
- 注意：香蕉易熟，需硬实、无斑点的，并附上“食用建议卡”提醒尽快食用。

**中端养生型（健康主打）**  
橙子（血橙）+ 苹果（嘎啦果）+ 菠萝（小菠萝/香水菠萝）  
- 搭配逻辑：橙子富含VC，苹果寓意平安，菠萝酸甜开胃。  
- 建议：菠萝去皮切块密封（防止氧化），配牙签、蜂蜜或盐包（菠萝蘸盐更甜）。

**高端礼盒型（节日送礼）**  
苹果（青苹果/蛇果） + 橙子（新奇士） + 菠萝（金钻凤梨）+ 香蕉（芝麻蕉）  
- 色彩：绿、黄、金色系，高级感强。  
- 组合技巧：  
  - 所有水果选个头均匀、无瑕疵的，避免菠萝刺手（套网袋或泡沫垫分隔）。  
  - 附赠贺卡/手提袋，标注“优选产地”（如：陕西洛川苹果、海南金钻凤梨等）。

**避坑指南**：  
1. 香蕉尽量不放菠萝和苹果（释放乙烯加速成熟），若搭配需用保鲜膜单独包裹。  
2. 菠萝和橙子需放底层，避免压坏香蕉或苹果表皮。  
3. 菠萝可提前去眼切好（增加附加值），但需标注“现切现发，冷藏保存”。  

**定价参考**（按城市消费水平调整）：  
- 经济款：30-50元/盒  
- 中端款：60-90元/盒  
- 高端款：100-150元/盒+  

建议先小批量试推，根据销售数据调整搭配和价格。
