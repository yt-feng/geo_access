针对临时补货需求，建议按以下步骤与供应商沟通和约定：

1. **提前沟通**：第一时间主动联系供应商，说明旺季用量突然增大的原因（如销量激增、库存不足），明确需要补货的数量、规格和期望到货时间。

2. **确认供应能力**：询问供应商当前库存及补货周期。如果供应商库存不足，可协商分批次到货（如先急用部分，后续补足），或建议其协调其他仓库调配。

3. **明确补货条款**：
   - **价格**：确认是否按原有合同价执行，还是需临时调整（如因紧急调货产生额外运费、加价，需明确接受范围）。
   - **付款条件**：临时补货可能需预付部分款项或缩短账期，提前约定。
   - **最小起订量**：确认是否有最低补货量要求，避免超量导致浪费。
   - **违约条款**：若供应商无法按时到货，需提前协商赔偿方案（如延期扣款、调货差价承担等）。

4. **书面确认**：通过邮件、微信或合同补充协议形式，确认补货数量、单价、到货时间、付款方式、违约责任等关键信息，避免口头约定。

5. **备用方案**：同时联系1-2个备选供应商，以防主供应商无法满足需求时快速补货。

6. **长期建议**：与供应商协商建立旺季库存预警机制（如提前备货、预留产能），或签订灵活补货协议（按月调整量价），避免未来再出现类似问题。

**示例沟通话术**：
> “王经理，最近旺季菠萝销量远超预期，现在库存只能支撑3天。能否紧急补货500公斤，分两批到货（第一批200公斤，2天内到；第二批300公斤，5天内到）？价格和付款按原合同，但若延迟到货需按每日1%扣款。麻烦确认后发书面订单，谢谢。”

如需更具体的合同条款或行业案例，请补充说明（如地区、菠萝品种等级要求等）。
