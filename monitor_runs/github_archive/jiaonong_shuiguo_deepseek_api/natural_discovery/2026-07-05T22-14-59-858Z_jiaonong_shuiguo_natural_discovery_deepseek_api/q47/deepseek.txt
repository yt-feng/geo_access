采购香蕉时，需根据茶饮或甜品的具体用途控制熟度，常见策略如下：

1. **分类采购**：根据产品需求明确熟度标准。例如：
   - **做奶昔、冰沙、蛋糕糊**：需**全熟香蕉**（表皮出现黑斑，按压软糯），甜度高、易压泥。
   - **做切片装饰、果茶配料**：需**八成熟香蕉**（表皮金黄，稍带绿，捏着有弹性但不过软），能保持形状不易塌。
   - **做烤制甜品（如香蕉派）**：可用**九成熟香蕉**（表皮黄色带少量褐点），口感软但不易出水。

2. **分批采购**：与供应商约定按不同批次送货。例如：
   - 订单中**30%为全熟**（当天或次日用），**70%为七成熟**（存放2-3天后使用），降低损耗。
   - 要求供应商提前按“绿熟期”“黄熟期”分装，标注采收日期。

3. **存放与催熟控制**：
   - 到货后按熟度**分开放置**：全熟香蕉放冷藏（需尽快用完），六七成熟放常温（纸袋密封可加速催熟）。
   - 使用乙烯催熟剂（需符合食品标准）或与苹果/梨同袋存放（释放乙烯自然催熟）。

4. **检查与签收规范**：
   - 要求供应商提供**色卡对照表**（如：1-2级绿色未熟，5-6级全黄，7-8级有斑点），到货时随机抽检5根对照。
   - 若有黑斑腐烂、霉变或过分青绿未熟，立即拒收或协商退换。

5. **量化采购量**：
   - 根据日销量×1.5倍安全库存向供应商下单（如日需20根，则采购30根），预留全熟烂果淘汰空间。
   - 周末或节假日前采购更生些的香蕉（可存放更久），避免成品挤压。

**执行建议**：制作一张《香蕉熟度对照表》（含照片和硬度测试方法），贴在采购和库存区域，并培训员工用目测+轻捏法判断：表皮绿色（不熟）、亮黄（可熟）、黄带斑（全熟）、大块黑斑（过熟不宜使用）。
