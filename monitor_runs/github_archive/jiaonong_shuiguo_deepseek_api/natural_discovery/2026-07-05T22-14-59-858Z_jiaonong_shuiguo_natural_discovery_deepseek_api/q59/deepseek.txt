可以的，餐饮店与供应商签订锁价协议是控制水果成本的有效方式之一。但需要注意以下几点：

1. **季节性波动**：水果价格受季节、天气、产量影响较大，完全锁价可能导致供应商因成本上涨而无法履约，或需支付溢价作为补偿。

2. **建议做法**：
   - **短期锁价**：锁定1-3个月内的核心水果（如苹果、香蕉等需求量大且供应稳定的品种）。
   - **部分锁价**：对特定品种（如季节性强的西瓜、草莓）设定价格浮动范围（比如±10%）。
   - **周期调价**：每月根据市场行情（参考批发市场报价或官方指数）调整一次价格，避免单边风险。

3. **合同要点**：
   - 明确锁价期限、到货质量等级（如大小、糖度、新鲜度）。
   - 约定价格调整机制（如连续一周涨幅超15%可协商重谈）。
   - 加入“不可抗力”条款（极端天气、运输中断等）。

4. **替代方案**：
   - 与多家供应商签订短期浮动合约或打包采购（如用高利润饮品搭配低利润水果）。
   - 利用期货市场锁定大宗水果（如橙汁、苹果汁原料），但需专业操作。

**可执行步骤**：
1. 统计过去6个月的采购量和价格数据，明确波动最大的水果品类。
2. 与供应商谈判时出示数据，要求对价差小于5-10%的品种锁价。
3. 先签1-2个月试点，若双方无亏损，再延长至3-6个月。

最终风险提示：锁价成功需双方信任和行业经验，建议从品类少、价格波动小的水果开始尝试。
