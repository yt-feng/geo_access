采购菠萝时判断熟度，建议从**外观、气味、手感、重量**四个方面入手，目标是选择**七八成熟**的菠萝（即表皮青黄相间、果肉已甜但尚未完全变软），这样既能减少运输和存放期间的损耗，又能在后续几天自然成熟销售。

---

### 1. 看“眼”和“皮”
- **表皮颜色**：  
  - **七八成熟**：表皮呈**青黄色或黄绿色**，黄色部分约占1/3到1/2，青硬部分较多。  
  - **过于生**：全青、坚硬。  
  - **过于熟**：全黄甚至褐色、发软有黑斑（极易腐烂，损耗大）。
- **果眼（凹陷处）**：  
  - 成熟度好的菠萝，果眼**饱满、凸起明显**，且暗绿色转为黄绿色。  
  - 果眼扁平、颜色深青的，偏生。

### 2. 闻“底”和“叶”
- **底部气味**：  
  - 靠近菠萝底部（果梗端）闻，有**淡淡清香**为佳。  
  - 无味则太生；有浓烈甜腻或酒精味则过熟或已变质。
- **叶片**：  
  - 叶片**翠绿挺立**，轻轻拉中间叶片不易脱落。  
  - 叶片发黄、干枯或一拉就掉，表示存放太久。

### 3. 按“皮”和“身”
- **手指按压**：  
  - **七八成熟**：果皮**微硬但有弹性**，轻按不会下陷。  
  - 太生：硬得像石头。  
  - 太熟：按压有凹陷、发软（一碰就坏）。
- **重量**：  
  - 同等大小选**手感较重**的，说明水分足、新鲜。  
  - 过轻则可能失水或空心，易损耗。

### 4. 提示“避坑”点
- **避免全青色**：这类菠萝虽然耐运输，但店里难以自然熟透（室温下容易先腐烂后变甜），且口感酸涩。  
- **避免全黄色**：全黄菠萝多数已过度成熟，运输中稍碰撞就易出现黑心、软烂。  
- **避免底部发黑**：果梗端有黑斑或霉点，说明内部已开始变质。

---

**采购建议总结**：  
- **首选**：表皮**青黄色、果眼饱满、叶片绿、有清香、按压微硬**的菠萝。  
- **采购后**：立即剔除有伤、过熟的，按熟度分级摆放（青的常温催熟，黄的放冷柜短期销售）。  
- **催熟小技巧**：青黄菠萝可和苹果或香蕉一起放到纸箱里（室温20-25℃），一般2-3天就能黄熟。
