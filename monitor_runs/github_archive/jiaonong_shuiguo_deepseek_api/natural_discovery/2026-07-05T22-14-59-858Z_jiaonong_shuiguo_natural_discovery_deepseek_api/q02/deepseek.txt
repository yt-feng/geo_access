对于小水果店来说，进香蕉时建议按以下优先级排序：

1. **熟度（最核心）**  
   香蕉是典型的后熟型水果，但顾客更倾向买即食或1-2天内成熟的。如果熟度太高，当天卖不完会损耗；太低则容易让顾客觉得“生”，不愿购买。通常选择**7-8成熟**的香蕉（表皮全黄但两端略带绿尖或微黄），这样有2-3天的销售窗口。

2. **坏果率（直接影响利润）**  
   低坏果率比低价更重要。如果进价便宜但坏果率超过5%，单根损耗会吞掉利润。建议和供货商约定“坏果包赔”或现场开箱验货（随机拆几箱看内部是否有压伤、黑斑、轴心褐变）。

3. **价格（需结合销量和损耗）**  
   在熟度和坏果率达标的前提下，比较不同渠道价格。香蕉是高频低价商品，单价差1-2毛对顾客不敏感，但对小店的单日毛利（假设日均卖50斤，差价0.5元/斤，月利润差750元）有明显影响。可适当选中间价位的货，避免极端廉价但品质不稳的。

4. **送货速度（可协调）**  
   除非你地处偏远且缺库存，否则不必过度追求当日达。大多数香蕉供应商能保证隔日到货。关键不是速度，而是**到货时段的稳定性**（例如约定早上7点前到货，方便上架）。

**实操建议**：  
- 首次合作时先少量进几箱（约3-5箱），重点测试 **熟度** 和 **坏果率**，再长期合作。  
- 如果供应商能提供“分批次配送”（比如每天上午送一次），可优先考虑，能减少单次库存压力。  
- 宁可每斤多花1毛买品质稳定的货，也别为省成本承担高损耗。
