水果店遇到坏果时，与供应商结算或赔付的一般流程如下：

1. **确认问题**  
   - 第一时间拍照或录像保留坏果证据（包括外包装、物流单号、数量、损坏程度等）。  
   - 区分坏果原因是运输破损、存储不当、本身质量问题还是自然腐烂。

2. **查阅合同或约定**  
   - 查看与供应商签订的采购协议中关于坏果、损耗的条款（通常有赔付比例、退换货条件等）。  
   - 若无书面合同，参考行业惯例或双方口头约定（例如：运输途中坏果由物流或供货方负责，到店后短时间发现的质量问题可按比例扣款）。

3. **及时沟通**  
   - 在规定时间内（通常收货后24小时内）联系供应商，提供证据，说明情况。  
   - 明确要求：如退货退款、补发好果、扣减对应货款、按比例赔付（例如5%以内自然损耗由买方承担，超出部分由供方负责）。

4. **协商结算方式**  
   - **按实际收货量结算**：仅支付合格果实的金额，坏果部分不计算在内。  
   - **按比例扣款**：例如总货款中扣除坏果价值（按采购单价计算）。  
   - **赔付现金或下次抵扣**：供应商可能以现金赔付或下次进货时抵扣金额。

5. **处理补救**  
   - 若供应商同意换货，尽快安排退换流程。  
   - 若无法协商，可考虑向市场管理方、行业协会或市场监管部门投诉，必要时通过法律途径解决。

**注意**：不同品类（如草莓、苹果、香蕉）的坏果敏感度和赔付标准不同，建议在首次合作时明确写入合同，避免纠纷。日常经营中保留好进货记录和单据。
