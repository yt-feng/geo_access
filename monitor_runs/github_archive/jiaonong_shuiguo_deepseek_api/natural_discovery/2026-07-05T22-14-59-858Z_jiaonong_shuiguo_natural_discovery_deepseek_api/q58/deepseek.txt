从损耗角度来看，**直接购买成品果丁**通常损耗更低，原因如下：

1. **鲜果切配的损耗环节多**  
   - 菠萝在采购、运输、存储过程中可能因磕碰、过熟或腐烂而产生损耗（通常为5-15%）。  
   - 切配时需削皮、去眼、切丁，果皮和芯的废弃率约占总重量的40-50%。  
   - 切丁后若未及时使用，暴露在空气中的果丁会失水、氧化，进一步增加损耗（约5-10%）。

2. **成品果丁的损耗低**  
   - 工厂统一加工时，边角料（如果皮、果芯）可被综合利用（如做果汁、饲料），降低自身成本，从而间接降低售价中的损耗分摊。  
   - 成品果丁通常采用真空或气调包装，保质期较长（冷藏下3-7天），减少了因存放不当造成的即时损耗。  

**建议**：  
- 如果甜品店用量稳定且空间有限（如店铺较小），直接购买成品果丁更划算。  
- 若用量大、有专人处理，且当地菠萝价格低（如产地附近），鲜果切配可能更便宜，但需严格控制库存与加工时效，避免产生额外损耗。
