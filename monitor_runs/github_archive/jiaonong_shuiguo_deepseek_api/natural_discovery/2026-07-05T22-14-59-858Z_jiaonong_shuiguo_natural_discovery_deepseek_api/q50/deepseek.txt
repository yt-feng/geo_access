对于茶饮店采购菠萝原料，**出汁率比甜度更关键**，但最佳方案是两者平衡。以下是详细分析：

1. **出汁率直接决定成本与产量**  
   茶饮店制作饮品（如果茶、菠萝汁基底）需要大量汁液。出汁率低的菠萝会导致原料浪费、成本上升，且每杯出品的用量不稳定。高出汁率（40%以上）能保证标准化生产，降低损耗。

2. **甜度可通过配方调整**  
   如果菠萝本身甜度不足，可通过添加糖浆、蜂蜜或其他水果（如苹果、梨）来平衡风味。但低出汁率无法通过后期调整弥补，饮品的口感和浓度会受影响。

3. **高端茶饮需兼具香味和酸度**  
   部分连锁店会优先选择甜度高、香气足的菠萝品种（如金钻、香水菠萝），因其果肉更香浓，适合制作菠萝特调或作为装饰；但如果用于榨汁，出汁率仍是基础。

4. **实际采购建议**  
   - **核心产品为菠萝果汁/冰沙类**：优先选出汁率高的品种（如台农17号、巴厘菠萝），成本更可控。  
   - **核心产品为装饰果肉或菠萝冻**：选甜度高、纤维少的品种（如金钻、牛奶菠萝）。  
   - **折中方案**：试用不同产区样品，测试甜度（12-16°Brix）和出汁率（40%-50%），并计算成本与出品匹配度。

**结论**：先确保出汁率≥35%-40%以保证出品稳定，再通过其他成分调整甜度；若预算允许，选择出汁率和甜度都较高的品种。
