通常情况下，水果采购与供应商谈判的“锁价”期限会因水果品种、市场波动、供需关系和合作模式而有所不同。一般来说，常见期限如下：

1. **短期锁价（1-2周）**：适用于应季水果（如荔枝、杨梅）或价格波动较大的品种，供应商通常愿意锁定较短时间，以规避风险。

2. **中期锁价（1-3个月）**：适用于大宗、存储性较好的水果（如苹果、柑橘、梨），在供应旺季或签订购销协议时较常见。

3. **长期锁价（半年以上）**：较少见，通常仅适用于加工用水果（如浓缩果汁原料）、通过期货或长期合约模式锁定，需要双方有较强信任和明确的分批交付机制。

**建议**：
- **明确协议条款**：在锁价协议中明确有效期、交付方式、质量标准、价格调整触发条件（如极端天气、运费大涨）。
- **分阶段锁价**：如果采购量大，可尝试分批锁价（如每月谈判一次），平衡价格风险和供应保障。
- **关注市场行情**：利用行业协会数据或产地信息网（如“一亩田”等平台）提前预判价格趋势，选择合适时机签订锁价合同。

如能提供具体水果品种和采购规模，可进一步给出针对性建议。
