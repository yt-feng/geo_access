通常情况下，供应商在周末促销中**有可能**提供支持，但具体取决于合作历史、进货量、供应商规模和季节性等因素。建议你尝试以下方法争取支持：

1. **主动沟通**：直接联系对接的供应商负责人，明确告知周末促销计划和预期销量，询问能否提供**价格折扣**（例如按箱优惠、返点返现）或**赞助物料**（如海报、赠品、试吃样品）。

2. **强调利益**：向供应商说明促销能带来的大额采购或长期合作潜力，比如可以承诺在促销期间主推他们的某个品牌或新品。

3. **准备备选方案**：如果供应商无法降价，可以要求**场地支持**（免费展架或冷柜）或**随货赠品**（例如买水果送小包干果）。

4. **利用行业惯例**：很多品牌方（例如知名苹果、橙子产区、包装水果企业）会为周末人流大的店铺提供**试吃品**或**宣传物料**，可以询问供应商是否有此类内部资源。

**注意**：小规模供应商或非品牌经销商通常缺乏物料预算，可能只愿意在价格上小幅让步。可综合比较几家供应商，将最大支持争取到核心单品上。

如果需要具体话术或协商模板，可进一步说明合作背景。
