在便利店销售菠萝切块时，主要需关注以下三个关键因素：

1. **保质期管理**  
   - 菠萝切块属于鲜切水果，易氧化变味，建议保质期控制在 **24-48小时**（冷藏条件4°C以下）。  
   - 每日检查库存，遵循“先进先出”原则，到期前2小时下架处理。  

2. **包装设计**  
   - 使用 **透气性适中的食品级托盘**（避免积水滋生细菌），覆盖透明保鲜膜或密封盖。  
   - 标注 **生产时间、保质期（精确到小时）** 及品名，并提醒“即食或冷藏”。  

3. **配送与补货频次**  
   - 若为门店自制，配送中心需 **每日至少1次** 补货（早高峰前完成）；若为第三方配送，建议 **每日早晚2次**（如9:00和15:00）。  
   - 监控销量，若上午时段售罄，需紧急加单（适合高客流门店）。  

**优先排序建议**：  
1. 保质期（安全底线）  
2. 配送频次（保障新鲜度）  
3. 包装（提升体验与品牌感）  

具体可执行动作：  
- 建立 **“2小时检查表”**：店员每小时记录货架温度与库存。  
- 对配送车安装 **温控记录仪**，确保全程冷链（4°C±1°C）。
