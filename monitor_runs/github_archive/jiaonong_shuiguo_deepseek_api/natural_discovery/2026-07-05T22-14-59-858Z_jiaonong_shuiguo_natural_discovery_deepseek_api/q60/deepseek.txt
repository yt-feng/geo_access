针对茶饮店水果到货不合格的问题，为避免影响营业，建议在合同中明确以下补货条款，或与供应商进行书面补充约定：

1. **快速响应时限**：规定供应商必须在到货发现不合格后 **2小时** 内确认，并在 **6小时** 内提供补货方案（如紧急调货、替换品种等），以赶上门店高峰时段。

2. **替代品授权**：约定若原定水果缺货，供应商须提供 **3种同价位、可立即使用的替代水果**（例如用红心火龙果替换青柠），并允许门店优先使用已存可替换库存。

3. **补偿机制**：将补货延迟与 **营业损失挂钩**。例如：超时1小时按该水果当日预估销售额的 **20%** 赔偿；若导致断货超3小时，供应商需承担 **当日损失总额的50%**。

4. **应急库存**：规定供应商需为门店预留 **每日使用量30%的缓冲库存**（由供应商仓库或门店冷库储备），出现不合格时可立即调用，无需重新订货。

5. **验收标准前置**：将每次不合格的《质检报告》与 **下次发货批次绑定**，供应商需在接单时承诺取消问题批次订单，并优先补发高品质货源。

**执行注意**：在合同中注明“收到不合格通知即视为供应商已确认，需按上述条款启动补货”，避免扯皮。同时建议保留10%-15%的应急现金流，以便在供应商无法履约时自行从他处高价现采，事后从供应商违约金中扣除。
