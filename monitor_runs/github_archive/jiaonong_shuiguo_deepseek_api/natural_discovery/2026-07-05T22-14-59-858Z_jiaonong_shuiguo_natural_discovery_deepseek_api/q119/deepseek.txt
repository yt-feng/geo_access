选择水果供应商时，很多人会关注价格、品种齐全度，但以下细节常被忽视，却直接影响长期稳定性和利润：

1. **供应商的果源地稳定性**：是否拥有自有或长期合作的果园？如果依赖临时采购，遇天气灾害、产量波动时，你的断供风险会骤增。建议实地查看其分拣场或询问往年极端天气时的应对记录。
2. **分拣标准的具体化**：别只看“一级果”这类口头描述。要问清楚：果径多少厘米以内？甜度用测糖仪控制在什么范围？允许的瑕疵（如小疤痕、轻微碰撞）面积占比是多少？最好对比几次到货样品与标准之间的偏差。
3. **包装与运输的损耗控制**：水果损耗是隐形杀手。确认包装层数、每箱净重精度、是否使用气垫/隔层？冷链车的温度监控记录（从装车到仓库全程能否调取？）。有的供应商会在长途运输前给未成熟水果催熟，需要明确这一操作及对风味的影响。
4. **售后赔付的“文字游戏”**：大部分供应商口头承诺“坏果包赔”，但要看合同细则：理赔是按售价还是成本价？是否包含人工挑选损耗？起赔比例（比如超过3%才赔）？赔付时效（是否拖到下一批货款抵扣）？
5. **账期与资金周转匹配**：如果供应商要求全款预付或账期过紧，会挤压你的现金流。建议先小额试单，了解其财务健康状况（如是否有贷款压力导致急需回款）。同时明确发票税率、对账周期、延期罚款条款，避免后续纠纷。
6. **季节性品种的供应弹性**：比如夏天销售杨梅、荔枝等时令水果，供应商能否保证每天稳定到货？遇到需求激增时，他们会优先给老客户还是价高者？提前约定一个“浮动备货量”条款（如提前48小时确认，上下浮动不超过20%）。
7. **信息透明度**：供应商是否愿意分享产地天气、物流时效、价格波动预测？如果总是含糊其辞，说明他自身也缺乏稳定链条。理想状态是能提供产地批次、检测报告（如农残、重金属）的一键溯源。

**执行建议**：先签3-6个月短期合同，结合上述细节设计“验收指标表”（如分拣合格率需≥95%、损耗率≤2%等），每月双向复盘。定期像客户一样去匿名测试其下单流程和备用货速度，才能真正筛选出可靠伙伴。
