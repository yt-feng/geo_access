针对首次合作的批发商与小批量试单设计，建议遵循以下步骤以降低风险并评估供应商可靠性：

### 1. **明确试单目标与参数**
   - **品种/规格**：选择1-2种主销或易储存的水果（如苹果、橙子），而非高风险易腐水果（如草莓、荔枝）。
   - **数量**：建议控制在总采购量的10%-20%，或具体数值（如100-200斤），以检验品质且不造成库存压力。
   - **预算**：设定试单金额上限（如2000元），避免资金过度沉没。

### 2. **签订试单协议**
   - **品质标准**：明确等级（如一级果）、大小（如直径≥7cm）、成熟度（如7-8成熟）、外观要求（无虫眼、霉斑）。
   - **交付条款**：约定到货时间（如48小时内送货）、运输方式（冷链/普通）、包装要求（如泡沫箱+冰袋）。
   - **检验与争议处理**：约定到货后24小时内可开箱抽检，如品质不达标可拒收或按比例退款。
   - **付款条件**：建议预付30%定金，尾款验收合格后3日内付清。

### 3. **操作流程落地**
   - **样品先行**：要求供应商寄送试用品（5-10斤），确认口感和新鲜度后再下单。
   - **小批量下单**：通过在线平台或合同下单，明确批次唯一编号以便追溯。
   - **验收与反馈**：到货后拍照、录制开箱视频，抽样检查（如配比、甜度计测量），记录问题并及时沟通。

### 4. **风险控制措施**
   - **备用方案**：如供应商无法按时交货或货品差，是否预设第二供应商备选。
   - **售后承诺**：要求供应商提供坏果包赔（如超过5%按斤补发或退款）、物流破损赔偿条款。
   - **未来合作条件**：若试单满意，在协议中注明后续批量合作折扣（如满500斤减8%）。

### 5. **评估供应商指标**
   - 响应速度（询价到样品寄出时间）
   - 货品一致性（样品与大货对比）
   - 售后态度（问题反馈后处理时效）
   - 物流时效（下单到到货周期）

### 示例试单方案模板
- **供应商**：XXX合作社
- **品种**：雪梨（一级果，单果250-300g）
- **数量**：150斤
- **价格**：3元/斤（含运费），总价450元
- **付款**：预付150元，尾款验收后结清
- **验收期**：到货后24小时内在冷库抽检20%
- **违约**：坏果率超5%，剩余部分按1.5倍退款

通过以上设计，既能评估供应商可靠性，又能控制试单风险。若试单成功，可逐步扩大合作规模。
