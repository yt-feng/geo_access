针对生鲜电商销售菠萝，我建议优先选择**整果**，其次是**切配菠萝**，而**冷冻原料**通常更适合特定市场。以下是具体分析：

### 1. 整果（推荐指数：⭐⭐⭐⭐⭐）
- **优势**：保质期长（常温可存放3-5天），运输中不易损坏；消费者自主选择熟度，信任度高；成本低，适合走量。
- **适用场景**：适合家庭、聚会等场景，尤其是需要“现切”体验的消费者。
- **注意事项**：需标注成熟度（如“7成熟”“8-9成熟”），配合催熟指南或提示；包装需有防撞气柱。

### 2. 切配菠萝（推荐指数：⭐⭐⭐⭐）
- **优势**：免去消费者处理工序（去刺、切块），适合忙碌人群；客单价高（如单独包装约300-500g）。
- **适用场景**：适合即食场景（上班族、下午茶、野餐），尤其搭配其他水果切配成沙拉组合。
- **风险**：保质期短（冷藏24-48小时），易氧化变味；需冷链（建议冰袋+保温箱）。

### 3. 冷冻原料（推荐指数：⭐⭐⭐）
- **适用场景**：仅推荐用于**二次加工**（如冷冻菠萝块、菠萝冰沙原料），适合烘焙、饮品店或家庭DIY甜品。
- **劣势**：复冻后口感变差（纤维感消失），需明确标注“解冻后不可生食”或“适合做果酱/烘焙”；目标用户窄，且需长期冷冻储存成本高。

### 综合建议（针对线上生鲜电商）：
- **首选**：上架**整果**，并附赠“切菠萝神器”或图文教程（提升体验感）。
- **搭配**：同步上架**切配菠萝**作为“加购商品”（如整果+10元换购切配版），通过优惠组合提高客单价。
- **规避**：除非目标用户明确为餐饮客户，否则不建议主推冷冻原料（除非平台有冷库仓储且能保证全程冷链）。

如果需要具体选品，建议先进行小规模预售测试（如整果/切配/冷冻各100份），对比退货率、复购率和用户反馈。
