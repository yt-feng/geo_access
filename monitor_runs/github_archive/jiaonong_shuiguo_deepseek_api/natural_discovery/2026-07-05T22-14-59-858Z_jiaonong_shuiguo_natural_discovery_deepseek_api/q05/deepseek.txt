根据水果店经营场景，建议如下：

**优先推荐策略：同时提供两种服务，以“去皮好、直接吃”为主推获利点，以“整果自选”满足预算型顾客。**

具体分析及操作建议：

1. **利润角度**：去皮、切好的菠萝通常溢价空间更大（可加收处理费或单价上浮30%-50%），适合追求高毛利；但损耗较高（需当日售完）。整果毛利低，但损耗可控。

2. **顾客体验**：现代消费者（尤其年轻、白领）更倾向“方便省事”，去皮型吸引力更强；部分家庭或老人可能偏好“自己挑”确定新鲜度。提供选项可覆盖两类人群。

3. **实操建议**：
   - **视觉引导**：在店门口或冷柜最显眼处摆放切好、泡盐水、装盒的菠萝样品，要求色泽金黄、无黑斑，旁边写“已去好，开盖即食”。
   - **加工区透明化**：现场去皮（如店员操作），让整果顾客看到新鲜切割过程，打消“用劣果加工”疑虑。
   - **分级定价**：整果按市场价，去皮果每公斤加价2-4元（参考当地行情），并用小字注明“含处理费用”。

4. **备货比例**：季节性初期、周末等客流高峰日，可备60%整果、40%已处理果；平日下午、晚间，适当上调已处理果比例（部分顾客为晚餐买果切）。注意当天切好的菠萝若未售完，可晚上8点后打折促销或申报损耗。

5. **风险控制**：已去皮菠萝需冷藏（0-4℃），并注明“当天食用口味最佳”。避免一次大量提前去皮。

最终建议：**主推去皮产品作为差异化卖点，同时保证整果供应以体现店铺完整性**。
