作为夫妻店新手，试单的核心是 **“小批量、多频次、压风险”**，避免一次亏大。针对香蕉、苹果、橙子这三类常见且差异大的水果，具体操作建议如下：

### 一、总体原则
1. **不要看单价只看总价**：有的供货商报低价但搭售高损耗品（比如必须带一批次果），试单要明确单品、单规格、单价格。
2. **不留“寄售”口头账**：和供货商讲清楚“账期”是进货后多久结款，试单最好现结或一周内结，避免长期拖欠导致品控问题无法追责。
3. **每次试单记录“到货状态+销售速度”**：到货时间、果品外观、甜度、单价、3天内销售百分比，对比不同供应的表现。

### 二、香蕉试单要点（高风险、高周转）
- **测货量**：第一次不要超过 **1-2箱**（每箱约28斤）。香蕉熟得快，进货3天卖不掉就可能变黑。
- **到货时间**：要求早上6-8点到店。夏天必须带冷藏车或泡沫箱到货，避免无冷链捂熟。
- **验收重点**：只看 **“把型”和“把色”**—— 水果把（柄）要青色、果体黄色占40%以下最耐放；全黄的到货当天就得卖，不买。
- **试单建议**：让供货商先发一把给你（5-6根），不压资金。看2天销售情况，如果卖得快再要一箱。

### 三、苹果试单要点（耐储存、重规格）
- **测货量**：第一次 **1-2箱**（每箱约40-42斤）。苹果耐放，但不同产地口感差异大。
- **核心测试**：切开一个**压仓果**（箱子底部的）和**面果**（上层）尝。苹果常有“上大下小、上甜下涩”情况。要求供货商允许你抽底验货。
- **重点看**：**品种真假**（比如红富士、嘎啦、花牛要认果形和香味）、**表面蜡感**（天然果粉 vs 后期打蜡）、**硬度**（软了说明冷库放太久）。
- **试单建议**：要求供货商写明白品种（如“阿克苏冰糖心”），最好提供两个产地（如烟台、陕西）各半箱对比，看哪个在你店里好卖。

### 四、橙子试单要点（最易踩坑）
- **测货量**：第一次 **1-2筐**（每筐约30-40斤）。橙子水分和酸度是生命线。
- **验收重点**：**当场切开两个看籽和内膜**。部分供货商用“纽荷尔脐橙”冒充“赣南脐橙”，但切开看瓣数、籽多少、果肉颜色可鉴别。另外，**捏果屁股**——软塌塌的说明失水严重。
- **小心套路**：有的供货商给你看的大果，但实际发小果或次果。试单时要求**按个数或公斤数固定规格**，不要按“箱/筐”模糊计价。
- **时间敏感**：橙子采摘后10-15天口感最佳，超过30天会过酸或干水。问供货商“这批果什么时候下树的？”

### 五、与供货商谈判话术参考
- **破冰**：“老板，我们新开店，想试一下您的货。先拿小单，如果品质好、卖得快，后面长期上量。”
- **压条件**：“能不能先付一半钱？如果我这边卖不动，三天内退剩下的货可以吗？”（很多批发市场允许，尤其淡季）
- **要支持**：“第一次合作，您能不能给我写个**到货时间表**？我按时间验收，如果迟到2小时以上，每一箱便宜5块行吗？”（卡死到货时效）

### 六、风险预案
- **如果卖不动**：立刻**分装特价**（比如把苹果切好、香蕉切段）按份卖，或者送熟客尝鲜。**不要等全部坏掉再处理**。
- **换供应商标杆**：同一品种找两家供货商都拿一箱，对比哪种卖得快、复购率高，淘汰差的。
- **记录追踪**：用excel或本子记下：供货商名、下单时间、交货时间、单果大小（可量3个果的直径平均值）、销售天数。超过3次“到货差”或“销售慢”的，直接换人。

**最终建议**：第一次去批发市场，带块小电子秤（称单果重量）、一把水果刀（现场切看品种）、一个本子（记电话号码和价格），不要被“熟人介绍”忽悠，坚持自己验收每一箱。祝开张大吉！
