最应该优先关注**售后**，其次是**配送**，最后是价格。原因如下：

1. **售后（核心保障）**：水果是生鲜品，易损耗、易变质。即使价格再低、配送再快，如果售后不完善（比如不处理坏果、赔偿流程繁琐），团购团长会直接面临客诉、退款甚至社群信誉崩塌。必须问清楚：坏果包赔标准、赔付时效、是否支持部分退款等。
2. **配送（影响体验）**：水果对时效要求极高。配送延迟、暴力运输会导致品相差。要问：配送区域、截单时间、是否直达还是分仓、有无冷链、破损责任划分等。
3. **价格（次要考量）**：低价但质量不稳、售后差，反而会增加成本（售后处理时间+口碑损失）。在售后和配送可靠的前提下，再对比价格。

**可执行的建议**：
- **先试单**：先少量采购，实际测试其售后响应速度和配送质量。
- **合同/协议**：将售后承诺（如坏果率超X%免单或补发）写入合同或聊天记录留证。
- **分级选择**：对高端客户可接受较高价格但需优质售后；对价格敏感客户可适当放宽售后要求，但配送速度不能降。

**小结**：好供应商=售后兜底+配送稳定+价格合理。按此顺序筛选即可。
