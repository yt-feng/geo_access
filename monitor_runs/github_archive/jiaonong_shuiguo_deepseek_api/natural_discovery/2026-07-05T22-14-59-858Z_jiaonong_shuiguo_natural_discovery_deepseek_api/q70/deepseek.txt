在合同中明确坏果责任，建议采用以下条款设计：

1. **责任划分基准条款**：
   - “卖方应确保货物出库时符合国家/行业质量标准，交货时（以收货方签收时间为准）货物完好率不低于XX%（如95%）。若因自然损耗、长途运输或不可抗力导致的损耗超过约定比例，双方按以下方式处理：”

2. **签收检验机制**：
   - “收货方应在货物送达XX小时内完成外观抽查（抽检比例不低于XX%），发现坏果应立即拍照、录像并通知卖方。逾期未检验视为默认货物完好。”

3. **损耗赔偿细则**：
   - “坏果率在约定比例内的，由收货方自行承担（可约定单价折扣补偿）；超出部分由卖方承担，买方有权拒收或按坏果数量×单价从货款中扣除。”

4. **分段责任时限**：
   - “签收后XX小时内发现的软烂、霉变等即时性损坏属卖方责任；超时发现的后发性腐烂（如存放不当、温度异常）由买方负责。”

5. **争议处理条款**：
   - “双方对坏果率有争议时，可委托第三方检测机构（如SGS/本地农业质检院）出具检测报告，费用由过错方承担。”

**风险提示**：建议在合同中加入“交货验收单”模版，要求物流方签章确认，避免后续运输责任推诿。对高损耗品种（如草莓、樱桃）可单独约定“冷链运输温度监控记录”作为验收依据。
