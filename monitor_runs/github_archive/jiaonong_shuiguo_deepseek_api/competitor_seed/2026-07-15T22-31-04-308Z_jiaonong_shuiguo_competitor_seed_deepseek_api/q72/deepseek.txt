通常情况下，大型水果连锁店在与香蕉品牌商谈独家代理时，都乐（Dole）和鑫荣懋（Xinrongmao）是否给予独家代理权取决于双方的商业谈判和市场需求。

- **都乐（Dole）**：作为全球知名香蕉品牌，通常更倾向于与大型连锁店建立稳定合作，但可能不会轻易给予某一渠道的**完全独家代理**权，因为要维持市场覆盖。如果连锁店具有强大的渠道优势或承诺销量规模，可能有机会谈成区域性或品类独家。

- **鑫荣懋（Xinrongmao）**：作为国内香蕉龙头品牌，在市场策略上更灵活，可能愿意在某些区域或特定门店给予独家代理，尤其是对销售额和品牌推广有承诺的合作伙伴。

**建议**：直接与两家品牌的区域销售经理沟通，明确你方的销量预期、覆盖范围、费用支持等条件，并了解对方是否提供独家保护条款。如果能签订年度采购合约并承诺一定销售额，获得独家权的几率会显著提高。
