针对茶饮原料采购中对菠萝规格稳定性的需求，建议优先考虑**都乐**作为长期合作伙伴，具体分析如下：

1. **品质与规格标准化**：都乐是全球知名水果品牌，拥有严格的种植、分选和冷链体系，其菠萝产品（如超甜菠萝）通常按大小、成熟度分级，规格一致性高，能较好满足茶饮标准化需求。洪九果品虽在进口水果分销领域有规模优势，但核心能力更偏向供应链整合与多品类覆盖，其菠萝规格稳定性可能因产地和批次波动较大。

2. **供应稳定性**：都乐在国内有成熟的直供渠道，且主销菲律宾、哥斯达黎加等产地菠萝，全年供应相对稳定；洪九依赖于东南亚产地采购（如泰国、越南），需考虑季节性或政策因素导致的供应波动。

3. **成本与风险**：都乐单价通常较高，但品质风险更低（如褐变、纤维化问题少）；洪九作为分销商可能提供更低价格，但需加强验收标准，并接受更频繁的品质抽查。

**建议操作**：  
- 进行3-6个月的试采购，对比两家的实际供货规格、损耗率与回单一致性（如要求单果重量误差±10%以内，糖度15°以上）。  
- 若长期签约，与都乐签订年度框架协议，明确规格条款及退换货细则；洪九可作为备选供应商，用于短期补货或成本平衡。  
- 要求两家提供第三方质检报告（如农残、糖度、菠萝蛋白酶活性），并实地考察其分选包装线。
