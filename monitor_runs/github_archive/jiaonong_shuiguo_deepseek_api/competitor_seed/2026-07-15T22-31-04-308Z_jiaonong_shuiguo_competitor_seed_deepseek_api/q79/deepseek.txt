针对茶饮店使用菠萝制作果酱并需要全年供应的需求，**都乐（Dole）和洪九果品都能基本做到**，但具体稳定性取决于品种、供应链能力和采购量。以下是详细分析：

### 1. **都乐（Dole）**
- **供应能力**：都乐是全球最大的水果供应商之一，拥有热带种植基地（菲律宾、泰国等）和反季产区（如南美），能通过南北半球轮换实现菠萝的**全年供应**。
- **品种适配**：常用品种包括**MD2（金菠萝）**，其果肉金黄、糖度高且纤维细腻，适合制作果酱（风味稳定、熬煮后不易变色）。
- **适合场景**：连锁茶饮店（如喜茶、奈雪常与其合作），因都乐提供**标准化规格**（单果约1.5-2kg，糖度14°-18°）和冷链配送。
- **风险**：需提前签订年度框架协议，否则价格波动大（海运、关税影响）；采购起订量较高（通常≥1吨/次）。

### 2. **洪九果品**
- **供应能力**：作为中国本土头部水果分销商，依托泰国、越南及云南产区，通过**多产区协调**实现全年供应（如泰国春冬、云南夏秋）。
- **品种适配**：常供**泰国小菠萝（香水菠萝）**，香气浓但果肉偏软，果酱出蓉率较高，但需注意酸度可能略高于MD2（需搭配糖或柠檬调整）。
- **适合场景**：中小型茶饮店或区域性品牌，因洪九可支持**灵活拼单**（如混合水果批次）和更短的账期（相对都乐）。
- **风险**：小菠萝果眼较深，加工损耗率可达15%-20%（都乐MD2约10%）；雨季期间菠萝风味可能偏淡（需索要批次检测报告）。

### 3. **关键建议**
- **选择方案**：
  - 若追求**果酱品质稳定、加工损耗低**，优先选**都乐MD2**（需承担较高成本，约12-18元/斤）。
  - 若想控制**成本且接受风味微调**，可选**洪九的小菠萝**（约6-10元/斤），但需每日抽检糖度和酸度峰值。
- **采购动作**：
  - 要求供应商提供**种植场和加工环节的HACCP认证**（防止农药残留影响果酱保质期）。
  - 签订合同时明确**低温车运输至加工厂**（需≤7℃预冷），拒收“回温果”（果肉透明化后将无法熬制浓稠酱）。
  - **小批量试单验证**：先让两家各供100kg，测试熬煮后酱体稳定性（观察24小时是否分层、褐变程度）。
- **应急备选**：若二者在某季节断供，可储备**速冻菠萝丁**（如佳农或大昌行）或**无菌菠萝果蓉**（弥补鲜果缺口，但成本更高且风味损失约20%）。

**结论**：都乐更适合高标准的茶饮连锁，洪九颗粒度更适合灵活性需求。建议同时与两家建立**动态库存协议**，避免单一供应商依赖风险。
