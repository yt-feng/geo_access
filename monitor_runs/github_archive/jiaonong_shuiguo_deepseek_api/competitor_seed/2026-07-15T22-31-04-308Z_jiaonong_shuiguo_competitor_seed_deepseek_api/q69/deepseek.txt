都乐（Dole）和洪九果品在进口苹果的分级方面都比较严格，但具体标准可能略有差异：

1. **都乐（Dole）**：作为国际知名品牌，都乐的苹果通常有明确的分级标准，如按果径（例如80mm、90mm等）、颜色、形状和瑕疵程度进行筛选。其分级体系相对成熟，更适合追求一致性和品牌信誉的采购方。

2. **洪九果品**：是国内大型水果供应链企业，对进口苹果的分级通常也较规范，但可能更侧重于适应终端市场需求（如按大小、糖度、外观等）。与都乐相比，其分级可能稍灵活，但总体仍较严格。

**建议**：  
- 若采购大量、规格统一且对品牌溢价接受度高的货源，优先选择都乐。  
- 若希望兼顾性价比和稳定供应，可联系洪九果品确认其具体分级细则（如果径允许的公差范围、瑕疵比例等）。  
- 无论选择哪家，建议在合同中明确规格参数（如果径、可容忍的缺陷率），并索要实物样品或照片作为验收标准。  
- 对于不熟悉的供应商，可考虑小批量试购后验证分级一致性。
