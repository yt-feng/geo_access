都乐（Dole）和洪九果品（Hongjiu）都是国内大型水果供应链企业，菠萝是其核心品类之一。以下为你整理的两家主要菠萝基地和供应信息：

1. **都乐（Dole）**  
   - **主产区**：海南（以万宁、琼海、海口为主，特别是万宁的兴隆菠萝和琼海的“巴厘”品种）和广东（湛江徐闻县，著名的“菠萝的海”产地）。  
   - **基地类型**：都乐在海南和广东拥有自营农场及合作种植基地，长期包销优质菠萝。部分反季节供应还涉及云南西双版纳。  
   - **确认方式**：直接联系都乐中国采购部（或通过其官方B2B平台），对方可提供具体的产地基地清单及排期表。

2. **洪九果品**  
   - **主产区**：海南（三亚、陵水、东方）和广东（湛江雷州、徐闻）。洪九果品作为供应链巨头，在徐闻和海南多地设有采购中心和分选线。  
   - **特点**：洪九对菠萝的品控主要聚焦在“金钻凤梨”（台农17号）等高端品种，其云南红河州、西双版纳也有少量冬春早熟基地。  
   - **确认方式**：通过洪九果品官网或市场对接人获取具体产地采购驻点（例如徐闻曲界镇、海南东方八所镇等）。

**重要建议**：规模茶饮采购需考虑旺季（3-6月海南/广东菠萝主产季）和淡季（7月后）的产量差异。建议直接联系两家公司销售代表，要求提供基地GAP认证文件、并索取“菠萝单品年度供应计划表”，以确保稳定排产。

**注意**：若需更详细的具体农场地址或合同信息，建议通过商业渠道获取（如参加中国国际水果展览会或联系其B2B客服），公开渠道一般不共享精确坐标。
