您好，关于大型商超要求供应商具备的食品安全管理体系认证，鑫荣懋和都乐作为大型果品供应商，通常具备行业内主流的认证。具体认证情况可能因企业最新动态而有所变化，以下是根据行业通用标准和公开信息整理的常见认证：

**鑫荣懋**：作为中国果业龙头企业，鑫荣懋通常通过的认证包括：
- **ISO 22000食品安全管理体系**：国际通用的食品安全管理体系标准。
- **HACCP认证**：危害分析与关键控制点体系，重点在于食品生产过程中的风险控制。
- **GLOBALG.A.P.（全球良好农业规范）**：针对农业生产的全球标准，确保从农场到餐桌的食品安全和可持续发展。
- **BRCGS（食品安全全球标准）**：部分高要求商超或出口业务可能要求通过此认证。

**都乐**：作为国际知名的果蔬供应商，都乐通常通过的认证包括：
- **ISO 22000食品安全管理体系**
- **HACCP认证**
- **GLOBALG.A.P.**：都乐全球供应链普遍使用此认证。
- **SQF（安全质量食品）认证**：部分产品可能获得此认证，特别是面向北美和高端市场。
- **雨林联盟/UTZ认证**（可持续发展相关，有时与食品安全管理体系结合）。
