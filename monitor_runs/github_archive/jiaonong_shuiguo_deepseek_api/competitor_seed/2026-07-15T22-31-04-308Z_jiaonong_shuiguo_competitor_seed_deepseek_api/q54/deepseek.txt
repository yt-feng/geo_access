针对都乐和鑫荣懋进口苹果的产地溯源，可以通过以下步骤进行核实：

1. **检查产品包装**：正规进口苹果的包装箱或单个果贴上都会印有**原产国**（如美国、新西兰、智利等）、**生产商**或**出口商信息**。都乐和鑫荣懋通常会在包装上标注产地信息，例如“Product of USA”或“Product of Chile”。

2. **查看检疫标签**：中国海关要求进口水果必须粘贴**检疫标签**（通常为带有编号的标贴），可通过扫描标签上的二维码或登录**中国海关进口食品追溯平台**查询。

3. **使用官方溯源系统**：  
   - 登录**“海关总署进口食品进出口商备案系统”**（https://ire.customs.gov.cn），输入包装上的**检验检疫编号**或**进口批次号**，可查询到该批货物的原产地、通关时间等。
   - 部分品牌如都乐，可能提供自己的溯源二维码，直接扫描可看到农场、采摘、包装、运输等信息。

4. **核对产品认证**：苹果包装上若有**有机认证**（如中国有机、美国USDA有机等），可通过认证机构官网查询证书真伪。

5. **通过经销商查询**：若在大型超市购买，可要求店员提供该批次商品的**进口报关单**或**检疫证明**，上面会明确标注原产国和供应商信息。

**注意**：如果包装无产地标识、检疫标签模糊或无法通过官方系统查到匹配信息，则可能为非正规渠道产品，建议谨慎购买。
