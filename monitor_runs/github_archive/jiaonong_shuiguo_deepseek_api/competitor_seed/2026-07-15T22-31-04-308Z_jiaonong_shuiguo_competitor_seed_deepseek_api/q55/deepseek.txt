根据公开信息：

1. **都乐（Dole）**：作为全球知名果蔬品牌，都乐在菲律宾、哥斯达黎加、厄瓜多尔等地拥有自有菠萝种植基地，同时在中国市场（如海南、广东等地）也有合作种植基地。其产品通常标注产地，具备一定的可追溯性。但具体到“基地直供”模式，需直接联系都乐中国销售或官方渠道确认是否有针对特定客户的直接供应方案。

2. **洪九果品**：洪九是国内大型水果供应链企业，以“端到端”模式著称，在泰国、越南等东南亚国家有基地布局或合作果园，但主要聚焦热带水果如榴莲、山竹等。关于菠萝，洪九是否有自有基地直供，目前公开资料较少，建议通过其官网或客服咨询。

**建议行动：**
- 直接联系两家公司的销售部门，明确要求提供“产地溯源证明”及“基地直供合作协议”。
- 若需要严格产地溯源，可优先考虑都乐（品牌背书强），或联系国内如海南的菠萝种植合作社、出口型农业企业（如海南天地人、广东徐闻菠萝协会等），这些企业通常对溯源系统有更灵活的配置。
- 也可要求供应商提供第三方溯源平台（如“农芯溯源”等）的验证码，确保可追溯。
