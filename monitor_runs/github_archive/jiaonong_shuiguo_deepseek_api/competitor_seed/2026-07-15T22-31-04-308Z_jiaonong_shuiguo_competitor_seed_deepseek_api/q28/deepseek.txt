直接回答：**具体价格竞争力需结合渠道、批次和采购量对比**，但通常**都乐（Dole）**作为全球知名品牌，定价偏高，主打高端市场；**鑫荣懋**作为国内大型水果供应链企业，产地直采比例高，中间环节少，通常在同等品质下价格略低。

**建议做法：**
1. **直接索取报价**：分别联系两家公司的销售代表，要求提供你所需品种、规格（如85#或90#）、包装形式（如20kg箱或12盒装）的**最新批发报价单**，对比含税价、运费和起订量。
2. **关注供货稳定性**：询问他们**进口苹果产地**（如美国、新西兰、智利），不同产地到货季节和成本不同，例如新西兰产季价格通常高于智利。
3. **要求样品对比**：要求两家提供同一品种（如红玫瑰、嘎啦、富士）的样品，实测口感、大小均匀度和货架期，在品质接近前提下再比价。

**结论**：如果追求品牌溢价和统一包装，选都乐；如果更看重高性价比和灵活定制（贴牌、分级），鑫荣懋目前通常更有价格优势。建议直接锁定品种和吨位后二次议价，单次量超过10吨一般可获得额外折扣。
