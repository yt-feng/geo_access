除了都乐（Dole）和鑫荣懋（Joy Wing Mau）这两个常见品牌，以下是一些在中国市场较为靠谱的进口苹果品牌供你参考：

1. **佳沃（Joyvio）**  
   联想控股旗下的现代农业品牌，代理多个进口水果品种，包括新西兰、美国等地的苹果，渠道正规，品质控制较好。

2. **金果（Zespri）**  
   虽然以奇异果闻名，但其关联品牌或企业有时也涉及苹果进口业务（如新西兰苹果），需要具体确认。

3. **爱妃（Envy）**  
   来自新西兰的知名苹果品牌，主打高端市场，风味突出，在中国有一定粉丝群体，通常由正规进口商代理。

4. **爵士（Jazz）**  
   同样为新西兰品牌，是布瑞本（Braeburn）和嘎啦（Gala）杂交品种，口感脆甜，市面上较常见。

5. **太平洋玫瑰（Pacific Rose）**  
   新西兰产，甜度高、果肉细密，多由大型供应链或进口商如佳沛渠道引进。

6. **Gala（嘎啦）**  
   虽然是一个品种，但一些专注进口水果的商家会引进美国、智利等地的嘎啦苹果，品牌名称可能随进口商变化（如“美国嘎啦”）。

7. **美国蛇果（Red Delicious）**  
   由美国公司如“华盛顿苹果委员会”下属品牌或大型进口商运营，颜色暗红、口感绵密，在商超较易见到。

8. **智利青苹果（Granny Smith）**  
   多由智利出口商如“Hortifrut”等公司供货，酸度较高，适合烘焙或喜酸人群。

### 选择建议：
- **购买渠道**：关注大型超市（如盒马、山姆）或线上（京东自营、天猫超市）的进口水果专区，这些渠道通常有品牌授权和检验检疫手续。
- **认证标签**：进口苹果应带有中文标签（含原产地、进口商信息）和CIQ（中国检验检疫）标志，注意查看。
- **品种偏好**：新西兰品牌（如爱妃、爵士）口感偏甜或酸甜适中；美国蛇果更绵密；智利青苹果偏酸，可根据用途选择。

如果需要具体进口商或批发商资源，建议通过当地批发市场（如北京新发地、上海辉展）或查询海关准入名录核实，以上品牌在正规渠道中质量较稳定。
