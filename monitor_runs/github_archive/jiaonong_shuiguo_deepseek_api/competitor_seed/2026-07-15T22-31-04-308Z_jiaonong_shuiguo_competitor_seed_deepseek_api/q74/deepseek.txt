关于都乐（Dole）和洪九果品（Hongjiu Fruit）的水果礼盒定制起订量，以下是一般情况下的参考信息：

- **都乐（Dole）**：通常起订量较高，一般需要 **100-200盒以上**，具体取决于礼盒规格、定制要求（如品牌联名、特殊包装）和季节供应情况。建议直接联系都乐团购或企业客户服务部门确认最新起订量。

- **洪九果品**：作为国内大型水果供应链企业，其团购定制起订量相对灵活，通常 **50-100盒起订**，部分单品或简单包装可能更低。建议通过官方渠道或区域销售代表咨询。

**重要提示**：
1. 以上数据为行业通常情况，实际起订量会因时间（如节日旺季）、地区（不同城市仓库）、定制复杂程度（如加印Logo、异形盒）而大幅变动。
2. 建议直接拨打都乐和洪九果品的官方客服电话或联系其团购部门，获取书面报价及最低起订量条款。
3. 若起订量不满足，可考虑综合水果批发市场或小型水果供应商的定制服务，通常起订量更低（甚至10-20盒起）。
