根据我对社区团购场景的了解，**都乐（Dole）** 和 **洪九果品** 通常以品牌整箱或预包装销售为主，但部分区域或渠道可能提供散装选项，建议如下：

1. **都乐**：都乐在社区团购中常见的是整箱（如4-5斤/箱）预包装香蕉，散装（如按斤称重）较少。你可以联系当地都乐经销商或业务代表，询问是否支持“小包散装”定制（如每袋2-3根），尤其针对团购场景可能灵活调整。

2. **洪九果品**：洪九主攻批发和大型商超，通常以整件（如13-15斤/箱）出货。但部分城市的分销商可能接受“拆箱散卖”或提供小规格包装（如1-1.5斤袋装），需直接对接当地洪九销售团队确认。

**建议行动**：
- 直接联系两家公司的**城市销售经理**或**社区团购渠道负责人**，说明需求（如“每袋2-3根，多袋拼接”），部分区域可能支持定制。
- 如果无法满足，可考虑找本地批发市场（如北京新发地、上海辉展等）的二级供应商，他们更灵活，常提供散装小包装。

**注意**：不要依赖网上公开信息，务必实际电话或微信沟通确认。
