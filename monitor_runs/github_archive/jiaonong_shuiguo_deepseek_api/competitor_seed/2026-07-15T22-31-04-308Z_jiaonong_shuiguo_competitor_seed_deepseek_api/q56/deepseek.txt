都乐（Dole）和鑫荣懋（Xinrongmao）在生鲜电商平台上的品牌知名度方面，**都乐总体更高**，尤其是在全国性和消费者认知层面。具体对比如下：

1. **都乐**：国际品牌，历史悠久（1851年创立），在香蕉品类中全球知名度极高，常年占据高端水果市场。在中国市场，都乐通过大型商超、电商平台和广告投放，积累了较高的消费者辨识度，尤其在进口香蕉领域，常被视为品质标杆。
2. **鑫荣懋**：中国本土头部水果企业，在产业链整合和B端渠道（如供应链服务）上实力强劲，但C端品牌知名度相对都乐偏低。其香蕉产品多以性价比和国产化定位为主，在特定区域（如华南）或电商细分渠道（如社区团购）有一定影响力，但全国性品牌曝光不如都乐。
