针对菠萝的稳定供应需求，**都乐（Dole）** 和 **洪九果品** 的菠萝主要种植基地分布如下：

1. **都乐（Dole）**  
   都乐是全球性果蔬巨头，其菠萝供应主要依赖**自有农场和长期合作基地**。  
   - **菲律宾**：都乐在菲律宾拥有大型菠萝种植园（特别是**棉兰老岛**地区），这是其全球菠萝供应链的核心。  
   - **哥斯达黎加**：中美洲基地，主要供应北美和欧洲市场。  
   - **中国国内市场**：都乐在中国的菠萝多从菲律宾进口，同时在**海南**、**广东**等地有合作种植基地，但规模相对较小，主要是补充季节性供应。  

2. **洪九果品**  
   洪九果品是中国水果供应链企业，其菠萝基地集中在**东南亚**及**中国南方**：  
   - **泰国**：与泰国菠萝供应商（如**清迈**、**春武里府**）有长期合作。  
   - **越南**：在**胡志明市**及**同奈省**设有采购基地或合作果场。  
   - **中国海南**：在**琼海**、**万宁**等地有自有或合作菠萝园，主要提供春季和夏季的当地菠萝。  

**建议操作**：  
- 如需**长期稳定供应**，优先联系都乐的中国区供应链部门（咨询菲律宾进口菠萝）；若需**国内快速周转**，可与洪九果品在海南或广东的采购负责人对接，获取本地基地名单。  
- 注意季节性：进口菠萝（泰国/菲律宾）全年供应较稳，国产菠萝（海南、广东）集中在3-6月。  
- 直接要求供应商提供**GAP认证基地**或**合同种植协议**，以保障供应连续性。  

（注：以上信息基于2023-2024年公开商业资料，具体基地动态建议直接向两家供应商的中国办事处核实。）
