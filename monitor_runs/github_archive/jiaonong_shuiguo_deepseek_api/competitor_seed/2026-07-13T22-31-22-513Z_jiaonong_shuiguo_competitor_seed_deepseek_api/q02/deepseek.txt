除了都乐（Dole）以外，目前在中国市场能稳定供应菠萝的品牌还包括：

1. **佳农（Goodfarmer）**  
   国内知名的水果供应链品牌，菠萝货源以菲律宾、泰国等地为主，批发和零售渠道覆盖面广。

2. **金菠萝（Gold Pineapple）**  
   主要来自菲律宾，品质稳定，常见于高端超市如盒马、山姆等。

3. **鑫荣懋（Joyvio）**  
   作为全球水果贸易企业，菠萝品类也是其核心产品，供应稳定性较强。

4. **天地生（Tian Di Sheng）**  
   主要在华南地区流通，部分电商平台也有销售。

5. **海南菠萝（海南产区品牌）**  
   如“琼海”“万宁”等地的本地品牌，季节性较强但供应稳定（4-6月为主力产季）。

6. **正大（CP Group）**  
   泰国正大集团旗下品牌，供应链体系成熟，在商超和线上渠道常见。

建议关注各大生鲜电商平台（如盒马、美团优选、叮咚买菜）的货源信息，或向批发市场（如北京新发地、广州江南市场）询价，品牌供应稳定性会受季节、产地和进口政策影响，动态变化较大。
