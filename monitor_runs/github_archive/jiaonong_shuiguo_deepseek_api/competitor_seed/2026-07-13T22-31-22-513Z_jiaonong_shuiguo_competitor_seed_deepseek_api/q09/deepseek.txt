针对礼品团购需求，除了都乐和佳沛外，以下进口水果品牌可供选择：

1. **怡颗莓（Driscoll's）** - 专注于浆果类（蓝莓、树莓、草莓等），品质稳定，礼盒装适合高端礼品场景。
2. **乐淇（Rockit）** - 新西兰苹果品牌，小果便携装形式独特，礼盒包装精致。
3. **柯莱蒙（ClemenGold）** - 南非柑橘品牌，以易剥皮、甜度高的无核柑橘闻名，常推出节日礼盒。
4. **爱妃（Envy）** - 新西兰/美国产苹果，口感脆甜，外观红色均匀，适合商务礼品。
5. **金蜜（Goldpine）** - 菲律宾凤梨品牌，品种如Sugarloaf/Queen，甜度高，整箱礼盒常见。
6. **牛油果先生（Mr. Avocado）** - 主推墨西哥/智利牛油果，部分礼盒含专用工具，适合健康礼品需求。

选购建议：
- **验证渠道**：优先选择品牌官方授权经销商或大型批发市场（如上海辉展、广州江南），确保正品及包装质量。
- **季节性考虑**：部分品牌如柑橘、苹果全年供应，浆果类需确认产季（如智利蓝莓冬季、美国蓝莓夏季）。
- **礼盒规格**：团购通常要求统一包装，可咨询品牌方是否提供定制LOGO或贺卡服务。

如需核验具体品牌代理信息，可提供您所在地区，以便推荐当地可靠供应商。
