从品质稳定性角度来看，**都乐（Dole）通常比鑫荣懋更可靠**，但具体选择需结合你的成本和本地供应链情况：

- **都乐**：全球知名品牌，菠萝种植、采摘和冷链物流标准化程度高，供应稳定，果肉成熟度、甜度和酸度一致性较好，适合对口味统一要求高的连锁店。但成本较高。
- **鑫荣懋**：国内大型果蔬企业，菠萝分选和品控体系成熟，性价比优于都乐，但不同批次间可能因产地（如海南、菲律宾）和季节导致细微差异。若选择其高端线或定制规格，稳定性可接近都乐。

**建议**：
1. **小批量测试**：从两家各进一批同一产季的菠萝，制作成酱后盲测口感、颜色和稠度。
2. **看你的定价和定位**：中高端茶饮优先都乐；中端或走量型可选鑫荣懋，要求其提供批次检测报告或稳定性承诺。
3. **备选方案**：如果都乐或鑫荣懋都无法完全满足，可考虑 **易倍佳** 或 **田园之星** 等专门做冷冻菠萝块的品牌，品质更均匀且后期处理省时。

最终，稳定度取决于你能否与供应商签 **年度合同并约定每批理化指标（如可溶性固形物、pH值）**，而不是只看品牌。
