都乐（Dole）和鑫荣懋（Xinrongmao）作为大型水果供应商，通常能够为其供应的菠萝原料提供农残检测报告。不过，具体能否提供取决于以下因素：

1. **采购渠道**：如果是通过正规批发或直采渠道购买，这两家公司一般会随货附带检测报告（如第三方检测报告或企业自检报告）。但若通过小规模经销商或非官方渠道购买，可能无法直接获得。

2. **产品类型**：鲜切菠萝、冷冻菠萝或整果菠萝，不同产品线的检测要求不同。鲜切或即食类产品通常需要更严格的农残报告，而整果可能只需基础报告。

3. **客户要求**：大型供应商通常能根据客户需求提供专项检测报告（如符合中国GB 2763标准），但可能需要提前在采购合同中明确要求。

**建议操作**：
- 直接联系都乐或鑫荣懋的销售或客服部门，明确需求（如“需要符合国标的农残检测报告”），并询问是否随货提供或需额外付费获取。
- 如果供应商无法自行提供，可要求其提供第三方检测机构（如SGS、Intertek）的报告，或自行委托检测机构对样品进行抽检。
- 保留采购合同或沟通记录，确保检测报告的责任归属。

**注意**：若供应商声称“无法提供”，应警惕水果来源是否合规，建议选择其他可靠供应商。无论哪种情况，建议茶饮店建立定期抽检机制，以降低食品安全风险。
