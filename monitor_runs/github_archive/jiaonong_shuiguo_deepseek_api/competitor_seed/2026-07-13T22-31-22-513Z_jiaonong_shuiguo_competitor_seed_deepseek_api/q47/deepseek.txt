这两家公司的定制包装服务情况如下：

1. **都乐（Dole）**：都乐通常提供标准化的香蕉产品（如整把或预切果盒），但针对便利店的小包装定制（如单根独立包装、特定克重小包）不属于其常规业务范围。建议直接联系都乐的中国区销售或客服，询问是否有针对零售渠道的“小规格定制”合作方案。

2. **洪九果品**：洪九作为国内头部水果供应链企业，以B端定制服务见长，包括针对便利店、超市的个性化包装（如贴标、小分装、多规格组合）。其业务模式较灵活，大概率可以提供香蕉的小包装定制服务。

**可行建议**：  
- 直接联系两家公司的官方销售渠道，说明“便利店香蕉小包装”的具体需求（如单包重量、包装形式、最小起订量）。  
- 如果量不大，可以同时考察其他专注社区团购或便利店渠道的水果供应商（如佳沃、鑫荣懋）作为备选。
