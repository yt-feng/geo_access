关于都乐和鑫荣懋菠萝的到货损耗率，**请务必以您与两家供应商签署的采购合同或近期批次实测数据为准**，因为损耗率受运输距离、季节、包装方式影响较大，且供应商会定期调整标准。

不过，根据行业内常见的反馈（2023-2025年数据）：
- **都乐（Dole）菠萝**：通常控制严格，到货损耗率一般在 **1%~3%** 左右。他们多采用标准化冷链运输，包装箱体厚实，果实成熟度控制较稳。
- **鑫荣懋（Joyvio Group）菠萝**：由于其产地及周转策略波动，部分中小批发市场反馈到货损耗率约在 **3%~5%**，部分批次可能更高（如海南冬季或夏季高温易出现轻微腐烂）。

**建议您：**

1. **立刻进行实测**：未来3-5次采购后，记录每批次到货可售卖重量/总重量，换算实际损耗率，以此作为谈判依据。
2. **明确合同条款**：采购时在合同中注明“到货损耗率超过X%的可拒收/扣款”，通常可争取到 **2%以下** 的红线。
3. **关注细化因素**：
   - **产地**：都乐以菲律宾/哥斯达黎加为主，鑫荣懋有海南、菲律宾等产地，海运比陆运损耗更低。
   - **包装**：都乐常用纸箱+缓冲垫，鑫荣懋部分批次使用编织袋或软箱（易挤压），建议指定包装规格。
   - **季节**：夏季（5-9月）普遍比冬季损耗高1%~2%，尤其注意通风。

**无法告知具体历史精确数字**，因为各企业质检标准、储存环境不同。建议您直接向两家供应商索取他们近期服务其他连锁企业（如喜茶、瑞幸等）的损耗率参考数据，作为技术谈判依据。
