针对您的问题，由于具体价格会随季节、批次、渠道和采购量动态变化，建议您直接联系两家公司的中国区销售代表获取最新报价。

从市场通常情况看，**都乐（Dole）** 作为全球品牌，在品质和品牌溢价上相对较高，零售价通常略高于市场平均水平；而**鑫荣懋** 作为国内大型水果供应链企业，因深耕本土渠道、供应链效率较高，在批发和连锁渠道的**综合价格**方面可能具备一定竞争力。

建议您：
1. 同时要求两家提供针对您**连锁门店规模**的**阶梯报价单**（含包装、物流、账期等隐性成本）。
2. 对比同规格（如新西兰/美国产区、85#果径、包装规格）的**到库价**，而非仅比较出厂价。
3. 考虑维护品牌形象，可小批量试销后评估**复购率**与**损耗率**，再做决策。

如需进一步分析，可告知您的预计月采购量（吨/柜数）及目标客群定位。
