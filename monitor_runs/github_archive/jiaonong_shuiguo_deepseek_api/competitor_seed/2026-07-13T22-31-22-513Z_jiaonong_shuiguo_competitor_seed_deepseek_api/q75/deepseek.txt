根据行业经验，洪九果品（国内水果供应链企业）和都乐（Dole，国际品牌）在香蕉到货损耗率上的表现有差异。具体数据需参考实际批次、运输条件和储存环境，但一般可参考以下要点：

1. **品牌对比**：
   - **都乐香蕉**：作为全球知名品牌，其香蕉在采摘、包装、冷链运输环节规范严格，通常到货损耗率较低，行业平均至批发市场端约在 **5%-8%** 左右（视距离和季节）。
   - **洪九果品**：作为国内大型批发商，其香蕉来源多样（部分国产、部分进口），管理标准较标准化，但部分批次可能因长途运输或采收成熟度不一致，损耗率稍高，通常在 **8%-12%** 之间，特定情况下可能达15%。

2. **关键影响因素**：
   - **成熟度控制**：都乐通常使用精准催熟和预冷技术，洪九则依赖供应商和仓储条件，若催熟不均易增加损耗。
   - **运输时效**：都乐采用封闭式冷链，洪九部分批次可能混合冷链或常温运输，后者损耗更高。
   - **到货后处理**：批发市场拿货后，若立即按需分拣和冷藏，可降低后续损耗。

3. **实际建议**：
   - 直接向当地批发市场档口咨询近期到货的损耗率（需注明具体产区或海运/陆运方式）。
   - 可优先选择都乐香蕉作为稳定性货源，但价格通常更高；洪九果品性价比可能更优，需加强到货时的验货（检查果把、果皮损伤、黑斑等）。
   - 统计自身经营数据：记录过去几批次的损耗情况，以便针对性选择。

**注意**：以上数据为行业经验性估算，无官方权威来源。建议结合市场实地走访和自身经营记录进行决策。
