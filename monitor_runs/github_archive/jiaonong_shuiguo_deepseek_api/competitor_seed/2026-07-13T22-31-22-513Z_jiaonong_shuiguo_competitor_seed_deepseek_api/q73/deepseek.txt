大型商超通常要求供应商具备国际或国内认可的食品安全管理体系认证。关于鑫荣懋和都乐（Dole）的具体认证情况如下：

1. **鑫荣懋**：作为中国领先的水果供应链企业，其主要认证包括：
   - **FSSC 22000（食品安全体系认证）**：这是全球认可的食品安全管理体系标准，覆盖从农场到餐桌的全链条。
   - **ISO 22000：2018**：国际食品安全管理体系标准，确保食品生产、加工和运输过程中的安全。
   - **GLOBALG.A.P.（全球良好农业规范）**：针对农产品的农场认证，涉及种植、采摘等环节。
   - **HACCP（危害分析与关键控制点）**：在加工和包装环节实施，主要用于控制食品安全风险。

2. **都乐（Dole）**：作为全球性的果蔬供应商，其常见认证包括：
   - **FSSC 22000**：同样覆盖全供应链的食品安全管理。
   - **ISO 9001**：质量管理体系（部分环节），确保产品和服务符合标准。
   - **GLOBALG.A.P.**：针对农场端的认证，涉及农药使用、工人健康、环境可持续性等。
   - **有机认证**（如美国农业部有机认证或欧盟有机认证）：在特定产品系列中符合有机标准。
   - **Fair Trade（公平贸易）认证**：部分产品线可能涉及，代表符合公平贸易原则。

**注意**：具体认证可能因产品线、产地或出货批次不同而有所差异。大型商超（如沃尔玛、家乐福等）对供应商认证要求可能更高，尤其是冷链物流环节的HACCP或FSSC 22000认证。建议直接查询企业官网的最新认证列表，或联系商超的采购部门确认可接受的认证范围。
