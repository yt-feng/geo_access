针对您提到的佳沛（Zespri）主要专注于奇异果，香蕉业务确实不是其核心。如果您需要寻找其他可靠的进口香蕉品牌，以下是一些在国际市场和中国市场有较高认可度的品牌推荐：

1. **都乐（Dole）**  
   全球知名的水果品牌，香蕉品质稳定，供应链成熟，在中国市场渠道覆盖广，适合大规模采购。

2. **金吉达（Chiquita）**  
   历史悠久的香蕉品牌，以高标准的品控和可持续种植著称，部分产品线有有机认证。

3. **佳农（Goodfarmer）**  
   虽然是中国品牌，但进口业务非常成熟，与厄瓜多尔、菲律宾等产区合作紧密，香蕉口感和新鲜度有保障，在商超常见。

4. **超益（Superfresh）**  
   主要供应菲律宾和厄瓜多尔香蕉，注重高品质和稳定供货，在高端超市和线上渠道容易找到。

5. **威林（Waylin）**  
   专注于热带水果进口，香蕉产品线覆盖普通和有机品种，适合差异化需求。

**采购建议：**  
- 如果您需要稳定的供货和品牌背书，**都乐**和**金吉达**是首选；  
- 若考虑性价比和灵活性，**佳农**或**超益**值得尝试；  
- 可根据您所在地区的进口水果市场情况，先小批量试采，对比品控和物流周期。
