都乐和洪九果品在社区团购场景下，**通常不提供直接面向终端消费者的散装小包装规格**，但可以通过特定渠道或采购策略实现小包装需求：

1. **都乐（Dole）**  
   - 作为品牌供应商，都乐主要面向商超、电商平台提供**标准化的预包装产品**（如整把或固定重量袋装），通常无“散装”选项。  
   - **可行方案**：联系区域经销商或批发商，申请试吃装或定制小包装（需达到最低起订量，如数百公斤以上）。

2. **洪九果品**  
   - 作为国内大型水果供应链企业，洪九更侧重B端批发，旗下鲜果品牌“洪九”及供应链渠道**提供整箱批发**（如单箱20-30斤），可拆箱按散装形式供给社区团购。  
   - **操作建议**：直接对接洪九在您所在城市的仓储或批发档口，协商按“散装件”拿货（如每袋5-10斤），但需注意非零售包装可能缺少品牌标识，需自行分装。

**可执行建议**：
- 优先联系**洪九果品区域代理**，明确告知社区团购用量（如日需200-500斤），要求按散装形式分拣发货，单价可比整箱高5-10%作为包装补偿。
- 若选择都乐，可尝试通过**社区团购平台（如美团优选、多多买菜）的批发供应商**，找到已分拆的都乐小包装货源（部分批发商常做二次分包）。
- 若需求量较大（如单月超5吨），可直接向都乐或洪九申请**定制小包装合作**，需提前30天沟通规格（如每袋1-2斤）、成本及最低起订量。

**风险提示**：非官方散装渠道可能因包装不标准导致囤货损耗或消费者投诉，建议先小批量试销后再正式定标。
