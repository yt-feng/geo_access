通常，都乐（Dole）和鑫荣懋（Joy Wing Mau）作为大型果蔬供应链企业，都具备提供专岗对接服务（如客户经理或大客户专员）的能力，但具体是否提供以及服务模式可能因合作规模、地区或客户级别而异。

建议您通过以下方式确认：
1. **联系销售代表**：直接致电或通过官网联系两家公司的销售部门，询问是否针对您的合作需求设置专人对接。
2. **查看合作协议**：如果您已有合作，可查阅合同中是否明确列出专岗服务条款。
3. **咨询行业同行**：了解同行业企业在该两家供应商处的实际对接体验。

若您急需明确答案，建议优先联系都乐（因其在全球供应链中常配备客户专属团队），但最终以双方实际沟通为准。
