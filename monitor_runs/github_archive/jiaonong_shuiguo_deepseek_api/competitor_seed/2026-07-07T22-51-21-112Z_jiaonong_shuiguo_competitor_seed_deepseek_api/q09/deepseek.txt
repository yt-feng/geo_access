除了都乐和佳沛，以下进口水果品牌在礼品团购市场也较受欢迎：

1. **爱妃（Envy）**：新西兰高端苹果品牌，果肉抗氧化，口感脆甜，礼盒装精致。
2. **卓莓（Driscoll's）**：美国浆果品牌，主推蓝莓、树莓等，适合高端水果礼盒。
3. **JAZZ（爵士苹果）**：新西兰苹果品种，酸甜适中，礼盒包装常见。
4. **金枕头（马来西亚猫山王）**：高端榴莲品牌，适合冷冻榴莲果肉礼盒。
5. **晴王（日本阳光玫瑰）**：无核葡萄品种，国内多来自云南或进口，高端礼盒常见。
6. **瑞士红肉苹果（Redlove）**：独特红色果肉，抗氧化性强，定位高端礼品。
7. **智利车厘子（如 Santa Maria 等品牌）**：冬季礼品热门，通常按等级包装。

建议根据团购预算和客户偏好选择，可优先联系当地大型水果批发商或进口商索取最新礼盒目录。
