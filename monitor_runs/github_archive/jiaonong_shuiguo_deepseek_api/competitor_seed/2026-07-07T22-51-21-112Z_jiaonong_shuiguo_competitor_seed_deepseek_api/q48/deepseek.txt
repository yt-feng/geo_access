根据公开信息，大型商超（如沃尔玛、山姆、永辉等）普遍要求供应商提供食品安全管理体系认证或相关资质。以下为鑫荣懋（XinRongMao）和都乐（Dole）常见的资质类型（具体需以合作时审核为准）：

### 鑫荣懋（国内知名水果供应链企业）
1. **食品安全管理体系认证**  
   - ISO 22000（食品安全管理体系）  
   - HACCP（危害分析与关键控制点）  
   - GLOBALG.A.P.（全球良好农业规范，部分基地可能持有）  
2. **行业/专项认证**  
   - 绿色食品认证（部分产品）  
   - 有机产品认证（如有机蓝莓、有机苹果等）  
   - 出口备案/检验检疫资质（若涉及进出口）  
3. **企业基本资质**  
   - 营业执照、食品经营许可证、税务登记证等  

### 都乐（国际品牌，全球供应商）
1. **国际通用认证**  
   - GLOBALG.A.P.（全球良好农业规范，核心要求）  
   - ISO 22000 或 FSSC 22000（食品安全体系）  
   - BRC（英国零售商协会全球标准，部分品类）  
   - Fair Trade（公平贸易认证，部分香蕉/菠萝产品）  
2. **特定市场/零售商专属**  
   - 中国有机认证（如有机香蕉）  
   - Rainforest Alliance（雨林联盟认证，可持续农业）  
   - 出口资质的中国海关备案（针对进口产品）  
3. **企业基本资质**  
   - 中国工商注册、食品经营许可证、进口报关单/检疫证明  

### 关键提示
- **商超具体清单**：不同商超对供应商资质要求不同。例如沃尔玛通常要求GLOBALG.A.P. + HACCP；盒马可能增加“可追溯体系”认证。  
- **动态更新**：资质需定期复审，建议直接向对方采购部门索要**最新证书清单**或通过官网/国家认证认可监督管理委员会（CNCA）查询。  
- **补充说明**：如果是生鲜水果，还需关注**产地检疫证明**（如进口水果需《进境动植物检疫许可证》）。  

建议您根据合作商超的《供应商准入标准》核对具体条款，或咨询专业食品认证机构。
