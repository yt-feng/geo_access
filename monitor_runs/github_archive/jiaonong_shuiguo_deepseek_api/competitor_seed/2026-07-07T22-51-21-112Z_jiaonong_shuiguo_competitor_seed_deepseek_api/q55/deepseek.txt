都乐（Dole）和洪九果品（Hongjiu Fruit）确实有基地直供模式，但需根据具体产地确认。

**都乐**：作为全球知名果蔬品牌，拥有自有或合作基地，在菲律宾、哥斯达黎加等主要菠萝产区有直采体系。在中国市场，都乐通常通过旗下供应链直接提供溯源信息，部分产品可扫码追溯产地。建议联系其官方销售渠道确认特定批次的可溯源性。

**洪九果品**：是中国头部水果供应链企业，以“端到端”直采闻名，在泰国、越南等东南亚菠萝产区有直采基地，并宣称可提供全程溯源服务。其官网和公开资料显示，产品附带产地、加工等追溯信息。

**建议**：
1. 直接联系两家公司的采购或客服部门，明确要求“提供产地溯源证明”，并索取基地信息（如GPS坐标、认证文件）。
2. 若需要更严格的追溯（如区块链扫码），可尝试询问是否支持第三方溯源平台（如信联、蚂蚁链）。
3. 考虑中小规模、专做有机或地标产品的供应商（如海南波罗蜜直供基地），可能溯源透明度更高。

注意：溯源信息的详细程度可能因批次不同而异，务必在采购合同或协议中明确溯源条款。
