都乐（Dole）是全球最大的水果供应商之一，拥有完善的全球供应链体系，在中国也布局了多个产区（如广东、广西、海南等），理论上能够实现菠萝的全年供应。但需注意：都乐产品线以鲜果为主，若制作果酱需要工厂级大批量菠萝原料，可能需要与其签约批量采购或定制冻品原料合同。

洪九果品（Hongjiu Fruit）是中国领先的水果供应链企业，主要做进口水果（如泰国金枕头榴莲、越南菠萝等）。他们确实能提供全年菠萝（通过泰国、菲律宾等产地轮换），但需确认是否有专门针对果酱加工类的工业级菠萝供应服务（其零售端更偏向生鲜精品果）。

**可行建议**：
- 两家都具备主导全年供应的能力，建议**直接联系两家公司的B2B部门**（都乐工业原料部或洪九大宗采购部），要求提供“全年菠萝冻块/果泥样本”及价格表。
- 如果规模不大，可尝试与**广东、海南本地菠萝加工厂**（如佳顺、绿源等）签约，他们通常能提供全年稳定供应且价格更低（使用速冻菠萝或浓缩汁）。
- 引入**第三方仓储代采**：让供应链公司（如万纬冷链）代采并统一分装。

注意：制作果酱对菠萝成熟度（如糖酸比、果胶含量）有要求，建议要求供应商提供**符合QB/T 13861-2017《果蔬酱》标准**的工业级原料，而非零售级鲜果。
