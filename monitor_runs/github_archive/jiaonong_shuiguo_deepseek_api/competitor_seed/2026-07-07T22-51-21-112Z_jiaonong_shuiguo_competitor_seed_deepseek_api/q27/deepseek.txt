洪九果品和鑫荣懋在香蕉产地直发方面都有布局，但具体基地情况如下：

1. **洪九果品**  
   洪九果品是国内领先的水果供应链企业，在香蕉领域有较为深入的源头布局。其在云南、广西、海南等国内香蕉主产区拥有合作或自建的种植基地，同时也在东南亚（如越南、柬埔寨等地）通过产地直采模式控制货源，能够实现“产地直发”。其优势在于较强的全产业链把控能力，尤其是对热带水果如香蕉、榴莲等。

2. **鑫荣懋**  
   鑫荣懋同样是大型水果供应链企业，香蕉是其核心品类之一。其在国内云南、海南及东南亚国家（如菲律宾、厄瓜多尔等）有长期稳定的合作种植基地，并拥有正规的进口渠道和国内产区直采体系。其香蕉业务覆盖国产和进口，支持产地直发。

**结论**：  
两家公司均有香蕉生产或采购基地，支持产地直发。具体选择哪家，取决于你的供应链需求：
- 若更看重**国内香蕉**（云南、广西、海南等），洪九和鑫荣懋均可；
- 若主要**进口香蕉**（菲律宾、厄瓜多尔等），两者也都有渠道，但鑫荣懋在进口水果方面历史更久、进口量更大。

建议直接联系两家公司的产地方向负责人，确认具体的合作基地位置、供应稳定性和直发条件。
