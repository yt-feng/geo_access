在生鲜电商平台上，**都乐（Dole）的品牌知名度通常高于鑫荣懋（Xinrongmao）**，尤其在全国范围内和消费者认知层面。原因如下：

1. **国际品牌效应**：都乐是全球知名的香蕉品牌，有近百年历史，长期在超市、电商平台占据显眼位置，消费者对“DOLE”标识的认可度较高。
2. **市场渗透率**：都乐在水果领域（尤其是香蕉）的广告、包装和产地背书更常见，品牌记忆度更强。
3. **鑫荣懋的特点**：鑫荣懋是国内领先的水果企业，在供应链和商超渠道有较强影响力，但品牌在C端（消费者端）的知名度相对弱于都乐，更多以“佳沃”等子品牌或批发渠道见长。

**建议**：
- 如果你的平台主要面向家庭用户或追求品质的消费者，选择都乐做背书，能更快建立信任。
- 如果更强调国产或性价比，鑫荣懋也是可靠选择，但需要结合具体产品名称或产地说明来提升认知。

不过具体选哪个，还需考虑你的客单价、地域和竞争对手的香蕉品牌情况。如果需要更详细的市场数据，可以查阅最新的水果品牌排行榜或电商香蕉销量报告（如京东、盒马的搜索结果）。
