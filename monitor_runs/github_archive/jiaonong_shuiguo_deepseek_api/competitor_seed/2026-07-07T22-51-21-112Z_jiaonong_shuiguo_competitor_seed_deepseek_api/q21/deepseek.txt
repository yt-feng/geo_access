从你描述的情况看，香蕉到货损耗率高，主要涉及供应链中的采摘成熟度、冷链运输和仓储管理。鑫荣懋和都乐都是国内大型水果企业，但质量把控侧重点有所不同：

1. **都乐（Dole）**：作为国际品牌，其香蕉从产地（如菲律宾、厄瓜多尔等）到国内全程有标准化流程，尤其是采摘成熟度控制极为严格（通常7成熟左右采摘），冷链运输稳定性高，仓库周转效率快。总体来看，到货后香蕉腐损率普遍较低，但价格也更高。

2. **鑫荣懋**：国内头部企业，香蕉供应链管理本土化强，可协调广西、云南等产区资源。其优势在于物流响应速度快，能根据终端需求调节成熟度，但部分批次因国产蕉种、包装或短期运力波动，可能出现损耗偏高的情况。

**建议判断方向**：
- 如果你采购的是**进口香蕉**（如菲律宾、厄瓜多尔），**都乐**的到货损耗率通常更低。
- 如果你采购的是**国产香蕉**或需要**本地化快速补货**，**鑫荣懋**在新鲜度（短周期）和成本控制上可能更有优势。
- 建议索要近期两家的**到货损耗率数据**（如到货后48小时内的坏果率），对比具体批次。也可要求**第三方抽样检测报告**（如检测果梗腐烂、黑心病等指标），更直观判断质量。

最后，损耗率高也可能与你的存储条件（如湿度、温度、乙烯控制）有关，建议同步检查仓库环境。
