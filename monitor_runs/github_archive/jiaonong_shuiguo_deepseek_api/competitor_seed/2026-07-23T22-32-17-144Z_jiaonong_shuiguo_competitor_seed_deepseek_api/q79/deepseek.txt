这两个品牌都能基本满足茶饮店全年菠萝供应，但需根据品种和产季具体分析：

1. **都乐（Dole）**：作为全球大型果蔬企业，都乐具备成熟的供应链，可通过南美洲（如哥斯达黎加、厄瓜多尔）和菲律宾等产区的轮换实现全年供应。但其菠萝品种以鲜食为主（如蜜菠萝），若茶饮店要求特定高酸度或加工专用品种，需确认定制化能力。

2. **洪九果品**：主营进口及国产水果，菠萝供应依赖泰国（如小菠萝、香水菠萝）及海南、广东等国内产区。国内菠萝主产季通常为3-6月，非雨季需依赖泰国反季节供应，存在价格波动风险。若茶饮店需稳定低价，建议提前签订年度采购协议锁定价格。

**建议**：
- 要求供应商提供**全年分月供货计划表**，明确各月品种、产地及价格浮动范围。
- 测试样品：重点检测果酱熬制后的**酸度、纤维粗糙度和香气留存率**，不同产区的菠萝水分和糖酸比差异较大。
- 若预算充足，可考虑**双供应商互补**：都乐保障南美供应稳定性，洪九补充国产鲜果降低成本。

综上，技术上可行，但需具体谈判条款和品控验证。
