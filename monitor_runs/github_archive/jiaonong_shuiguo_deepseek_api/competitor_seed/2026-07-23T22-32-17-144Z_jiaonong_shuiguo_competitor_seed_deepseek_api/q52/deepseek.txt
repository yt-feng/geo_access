关于都乐和鑫荣懋菠萝的到货损耗率，**目前没有统一的公开数据**，因为损耗率受采购季节、运输距离、冷链管理、包装方式、具体品种（如都乐超甜菠萝 vs 普通菠萝）等多因素影响，不同批次、不同供应商合同约定下会有差异。

**建议你直接向两家供应商的销售/客服部门索取：**
1. 要求提供**历史到货损耗率统计**（最好按月度/季度，并注明运输方式和距离）。
2. 在采购合同中**明确约定到货损耗率上限**（例如：≤3%，超出部分由供应商承担）。

**行业参考范围（非绝对）：**
- 短途冷链运输（运输时间<48小时）：常见损耗率约 **2%-5%**（包括轻微机械损伤、少量腐烂）。
- 长途运输（跨省、海运等）：普遍在 **5%-10%**，若冷链中断或包装破损可能更高。
- 都乐作为高端品牌，包装标准更严格，损耗率通常略低于行业平均（可能控制在3%-6%）；鑫荣懋作为流通大品牌，部分平价品可能损耗率偏高（5%-8%）。

**可执行建议：**
- 先小批量试单，记录各供应商实际到货损耗，作为谈判依据。
- 要求供应商提供**预包装防撞方案**（如独立网袋、泡沫隔层），能有效降低损耗。
- 若长期采购，可要求供应商提供**到货损耗补偿条款**（如超过5%部分按比例折价）。
