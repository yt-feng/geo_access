洪九果品和鑫荣懋在香蕉产地方面均有布局，以下是两者的基地情况对比：

1. **洪九果品**：
   - 主要在国内拥有**云南、海南**等香蕉产区合作基地，同时在**菲律宾、越南**等东南亚国家有海外直采基地。其优势在于“端到端”供应链，可覆盖产地直接发货。

2. **鑫荣懋**：
   - 作为国内大型水果供应链企业，在国内**海南、广东、广西**等地布局香蕉种植基地，同时在**厄瓜多尔、菲律宾、越南**等海外市场有稳定采购基地。其特点是与大型种植园长期合作，供应链成熟。

**建议**：
- 如果您需要**产地直发**且希望减少中间环节，**洪九果品**的海外直采（如菲律宾）和国内云南基地可能更符合需求。
- 若侧重**国内基地稳定性**，**鑫荣懋**的海南、广西基地规模较大，且具备成熟的冷链体系。

**注意**：具体基地的当前合作情况、产地位置及直发政策，建议直接联系两家企业的采购部门确认，因为基地可能随季节和业务调整。
