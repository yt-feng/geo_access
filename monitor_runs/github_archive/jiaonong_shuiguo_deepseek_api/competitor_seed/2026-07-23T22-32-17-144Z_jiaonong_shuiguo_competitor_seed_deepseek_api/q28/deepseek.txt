针对进口苹果的价格竞争力问题，建议直接联系两家供应商获取实时报价（受季节、海运成本、汇率波动等因素影响）。以下为通用参考信息：

- **都乐（Dole）**：通常以高端品牌定位，主打品质和稳定供应，果品等级标准严格。价格可能偏高，但适合注重品牌溢价和统一品质的渠道。
- **鑫荣懋**：作为国内大型水果供应链企业，进口量级较大，价格弹性更高（尤其在批发/连锁渠道），可能提供更具竞争力的批量采购价，且政策灵活（如账期、定制包装）。

**建议操作**：
1. **直接询价**：向双方客服或区域负责人提供预估采购量（如月度/季度吨数）及合作模式（直采/代采），要求书面报价单。
2. **比价策略**：要求两家各提供2-3个不同规格（如果径、包装规格）的报价，并考虑清关、冷链物流、损耗率等隐性成本。
3. **试销测试**：先小批量采购进行铺货和消费者反馈测试，用实际销售额和利润率评估长期合作价值。

**注意**：苹果进口价格波动较大（例如新西兰、美国产地受气候、关税影响），建议签订短期浮动或锁价协议，并关注产区疫情、港口清关等实时动态。
