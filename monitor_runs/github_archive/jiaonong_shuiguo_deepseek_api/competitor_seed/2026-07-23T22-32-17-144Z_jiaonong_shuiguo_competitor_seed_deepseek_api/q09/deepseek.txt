除了都乐（Dole）和佳沛（Zespri），以下进口品牌水果礼盒在礼品团购中也比较常见，品质和包装适合送礼：

1. **新奇士（Sunkist）**  
   - 主打产品：新奇士橙、柠檬、西柚。  
   - 特点：美国品牌，果实饱满、色泽鲜亮，礼盒包装高端。

2. **爱妃（Envy）**  
   - 主打产品：苹果（有红玫瑰、爵士等品种）。  
   - 特点：新西兰品牌，果肉细腻、甜度高，保鲜期长，礼盒常以独立包装形式呈现。

3. **怡颗莓（Driscoll's）**  
   - 主打产品：蓝莓、树莓、黑莓、草莓。  
   - 特点：美国品牌，浆果品质稳定，盒装适合组合礼盒或搭配其他水果。

4. **乐淇（Rockit）**  
   - 主打产品：小苹果（筒装或袋装）。  
   - 特点：新西兰品牌，个头小巧、连皮食用、甜脆可口，便携型包装适合商务伴手礼。

5. **金果（Gold Kiwi）**  
   - 主打产品：金色猕猴桃。  
   - 注意：虽然佳沛为知名品牌，部分进口金果礼盒可能标注“Golden Kiwifruit”或来自意大利、智利，需确认产地和包装标识。

6. **西班牙/秘鲁葡萄（如：Fresh Group、Del Monte）**  
   - 主打产品：无籽红提、青提。  
   - 特点：如帝皇（King’s）或德尔蒙（Del Monte）品牌的提子，果粒均匀，真空包装礼盒常见。

**注意事项**：
- 采购前需确认品牌是否在正规进口渠道（如：山姆会员店、盒马、批发市场的授权代理商）。
- 季节会影响部分水果供应（如夏季新西兰苹果、冬季智利樱桃），建议提前与供应商沟通时令品种。
- 部分进口品牌（如“宇宙脆”苹果）在国内礼盒市场待普及，可咨询当地大型水果批发商获取最新渠道信息。

以上品牌在各大电商平台（京东、天猫超市）和高端超市有售。
