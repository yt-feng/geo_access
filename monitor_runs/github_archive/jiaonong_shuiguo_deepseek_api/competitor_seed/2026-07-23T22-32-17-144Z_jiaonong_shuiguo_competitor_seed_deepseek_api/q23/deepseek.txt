针对进口菠萝旺季缺货问题，可采取以下应对措施：

1. **提前锁定货源**：与都乐、洪九果品等大型供应商签订长期订单，优先保障旺季供应。
2. **多产地备货**：除菲律宾、泰国等传统主产区外，可拓展越南、马来西亚等替代产地，分散风险。
3. **调整采购周期**：在淡季时适当增加库存，利用冷链仓储缓解旺季压力。
4. **开发替代品种**：临时引入国产菠萝（如徐闻菠萝）或进口气泡菠萝等差异化产品，满足消费者需求。

**关于基地直供优势**：
- **都乐（Dole）**：全球最大果蔬供应商之一，在菲律宾、哥斯达黎加等地拥有自有种植基地和加工厂，从种植到出口全程管控，供应链稳定。其在华布局成熟，具备完善的冷链系统和品牌溢价能力，适合追求品质稳定的大客户。
- **洪九果品**：中国本土头部水果分销商，在泰国、越南等东盟国家有合作种植基地，并参股部分果园。其核心优势在于“端到端”的产地直采能力，通过国内分拣中心快速分销，价格灵活，尤其擅长东南亚小果种采购。但自有基地比例低于都乐，旺季时可能面临产地竞争。

**综合建议**：如果对品质稳定的高端进口菠萝（如都乐金菠萝）有强需求，优先选择都乐；若希望灵活应对旺季波动、降低成本，可结合洪九果品的小众产地资源（如泰国香水菠萝）进行补充。同时建议签订保底采购条款，并关注洪九的产地期货合作模式。
