除了都乐（Dole）和鑫荣懋（Joyvio），以下几个进口苹果品牌在市场上也较为知名且信誉较好，可供参考：

1. **爱妃（Envy）**  
   - 新西兰品牌，以红玫瑰苹果（Pacific Rose）为主打，口感脆甜、香气浓郁，成熟后耐储存。
   - 可通过京东、山姆会员店等渠道购买。

2. **爵士（Jazz）**  
   - 新西兰与智利均有生产，果肉紧实、酸甜平衡，适合冷藏保存。
   - 常见于盒马、百果园等精品水果店。

3. **红粉佳人（Pink Lady）**  
   - 澳大利亚品种，以粉红外观和酸甜可口著称，果肉细嫩。
   - 主要在华北、华东高端超市有售。

4. **嘎啦（Gala）**  
   - 美国、新西兰、智利广泛种植，果形小巧，口感酥甜，适合即食。
   - 各大商超易买到，注意认准进口标识（如“智利产”）。

5. **青森（Aomori）**  
   - 日本品牌，主打“世界一号”“富士”等品种，果实大、水分足，糖度高。
   - 主要在高端生鲜电商（如每日优鲜、美团买菜）预售或进口超市少量供应。

6. **红露（Red Velvet）**  
   - 中国种植、进口品牌（如新西兰、美国），果皮深红，肉质密实，适合长期存放。
   - 部分批发市场或会员制超市（如Costco）有售。

**选购建议**：  
- 注重口感：爱妃、爵士（酸甜平衡）、红粉佳人（酸甜突出）  
- 注重耐储性：红露、嘎啦（冷藏可放1-2周）  
- 追求稀缺感：青森（价格较高，产量少）

**注意**：购买时查验包装标签的“原产地”和“进口商”信息，避免买到仿冒或混装产品。部分品牌（如爱妃）有官方防伪码可扫。
