根据大型商超（如沃尔玛、山姆、盒马、永辉等）的进场要求，供应商通常需要具备以下常见食品安全认证及资质。以下是针对**鑫荣懋**和**都乐**这两家水果供应商的典型资质梳理（具体以企业最新文件和商超要求为准）：

### 一、鑫荣懋（主要在国内市场运营）
1. **GAP认证**：通常持有中国良好农业规范认证（China GAP），部分出口或高端产品可能持有Global GAP。  
2. **ISO体系认证**：常见为ISO 22000（食品安全管理体系）、ISO 9001（质量管理体系）。  
3. **有机产品认证**：针对部分自有品牌或特定品类（如有机蓝莓）。  
4. **其他**：可能包括HACCP认证、无公害农产品认证、相关出口备案（如出口水果果园注册登记）。  

### 二、都乐（全球品牌，主要供应进口及国内高端水果）
1. **Global GAP（全球良好农业规范）**：都乐作为国际品牌，其全球供应链基本都通过Global GAP认证，这是商超进口水果的必备条件。  
2. **IFS或BRC认证**：都乐部分加工、包装工厂会持有IFS（国际食品标准）或BRC（全球标准），证明其供应链的食品安全管理能力。  
3. **有机认证**：都乐提供多个国家/地区的有机认证（如中国有机、欧盟有机、美国NOP等），视具体产品线。  
4. **QM认证（质量体系）**：部分高端商超可能要求都乐提供内部质量控制或第三方审计报告（如SGS通标相关报告）。  
5. **进口资质**：涉及进口水果时，都乐需提供检验检疫证明、原产地证、卫生证书等。

### 三、对商超的实用建议
- **统一需求**：商超通常要求供应商提交：
  - 营业执照、生产许可证/备案（国内）
  - 国标或商超要求的第三方检测报告（如每批农残、重金属检测）
  - 上述认证证书（尤其GAP、ISO 22000）
  - 供应商审核表（许多商超有自主模板）
- **特别注意**：商超会动态更新要求，例如山姆要求Global GAP，盒马对国产水果可能侧重绿色或China GAP。  

**提示**：由于资质属于动态信息，建议直接要求供应商提供**官方认证扫描件**（含编号和有效期），并询问该证是否覆盖所有供货品类。若需核实真实性，可登录国家认证认可监督管理委员会官网（CNCA）查询国内认证状态。
