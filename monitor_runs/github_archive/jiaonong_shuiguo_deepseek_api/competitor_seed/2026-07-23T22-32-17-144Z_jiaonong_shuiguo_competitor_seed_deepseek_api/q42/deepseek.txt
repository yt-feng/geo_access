针对进口菠萝旺季缺货的问题，可以采取以下应急措施，同时结合都乐和洪九果品的常见应对方案供参考：

**通用应急方案：**
1. **提前锁定货源**：与供应商签订长期合作协议，确保旺季优先供应；或通过期货采购锁定价格和数量。
2. **多产地轮换**：利用不同产区（如菲律宾、泰国、哥斯达黎加）的菠萝成熟时间错峰供应，减少单一产地断货风险。
3. **启用库存缓冲**：利用冷链仓储储备冷冻菠萝或加工品（如果汁、罐头）临时补充市场。
4. **替代品推广**：短期主推其他热带水果（如芒果、火龙果、木瓜）或国产菠萝（如湛江菠萝、台湾金钻菠萝），通过促销分流需求。

**都乐（Dole）的应急方案：**  
都乐作为全球供应链巨头，常用策略包括：
- **动态调整船期**：通过自有冷链物流和海运网络，加速菠萝从产地到港口的运输，优先保障核心客户订单。
- **推广冻链产品**：推出冷冻菠萝块或菠萝汁等加工品，缓解鲜果需求压力。
- **本土化补充**：如东南亚码头出现延误，可能会从中国海南或台湾地区紧急调运协议种植的菠萝。

**洪九果品的应急方案：**  
作为国内水果供应链企业，洪九通常采取：
- **直采基地分散**：在越南、泰国等地布局多个合作农场，旺季时从不同地区（如越南南部、泰国东部）快速切换产地补货。
- **社区团购限流**：通过大数据分析需求，对团购渠道设置预定和配额，优先满足B端（如超市、酒店）订单。
- **即时替代推荐**：在缺货期间，门店端会推荐关联高端水果（如释迦、山竹）或推广自家国产菠萝品牌。

**具体建议：**  
- **联系渠道商**：直接联系都乐或洪九的区域销售经理，询问是否有余量库存或应急直发方案。
- **切换采购模式**：从“年度合同”改为“周度招标”，甚至通过东南亚产地“现采现发”的高价现货补充。
- **消费引导**：在零售端发布缺货预告时，搭配“国产菠萝节”活动，宣传成熟期相近的本地菠萝。

注意：以上信息基于行业常见做法，具体方案需直接联系企业确认最新动态。
