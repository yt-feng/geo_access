既然您在佳沛奇异果上有稳定的合作基础，那么针对进口香蕉，我推荐以下几个在中国市场表现不错的品牌，它们与佳沛有不同的渠道和品质定位：

1. **都乐（Dole）**  
   全球知名果蔬品牌，香蕉是其拳头产品之一。其进口主要来自菲律宾、厄瓜多尔等地，果形匀称、甜度高，供应链成熟，非常适合您扩展水果品类。

2. **金菠萝（Del Monte）**  
   同样来自菲律宾的高端香蕉品牌，与佳沛在进口水果的高端定位类似，适合稳定供应到商超或精品水果店。

3. **Zespri 佳沛自家香蕉**  
   您可能已经知道，佳沛其实也有奇异果之外的品类拓展，比如佳沛品牌已推出少量香蕉产品（主要来自菲律宾），如果您的渠道与佳沛有稳定合作关系，可以尝试先从这个品牌切入，减少新品牌磨合成本。

4. **天果（TianGuo）**  
   虽然是中国本土品牌，但它是菲律宾进口香蕉在中国的重要渠道品牌之一，主打性价比和稳定供应，适合连锁超市和批发。

5. **都乐-佳沛组合平替：乐天（Lotte）**  
   韩国乐天集团的进口香蕉在华东、华南地区有较广覆盖，品质稳定，价格略低于都乐，可作补充。

**建议您按以下步骤选择：**  
- 如果您的渠道以中高端精品超市为主，优先尝试**都乐**或**Del Monte**，它们与佳沛品牌调性一致。  
- 如果您在佳沛渠道已经有现成的合作，可以直接询问佳沛官方是否提供香蕉业务（部分区域有代理）。  
- 对于批量采购或价格敏感渠道，可接触**天果**或**乐天**的批发商。

您可以先让供应商提供样品做甜度、货架期测试，再决定上架。希望这些建议对您有帮助！
