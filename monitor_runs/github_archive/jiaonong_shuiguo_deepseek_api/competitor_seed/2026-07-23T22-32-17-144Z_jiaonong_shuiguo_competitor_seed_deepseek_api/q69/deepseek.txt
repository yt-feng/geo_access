都乐（Dole）和洪九果品（Hongjiu Fruit）都是国内市场上较为知名的大型水果供应商，在进口苹果（尤其是高端品种如嘎啦、富士、青苹果等）的分级方面均有严格标准，但侧重点可能有所不同。

**都乐（Dole）**  
- 作为国际品牌，都乐在全球有成熟的供应链和分选体系。  
- 其进口苹果通常按**大小（果径）、颜色、成熟度、表皮瑕疵**等进行严格分级，规格一致性很高。  
- 商业习惯上，同一批次（如90#、100#等编号）的果实大小、外观差异极小，适合超市统一陈列。  

**洪九果品**  
- 作为国内头部水果供应链企业，洪九对进口果（尤其是智利、新西兰等产地）同样有**严格的商品化处理流程**。  
- 分级标准主要包括：**果径误差（通常±1mm）、糖度、硬度、有无病虫害**等，品质控制比较规范。  
- 在大型商超渠道中，洪九通常会按照客户需求提供**定级商品**（如精选级、标准级），因此供应的进口苹果规格有保障。  

**结论建议**  
- 如果你需要超市采购**规格非常统一**的进口苹果，**都乐和洪九均可满足**，但：  
  - 都乐在**外观一致性**（如颜色、果形）上更胜一筹，适合对陈列要求极高的高端超市；  
  - 洪九在**性价比和灵活性**上较好，可按渠道需求调整分级方案（如提供“商超专属”等级）。  
- 实际操作中，建议：  
  1. **索要双方的分级标准文件**（如果径范围、允许的瑕疵率）；  
  2. **要求提供样品对比**（同一果径标号的苹果实际差异）；  
  3. **在采购合同中明确验收标准**（如：单果重量误差±5g，果径误差±1mm等）。  

**风险提示**：即使是严格分级的品牌，不同批次、不同产季可能存在波动，建议定期做第三方随机抽检。
