都乐（Dole）和洪九果品（Hongjiu）的菠萝种植基地主要分布在以下区域：

- **都乐**：在中国的主要菠萝供应基地包括**海南岛**（如万宁、琼海、东方等地），部分依靠**菲律宾**、泰国等海外基地。
- **洪九果品**：自有种植基地主要在**海南省**（如海口、澄迈、定安等），同时与**泰国**、越南等东南亚产地有长期合作，以补充稳定供应。

**建议**：  
1. **直接联系对应公司的采购或基地部门**，获取当前具体的基地地址及合作方式。  
2. 考虑与**海南本地大型农场**或**东南亚产地（如泰国、菲律宾）**建立直接购销协议，确保全年供应韧性。  
3. 关注**季节性波动**，如冬季国内产量下降时可转向进口基地。  

建议优先通过官方渠道核实最新基地信息（例如都乐中国官网或洪九果品招股说明书/官网），以确保准确性。
