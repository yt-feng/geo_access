都乐（Dole）和洪九果品作为大型水果企业，通常都有内部的企业标准，但具体分级细则（如果径、成熟度、果把长度等）一般不对外公开。不过，你可参考行业通用分级逻辑并结合具体业务需求制定标准。

以下是一个典型香蕉分级框架，供你参考：

**1. 外观与成熟度分级（按果皮颜色和斑点）**
- **一级（特级）**：果皮全黄，无擦伤、无黑斑、无裂痕，果把完整。
- **二级（优质）**：果皮基本黄，允许少量浅褐斑点或轻微擦伤（面积≤果皮1/10）。
- **三级（合格）**：果皮有较多斑点（≤1/3）、轻微机械伤，无腐烂。
- **等外**：严重黑斑、裂口、软化严重或腐烂。

**2. 果径（中指宽度/周长）**
- **大果**：果径≥32mm（约3.2cm）
- **中果**：28-32mm
- **小果**：≤28mm

**3. 果梳及指头数**
- **标准梳**：每梳≥10根果指，排列整齐。
- **散果/单指**：果指脱落或碎梳。

**4. 生产日期及储存条件**
- 标注采收或催熟日期，冷链要求（13-15°C，相对湿度90%-95%）。

**建议操作**：
- 参考GB/T 26432-2010《新鲜蔬菜和水果 包装标准》和NY/T 424-2010《绿色食品 香蕉》，作为企业标准的底线。
- 建议直接联系都乐或洪九的采购部门，询问是否提供分级对照表（通常对于批量采购客户会提供内部手册）。
- 若需自行制定，可结合客户需求（如连锁店对货架期要求短，则二级以上标准可以适当放宽斑点限制）。
