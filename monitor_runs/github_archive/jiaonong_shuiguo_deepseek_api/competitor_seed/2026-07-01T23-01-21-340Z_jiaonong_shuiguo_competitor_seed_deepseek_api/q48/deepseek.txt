大型商超（如沃尔玛、山姆、永辉、盒马等）对果蔬类供应商的常见食品安全资质要求一般包括：**GLOBALG.A.P.（全球良好农业规范）认证**、**BRC（英国零售商协会）食品安全标准认证**、**ISO 22000（食品安全管理体系）**或**HACCP（危害分析与关键控制点）认证**。对于出口及高端渠道，还可能需要**有机认证**或**非转基因认证（IP认证）**。

以下是鑫荣懋和都乐作为大型国际果蔬供应商通常会持有的资质（信息来源为公开行业资料及企业披露）：

### 1. 鑫荣懋
鑫荣懋是国内领先的水果供应链企业，其产品线包括进口和国产水果。常见资质包括：
- **GLOBALG.A.P.**：覆盖其自有及合作农场的种植与采收环节。
- **BRC 仓储与配送认证**：针对冷链仓储与物流中心，确保储藏和运输条件符合国际标准。
- **ISO 22000 / HACCP**：其加工包装中心（如深圳、北京等）设立有食品安全管理体系，通过相应认证。
- **中国有机认证**：部分产地及产品（如有机蓝莓、有机猕猴桃）持有有机认证。
- **绿色食品认证**：针对其部分国产水果（如新疆葡萄、蜜瓜等）获得绿色食品标志。

### 2. 都乐
都乐（Dole）是全球大型果蔬公司，进口产品较多，其品牌在商超渠道通常需同时具备原产国和出口目的国标准认证。常见资质包括：
- **GLOBALG.A.P.**：核心准入项，覆盖香蕉、菠萝等主要产品。
- **BRC 仓储与配送认证**：其上海、北京等地的配送中心具有该认证，满足商超对冷链管理的审核。
- **ISO 14001（环境管理体系）**与**OHSAS 18001（职业健康安全）**：部分商超要求体系认证，都乐通常提供。
- **美国有机认证（USDA Organic）**或**中国有机认证**：针对其有机产品线（如有机香蕉、苹果等）。
- **非转基因（Non-GMO）认证**：主要针对香蕉、木瓜等品项，满足部分商超的专项要求。

### 特别说明
- **商超渠道差异**：不同商超（如：山姆要求供应商通过BSCI或SMETA验厂；盒马要求供应商需备案食用农产品电子证照；永辉看中源头产地可追溯体系）对认证名单可能不同。建议你**直接询问该商超的采购或品控部门**，获取其**准入门槛清单**。
- **资质时效性**：食品类认证通常为1-3年有效，且需年度监督审核。建议通过以下方式核实最新状态：
  - 联系对应企业**客服或大客户部门**索取最新证书副本（扫描件/复印件）。
  - 通过**认证机构官网**（如GLOBALG.A.P.数据库查询会员号）或**中国认监委（CNCA）**核实有机/绿色食品有效期。
- **最小销售单元**：对于商超，还需确认具体产品（如预包装水果、切配果盒）是否满足**生产许可证（SC）**要求（如适用）。
