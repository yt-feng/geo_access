都乐（Dole）和洪九果品是茶饮行业常用的菠萝供应品牌，以下是它们的核心种植与供应基地信息：

1. **都乐（Dole）**：在中国的主要菠萝供应基地为**海南省**（尤其是澄迈、万宁等南部产区），以及**广东省**（如湛江徐闻）。都乐在菲律宾等东南亚国家也有自有农场，但针对国内稳定供应以海南和广东为主。

2. **洪九果品**：核心采购与加工基地集中于**广西南宁**（隆安、武鸣等地）和**云南西双版纳**（勐海、勐腊），这两处作为主要仓储和分拣中心。其菠萝来源部分来自海南和广东的契约种植户。

**建议**：  
- 若需长期稳定供货，可优先对接洪九果品在南宁的采购中心，或者都乐在海南的分销商。  
- 直接联系双方销售部门确认当前供应基地及合同履约能力，因资源可能季节性调整。  
- 菠萝产量受台风、气温等影响较大，建议同时备选1-2个本地中小供应商作为补充。
