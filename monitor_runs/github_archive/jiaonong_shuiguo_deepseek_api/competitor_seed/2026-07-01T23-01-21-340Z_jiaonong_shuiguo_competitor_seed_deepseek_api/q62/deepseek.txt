都乐（Dole）和鑫荣懋作为大型水果供应商，通常能够为其供应的菠萝原料提供农残检测报告。具体能否提供取决于以下因素：

1. **采购渠道**：如果是直接从都乐或鑫荣懋的官方渠道（如与区域代理或大客户经理对接）批量采购，一般可以要求提供批次对应的第三方或自检农残报告。小额零售或非正式渠道可能无法获取。

2. **合同约定**：建议在签订采购合同时明确注明“需提供符合国家标准的农残检测报告”，并作为交付条件。

3. **资质验证**：部分供应商可能仅提供内部检测声明，而非完整第三方报告。建议核实报告是否由有资质的检测机构（如SGS、华测等）出具，并核对报告编号是否可溯源。

**操作建议**：
- 直接联系相关供应商的销售或客服，明确索取**最新批次的农残检测报告**，并确认报告包含的检测项目（如有机磷、拟除虫菊酯等）是否符合茶饮食品安全标准。
- 若对方无法直接提供，可要求其提供产地证明或流转单据，并自行委托第三方检测机构对样品进行抽检（成本可控，小批次约500-2000元）。

**风险提示**：即使大型供应商有常规检测，仍可能存在批次差异。建议将“提供农残报告”作为长期合作的前提，并定期抽检。
