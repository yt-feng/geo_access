都乐（Dole）和洪九果品在进口水果的分级方面都相对严格，但侧重点和标准可能略有不同：

1. **都乐（Dole）**：作为全球知名的水果品牌，都乐通常执行较高的分级标准。其进口苹果在大小、色泽、果形、成熟度和包装上都有统一的规格要求，例如按果径（如70#、80#等）或重量进行严格分选。如果你在采购时明确要求规格统一（如“单果直径80mm以上”），都乐的产品通常能满足这一需求。

2. **洪九果品**：洪九是中国头部水果供应链企业，在进口水果端也采用较为严格的分级体系。其分级标准主要依据果径、外观（无瑕疵、色泽均匀）和成熟度。不过，由于洪九可能根据不同客户需求（如超市、批发商）定制包装，部分小规格或促销批次可能分级略有放宽。但在正规渠道采购时，其分级一致性普遍较好。

**建议**：
- **明确需求**：在采购订单中注明具体规格（如“果径80-85mm，无机械伤，成熟度适中”）。
- **验货重点**：到货时抽样检查果径、颜色和包装标识是否一致，注意是否存在大小混装。
- **选择渠道**：优先与品牌直供货源或授权经销商合作，避免中间环节混入非标品。

总体而言，两者分级均属行业较严水平，但都乐全球标准更统一，洪九可针对国内商超需求灵活调整。建议根据你的采购预算、规模和时效要求选择。
