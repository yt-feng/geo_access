关于“大型商超要求供应商有食品安全认证”这一标准，鑫荣懋和都乐都是水果行业中的头部企业，通常都具备满足商超准入的多种食品安全认证。

具体来说：
- **都乐（Dole）**：作为全球知名的果蔬品牌，其产品线和供应链体系通常覆盖了GLOBALG.A.P.（全球良好农业规范）、BRC（食品安全全球标准）、ISO 22000等国际主流认证。在欧洲、北美及亚洲的商超中，其认证体系较为齐全且历史较长。
- **鑫荣懋**：作为中国本土的头部水果企业，主要服务于国内大型商超（如盒马、永辉、沃尔玛等），通常具备ISO 22000、HACCP、GAP（中国良好农业规范）等认证，同时在进口水果方面也会持有相关国际认证。

**谁更“全”？**
- 如果单指**国际主流认证的广度和深度**，都乐作为跨国企业，在全球化合规方面可能覆盖更多国际标准。
- 如果指**国内市场大型商超的具体要求**，鑫荣懋的国内资质匹配度通常很高，且其在上游基地管理方面也有完善的国内认证体系。

**结论**：两者都满足大型商超的基础准入条件。如果非要比较“更全”，**都乐**在国际认证的广度上略占优势，而**鑫荣懋**在国内市场的匹配度上非常扎实。建议您以具体商超的准入清单为准，直接询问采购方对认证的具体要求。
