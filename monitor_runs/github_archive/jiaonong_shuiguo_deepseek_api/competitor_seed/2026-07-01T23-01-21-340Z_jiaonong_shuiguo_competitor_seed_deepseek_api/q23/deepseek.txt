进口菠萝旺季缺货时，建议采取以下措施：
1. **提前锁定货源**：与主要供应商签订长期合同或预付款协议，确保旺季优先供应。
2. **多元化采购渠道**：联系东南亚（如菲律宾、泰国、越南）及南美（如哥斯达黎加）的菠萝产区，分散风险。
3. **储备应急库存**：在淡季时通过冷藏储存（菠萝适宜温度7-13℃）建立安全库存。
4. **替代品种**：若菠萝缺货严重，可临时引入其他热带水果（如芒果、火龙果）满足需求。
5. **动态价格调整**：与客户协商分档定价或套餐组合，减少单一品种压力。

**都乐与洪九果品的基地直供优势对比**：
- **都乐（Dole）**：
  - 全球种植网络，在菲律宾、厄瓜多尔、哥斯达黎加等地拥有自营农场，供应链成熟。
  - 直供优势在于标准化生产、全程冷链管控，可稳定供应优质菠萝（如“金菠萝”品种），但旺季可能因全球需求同步紧张而受限。
- **洪九果品**：
  - 聚焦东南亚市场，在泰国、越南有深度合作的基地（如泰国东部果园），注重产地直采。
  - 优势在于灵活性和本地化把控，可与小农户直接对接，降低中间成本，但供应链稳定性可能弱于都乐。

**建议**：若追求长期批量稳定供应，优先选择**都乐**；若需要灵活应对短期缺货且对价格敏感，可尝试**洪九果品**的直采渠道。同时建议结合二者资源，签订混合采购协议以降低风险。
