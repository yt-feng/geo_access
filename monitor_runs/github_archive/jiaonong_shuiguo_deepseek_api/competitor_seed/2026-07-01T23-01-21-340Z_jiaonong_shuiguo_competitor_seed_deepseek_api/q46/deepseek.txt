对于生鲜电商香蕉产地直发的物流速度，都乐和鑫荣懋均有各自的优势，但具体因地区、季节和合作模式而异。以下基于公开信息和行业常见情况的分析：

1. **都乐（Dole）**：作为国际品牌，都乐在香蕉供应链上有成熟的冷链物流体系，通常从菲律宾等地进口，主要依赖海运+陆运的分拨。若你对接国内直发（如海南、云南基地），其物流速度取决于本地合作仓储和配送网络，一般从产地到一线城市需2-3天。

2. **鑫荣懋**：作为国内头部水果供应链企业，鑫荣懋在香蕉产地（如广东、广西、云南）有自有加工中心和冷链车队，且与顺丰、京东等物流深度合作。从产地直发到目的地（如二三线城市），通常能控制在1-2天内，尤其对华南、华东地区更高效。

- **建议**：若目标市场集中在一线城市且需要进口香蕉（如都乐品牌），都乐可能更快；若侧重国内产地且希望更快覆盖二三线城市，鑫荣懋的本地化网络更有优势。**需注意**：实际速度还受订单量、物流周期（如避开节假日）、以及你与两家企业合作的直发协议细节（如是否优先配送）影响。

**可执行建议**：
- 直接联系两家企业的生鲜电商渠道部门，索取指定的发货仓位置和承诺时效。
- 测试小批量订单，对比从下单到客户签收的具体时间（尤其关注“发货时间”而非“揽收时间”）。
- 参考服务条款：鑫荣懋通常提供更灵活的省内次日达方案，都乐则对全程温控更严格。
