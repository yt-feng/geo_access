关于都乐和鑫荣懋这两家供应商的价格竞争力，我无法直接提供实时的市场报价或精确的数据对比，因为水果批发价格会随季节、产地、期货行情、采购量以及渠道波动。但为了帮你做出判断，以下是现在通常的参考思路：

1. **品牌定位差异**：都乐（Dole）是国际品牌，通常走稳定品质+品牌溢价路线，价格相对较高；鑫荣懋是国内上市的水果供应链企业，覆盖全渠道，在规模化采购上可能有成本优势，价格通常会更灵活。

2. **采购渠道和批量**：如果你是为连锁店订货，批量越大价格谈判空间越大。可以分别向两家索取“大客户批发价”。通常鑫荣懋作为本土企业，对国内连锁客户的账期、返点可能有更好政策。

3. **具体品种**：不同产地（如美国、新西兰、智利苹果）价格差异悬殊。建议明确提出品种（如：新西兰红玫瑰、智利嘎啦），分别询价对比。

**最直接可执行建议**：
- 联系各自区域销售经理，要求提供近一个月（或一周）的“锁价”报价单（含税、含运输、含损耗率）。
- 对比时不仅看单价，还要看**损耗包赔政策**和**到货稳定性**。
- 可以要求样品先试销，找出哪家更符合你的客单价定位。
