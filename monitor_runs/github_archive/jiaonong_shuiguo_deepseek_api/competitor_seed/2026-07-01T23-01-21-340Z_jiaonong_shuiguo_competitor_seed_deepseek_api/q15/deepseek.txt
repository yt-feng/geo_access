除了都乐（Dole）和鑫荣懋（Joyvio）外，市场上还有以下几个较为靠谱的进口苹果品牌，供对比参考：

1. **爱妃（Envy）**  
   - 特点：主打高端市场，果肉紧实、甜度高，耐储存，产自新西兰。
   - 建议：适合追求极致口感和送礼场景。

2. **纽西兰（Zespri）**  
   - 特点：虽然以奇异果闻名，但旗下苹果品牌如“爵士”（Jazz）、“太平洋玫瑰”（Pacific Rose）等也较知名，产自新西兰。
   - 建议：适合偏好新西兰果味、注重品牌信誉的消费者。

3. **美国华盛顿苹果（Washington Apple）**  
   - 特点：产自美国华盛顿州，品种包括红蛇果（Red Delicious）、嘎啦（Gala）等，果形统一、脆甜。
   - 建议：适合日常食用，性价比较高。

4. **加州阳光（Sunkist）**  
   - 特点：以柑橘闻名，但旗下进口苹果（如富士、青苹果）品质稳定，产地美国。
   - 建议：如需多种采购，可作为补充选择。

5. **意大利/法国品种（如Pink Lady）**  
   - 特点：粉红女士（Pink Lady）源自澳洲，但欧洲产区（意大利、法国）品质更优，甜酸平衡，适合烘焙。
   - 建议：适合对甜度有特殊要求的用户。

**选择建议**：
- 如果侧重甜度和耐储存，优先爱妃或华盛顿苹果；
- 如果追求品牌口碑和多样性，可考虑纽西兰或加州阳光；
- 尝试小众高端款，可关注意大利的“Red Love”等品种。

实际购买时，注意查看产地、认证标识（如GLOBALG.A.P.）及包装日期，避免假冒。如有特定超市（如山姆、盒马）采购需求，可进一步询问其常驻品牌。
