针对您关注的菠萝产地溯源及基地直供问题，根据目前公开信息：

1. **都乐（Dole）**  
   都乐在全球有多处自有或合作农场，在中国市场销售的菠萝（通常为菲律宾或哥斯达黎加品种）均来自其国际供应链，具备产地可追溯性。但都乐在国内的基地直供模式主要针对大型商超或批发渠道，并非直接面向所有终端客户开放采购。建议您通过都乐中国官网或客服（400-820-1888）确认其国内直供合作的具体条件。

2. **洪九果品**  
   洪九果品是国内头部水果供应链企业，主攻东南亚进口水果。其菠萝供应主要来自泰国、菲律宾等地的合作种植基地，并在国内有分拣中心。洪九通常通过批发市场或B2B平台（如“洪九到家”）对接采购商，需提前联系其大客户部门（可拨打官网电话023-6866 8888）获取基地直供或产地溯源的详细信息。

**实用建议**：  
- 若要求**精准溯源到具体种植基地**，建议优先联系**都乐**（国际溯源体系较成熟）；若偏好**国内供应链灵活合作**，可咨询**洪九果品**。  
- 也可直接查询农业农村部“农产品质量安全追溯平台”或地方农业部门的基地备案信息，获取更权威的供应商名录。  
- 批量采购时，建议要求供应商提供**产地证、检疫证明、批次号及包装二维码**，以满足核心溯源需求。
