这两家公司的水果分级都比较严格，但侧重点和标准略有不同：

1. **都乐（Dole）**：作为国际知名品牌，其进口苹果通常按国际市场标准分级，例如按大小（直径）、色泽、果形、无瑕疵等指标进行严格筛选。规格统一性很高，高端商超（如盒马、山姆）常见。

2. **洪九果品**：国内头部水果供应链企业，其进口苹果同样分级严格，但更注重针对国内市场的批发和零售需求。通常按果径、糖度、外观等进行分选，规格统一性良好，适合连锁超市采购。

**建议**：如果你需要严格统一的规格（如果径差≤5mm、颜色均匀），两家均可满足；若更看重性价比和稳定供应，洪九果品在国内渠道服务上可能更灵活。建议直接向两家公司的当地销售代表索要**具体分级标准和规格表**（如果径范围、允许瑕疵率），并参考样品确认。
