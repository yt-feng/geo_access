针对礼品团购需求，除都乐（Dole）和佳沛（Zespri）外，以下进口水果品牌值得考虑：

1. **爱妃（Envy）**  
   - 苹果品牌，以果肉抗氧化、甜脆闻名，礼品装常为红色礼盒，适合高端送礼。
   
2. **新奇士（Sunkist）**  
   - 主打柑橘类（橙子、柠檬），品牌认知度高，有标准礼盒装，价格适中。
   
3. **超甜蕉（Dole旗下高端线）**  
   - 虽然属于都乐集团，但“超甜蕉”作为独立子品牌在团购中常见，主打甜度更高、包装更精致。
   
4. **怡颗莓（Driscoll's）**  
   - 浆果品牌（蓝莓、树莓、黑莓），礼盒以塑料盒或纸盒装为主，适合小批量精美包装。
   
5. **吉尔德（Galla）**  
   - 美国苹果品牌（如“爵士苹果”），包装简约，性价比优于爱妃。
   
6. **蓝宝实（Bluesky）**  
   - 新西兰车厘子品牌，夏季是团购旺季，礼盒装规格多样（如2磅、5磅）。
   
7. **都乐（Dole）的其他品类**  
   - 例如都乐法国小黄瓜、都乐凤梨（金菠萝）等，可搭配组合礼盒。
   
8. **其他渠道可查**  
   - 部分区域性进口品牌：如智利车厘子品牌、美国新奇士、澳洲芒果品牌（如Calypso、Kensington Pride）等，可通过批发市场或团购平台查询最新供应。

**建议**：  
- 团购前联系当地水果批发商或进口商确认货源稳定性（尤其是季节性水果如车厘子、芒果）。  
- 礼盒包装建议选择品牌官方通用礼盒（如爱妃的红苹果礼盒、新奇士的透明窗纸盒），避免自行定制增加成本。  
- 对于单位团购，可要求品牌方提供**团购专属折扣**或认证合作证明（如发票、品牌授权书）。
