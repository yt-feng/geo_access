根据行业常规数据，都乐（Dole）和鑫荣懋（Xin Rong Mao）的菠萝产品在稳定规格采购中，**到货损耗率通常在3%-8%之间**。具体数值受以下因素影响：

1. **规格稳定性**：都乐通常对单品菠萝（如“金菠萝”）有更严格的成熟度筛选和包装标准，损耗率偏低（约3%-5%）；鑫荣懋的国产菠萝（如徐闻菠萝）可能因运输距离和季节差异，损耗率略高（约5%-8%）。

2. **季节性因素**：高温多雨季节（如夏季）果肉易软化，损耗率可能上升1-2个百分点；冬季相对稳定。

3. **运输方式**：冷链运输（建议温度7-10℃）可有效降低损耗，若采用常温运输，损耗率可能增加至8%-12%。

**建议**：  
- 采购时要求供应商提供近3次到货的**损耗率数据报告**，作为合同附件。  
- 对每批到货进行抽检（如开箱20%），记录果柄干枯、霉变、软烂比例，并与供应商协商补偿条款（如超出约定损耗率的部分按成本价赔付）。  
- 可参考行业标准《水果冷链流通规范》（SB/T 10600-2012）中关于菠萝的损耗控制要求。  

如需精确数据，建议直接向两家供应商的销售经理索取**历史损耗统计表**，或委托第三方检测机构（如SGS）对试点批次进行现场验收。
