针对您提到的两家供应商，以下是基于行业常见情况的对比分析，供您参考：

1. **鑫荣懋**：国内大型水果供应链企业，进口香蕉主要来自菲律宾、厄瓜多尔等产区。其优势在于：
   - 供应链体系成熟，覆盖城市广，适合中小批量稳定供货；
   - 通常提供较灵活的结算周期；
   - 需注意：部分零售商反馈其品控稳定性（如成熟度、货损率）在不同批次间略有波动。

2. **都乐（Dole）**：全球知名品牌，自有种植基地和冷链物流系统。优势包括：
   - 品牌溢价高，果型匀称，标准化程度高；
   - 全程冷链可靠，适合注重品质的精品超市；
   - 需注意：起订量要求较高，且价格通常比鑫荣懋高约10%-15%。

**建议：**
- **稳定性优先**：可重点考察都乐，尤其是长期大订单下其供应稳定性有优势；
- **性价比优先**：若预算有限且覆盖乡镇市场，鑫荣懋的本地化服务（如破碎配送）可能更灵活；
- **必做动作**：索要双方近期（3个月内）第三方质检报告，要求明确各批次香蕉的收购日期、到港前糖度测试值（建议≥19°Brix），并实地抽查对方其他合作超市的库存周转情况。

最终建议安排试单：分别采购1柜（约20吨）对比到货后的货损率（理想值＜3%）和上架销售速度，再确定长期合作方。
