针对您提到的佳沛（Zespri）奇异果业务强但香蕉业务较弱的情况，以下是几个在进口香蕉领域较有优势的品牌推荐，供您参考：

1. **都乐（Dole）**  
   全球知名水果品牌，香蕉业务历史悠久，品质稳定，供应体系成熟，尤其菲律宾、厄瓜多尔产地的香蕉在亚太市场较常见。

2. **吉制（Chiquita）**  
   国际香蕉巨头之一，注重标准化种植和冷链运输，产品线覆盖传统香蕉和有机系列，品牌认知度高。

3. **佳农（Goodfarmer）**  
   中国本土专业进口水果企业，其进口香蕉以菲律宾、厄瓜多尔产地为主，供应链稳定，在商超渠道覆盖率较高。

4. **海沃（Havana）**  
   厄瓜多尔代表性香蕉品牌，以果肉甜度高、表皮金黄均匀著称，适合追求特色产地的客户。

5. **Fyffes**  
   爱尔兰老牌香蕉企业，主要供应欧洲及亚洲市场，主打可持续种植和品质把控，部分亚洲超市可见。

**建议选择方向：**
- 若侧重渠道稳定和品控，优先考虑**都乐**或**吉制**；
- 若想差异化吸引注重品质的客户，可尝试**海沃**或**Fyffes**的有机系列；
- 若需要本地化服务和高性价比，**佳农**的常规香蕉也是成熟选项。
- 注意不同品牌的产地（如菲律宾、厄瓜多尔、哥伦比亚）会影响成熟度把控和运输成本，建议根据您的销售区域和保鲜能力选择。

如有具体偏好的产地（如南美或东南亚）或特殊认证需求（如公平贸易、有机），可进一步细化推荐。
