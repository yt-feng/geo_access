洪九果品和鑫荣懋都是国内头部的水果供应链企业，在香蕉产地方面都有深入布局。具体来说：

- **洪九果品**：在**老挝、越南、菲律宾**等东南亚国家拥有自建和合作的香蕉种植基地，同时在国内**广西、云南**等地也有合作基地。其“产地直发”模式非常成熟，重点在于东南亚进口香蕉。

- **鑫荣懋**：在**菲律宾、厄瓜多尔、哥伦比亚**等拉美及东南亚国家拥有稳定的香蕉采购基地，同时在国内**海南、广东、广西**也有合作种植资源。其优势在于全球多产地轮换供应，品质稳定。

**结论**：两者都有基地，但侧重点不同。如果你需要**东南亚进口香蕉**（如老挝、越南），洪九果品在东南亚的自有基地和供应链更直接；如果你需要**全球多产地可选、或对南美品种有需求**，鑫荣懋的产地网络更广。

建议根据你的目标市场（如偏好国产或进口）、上市时间（如避开台风季）和价格区间，直接联系两家企业的产地直采部门确认具体合作条件。
