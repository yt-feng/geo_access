针对大型商超对供应商的食品安全认证要求，鑫荣懋和都乐作为国内领先的水果供应链企业，通常具备以下主要资质和认证（具体以实际查验为准）：

**鑫荣懋的常见资质：**
- **ISO 22000食品安全管理体系认证**（覆盖从种植到零售的全链条）
- **HACCP危害分析与关键控制点认证**
- **GlobalG.A.P.全球良好农业规范认证**（部分合作基地）
- **有机产品认证**（对应产品线）
- **BRC英国零售商协会食品标准认证**或**IFS国际食品标准认证**（部分出口或高端渠道）
- **绿色食品认证**（部分产品）
- **无公害农产品认证**
- **产地检验检疫备案**及每批次的**第三方农残检测报告**
- **出口果园/包装厂注册**（如针对出口市场）

**都乐的常见资质：**
- **ISO 22000认证**
- **HACCP认证**
- **GlobalG.A.P.认证**（覆盖自有农场及合作种植基地）
- **有机认证**（如美国农业部USDA有机、中国有机等）
- **雨林联盟认证**（可持续发展相关）
- **每批次产品的**农残检测合格报告**（符合中国及出口国标准）
- **海关AEO高级认证**（如涉及进口业务）
- **部分产品拥有**无公害农产品/绿色食品认证**

**给您的行动建议：**
1. **直接索证**：向供应商的客服或品控部门索取最新的资质电子版或扫描件（注意查看有效期）。
2. **注意品类差异**：不同水果或加工品（如鲜果 vs 果切/果汁）所需的认证可能不同，需确认具体供货品类。
3. **关注动态更新**：大型商超（如沃尔玛、山姆、盒马等）会定期更新准入供应商目录及证书要求，建议直接咨询该商超的采购或质控部门，获取其认可的**最低证书清单**。
