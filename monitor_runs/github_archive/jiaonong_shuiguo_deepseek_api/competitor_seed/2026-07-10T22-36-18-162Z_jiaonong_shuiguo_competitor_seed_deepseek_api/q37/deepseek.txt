您好！关于菠萝供应商的产地溯源透明度，根据公开信息和行业评价，都乐（Dole）和洪九果品各有特点，但整体上**都乐在产地溯源方面通常被认为更透明**。以下是具体分析：

### 都乐（Dole）
- **优势**：都乐是全球知名水果品牌，拥有完善的供应链管理系统。其产品通常带有可追溯的条形码或二维码，消费者可以查询到菠萝的种植地、采摘日期、加工信息等。都乐还公开其部分产地的种植标准（如哥斯达黎加、菲律宾等），并参与第三方认证（如GlobalG.A.P.），增强透明度。
- **适用场景**：适合对品牌信誉、全链条信息透明有高要求的企业或消费者。

### 洪九果品
- **优势**：洪九果品是中国头部水果供应链企业，聚焦东南亚进口水果（如泰国、越南菠萝）。其溯源体系依托“端到端”模式，在产地端建立了直采和分选中心，部分产品有产地信息标识，但公开透明度较弱。洪九在B端市场（商超、批发）有较强整合能力，但普通消费者层面的溯源信息不如都乐详细。
- **不足**：洪九果品作为供应链企业，更多服务于大宗贸易，其溯源信息可能更集中在商业合作层面，而非直接面向公众的完整透明展示。

### 建议
- **选择都乐**：如果您需要向消费者提供明确的产地追溯（如通过扫码查看种植地、保质期等），或对品牌背书有较高依赖，都乐是更优选择。
- **选择洪九果品**：如果您在大宗采购（如批发、餐饮供应）中，更看重性价比、供应链稳定性，且不追求极致的终端可查，可以考虑洪九——但建议您在合作合同中明确要求提供具体产地和溯源文件。

### 如何验证？
- **要求供应商提供**：无论选择哪一家，建议直接要求对方提供以下具体信息：
  - 菠萝的具体产地（农场名称或GPS坐标）
  - 采摘日期和运输记录
  - 第三方检测报告（农药残留、种植标准）
- **自行测试**：可向销售方索取样品，扫描包装上的追溯码（都乐通常支持），检查信息是否完整。
