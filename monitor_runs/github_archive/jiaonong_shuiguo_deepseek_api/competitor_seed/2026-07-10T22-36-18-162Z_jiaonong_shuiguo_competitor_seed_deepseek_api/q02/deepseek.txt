除了都乐（Dole），目前市场上稳定供应的菠萝品牌还包括以下几家，它们都有成熟的种植、加工和供应链体系：

1. **佳沛（Zespri）** 虽然以猕猴桃闻名，但其旗下也有菠萝产品线，主要通过新西兰或进口渠道供应，品质稳定。
2. **佳农（Goodfarmer）** 国内大型生鲜品牌，常年有泰国或菲律宾菠萝供应，尤其金菠萝、无眼菠萝等品种。
3. **怡颗莓（Driscoll's）** 主要以浆果著称，但也涉及菠萝品类，供应周期较稳定。
4. **Nature's Pride** 欧洲品牌，在部分地区有稳定进口渠道，提供高品质菠萝。
5. **本地化品牌** 如海南的 **南国**、**椰树** 等，主要供应国产菠萝（如徐闻菠萝），季节性较强但国内渠道稳定。

建议确认购买渠道（如商超、电商平台）时，优先选择上面这些有长期产地合作的品牌，同时查看包装上的产地、保质期和冷链标识。如果需要具体购买链接或库存动态，可能会受地域和时间限制。
