直接回答：**无法直接比较两者当前的具体价格**，因为进口苹果价格受产地、规格、包装、运输成本及实时汇率影响较大，且两家供应商对B端客户的报价通常需根据批量、账期、合作方式单独洽谈。

不过，可以提供两个判断依据和具体行动建议：

1. **行业通常情况**：  
   - **都乐（Dole）** 作为全球性品牌，其进口苹果（如美国、智利产）在品牌溢价、供应链稳定性和品质标准上较高，但同等级产品**裸价（未含品牌附加费）可能略高于鑫荣懋**。  
   - **鑫荣懋（Joy Wing Mau）** 是国内头部水果供应链企业，更擅长通过**产地直采、拼船/散货海运**降低单次采购成本，且对国内渠道（如商超、批发市场）的报价**可能在毛利空间上更灵活**。  
   - **关键变量**：若都乐提供品牌营销支持（如统一陈列、海报），则其“综合成本”需计入营销费节省。

2. **获取准确报价的方式**：  
   - **立即行动**：联系两家公司的**区域销售经理**，给出你的预估年采购量（如20-50吨/年）、期望品级（如80#果、一级果）和配送城市，要求给出**含税到仓价**。  
   - **对比维度**：不要只看单价，需明确是否包含：  
     - 冷链物流费  
     - 坏果赔付比例（如行业标准5%范围内的免费补货）  
     - 期货锁定价格（如提前预付定金锁定智利车厘子式苹果的远期价格）  
   - **参考工具**：如果要快速验证，可查看**北京新发地、广州江南市场**的每日批发报价（如“85#加力果”），但需注意地头价与终端价的差价通常包含3-5%的渠道损耗。

3. **判断竞争力**：  
   - 若你的门店**主打高端形象**，都乐的品牌背书可能带来更高客单价，此时价格竞争力弱化；  
   - 若追求**走量利润率**，鑫荣懋的直采模式+无品牌溢价通常更优，但需自己承担品牌宣传成本。

**结论**：建议**同时索取两家当季代表性品种（如美国绿星苹果、新西兰爵士苹果）的吨车价报价单**，要求注明不含税/含税、包邮/自提，并让销售承诺“对比市场同等品级最低价”。价格竞争力最终取决于你的采购量级和账期谈判结果。
