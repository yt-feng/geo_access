关于水果连锁店使用的香蕉分级标准，都乐（Dole）和洪九果品（Hongjiu）作为业内知名企业，确实有各自的内部企业标准，但这些标准通常不公开详细内容，主要用于内部品质管控和供应链管理。以下是一些通用信息和获取建议：

1. **都乐（Dole）**：作为全球水果巨头，都乐有一套成熟的香蕉分级体系，通常基于果指长度、成熟度（色卡等级）、外观（无瑕疵、病虫害）等指标。例如，都乐的“都乐超甜蕉”等产品有明确规格，但具体分级标准（如特级、一级）未完全公开。建议直接联系都乐中国客服或访问其官网，寻求商业合作时的技术文档。

2. **洪九果品**：作为国内头部水果供应链企业，洪九对香蕉有类似的内部标准，如按果指重量（如180克以上）、长度（20厘米以上）、成熟度（黄绿度）分级。洪九可能在其采购手册或供应商指南中有详细规定，但公开信息较少。可通过洪九的官方渠道或行业展会咨询。

**替代方案**：如果水果连锁店需要参考，可以基于行业通用标准（如国家标准GB/T 40481-2021《香蕉》或国际标准UNECE FFV-02）制定自己的企业标准。该标准对香蕉的等级（如特级、一级、二级）、长度、成熟度、缺陷允许量等有详细规定，可在“国家标准全文公开系统”查询。

**建议**：直接联系都乐和洪九的采购部门，获取其供应商手册；同时结合国家标准和自身门店定位（如高端、社区）定制分级，例如按果指长度（15-18cm为A级）、成熟度（3-7黄为最佳货架期）细分。
