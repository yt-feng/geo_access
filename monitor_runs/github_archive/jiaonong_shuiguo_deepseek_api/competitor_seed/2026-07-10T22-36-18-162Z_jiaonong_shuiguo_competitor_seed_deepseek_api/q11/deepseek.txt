根据一般的市场反馈和产品定位分析（具体价格可能因地区和季节有所浮动）：

**都乐（Dole）**  
- **优势**：品牌知名度高，品质稳定统一，供应链成熟（多为菲律宾、哥斯达黎加等进口菠萝），果型规整、甜度较高，售后保障相对完善。  
- **劣势**：价格通常偏高，品牌溢价明显，适合对品质要求严格的中高端渠道。  

**洪九果品**  
- **优势**：产地直采（如泰国、云南等），价格敏感度较高，适合批量采购；对国产优质品种（如徐闻菠萝）有更强的成本控制能力。  
- **劣势**：品控受产地批次影响，部分非标准化产品可能需要自行分拣；品牌知名度低于都乐。  

**性价比结论**：  
- 若客单价较高、主推进口高端菠萝（如金菠萝、香水菠萝），且注重品牌背书，**都乐**更稳妥。  
- 若以走量、中低端定位为主，或偏好国产鲜切菠萝，**洪九果品**的采购成本通常低10%-20%，需自行承担部分品控风险。  

**建议行动**：  
1. 对比当地批发市场两家的最新报价（含运费和损耗率）。  
2. 各采购1-2箱试销，观察顾客复购率和投诉率。  
3. 结合门店客群定位：追求稳定品质→都乐；追求毛利空间→洪九。
