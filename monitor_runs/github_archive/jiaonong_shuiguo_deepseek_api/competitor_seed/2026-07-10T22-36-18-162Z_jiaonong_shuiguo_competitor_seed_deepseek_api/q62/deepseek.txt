通常情况下，大型水果供应商（如都乐、鑫荣懋）在向茶饮店、餐饮企业等B端客户供货时，是可以提供权威第三方农残检测报告的，但具体能否提供取决于以下条件：

1. **采购渠道**：如果是通过正规经销商或直接对接厂家采购，通常可以要求随货提供批次检测报告。如果通过批发市场或零散采购，需自行向供应商索取。
2. **客户类型**：都乐和鑫荣懋对茶饮店等商业客户通常支持提供，但个人零售购买一般无法获取。
3. **产品批次**：需要明确指定采购的菠萝批次（如产地、生产日期），供应商会针对该批次出具报告。

**建议操作步骤**：
- 直接联系都乐和鑫荣懋的销售客服或区域代理商，说明是茶饮店采购，要求提供对应菠萝批次的第三方检测报告（如SGS、华测等）。
- 如果对方无法提供，可考虑更换供应商或要求其提供产地自检报告作为辅助（但商业合规上应要求第三方报告）。

**注意**：部分情况下，大型供应商的“产品规格书”中可能包含农残检测合格声明，但这不等于完整的检测报告，建议以正式报告为准。若对方拒绝提供，需警惕货源合规性。
