针对您提到的都乐（Dole）和洪九果品这两个品牌，通常情况下：

1. **都乐（Dole）**：在国内市场，都乐主要销售标准化预包装产品（如3-5根装、袋装或盒装），一般很少提供散装（裸装、无包装）销售。如果您希望获取小包装的直接供应，可能需要与区域经销商确认是否有定制化散装的可能性（通常起订量较大）。

2. **洪九果品**：洪九果品以大宗批发和供应链服务为主，其业务模式包括为社区团购、商超等提供定制化包装。他们确实**可能接受散装或简易包装**的订单，尤其面向B端客户时，可以协商按重量、数量等需求进行分装。建议直接联系洪九的区域销售负责人，说明您的社区团购具体需求（比如每日需求量、包装规格、物流要求等），确认是否支持散装分货。

**建议操作**：
- 联系都乐官方或当地经销商，询问最小起订量及能否提供无品牌简易包装。
- 联系洪九果品总部或区域分公司（通过官网或行业展会），通常其大客户经理会针对社区团购等B端场景给出灵活方案。
- 如果对方坚持标准化包装，可咨询是否有“最小包装单位较小的”（如1公斤以下包装）选项，接近散装需求。

对于社区团购，也可考虑直接对接产地或农批市场的二级批发商，他们对散装小包装的灵活度通常更高。
