根据公开信息，**鑫荣懋**和**都乐（Dole）**作为国内大型水果供应链企业，其冷链物流配送网络覆盖情况如下：

**鑫荣懋：**
- 拥有覆盖全国主要城市的冷链物流网络，重点服务大型商超（如沃尔玛、山姆、盒马等）。
- 公开资料显示，其配送范围已覆盖**北京、上海、广州、深圳、成都、武汉、杭州、南京、重庆、西安、沈阳、济南、青岛、福州、厦门**等50多个核心城市，并可通过区域分仓实现二三线城市及地级市的延伸。

**都乐（Dole）：**
- 主要依托自建或合作的冷链体系，满足商超的高标准配送需求。
- 主要覆盖**上海、广州、深圳、北京、成都、武汉、杭州、南京、厦门、福州、大连、青岛、济南、西安、沈阳**等华东、华南、华中及华北重点城市。南方及沿海区域的覆盖密度较高。

**说明：**
1. 以上信息为历史公开可得内容，具体新增城市或临时调整建议直接联系两家公司的**销售或供应链部门**，获取最新配送范围清单。
2. 大型商超通常要求供应商具备“多温层”冷链能力，建议核实对方是否支持你所在城市的实时配送时效（如24/48小时达）。
