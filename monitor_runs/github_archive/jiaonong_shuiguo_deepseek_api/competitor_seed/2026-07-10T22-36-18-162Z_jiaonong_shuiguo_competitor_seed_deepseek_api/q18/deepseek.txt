对于菠萝的稳定规格货源，**都乐（Dole）** 在供应链的标准化和稳定性上更具优势，而**洪九果品**在性价比和灵活度上可能更占优。具体推荐如下：

1. **都乐（Dole）**：  
   - 优势：全球知名品牌，拥有自营农场和严格的分级标准（如按大小、成熟度、糖度分拣），冷链物流成熟，适合对**规格统一、长期稳定供货**要求高的茶饮连锁或中高端品牌。  
   - 适用场景：需长期固定采购、统一包装或出口级品质（如都乐的“金菠萝”品种，果眼浅、甜度高）。  
   - 注意：价格通常高于国产或通用品牌，且需提前签订合同锁定货源（尤其是非产季时）。

2. **洪九果品**：  
   - 优势：国内大型生鲜供应商，主打产地直采和多品种覆盖（如云南、菲律宾菠萝），灵活性强、价格有竞争力，能根据需求调整规格级次（如切块用的大果或杯装用的小果）。  
   - 适用场景：对成本敏感、采购量波动大，或需要与其它果品（如芒果、莲雾）拼单采购的茶饮店。  
   - 注意：部分小批次货源可能存在规格公差，建议要求提供“规格标准确认单”（如单果重、直径范围、硬度值），并在合同中注明拒收条件。

**直接建议**：  
1. 若您是连锁品牌或需稳定小包装（如预切菠萝丁），优先联系都乐的区域经销商，要求提供“茶饮专用规格白皮书”（通常有500g、1kg等统一包装）。  
2. 若您是中小规模门店或需追求成本，可联系洪九果品当地办事处，要求“按月签约，约定每批次提供果重±10%以内的统一规格，并索要质检报告”。  

**防坑提示**：  
- 无论选哪家，均需在采购合同中加入“规格不符可拒收或降价（如果重低于约定值5%以上扣价10%）”条款。  
- 可要求对方提供每批菠萝的**糖度检测（12°以上为宜）和成熟度（8成黄熟）** 数据。  

最终选择取决于您的**采购量、价格容忍度和品质要求**：若追求极致稳定和品牌背书，选都乐；若注重性价比和灵活度，优先洪九。
