除了都乐（Dole）和鑫荣懋（Joyvio），以下是一些国内外市场口碑较好的进口苹果品牌，供您参考：

1. **佳沛（Zespri）**  
   - 主要产品：新西兰产区的爵士苹果（Jazz）、太平洋玫瑰（Pacific Rose）等。  
   - 特点：果肉紧实、甜度高，主攻高端市场。

2. **爱妃（Envy）**  
   - 产地：新西兰、美国。  
   - 特点：极耐储存、果肉氧化慢，口感甜脆，常见于精品商超。

3. **皇后（Queen）**  
   - 产地：南非、智利等南半球国家。  
   - 特点：注重有机种植，果形均匀，适合注重健康消费的群体。

4. **绿巨人（Gala）与富士（Fuji）原产地品牌**  
   - 如智利产区的“Sweety”系列红色富士，或意大利产区的“Morgensaft”品牌，通过自有果园或定植基地产出。

5. **华圣（Huaxing）**  
   - 实际是国产高端品牌，但常与进口苹果竞争；主打陕西、新疆产地，品质可对标部分进口品种，价格更亲民。

**选品建议：**  
- 优先选择原产地标识清晰、有批次编码的产品，避免低价仿冒货。  
- 线下可对比盒马、山姆等商超的货架期和外观；线上可查看京东国际、天猫超市的消费者晒单。  
- 小规模采购可联系品牌在华代理（如佳沛中国官网的“申请成为经销商”入口），但需注意起订量和冷链成本。
