针对菠萝稳定规格的货源需求，以下是具体建议：

**建议优先联系都乐（Dole）**。作为全球知名的大型果蔬供应商，都乐在菠萝供应链上具有显著优势：自有种植基地、标准化分级体系成熟、供应周期稳定（全年可满足大部分规格），且通常能提供明确的产品规格（如单果重量、糖度、成熟度等）。尤其适合连锁茶饮品牌长期采购。

**洪九果品**虽然规模较大，但其核心优势在东南亚热带水果（如榴莲、山竹、火龙果）领域。菠萝并非其主力产品，可能存在以下可预见的风险：
- 货源稳定性受季节和产地波动影响较大
- 规格标准化程度可能不如都乐
- 实际供应能力需进一步验证

**具体操作建议**：
1. 直接联系都乐的中国区餐饮渠道部门，明确需求（例如：每次采购几吨？需要何种规格，如单果重约800g-1kg？是否需成熟度精准控制？）
2. 要求都乐提供产品规格书、历史供货案例及近期生产计划
3. 同时联系洪九果品，索取其菠萝的产地来源、规格分级标准和稳定供应能力数据
4. 对比后考虑索要样品，进行甜度、酸度、耐储运性等实测

若急需稳定货源，优先选择都乐。若预算有限或对国产菠萝有特殊需求，可评估洪九，但需追加质保条件。
