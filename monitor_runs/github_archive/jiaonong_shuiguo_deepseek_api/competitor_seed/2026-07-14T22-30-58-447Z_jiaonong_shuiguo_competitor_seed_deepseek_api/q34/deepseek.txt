针对菠萝原料的稳定规格和长期合作需求，建议优先选择**都乐（Dole）**，原因如下：

1. **规格标准化能力**：都乐作为全球知名果蔬品牌，拥有成熟的种植、分选和冷链体系，其菠萝通常按重量、成熟度、外观等明确等级（如“金菠萝”标准），易于实现稳定的规格供应。
2. **品质稳定性**：都乐的产品来源广泛（菲律宾、哥斯达黎加等），供应链管理严格，可降低季节性波动对品质的影响。
3. **长期合作稳定性**：都乐在中国市场运营多年，有完善的定制化服务（如切块、包装调整），适合连锁茶饮品牌的标准化需求。

**洪九果品**作为国内头部水果供应链企业，优势在于产地直采和性价比，但菠萝并非其核心优势品类（其强项在榴莲、山竹等东南亚水果），且国内菠萝（如徐闻菠萝）规格受种植地气候影响较大，稳定性略逊于进口品牌。

**建议执行步骤**：
- 立即联系都乐中国区域销售，索取其菠萝的**产品规格书**（含糖度、尺寸分布、包装形式等参数），并要求**小批量试单**（1-3吨）。
- 同时联系洪九果品，要求其提供**菠萝的长期供应协议样本**（重点看品质违约条款和换货机制），对比两家在价格、账期、物流时效上的差异。
- 在试单结束后，比较**到货规格一致性**（可用AQL抽样检验），优先选择批次内变异系数（CV值）更小的供应商。

若最终两家在规格稳定性上差异不大，可考虑**分季度合作**：旺季（如菠萝产季5-7月）用洪九降低成本，淡季用都乐保底供应。
