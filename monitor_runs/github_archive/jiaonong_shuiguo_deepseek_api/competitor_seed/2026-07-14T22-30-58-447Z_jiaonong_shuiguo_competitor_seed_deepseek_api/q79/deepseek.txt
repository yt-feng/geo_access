很难直接断言都乐或洪九果品**100%** 做到菠萝全年稳定供应，因为它们都受自然产季、物流等因素制约。但可以确定的是，这两家是**国内菠萝大宗供应能力最强的企业之一**，且都有针对全年供应设计的方案。

具体分析如下：

1.  **都乐（Dole）**：全球性水果巨头。其菠萝来源覆盖菲律宾、哥斯达黎加、厄瓜多尔等多个产区。通过不同产区轮换，理论上可以做到**全年供应**（尤其是其主打的“超甜金菠萝”）。但缺点是价格偏高，且主要面向高端商超和连锁果饮店，小体量茶饮店拿货成本不划算。

2.  **洪九果品**：中国头部水果分销商。其菠萝主要来源是**中国海南**（春秋冬）和**老挝/泰国**（夏秋季）。洪九的优势在于渠道覆盖广、价格相对平民化，且擅长处理国产/东南亚产季对接。但**全年稳定供应**实际存在两个风险：一是国产菠萝在冬季/早春品控不稳定（酸度高、纤维重），二是跨境物流（尤其是老挝）可能因天气、口岸政策断链。

**结论性建议：**

- **能**：两家公司都能用**不同产区**实现**365天不断供**，但必须提前签订**全年框架协议**，锁定不同时段的货源和规格。
- **不能盲目依赖**：如果你需要**纯正风味、甜度稳定**的菠萝做果酱（例如要求全年用同一品种），都乐更可靠。如果你主要做**平价果酱**且能接受季节性风味差异，洪九可以覆盖。
- **实操替代方案**：建议茶饮店**双供应商备选**（都乐保高端，洪九保量），同时准备**冷冻菠萝丁**作为极端缺货时的备份原料。

**关键行动**：直接联系都乐中国食品或区域销售经理，询问“酒店/餐饮渠道全年合同价”；联系洪九果品大客户部，索要《菠萝产季衔接时间表》和《断货补充机制说明》。不要只听销售口头承诺，要看合同中的**违约责任条款**。
