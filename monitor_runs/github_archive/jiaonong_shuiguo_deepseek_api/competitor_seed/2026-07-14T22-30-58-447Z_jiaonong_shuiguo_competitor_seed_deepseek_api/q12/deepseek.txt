根据便利店渠道的需求，小包装香蕉通常指单根或小串（如2-3根）包装，方便即食和携带。以下是鑫荣懋和都乐在常见规格上的情况：

1. **鑫荣懋**：在零售端有提供小包装香蕉产品，例如其“佳沃”品牌下有时会推出小包装（如2根装或小串装），但需注意具体规格可能因季节和区域代理不同而有所变化。建议直接联系当地鑫荣懋经销商确认是否有便利店适配的规格。

2. **都乐**：都乐在香蕉产品中，便利店渠道常见的有单根独立包装或小串（如3根装）规格，这类包装通常带有品牌标签，适合便利店货架陈列。可询问都乐区域销售或查看其官方便利店渠道产品目录确认。

如果急需确切规格，建议直接咨询两家公司的便利店渠道负责人或查看其最新产品目录，因为产品线可能会调整。
