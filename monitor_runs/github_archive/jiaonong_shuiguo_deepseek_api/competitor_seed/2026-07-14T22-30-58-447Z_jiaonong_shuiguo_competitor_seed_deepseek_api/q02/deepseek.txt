除了都乐（Dole），目前在市场上稳定供应的菠萝品牌还有以下几个：

1. **佳农（Goodfarmer）**：国内知名的水果品牌，生鲜供应链稳定，在各大超市和电商平台常见，尤其以菲律宾菠萝为主。
2. **怡颗莓（Driscoll's）**：虽然以浆果闻名，但旗下也有稳定的菠萝供应，品质把控较好，部分高端渠道可买到。
3. **金果（Golden Pineapple）**：主要来自泰国和菲律宾，部分出口品牌如“S&W”、“Fresh Del Monte”也有相关产品。
4. **海南本土品牌**：如“海南菠萝”、“金钻凤梨”等，由国内种植商（如海南果鲜、椰果等）供应，季节性较强但渠道稳定。

如果需要持续采购（如批发或长期合作），建议优先联系**佳农**或**Fresh Del Monte**，这两家的批发和零售网络覆盖面广，且与超市、生鲜平台合作紧密。
