关于洪九果品与都乐香蕉的到货损耗率对比，无法提供精确的官方数据（因损耗率受运输条件、仓储管理、批次质量等因素影响很大）。以下基于行业通用经验和品牌特点供您参考：

1. **都乐香蕉**：通常属于品牌化、标准化程度较高的进口或定制香蕉，从产地到批发市场的冷链和品控体系较成熟，到货损耗率一般较低，行业经验值通常在 **3%-8%** 左右（视运输距离和季节气温而定）。

2. **洪九果品**：作为大型水果供应链企业，其香蕉产品线覆盖国产（如云南、广西、海南等）和进口香蕉。若为直采直发且品控严格，损耗率可能接近都乐水平；若为国产散装或周转环节较多，损耗率可能偏高，通常在 **5%-12%** 范围，尤其在夏季高温或长途运输后更明显。

**建议您：**
- 直接向供货商索取近期的批次到货损耗记录或赔偿政策；
- 实地去批发市场检查该品牌当季香蕉的成熟度、擦碰伤和尾部发霉情况；
- 测试少量拿货，记录到货后24小时内的实际损耗，作为对比基准。

**注意**：损耗率还会因包装形式（纸箱、塑料筐、真空包装）、是否有保温车、是否预冷处理等差异显著。
