关于水果连锁店的香蕉分级标准，**都乐（Dole）** 和 **洪九果品** 作为大型水果企业，通常有内部的企业标准，但具体标准文件（如详细的分级参数）一般不对外公开。以下是根据行业通用实践和公开信息整理的内容：

1. **都乐**：作为国际知名品牌，都乐通常遵循或参考国际行业标准（如欧盟或美国USDA标准），主要根据香蕉的**成熟度、果皮颜色、长度、弯曲度、表面瑕疵**等指标进行分级。例如，果皮颜色分为1-7级（绿到全黄），果身长度通常要求≥14厘米，无严重疤痕或腐烂。具体标准可能因销售渠道（如超市、批发）不同而调整。

2. **洪九果品**：作为国内头部水果企业，洪九果品执行类似的分级制度，但标准可能更贴近国内消费习惯。通常根据**果指长度、整齐度、成熟度（如青熟、微黄、全黄）、表面损伤程度**等分为特级、一级和二级。例如，特级香蕉要求果指长度≥16厘米，无裂口、无黑斑，成熟度基本一致。
