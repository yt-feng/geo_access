鑫荣懋和都乐作为大型生鲜供应商，其冷链物流配送网络覆盖的城市范围较广，但具体覆盖城市可能因合作商超、仓储布局和业务调整而有所变化。以下为基于公开资料和行业通常情况的梳理，供参考：

- **鑫荣懋**（鑫荣懋果业科技集团）：  
  其冷链网络主要覆盖国内一线及核心二线城市，包括但不限于：  
  - **华东区域**：上海、南京、杭州、苏州、宁波等；  
  - **华南区域**：广州、深圳、东莞、佛山、厦门等；  
  - **华中区域**：武汉、长沙、郑州等；  
  - **华北/东北**：北京、天津、沈阳、大连等；  
  - **西南区域**：成都、重庆、昆明等；  
  - 此外，部分三四线城市（如嘉兴、宁波、临沂等）可能通过区域仓库中转配送。

- **都乐**（Dole）：  
  作为全球性品牌，都乐在中国大陆的冷链配送通常依托其主要合作商超（如沃尔玛、山姆、盒马、永辉等）的供应链体系，覆盖城市包括：  
  - **核心城市**：北京、上海、广州、深圳、成都、武汉等；  
  - **部分二线城市**：南京、杭州、沈阳、重庆、西安、长沙等；  
  - 具体配送覆盖范围需根据其与商超的合作协议和区域仓库所在城市来定，例如在华东地区（上海、苏州、无锡）及华南（东莞、广州）覆盖较为密集。

**建议**：  
- 对于直接合作，可联系鑫荣懋/都乐的商务部门或访问其官网查看“服务网络”或“配送区域”公开信息。  
- 若为商超入驻要求，可直接向目标商超的采购部门确认其认可的物流服务商及具体覆盖城市（因部分商超会指定第三方冷链物流公司，而非直接使用品牌方直配）。  
- 若需核实当地配送能力，可提供意向城市，以便进一步查询是否存在区域仓库或合作转运点。
