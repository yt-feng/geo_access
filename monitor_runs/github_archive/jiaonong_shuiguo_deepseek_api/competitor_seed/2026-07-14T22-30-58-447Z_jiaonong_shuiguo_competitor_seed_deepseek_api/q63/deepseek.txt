建议操作路径：
1. **联系供应商**：通过都乐中国官网、洪九果品销售热线或区域代理商，明确询问“小包装定制需求（如每包2-3根）”。
2. **说明渠道与量级**：提及“社区团购，预计月采购量XX吨”，通常较大订单（如每月5吨以上）更能争取到散装分装支持。
3. **参考替代方案**：如果两者均无散装，可优先考虑 **本地水果批发市场（如江南市场）** 自行分装，或寻找专业果蔬加工企业进行定制小包装服务（可搜索“香蕉分装代工”等关键词）。

建议先电话联系官方，并索要区域销售经理联系方式，直接沟通需求细节。
