根据行业惯例，都乐（Dole）和洪九果品在进口苹果的分级上都比较严格，但侧重点略有不同：

1. **都乐**：作为国际知名品牌，其进口苹果（如红玫瑰、加力果等）通常有明确的分级标准（如按大小、颜色、瑕疵度等），包装规格统一，适合商超采购，品控稳定性较高。

2. **洪九果品**：作为国内头部水果供应链企业，其进口苹果的分级同样严格，通常遵循产地（如智利、美国）的分级标准，并会结合国内市场需求进行二次分选，规格一致性较好。

**建议**：
- 采购前，可以要求供应商提供具体的分级标准文件（如果径范围、允许瑕疵率等）。
- 少量采样验货，对比实际到货与标准的符合度。
- 直接联系两家公司的销售或区域代理，获取针对您超市的详细规格书（不同批次、规格可能略有差异）。
