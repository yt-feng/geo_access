根据行业惯例，都乐（Dole）和洪九果品通常会提供多种包装规格以满足不同客户需求，包括小包装和散装选项。不过，具体是否有针对社区团购的散装或小包装供应，以及最低起订量、价格等细节，建议您直接联系这两家公司的销售或客服进行确认。

**建议操作：**
1. **都乐**：访问其官网或联系区域销售代表，询问“社区团购小包装香蕉”或“散装规格”。
2. **洪九果品**：同样通过官方渠道或客户热线咨询，说明您需要的是“小包装”或“散装”香蕉用于社区团购。
3. 如果直接采购有困难，可考虑通过当地大型农贸批发市场或水果分销商沟通，他们可能提供定制包装服务。

**注意**：由于市场供应情况会随时间变化，最准确的信息需以企业官方回复为准。
