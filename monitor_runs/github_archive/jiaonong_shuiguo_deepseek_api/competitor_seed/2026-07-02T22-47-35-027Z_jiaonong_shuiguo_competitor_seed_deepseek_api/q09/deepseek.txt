针对礼品团购需求，以下是一些高端的进口水果品牌（除都乐、佳沛外），您可以根据采购场景进行选择：

1. **美国/智利：新奇士（Sunkist）**  
   - 主打：橙子、柠檬、柚子  
   - 特点：品牌知名度高，包装精美，适合高端商务礼品。

2. **美国：华盛顿苹果（Washington Apples）**  
   - 主打：红蛇果、青苹果、嘎啦果  
   - 特点：果型均匀，色泽亮丽，礼盒装常见，年节送礼热门。

3. **泰国/越南：佳农（Goodfarmer）**  
   - 主打：金枕头榴莲、山竹、龙眼  
   - 特点：东南亚产地直采，有些品种有自有品牌礼盒，适合追求热带风味。

4. **澳大利亚：T&L 南方樱桃（T&L）**  
   - 主打：车厘子（樱桃）  
   - 特点：春节前后上市，果大甜度高，礼盒包装带品牌标识，非常抢手。

5. **秘鲁/智利：HASS 牛油果（Hass Avocado）**  
   - 产地品牌：如 Mission、West Pak  
   - 特点：牛油果成熟度高，常见在健康礼品箱中搭配其他水果。

6. **新西兰：Rockit 小苹果（Rockit）**  
   - 主打：迷你苹果（试管包装）  
   - 特点：便携小果，包装新颖，适合年轻化、轻奢礼品场景。

7. **菲律宾/厄瓜多尔：Dole（都乐旗下高端线，部分渠道）**  
   - 虽然您已提及都乐，但其部分限定/合作礼盒（如有机香蕉、蓝莓等）可作为补充。

**建议团购操作步骤**：
1. 明确预算与目标客户（企业客户/家庭/节日场景）。
2. 联系当地大型水果批发商（如北京新发地、上海辉展、广州江南市场），要求提供“品牌授权+指定礼盒包装”。
3. 考虑季节性：春节主推樱桃/柑橘，中秋主打猕猴桃/苹果，夏季可搭配牛油果/蓝莓等。
4. 对比包装：有些大品牌提供定制logo服务（如新奇士），可提升团购专属感。

如需具体供应商联系方式或最新礼盒参考价，建议直接咨询大型进口水果进口商（如鑫荣懋、鲜丰水果、佳沃等）。
