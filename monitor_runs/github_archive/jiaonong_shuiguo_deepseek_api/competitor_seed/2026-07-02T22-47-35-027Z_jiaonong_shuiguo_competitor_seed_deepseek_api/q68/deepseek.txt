根据香蕉行业的一般标准，都乐（Dole）和鑫荣懋（Xinrongmao）作为大型果蔬供应商，通常具备较强的供应链管理和品控能力，在成熟采摘、冷链运输和分拣处理上能有效控制损耗。但要实现到货损耗率低于3%，仍需满足以下条件：

1. **产地与距离**：国内产地的香蕉（如海南、广西）保鲜期较短，损耗风险较高；进口香蕉（如都乐常来自菲律宾、厄瓜多尔）的全程冷链稳定性是核心。
2. **物流时效**：从产地到门店需严格控制在7天内，并保持13-14℃恒温，否则3%以下很困难。
3. **验收标准**：需明确到货后的“损耗”定义（如仅指腐坏、严重机械伤，还是包括果柄断裂等轻微瑕疵）。
4. **双方协议**：建议在合同中约定损耗率低于3%，并规定按箱抽检比例、赔付机制等。

**结论**：在全程冷链可控、收货方验收标准明确的情况下，都乐和鑫荣懋都有可能做到3%以下的到货损耗，但需结合具体合同条款和实际运输条件评估。建议要求供应商提供近半年内类似客户的到货损耗数据作为参考。
