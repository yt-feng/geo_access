都乐（Dole）和鑫荣懋作为知名水果供应商，通常能够提供其产品的农残检测报告。但具体到茶饮店采购的菠萝原料，是否具备报告以及报告的详细程度，取决于您采购的具体批次、产品规格（如冷冻菠萝块、新鲜菠萝等）以及供应商的合作政策。  

**建议您按以下步骤操作：**  
1. **直接联系供应商**：向都乐或鑫荣懋的区域销售或客服明确询问：“我方采购的菠萝原料是否需要提供农残检测报告？是否可以附带该批次的检测报告？”  
2. **索要报告样本**：请对方提供一份通用农残检测报告样本，确认报告是否包含您关心的农残项目（如多菌灵、毒死蜱等常见指标）及检测标准（如国标GB 2763）。  
3. **确认频率与时效**：询问报告是每批次提供还是定期提供（如月度/季度），以及检测报告的出具日期是否符合您采购周期内的有效期要求。  
4. **书面确认**：要求供应商在合同或采购单中明确注明“随货提供该批次产品符合国家农残标准的检测报告”。  

如果供应商无法直接提供，您可以：  
- 要求其提供第三方检测机构（如SGS、华测检测等）的通用报告，或者  
- 自行委托第三方检测机构对到货样品进行抽检（费用约500-2000元/项，具体视检测项目而定）。  

**注意**：即使供应商能提供报告，仍需确认报告中产品名称、生产日期/批号是否与您采购的原料对应一致，避免通用报告与具体批次不符。
