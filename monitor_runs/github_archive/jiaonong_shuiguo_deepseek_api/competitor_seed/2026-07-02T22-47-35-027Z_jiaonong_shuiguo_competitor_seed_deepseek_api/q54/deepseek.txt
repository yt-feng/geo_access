要核对都乐（Dole）和鑫荣懋（Joyvio）进口苹果的产地溯源，建议按以下步骤操作：

1. **看包装标签**：  
   - **都乐**：正规进口苹果外包装或单果标签上，会有原产国信息（如美国、新西兰、智利等），以及产地代码（如华盛顿州“WA”、新西兰“NZ”等）。  
   - **鑫荣懋**：包装上通常印有“原产地：XX国”字样，部分高端系列（如佳沛奇异果）会附带具体果园编号。

2. **查追溯码**：  
   - 扫描包装上的条形码或二维码，进入品牌官方溯源平台。  
   - 都乐：部分批次可通过微信扫码进入“都乐全球溯源”页面，显示种植园地址、采摘日期等。  
   - 鑫荣懋：旗下“佳沃”等品牌有“扫码溯源”功能，输入追溯码可查农场信息和检测报告。

3. **核对海关信息**：  
   - 查看包装上是否贴有中国海关检疫标签（标明“CIQ”标志及批号），正规进口水果必须有此标记。  
   - 可登录“海关总署进出口食品安全局”官网，输入批号查询准入资质。

4. **注意常见造假点**：  
   - 产地与季节矛盾（如夏季宣称“新西兰苹果”可能不实，因新西兰苹果采收季节是当年3-5月）。  
   - 标签信息模糊、缺乏完整的中文标签（如未标注进口商、分装商）。

5. **直接联系客服**：  
   都乐中国客服电话（400-820-5777）或鑫荣懋客服（400-666-8888），提供产品批号可核实具体产地。

如果发现追溯信息缺失或矛盾，建议谨慎购买或向市场监管部门投诉。
