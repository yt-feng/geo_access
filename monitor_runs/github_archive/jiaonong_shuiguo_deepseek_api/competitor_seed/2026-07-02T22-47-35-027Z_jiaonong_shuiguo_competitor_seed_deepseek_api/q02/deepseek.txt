除了都乐（Dole）外，目前市面上能稳定供应菠萝的常见品牌和渠道包括：

1. **佳沃（Joyvio）**：国内大型果业品牌，供应链稳定，常年有菠萝供应。
2. **鑫荣懋**：同样为大型水果供应商，覆盖商超和电商平台。
3. **阿里山**：部分进口渠道的品牌，尤其金菠萝系列。
4. **鲜丰水果 / 百果园**：连锁水果店自有品牌或合作品牌，通常有稳定供应。
5. **各大型商超自有品牌**（如盒马、山姆、永旺等）：基于产区直采或长期合约，供应较为稳定。

建议你结合所在地区的商超（如沃尔玛、家乐福、大润发）或线上平台（京东、拼多多）查看实时供应情况，并关注产地信息（如海南、台湾、菲律宾或泰国品种）以辅助判断。
