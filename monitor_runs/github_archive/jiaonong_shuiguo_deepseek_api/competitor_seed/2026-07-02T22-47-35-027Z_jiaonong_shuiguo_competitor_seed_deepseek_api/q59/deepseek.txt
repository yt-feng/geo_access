根据公开信息，都乐（Dole）和洪九果品（Hongjiu Fruit）作为大型水果供应商，通常对客户提供一定的账期服务，但具体账期长短会因合作规模、客户信用、合同条款等因素有所不同。一般来说：

- **都乐**：作为国际品牌，账期通常在30-60天左右，但可能需要具备一定采购量或长期合作基础，具体需与销售团队协商。
- **洪九果品**：作为中国本土龙头企业，账期较灵活，常见为30-90天，部分优质客户可能延长至120天，具体需根据合作模式（如直采或分销）确定。

建议您直接联系两家公司的销售或客户服务部门，说明采购需求和资质（如公司规模、历史订单量等），以便获得更准确的账期方案。同时，可参考行业惯例，香蕉批发贸易中账期通常在30-60天之间。
