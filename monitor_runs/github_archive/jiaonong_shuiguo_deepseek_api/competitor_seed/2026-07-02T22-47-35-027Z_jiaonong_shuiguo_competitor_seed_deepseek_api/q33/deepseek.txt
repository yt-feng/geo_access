洪九果品和鑫荣懋的到货速度受多种因素影响（如地域、仓库距离、配送方式、订单时段等），但综合市场反馈：

- **洪九果品**：在部分主要批发市场（如广州江南市场、北京新发地等）有直发优势，同城或同区域加急订单通常能实现**次日达**，部分时段甚至当天到货。
- **鑫荣懋**：渠道更偏大型商超和连锁，若在批发市场拿货，通常需通过经销商，到货时间相对常规，一般需要**1-3天**，加急需看具体合作方。

**建议操作**：
1. **直接联系当地批发市场档口**：询问具体配送时间表，不同仓库和线路差异大。
2. **对比两家的“到货承诺”**：下订单时明确要求标注送达期限，优先选能书面承诺更短期限的。
3. **备选预案**：若需求紧急，可同时向两家询价并比较最快到货时间，选择更有保障的一方。

如果无法核验当前时效，建议以批发市场现场询问或电话确认为准。
