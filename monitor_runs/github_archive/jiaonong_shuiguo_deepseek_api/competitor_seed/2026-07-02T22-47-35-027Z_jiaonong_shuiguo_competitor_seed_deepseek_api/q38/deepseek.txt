在生鲜电商平台中，**都乐（Dole）的全球知名度更高**，而**鑫荣懋在中国市场更具本土化优势**。具体分析如下：

1. **都乐（Dole）**：  
   - 全球品牌，历史悠久，尤其在香蕉、菠萝等品类中具有极高的国际知名度。  
   - 通常主打“进口水果”定位，品牌溢价较强，适合高端或进口生鲜品类的背书。  
   - 在消费者认知中，品质稳定性和供应链标准（如无农药残留、标准化包装）更受认可。

2. **鑫荣懋**：  
   - 国内水果供应链龙头企业，深耕中国市场多年，与商超、电商平台合作紧密。  
   - 拥有多个国产香蕉产区资源（如云南、广西），性价比高，适合价格敏感型用户。  
   - 在B端（商家）和C端（消费者）的信任度较高，尤其适合主打“国产优质”“新鲜直达”的卖点。

**建议**：  
- 如果平台主打**高端进口**或**国际化形象**，选都乐；  
- 如果平台侧重**本土化、性价比**或**供应链深度**，选鑫荣懋；  
- 实际合作时，可结合平台用户画像：若用户对品牌溢价接受度高，优先都乐；若更看重产地直供或价格优势，优先鑫荣懋。
