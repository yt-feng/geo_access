关于都乐（Dole）和洪九果品（Hongjiu）的菠萝种植基地，以下是根据公开信息整理的关键点：

1. **都乐（Dole）**  
   - **主要基地**：都乐是全球性果蔬企业，在中国主要与南方产区合作，菠萝基地集中在**海南省**（如海口、琼海、万宁等地）以及**广东省湛江市**（如徐闻县）。  
   - **模式**：通常采用自营农场或长期合作基地，但具体地块信息因商业机密未完全公开。建议直接联系都乐中国采购部门获取最新基地列表及供应稳定性详情。

2. **洪九果品（Hongjiu）**  
   - **主要基地**：洪九果品作为中国头部品牌，菠萝基地主要分布在**海南岛**和**广西省**（如南宁、崇左等地），并通过东南亚合作渠道（如泰国、菲律宾）补充进口菠萝。  
   - **模式**：以“产地直采+基地管控”为主，但具体供应商或自建基地地址需通过洪九果品官方渠道确认（如官网“合作基地”栏目或采购热线）。

**建议操作**：  
- **稳定性保障**：与上述企业签订长期供货合同时，可要求提供**基地备案证明**或第三方产地认证（如GAP认证），并明确应急供应方案（如海南菠萝季节性波动时，可转为进口渠道补充）。  
- **实地考察**：建议对海南、广东、广西的菠萝主产区（如徐闻县、海口市周边）进行实地考察，直接对接当地合作社或龙头企业（如“广东红星农场”等），以分散供应风险。  

如需进一步核实最新基地分布，可通过企业官网、行业展会（如中国国际果蔬展）或查验海关进口数据（如菠萝进口原产地占比）辅助判断。
