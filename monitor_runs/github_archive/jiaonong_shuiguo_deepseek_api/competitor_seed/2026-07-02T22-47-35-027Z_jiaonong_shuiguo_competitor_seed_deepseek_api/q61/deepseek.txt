1. **直接联系企业**：访问都乐或洪九果品的官方网站、客服或销售代表，询问其企业标准文件或分级流程。
2. **参考行业标准**：中国有《香蕉》（GB/T 19335-2014）等国家标准，可用于基础分级（如按果指长度、外观、成熟度等）。连锁店可在此基础上结合自身需求调整。
3. **自主制定标准**：根据销售场景（如鲜食、烘焙）和客户需求，制定简单可行的分级表（如A级：无瑕疵、长度≥18cm；B级：轻微瑕疵、长度≥15cm）。
