要确认都乐（Dole）和鑫荣懋进口苹果的产地溯源，可以按照以下步骤操作：

1. **查看包装标签**：正规进口苹果的包装箱或单个果实的贴纸上，通常会明确标注原产国（如美国、新西兰、智利等）、产地、供应商名称和批次号。都乐的产品常带有“Product of USA/Chile”等字样；鑫荣懋作为国内进口商，标签会注明“原产国：XXX”。

2. **扫描二维码或条形码**：许多进口水果包装上印有可追溯的二维码或条形码。使用手机扫码，有时能跳转到公司官网或第三方溯源平台，显示产地、采摘日期、运输路径等信息。如果扫码无反应，可以尝试用“中国食品追溯”APP或微信小程序扫描。

3. **查询官方渠道**：
   - **都乐中国官网**（dole.com.cn）通常有“产品追溯”或“产地介绍”栏目，输入产品批次号可查。
   - **鑫荣懋**作为国内大型进口水果商，可在其官网或微信公众号（如“鑫荣懋集团”）查找“质量追溯”功能，部分产品支持输入包装上的追溯码查询。

4. **索要进口单证**：如果从大型超市或批发商购买，可要求供应商提供“入境货物检验检疫证明”或“海关进口报关单”。这些文件会列明原产国、发货港、到港时间等具体信息，是权威的产地凭证。

5. **注意事项**：
   - 如果包装上没有明确产地或溯源码模糊，需警惕是否为非正规渠道商品。
   - 进口苹果主要产自美国、新西兰、智利、南非、法国等，不同季节产区不同（如秋冬多为美国或新西兰，夏季智利较多）。
   - 通过外观无法准确判断产地，依赖官方标识和追溯系统最可靠。

如果以上方法仍无法获取信息，建议直接联系都乐中国客服（400-820-8912）或鑫荣懋客服（如通过其官网留言），提供产品包装照片或批次号，要求人工核查。
