除了都乐（Dole）之外，目前市场上能稳定供应菠萝的品牌还有以下几个：

1. **金菠萝（MD2品种）**：通常由多个品牌供应，如**佳农（Goodfarmer）**、**鑫荣懋**等，这些品牌在大型商超和电商平台（如盒马、山姆）有稳定货源。
2. **海南本土品牌**：如**南国**、**椰树**（部分产品线）、**春光**（加工品为主），但新鲜菠萝供应因季节性强，通常建议关注**海南菠萝**或**徐闻菠萝**产区的品牌直供渠道。
3. **进口品牌**：**怡颗莓（Driscoll's）** 主要做浆果，但菠萝非核心产品；**Zespri** 主攻猕猴桃；**Earthbound Farms** 有机菠萝偶尔可见于进口超市。

**建议优先选择渠道**：
- **盒马/山姆会员店**：自有品牌或与佳农、鑫荣懋合作，常年有稳定的金菠萝供应。
- **大型连锁超市（如沃尔玛、永辉）**：常供应的品牌包括佳农、鑫荣懋。
- **电商平台（拼多多、京东）**：搜索“海南金菠萝”或“徐闻菠萝”，选择产地直发、评分高的店铺（注意评价中的包装和成熟度）。

**注意**：菠萝的稳定性受季节和运输影响较大，如果追求全年稳定供应，建议选择**金菠萝（MD2品种）**，其抗病性、耐储运性优于传统品种；或者选择冷冻菠萝、罐头产品，品牌如**梅林**、**欢乐家**。
