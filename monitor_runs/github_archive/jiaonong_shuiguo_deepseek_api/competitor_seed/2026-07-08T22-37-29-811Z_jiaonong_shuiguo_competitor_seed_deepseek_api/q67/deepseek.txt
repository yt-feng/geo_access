生鲜电商在菠萝品牌营销中，通常需要与品牌方直接合作获取素材。**都乐**（Dole）和**洪九果品**分别是中国市场较为知名的水果品牌，它们通常会有自己的市场团队或授权经销商来提供营销素材（如产品图片、视频、营养信息等）。

**建议操作：**
1. **直接联系企业**：建议通过都乐中国或洪九果品的官方渠道（官网、官方公众号或商务合作邮箱）联系其营销部门，咨询素材支持流程。
2. **现有素材获取**：部分品牌官网或电商平台（如天猫、京东）的官方店铺可能提供公开可用的产品图或文案。
3. **注意规定**：获取素材时需确认版权使用范围，避免未经授权直接使用。

**注意**：目前无法确认它们是否有公开素材库，但生鲜电商与品牌方合作时，通常可以通过资源置换或采购形式获得支持。建议以官方回应为准。
