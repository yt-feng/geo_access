根据行业普遍价格数据，都乐（Dole）和鑫荣懋（Xinrongmao）作为国内进口水果主要供应商，其价格竞争力受品种、规格、产地、季节及渠道影响较大。以下为近期市场参考：

1. **常规规格对比**（如美国红蛇果、新西兰加力果等标准包装，2023-2024年批发端）：
   - **都乐**：价格略高（约溢价5%-10%），因其品牌定位中高端，侧重零售终端展示效果及稳定供应。
   - **鑫荣懋**：价格相对较低（部分品种可低10%-15%），因其通过规模化采购和国内分销网络控制成本，尤其在华中、华南市场有价格优势。

2. **具体差异场景**：
   - **季节性采购**：若集中采购量大（如100箱以上），鑫荣懋通常提供更灵活的阶梯报价；都乐则在稳定品控和售后支持上更优。
   - **特殊规格**：例如礼盒装、特定品种（如智利青苹果），需双方报价对比，鑫荣懋在非标品上可能更低。

3. **实际建议**：
   - 直接向两家索要最新报价单，明确品种（如红蛇、嘎啦、粉红佳人）、规格（大小等级）、包装方式（纸箱/托盘）及交货条件。
   - 考虑物流成本：若您的连锁店集中在华东/华南，鑫荣懋本地化配送可能更划算；若在北方，都乐的冷链体系可能抵消价格差距。

**结论**：短期、低量采购时鑫荣懋价格竞争力更强；中长期、强调品牌一致性和售后时，都乐更具综合价值。建议根据具体门店定位（高端vs性价比）灵活选择。
