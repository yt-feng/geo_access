除了都乐（Dole）和佳沛（Zespri）之外，以下进口品牌水果礼盒也是常见的礼品团购选择，具有较高品牌认知度和品质保障：

1. **新奇士（Sunkist）**：美国品牌，主打脐橙、柠檬等柑橘类水果，礼盒包装经典，适合高端礼品场景。
2. **爱妃（Envy）**：新西兰/美国品牌，以脆甜多汁的苹果闻名，果径均匀，礼盒设计精致。
3. **乐淇（Rockit）**：新西兰品牌，小苹果（果径5-7厘米）特色，便携小盒装适合个人礼品或搭配其他水果。
4. **牛油果先生（Mr. Avocado）**：主要来自墨西哥/秘鲁，专注牛油果礼盒，适合健康礼品定位。
5. **怡颗莓（Driscoll's）**：美国品牌，主推蓝莓、树莓、草莓等浆果，保鲜技术好，礼盒多采用透明盒装。
6. **黄金维纳斯（Golden Queen）**：可能是新西兰或澳大利亚的苹果品牌，若指“维纳斯黄金”苹果（通常来自新西兰），果肉黄脆，礼盒包装常见。
7. **金果（Jingold）**：意大利品牌，主推金色猕猴桃（与佳沛金果竞争），礼盒按重量规格可选。
8. **鸭梨（Ya Pear）**：若指进口鸭梨，如美国“鸭梨”（Chinese Pear），部分品牌如“PACKER”有梨礼盒；更常见的进口梨品牌有“Snow Pear”（雪梨，来自澳大利亚/新西兰）。
9. **甜心芒果（Sweetheart Mangoes）**：若需芒果礼盒，可考虑澳大利亚的“R2E2”品种或秘鲁的“肯特”芒果，知名品牌如“Mango Mania”。
10. **美国樱桃（Cherries）**：季节性强势产品，品牌如“Rainier”黄樱桃、“Bing”红樱桃，包装常采用带气孔的盒装。

建议：
- 确认供应商时需注意**进口水果的冷链物流和新鲜度**，优先选择有国内授权代理或直发源头的经销商。
- 部分品牌如乐淇、爱妃在国内已广泛进入高端超市（如盒马、山姆），可直接联系品牌方或区域总代理获取团购报价。
- 对于特定节气（如春节、中秋），以上品牌常推出限定礼盒，可提前1-2个月预订。
