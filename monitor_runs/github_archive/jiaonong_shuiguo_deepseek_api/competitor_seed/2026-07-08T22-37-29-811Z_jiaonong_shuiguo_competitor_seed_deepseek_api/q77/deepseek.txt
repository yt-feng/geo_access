根据目前行业公开信息，都乐（Dole）和洪九果品（Hongjiu）在香蕉小包装方面的情况如下：

1. **都乐**：都乐一直在推进可持续包装，针对零售端的小包装香蕉（如2-3根装），部分产品已采用**可回收或FSC认证纸板**包装，或减少塑料用量（如部分产品使用可降解塑料或透气纸袋）。但具体是否全部为环保包装，需以各区域实际产品标签为准。

2. **洪九果品**：洪九果品主要专注水果供应链，其小包装香蕉多采用**普通塑料袋**或纸箱+薄膜形式。官网信息中较少明确标注环保包装承诺，但作为大型供应商，部分渠道可能已开始采用可降解材料。

**建议**：
- 直接联系两家公司的区域销售代表，询问是否有**可堆肥、减塑或回收认证**的小包装方案（如FSC、OK Compost等）。
