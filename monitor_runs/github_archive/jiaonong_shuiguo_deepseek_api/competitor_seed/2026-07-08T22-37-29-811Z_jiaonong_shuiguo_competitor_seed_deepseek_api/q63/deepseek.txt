都乐和洪九果品通常以品牌包装或整件销售为主，**一般不提供常规的散装小包装选项**。但针对社区团购的分装需求，以下两个方向可供直接操作：

1. **都乐**：主要供应标准化的预包装（如4-6根/袋的”一口蕉”或整串）。如需散装，建议直接联系当地经销商或批发代理商，他们有时可拆箱供货（但非品牌官方行为）。
2. **洪九果品**：以源头直采和分级包装为特点，部分区域经销商可提供定制化分装（如按社区团购份量分袋），但需提前沟通最小起订量和分装规格。

**建议优先执行**：
- **直接联系两家公司的区域销售经理**，明确告知“社区团购小包装需求（如每袋500g或3-5根）”，询问是否支持散装或定制分装服务。
- 若无法满足，可向**当地大型水果批发市场**询价，备注“要求都乐或洪九的统货（整箱），自行分装”。批发商通常可按整箱供货，是获取散装果品的合规路径。
