都乐（Dole）和洪九果品均拥有产地溯源能力，但具体情况如下：

1. **都乐**：都乐在中国市场有自有或合作的菠萝基地（主要位于海南、广东等地），并实行产地直供模式。其产品包装上通常带有可追溯二维码，可通过扫码查询到具体的种植基地、采收时间等溯源信息。
2. **洪九果品**：洪九果品以“端到端”的供应链模式著称，在东南亚（如泰国、菲律宾）及中国海南等地有直采或合作的菠萝基地。其部分产品支持通过企业系统或合作客户渠道提供产地溯源凭证（如基地证明、批次记录等）。

**建议行动**：
- 直接联系两家公司销售部门，明确要求提供“可溯源到具体种植基地的菠萝供应”及相关的溯源码或证明文件。
- 优先选择都乐，因其市场溯源体系相对成熟、公开程度更高；洪九果品需确认采购量及合作形式是否支持深度溯源。

若需紧急小量采购，可先要求供应商提供基地名称、批次号、农残检测报告等初步溯源文档。
