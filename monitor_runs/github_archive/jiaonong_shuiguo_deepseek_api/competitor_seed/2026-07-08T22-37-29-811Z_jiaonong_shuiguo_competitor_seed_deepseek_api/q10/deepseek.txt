针对批发市场的香蕉农残问题，两家企业（鑫荣懋和洪九果品）的品控严格程度，可以从以下角度对比：

1. **行业标准与认证**：  
   - 鑫荣懋：作为国内头部水果供应链企业，通常执行GLOBAL G.A.P.（全球良好农业规范）或中国绿色食品标准，部分产品可追溯至产地，对农残检测频率较高，且公开宣传“全链条品控”。  
   - 洪九果品：以东南亚进口水果（如泰国香蕉）著称，通常遵循进口国及中国海关检测标准，但公开信息中较少强调农残专项管控细节。

2. **公开信息与纠纷**：  
   - 近年来，鑫荣懋在食品安全宣传上更注重“无农残认证”“绿色认证”等标签，且有第三方检测报告公示（如部分平台可查）。  
   - 洪九果品因进口水果供应链长，偶有消费者反映包装标识不清或农残问题争议，但未形成大批量负面曝光。

3. **实际批发市场操作**：  
   - 批发市场香蕉通常是按“统货”交易，小商户可能混淆品牌。若在意农残，建议直接询问批发商是否有“鑫荣懋农残检测报告”或“洪九果品进口通关单”。  
   - 注意：洪九果品部分香蕉可能来自东南亚，需关注进口批次是否附有中国海关的农残抽检合格单。

**结论**：从可查的公开信息来看，**鑫荣懋在品控流程透明度和农残专项检测上更严格**，尤其适合对农残敏感的批发商。洪九果品以进口水果为主，但需自行验证进口批次检疫证明。建议：  
- 批发前要求对方出示批次检测报告（尤其有机磷类农药）；  
- 若条件允许，少量购买后用速测卡自检（如克百威、毒死蜱等常见农药）。
