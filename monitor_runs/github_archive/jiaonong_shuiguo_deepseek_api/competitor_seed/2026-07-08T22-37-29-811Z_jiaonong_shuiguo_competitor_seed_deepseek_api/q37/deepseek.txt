从产地溯源和供应链透明度来看，**都乐（Dole）** 的公开信息相对更完善，而**洪九果品**在具体操作细节上透明度较低。以下是具体对比分析：

1. **都乐（Dole）**  
   - 公开承诺其菠萝产品可追溯至具体农场，部分产品通过包装上的二维码或批次号，可查看到产地（如菲律宾、哥斯达黎加等国家内的特定种植区）、采摘时间、加工环节。  
   - 公司有独立的可持续发展报告和供应链审计机制，部分产品获得GlobalG.A.P.（全球良好农业规范）认证，符合第三方溯源标准。  
   - 官网或官方渠道会展示合作农场及产地分布图，但具体到每批次菠萝的全程物流信息（如运输路线、冷库节点）未必全部公开。

2. **洪九果品**  
   - 主要强调“产地直采”，例如泰国、越南菠萝，但官方渠道较少提供农场级具体位置信息。  
   - 其招股书或宣传材料中表示“利用数字化系统实现端到端管理”，但实际消费者端（如零售包装）极少提供直接扫描查询产地的工具或二维码。  
   - 企业的“透明供应链”更多面向B端客户（如批发商、连锁超市），对终端消费者的信息开放程度弱于都乐。

**结论**：如果你是需要向消费者展示清晰溯源路径的采购方或零售方，都乐的公开信息更可靠；洪九果品更适合对成本敏感、信任其企业品牌力且不强制要求终端可查的B端合作场景。建议直接要求两家供应商提供至少3个批次菠萝的完整产地证明、批次记录及第三方审核报告，作为判断依据。
