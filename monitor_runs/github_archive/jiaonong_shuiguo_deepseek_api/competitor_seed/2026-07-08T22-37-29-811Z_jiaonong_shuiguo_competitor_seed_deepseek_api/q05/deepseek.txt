针对茶饮店长期合作的菠萝原料需求，**建议优先选择都乐（Dole），洪九果品可作为备选或补充渠道**。

**具体分析如下：**

1. **品质稳定性与标准化：**
   - **都乐**：作为全球领先的水果供应商，菠萝多为标准化种植（如菲律宾、哥斯达黎加等产区），果形、甜度、成熟度控制严格，批次差异小。适合茶饮店对口味一致性的要求，尤其是需要统一风味配方（如菠萝果酱、鲜切果肉）。
   - **洪九果品**：主攻东南亚水果，但更依赖产地直采（如泰国、越南），品质受产地气候、采摘时间影响较大，不同批次可能存在酸甜度波动。适合对风味个性化要求高、能灵活调整配方的门店。

2. **供应链与成本：**
   - **都乐**：供应链成熟，物流冷链体系完善，到货损耗率低（通常在3%以内）。但价格偏高，且可能要求最低订货量（如整箱或托盘起订），适合日均消耗量大、对供应链稳定性要求高的连锁品牌。
   - **洪九果品**：采用“产地+批发市场”模式，单次起订量可更低（如按公斤采购），价格通常低于都乐15-20%。但到货品质受运输、分拣环节影响，损耗率可能达5-10%。适合中小门店或对成本敏感、能接受波动性库存的店铺。

3. **特色需求匹配：**
   - 若主打**鲜切果肉**：都乐金菠萝（MD2品种）果肉厚实、芯软、甜度平均，无需大量人工去芯。洪九的泰国小菠萝风味更浓郁，但需手工修整，人工成本增加。
   - 若制作**菠萝汁**：都乐可直接提供冷冻或浓缩菠萝汁原料，洪九则更适合鲜果现榨（需额外处理纤维）。

4. **突发风险应对：**
   - 都乐全球化种植（菲律宾、厄瓜多尔等），可规避单一产区气候灾害影响。洪九依赖产区（如泰国春蓬府），若遇台风、洪涝可能导致断供。

**执行建议：**

- **短期测试**：先采购都乐的一个发货批次（如1-2周用量），评估出汁率、甜度后，与洪九同规格产品对比（要求采购单上标注产地、成熟度）。
- **长期策略**：
  - **连锁门店（5家以上）**：与都乐签年度框架协议，锁定价格浮动范围（通常±5%），要求提供代加工服务（如切片、冷冻颗粒）。
  - **单店或小店**：以洪九为主（灵活采购），每批到货后快速糖度检测（用手持折光仪），若低于12°Brix则退回或申请降价。同时预留都乐作为应急备选（提前签约，按周提货）。
- **风险控制**：合同中明确要求“农残检测报告”（GB/T 5009标准）和“无催熟剂证明”，尤其是洪九的进口泰国菠萝需核查海关检疫凭证。

**最终判断**：除非您对成本极度敏感或门店日均用量低于20公斤，否则都乐在长期合作中的综合隐形成本（损耗、人工、品控）更低。
