除了都乐（Dole）和鑫荣懋，以下是其他较为靠谱的进口苹果品牌或供应商，供您参考：

1. **佳沛（Zespri）**：虽然以猕猴桃闻名，但其苹果业务也较为稳定，主要供应新西兰品种（如爵士苹果、爱妃苹果），品质较高，尤其适合高端市场。
2. **爱妃（Envy）**：新西兰品牌，受商标保护，口感甜脆、耐储存，适合零售和礼品渠道。
3. **爵士（Jazz）**：新西兰品牌，果肉紧实、酸甜均衡，常与爱妃捆绑供应。
4. **红园（Red Star）**：美国品牌，主要供应嘎啦、富士等品种，性价比不错，适合商超及批发。
5. **普莱姆（Prime）**：加拿大品牌，主打有机或常规嘎啦苹果，口感鲜脆，适合对货源品质要求较高的客户。
6. **金果园（Golden Orchard）**：南美（如智利、阿根廷）的供应商，供货期较长（尤其是反季节），能覆盖国内夏季市场，价格相对较低。

**选择建议**：
- 若需要高端、新奇特品种，优先考虑新西兰的爱妃、爵士或其授权的国内代理商（如鑫荣懋部分品类）。
- 若追求稳定、高性价比的常规品种，可关注美国或南美品牌（如红园、金果园）。
- 推荐直接联系上述品牌的国内总代理（如佳沛中国、爱妃中国），或通过大型进口果商（如鑫荣懋、都乐）的一级渠道调货。

**提醒**：进口苹果受季节和贸易政策影响较大，建议提前询问产地、品种、到货时间及检疫证明，并小批量试货。如果对某一品牌感兴趣，可进一步要求具体的产地合作社或出口商名称以核实真实性。
