对于茶饮店使用菠萝制作果酱并需要全年供应，都乐（Dole）和洪九果品（Hongjiu Fruit）这两家公司都有可能满足需求，但具体能力取决于它们的供应链和业务模式。

### 都乐（Dole）
- **供应链能力**：都乐是全球最大的新鲜水果和蔬菜供应商之一，拥有全球化的种植、采购和物流网络。它在多个热带地区（如菲律宾、泰国、哥斯达黎加等）有菠萝种植基地，能够实现全年供应。
- **适用性**：都乐通常供应新鲜菠萝，适用于茶饮店直接使用或加工成果酱。其大规模生产和稳定供应能力较强，但价格可能偏高。
- **建议**：如果茶饮店需要新鲜菠萝并注重品牌和稳定性，都乐是一个可靠选择。直接联系都乐的区域分销商或B2B部门，确认全年供应计划和价格。

### 洪九果品
- **供应链能力**：洪九果品是中国领先的水果供应链管理公司，专注于从全球产地（如泰国、越南、菲律宾等）进口水果，并在中国分销。它拥有较强的产地直采和仓储物流能力，能提供全年菠萝供应。
- **适用性**：洪九果品既供应新鲜菠萝，也提供加工服务（如速冻或罐装菠萝），更适合茶饮店进行果酱生产。其灵活性和成本控制可能优于都乐。
- **建议**：洪九果品更适合国内茶饮店，特别是需要定制批次和价格优势的场景。联系其业务部门，询问是否提供菠萝原料的加工或定制供应。

### 实际操作建议
1. **明确需求**：确定茶饮店果酱生产所需的菠萝形态（新鲜、速冻、罐头或预切丁），以及每月需求量。
2. **直接联系**：
   - 访问都乐官网或联系其中国区客服，询问年度供应合同和具体条款。
   - 联系洪九果品的销售团队，了解其菠萝供应计划、冷链能力和价格。
3. **备用方案**：如果这两家无法满足全年供应，可考虑其他供应商（如佳沃、百果园）或使用冷冻菠萝作为替代。

**结论**：都乐和洪九果品均具备全年菠萝供应的能力，但都乐更全球化和稳定，洪九果品更本土化和灵活。建议根据茶饮店的预算、规模和加工需求选择，并直接对接以获取具体方案。
