根据我的知识库（截至2025年5月），以下是关于都乐（Dole）和洪九果品在菠萝产地溯源及基地直供方面的信息：

1. **都乐（Dole）**：都乐是全球知名的果蔬品牌，在中国市场主要销售进口菠萝（如菲律宾、哥斯达黎加等产地），以及部分国内合作基地的菠萝。都乐通常有较为完善的供应链和产地溯源系统，但**不一定是自建基地直供**，更多是通过与大型种植基地或农场合作，实现从产地到销售的全程追溯。若您需要基地直供，建议直接联系都乐中国或查看其官网的合作基地名单。

2. **洪九果品**：洪九果品是一家中国本土的水果供应链企业，专注于“端到端”的供应链服务，在全国多地（如海南、广西、云南等）拥有合作或自建的菠萝种植基地。洪九果品强调“基地直采、产地直供”，通常能提供较为详细的产地溯源信息（如具体果园、采摘时间等）。因此，**洪九果品较有可能提供基地直供的菠萝**，并支持溯源要求。

**建议操作**：
- 直接联系洪九果品的销售或供应链部门，明确询问“菠萝基地直供及产地溯源”的具体方案（如是否提供果园产地证明、批次追溯码等）。
- 对于都乐，可询问其菠萝产地是否为中国本土（如海南基地）以及是否支持定制化溯源服务。

注意：由于供应商的合作政策和基地情况会动态变化，建议在联系时要求提供最新的合作基地名单和溯源能力证明。
