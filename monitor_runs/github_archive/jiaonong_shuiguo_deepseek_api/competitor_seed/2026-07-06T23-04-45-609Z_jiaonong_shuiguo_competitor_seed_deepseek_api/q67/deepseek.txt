通常生鲜电商平台与水果供应商的合作会涉及品牌营销支持，包括提供素材（如产品图片、视频、宣传文案等）。都乐（Dole）和洪九果品都是知名的水果品牌，但具体是否提供素材支持，可能取决于双方的合作协议、合作关系以及电商平台的需求。

- **都乐（Dole）**：作为国际知名水果品牌，通常会有专业的营销团队和素材库。如果生鲜电商平台与都乐有正式合作或采购关系，理论上可以协商获取品牌营销素材。建议直接联系都乐的商务或市场部门，确认合作可能。

- **洪九果品**：作为国内水果供应链企业，可能也会为合作伙伴提供相应营销支持。同样，需根据具体合作情况（如采购量、合作模式）来评估。建议联系洪九的客户经理或营销团队了解。

您可以通过以下方式确认：
1. **联系供应商**：直接向都乐或洪九的商务人员询问素材提供政策。
2. **查询合作协议**：查看与这两家供应商的合作合同或邮件记录，是否有素材支持条款。
3. **联系电商平台**：如果平台已有合作，可咨询平台采购或运营团队。

总之，素材提供并非默认服务，通常需要主动沟通。建议优先联系相关负责人，明确需求和合作可能性。
