都乐（Dole）和洪九果品都是国内市场上知名的大型水果供应商，在商品化处理和分级方面较为严格。
具体到进口苹果：

1.  **都乐**：作为国际品牌，其对进口水果的分级标准非常严格。通常按照大小、色泽、果形、糖度等指标进行标准化分选，规格统一性很高，适合对品质一致性有明确要求的采购（如商超、高端渠道）。
2.  **洪九果品**：作为国内头部水果供应链企业，其核心优势在于源头直采和标准化分选。对于进口苹果，其分级流程同样严格，会按市场流通标准（如每箱重量、每箱粒数、果径范围）进行筛选，规格统一性有保障。

**采购建议**：
-   如果追求**国际标准+品牌溢价**（如进入精品超市），都乐是稳妥选择。
-   如果追求**性价比+本土化渠道覆盖**（如批发市场、社区团购），洪九果品同样能做到严格的规格统一，且价格通常更具竞争力。

**实际操作层面**：
建议直接向两家公司的销售或采购部门索取**具体品项（如Gala、Red Delicious等）的规格书**，确认其每个等级对应的**果径、单果重、包装方式**等具体参数，并进行样品比对。严格的分级意味着苹果大小、颜色、成熟度波动范围很小，但无法做到100%完全相同，这一点在签订合同时明确容差范围即可。
