关于菠萝的稳定规格货源，两家公司均有供应能力，但侧重点不同：

**都乐（Dole）**  
- 全球化的水果供应商，菠萝多为菲律宾、哥斯达黎加等产地，规格标准化程度高（如大小、成熟度、糖度）。  
- 适合需长期稳定、出口级品质的客户，通常以整柜（20-40尺）起订。  
- 优点：品控严格，全年供应周期稳定（如菲律宾菠萝主产区）。  
- 缺点：价格偏高，最小起订量大，对中小型客户不够灵活。

**洪九果品**  
- 国内大型水果供应链企业，菠萝以泰国、海南、广东等地为主。  
- 更适合中小批量、多批次采购，可定制规格（如单果重、切割方式）。  
- 优点：价格阶梯灵活，能配合国内茶饮门店的补货节奏（如分箱/散货）。  
- 缺点：部分产区受季节影响（如广东菠萝旺季为4-7月），需确认全年供应能力。

**建议：**  
1. **订单量小且需要频繁补货** → 优先洪九果品，对接其品控部门明确规格（如：单果重1.2-1.5kg、糖度14°以上）。  
2. **大批量、全年稳定供应** → 直接联系都乐中国区销售（如华东/华南办事处），要求提供“茶饮专项规格表”及批次质检报告。  
3. **进一步确认**：要求双方提供近3个月同一品种菠萝的批次规格记录（含果径、糖度、瑕疵率），对比稳定性和售后响应。  

（注：可要求试单，例如洪九支持500箱起试，都乐需整柜试。）
