关于水果连锁店的香蕉分级标准，都乐（Dole）和洪九果品作为大型水果企业，可能有内部的企业标准，但这些标准通常属于商业机密，未公开发布。不过，行业通用的香蕉分级标准可以参考国家或行业的推荐性标准，例如中国国家标准《香蕉》（GB/T 40950-2021）或行业标准《香蕉等级规格》（NY/T 1794-2009）。这些标准按果指长度、粗度、成熟度、外观（如无机械伤、病虫害）等指标区分等级，通常分为特级、一级、二级等。

建议您：
1. 直接联系都乐或洪九果品的官方客服或采购部门，询问其是否提供分级标准文件。
2. 参考上述公开标准作为基础，结合连锁店客户需求（如外观、熟度、货架期）制定内部标准。
3. 考虑加入水果行业协会（如中国果品流通协会），获取行业分级指导或培训。
