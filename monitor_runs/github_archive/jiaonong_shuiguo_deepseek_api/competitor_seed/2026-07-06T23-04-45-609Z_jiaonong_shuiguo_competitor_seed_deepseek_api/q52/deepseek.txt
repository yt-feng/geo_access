针对菠萝这类易损耗水果，**都乐（Dole）**和**鑫荣懋**作为大型供应商品控较为严格，但具体到货损耗率并无官方统一公开数据（会因产地、季节、运输距离、包装批次、终端仓储条件差异较大）。

根据行业经验数据和多家连锁茶饮品牌的实测反馈（非官方承诺值），通常情况如下：

-   **都乐菠萝：** 常规冷链或气调运输下，**到货损耗率一般在 3% - 6%** 左右。其包装（如单果套袋、纸箱或塑料筐）和产地处理标准较成熟，正常情况下的硬损耗（如严重腐烂、霉变）较低，但更可能出现的是**表面轻微压伤或成熟度不均**导致的内部损耗（切开后局部不可用）。
-   **鑫荣懋菠萝：** 与都乐相近，**常规损耗率约在 4% - 7%** 范围。如果批次中选用的是针对茶饮市场的定制规格（如大小分级更严格），损耗可能稍低；但某些走量批次，若运输中堆叠不当或果皮较薄，**机械伤（碰伤、压伤）导致的水烂和褐变**可能略高于都乐。

**关键建议（可直接执行）：**

1.  **要求合同约定上限**：在采购谈判/合同中，明确写入**“到货损耗率（含严重腐烂、霉变、机械伤导致不可用）不得超过5%”**。超出部分按比例扣减货款或要求补货。这是行业常见操作。
2.  **执行“开箱全检”**：每次到货必须按**开箱比例（如10箱抽1箱）** 检查底部和中间层（泡棉、纸板间）。重点关注**果蒂处是否发霉、果身是否有软烂（非轻微磕碰）**。拍照留证。
3.  **追加“内部损耗”条款**：菠萝切开后，若发现**严重黑心、水心、烂心超过2%**，应在8小时内向供应商提供切果视频报损。这种“隐藏损耗”通常不在简单的目视目测里，务必定入质检条款。

**最直接获知稳定数据的方法**：请两家公司的区域销售经理或客服，针对**你所在城市（冷链距离）**和**你指定的采购批次规格**，提供一份**近30日同规格产品到该仓库的历史平均损耗率报告**。他们有后台数据，通常愿意提供（作为比价/合作条件）。
