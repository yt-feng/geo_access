根据市场反馈和行业口碑，都乐（Dole）和鑫荣懋都是国内较为知名的进口香蕉供应商，但侧重点略有不同：
1. **都乐**：全球品牌，历史较长，香蕉品控和冷链物流比较成熟，供应相对稳定，但终端价格通常偏高。
2. **鑫荣懋**：国内水果供应链头部企业，与多家国际产地（如菲律宾、厄瓜多尔）有长期协议，近年进口香蕉的批次稳定性和价格竞争力有不错表现，且国内仓储配送网点更密集。

**建议判断依据**：
- 如果超市定位中高端、客单价较高，可优先考虑都乐，品牌背书和包装统一性更强。
- 如果注重进货成本、要求灵活调货或需要本地化配送服务，可优先试采鑫荣懋，并索要最近的批次质检报告和供货周期承诺。
- 最稳妥的方法是：联系两家**索要最近3个月的供货批次记录、到货损耗率数据以及针对你超市的合同价格浮动条款**，并安排同规格小批量试单对比实际到货质量。

另外，进口香蕉市场受海运、产地气候影响较大，建议同时预留1-2家备选供应商（如佳农、金果等），避免单一依赖。
