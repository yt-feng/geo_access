除了都乐（Dole）之外，以下品牌在国内渠道也具备稳定的菠萝供应能力（主要涵盖鲜果、加工品和特定产区品牌）：

1. **金菠萝（金菲/Brand of Fresh Del Monte）**  
   - 全球供应链成熟，常见于商超和生鲜平台，主打无眼菠萝、甜度较高。

2. **佳沃（Joyvio）**  
   - 国内知名水果品牌，依托国内产区（如海南、广东）和进口渠道，品控较稳定。

3. **海南热带水果产区品牌（如“海吉岛”“南国”）**  
   - 以海南本地菠萝为主，适合社区团购和电商直发，季节性但保供较好。

4. **菲律宾 Del Monte（德尔蒙特）**  
   - 与都乐同属全球大牌，天猫/京东有旗舰店，供应稳定。

5. **Sysco 或 Direct Source**  
   - 针对B端（餐饮、酒店）的供应商或自有品牌，适合批量采购。

6. **泰国进口品牌（如“金枕头”系列）**  
   - 侧重小菠萝、手撕菠萝品类，渠道集中于山姆、盒马等精品超市。

**建议：** 核心渠道如盒马、叮咚买菜等平台上可常搜索“菠萝”筛选品牌，关注“产地直采”和“冷链配送”标识。若需稳定长期供应，可优先选择有国内自有基地或长期进口合作的品牌（如佳沃、金菠萝）。
