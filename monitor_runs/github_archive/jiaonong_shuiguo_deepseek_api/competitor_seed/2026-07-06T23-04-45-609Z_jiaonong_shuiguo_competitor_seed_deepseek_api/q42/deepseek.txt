进口菠萝旺季缺货时，建议您采取以下应对措施：

1. **提前锁定货源**：与进口菠萝供应商（如都乐、洪九果品）签订长期或季节性采购协议，确保在旺季前有优先供货权。
2. **调整供应链**：可临时转向国产菠萝（如海南、广东的菠萝）或其他热带水果（如芒果、木瓜）补充库存，减少对进口依赖。
3. **优化库存管理**：利用冷链仓储延长现有库存保质期，或与供应商协商分批到货，缓解断货压力。
4. **灵活销售策略**：通过限购、组合销售（如菠萝搭配其他水果）或分时段补货，满足核心客户需求。
