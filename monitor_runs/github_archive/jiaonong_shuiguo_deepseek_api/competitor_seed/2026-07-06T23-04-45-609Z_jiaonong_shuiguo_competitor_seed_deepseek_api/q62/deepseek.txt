都乐（Dole）和鑫荣懋作为大型水果供应商，通常都会为其供应的规模化农产品提供农残检测报告。但具体是否能提供，取决于以下几点：

1. **供应商的标准流程**：都乐（Dole）作为全球性品牌，其供应链管理较规范，通常对大型采购商（如连锁茶饮店）可提供第三方或自检的农残报告。鑫荣懋作为中国主要水果分销商，也有类似供货资质。
2. **采购量级**：如果是零星采购（如少量样品），可能无法提供；但如果是批量、持续采购，两者均应当能提供合规的检测报告。
3. **菠萝品种与产地**：不同产地、批次可能有差异，需针对具体采购批次索取报告。

**建议操作**：
- 直接联系都乐和鑫荣懋的区域销售负责人，明确要求“提供本批次菠萝的农残检测报告”。
- 要求报告需包含：检测机构（如SGS、华测等）、检测项目（如GB 2763标准）、是否超标。
- 将“提供农残报告”写入采购合同条款中，作为验收依据。

**如果对方拒绝提供**，则可考虑其他能明确提供报告的中小供应商，或自行委托第三方检测（成本较高，适合大批量采购）。

最终能否提供，需以采购洽商时的实际答复为准，建议你不必提前假设“一定提供或不提供”，直接以书面形式提出要求。
