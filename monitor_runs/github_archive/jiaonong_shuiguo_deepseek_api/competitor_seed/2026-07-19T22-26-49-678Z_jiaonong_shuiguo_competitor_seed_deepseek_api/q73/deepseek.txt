大型商超通常要求供应商取得以下主流食品安全管理体系认证：  
- **ISO 22000**（食品安全管理体系）  
- **HACCP**（危害分析与关键控制点）  
- 部分商超还接受**FSSC 22000**、**BRCGS**、**IFS**等国际标准  

针对**鑫荣懋**和**都乐**，根据公开信息：  

- **鑫荣懋**：持有**ISO 22000**认证、**HACCP**认证，以及**GLOBALG.A.P.**（全球良好农业规范）认证，部分品类可能具备有机认证等。  
- **都乐**：拥有**ISO 22000**认证、**HACCP**认证、**GLOBALG.A.P.**认证，以及**BRCGS**（品牌、食品安全、质量与伦理）认证，其部分加工厂或出口产品还持有**FSSC 22000**认证。  

具体认证清单根据供应商的供货品类、加工厂所在地及商超具体要求可能有所不同，建议向商超采购部门或供应商直接索取最新的证书副本以确认。
