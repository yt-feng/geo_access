通常情况下，大型果蔬供应商如都乐（Dole）和鑫荣懋会为批量采购的客户（特别是茶饮店、餐饮连锁等商业客户）提供**农残检测报告**。具体能否提供需确认以下情况：

1. **针对商业客户**：两家公司均有完善的供应链质量管理体系，**主动向签约或批量采购的客户提供第三方（如SGS、CTI等）的农残检测报告**，以确保符合食品安全标准。例如都乐会对菠萝等产品进行常规农残筛查。
2. **散客/小批量采购**：若您只是零星采购（如零售购买），可能无法直接获取报告，需与销售代表确认是否有通用报告（如批次检测报告）。

**建议操作步骤**：
- **直接联系销售/客服**：拨打都乐或鑫荣懋的官方客服或对接本地经销商，询问“申请菠萝农残检测报告”的具体流程。通常需提供采购量、合同或意向订单。
- **索取“批次检测报告”**：告知对方您需用于茶饮店原料合规审核，要求提供**对应批次的第三方检测报告**（含检测项目、标准、结果）。
- **自行送检备选**：若对方无法提供（例如个体小商户），可自费送样至专业检测机构（如华测、中检）做常规农残项目（如有机磷、菊酯类），费用约几百元，3-5天出报告。

一般正规供应商（尤其是大型企业）均会配合客户需求提供合规文件。若采购量较小，建议优先联系本地经销商或阿里巴巴上的授权分销商，说明用途后索取。
