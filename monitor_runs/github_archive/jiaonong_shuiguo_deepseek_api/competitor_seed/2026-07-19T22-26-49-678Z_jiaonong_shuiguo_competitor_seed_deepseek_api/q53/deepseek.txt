针对您提到的都乐和洪九果品，两家公司均可签订长期供货合同，但需根据您的具体需求评估合作条件：

1. **都乐（Dole）**：作为国际大型水果企业，具备稳定的香蕉供应链和标准化品控能力，通常支持签订长期框架协议。建议直接联系其中国区业务部门，明确需求量、质量标准（如规格、成熟度）及价格调整机制（如随行就市或固定周期调价）。

2. **洪九果品**：作为国内头部水果供应链公司，覆盖进口与国产香蕉，灵活度较高。可协商签订长期合同，尤其适合对区域配送、小批量高频次补货有需求的连锁餐饮。需重点关注其产地直采能力、冷链物流时效及价格波动条款。

**操作建议**：
- 提前准备近3-6个月的历史用量数据，作为合同谈判基础；
- 明确合同关键条款：最低/最高供货量、品控标准、违约赔偿、不可抗力处理（如台风影响产区）；
