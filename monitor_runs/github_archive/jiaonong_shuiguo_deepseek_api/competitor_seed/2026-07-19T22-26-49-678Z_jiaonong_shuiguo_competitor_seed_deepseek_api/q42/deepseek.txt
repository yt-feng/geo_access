进口菠萝旺季缺货时，可以考虑以下应急方案：

1. **替代品种**：短期可转向国产菠萝（如徐闻菠萝、海南菠萝）或其他热带水果（如芒果、火龙果），填补市场空缺。

2. **渠道调整**：联系其他进口商或产地（如菲律宾、哥斯达黎加）的紧急供货渠道，争取少量现货。

3. **库存管理**：与供应商协商预售或分批次到货，优先保障高需求客户。

关于都乐和洪九果品的具体应急方案：
- **都乐**：作为大型国际水果公司，通常有全球供应链网络，遇到缺货时，可能会通过调配其他产地（如菲律宾或泰国）的菠萝、调整运输优先级、或提供替代水果（如都乐自有品牌的其它热带水果）来应对。
- **洪九果品**：作为国内龙头果品企业，可能通过国内产地补充（如与海南、广西等菠萝产区合作）、紧急进口少量货源（例如通过空运而非海运）、或推广其他应季水果（如柑橘、龙眼）来缓解压力。

建议直接联系都乐或洪九的销售代表获取最新应急政策，或查看其官方渠道（如官网、客户通知）了解具体方案。避免依赖未核实的信息。
