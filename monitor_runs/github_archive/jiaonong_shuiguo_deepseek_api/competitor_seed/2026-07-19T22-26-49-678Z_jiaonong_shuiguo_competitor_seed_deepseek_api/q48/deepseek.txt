大型商超（如沃尔玛、山姆、盒马、永辉等）对供应商的食品安全认证有严格准入要求。鑫荣懋和都乐作为国内头部水果供应链企业，通常持有以下核心资质（以下信息依据行业通行要求及企业公开信息整理）：

### 一、鑫荣懋（鑫荣懋果业科技集团）

- **必持资质**  
  - **BRCGS食品安全全球标准**（A级或B级）：英标认证，全球零售端（尤其是山姆、沃尔玛）的通用门槛。  
  - **ISO 22000** 或 **HACCP**：国内商超基础准入要求。  
  - **GlobalG.A.P.**：全球良好农业规范认证，覆盖其自营或合作农场。  
  - **绿色食品认证**：部分品牌包装或基地产品持有。  
  - **出口备案/检疫许可**：若涉及进口水果，需具备进口水果资质（冷链相关）。

- **其他加分项**  
  - 各地商超对应的具体版本（如山姆对BRC有星标等级要求）。  
  - 每年接受SGS、Intertek等第三方审核（依据其招投标文件）。  

### 二、都乐（Dole）

- **必持资质**  
  - **GlobalG.A.P.**：全球自有农场及合作农场必备。  
  - **BRCGS**（多数工厂为A级）：都乐上海/广州等包装工厂常备。  
  - **ISO 14001/ISO 22000**：部分下游加工环节持有。  
  - **Sustainability认证**：如Rainforest Alliance（雨林联盟）或有机认证（针对特定单品，如香蕉）。  
  - **HACCP**：基础食品生产资质。  

- **海外进口类特有**  
  - **CIQ入境检验检疫证明**：官方通关文件，批次出具。  
  - **FDA认证**（美国产地品项）或**EU有机标识**（欧盟品项）。  

### 三、两者共同特征、查证建议

- **商超具体门槛**：核心看“大商超准入目录”。例如：  
  - **沃尔玛**通常要求BRC或FSSC 22000（二选一）；  
  - **山姆**严苛度最高，可能要求BRC A级+GlobalG.A.P.认证；  
  - **盒马/永辉**可能只要求ISO 22000+HACCP+产地检疫。  
- **查询核实渠道**：  
  - 直接向双方客服或销售索取“商超认证证书清单”（通常会展示在品牌PPT中）；  
  - 在**鑫荣懋/都乐官网**的“社会责任/质量安全”板块（或通过企业合作邮箱索取）；  
  - 如有中标文件/供货合同，商超的“供应商评审报告”会列出具体认证项。  

**关注点**：关键看其与哪类商超签约（如山姆要求BRC且部分地区需A级，而普通永辉可能只需HACCP+普通ISO）。建议向具体采购方索取“准入认证清单”做精确匹配。
