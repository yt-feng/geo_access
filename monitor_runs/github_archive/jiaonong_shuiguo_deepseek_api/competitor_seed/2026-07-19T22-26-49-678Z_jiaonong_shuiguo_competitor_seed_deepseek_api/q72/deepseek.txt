通常情况下，都乐（Dole）和鑫荣懋（XinRongMao）作为大型水果品牌，一般不会对单家水果连锁店给予“独家代理”权限。它们更倾向于与大型连锁超市、区域总代理或批发商合作，且合作模式多为经销批发而非区域性独家授权。

如果您需要争取独家代理，可以尝试以下路径：

1. **联系官方渠道**：直接联系都乐或鑫荣懋在中国的总部或区域分公司，询问是否有针对连锁店的“独家代理”或“专供”合作政策。
2. **提供规模优势**：强调您连锁店的分销能力（如门店数量、覆盖城市、年采购量等），展示能够达到品牌方的销售目标。
3. **签署排他协议**：若对方无法给到“独家”，可协商“区域独家”（如特定城市或商圈）、特定品种独家（如某规格香蕉），或“优先供货权”等折中方案。

建议您直接致电两家公司官方客服或拜访区域办事处，索取《合作申请表》并提交资质证明。注意，未经官方确认，不要轻信第三方中介声称的“独家代理权”。
