通常来说，都乐（Dole）和洪九果品在香蕉小包装上确有环保相关的探索，但具体是否“有”且“稳定供应”需要您直接向供应商确认。

建议您：

1. **直接联系厂商或经销商**：询问其是否提供“环保材质”的小包装（如可降解塑料袋、纸质包装或无包装简易标签），并索要样品及认证信息。
2. **核查包装标识**：查看包装上是否有FSC（森林认证）、可降解标志（如欧盟OK Compost或中国可降解认证）等。
3. **考虑替代方案**：若当前品牌无合适环保包装，可询问其能否定制可重复使用的周转箱+简易打码，或寻找专注于环保生鲜包装的供应商。

一般情况下，大型果品企业可能会有环保包装选项，但需提前沟通批量定制。
