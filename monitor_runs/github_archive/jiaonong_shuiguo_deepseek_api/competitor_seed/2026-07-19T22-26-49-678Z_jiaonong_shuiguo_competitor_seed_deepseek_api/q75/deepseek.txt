根据行业内的公开资料和一般经验，洪九果品和都乐香蕉在批发市场的到货损耗率存在一定差异，具体如下：

1. **都乐香蕉**（品牌化、标准化程度高）：  
   - 到货损耗率通常在 **2%-5%** 左右。  
   - 原因：都乐采用全程冷链控温、标准化包装（如纸箱带气孔、控熟剂处理），且从产地到批发环节的品控严格，因此运输和仓储中因高温、挤压导致的青熟不均、腐烂比例较低。

2. **洪九果品**（注重性价比，多走产地直采）：  
   - 到货损耗率一般在 **5%-10%** 区间，部分批次可能更高（如遇到运输延误或天气异常时）。  
   - 原因：洪九以采购成本较低、多级批发网络为特点，部分货物可能未采用全程冷链或包装规格不够统一，导致因温度波动、堆叠挤压产生的青斑、水烂现象较多。另外其部分香蕉在产地催熟技术参差，到货后黄熟度不一致可能加速后续零售端损耗。

**关键影响因素**：
- **运输时效与温度**：若当天提货且冷链达标，两者损耗差距可能缩小至1%-2%。
- **季节与产地**：例如夏季高温导致都乐的损耗可能升至4%-5%，洪九可能升至8%-12%。
- **批发市场管理**：若市场堆积、通风差，洪九的损耗可能明显放大。

**建议**：
- 若追求低损耗（且资金充足），优先选择都乐，尤其适合需要稳定品质的超市或连锁店。
- 若对价格敏感并可接受较高损耗（例如计划快速分销、用价格弥补损耗），可考虑洪九，但需实地查看批次黄熟度，并选择到货时间短、包装完好的货源。

（注意：以上数据为行业经验估算，实际损耗需结合具体批次、市场环境及你的采购量验证。）
