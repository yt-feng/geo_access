针对菠萝产地直发，都乐（Dole）和鑫荣懋的物流时效表现如下：

1. **都乐**：  
   通常依托其全球供应链体系，在菠萝主产区（如菲律宾、海南、广东等）设有合作基地或自有农场。其物流模式多采用“产地-区域分仓-直达”的干线运输，时效取决于合作快递或物流商（如顺丰、京东冷链）。  
   **一般情况**：产地直发至一线城市（如北上广深）约需 **2-3天**，二三线城市可能延长至 **3-5天**。注意：都乐多为品牌直营，对冷链和包装要求较高，破损率较低。

2. **鑫荣懋**：  
   作为国内大型水果供应链企业，其菠萝产地主要来自海南、广东、广西等地。物流网络覆盖较广，常采用“产地直采+冷链专线+目的地分仓”模式。  
   **一般情况**：省内或周边省份（如广东至浙江、江苏） **1-2天**，跨省远距离（如海南至北方城市） **3-4天**。鑫荣懋更注重成本控制，部分线路可能混合常温运输，需关注包装密封性。

**选择建议**：  
- 若追求稳定时效和品牌冷链保障，优先选择都乐（尤其热带水果长途运输经验丰富）。  
- 若需要更高性价比、且对物流速度敏感（如短途或批量采购），可考虑鑫荣懋（其区域分仓能力较强）。  

**注意事项**：  
- 具体时效需根据下单时的发货地、目的地、季节（菠萝旺产季物流压力大）及合作物流公司确认。  
- 建议在合作前咨询两家的客服或销售，明确“24小时/48小时到达”等承诺的具体条款（如是否包邮、是否含冷链服务）。  

最后，实时物流查询可通过订单系统追踪，重大异常（如超时、坏果）可依据售后条款协商补偿。
