对于长期合作年单的意愿，通常取决于双方的业务匹配度、供应链稳定性及合作条款。以下是基于行业惯例的分析：

1. **鑫荣懋**：作为国内头部水果供应链企业，其规模化、标准化程度较高，倾向于长期稳定合作。若您的需求量大、账期规范，对方可能更愿意签约。但需注意其客户集中度，大客户合作条款可能较严格。

2. **洪九果品**：更强调灵活性和渠道下沉，对中小客户或定制化需求配合度可能更高。若您的业务模式需快速响应或品类调整，洪九可能更主动。但其对终端市场依赖较强，年单条款需明确价格波动风险分担。

**建议**：
- 直接向两家公司采购或销售团队提出长期合作意向，重点沟通**最低起订量、价格锁定周期、质量纠纷处理机制**。
- 对比双方对**账期、违约赔偿**的接受度——年单本质是风险置换，愿意分担更多价格波动风险的一方通常协议意愿更强。
- 若您的订单占其销量5%以上，可要求对方提供独家区域或品类优先权作为签约条件。

最终结论：**无明显谁能“更愿意”，需根据您提出的具体条款（如最低采购额、调价机制）评估响应速度**。建议同步询价，用竞争性谈判测试诚意。
