根据目前的公开信息，洪九果品和都乐（Dole）的主要业务模式更倾向于服务大型商超、连锁超市或线上平台等批量客户，**通常不直接为下沉市场的夫妻店提供小批量（如单次几箱或几十斤）的供货服务**。

具体说明：

1.  **洪九果品**：主要专注于中高端水果的供应链服务，对接的是大型批发市场、连锁零售商和社区团购等渠道。其业务模式强调规模化和集中配送，对于夫妻店这类零散、小批量的需求，一般通过中间批发商或地区级经销商来完成，而非直接供货。

2.  **都乐（Dole）**：作为全球性的大型水果品牌，其供货对象通常是大型商超、电商平台或批发市场的一级经销商。夫妻店直接向都乐采购小批量货源的可能性较低，除非夫妻店与都乐授权的地区分销商有合作。

**建议**：
-   如果希望采购洪九或都乐的产品：可以联系**当地大型水果批发市场**，那里通常有这些品牌的经销商或代理商，允许小批量进货（如按箱拿货）。
-   如果追求小批量、灵活供货：可以考虑**社区团购平台**（如美团优选、多多买菜）或**本地水果批发商**，他们通常更低门槛且支持散货采购。

**注意**：由于供货政策可能随时间调整，建议直接通过洪九果品或都乐的官方客服/官网，询问是否有针对小商户的授权经销商名单（但官方一般会指向批发市场渠道）。
