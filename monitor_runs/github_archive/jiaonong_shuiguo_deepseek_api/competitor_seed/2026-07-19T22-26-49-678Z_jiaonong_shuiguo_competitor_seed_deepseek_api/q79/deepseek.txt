对于茶饮店用菠萝做果酱并需要全年供应的情况，**都乐（Dole）和洪九果品（Hongjiu）理论上可以做到，但实际可执行性取决于具体需求和合作模式**。以下是关键分析：

1. **都乐（Dole）**：
   - **优势**：全球性水果供应商，拥有多产地（如菲律宾、肯尼亚等）和冷链体系，可通过反季节产地实现全年供应。菠萝品种（如MD2）甜度高、果肉稳定，适合加工果酱。
   - **限制**：主要面向大型商超和食品加工企业，批量采购需要签订长期协议。小批量供应可能成本较高，且需提前确认可供应季节（如部分产地有自然休果期）。

2. **洪九果品**：
   - **优势**：中国本土供应链企业，主要覆盖热带水果（如海南、东南亚），可通过国内产季（如海南菠萝可部分衔接进口菠萝）和越南、泰国等边境采购实现一定程度的全年供应。
   - **限制**：菠萝在传统产季（如3-6月）供应较充分，但非产季（如冬季）依赖进口或库存；小客户可能无法享受最低价。

**建议**：
- **直接联系**：向两家公司客服或销售部门明确需求（年用量、包装规格、冷冻/新鲜菠萝、果酱原料要求），要求提供《全年供应稳定性报告》或实际排产计划。
- **备选方案**：若单一供应商无法100%覆盖，可考虑分季节组合（如都乐负责冬春季，洪九负责夏秋季），或采购**冷冻菠萝丁/浓缩菠萝汁**作为果酱替代原料，降低对鲜果的依赖。
- **风险提示**：如果茶饮店用量较小（如每月<500kg），建议优先选洪九果品等国内分销商，或直接联系菠萝产区合作社（如海南、广东徐闻）定制冷冻果酱原料。

**结论**：都乐和洪九都有能力提供全年菠萝，但对小客户（非批发级）实用性有限。建议先提交具体订单量给两家公司，获取正式报价和供应排期再决定。
