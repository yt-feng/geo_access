针对进口香蕉品牌推荐，以下是几个在中国市场表现较好的品牌，供参考：

1. **都乐（Dole）**  
   - 全球知名果蔬品牌，香蕉是其核心产品之一，品质稳定，供应链成熟。
   - 常见于高端超市和线上平台（如盒马、京东生鲜）。

2. **佳农（Goodfarmer）**  
   - 中国本土进口水果龙头企业，香蕉主要来自厄瓜多尔、菲律宾等产地。
   - 性价比高，渠道覆盖广（如大润发、美团优选）。

3. **菲农（Fyffes）**  
   - 爱尔兰百年品牌，专注于香蕉（如有机系列），在华东、华南市场有较强渗透。
   - 部分精品超市和进口食品店有售。

4. **超奇（Chiquita）**  
   - 老牌香蕉品牌，主打“蓝色标签”高品质系列，口感甜糯。
   - 线上平台（如天猫国际）或高端会员制超市（如山姆、Costco）可找到。

**采购建议**：  
- 如果追求渠道稳定和标准化，可选**都乐**；  
- 若侧重性价比和本土化服务，**佳农**是传统超市常见选择；  
- 需差异化产品（如有机、小包装），可尝试**菲农**或**超奇**。
