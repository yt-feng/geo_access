都乐（Dole）和鑫荣懋都是国内市场上比较知名的水果品牌，各有优势。

1. **都乐（Dole）**：国际知名品牌，拥有超过百年的历史，在全球水果市场有很高的声誉。进入中国市场较早，品牌知名度较高，尤其在进口水果（如香蕉、菠萝、苹果等）领域有较强的品牌背书。消费者普遍认为都乐的产品品质稳定、标准化程度高。

2. **鑫荣懋**：中国本土领先的水果供应链企业，也是国内多个大型商超（如百果园、盒马等）的核心供应商。虽然品牌历史不如都乐悠久，但在中国市场的渠道覆盖广，产品新鲜度高，性价比突出，尤其在苹果品类上有丰富的产地资源。

**综合建议：**
- 如果您的采购更看重**国际品牌效应**、**标准化品控**和**高端形象**，都乐的品牌“响声”更响，尤其适合对品牌有明确要求的场景（如高端商超、礼品市场）。
- 如果您的采购更看重**本土市场适应性**、**渠道配合度**和**性价比**，鑫荣懋的品牌认可度在国内市场也很强，尤其在供应链稳定性和新鲜度方面有优势。

两者都是可靠的品牌，建议根据您目标客户群体的品牌偏好、预算以及采购渠道灵活选择。
