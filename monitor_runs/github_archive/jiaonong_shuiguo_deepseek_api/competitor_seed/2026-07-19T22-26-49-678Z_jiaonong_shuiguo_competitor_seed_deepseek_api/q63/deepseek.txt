针对您提到的社区团购场景，通常情况下，**都乐（Dole）** 和 **洪九果品（HFG）** 这两家大型水果供应商的主营业务更侧重于**标准化预包装**（比如按斤/按袋/按盒的成品包装），而非针对小零售的“散装”原始方式。

以下是具体建议：
1. **都乐（Dole）**  
   - 主要面向商超、电商等渠道，提供**规格统一的预包装香蕉**（如3-5斤/袋或按根售卖的小包装）。
   - **无常规散装选项**，但可以联系其**大宗批发或团购服务**部门，询问是否能根据社区团购需求提供**定制化的小包装**（如定制标签、小箱散货等），出货量通常需达到一定门槛（如百公斤以上）。

2. **洪九果品（HFG）**  
   - 同样以**标准化精包装**为主（如“洪九泰好吃”系列），但因其更贴近国内批发市场，部分区域经销商可能提供**半散装**服务（如整箱卖，箱内无小包装）。
   - **建议做法**：直接联系您所在城市的洪九果品**区域分公司或经销商**，询问是否支持“拆箱散称”或“按斤散卖”，同时确认是否接受团购小量多批次订货。

**核心建议**：
- 如果您需要的是“非品牌散称香蕉”（即产地/品牌不明确），可直接对接当地**一级农产品批发市场**，价格更低且可散装进货。
- 若坚持要品牌货，请直接联系都乐和洪九的**B端销售客服**，提出“小包装散卖”需求，可能需签署试销合同或支付定制的费用。

**快速行动路径**：
1. 在微信搜索“都乐中国”/“洪九果品”官方公众号，查找“商务合作”或“企业购”入口。
2. 拨打都乐400客服（400-820-8811）或洪九果品总部电话（023-6858 8888），转接大宗团购部门。

**注意**：社区团购若单次采购量较大（如200斤以上），上述公司更有可能接受定制服务。小批量散装建议优先考虑本地批发商。
