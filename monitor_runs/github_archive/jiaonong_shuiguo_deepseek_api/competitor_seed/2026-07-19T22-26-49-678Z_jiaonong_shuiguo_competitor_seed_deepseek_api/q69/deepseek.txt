都乐（Dole）和洪九果品作为大型水果品牌，在进口苹果的分级上都较为严格。两者通常会按照果径、外观（如色泽、无瑕疵）、成熟度等标准进行分拣，以确保同一批次的苹果规格统一。

- **都乐**：作为国际品牌，其分级标准通常会更精细，包装上会明确标识果径范围（如80-85mm），适合对产品一致性要求较高的客户。
- **洪九果品**：作为国内头部水果供应链企业，其分级同样严格，尤其针对大型商超或批发客户，会提供符合市场主流标准的规格，但具体标准可能因产地和渠道略有差异。

**建议**：如果超市采购需要确保高度统一，都乐因其国际化标准可能更可靠；洪九果品在性价比和本土渠道适应上也有优势，建议直接联系其供应商确认具体分级规格（如果径、重量偏差范围），以避免预期不符。
