通常情况下，都乐（Dole）和鑫荣懋作为大型水果供应商，其香蕉产品的到货损耗率可以控制在较低水平，但要达到“低于3%”这个具体指标，需要区分以下情况：

1. **都乐**：作为全球领先的香蕉供应商，其供应链管理成熟，在产地采后处理、冷链运输、成熟度控制方面有严格标准。如果餐饮连锁对香蕉的**成熟度（如青蕉或黄蕉）** 有明确要求，且合作方按协议执行（如指定催熟流程、缩短运输时间），到货损耗率可能接近3%甚至更低。但若要求的是**即食黄蕉**，则由于运输及存储期极短，损耗率通常较高，可能难以稳定低于3%。

2. **鑫荣懋**：作为国内大型水果供应链企业，其在香蕉的国产化采购、催熟技术和终端配送方面能力较强。若采购的是**未催熟青蕉**并自行控制催熟环节，到货损耗率可以控制在3%以内；若要求直接配送**已催熟黄蕉**到门店，则损耗率可能受配送距离、门店存储条件影响，通常为3%-5%。

**关键建议**：
- 与供应商签订合同时，需明确“损耗”的定义（如：腐烂、机械伤、黑心病等），并约定到货验收标准（如开箱抽查比例、不合格品赔付方案）。
- 可要求供应商提供其历史到货损耗率数据，或先进行小批量试单（如10-20箱），通过实际到货检测确认能否达标。
- 注意：即使供应商能承诺低于3%的损耗，餐饮连锁自身的验收、存储和周转速度也会影响最终损耗，建议同步优化门店端的香蕉管理流程。
