生鲜电商如果需要菠萝品牌营销支持，**都乐（Dole）和洪九果品通常可以提供相关素材**，但具体情况取决于合作性质和需求：

1. **都乐**：作为全球知名水果品牌，通常拥有成熟的品牌视觉素材（如产品图片、视频、品牌故事等），且愿意与生鲜电商合作推广。建议直接联系其市场部或区域负责人，说明合作意向（如联合促销、社群推广等），一般能获得授权使用的素材或定制化内容支持。

2. **洪九果品**：作为国内头部水果供应链企业，同样具备提供产品宣传素材的潜力，尤其针对特定产区（如海南菠萝、徐闻菠萝等）有产地溯源素材。但需注意，洪九更偏向B端渠道，可能需要通过采购合作或联合活动来获取素材授权。

**建议操作步骤**：
- **明确需求**：界定需要哪些素材（如高清产品图、短视频、营养成分表、品牌Logo等）。
- **对接渠道**：通过官方客服、企业微信或行业展会（如亚洲果蔬展）联系品牌市场团队。
- **谈判合作**：确认素材使用权限（是否可修改、是否需标注来源、是否涉及费用），或提出联合营销方案（如共同承担推广成本，换取素材支持）。

**注意事项**：
- 如果仅是常规销售而非深度合作，部分品牌可能不会提供免费素材，但可协商通过采购数量或分成比例换取。
- 建议同时与**佳沛、百果园、鑫荣懋**等其他水果品牌沟通，扩大素材获取范围。

总之，直接联系品牌方确认是最高效的方式，同时准备好具体的合作方案可以提高成功率。
