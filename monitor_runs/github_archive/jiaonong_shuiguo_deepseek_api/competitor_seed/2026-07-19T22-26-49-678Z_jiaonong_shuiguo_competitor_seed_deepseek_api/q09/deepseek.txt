除了都乐（Dole）和佳沛（Zespri），以下是其他值得推荐的进口品牌水果礼盒选项，适合礼品团购场景：

1. **新奇士（Sunkist）**  
   美国品牌，主打橙子、柠檬、葡萄柚等柑橘类水果，品质稳定，包装精美，适合高端礼盒。

2. **爱妃（Envy）**  
   新西兰/美国品牌，以苹果为主，口感脆甜、抗氧化，红色包装礼盒有高级感，适合送礼。

3. **乐淇（Rockit）**  
   新西兰品牌，小苹果（筒装），主打便携和精致外观，适合小礼品或混合礼盒搭配。

4. **丹烁（Dazzle）**  
   新西兰/澳大利亚品牌，苹果品种，颜色红亮、果肉紧实，礼盒外观吸引人。

5. **怡颗莓（Driscoll's）**  
   美国品牌，以蓝莓、树莓、草莓等浆果为主，适合组合礼盒（需冷链运输），适合注重健康的高端客户。

6. **金果（Jingold）**  
   意大利品牌，主打猕猴桃（绿肉和黄肉），品质稳定，可替代或补充佳沛产品线。

7. **哈斯（Hass）牛油果**  
   主要来自墨西哥或加州，牛油果礼盒适合健康/轻食人群，可搭配其他水果或食品。

8. **金都一号**（国产但不限于）  
   虽然是中国品牌，但“金都一号”火龙果在高端礼盒中常见，且东南亚进口火龙果品牌如**越南红心火龙果**也受欢迎。

**团购建议：**
- 若客户偏好混合礼盒，可搭配新奇士橙子 + 爱妃苹果 + 怡颗莓（需冷链）。
- 注意季节性：进口水果如智利车厘子（属于季节限定，适合春节礼盒）。
- 建议向供应商索要报关单或原产地证明，以体现品牌正品保障。
