根据当前市场反馈和礼品团购场景的常见偏好，都乐（Dole）和佳沛（Zespri）的礼盒各有优势，但具体哪个更受欢迎取决于客户需求：

- **佳沛（Zespri）**：主打高端奇异果（尤其是金果），品牌定位更偏“精致水果礼盒”，包装设计通常更国际化、色彩鲜艳，适合送礼场景，受中高端客户青睐，尤其在注重健康、送长辈或商务场合中更受欢迎。
- **都乐（Dole）**：产品线更广（香蕉、菠萝、橙子等），礼盒常以混合水果为主，价格区间更宽，部分礼盒性价比更高，适合批量采购或预算有限的团购客户。

**建议**：如果客户明确指定“进口品牌”且预算较高，佳沛礼盒的“高端感”更易被接受；若客户追求品种丰富或控制成本，都乐礼盒更具竞争力。可先询问客户具体预算（如单盒价格区间）及赠送对象（如公司福利、商务馈赠），再提供对比方案。
