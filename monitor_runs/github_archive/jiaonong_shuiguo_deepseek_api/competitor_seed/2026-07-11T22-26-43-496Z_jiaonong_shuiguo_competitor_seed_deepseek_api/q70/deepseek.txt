都乐（Dole）和鑫荣懋（Xinrongmao）均为大型水果供应商，通常具备一定的产地溯源和品控能力，但具体情况需根据合作需求和产品批次确认：

1. **都乐（Dole）**  
   - 都乐是全球知名水果品牌，其香蕉产品通常有完善的供应链可追溯系统，可以提供产地信息（如菲律宾、厄瓜多尔等种植基地）和品控证明（如GlobalG.A.P.、有机认证等）。  
   - 建议直接联系都乐中国销售或客户服务，说明您需要的“产地溯源”和“品控证明”具体类型（如批次追溯、检测报告、认证证书等），以便确认能否提供对应文件。

2. **鑫荣懋（Xinrongmao）**  
   - 鑫荣懋是中国大型果品企业，主营进口和国产水果，其香蕉合作产地包括菲律宾、云南等。该公司通常提供基础溯源信息（如原产地、到货批次）和品控报告（如农残检测、检疫证明）。  
   - 需要提前与采购团队确认：是否支持按客户要求出具定制化品控证明（如第三方检测报告）、是否公开溯源二维码等。

**建议行动步骤**：
- 直接联系两家公司的销售或客服部门，明确要求：  
  - 产地溯源文件（如原产地证明、种植园信息、物流路径）  
  - 品控证明（如第三方检测报告、内部质检记录、认证副本）  
- 询问是否支持定制化追溯（如通过二维码或系统查询批次信息）。  
- 若需长期合作，可要求提供样品试点追溯流程。  

**提示**：大型供应商一般可以满足合规溯源需求，但具体呈现形式可能不同；建议以书面形式（如邮件）明确要求，避免口头沟通误差。
