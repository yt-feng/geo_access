针对您提到的鑫荣懋（Xinrongmao）和都乐（Dole）两家大型水果供应商，在大型商超（如沃尔玛、家乐福、永辉、盒马等）的准入要求下，通常会要求持有以下核心食品安全认证及资质。以下是两家公司的常见资质汇总：

### 一、行业通用基础资质（两者均具备）
- **营业执照**：合法经营主体。
- **食品经营许可证**：或“食品生产许可证”，涉及分装或加工环节。
- **进出口相关资质**（如需）：如海关备案、进口水果检疫许可证（植物检疫证明）、原产地证。
- **产品质检报告**：每批次或定期送检的第三方检测报告（符合GB 2763农残标准）。

### 二、重点食品安全认证

| 认证类别 | 鑫荣懋常见资质 | 都乐常见资质 | 商超要求缘由 |
| :--- | :--- | :--- | :--- |
| **国际通用体系认证** | **GLOBALG.A.P.（全球良好农业规范）** <br>**BRCGS（食品安全全球标准）**<br>（部分自有基地及加工厂持有） | **GLOBALG.A.P.**（作为全球供应商必备）<br>**ISO 9001 / ** **HACCP** （加工环节）<br>**有机认证**（特定产品，如欧盟有机、美国USDA） | 证明源头种植、加工、仓储环节符合国际食品安全及质量管理标准。 |
| **国内官方/行业认证** | **绿色食品认证**（部分产品，如猕猴桃、苹果）<br>**中国良好农业规范（ChinaGAP）** | **绿色食品认证**（如都乐香蕉、菠萝）<br>**中国有机产品认证**（部分产品） | 满足国内“绿色、有机”标签法规及商超对特色认证商品的需求。 |
| **特定产品/追溯** | **- 供港澳水果备案基地**（部分产品）<br>**- 农产品地理标志**（合作基地产品） | **- 多个自有品牌认证**（如Dole Organic、Dole Sunshine）<br>**- 可持续认证**：Rainforest Alliance（雨林联盟，热带水果） | 商超要求可追溯、可持续，尤其对高风险水果（如香蕉、浆果）更严格。 |
| **社会责任/环境** | **- Sedex（供应商商业道德数据交换）** 会员<br>**- 社会责任审核**（部分商超要求SMETA） | **- Sedex / SIZA（可持续农业倡议）** 审核<br>**- 全球劳工责任项目** | 大型商超如沃尔玛、TESCO近年强制要求社会责任审核。 |

### 三、特定商超的个性化要求（举例）
- **沃尔玛**：会额外要求供应商完成 **Walmart Sustainable Package**（可持续包装承诺）及年度 **Responsible Sourcing Audit**。
- **Costco（开市客）**：对供应商有更严苛的 **Costco Quality Assurance Program**，通常要求持有 **GFSI（全球食品安全倡议）** 认可的认证（如BRC、FSSC 22000）。
- **盒马 / 山姆**：关注 **全链路溯源体系**，鑫荣懋和都乐均提供从产地到货架的二维码溯源，且需接受月度飞行检查。

### 四、总结：两者对比
- **相同点**：都持有 **GLOBALG.A.P.** 源头认证 + **HACCP** 或 **ISO 22000** 加工厂认证，且均满足大型商超的基础“三证”（营业执照、许可证、检测报告）。
- **差异点**：
  - **都乐**更强调 **全球统一标准**（如Rainforest Alliance、USDA Organic）和**自有品牌认证**，国际化业务更依赖GFSI认证体系。
  - **鑫荣懋**在**国内绿色/有机认证**及**中国良好农业规范（ChinaGAP）** 上投入较多，同时受益于**粤港澳大湾区供港基地优势**，相关检疫资质更全。

**建议**：在与商超合作时，可以直接要求对方提供 **《供应商资质清单》** 或 **《合格供应商证明文件》**，重点核对 **外部第三方认证证书的有效期** 及 **是否处于商超白名单名录中**。两者通常都能满足95%以上商超要求，仅个别“超高端”渠道（如某些有机食品专柜）会额外指定认证。
