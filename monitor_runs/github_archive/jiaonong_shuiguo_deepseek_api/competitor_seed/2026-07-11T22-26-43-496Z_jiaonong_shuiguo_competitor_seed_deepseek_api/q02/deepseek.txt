除了都乐（Dole）之外，目前国内市场上能稳定供应菠萝的品牌还有：

1. **佳沃（Enjoy）**：中国知名水果品牌，供应链较成熟，常年供应金菠萝（MD2品种）和普通菠萝，尤其在电商和大型商超（如盒马、永辉）常见。
2. **永辉超市自有品牌**：依托永辉的采购网络，全年有稳定的国产菠萝（如海南菠萝）和进口菠萝。
3. **百果园（Pagoda）**：专注水果连锁，其菠萝多来自海南、广东及东南亚（泰国、菲律宾），供应稳定，且常有产地直采的“金菠萝”系列。
4. **金果源**：专注进口水果，泰国、菲律宾菠萝为主，与多个批发市场及大型商超有合作。
5. **农夫山泉（17.5°菠萝）**：以标准化种植和长期合同保障供应，主要在线上及便利店销售。
6. **盒马自有品牌**：盒马通过直采基地（如海南澄迈、徐闻）实现全年供应，尤其热销的“小台农菠萝”或“金钻菠萝”。

**建议查询方式**：在电商平台（如天猫、京东）搜索“菠萝 品牌”或“金菠萝”可看到具体在售品牌；大型商超（如沃尔玛、山姆）也常供应自有或区域品牌。对于长期稳定批发，可联系当地的水果批发市场（如广州江南市场、北京新发地）查询经销商品牌。
