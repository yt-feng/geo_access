针对都乐和洪九果品两个品牌的菠萝性价比比较，建议从以下维度分析：

1. **产地与品种**  
   - 都乐菠萝多为菲律宾进口（如金菠萝），果肉细腻、甜度高，无硬芯，但运输成本高。  
   - 洪九果品菠萝以国内海南、云南产地为主（如徐闻菠萝），性价比突出，但需注意品种差异（部分品种可能存在纤维较粗问题）。

2. **价格与规格**  
   - 都乐金菠萝：批发价通常为6-8元/斤（需整箱采购，规格约10-15斤/箱）。  
   - 洪九徐闻菠萝：批发价约2-4元/斤（带箱约25斤/箱，部分区域可议价）。  
   - 建议计算**出肉率**：都乐出肉率约55%-60%，洪九约45%-50%（皮厚稍多），按可食部分算，都乐实际成本约12-13元/斤，洪九约4-8元/斤。

3. **损耗与保鲜**  
   - 进口菠萝需冷链运输，到店损耗约5%-10%；国产菠萝在本地供应链完善的情况下，损耗可控制在3%-5%。

4. **消费群体匹配**  
   - 若门店定位高端精品：都乐更易接受，建议定价在19.9-25元/个（约2斤左右）。  
   - 若走社区低客单价销量：徐闻菠萝可定价在9.9元/个（约2-3斤/个），复购率高。

**可执行建议**：  
- 引入洪九果品的海南金钻凤梨（皮薄无黑眼，出肉率高达65%），单价约5-7元/斤，平衡品质与成本。  
- 首批采购建议少量试销（每店10-15箱），对比实际损耗与销量后，再决定是否扩大品类。
