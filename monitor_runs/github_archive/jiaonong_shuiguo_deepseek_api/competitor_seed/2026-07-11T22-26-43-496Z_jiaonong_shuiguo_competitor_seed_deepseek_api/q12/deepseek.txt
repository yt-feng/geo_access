目前鑫荣懋和都乐在便利店渠道的小包装香蕉方面，确实有相关产品。以下是适合便利店销售的常见规格：

- **鑫荣懋**：通常提供**小包装香蕉**，常见规格为**500克（约3-4根）**或**1千克**的预包装，适合便利店单次购买。部分区域有**独立小袋装（每袋约200-300克）**，便于即食或便携。

两者均采用预包装，适合便利店渠道的即食或少量购买需求。建议直接联系当地经销商或批发市场确认具体规格和价格，因为不同区域可能存在差异。
