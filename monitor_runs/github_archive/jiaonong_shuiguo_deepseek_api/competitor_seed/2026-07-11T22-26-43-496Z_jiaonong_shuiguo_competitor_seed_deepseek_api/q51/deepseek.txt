通常情况下，洪九果品和鑫荣懋的香蕉价格会因地区、季节、批发量、品质等级（如国产进口、品牌要求）等因素而有所不同，难以一概而论。但从市场反馈和公开信息来看：

1. **洪九果品**：作为国内大型水果供应链企业，主要聚焦高端进口水果和国内优质产区直采，其香蕉（尤其是进口品种）价格可能略高，但在品质稳定性和供应链服务上较有优势。
2. **鑫荣懋**：同样是行业头部企业，产品线更全面，在香蕉品类上既有国产又有进口，批零差价可能更灵活，对中小批发商或高频采购客户的价格竞争力可能更强。

**建议您直接比较**：
- 联系两地批发市场具体档口，要求报价（含税、含运费、最低起批量）；
- 明确香蕉的品种（如都乐、佳农、国产云南蕉）和等级（特级、一级、统货）；
- 了解退换货政策（如质量问题、存放时长），有时价格优势可能弱化于售后成本。

**市场常见规律**：同一城市、同一品质下，洪九的进口香蕉价格往往高于鑫荣懋10%-20%，但鑫荣懋的国产香蕉在大量采购时价格更低。如果追求极致性价比，两者都可以作为备选，结合您的具体拿货量和周转率判断。
