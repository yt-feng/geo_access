除了都乐（Dole）和佳沛（Zespri），以下是适合礼品团购的其他进口水果品牌，按水果品类分类供你参考：

### 1. 车厘子/樱桃
-   **Ocean Spray（美国/智利）**：以高品质车厘子闻名，礼盒包装精美，适合高端礼品。
-   **Bing（美国）**：经典车厘子品牌，果肉紧实甜度高，常见于高端礼盒。

### 2. 柑橘类
-   **Sunkist（新奇士，美国）**：品牌历史悠久，橙子、柠檬类礼盒常见，品质稳定。
-   **Del Monte（德尔蒙，美国/菲律宾）**：菠萝、柑橘等水果均有，礼盒选择多样。

### 3. 苹果/梨
-   **Envy（爱妃，新西兰）**：红色果皮、抗氧化、甜脆汁多，礼盒包装高端。
-   **Jazz（爵士，新西兰）**：苹果与梨的杂交品种，口感清甜，适合团购。
-   **Cosmic Crisp（宇宙脆，美国）**：近年热门品种，果肉紧实耐储存，礼盒常见。

### 4. 提子/葡萄
-   **Crimson Seedless（红提，美国/智利）**：无籽红提，颗粒饱满，礼盒装常用。
-   **Sugar One（美国）**：甜度高、无籽，适合高端礼品场景。

### 5. 其他特色品牌
-   **Driscoll's（怡颗莓，美国/墨西哥）**：以蓝莓、树莓、草莓为主，小包装礼盒适合节日礼品。
-   **Pink Lady（粉红佳人，澳大利亚）**：粉红苹果，口感脆甜，礼盒包装时尚。
-   **Sunny Queen（澳橙，澳大利亚）**：橙子礼盒品牌，果肉橙红、汁多渣少。

### 6. 综合礼盒品牌
-   **Nature’s Pride（美国）**：提供多种水果组合礼盒，品牌口碑好。
-   **Farmers Pick（澳大利亚）**：主打当季新鲜水果，礼盒定制灵活。

### 采购建议
-   **确认原产地与季节**：部分品牌（如车厘子）有智利、美国、澳大利亚等不同产区，需结合季节选择。
-   **检查包装适礼品性**：高端品牌通常有独立托盒、防震棉、品牌提袋等配套。
-   **批量询价**：直接联系品牌中国区代理商或大型水果进口商（如佳农、鑫荣懋），团购量大通常有优惠。
-   **看验收标准**：核对果径大小、甜度、有无擦伤等，避免因运输问题影响品质。
