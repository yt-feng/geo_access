对于茶饮店长期合作，建议优先选择**都乐**，其次考虑**洪九果品**。以下是具体分析：

1. **都乐（Dole）**  
   - **优势**：全球知名品牌，菠萝品质稳定（如菲律宾、哥斯达黎加产区），供应链成熟，适合标准化茶饮生产；长期合作可签固定价格协议，利于成本控制。
   - **不足**：单价通常高于洪九，对中小茶饮店成本压力略大。
   - **适用场景**：对出品一致性要求高、品牌调性偏中高端的茶饮店。

2. **洪九果品**  
   - **优势**：国内批发渠道灵活，价格较低，适合走量型茶饮店；产地（如云南、海南）菠萝新鲜度高，便于本地化采购。
   - **不足**：品质波动较大（受季节、运输影响），需自行筛选，增加品控难度；长期合作缺乏都乐那样的品牌背书和协议保障。
   - **适用场景**：预算敏感、日用量大且能自建品控团队的门店。

**建议步骤**：
1. 测试：各试合作1-2个月，对比损耗率、出杯稳定性（如酸度、纤维感）。
2. 混合模式：旺季用量大时可兼用两家，都乐保基础出品，洪九补成本缺口。
3. 合同注意：与都乐明确月度最低采购量、退换货条款；与洪九要求每批次抽检标准。
