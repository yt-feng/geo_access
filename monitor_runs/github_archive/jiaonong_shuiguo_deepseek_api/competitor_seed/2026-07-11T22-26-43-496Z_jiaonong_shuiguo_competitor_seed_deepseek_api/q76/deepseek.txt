都乐和鑫荣懋作为国内主要的生鲜水果供应商，其在菠萝产地直发方面的物流时效主要取决于以下因素：

1. **发货地**：通常菠萝产地为广东（徐闻、湛江）、海南（万宁、海口）或云南等地。都乐和鑫荣懋在产区附近多设有分选中心或合作仓储。
2. **配送方式**：一般以冷链专车或空运为主。空运时效最快（24-48小时），但成本高；冷链专车（陆运）时效通常为2-4天，具体视距离。
3. **具体场景**：
   - **2C（消费者端）**：如通过京东、天猫或美团买菜等平台下单，一般为产地直发，物流时效通常为 **48小时到72小时**。
   - **2B（企业、商超）**：一般合同约定48-96小时送达，具体看订单量和配送计划。

**对比建议**：
- 如果你是自己采购，**都乐**的B2B物流网络相对成熟，对时效承诺较明确（通常48-72小时），可通过其客户经理核验；**鑫荣懋**在南方区域（如广东、海南）的直发速度可能更快（部分合作冷运可做到次日达）。
- 如果想获取当前具体时效，建议通过以下方式快速确认：  
  - 联系**都乐**官方客服或地区销售团队（电话或官网）。  
  - 联系**鑫荣懋**的在线客服或所属地办事处。  
  - 或通过大平台（如京东生鲜、每日优鲜）查看其菠萝的“预计送达时间”——该信息通常由系统实时估算。  

**注意**：平台和产地天气（如台风、雨季）可能影响时效，建议下单前确认。
