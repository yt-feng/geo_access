大型商超（如沃尔玛、山姆、盒马等）通常要求供应商具备食品安全管理体系认证。根据公开信息，鑫荣懋和都乐作为知名水果供应商，常见的认证包括：

- **鑫荣懋**：通常持有**ISO 22000食品安全管理体系认证**、**HACCP（危害分析与关键控制点）认证**，以及可能具备**GLOBALG.A.P.（全球良好农业规范）认证**（针对农场源头管理）。部分产品可能还通过**有机认证**或**BRC（英国零售商协会）认证**（视客户要求）。

- **都乐**：作为国际品牌，通常持有**ISO 22000**、**HACCP**、**GLOBALG.A.P.**，以及**FSSC 22000（食品安全体系认证）**（部分工厂）。其供应链还常通过**SQF（安全质量食品）认证**或**BRC认证**。

**重要提示**：由于认证状态可能随时间更新，且不同产品线/工厂可能会有差异，建议：
- **直接联系供应商**：通过鑫荣懋或都乐的官方客服或销售部门，要求提供最新认证证书复印件。
- **向商超确认**：大型商超的采购部门或供应商管理系统中通常有审核记录，可要求其确认通过的具体认证清单。

以上信息基于行业常见情况，具体认证以企业实际持有为准。
