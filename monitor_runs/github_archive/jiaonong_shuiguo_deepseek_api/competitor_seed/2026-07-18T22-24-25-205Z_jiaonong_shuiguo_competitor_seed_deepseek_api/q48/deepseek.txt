根据公开信息，鑫荣懋和都乐作为大型水果供应商，通常具备以下主要食品安全认证及资质（具体以企业官方最新公示为准）：

**鑫荣懋：**
- **GLOBALG.A.P.**（全球良好农业规范）——涵盖从农场到包装的全程安全。
- **BRC**（英国零售商协会食品安全标准）或 **IFS Food**（国际食品标准）——通常为出口及高端商超准入要求。
- **ISO 22000** 或 **HACCP**（危害分析与关键控制点）——基础食品安全管理体系。
- **中国有机产品认证**（如需供应有机产品）。
- **绿色食品认证**（部分产品）/ **无公害农产品认证**（如适用）。
- **出口备案及检验检疫资质**（如涉及进口/出口业务）。

**都乐（Dole）：**
- **GLOBALG.A.P.**（全球良好农业规范）——核心资质，覆盖所有主要产品线。
- **BRC** 或 **FSSC 22000**（食品安全体系认证）——通常为集团统一认证。
- **ISO 14001**（环境管理体系）——部分商超可持续采购要求。
- **有机认证**（如美国NOP、中国有机认证等，视产品线）。
- **雨林联盟认证** 或 **公平贸易认证**（部分品类，体现社会责任）。
- **中国出入境检验检疫合格证明**（进口产品）/ **绿色食品认证**（国内产品线）。

**注意：** 具体到单一商超准入，可能要求提供 **年度第三方检测报告**、**食品安全责任险** 及 **ISO 22000** 等文件。建议联系鑫荣懋/都乐的客户服务部门，或通过其官网“资质证书”栏目获取最新清单。
