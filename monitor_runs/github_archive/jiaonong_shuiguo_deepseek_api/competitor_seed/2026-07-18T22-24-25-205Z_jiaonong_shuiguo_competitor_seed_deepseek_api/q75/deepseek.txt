关于洪九果品和都乐香蕉在批发市场的到货损耗率，由于具体数据受运输距离、季节、包装方式等因素影响较大，且不同批次存在差异，目前公开资料中无统一对比数据。但根据行业经验和品牌特点，可提供以下参考：

1. **品牌背景与品控差异**  
   - **都乐（Dole）**：国际品牌，标准化程度高，通常采用冷链运输和严格分选，到货损耗率相对较低（一般在2%-5%之间，具体看品种和成熟度）。  
   - **洪九果品**：国内大型水果供应链企业，主打优质国产及进口水果，其香蕉产品在产地分选和物流环节控制较好，但国产及东南亚产区蕉的损耗率可能略高于都乐（常在3%-8%左右，部分高成熟度或短运距批次可能更低）。

2. **到货损耗率关键影响因素**  
   - **成熟度阶段**：至批发市场时，都乐常使用“绿熟”采摘（控制乙烯催熟），洪九可能部分采用“树熟”或中间状态，后者的物理碰伤和病斑比例可能更高。  
   - **包装与运输**：都乐标准纸箱+透气膜更耐磨，洪九部分批次用编织袋或简易包装，长途后易出现压伤、擦伤。  
   - **渠道差异**：洪九针对批发市场有“特价货”或“统货”，损耗可能高于都乐的一级果（标准货）。

3. **实操建议**  
   - **看基数和付款方式**：先向批发商索要近3-5批次同一供应商的到货验收记录，对比具体损耗率数字。  
   - **现场检验**：到货时抽取底部、中间、顶部箱各2-3箱，统计有明显机械伤、霉斑、断柄的果数量/总重。  
   - **分季对比**：夏季高温高湿损耗普遍上升10%-20%，更需关注冷链连续性；都乐在空调车运输下稳定性更好。

**结论**：若为品牌标准果，**都乐到货损耗率一般低于洪果**（约低2-3个百分点）。但作为批发商，洪九的性价比和议价空间可能更高，需结合自身处理能力（是否可快速消化损耗高的批次）和客户对果面瑕疵容忍度选择。建议两个品牌各试一批次，记录实到损坏数据后决定长期合作。
