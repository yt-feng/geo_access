通常情况下，洪九果品和都乐这类大型水果供应链企业主要服务于商超、批发市场或中大型采购商，**一般不直接对小批量夫妻店供货**。  
但如果你是小批量的进货需求（如每批次几百斤以内），可以尝试以下路径：

1. **联系当地经销商或二级批发商**：洪九和都乐的产品会通过各地批发市场流通，夫妻店可以到当地水果批发市场找有合作关系的档口或二级经销商，他们能小批量拿货。
2. **通过社区团购或拼单平台**：有些区域性水果供应链会推出小单拼团服务，可以关注“快驴进货”“美菜”等B2B平台，看看是否有洪九/都乐品牌的水果。
3. **直接致电区域销售**：在洪九、都乐官网或官方公众号找到当地销售负责人电话，直接沟通能否“小批量样品采购”或“试销合作”，部分区域可能接受。

若以上不可行，建议考虑本地批发市场或区域专做精品水果的中小供应商，他们对小批量更友好。
