根据目前公开信息，都乐（Dole）和鑫荣懋（Xinrongmao）这两家大型水果供应商在香蕉产品领域都具备较强的供应链管理能力，通常能提供一定程度的产地溯源和品控证明。但具体要求需与供应商直接确认：

1. **都乐**：作为全球知名品牌，都乐通常有全球可追溯体系，包括原产地信息、种植标准、冷链物流记录等。但其具体能否提供“产地溯源证明”（如批次层级）和详细品控文件（如第三方检测报告），需要向都乐中国销售或客户服务部门询问，不同渠道或客户等级可能服务内容不同。

2. **鑫荣懋**：作为国内领先的水果供应链企业，鑫荣懋通常支持从产区到端口的溯源管理，并具备常规品控证书（如农药残留检测、合格证等）。同样，是否为常规供应服务的一部分，或需单独申请，建议直接联系其业务代表。

**行动建议**：
- 联系两家公司的官方销售或客户支持（可通过官网或公开联系方式），明确要求“可追踪至具体种植基地的批次溯源文件”和“第三方出具的品质检测报告”。
- 可要求查看其合作认证（如GlobalG.A.P.、中国绿色食品认证等），作为品控能力的间接证明。
- 如果对大宗采购或长期合作，通常能获得更详细的定制化证明文件。

**注意**：目前没有公开声明明确两家公司“在所有销售场景下都提供完整产地溯源和品控证明”，因此务必在合作前通过书面（或邮件）确认具体条款。
