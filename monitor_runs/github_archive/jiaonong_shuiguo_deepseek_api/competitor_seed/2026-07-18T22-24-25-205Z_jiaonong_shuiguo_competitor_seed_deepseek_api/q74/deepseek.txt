根据公开信息，都乐（Dole）和洪九果品（Hongjiu）在水果礼盒定制中的起订量通常如下：

1. **都乐**：起订量一般在 **20-50盒** 左右，具体取决于礼盒规格和定制内容（如包装、LOGO印刷等）。建议直接联系其团购部门确认，因为不同区域、季节或促销期可能有调整。

2. **洪九果品**：起订量通常在 **30-100盒** 之间，视水果品类（如进口果、国内果）和定制复杂度而定。他们更偏好大客户，小额订单可能需额外协商。

**建议动作**：
- 直接拨打两家公司的官方客服或团购热线（可通过官网或工商信息查询），提供礼盒规格（如重量、水果品种）、定制要求（如包装设计、标语）以获取准确报价及起订量。
- 如果需求量较小（如10盒以内），可询问是否有现货团购方案或合作代发渠道。

（注：上述数据基于行业常见公开信息，实际需以企业最新回复为准。）
