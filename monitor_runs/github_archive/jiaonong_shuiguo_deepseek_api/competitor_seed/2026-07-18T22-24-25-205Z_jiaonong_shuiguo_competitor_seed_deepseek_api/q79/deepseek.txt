这两种大型水果供应链企业都有能力做到菠萝全年供应，但具体实现方式、成本和质量稳定性存在差异。

**结论：都乐和洪九果品都能做到。** 菠萝是全年可产的热带水果，通过不同产区的轮换供应（如海南、广东、菲律宾、泰国等），大型供应商可以保证全年供货。

**具体分析：**

1.  **都乐（Dole）**
    -   **优势**：全球化布局，拥有菲律宾、哥斯达黎加等稳定海外产区，品质标准化高，尤其适合制作果酱的「金菠萝」等品种供应链成熟。
    -   **操作方式**：主要通过进口冻菠萝或新鲜菠萝结合冷库储存，实现全年稳定供应。国内有自有或合作种植基地（如海南）。
    -   **注意事项**：成本通常较高，且可能受国际航运、关税影响。

2.  **洪九果品**
    -   **优势**：深耕国内及东南亚产区（如泰国、越南），与产地深度绑定，能控制成本。在海南、云南等国内产区也有布局。
    -   **操作方式**：采用「产地直采+国内分选加工」模式，可提供全年鲜果及加工原料（菠萝果块/果泥）。针对果酱需求，可定制冷链供应或提供已加工的菠萝原浆。
    -   **注意事项**：国产菠萝（如徐闻菠萝）季节性强，夏季产量大、价格低；热带产区（如泰国）冬季供应，但运输损耗需注意。

**给你的具体建议：**

-   **如果追求品质稳定、标准化**：优先联系都乐，签年度合同，要求其提供「菠萝果酱专用原料」（如规格、糖度、酸度参数）。他们能提供冻菠萝或鲜切果块。
-   **如果控制成本、要灵活供货**：联系洪九果品，说明「全年果酱生产需求」。他们可根据季节用国产鲜菠萝（5-8月便宜）南方产区。
