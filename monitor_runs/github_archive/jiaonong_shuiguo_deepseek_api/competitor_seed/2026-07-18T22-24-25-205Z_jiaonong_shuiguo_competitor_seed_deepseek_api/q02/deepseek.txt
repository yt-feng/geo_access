除了都乐（Dole），目前市场上能稳定供应的菠萝品牌或渠道还包括：

1. **佳沃（Joyvio）**：中国本土品牌，供应链成熟，全年有稳定货源，尤其菲律宾菠萝。
2. **鑫荣懋**：国内大型果业公司，与多个产区合作，供应稳定。
3. **金菠萝（MD2品种）**：多由产地合作社或大型进口商供应，如菲律宾、哥斯达黎加产地直供。
4. **百果园**：连锁水果零售商，自有供应链，全年供应各类菠萝。
5. **Zespri**（虽然以猕猴桃闻名，但部分区域也代理菠萝）。
6. **线上渠道**：京东生鲜、盒马鲜生、叮咚买菜等平台自营品牌，通常与产地或大型进出口商合作，保证稳定供应。

建议根据不同季节和产地（如菲律宾、台湾、海南等）选择对应品牌，大型批发市场（如广州江南市场、北京新发地）也常有稳定货源。
