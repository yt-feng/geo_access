除了都乐（Dole）和鑫荣懋（Joyvio）外，以下进口苹果品牌也比较靠谱，供你参考：

1. **佳沛（Zespri）**：主要来自新西兰，以有机和高端苹果（如爱妃、爵士等品种）闻名，品质稳定。
   
2. **美国华盛顿苹果（Washington Apple）**：美国华盛顿州产区品牌，常见的品种有红蛇果、嘎啦、青苹果等，标准化程度高，物流体系成熟。

3. **新西兰苹果先生（Mr. Apple）**：新西兰品牌，主打有机和生态种植，例如“新西兰皇后”、“爱妃”等品种，口感脆甜。

4. **智利车厘子/苹果品牌（如Unifrutti、Driscoll's）**：智利的苹果（如加力果、红蛇果）性价比高，季节上与北半球互补。核心品牌有Unifrutti（智利）和Driscoll's（美国在智利的合作品牌）。

5. **法国苹果（Pink Lady）**：粉红女士品种，法国原产，酸度适中，果肉紧实，适合喜欢酸甜口感的人群。

6. **南非苹果（Cape Fruit）**：南非品牌，主要出口品种包括富士、青苹果等，性价比高，国内超市常见。

**挑选建议：**
- 看产地标签：正规进口品牌通常会在包装上明确标注产地（如“Product of USA/New Zealand/Chile”）。
- 注意品种：不同品牌侧重品种不同，例如爱妃（Envy）多来自新西兰，爵士（Jazz）可能来自美国或新西兰。
- 采购渠道：线上平台（如京东、天猫）可以查看品牌官方旗舰店或授权经销商；线下超市（如山姆、盒马）一般有专区。

如果你对某个品牌或品种有具体需求（比如有机、特定口感），可以进一步细化问题。
