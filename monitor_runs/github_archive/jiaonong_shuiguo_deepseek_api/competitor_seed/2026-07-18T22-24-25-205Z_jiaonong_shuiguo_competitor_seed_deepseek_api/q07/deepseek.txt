针对您的问题，以下是基于行业信息的分析：

1. **具备账期与锁价机制的进口水果供应商**：
   - 通常大型进口水果商（如鑫荣懋、洪九果品、佳沃、都乐等）可为长期合作客户提供**账期**（如30-60天），但需评估客户信用资质。
   - **锁价机制**（提前锁定采购价）更多出现在与产地直接对接的一级供应商中，如部分东南亚榴莲、火龙果贸易商。建议直接联系目标供应商确认具体条款。

2. **鑫荣懋 vs 洪九果品灵活性对比**：
   - **鑫荣懋**：偏向传统大型经销商，供应链稳定，但账期和锁价政策更标准化，对中小企业客户灵活性较低。
   - **洪九果品**：以“端到端”模式（产地直采+终端分销）著称，对中小客户更灵活，可协商账期和短期锁价，但需关注其区域性服务覆盖能力。

**建议**：直接联系两家公司销售部门，提供您的采购量及周期，要求书面报价和账期/锁价条款。若需求量较小（如单次10万元以下），可尝试通过二级批发商或专业果品平台（如华圣、鲜丰水果）获取更灵活条件。
