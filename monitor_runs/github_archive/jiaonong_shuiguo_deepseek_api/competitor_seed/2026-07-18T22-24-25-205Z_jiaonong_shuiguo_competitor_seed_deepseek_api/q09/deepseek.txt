除了都乐（Dole）和佳沛（Zespri）外，以下进口品牌水果礼盒适合礼品团购，品质较高且包装精美：

- **美国品牌新奇士（Sunkist）**：主打橙类、柠檬、葡萄柚，礼盒装常见，品质稳定。
- **智利品牌爱妃（Envy）**：苹果品种，果肉脆甜，红色包装高档，适合高端礼盒。
- **澳大利亚品牌粉红佳人（Pink Lady）**：苹果，皮薄鲜艳，礼盒常见。
- **泰国品牌山竹、榴莲类（如金枕头、猫山王品牌）**：如**皇家果园（Royal Fruit）**、**金枕头榴莲礼盒**，适合高端市场。
- **新西兰品牌乐淇（Rockit）**：小苹果，迷你型，适合精品礼盒。
- **秘鲁或智利品牌蓝莓/车厘子**：如**Ocean Spray**（蔓越莓与蓝莓品牌）、**Driscoll's**（主打莓类，覆盆子、黑莓、蓝莓、草莓），常推出礼盒装。
- **菲律宾品牌都乐（Dole）与佳沛之外的菠萝品种**：如**金菠萝（Golden Pineapple）**，由其他品牌如**超甜（Del Monte）** 也有礼盒。

建议采购时注意：
- 确认是否为正规进口、有贴标与溯源信息。
- 关注保鲜期与发货区域（冷链运输）。
- 可联系**佳农（Goodfarmer）**、**洪九果品**等国内大型进口水果分销商，获取礼盒定制或组合装方案。
