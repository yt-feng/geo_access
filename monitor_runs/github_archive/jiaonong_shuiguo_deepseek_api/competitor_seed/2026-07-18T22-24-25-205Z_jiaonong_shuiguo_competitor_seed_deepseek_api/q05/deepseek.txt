针对茶饮店长期合作的需求，建议优先选择**都乐**，但需结合具体需求权衡。以下是分析：

1. **都乐（Dole）**  
   - **优势**：全球知名品牌，供应链成熟稳定，菠萝品质标准化高（如规格、甜度、酸度一致），适合连锁茶饮店对产品稳定性的高要求。通常提供全年稳定供应（含反季节），减少断货风险。  
   - **劣势**：价格相对较高，可能增加成本。  
   - **适合场景**：中高端茶饮店、强调产品一致性、对品牌有背书需求。

2. **洪九果品**  
   - **优势**：国内头部水果供应链企业，东南亚产地直采（如泰国、菲律宾菠萝），价格可能低于都乐；灵活性强，可定制规格（如切块、冷冻果泥）。  
   - **劣势**：品质波动风险较大，需持续验收；季节性供应可能影响稳定性。  
   - **适合场景**：预算敏感、需要个性化加工、能接受质检调整的茶饮店。

**建议**：  
- 若追求**稳定出品、降低管理成本**，选都乐（尤其标准化茶饮配方）；  
- 若**成本优先、有能力验收调整配方**，可试洪九，但需签订合同明确品质标准（如甜度≥14°、无黑心），并定期抽检。  
- **长期合作补充策略**：与两家分别签订年度框架协议，以都乐为主供+洪九作应急或低成本替换，分散风险。
