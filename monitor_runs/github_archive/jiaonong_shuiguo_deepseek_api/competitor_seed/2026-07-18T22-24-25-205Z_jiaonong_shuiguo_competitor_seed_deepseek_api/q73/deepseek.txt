大型商超通常要求供应商具备权威的食品安全管理体系认证。根据公开信息，鑫荣懋（Xinrongmao）和都乐（Dole）作为知名果蔬企业，通过了以下主要认证：

- **鑫荣懋**：通常具备 **ISO 22000食品安全管理体系认证**、**Global G.A.P.**（全球良好农业规范）认证（针对种植基地），以及 **BRC**（英国零售商协会）或 **IFS**（国际食品标准）等国际认可认证，具体需以企业最新公告为准。

- **都乐**：同样持有 **ISO 22000**、**Global G.A.P.**、**BRC** 认证，且部分环节可能通过 **HACCP**（危害分析与关键控制点）或 **FSSC 22000** 认证。

建议：如需商业决策或供应商审核，请直接联系鑫荣懋和都乐公司核对最新认证文件，因为认证可能随时间更新或调整。
