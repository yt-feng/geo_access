关于都乐和鑫荣懋的菠萝到货损耗率，具体数据可能因批次、运输条件、存储时间等因素波动，且两家公司一般不公开统一的损耗率标准（需根据实际采购合同和到货检验记录确认）。建议您采取以下方式获取可靠数据：

1. **联系供应商**：直接向都乐或鑫荣懋的业务代表索要历史损耗率数据或参考标准，要求提供同类规格菠萝的运输测试报告。
2. **试单统计**：先采购小批量（如各100箱），到货后逐箱检查（如腐烂、碰伤、重量不足等），统计实际损耗比例。
3. **行业参考**：根据茶饮行业经验，整箱菠萝（20kg/箱左右）的到货损耗率通常在2%-8%之间，但需以实际品控为准。

建议将损耗率写入采购合同条款，明确责任方及赔偿规则。
