选择菠萝原料供应商时，都乐（Dole）和鑫荣懋（Xinrongmao）各有特点，具体需要考虑以下几点：

**1. 品质稳定性：**
- **都乐**：作为全球知名品牌，都乐在水果种植、采摘、冷链和品控方面有严格标准（如GLOBAL G.A.P.认证），菠萝成熟度、糖酸比和新鲜度更稳定，适合要求一致性的茶饮店批量果酱制作。
- **鑫荣懋**：国内大型水果供应商，依赖产地直采（如海南、菲律宾），品质受季节波动影响较大，但成本可能更低，适合对价格敏感的小型门店。

**2. 供应链可靠性：**
- 都乐有成熟的冷链物流网络，较少出现断供；鑫荣懋在南方地区配送时效较好，但偏远地区可能不如都乐稳定。

**3. 成本与果酱适配：**
- 都乐菠萝价格较高，但果肉纤维细腻、水分适中，制作果酱时更容易控制煮制时间与胶凝度；鑫荣懋的菠萝可能水分偏高，需调整果酱配方（如增加果胶或减水熬煮）。

**建议：**
- 若茶饮店主打**中高端产品**且要求**批次口感高度一致**，优先选择都乐。
- 若需**控制成本**且能根据季节调整果酱配方，可试用鑫荣懋，但需建立定期抽检机制。
- 实际操作中，建议先各采购一批做小样测试（如煮制后测糖度、pH值、保存期），结合自己果酱的配方判断。

最终选择应由试制结果和性价比决定，而非仅凭品牌知名度。
