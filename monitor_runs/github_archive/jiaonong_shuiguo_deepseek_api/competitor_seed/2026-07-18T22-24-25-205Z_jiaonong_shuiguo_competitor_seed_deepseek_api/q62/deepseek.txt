都乐（Dole）和鑫荣懋（XinRongMao）作为大型水果供应商，通常能够提供所供应产品的农残检测报告，但具体能否提供、报告覆盖范围及获取方式需根据以下几点确认：

1. **都乐（Dole）**：作为全球知名的果蔬品牌，其在国内的菠萝产品通常具备完善的合规文件，包括农残检测报告。建议直接联系都乐在中国的销售团队或客服，说明需要菠萝原料的批量采购报告，要求提供第三方检测机构的报告（如SGS、Intertek等）。部分情况下，报告可能需额外申请或收费。

2. **鑫荣懋**：作为国内头部水果供应链企业，其产品同样需符合市场准入要求。可通过其官方渠道（官网、销售代表）申请菠萝原料的农残检测报告。通常大宗采购客户可免费获取，散客可能需要单独协商。

**操作建议**：
- **直接联系**：通过都乐中国官网或鑫荣懋客服电话，明确询问“菠萝原料是否自带农残检测报告，能否提供PDF版本”。
- **细化需求**：说明需检测的项目（如GB 2763标准中的常规农残），确认报告是否涵盖你的原料批次。
- **备选方案**：若供应商无法提供，可委托第三方检测机构（如华测、谱尼测试）对采购批次自行检测，成本约200-500元/项。

注意：报告需注意有效期（通常为1年内）、检测机构资质（CMA/CNAS认证），且需与供货批次对应。如对方以“商业机密”为由拒绝，可能需签订保密协议或要求提供脱敏版本。
