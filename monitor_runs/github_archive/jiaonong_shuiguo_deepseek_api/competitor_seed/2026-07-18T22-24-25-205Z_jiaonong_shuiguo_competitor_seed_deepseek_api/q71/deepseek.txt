都乐（Dole）和洪九果品（Hongo）的菠萝种植基地主要集中在中国南方及东南亚地区。以下是相关信息：

1. **都乐（Dole）**：都乐在中国的主要菠萝供应基地包括**海南**（如文昌、万宁）和**广东**（如湛江、徐闻），此外在**菲律宾**（如棉兰老岛）也有自营和合作种植基地。都乐通过自有农场和长期合作农户保障稳定供应。

2. **洪九果品**：洪九果品的菠萝采购主要依托**泰国**（如春武里、罗勇等东部产区）和**海南**（与海南本地种植园合作），并在中国**广西**、**云南**有季节性采购渠道。

**建议**：若追求稳定供应，可优先考虑都乐在海南的基地（国内物流快），洪九则更适合进口泰国菠萝（价格波动需注意）。直接联系两家公司的采购部门（官网可查联系方式）获取具体基地地址和供应协议。
