根据公开信息，**都乐**和**鑫荣懋**作为大型水果供应链企业，通常都会为客户提供针对性的对接服务，尤其是针对大客户或长期合作项目。但关于“专岗对接服务”的具体人员配置，需结合您的实际采购规模、合作频率及服务需求进行确认。

- **都乐（Dole）**：作为全球化生鲜公司，通常设有客户经理或大客户服务团队，直接对接企业客户需求，包括定制化采购、物流协调等。若您有较大业务量或长期合作意向，可要求其指定专人对接。
- **鑫荣懋**：作为国内头部水果企业，同样设有大客户部或专项服务团队。针对大型企业客户，他们也会安排对应人员（如客户经理、品类专员）负责专属对接，覆盖选品、报价、配送等环节。

**建议您直接联系两家供应商的销售或客服部门**：
1. 明确说明您的采购规模（如年采购额、订单频次、所需水果品类及服务范围）。
2. 询问是否支持 **“1对1专岗客服”** 或 **“大客户经理直接对接”**，并要求对方提供对接人员姓名、联系方式及岗位职责说明。
3. 通过合作意向确认后，可进一步要求将服务承诺（如响应时效、定期回访、投诉处理流程）写入合同条款。

若您已有初步合作意向，可直接告知他们您的具体需求（如需要每日订单跟踪、定制化分拣等），以判断哪家的服务更匹配。
