关于产地溯源，都乐（Dole）和洪九果品在菠萝供应上均有相关布局：

1. **都乐（Dole）**：都乐是全球知名的果蔬品牌，在中国市场主要供应菲律宾菠萝（如“都乐超甜蕉”同体系），其基地直供模式明确——在菲律宾拥有自有农场，并在中国设有分选包装中心，可提供从种植到销售的全程溯源信息。因此，都乐可以满足产地溯源需求。

2. **洪九果品**：洪九果品是国内领先的多品牌鲜果集团，其菠萝供应主要来自泰国和海南基地。洪九在海南有合作或自建的菠萝种植基地，并采用“端到端”直采模式，能够提供产地溯源服务。例如，其“洪九泰好吃”品牌下的菠萝产品通常支持溯源。

**建议**：若需稳定、高标准的溯源体系，优先联系都乐官方渠道（如官网或大客户部门）；若需要国内产地（如海南）或灵活的小批量供应，可对接洪九果品本地业务团队。在采购前，建议直接要求对方提供批次对应的质检报告和产地证明文件。
