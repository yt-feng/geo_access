关于都乐（Dole）和鑫荣懋的菠萝到货损耗率，这属于企业级供应链数据，通常属于商业机密，且损耗率会因运输距离、季节、包装方式以及接收时的验收标准而有所差异，**没有公开的统一数据**。

为了便于你进行采购决策，可以参考以下行业经验数据和实际操作建议：

**1. 行业一般参考范围**
-   **都乐**：作为国际品牌，其供应链品控较为严格，包装标准化程度高。常规冷链运输下，到货后的外观损耗率（如腐烂、严重机械伤）通常在 **1%-3%** 左右（需剔除因催熟问题导致的不合格）。
-   **鑫荣懋**：国内头部水果分销商，供应链经验丰富。常规冷链运输下，外观损耗率一般在 **2%-5%** 左右（需注意不同产地、不同季节的菠萝质量波动可能较大）。

**2. 影响损耗率的关键因素**
-   **包装方式**：纸箱装（易压伤） vs 塑料周转筐/托盘（抗压性更好）。强烈建议采购时要求采用**带缓冲层**的包装。
-   **成熟度**：采购全熟黄菠萝 vs 绿熟菠萝。全熟果损耗率高，一般专业茶饮采购建议选择**七八成熟（约1/3转黄）**的菠萝，到店存放2-3天后使用，损耗更可控。
-   **运输条件**：冷链车（推荐2-8℃） vs 普通货车。非冷链运输在夏季损耗可能直接翻倍。

**3. 给你的可执行建议**

-   **第一步：要求供应商提供试发样品**：在签订批量合同前，要求两家供应商各发一箱（约12-20kg）**指定规格**（如称重、颜色、糖度）的菠萝到你的仓库或门店，你方现场记录**开箱损耗率**。
-   **第二步：在合同中明确约定损耗标准**：这是最核心的条款。例如：
    > “双方约定：菠萝到货后，**外观可见腐烂、软烂、严重机械伤（深度超过1cm）** 的损耗率不得超过【3%】。若超出此比例，超出部分由供应商按采购单价全额赔偿，并于【24小时】内完成补发或退款。”

-   **第三步：建立验收标准**：
    -   **当场查验**：到货后1小时内开箱检查。
    -   **拍照留证**：对有问题的菠萝和包装箱拍照，并立即发给供应商确认。
    -   **按箱理赔**：要求供应商对整箱出问题的批次（如整箱腐烂）进行单独处理。

**总结**：
没有官方数据能告诉你具体数字。**最可靠的做法是：以试发和合同条款为准，将损耗风险通过合同转移给供应商，而不是承担经验估计的比例。** 建议你先与都乐或鑫荣懋的销售经理沟通，要求他们提供该笔订单可落地的损耗承诺（通常是按百分比扣款或补货），而非口头说“行业标准”。
