根据公开信息，**都乐（Dole）** 和 **洪九果品** 均具备产地直供和溯源能力，但具体模式有所不同：

1. **都乐（Dole）**  
   - 都乐是全球知名水果品牌，其菠萝主要来自 **菲律宾、哥斯达黎加** 等自有或合作农场。  
   - 都乐提供 **全程可追溯系统**，可通过包装上的溯源码追溯产地、采摘时间等信息。  
   - 在中国市场，都乐与大型商超（如盒马、山姆）合作，支持基地直发。

2. **洪九果品**  
   - 洪九果品主要经营 **东南亚水果**，菠萝核心产区为 **泰国**（如“普吉”品种）。  
   - 洪九采用 **“端到端”直采模式**，部分品种（如金菠萝）实现产地直供，并配备溯源码。  
   - 需注意：洪九的菠萝业务以 **B端批发** 为主，小批量零售需通过其合作渠道（如社区团购、电商平台）。

**建议操作**：  
- 若为商业采购，可直接联系两家公司的 **销售部** 索取产地溯源方案；  
- 若为小批量需求，可通过 **叮咚买菜、美团优选** 等平台选择标注“都乐/洪九+溯源码”的产品，扫码验证；  
- 在采购合同中明确要求提供 **产地证明、报关单（进口菠萝）** 及第三方检测报告，确保溯源真实性。
