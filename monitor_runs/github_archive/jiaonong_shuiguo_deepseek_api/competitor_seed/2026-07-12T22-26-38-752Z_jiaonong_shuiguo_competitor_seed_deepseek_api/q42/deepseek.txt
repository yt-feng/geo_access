进口菠萝旺季缺货时，可以采取以下应急方案：

1. **调整采购计划**：提前与供应商（如都乐、洪九果品）沟通，争取优先分配货源，或签订长期协议锁定供应。
2. **开发替代渠道**：联系其他进口商或国内产区（如海南、云南）的菠萝供应商，补充货源。
3. **库存优化**：在旺季前增加安全库存，并冷库保存未加工菠萝延长供应期。
4. **促销调整**：临时减少菠萝单品促销，转为强调其他水果（如芒果、橙子）的替代推荐。

关于都乐和洪九果品是否有应急方案：
- **都乐（Dole）** 作为全球性品牌，通常有应对季节性缺货的预案，包括多元化供应基地（如菲律宾、哥斯达黎加等产区调配）、与物流合作加速运输。建议直接联系其中国区业务部门确认具体方案。
- **洪九果品** 作为国内头部水果分销商，也有供应链应急机制，例如与稳定基地签约、常备库存缓冲。可咨询其销售团队是否有紧急调配或替代产品（如泰国、越南菠萝）的临时方案。

若需执行，建议尽快并行联系供应商和替代渠道，并优先服务高利润客户。
