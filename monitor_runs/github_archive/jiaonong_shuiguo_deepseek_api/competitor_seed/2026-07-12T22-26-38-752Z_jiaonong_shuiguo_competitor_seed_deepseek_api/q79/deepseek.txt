都乐（Dole）和洪九果品（Hongjiu）作为国内大型水果供应商，理论上**能够做到菠萝的全年供应**，但实际操作中需要具体分析：

- **都乐**：拥有全球化种植基地（如菲律宾、哥斯达黎加等热带产区），通过不同产地的轮换采收，可以做到全年供应鲜菠萝。但其供应链更偏向鲜果流通，如果用于加工果酱，需确认是否有稳定的工业级果采购或冷库储备。
- **洪九果品**：以泰国、越南等东南亚菠萝为主，通过产地直采和冷链物流可维持全年供应。但需注意其菠萝主要为鲜食果品（如金钻、香水菠萝），若用于果酱生产，需提前沟通品规和糖度要求。

**建议你的茶饮店：**
1. 直接联系两家公司的**大客户部**或**加工渠道**，明确告知“用于果酱加工”的需求，询问是否有稳定的工业级菠萝供应（成熟度更高、价格更稳定）。
2. 考虑 **“鲜菠萝+冷冻菠萝泥”** 结合方案：全年可用都乐/洪九的鲜菠萝（旺季），淡季改用冷冻菠萝泥（如都乐已有此类加工产品）。
3. 若独家依赖一家，需签订**全年框架协议**并约定断供赔偿条款，因为台风、疫情等仍可能影响产地轮换。

**可执行性判断**：大概率可行，但需主动对接商务条件，不能默认“公开市场能买到即等于长期稳定供应”。
