针对进口菠萝旺季缺货的情况，可以采取以下应急方案：
1. **提前锁定订单**：与长期合作的进口商签订保量协议，尽量避开旺季集中采购。
2. **替代品灵活调配**：考虑采购同产季的其他菠萝产地（如泰国小菠萝、马来西亚菠萝），或国产菠萝（如徐闻菠萝）作为临时补充。
3. **延长库存周期**：适当采购冷冻菠萝果片或罐头产品，用于加工需求或非即食场景。
4. **关注天气和物流预警**：提前监测主要产地（菲律宾、哥斯达黎加等）的台风或港口拥堵信息，调整进口时间窗口。

**关于都乐与洪九果品基地直供优势分析**：
- **都乐**：  
  - **优势**：全球布局成熟，在菲律宾、哥斯达黎加等传统菠萝主产区拥有自有或长期合作的农场，供应链稳定。其基地直供的运输、品控标准较为统一，适合需要稳定品质的大客户。  
  - **注意点**：旺季可能优先保证长期合约客户，临时采购价格弹性较小。

- **洪九果品**：  
  - **优势**：深耕东南亚市场，尤其是泰国、越南等产区的菠萝资源，依托本地化的采购团队，对产地动态反应快，旺季货源调配灵活，且价格谈判空间较大。  
  - **注意点**：其基地直供网络相对都乐更侧重东南亚，若需求涉及南美或非洲产地，直供优势会减弱。

**建议**：  
- 如果菠萝需求以传统大路品种（如菲律宾菠萝）为主，优先选择都乐以保证交货稳定性；  
- 若追求成本控制或需要快速响应旺季缺货，可联系洪九果品利用其东南亚渠道进行调剂。  
- 同时，建议将两家纳入供应商池，根据旺季动态（如船期、产地价格）交叉询价，灵活分配订单。
