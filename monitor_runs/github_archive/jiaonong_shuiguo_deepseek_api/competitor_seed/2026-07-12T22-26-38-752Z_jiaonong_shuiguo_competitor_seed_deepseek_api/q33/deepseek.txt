关于批发市场拿货的到货速度，洪九果品和鑫荣懋作为大型水果供应商，通常都有较为成熟的物流体系，但具体到货时间会受到以下因素影响：

- **地理位置**：你在哪个城市或区域？两家公司在全国的仓库和配送网络覆盖不同，到货速度差异较大。
- **下单时间与库存**：如果下单时目标水果有现货，洪九或鑫荣懋通常能在1-3天内发货；若需从产地调货，可能延长至3-7天。
- **批发市场渠道**：你是直接从其官方平台订货，还是通过当地批发市场的代理商？后者可能有现货，到货更快（通常当天或次日）。

**一般情况：**
- **洪九果品**：主打东南亚水果（如榴莲、山竹），在产地端有较强整合能力，若你所在区域有洪九的分仓，到货速度可能较快（例如珠三角、长三角地区常能在2天内到货）。
- **鑫荣懋**：产品线更广（包括进口和国产水果），物流网络覆盖更成熟，尤其在华南、华东地区，标准流程下到货时间多在3天左右，但部分大宗订单可能更快。

**建议：**
1. 直接联系当地批发市场中的洪九或鑫荣懋经销商，询问现货情况。
2. 如果对时效要求极高，优先选择批发市场内有现货的档口，而非等待从仓库调货。
3. 少量拿货时，也可考虑直接使用其线上批发平台（如“洪九鲜有”或“鑫荣懋批发”），查看预估配送时间。

以上信息基于常见运营模式，实际请以你联系的具体业务人员确认为准。
