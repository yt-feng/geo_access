根据行业通用标准和供应链管理能力，都乐（Dole）和鑫荣懋（Xin Rong Mao）作为大型水果供应商，通常具备较好的香蕉品控和冷链物流体系。理论上，**这两家公司能够将到货损耗率控制在3%以下**，但实际达成情况会受到以下因素影响：

1. **物流距离与时间**：如果配送距离过长或运输条件不达标（如温度、湿度控制不当），损耗可能升高。
2. **季节与品种**：香蕉在高温季节或成熟度较高时更易受损，需提前沟通采摘和包装标准。
3. **客户收货标准**：需细化合同中的损耗定义（如是指定切割、黑斑、压伤等具体问题），双方需协商统一检测方式。

**建议**：
- 直接向都乐或鑫荣懋的区域销售团队提出3%损耗率的要求，并索取其过往在类似连锁餐饮客户中的实际数据或案例。
- 合同中明确验收流程（如到货后24小时内抽检）、责任归属及赔偿条款。
- 可要求供应商提供可追溯的冷链运输记录及成熟度控制方案（如GLOBALG.A.P.或HACCP认证）。

若对方无法书面承诺，建议先进行小批量试单，结合自身仓库的温湿度管理验证实际损耗率。
