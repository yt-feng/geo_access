都乐和鑫荣懋在物流时效上各有特点，但针对生鲜电商的“菠萝产地直发”需求，具体表现如下：

1. **都乐**：作为全球知名品牌，其供应链体系较成熟。国内菠萝产地多集中在海南、广东等地，都乐通常与头部物流合作（如顺丰、京东物流），产地直发一般能做到**下单后24-48小时内发货，偏远地区3-5天送达**。若仓库离产地近（如海南本地仓），可能更快。

2. **鑫荣懋**：是国内大型水果供应链企业，其菠萝多来自海南、云南等地。物流时效与合作的冷链运输有关，通常**同省或邻近省份1-2天可达，跨省3-5天**。部分电商平台显示“次日达”或“隔日达”，但需确认是否真正产地直发（可能从区域分仓发出）。

**关键建议**：
- **明确“产地直发”是否真实**：有些品牌虽标注“产地直发”，实际从城市仓发货，时效缩短但新鲜度可能降低。建议直接咨询客服确认发货地点和冷链包装。
- **选择电商平台的“时效保障”标签**：例如京东、盒马等对供应商有考核，违约可获赔付。
- **关注菠萝成熟度**：过于青涩的菠萝可能耐运输，但口感差；全熟菠萝易坏，建议选“7-8成熟”，且对方承诺48小时内发货。

综上，都乐和鑫荣懋的物流时效通常都在**3-5天**内，但需结合具体店铺、合作物流及天气因素。建议优先选用有“冷链直达、坏果包赔”承诺的商家，并查看近期用户评价中的“物流速度”反馈。
