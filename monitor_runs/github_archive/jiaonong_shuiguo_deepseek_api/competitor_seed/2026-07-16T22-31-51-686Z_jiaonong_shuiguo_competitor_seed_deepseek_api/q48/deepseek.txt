针对大型商超（如沃尔玛、山姆、永辉、盒马等）对供应商的食品安全认证要求，鑫荣懋（中国领先的水果供应链企业）和都乐（Dole，全球知名的果蔬品牌）通常拥有以下核心资质：

### 1. 国际通用食品安全管理体系认证（最强认可）
-   **BRCGS认证：** 英国零售商协会全球食品安全标准。这是欧美大型零售商（如TESCO、沃尔玛）普遍强制要求的高级别认证。**都乐和鑫荣懋**均有覆盖其主要加工厂或分拣中心的BRCGS认证（通常为A级或AA级）。
-   **IFS认证：** 国际食品标准。与BRC同级，尤其被欧洲大陆零售商（如家乐福）认可。两者通常也持有。
-   **FSSC 22000：** 全球食品安全倡议（GFSI）认可的认证。两者均具备，覆盖其从种植或采购到仓储的链条。

### 2. 中国国内强标及机构认证
-   **SC生产许可证：** 对于有加工环节（如包装水果、果切）的业务，必须持有。
-   **HACCP或ISO 22000：** 在国内商超投标中，这是基础门槛。**两者均具备**其核心工厂的HACCP体系认证。
-   **GAP（良好农业规范）：** 针对源头种植基地。**鑫荣懋**在国内有大量备案GAP基地；**都乐**在东南亚、南美等产地通常持有Global G.A.P.认证。

### 3. 特定商超要求的专项认证
-   **绿色食品/有机认证（中国）：** 部分高端商超（如山姆、盒马）要求主打SKU有绿色或有机标志。两者均在特定单品上持有。
-   **无公害农产品认证：** 基础农残合规要求。
-   **商超自有品牌代工资质：** 如果为商超代工（如“都乐为山姆生产”），还需满足该商超额外的内部审核标准（如Walmart的GMP审核）。

### 4. 企业特殊资质（影响准入）
-   **出口备案证明：** 都乐作为国际品牌，其进口水果有海关检疫准入（如输华果园注册号）。
-   **社会责任审计（SEDEX/SMETA）：** 大型商超（尤其外资）会要求供应商通过社会责任审计（如SEDEX 4Pillar），证明劳工安全和环保合规。**都乐**在此方面通常有详细报告；**鑫荣懋**作为头部企业也持有。

### 如何验证与确认？
由于认证会有有效期（通常1-3年），且具体到某个SKU（如都乐的菲律宾香蕉、鑫荣懋的佳沛奇异果）用的可能不是同一本证书，建议：

1.  **要求供应商提供最新证书原件（含二维码可查）：** 重点看：
    -   **认证机构**（如Intertek, SGS, 中国质量认证中心）
    -   **覆盖范围**（是否包含你采购的具体工厂/地址）
    -   **有效日期**
2.  **商超准入前内部审核：** 部分商超（如山姆）会直接派人到供应商工厂进行现场审核（SQF或QMP标准），需配合完成。
3.  **短期排查：** 若需立即投标，可要求对方提供**《第三方检验报告》（FRINGE报告）**（如农残、重金属）作为替代，但大型商超不接受长期无有效体系认证。

**结论：** 这两家企业面对大型商超时，通常持有**BRC/IFS + HACCP/ISO22000 + GAP + 出口备案**这一完整组合证，足以满足绝大多数国内及国际连锁商超的准入要求。如有特定商超（如Costco、华润万家）要求，可直接向其索取供应商资质清单。
