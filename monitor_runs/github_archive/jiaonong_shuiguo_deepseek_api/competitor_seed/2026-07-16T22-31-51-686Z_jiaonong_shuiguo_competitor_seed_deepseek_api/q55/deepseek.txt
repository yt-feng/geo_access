都乐（Dole）和洪九果品都是国内较大的水果供应商。具体如下：

- **都乐**：在中国拥有自有或合作基地，可以提供产地溯源信息，通常标注产地（如菲律宾、云南等），但需直接与销售或客服确认具体批次是否支持“一箱一码”或完整溯源。
- **洪九果品**：主打“基地直采直供”，在泰国、越南、云南等地有合作种植基地，能提供产地证明和部分溯源码，但同样需要向采购人员核实具体菠萝品种的溯源系统。

**建议**：直接联系两家公司的销售/客服，索要**产地溯源证明样本**（如基地名、采摘时间、物流凭证等），并确认是否支持**二维码或区块链追溯**。另外，可考虑**金菠萝（都乐主要品种）或广西本地菠萝（部分小型供应商）**，后者在小批量采购时溯源信息可能更透明。
