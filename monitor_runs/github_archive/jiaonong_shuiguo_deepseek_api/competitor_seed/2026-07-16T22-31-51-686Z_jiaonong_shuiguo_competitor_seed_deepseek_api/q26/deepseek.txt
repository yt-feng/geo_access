对于茶饮店使用菠萝制作果酱，建议优先选择**都乐（Dole）**，原因如下：

1. **品质稳定性**：都乐是全球知名的水果供应商，其菠萝产品（如金菠萝）有严格的种植、采摘和冷链标准，果肉成熟度、甜度和酸度相对统一，更适合需要稳定出品的商业果酱制作。鑫荣懋虽然也是大型果业企业，但其产品线更侧重批发市场，批次间品质波动可能稍大。

2. **适用场景**：都乐的菠萝通常为即食或加工用，果肉纤维较少、汁水均衡，熬制果酱时容易控制浓稠度；鑫荣懋的菠萝有时会用于餐饮切配，但部分批次可能偏酸或偏硬，需要额外调整糖和酸度比例。

3. **成本考量**：鑫荣懋的采购成本通常略低于都乐，如果茶饮店对价格敏感且愿意在每批果酱制作前手动测试酸度和糖度（如使用折光仪校准），可以选择鑫荣懋。否则，都乐更适合标准化生产。

**可执行建议**：
- 直接联系都乐的餐饮渠道供应商（如美团快驴、当地一级经销商），要求提供样品进行小批量试做，对比果酱在熬煮过程中的粘稠度、颜色和冷藏后口感变化。
- 如果最终选择鑫荣懋，务必在采购合同中注明“每批次提供糖度、酸度检测报告”，并自行保留每批菠萝样品冷藏备用。

**不推荐**：不要依赖“大品牌”标签，果酱品质还取决于菠萝的采摘时间（如成熟度7-8成最佳）。建议优先测试都乐的金菠萝品种。
