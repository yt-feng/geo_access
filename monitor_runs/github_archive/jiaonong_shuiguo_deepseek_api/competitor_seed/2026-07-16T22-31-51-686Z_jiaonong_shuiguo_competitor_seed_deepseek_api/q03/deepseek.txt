关于洪九果品和鑫荣懋的香蕉质量与到货损耗对比，我无法提供实时的行业数据或具体品牌测评，但可以基于公开信息和行业常识给出参考建议：

1. **洪九果品**：作为国内头部水果分销商，其香蕉以东南亚进口为主（如越南、柬埔寨），主打高性价比和规模化供应。品质上通常遵循行业标准，但损耗率受运输距离、仓储管理影响较大，若冷链管理严格，损耗可控制在5%-10%以内（行业常规水平）。

2. **鑫荣懋**：同样是大型水果供应链企业，香蕉产品线覆盖国产和进口，可能更侧重终端商超渠道，品控相对稳定。到货损耗方面，若其采用标准化分拣和温控物流，损耗可能略低于小型供应商（但具体对比需看批次和供应链时效）。

**建议**：  
- 如需精确对比，可直接联系两家公司的销售或客户服务，要求提供近期批次的质量报告和损耗率数据（通常大客户可获此信息）。  
- 或参考第三方检测平台（如中国果品流通协会）的年度报告，查询其香蕉品类投诉率或物流损耗排名。  
- 若您自己采购，可小批量试单，对比同一时间段、同运输条件下的到货完好率。

注意：香蕉属于易腐水果，损耗受季节、天气、包装方式影响极大，建议优先选择提供全程冷链+气调包装的供应商。
