除了都乐（Dole），目前市场上稳定供应的菠萝品牌和渠道还有以下几个：

1. **佳沃（Joyvio）**  
   国内大型水果企业，全年有稳定的菠萝供应（主要来自菲律宾、海南），在大型商超（如盒马、永辉）和电商平台（京东自营、天猫）均有销售。

2. **鑫荣懋（Xinrongmao）**  
   大型水果供应链公司，旗下自有品牌或代理进口菠萝（如菲律宾菠萝），供应稳定，批发市场和连锁超市常见。

3. **金菠萝（Golden Pineapple）**  
   特指菲律宾进口的“MD2”品种（如都乐同品种），很多品牌（如元祥、洪九）均有代理，主要标注为“超甜金菠萝”，常年在精品超市（如山姆、Ole'）和社区团购平台供应。

4. **海南本土品牌（如“南国”、“椰岛”、“农垦”）**  
   主要供应海南菠萝（如“巴厘”品种），季节性强（3-6月为主），但部分合作社或种植基地（如“海南天地人”）在收果期供应量较大且稳定。

5. **水果连锁超市（百果园、鲜丰水果）**  
   自营品牌或合作采购，常年供应都乐、佳农、金菠萝等多品牌，购买便利。

**建议**：  
- 如果追求全年稳定供应，优先选**佳沃**或**鑫荣懋**的菲律宾金菠萝。  
- 如果希望性价比高，可关注**百果园**或**京东自营**的进口菠萝促销。  
- 若不介意季节性，可尝试**海南产地直销**的鲜切菠萝（社区团购或拼多多上的产地直发）。  

（注：以上信息基于2023-2024年市场常见供应情况，具体库存需以购买时的商家页面或门店咨询为准。）
