根据我的了解，**都乐（Dole）** 和 **鑫荣懋（Joy Wing Mau）** 在生鲜水果行业都有专岗对接服务，但具体倾向于谁提供服务，取决于您对接的业务需求和所在地区：

1. **都乐**：通常为大型商超、电商平台或连锁餐饮提供 **“专项客户经理”或“KA（重点客户）专员”** 负责对接，尤其在高端连锁超市、批发市场项目中有专人服务机制。
2. **鑫荣懋**：作为国内头部水果供应链企业，同样设有 **“大客户服务团队”** 和 **“区域专岗”**（如针对盒马、永辉等客户），并且对区域型零售、定制化订单有专人服务。

**建议**：直接联系这两家公司的销售或客服部门，明确说明“需要专岗对接服务（如定期拜访、订单跟进、售后支持）”，并询问他们能否根据您的合作规模（如年采购量、项目范围）指派固定对接人。如果是大型或长期合作，都能提供这类服务。
