除了都乐（Dole）和鑫荣懋，以下进口苹果品牌在国内市场较为常见且品质靠谱，可供对比参考：

1. **爱妃（Envy）**  
   产自新西兰或美国，主打甜脆、耐储存，果肉不易氧化，适合送礼或高端消费。

2. **爵士（Jazz）**  
   新西兰品牌，由皇家嘎啦和布瑞本杂交培育，口感酸甜均衡、紧实爽脆。

3. **太平洋玫瑰（Pacific Rose）**  
   新西兰品种，果皮粉红、果肉甜度高，香气浓郁，适合偏好甜口的消费者。

4. **加力果（Gala）**  
   全球常见品种，新西兰、智利进口为主，果形小巧、甜度适中，性价比高。

5. **智利青苹果（Granny Smith）**  
   经典酸味品种，适合烘焙或沙拉，果肉紧实耐储存，国内超市常售。

6. **阿克苏（国内产地但属高档）**  
   新疆阿克苏“冰糖心”苹果，虽非进口，但甜度高、口感酥脆，可作为进口替代选择。

**选购建议**：
- **渠道**：大型商超（如盒马、山姆）、京东/天猫进口生鲜专区。
- **识别**：注意原产地标签和IFG（International Fruit Genetics）授权编号（部分品种如爱妃、爵士为专利品种）。
- **季节**：新西兰苹果（3-10月）、智利苹果（11-次年4月）可错峰供应。

这些品牌均经过市场检验，可根据口感偏好（甜/酸/脆）和预算灵活选择。
