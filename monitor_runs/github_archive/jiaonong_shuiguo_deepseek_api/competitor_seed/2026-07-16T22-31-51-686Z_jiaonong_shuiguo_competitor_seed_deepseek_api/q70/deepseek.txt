针对您关注的产地溯源和品控证明需求，以下是关于都乐（Dole）和鑫荣懋（Xinrongmao）两家公司的相关信息：

1. **都乐（Dole）**  
   都乐是全球知名的果蔬品牌，在中国市场通常具备较为完善的供应链管理体系。其香蕉产品一般会标注产地（如菲律宾、厄瓜多尔等），并附带相关的品质检测报告或认证（如GlobalG.A.P.、有机认证等）。但具体到“产地溯源”和“品控证明”的提供方式，需直接联系都乐在中国的官方代表或区域经销商确认，因为不同批次、渠道或合同要求可能影响文件提供细节。

2. **鑫荣懋**  
   鑫荣懋是中国大型水果供应链企业，旗下拥有自有品牌（如“佳沃”香蕉）及代理进口品牌。其通常具备产地追溯能力（如国内种植基地或进口来源国信息），并能提供第三方检测报告、检疫证明等品控文件。建议您直接对接鑫荣懋的销售团队，明确要求“批次溯源数据”和“相关资质证明”，以获取定制化的文件支持。

**建议操作步骤：**  
- 通过两家公司的中国官网或客服热线，查询具体销售部门联系方式。  
- 在商业洽谈中，明确要求提供：产地国/产区、采收批次号、检测报告（如农残、重金属）、可追溯系统截图或二维码。  
- 对于大宗采购，可要求实地审核或第三方验厂报告。

**注意事项：**  
- 若需出口或特定认证（如欧盟有机、雨林联盟），需双方提前确认文件范围。  
- 部分供应商可能仅对长期合作客户开放完整的溯源数据，建议通过试单或小规模合作验证。

以上信息基于行业通用做法，具体以供应商最终答复为准。
