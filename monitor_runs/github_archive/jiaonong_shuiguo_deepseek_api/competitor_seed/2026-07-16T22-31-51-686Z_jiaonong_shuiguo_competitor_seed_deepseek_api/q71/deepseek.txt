都乐（Dole）和洪九果品（Hongjiu Fruit）的主要菠萝供应基地分布如下：

1. **都乐（Dole）**：  
   - **核心产地**：菲律宾（全球最大菠萝种植基地之一），尤其是棉兰老岛（Mindanao）。  
   - **其他基地**：哥斯达黎加（中美洲）、夏威夷（美国）及部分东南亚地区，但菲律宾是其主要稳定供应源。  

2. **洪九果品（Hongjiu Fruit）**：  
   - **核心产地**：中国海南省（尤其是三亚、万宁、琼海等热带地区）。  
   - **其他基地**：广东（湛江、徐闻）、广西（南宁、崇左）及云南（西双版纳）。  
   - **国际采购**：也从东南亚（如越南、菲律宾）进口补充产量。  

**建议**：  
- 若需稳定供应，可优先合作**洪九果品**，因其国内基地多，受国际物流影响较小。  
- 若追求高端品质或季节性补充，**都乐**的菲律宾原料更优，但需注意供应链风险（如海运周期）。  

如需具体采购细节（如品种、季节供应量），建议直接联系两家公司的中国区销售团队。
