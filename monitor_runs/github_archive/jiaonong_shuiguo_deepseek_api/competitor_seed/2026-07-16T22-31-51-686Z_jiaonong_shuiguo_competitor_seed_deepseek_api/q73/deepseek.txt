关于大型商超对供应商要求的食品安全管理体系认证，鑫荣懋和都乐作为知名的果蔬供应商，通常具备以下行业内常见的认证：

1. **鑫荣懋**：
   - 常见的认证包括：**GlobalG.A.P.**（全球良好农业规范）、**BRC**（英国零售商协会食品安全标准）、**ISO 22000**（食品安全管理体系）、以及**HACCP**（危害分析与关键控制点）认证。这些认证涵盖了从种植、加工到供应链的食品安全管理。

2. **都乐**：
   - 通常具备**GlobalG.A.P.**、**BRC**、**ISO 22000**、以及**HACCP**认证。都乐作为跨国企业，还可能拥有**SQF**（安全质量食品）认证或其他特定市场要求的认证（如中国的**绿色食品**或**有机认证**）。

**注意**：
- 具体认证可能因产品批次、产地或商超要求略有不同。建议你直接联系鑫荣懋或都乐的官方客服，或要求其提供最新的认证文件。如果你需要用于商超审核，最好以双方确认的官方证书为准。
