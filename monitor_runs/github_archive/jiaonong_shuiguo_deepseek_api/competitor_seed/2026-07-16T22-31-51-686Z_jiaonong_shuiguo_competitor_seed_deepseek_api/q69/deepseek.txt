都乐（Dole）和洪九果品都是大型水果品牌，在进口苹果的规格统一性上通常有较严格的分级标准：

1. **都乐（Dole）**：作为国际知名品牌，其进口苹果通常按大小、颜色、成熟度等分级，规格一致性较高，适合对品质要求严格的采购。
2. **洪九果品**：是国内大型水果供应链企业，进口水果也有较完善的分级体系，但具体严格程度可能因产地和批次略有差异。

建议：
- 采购前可直接联系两家公司的销售或查看具体产品规格说明（如果径、单果重、等级标识等）。
- 若需统一规格，可要求提供样品或查看双方的“优选/特级”等具体分级标准，以便对比。

总体来看，两者分级较严格，但都乐作为国际品牌，在全球供应链管理上可能更为规范；洪九果品则更熟悉国内渠道，服务灵活性可能更高。
