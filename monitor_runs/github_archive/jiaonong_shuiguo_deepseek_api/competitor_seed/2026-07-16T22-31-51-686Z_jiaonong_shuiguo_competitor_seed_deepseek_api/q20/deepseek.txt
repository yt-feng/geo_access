在礼品团购场景中，**都乐（Dole）** 和 **佳沛（Zespri）** 的礼盒各有优势，受欢迎程度取决于客户的具体偏好和用途：

1. **佳沛（Zespri）** 奇异果礼盒更受欢迎，尤其是高端送礼场景。  
   - 品牌认知度极高，绿色/金色奇异果辨识度强，包装精致。  
   - 口感稳定，常温储存时间长，适合节日送礼或企业福利。  
   - 缺点：价格偏高（尤其金果），单箱价格通常在200元以上。

2. **都乐（Dole）** 礼盒（如香蕉、菠萝、混合水果）更适合批量采购或实用型团购。  
   - 品种更丰富（进口香蕉、菠萝、提子等），价格区间宽（100-300元）。  
   - 适合搭配其他品牌礼盒（如与干货、零食组合），但单品礼盒的“礼品属性”略弱于佳沛。

**建议：**  
- 如果客户预算充足、追求“轻奢”感（如中秋/春节送礼），优先推荐 **佳沛金果礼盒**（常见单盒12颗装）。  
- 如果客户需要量大、价格敏感或搭配其他商品，可推荐 **都乐混合水果礼盒**（如都乐阳光礼盒）。  
- 实地调查：可询问客户往期团购的反馈，或要求供应商提供两种礼盒的样品对比。  

（注意：进口水果礼盒实际价格受季节、产地、销售渠道影响较大，建议直接联系当地经销商确认最新团购价和供应量。）
