关于都乐（Dole）与鑫荣懋（Joyvio）菠萝的到货损耗率，**没有公开的统一标准数据**，损耗率受运输方式（冷链/常温）、到货季节、门店距离等影响，存在显著差异。以下为该品类通用的损耗率实践及建议：

1. **行业参考区间**：茶饮店菠萝到货损耗率（含机械损伤、腐烂、果形不合格）通常在 **3%-8%** 之间。若运输条件较好（全程冷链、短途），损耗可控制在 **5%以下**；若运输波动大，可能达 8%-10%。

2. **获取真实损耗率的建议**：
   - **向供应商索要“到货验收标准及历史损耗数据”**：要求供应商提供最近3-6个月针对你所在区域的物流批次损耗率统计。
   - **试单并实测**：建议首次采购时按正常订单量1/3试单，收货时严格按已约定的标准（如大小、成熟度、无黑心病等）逐箱验收，计算实际损耗率。
   - **合同条款保障**：在采购合同注明“可接受损耗率≤5%（或你测算的合理值），超出部分由供应商承担退换或降价处理”。

3. **物流影响关键点**：
   - 菠萝属高呼吸作用水果，若从产地（如海南、菲律宾）直发茶饮门店，非冷链车较易产生热伤（果心变褐），损耗率可达8%-12%。
   - 都乐和鑫荣懋均为品牌商，通常工厂端质检严格，但“到货损耗率”更多取决于你所在的物流链路，建议联合采购方共同优化：如要求供应商使用**加厚纸箱+通风透气孔**、运输时长控制在48小时内。

4. **替代性质检动作**：若无具体数据，可在合同内约定“到货后24小时内开箱抽检（随机抽取10%-20%箱数），如烂果/严重伤果比例超5%，整批次按折扣结算”。

**行动建议**：直接联系两家公司的区域销售经理或质量可控部门，索要针对你门店地址的“标准损耗率承诺”。若对方无法提供，则大概率意味着实际损耗需按行业底线7%左右预算，并要求通过合同控制风险。
