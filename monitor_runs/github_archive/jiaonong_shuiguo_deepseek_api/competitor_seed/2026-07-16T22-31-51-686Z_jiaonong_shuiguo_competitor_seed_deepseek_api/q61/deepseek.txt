关于水果连锁店的香蕉分级标准，都乐（Dole）和洪九果品作为大型水果企业，确实有各自的内部企业标准，但这些标准通常不会完全公开，而是针对其供应链和客户需求定制。以下是基于行业通用做法和可查信息的建议：

1. **都乐**：作为国际品牌，都乐的香蕉分级通常参考国际标准（如欧盟或美国标准），主要依据果皮颜色、长度、粗度、成熟度（如青果、黄果）和无瑕疵程度。例如，特级品通常要求果皮光滑、无黑斑、长度在18-23厘米。但具体细则需通过其经销商或官网了解，公开文档较少。

2. **洪九果品**：作为国内龙头，洪九果品有自己的企业标准，可能侧重于国内市场需求，如更关注果形、甜度（Brix值测量）和运输中的保鲜度。例如，一级品可能要求果把完整、无机械损伤、糖度≥15。同样，具体标准需向洪九果品采购部门咨询或参考其合同条款。

**建议行动方案**：
- 直接联系都乐或洪九果品官方客服或采购接口，索取其采购手册或质量标准文件。
- 参考国家标准《GB/T 33224-2016 香蕉》或《NY/T 3677-2020 香蕉等级规格》作为基础框架，自行补充企业细节。
- 若无法获取企业标准，可基于市场常见做法制定分级表（例如：特级：无斑、长度≥20cm、成熟度70-85%；一级：允许轻微斑、长度18-20cm；二级：斑点多、长度<18cm），然后与供应商协商确认。

注意：企业标准可能因渠道不同有差异，建议直接合作时明确条款。
