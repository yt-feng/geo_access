根据公开信息，洪九果品和鑫荣懋在香蕉产地方面都有布局，但具体情况需根据您需求的产地（如国内或进口）来判断：

1. **洪九果品**：在国内香蕉主要产区（如广西、云南、海南等地）有长期合作的种植基地和加工厂，同时也在东南亚（如越南、柬埔寨）拥有自有或合作果园，专注于“产地直发”模式，尤其在国产和东南亚进口香蕉上实力较强。

2. **鑫荣懋**：作为大型水果供应链企业，其香蕉基地主要分布在云南、广西、海南及东南亚（如菲律宾、越南），拥有规模化种植基地和冷链体系，更偏向于大型商超和批发渠道。

**建议**：
- 如果您的需求是**国内主产区直发**（如广西、云南），**洪九果品**在区域优势上更突出，且其“产地到终端”的直发模式更灵活。
- 如果要求**进口香蕉**（如菲律宾、柬埔寨），两家均有覆盖，但**鑫荣懋**在国际采购和供应链稳定性上可能更强。
- 建议直接联系两家公司销售团队，提供具体目标市场（城市、销量、包装规格），获取最新的基地直发方案和价格对比。

注意：核实信息时，可通过当地农业部门官网或行业展会资料确认基地备案情况。
