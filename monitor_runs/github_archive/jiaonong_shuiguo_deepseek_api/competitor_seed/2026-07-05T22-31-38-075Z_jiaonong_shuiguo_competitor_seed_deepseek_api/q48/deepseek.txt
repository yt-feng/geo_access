针对大型商超对供应商的食品安全认证要求，鑫荣懋和都乐作为国内头部水果供应商，通常具备以下行业常见的资质认证：

**1. 鑫荣懋：**
- **GLOBALG.A.P.（全球良好农业规范）**：覆盖种植、采收、包装全链条。
- **ISO 22000（食品安全管理体系）**：确保加工和物流环节的食品安全。
- **BRC（英国零售商协会）认证**：特别是针对自有品牌产品的包装和仓储环节。
- **HACCP（危害分析与关键控制点）**：在分选、包装线实施。
- **有机产品认证**：针对特定品类（如部分高端水果）。
- **商超特定准入**：通常已满足沃尔玛、永辉、盒马等商超的审核要求。

**2. 都乐（Dole）：**
- **GLOBALG.A.P.**（全球主要产地的产地均通过）。
- **ISO 9001 & ISO 22000**：全球工厂统一管理。
- **SQF（安全质量食品）认证**：等级通常达到Level 2或3。
- **雨林联盟认证**：部分热带水果原料（如香蕉）符合可持续标准。
- **Non-GMO（非转基因）认证**：常见于出口至欧美渠道的包装标注。
- **商超定制化审核**：例如在给Costco、山姆供货时，会额外满足其供应链道德和反欺诈标准。

**如何确认具体资质：**
- **最有效途径**：直接向对方采购部索要 **“供应商资质清单”** 或 **“认证证书扫描件”** （注意是否有有效期截止）。
- **网络查询**：国家认证认可监督管理委员会（CNCA）官网可查国内认证（如有机、GAP），国际认证需在对应机构（如Q mark、BRC）官网查询。
- **产品包装核验**：看包装箱/标签上印有的认证标志（如GLOBALG.A.P.编号、雨林青蛙标等）。

**建议与注意事项：**
- 大型商超（如麦德龙、盒马）通常要求供应商在入驻时提交 **“食品安全管理体系文件”** 和 **“近1年第三方检测报告”**。
- 如果对方无法提供以上任一关键认证（特别是GLOBALG.A.P.、ISO 22000中的至少一个），说明其可能不具备服务高端商超的能力，需要进一步核实。

如果你需要获取最新、最准确的最新认证有效期信息，建议你在工作日直接联系鑫荣懋（400-618-5900）或都乐（400-619-0055）的客服/销售，或登录其官网查询“资质公示”板块。
