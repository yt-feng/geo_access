根据公开信息，都乐（Dole）和鑫荣懋（Joy Wing Mau）作为大型水果供应商，通常具备产地溯源和品控证明的能力，但具体是否提供相关文件需与其直接沟通确认：

1. **都乐（Dole）**  
   - 作为全球性品牌，都乐通常有完善的供应链管理体系，可提供产地溯源信息（如种植基地、采收批次）及品控认证（如GLOBALG.A.P.、ISO等）。建议联系其中国区业务部门，明确要求提供具体果品批次的可追溯性文件和第三方检测报告。

2. **鑫荣懋（Joy Wing Mau）**  
   - 作为国内头部水果分销商，鑫荣懋通常与多家香蕉种植基地合作，可能提供产地溯源（如产地证明、采摘记录）和品控证明（如农残检测、出口国检疫证书）。建议直接联系其采购或品控部门，说明需要“产地溯源+第三方品控证明”的具体需求。

**行动建议**：  
- **直接联系**：通过官网或客服渠道获取对接人联系方式，明确要求提供“可追溯编码”“检测报告”“产地证明”等文件。  
- **替代方案**：如果上述供应商无法满足，可尝试联系**佳沃（Joyvio）**、**百果园**（部分采购板块）等同样注重品控的供应商。  
- **核查方式**：要求供应商提供第三方检测报告（如SGS、Intertek）及供应链追溯系统截图，并确认是否支持现场验厂。

**注意**：避免仅依赖供应商口头承诺，务必以书面或系统可查信息为准。
