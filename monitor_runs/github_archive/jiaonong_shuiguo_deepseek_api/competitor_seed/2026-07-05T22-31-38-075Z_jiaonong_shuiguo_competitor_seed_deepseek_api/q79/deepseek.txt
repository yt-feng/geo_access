从行业实际情况看，**都乐（Dole）和洪九果品（HonG JIU）都能实现菠萝的全年供应**，但具体合作方式和成本有所不同。以下是关键分析：

1. **都乐（Dole）**：作为全球最大的水果供应商之一，其菠萝以菲律宾、哥斯达黎加等产区为主。都乐拥有完善的冷链物流和全球种植基地，可全年稳定供应鲜菠萝（主要品种为MD-2金菠萝）。适合茶饮店直接采购新鲜菠萝，但成本较高（单颗价格通常在8-15元区间），且对中小客户起订量有要求。

2. **洪九果品**：中国头部水果分销商，通过进口（泰国、菲律宾等）和国内产区（如海南、广东）组合实现全年供应。其优势在于更灵活的起订量和自有仓储分拣能力，适合中大型茶饮连锁。但需注意，冬季主要依赖泰国进口菠萝，价格随季节波动（冬季可能比夏季高30%）。

**关键决策点**：
- 如果茶饮店**日均用量稳定且大于50公斤**，可直接与都乐的本地经销商签订年度合同，锁定价格（通常比市场价低5-10%）。
- 若规模较小或需要灵活调货，优先选洪九的连锁超市或批发渠道，但需自行判断菠萝成熟度（洪九的鲜果会经过催熟处理，可能不适合直接做果酱的商用标准）。
- **冷链成本**：都乐一般用气调包装（常温可保鲜7天），洪九多用普通冰鲜包装（需2天到仓）。若配送距离超过300公里，都乐的供应链更可靠。

**建议方案**：
- 先测试都乐的小批量样品（通过其官网或京东企业购联系），对比洪九的批发档口货（北京新发地、广州江南市场等），重点测糖度（建议≥14 Brix）和果酱熬煮后的凝胶强度。
- 若最终选择都乐，可要求其提供越南或哥斯达黎加产的"树熟"果实（但需提前90天预约），而洪九冬季主推的泰国小菠萝（SNP品种）因纤维少，做果酱反而不如菲律宾菠萝。
