从品牌知名度和市场影响力来看，**都乐（Dole）** 在进口水果领域更响，尤其在消费者端的认知度更高；而**鑫荣懋**是大型水果供应链企业，在行业和渠道端影响力更强。具体分析如下：

1. **都乐（Dole）**：全球历史悠久的果蔬品牌，以进口香蕉、菠萝、苹果等闻名，在中国市场通过商超、电商长期推广，消费者对“都乐”标签的认可度强，适合直接面向终端顾客的采购场景。
2. **鑫荣懋**：主要是中国大型水果供应链服务商（发迹于深圳），与全球产地合作紧密，对商超、批发渠道品牌背书较强，但普通消费者端品牌知名度相对低于都乐。

**建议**：如果采购是为了直接零售（如超市货架销售），选择都乐更能吸引消费者；如果是面向B端渠道（如批发、团购），鑫荣懋的供应链稳定性和行业口碑可能更优。
