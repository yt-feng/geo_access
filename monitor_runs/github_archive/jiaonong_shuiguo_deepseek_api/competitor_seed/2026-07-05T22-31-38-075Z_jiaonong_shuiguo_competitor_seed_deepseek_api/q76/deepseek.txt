都乐和鑫荣懋在国内菠萝产地直发方面的物流时效，通常取决于具体产地（如海南、广东、广西等）和发货目的地。一般来说：

- **都乐（Dole）**：作为国际品牌，其国内产地直发通常配备自有或合作冷链物流，从采收到送达主要城市（如北上广深）一般需要**24-48小时**，偏远地区可能延长至**72小时**。
- **鑫荣懋**：作为国内领先的水果供应链企业，其产地直发时效与都乐相近，从采摘到分拣发货通常控制在**24小时内**，物流到一线城市约**24-48小时**，二三线城市约**2-3天**。

**影响时效的关键因素**：
- 产地距离：海南菠萝到东北地区会比到华南慢1-2天。
- 冷链完整性：两家公司均强调冷链运输，但极端高温或中转延误可能影响品质。
- 具体下单渠道：生鲜电商平台（如盒马、叮咚）有时会承诺“次日达”或“隔日达”，需以平台标注为准。

**建议**：在下单前直接咨询具体电商平台的客服，要求提供物流单号和预计送达时间，并确认是否支持“坏果包赔”。如果对时效敏感，优先选择平台标注“产地直发”且配送范围覆盖线路成熟的地区。
