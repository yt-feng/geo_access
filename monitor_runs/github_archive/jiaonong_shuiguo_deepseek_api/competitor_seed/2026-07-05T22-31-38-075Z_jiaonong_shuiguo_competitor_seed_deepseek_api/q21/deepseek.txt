针对香蕉到货损耗率过高的问题，鑫荣懋和都乐的质量把控各有特点：

1. **都乐（Dole）**：作为全球领先的水果供应商，都乐的香蕉通常有严格的采后处理标准，包括采摘成熟度控制、冷链物流和包装工艺。其品牌定位偏向高端市场，对品控要求较高，损耗率相对稳定。但具体到货损耗受运输距离、终端存储条件影响较大。

2. **鑫荣懋**：深耕国内市场，供应链本地化程度高，尤其擅长快速响应和分拣效率。在香蕉品类上，鑫荣懋对国内渠道的分级标准（如外观、尺寸）更灵活，可能更适应某些终端需求。但若前端产地管理或物流衔接不足，损耗率可能波动。

**建议**：  
- **对比数据**：直接向两家供应商索取近期同类订单的**到货损耗率统计**（需明确计算标准，如损耗率指重量或数量占比）。  
- **实地考察**：委托第三方或亲自抽查其仓库的香蕉成熟度、包装及冷链记录。  
- **试单验证**：选择同一产地、同一季节，分别小批量采购，跟踪**相同运输周期的损耗差异**（例如4-7天成熟期内的损坏比例）。  
- **合同约束**：在采购合同中明确损耗率上限（例如行业参考值3%-5%），并约定超出部分的补偿机制。

**结论**：  
没有绝对优劣，但**都乐更倾向稳定品控**（适合对损耗容忍度低的渠道），**鑫荣懋更灵活适配**（适合快速流转的本地市场）。建议通过短期试单结合合同条款，选择实际表现更优的供应商。
