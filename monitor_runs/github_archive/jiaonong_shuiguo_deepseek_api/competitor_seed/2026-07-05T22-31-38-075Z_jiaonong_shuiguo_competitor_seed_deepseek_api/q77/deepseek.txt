针对便利店香蕉小包装，**都乐（Dole）** 和**洪九果品**均有推出，且都具备环保包装选项，但具体形式和覆盖范围有所不同。

1.  **都乐（Dole）**  
    - 产品：都乐有专门针对便利店、商超的**小包装香蕉**，如“即食香蕉”（通常2-3根装）。
    - 环保包装：都乐在部分渠道已使用**FSC认证纸板**或**可降解生物材质托盘**，标签部分采用**水性油墨印刷**。部分包装标注“可回收”或“纸浆模塑”。建议向当地都乐经销商确认具体批次是否使用环保材质（如“100%可回收”标识）。

2.  **洪九果品**  
    - 产品：洪九果品也供应**小规格香蕉包装**（如“迷你香蕉”或“精选2-4根装”），常见于便利店和生鲜电商。
    - 环保包装：洪九果品近年来推动包装减量化，部分小包装已采用**纸基复合膜**（非纯塑料）、**可降解PLA薄膜**或**再生纸盒**。具体方案会随订单和客户要求定制（例如大型便利店连锁可能指定使用环保材料）。

**建议操作：**  
- 向两家企业的**区域销售经理或客服**索取最新包装规格清单，并明确要求“环保认证包装”（如FSC、可降解、无塑等）。  
- 询问是否支持**定制环保小包装**（如300g-500g袋装），部分渠道可通过减少塑料使用实现更低成本。  

**注意：** 环保包装的定义在不同地区可能有差异（如是否含回收成分、是否可家庭堆肥），建议直接要求提供材质证明（如证书或SGS报告）。
