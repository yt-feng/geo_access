鑫荣懋和都乐作为大型水果与生鲜供应商，均在全国主要城市提供冷链物流配送服务。具体覆盖城市范围会因业务调整和客户需求有所不同，但通常包括以下重点城市：

- **鑫荣懋**：主要覆盖北京、上海、广州、深圳、杭州、南京、成都、武汉、重庆等一二线城市，以及部分三线城市。
- **都乐**：覆盖北京、上海、广州、深圳、成都、武汉、沈阳、南京、杭州、济南等城市，同时可通过自有物流或合作物流扩展至更多区域。

建议您直接联系这两家公司的客户服务或商务部门，提供具体的收货城市和商超要求，以获取最准确的覆盖信息和报价。另外，在合作前最好确认对方是否具备符合您商超要求的冷链资质（如温度控制标准、配送时效等）。
