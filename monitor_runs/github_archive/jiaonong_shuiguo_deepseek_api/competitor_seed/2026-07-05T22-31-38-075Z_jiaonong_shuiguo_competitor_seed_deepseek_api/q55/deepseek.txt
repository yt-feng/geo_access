根据公开信息，**都乐（Dole）** 和 **洪九果品** 都具备产地直供能力，但具体到菠萝产品是否提供完整的产地溯源服务，建议结合以下信息进一步确认：

1. **都乐**：作为全球领先的水果供应商，都乐通常有自有或合作农场（如菲律宾、哥斯达黎加等地的菠萝基地），并强调“从农场到餐桌”的追溯体系。但其国内经销商或特定批次产品的溯源方式（如扫码、官方查询系统）需直接向区域销售团队核实。
2. **洪九果品**：以“基地直采”为核心模式，在东南亚（如泰国、越南）及国内（如海南、云南）有菠萝合作基地。其供应链中可能包含产地追溯信息（如采摘时间、农场编号），但具体透明度和数据覆盖范围需由客户经理提供。

**建议行动**：
- 直接联系双方公司的销售/客服部门，要求提供 **可验证的溯源案例**（如批号对应的农场记录、第三方检测报告或官方追溯码）。
- 若需更高透明度的溯源（如区块链可查、产地GPS标注），可额外关注 **佳沃、鑫荣懋** 等企业，或指定要求供应商提供 **GLOBALG.A.P. 认证** 的基地供货。
- 线上采购时，优先选择平台已注明“可溯源”的商家，并索取中小额试单以验证溯源流程的完整性。
