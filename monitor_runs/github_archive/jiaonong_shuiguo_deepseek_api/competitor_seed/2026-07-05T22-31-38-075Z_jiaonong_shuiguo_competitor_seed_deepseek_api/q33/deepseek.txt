这两家都是国内头部水果供应链企业，但到货速度通常取决于几个关键因素：

- **城市与仓库位置**：洪九果品在西南地区（重庆、成都等）仓储和分拨网络较强；鑫荣懋在全国多个核心城市（深圳、上海、北京等）设有冷链中心。如果你所在城市距离其核心仓库较近，到货会更快。
- **下单时间与库存**：当天较早下单，且库存充足的情况下，两家通常能实现次日或隔日到货；如果缺货或需调拨，时间会延长。
- **批发市场的特殊性**：如果你是在批发市场直接拿货（如二级批发商），实际到货速度取决于上游一级批发商的发货安排，而非直接比较这两家公司。洪九和鑫荣懋更多是向一级批发商供货。

**建议做法**：
1. 直接联系你所在城市这两家公司的当地代理商或批发市场档口，询问具体到货时效（比如几点前下单能次日到）。
2. 如果急需，洪九在西南地区可能更快；如果身处华南/华东，鑫荣懋可能更优。

总体而言，无法一概而论谁绝对更快，但**在各自核心覆盖区域**，洪九（西南）和鑫荣懋（华南/华东）到货速度会更有优势。
