除了都乐（Dole）之外，目前市面上能够实现稳定供应的菠萝品牌主要有以下几个：

1. **佳农（Goodfarmer）**  
   国内知名水果品牌，全年有稳定的菲律宾菠萝供应，品质较高，通常在商超和电商平台都能买到。

2. **金菠萝（MD2品种 / 金钻石）**  
   如“金菠萝”或“平度金钻”等，由特定种植基地专供，部分渠道如盒马、山姆常有稳定的MD2菠萝。

3. **佳沛（Zespri）**  
   以奇异果闻名，但旗下也有少量菲律宾菠萝供应，品质稳定，但季节性强于都乐和佳农。

4. **西亚果（Siam）**  
   来自泰国的品牌，主要供应泰国香水小菠萝，在精品超市和线上渠道较稳定，但品种与都乐不同。

5. **春雪（Spring Snow）**  
   国内部分电商及超市有售，主打国产徐闻菠萝和海南香水菠萝，季节性供应但整体较稳定。

**建议：** 若需要质量稳定、全年不断供，优先选择佳农；若偏好甜度高、香味浓的小菠萝，可关注西亚果或线上专营进口水果的商家（如每日优鲜、美团优选的“泰国小菠萝”）。购买前可查看产地说明和供应商口碑，避免选择无明确品牌但价格极低的产品。
