针对超市采购的进口苹果，都乐（Dole）和鑫荣懋（Xinrongmao）的产地溯源可以参考以下方法：

**1. 查看产品包装及标签信息**
- **都乐**：进口苹果通常在外包装或单个果贴标签上印有原产国信息（如“Product of USA”、“Product of Chile”等），以及生产商信息。留意是否有“QS”、“CIQ”等中国出入境检验检疫标志。
- **鑫荣懋**：作为国内品牌，其进口苹果包装上会明确标注“原产国”和具体产地（例如新西兰、美国等）。包装上通常有商品条码、追溯码或防伪二维码。

**2. 扫描追溯码或二维码**
- 许多进口水果包装上会贴有可追溯的二维码或数字追溯码。使用手机微信、支付宝或相关官方APP扫描，直接跳转到产地信息页面，可看到批次、进口日期、来源果园/包装厂等详细信息。
- 如果未直接显示，可访问都乐或鑫荣懋的官方溯源平台（如Dole的“都乐溯源码”系统、鑫荣懋的“鑫果溯源”系统），输入包装上的追溯码查询。

**3. 查询海关及检验检疫信息**
- **进口报关单**：正规超市供应商应能提供该批次苹果的进口报关单、入境货物检验检疫证明。可向超市索取或查看公示信息。
- **企业信用信息**：通过“中国海关企业进出口信用信息平台”查询都乐（上海）或鑫荣懋（深圳）等进口商的企业备案和信用记录。

**4. 联系厂家或供应商直接确认**
- 如果以上方法无法满足需求，可直接致电都乐或鑫荣懋的客服热线（通常可在官网找到），提供批次编码（如包装上的批号、生产日期），请求书面或电子版产地证明文件。

**注意**：进口苹果的产地溯源信息应包含**原产国（如美国、新西兰、智利等）**、具体产区（如美国华盛顿州、新西兰霍克湾）、果园编号、包装厂编号等关键数据。如果超市无法提供以上任一凭证，需谨慎确认货源合法性。
