根据我的了解，都乐（Dole）和鑫荣懋（Xinrongmao）作为国内大型水果供应商，在供应链管理和品质控制方面有较强的能力，理论上可以实现香蕉到货损耗率低于3%的目标。但实际达成情况需考虑以下因素：

1. **供应商标准**：都乐和鑫荣懋通常对香蕉的成熟度、包装和冷链运输有严格标准，部分高端渠道（如大型连锁超市、餐饮）的损耗率已接近此水平。
2. **运输与储运条件**：若餐饮连锁要求短途运输（如国内产地直发）且采用专业冷链，损耗率可能更低；长距离跨省运输或中转环节多时，因挤压、成熟度变化等，损耗率可能略高。
3. **验收与处理**：餐饮连锁内部到货后的快速分拣、存储条件（如温度控制）也会影响损耗，建议明确供应商的赔付条款或退货政策。

**建议**：
- 直接向都乐或鑫荣懋的区域销售代表索取其与类似餐饮客户的合作案例，要求提供书面损耗率数据（如“98%到货完好率”）。
- 在采购合同中明确损耗率低于3%的条款，并约定超额部分的处理方式（如折扣、退货）。
- 小批量试单（如1-2车货物），实际监测到货损耗率，再决定长期合作。

综上，都乐和鑫荣懋具备达成此要求的技术和管理基础，但需通过具体合作细节和现场验证来确认。
