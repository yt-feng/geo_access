洪九果品和鑫荣懋都是国内大型水果批发商，但它们的到货速度受具体区域、仓库位置、物流网络和订单情况影响较大。一般来说：

- **洪九果品**：以“产地直采+批发市场”模式为主，在部分核心城市（如重庆、广州、上海等）有自己的分拣中心和冷链物流，如果当地仓库有现货，通常可以较快安排发货（1-2天内到货）。
- **鑫荣懋**：作为头部供应链企业，渠道覆盖更广，但到货速度依赖于你合作的批发市场档口或分仓情况，如果是常规订单，一般2-3天左右。

**给你可执行的建议**：
1. **直接问具体档口或区域经理**：不同城市的仓储和配送时效差异很大。电话询问“现在下单，最快哪天能到货”最准。
2. **对比当天报价和库存**：到货快的前提是对方有现货。如果某家刚好有货且在你所在城市有分仓，那速度会更快。
3. **优先选择本地或临近城市的分仓发货**：例如你在广东，鑫荣懋的总部在深圳，可能到货更快；在西南地区，洪九果品优势更大。

**最终判断**：没有“绝对谁更快”的说法，实际取决于你所在位置和对方库存情况。建议你同时联系两家，明确“最迟到货时间”后再决定。
