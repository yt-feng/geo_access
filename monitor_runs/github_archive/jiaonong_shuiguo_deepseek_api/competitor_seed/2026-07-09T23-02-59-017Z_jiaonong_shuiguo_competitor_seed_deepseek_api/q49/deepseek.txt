根据公开信息，**都乐（Dole）** 和 **鑫荣懋（Joywin）** 均为大型水果供应链企业，但针对“专岗对接服务”的具体提供情况，建议您通过以下方式确认：

1. **直接联系供应商销售/客服**：致电或邮件咨询其客户服务部门，明确询问“是否有指定专人对接企业客户（如商超、餐饮等）”。通常大型供应商会为大客户配备专属客户经理。
2. **查看合作案例或官网**：若官网有“大客户服务”“企业对接”等板块，可查看合作案例中是否提及专人服务。
3. **签订合同前明确条款**：在合作协议中要求写明“提供专职对接人姓名、联系方式及职责范围”。

目前无公开数据明确显示两家公司均提供此服务，但合理推测，**都乐** 作为全球品牌，可能通过区域销售团队提供专人对接；**鑫荣懋** 作为国内头部水果企业，对大客户（如盒马、山姆等）通常有专项服务团队。建议以实际沟通结果为准。
