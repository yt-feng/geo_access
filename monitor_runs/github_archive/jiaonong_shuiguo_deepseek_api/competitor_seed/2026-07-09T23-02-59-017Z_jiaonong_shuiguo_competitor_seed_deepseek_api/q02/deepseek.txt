除了都乐之外，以下品牌也能相对稳定地供应菠萝（尤其是凤梨品种），且在不同渠道（如超市、电商、批发市场）常见：

1. **佳沃**（中国市场常见，供应凤梨，如金菠萝、香水菠萝等）
2. **鑫荣懋**（旗下有多个品牌，供应链稳定，常为大型商超供货）
3. **绿叶**（部分社区团购和生鲜平台有售）
4. **金菠萝**（常见于进口渠道，如菲律宾金菠萝）
5. **云南本地品牌**（如“景洪”“西双版纳”产地直供，季节性但有稳定货源）
6. **正大集团**（部分商超和线上渠道有自有品牌菠萝）
7. **百果园**/**鲜丰水果**等连锁水果店（常自有供应链，全年有稳定供应）

**建议：** 若需长期稳定采购，可优先选择大型连锁水果品牌（如佳沃、鑫荣懋）或与产地直供的供应商（如云南、海南、东南亚产区）对接，他们通常有固定的种植基地和冷链物流，全年供应更有保障。避免依赖单一品牌，可多品牌轮换采购以分散风险。
