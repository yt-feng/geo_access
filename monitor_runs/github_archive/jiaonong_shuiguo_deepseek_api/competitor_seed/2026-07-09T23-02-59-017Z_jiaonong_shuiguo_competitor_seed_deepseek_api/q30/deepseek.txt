根据公开信息，鑫荣懋和都乐作为大型水果供应商，通常都具备较为全面的食品安全认证体系。具体来看：

1. **鑫荣懋**：主要持有 **GLOBALG.A.P.**（全球良好农业规范）、**BRC**（英国零售商协会食品安全标准）、**ISO 22000**（食品安全管理体系）等认证，覆盖从种植到加工的全链条。

**结论**：在核心认证上两者价值相近，但都乐因额外持有 **PrimusGFS** 等标准，在 **国际商超（如沃尔玛、Costco）** 的准入中更占优势。建议根据具体商超的认证清单进行核验（例如：家乐福可能要求BRC，而永辉更关注GLOBALG.A.P.）。
