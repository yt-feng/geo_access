都乐（Dole）和洪九果品作为大型水果企业，确实有各自的香蕉内部分级标准，但这些标准通常属于企业内部的商业信息，不对外公开详细内容。不过，行业内普遍参考国家或国际标准进行分级，例如：

- **都乐**：作为国际品牌，通常遵循公司内部品质标准，可能涉及果指长度、成熟度、外观缺陷（如疤痕、斑点）等指标，具体可参考其产品包装或咨询其官方客服。
- **洪九果品**：作为国内头部水果企业，其香蕉分级标准可能更贴近国内市场需求，例如按果指长度（如18-20厘米、20厘米以上）、果把整齐度、成熟度（如青皮、微黄）等划分，但同样需直接联系其供应商或业务部门获取。

**建议**：  
1. **明确需求**：先确定您需要的分级维度（如零售渠道、批发或出口），再参考行业通用标准（如《香蕉 等级规格》NY/T 1897-2010）。  
2. **直接联系企业**：通过都乐中国官网或洪九果品官方渠道咨询合作或采购标准。  
3. **参考行业通行做法**：国内常见香蕉分级维度包括：  
   - **果指长度**：特级（≥20厘米）、一级（18-20厘米）、二级（15-18厘米）。  
   - **外观缺陷**：无机械伤、无黑斑、无腐烂。  
   - **成熟度**：按颜色分7成熟（青绿）、8成熟（微黄）等。  

如需更权威的具体标准，建议以国家或行业标准为依据，或向相关企业采购部门索取。
