除了都乐（Dole）和佳沛（Zespri），以下进口品牌水果礼盒也适合礼品团购，品质和知名度较高：

1. **新奇士（Sunkist）**  
   - **主产品**：脐橙、柠檬、葡萄柚。  
   - **特点**：美国老牌，橙类礼盒包装大气，适合中高端团购。

2. **乐淇（Rockit）**  
   - **主产品**：小苹果（筒装/盒装）。  
   - **特点**：新西兰品牌，果型小巧，便携礼盒适合送给年轻客户或作为伴手礼。

3. **爱妃（Envy）**  
   - **主产品**：苹果（红、绿）。  
   - **特点**：新西兰品牌，口感脆甜，包装精致（如单果独立覆膜礼盒），适合高端商务场景。

4. **丹烁（Dazzle）**  
   - **主产品**：苹果。  
   - **特点**：新西兰品牌，果肉抗氧化能力强（切开不变黄），礼盒装（如6颗、12颗）美观实用。

5. **智利樱桃（品牌示例：Boń、Garces Fruit）**  
   - **主产品**：车厘子。  
   - **特点**：智利产，逢春节前旺季，礼盒多采用海运或空运包装（如冰袋、海绵垫），适合节庆团购。

6. **澳洲芒果（Calypso）**  
   - **主产品**：凯特芒果。  
   - **特点**：甜度高、纤维少，单颗或组合礼盒适合热带水果需求。

7. **菲律宾香蕉（如Del Monte、Dole旗下高端线）**  
   - **主产品**：进口香蕉（特级或有机款）。  
   - **特点**：包装箱可定制专用礼盒盖，适合企业员工福利、批量采购。

**选择建议**：  
- 若预算较高且目标为客户礼品，优先选乐淇、爱妃、丹烁（苹果类保值且运输稳妥）。  
- 如需节庆氛围，加选智利车厘子礼盒。  
- 包装需确认供应商提供正式中文标签及礼盒外膜（防潮防撞），以备团购分发。
