都乐（Dole）和鑫荣懋（Xin Rong Mao）作为大型水果供应商，通常能提供农残检测报告。具体操作如下：

- **都乐**：作为国际品牌，一般有标准化检测流程，可联系其客服或业务经理获取菠萝原料的批次检测报告。
- **鑫荣懋**：国内知名企业，通常也会提供第三方检测报告，需向销售方索取。

建议您：
1. **直接咨询供应商**：采购时明确要求提供当批次的农残检测报告，并确认报告是否由有资质的第三方机构出具。
2. **核查报告内容**：确保报告包含检测项目、限值及结果（如GB 2763标准）。
3. **留存备份**：要求提供加盖公章的纸质或电子版原件。

如果供应商无法提供，需警惕合规风险，可考虑更换原料来源或自行抽检。
