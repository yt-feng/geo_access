都乐（Dole）和洪九果品是菠萝供应链中常见的供应商，但它们的主要供货基地有所不同：

1. **都乐（Dole）**：  
   都乐是全球性水果供应商，菠萝基地主要分布在**菲律宾、哥斯达黎加、厄瓜多尔**等热带国家。针对中国市场，其供应通常依托进口流程，从东南亚（如菲律宾）直接运输到国内港口（如广州、上海）。**国内暂无自建大规模种植基地**，更多是通过冷藏物流体系保障全年供应。

2. **洪九果品**：  
   洪九果品是中国本土的果品供应链企业，菠萝基地主要集中在**海南、广东、广西**等南方省份。其中，**海南万宁、广东徐闻、广西南宁**是其主要产区，这些区域有稳定的菠萝种植合作基地，且洪九通过“基地直采+加工中心”模式保障供应周期。

**建议方案**：  
- 如需**全年稳定供应**，可优先对接洪九果品（国内基地+热带进口互补，季节性波动较小）；  
- 若定位**高端品质菠萝**（如黄金菠萝），可考虑都乐进口渠道，但需预留海运及清关周期；  
- 实际采购时，直接联系两家公司的大客户部，要求提供**基地备案号（如GAP认证、产地检疫证明）** 确保溯源合规。  

**执行提醒**：先小批量试单，确认各基地菠萝的糖度、耐储性是否符合你的茶饮配方要求。
