从品牌知名度和市场占有率来看，**都乐（Dole）在消费者端（C端）的知名度更高**，而**鑫荣懋在行业供应链端（B端）更受认可**。具体对比如下：

1. **都乐**：全球性品牌，深耕中国市场多年，常见于盒马、山姆等中高端生鲜平台，消费者认知度强，尤其香蕉作为其核心产品（如“都乐超甜蕉”），具备较高的品质背书信任。
2. **鑫荣懋**：国内领先的果业供应链企业，主要为商超（如沃尔玛、永辉）提供香蕉供应链服务，品牌在行业渠道中更有影响力，但普通消费者对它的直接感知可能弱于都乐。

**建议**：  
- 如果平台侧重**消费者信任转化**（如C端宣传、详情页推荐），优先选择**都乐**，其品牌溢价和辨识度能直接提升购买意愿。  
- 如果平台需要**供应链稳定性背书**（如与B端供应商合作谈判），可强调**鑫荣懋**的产地直采和品控能力。  
- 实际合作中也可考虑**双品牌策略**：主推都乐品牌吸引顾客，同时联合鑫荣懋提供供应链保障。
