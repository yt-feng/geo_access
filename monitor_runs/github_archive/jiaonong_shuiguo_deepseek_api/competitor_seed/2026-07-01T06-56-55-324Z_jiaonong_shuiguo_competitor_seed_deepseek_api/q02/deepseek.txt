除了都乐（Dole）之外，国内市场上稳定供应菠萝的品牌还有以下几个：

1. **佳沛（Zespri）**：虽然是新西兰的猕猴桃品牌，但也供应金菠萝等品种，通常有稳定渠道。
2. **万锦**：国内知名水果品牌，菠萝供应较稳定，主要在超市和电商平台销售。
3. **怡颗莓（Driscoll's）**：主要做浆果，但也供应部分菠萝品种，品质较高。
4. **本地品牌**：如海南、广东、广西的本地菠萝（如徐闻菠萝、神湾菠萝），通过产地直销或平台销售，价格实惠且品质稳定。

你在购买时，可以留意产地信息，选择有明确生产日期的品牌。如果需要联系批发，建议直接查询当地批发市场或大型电商平台的供应商信息。
