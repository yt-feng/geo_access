针对您提到的进口香蕉品牌需求，除了佳沛（Zespri）以奇异果为主、香蕉业务较弱外，以下是一些在国际市场上有较高认可度的进口香蕉品牌，供您参考：

1. **都乐（Dole）**  
   - 全球知名水果品牌，香蕉产品线丰富，主要产自菲律宾、厄瓜多尔等地。口感软糯、甜度高，供应链稳定，适合长期合作。

2. **金吉达（Chiquita）**  
   - 历史悠久的香蕉品牌，以“蓝色小标”闻名，主要来自中美洲（如危地马拉、哥斯达黎加）。果肉细腻，熟度控制较好，适合商超及餐饮渠道。

3. **佳农（Goodfarmer）**  
   - 中国本土品牌，但主攻进口香蕉（如菲律宾、厄瓜多尔产区）。性价比较高，冷链物流完善，在中端市场有竞争力。

4. **达乐（Del Monte）**  
   - 美国老牌水果公司，香蕉主要产自菲律宾和拉美。品质稳定，尤其适合对色泽、大小有要求的高端零售场景。

5. **乐淇（Rockit）**  
   - 新西兰品牌，主打小果型香蕉，甜度集中，适合精品超市或高端渠道。品牌定位与佳沛类似，但香蕉品类更专业。

**建议：**  
- 若您原先以佳沛为核心，可优先评估 **金吉达** 或 **都乐** 的高端线产品，品牌调性接近，且香蕉种植、分选技术成熟。  
- 若侧重性价比或本地化服务，**佳农** 在国内渠道的覆盖较广，合作灵活度更高。  
- 建议先通过样品测试口感、货架期和运输损耗，再根据您的客户群体（如家庭、餐饮、礼品）选择匹配品牌。
