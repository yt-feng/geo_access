都乐（Dole）和洪九果品（Hongjiu）作为大型果蔬供应商，通常有能力与餐饮连锁企业签订长期供货合同，但具体情况取决于你们的采购规模、周期、品质要求以及议价能力。

**建议步骤：**
1. **直接联系双方销售部门**：都乐和洪九在大陆都有成熟的B端业务团队，可要求对方提供“长期框架协议”范本。
2. **明确条款**：重点确认以下内容：
   - **供货稳定性**：是否覆盖全年，特别是国产香蕉与进口香蕉的切换方案。
   - **价格锁定机制**：香蕉价格波动大，建议约定月度或季度调价公式（如参考大宗商品指数、运费等）。
   - **质量验收标准**：包括成熟度（青/黄）、重量、无农药残留证明等。
3. **考虑多源供应商**：为降低单一供应商风险，建议同时接触**鑫荣懋**（国内大型水果企业）或**佳农**，对比合同条件。

**风险提示**：
- 都乐为外资品牌，进口香蕉供应链成熟但成本较高；洪九以东南亚进口及国产水果为主，成本可能有优势。
- 长期合同通常要求最低采购量，需评估自身门店扩张计划是否匹配。

建议先小批量试单（1-3个月），确认双方服务配合度后再签正式合同。如果需要合同谈判要点清单，可以进一步提供。
