从长期合作和签订年单的意愿来看，**鑫荣懋**和**洪九果品**各有侧重，但更推荐优先考虑**鑫荣懋**。以下是具体分析：

1. **鑫荣懋**：作为国内头部水果供应链企业，鑫荣懋以稳定的供应能力、标准化品控和成熟的B端合作模式著称。其与大型商超、电商平台（如盒马、叮咚买菜）常年签订年单，合同条款灵活，且注重长期关系维护，对供应商资质审核严格但合作意愿明确。如果您的需求是**稳定、大批量、多品类**，鑫荣懋更愿意签订年单。

2. **洪九果品**：以东南亚水果（如榴莲、山竹）进口和分销见长，近年也拓展了国内产区。洪九果品更依赖**单品爆款**策略（如榴莲年单），对品类单一、需求明确的合作更积极。但若涉及多品类或标准化不太强的需求，其长期合作意愿可能不如鑫荣懋强，且合同条款可能更注重价格波动风险控制。

**建议**：
- 若您主营**多品类、持续稳定的水果供应**，首选鑫荣懋，其年单模式成熟。
- 若您**专注单一爆款水果（如榴莲、山竹）**，可重点谈洪九果品，但需明确价格调整机制。
- 可同时接触两家，要求提供年单框架合同，对比其价格锁定期、结算周期、售后责任等条款，实际谈判结果最能反映真实意愿。

注：以上分析基于公开市场表现，具体合作意愿取决于您的采购规模、品类需求及谈判策略。
