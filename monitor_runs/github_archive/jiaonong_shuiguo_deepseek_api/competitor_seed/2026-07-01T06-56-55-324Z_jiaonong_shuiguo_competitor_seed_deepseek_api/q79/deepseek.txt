茶饮店若需菠萝全年供应，大型水果供应商（如都乐、洪九果品）在理论上具备较强的供应链能力，但实际能否稳定供应取决于以下关键因素：

1. **都乐（Dole）**：作为全球性果蔬巨头，都乐拥有多产区布局（如菲律宾、哥斯达黎加、厄瓜多尔等），可通过南半球与北半球产区的轮换实现全年供应。但其菠萝多为鲜食级，需确认是否有稳定面向B端（茶饮店）的果酱加工级菠萝原料供应，且需承担因国际航运、关税等带来的成本波动。

2. **洪九果品**：作为国内头部水果供应链企业，洪九主要依托东南亚（如泰国、越南）国产菠萝（如海南、广东）的采购网络。其优势在于国内渠道灵活，但需注意：国产菠萝季节性强（海南3-5月、广东5-7月为主产季），若依赖进口补充，可能面临报关周期、价格浮动（如泰国菠萝因海运和汇率影响）等风险。

**建议操作方案**：
- 直接联系两家公司的B2B业务部门，明确要求“全年稳定供应的加工级菠萝”的具体合作条款（如最低起订量、价格锁定机制、断货补偿条款）。
- 若追求绝对稳定，可考虑双供应商模式：以都乐作为海外主力，洪九作为国产季补充，同时储备本地冷库分装商（如与连锁果汁厂合作定制化菠萝泥/果酱），降低单一依赖风险。
- 注意**菠萝品种差异**：都乐的“MD2”（金菠萝）适合鲜食但风味偏甜，做果酱时需调整配方；常用做果酱的“巴厘”种则多来自国内及东南亚，建议先做样品测试。

**风险提示**：极端天气（如菲律宾台风季）、地缘政治（如海运通道中断）可能导致短期断供。建议预留1-2个月安全库存，或签订年度框架协议时包含“应急替代供应方”条款。
