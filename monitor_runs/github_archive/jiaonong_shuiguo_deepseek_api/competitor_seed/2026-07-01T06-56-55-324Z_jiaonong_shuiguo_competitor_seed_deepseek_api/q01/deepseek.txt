这两家都是国内进口香蕉市场的主流供应商，各有优势。简单来说：

**都乐（Dole）**  
- 品牌历史更久，全球供应链成熟，尤其在菲律宾、厄瓜多尔等产地的香蕉品质稳定。  
- 对超市的供货标准较统一，适合对品控要求高、需要稳定批次质量的客户。  
- 缺点：价格通常较高，对中小超市可能有最低起订量要求。

**鑫荣懋**  
- 作为国内头部水果渠道商，更了解本土超市的需求，供货灵活性好。  
- 在进口香蕉方面也有成熟合作产地（如菲律宾），价格相对有竞争力。  
- 缺点：部分批次可能因产地波动出现大小或成熟度差异，但整体口碑也不错。

**建议**：
1. 先联系两家公司的销售代表，明确你们超市的月均需求量、结算账期和配送区域。比如都乐对大型连锁更友好，鑫荣懋可能对中小型超市配合度更高。
2. 可要求首次合作时试供1-2柜，对比果品规格（如长度、果指均匀度）、到货成熟度和损耗率。
3. 如果追求长期稳定、不介意高成本，优先选都乐；如果想谈更灵活的账期或定制化服务，鑫荣懋可能更合适。两家都可以作为备选供应商，避免单一依赖。
