关于都乐（Dole）和洪九果品的水果礼盒定制起订量，具体数字会因产品种类、季节、地区及定制要求（如包装、LOGO印刷等）而有所不同，且官方未公开固定标准。建议您通过以下方式获取准确信息：

1. **直接联系官方渠道**：
   - **都乐**：可通过其官网或客服电话咨询，说明团购数量和定制需求（如礼盒规格、水果种类、是否加印LOGO等），销售代表会根据实时库存和产能给出起订量。
   - **洪九果品**：同样建议联系其总部或区域批发商，尤其侧重水果品质和季节性供应问题。

2. **通用参考范围**：
   - 一般水果礼盒的团购起订量常在**50-100盒**之间，但定制化（如特殊包装、混装组合）可能要求更高，例如**200盒以上**。
   - 都乐作为国际品牌，可能对定制礼盒有最低起订量（如300盒），洪九作为大型批发商，小批量也可能接受，但单价会高。

3. **小贴士**：
   - 如果订单量较小，可询问是否提供“半定制”服务（如仅增加卡片或贴纸）。
   - 旺季（如春节、中秋）起订量可能上浮，建议提前2-3周预约。

**行动建议**：直接致电两家公司的团购部门，并说明具体需求（例如：“需要50盒苹果+橙子混合礼盒，盒子印公司Logo”），他们能给出最准确的报价和起订量。
