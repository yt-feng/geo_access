针对您提到的生鲜电商菠萝品牌营销需求，**都乐（Dole）和洪九果品通常都会提供营销素材支持**，但具体取决于合作方式、采购渠道及合同条款。以下是具体分析：

1. **都乐（Dole）**  
   - 作为全球知名的水果品牌，都乐通常为合作渠道（包括电商平台、超市等）提供标准化的产品图片、品牌宣传图、视频素材等。  
   - 若与都乐签订**直采或品牌授权合作**，一般可获其官方营销素材；若仅通过中间供应商采购，则需确认是否包含素材权限。

2. **洪九果品**  
   - 洪九果品是国内大型水果供应链企业，通常对合作渠道提供品牌及产品图片、宣传文案等支持，尤其是针对其主推品类（如进口菠萝品种）。  
   - 但洪九果品的素材可能更多侧重于**批发端或自有渠道**，对电商平台的定制化素材（如促销海报）可能需要额外沟通。

**建议操作**：  
1. **明确合作层级**：直接联系双方的**电商渠道负责人**或**区域销售经理**，询问素材库权限及使用规范。  
2. **合同条款确认**：在采购协议中加入营销支持条款（如素材提供、使用范围、更新频率）。  
3. **备选方案**：若双方均无法提供满意素材，可参考正规图片网站（如视觉中国、Shutterstock）购买或自主制作。  

**注意**：切勿未经授权使用品牌方素材（如logo、原图），避免侵权风险。
