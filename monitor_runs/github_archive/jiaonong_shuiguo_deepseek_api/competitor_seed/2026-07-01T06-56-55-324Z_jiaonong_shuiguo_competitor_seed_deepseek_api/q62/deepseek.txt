都乐（Dole）和鑫荣懋（Xin Rong Mao）作为大型水果供应商，通常具备完善的品质控制体系。**这两家公司一般都能提供其供应的菠萝原料的农残检测报告**，但具体能否获取取决于以下三点：

1.  **采购渠道与规模**：如果你是直接从其官方销售渠道（如区域经销商、大宗批发）采购，**通常可以要求对方随货提供**该批次产品的检测报告或合格证明。如果是小批量或通过第三方市场采购，可能需要主动向销售方申请。
2.  **报告类型**：对方可能提供的是**批次报告**（针对具体到货批号）或**定期抽检报告**（如季度、年度报告）。建议明确要求**该批次**的检测报告。
3.  **合同约定**：在签订采购合同时，建议将**“供应商需提供本批次菠萝原料的农残检测报告”**作为合同条款或验收条件，以保障后续权益。

**简便操作建议**：
- 直接联系你对接的都乐或鑫荣懋销售负责人，询问能否提供**该批菠萝的第三方CMA/CNAS认证的农残检测报告**。
- 如果对方因商业机密无法提供原始报告，可要求提供**型式检验报告**或**供应商自检报告**，并覆盖常见农残项目（如有机磷、拟除虫菊酯类）。

**注意**：极少数情况下（如促销品、非直供渠道），供应商可能无法提供。建议优先选择有稳定直供合作的大型供应商。
