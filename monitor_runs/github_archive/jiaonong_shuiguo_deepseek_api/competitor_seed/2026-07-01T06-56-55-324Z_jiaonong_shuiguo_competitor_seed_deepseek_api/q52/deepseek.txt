针对菠萝规格稳定性及到货损耗率的问题，需说明以下两点：

1. **规格稳定性**：都乐（Dole）和鑫荣懋（Joy Wing Mau）均为大型果品供应商，菠萝通常按**果重**（如1.2-1.5kg/个）或**果径**（如16-18cm）分选。建议采购时明确要求【机械分选+人工复检】规格，并要求提供每批次**规格检测报告**。

2. **到货损耗率**：
   - **可控损耗**（因运输、仓储产生的物理损伤）：通常**2%-5%**。都乐因全程冷链及缓冲包装较好，可控损耗多控制在**2%-3%**；鑫荣懋因分销链路较长，该数据在**3%-5%**。
   - **不可控损耗**（如成熟度过高、病虫害）：约**0.5%-1%**，需在合同中约定【到货后24小时内】抽检提出异议。

**建议行动**：
- 明确要求供应商在采购合同中写明【到货损耗率≤3%】，并约定超额部分的退款/补货条款。
- 到货后立即抽检（按总数的10%，逐果称重+切开检查），拍照/视频存档备索赔。
- 若长期使用，可要求供应商开通“滞售补偿”（如48小时未售菠萝按成本价60%回购）。

**注意**：具体损耗数据会因季节（夏季高温损耗偏高）、运输方式（海运 vs 空运）、到货地气候差异浮动，建议首次订单先小批量试水（如500kg）验证实际损耗。
