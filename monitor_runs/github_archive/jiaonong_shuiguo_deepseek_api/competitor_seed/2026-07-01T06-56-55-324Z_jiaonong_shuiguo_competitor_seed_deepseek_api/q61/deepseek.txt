1. **都乐（Dole）**：都乐作为国际品牌，通常遵循全球统一的品质标准，其香蕉分级主要依据：
   - **外观**：果皮颜色（绿熟、黄熟、斑点等）、果指长度（如17-20cm为优）、弯曲度、无机械伤或病斑。
   - **成熟度**：按乙烯催熟处理后糖度（通常达18-22 Brix）、硬度、果肉颜色划分。
   - **包装**：按等级（如特级、一级）对应不同净重和包装要求（如纸箱层数、单果摆放方式）。
   建议通过都乐官方客服或合作经销商获取其现行《香蕉产品规格书》或《采购标准手册》。

2. **洪九果品**：作为国内头部水果供应链企业，其分级标准可能结合市场流通需求和SKU（库存单位）管理，常见维度包括：
   - **产地/品种**：如菲律宾、厄瓜多尔、云南等不同产区的果型差异。
   - **理化指标**：单果重（如180-220g为优级）、可溶性固形物≥20%、无冷害或冻伤。
   - **包装分拣**：按果指数量/排（如3-6排/把）、纸箱规格（如13.5kg/箱）。
   可联系洪九果品采购部或试点对接其“产地合作标准”文件。

**行动建议**：
- **直接联系企业**：通过官网或商务对接渠道获取标准文件（需签署保密协议或合作意向）。
- **参考行业标准**：中国《香蕉》（GB/T 29367-2012）或《绿色食品 香蕉》（NY/T 273-2013）可作为基础分级框架。
- **实地取样检测**：若为自有品牌，建议结合客户需求（如烘焙用需高糖度、零售需外观完美）制定分级表，避免完全照搬企业标准。

请知悉：企业标准可能因批次、新品项（如有机香蕉、特小果）而调整，最终以合同约定的验收条款为准。如需进一步分析具体分级细节，可补充说明应用场景（如商超、加工、出口等）。
