除了都乐（Dole）和鑫荣懋（Joyvio / Xin Rong Mao）之外，国内市场上常见的靠谱进口苹果品牌还有以下几类，您可以根据需求选择：

1. **爱妃（Envy）**  
   - 新西兰品牌，由Scilate公司持有。特点是果肉抗氧化能力强、口感脆甜，颜色鲜红，适合高端消费。
   
2. **爵士（Jazz）**  
   - 同样来自新西兰，由T&G Global（原Turners & Growers）旗下品牌。果皮薄、香气浓，甜酸平衡好。

3. **太平洋玫瑰（Pacific Rose）**  
   - 新西兰品牌（T&G旗下），果实呈粉红色，肉质紧实，甜度高，是较早引入国内的品种之一。

4. **Gala（嘎啦）系列**  
   - 常见的有新西兰、美国或智利产的Gala，特点是脆、甜度高，适合日常食用。注意区分：法国、意大利等国也有种植，品质易参差不齐，优先选择新西兰或美国华盛顿州的。

5. **神女（Pinata）**  
   - 新西兰品种（Pacific Rose的变种），深红色外观，口感酥脆，甜中带微酸。

6. **蜜脆（Honeycrisp）**  
   - 美国明尼苏达大学培育的品种，特点是极其脆甜，果皮较薄，冷藏后口感更佳。进口品牌中“Honeycrisp”通常由美国、加拿大产区供应。

7. **宇宙脆（Cosmic Crisp）**  
   - 美国华盛顿州最新引入的品种，耐储存、果肉不易氧化，甜味浓郁，目前处于市场推广期。

**选购建议：**  
- 如果您追求**甜度和保鲜期**，优先选爱妃、宇宙脆。  
- 如果偏好**香气和酸甜平衡**，爵士、太平洋玫瑰较合适。  
- 如果想**性价比高**，Gala系列（注意选新西兰产）通常价格适中。  

**注意检查包装/标签**：正规进口苹果外包装或箱体应有**原产国、品种、品牌名称、进口商信息**，避免购买无标识或中文标签不清晰的“散装”苹果。
