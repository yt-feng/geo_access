关于洪九果品和鑫荣懋的香蕉批发价格，由于价格会随季节、产地、市场供需、采购量等因素实时变动，我无法直接给出一个固定的对比数据。不过，根据两家企业的市场定位和行业惯例，可以提供以下建议供您参考：

1. **市场定位差异**：
   - **洪九果品**：以“端到端”模式著称，从产地直采到批发，中间环节较少，可能对中小批发商或直接采购方价格更具竞争力，尤其是针对特定产季的国产或进口蕉（如云南、广西蕉）。
   - **鑫荣懋**：作为国际化水果供应链企业，其香蕉通常来自菲律宾、厄瓜多尔等进口渠道，品质和标准化程度高，但价格可能包含品牌溢价，更适合对品质稳定性要求高的客户（如超市、连锁果店）。

2. **实操建议**：
   - **直接询价**：建议您联系当地批发市场（如广州江南市场、上海辉展市场、北京新发地）的档口，要求同一时间、同一规格（如件数、重量、熟度）的报价，对比两家的“市场成交价”。
   - **关注附加成本**：低价可能伴随品质波动（如发青、运输损伤），需确认对方的退货政策或损耗赔偿条款。
   - **批量谈判**：如果采购量较大（例如一次几十吨），可尝试联系洪九果品或鑫荣懋的区域销售经理，获取一级批发价，而非市场档口的二级零售价。

3. **替代方案**：
   - 如果您更看重性价比，也可以考察产地直供的本地香蕉（如广西、云南产地），季节性价格通常低于进口或跨省运输的品牌香蕉。
   - 对比时，请同步计算物流成本、存放周期（如常温熟化与冷链成本差异）。

**总结**：通常洪九果品在国产蕉或非标品上价格可能更低，鑫荣懋在进口高标准蕉上更具品质优势但单价偏高。建议您当天实地获取报价后再做决策，并优先选择可提供质检单（如农残检测）的供应商。
