鑫荣懋和都乐都是大型水果供应商，通常都具备主流食品安全认证（如Global G.A.P.、BRC、ISO 22000、HACCP等）。由于认证覆盖范围、更新频率和具体品类差异，无法简单判断“谁更全”。

建议您采取以下步骤自行核实：
1. **直接索要认证清单**：联系两家公司的销售或品控部门，要求提供当前有效的认证证书复印件（注意查看有效期和覆盖基地/产品范围）。
2. **对比认证类型**：关键认证包括：
   - **基础级**：ISO 22000、HACCP（食品安全管理体系）。
   - **产地级**：Global G.A.P.（全球良好农业规范）。
   - **出口级**：BRC、IFS、FSSC 22000等（部分国际商超要求）。
   - **有机/特殊认证**：若商超有要求，需同时对比有机证书、无公害认证等。
3. **关注覆盖范围**：例如，有Global G.A.P.认证但仅限某几个种植基地，或BRC仅针对某加工厂——需确认是否覆盖你采购的全部品类和产地。
4. **查询官网或第三方平台**：比如在“国家认证认可监督管理委员会”官网查询部分认证信息，或直接查看企业官网的“质量保障”“企业荣誉”页面。

**结论**：没有公开资料能证明哪家“更全”。最可靠的做法是基于商超对认证种类的具体要求（例如商超要求必须有Global G.A.P.且覆盖柑橘类），分别向两家索取对应品种和产区的认证文件，进行比对。通常大型商超会同时批准两家供应商，只要核心认证达标即可。
