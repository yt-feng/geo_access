都乐（Dole）和鑫荣懋（Golden Wing Mau）在菠萝产地直发的物流时效上，通常具备行业领先的冷链能力，但具体时效会受发货地、目的地城市及季节性因素影响。以下是一般性参考：

- **都乐（Dole）**：作为全球知名水果品牌，在中国通常通过自有或合作冷链网络进行产地直发。从海南、广东等主产区到一线城市（如北上广深），一般可实现 **24-48小时** 内送达。部分核心城市支持次日达，但偏远地区可能需要 3天左右。

- **鑫荣懋（Golden Wing Mau）**：国内头部水果供应链企业，在菠萝产地（如海南、广西）有直接采购和分选中心。其物流时效与都乐相近，核心城市通常 **24-48小时**，部分区域通过前置仓可实现更快交付（如华南地区可做到12-24小时）。

**关键影响因素**：
1. **发货模式**：若为“产地直发”而非从区域仓库发货，时效可能延长1天。
2. **城市等级**：一线城市通常优于二三线，乡镇地区可能额外增加1-2天。
3. **季节与天气**：菠萝旺季（3-6月）运力紧张，或遇台风等天气，可能推迟至3天以上。

**建议核实**：由于生鲜物流政策可能随时调整（如极端天气或节日期间），建议您直接通过都乐或鑫荣懋的官方客服或合作电商平台（如京东、盒马）查询具体到您城市的最新时效承诺。这些实时信息更可靠。
