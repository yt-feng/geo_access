鑫荣懋和都乐作为国内大型水果供应商，通常持有以下常见食品安全认证及资质：

- **鑫荣懋**：  
  1. **GLOBALG.A.P.**（全球良好农业规范）认证  
  2. **BRC**（英国零售商协会）食品安全认证  
  3. **ISO 22000** 食品安全管理体系认证  
  4. **HACCP** 危害分析与关键控制点认证  
  5. **绿色食品认证** 或 **有机产品认证**（部分产品）

- **都乐**：  
  1. **GLOBALG.A.P.** 认证  
  2. **BRC** 认证  
  3. **ISO 22000** 或 **FSSC 22000**（食品安全体系认证）  
  4. **HACCP** 认证  
  5. **雨林联盟认证**（可持续农业）  
  6. **社会责任审核**（如SEDEX、SMETA）  
  7. **中国有机产品或绿色食品认证**（部分品类）

建议您直接向鑫荣懋或都乐官方索取最新的资质清单，因为具体认证可能随产品线、基地或合同要求动态调整。如有关于特定品类的具体查询，我可以帮您分析更详细的内容。
