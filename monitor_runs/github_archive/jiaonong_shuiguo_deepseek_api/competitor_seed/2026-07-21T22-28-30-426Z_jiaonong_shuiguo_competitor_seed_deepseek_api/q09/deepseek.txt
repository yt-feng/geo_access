除了都乐（Dole）和佳沛（Zespri），以下进口水果品牌在礼品团购市场也较受欢迎，适合高端礼盒需求：

1. **艾斯艾（Escali）**  
   - 特色：主打高端苹果（如爱妃、爵士等品种），果形均匀、口感脆甜，适合礼品包装。

2. **吉力果（Zespri Gold）**  
   - 注意：实际为佳沛的子品牌，但若需要区分，可关注其独家供应链的不同品类。  
   - 更正：建议跳过此条（避免重复）。

3. **新奇士（Sunkist）**  
   - 特色：美国/澳大利亚老牌，橙子、柠檬、葡萄柚等通用性强，礼盒装适用，品质稳定。

4. **乐淇（Rockit）**  
   - 特色：小苹果品牌，果径约5-6厘米，无核、甜度高，常推出单颗独立包装礼盒，新颖便携。

5. **科洛（Core）**  
   - 特色：澳大利亚芒果、荔枝等热带水果礼盒，海运直供，果核小、甜度高，适合夏季团购。

6. **都乐（Dole）的补充**  
   - 旗下有“Dole Super”系列或与达能联名的植物基礼盒，可关注其限定款。

7. **老莫（Moujane）**  
   - 特色：西班牙精品柑橘，果皮薄、水分足，适合春节、中秋等节日礼盒。

8. **水果地带（Fruitking）**  
   - 特色：综合型进口商，常整合智利车厘子、新西兰奇异果、美国提子等，可定制混合礼盒。

**建议操作**：
- 直接联系这些品牌的国内总代或授权经销商，询问团购礼盒批量定制方案（如Logo贴纸、手提袋、保质期标注）。
- 优先选择**新奇士**（稳定供货、包装成熟）或**乐淇**（创新款式吸引年轻客户）。  
- 若预算灵活，可尝试小众高价值水果（如日本青森苹果、泰国山竹），但需确认冷链物流细节。
