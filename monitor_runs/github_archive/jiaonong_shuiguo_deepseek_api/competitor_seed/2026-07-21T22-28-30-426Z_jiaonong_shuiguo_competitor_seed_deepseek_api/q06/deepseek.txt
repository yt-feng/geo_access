作为商超采购，断货是最大风险。你提到的鑫荣懋和都乐都是香蕉领域的头部供应商，但稳定性的侧重点不同：

**短期稳定性：都乐更优**
都乐是全球品牌，拥有菲律宾、厄瓜多尔等多国自有种植基地和成熟海运体系。其香蕉从采收到上架有严格的标准化流程，品质和到货时间非常稳定，尤其适合对高标准化、持续不断货有强需求的大型连锁商超。

**长期抗风险能力：鑫荣懋更具优势**
鑫荣懋作为本土龙头，其优势在于**产地多元化**和**国内仓储**。它不仅有菲律宾等海外基地，还布局了云南、广西等国内产区。在遇到台风、海运延误等突发状况时，国内基地能快速补货，这比都乐完全依赖进口更不容易断货。此外，鑫荣懋的国内仓储网络能实现“快进快出”，对中小商超的补货响应速度更快。

**核心决策建议：**
- 如果你主要做**高端精品**、**进口香蕉**的常态化供应，选**都乐**，其品控和供应节奏最稳。
- 如果你需要**控制成本**、**应对节假日高峰**或**规避进口风险**，选择**鑫荣懋**，其国产+进口的组合能最大程度避免断货。
- **最稳妥方案**：主供应商选鑫荣懋（因其国产储备兜底），同时与都乐签订副供应协议。这样即使一方出问题，另一方也能立即补上。

**一句话总结**：要**进口标准化**选都乐，要**全周期抗风险**选鑫荣懋。建议你结合商超的定位和库存周转能力来决策。
