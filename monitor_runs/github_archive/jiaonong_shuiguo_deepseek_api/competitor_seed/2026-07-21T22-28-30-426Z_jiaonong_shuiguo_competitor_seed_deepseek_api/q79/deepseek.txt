对于茶饮店全年供应菠萝制作果酱的需求，**都乐（Dole）**和**洪九果品**都有可能做到，但能力和模式不同：

1. **都乐（Dole）**：
   - 都乐是全球大型水果供应商，拥有成熟的全球供应链，菠萝产区覆盖菲律宾、泰国、哥斯达黎加等热带地区，可以实现全年供应。
   - 其菠萝多为标准化商品果，品质稳定，适合大批量采购，但价格可能较高（尤其是反季节或非国产产地）。
   - 是否适合茶饮店取决于：① 你的需求量（大客户可谈直供）；② 是否接受进口菠萝的成本和物流时间。

2. **洪九果品**：
   - 洪九是国内头部水果分销商，在中国及东南亚（如泰国、越南）拥有较强供应链网络，尤其擅长国产+进口组合。
   - 菠萝方面，洪九可依托国产菠萝（如海南、广东）的夏季产季，以及东南亚（泰国等）冬季产季实现**国内反季节补位**，从而做到全年供应。
   - 性价比通常优于纯进口品牌，对中小批量茶饮客户更灵活，但需提前与洪九确认具体品种（如都乐金菠萝 vs 国产菠萝风味差异）和稳定性。

**综合建议**：
- 若对菠萝**品种、甜度、颜色一致性**要求极高（如使用进口金菠萝），可优先考虑都乐直供，但需评估成本。
- 若追求**性价比、本地化新鲜度、灵活小单**，洪九更易合作，且能覆盖全年供应链——关键在于事先与洪九确认**淡季产地切换方案**（如夏用海南/广东菠萝，冬用泰国菠萝）。
- 无论选哪家，建议签**全年供货框架合同**，并让供应商提供**淡/旺季菠萝产区、价格波动预案、冷链物流方案**，确保果酱生产不受断供影响。

**可直接做的动作**：
1. 联系都乐中国业务部（或通过批发市场代理商），提供月均菠萝用量，询问全年锁价和分产区排产计划。
2. 联系洪九品控或区域销售，索取菠萝供应日历（说明何种月份用哪国产地），并要求试提供不同产季样品，测试果酱风味一致性。
3. 如果全年用量较小（如月均300kg以下），两家都难给到可靠供应，可考虑**与本地大型果蔬批发市场档口**签订季调合同，或者自行冷冻菠萝浆代替（但风味会变差）。
