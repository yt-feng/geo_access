您好！关于都乐（Dole）和洪九果品（Hongjiu）的菠萝种植基地具体位置，以下是我基于公开信息整理的概述，但请注意，农业企业的基地分布可能随时间调整，建议您在做采购决策前，直接联系相关公司以获取最新、最准确的产地信息或采购对接。

1. **都乐（Dole）**
   - **主要基地**：都乐是全球性果蔬供应商，其菠萝种植基地主要集中在**菲律宾**（如布基农省、南哥打巴托省等）以及**哥斯达黎加**（全球最大的菠萝出口国之一，都乐在当地有大型农场）。在中国市场，都乐的菠萝通常以进口为主，可能通过上海、广州等港口的分拨中心供应。
   - **建议**：如需国内稳定供应，可咨询都乐中国销售团队是否有云南或海南的合作种植项目（部分进口商有时会支持国内补货）。

2. **洪九果品（Hongjiu）**
   - **主要基地**：洪九果品是中国领先的果品供应链企业，其菠萝采购来源包括**泰国（如巴蜀府、春武里府）** 和**菲律宾**。在国内，洪九在多省份有分拣加工中心（如重庆、广州等），但自有的种植基地信息较少，主要依赖全球产地直采。
   - **建议**：联系洪九的区域采购经理，明确其菠萝供应是否以进口为主，或是否有国内合作农场。

**关键建议**：
- 若您需要长期稳定供应，建议直接联系两家公司的采购部门，获取**基地认证证明**、**批次供应能力**及**季节性波动应对方案**。
- 同时可考察国内菠萝主产区（如广东徐闻、海南万宁）的合作社，作为补充。
- 所有供应商都应提供**农残检测报告**和**追溯码**，并签订长期供货协议中明确质量标准（如糖度、黑心率等）。

请务必以官方渠道（官网、客服或实地考察）核实基地信息，避免依赖非公开的第三方资料。祝您采购顺利！
