都乐（Dole）和洪九果品（Hongjiu）作为大型水果供应商，通常主要提供整箱或批量包装。针对社区团购需求的“散装选项”（如小包装、按个或按斤拆分），这两家公司的直供渠道通常不直接提供此类产品，而是通过经销商或下游批发市场进行二次分装。

### 建议操作步骤：
1. **直接联系官方销售**：  
   - 都乐：联系其企业团购或批发部门，询问是否接受“小包装定制”要求（通常起订量较高，如50-100箱）。  
   - 洪九果品：类似，可咨询其区域代理或销售经理，确认是否有小包装分装服务。  
2. **寻找二级分销商**：  
   - 从当地水果批发市场、社区团购供应商平台（如快驴、美菜）或1688等B2B平台搜索“都乐/洪九香蕉 散装”，可能有中间商提供预分装服务。  
3. **替代方案**：  
   - 直接采购整箱香蕉后自行分装（需注意保存和损耗问题）。  
   - 寻找本地中小品牌或特色香蕉（如海南、云南产区），这类供应商常可提供小包装定制服务。

### 注意事项：
- 散装通常意味着包装、分拣成本上升，需确认是否接受价格上浮。  
- 建议先少量试购（如一箱），确认品质和物流稳定性后再扩量。  

如果急需明确答案，最快的方式是直接致电两家公司客服或区域经销商（官网可查联系方式），避免通过第三方渠道延误信息。
