根据公开信息，都乐（Dole）和洪九果品均制定了针对香蕉的分级企业标准，但具体细节未完全公开。以下是可获得的参考信息：

1. **都乐（Dole）**：作为国际品牌，都乐通常遵循内部严格的分级体系，主要依据香蕉的**长度、粗细、成熟度（颜色卡）、果面瑕疵（如疤痕、黑点比例）** 等指标。例如，一级品通常要求果形均匀、无机械损伤、成熟度在3-5级（按颜色卡判断）。建议直接联系都乐中国客服或参考其官网的产品说明获取具体标准。

2. **洪九果品**：国内头部水果供应链企业，其香蕉分级多参考**行业通用标准**（如《香蕉》GB/T 27635-2011国家标准），并结合自有渠道需求调整。常见分级维度包括：**单果重（一般分为120-200g等区间）、果梳整齐度、色泽（如青黄比例）、农药残留检测**等。洪九果品官网未公开详细分级文档，但可以通过其批发市场（如北京新发地、嘉兴水果市场）的采购手册或客户经理了解。

**建议**：对于连锁店的实际操作，可直接：
- 联系都乐或洪九果品的销售代表，索要其**内部品控手册**或**采购规格书**（属于商业机密，但合作客户通常可获取）。
- 参考国家标准《香蕉》（GB/T 27635-2011）或行业标准《香蕉等级规格》（NY/T 1792-2009），这些公开标准提供了通用分级框架（如将香蕉分为特级、一级、二级，按果指长度、弯曲度、缺陷面积等划分）。
