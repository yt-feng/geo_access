除了都乐（Dole）和鑫荣懋（Joywing），以下是一些在国内外市场较为知名且口碑较好的进口苹果品牌，可供对比参考：

1. **美国华盛顿苹果（Washington Apples）**  
   - 美国华盛顿州官方品牌，涵盖嘎啦、富士、红蛇果等多个品种。  
   - 品质稳定，出口全球，国内大型商超常见。

2. **新西兰爱妃苹果（Envy）**  
   - 以高甜度、耐储存著称，果肉不易氧化，适合鲜食和烘焙。  
   - 由新西兰Bostock集团等主要出口商运营。

3. **日本青森苹果（Aomori Apples）**  
   - 品种如“富士”、“王林”等，主打高品质和精致包装。  
   - 通过日本农产品出口渠道进入中国，价格较高。

4. **澳大利亚粉红女士（Pink Lady）**  
   - 澳大利亚Brambles集团（现Costa集团）品牌，酸糖平衡，表皮有独特粉色条纹。  
   - 出口至全球，在中国高端市场有稳定供应。

5. **法国嘎啦苹果（Gala）**  
   - 法国多家合作社（如Agrial、Socofin）运营，口感脆甜，适合即食。  
   - 常见于欧洲和中国进口超市。

6. **智利青苹果（Granny Smith）**  
   - 智利主要出口商（如Subsole、Unifrutti）供应，酸度高，常用于烘焙或沙拉。  
   - 与北半球反季节互补，中国市场占有率高。

7. **南非苹果（如Fuji、Royal Gala）**  
   - 南非核心出口商（如Capespan、Pilgrim Bokomo）提供，品质不输智利，价格相对略低。

**选购建议**：  
- 优先选择有贴标、可追溯产地的品牌（如产品包装上的品牌和产地信息）。  
- 部分高端品牌（如日本青森）可能需通过精品超市或电商进口渠道购买。  
- 可以关注水果批发市场或大型连锁超市（如盒马、山姆、Costco）的定期采购清单。

如需具体品种对比（口感、价格区间），可以进一步提供细节需求。
