大型商超（如沃尔玛、山姆、盒马等）通常要求供应商具备国际或国内认可的食品安全管理体系认证。鑫荣懋（中国知名水果供应链企业）和都乐（Dole，全球果蔬品牌）常见的认证包括：

1. **FSSC 22000（食品安全管理体系认证）**  
   - 适用于食品制造商和供应链，是GFSI（全球食品安全倡议）认可的认证。

2. **BRCGS（原BRC，食品安全全球标准）**  
   - 都乐作为国际品牌，通常持有BRC认证，用于满足零售商（尤其是欧洲和亚太地区）的审核要求。

3. **IFS Food（国际食品标准）**  
   - 类似BRC，属于GFSI认可的认证，都乐和鑫荣懋可能持有。

4. **ISO 22000（食品安全管理体系）**  
   - 两者都可能获得此基础认证，结合HACCP管理。

5. **GLOBALG.A.P.（全球良好农业规范）**  
   - 针对农场端的生产标准，都乐在产区通常持有该认证，鑫荣懋也可能应用于其合作基地。

6. **中国国内认证（如SC、HACCP、有机认证）**  
   - 在中国大陆运营，两者均需满足《食品安全法》要求，包括生产许可证（SC）和可能的HACCP认证。

**注意**：具体认证清单需以企业官网或官方文件为准（例如供应商资质审核结果）。对于直接采购需求，建议联系其客户服务部门或查询商超的“供应商准入名单”。以上信息为行业常见实践，但并非实时认证状态。
