通常情况下，大型水果供应商如**都乐（Dole）**和**鑫荣懋**（主要做进口和国产高端水果供应）都能提供其产品的农残检测报告。具体能否提供取决于以下因素：

1. **都乐（Dole）**：作为全球知名的果蔬品牌，其菠萝产品（如金菠萝）通常有标准化的检测流程。如果是通过正规渠道（如官方经销商、大型批发商、直接采购）购买，一般可以提供符合国家标准的第三方检测报告（如SGS或CTI等机构出具）。建议你在采购前直接向都乐的国内销售团队或经销商索取。

2. **鑫荣懋**：作为国内大型水果供应链企业，其供应的菠萝（包括进口和国产品种）通常也具备必要的检测报告。同样，需要联系其销售负责人或采购部门确认。如果是针对特定批次，可能需要协商后提供。

**行动建议：**
- 在采购前，明确告知供应商你需要**农残检测报告（包含具体检测项目和标准，如GB2763-2021）**，并询问是否随货提供或可提前提供。
- 如果对方无法提供，可要求提供**第三方检测机构的报告**（如中国检验认证集团、SGS、华测检测等），或协商自行送检的费用分摊。

**注意：** 如果是从水果批发市场、小规模经销商处购买，可能无法直接提供报告，建议优先选择上述正规渠道。另外，可要求供应商在合同或订单中注明“需提供农残检测报告”。
