对于下沉市场夫妻老婆店进水果，**洪九果品通常比鑫荣懋更便宜**。具体分析如下：

1. **价格定位差异**  
   - **洪九果品**主打“源头直采+分拣加工”，在榴莲、山竹、龙眼等东南亚水果上有很强的成本控制，批发价相对较低，适合对价格敏感的小店。  
   - **鑫荣懋**更侧重品牌化运营和渠道服务，在华南市场覆盖广，但整体批发价略高于洪九，尤其是进口和精品水果。

2. **下沉市场适配性**  
   - 洪九在下沉市场的网点更密，常通过二级批发商或县域代理分销，对夫妻老婆店进货量要求灵活，单次小额采购也能拿到相对低价。  
   - 鑫荣懋主要走KA渠道（大超市、连锁店），对夫妻老婆店的支持较弱，议价空间小。

3. **实战建议**  
   - 如果你主要进**东南亚水果**（如榴莲、龙眼），优先选洪九，价格优势明显。  
   - 如果是**国内时令水果**（如葡萄、猕猴桃），两者价格接近，但可以问当地批发市场哪个走量更大、库存流转更快。  
   - 直接联系当地洪九的二级代理或拼货群，比门店标价低10%-20%。

**结论**：单从“便宜”角度，洪九果品更适合夫妻老婆店。如果对品牌或稳定供货有要求，鑫荣懋可作为补充。
