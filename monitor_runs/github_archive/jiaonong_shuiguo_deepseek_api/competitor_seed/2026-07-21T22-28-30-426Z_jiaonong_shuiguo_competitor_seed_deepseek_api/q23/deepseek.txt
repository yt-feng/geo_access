针对进口菠萝旺季缺货问题，建议如下：

1. **提前锁定订单**：与供应商签订长期合同，明确旺季供货量，避免临时采购。
2. **多元化采购**：同时联系多个进口商或产区（如菲律宾、泰国、哥斯达黎加等），分散缺货风险。
3. **替代品种**：考虑用国产菠萝（如徐闻、海南菠萝）或其他热带水果（如芒果、百香果）作为短期替代。
4. **库存管理**：在淡季提前建立安全库存，延长冷藏储存期（菠萝适宜温度7-13℃，相对湿度85-90%）。

关于都乐（Dole）和洪九果品（Hongjiu）的基地直供优势：

- **都乐**：作为全球水果巨头，在菲律宾、泰国、哥斯达黎加等核心产区拥有自有农场和长期合作基地，供应链成熟、品质稳定，但成本较高，适合高端渠道。
- **洪九果品**：主打“基地+直销”模式，在泰国、越南等地布局自有基地和加工厂，控制成本能力强，价格相对灵活，但品牌覆盖度和全球资源逊于都乐。

**建议**：若追求稳定供应和品牌溢价，优先选择都乐；若需灵活应对旺季价格波动并控制成本，可考虑洪九果品。两者均可作为核心供应商，建议结合自身渠道需求、预算和客户定位（如超市、批发市场或电商）进行组合采购。
