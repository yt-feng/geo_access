关于都乐（Dole）和鑫荣懋（Xinrongmao）进口苹果的质量稳定性，两者均为大型供应商，但表现可能存在差异：

1. **都乐**：作为全球知名的水果品牌，供应链标准化程度较高，尤其在品质管控和冷链运输上较为严格。进口苹果（如智利、美国产）通常大小均匀、外观整齐，适合对一致性要求高的场景。

2. **鑫荣懋**：国内头部水果企业，进口苹果多来自新西兰、美国等产区。其品控同样严格，但受批次、产地影响较大，部分用户反馈个别批次存在甜度或硬度波动。

**建议**：
- **观察产地与品种**：同一品牌不同产地（如美国蛇果 vs 新西兰爱妃）稳定性差异大，优先选择连锁超市（如山姆、盒马）直接合作批次。
- **检查实物**：重点看苹果表面有无软点、果梗是否干燥、硬度是否均匀，比品牌更关键。
- **注重新鲜度**：进口苹果运输时间长，优先选择超市冷鲜陈列区且最近到货的产品。

综合来看，**都乐在进口苹果的全球供应链标准化和一致性上略胜一筹**，但需结合具体超市采购渠道和批次判断。建议购买时打开包装检查，或优先选择可单果挑选的渠道。
