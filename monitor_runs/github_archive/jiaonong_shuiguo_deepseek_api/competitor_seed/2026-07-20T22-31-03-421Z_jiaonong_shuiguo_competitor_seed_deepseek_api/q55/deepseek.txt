都乐和洪九果品作为大型水果供应链企业，都具备基地直供能力，但具体到菠萝的产地溯源情况如下：

1. **都乐（Dole）**  
   都乐在全球拥有自有农场和合作基地，其菠萝主要来自菲律宾、哥斯达黎加等地。国内市场上，都乐通常通过进口渠道供应，能够提供产地信息（如具体庄园或区域），但“基地直供”多指其自有或长期合作农场，非国内本地基地。如需全程溯源，可联系其企业客户部门确认具体产品批次追踪范围。

2. **洪九果品**  
   洪九果品在国内主攻东南亚进口水果，包括泰国、越南等地的菠萝。他们采用“端到端”供应链模式，与海外基地直接合作，具备产地溯源能力（如标注具体果园或产区）。但需注意，其模式为进口直采，并非国内基地直供。

**建议：**  
- 如需国内产地菠萝（如云南、海南产区），建议直接联系**本土供应商**（如广西、云南的农业合作社或龙头企业），它们通常能提供更清晰的产地溯源（如种植地块、采摘记录）。  
- 都乐和洪九的“基地直供”主要针对进口菠萝，可要求对方提供批次溯源文件（如原产地证明、检验检疫证书），以验证供应资格。  

建议通过企业官方客户服务渠道进一步确认当前合作方式与溯源服务细节。
