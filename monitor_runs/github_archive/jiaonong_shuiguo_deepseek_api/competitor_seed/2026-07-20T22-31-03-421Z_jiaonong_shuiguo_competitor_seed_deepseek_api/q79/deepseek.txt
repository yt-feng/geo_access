对于茶饮店需要菠萝全年供应的问题，**都乐（Dole）和洪九果品都有可能满足需求，但具体能否实现取决于你的采购量、品质要求和供应链协议**。

以下是分析：

1. **都乐**：作为全球大型果蔬供应商，都乐在南美、东南亚等地区有稳定的菠萝种植基地，能够通过南北半球产地轮换（如菲律宾、哥斯达黎加等）实现全年供应。但建议你与都乐确认是否提供“工业用菠萝”（如切块、冻果等加工形式），及是否有针对茶饮店的小批量或定制化服务。部分包销或定制协议可能需要一定起订量（如整柜冷链运输）。

2. **洪九果品**：作为国内头部水果供应链企业，洪九在泰国、越南等东南亚产区有布局，具备全年供应热带水果的能力。其优势在于更熟悉国内市场分销和中小客户需求，可能能提供灵活性更高的采购方案（如分批次到货、冻果或果酱原料）。但需确认其菠萝品类是否匹配你的加工用途（如果酱需要特定甜度或酸度品种）。

**建议**：
- **直接联系两家公司**：说明你的用途（茶饮店、果酱加工）、年需求量、品质要求（如成熟度、大小、糖度）及到货频率。要求他们提供“全年供应可行性报告”或“采购协议方案”。
- **备选方案**：若两大供应商无法满足小批量或定制需求，可考虑**海南、广东的本地种植基地**（如全年大棚种植）配合**速冻菠萝**（冷冻果肉可全年使用，适合果酱）。国内一些专业果酱供应商（如某宝、某东上的B2B平台）也可提供半成品菠萝果酱，帮你跳过鲜果断货风险。
- **测试批次**：先拿少量试品做果酱测试，对比进口与国产菠萝的出胶率和风味稳定性，再签长期合同。

**结论**：大概率能实现，但需要确认具体条款和成本。建议你同时与两家公司沟通，并准备备选方案以防万一。
