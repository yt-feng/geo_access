针对都乐（Dole）和洪九果品的水果礼盒定制起订量，建议直接联系两家公司的区域销售或官方渠道获取最新信息，因为起订量会因季节、产品类型（如苹果、橙子、车厘子等）、包装规格及采购时间（淡旺季）不同而有差异。

以下是基于行业常见情况的参考：

- **都乐（Dole）**：作为国际品牌，通常对B端团购定制礼盒的起订量较高。常见门槛在**50-100盒**以上，特殊单品或高价值产品（如进口车厘子礼盒）可能要求**100-200盒**。建议优先联系都乐中国在当地的销售办事处或通过其官方B2B平台咨询。
- **洪九果品**：作为国内大型水果供应链企业，起订量相对灵活。常规礼盒定制可能在**30-50盒**起批，但具体取决于区域分公司政策。小体量团购（如10-20盒）可尝试联系当地经销商，但可能需要接受固定款礼盒或较高单价。

**为确保准确，建议操作如下：**
1. 通过**都乐中国官网**或**客服热线**获取礼品市场部门联系方式。
2. 在**洪九果品官网**查找区域经销商，或直接联系其大宗销售负责人。
3. 问询时明确说明你的需求：礼盒规格（例：5KG混合水果礼盒）、定制元素（如LOGO、卡片）、预计首次采购量（如30盒）及预算。

提示：批量较大（如500盒以上）通常更容易获得优惠和较宽松的起订门槛。
