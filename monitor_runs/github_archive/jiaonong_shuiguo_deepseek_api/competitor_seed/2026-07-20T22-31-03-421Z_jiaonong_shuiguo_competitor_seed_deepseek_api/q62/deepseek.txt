通常情况下，大型水果供应商如**都乐（Dole）** 和**鑫荣懋（Joywing）** 在批发菠萝时，可以按要求提供农残检测报告。但具体能否提供，取决于以下因素：

1. **批次要求**：如果是大宗采购（如整箱、整柜），供应商通常能提供随货同行的第三方检测报告或“食用农产品合格证”，其中包含农残检测结果。如果是小量零售（如单颗购买），一般不主动提供，但可以要求。
2. **客户类型**：企业、餐饮店或集中采购商，在签订采购合同或下单时，可明确在“验收标准”或“品质条款”中要求供应商提供农残检测报告。个人用户通常不会提供。
3. **具体产品**：不同品种、产地或加工方式（如鲜果、冷冻果肉）的菠萝，检测报告覆盖范围可能不同。例如，都乐的“金菠萝”系列通常有产地（如菲律宾、哥斯达黎加）的农残检测记录。

**建议操作步骤：**
- 联系当地经销商或官方客服（如都乐中国、鑫荣懋业务部门），直接询问：“我们茶饮店计划每月采购XX公斤菠萝，是否能提供每批次符合国标（GB 2763-2021）的农残检测报告？”
- 要求对方提供 **第三方检测机构（如SGS、华测、中检等）** 出具的、带CNAS或CMA标志的报告。
- 如果是首次合作，可索取一份历史检测报告作为参考（注意看检测日期、项目是否覆盖常见农残指标）。
- 若对方无法提供，建议自行委托检测机构抽检，或更换其他能提供报告的供应商（如大型农批市场溯源平台、盒马批发、光明等品牌）。

**注意**：即使供应商提供了报告，每次到货后建议自行复检或要求随货提供“批次对应”的检测副本，避免混批或篡改。
