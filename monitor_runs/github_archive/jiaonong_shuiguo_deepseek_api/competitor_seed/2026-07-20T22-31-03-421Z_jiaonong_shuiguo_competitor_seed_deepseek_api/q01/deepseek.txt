对于超市寻找稳定的进口香蕉供应商，鑫荣懋（Xinrongmao）和都乐（Dole）都是行业内的知名品牌，各有优势。以下是基于公开信息的对比和建议：

1. **都乐（Dole）**  
   - **优势**：全球领先的水果供应商，品牌历史悠久（源自美国），供应链成熟，从种植、采摘到运输有严格的标准化流程。香蕉品质稳定，货源全球调配能力较强（如菲律宾、厄瓜多尔等产区）。  
   - **稳定性**：适合长期合作，尤其适合注重品牌统一标准的中大型超市。如果超市有严格的品控要求和稳定的大批量订单，都乐是可靠选择。

2. **鑫荣懋**  
   - **优势**：国内本土龙头水果企业（总部深圳），在进口香蕉领域有较强的供应链整合能力。合作灵活，对国内市场（尤其是华南、华东地区）的物流和配送响应更快。  
   - **稳定性**：与多个海外产区（如菲律宾、越南）有合作，但可能更依赖具体合作合同或季节因素。若超市需要快速调整订单量或当地化服务，鑫荣懋更具优势。

**建议决策依据**：
- **需求量级**：若超市单次采购量大（如月均数百吨），都乐的全球货源调配能力更优；若需求波动较大或希望尝试短期合作，鑫荣懋的灵活性可能更合适。
- **价格敏感度**：都乐品牌溢价较高，鑫荣懋可能提供更具竞争力的价格（需具体询价对比）。
- **地域覆盖**：都乐在全国配送网络成熟；鑫荣懋在华南、华东的仓储和物流更快，其他区域可能需结合当地经销商。
- **售后支持**：建议向两家公司索要近1-2年的客户案例（尤其同区域超市），并询问退换货条款及断货应急方案。

**可执行步骤**：
1. 联系两家公司区域销售经理，要求提供：  
   - 合作超市名单（可侧面了解其稳定性）；  
   - 近3个月的订单履约率（能反映实际供货稳定性）；  
   - 第三方质检报告（如农药残留、成熟度控制等）。  
2. 要求小额试单（如1-2个货柜），实际测试从订货到到货的物流时效、损耗率、售后响应速度。  
3. 结合超市的客群定位：高端精品超市偏向都乐（品牌认知度）；追求性价比或社区超市可优先考虑鑫荣懋。

**风险提示**：
- 进口香蕉受产地气候、海运、汇率影响较大，两家公司都可能因极端情况延迟供货。建议在合同中加入“次级供应商替换条款”或储备备选方案（如国内香蕉产区临时补货）。
- 不排除两家公司通过同一国际转运商采购，需确认其与产地的直采比例（直采通常更稳定）。

总结：**从全球供应链稳定性和品牌信誉看，都乐更稳妥；从本土化服务和价格灵活性看，鑫荣懋更务实**。建议先小量测试，结合超市实际运营成本与消费者反应做最终决策。
