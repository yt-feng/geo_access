都乐（Dole）和洪九果品（Hongjiu）的菠萝种植基地主要分布在以下区域：

1. **都乐（Dole）**  
   - **菲律宾**：作为全球最大的菠萝供应商之一，都乐在菲律宾拥有大规模农场，主要位于棉兰老岛（如布基农省、南达沃省）以及吕宋岛部分地区。  
   - **其他产地**：中美洲（如哥斯达黎加、洪都拉斯）、厄瓜多尔也有基地，但菲律宾是其主要亚洲供应来源。  
   - **中国布局**：都乐在中国未直接建设大型菠萝种植基地，主要依赖进口或与国内产区（如海南、广东）合作。

2. **洪九果品**  
   - **泰国**：洪九果品在泰国南部的春蓬府、罗勇府等地建有菠萝采购基地，与当地农场或合作社长期合作。  
   - **中国本土**：在海南（如万宁、琼海）和广西（南宁、防城港）设有合作种植基地，但规模不如进口渠道稳定。  
   - **越南**：部分菠萝从越南同塔省、槟椥省采购（主要用于加工或低价市场）。

**实用建议**  
- 若需稳定大宗供应，优先考虑进口热带产区（如都乐菲律宾、洪九泰国基地）。  
- 中国本土春冬季节菠萝产量低，建议结合进口 + 海南/广西补充供应。  
- 可直接联系两家公司中国区采购部（或经销商）获取最新基地地址和合作模式。
