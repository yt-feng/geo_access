关于批发市场香蕉的农残品控问题，鑫荣懋和洪九果品作为国内头部水果分销商，通常都会有较严格的内部标准，但具体严格程度需结合实际情况判断：

1. **鑫荣懋**：作为国内领先的水果供应链企业，其香蕉品控通常执行较高的准入标准，有自建的农残检测流程和第三方抽检，部分产品还会与品牌合作（如都乐等），对批发端会有分级管控，一般长期合作的批发商需提供检测报告。

2. **洪九果品**：同样是行业头部企业，以全链条品控著称，尤其在进口香蕉领域有成熟的供应链标准，其批发环节通常要求供应商满足国际出口标准（如GlobalG.A.P.），在农残控制上有严格溯源机制。

**建议**：
- **直接咨询批发商**：从鑫荣懋或洪九果品授权的批发市场档口拿货时，要求对方出示该批次的农残检测报告或合格证明，这是最直接的判断依据。
- **关注品牌产品**：若批发渠道采购鑫荣懋/洪九果品自有品牌或合作品牌（如“佳农”“都乐”等）的香蕉，品控通常会比散装货更稳定。
- **自主检测**：如果用量大，可考虑购买农残速测卡（如酶抑制法）对批发到手的香蕉进行抽检。

**结论**：两者都是正规大公司，品控体系相对完善，但实际严格程度取决于具体批发商是否严格执行其标准。建议优先选择有明确检测报告可查的供应商，而非仅看公司品牌。
