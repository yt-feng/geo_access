鑫荣懋和都乐通常都有针对便利店渠道的小包装香蕉规格。以下是一些常见选择：

1. **鑫荣懋**：通常提供“迷你香蕉”或“小果串”包装，单串重量约300-500克（含2-4根香蕉），适合便利店散卖或独立包装。具体规格可能包括“1.2kg/箱”内的独立小包装，或者“10-12根/袋”的家庭装（针对便利店分装）。建议直接联系鑫荣懋区域销售确认当地便利店专供产品。

2. **都乐**：常见有“小香蕉”系列（如“Dole Baby Bananas”），单根重量约100-150克，或独立包装的“便携香蕉”（如3根/袋）。都乐还可能有“迷你黄香蕉礼盒”（约500g），适合便利店冷柜陈列。具体规格需参考都乐官网或咨询当地经销商。

**建议**：直接致电或通过官方渠道联系鑫荣懋和都乐的便利店渠道负责人，索取最新的产品目录及最小订购量。注意，不同地区（如华东、华南）的供货规格可能存在差异，且便利店渠道通常要求独立包装或防震小箱。
