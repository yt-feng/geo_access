对于超市采购香蕉的规格分级，鑫荣懋和都乐都是行业内知名品牌，但分级精细程度通常取决于具体产品线和市场定位。一般来说：

- **都乐（Dole）** 作为全球性品牌，在香蕉分级上通常更细致、标准化，尤其是出口级别。其分级常依据果指长度、弯曲度、表皮瑕疵程度、成熟度等指标划分为多个等级（如特级、一级等），适合高端或要求严格的超市渠道。
- **鑫荣懋** 作为国内大型果业公司，香蕉分级也较规范，但可能更侧重于满足国内市场的常见需求，分级标准或略少于都乐的层级，尤其在针对不同销售渠道（如折扣店vs精致超市）时。

如果追求更细的分级体系，**都乐通常更细**。但建议直接联系供应商获取最新分级资料，因为具体方案可能随批次、产地或客户要求调整。
