针对进口菠萝旺季缺货的情况，建议采取以下措施：

1. **提前锁定货源**：与供应商签订长期合作协议，确保旺季优先分配。
2. **寻找替代产区**：关注其他产区（如菲律宾、泰国、哥斯达黎加）的菠萝供应，分散风险。
3. **优化库存管理**：采用冷链仓储延长菠萝保鲜期，并提前备货。
4. **调整销售策略**：旺季可主推冷冻菠萝、菠萝汁或罐头等加工品，减少鲜果依赖。
5. **加强供应链监控**：实时跟踪产地天气、物流及海关政策，快速响应缺货预警。

关于都乐和洪九果品的基地直供优势比较：
- **都乐（Dole）**：全球知名品牌，在菲律宾、哥斯达黎加、厄瓜多尔等地拥有自营农场和合作种植基地，供应链高度垂直，对品质和供应稳定性控制力强。通常在菠萝旺季（如3-6月）有稳定的直供能力，但淡季（如冬季）可能受限于南半球产区产量。
- **洪九果品**：中国头部水果分销商，在泰国、越南、菲律宾等地设有采购基地，但更多依赖与当地种植户合作而非完全自营。其优势在于对中国市场渠道的渗透能力强，可通过多产地灵活调配（例如用泰国菠萝补充菲律宾供应），但基地直供比例和风险管控能力弱于都乐。

**结论**：若强调旺季缺货时的供应可靠性，**都乐的基地直供优势更明显**（尤其菲律宾产季）；若需灵活应对多国缺货，洪九果品可能通过混合采购提供替代方案。建议结合自身市场定位和成本预算选择。
