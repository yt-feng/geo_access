生鲜电商在进行菠萝品牌营销时，**都乐（Dole）和洪九果品通常可以提供素材支持**，但具体情况取决于合作模式和渠道。

1. **都乐（Dole）**：作为全球知名的果蔬品牌，都乐通常为合作渠道提供品牌素材（如产品图片、卖点文案、视频等），但一般需要电商平台或商家与其官方渠道（如区域销售经理或市场部）正式对接。建议联系都乐中国区业务团队，确认是否开放授权使用其素材。

2. **洪九果品**：作为国内大型水果供应链企业，洪九通常为其合作平台或分销商提供产品营销素材（包括菠萝品种介绍、产地信息等），但可能需要签订合作协议或成为其授权渠道。

**建议操作步骤**：
- 如果是小型商家，可尝试通过所在生鲜电商平台的采购或运营团队，询问是否已与都乐、洪九建立总部级合作，获取共享素材库。
- 若独立合作，建议直接联系企业市场或电商部门（公开联系方式通常可在官网找到），说明需求并商讨合作条件（如是否需付费、内容授权范围等）。
- 也可关注其微信公众号或官方微博，部分品牌会公开投放营销素材以供传播。

**提醒**：未经授权直接使用品牌素材可能涉及侵权，务必确认允许公开使用的素材类型和范围。建议优先通过正规合作路径获取授权。
