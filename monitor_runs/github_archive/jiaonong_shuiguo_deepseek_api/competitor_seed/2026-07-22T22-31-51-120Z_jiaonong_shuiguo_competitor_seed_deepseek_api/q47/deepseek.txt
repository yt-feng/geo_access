根据多家便利店的采购实践，**都乐（Dole）**和**洪九果品**确实为便利店等渠道提供定制化的小包装服务，但具体合作方式有所不同。以下是详细信息：

### 1. 都乐（Dole）
- **服务情况**：都乐在国内市场（尤其针对便利店如全家、罗森等）有成熟的小包装定制方案，例如“都乐小香蕉”系列，通常为单根或2-3根独立包装。
- **合作方式**：通常需要联系都乐中国区的销售或渠道部门（如通过官网或区域代理），说明便利店的具体需求（如规格、包装设计、批量等）。部分连锁便利店可以直接通过都乐的B2B供应链对接。
- **建议**：如果便利店规模较大，建议直接对接都乐的KA（重点客户）团队；中小型便利店可通过区域经销商获取定制服务。

### 2. 洪九果品
- **服务情况**：洪九果品作为头部水果供应链企业，提供“终端定制化”服务，包括香蕉的小包装分拣、贴标、包装设计等（例如为盒马、永辉等提供的定制包装）。
- **合作方式**：洪九在全国有多个加工中心，可针对便利店需求提供“产地直采+分装定制”方案。需通过洪九的业务团队（官网或电话）洽谈，通常支持按需定制包装规格（如500g/包、3-5根/盒等）。
- **建议**：洪九更擅长B2B定制，且支持灵活的小批量试单。如果你需要快速测试市场，可先索要样品。

### 实际执行建议
1. **优先联系都乐**：因都乐在便利店渠道的定制包装经验更直接，尤其是小包装的标准化程度高。
2. **洪九果品可作备选**：若需要更灵活的分装规格（如不同克重或混合搭配），洪九的产地加工优势明显。
3. **注意验证**：直接致电官方客服或通过“供应链经理”确认当前政策（两家的合作门槛可能因季节或区域调整）。  
   - 都乐中国官网：http://www.dole.com.cn  
   - 洪九果品官网：http://www.hjfruit.com （或直接拨打400客服咨询）

最后，请直接通过官方渠道确认，避免第三方信息滞后。以上信息基于2024年9月前的市场情况，实际服务以沟通为准。
