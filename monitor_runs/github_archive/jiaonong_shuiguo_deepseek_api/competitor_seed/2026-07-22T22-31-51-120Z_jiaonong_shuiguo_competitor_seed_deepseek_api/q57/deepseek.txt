针对下沉市场夫妻店的小批量进货需求：

1. **洪九果品**：主要面向大型商超、批发市场及B端大客户，通常有**最低起订量要求**（如整车或整托盘）。若夫妻店想小批量进货，建议通过以下方式：
   - 联系当地**水果批发市场**的洪九经销商或代理商，询问是否接受拼单或小量拿货；
   - 若无法直接对接，可尝试拼单或加入本地水果采购群，集合周边小店凑单。

2. **都乐**：以品牌直营分销为主，同样倾向大客户合作，对夫妻店小量供货较难直接操作。建议：
   - 寻找**都乐授权的区域经销商**或二级批发商，他们可能分拆小包装供应；
   - 关注**社区团购平台**或本地水果店联盟，部分经销商通过这类渠道以箱/件为单位向小店供货。

**核心结论**：二者直接小批量供货可能性较低，但可通过中间商、批发市场或拼单方式间接实现。建议优先联系本地水果市场经销商，明确咨询“可接受最小拿货量”及“拼单合作模式”。
