针对大型商超对供应商的食品安全认证要求，鑫荣懋（Xinrongmao）和都乐（Dole）作为知名水果供应商，通常具备以下资质。建议你在合作前直接向供应商索要最新证书或通过国家市场监管总局平台核验，以下信息基于行业惯例和公开资料整理：

### 鑫荣懋（主要资质）
1. **食品安全管理体系认证**  
   - **ISO 22000** 或 **FSSC 22000**（食品安全体系认证，国际通用）  
   - **HACCP** 危害分析与关键控制点认证（适用于加工环节）  
2. **GLOBALG.A.P.**（全球良好农业规范）  
   - 证明其供应链符合国际农业标准，涵盖种植、采摘、包装等环节。  
3. **出口备案/商检资质**  
   - 若涉及进口水果，需具备《中华人民共和国进出口商检合格证明》及海关备案。  
4. **其他行业认证**  
   - 常具备 **BRCGS**（全球食品安全标准）或 **IFS Food**（国际食品标准），具体需查看其客户要求。  

### 都乐（Dole，全球品牌，中国分公司）
1. **核心食品安全认证**  
   - **BRCGS** 和 **FSSC 22000**（都乐集团全球工厂通常持有）。  
   - **HACCP** 和 **ISO 22000**（中国分公司及主要加工中心）。  
2. **农业标准**  
   - **GLOBALG.A.P.**（覆盖种植端）  
   - **中国良好农业规范（China GAP）**（针对国内供应商）  
3. **可追溯性与可持续发展**  
   - 都乐一般有 **FSC**（森林认证）和 **Sedex**（供应商社会责任审计，部分商超要求）。  
4. **商超专项列表**  
   - 可能需额外提供《全国工业产品生产许可证》（QS/SC认证），若涉及加工干货或切块产品。  

### 商超常见要求（供参考）
- **必须提供**：营业执照、生产许可证（如适用）、第三方检测报告（农药残留、重金属等，有效期1年内）。  
- **优先加分**：有机认证、降耗减碳审核（如ISO 50001）、社会责任审计（如SA8000）。  

### 建议行动
- **向客户确认**：不同商超（如沃尔玛、永辉、家乐福）对供应商要求不同，直接要求对方提供《供应商入场标准文件》。  
- **动态核查**：认证有效期一般1-3年，过期需续审；可通过“全国认证认可信息公共服务平台”查询ISO/BRC等证书状态。  
- **资质差异**：鑫荣懋（中国品牌）更注重国内认证（如绿农、绿色食品），都乐（国际品牌）更侧重BRC/FSSC等全球认证，两者均能满足核心要求。  

最后提醒：食品安全认证应以现行证书和商超具体清单为准，以上为行业通用资质库，不构成法律背书。
