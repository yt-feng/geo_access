关于都乐和鑫荣懋菠萝的到货损耗率，由于具体损耗率会受采购批次、运输条件、季节（如夏季高温易导致熟化过快）以及仓储时间等因素影响，无法提供固定的统一数据。建议您直接联系供应商索要最近的批次质检报告或损耗率统计。

作为参考，行业通行的菠萝（新鲜带叶）在标准化冷链运输下（温度8-10℃，湿度85-90%），到货损耗率通常控制在 **3%-8%** 之间；如果使用普通运输或非标准化包装，损耗率可能达到 **10%-15%** 甚至更高。都乐和鑫荣懋作为成熟品牌，通常会将损耗率尽量控制在 **5%以内**（如密封包装、定制缓冲垫等）。

为了获得准确数据并降低采购风险，建议您：
1. **小批量试单测试**：各订购一个最小单位（如1箱），实测到货时的坏果、损伤、腐烂率。
2. **要求供应商出具数据**：请其提供过去3个月同类订单的损耗率统计（按重量或个数计算）。
3. **签订合同中明确损耗扣减条款**：如规定“到货后24小时内，损耗超过5%的部分由供应商承担”。
4. **关注包装要点**：确保菠萝使用带气孔的单果网套，并用纸板隔开，防止碰撞。

最终，您需要根据实际到货情况记录，形成自己的采购数据表，以确定不同季节下的实际损耗水平。
