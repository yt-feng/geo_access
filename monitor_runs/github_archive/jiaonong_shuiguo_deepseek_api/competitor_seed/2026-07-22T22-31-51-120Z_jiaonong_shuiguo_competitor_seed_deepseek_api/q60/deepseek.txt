针对进口菠萝的高损耗问题，都乐（Dole）和鑫荣懋这类大型水果企业通常采用以下供应链关键控制点来降低到货损耗率，你可以参考这些做法进行自查或改进：

1. **产地源头严格分级**：
   - **采收熟度控制**：菠萝通常在**七成熟（果色微微转黄）** 时采收。过青（后熟困难且酸度高）或过熟（极易在运输中发酵腐烂）都会导致高损耗。
   - **分级筛选**：通过机械分选剔除非标果（畸形、虫眼、早期软腐），只保留标准大小且无外伤的果实。

2. **冷链与预冷处理**：
   - **强制预冷**：菠萝采摘后立即进入**差压预冷**或**真空预冷**，在24小时内将果心温度降至**8-10℃**（不能低于7℃，否则易冷害）。
   - **全程恒温**：运输及仓储温度严格维持在**8-12℃**，并避免温度剧烈波动（温度波动会导致果肉软化、褐变）。

3. **包装与物理防护**：
   - **缓冲包装**：使用**带孔瓦楞纸箱**（每规格12-15个果）加**珍珠棉隔层**，每颗果实包裹泡沫网套防止碰撞。
   - **防潮纸**：在箱底铺防潮吸水纸，吸收冷凝水，减少霉菌滋生。

4. **快速转运与动销**：
   - **催熟与上架时间压缩**：到港后快速经过熏香催熟（乙烯），严格控制催熟度（果皮转黄30%以内）。从到港到门店上架控制在**48小时内**。
   - **门店动态检查**：要求门店每日检查菠萝，将**变软、有汁液渗出**的果第一时间折扣出清，避免传染好果。

**针对性建议**：
- **重点核查到货温度**：如果你发现到货时果箱内壁有水珠或果蒂发黑，通常说明冷链中断。建议在合同里明确**“到货果心温度≤12℃”** 和**“果蒂切口新鲜无褐变”** 作为验收标准。
- **使用防霉剂**：部分大贸商会在菠萝切割处（果眼部位）喷洒微量专用防霉剂（如扑海因），但需符合进口残留标准。建议确认供应商是否使用。

你的高损耗问题是集中在**腐坏**（黑软、霉斑）还是**机械伤**（压坏、切口开裂）？这两类原因对应的控制点差异很大。
