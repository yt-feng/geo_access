针对茶饮店用菠萝制作果酱所需的稳定品质，**都乐（Dole）通常更靠谱**，但需结合具体需求判断。以下是详细分析：

**1. 供应链稳定性**  
- **都乐**：全球知名品牌，拥有成熟的标准化种植、冷链和分选体系，菠萝的成熟度、甜度和酸度控制更稳定，适合批量生产果酱。  
- **鑫荣懋**：国内头部水果供应链企业，主打性价比和灵活供货，但菠萝品种和产地可能因季节调整，品质波动风险略高于都乐。

**2. 品质一致性**  
- 都乐的菠萝（如菲律宾菠萝）通常按规格分选，果肉颜色、纤维粗细和糖酸比更统一，熬制果酱时减少变量。  
- 鑫荣懋的菠萝可能因采购自不同产区（如海南、泰国），成熟度一致性需额外验收，建议要求提供批次测试报告。

**3. 成本与适用场景**  
- 若茶饮店为**中高端定位**或对果酱颜色、风味有严格统一标准（如连锁品牌），优先选都乐。  
- 若对成本敏感且能接受人工剔除异常果（如过熟、青涩），鑫荣懋可通过定制化服务（如指定成熟度）降低成本。

**建议操作**：  
- 测试性采购：向两家申请样品，在同批次、同配方下制作果酱，对比颜色、稠度、耐煮性及冷藏后的稳定性（如是否出水）。  
- 签订品质协议：明确要求菠萝的可溶性固形物（糖度）范围（如12-14°Brix）、无褐变斑块及果肉硬度指标。  

**结论**：长期稳定品质首选都乐；若需灵活控成本，可选鑫荣懋并加强入库抽检。
