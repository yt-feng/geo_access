除了都乐（Dole），目前市场上还有以下品牌能够稳定供应菠萝，主要分为国产和进口两类：

1. **国产主流品牌**  
   - **徐闻菠萝**：广东湛江徐闻县是中国最大菠萝产区，当地有“愚公楼”“菠萝的海”等区域品牌，通过盒马、叮咚买菜及社区团购渠道供应。  
   - **海南菠萝**：海南本地品牌如“南国”“热带阳光”，在沃尔玛、永辉及线上京东自营有售，供应期通常为3-6月。  
   - **云南菠萝**：西双版纳等地的“高山菠萝”或小菠萝（如“甜蜜蜜”品种），部分通过拼多多、直播电商销售，全年供应量相对小但稳定。

2. **进口替代品牌**  
   - **金菠萝（Del Monte）**：全球第二大新鲜果蔬供应商，品质与都乐相当，国内通过山姆、Costco及部分精品超市（Ole'、盒马鲜生）销售。  
   - **超甜菠萝（Sweet Honey）**：以菲律宾产为主，在盒马、每日优鲜等平台常见，特点是糖度高、纤维少。  
   - **非转基因品牌**：如“超级菠萝”（Super Pineapple）或“黄金菠萝”（Golden Pineapple），在京东国际、天猫进口超市有售，价格通常较都乐低10%-20%。

3. **稳定供应建议**  
   - **季节性注意**：国产菠萝旺季在3-6月，进口菠萝（如菲律宾、泰国）全年供应，但冬季价格偏高。可优先选择菲律宾产地的金菠萝或超甜菠萝。  
   - **采购渠道**：大型商超（永辉、大润发）的国产菠萝到货较规律；线上生鲜平台（美团优选、多多买菜）的徐闻菠萝性价比高，但需注意物流时效。  
   - **避免小品牌**：部分区域小品牌（如某些台湾凤梨仿冒品）可能断供，认准“出口备案基地”或“绿色食品”认证的产地。

如果追求持续稳定供应，建议同时与徐闻菠萝产地合作社（如“广东菠萝协会”）及进口商（如鑫荣懋、佳沃）建立长期合作**。
