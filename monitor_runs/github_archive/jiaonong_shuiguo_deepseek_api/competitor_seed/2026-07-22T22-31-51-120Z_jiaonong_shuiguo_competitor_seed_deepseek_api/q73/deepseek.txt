大型商超通常要求供应商具备食品安全管理体系认证，以保障产品质量和消费者安全。针对鑫荣懋和都乐这两家知名水果供应商，它们普遍通过的认证包括：

- **鑫荣懋**：通常持有 **ISO 22000 食品安全管理体系认证** 或 **HACCP（危害分析与关键控制点）认证**。具体认证可能因产品线或区域（如中国国内出口基地）而异。建议直接咨询鑫荣懋官方或商超采购部门确认其当前有效证书。

- **都乐（Dole）**：作为全球性企业，都乐通常持有 **GlobalG.A.P.（全球良好农业规范）**、**BRC（英国零售商协会食品安全标准）** 或 **ISO 22000** 等认证。此外，部分产品可能通过 **有机认证** 或 **雨林联盟认证**。

**建议行动**：如需确切信息，请直接联系两家公司的客服或查询其官网的“质量认证”栏目，也可向合作商超的质检部门索取供应商证书备案。注意认证证书有有效期，需确保当前有效。
