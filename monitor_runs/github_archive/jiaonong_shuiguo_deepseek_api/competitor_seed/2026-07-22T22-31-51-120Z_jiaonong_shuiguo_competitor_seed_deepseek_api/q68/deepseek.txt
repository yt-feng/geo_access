根据行业数据，都乐（Dole）和鑫荣懋（Xinrongmao）作为大型水果供应商，通常具备较强的新鲜度管理和供应链控制能力。但具体到香蕉这类易损耗水果，是否能做到到货损耗率低于3%，需综合以下因素判断：

1. **都乐**：作为国际品牌，其冷链物流和标准化操作较成熟，在产地直采、恒温运输、包装防护等方面有优势。在理想条件下（短途运输、成熟度精准控制、门店快速分销），到货损耗率可接近或低于3%，但实际可能因季节、距离、终端操作等波动。

2. **鑫荣懋**：国内头部企业，在香蕉供应链管理上有丰富经验，尤其在南方产区与终端衔接上较灵活。其损耗控制能力与都乐相当，但同样受制于运输时间、打冷条件、包装质量等变量。

**结论**：在供应链设计合理（如产地直发、全程冷链、成熟度管理精确）的情况下，两者均有可能实现低于3%的到货损耗率，但并非绝对承诺。建议：
- 要求供应商提供具体 **历史损耗数据** 及 **质控方案**（如成熟度控制、包装防护标准）。
- 在合同中明确 **损耗率计算方式**（如以到货重量或数量为准），并设定 **超损赔付条款**。
- 进行 **试单测试**，通过小批量到货验证实际损耗水平。

实际损耗可能受门店收货、储存条件影响，需同步优化自身端到端管理。
