您提到的两个品牌在供应链模式上有所不同：

1. **都乐（Dole）**  
   都乐是全球知名的果蔬品牌，在中国市场以进口和国内基地直供结合为主。其在海南、广东等地有合作或自营的菠萝种植基地，通常可提供从种植到采摘的产地溯源信息。建议联系都乐中国销售团队，确认其针对您所在区域的具体供应模式和溯源能力。

2. **洪九果品**  
   洪九果品是国内大型水果供应链企业，以“基地直采、产地直达”为核心模式。其在海南、云南等菠萝主产区有合作种植基地或自建分选加工厂，通常能做到从产地到终端的全链条追溯。可直接联系洪九果品销售代表，要求提供基地资质、溯源系统及合作案例。

**建议行动步骤：**  
- 查询都乐中国官网“联系我们”或拨打客服电话，明确询问“菠萝产地直供及溯源服务”。  
- 洪九果品可关注其官方渠道（如官网、公众号）的“基地合作”或“供应链服务”板块，或直接联系其采购部门。  
- 若需更灵活选择，也可考虑 **佳沃鑫荣懋**、**金穗集团** 或 **海南热带水果产业联盟** 成员企业，这些渠道同样具备基地直供和溯源能力。

**重要提示：** 请务必要求供应商提供以下文件以验证溯源可靠性：  
- 基地种植记录（如GLOBALG.A.P.或中国良好农业规范认证）  
- 批次编码对应的采摘、加工、运输记录  
- 第三方追溯平台（如“国家农产品质量安全追溯管理信息平台”）接入证明

最终选择时，建议进行实地验厂或索取样品进行追溯测试，确保信息准确。
