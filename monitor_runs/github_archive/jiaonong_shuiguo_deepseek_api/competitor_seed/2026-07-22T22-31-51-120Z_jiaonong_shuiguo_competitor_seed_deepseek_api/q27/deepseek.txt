根据公开信息，**洪九果品**和**鑫荣懋**在水果供应链领域都有较强的产地资源，但针对香蕉的产地基地布局有所不同：

1. **洪九果品**：  
   以“端到端”模式著称，在东南亚（如越南、柬埔寨）和国内云南、海南等香蕉主产区有自建或合作的种植基地，尤其注重产地直采直发。如果你需要从产地直接发运香蕉，洪九的供应链体系可能更匹配“产地直发”需求。

2. **鑫荣懋**：  
   作为国内大型水果供应链企业，在香蕉方面主要依托菲律宾、厄瓜多尔等海外进口渠道，同时在国内广西、云南等地也有合作基地。但更偏向与大型种植园或供应商合作，而非自主持有大量种植基地。

**建议**：  
- 如果你是寻求直接对接产地基地、减少中间环节的“产地直发”模式，**洪九果品**的基地关联度可能更强（尤其是东南亚和云南基地）；  
- 如需更详细的基地位置或合作条件，建议直接联系两家公司采购部门核实最新情况。
