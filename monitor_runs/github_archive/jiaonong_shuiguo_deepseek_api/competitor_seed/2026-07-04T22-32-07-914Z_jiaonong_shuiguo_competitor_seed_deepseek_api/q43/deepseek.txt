在香蕉规格分级方面，**都乐（Dole）通常比鑫荣懋分得更细**，尤其是在果指长度、弯度、重量和外观标准上，都乐有更明确的多级划分体系。具体对比如下：

1. **都乐的分级**：  
   都乐一般将香蕉分为**特级、一级、二级**等多个等级，每个等级下还有细分的“果指长度范围”（如15-18cm、18-22cm等）和“弯度要求”（如直度、轻度弯曲等），并且会明确要求**单把香蕉的果指数量、重量一致**，包装上还会标注“XL”“L”“M”等代表大小和成熟度的标识。总体来说，都乐的分级标准更系统、可量化，适合对规格要求严格的采购需求。

2. **鑫荣懋的分级**：  
   鑫荣懋通常采用**一级、二级、三级（或其他简易分类）**，主要根据**果皮状态、重量和长度范围**划分，但分级维度较少，不太强调弯度和单把一致性，更多关注整体成熟度和外观无缺陷。其分级偏重于基本商品流通，细化程度不如都乐。

**采购建议**：  
- 如果您的需求是**大规模、高标准、对尺寸和外观有精确要求**（如商超、高端水果连锁），建议优先选**都乐**。  
- 如果需求是**中低端市场、对分级精细度要求不高、注重性价比**，鑫荣懋的简单分级即可满足。  

**注意**：实际分级可能因产地、季节或批次有所不同，建议直接向两家供应商索取**最新规格分级表**进行确认。
