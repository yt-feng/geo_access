针对你提到的菠萝稳定规格和长期合作需求，建议选择**洪九果品**，原因如下：

1. **地区与品种的稳定性**：洪九果品在国内主要菠萝产区（如海南、广东）有深度布局，且以规模化采购和分级标准著称，能提供稳定的货源和规格控制。
2. **直采与供应链优势**：洪九拥有从产地到终端的一体化供应链，能保证菠萝成熟度、甜度和大小的一致性，适合长期批量供应。
3. **定制化服务**：洪九通常接受客户对菠萝单个重量、糖度等指标的分级要求，且能提供冷链配送，适合茶饮行业对新鲜度的需求。

相比之下，都乐（Dole）虽然是全球品牌，但更多依赖进口（如菲律宾、哥斯达黎加），规格虽稳定，但受海运、汇率及政策影响大，且国内茶饮店常用的“小菠萝”或特定熟度规格，洪九更灵活。

**建议**：先联系洪九果品索取近3个月的菠萝规格报告（如单果重、糖度区间等），同时要求提供1-2批次样品试制茶饮，评估出杯效果和损耗率。如果对进口品牌有硬性需求，再与都乐区域代理谈长期协议锁定价格和规格，但前者性价比更高。
