关于洪九果品和都乐香蕉在批发市场的到货损耗率，目前没有公开的权威统计数据直接对比两者数据，因为损耗率受运输距离、天气、储存条件、批发市场处理效率等多种因素影响。但可以根据行业实际情况给出参考建议：

1. **都乐香蕉**：作为国际品牌，其供应链标准化程度较高，通常采用冷链运输、预冷处理和标准化包装，从产地到批发市场的损耗率一般控制在 **5%-10%** 左右（具体视物流和仓储条件）。  
2. **洪九果品**：作为国内头部水果供应链企业，其香蕉业务主要依靠产地直采和规模化分拣，但部分批发市场环节可能因非全程冷链或散装销售导致损耗偏高，行业估算其损耗率通常为 **8%-15%**，具体需结合到货时的成熟度及市场周转速度判断。

**建议**：  
- 若追求低损耗，优先选择都乐，尤其适合销售周期短、对品相要求高的场景。  
- 若考虑成本，洪九果品价格可能更低，但需现场检查到货香蕉的“青度”（建议选择7-8成熟果实，避免过熟或青涩）及是否有机械损伤。  
- **直接验证方法**：在批发市场现场随机抽查10箱，统计有擦伤、黑斑、腐烂的果体占比，再结合厂家提供的到货合格率文件综合判断。
