关于都乐和洪九果品在菠萝种植及供应基地的具体信息，目前公开资料中较难查到非常细致的产地分布。不过根据行业普遍情况和品牌公开信息，可以为你提供以下参考：

**都乐（Dole）菠萝基地：**
都乐是全球知名的果蔬供应商，其菠萝主要来源于南美和东南亚等热带地区。在中国市场，都乐有部分菠萝来自菲律宾、泰国等东南亚国家。如你在中国大陆采购，可重点关注都乐在**菲律宾**的种植基地（其是全球最大的菠萝出口国之一），或其在**广东、海南**等国内地区设立的合作种植农场。具体供应基地会随季节和市场调整，建议直接联系都乐中国销售团队获取最新产地信息。

**洪九果品（Hongjiu Fruit）菠萝基地：**
洪九果品是国内大型水果供应链企业，其菠萝主要依赖于**中国海南、广东**以及**泰国、越南**等东南亚国家。特别是其“洪九泰宝”等品牌菠萝，大多来自泰国南部或越南的长期合作种植基地。此外，洪九也在国内云南等地有少量试种基地。由于洪九以“产地直采”为主，建议通过其官方客服或采购部门了解最新签约基地及季节性供应计划。

**采购建议：**
1. **直接联系官方**：通过都乐中国官网或洪九果品官网、客服渠道获取当前合作的菠萝基地名单和供应能力。
2. **关注季节性**：国内菠萝主产季为2-5月（海南、广东），而泰国、菲律宾等可全年供应，但降雨期品质可能波动。
3. **补充备案**：建议同时联系国内其他大型菠萝产区（如徐闻、万宁）的合作社，作为备用供应源，避免过度依赖单一品牌。

请注意，以上信息基于公开渠道整理，具体基地位置和合作模式可能随企业战略调整而变化。如需验核，请以品牌方最新回复为准。
