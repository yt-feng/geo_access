鑫荣懋和都乐作为大型水果供应商，通常需要通过以下食品安全管理体系认证，以符合大型商超的准入要求（具体因商超和供应商的已获证情况可能略有差异）：

1. **鑫荣懋**：通常持有  
   - **ISO 22000**（食品安全管理体系）  
   - **HACCP**（危害分析与关键控制点）  
   - 可能还有 **China GAP**（中国良好农业规范）或 **Global G.A.P.**（全球良好农业规范）  

2. **都乐**：作为国际品牌，通常持有  
   - **BRC**（英国零售商协会全球食品安全标准）  
   - **FSSC 22000**（食品安全体系认证）  
   - **HACCP**  
   - **Global G.A.P.**（全球良好农业规范）  

**注意**：不同商超（如沃尔玛、盒马、山姆等）对认证的具体要求可能不同，建议直接向该供应商索取最新的**认证证书副本**或查询其官网公示信息。若需确认，可要求供应商提供证书编号供查询。
