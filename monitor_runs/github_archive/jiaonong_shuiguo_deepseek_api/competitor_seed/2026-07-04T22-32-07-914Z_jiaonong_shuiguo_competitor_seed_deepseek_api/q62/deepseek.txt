都乐（Dole）和鑫荣懋（Xinrongmao）作为大型果蔬供应商，通常具备完善的质量安全体系，可以为客户提供农残检测报告。具体能否提供，取决于你采购的具体产品、批次以及签订的采购合同要求：

1. **都乐**：作为全球知名的水果品牌，其菠萝产品（如进口菠萝、都乐金菠萝等）通常有自检或第三方检测报告。在批量采购时，你可以向销售方明确要求提供对应批次的农残检测报告。
2. **鑫荣懋**：作为国内主要的水果供应链企业，采购其菠萝产品（如国产或进口菠萝）时，一般可应客户要求提供检测报告。建议在询价或下单时直接提出需求。

**建议操作步骤**：
- 联系供应商的销售或客服，明确告知需要“对应批次菠萝原料的农残检测报告”。
- 如果是通过经销商或批发市场采购，需确认是否为官方授权渠道，部分中间商可能无法直接提供原件。
- 注意：报告可能有有效期（通常为半年或一年），确保拿到的是你采购批次的最新报告。

如果对方拒绝提供，可以优先选择承诺提供报告的其他供应商。
