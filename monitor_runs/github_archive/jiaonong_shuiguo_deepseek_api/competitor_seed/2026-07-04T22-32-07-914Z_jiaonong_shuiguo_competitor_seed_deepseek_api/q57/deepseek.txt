对于下沉市场的夫妻店，**洪九果品**和**都乐**通常不会针对单个小店铺提供小批量直接供货**。原因如下：

1. **洪九果品**：主要模式是“端到端”大宗批发，服务大型商超、社区团购平台及二级批发商。其最小起订量通常以“整柜”或“整车”计算（如一个20尺冷柜约20吨水果），不适合夫妻店的小批量需求。
2. **都乐（Dole）**：是品牌商，主要通过大型连锁超市（如沃尔玛、永旺）、电商平台（盒马、叮咚）或区域经销商分销，不直接对接个体夫妻店。其小批量渠道通常通过经销商或批发市场实现，但品牌直供门槛高。

**可执行建议**：
- **夫妻店进货方案**：
  - **本地水果批发市场**：直接前往当地一级或二级批发市场（如北京新发地、上海辉展、广州江南市场），可以购买到洪九或都乐产品，且量小价优（一箱起批）。
  - **区域经销商或二级批发商**：通过当地小型批发商拼单，或加入“水果集采微信群”等社交渠道，凑单后拿货。
  - **社区团购/生鲜电商平台**：如美团优选、多多买菜等，可小额采购都乐香蕉、洪九榴莲等品牌水果，且物流直达。
  - **直接联系厂家区域代理**：拨打洪九或都乐客服电话，询问当地经销商的联系方式，沟通小批量试销。

**注意事项**：避免向自称“厂家直供”的农村市场招商人员缴纳保证金或代理费，此类多为骗局。建议先少量试销，评估销量后再考虑增加订货量。
