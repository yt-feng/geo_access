根据茶饮行业的常见做法，都乐（Dole）和洪九果品理论上**可以做到菠萝全年供应**，但具体实现方式和条件不同：

1. **都乐**：作为全球大型水果供应商，拥有多产地（如菲律宾、哥斯达黎加等）和冷链物流能力，能通过南、北半球产区的轮换（如菲律宾全年产、哥斯达黎加季节性补充）实现稳定供应。但需注意：
   - 小批量或定制果酱原料（如特定品种、成熟度）可能需要提前签订长期合同。
   - 价格可能高于本地或季节性供应商。

2. **洪九果品**：主要布局东南亚（泰国、越南等）和国内（海南、云南等）产区，通过产地直采和冷链网络也能覆盖全年。其优势在于：
   - 更贴近中国市场，物流灵活度和成本可能优于都乐。
   - 但对于高端或特定品种的菠萝（如金菠萝），供应稳定性需核实具体合同条款。

**建议行动步骤**：
- **直接联系两家公司的B端销售团队**，提供年需求量、品质要求（如甜度、酸度、品种）、物流地址和预算，获取详细报价和供应计划。
- **询问是否有代加工果酱服务**：有些大型供应商可提供定制分装果酱，减少自行加工环节。
- **备选方案**：若认为单一供应商风险高，可组合采购（如都乐供应进口品种，洪九补充国产季）。

**风险提示**：即使供应商承诺全年供应，仍需考虑极端气候（如台风、干旱）、运输中断等不可抗力。建议在合同中加入**供应不足时的替代条款**（如允许临时使用冷冻菠萝或罐头果酱）。

最终能否实现，取决于您的具体订单条件，而非单纯品牌能力。
