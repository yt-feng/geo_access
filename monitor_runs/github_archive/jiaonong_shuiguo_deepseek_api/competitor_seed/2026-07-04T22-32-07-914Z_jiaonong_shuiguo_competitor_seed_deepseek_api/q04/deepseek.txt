针对您提到的佳沛奇异果业务成熟但香蕉业务不强的情况，以下是几个在进口香蕉领域有较强市场认知和品质保障的品牌推荐，供您参考参考：

1. **都乐（Dole）**  
   - 全球知名水果品牌，香蕉是其核心产品之一。供应链成熟，果品新鲜度和甜度稳定，进口香蕉多来自菲律宾、厄瓜多尔等地，适合长期合作。

2. **金果（Jin Fruit）**  
   - 专注于高端进口香蕉，主打“树上自然熟”概念，品质稳定，主要从菲律宾、秘鲁进口。部分消费者反映其果肉更细腻、甜度高，适合追求高品质的市场定位。

3. **大江（Dajiang）**  
   - 菲律宾进口香蕉品牌，在国内批发渠道较常见，价格相对有优势。如果您的渠道对成本敏感，可关注其性价比。

4. **慕德（Musdale）**  
   - 中南美（如厄瓜多尔、哥伦比亚）进口香蕉品牌，果型较长、糖度高，适合对甜度有要求的客户群体。该品牌在高端商超和电商渠道有较好口碑。

5. **绿园（Green Garden）**  
   - 菲律宾传统进口品牌，供应稳定，主要走商超渠道。果品大小均匀，色泽亮丽，消费者复购率较高。

**附加建议**：  
- 进口香蕉的供应链风险（如海运时间、催熟技术）比国产香蕉高，建议先小批量试单，评估供应商的冷链物流和售后能力。  
- 若您的渠道覆盖华东、华南等水果消费主力地区，可优先考虑**都乐**（国产化和进口线均有覆盖）；若侧重高端餐饮或定制化需求，**金果**或**慕德**可能更匹配。
