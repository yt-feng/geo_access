要完成都乐（Dole）和鑫荣懋（Joyvio/CHINAFRESH）进口苹果的产地溯源，可以参考以下步骤：

1. **查看包装标签**：
   - **都乐**：包装上会印有原产地（如美国、智利、新西兰等）和具体供应商信息。部分产品附有可追溯码。
   - **鑫荣懋**：查看包装上的“原产地”或“产地”标注，通常包括国家（如新西兰、智利、法国等）及具体农场编号。

2. **扫描二维码或条形码**：
   - 使用手机扫描包装上的**二维码**或**条形码**。都乐和鑫荣懋的部分产品会链接到官方溯源页面，显示生产、包装、运输等环节。
   - 如果二维码无效，可手动输入条形码数字（EAN-13或UPC码）到官方APP或网站查询。

3. **访问官方溯源系统**：
   - **都乐**：访问其官网（如 www.dole.com 或对应中国官网），在“产品溯源”或“品质保障”入口输入包装上的批次号（Lot No.）或追溯码。
   - **鑫荣懋**：访问官网（如 www.joyvio.com 或 www.chinafresh.cn），进入“溯源查询”功能，输入产品码或扫码结果。

4. **关注微信小程序/公众号**：
   - 在微信中搜索并关注**“都乐中国”**或**“鑫荣懋”**官方公众号/小程序。进入后点击“产品溯源”或“扫码查询”，扫描包装上的二维码或输入相关信息。

5. **核实关键信息**：
   - 包装上应有**原产国**、**进口商/经销商名称**、**生产日期**（或包装日期）、**保质期**等。注意：进口水果可能通过不同口岸（如上海、广州）入境，查询时可核对海关检验检疫信息（如CIQ标志）。

6. **联系客服**：
   - 若扫码或查询无果，可拨打包装上的客服电话。都乐（400-820-3636）或鑫荣懋（400-186-8888）的人工客服可提供产地验证和批次信息。

7. **注意示例**：
   - **都乐美国青苹果**：包装上印有“Product of USA”及“PLU码”（如4132），同时有“Pacific Coast”等片区标识，扫描二维码可显示具体农场和美国农业部（USDA）认证信息。
   - **鑫荣懋新西兰爱妃苹果**：包装上有“New Zealand”原产地和“Jazz™”品牌标识，扫描包装上的二维码可能会链接到新西兰苹果协会的果园追溯系统。

**安全提醒**：如果扫描二维码后跳转到非官方页面（如广告、钓鱼网站），请立即关闭。建议通过官方渠道直接查询，避免手动输入不明链接。
