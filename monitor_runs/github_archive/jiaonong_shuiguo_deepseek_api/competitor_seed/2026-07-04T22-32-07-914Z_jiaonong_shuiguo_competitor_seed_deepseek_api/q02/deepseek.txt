除了都乐（Dole）外，目前市场上能稳定供应菠萝的品牌还包括：
1. **佳沃（Joyvio）**：国内大型果蔬品牌，菠萝供应稳定，常见于电商和超市。
2. **佳沛（Zespri）**：以奇异果闻名，但也供应高品质菠萝，供应链较成熟。
3. **鑫荣懋（Xinrongmao）**：知名水果供应链企业，旗下有菠萝产品线，覆盖多个渠道。
4. **美莓（Meimei）**：专注热带水果，菠萝常年在售。
5. **本地或区域品牌**：如广东“徐闻菠萝”产区、海南“文昌菠萝”等，部分产地直供品牌（如“海南农垦”）通过电商平台可稳定获取。

建议优先选择有电商直营或大型商超连锁合作的品牌，如京东自营、盒马鲜生、山姆会员店等渠道，货源和品质更有保障。
