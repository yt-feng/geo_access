关于水果连锁店所需的香蕉分级标准，都乐（Dole）和洪九果品这类大型企业通常会有自己的内部企业标准，但这些标准的具体内容（如详细分级指标、文件编号等）一般不对外公开披露。因此，我无法直接提供这两家公司的企业标准文本。

不过，您可以参考以下途径获取权威的分级依据：

1. **行业推荐标准**：目前国内香蕉分级普遍参考 **GB/T 9827-2008《香蕉》** 国家推荐标准。该标准对香蕉的果指长度、果径、成熟度、果面缺陷等有明确的分级（如一级、二级、三级）。
2. **自有标准制定建议**：如果连锁店需要建立分级体系，建议以国家标准为基础，结合销售场景（如鲜食、加工）和终端顾客对表皮色泽、硬度、口感的需求，制定可量化的操作规范（例如：单果重量、果糖含量、无黑斑比例）。
3. **直接咨询渠道**：您可以通过都乐（中国）或洪九果品的**官方客服或业务采购部门**，询问其针对B端客户的香蕉供货分级标准。通常大型供应商会为长期合作客户提供品控手册。

由于我无法验证未公开的企业文件，建议优先采用国家标准作为统一依据，并联系供应商获取其具体分级要求，避免混淆。
