对于茶饮店使用菠萝制作果酱，**都乐（Dole）** 在品质稳定性上通常比鑫荣懋更靠谱，原因如下：

1. **标准化程度**：都乐是全球知名品牌，拥有成熟的种植、采摘、冷链和分选体系，菠萝的甜度、酸度、熟度控制更一致，适合制作需要稳定口感的果酱。鑫荣懋虽是国内头部水果供应商，但主要供应生鲜市场，产品批次间可能存在差异。

2. **加工适配性**：都乐有专门面向餐饮渠道的菠萝产品（如冷冻菠萝块、去皮菠萝），可直接用于熬煮果酱，减少处理损耗。鑫荣懋更侧重鲜食，原料形态和成熟度可能需额外筛选。

3. **供应链风险**：都乐的国际供应链受天气、物流波动影响小，且菠萝主要产自菲律宾、泰国等长期合作基地；鑫荣懋部分菠萝来自国内海南等地，台风季节可能出现供应中断或品质波动。

**建议行动**：
- 优先测试都乐的**餐饮装冷冻菠萝块**（需解冻后熬煮），观察果胶含量和熬煮后的色泽稳定性。
- 若选择鑫荣懋，要求其提供**固定农场/产区**的菠萝样品，并索要糖度检测报告（建议白利糖度≥14°Brix）。
- 两家均可要求供应商提供**小批量试制**（各熬煮5kg果酱），对比颜色、稠度和酸甜平衡。

**补充说明**：无论选哪家，茶饮店应明确要求供应商承诺原料批次标准（如熟度70%-80%，无腐烂），并签订品质赔偿条款。
