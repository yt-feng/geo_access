都乐（Dole）和洪九果品（Hongjiu Fruit）在水果供应链方面均有较强实力，针对菠萝的产地直供和溯源能力，情况如下：

1. **都乐（Dole）**：都乐是全球知名的果蔬品牌，在中国市场有成熟的供应链体系。其菠萝产品通常来自菲律宾、哥斯达黎加等自有或合作农场，具备完善的产地溯源系统。都乐在中国也设有加工和配送中心，能够提供从种植到销售的全链条追溯信息。如果你关注的是“基地直供”，都乐通常以自有农场或长期合作基地为主，可以满足直供需求。

2. **洪九果品（Hongjiu Fruit）**：洪九果品是国内领先的水果供应链企业，以“端到端”的供应链管理著称。在菠萝品类上，洪九有专门的源头直采基地，主要依托其在东南亚（如泰国、菲律宾）以及中国海南、广东等地的合作种植基地。洪九强调“基地直供”模式，能够提供产地溯源服务，但具体到菠萝品种（如香水菠萝、金菠萝等），建议直接联系其业务部门确认当前可溯源的基地范围。

**建议**：
- 如果你需要批量采购并确认溯源能力，可直接联系都乐或洪九果品的销售代表，要求提供具体的菠萝产地证明、农药使用记录、采摘到发货的物流编码等溯源文件。
- 如果是小批量或零售渠道，可通过都乐官方电商平台或洪九果品的合作商超（如盒马、永辉等）查看产品包装上的溯源二维码或标签。

**注意**：都乐和洪九均有能力做到基地直供，但具体到某个季节（如菠萝淡季）或特定品种，直供范围可能有所调整，建议以官方最新回复为准。
