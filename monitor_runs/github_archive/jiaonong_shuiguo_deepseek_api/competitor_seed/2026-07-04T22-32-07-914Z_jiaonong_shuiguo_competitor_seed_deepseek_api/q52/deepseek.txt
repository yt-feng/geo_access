关于菠萝原料的到货损耗率，通常取决于采购规格（如糖度、成熟度、包装方式）、供应链距离和季节性。针对都乐（Dole）和鑫荣懋（Joy Wing Mau）这两个供应商，行业经验数据如下（供你参考，具体需以实际批次和合同约定为准）：

**1. 都乐（Dole）菠萝：**
- **常见损耗率范围：** 约5%-10%（含包装及运输中的轻微机械损伤、自然腐烂）。
- **说明：** 都乐通常有严格的品控和冷链标准，如果采购的是标准化出口级菠萝（如整箱或切块原料），损耗率通常较低。建议要求供应商提供“新鲜度保证”和“到货验收标准”。

**2. 鑫荣懋（Joy Wing Mau）菠萝：**
- **常见损耗率范围：** 约8%-15%（鑫荣懋是国内分销商，可能涉及多环节中转或不同产地批次，损耗率波动稍大）。
- **说明：** 若采购其产地直供的菠萝（如海南、菲律宾等），建议明确到货成熟度，因为越熟损耗率越高。对冷藏运输、到货温度有明确协议可降低损耗。

**3. 更精准的操作建议：**
- **要求供应商提供历史数据：** 在采购合同中写明“允许损耗率（如≤10%）”，约定超出部分由供应商承担，或按比例折价。
- **实测：** 首次合作时，对每批到货抽检（例如开箱称重、剔除腐烂/损伤果），计算实际损耗率。持续跟踪3-5批后，双方均能锁定稳定预期。
- **季节差异：** 雨季/高温季节菠萝易发软、霉变，损耗率可能上浮2-5%。建议在合同中附加“季节性损耗补偿条款”。

**4. 降低损耗的常用措施（茶饮原料）：**
- 要求供应商按“茶饮行业专用标准”挑选菠萝（如糖度14-16、果肉硬度适中、无黑心）。
- 到货后立即分级处理：高损耗批次（如开箱即有10%以上烂果）可要求退换货。
- 考虑采购冷冻菠萝浆或NFC原浆替代鲜果（损耗率可降至1%以内，但需测试风味）。

**最终建议：**  
向都乐或鑫荣懋的销售员索要其“供应给茶饮连锁”的历史损耗数据（通常他们会提供书面参考）。如果没有，可参考上述范围作为谈判起点，重点要求**到货合同明确损耗上限**并留有检验时效条款（例如到货后2小时内抽样验收）。
