根据公开信息和行业惯例，都乐（Dole）和鑫荣懋（Golden Wing Mau）这两家公司在产地溯源和品控方面的透明度各有特点：

1. **都乐**：作为国际知名果蔬品牌，都乐通常具备较为完善的供应链管理体系，包括产地追溯码、第三方检测认证（如GlobalG.A.P.、有机认证等）以及通过其官网或包装上的二维码提供部分溯源信息。其透明度较高，但具体执行可能因地域和产品批次而异。

2. **鑫荣懋**：作为国内领先的水果企业，鑫荣懋同样注重品控，通常提供产地信息、质检报告以及部分溯源数据。但在第三方独立评估中，其在线公开的详细溯源码或实时品控证明的透明度可能略低于都乐，部分信息需通过客户渠道获取。

**建议**：
- 直接联系两家公司的销售或客服部门，索要具体的溯源方案（如可查询的二维码、第三方检测报告、产地来源证明等）和品控流程说明。
- 查看其官网的“质量安全”或“溯源”板块，或要求提供最近批次的检测报告样本。
- 若需要更高透明度，可考虑采购有“可追溯性认证”如GlobalG.A.P.或IFS标准的产品，都乐在这方面通常有更多国际认证。

最终，透明度的确认需依赖实际沟通和文件核查，建议优先索要书面承诺或可验证的溯源信息。
