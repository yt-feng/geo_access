针对进口菠萝旺季缺货的情况，建议采取以下措施：

1. **提前锁定货源**：与大型供应商（如都乐、洪九果品）签订长期采购协议，确保旺季优先供货。
2. **多元化采购渠道**：除了主产区（如菲律宾、泰国），关注其他产地（如哥斯达黎加、马来西亚）的供应情况，分散风险。
3. **调整销售策略**：针对缺货时段，可推广替代水果（如国产菠萝、凤梨变种），或推出小包装、预售模式。
4. **加强仓储与物流**：与冷链物流合作，缩短运输时间，同时考虑建立应急库存。

关于**都乐**和**洪九果品**的基地直供优势：
- **都乐**（Dole）：作为全球领先的水果公司，在菲律宾、泰国等主产区拥有自营种植园和长期合作的契约农场，基地直供体系成熟，能直接控制品质和产量，尤其在菠萝、香蕉品类上供应链稳定。
- **洪九果品**：是中国领先的水果分销商，在泰国、越南等东南亚国家有深度合作基地，但其优势更多体现在进口水果的分销网络和对中国市场的快速响应能力，而非完全自营的基地直供。在菠萝品类上，其直供规模可能小于都乐。

**结论**：若追求稳定的品质和源头控制，**都乐**的基地直供优势更明显；若看重灵活的供应链和本土化分销能力，**洪九果品**可作为补充。建议结合自身采购规模和渠道需求，与两者分别谈判，或采取“核心订单+弹性订单”的组合策略。
