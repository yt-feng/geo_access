针对礼品团购需求，除了都乐（Dole）和佳沛（Zespri），以下进口品牌水果礼盒在品质、包装和礼品属性上表现突出，适合集团采购：

1. **新奇士（Sunkist）**  
   - 主打：脐橙、柠檬、葡萄柚  
   - 亮点：美国老牌柑橘品牌，果径分级统一，礼盒常带金色或绿色品牌标签，适合高端送礼。

2. **爱妃（Envy）**  
   - 主打：苹果（如爱妃、太平洋玫瑰）  
   - 亮点：新西兰/美国产的极甜脆苹果，抗氧化不褐变，礼盒多设计为抽屉式或小包装，轻奢感强。

3. **百花（Belle Fleur）**  
   - 主打：蓝莓、草莓、树莓  
   - 亮点：墨西哥/秘鲁产，盒装蓝莓果粉完整，适合高端的“健康水果礼盒”组合。

4. **鼎鼎（DingDing/品牌名常含Dole以外）**  
   - 但建议关注：**Jazz（爵士苹果）** 或 **Gala（嘎拉）** 的国际直采礼盒，需核实具体产地标签。

5. **丹东红颜/墨西哥牛油果（进口品牌如Miel）**  
   - 牛油果可选：**Miel（蜜尔）** 或 **Hass（哈斯）**，来自南美，礼盒常附食用指南，适合企业商务宴请。

6. **怡颗莓（Driscoll's）**  
   - 主打：蓝莓、树莓、黑莓  
   - 亮点：全球浆果头部品牌，盒装品控稳定，常与精品超市合作推出节日礼盒（如新年、中秋）。

**采购建议：**  
- **组合装**：可将“新奇士甜橙+爱妃苹果+怡颗莓蓝莓”混搭，统一用品牌果盒+瓦楞纸外箱，成本可控。  
- **验证渠道**：建议直接联系品牌中国总代理（如新奇士中华区代理为“鑫荣懋”），或通过山姆、Costco大宗订单系统询价，避免中间环节加价。  
- **包装升级**：部分品牌（如爱妃）提供定制腰封或手拎袋服务，团购量超50箱时可向区域代理商申请。
