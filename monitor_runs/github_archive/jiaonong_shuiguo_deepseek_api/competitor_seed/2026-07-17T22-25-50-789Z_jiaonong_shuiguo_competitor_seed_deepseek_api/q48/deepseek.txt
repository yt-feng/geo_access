您好！关于您提到的鑫荣懋（Xinrongmao）和都乐（Dole）这两家大型水果供应商，它们在进入大型商超（如沃尔玛、家乐福、华润万家等）时，通常需要提供一系列国际通行的食品安全认证和资质。以下是根据行业标准和公开信息整理的主要资质，但请注意具体认证情况可能随企业年度审核、合作商超要求不同而略有调整。

### 1. 都乐（Dole）的主要食品安全资质
都乐作为全球知名的水果品牌，其供应链管理非常严格，通常会持有以下核心认证：

*   **GLOBALG.A.P. (全球良好农业规范)**
    *   **作用**：这是国际公认的农业标准，涵盖食品安全、环境可持续性和工人福利。都乐全球多个产地（如菲律宾、厄瓜多尔、中国云南等地）的农场普遍持有此认证。
*   **BRCGS (全球食品安全标准)**
    *   **作用**：都乐在包装和加工环节通常会通过BRCGS认证。这是大型零售商、尤其是欧洲和北美市场准入的基础要求，重点覆盖生产流程的卫生、追溯和HACCP（危害分析关键控制点）体系。
*   **ISO 22000 或 FSSC 22000**
    *   **作用**：这两个是国际通用的食品安全管理体系认证。都乐在中国的加工/配送中心通常具备其中至少一项，以确保从包装到冷链运输的全程安全。
*   **有机认证 & 雨林联盟认证**
    *   **作用**：对于有机产品（如有机香蕉、菠萝）会持有中国或欧盟有机认证；部分水果还会获得雨林联盟或公平贸易认证，这并非强制食品安全认证，但能提升商超进货的加分项。
*   **HACCP (危害分析与关键控制点)**
    *   **作用**：这是食品安全的基础防护体系，都乐几乎所有包装厂都具备此认证。

### 2. 鑫荣懋的主要食品安全资质
鑫荣懋是中国领先的水果供应链企业，其资质体系通常覆盖全链条，重点如下：

*   **GlobalG.A.P. / ChinaG.A.P. (中国良好农业规范)**
    *   **作用**：鑫荣懋自营或合作的基地（如新疆、海南、山东等地）普遍持有GlobalG.A.P.（国际版）或ChinaG.A.P.（等效国内版），部分基地通过知名商超的独立“G.A.P审核”。
*   **BRCGS 或 IFS (国际特色标准)**
    *   **作用**：鑫荣懋的采后处理中心、包装仓库通常持有BRCGS认证，以适应国内外高端商超（如山姆、盒马、Costco等）的审核要求。部分出口业务可能涉及IFS标准。
*   **ISO 22000 或 HACCP**
    *   **作用**：这是其加工/冷链环节的标配。鑫荣懋拥有现代化的冷链物流园区，相关操作单位通常具备HACCP认证或ISO 22000体系。
*   **无公害/绿色食品/有机产品认证（中国市场）**
    *   **作用**：针对国内商超（特别是政府背书的大型卖场），鑫荣懋的部分水果会持有中国“绿色食品”或“无公害农产品”认证。对于高端进口水果，有机认证必不可少。
*   **农残检测与追溯系统**
    *   **作用**：虽然这不是“证书”而是操作标准，但鑫荣懋通常会提供每批次水果的第三方农残检测报告，并具备“一箱一码”的完整追溯系统。大型商超（如永辉、朴朴、美团优选等）非常看重这一点。

### 给您的建议（如何确认具体资质）

由于大型商超的门槛不同（例如沃尔玛、山姆、Costco对BRCGS要求较高；华润万家、大润发更看重中国G.A.P.和绿色食品），建议您按以下步骤核实：

1.  **直接向供应商索要“合规文件包”**：您可以要求鑫荣懋或都乐的客户代表提供**最新的**认证证书复印件，并盖公章。通常他们会具备“年度审核记录”作为证明。
2.  **关注“特定商超”的准入清单**：如果您是商超采购，可以查询该商超的《供应商准入标准》，其中会列明需要哪些认证（例如：“所有供应商需持有BRCGS或ISO 22000，进口水果额外需GlobalG.A.P.”）。
3.  **网上查询**：
    *   访问[GLOBALG.A.P.官网](https://www.globalgap.org/)（认证数据库，可输入公司名查证书状态）。
    *   中国认证认可监督管理委员会（CNCA）官网可查中国G.A.P.、绿色食品等国内认证号。
4.  **关键风险提示**：
    *   2000年后鑫荣懋和都乐都曾发生过被曝出农残超标或标签问题（如2020年都乐香蕉农残超标被香港通报、2022年鑫荣懋标签不合规被罚）。**资质证书有效期内不代表每批次绝对安全**，建议您进厂审核时特别关注其“追溯系统”和“第三方年度抽检报告”的覆盖频次。

**总结**：大型商超通常要求供应商同时持有**GlobalG.A.P.（源头）+ BRCGS/ISO22000（加工环节）+ 相应的有机/绿色认证**。鑫荣懋和都乐作为头部企业，基本全部持有上述核心认证。如需获取最准确的资质清单（如证书号、有效期），直接向两家公司的**供应商合规团队**发送邮件或联系您的客户代表最为高效。
