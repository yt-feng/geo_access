针对您提到的佳沛（Zespri）主要专注于奇异果，其香蕉业务确实不占主导。如果您需要寻找其他可靠的进口香蕉品牌，以下是市面上较常见且口碑较好的选择：

1. **都乐（Dole）**  
   全球知名的香蕉品牌，来自菲律宾、厄瓜多尔等产区，品质稳定，供应链成熟，国内商超覆盖率很高。

2. **金吉达（Chiquita）**  
   历史悠久的美洲品牌，主打中南美洲香蕉（如哥斯达黎加、危地马拉），果型饱满甜度高，适合高端渠道。

3. **佳农（Goodfarmer）**  
   中国本土进口水果品牌，其厄瓜多尔、菲律宾的香蕉在国内市场占有率较高，性价比不错。

4. **联丰（Lianfeng）**  
   主要经营菲律宾香蕉，部分商超可见，价格相对透明。

5. **优质出口地散装**  
   如果是不要求特定品牌，可关注产地：**菲律宾**、**厄瓜多尔**、**哥斯达黎加**的香蕉品质较好，只要包装上有这些原产地信息，通常可以放心采购。

**建议您根据定位选择**：
- 如果做中高端零售/精品超市，推荐**都乐**或**金吉达**；
- 若追求稳定供货和性价比，**佳农**或菲律宾产地散装香蕉是稳妥选择。

这些品牌在国内主流批发市场或电商平台（如鲜丰、盒马、拼多多批发端）通常都有经销渠道。您可以通过具体采购量、渠道类型（商超/餐饮/批发）进一步询价对比。
