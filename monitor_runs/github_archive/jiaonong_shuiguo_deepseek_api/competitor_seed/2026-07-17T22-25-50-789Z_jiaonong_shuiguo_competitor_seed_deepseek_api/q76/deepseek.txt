都乐和鑫荣懋在生鲜电商的菠萝产地直发中，物流时效通常如下：

- **都乐**：作为全球知名品牌，其冷链物流体系较为完善，通常能在**下单后1-3天**内完成配送（视距离和电商平台政策），覆盖范围广，但需注意是否指定“产地直发”服务，部分渠道可能通过区域仓库中转。
- **鑫荣懋**：作为国内大型水果供应链企业，其产地直发时效与都乐相近，一般**1-3天**内发货并完成配送，尤其在南方主产区（如海南、广东）时效更快，部分偏远地区可能延长至3-5天。

**建议**：
1. 在电商平台（如盒马、京东、拼多多）下单前，直接查看商品详情页的“发货地”和“物流承诺”（如“24小时发货”、“次日达”等）。
2. 若急用，优先选择带有“产地直发”标识和明确时效保障的商家，或联系客服确认具体物流时间（尤其是促销期间可能延迟）。
3. 注意：两品牌的速递时效受季节、天气、区域和平台库存影响，实际以订单物流轨迹为准。
