要验证都乐（Dole）和鑫荣懋（Xinrongmao）进口苹果的产地溯源，可以按以下步骤操作：

1. **看包装标签**：正规进口苹果的包装箱或水果自带的贴纸上会标注“原产国”（如美国、智利、新西兰等）。都乐和鑫荣懋的进口苹果通常会在显眼位置印有产地信息。
2. **扫描二维码**：部分产品包装上有追溯二维码，用微信扫描后可查看产地产地、采摘日期、运输记录等。
3. **查官方渠道**：登录都乐或鑫荣懋的官方网站或微信公众号，输入产品批次号或条形码，可查询详细信息。
4. **看检验检疫证明**：超市货架旁或包装箱内可能有进口水果的《入境货物检验检疫证明》副本，上面会注明产地和入境日期。
5. **核对条形码**：进口苹果的条形码开头两位数字代表国家（如00-09是美国，94是新西兰），可扫码或手动查询验证。

如果包装信息不完整或无法扫描，可直接咨询超市生鲜负责人索要进货单据或追溯信息。注意，仅有中文标签而无外文原产国信息的可能为国产冒充。
