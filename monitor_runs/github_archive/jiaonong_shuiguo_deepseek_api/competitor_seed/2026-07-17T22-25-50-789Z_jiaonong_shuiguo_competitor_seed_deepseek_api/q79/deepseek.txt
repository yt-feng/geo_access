根据我的了解，都乐和洪九果品作为大型水果供应商，具备一定的供应链能力，但能否做到全年稳定供应菠萝，取决于以下几个关键因素：

1. **菠萝的产地与季节**：中国菠萝主产区在广东、海南、广西等地，自然成熟期集中在3-6月。要实现全年供应，通常需要依赖进口（如菲律宾、泰国等热带地区）或冷库储存反季节菠萝。

2. **都乐（Dole）**：都乐是全球性水果企业，业务覆盖北美、东南亚等产区，具有较强的进口和冷链分拨能力。理论上，它可以利用全球采购网络（如通过菲律宾、哥斯达黎加等地的菠萝基地）实现全年供应，但成本较高，且需提前签订长期合约。

3. **洪九果品**：洪九是国内水果供应链公司，在泰国、越南等东南亚国家有直采基地，能稳定供应热带水果。但菠萝的全年供应需依赖其海外基地的产季衔接能力，同时关注仓储和运输损耗控制。

4. **实际可行性**：两家公司都可能做到全年供应，但需要提前沟通具体需求（如品种、规格、供应量、价格区间、物流频率等）。小型茶饮店若用量不大，建议优先询问其“定制化供应方案”或与其他本地/进口分销商组合采购。

5. **替代方案**：如果全年供应不可行或成本过高，可考虑：
   - 在旺季制作冷冻菠萝果酱储存；
   - 使用冷冻菠萝果肉（如维果等品牌）替代鲜果；
   - 与多个小型供应商合作，在非产季用进口菠萝补充。

**结论**：都乐和洪九果品在资源上具备潜力，但需明确谈判具体供应能力。建议直接联系其销售部门，明确要求“全年菠萝原料供应”并提供预估用量，以获取准确报价和排期。
