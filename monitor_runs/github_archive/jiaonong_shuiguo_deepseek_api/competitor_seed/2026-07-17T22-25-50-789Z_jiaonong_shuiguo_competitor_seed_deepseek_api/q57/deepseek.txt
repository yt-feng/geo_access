针对下沉市场夫妻店小批量进货的需求，**洪九果品和都乐目前均没有针对这类客户的小批量直接供货服务**。具体原因如下：

- **洪九果品**：主要面向大型商超、批发市场和连锁水果店，其商业模式以B2B为主，通常需要达到一定起订量（如整箱或整车），且要求客户具备一定规模或资质，不倾向服务个体夫妻店。
- **都乐（Dole）**：作为国际品牌，在中国主要通过大型零售商（如沃尔玛、盒马）或一级经销商分销，对夫妻店的小订单接触极少，通常不直接对接小批量需求。

**可行替代建议：**
1. **本地农贸批发市场**：直接前往所在城市的大型水果批发市场，以现金或少量批量拿货，起订量灵活。
2. **区域性批发商或二批商**：联系当地水果批发商或经销商，他们通常能接受小批量（如几箱起批）配送，且价格更灵活。
3. **线上批发平台**：使用采源宝、1688等平台搜索水果，筛选“支持小单”的供应商（注意关注运费和保质期）。
4. **品牌官方渠道咨询**：直接联系洪九或都乐的当地经销商（可通过官网或客服获取联系方式），询问是否有零散拿货政策，部分非核心市场可能灵活妥协。

**行动建议：** 优先采用方案1或2，成本更优且可控；如需明确政策，可拨打洪九或都乐官方电话验证。
