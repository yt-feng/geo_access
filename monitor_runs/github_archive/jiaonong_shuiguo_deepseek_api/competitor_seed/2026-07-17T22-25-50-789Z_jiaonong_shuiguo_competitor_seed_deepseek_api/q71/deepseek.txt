针对菠萝的稳定供应需求，以下是都乐（Dole）和洪九果品（Hongjiu）菠萝种植基地的主要区域信息（基于公开资料和行业信息）：

1. **都乐（Dole）菠萝基地**：  
   - **主要区域**：都乐在全球（尤其在菲律宾、泰国、厄瓜多尔等地）有大规模自有菠萝种植园。在中国市场，其供应通常依赖进口，但在中国境内（如海南、广东、广西）也设有合作基地或加工中心，具体稳定供应点需与都乐中国官方确认（可通过其官网或联系销售代表获取最新基地名单）。  
   - **建议**：联系都乐中国区业务团队（电话或官网渠道），明确询问其国内合作种植基地及稳定供应计划。

2. **洪九果品菠萝基地**：  
   - **主要区域**：洪九果品在国内以进口及国产高端水果分销为主，其菠萝主要采购自海南（如万宁、琼海、乐东等地）的规模化种植基地，同时在云南（如西双版纳）也有合作产区。洪九果品有自有或长期合作基地，但具体地点属于商业信息，需直接与洪九果品采购或销售部门沟通（可通过其官网或400热线）。  
   - **建议**：致电洪九果品总部或其华南区域采购中心（例如在海南的办事机构），明确索要其菠萝种植合作基地列表。

**若急需当前稳定供应**，可考虑以下操作：  
- 联系两大公司的**区域销售经理**或**采购对接人**，要求提供年度供应协议及基地审核。  
- 根据行业经验，**海南菠萝（巴厘、金钻等品种）** 在3-6月主产期供应稳定，**泰国/菲律宾进口菠萝**（如洪九、都乐常进口）可全年供应但成本较高。  
- 备选方案：直接联系**海南/广东的菠萝种植合作社**（如海南万宁菠萝协会、湛江菠萝产地集散中心），此类渠道能提供更灵活的产地直采。  

**注意**：具体基地位置及合作细节需以企业提供的官方合同及现场考察为准，切勿轻信未经核实的非官方信息。
