对于社区团购的香蕉小包装需求，关于**都乐**和**洪九果品**是否有散装选项，以下是基于行业常识的建议：

1. **都乐**：通常主要供应标准化包装的香蕉产品，如整箱或预包装小包装（如4-6根装），**散装直供社区的选项较少**。其渠道更多面向商超、电商等中高端市场，散装批发可能需特殊定制或通过代理商协商。

2. **洪九果品**：作为大型果品供应链企业，更多提供**大宗批发及分拣加工服务**，较可能支持**散装或定制小包装**的香蕉产品，尤其适合社区团购这类需要灵活分装的需求。建议直接联系其当地业务部门，询问是否有散装提货、分装服务或可签署定制协议。

**建议行动**：
- **首选洪九果品**：直接联系其客户或采购部门，明确说明社区团购对规格、包装（如：每小包3-5根，无品牌包装）、频率及配送要求。
- **次选都乐**：可尝试联系其当地代理商或批发市场摊位，询问是否接受小批量散装提货（通常价格可能较高或需最低起订量）。
- **备选方案**：若以上均无合适散装选项，可考虑从当地大型水果批发市场（如中农批、果品批发市场）直接采购整箱香蕉，并由社区团长自行分装。

**注意**：具体情况可能因区域、时令及合作关系而异，建议直接电话询价和确认库存，以便获得准确报价和起订量要求。
