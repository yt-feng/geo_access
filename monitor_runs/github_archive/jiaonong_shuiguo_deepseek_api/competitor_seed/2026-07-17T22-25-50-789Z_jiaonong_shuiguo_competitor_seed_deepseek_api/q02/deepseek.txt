除了都乐（Dole），目前市场上能稳定供应菠萝的品牌还有：

- **佳农（Goodfarmer）**：国内常见品牌，主要供应菲律宾或国内产菠萝，全年基本有货，供应链较稳定。
- **金菠萝（MD2品种）**：一些产地直供品牌如“菲律宾进口金菠萝”或“海南金菠萝”常见于盒马、山姆等渠道，包装上可能标注“Golden Pineapple”或品牌名“Del Monte”（地扪）。
- **地扪（Del Monte）**：国际大品牌，供应菲律宾或哥斯达黎加菠萝，品质稳定，常见于高端超市和电商。
- **海南本土品牌**：如“海垦”“南国”“琼绿”等，季节性供应海南菠萝，春夏秋三季较多，冬季可能减少。
- **国内合作社或鲜果直供品牌**：如“鲜丰”“百果园”自有品牌菠萝，通常有稳定产地合作。

建议选择时注意查看包装上的产地和品种（如MD2为常见甜度高、酸度低的金菠萝品种）。如果能明确购买渠道（如线下超市、线上平台），我可以进一步推荐具体品牌。
